# RQ3.3 Early-Stopping Results

Structure:

- `output/<model_alias>/<task>/results.jsonl`: online early-stopping outputs.
- `output/<model_alias>/<task>/txt_output/`: per-task text artifacts when generated.
- `output/<model_alias>/<task>/code_output/`: extracted code artifacts for code-generation tasks.

Current model aliases: `qwen`, `deepseek`, `glm`.
