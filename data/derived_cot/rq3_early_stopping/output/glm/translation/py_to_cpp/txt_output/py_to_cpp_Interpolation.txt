    @staticmethod
    def interpolate_1d(x, y, x_interp):
        y_interp = []
        for xi in x_interp:
            for i in range(len(x) - 1):
                if x[i] <= xi <= x[i+1]:
                    yi = y[i] + (y[i+1] - y[i]) * (xi - x[i]) / (x[i+1] - x[i])

Key translation notes:

- **`zip` behavior**: Python's `zip(x_interp, y_interp)` iterates up to the shorter list. In C++, this is replicated with `std::min(x_interp.size(), y_interp.size())` and indexed access.
- **`range(len(x) - 1)`**: When `len(x)` is 0 or 1, this produces an empty range in Python. The C++ condition `i + 1 < x.size()` naturally handles this since `size_t` underflow is avoided (0+1 < 0 is false for unsigned).
- **Chained comparison** `x[i] <= xi <= x[i+1]` is expanded to `x[i] <= xi && xi <= x[i + 1]`.
- **Silent skip on no match**: If no interval contains `xi` (or `yi`), nothing is appended — matching the Python behavior where the loop simply doesn't `break` and no `append` occurs.
- **Floating-point division**: Using `double` throughout ensures float division matches Python's `/` operator.
