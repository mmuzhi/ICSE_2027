=== SOURCE CODE (java) ===
package org.example;

public class BankAccount {
    private int balance;

    public BankAccount(int balance) {
        this.balance = balance;
    }

    public BankAccount() {
        this(0);
    }

    public int deposit(int amount) {
        if (amount < 0) {
            throw new IllegalArgumentException("Invalid amount");
        }
        this.balance += amount;
        return this.balance;
    }

    public int withdraw(int amount) {
        if (amount < 0) {
            throw new IllegalArgumentException("Invalid amount");
        }
        if (amount > this.balance) {
            throw new IllegalArgumentException("Insufficient balance.");
        }
        this.balance -= amount;
        return this.balance;
    }

    public int viewBalance() {
        return this.balance;
    }

    public void transfer(BankAccount otherAccount, int amount) {
        this.withdraw(amount);
        otherAccount.deposit(amount);
    }

    public static void main(String[] args) {
        BankAccount account1 = new BankAccount();
        BankAccount account2 = new BankAccount();
        account1.deposit(1000);
        account1.transfer(account2, 300);
        System.out.println("account1.balance = " + account1.viewBalance());
        System.out.println("account2.balance = " + account2.viewBalance());
    }
}

- Java's `IllegalArgumentException` is an unchecked exception. In C++, we can throw `std::invalid_argument` (or `std::runtime_error`). Java's `IllegalArgumentException` is thrown for invalid amount (negative) and insufficient balance. We'll use `std::invalid_argument` for consistency, though we could also use `std::logic_error`. The behavior should be identical: terminate program if uncaught? In C++, if exception not caught, program terminates with message. Java also terminates with stack trace. So it's fine.

- Constructors: default constructor calls the parameterized constructor with 0. In C++, we can use delegation (C++11) or just set balance to 0.

- Methods: deposit returns new balance, throws if negative. withdraw throws if amount negative or > balance. viewBalance returns balance. transfer calls withdraw and deposit on other account.

- Main: creates two accounts with default constructor, deposit 1000, transfer 300, print balances.
