        account1.balance = 700 account2.balance = 300

class BankAccount:
    def __init__(self, balance=0):
        self.balance = balance

    def deposit(self, amount):
        if amount < 0:
            raise ValueError("Invalid amount")
        self.balance += amount
        return self.balance

    def withdraw(self, amount):
        if amount < 0:
            raise ValueError("Invalid amount")
        if amount > self.balance:
            raise ValueError("Insufficient balance.")
        self.balance -= amount
        return self.balance

    def view_balance(self):
        return self.balance

    def transfer(self, other_account, amount):
        self.withdraw(amount)
        other_account.deposit(amount)

=== RAW ANSWER ===

To translate the given Python bank account system into C++, we need to replicate the same functionality with idiomatic C++ while maintaining the exact behavior, including error handling and edge cases.

### Approach
1. **Class Definition**: Define a `BankAccount` class with a constructor that initializes the balance to 0 by default.
2. **Exception Handling**: Use `std::runtime_error` for exceptions, as C++ does not have a direct equivalent of Python's `ValueError`. The exception messages are kept the same.
3. **Member Functions**: Implement `deposit`, `withdraw`, `view_balance`, and `transfer` methods to mirror the Python class's behavior:
   - `deposit`: Adds a positive amount to the balance; throws an exception for negative amounts.
   - `withdraw`: Subtracts a valid amount (checked for negativity and sufficiency); throws exceptions for invalid or insufficient amounts.
   - `view_balance`: Returns the current balance.
   - `transfer`: Uses `withdraw` and `deposit` to transfer funds between accounts.

### Solution Code
```cpp
#include <stdexcept>
#include <string>

class BankAccount {
public:
    BankAccount(int balance = 0) : balance(balance) {}

    int deposit(int amount) {
        if (amount < 0) {
            throw std::runtime_error("Invalid amount");
        }
        balance += amount;
        return balance;
    }

    int withdraw(int amount) {
        if (amount < 0) {
            throw std::runtime_error("Invalid amount");
        }
        if (amount > balance) {
            throw std::runtime_error("Insufficient balance.");
        }
        balance -= amount;
        return balance;
    }

    int view_balance() const {
        return balance;
    }

    void transfer(BankAccount& other_account, int amount) {
        withdraw(amount);
        other_account.deposit(amount);
    }

private:
    int balance;
};
```

### Explanation
- **Constructor**: Initializes the account balance with a default value of 0.
- **deposit**: Increases the balance by the specified amount if it's non-negative; otherwise, throws an exception.
- **withdraw**: Decreases the balance by the specified amount if it's non-negative and does not exceed the current balance; otherwise, throws an appropriate exception.
- **view_balance**: Returns the current balance without modification.
- **transfer**: Transfers a specified amount from the current account to another account by first withdrawing the amount and then depositing it into the target account.

This C++ implementation ensures that all edge cases and error conditions are handled identically to the Python version, maintaining the same behavior and exception messages.
