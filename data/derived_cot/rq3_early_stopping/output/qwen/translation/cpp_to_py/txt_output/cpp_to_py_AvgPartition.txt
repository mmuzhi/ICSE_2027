#include <vector>

class AvgPartition {
public:
    AvgPartition(const std::vector<int>& lst, int limit);
    std::pair<int, int> set_num() const;
    std::vector<int> get(int index) const;

private:
    std::vector<int> lst;
    int limit;
};
AvgPartition::AvgPartition(const std::vector<int>& lst, int limit)
    : lst(lst), limit(limit) {}

std::pair<int, int> AvgPartition::set_num() const {
    int size = lst.size() / limit;
    int remainder = lst.size() % limit;
    return { size, remainder };
}

std::vector<int> AvgPartition::get(int index) const {
    auto [size, remainder] = set_num();
    int start = index * size + std::min(index, remainder);
    int end = start + size;
    if (index + 1 <= remainder) {
        end += 1;
    }
    return std::vector<int>(lst.begin() + start, lst.begin() + end);
}

To solve this problem, we need to translate a C++ class `AvgPartition` into an equivalent Python class. The class is designed to partition a list of integers into buckets with a specified limit, where the first few buckets may have one more element than the rest. The solution must maintain the same behavior, including handling edge cases and exceptions.

### Approach
1. **Class Initialization**: The constructor takes a list of integers and a limit, storing them as instance variables.
2. **Set Number Method**: This method computes the size of each bucket and the remainder elements that will be distributed across the first few buckets. The size is calculated as the integer division of the list length by the limit, and the remainder is the leftover elements.
3. **Get Method**: This method retrieves the elements of a specified bucket. The start and end indices for the bucket are calculated based on the bucket size and any remainder elements. The method ensures that if the bucket index is out of bounds, an empty list is returned.

### Solution Code
```python
class AvgPartition:
    def __init__(self, lst, limit):
        self.lst = lst
        self.limit = limit

    def set_num(self):
        size = len(self.lst) // self.limit
        remainder = len(self.lst) % self.limit
        return (size, remainder)

    def get(self, index):
        size, rem = self.set_num()
        start = index * size + min(index, rem)
        end = start + size
        if index < rem:
            end += 1
        return self.lst[start:end]
```

### Explanation
1. **Initialization**: The `__init__` method initializes the class with the given list and limit, storing them for later use.
2. **Set Number Method**: The `set_num` method calculates the size of each bucket (excluding any remainder elements) and the number of elements left over after dividing the list length by the limit. These values are returned as a tuple.
3. **Get Method**: The `get` method calculates the start and end indices of the specified bucket. The start index is computed by multiplying the bucket index by the bucket size and adding the minimum of the index and the remainder. The end index is adjusted by adding one if the bucket index is within the remainder elements. The method then returns the slice of the list from the start to end indices.

This approach ensures that the Python class behaves identically to the original C++ class, maintaining the same logic for partitioning the list into buckets, handling edge cases, and managing exceptions.
