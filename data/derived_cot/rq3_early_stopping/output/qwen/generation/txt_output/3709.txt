=== ANSWER ===

To solve this problem, we need to determine if there exists a substring of exactly length `k` in the given string `s` that meets specific conditions. The substring must consist of only one distinct character, and if there are characters immediately before or after the substring, they must be different from the character in the substring.

### Approach
1. **Iterate Over Possible Substrings**: For each possible starting index `i` in the string, check the substring of length `k` starting at `i`.
2. **Check Uniform Character**: Ensure all characters in the substring are the same. If they are not, skip to the next substring.
3. **Check Neighboring Characters**: If the substring is uniform, check the character immediately before and after the substring. If either neighboring character is the same as the substring's character, skip to the next substring. If both neighbors are different (or do not exist), the substring meets all conditions.
4. **Return Result**: As soon as a valid substring is found, return `True`. If no valid substring is found after all checks, return `False`.

### Solution Code
```python
class Solution:
    def hasSpecialSubstring(self, s: str, k: int) -> bool:
        n = len(s)
        for i in range(0, n - k + 1):
            char = s[i]
            all_same = True
            for j in range(i, i + k):
                if s[j] != char:
                    all_same = False
                    break
            if not all_same:
                continue
            if i > 0 and s[i-1] == char:
                continue
            if i + k < n and s[i+k] == char:
                continue
            return True
        return False
```

### Explanation
1. **Initialization**: The length of the string `s` is stored in `n`.
2. **Loop Through Substrings**: The loop runs from the start of the string to the position where a substring of length `k` can still be formed.
3. **Check Uniformity**: For each starting index `i`, the code checks if all characters in the substring `s[i:i+k]` are the same. If not, it skips further checks for this substring.
4. **Check Neighbors**: If the substring is uniform, the code checks the character before `i` (if it exists) and the character after the substring (at index `i + k`). If either of these characters matches the substring's character, the substring is invalid.
5. **Return Result**: If a valid substring is found, the function immediately returns `True`. If no valid substring is found after all iterations, it returns `False`.

This approach efficiently checks each possible substring of length `k` for the required conditions, ensuring optimal performance given the problem constraints.
