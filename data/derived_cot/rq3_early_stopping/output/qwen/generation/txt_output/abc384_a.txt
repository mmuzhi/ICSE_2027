=== ANSWER ===

To solve this problem, we need to replace every character in a given string that is not a specified character `c_1` with another character `c_2`. The solution involves reading the input values, processing the string according to the given conditions, and outputting the modified string.

### Approach
1. **Input Parsing**: The first line of input contains an integer `N` (the length of the string) followed by two lowercase English letters `c_1` and `c_2`. The second line contains the string `S` of length `N`.
2. **String Processing**: Iterate through each character in the string `S`. For each character, check if it matches `c_1`. If it does, retain the character; otherwise, replace it with `c_2`.
3. **Output the Result**: Construct the modified string by appending each processed character and print it.

### Solution Code
```python
def main():
    data = input().split()
    n = int(data[0])
    c1 = data[1]
    c2 = data[2]
    s = input().strip()
    result = []
    for char in s:
        if char == c1:
            result.append(char)
        else:
            result.append(c2)
    print(''.join(result))

if __name__ == "__main__":
    main()
```

### Explanation
1. **Input Parsing**: The first line of input is split into three parts: the integer `N` (length of the string), `c_1`, and `c_2`. The second line is the string `S` to be processed.
2. **String Processing**: The code processes each character in `S` by checking if it matches `c_1`. If it does, the character is added to the result list as-is. Otherwise, `c_2` is added to the result list.
3. **Output**: The result list is joined into a single string and printed. This approach efficiently processes each character in linear time, making it optimal for the given constraints. 

This solution ensures that all characters not equal to `c_1` are replaced with `c_2`, producing the desired output string. The simplicity and efficiency of the approach make it suitable for the problem constraints.
