=== ANSWER ===

To solve this problem, we need to determine whether Snuke is following Takahashi's instructions correctly when raising his hands to eat takoyaki. The instructions specify that Snuke should raise only his left hand if he wants to eat takoyaki, and only his right hand otherwise. The solution involves checking the given inputs (L and R) to see which hand(s) Snuke is raising and then determining the appropriate output.

### Approach
1. **Problem Analysis**: The problem requires us to check two conditions based on the values of L and R (each being either 0 or 1). 
   - If L is 1 and R is 0, Snuke is raising only his left hand, so the output should be "Yes".
   - If L is 0 and R is 1, Snuke is raising only his right hand, so the output should be "Yes".
   - If both L and R are 1, Snuke is raising both hands, which is invalid.
   - If both L and R are 0, Snuke is raising no hands, which is also invalid.
2. **Algorithm Selection**: The solution involves straightforward conditional checks to determine the correct output based on the values of L and R. No complex algorithms are needed; simple if-else statements suffice.

### Solution Code
```python
def main():
    data = input().split()
    L = int(data[0])
    R = int(data[1])
    
    if L == 1 and R == 0:
        print("Yes")
    elif L == 0 and R == 1:
        print("Yes")
    else:
        print("Invalid")

if __name__ == "__main__":
    main()
```

### Explanation
- **Input Handling**: The input is read as two integers, L and R, from standard input.
- **Condition Checks**: 
  - The first condition checks if Snuke is raising only his left hand (L=1 and R=0) and prints "Yes" if true.
  - The second condition checks if Snuke is raising only his right hand (L=0 and R=1) and prints "Yes" if true.
  - If neither condition is met, the output is "Invalid", which covers cases where both hands are raised or no hands are raised.
- **Output**: The result is printed based on the conditions evaluated.

This approach efficiently checks the given conditions and produces the correct output with minimal computational overhead.
