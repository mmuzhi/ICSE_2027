=== ANSWER ===

To solve this problem, we need to process a given string consisting of digits and extract all occurrences of the digit '2'. The solution involves iterating through each character in the string, checking if it is a '2', and if so, including it in the result. The result is formed by concatenating all these '2's in the order they appear in the original string.

### Approach
1. **Problem Analysis**: The problem requires us to filter out all characters from the input string except the digit '2'. The remaining characters should be concatenated in their original order to form the output string. Given that the input string contains at least one '2', we can proceed with the solution without handling edge cases for empty strings or absence of '2'.
2. **Intuition**: The solution involves a simple traversal of the string. For each character encountered, if it is '2', it is added to the result. This approach ensures that we maintain the original order of the '2's in the output.
3. **Algorithm Selection**: The algorithm used here is a linear scan through the string, checking each character. This approach is efficient with a time complexity of O(n), where n is the length of the string, as each character is processed exactly once.
4. **Optimization**: The solution is already optimal for the given problem constraints (string length up to 100). No further optimization is needed.

### Solution Code
```python
def main():
    S = input().strip()
    result = ''.join(char for char in S if char == '2')
    print(result)

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The input string is read from standard input and stripped of any leading or trailing whitespace.
2. **Filtering Characters**: Using a list comprehension, each character in the string is checked. If the character is '2', it is included in the result list.
3. **Concatenating Result**: The filtered characters are joined into a single string, which is then printed as the output.
This approach efficiently processes the input string in linear time, ensuring optimal performance while maintaining simplicity and clarity.
