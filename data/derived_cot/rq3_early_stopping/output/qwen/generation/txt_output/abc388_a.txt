=== ANSWER ===

To solve this problem, we need to generate a new string by concatenating the first character of the input string with the substring "UPC". The input string is guaranteed to have the first character as an uppercase letter and the subsequent characters as lowercase letters. 

### Approach
1. **Read the Input String**: The input is read from standard input.
2. **Extract the First Character**: The first character of the string is isolated.
3. **Concatenate "UPC"**: The extracted first character is combined with the string "UPC".
4. **Output the Result**: The concatenated string is printed as the output.

This approach efficiently leverages string manipulation to achieve the desired result with minimal computational overhead.

### Solution Code
```python
def main():
    S = input().strip()
    result = S[0] + "UPC"
    print(result)
```

### Explanation
1. **Reading Input**: The `input()` function reads the entire input string, and `strip()` ensures any extraneous whitespace is removed.
2. **Extracting First Character**: The first character of the string is accessed using `S[0]`.
3. **Concatenation**: The first character is concatenated with the string "UPC" using the `+` operator.
4. **Output**: The resulting string is printed directly.

This solution is straightforward and efficient, handling all specified constraints and edge cases naturally. The code is concise and easy to understand, making it suitable for the given problem.
