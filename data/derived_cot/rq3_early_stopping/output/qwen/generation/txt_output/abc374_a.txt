=== ANSWER ===

To solve this problem, we need to determine if a given string ends with the substring "san". The solution involves checking the last three characters of the string to see if they match "san".

### Approach
1. **Problem Analysis**: The problem requires checking if the input string ends with the specific substring "san". The solution can be derived by examining the last three characters of the string.
2. **Intuition**: By using string slicing, we can easily access the last three characters of the string. If these characters are "san", the answer is "Yes"; otherwise, it is "No".
3. **Algorithm Selection**: The algorithm involves a straightforward check using string slicing. The slicing operation `S[-3:]` extracts the last three characters of the string. This is efficient and leverages Python's built-in string handling capabilities.
4. **Complexity Analysis**: The solution operates in constant time O(1) since it only checks a fixed number of characters (three) regardless of the string's length.

### Solution Code
```python
S = input().strip()
if S.endswith('san'):
    print("Yes")
else:
    print("No")
```

### Explanation
1. **Input Handling**: The input string is read and stripped of any leading or trailing whitespace to ensure accurate processing.
2. **String Check**: The `endswith` method is used to check if the string ends with "san". This method efficiently verifies the required suffix without manually handling string indices.
3. **Output Decision**: Based on the result of the `endswith` check, the program prints "Yes" if the string ends with "san" and "No" otherwise. This approach is concise and leverages Python's built-in string methods for simplicity and efficiency.
