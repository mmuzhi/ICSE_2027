=== ANSWER ===

To solve this problem, we need to determine the number of empty squares on an 8x8 grid where a piece can be placed such that it cannot be captured by any existing pieces. The capture condition is defined as any existing piece in the same row or column as the placed piece.

### Approach
1. **Problem Analysis**: The grid consists of 64 squares, each either empty ('.') or occupied by a piece ('#'). A piece placed on an empty square can be captured if there is any existing piece in the same row or column. The goal is to count all empty squares where no such existing pieces exist in their respective row or column.

2. **Intuition**: For each empty square, we need to check if its entire row and column are free of any existing pieces. If both conditions are met, the square is a valid placement.

3. **Algorithm Selection**: 
   - **Preprocessing**: First, we preprocess the grid to mark which rows and columns contain at least one existing piece. This allows us to quickly check the safety of any empty square.
   - **Counting Valid Squares**: For each empty square, we verify if its row and column are marked as safe (i.e., no existing pieces). If both conditions are satisfied, the square is counted.

4. **Complexity Analysis**: The algorithm involves two passes over the grid: one to preprocess the rows and columns (O(64) operations) and another to count valid squares (another O(64) operations). The overall complexity is O(n^2), which is efficient for the given constraints (8x8 grid).

### Solution Code
```python
def main():
    grid = []
    for _ in range(8):
        grid.append(input().strip())
    
    row_has_piece = [False] * 8
    col_has_piece = [False] * 8
    
    for i in range(8):
        for j in range(8):
            if grid[i][j] == '#':
                row_has_piece[i] = True
                col_has_piece[j] = True
                
    count = 0
    for i in range(8):
        for j in range(8):
            if grid[i][j] == '.' and not row_has_piece[i] and not col_has_piece[j]:
                count += 1
                
    print(count)

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The grid is read from standard input, with each line representing a row of the grid.
2. **Preprocessing**: Two boolean arrays, `row_has_piece` and `col_has_piece`, are initialized to keep track of rows and columns that contain at least one '#' (existing piece).
3. **Marking Rows and Columns**: For each cell in the grid, if the cell contains a '#', the corresponding row and column in the preprocessing arrays are marked as `True`.
4. **Counting Valid Squares**: For each empty cell ('.'), the algorithm checks if its row and column are marked as safe (i.e., no existing pieces). If both conditions are met, the cell is counted as a valid placement.
5. **Output**: The total count of valid squares is printed.

This approach efficiently checks all possible placements by leveraging preprocessing to minimize repeated checks, ensuring optimal performance for the problem constraints.
