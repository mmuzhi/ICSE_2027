=== ANSWER ===

To solve this problem, we need to rearrange the digits of a given three-digit integer in two specific orders and then print the resulting integers. The first rearrangement involves placing the tens and ones digits first, followed by the hundreds digit. The second rearrangement involves placing the ones digit first, followed by the hundreds and tens digits.

### Approach
1. **Input Handling**: Read the input three-digit integer as a string to easily access each digit.
2. **Digit Extraction**: Extract the hundreds, tens, and ones digits from the string.
3. **Rearrangement**: 
   - For the first rearrangement, form a number by placing the tens digit in the hundreds place, the ones digit in the tens place, and the hundreds digit in the ones place.
   - For the second rearrangement, form a number by placing the ones digit in the hundreds place, the hundreds digit in the tens place, and the tens digit in the ones place.
4. **Output**: Print the two resulting integers separated by a space.

### Solution Code
```python
def main():
    N_str = input().strip()
    a = int(N_str[0])
    b = int(N_str[1])
    c = int(N_str[2])
    first_num = 100 * b + 10 * c + a
    second_num = 100 * c + 10 * a + b
    print(f"{first_num} {second_num}")

if __name__ == '__main__':
    main()
```

### Explanation
1. **Reading Input**: The input is read as a string to facilitate individual digit extraction.
2. **Digit Extraction**: The first character of the string (index 0) is the hundreds digit (a), the second character (index 1) is the tens digit (b), and the third character (index 2) is the ones digit (c).
3. **First Rearrangement**: The number formed by the tens, ones, and hundreds digits is calculated as `100*b + 10*c + a`.
4. **Second Rearrangement**: The number formed by the ones, hundreds, and tens digits is calculated as `100*c + 10*a + b`.
5. **Output**: The two resulting numbers are printed in the specified order, separated by a space.

This approach efficiently rearranges the digits using basic arithmetic operations and string manipulation, ensuring clarity and correctness for the given problem constraints.
