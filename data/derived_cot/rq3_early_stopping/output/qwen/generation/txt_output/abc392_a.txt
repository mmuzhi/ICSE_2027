def main():
    data = input().split()
    a = int(data[0])
    b = int(data[1])
    c = int(data[2])
    
    if a * b == c or a * c == b or b * c == a:

if __name__ == "__main__":

To solve this problem, we need to determine if there exists a permutation of the given three integers such that the product of the first two integers equals the third integer. 

### Approach
1. **Problem Analysis**: The problem requires checking all possible permutations of the three given integers to see if any permutation satisfies the condition \(B_1 \times B_2 = B_3\). Since there are only three integers, there are six possible permutations, but we can efficiently check the condition without generating all permutations by directly verifying the three possible pairs of products.
2. **Key Insight**: For three integers \(a\), \(b\), and \(c\), the condition \(B_1 \times B_2 = B_3\) can be satisfied in three scenarios:
   - \(a \times b = c\)
   - \(a \times c = b\)
   - \(b \times c = a\)
3. **Algorithm Selection**: The solution involves reading the three integers, checking the three conditions mentioned above, and printing "Yes" if any condition is met, otherwise printing "No".

### Solution Code
```python
def main():
    data = input().split()
    a = int(data[0])
    b = int(data[1])
    c = int(data[2])
    
    if a * b == c or a * c == b or b * c == a:
        print("Yes")
    else:
        print("No")

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The input consists of three integers read from standard input.
2. **Checking Conditions**: The code checks if any of the three conditions \(a \times b = c\), \(a \times c = b\), or \(b \times c = a\) is true. If any condition is satisfied, it prints "Yes"; otherwise, it prints "No".
3. **Efficiency**: The solution efficiently checks all possible pairs of products in constant time \(O(1)\), making it optimal for the given constraints. The approach leverages the small input size to avoid unnecessary computations, ensuring quick and accurate results.
