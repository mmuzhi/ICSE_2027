def main():
    import sys
    data = sys.stdin.read().split()
    n = int(data[0])
    c = int(data[1])
    times = list(map(int, data[2:2+n]))
    
    candies = 0
    last_candy_time = -1  # no candy received initially
    
    for t in times:
        if t - last_candy_time >= c:
            candies += 1
            last_candy_time = t
            
    print(candies)

if __name__ == "__main__":
    main()

To solve this problem, we need to determine how many candies Takahashi will receive when pressing a button N times at specific times, given that he receives a candy only if at least C seconds have elapsed since his last candy.

### Approach
1. **Initialization**: Read the input values, which include the number of button presses (N), the threshold time (C), and the list of times at which the button is pressed.
2. **Tracking Candies**: Initialize a counter for the number of candies and a variable to keep track of the last time a candy was received (initially set to -1, indicating no candy has been received yet).
3. **Iterate Through Presses**: For each press time in the list:
   - Check if the current time minus the last candy time is at least C. If it is, increment the candy counter and update the last candy time to the current time.
4. **Output Result**: After processing all presses, print the total number of candies received.

This approach efficiently checks each press time against the last candy time to determine if a candy should be awarded, ensuring optimal performance with a linear pass through the list of times.

### Solution Code
```python
def main():
    import sys
    data = sys.stdin.read().split()
    n = int(data[0])
    c = int(data[1])
    times = list(map(int, data[2:2+n]))
    
    candies = 0
    last_candy_time = -1
    
    for t in times:
        if t - last_candy_time >= c:
            candies += 1
            last_candy_time = t
            
    print(candies)

if __name__ == "__main__":
    main()
```

### Explanation
- **Initialization**: The input is read and parsed into integers N (number of presses), C (threshold time), and a list of times T.
- **Tracking Candies**: The variable `candies` counts the number of candies received, starting at 0. `last_candy_time` is initialized to -1 to indicate no prior candy.
- **Processing Presses**: For each time in the list, the code checks if the elapsed time since the last candy (current time minus `last_candy_time`) is at least C. If true, the candy count is incremented, and `last_candy_time` is updated to the current time.
- **Output**: The total number of candies is printed after processing all presses.

This method ensures that each press is evaluated in constant time, leading to an efficient O(N) solution where N is the number of button presses.
