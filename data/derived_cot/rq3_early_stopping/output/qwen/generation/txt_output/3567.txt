=== ANSWER ===

To solve this problem, we need to convert a given Gregorian calendar date in the "yyyy-mm-dd" format into its binary representation. The binary representations of the year, month, and day should be concatenated in the same "yyyy-mm-dd" format without any leading zeros.

### Approach
1. **Parse the Input Date**: The input string is split into year, month, and day components using the hyphen '-' as a delimiter.
2. **Convert to Integers**: Each component (year, month, day) is converted from a string to an integer.
3. **Convert to Binary**: Each integer is then converted to its binary representation. The built-in Python function `bin()` is used for this conversion, and the resulting binary string is stripped of the '0b' prefix.
4. **Reformat the Result**: The binary strings of the year, month, and day are combined into the desired output format, separated by hyphens.

### Solution Code
```python
class Solution:
    def convertDateToBinary(self, date: str) -> str:
        parts = date.split('-')
        y = int(parts[0])
        m = int(parts[1])
        d = int(parts[2])
        return bin(y)[2:] + '-' + bin(m)[2:] + '-' + bin(d)[2:]
```

### Explanation
1. **Parsing the Input**: The input string is split into three parts (year, month, day) using the `split('-')` method. For example, "2080-02-29" is split into ["2080", "02", "29"].
2. **Conversion to Integers**: Each part of the split string is converted to an integer. This step ensures that leading zeros (if any) in the month or day are handled correctly during binary conversion.
3. **Binary Conversion**: The `bin()` function converts each integer to its binary representation. The `[2:]` slicing removes the '0b' prefix that Python adds to binary strings. For instance, `bin(2080)[2:]` results in "100000100000".
4. **Result Construction**: The binary strings of the year, month, and day are concatenated with hyphens in the same order as the original date string. The final result is a string where each component is its binary representation, maintaining the original date format.

This approach efficiently handles the conversion and reformatting of the date components into their binary representations, ensuring correctness and simplicity.
