=== ANSWER ===

To solve this problem, we need to extract two digits from a given 3-character string where the first and third characters are digits, and the second character is 'x'. The goal is to compute the product of these two digits and output the result.

### Approach
1. **Input Handling**: Read the input string which is guaranteed to be exactly 3 characters long.
2. **Character Extraction**: The first and third characters of the string are the digits we need. The second character is always 'x' as per the problem constraints.
3. **Conversion and Multiplication**: Convert the first and third characters from strings to integers, then multiply these integers to get the result.
4. **Output**: Print the product as an integer.

This approach efficiently leverages string manipulation and basic arithmetic operations to solve the problem. The constraints ensure that the input is always valid, simplifying the extraction and conversion steps.

### Solution Code
```python
def main():
    S = input().strip()
    first_digit = int(S[0])
    second_digit = int(S[2])
    result = first_digit * second_digit
    print(result)

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The input string `S` is read and stripped of any leading or trailing whitespace (though the problem guarantees a 3-character string, this step ensures robustness).
2. **Extracting Digits**: The first character `S[0]` and the third character `S[2]` are converted from strings to integers using the `int()` function.
3. **Multiplication**: The two integers obtained from the string characters are multiplied together.
4. **Output Result**: The result of the multiplication is printed directly.

This solution efficiently handles the problem requirements with minimal steps, ensuring clarity and correctness. The constraints guarantee that the input format is correct, so no additional validation is necessary.
