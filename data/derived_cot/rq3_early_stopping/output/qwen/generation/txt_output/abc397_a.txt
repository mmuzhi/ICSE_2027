36.6

=== ANSWER ===

To solve this problem, we need to classify Takahashi's body temperature into one of three categories based on its value. The classification is straightforward and involves checking the temperature against specific thresholds.

### Approach
1. **Input Handling**: Read the input temperature value, which is provided as a string with one decimal place.
2. **Conversion**: Convert the input string into a floating-point number for numerical comparison.
3. **Classification**: Use conditional statements to determine the category:
   - If the temperature is 38.0 or higher, it is classified as "High fever" (output 1).
   - If the temperature is between 37.5 and 38.0 (exclusive), it is classified as "Fever" (output 2).
   - If the temperature is below 37.5, it is classified as "Normal" (output 3).

### Solution Code
```python
def main():
    X = float(input().strip())
    if X >= 38.0:
        print(1)
    elif X >= 37.5:
        print(2)
    else:
        print(3)

if __name__ == '__main__':
    main()
```

### Explanation
1. **Reading Input**: The input is read as a string and converted to a float to handle decimal values accurately.
2. **Condition Checks**: The code checks the temperature against the defined thresholds:
   - The first condition checks if the temperature is 38.0 or higher, printing 1 for "High fever".
   - The second condition checks if the temperature is between 37.5 and 38.0, printing 2 for "Fever".
   - The final condition covers temperatures below 37.5, printing 3 for "Normal".
3. **Output**: The result is printed based on the condition met, ensuring the correct classification is provided.

This approach efficiently categorizes the temperature into the appropriate category using simple conditional checks, ensuring clarity and correctness.
