### Solution Code
```python
class Solution:
    def findMaxK(self, nums: List[int]) -> int:
        s = set(nums)
        max_val = -1
        for num in s:
            if num >= 0 and -num in s:
                if num > max_val:
                    max_val = num
        return max_val
```

### Explanation
1. **Set Conversion**: The input list `nums` is converted into a set `s` to allow O(1) lookups.
2. **Tracking Maximum Value**: Initialize `max_val` to -1. This variable will store the largest valid `k` found during iteration.
3. **Iteration and Check**: For each unique element `num` in the set:
   - If `num` is non-negative and its negative counterpart `-num` exists in the set, update `max_val` if `num` is greater than the current `max_val`.
4. **Result**: After processing all elements, `max_val` holds the largest `k` such that both `k` and `-k` are in the list. If no such `k` exists, the function returns -1.
