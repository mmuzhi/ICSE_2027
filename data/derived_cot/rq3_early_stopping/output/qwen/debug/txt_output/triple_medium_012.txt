=== RAW ANSWER ===

To solve this problem, we need to find the maximum gap between any two consecutive elements in a sorted array. The approach involves sorting the array first, then iterating through the sorted array to compute the differences between consecutive elements and keeping track of the maximum difference encountered.

### Approach
1. **Sort the Array**: The first step is to sort the given array in ascending order. This allows us to easily compute the differences between consecutive elements, which is crucial for finding the maximum gap.
2. **Handle Edge Cases**: If the array has fewer than two elements, the maximum gap is zero since there are no pairs to compare.
3. **Iterate and Track Maximum Gap**: By iterating through the sorted array, we compute the difference between each consecutive pair of elements. During each iteration, we update the maximum gap if the current difference is larger than the previously recorded maximum gap.

### Solution Code
```python
class Solution:
    def maximumGap(self, nums: list) -> int:
        nums = sorted(nums)
        max_gap = float('-inf')
        if len(nums) < 2:
            return 0
        for i in range(len(nums)-1):
            gap = nums[i+1] - nums[i]
            if max_gap < gap:
                max_gap = gap
        return max_gap
```

### Explanation
1. **Sorting the Array**: The array is sorted to arrange elements in ascending order, which simplifies the process of finding consecutive elements and their differences.
2. **Edge Case Handling**: If the array contains only one or zero elements, the function returns 0 immediately since no gaps can exist.
3. **Tracking Maximum Gap**: The loop runs from the first element to the second last element, calculating the difference between each element and its successor. The maximum gap encountered during this process is stored and returned at the end. This approach efficiently narrows down the solution to consecutive elements, ensuring optimal performance with a time complexity of O(n log n) due to the sorting step, followed by O(n) for the iteration.
