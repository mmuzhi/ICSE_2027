To solve this problem, we need to determine the number of ways to split a given binary string into three parts such that each part contains the same number of '1's. The solution involves checking specific conditions and calculating the valid splits based on the positions of the '1's in the string.

### Approach
1. **Check for No '1's**: If the string contains no '1's, the number of ways to split the string into three parts (each containing zero '1's) is given by the combination formula \( \binom{n-1}{2} \), where \( n \) is the length of the string. This is because we need to choose two split positions from the \( n-1 \) possible gaps between characters.
2. **Check Divisibility by 3**: If the total number of '1's is not divisible by 3, it's impossible to split the string into three parts with an equal number of '1's, so the answer is 0.
3. **Calculate Valid Splits**: If the number of '1's is divisible by 3, we first build a list of indices where '1's occur. The valid splits are determined by the gaps between the \( k \)-th and \( (k+1) \)-th '1's, and the \( 2k \)-th and \( (2k+1) \)-th '1's, where \( k \) is one-third of the total '1's. The product of these gaps gives the number of valid ways to split the string.

### Solution Code
```python
class Solution:
    def numWays(self, s: str) -> int:
        MOD = 10**9 + 7
        N = len(s)
        c1 = s.count('1')
        if c1 == 0:
```
