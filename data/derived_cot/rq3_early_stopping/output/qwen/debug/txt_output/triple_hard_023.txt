=== RAW ANSWER ===

To solve this problem, we need to find the median of two sorted arrays efficiently. The median is the middle value in an ordered list of numbers. If the combined length of the two arrays is even, the median is the average of the two middle values.

### Approach
1. **Merge the Arrays**: We will merge the two sorted arrays into a single sorted array by comparing elements from both arrays and appending the smaller element to the result array.
2. **Calculate the Median**: After merging, we determine the median based on the length of the merged array:
   - If the length is odd, the median is the middle element.
   - If the length is even, the median is the average of the two middle elements.

### Solution Code
```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        p1 = 0
        p2 = 0
        new = []
        while p1 < len(nums1) and p2 < len(nums2):
            if nums1[p1] < nums2[p2]:
                new.append(nums1[p1])
                p1 += 1
            else:
                new.append(nums2[p2])
                p2 += 1
        while p1 < len(nums1):
            new.append(nums1[p1])
            p1 += 1
        while p2 < len(nums2):
            new.append(nums2[p2])
            p2 += 1
        
        if len(new) % 2 == 1:
            median = float(new[len(new) // 2])
        else:
            index = len(new) // 2
            median = (new[index] + new[index - 1]) / 2.0
        
        return median
```

### Explanation
1. **Merging Arrays**: The algorithm uses two pointers to traverse both arrays simultaneously, appending the smaller of the two current elements to the merged array. This ensures the merged array remains sorted.
2. **Handling Remaining Elements**: After one array is exhausted, the remaining elements from the other array are appended to the merged array.
3. **Finding the Median**: The median is determined by checking the length of the merged array. If the length is odd, the median is the middle element. If even, it is the average of the two middle elements.

This approach efficiently merges the arrays in O(n) time, where n is the total number of elements in both arrays, and calculates the median in O(1) time. The space complexity is O(n) to store the merged array.
