To solve this problem, we need to generate a list of numbers from 1 to n in lexicographical order. The initial approach converts each number to a string and then sorts the list lexicographically. This ensures that the numbers are ordered based on their string representations, which is the correct approach for lexicographical ordering.

### Approach
1. **Convert Numbers to Strings**: First, we convert each number from 1 to n into a string. This allows us to leverage the lexicographical ordering of strings.
2. **Sort Lexicographically**: We then sort the list of strings in lexicographical order. This step ensures that the numbers are ordered based on their string representations, which is the requirement for lexicographical ordering.
3. **Convert Back to Integers**: Finally, we convert each string back to an integer to return the result as a list of integers.

This approach efficiently handles the lexicographical ordering by utilizing the built-in string comparison, which is both straightforward and effective.

### Solution Code
```python
class Solution:
    def lexicalOrder(self, n: int) -> List[int]:
        lst = [str(i) for i in range(1, n+1)]
        lst.sort()
        return [int(x) for x in lst]
```

### Explanation
1. **String Conversion**: The list comprehension `[str(i) for i in range(1, n+1)]` converts each number from 1 to n into a string. For example, if `n` is 13, the list becomes `['1', '2', '3', ..., '13']`.
2. **Lexicographical Sorting**: The `sort()` method sorts the strings in lexicographical order. For instance, `'1'` comes before `'10'` because `'1'` is a prefix of `'10'`, and in lexicographical order, shorter strings come before longer ones if they share a common prefix.
3. **Integer Conversion**: The list of sorted strings is then converted back to integers using `[int(x) for x in lst]`, resulting in the desired lexicographical order of numbers.

This method ensures that the numbers are ordered correctly based on their string representations, meeting the problem's requirements efficiently.
