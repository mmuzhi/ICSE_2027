To solve this problem, we need to construct a quadtree from a given grid. The quadtree is a tree data structure where each node represents a square region in the grid. The root node represents the entire grid, and each non-leaf node has four children representing the four quadrants of the region. The leaves of the quadtree represent individual cells in the grid.

### Approach
1. **Problem Analysis**: The problem involves recursively dividing the grid into quadrants until each quadrant is a single cell (leaf node). If all four quadrants of a node have the same value and are leaves, the node is also a leaf. Otherwise, the node is non-leaf with four children.
2. **Intuition**: The solution uses a divide-and-conquer approach. The grid is split into four equal parts (quadrants) recursively until each part is a single cell. At each step, if all four quadrants of the current node have the same value, the node is a leaf; otherwise, it has four children.
3. **Algorithm Selection**: The algorithm constructs the quadtree by recursively processing each quadrant of the grid. The base case is when the quadrant is a single cell (leaf node). For larger quadrants, the algorithm checks if all four sub-quadrants are leaves with the same value; if so, it creates a leaf node; otherwise, it creates a non-leaf node with the four sub-nodes.
4. **Complexity Analysis**: The time complexity is O(n^2) in the worst case, where n is the grid size, because each cell is processed once. The space complexity is O(log n) due to the recursion depth, which is the depth of the quadtree.
