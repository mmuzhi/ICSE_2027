=== RAW ANSWER ===

To solve this problem, we need to find the median of two sorted arrays efficiently. The median is the middle value in an ordered list of numbers. If the combined length of the two arrays is even, the median is the average of the two middle numbers.

### Approach
1. **Merge the Arrays**: Combine the two sorted arrays into a single sorted array. This is done by comparing elements from both arrays and appending the smaller element to the result array until one array is exhausted.
2. **Handle Remaining Elements**: After one of the arrays is fully processed, append the remaining elements from the other array to the result array.
3. **Calculate the Median**: 
   - If the combined length of the merged array is odd, the median is the middle element.
   - If the combined length is even, the median is the average of the two middle elements.

### Solution Code
```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        p1 = 0
        p2 = 0
        new = []
        while p1 < len(nums1) and p2 < len(nums2):
            if nums1[p1] < nums2[p2]:
                new.append(nums1[p1])
                p1 += 1
            else:
                new.append(nums2[p2])
                p2 += 1
        while p1 < len(nums1):
            new.append(nums1[p1])
            p1 += 1
        while p2 < len(nums2):
            new.append(nums2[p2])
            p2 += 1
        
        n = len(new)
        if n % 2 == 0:
            index = n // 2
            median = (new[index] + new[index - 1]) / 2.0
        else:
            median = float(new[n // 2])
        return median
```

### Explanation
1. **Merging Arrays**: The algorithm starts by initializing two pointers, `p1` and `p2`, to the start of each array. It then iterates through both arrays, comparing elements and appending the smaller one to the result array `new`. This ensures the merged array remains sorted.
2. **Appending Remaining Elements**: Once one array is exhausted, the remaining elements from the other array are appended to `new` in their natural order.
3. **Finding the Median**: The median is determined based on the length of the merged array. For an odd length, the median is the middle element. For an even length, it is the average of the two middle elements. The result is returned as a float to handle both cases appropriately.

This approach efficiently merges the arrays and computes the median in linear time, making it suitable for large input sizes.
