For each candidate speed `mid`, we calculate the total travel time by summing the ceiling of each of the first `n-1` segments divided by `mid` and adding the last segment divided by `mid`.

### Solution Code
```python
import math

class Solution:
    def minSpeedOnTime(self, dist: List[int], hour: float) -> int:
        n = len(dist)
        if hour < n - 1:
            return -1
        
        left, right = 1, 10**18
        
        while left < right:
            mid = (left + right) // 2
            total_time = 0.0
            for i in dist[:-1]:
                total_time += math.ceil(i / mid)
            total_time += dist[-1] / mid
            
            if total_time <= hour:
                right = mid
            else:
                left = mid + 1
        
        return left
```
