class Solution:
    def minOperationsMaxProfit(self, customers: List[int], boardingCost: int, runningCost: int) -> int:
        maxProfit = -10**18  # a very small number
        ans = 0
        curRounds = 0
        curCustomers = 0
        rem = 0
        i = 0
        n = len(customers)
        while i < n or rem:
            if i < n:
                rem += customers[i]
                i += 1
            curRounds += 1
            if rem >= 4:
                rem -= 4
                curCustomers += 4
            else:
                curCustomers += rem
                rem = 0
            curProfit = curCustomers * boardingCost - curRounds * runningCost
            if curProfit > maxProfit:
                maxProfit = curProfit
                ans = curRounds
        return ans if maxProfit > 0 else -1

The Ferris wheel can board up to 4 customers per round, and each round incurs a running cost. The profit is calculated as the total boarding cost minus the total running cost. The goal is to find the earliest round where the profit is maximized.

### Approach
1. **Initialization**: Start with initial values for maximum profit (`maxProfit`), answer (`ans`), current rounds (`curRounds`), current customers boarded (`curCustomers`), and remaining customers (`rem`). Initialize `ans` to -1 to indicate no solution found initially.
2. **Simulation Loop**: While there are customers left to process (either in the current list or remaining from previous rounds), simulate each round:
   - **Customer Arrival**: Add customers arriving at the current minute to the remaining customers.
   - **Boarding Customers**: Board up to 4 customers in each round. Update the remaining customers and the total customers boarded.
   - **Profit Calculation**: Calculate the current profit based on the total customers boarded and the number of rounds.
   - **Update Maximum Profit**: If the current profit exceeds the recorded maximum profit, update `maxProfit` and `ans` to the current round number.
3. **Result Determination**: After processing all customers and remaining customers, return the round number where the maximum profit was first achieved. If no positive profit is found, return -1.

### Solution Code
```python
class Solution:
    def minOperationsMaxProfit(self, customers, boardingCost, runningCost):
        maxProfit = -10**18
        ans = -1

        i = 0

```

### Explanation
- **Initialization**: The variables `maxProfit`, `ans`, `curRounds`, `curCustomers`, and `rem` are initialized to track the maximum profit, the corresponding round, and the current state of customers and rounds.
- **Loop**: The loop continues until all customers have been processed and there are no remaining customers. In each iteration:
  - **Customer Arrival**: If there are more customers in the list, they are added to the remaining customers (`rem`).
  - **Boarding**: Up to 4 customers are boarded in each round. The remaining customers are adjusted accordingly.
  - **Profit Calculation**: The profit for the current state is calculated and compared to the maximum profit recorded so far. If it exceeds the maximum, the maximum profit and the corresponding round are updated.
- **Result**: After the loop, if the maximum profit is positive, the corresponding round number is returned; otherwise, -1 is returned indicating no profitable operation was found.

This approach efficiently simulates each round, dynamically updates the profit, and ensures the earliest round with the maximum profit is identified.
