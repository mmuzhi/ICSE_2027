To solve this problem, we need to construct a quad-tree from a given grid where each node represents a quadrant of the grid. The quad-tree is built recursively by dividing the grid into four equal parts until each part is a single cell (leaf node). If all four quadrants of a node have the same value, the node is marked as a leaf; otherwise, it is an internal node with four children.

### Approach
1. **Base Case Handling**: If the grid is a single cell (level 1), return a leaf node with the cell's value.
2. **Recursive Division**: For larger grids, divide the grid into four quadrants (top-left, top-right, bottom-left, bottom-right) recursively.
3. **Leaf Node Check**: After building the four quadrants, check if all four are leaves and have the same value. If so, create a new leaf node with that value. Otherwise, create an internal node with the four children.
