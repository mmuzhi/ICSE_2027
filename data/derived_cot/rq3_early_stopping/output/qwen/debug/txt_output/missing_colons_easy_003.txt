To solve this problem, we need to find the mode(s) (most frequently occurring element(s)) in a binary search tree (BST). The approach involves traversing the tree to count the occurrences of each value and then determining which values have the highest count.

### Approach
1. **Tree Traversal**: We use a depth-first search (DFS) approach to traverse the tree. This involves visiting each node, processing its value, and then recursively visiting its left and right children.
2. **Frequency Counting**: As we traverse the tree, we maintain a dictionary to keep track of how many times each value appears.
3. **Finding Modes**: After traversing the entire tree, we determine the maximum frequency from the dictionary. Any value(s) that occur with this maximum frequency are considered the mode(s) of the tree.

### Solution Code
```python
class Solution:
    def findMode(self, root: Optional[TreeNode]) -> List[int]:
        def dfs(node):
            if not node:
                return
            freq[node.val] += 1
            dfs(node.left)
            dfs(node.right)
        
        freq = {}
        dfs(root)
        max_cnt = 0
        ans = []

            elif cnt == max_cnt:
                ans.append(num)
        return ans
```

### Explanation
1. **Tree Traversal**: The `dfs` function recursively visits each node in the tree. For each node, it increments the count of the node's value in the frequency dictionary.
2. **Frequency Counting**: The dictionary `freq` is populated during the traversal, with keys being the node values and values being their respective counts.
3. **Finding Modes**: After the traversal, we iterate through the frequency dictionary to find the maximum count. All values that have this maximum count are collected into the result list `ans`, which is then returned.

This approach efficiently counts the occurrences of each value in the BST and identifies the most frequent values, ensuring optimal performance with a time complexity of O(n), where n is the number of nodes in the tree. The space complexity is also O(n) due to the storage required for the frequency dictionary.
