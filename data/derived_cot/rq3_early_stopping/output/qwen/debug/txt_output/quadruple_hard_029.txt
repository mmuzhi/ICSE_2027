=== RAW ANSWER ===

To solve this problem, we need to find the median of two sorted arrays efficiently. The median is the middle value in an ordered list of numbers. If the combined length of the two arrays is even, the median is the average of the two middle values; if it's odd, the median is the middle value.

### Approach
1. **Merge the Arrays**: Combine the two sorted arrays into a single sorted array. This is done by comparing elements from both arrays and appending the smaller one to the result list, then moving the pointer forward in the respective array.
2. **Calculate the Median**: After merging the arrays, determine the median based on the length of the resulting array. If the length is even, the median is the average of the two middle elements; if odd, it's the middle element.

### Solution Code
```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        new = []
        p1 = 0
        p2 = 0
        
        while p1 < len(nums1) and p2 < len(nums2):
            if nums1[p1] < nums2[p2]:
                new.append(nums1[p1])
                p1 += 1
            else:
                new.append(nums2[p2])
                p2 += 1
        
        while p1 < len(nums1):
            new.append(nums1[p1])
            p1 += 1
            
        while p2 < len(nums2):
            new.append(nums2[p2])
            p2 += 1
        
        n = len(new)
        if n % 2 == 0:
            index1 = n // 2 - 1
            index2 = n // 2
            median = (new[index1] + new[index2]) / 2.0
        else:
            median = new[n // 2]
        return median
```

### Explanation
1. **Merging Arrays**: The algorithm starts by initializing two pointers, `p1` and `p2`, to the start of each array. It then iterates through both arrays, comparing elements at the current pointers and appending the smaller element to the result list `new`. The corresponding pointer is incremented after each comparison.
2. **Appending Remaining Elements**: Once one of the arrays is exhausted, the remaining elements from the other array are appended to the result list in their original order.
3. **Finding the Median**: The median is calculated based on the length of the merged array. If the length is even, the median is the average of the two middle elements (indices `n//2 - 1` and `n//2`). If the length is odd, the median is the middle element (index `n//2`).

This approach efficiently merges the two sorted arrays in linear time, O(m + n), where m and n are the lengths of the two arrays, and then calculates the median in constant time. The space complexity is O(m + n) due to the storage required for the merged array.
