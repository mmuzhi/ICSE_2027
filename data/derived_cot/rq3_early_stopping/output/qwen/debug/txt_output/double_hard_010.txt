=== RAW ANSWER ===

To solve this problem, we need to find the median of two sorted arrays efficiently. The median is the middle value in an ordered list of numbers. If the combined length of the two arrays is even, the median is the average of the two middle values; otherwise, it is the middle value.

### Approach
1. **Merge the Two Arrays**: Since both arrays are already sorted, we can merge them into a single sorted array by comparing elements from each array and appending the smaller element to the new array.
2. **Calculate the Median**: Once the merged array is created, we determine the median based on the length of this array. If the length is odd, the median is the middle element. If the length is even, the median is the average of the two middle elements.

The key insight here is that merging two sorted arrays can be done in linear time, O(m + n), where m and n are the lengths of the two arrays. This approach ensures efficiency and correctness.

### Solution Code
```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        p1 = 0
        p2 = 0
        new = []
        while p1 < len(nums1) and p2 < len(nums2):
            if nums1[p1] < nums2[p2]:
                new.append(nums1[p1])
                p1 += 1
            else:
                new.append(nums2[p2])
                p2 += 1
        while p1 < len(nums1):
            new.append(nums1[p1])
            p1 += 1
        while p2 < len(nums2):
            new.append(nums2[p2])
            p2 += 1

        n = len(new)
        if n % 2 == 0:
            index = n // 2
            median = (new[index - 1] + new[index]) / 2.0
        else:
            median = new[n // 2]
        return median
```

### Explanation
1. **Merging Arrays**: The algorithm starts by initializing two pointers, `p1` and `p2`, to the start of each array. It then compares elements from both arrays and appends the smaller element to the new array, moving the respective pointer forward. This process continues until one of the arrays is exhausted.
2. **Appending Remaining Elements**: After one array is exhausted, the remaining elements from the other array are appended to the new array in their sorted order.
3. **Finding the Median**: The median is calculated based on the length of the merged array. If the length is even, the median is the average of the two middle elements. If the length is odd, the median is the middle element.

This approach efficiently combines the two sorted arrays and computes the median in linear time, making it optimal for large input sizes.
