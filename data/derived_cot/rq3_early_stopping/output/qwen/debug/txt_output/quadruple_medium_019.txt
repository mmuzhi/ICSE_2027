=== RAW ANSWER ===

To solve this problem, we need to construct a quad-tree from a given binary grid. The quad-tree is built such that each node represents a subregion of the grid. If a subregion has all the same values (0 or 1), the node is a leaf; otherwise, it is a non-leaf node with four children representing the four quadrants of the subregion.

### Approach
1. **Base Case Handling**: If the subgrid is 1x1, return a leaf node with the value of the single cell.
2. **Recursive Division**: For larger grids, divide the current grid into four quadrants. Recursively build the quad-tree for each quadrant.
3. **Merging Condition**: After building the four child nodes, check if all four are leaves and have the same value. If so, merge them into a single leaf node. Otherwise, create a non-leaf node with the top-left child's value and the four children.

### Solution Code
```python
class Solution:
    def construct(self, grid: List[List[int]]) -> 'Node':
        level = len(grid)
        if level == 0:
            return None
        return self.buildTree(grid, 0, level, 0, level, level)
        
    def buildTree(self, grid, rs, re, cs, ce, level):
        if level == 1:
            return Node(grid[rs][cs], True, None, None, None, None)
        
        next_level = level // 2
        tl = self.buildTree(grid, rs, re - next_level, cs, ce - next_level, next_level)
        tr = self.buildTree(grid, rs, re - next_level, ce - next_level, ce, next_level)
        bl = self.buildTree(grid, re - next_level, re, cs, ce - next_level, next_level)
        br = self.buildTree(grid, re - next_level, re, ce - next_level, ce, next_level)
        
        if tl.isLeaf and tr.isLeaf and bl.isLeaf and br.isLeaf and tl.val == tr.val == bl.val == br.val:
            return Node(tl.val, True, None, None, None, None)
        else:
            return Node(tl.val, False, tl, tr, bl, br)
```

### Explanation
1. **Base Case Handling**: The function checks if the current subgrid is 1x1. If so, it returns a leaf node with the value of that cell.
2. **Recursive Division**: For larger grids, the function divides the current grid into four quadrants. Each quadrant is processed recursively to build the corresponding child nodes.
3. **Merging Condition**: After building all four child nodes, the function checks if all are leaves and have the same value. If true, it returns a new leaf node with that value. Otherwise, it returns a non-leaf node with the top-left child's value and the four children as its subtrees.

This approach efficiently constructs the quad-tree by leveraging recursion and merging when possible, ensuring optimal performance and correctness.
