To solve this problem, we need to construct a quad-tree from a given grid of integers (0 and 1). The quad-tree is built by recursively dividing the grid into quadrants until each quadrant is either a single cell (leaf node) or contains uniform values (all 0 or all 1). 

### Approach
1. **Problem Analysis**: The task is to build a quad-tree where each node represents a subspace of the grid. The node can either be a leaf (if the subspace contains all the same values) or have four children (top-left, top-right, bottom-left, bottom-right) if the subspace contains mixed values.
2. **Intuition**: The solution involves recursively dividing the grid into four quadrants. For each quadrant, if it is a single cell, it becomes a leaf node. If the quadrant contains mixed values, it is further divided. The parent node is created only if all four children are leaves with the same value.
3. **Algorithm Selection**: The algorithm uses a divide-and-conquer approach. The grid is split into four equal parts at each recursive step. The base case is when the grid part is a single cell. For larger grids, the algorithm checks if all four quadrants are uniform and create a leaf node if true; otherwise, it creates a non-leaf node with the four children.
4. **Complexity Analysis**: The time complexity is O(n^2) where n is the grid size, as each cell is processed once. The space complexity is O(n^2) due to the recursion stack and storing the tree structure.
