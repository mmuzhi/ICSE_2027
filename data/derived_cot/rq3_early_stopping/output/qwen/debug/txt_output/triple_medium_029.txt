The solution involves using a backtracking approach to explore all possible ways to split the string into four parts, ensuring each part is a valid number.

### Approach
1. **Backtracking**: The solution uses a recursive backtracking method to explore all possible ways to split the string into four parts. The function `backtrack` is called recursively to build each part of the IP address.
2. **Base Case**: When the entire string is processed (i.e., the current index `i` equals the length of the string and exactly four parts are formed), the current combination of parts is added to the result list.
3. **Invalid Cases**: If more than four parts are formed or the current index exceeds the string length, the recursion backtracks.
4. **Handling Zero**: If the current character is '0', it can only be a single-digit part (since leading zeros are not allowed in multi-digit numbers).
5. **Non-Zero Handling**: For non-zero characters, the function tries to form parts of length 1, 2, or 3 digits, ensuring the numeric value is less than 256.

```python
class Solution:
    def restoreIpAddresses(self, s: str) -> List[str]:
        res = []
        cur = []
        def backtrack(i):
            if i == len(s) and len(cur) == 4:
                res.append(".".join(cur))
                return
            if len(cur) > 4 or i > len(s):
                return
            if s[i] == '0':
                cur.append(s[i])
                backtrack(i+1)
                cur.pop()
                return
            j = 0
            while j < 3 and i+j < len(s):
                num = int(s[i:i+j+1])
                if num < 256:
                    cur.append(s[i:i+j+1])
                    backtrack(i+j+1)
                    cur.pop()
                j += 1
        backtrack(0)
        return res
```

1. **Initialization**: The `res` list stores all valid IP addresses, and `cur` list keeps track of the current parts being formed.
2. **Backtracking Function**: The `backtrack` function starts from index `i` and processes the string recursively:
   - **Base Case**: If the entire string is processed (`i == len(s)`) and exactly four parts are formed (`len(cur) == 4`), the current parts are joined into a string and added to `res`.
   - **Invalid Cases**: If more than four parts are formed or the index exceeds the string length, the function returns without further processing.
   - **Zero Handling**: If the current character is '0', it is added as a single part, and the function proceeds to the next index.
   - **Non-Zero Handling**: For non-zero characters, the function tries to form parts of up to three digits, checks if the numeric value is less than 256, and proceeds recursively if valid.
3. **Result**: After processing the entire string, the function returns the list of all valid IP addresses.
