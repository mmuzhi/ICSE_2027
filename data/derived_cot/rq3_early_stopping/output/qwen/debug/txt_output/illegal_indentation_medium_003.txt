1. **Initialization**: Start with digits from 1 to 9 using a queue (deque for efficient popping from the front).
2. **BFS Exploration**: For each number dequeued, check if it lies within the specified range `[low, high]`. If it does, add it to the result list.
3. **Digit Appending**: For each number, determine the last digit. If this digit is less than 8, append the next digit (last digit + 1) to form a new number and enqueue it. This ensures that the digits remain in strictly increasing order.
4. **Termination**: The process continues until the queue is empty, ensuring all possible sequential digits are generated.

### Solution Code
```python
from collections import deque

class Solution:
    def sequentialDigits(self, low: int, high: int) -> list[int]:
        out = []
        queue = deque(range(1, 10))
        while queue:
            elem = queue.popleft()
            if low <= elem <= high:
                out.append(elem)
            last_digit = elem % 10
            if last_digit < 9:
                next_num = elem * 10 + (last_digit + 1)
                queue.append(next_num)
        return out
```

### Explanation
1. **Initialization**: The queue starts with digits 1 through 9, representing the smallest possible sequential digits.
2. **BFS Loop**: For each number dequeued, it checks if the number is within the range `[low, high]`. If it is, the number is added to the result list.
3. **Digit Appending**: The last digit of the current number is found using modulo operation. If this digit is less than 8, the next sequential digit (last digit + 1) is appended to the number, forming a new number which is enqueued for further processing.
4. **Efficiency**: The BFS ensures that all numbers are processed in order, and each number is generated exactly once, making the solution efficient and correct for the problem constraints.
