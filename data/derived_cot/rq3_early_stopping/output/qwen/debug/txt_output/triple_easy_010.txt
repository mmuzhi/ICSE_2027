### Solution Code

```python
class Solution:
    def findComplement(self, num: int) -> int:
        if num == 0:
            return 1
        s = bin(num)[2:]
        s = s.replace('0', '2').replace('1', '0').replace('2', '1')
        return int(s, 2)
```

1. **Binary Conversion**: The `bin(num)` function converts the number to a binary string prefixed with '0b'. The `[2:]` slice removes the '0b' prefix.
2. **Bit Flipping**: The `replace` method is used to swap each '0' to '1' and each '1' to '0'. This is done in two steps: first replace '0' with a temporary placeholder '2', then replace '1' with '0', and finally replace '2' with '1'.
3. **Conversion to Integer**: The modified binary string is converted back to an integer using `int(s, 2)`, where the second argument specifies the base (binary) for the conversion.
4. **Edge Case Handling**: If the input number is 0, the function directly returns 1, as the complement of 0 in binary is 1.
