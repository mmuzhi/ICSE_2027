=== RAW ANSWER ===

To solve this problem, we need to count the number of 3x3 contiguous submatrices within a given grid that form a magic square. A magic square is defined as a 3x3 matrix where all numbers from 1 to 9 are used exactly once, and the sums of the numbers in each row, column, and both main diagonals are all equal to 15.

### Approach
1. **Problem Analysis**: The task involves checking every possible 3x3 submatrix in the grid to determine if it meets the criteria of a magic square. The key properties of a magic square are:
   - All numbers from 1 to 9 are present exactly once.
   - The sum of each row, column, and both main diagonals is 15.

2. **Intuition**: For each possible top-left corner of a 3x3 submatrix in the grid, we extract the submatrix and check if it satisfies the magic square conditions. 

3. **Algorithm Selection**: 
   - Iterate over all possible starting positions (i, j) where a 3x3 submatrix can fit within the grid.
   - For each starting position, extract the 3x3 submatrix.
   - Check if the submatrix contains all numbers from 1 to 9 exactly once and if the sums of its rows, columns, and diagonals are all 15.

4. **Complexity Analysis**: 
   - The algorithm processes each 3x3 submatrix in the grid. If the grid has dimensions M x N, there are (M-2) * (N-2) such submatrices.
   - Each submatrix check involves verifying 9 unique numbers and summing 3 rows, 3 columns, and 2 diagonals, which is a constant-time operation.
   - Thus, the overall time complexity is O(M*N*9), which simplifies to O(M*N) since 9 is a constant factor.

### Solution Code
```python
class Solution:
    def numMagicSquaresInside(self, grid: List[List[int]]) -> int:
        M, N = len(grid), len(grid[0])
        res = 0
        
        def is_magic(arr):
            nums = []
            for row in arr:
                nums.extend(row)
            if len(nums) != 9:
                return False
            if set(nums) != set(range(1, 10)):
                return False
            
            for i in range(3):
                if sum(arr[i]) != 15:
                    return False
            
            for j in range(3):
                col_sum = arr[0][j] + arr[1][j] + arr[2][j]
                if col_sum != 15:
                    return False
            
            if arr[0][0] + arr[1][1] + arr[2][2] != 15:
                return False
            if arr[0][2] + arr[1][1] + arr[2][0] != 15:
                return False
            
            return True
        
        for i in range(M - 2):
            for j in range(N - 2):
                matrix = []
                for x in range(i, i + 3):
                    matrix.append(tuple(grid[x][j:j+3]))
                if is_magic(matrix):
                    res += 1
        
        return res
```
