To solve this problem, we need to find the third maximum number in a given list of integers. If there are fewer than three distinct numbers, we should return the maximum number from the list. 

### Approach
1. **Problem Analysis**: The task is to determine the third largest distinct number in the input list. If there are fewer than three distinct numbers, the solution should return the maximum number available.
2. **Intuition**: By converting the list to a set, we eliminate duplicate values, focusing only on distinct numbers. Sorting the distinct numbers in descending order allows us to easily access the third largest element if it exists.
3. **Algorithm Selection**: 
   - **Step 1**: Convert the input list to a set to remove duplicates.
   - **Step 2**: Sort the distinct numbers in descending order.
   - **Step 3**: Check if there are at least three distinct numbers. If not, return the largest number (first element of the sorted list). Otherwise, return the third largest number (third element in the sorted list).
4. **Complexity Analysis**: The operations involve converting the list to a set (O(n)), sorting the set (O(m log m) where m is the number of distinct elements), and accessing the third element (O(1)). The overall complexity is dominated by the sorting step, making it O(n log n) in the worst case.

### Solution Code
```python
class Solution:
    def thirdMax(self, nums: List[int]) -> int:
        unique_nums = sorted(set(nums), reverse=True)
        if len(unique_nums) < 3:
            return unique_nums[0]
        else:
            return unique_nums[2]
```

### Explanation
1. **Conversion to Set**: The input list is converted to a set to eliminate duplicate values, ensuring we only work with distinct numbers.
2. **Sorting**: The distinct numbers are sorted in descending order to easily access the largest, second largest, and third largest elements.
3. **Condition Check**: If the number of distinct elements is less than three, the solution returns the largest number (first element of the sorted list). Otherwise, it returns the third largest number (third element in the sorted list).
This approach efficiently handles both scenarios (presence or absence of three distinct numbers) and ensures correctness by leveraging sorting and set operations.
