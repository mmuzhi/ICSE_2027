3. **DFS Traversal**: For each course, if it hasn't been visited, perform DFS. During DFS, mark the course as visited and add it to the recursion stack. For each prerequisite, if it hasn't been visited, recursively check for cycles. If it has been visited and is in the recursion stack, a cycle is detected.
4. **Result Determination**: If any cycle is detected during the traversal, return `False`. If no cycles are found after checking all courses, return `True`.

    def canFinish(self, numCourses: int, prerequisites: List[List[int]]) -> bool:
        graph = {i: [] for i in range(numCourses)}
        for a, b in prerequisites:

        visited = set()
        rec_stack = set()
        
        def dfs(course):
            visited.add(course)
            rec_stack.add(course)
            for p in graph[course]:
                if p not in visited:
                    if not dfs(p):
                        return False
                elif p in rec_stack:
                    return False

            return True
        
        for i in range(numCourses):
            if i not in visited:
                if not dfs(i):
                    return False
        return True

1. **Graph Construction**: The graph is built by iterating through the prerequisites. For each prerequisite pair `[a, b]`, an edge from `b` to `a` is added, indicating that `a` is a prerequisite for `b`.
2. **Cycle Detection**: The DFS function checks for cycles by marking nodes as visited and tracking nodes in the current recursion stack. If a node is encountered that is already in the recursion stack, a cycle exists.
3. **DFS Traversal**: The main loop iterates over all courses. For each unvisited course, DFS is initiated. If DFS returns `False` (indicating a cycle), the function immediately returns `False`.
4. **Result**: If no cycles are detected during the traversal of all courses, the function returns `True`, indicating all courses can be finished in some order.
