=== RAW ANSWER ===

To solve this problem, we need to find the median of two sorted arrays without merging them entirely. The median is the middle value in an ordered list of numbers. If the combined length of the two arrays is even, the median is the average of the two middle numbers; otherwise, it is the middle number.

### Approach
1. **Initialization**: Start with two pointers, `p1` and `p2`, initialized to 0. These pointers will traverse the two arrays.
2. **Merging Arrays**: Compare elements from both arrays using the pointers. Append the smaller element to a result list and move the corresponding pointer forward. This step ensures that the elements are merged in sorted order.
3. **Remaining Elements**: After one array is exhausted, append the remaining elements of the other array to the result list.
4. **Finding Median**: Calculate the median based on the length of the merged list. If the length is odd, the median is the middle element. If even, it is the average of the two middle elements.

### Solution Code
```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        p1 = 0
        p2 = 0
        merged = []
        
        while p1 < len(nums1) and p2 < len(nums2):
            if nums1[p1] < nums2[p2]:
                merged.append(nums1[p1])
                p1 += 1
            else:
                merged.append(nums2[p2])
                p2 += 1
        
        while p1 < len(nums1):
            merged.append(nums1[p1])
            p1 += 1
            
        while p2 < len(nums2):
            merged.append(nums2[p2])
            p2 += 1
            
        n = len(merged)
        if n % 2 == 1:
            median = merged[n//2]
        else:
            median = (merged[n//2 - 1] + merged[n//2]) / 2.0
        
        return median
```

### Explanation
1. **Initialization**: The pointers `p1` and `p2` start at the beginning of each array.
2. **Merging**: The while loop continues until one of the arrays is fully traversed. In each iteration, the smaller element between `nums1[p1]` and `nums2[p2]` is added to the merged list, and the corresponding pointer is incremented.
3. **Remaining Elements**: Once one array is exhausted, the remaining elements of the other array are appended to the merged list in their natural order.
4. **Median Calculation**: The median is determined based on the length of the merged list. If the length is odd, the median is the middle element. If even, it is the average of the two middle elements. This approach efficiently finds the median without fully merging the arrays, optimizing both time and space complexity.
