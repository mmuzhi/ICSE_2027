To solve this problem, we need to find an arrangement (path) that uses each edge exactly once in a directed graph. This is known as an Eulerian path. The solution involves checking the necessary conditions for the existence of an Eulerian path and then using a depth-first search (DFS) to construct the path.

### Approach
1. **Graph Construction and Degree Calculation**: 
   - Build a directed graph from the given pairs and compute the net out-degree for each node (out-degree minus in-degree).

2. **Eulerian Path Conditions**:
   - For an Eulerian path to exist in a directed graph, the net degrees of all nodes must satisfy one of two conditions:
     - All nodes have a net degree of zero (Eulerian cycle).
     - Exactly one node has a net degree of +1 (start node), one node has a net degree of -1 (end node), and all other nodes have a net degree of zero.

3. **DFS Traversal**:
   - If the conditions are met, perform a DFS starting from the start node (or any node in the case of an Eulerian cycle) to traverse all edges. The DFS removes edges as it traverses them, ensuring each edge is used exactly once.
   - The traversal order is reversed to form the correct path.

### Solution Code
```python
from collections import defaultdict
from typing import List

class Solution:
    def validArrangement(self, pairs: List[List[int]]) -> List[List[int]]:
        graph = defaultdict(list)
        degree = defaultdict(int)
        for x, y in pairs:
            graph[x].append(y)
            degree[x] += 1
            degree[y] -= 1
        
        start = None
        end = None
        for node, d in degree.items():
            if d == 1:
                start = node
            elif d == -1:
                end = node
            elif d != 0:
                return []
        
        if start is None and end is None:
            start = next(iter(graph.keys())) if graph else None
        elif start is None:
            return []
        elif end is None:
            return []
        
        ans = []
        def dfs(x):
            while graph[x]:
                neighbor = graph[x].pop()
                dfs(neighbor)
            ans.append(x)
        
        dfs(start)
        ans.reverse()
        return [[ans[i], ans[i+1]] for i in range(len(ans)-1)]
```

3. **DFS Traversal**: Starting from the identified start node (or any node if it's an Eulerian cycle), the DFS recursively traverses the graph, popping edges from the list as it goes. The traversal order is reversed to form the correct path.
4. **Path Construction**: The reversed list of nodes is converted into a list of edges, which is returned as the result.
