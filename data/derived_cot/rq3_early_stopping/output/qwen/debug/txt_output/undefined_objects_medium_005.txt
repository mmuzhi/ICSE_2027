To solve this problem, we need to determine the minimum number of steps required to transform string `s` into string `t` by only adding characters. Each step allows us to add one character to `s`. The solution involves comparing the frequency of characters in both strings and calculating the deficit of characters in `s` compared to `t`.

### Approach
1. **Frequency Counters**: Use the `Counter` class from the `collections` module to count the occurrences of each character in both strings `s` and `t`.
2. **Calculate Deficit**: For each character in `t`, check if its count in `s` is less than its count in `t`. If so, the difference between these counts gives the number of characters needed to be added to `s` to match `t`.
3. **Sum the Deficits**: Sum all the deficits calculated in the previous step to get the total number of steps required.

### Solution Code
```python
from collections import Counter

class Solution:
    def minSteps(self, s: str, t: str) -> int:
        cnt1 = Counter(s)
        cnt2 = Counter(t)
        sm = 0
        for char in cnt2:
            if cnt1[char] < cnt2[char]:

        return sm
```

3. **Result**: The total steps required to transform `s` into `t` by adding characters is the sum of all deficits, which is returned as the result.
