### Solution Code
```python
class Solution:
    def maximumNumber(self, num: str, change: List[int]) -> str:
        num_list = list(num)
        on = False
        for i, ch in enumerate(num_list):
            x = int(ch)
            if x < change[x]:
                on = True
                num_list[i] = str(change[x])
            elif x > change[x] and on:
                break
        return ''.join(num_list)
```

### Explanation
1. **Initialization**: The input string is converted into a list for easier manipulation. A boolean flag `on` is initialized to `False` to track whether replacements are allowed.
2. **Iteration and Replacement**: For each digit in the string:
   - The current digit is converted to an integer `x`.
   - If `x` is less than the corresponding value in the `change` list (`change[x]`), the digit is replaced, and the `on` flag is set to `True`.
   - If `x` is greater than `change[x]` and the `on` flag is `True`, the loop breaks immediately to stop further replacements.
3. **Result Construction**: The modified list is joined back into a string and returned as the result.

This approach ensures that the number is maximized by replacing digits from left to right while stopping at the first digit that cannot be replaced, thus optimizing the transformation process.
