To solve this problem, we need to calculate the total points based on a series of operations performed on a stack of scores. The operations include doubling the last score, canceling the last operation, adding the last two scores, or directly inputting a score. The challenge is to correctly interpret each operation and maintain the stack of scores accordingly.

### Approach
1. **Initialization**: Start with an empty stack to keep track of the scores.
2. **Iterate through Operations**: For each operation in the list:
   - **'D' Operation**: Double the last score in the stack and push the result back onto the stack.
   - **'C' Operation**: Remove the last score from the stack.
   - **'+' Operation**: Add the last two scores in the stack and push the result onto the stack.
   - **Default Operation**: Convert the operation string to an integer and push it onto the stack.
3. **Summing Scores**: After processing all operations, sum all the scores in the stack to get the total points.

### Solution Code
```python
class Solution:
    def calPoints(self, operations: List[str]) -> int:
        stack = []
        for op in operations:
            if op == 'D':
                stack.append(2 * stack[-1])
            elif op == 'C':
                stack.pop()
            elif op == '+':
                stack.append(stack[-1] + stack[-2])
            else:
                stack.append(int(op))
        return sum(stack)
```

### Explanation
- **Stack Handling**: The stack is used to keep track of the current scores. Each operation modifies the stack in a specific way.
- **Operations Processing**:
  - **'D'**: This operation doubles the last score by accessing the top element of the stack and pushing the doubled value back onto the stack.
  - **'C'**: This operation removes the last score from the stack, effectively canceling the last operation.
  - **'+'**: This operation adds the last two scores in the stack and pushes the result back onto the stack.
  - **Default**: Any other operation is treated as a direct score input, converted from a string to an integer and pushed onto the stack.
- **Result Calculation**: The sum of all elements in the stack after processing all operations gives the total points.

This approach efficiently processes each operation in linear time, ensuring that all operations are handled correctly and the final result is computed accurately.
