To solve this problem, we need to determine if two given lists of strings, when concatenated, form the same string. The solution involves joining all strings in each list and then comparing the resulting strings.

### Approach
1. **Join Strings**: Concatenate all strings in the first list (`word1`) into a single string.
2. **Join Strings**: Similarly, concatenate all strings in the second list (`word2`) into another single string.
3. **Compare Strings**: Check if the two resulting strings are identical. If they are, return `True`; otherwise, return `False`.

The key insight here is that the order of characters in the concatenated strings must be the same for both lists to be considered equal. The solution efficiently checks this by directly comparing the concatenated strings.

### Solution Code
```python
class Solution:
    def arrayStringsAreEqual(self, word1: List[str], word2: List[str]) -> bool:
        word3 = "".join(word1)
        word4 = "".join(word2)
        return word3 == word4
```

### Explanation
1. **Joining Strings**: The `join()` method is used to concatenate all strings in each list. For example, `"".join(word1)` combines all elements of `word1` into a single string without any separators.
2. **Comparison**: The resulting strings from both lists are compared using the equality operator (`==`). This comparison checks if both strings contain exactly the same sequence of characters.
3. **Return Result**: The function returns `True` if the strings are identical and `False` otherwise. This approach efficiently handles the problem by leveraging simple string operations and comparison.
