To solve this problem, we need to find the maximum gap between any two consecutive numbers in a sorted list of integers. The initial code provided attempts to solve this problem but contains several bugs, including incorrect variable names, syntax errors, and logical mistakes.

### Approach
1. **Sorting the Array**: The first step is to sort the input array in ascending order. This allows us to easily compute the differences between consecutive elements.
2. **Handling Edge Cases**: If the array has fewer than two elements, the maximum gap is zero since no pairs exist.
3. **Tracking Maximum Gap**: Initialize a variable to keep track of the maximum gap found. Iterate through the sorted array, compute the difference between each pair of consecutive elements, and update the maximum gap whenever a larger difference is found.

### Solution Code
```python
class Solution:
    def maximumGap(self, nums: List[int]) -> int:
        if len(nums) < 2:
            return 0
        nums = sorted(nums)
        max_diff = -1
        for i in range(len(nums) - 1):
            gap = nums[i+1] - nums[i]
            if gap > max_diff:
                max_diff = gap
        return max_diff
```

### Explanation
1. **Sorting**: The array is sorted to ensure that we can efficiently compute the differences between consecutive elements. For example, given `[5, 10, 3]`, sorting results in `[3, 5, 10]`.
2. **Edge Case Handling**: If the array contains only one element (e.g., `[5]`), the function returns `0` because there are no pairs to compare.
3. **Tracking Maximum Gap**: The variable `max_diff` is initialized to `-1` to start tracking the largest gap. For each consecutive pair in the sorted array, the difference is calculated. If this difference is larger than the current `max_diff`, `max_diff` is updated. For instance, in the sorted array `[3, 5, 10]`, the gaps are `2` and `5`, so the maximum gap is `5`.

This approach ensures that we efficiently find the maximum gap by leveraging sorting and a linear scan through the array, making it both correct and optimal.
