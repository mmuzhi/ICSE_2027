### Approach
1. **Stack Initialization**: We start with an empty stack to keep track of the scores.
2. **Processing Operations**: For each operation in the input list:
   - **'D'**: Double the last score in the stack and push the result onto the stack.
   - **'C'**: Remove the last score from the stack.
   - **'+'**: Add the last two scores in the stack and push the result onto the stack.
   - **Other Strings**: Treat the string as an integer and push it onto the stack.
3. **Summing Scores**: After processing all operations, compute the sum of all elements in the stack.

### Solution Code
```python
class Solution:
    def calPoints(self, operations: List[str]) -> int:
        stack = []
        for op in operations:
            if op == 'D':
                stack.append(2 * stack[-1])
            elif op == 'C':
                stack.pop()
            elif op == '+':
                stack.append(stack[-1] + stack[-2])
            else:
                stack.append(int(op))
        return sum(stack)
```

### Explanation
- **Stack Handling**: The stack is used to maintain the sequence of scores, allowing efficient access and modification of the last element.
- **Operations Processing**:
  - **'D'**: The last score is doubled and pushed onto the stack.
  - **'C'**: The last score is removed from the stack.
  - **'+'**: The sum of the last two scores is computed and pushed onto the stack.
  - **Other Strings**: The string is converted to an integer and pushed onto the stack.
- **Sum Calculation**: The total sum of all scores in the stack is returned after processing all operations. This approach efficiently handles each operation in linear time, making it optimal for the given problem constraints.
