=== RAW ANSWER ===

To solve this problem, we need to add 1 to a non-negative integer represented as a list of digits and return the result as a list of digits. The key challenge is handling the carry-over when incrementing the number, especially when the digits are all 9s.

### Approach
1. **Convert Digits to String**: Combine all the digits into a single string to form the original number.
2. **Increment the Number**: Convert the string to an integer, increment it by 1, and then convert it back to a string.
3. **Convert Back to Digits**: Split the resulting string into individual digits and convert each digit back to an integer. This handles the carry-over naturally, as converting the incremented number back to a string automatically appends any necessary digits (e.g., 99 becomes 100).

This approach efficiently handles all cases, including those where the increment causes an increase in the number of digits (like 999 becoming 1000). The solution avoids manual carry-over handling by leveraging string conversion, which simplifies the logic and ensures correctness.

### Solution Code
```python
class Solution:
    def plusOne(self, digits: List[int]) -> List[int]:
        s = ''.join(map(str, digits))
        i = int(s) + 1
        return list(map(int, str(i)))
```

### Explanation
1. **String Conversion**: The digits are joined into a string to form the original number. For example, the list `[1, 2, 3]` becomes the string "123".
2. **Increment**: The string is converted to an integer, incremented by 1, and then converted back to a string. For example, "123" becomes 124, which is then converted back to "124".
3. **Digit Conversion**: The resulting string is split into individual characters, each converted back to an integer. For example, "124" becomes the list `[1, 2, 4]`.

This method efficiently handles all edge cases, including numbers ending with one or more 9s, by automatically managing the carry-over through the string conversion process. The solution is concise, readable, and ensures correctness for all valid inputs.
