The window slides over the string by the length of one word at a time, updating the frequency dictionary by removing the leftmost word and adding the new word on the right.

3. **Algorithm Selection**: 
   - **Frequency Dictionary**: We use a dictionary to count occurrences of each word in the current window.
   - **Sliding Window**: The window starts at each possible offset (from 0 to the length of the first word minus one) and slides one word at a time, updating the frequency dictionary and checking if it matches the frequency of words in the input list.

4. **Complexity Analysis**: 
   - **Time Complexity**: O(n * m), where `n` is the length of the string `s` and `m` is the total length of all words. For each starting offset, we slide the window across the string, performing constant-time operations for each slide.
   - **Space Complexity**: O(k), where `k` is the number of distinct words in the input list, due to the frequency dictionary.

```python
import collections
from typing import List

class Solution:
    def findSubstring(self, s: str, words: List[str]) -> List[int]:
        if not s or not words:
            return []
        wlen = len(words[0])
        slen = wlen * len(words)
        n = len(s)
        if n < slen:
            return []
        occ = collections.Counter(words)
        
        def test():
            if len(track) != len(occ):
                return False
            for key in occ:
                if track.get(key, 0) != occ[key]:
                    return False
            return True
        
        res = []
        for k in range(wlen):
            track = collections.Counter()
            for i in range(k, k + slen, wlen):
                word = s[i:i + wlen]
                if word in occ:
                    track[word] += 1
            if test():
                res.append(k)
            start_index = k + wlen
            while start_index + slen <= n:
                word_to_remove = s[start_index - wlen:start_index]
                if word_to_remove in occ:
                    track[word_to_remove] -= 1
                word_to_add = s[start_index + slen - wlen:start_index + slen]
                if word_to_add in occ:
                    track[word_to_add] += 1
                if test():
                    res.append(start_index)
                start_index += wlen
        return res
```

### Explanation
1. **Initial Checks**: The code first checks if the string `s` or the list `words` is empty, returning an empty list if true. It also checks if the string is shorter than the combined length of all words, in which case no valid substring exists.

2. **Frequency Dictionary**: The `occ` dictionary stores the count of each word from the input list.

3. **Test Function**: The `test` function checks if the current window's word frequencies match those in the `occ` dictionary.

4. **Sliding Window**: For each starting offset `k` (from 0 to the length of the first word minus one), the code initializes a frequency dictionary `track` and builds it by processing each word in the initial window. If the window's frequencies match `occ`, the starting index `k` is added to the result.

5. **Window Sliding**: The window slides one word at a time. For each new position, the leftmost word is removed from `track` and the new word added to the right is included. The `test` function is called again to check if the updated window matches the required word frequencies.

6. **Result Compilation**: Valid starting indices are collected in the result list and returned after processing all possible windows.

This approach efficiently checks all possible substrings by leveraging a sliding window and a frequency dictionary, ensuring optimal performance and correctness.
