To solve this problem, we need to rearrange the elements of an array such that all elements less than the pivot come first, followed by elements equal to the pivot, and then elements greater than the pivot. The relative order of elements in each category should remain the same as in the original array.

### Approach
1. **Initialization**: We initialize three empty lists to hold elements less than, equal to, and greater than the pivot.
2. **Iteration**: We iterate through each element in the input array:
   - If the element is less than the pivot, it is added to the first list.
   - If the element is equal to the pivot, it is added to the second list.
   - If the element is greater than the pivot, it is added to the third list.
3. **Combining Lists**: The final result is obtained by concatenating the three lists in the order: less than pivot, equal to pivot, and greater than pivot.

### Solution Code
```python
class Solution:
    def pivotArray(self, nums: List[int], pivot: int) -> List[int]:
        lst, val, tmp = [], [], []
        for i in nums:
            if i < pivot:
                lst.append(i)
            elif i == pivot:
                val.append(i)
            else:
                tmp.append(i)
        return lst + val + tmp
```

### Explanation
- **Initialization**: Three lists (`lst`, `val`, `tmp`) are initialized to store elements based on their relation to the pivot.
- **Iteration**: For each element in the input array, the code checks its value relative to the pivot:
  - Elements less than the pivot are added to `lst`.
  - Elements equal to the pivot are added to `val`.
  - Elements greater than the pivot are added to `tmp`.
- **Combining Lists**: The result is formed by concatenating `lst`, `val`, and `tmp` in that order, ensuring the elements are grouped correctly while maintaining their original order within each group.

This approach efficiently categorizes each element in linear time, making it optimal for the given problem constraints.
