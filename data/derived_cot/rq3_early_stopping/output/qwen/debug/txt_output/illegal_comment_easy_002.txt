=== RAW ANSWER ===

To solve this problem, we need to compute the alternating digit sum of a given integer. The alternating digit sum is calculated by summing the digits at even indices (0-indexed) and subtracting the digits at odd indices. 

### Approach
The approach involves the following steps:
1. Convert the integer `n` into a string to facilitate digit-by-digit processing.
2. Iterate over each digit in the string representation of `n`.
3. For each digit at an even index (0-indexed), add its integer value to a running total.
4. For each digit at an odd index, subtract its integer value from the running total.
5. Return the final computed total.

### Solution Code
```python
class Solution:
    def alternateDigitSum(self, n: int) -> int:
        s = str(n)
        count = 0
        for i in range(len(s)):
            if i % 2 == 0:
                count += int(s[i])
            else:
                count -= int(s[i])
        return count
```

### Explanation
1. **String Conversion**: The integer `n` is converted to a string `s` to allow easy access to each digit via indexing.
2. **Iteration**: The loop runs from the first digit to the last digit of the string.
3. **Alternating Sum Calculation**: 
   - For even indices (i.e., positions 0, 2, 4, ...), the corresponding digit's integer value is added to `count`.
   - For odd indices (i.e., positions 1, 3, 5, ...), the corresponding digit's integer value is subtracted from `count`.
4. **Result**: The final value of `count` after processing all digits is returned, which is the alternating digit sum of the input integer.

This approach efficiently processes each digit in linear time, making it optimal for the given problem constraints.
