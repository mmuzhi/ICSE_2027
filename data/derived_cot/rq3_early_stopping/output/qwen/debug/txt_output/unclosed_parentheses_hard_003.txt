The key challenge is to determine which nodes to skip to minimize the total cost while ensuring all trips are covered.

### Approach

2. **Frequency Calculation**: For each node, calculate how many times it is visited across all trips. This involves finding the Lowest Common Ancestor (LCA) for each trip and updating the frequency counts along the path from each node to the LCA.
3. **Dynamic Programming (DP)**: Use a DP approach where each node has two states:

   - **State 1**: The node is skipped (half price is paid).
   The DP function computes the minimum cost for each state by considering the children's states and combining them appropriately.

                stack.append(neighbor)
                
        def lca(a, b):
            if level_arr[a] < level_arr[b]:
                a, b = b, a
            while level_arr[a] > level_arr[b]:
                a = parent_arr[a]
            if a == b:

            return a

                x = parent_arr[x]
            freq[l] += 1
            x = j

                freq[x] += 1
                x = parent_arr[x]
