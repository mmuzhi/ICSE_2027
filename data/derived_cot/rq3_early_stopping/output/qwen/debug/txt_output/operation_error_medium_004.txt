To solve this problem, we need to calculate the minimum fuel cost required to transport all people from every city (node) to the root city (node 0) in a tree-structured graph. Each bus can carry up to 'seats' number of passengers, and the fuel cost is determined by the number of buses used, with each bus costing 1 unit of fuel regardless of the number of passengers it carries.

### Approach
1. **Graph Representation**: Represent the cities and roads as an adjacency list. This helps in traversing the tree structure efficiently.
2. **Tree Traversal**: Perform a depth-first search (DFS) starting from the root node (node 0). During the traversal, count the number of people in each subtree.
3. **Bus Calculation**: For each subtree, calculate the number of buses needed to transport all people in that subtree to the parent node. The number of buses required is the ceiling of the division of the total people by the bus capacity (seats). This ensures that any remaining passengers after dividing by the bus capacity are accommodated in an additional bus.
4. **Summing Fuel Cost**: Accumulate the fuel cost by summing the number of buses required for each subtree during the DFS traversal.

        visited = [0] * n
        visited[0] = 1
        
        res = [0]
        
        def dfs(i):
            if visited[i] == 1:
                return 0
            visited[i] = 1
            tot = 1
            for neighbor in adjacencyList[i]:
                tot += dfs(neighbor)

            res[0] += buses

### Explanation
1. **Graph Representation**: The adjacency list is built by iterating through the roads and adding each connection to the respective node's list. This ensures that the graph is represented correctly for traversal.
2. **DFS Traversal**: The DFS function starts from each child of the root node (node 0) and recursively visits all nodes in the subtree. During traversal, it counts the number of people (nodes) in the subtree.
3. **Bus Calculation**: For each subtree, the number of buses required is calculated using ceiling division. This is done by `(tot + seats - 1) // seats`, which ensures that any remainder from the division results in an additional bus being used.
4. **Fuel Cost Accumulation**: The total fuel cost is accumulated in the `res` array during the DFS traversal. The result is the sum of all buses required across all subtrees, providing the minimum fuel cost to transport all people to the root node.

This approach efficiently traverses the tree, calculates the required buses for each subtree, and sums the fuel costs to determine the minimum fuel needed for the entire transportation process.
