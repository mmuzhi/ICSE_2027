=== RAW ANSWER ===

To solve this problem, we need to find the median of two sorted arrays efficiently. The median is the middle value in an ordered list of numbers. If the combined length of the two arrays is even, the median is the average of the two middle numbers.

### Approach
1. **Merge the Arrays**: Combine the two sorted arrays into a single sorted array. This can be done by comparing elements from both arrays and appending the smaller element to the result array until one of the arrays is exhausted. The remaining elements of the non-exhausted array are then appended to the result.
2. **Calculate the Median**: Once the arrays are merged into a single sorted array, the median can be determined by checking the length of the resulting array. If the length is odd, the median is the middle element. If the length is even, the median is the average of the two middle elements.

### Solution Code
```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        new = []
        p1 = 0
        p2 = 0
        
        while p1 < len(nums1) and p2 < len(nums2):
            if nums1[p1] < nums2[p2]:
                new.append(nums1[p1])
                p1 += 1
            else:
                new.append(nums2[p2])
                p2 += 1
                
        while p1 < len(nums1):
            new.append(nums1[p1])
            p1 += 1
            
        while p2 < len(nums2):
            new.append(nums2[p2])
            p2 += 1
            
        n = len(new)
        if n % 2 == 1:
            median = new[n//2]
        else:
            median = (new[n//2 - 1] + new[n//2]) / 2.0
            
        return median
```

### Explanation
1. **Merging Arrays**: The code initializes two pointers, `p1` and `p2`, to traverse `nums1` and `nums2` respectively. By comparing elements at these pointers, the smaller element is appended to the result array `new`, and the corresponding pointer is incremented. This process continues until one of the arrays is fully traversed. The remaining elements of the non-exhausted array are then appended to `new`.
2. **Finding the Median**: The median is calculated based on the length of `new`. If the length is odd, the median is the middle element. If even, it is the average of the two middle elements. The result is returned as a float to handle both cases appropriately.

This approach ensures that the solution is both correct and efficient, handling all edge cases including even and odd lengths of the combined arrays.
