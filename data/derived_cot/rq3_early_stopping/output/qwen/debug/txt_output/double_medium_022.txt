### Approach
The efficient approach involves using a monotonic stack to determine the left and right boundaries for each element where it is the minimum in any subarray. Here's the step-by-step breakdown:

1. **Left Boundaries Calculation**: For each element, find the nearest previous element that is smaller than the current element. This helps in determining the range where the current element is the minimum to the left.
2. **Right Boundaries Calculation**: Similarly, for each element, find the nearest next element that is smaller than the current element. This helps in determining the range where the current element is the minimum to the right.
3. **Sum Calculation**: For each element, multiply its value by the number of subarrays where it is the minimum (determined by the left and right boundaries) and accumulate the result, applying modulo \(10^9 + 7\) to handle large numbers.

class Solution:
    def sumSubarrayMins(self, arr: List[int]) -> int:
