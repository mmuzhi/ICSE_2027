To solve this problem, we need to determine if all the frequencies of the numbers in the given array are unique. The solution involves counting the occurrences of each number in the array and then checking if these counts are all distinct.

### Approach
1. **Count Frequencies**: Use the `Counter` class from the `collections` module to count the occurrences of each number in the array. This gives us a dictionary where keys are the numbers and values are their respective counts.
2. **Check Uniqueness of Counts**: Convert the values (counts) of the dictionary into a set to eliminate duplicates. If the length of this set is equal to the number of distinct numbers (keys) in the dictionary, it means all counts are unique. Otherwise, there are duplicate counts.

### Solution Code
```python
from collections import Counter

class Solution:
    def uniqueOccurrences(self, arr: List[int]) -> bool:
        d = Counter(arr)
        return len(set(d.values())) == len(d)
```

### Explanation
1. **Counting Frequencies**: The `Counter` class efficiently counts the occurrences of each element in the array, storing them in a dictionary-like structure where keys are elements and values are their counts.
2. **Checking Uniqueness**: By converting the values (counts) of the dictionary into a set, we can check for duplicates. If the set's length matches the number of distinct keys (elements), it means all counts are unique. The solution returns `True` if this condition is met, otherwise `False`.

This approach ensures that we efficiently and correctly determine if all element frequencies in the array are unique, leveraging Python's built-in data structures for simplicity and clarity.
