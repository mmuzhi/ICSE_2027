To solve this problem, we need to modify an array based on a series of operations where each operation specifies replacing every occurrence of a particular value with another value. The challenge is to ensure that all occurrences of the specified value in the array are replaced in each operation, considering the sequential nature of the operations.

### Approach
1. **Problem Analysis**: The problem requires processing each operation in sequence, replacing all instances of the old value with the new value in the array. The key observation is that each operation affects the entire array, and subsequent operations should consider the modified array from previous operations.
2. **Intuition**: For each operation, we scan the entire array to replace all occurrences of the old value with the new value. This ensures that all instances are updated, even if they appear multiple times in the array.
3. **Algorithm Selection**: The solution involves iterating through each operation and then through each element of the array. For each element matching the old value in the current operation, it is replaced with the new value. This approach guarantees that all occurrences are updated in each operation.
4. **Complexity Analysis**: The time complexity is O(m * n), where m is the number of operations and n is the number of elements in the array. This is because for each operation, we potentially scan the entire array once. The space complexity is O(1) as we modify the array in place (or use a copy if specified, but the problem allows in-place modification).

### Solution Code
```python
class Solution:
    def arrayChange(self, nums: List[int], operations: List[List[int]]) -> List[int]:
        for op in operations:
            old, new = op
            for i in range(len(nums)):
                if nums[i] == old:
                    nums[i] = new
        return nums
```

### Explanation
1. **Initialization**: The solution starts by iterating over each operation in the given list.
2. **Operation Processing**: For each operation, the old value and new value are extracted. The array is then scanned from start to end.
3. **Replacement**: During the scan, every element that matches the old value is replaced with the new value. This ensures all occurrences of the old value are updated in the current operation.
4. **Result**: After processing all operations, the modified array is returned. The solution efficiently handles the sequential nature of operations by updating the array in place, ensuring all subsequent operations operate on the latest version of the array.

This approach guarantees that each operation fully updates the array, considering all previous modifications, leading to the correct final result.
