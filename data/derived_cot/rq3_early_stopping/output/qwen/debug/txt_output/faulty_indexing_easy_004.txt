To solve this problem, we need to increment a number represented as a list of digits by one and return the result as a list of digits. The key challenge is handling cases where incrementing the number leads to a carry-over, such as when the number is all 9s.

### Approach
1. **Convert Digits to String**: The list of digits is converted into a string to facilitate arithmetic operations.
2. **Increment the Number**: The string is converted to an integer, incremented by one, and then converted back to a string.
3. **Convert Back to List**: The resulting string is split into individual digits, which are converted back into a list of integers.
4. **Return the Result**: The list of digits is returned as the solution.

This approach efficiently handles carry-overs by leveraging Python's ability to handle large integers and string manipulations. The solution ensures that all edge cases, such as numbers ending with 9s, are correctly processed.

### Solution Code
```python
class Solution:
    def plusOne(self, digits: List[int]) -> List[int]:
        s = ''.join(map(str, digits))
        i = int(s) + 1
        li = list(map(int, str(i)))
        return li
```

### Explanation
1. **String Conversion**: The input list of digits is joined into a single string. For example, `[1, 2, 3]` becomes `"123"`.
2. **Increment Operation**: The string is converted to an integer, incremented by one, and then converted back to a string. For example, `"123"` becomes `124` as a string.
3. **Digit Splitting**: The resulting string is split into individual characters, each converted back to an integer. For example, `"124"` becomes `[1, 2, 4]`.
4. **Result Return**: The list of integers is returned, which represents the incremented number. This approach efficiently handles all cases, including those with carry-overs (e.g., `[9, 9]` becomes `[1, 0, 0]`).
