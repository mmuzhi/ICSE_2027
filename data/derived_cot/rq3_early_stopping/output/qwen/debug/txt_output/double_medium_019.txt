    def corpFlightBookings(self, bookings: List[List[int]], n: int) -> List[int]:

Each booking specifies a range of flights [l, r] and the number of seats to book. The challenge is to efficiently compute the total bookings for each flight without resorting to a brute-force approach, which would be too slow for large inputs.

### Approach
1. **Difference Array Technique**: This technique is used to efficiently handle range updates. We initialize an array `arr` of size `n+1` with zeros. For each booking [l, r, seats], we:
   - Add `seats` to `arr[l-1]` (since flight numbers start at 1, we adjust the index by subtracting 1).
   - Subtract `seats` from `arr[r]` if `r < n` (to cancel the effect of the booking after the last flight in the range).

2. **Prefix Sum Calculation**: After processing all bookings, the array `arr` will contain the necessary adjustments. The actual number of bookings for each flight is obtained by computing the prefix sum of the first `n` elements of `arr`. This gives us the cumulative number of bookings up to each flight.
