### Approach
1. **Sort the List**: The first step is to sort the input list in ascending order. This allows us to easily compute the differences between adjacent elements.
2. **Handle Edge Cases**: If the list has fewer than two elements, the maximum gap is zero since there are no adjacent elements to compare.

### Solution Code
```python
class Solution:
    def maximumGap(self, nums: list) -> int:
        nums = sorted(nums)
        if len(nums) < 2:
            return 0
        max_gap = 0
        for i in range(len(nums) - 1):
            gap = nums[i+1] - nums[i]
            if gap > max_gap:
                max_gap = gap
        return max_gap
```

### Explanation
1. **Sorting the List**: The input list is sorted to ensure that adjacent elements are in ascending order, which simplifies the calculation of gaps between elements.
2. **Edge Case Handling**: If the list contains fewer than two elements, the function immediately returns 0 because no gaps can exist.
3. **Finding Maximum Gap**: The loop iterates through each adjacent pair in the sorted list, calculates the difference between them, and updates the maximum gap found. The final result is the largest difference encountered during the iteration.

This approach efficiently computes the maximum gap by leveraging sorting and a single pass through the list, ensuring optimal performance with a time complexity of O(n log n) due to the sorting step.
