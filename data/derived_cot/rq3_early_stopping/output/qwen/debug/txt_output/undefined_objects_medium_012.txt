=== RAW ANSWER ===

To solve this problem, we need to find the maximum product of the sums of two non-empty subtrees obtained by removing exactly one edge from the given binary tree. The solution involves traversing the tree and calculating the sum of each subtree, then determining the maximum product of the sums of the two resulting subtrees.

### Approach
1. **Tree Sum Calculation**: First, compute the total sum of all nodes in the tree using a helper function `tsum`.
2. **Subtree Sum and Product Calculation**: Traverse the tree recursively, calculating the sum of each subtree. For each node, consider the product of the sum of the left subtree and the sum of the rest of the tree (total sum minus left subtree sum), and similarly for the right subtree.
3. **Tracking Maximum Product**: During the traversal, keep track of the maximum product encountered by comparing the products calculated at each node.

### Solution Code
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right

class Solution:
    def maxProduct(self, root: Optional[TreeNode]) -> int:
        sm = self.tsum(root)
        mx = [0]
        self.fun(root, sm, mx)
        return mx[0] % (10**9 + 7)
    
    def tsum(self, root):
        if root is None:
            return 0
        return root.val + self.tsum(root.left) + self.tsum(root.right)
    
    def fun(self, root, sm, mx):
        if root is None:
            return 0
        a = self.fun(root.left, sm, mx)
        b = self.fun(root.right, sm, mx)
        current_sum = a + b + root.val
        product_left = a * (sm - a)
        product_right = b * (sm - b)
        if product_left > mx[0]:
            mx[0] = product_left
        if product_right > mx[0]:
            mx[0] = product_right
        return current_sum
```

### Explanation
1. **Tree Sum Calculation**: The `tsum` function recursively calculates the total sum of all nodes in the tree. This sum is crucial for determining the product of the sums of the two subtrees formed by removing an edge.
2. **Subtree Sum and Product Calculation**: The `fun` function recursively traverses the tree, calculating the sum of each subtree. For each node, it computes the product of the sum of the left subtree and the sum of the remaining tree (total sum minus left subtree sum), and similarly for the right subtree.
3. **Tracking Maximum Product**: During the traversal, the maximum product encountered is stored in the list `mx`. The modulus operation at the end ensures the result is within the required constraints.

This approach efficiently explores all possible splits by leveraging recursive traversal and dynamic sum calculations, ensuring the maximum product is found.
