=== RAW ANSWER ===

To solve this problem, we need to determine the number of rods that have all three colors (Red, Green, Blue) attached to them. The input string alternates between color characters and rod identifiers. For example, the string "RGB0GRB1" indicates that rod '0' has colors R, G, and B, while rod '1' has colors G and B.

### Approach
1. **Problem Analysis**: The problem requires counting rods that have all three colors (R, G, B) associated with them. The input string is structured such that each color is followed by a rod identifier. For instance, "R0" means color R is attached to rod '0'.
2. **Intuition**: We need to group colors by their respective rods and check if any rod has all three distinct colors.
3. **Algorithm Selection**: We use a dictionary to map each rod to a set of colors. By iterating through the string in steps of two characters, we can extract each color and its corresponding rod, then add the color to the rod's set in the dictionary.
4. **Complexity Analysis**: The algorithm processes each character in the string exactly once, resulting in a time complexity of O(n), where n is the length of the string. The space complexity is O(k), where k is the number of distinct rods, as each rod's colors are stored in a set within the dictionary.

### Solution Code
```python
class Solution:
    def countPoints(self, rings: str) -> int:
        rod_colors = {}
        i = 0
        n = len(rings)
        while i < n:
            if i + 1 < n:
                color = rings[i]
                rod = rings[i + 1]
                if rod not in rod_colors:
                    rod_colors[rod] = set()
                rod_colors[rod].add(color)
            i += 2
        count = 0
        for colors in rod_colors.values():
            if len(colors) == 3:
                count += 1
        return count
```

### Explanation
1. **Initialization**: We start by initializing an empty dictionary `rod_colors` to store each rod's associated colors.
2. **String Processing**: We iterate through the string in steps of two characters. For each iteration, we check if the next character exists (to avoid index errors). The current character is a color, and the next character is a rod identifier.
3. **Dictionary Update**: For each rod identified, we ensure its entry exists in the dictionary (initializing an empty set if it doesn't). We then add the color to the rod's set.
4. **Counting Valid Rods**: After processing all characters, we count how many rods have exactly three distinct colors in their sets. This count is returned as the result.

This approach efficiently groups colors by rods and checks for completeness, providing an optimal solution to the problem.
