To solve this problem, we need to generate all numbers between a given range [low, high] that consist of sequential digits. Sequential digits numbers are those where each digit is exactly one more than the previous digit, such as 123 or 12345.

### Approach
The approach involves using a breadth-first search (BFS) approach to generate all possible sequential digits numbers. We start with digits from 1 to 9 and iteratively build longer numbers by appending the next digit in sequence. For each number generated, we check if it falls within the specified range [low, high]. If it does, we add it to our result list. The process continues until all possible sequential digits numbers are generated.

### Solution Code
```python
from collections import deque

class Solution:
    def sequentialDigits(self, low, high):
        out = []
        queue = deque(range(1, 10))
        while queue:
            elem = queue.popleft()
            if low <= elem <= high:
                out.append(elem)
            last_digit = elem % 10
            if last_digit < 9:
                next_num = elem * 10 + (last_digit + 1)
                queue.append(next_num)
        return out
```

### Explanation
1. **Initialization**: We start by initializing a queue with digits from 1 to 9. These are the base numbers from which we will build longer sequential digits numbers.
2. **BFS Processing**: Using a queue, we process each number by removing it from the front of the queue. For each number, we check if it lies within the range [low, high]. If it does, we add it to our result list.
3. **Building Sequential Numbers**: For each number, we determine the last digit. If this digit is less than 9, we can extend the number by appending the next digit (last digit + 1). This new number is then added to the queue for further processing.
4. **Termination**: The process continues until the queue is empty, ensuring all possible sequential digits numbers within the range are generated.

This approach efficiently generates all valid sequential digits numbers by leveraging BFS, ensuring that each number is processed exactly once and all possible extensions are considered. The use of a queue allows us to systematically explore all possible sequences of digits.
