To solve this problem, we need to remove duplicates from a sorted array such that each element appears at most twice. The solution involves traversing the array and keeping track of the next available position to write the elements, ensuring that no element appears more than twice.

### Approach
1. **Problem Analysis**: The problem requires modifying the array in-place to remove duplicates while allowing up to two occurrences of each element. The array is given in non-decreasing order, which allows us to compare elements efficiently.
2. **Intuition**: By iterating through the array starting from the third element (index 2), we can compare each element with the element two positions before the current write position (`ans-2`). If they are different, the current element is copied to the next available position (`ans`), and `ans` is incremented.
3. **Algorithm Selection**: The algorithm uses a single pass through the array, maintaining a pointer (`ans`) that tracks the next position to write. This approach ensures that each element is processed in constant time, leading to an efficient solution.
4. **Complexity Analysis**: The algorithm runs in O(n) time, where n is the length of the array, as each element is processed exactly once. The space complexity is O(1) since no additional data structures are used.

### Solution Code
```python
class Solution:
    def removeDuplicates(self, nums: List[int]) -> int:
        n = len(nums)
        if n <= 2:
            return n
        ans = 2
        for i in range(2, n):
            if nums[i] != nums[ans-2]:
                nums[ans] = nums[i]
                ans += 1
        return ans
```

### Explanation
1. **Initial Check**: The code first checks if the array length is 2 or less. If so, it returns the length immediately since no duplicates need to be removed.
2. **Initialization**: The variable `ans` is initialized to 2, indicating that the first two elements are already considered part of the result.
3. **Iteration**: Starting from the third element (index 2), the code checks if the current element is different from the element two positions before the current `ans`. If they differ, the current element is placed at the `ans` position, and `ans` is incremented.
4. **Result**: The value of `ans` after processing all elements gives the length of the modified array where each element appears at most twice.

This approach efficiently processes the array in linear time with constant space, ensuring optimal performance and correctness.
