To solve this problem, we need to determine if two given lists of strings, when concatenated, form identical strings. The provided code attempts to solve this by joining the strings from both lists and comparing them, but it contains a syntax error in the comparison operator.

### Approach
The approach involves:
1. **Joining Strings**: Concatenate all strings in each list into a single string.
2. **Comparison**: Check if the concatenated strings from both lists are identical.

The primary issue in the provided code is the use of the assignment operator `=` instead of the comparison operator `==` during the string comparison. This mistake causes a syntax error because `=` is not allowed in the condition of an `if` statement. The fix involves replacing `=` with `==` to correctly compare the strings.

### Solution Code
```python
class Solution:
    def arrayStringsAreEqual(self, word1: List[str], word2: List[str]) -> bool:
        word3 = "".join(word1)
        word4 = "".join(word2)
        return word3 == word4
```

### Explanation
1. **Joining Strings**: The `"".join(word1)` and `"".join(word2)` methods concatenate all strings in `word1` and `word2` into single strings `word3` and `word4`, respectively.
2. **Comparison**: The `if` condition checks if `word3` is equal to `word4` using the correct `==` operator. If they are equal, the function returns `True`; otherwise, it returns `False`.

This approach efficiently checks for equality by leveraging Python's string joining and comparison capabilities, ensuring correctness and clarity.
