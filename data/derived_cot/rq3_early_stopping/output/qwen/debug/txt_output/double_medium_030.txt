To solve this problem, we need to sort an array of 0s, 1s, and 2s in-place using the Dutch National Flag algorithm. The algorithm efficiently partitions the array into three sections: all 0s, all 1s, and all 2s, using three pointers to track the positions where each number should be placed.

### Approach
1. **Initialization**: We initialize three pointers:
   - `low` (index 0): Points to the next position where a 0 should be placed.
   - `mid` (index 0): Current element being processed.
   - `high` (index len(nums)-1): Points to the next position where a 2 should be placed.
2. **Processing Loop**: While `mid` is less than or equal to `high`:
   - If the current element (`nums[mid]`) is 0, swap it with the element at `low` and increment both `low` and `mid`.
   - If the current element is 1, simply move `mid` forward.
   - If the current element is 2, swap it with the element at `high` and decrement `high`.
3. **Termination**: The loop terminates when `mid` exceeds `high`, ensuring all elements are correctly placed.

This approach ensures that the array is sorted in a single pass with O(n) time complexity and O(1) space complexity, as it operates in-place.

### Solution Code
```python
class Solution:
    def sortColors(self, nums: List[int]) -> None:
        low = mid = 0
        high = len(nums) - 1
        while mid <= high:
            if nums[mid] == 0:
                nums[low], nums[mid] = nums[mid], nums[low]
                low += 1
                mid += 1
            elif nums[mid] == 1:
                mid += 1
            else:
                nums[mid], nums[high] = nums[high], nums[mid]
                mid += 1
                high -= 1
```

### Explanation
- **Initialization**: The pointers `low`, `mid`, and `high` are initialized to 0, 0, and the last index of the array, respectively.
- **Loop Execution**: The loop runs while `mid` is within the bounds of the array (`mid <= high`).
  - **Handling 0s**: When a 0 is encountered at `mid`, it is swapped with the element at `low`, and both `low` and `mid` are incremented. This places the 0 in the correct section.
  - **Handling 1s**: If the element at `mid` is 1, only `mid` is incremented, as 1s are already in their correct position.
  - **Handling 2s**: When a 2 is found at `mid`, it is swapped with the element at `high`, and both `mid` and `high` are adjusted. This places the 2 in the correct section.
- **Termination**: The loop exits once all elements are processed, resulting in a sorted array with all 0s followed by 1s and then 2s.

This method efficiently sorts the array in linear time with constant space, making it optimal for the given problem constraints.
