=== RAW ANSWER ===

To solve this problem, we need to construct a quad-tree from a given grid. The quad-tree is a hierarchical data structure where each node can have up to four children, representing the four quadrants of the grid. The goal is to build this tree recursively by dividing the grid into smaller quadrants until each leaf node represents a single cell.

### Approach
1. **Base Case Handling**: If the current grid is a single cell (i.e., level 1), return a leaf node with the value of that cell.
2. **Recursive Division**: For grids larger than 1x1, divide the grid into four quadrants (top-left, top-right, bottom-left, bottom-right) and recursively build the quad-tree for each quadrant.
3. **Leaf Node Check**: After processing all four quadrants, check if all four child nodes are leaves and if they have the same value. If so, the current node can be a leaf node with that value. Otherwise, the current node is an internal node with the four child nodes.

### Solution Code
```python
class Solution:
    def construct(self, grid: List[List[int]]) -> 'Node':
        level = len(grid)
        root = None
        if level >= 1:
            root = self.buildTree(grid, 0, level, 0, level, level)
        return root
        
    def buildTree(self, grid, rs, re, cs, ce, level):
        if level == 1:
            return Node(grid[rs][cs], True, None, None, None, None)
        
        next_level = level // 2
        tl = self.buildTree(grid, rs, rs + next_level, cs, cs + next_level, next_level)
        tr = self.buildTree(grid, rs, rs + next_level, cs + next_level, cs + next_level * 2, next_level)
        bl = self.buildTree(grid, rs + next_level, rs + next_level * 2, cs, cs + next_level, next_level)
        br = self.buildTree(grid, rs + next_level, rs + next_level * 2, cs + next_level, cs + next_level * 2, next_level)
        
        if tl.isLeaf and tr.isLeaf and bl.isLeaf and br.isLeaf:

            else:
                new_node = Node(tl.val, False, tl, tr, bl, br)
        else:
            new_node = Node(tl.val, False, tl, tr, bl, br)
```
