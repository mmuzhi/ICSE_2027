### Approach

2. **Satisfies Trip Check**: For each midpoint time during the binary search, calculate the total number of trips all buses can complete in that time. This is done by summing `mid // t` for each bus time `t` in the array.
3. **Binary Search Adjustment**: If the total trips meet or exceed `totalTrips`, adjust the search to find a potentially smaller time by setting `right` to `mid`. Otherwise, search for a larger time by setting `left` to `mid + 1`.
4. **Result**: The binary search concludes when `left` equals `right`, returning this value as the minimum time required.

### Solution Code
```python
class Solution:
    def satisfiesTrip(self, mid: int, time: List[int], totalTrips: int) -> bool:
        trip = 0
        for t in time:
            trip += mid // t
        return trip >= totalTrips

    def minimumTime(self, time: List[int], totalTrips: int) -> int:
        time.sort(reverse=True)
        minimum = min(time)
        left = minimum
        right = minimum * totalTrips
        
        while left < right:
            mid = (left + right) // 2
            if self.satisfiesTrip(mid, time, totalTrips):
                right = mid
            else:
                left = mid + 1
        return left
```

- **Binary Search**: The binary search efficiently narrows down the possible time values from the smallest possible time (`min(time)`) to the largest (`min(time) * totalTrips`).

- **Adjusting Search Bounds**: Depending on whether the current midpoint time satisfies the condition, the search bounds are adjusted to find the minimum time. If the condition is met, the upper bound is reduced; otherwise, the lower bound is increased.
- **Result**: The loop exits when `left` equals `right`, which is the minimum time required for the buses to complete at least `totalTrips` trips.

This approach ensures that we efficiently find the solution using binary search, leveraging the fact that the problem is monotonic—either a time is sufficient or it is not, allowing us to use binary search for optimal performance.
