To solve this problem, we need to compute the sum of the minimum values of all possible contiguous subarrays in a given array. The challenge is to do this efficiently without resorting to a brute-force approach, which would be too slow for large arrays.

### Approach
The efficient approach involves using a monotonic stack to keep track of elements and their contributions as minimums in subarrays. The key idea is to determine for each element in the array, how many subarrays it is the minimum of. This is done by finding the nearest smaller elements to the left and right of each element. 

1. **Left Boundaries**: For each element, find the first element to the left that is smaller than it. The number of subarrays where this element is the minimum is determined by its position relative to this left boundary.
2. **Right Boundaries**: Similarly, find the first element to the right that is smaller than or equal to it. The right boundary helps in determining the end of subarrays where the current element is the minimum.
3. **Contribution Calculation**: For each element, multiply its value by the number of subarrays where it is the minimum (calculated using left and right boundaries) and accumulate the result.

This approach ensures that we efficiently compute the sum in linear time using a stack, making it suitable for large input sizes.

### Solution Code
```python
class Solution:
    def sumSubarrayMins(self, arr: List[int]) -> int:
        n = len(arr)
        left_bound = [-1] * n
        stack = []
        for i in range(n):
            while stack and arr[stack[-1]] > arr[i]:
                stack.pop()
            left_bound[i] = stack[-1] if stack else -1
            stack.append(i)
        
        right_bound = [n] * n
        stack = []
        for i in range(n-1, -1, -1):
            while stack and arr[stack[-1]] >= arr[i]:
                stack.pop()
            right_bound[i] = stack[-1] if stack else n
            stack.append(i)
        
        res = 0
        mod = 10**9 + 7
        for i in range(n):
            res = (res + arr[i] * (i - left_bound[i]) * (right_bound[i] - i)) % mod
        
        return res
```

### Explanation
1. **Left Boundaries Calculation**: We iterate through the array from left to right, using a stack to keep track of indices. For each element, we pop elements from the stack that are greater than the current element, as they can no longer be the minimum in future subarrays. The remaining top of the stack gives the left boundary for the current element.
2. **Right Boundaries Calculation**: We iterate through the array from right to left, again using a stack. For each element, we pop elements that are greater than or equal to the current element to determine the right boundary.
3. **Sum Calculation**: For each element, the number of subarrays where it is the minimum is the product of the distances to its left and right boundaries. The element's value multiplied by this product gives its total contribution to the sum. The result is accumulated with modulo \(10^9 + 7\) to handle large numbers.

This approach efficiently computes the desired sum in \(O(n)\) time with \(O(n)\) space complexity, making it optimal for the problem constraints.
