To solve this problem, we need to find the longest cycle in a directed graph where each node has at most one outgoing edge. The solution involves traversing the graph using Depth-First Search (DFS) while keeping track of visited nodes and the depth at which each node was first encountered in the current traversal path. If a node is encountered again in the same path, a cycle is detected, and its length is calculated based on the difference in depths.

### Approach
1. **Graph Traversal**: For each unvisited node, initiate a DFS traversal. During traversal, mark nodes as visited to avoid reprocessing.
2. **Cycle Detection**: While traversing, maintain a set of nodes in the current path. If a node is encountered that is already in the current path, a cycle is detected.
3. **Cycle Length Calculation**: The length of the cycle is determined by the difference between the current depth and the depth at which the node was first encountered in the current path.
4. **Tracking Maximum Cycle**: Keep track of the maximum cycle length encountered during the traversal.

### Solution Code
```python
class Solution:
    def longestCycle(self, edges: List[int]) -> int:
        n = len(edges)
        bl = [False] * n
        mx = -1
        
        for i in range(n):
            if not bl[i]:
                x = i
                l = 0
                st = set()
                while x != -1 and not bl[x]:
                    bl[x] = True
                    st.add(x)
                    l += 1
                    x = edges[x]
                if x != -1 and x in st:
                    cycle_length = l - st.index(x) - 1
                    if cycle_length > mx:
                        mx = cycle_length
        return mx
```
