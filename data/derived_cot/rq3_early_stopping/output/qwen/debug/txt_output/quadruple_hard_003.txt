To solve this problem, we need to find the longest substring within a given word that does not contain any forbidden substrings. The solution involves using a trie data structure to efficiently check for forbidden substrings and a sliding window approach to determine the longest valid substring.

### Approach
1. **Trie Construction**: Build a trie from all forbidden substrings. Each node in the trie represents a character, and an "end" marker indicates the completion of a forbidden substring.
2. **Forbidden Substring Check**: For any substring, traverse the trie to check if it contains any forbidden substring. If a forbidden substring is found, return its length; otherwise, return 0.
3. **Sliding Window**: Use a sliding window approach to explore all possible substrings from the end of the word to the beginning. For each starting index, expand the window to the right as much as possible without including any forbidden substring. The length of the valid substring is then recorded, and the maximum length is tracked.

### Solution Code
```python
class Solution:
    def longestValidSubstring(self, word: str, forbidden: List[str]) -> int:
        trie = {}
        for f in forbidden:
            t = trie
            for c in f:
                if c not in t:
                    t[c] = {}
                t = t[c]
            t["end"] = True
        
        def isForbidden(s):
            t = trie
            for idx, c in enumerate(s):
                if c not in t:
                    return 0
                t = t[c]
                if "end" in t:
                    return idx + 1
            return 0
        
        n = len(word)
        res = 0
        j = n + 1
        for i in range(n-1, -1, -1):
            if j > i:
                truc = isForbidden(word[i:j])
                if truc:
                    j = i + truc - 1
            res = max(res, j - i)
        return res
```

1. **Trie Construction**: The trie is built by iterating over each forbidden substring. For each character in a forbidden substring, a new node is created if it doesn't exist. An "end" marker is added at the last character to denote the completion of a forbidden substring.
2. **Forbidden Substring Check**: The `isForbidden` function checks if a given substring contains any forbidden substring by traversing the trie. If a forbidden substring is found, its length is returned; otherwise, 0 is returned.
3. **Sliding Window**: The main loop starts from the end of the word and moves backward. For each starting index `i`, the window extends to `j` (initially set to the end of the word). The `isForbidden` function checks if the substring from `i` to `j` contains any forbidden substring. If it does, `j` is adjusted to exclude the forbidden substring.
