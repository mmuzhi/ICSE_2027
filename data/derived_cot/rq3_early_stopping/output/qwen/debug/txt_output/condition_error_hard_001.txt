To solve this problem, we need to find all pairs of indices (i, j) such that the concatenation of the words at these indices forms a palindrome. The solution involves efficiently checking for these pairs by leveraging a hash map to store reversed words and their indices.

### Approach
1. **Problem Analysis**: The task is to find pairs of words from the given list that, when concatenated, form a palindrome. A palindrome reads the same forwards and backwards. The challenge is to do this efficiently without resorting to a brute-force O(n^2) approach.

2. **Intuition**: For any two words `word1` and `word2`, their concatenation `word1 + word2` will be a palindrome if `word2` is the reverse of a suffix of `word1` and the prefix of `word1` (before the split) is a palindrome. Alternatively, `word1` could be the reverse of a suffix of `word2` and the prefix of `word2` is a palindrome.

3. **Algorithm Selection**: 
   - **Hash Map for Reversed Words**: Create a dictionary where each key is the reversed version of a word, and the value is the list of indices where that reversed word appears. This allows quick lookup of words that could form a palindrome when concatenated with the current word.
   - **Palindrome Check**: For each word, split it into two parts at every possible position. Check if the left part is a palindrome. If it is, then the right part should be the reverse of some word in the hash map. Similarly, check if the right part is a palindrome and the left part is the reverse of some word in the hash map.

4. **Complexity Analysis**: 
   - **Time Complexity**: O(n * m), where n is the number of words and m is the average length of the words. For each word, we check all possible splits and perform palindrome checks, which is O(m) for each split.
   - **Space Complexity**: O(n * m), due to the storage of reversed words and their indices in the hash map.

class Solution:
    def palindromePairs(self, words: List[str]) -> List[List[int]]:
        backward = {}
        for i, word in enumerate(words):
            reversed_word = word[::-1]
            if reversed_word not in backward:
                backward[reversed_word] = []
            backward[reversed_word].append(i)
        
        res = []
        n = len(words)
        
        for i, word in enumerate(words):
            m = len(word)
            # Case 1: The current word is a palindrome. Then we can pair it with the empty string if it exists.
            if word == "":
                continue
            
            # Case 2: The current word is the first part. We split the word into two parts: prefix and suffix.
            # For each split, if the prefix is a palindrome, then the suffix should be the reverse of some word in the backward map.
            for j in range(m + 1):
                prefix = word[:j]
                suffix = word[j:]
                
                # If the prefix is a palindrome, then we can pair the current word with the reversed of the suffix.
                if prefix == prefix[::-1]:
                    # If the suffix is not empty, we need to find the reversed of the suffix in the backward map.
                    if suffix in backward:
                        for idx in backward[suffix]:
                            if idx != i:
                                res.append([i, idx])
                
                # If the suffix is a palindrome, then we can pair the current word with the reversed of the prefix.
                if suffix == suffix[::-1]:
                    # If the prefix is not empty, we need to find the reversed of the prefix in the backward map.
                    if prefix and prefix in backward:
                        for idx in backward[prefix]:
                            if idx != i:
                                res.append([idx, i])
        
        return res
