=== RAW ANSWER ===

To solve this problem, we need to perform a level-order traversal of a binary tree from bottom to top. The standard level-order traversal (BFS) processes nodes level by level from top to bottom. To achieve the bottom-up order, we reverse the list of levels obtained from the BFS.

### Approach
1. **BFS Traversal**: We use a queue to traverse the tree level by level. Starting from the root node, we process each node, enqueue its children (left first, then right), and record the values of nodes at each level.
2. **Reversing Levels**: After collecting all levels in the order they are processed (top to bottom), we reverse the list of levels to present them from bottom to top.

### Solution Code
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def levelOrderBottom(self, root: Optional[TreeNode]) -> List[List[int]]:
        def checkReverse(ans):
            l, r = 0, len(ans)-1
            while l <= r:
                ans[l], ans[r] = ans[r], ans[l]
                l += 1
                r -= 1
            return ans

        if not root:
            return []
        q, ans = [root], []
        while q:
            n = len(q)
            level = []
            for i in range(n):
                curr = q.pop(0)
                level.append(curr.val)
                if curr.left:
                    q.append(curr.left)
                if curr.right:
                    q.append(curr.right)
            ans.append(level)
        return checkReverse(ans)
```

### Explanation
1. **BFS Traversal**: The code initializes a queue with the root node. It processes each node in the queue, appends their values to the current level, and enqueues their children. This ensures that nodes are processed level by level from top to bottom.
2. **Reversing Levels**: The `checkReverse` function reverses the list of levels in-place using two pointers (left and right) to swap elements until they meet in the middle. This effectively reverses the order of the levels, providing the bottom-up traversal as required.

The solution efficiently handles edge cases, such as an empty tree, by returning an empty list. The BFS ensures all nodes are visited, and the reversal function correctly orders the levels from bottom to top.
