To solve this problem, we need to determine the number of plates between candles in substrings of a given string for multiple queries. The string consists of characters '|' (candles) and '.' (plates). For each query, we need to find the substring specified by the left and right indices and count the number of plates between the first and last candles in that substring. If there are no candles or only one candle in the substring, the result should be 0.

### Approach
1. **Identify Candle Positions**: First, we collect all the indices where candles ('|') are located in the string. This helps in quickly locating the nearest candles for any given substring.
2. **Binary Search for Candle Indices**: For each query, we use binary search to find the first candle index greater than or equal to the left index of the query and the last candle index less than or equal to the right index of the query. This is done efficiently using binary search since the list of candle positions is sorted.
3. **Calculate Plates Between Candles**: Once we have the first and last candle positions in the substring, we calculate the number of plates between them by subtracting the number of candles (including both endpoints) from the total length of the substring between these two candles.

### Solution Code
```python
class Solution:
    def platesBetweenCandles(self, s: str, queries: List[List[int]]) -> List[int]:
        candies = [i for i, c in enumerate(s) if c == '|']
        
        def bns(x: int) -> int:
            l, r = 0, len(candies) - 1
            while l <= r:
                m = (l + r) // 2
                if candies[m] < x:
                    l = m + 1
                else:
                    r = m - 1
            return l
        
        ans = []
        for a, b in queries:
            l_idx = bns(a)
            r_idx = bns(b + 1) - 1
            if l_idx < r_idx:
                plates = candies[r_idx] - candies[l_idx] - (r_idx - l_idx)
                ans.append(plates)
            else:
                ans.append(0)
        return ans
```

1. **Candle Position Identification**: The code first creates a list `candies` that stores the indices of all candles in the string. This list is sorted by nature since it is built by iterating through the string in order.
2. **Binary Search Function (`bns`)**: This helper function performs binary search on the `candies` list to find the first candle index that is greater than or equal to a given value `x`. The function efficiently narrows down the search space by adjusting the left and right pointers based on the comparison between the middle element and `x`.
3. **Query Processing**: For each query, the code determines the leftmost and rightmost candles within the specified substring using the binary search function. If such candles exist (i.e., there are at least two candles in the substring), it calculates the number of plates between them by subtracting the number of candles from the total length of the substring between these candles.
