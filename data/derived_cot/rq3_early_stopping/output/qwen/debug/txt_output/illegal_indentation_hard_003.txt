To solve this problem, we need to find the maximum nice divisor of a given integer `primeFactors`. A nice divisor is defined as a divisor that is not divisible by any prime other than 2 and 3. The solution involves factorizing the given number into its prime factors of 2 and 3, then computing the product of these factors to determine the maximum nice divisor.

### Approach
1. **Initial Check**: If the input `primeFactors` is 1, return -1 since there are no divisors greater than 1. For values 2 or 3, return the number itself as they are trivial nice divisors.
2. **Factorization**: For numbers greater than 3, factorize the input into its exponents of 2 and 3. This involves repeatedly dividing the number by 2 and 3 until it is no longer divisible by these primes.
3. **Compute Result**: The maximum nice divisor is the product of \(2^a \times 3^b\), where `a` and `b` are the exponents of 2 and 3, respectively. If the product is 1 (indicating no factors of 2 or 3 were found), return -1.

### Solution Code
```python
class Solution:
    def maxNiceDivisors(self, primeFactors: int) -> int:
        if primeFactors == 1:
            return -1
        if primeFactors <= 3:
            return primeFactors
        
        a = 0
        temp = primeFactors
        while temp % 2 == 0:
            a += 1
            temp //= 2
        
        b = 0
        while temp % 3 == 0:
            b += 1
            temp //= 3
        
        d = (2 ** a) * (3 ** b)
        if d == 1:
            return -1
        
        return d
```
