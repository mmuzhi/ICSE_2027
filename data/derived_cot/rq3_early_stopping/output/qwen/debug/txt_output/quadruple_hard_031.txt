To solve this problem, we need to find the longest substring within a given word that does not contain any forbidden substrings. The solution involves using a trie data structure to efficiently check for forbidden substrings and a two-pointer technique to determine the longest valid substring.

### Approach
1. **Trie Construction**: Build a trie from all forbidden substrings. Each node in the trie will store whether it marks the end of a forbidden substring and the length of that substring.
2. **Two-Pointer Technique**: Use two pointers, `i` and `j`, to represent the current substring being checked. The pointer `j` expands the substring by moving forward, while `i` contracts the substring when a forbidden substring is found.
3. **State Tracking**: Maintain a state in the trie that represents the current matching path. As we move `j` forward, we update the state by traversing the trie with the current character. If we reach an "end" node in the trie, it means a forbidden substring has been found, and we need to move `i` forward to exclude this substring.
4. **Length Calculation**: For each valid substring found, update the maximum length accordingly.

### Solution Code
```python
class Solution:
    def longestValidSubstring(self, word: str, forbidden: List[str]) -> int:
        if not word:
            return 0
        
        trie = {}
        for f in forbidden:
            t = trie
            for c in f:
                if c not in t:
                    t[c] = {}
                t = t[c]
            t['end'] = True
            t['length'] = len(f)
        
        state = trie
        j = 0
        res = 0
        
        for i in range(len(word)):
            state = trie
            while j < len(word):
                c = word[j]
                if c in state:
                    state = state[c]
                    if 'end' in state:
                        forbidden_length = state['length']
                        j = j - forbidden_length + 1
                        break
                    if 'length' in state and j - state['length'] + 1 >= i:
                        j = j - state['length'] + 1
                        break
                else:
                    break
                j += 1
            res = max(res, j - i)
        return res
```
