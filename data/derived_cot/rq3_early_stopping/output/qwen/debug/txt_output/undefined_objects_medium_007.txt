To solve this problem, we need to construct a quad-tree from a given grid. The quad-tree is built by recursively dividing the grid into four quadrants until each quadrant is a single cell (leaf node) or until all cells in a quadrant have the same value (leaf node). 

### Approach
1. **Problem Analysis**: The grid is divided into four quadrants. If all four quadrants have the same value, the current node is a leaf node with that value. Otherwise, it is an internal node with four children representing the quadrants.
2. **Intuition**: The solution involves a recursive approach where each node is built by checking the top-left, top-right, bottom-left, and bottom-right quadrants. If all four quadrants are leaves and have the same value, the current node is a leaf; otherwise, it is an internal node with the four quadrants as children.
3. **Algorithm Selection**: The algorithm uses a divide-and-conquer strategy. The grid is split into four equal parts, and each part is processed recursively until the base case (single cell) is reached.
4. **Complexity Analysis**: The time complexity is O(n^2) where n is the grid size, as each cell is processed once. The space complexity is O(log n) due to the recursion depth, which corresponds to the depth of the quad-tree.

class Solution:
