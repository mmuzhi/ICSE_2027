=== RAW ANSWER ===

To solve this problem, we need to find the median of two sorted arrays efficiently. The median is the middle value of a sorted list. If the combined length of the two arrays is even, the median is the average of the two middle values; otherwise, it is the middle value.

### Approach
1. **Merge the Arrays**: Combine the two sorted arrays into a single sorted array. This can be done by using two pointers to traverse both arrays and appending the smaller elements from either array in order.
2. **Calculate the Median**: After merging, determine the median by checking the length of the combined array. If the length is odd, the median is the middle element. If the length is even, the median is the average of the two middle elements.

### Solution Code
```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        new = []
        p1 = 0
        p2 = 0
        
        while p1 < len(nums1) and p2 < len(nums2):
            if nums1[p1] < nums2[p2]:
                new.append(nums1[p1])
                p1 += 1
            else:
                new.append(nums2[p2])
                p2 += 1
                
        while p1 < len(nums1):
            new.append(nums1[p1])
            p1 += 1
            
        while p2 < len(nums2):
            new.append(nums2[p2])
            p2 += 1
            
        n = len(new)
        if n % 2 == 1:
            return float(new[n//2])
        else:
            return (new[n//2 - 1] + new[n//2]) / 2.0
```

### Explanation
1. **Merging Arrays**: The code initializes an empty list `new` and two pointers `p1` and `p2` to traverse `nums1` and `nums2` respectively. The while loop continues as long as both pointers are within their respective array bounds. In each iteration, the smaller element between `nums1[p1]` and `nums2[p2]` is appended to `new`, and the corresponding pointer is incremented.
2. **Appending Remaining Elements**: After one of the arrays is exhausted, the remaining elements of the other array are appended to `new` in their original order.
3. **Finding the Median**: The median is determined based on the length of `new`. If the length is odd, the median is the middle element. If even, it is the average of the two middle elements. The result is returned as a float to handle both cases appropriately.

This approach efficiently merges the two sorted arrays and calculates the median in linear time, O(n + m), where n and m are the lengths of the two arrays, making it optimal for large input sizes.
