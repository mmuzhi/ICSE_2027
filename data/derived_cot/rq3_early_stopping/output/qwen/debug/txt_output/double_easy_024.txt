The solution involves using a frequency dictionary to keep track of the occurrences of each character and then processing the characters in two phases: increasing order and decreasing order.

### Approach
1. **Frequency Dictionary**: We first build a frequency dictionary to count the occurrences of each character in the input string.
2. **Sorted Unique Characters**: We then extract the unique characters from the frequency dictionary and sort them in increasing order.
3. **Two-Phase Processing**: 
   - **Increasing Phase**: Iterate through the sorted unique characters and append each character to the result string while decrementing its frequency. If the frequency drops to zero, remove the character from the dictionary.
   - **Decreasing Phase**: Iterate through the sorted unique characters in reverse order (decreasing order) and append each character to the result string while decrementing its frequency. Again, remove the character if its frequency becomes zero.
4. **Repeat Until Completion**: Continue the two-phase processing until the frequency dictionary is empty, ensuring all characters are used in the result string.

class Solution:

        freq = {}
        for char in s:
            freq[char] = freq.get(char, 0) + 1
        
        letters = sorted(freq.keys())
        left = 0
        right = len(letters) - 1
        res = []
        
        while left <= right:
            # Move left pointer until a letter with positive frequency is found
            while left <= right and freq[letters[left]] == 0:

            if freq[letters[left]] == 0:

                right -= 1
            if left > right:
