To solve this problem, we need to find the second greater element for each element in the given array. The second greater element for an element at index `i` is the smallest element to the right of `i` that is greater than the first greater element of `i`. If no such element exists, we return -1.

### Approach
1. **First Greater Element Identification**: We first identify the first greater element for each element in the array. This is done using a stack-based approach where we traverse the array from left to right. For each element, we pop elements from the stack that are smaller than the current element and record the current element as the first greater element for the popped indices.
2. **Second Greater Element Identification**: Using the indices obtained from the first pass, we then identify the second greater element. We traverse the array again from left to right, using another stack to record the next greater element for each index. The second greater element for an index `i` is the next greater element of the index where the first greater element of `i` is located.
3. **Result Construction**: For each element, if a first greater element exists, we use the result from the second pass to determine the second greater element. If no second greater element exists, we return -1.

### Solution Code
```python
class Solution:
    def secondGreaterElement(self, nums: List[int]) -> List[int]:
        n = len(nums)
        first_greater = [-1] * n
        first_greater_index = [-1] * n
        stack = []
        for i in range(n):
            while stack and nums[stack[-1]] < nums[i]:
                idx = stack.pop()
                first_greater[idx] = nums[i]
                first_greater_index[idx] = i
            stack.append(i)
        
        next_greater2 = [-1] * n
        stack = []
        for i in range(n):
            while stack and nums[stack[-1]] < nums[i]:
                idx = stack.pop()
                next_greater2[idx] = nums[i]
            stack.append(i)
        
        ans = [-1] * n
        for i in range(n):
            if first_greater[i] != -1:
                ans[i] = next_greater2[first_greater_index[i]]
        return ans
```
