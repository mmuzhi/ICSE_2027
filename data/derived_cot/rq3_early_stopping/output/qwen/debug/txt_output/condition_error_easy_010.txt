To solve this problem, we need to convert a given integer into its corresponding Excel column title. Excel column titles follow a base-26 system where each digit is represented by a letter from 'A' to 'Z'. The key insight is that this system does not have a zero, so we need to adjust our approach to handle the conversion correctly.

### Approach
1. **Problem Analysis**: The problem requires converting a number to an Excel column title where each digit in the title corresponds to a letter in the alphabet. For example, 1 becomes "A", 26 becomes "Z", 27 becomes "AA", and so on.
2. **Insight**: The conversion can be approached by repeatedly dividing the number by 26, adjusting for the fact that there is no zero in the system. Each division step gives a remainder that corresponds to a letter in the alphabet. If the remainder is zero, it means the current digit should be 'Z', and we need to adjust the quotient by subtracting 1 to account for the absence of zero.
3. **Algorithm Selection**: The algorithm involves a loop where we repeatedly divide the number by 26, adjusting the remainder and quotient appropriately. The letters are built from the least significant digit to the most significant digit, and then the result is reversed to get the correct order.
4. **Complexity Analysis**: The time complexity is O(log n) because each iteration reduces the number by a factor of 26. The space complexity is O(log n) to store the result string.

### Solution Code
```python
class Solution:
    def convertToTitle(self, n: int) -> str:
        s = ""
        while n > 0:
            n, r = divmod(n-1, 26)
            s = chr(65 + r) + s
        return s
```

### Explanation
1. **Initialization**: Start with an empty string `s` to build the result.
2. **Loop**: While the number `n` is greater than zero:
   - **Adjustment**: Subtract 1 from `n` to convert it to a zero-based index.
   - **Division**: Use `divmod` to get the quotient and remainder when dividing by 26.
   - **Character Mapping**: Map the remainder to a corresponding letter (0 to 'A', 1 to 'B', ..., 25 to 'Z').
   - **String Construction**: Prepend the mapped character to the result string `s`.
3. **Result**: After processing all digits, the string `s` contains the Excel column title in the correct order.

This approach efficiently handles the conversion by leveraging base-26 arithmetic with adjustments for the absence of a zero digit, ensuring accurate and optimal results.
