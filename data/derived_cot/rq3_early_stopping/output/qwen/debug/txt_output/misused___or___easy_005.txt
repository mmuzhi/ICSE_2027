### Solution Code
```python
class Solution:
    def countGoodSubstrings(self, s: str) -> int:
        count = 0
        for i in range(len(s) - 2):
            x = s[i:i+3]
            if x.count(x[0]) == 1 and x.count(x[1]) == 1 and x.count(x[2]) == 1:
                count += 1
        return count
```

### Explanation
1. **Initialization**: We initialize a counter `count` to zero to keep track of the number of valid substrings.
2. **Iteration**: We loop through each possible starting index `i` of a substring of length 3. The loop runs from `0` to `len(s) - 3` to ensure we do not go out of bounds.
3. **Substring Check**: For each starting index `i`, we extract the substring `s[i:i+3]`. We then check if each character in this substring appears exactly once using the `count` method.
4. **Count Update**: If all three characters in the substring are distinct (each appears exactly once), we increment the counter `count`.
5. **Result**: After processing all possible substrings, the function returns the total count of valid substrings.

This approach efficiently checks each substring of length 3 in linear time, ensuring optimal performance for the given problem.
