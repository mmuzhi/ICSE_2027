=== RAW ANSWER ===

To solve this problem, we need to find the median of two sorted arrays. The median is the middle value in an ordered list of numbers. If the combined length of the two arrays is even, the median is the average of the two middle numbers; if it's odd, the median is the middle number.

### Approach
1. **Merge the Arrays**: Since both arrays are already sorted, we can merge them into a single sorted array by comparing elements from both arrays and appending the smaller element to the result array. This ensures the merged array remains sorted.
2. **Calculate the Median**: After merging, the median can be found by checking the length of the merged array. If the length is even, the median is the average of the two middle elements; if odd, it's the middle element.

### Solution Code
```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        p1 = 0
        p2 = 0
        new = []
        
        while p1 < len(nums1) and p2 < len(nums2):
            if nums1[p1] < nums2[p2]:
                new.append(nums1[p1])
                p1 += 1
            else:
                new.append(nums2[p2])
                p2 += 1
                
        while p1 < len(nums1):
            new.append(nums1[p1])
            p1 += 1
            
        while p2 < len(nums2):
            new.append(nums2[p2])
            p2 += 1
            
        n = len(new)
        if n % 2 == 0:
            index = n // 2
            median = (new[index] + new[index-1]) / 2
        else:
            median = float(new[n//2])
        return median
```

### Explanation
1. **Initialization**: We start by initializing two pointers, `p1` and `p2`, to 0. These pointers will traverse the two input arrays. We also initialize an empty list `new` to store the merged array.
2. **Merging Arrays**: The first loop runs as long as both pointers are within their respective array bounds. In each iteration, the smaller of the current elements from `nums1` and `nums2` is appended to `new`, and the corresponding pointer is incremented.
3. **Remaining Elements**: After one of the arrays is exhausted, the remaining elements from the other array are appended to `new`.
4. **Finding the Median**: The length of the merged array `new` determines the median calculation. If the length is even, the median is the average of the two middle elements. If the length is odd, the median is the middle element. The result is returned as a float to handle both cases appropriately.
