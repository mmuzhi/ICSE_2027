To solve this problem, we need to construct a quad-tree from a given grid of size n x n, where n is a power of 2. The quad-tree is built by recursively dividing the grid into four quadrants until each quadrant is a single cell (leaf node). If all four quadrants of a node have the same value, the node is marked as a leaf; otherwise, it is an internal node with four children.

### Approach
1. **Base Case Handling**: If the current grid is a single cell (level 1), return a leaf node with the cell's value.
2. **Recursive Division**: For grids larger than 1x1, divide the grid into four quadrants of equal size (level // 2). Recursively build the quad-tree for each quadrant.
3. **Leaf Node Check**: After building the children nodes, check if all four children are leaves and have the same value. If so, the current node is a leaf with that value. Otherwise, it is an internal node with the four children.
