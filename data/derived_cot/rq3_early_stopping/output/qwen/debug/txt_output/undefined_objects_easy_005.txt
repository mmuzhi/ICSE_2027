To solve this problem, we need to find the mode(s) (most frequent value(s)) in a binary tree. The approach involves traversing the tree to count the frequency of each node's value and then determining which values have the highest frequency.

### Approach
1. **Tree Traversal**: We use Breadth-First Search (BFS) to traverse the tree. BFS is suitable here because it efficiently processes each node level by level, ensuring all nodes are visited.
2. **Frequency Counting**: During traversal, we maintain a dictionary to count occurrences of each node's value.
3. **Finding Modes**: After counting frequencies, we determine the maximum frequency and collect all values that have this frequency.

### Solution Code
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def findMode(self, root: Optional[TreeNode]) -> List[int]:
        from collections import deque
        freq = {}
        if not root:
            return []
        
        queue = deque([root])
        while queue:
            curr = queue.popleft()
            freq[curr.val] = freq.get(curr.val, 0) + 1
            if curr.left:
                queue.append(curr.left)
            if curr.right:
                queue.append(curr.right)
        
        max_cnt = 0
        ans = []
        for val, cnt in freq.items():
            if cnt > max_cnt:
                max_cnt = cnt
                ans = [val]
            elif cnt == max_cnt:
                ans.append(val)
        return ans
```

### Explanation
1. **Tree Traversal**: The BFS starts from the root node. Each node is processed by popping it from the front of the queue, and its children are added to the end of the queue. This ensures that all nodes are visited in level order.
2. **Frequency Counting**: For each node, its value is recorded in a dictionary (`freq`), where the key is the node's value and the value is the count of occurrences. The `get` method is used to efficiently retrieve and increment the count.
3. **Finding Modes**: After traversal, the maximum frequency (`max_cnt`) is determined by iterating through the dictionary. All values with this maximum frequency are collected into the result list (`ans`), which is then returned.

This approach efficiently counts the frequencies and identifies the modes, ensuring optimal performance with a time complexity of O(n), where n is the number of nodes in the tree.
