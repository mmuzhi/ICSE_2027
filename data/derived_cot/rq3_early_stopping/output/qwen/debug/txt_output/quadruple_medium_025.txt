To solve this problem, we need to restore all possible valid IP addresses from a given string. An IP address consists of four parts, each ranging from 0 to 255, and each part must not have leading zeros unless the part is exactly "0". The solution involves generating all possible combinations of four parts that form a valid IP address from the input string.

### Approach
1. **Backtracking**: We use a backtracking approach to explore all possible ways to split the string into four parts. The algorithm starts from the beginning of the string and tries to form each part by considering 1 to 3 digits (since parts cannot exceed 3 digits).
2. **Base Case**: When the entire string is processed and four parts are formed, the current combination is added to the result list.
3. **Pruning**: The algorithm prunes invalid paths early. For instance, if a part starts with '0' and has more than one digit, it is skipped. Also, parts exceeding 255 are not considered.
4. **Recursion**: The function recursively processes the remaining string after each part is formed, ensuring that the parts are valid and the entire string is consumed.

### Solution Code
```python
class Solution:
    def restoreIpAddresses(self, s: str) -> List[str]:
        res = []
        cur = []
        
        def backtrack(i):
            if i == len(s) and len(cur) == 4:
                res.append(".".join(cur))
                return
            if len(cur) > 4 or i >= len(s):
                return
            if s[i] == '0':
                cur.append('0')
                backtrack(i + 1)
                cur.pop()
                return
            j = 0
            while j < 3 and i + j < len(s):
                num = int(s[i:i+j+1])
                if num < 256:
                    cur.append(s[i:i+j+1])
                    backtrack(i + j + 1)
                    cur.pop()
                j += 1
        
        backtrack(0)
        return res
```

### Explanation
1. **Initialization**: The `res` list stores all valid IP addresses, and `cur` list keeps track of the current parts being formed.
2. **Backtracking Function**: The `backtrack` function starts at index `i` and processes the string recursively:
   - **Base Case**: If the entire string is processed (`i == len(s)`) and four parts are formed (`len(cur) == 4`), the current parts are joined into a string and added to `res`.
   - **Pruning**: If more than four parts are formed or the index `i` exceeds the string length, the function returns without further processing.
   - **Zero Handling**: If the current character is '0', it is added as a part, and the function proceeds to the next index.
   - **Digit Processing**: For non-zero characters, the function tries parts of 1 to 3 digits. Each part is checked to ensure it is less than 256. Valid parts are added to `cur`, and the function proceeds recursively.
3. **Result Compilation**: After the backtracking completes, the `res` list contains all valid IP addresses, which is returned as the result.

This approach efficiently explores all valid combinations while pruning invalid paths early, ensuring optimal performance and correctness.

class Solution:
    def restoreIpAddresses(self, s: str) -> List[str]:
        res = []
        cur = []
        
        def backtrack(i):
            if i == len(s) and len(cur) == 4:
                res.append(".".join(cur))
                return
            if len(cur) > 4 or i >= len(s):
                return
            if s[i] == '0':
                cur.append('0')
                backtrack(i + 1)
                cur.pop()
                return
            j = 0
            while j < 3 and i + j < len(s):
                num = int(s[i:i+j+1])
                if num < 256:
                    cur.append(s[i:i+j+1])
                    backtrack(i + j + 1)
                    cur.pop()
                j += 1
        
        backtrack(0)
        return res
