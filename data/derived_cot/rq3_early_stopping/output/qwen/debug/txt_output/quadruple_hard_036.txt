=== RAW ANSWER ===

To solve this problem, we need to find the median of two sorted arrays. The median is the middle value in an ordered list of numbers. If the combined length of the two arrays is even, the median is the average of the two middle numbers; if it's odd, the median is the middle number.

### Approach
1. **Merge the Arrays**: Since both input arrays are sorted, we can merge them into a single sorted array by comparing elements from both arrays and appending the smaller one to the result array.
2. **Calculate the Median**: After merging the arrays, we determine the median based on whether the total number of elements is odd or even. For an odd count, the median is the middle element. For an even count, it's the average of the two middle elements.

### Solution Code
```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        p1 = 0
        p2 = 0
        new = []
        while p1 < len(nums1) and p2 < len(nums2):
            if nums1[p1] < nums2[p2]:
                new.append(nums1[p1])
                p1 += 1
            else:
                new.append(nums2[p2])
                p2 += 1
        while p1 < len(nums1):
            new.append(nums1[p1])
            p1 += 1
        while p2 < len(nums2):
            new.append(nums2[p2])
            p2 += 1
        
        n = len(new)
        if n % 2 == 1:
            median = new[n//2]
        else:
            median = (new[n//2 - 1] + new[n//2]) / 2.0
        
        return median
```

### Explanation
1. **Merging Arrays**: The algorithm uses two pointers to traverse both arrays. It compares elements from both arrays and appends the smaller element to the result array. This ensures the merged array is sorted.
2. **Handling Remaining Elements**: After one of the arrays is exhausted, the remaining elements from the other array are appended to the result array.
3. **Finding the Median**: The median is calculated by checking the middle element(s) of the merged array. If the total number of elements is odd, the median is the middle element. If even, it's the average of the two middle elements.

This approach efficiently merges the two sorted arrays and computes the median in linear time relative to the combined length of the input arrays, making it optimal for large datasets.
