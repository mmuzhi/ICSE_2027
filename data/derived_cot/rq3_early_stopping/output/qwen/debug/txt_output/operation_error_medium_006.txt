To solve this problem, we need to identify all words in `words1` that are "supersets" of all words in `words2`. A word in `words1` is considered a superset if it contains every letter that appears in any word of `words2`, with at least the same frequency as the maximum frequency of each letter in `words2`.

### Approach
1. **Problem Analysis**: The key observation is that for a word in `words1` to be a superset of all words in `words2`, it must include each letter from `words2` with a frequency at least as high as the maximum frequency of that letter in any word of `words2`.
2. **Intuition**: We first determine the maximum frequency of each letter across all words in `words2`. Then, for each word in `words1`, we check if its letter frequencies meet or exceed these maximum frequencies.
3. **Algorithm Selection**: 
   - **Frequency Calculation**: For each word in `words2`, compute its letter frequencies and keep track of the maximum frequency for each letter.
   - **Validation**: For each word in `words1`, compare its letter frequencies against the maximum frequencies derived from `words2`. If all letters in `words2` are present in the word from `words1` with sufficient frequencies, the word is included in the result.
4. **Complexity Analysis**: The algorithm processes each word in `words2` once to compute maximum frequencies, and each word in `words1` once to validate. The time complexity is O(N*M), where N is the total number of letters in `words2` and M is the total number of letters in `words1`. The space complexity is O(1) for the frequency counters, as the number of distinct letters is limited.

### Solution Code
```python
from collections import Counter

class Solution:
    def wordSubsets(self, words1: List[str], words2: List[str]) -> List[str]:
        w2_max = Counter()
        for word in words2:
            c = Counter(word)
            for letter, count in c.items():
                if count > w2_max.get(letter, 0):
                    w2_max[letter] = count
        return [w1 for w1 in words1 if Counter(w1) >= w2_max]
```

### Explanation
1. **Frequency Calculation**: The code initializes a `Counter` (`w2_max`) to store the maximum frequency of each letter found in `words2`. For each word in `words2`, it calculates the letter frequencies and updates `w2_max` to reflect the highest frequency encountered for each letter.
2. **Validation**: For each word in `words1`, the code checks if its letter frequencies (obtained via `Counter(w1)`) are at least as high as those stored in `w2_max`. The list comprehension filters and returns all words in `words1` that meet this condition.
3. **Efficiency**: The solution efficiently processes each word in both lists by leveraging Python's `Counter` class for frequency counting, ensuring optimal performance and correctness.
