To solve this problem, we need to find for each element in `nums1` the next greater element in `nums2`. The next greater element for an element `i` in `nums1` is the first element in `nums2` that is greater than `i` and appears after the first occurrence of `i` in `nums2`. If no such element exists, we return -1 for that element in `nums1`.

### Approach
1. **Problem Analysis**: The task involves iterating through each element in `nums1` and finding its next greater element in `nums2`. The solution must efficiently locate the first occurrence of each element in `nums1` within `nums2` and then check subsequent elements in `nums2` for a greater value.
2. **Intuition**: For each element in `nums1`, we first locate its position in `nums2`. If the element is not found, the result for that element is -1. If found, we then search from its position onward in `nums2` for the next greater element. The first greater element encountered is the answer; otherwise, the result is -1.
3. **Algorithm Selection**: The algorithm involves nested loops. The outer loop processes each element in `nums1`, and the inner loop searches `nums2` for the element and its next greater element. Using Python's `index` method helps locate the first occurrence efficiently.
4. **Complexity Analysis**: The algorithm has a time complexity of O(n*m), where `n` is the length of `nums1` and `m` is the length of `nums2`. This is because for each element in `nums1`, we potentially scan the entire `nums2` list. The space complexity is O(n) to store the results.

### Solution Code
```python
class Solution:
    def nextGreaterElement(self, nums1: List[int], nums2: List[int]) -> List[int]:
        l = []
        for i in nums1:
            try:
                idx = nums2.index(i)
            except ValueError:
                l.append(-1)
                continue
            found = False
            for k in range(idx + 1, len(nums2)):
                if nums2[k] > i:
                    l.append(nums2[k])
                    found = True
                    break
            if not found:
                l.append(-1)
        return l
```

### Explanation
1. **Initialization**: We initialize an empty list `l` to store the results.
2. **Outer Loop**: For each element `i` in `nums1`, we attempt to find its index in `nums2` using `nums2.index(i)`. If `i` is not found (raises `ValueError`), we append -1 to `l` and skip further processing for this element.
3. **Inner Loop**: If `i` is found in `nums2`, we search from the next position of `i` to the end of `nums2` for the first element greater than `i`. If found, we append this element to `l` and mark `found` as `True`. If no such element is found, we append -1 to `l`.
4. **Result**: The list `l` is returned, containing the next greater element for each element in `nums1` or -1 if no greater element exists.

This approach efficiently handles the problem by leveraging Python's built-in methods and straightforward loops, ensuring correctness while maintaining clarity.
