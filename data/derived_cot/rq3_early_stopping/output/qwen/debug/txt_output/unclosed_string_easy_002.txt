=== RAW ANSWER ===

To solve this problem, we need to process a series of operations on a stack of scores and compute the final sum of all scores. The operations include doubling the last score, removing the last score, adding the last two scores, and pushing a new score. The key challenge is to correctly handle each operation while maintaining the integrity of the stack.

### Approach
1. **Stack Initialization**: Start with an empty stack to keep track of the scores.
2. **Processing Operations**: For each operation in the input list:
   - **'D' Operation**: Double the last score in the stack and push the result onto the stack.
   - **'C' Operation**: Remove the last score from the stack.
   - **'+' Operation**: Add the last two scores in the stack and push the result onto the stack.
   - **Digit String**: Convert the string to an integer and push it onto the stack.
3. **Sum Calculation**: After processing all operations, sum all the elements in the stack to get the final result.

The solution assumes that all operations are valid, meaning that operations like 'D' and '+' will only be encountered when the stack has the required elements (at least one for 'D' and at least two for '+').

### Solution Code
```python
class Solution:
    def calPoints(self, operations: List[str]) -> int:
        stack = []
        for i in operations:
            if i == 'D':
                stack.append(2 * stack[-1])
            elif i == 'C':
                stack.pop()
            elif i == '+':
                stack.append(stack[-1] + stack[-2])
            else:
                stack.append(int(i))
        return sum(stack)
```

### Explanation
- **Stack Handling**: The stack is used to keep track of the current scores. Each operation modifies the stack accordingly.
- **Operations Processing**:
  - **'D'**: The last score is doubled and pushed onto the stack.
  - **'C'**: The last score is removed from the stack.
  - **'+'**: The sum of the last two scores is computed and pushed onto the stack.
  - **Digit String**: The string is converted to an integer and pushed onto the stack.
- **Sum Calculation**: The final sum is obtained by summing all elements in the stack after processing all operations.

This approach efficiently processes each operation in linear time, making it optimal for the given problem constraints. The space complexity is also linear, as the stack may grow with the number of valid scores.
