=== RAW ANSWER ===

To solve this problem, we need to perform a level-order traversal of a binary tree and return the nodes' values from bottom to top. The level-order traversal (BFS) is done first, and then the order of levels is reversed to achieve the bottom-up order.

### Approach
1. **Base Case Handling**: If the root is None, return an empty list.
2. **BFS Initialization**: Use a queue to perform BFS starting from the root node.
3. **Level Processing**: For each level, process all nodes in the queue, record their values, and enqueue their non-None children.
4. **Reverse Levels**: After processing all levels, reverse the list of levels to get the bottom-up order.

### Solution Code
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def levelOrderBottom(self, root: Optional[TreeNode]) -> List[List[int]]:
        def checkReverse(ans):
            l, r = 0, len(ans) - 1
            while l < r:
                ans[l], ans[r] = ans[r], ans[l]
                l += 1
                r -= 1
            return ans

        if not root:
            return []
        q = [root]
        ans = []
        while q:
            n = len(q)
            level = []
            for i in range(n):
                curr = q.pop(0)
                level.append(curr.val)
                if curr.left:
                    q.append(curr.left)
                if curr.right:
                    q.append(curr.right)
            ans.append(level)
        return checkReverse(ans)
```

### Explanation
1. **Base Case Handling**: The function checks if the root is None and returns an empty list immediately.
2. **BFS Initialization**: The queue starts with the root node. The `ans` list will store each level's node values.
3. **Level Processing**: For each node in the current level, its value is added to the current level's list. The node's non-None children are enqueued for processing in subsequent levels.
4. **Reverse Levels**: The `checkReverse` function reverses the `ans` list in-place by swapping elements from the start and end moving towards the center. This gives the bottom-up order of levels.

This approach efficiently performs a level-order traversal and ensures the levels are presented in the required bottom-up order by reversing the list of levels.
