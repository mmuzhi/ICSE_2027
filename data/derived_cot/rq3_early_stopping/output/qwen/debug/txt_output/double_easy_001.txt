class Solution:
    def canThreePartsEqualSum(self, arr: List[int]) -> bool:
        total = sum(arr)
        if total % 3 != 0:
            return False
        each_sum = total // 3
        count = 0
        current = 0
        for i in range(len(arr)):
            current += arr[i]
            if current == each_sum:
                count += 1
                current = 0
            if count == 2 and i < len(arr)-1:
                return True
        return False

The solution involves checking if the total sum of the array is divisible by 3 and then verifying if we can partition the array into three parts with the required sum.

### Approach
1. **Check Total Sum Divisibility**: First, we calculate the total sum of the array. If the total sum is not divisible by 3, it's impossible to split the array into three parts with equal sums, so we return False immediately.
2. **Calculate Target Sum**: If the total sum is divisible by 3, we compute the target sum each part should achieve (i.e., total sum divided by 3).
3. **Traverse the Array**: We traverse the array while accumulating the sum of elements. Whenever the accumulated sum equals the target sum, we reset the accumulator and increment a counter.
4. **Check for Three Parts**: After accumulating the sum, if the counter reaches 2 and there are elements left in the array, we return True because the remaining elements will form the third part with the required sum. If the entire array is processed without finding three parts, we return False.

### Solution Code
```python
class Solution:
    def canThreePartsEqualSum(self, arr: List[int]) -> bool:
        total = sum(arr)
        if total % 3 != 0:
            return False
        each_sum = total // 3
        count = 0
        current = 0
        for i in range(len(arr)):
            current += arr[i]
            if current == each_sum:
                count += 1
                current = 0
            if count == 2 and i < len(arr) - 1:
                return True
        return False
