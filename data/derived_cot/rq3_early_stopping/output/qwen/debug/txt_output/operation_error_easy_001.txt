To solve this problem, we need to determine if all the occurrences (frequencies) of distinct elements in an array are unique. The solution involves counting the frequency of each element in the array and then checking if these frequencies themselves are all distinct.

### Approach
1. **Count Frequencies**: Use the `Counter` class from the `collections` module to count the occurrences of each element in the array. This gives us a dictionary where keys are the distinct elements and values are their respective counts.
2. **Check Frequency Uniqueness**: Extract the values (frequencies) from the dictionary and check if all these values are unique. This can be done by comparing the length of the set of frequencies to the length of the dictionary. If they are equal, all frequencies are unique; otherwise, there are duplicate frequencies.

### Solution Code
```python
from collections import Counter

class Solution:
    def uniqueOccurrences(self, arr: List[int]) -> bool:
        d = Counter(arr)
        return len(set(d.values())) == len(d)
```

### Explanation
1. **Counting Frequencies**: The `Counter` class efficiently counts the occurrences of each element in the input array. For example, if the input array is `[1, 2, 2, 3, 3, 3]`, the `Counter` will produce `{1: 1, 2: 2, 3: 3}`.
2. **Checking Uniqueness**: The set of values from the `Counter` dictionary is created to eliminate duplicates. If the length of this set is equal to the number of distinct elements (keys) in the dictionary, it means all frequencies are unique. For instance, in the example above, the set of values `{1, 2, 3}` has a length of 3, which matches the number of distinct elements, so the function returns `True`. Conversely, if there were duplicate frequencies, such as in `[1, 1, 2, 2]` (frequencies `{2: 2}`), the set length would be 1, which is less than the number of distinct elements (2), so the function returns `False`.

This approach efficiently checks for unique occurrences by leveraging the properties of sets and dictionaries to ensure optimal performance.
