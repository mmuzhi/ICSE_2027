=== RAW ANSWER ===

To solve this problem, we need to find the k-th smallest factor of a given integer n. The solution involves iterating through all possible factors of n in increasing order and checking if each number divides n without a remainder. If we find the k-th factor during this iteration, we return it immediately. If we finish the iteration without finding the k-th factor, we return -1.

### Approach
1. **Iterate through numbers from 1 to n**: We check each number in this range to see if it is a factor of n (i.e., if n divided by the number leaves no remainder).
2. **Count factors**: For each factor found, we increment a counter. If the counter reaches k, we return the current factor immediately.
3. **Return -1 if not found**: If the loop completes without the counter reaching k, it means there are fewer than k factors, so we return -1.

This approach efficiently checks each potential factor in ascending order, ensuring the first k factors found are the smallest ones. The loop breaks early once the k-th factor is found, optimizing performance.

### Solution Code
```python
class Solution:
    def kthFactor(self, n: int, k: int) -> int:
        count = 0
        for i in range(1, n + 1):
            if n % i == 0:
                count += 1
                if count == k:
                    return i
        return -1
```

### Explanation
- **Initialization**: We start with a counter `count` set to 0 to keep track of how many factors we have found.
- **Loop through numbers**: We iterate from 1 to n. For each number `i`, we check if `i` is a factor of `n` by verifying if `n % i` equals 0.
- **Count and check**: Each time we find a factor, we increment `count`. If `count` equals `k`, we return `i` immediately, as it is the k-th factor.
- **Return -1 if not found**: If the loop completes without finding the k-th factor, we return -1, indicating there are fewer than k factors.

This approach efficiently finds the k-th smallest factor by leveraging a simple loop and early termination, ensuring optimal performance.
