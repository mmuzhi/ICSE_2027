To solve this problem, we need to find all self-dividing numbers within a given range [left, right]. A self-dividing number is a number that is divisible by each of its digits. The solution involves checking each number in the range to determine if it meets this criterion.

### Approach
1. **Iterate through the range**: For each number in the range from `left` to `right` (inclusive), check if it is a self-dividing number.
2. **Check each digit**: For each number, convert it to a string to process each digit individually.
3. **Handle division by zero**: If any digit is zero, the number cannot be self-dividing, so we skip further checks for that number.
4. **Check divisibility**: For each non-zero digit, verify if the number is divisible by the digit. If any digit fails this check, the number is not self-dividing.
5. **Collect results**: If all digits meet the divisibility condition, add the number to the result list.

### Solution Code
```python
class Solution:
    def selfDividingNumbers(self, left: int, right: int) -> List[int]:
        res = []
        for i in range(left, right + 1):
            ch = True
            for j in str(i):
                if j == '0':
                    ch = False
                    break
                if i % int(j) != 0:
                    ch = False
                    break
            if ch:
                res.append(i)
        return res
```

### Explanation
1. **Range Inclusion**: The loop runs from `left` to `right` (inclusive) to ensure all numbers in the specified range are checked.
2. **Digit Processing**: Each number is converted to a string to iterate over its digits individually.
3. **Zero Handling**: If a digit is zero, the number is immediately marked as not self-dividing, and the loop breaks early for efficiency.
4. **Divisibility Check**: For each non-zero digit, the number is checked for divisibility. If any digit does not divide the number evenly, the number is marked as not self-dividing, and the loop breaks early.
5. **Result Collection**: Numbers that pass all checks are added to the result list, which is returned at the end.

This approach efficiently checks each number in the range, leveraging early termination during digit checks to optimize performance. The solution handles edge cases such as numbers containing zero digits correctly.
