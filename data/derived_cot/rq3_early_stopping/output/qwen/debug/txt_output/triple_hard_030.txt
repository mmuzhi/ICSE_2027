To solve this problem, we need to find an Eulerian trail in a directed graph represented by a list of pairs. An Eulerian trail is a path that uses every edge exactly once. The solution involves constructing the graph, determining the start node based on the degrees of the nodes, and performing a depth-first search (DFS) to traverse the graph and record the trail.

### Approach
1. **Graph Construction**: Build a graph using adjacency lists and a dictionary to keep track of the net out-degree of each node. The net out-degree is calculated as the number of outgoing edges minus the number of incoming edges.
2. **Start Node Identification**: The start node of the Eulerian trail is determined by the node with a net out-degree of 1. If all nodes have a net out-degree of 0, any node can be chosen as the start (indicating an Eulerian circuit).
3. **DFS Traversal**: Perform a DFS starting from the identified node. During traversal, edges are removed as they are used, ensuring each edge is visited exactly once. The nodes are recorded in reverse order during traversal.
4. **Result Construction**: The recorded nodes are reversed to form the correct trail order. The trail is then converted into a list of edges.

### Solution Code
```python
from collections import defaultdict

class Solution:
    def validArrangement(self, pairs: List[List[int]]) -> List[List[int]]:
        graph = defaultdict(list)
        degree = defaultdict(int)
        for x, y in pairs:
            graph[x].append(y)
            degree[x] += 1
            degree[y] -= 1
        
        start = None
        for node, d in degree.items():
            if d == 1:
                start = node
                break
        if start is None:
            start = next(iter(graph.keys())) if graph else None
        
        ans = []
        
        def dfs(node):
            while graph[node]:
                neighbor = graph[node].pop()
                dfs(neighbor)
            ans.append(node)
        
        if start is not None:
            dfs(start)
        
        ans.reverse()
        return [[ans[i], ans[i+1]] for i in range(len(ans)-1)]
```

### Explanation
1. **Graph Construction**: The graph is built using a defaultdict where each node maps to a list of its neighbors. The degree dictionary tracks the net out-degree of each node by incrementing for outgoing edges and decrementing for incoming edges.
2. **Start Node Identification**: The start node is the first node found with a net out-degree of 1. If no such node exists, any node from the graph is chosen (indicating an Eulerian circuit).
3. **DFS Traversal**: The DFS function recursively traverses the graph, removing edges as they are used. Nodes are appended to the answer list when all outgoing edges are exhausted.
4. **Result Construction**: The answer list is reversed to present the trail in the correct order. The trail is then converted into a list of edges by pairing consecutive nodes in the answer list.

This approach efficiently constructs the Eulerian trail by leveraging graph traversal and careful management of node degrees, ensuring all edges are used exactly once.
