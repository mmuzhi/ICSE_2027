To solve this problem, we need to find the minimum difference between any two elements in a contiguous subarray of length `k` within the given array `nums`. The solution involves using a sliding window approach to efficiently explore all possible subarrays of length `k` and determine the smallest difference between any two elements within these subarrays.

### Approach
1. **Sliding Window Technique**: We use a sliding window of fixed size `k` to traverse the array. This allows us to efficiently consider each contiguous subarray of length `k` exactly once.
2. **Sorting Each Window**: For each window, we sort the elements to easily find the minimum difference between adjacent elements, which will be the smallest difference in that window.
3. **Tracking Minimum Difference**: As we process each window, we keep track of the smallest difference encountered so far by comparing the minimum difference of the current window with the overall minimum.

### Solution Code
```python
class Solution:
    def minimumDifference(self, nums: List[int], k: int) -> int:
        n = len(nums)
        if k > n:
            return 0
        
        res = float('inf')
        l, r = 0, k
        while r < len(nums):
            window = nums[l:r]
            window.sort()
            min_diff = float('inf')
            for i in range(1, len(window)):
                diff = window[i] - window[i-1]
                if diff < min_diff:
                    min_diff = diff
            if min_diff < res:
                res = min_diff
            l += 1
            r += 1
        
        return res
```

### Explanation
1. **Initialization**: The function starts by checking if `k` is larger than the array length `n`. If so, it returns 0 since no subarray of length `k` exists.
2. **Sliding Window Setup**: The window is initialized with left pointer `l` at 0 and right pointer `r` at `k`. This defines the first window of size `k`.
3. **Processing Each Window**: For each window, the elements are sorted. The minimum difference between adjacent elements in the sorted window is computed.
4. **Updating Result**: The smallest difference found in the current window is compared with the overall minimum (`res`). If it's smaller, `res` is updated.
5. **Window Movement**: The window is moved by incrementing both `l` and `r` by 1, ensuring the window size remains `k`.
6. **Result Return**: After processing all possible windows, the smallest difference found (`res`) is returned.

This approach efficiently explores all contiguous subarrays of length `k` using a sliding window and ensures the solution is found in optimal time by leveraging sorting within each window.
