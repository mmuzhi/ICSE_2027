To solve this problem, we need to find a valid arrangement of given pairs such that they form a sequence where each consecutive pair shares a common element, and all pairs are used exactly once. This is essentially finding an Eulerian trail in a directed graph where each edge is represented as a directed edge from one node to another.

### Approach
1. **Graph Construction**: Represent the graph using adjacency lists where each node points to its neighbors. Also, maintain a dictionary to track the net out-degree of each node (out-degree minus in-degree).
2. **Identify Start Node**: The start node for the Eulerian trail is the node with a net out-degree of 1. If no such node exists, the graph might be an Eulerian circuit, and any node can be chosen as the start.
3. **Depth-First Search (DFS)**: Perform a DFS starting from the identified node. During the traversal, each time we visit a node, we remove an edge from the adjacency list and recursively visit the neighbor node. Once all edges from a node are processed, we append the node to the result list.
4. **Reverse the Result**: The nodes are collected in reverse order during the DFS. Reversing this list gives the correct sequence of nodes for the Eulerian trail.
5. **Convert to Pairs**: Convert the sequence of nodes into pairs of consecutive nodes to form the final arrangement.

### Solution Code
```python
from collections import defaultdict

class Solution:
    def validArrangement(self, pairs: List[List[int]]) -> List[List[int]]:
        graph = defaultdict(list)
        degree = defaultdict(int)
        for x, y in pairs:
            graph[x].append(y)
            degree[x] += 1
            degree[y] -= 1
        
        start_node = None
        for k in degree:
            if degree[k] == 1:
                start_node = k
                break
        
        if start_node is None:
            start_node = next(iter(degree.keys()))
        
        ans = []
        def dfs(x):
            while graph[x]:
                y = graph[x].pop()
                dfs(y)
            ans.append(x)
        
        dfs(start_node)
        ans.reverse()
        return [[ans[i], ans[i+1]] for i in range(len(ans)-1)]
```

### Explanation
1. **Graph Construction**: The graph is built using a defaultdict where each node maps to a list of its neighbors. The `degree` dictionary keeps track of the net out-degree for each node.
2. **Start Node Identification**: The start node is identified as the node with a net out-degree of 1. If no such node exists, the first node in the graph is chosen as the start.
3. **DFS Traversal**: The DFS function processes each node by recursively visiting its neighbors, removing each edge from the adjacency list as it is used. Once all edges from a node are processed, the node is added to the result list.
4. **Result Construction**: The result list is reversed to get the correct sequence of nodes. The sequence is then converted into pairs of consecutive nodes to form the final arrangement.
