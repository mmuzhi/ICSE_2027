To solve this problem, we need to determine the maximum profit achievable by buying and selling a stock at most once. The solution involves iterating through the stock prices and calculating the profit from each potential transaction, ensuring that we only consider profitable transactions.

### Approach
1. **Problem Analysis**: The problem requires finding the maximum profit from buying and selling a stock once. The key insight is that the maximum profit is the difference between the highest and lowest prices, but only if the highest price occurs after the lowest price.
2. **Intuition**: By iterating through the list of prices, we can keep track of the minimum price encountered so far. For each subsequent price, if it is higher than the minimum, the difference between the current price and the minimum price gives a potential profit. Summing all such positive differences yields the maximum profit.
3. **Algorithm Selection**: The algorithm involves a single pass through the list of prices, maintaining the minimum price encountered. For each price, if it is higher than the current minimum, the difference is added to the profit. This approach efficiently computes the maximum profit in O(n) time with O(1) space complexity.
4. **Complexity Analysis**: The algorithm runs in linear time, O(n), where n is the number of days (prices). This is optimal for the problem as we need to examine each price at least once.

### Solution Code
```python
class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        if len(prices) < 2:
            return 0
        min_price = prices[0]
        max_profit = 0
        for i in range(1, len(prices)):
            if prices[i] < min_price:
                min_price = prices[i]
            else:
                profit = prices[i] - min_price
                if profit > max_profit:
                    max_profit = profit
        return max_profit
```

### Explanation
1. **Initialization**: The function starts by checking if there are fewer than 2 days of prices, in which case no transaction is possible, returning 0.
2. **Tracking Minimum Price**: The algorithm initializes `min_price` to the first day's price and `max_profit` to 0.
3. **Iterating Through Prices**: For each subsequent day, if the current price is lower than `min_price`, `min_price` is updated. Otherwise, the potential profit from buying at `min_price` and selling at the current price is calculated. If this profit exceeds the current `max_profit`, `max_profit` is updated.
4. **Result**: After processing all prices, the function returns `max_profit`, which holds the highest profit achievable from a single transaction.

This approach efficiently computes the maximum profit by leveraging a single pass through the data, ensuring optimal performance and correctness.
