To solve this problem, we need to find the longest harmonious subsequence in an array of integers. A harmonious subsequence is defined as a subsequence where all elements are either the same or differ by exactly 1. The goal is to find the longest such subsequence.

### Approach
1. **Frequency Map Creation**: First, we create a frequency map (dictionary) to count the occurrences of each number in the input array. This helps in efficiently checking the counts of consecutive numbers later.
2. **Iterate Through Frequency Map**: For each number in the frequency map, we check if the next consecutive number (current number + 1) exists in the map. If it does, the potential length of the harmonious subsequence is the sum of the counts of the current number and its consecutive number.
3. **Track Maximum Length**: We keep track of the maximum length encountered during the iteration. If no such consecutive numbers exist, the result remains 0, indicating no harmonious subsequence was found.

### Solution Code
```python
class Solution:
    def findLHS(self, nums: List[int]) -> int:
        count_map = {}
        for num in nums:
            count_map[num] = count_map.get(num, 0) + 1
        
        result = 0
        for num in count_map:
            if num + 1 in count_map:
                result = max(result, count_map[num] + count_map[num + 1])
        return result
```

### Explanation
1. **Frequency Map Creation**: The loop iterates through each number in the input array, updating the count of each number in the `count_map` dictionary. This ensures we know how many times each number appears in the array.
2. **Checking Consecutive Numbers**: For each number in the dictionary, we check if the next consecutive number (current number + 1) exists. If it does, the sum of their counts is a candidate for the longest harmonious subsequence.
3. **Result Calculation**: The maximum value encountered during these checks is stored in `result`, which is returned as the length of the longest harmonious subsequence. If no consecutive numbers are found, the result remains 0.

This approach efficiently leverages the frequency map to minimize repeated calculations and ensures optimal performance with a time complexity of O(n), where n is the number of elements in the input array.
