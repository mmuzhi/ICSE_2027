To solve this problem, we need to determine if two strings are almost equal. Two strings are considered almost equal if they have the same length and at most two positions where the characters differ. If there are exactly two differing positions, the characters at these positions must be the same but swapped between the two strings.

### Approach
1. **Check Lengths**: If the lengths of the two strings are different, they cannot be almost equal.
2. **Identify Differences**: Iterate through each character position in the strings and record the differing characters.
3. **Check Conditions**:
   - If there are no differences, the strings are identical and thus almost equal.
   - If there are exactly two differences, check if the characters at these positions are the same but swapped (i.e., the first differing character in the first string matches the second differing character in the second string, and vice versa).
   - If there are more than two differences or exactly one difference, the strings are not almost equal.

### Solution Code
```python
class Solution:
    def areAlmostEqual(self, s1: str, s2: str) -> bool:
        if len(s1) != len(s2):
            return False
        if s1 == s2:
            return True
        
        diff = []
        for i in range(len(s1)):
            if s1[i] != s2[i]:
                diff.append((s1[i], s2[i]))
                if len(diff) > 2:
                    return False
        
        if len(diff) == 0:
            return True
        elif len(diff) == 2:
            a, b = diff[0]
            c, d = diff[1]
            return a == d and b == c
        else:
            return False
```

### Explanation
1. **Length Check**: The function first checks if the lengths of the two strings are different. If they are, it returns `False` immediately.
2. **Identical Strings Check**: If the strings are identical, it returns `True` since they trivially satisfy the almost equal condition.
3. **Difference Tracking**: The function iterates through each character position in the strings. For each position where the characters differ, it records the pair of differing characters. If more than two differences are found during this iteration, it returns `False` early.
4. **Difference Handling**:
   - If no differences are found, the strings are identical, and the function returns `True`.
   - If exactly two differences are found, it checks if the characters at these positions are the same but swapped (i.e., the first differing character in the first string matches the second differing character in the second string, and vice versa). If so, it returns `True`; otherwise, `False`.
   - For any other number of differences (like one or more than two), the function returns `False`.

This approach efficiently checks the almost equal condition by leveraging early termination for invalid cases and clear conditions for valid cases.
