To solve this problem, we need to add 1 to a number represented as a list of its digits and return the result as a list of digits. The key challenge is handling cases where adding 1 causes a carry-over, such as when the input is [9], which should become [1, 0].

### Approach
1. **Convert Digits to String**: The list of digits is first converted into a string by joining each digit. This allows us to handle the number as a whole.
2. **Convert String to Integer**: The concatenated string is then converted into an integer, and 1 is added to this integer.
3. **Convert Result Back to Digits**: The incremented integer is converted back into a string, and each character (digit) is converted back into an integer, resulting in a list of digits.

This approach efficiently handles all cases, including those where adding 1 causes a carry-over, by leveraging Python's ability to handle arbitrary large integers and simple string manipulations.

### Solution Code
```python
class Solution:
    def plusOne(self, digits: List[int]) -> List[int]:
        s = ''.join(map(str, digits))
        i = int(s) + 1
        return list(map(int, str(i)))
```

### Explanation
1. **String Conversion**: The input list of digits is converted into a string by joining each digit. For example, [1, 2, 3] becomes "123".
2. **Integer Addition**: The string is converted to an integer, and 1 is added to this integer. For example, "123" becomes 124.
3. **Back to Digits**: The resulting integer is converted back into a string, and each character is converted into an integer. For example, 124 becomes [1, 2, 4].

This method ensures that all edge cases, including carry-over scenarios, are handled correctly. The solution is both concise and efficient, leveraging Python's built-in functions for simplicity and clarity.
