To solve this problem, we need to generate all valid IP addresses from a given string. An IP address consists of four parts, each part being a number between 0 and 255. The numbers cannot have leading zeros unless the part is exactly "0". 

### Approach
1. **Backtracking**: We use a backtracking approach to explore all possible ways to split the string into four parts. The algorithm recursively tries to form each part by considering 1 to 3 digits (unless the part is "0", in which case only one digit is allowed).
2. **Base Case**: When the entire string is processed and exactly four parts are formed, the IP address is added to the result.
3. **Validation**: During the backtracking, each part is checked to ensure it is a valid number (0-255) and does not have leading zeros unless it is "0".

### Solution Code
```python
class Solution():
    def restoreIpAddresses(self, s: str) -> List[str]:
        res = []
        cur = []
        
        def backtrack(i):
            if i == len(s) and len(cur) == 4:
                res.append(".".join(cur))
                return
            if len(cur) > 4 or i > len(s):
                return
            if s[i] == '0':
                cur.append(s[i])
                backtrack(i + 1)
                cur.pop()
                return
            j = 0
            while j < 3 and i + j < len(s):
                if int(s[i:i+j+1]) < 256:
                    cur.append(s[i:i+j+1])
                    backtrack(i + j + 1)
                    cur.pop()
                j += 1
        
        backtrack(0)
        return res
```

2. **Backtracking Function**: The nested `backtrack` function processes the string starting from index `i`:
   - **Base Case**: If the entire string is processed (`i == len(s)`) and exactly four parts are formed (`len(cur) == 4`), the current parts are joined into a string and added to `res`.

   - **Handling '0'**: If the current character is '0', it is added as a part, and the function proceeds to the next index.
   - **Non-zero Characters**: For non-zero characters, the function tries parts of length 1 to 3 digits (if possible) and checks if the formed number is less than 256.                     cur.append(s[i:i+j+1])
                    backtrack(i + j + 1)
                    cur.pop()
                j += 1
        
        backtrack(0)
        return res
