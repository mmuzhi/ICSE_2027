To solve this problem, we need to find the closest meeting node in a directed graph where each node has at most one outgoing edge. The meeting node is defined as the node that can be reached from both starting nodes (node1 and node2) with the smallest maximum distance from either starting node. If no such node exists, we return -1.

### Approach
1. **Problem Analysis**: The graph is represented as a list where each element indicates the next node for the corresponding node (or -1 if there is no outgoing edge). The goal is to compute the shortest distances from both starting nodes to all other nodes and then determine the node with the smallest maximum distance from either starting node.
2. **Intuition**: We use Breadth-First Search (BFS) from each starting node to compute the shortest distances to all reachable nodes. BFS is suitable here because it naturally explores nodes level by level, ensuring the shortest path is found first.
3. **Algorithm Selection**: We perform BFS twice—once from node1 and once from node2—to compute two distance arrays (`dist1` and `dist2`). For each node, if it is reachable from both starting nodes, we compute the maximum of the two distances. The node with the smallest such maximum distance is our answer. If multiple nodes have the same maximum distance, the smallest node index is chosen.
4. **Complexity Analysis**: The BFS operations run in O(N) time each, where N is the number of nodes, since each node and edge is processed at most once. The final pass through the distance arrays is also O(N). Thus, the overall complexity is O(N), which is efficient for the problem constraints.

### Solution Code
```python
import collections

class Solution:
    def closestMeetingNode(self, edges: List[int], node1: int, node2: int) -> int:
        n = len(edges)
        INF = float('inf')
        dist1 = [INF] * n
        dist2 = [INF] * n

        # BFS for node1
        q = collections.deque()
        dist1[node1] = 0
        q.append(node1)
        while q:
            cur = q.popleft()
            nxt = edges[cur]
            if nxt != -1 and dist1[nxt] == INF:
                dist1[nxt] = dist1[cur] + 1
                q.append(nxt)

        # BFS for node2
        q = collections.deque()
        dist2[node2] = 0
        q.append(node2)
        while q:
            cur = q.popleft()
            nxt = edges[cur]
            if nxt != -1 and dist2[nxt] == INF:
                dist2[nxt] = dist2[cur] + 1
                q.append(nxt)

        ans = -1
        min_d = INF
        for i in range(n):
            if dist1[i] != INF and dist2[i] != INF:
                d = max(dist1[i], dist2[i])
                if d < min_d:
                    min_d = d
                    ans = i
                elif d == min_d and i < ans:
                    ans = i

        return ans
```

### Explanation
1. **Initialization**: We initialize two distance arrays (`dist1` and `dist2`) with infinity (`INF`), except for the starting nodes (node1 and node2), which are set to 0.
2. **BFS Execution**: Using a queue, we perform BFS from each starting node. For each node processed, we check its outgoing edge. If the edge leads to an unvisited node, we update its distance and enqueue it.
3. **Finding the Meeting Node**: After computing distances from both starting nodes, we iterate through all nodes. For each node reachable from both starting nodes, we compute the maximum of the two distances. The node with the smallest such maximum distance is selected. If multiple nodes have the same maximum distance, the smallest node index is chosen.
4. **Result Handling**: If no valid meeting node is found, we return -1; otherwise, we return the index of the closest meeting node. This approach efficiently handles the problem by leveraging BFS to explore the graph and then selecting the optimal node based on computed distances.

        # BFS for node1
        q = collections.deque()
        dist1[node1] = 0
        q.append(node1)
        while q:
            cur = q.popleft()
            nxt = edges[cur]
            if nxt != -1 and dist1[nxt] == INF:
                dist1[nxt] = dist1[cur] + 1
                q.append(nxt)

        # BFS for node2
        q = collections.deque()
        dist2[node2] = 0
        q.append(node2)
        while q:
            cur = q.popleft()
            nxt = edges[cur]
            if nxt != -1 and dist2[nxt] == INF:
                dist2[nxt] = dist2[cur] + 1
                q.append(nxt)

        ans = -1
        min_d = INF
        for i in range(n):
            if dist1[i] != INF and dist2[i] != INF:
                d = max(dist1[i], dist2[i])
                if d < min_d:
                    min_d = d
                    ans = i
                elif d == min_d and i < ans:
                    ans = i

        return ans
