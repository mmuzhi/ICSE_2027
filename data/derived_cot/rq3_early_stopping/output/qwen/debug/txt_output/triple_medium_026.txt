### Solution Code
```python
from collections import Counter

class Solution:
    def minSteps(self, s: str, t: str) -> int:
        cnt1 = Counter(s)
        cnt2 = Counter(t)
        sm = 0
        for char in cnt2:
            sm += max(0, cnt2[char] - cnt1.get(char, 0))
        return sm
```

### Explanation
1. **Frequency Counters**: The `Counter` objects `cnt1` and `cnt2` store the frequency of each character in strings `s` and `t`, respectively.
2. **Character Comparison**: For each character in `t` (accessed via `cnt2`), we check if it exists in `s` (`cnt1`). If it does, we compute the difference between the counts in `t` and `s`. 3. **Result Calculation**: The total steps required to transform `s` into `t` by inserting characters is the sum of all these differences, which is returned as the result.

This approach efficiently calculates the minimal insertions by leveraging frequency counts, ensuring optimal performance with a time complexity of O(n), where n is the length of the longer string.
