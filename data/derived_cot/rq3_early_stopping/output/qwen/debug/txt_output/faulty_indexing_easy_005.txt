The solution involves sorting each row in ascending order and then sequentially removing the largest remaining values from each row in each operation.

### Approach

2. **Iterate Over Columns**: For each column index (from 0 to the number of columns minus one), we remove the value at the corresponding position from each row. Specifically, for the j-th operation, we remove the j-th largest value from each row (i.e., the value at position `-j` in the sorted row).
3. **Sum Removed Values**: During each operation, we sum all the values removed from each row. This sum is accumulated across all operations to get the final result.

### Solution Code
