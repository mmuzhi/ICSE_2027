To solve this problem, we need to find the maximum profit achievable by buying and selling a stock at most once. The solution involves tracking the minimum price encountered while iterating through the stock prices and calculating the maximum profit by comparing each subsequent price with this minimum.

### Approach
1. **Initialization**: If the list of prices has fewer than 2 elements, return 0 since no transaction can be made.
2. **Tracking Minimum Price**: Iterate through the list of prices starting from the second element. For each price, check if it is lower than the current minimum price. If it is, update the minimum price.
3. **Calculating Maximum Profit**: Whenever a price is higher than the current minimum price, calculate the potential profit by subtracting the minimum price from the current price. Update the maximum profit if this potential profit is greater than the previously recorded maximum profit.
4. **Return Result**: After processing all prices, return the maximum profit found. If no profit is possible, return 0.

### Solution Code
```python
class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        if len(prices) < 2:
            return 0
        min_price = prices[0]
        max_profit = 0
        for i in range(1, len(prices)):
            if prices[i] < min_price:
                min_price = prices[i]
            else:
                max_profit = max(max_profit, prices[i] - min_price)
        return max_profit
```

### Explanation
- **Initialization**: The function first checks if there are at least two prices. If not, it returns 0 because no transaction is possible.
- **Tracking Minimum Price**: The variable `min_price` is initialized to the first price in the list. As we iterate through the list, `min_price` is updated whenever a lower price is encountered.
- **Calculating Maximum Profit**: For each subsequent price, if it is higher than `min_price`, the potential profit (current price minus `min_price`) is calculated. The maximum of this potential profit and the previously recorded `max_profit` is stored in `max_profit`.
- **Result**: After processing all prices, the function returns `max_profit`, which holds the highest profit achievable by buying at the lowest price and selling at the highest subsequent price. If no profit is found, it returns 0.

This approach efficiently computes the maximum profit in a single pass through the list, making it optimal with a time complexity of O(n) and a space complexity of O(1).
