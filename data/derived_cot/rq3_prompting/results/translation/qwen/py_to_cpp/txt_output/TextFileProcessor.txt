3. **File Operations**: Utilize `std::ifstream` and `std::ofstream` for file reading and writing operations.
4. **Error Handling**: Throw exceptions for invalid JSON or file I/O errors to match the Python behavior.
5. **Character Filtering**: Process the file content by removing non-alphabetic characters using standard C++ string manipulation.

### Solution Code
```cpp
#include <fstream>
#include <string>
#include <stdexcept>
#include <nlohmann/json.hpp>
#include <cctype>

class TextFileProcessor {
public:
    TextFileProcessor(const std::string& file_path) : file_path(file_path) {}

    template<typename T>
    T read_file_as_json() {
        std::ifstream file(file_path);
        if (!file.is_open()) {
            throw std::runtime_error("Could not open file for JSON reading");
        }

        nlohmann::json json_data;
        file >> json_data;
        return json_data.get<T>();
    }

    std::string read_file() {
        std::ifstream file(file_path);
        if (!file.is_open()) {
            throw std::runtime_error("Could not open file for reading");
        }

        std::string content;
        std::string line;
        while (std::getline(file, line)) {
            content += line;
        }
        return content;
    }

    void write_file(const std::string& content) {
        std::ofstream file(file_path);
        if (!file.is_open()) {
            throw std::runtime_error("Could not open file for writing");
        }

        file << content;
    }

    std::string process_file() {
        std::string content = read_file();
        std::string processed;
        for (char c : content) {
            if (std::isalpha(static_cast<unsigned char>(c))) {
                processed += c;
            }
        }
        write_file(processed);
        return processed;
    }

private:
    std::string file_path;
};
```

### Explanation
1. **Constructor**: Initializes the file path.
2. **JSON Reading**: `read_file_as_json` reads the file as JSON and returns it as a specified type (e.g., `dict`, `list`). It uses the `nlohmann/json` library for parsing.
3. **File Reading**: `read_file` reads the entire file content into a string.
4. **File Writing**: `write_file` writes the provided content to the file, overwriting any existing content.
5. **Processing**: `process_file` reads the file, filters out non-alphabetic characters, writes the result back, and returns it. The filtering is done using `std::isalpha`.

This C++ implementation maintains the same functionality as the Python class, including error handling and behavior for edge cases, while using idiomatic C++ constructs where appropriate.
