=== RAW ANSWER ===

The function `str2datetime` processes the input string '1970-01-01T00:00:00.0Z' as follows:

1. Checks if the string is "0000-00-00T24:60:60Z", which it is not.
2. Extracts the fractional seconds (milliseconds/microseconds) part: `ms = string[20:-1]`.
   - The string indices: position 20 is '0' (after '.' and before 'Z'), and `-1` is the last character ('Z'), excluded. Thus, `ms = '0'`.
3. Pads `ms` to 6 digits (microseconds) by appending zeros: `ms += "000000"[:6 - len(ms)]` → `6 - 1 = 5`, so appends "00000" → `ms = '000000'`.
4. Parses the datetime components:
   - Year: `int('1970')` → 1970
   - Month: `int('01')` → 1
   - Day: `int('01')` → 1
   - Hour: `int('00')` → 0
   - Minute: `int('00')` → 0
   - Second: `int('00')` → 0
   - Microsecond: `int('000000')` → 0
5. Constructs and returns a `datetime.datetime` object: `datetime.datetime(1970, 1, 1, 0, 0, 0, 0)`.

In Python, when this object is represented (e.g., via `repr()`), seconds and microseconds are omitted if zero, resulting in `datetime.datetime(1970, 1, 1, 0, 0)`.
