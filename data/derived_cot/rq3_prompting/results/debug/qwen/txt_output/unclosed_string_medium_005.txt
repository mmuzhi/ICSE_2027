To solve this problem, we need to compute the sum of the minimum values of all possible subarrays in a given array. The challenge is to do this efficiently, especially since the array can be very large, making a brute-force approach infeasible.

### Approach
The efficient approach involves using a monotonic stack to keep track of elements and their positions, allowing us to compute the contribution of each element as the minimum in various subarrays. The key steps are:

1. **Compute Left Boundaries (L)**: For each element in the array, find the index of the previous element that is strictly less than the current element. This is done by traversing the array from left to right and using a stack to maintain indices of elements in non-decreasing order.
2. **Compute Right Boundaries (R)**: For each element in the array, find the index of the next element that is strictly less than the current element. This is done by traversing the array from right to left and using a stack to maintain indices of elements in non-decreasing order.
3. **Calculate Contributions**: For each element, the number of subarrays where it is the minimum is given by the product of the distances to the left and right boundaries. Multiply this by the element's value and accumulate the result, applying modulo \(10^9 + 7\) at each step to prevent overflow.

### Solution Code
```python
class Solution:
    def sumSubarrayMins(self, arr: List[int]) -> int:
        n = len(arr)
        L = [-1] * n
        R = [n] * n
        
        stack = []
        for i in range(n):
            while stack and arr[stack[-1]] >= arr[i]:
                stack.pop()
            if stack:
                L[i] = stack[-1]
            else:
                L[i] = -1
            stack.append(i)
        
        stack = []
        for i in range(n-1, -1, -1):
            while stack and arr[stack[-1]] >= arr[i]:
                stack.pop()
            if stack:
                R[i] = stack[-1]
            else:
                R[i] = n
            stack.append(i)
        
        res = 0
        mod = 10**9 + 7
        for i in range(n):
            res = (res + arr[i] * (i - L[i]) * (R[i] - i)) % mod
        
        return res
```

### Explanation
1. **Left Boundaries (L)**: The stack is used to keep track of indices in a non-decreasing order. For each element, we pop elements from the stack until we find one that is less than the current element. The remaining stack top gives the previous smaller element's index, which is stored in L.
2. **Right Boundaries (R)**: Similarly, we traverse the array from right to left, using a stack to find the next smaller element. The stack helps maintain indices in non-decreasing order, and the top of the stack after popping gives the next smaller element's index, stored in R.
3. **Contribution Calculation**: For each element at index `i`, the number of subarrays where it is the minimum is calculated as `(i - L[i]) * (R[i] - i)`. This product is multiplied by the element's value and added to the result, with modulo \(10^9 + 7\) applied to keep the numbers manageable.

This approach efficiently computes the desired sum in linear time using stack operations, making it suitable for large input sizes.

        R = [n] * n
        
        stack = []
        for i in range(n):
            while stack and arr[stack[-1]] >= arr[i]:
                stack.pop()
            if stack:
                L[i] = stack[-1]
            else:
                L[i] = -1
            stack.append(i)
        
        stack = []
        for i in range(n-1, -1, -1):
            while stack and arr[stack[-1]] >= arr[i]:
                stack.pop()
            if stack:
                R[i] = stack[-1]
            else:
                R[i] = n
            stack.append(i)
        
        res = 0
        mod = 10**9 + 7
        for i in range(n):
            res = (res + arr[i] * (i - L[i]) * (R[i] - i)) % mod
        
        return res
