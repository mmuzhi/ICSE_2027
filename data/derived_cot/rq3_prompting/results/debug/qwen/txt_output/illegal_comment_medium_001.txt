To solve this problem, we need to determine if it's possible to finish all courses given the prerequisites. The key insight is to detect cycles in the directed graph formed by the course dependencies. If there's a cycle, it's impossible to finish all courses; otherwise, it is possible.

### Approach
1. **Graph Construction**: Represent the courses and their dependencies as a directed graph where an edge from course `a` to course `b` indicates that `a` is a prerequisite for `b`.
2. **Cycle Detection**: Use Depth-First Search (DFS) to detect cycles in the graph. During DFS, maintain two sets: one for nodes currently in the recursion stack (to detect cycles) and another for nodes that have been fully processed (no cycles detected in their subtree).
3. **DFS Traversal**: For each course, if it hasn't been processed, initiate DFS. During DFS, mark the course as being visited and check its prerequisites. If any prerequisite leads to a cycle, return false. If no cycle is found, mark the course as processed.

### Solution Code
```python
class Solution:
    def canFinish(self, numCourses: int, prerequisites: List[List[int]]) -> bool:
        pre = defaultdict(list)
        for a, b in prerequisites:
            pre[b].append(a)

        rec_stack = set()

            if course in visited:
```

### Explanation
1. **Graph Construction**: The graph is built using a dictionary where each key is a course, and the value is a list of its prerequisites. This ensures that for each course, we know which courses must be completed before it.
2. **Cycle Detection**: The DFS function checks if a course is currently in the recursion stack (indicating a cycle) or has been visited (indicating no cycle was found in its subtree). If a course is found in the recursion stack, a cycle exists, and the function returns false.
3. **DFS Traversal**: The main loop iterates over all courses. For each unvisited course, it initiates DFS. If DFS finds a cycle, the function immediately returns false. If all courses are processed without detecting any cycles, the function returns true, indicating all courses can be finished.
