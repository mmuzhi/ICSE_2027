=== RAW ANSWER ===

To solve this problem, we need to reconstruct the original array from a list of adjacent pairs. The array is a linear sequence, meaning the graph formed by these pairs is a path graph with two endpoints. The solution involves traversing this graph starting from one endpoint to the other, collecting nodes in the order they are visited.

### Approach
1. **Graph Construction**: Build an adjacency list from the given list of adjacent pairs. Each node will have a list of its neighbors.
2. **Identify Endpoints**: The endpoints of the path graph are nodes with exactly one neighbor (degree 1). We start the traversal from one such endpoint.
3. **Depth-First Search (DFS)**: Use a stack to perform DFS. The stack keeps track of nodes to visit next. For each node, we visit all its unvisited neighbors and mark them as visited. The traversal continues until all nodes are visited, collecting them in the order they are visited.

### Solution Code
```python
class Solution:
    def restoreArray(self, adjacentPairs: List[List[int]]) -> List[int]:
        graph = {}
        for u, v in adjacentPairs:
            graph.setdefault(u, []).append(v)
            graph.setdefault(v, []).append(u)
        
        ans = []
        seen = set()
        stack = [next(x for x in graph if len(graph[x]) == 1)]
        while stack:
            n = stack.pop()
            ans.append(n)
            seen.add(n)
            for nn in graph[n]:
                if nn not in seen:
                    stack.append(nn)
        return ans
```

### Explanation
1. **Graph Construction**: The code first constructs an adjacency list from the input list of adjacent pairs. Each node's neighbors are stored in a list.
2. **Finding Start Node**: The traversal starts from a node with only one neighbor (an endpoint). This is found using a generator expression that selects the first node with degree 1.
3. **DFS Traversal**: Using a stack, the code performs DFS. Nodes are popped from the stack, added to the result list, and their unvisited neighbors are pushed onto the stack. This ensures that nodes are visited in the order they are encountered, reconstructing the original array.
4. **Result Compilation**: The result list is built during the traversal, collecting nodes in the order they are visited, which forms the reconstructed array.

class Solution:
    def restoreArray(self, adjacentPairs: List[List[int]]) -> List[int]:
        graph = {}
        for u, v in adjacentPairs:
            graph.setdefault(u, []).append(v)
            graph.setdefault(v, []).append(u)
        
        ans = []
        seen = set()
        stack = [next(x for x in graph if len(graph[x]) == 1)]
        while stack:
            n = stack.pop()
            ans.append(n)
            seen.add(n)
            for nn in graph[n]:
                if nn not in seen:
                    stack.append(nn)
        return ans
