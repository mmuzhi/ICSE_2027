To solve this problem, we need to find the longest continuous subsequence of numbers that differ by at most 1 in an array of integers. The solution involves counting the occurrences of each number and then checking for pairs of consecutive numbers (like x and x+1) to determine the longest possible subsequence.

### Approach
1. **Frequency Map Construction**: First, we construct a frequency map (dictionary) where each key is a unique number from the input array, and the value is its frequency in the array. This helps in efficiently checking the counts of consecutive numbers later.
2. **Finding Longest Harmonic Subsequence**: For each number in the frequency map, if the next consecutive number (num + 1) exists in the map, the candidate length is the sum of the frequencies of the current number and the next consecutive number. Otherwise, the candidate length is just the frequency of the current number. The maximum of all these candidate lengths is the answer.

### Solution Code
```python
from collections import defaultdict

class Solution:
    def findLHS(self, nums: List[int]) -> int:
        count_map = defaultdict(int)
        for num in nums:
            count_map[num] += 1
        
        result = 0
        for num in count_map:
            if num + 1 in count_map:
                candidate = count_map[num] + count_map[num + 1]
            else:
                candidate = count_map[num]
            if candidate > result:
                result = candidate
        return result
```

1. **Frequency Map Construction**: The code uses a `defaultdict` to count occurrences of each number in the input array. This allows efficient lookup of counts for any number.
2. **Iterating Through the Map**: For each number in the map, the code checks if the next consecutive number (num + 1) exists. If it does, the candidate length is the sum of the frequencies of the current number and the next consecutive number. If not, the candidate length is simply the frequency of the current number.
3. **Tracking Maximum Length**: The code keeps track of the maximum candidate length encountered during the iteration and returns this value as the result. This approach ensures that we efficiently find the longest subsequence of numbers that differ by at most 1. 

This solution efficiently handles the problem by leveraging a frequency map to minimize repeated calculations and ensures optimal performance with a time complexity of O(n), where n is the number of elements in the input array.
