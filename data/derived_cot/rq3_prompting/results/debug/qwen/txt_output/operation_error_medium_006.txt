To solve this problem, we need to identify words in `words1` that are supersets of every word in `words2`. A word is considered a superset of another if it contains every letter from the other word with at least the same frequency. 

### Approach
1. **Problem Analysis**: The key observation here is that for a word in `words1` to be a superset of every word in `words2`, it must include each letter that appears in any word of `words2` with a frequency at least as high as the maximum frequency of that letter in any word of `words2`.
2. **Intuition**: Instead of checking each word in `words1` against every word in `words2`, we can precompute the maximum frequency of each letter across all words in `words2`. Then, for each word in `words1`, we check if it meets or exceeds these maximum frequencies for all letters.
3. **Algorithm Selection**: 
   - **Step 1**: For each word in `words2`, compute its letter frequencies using `Counter` from the `collections` module.
   - **Step 2**: For each letter encountered, keep track of the maximum frequency it appears in any word of `words2`.
   - **Step 3**: For each word in `words1`, check if its letter frequencies meet or exceed the precomputed maximum frequencies from `words2`.
4. **Complexity Analysis**: The algorithm efficiently processes each word in `words2` once to build the frequency map and each word in `words1` once to check the superset condition. The time complexity is O(n*m), where n is the number of words in `words2` and m is the number of words in `words1`.

### Solution Code
```python
from collections import Counter

class Solution:
    def wordSubsets(self, words1: List[str], words2: List[str]) -> List[str]:
        w2_max = {}
        for c in map(Counter, words2):
            for char, count_val in c.items():
                w2_max[char] = max(w2_max.get(char, 0), count_val)
        return [w1 for w1 in words1 if Counter(w1) >= w2_max]
```

### Explanation
1. **Frequency Map Construction**: The code first processes each word in `words2` to build a dictionary (`w2_max`) that stores the highest frequency of each letter found in any word of `words2`. This is done by iterating over each word, converting it into a `Counter` (which counts letter frequencies), and updating the maximum frequency for each letter encountered.
2. **Superset Check**: For each word in `words1`, the code checks if its letter frequencies (obtained by converting the word to a `Counter`) meet or exceed the corresponding frequencies in `w2_max`. The condition `Counter(w1) >= w2_max` ensures that all letters in `w2_max` are present in `w1` with at least the required frequency.
3. **Efficiency**: By precomputing the maximum frequencies from `words2`, the solution avoids redundant checks for each word in `words1`, leading to an efficient and clear implementation. The use of `Counter` simplifies frequency counting and comparison.
