To solve this problem, we need to determine if all the frequencies of the numbers in the given array are unique. The solution involves counting the occurrences of each number in the array and then checking if these counts are all distinct.

### Approach
1. **Count Occurrences**: Use the `Counter` class from the `collections` module to count the occurrences of each number in the array.
2. **Check for Unique Counts**: Convert the counts into a list and then check if the length of this list is equal to the length of the set created from the list. If they are equal, it means all counts are unique; otherwise, there are duplicate counts.

### Solution Code
```python
from collections import Counter

class Solution:
    def uniqueOccurrences(self, arr: List[int]) -> bool:
        d = Counter(arr)
        l = list(d.values())
        if len(l) == len(set(l)):
            return True
        else:
            return False
```

1. **Counting Occurrences**: The `Counter` class is used to count how many times each number appears in the array. This gives us a dictionary where keys are the numbers and values are their respective counts.
2. **Checking Uniqueness**: The counts are converted into a list. We then compare the length of this list with the length of the set created from the list. If the lengths are the same, it means all elements in the list are unique, so we return `True`. Otherwise, we return `False`.
