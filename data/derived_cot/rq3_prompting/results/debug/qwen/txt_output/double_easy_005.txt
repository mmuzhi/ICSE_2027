### Solution Code
```python
class Solution:
    def calPoints(self, operations: List[str]) -> int:
        stack = []
        for i in operations:
            if i == 'D':
                stack.append(2 * stack[-1])
            elif i == 'C':
                stack.pop()
            elif i == '+':
                stack.append(stack[-1] + stack[-2])
            else:
                stack.append(int(i))
        return sum(stack)
```

- **Addition Operation**: The sum of the last two elements is computed and pushed onto the stack using `stack[-1] + stack[-2]`.
- **Result Calculation**: The sum of all elements in the stack is returned, which includes all valid operations processed correctly.
