To solve this problem, we need to find the median of two sorted arrays efficiently. The median is the middle value in an ordered list of numbers. If the combined length of the two arrays is even, the median is the average of the two middle values.

### Approach
1. **Merge the Arrays**: Combine the two sorted arrays into a single sorted array by comparing elements from both arrays and appending the smaller one to a new list. This step ensures that the resulting array is sorted.
2. **Calculate the Median**: Once the arrays are merged into a single sorted array, determine the median based on whether the length of the new array is even or odd. For an even length, the median is the average of the two middle elements. For an odd length, the median is the middle element.

### Solution Code
```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        new = []
        p1 = 0
        p2 = 0

            if nums1[p1] < nums2[p2]:

                p1 += 1
            else:
                new.append(nums2[p2])
                p2 += 1
        
        while p1 < len(nums1):
            new.append(nums1[p1])
            p1 += 1
            
        while p2 < len(nums2):
            new.append(nums2[p2])
            p2 += 1

        n = len(new)

            index = n // 2
            median = (new[index] + new[index-1]) / 2
        else:
            median = float(new[n//2])
        return median
```

### Explanation
1. **Merging Arrays**: The algorithm starts by initializing two pointers, `p1` and `p2`, to the beginning of each array. It then iterates through both arrays, comparing elements at the current pointers and appending the smaller element to the new list. The pointer for the array from which the element was taken is incremented. This process continues until one of the pointers reaches the end of its respective array.
2. **Appending Remaining Elements**: After one array is exhausted, the remaining elements from the other array are appended to the new list in their original order.
3. **Finding the Median**: The median is calculated based on the length of the new array. If the length is even, the median is the average of the elements at positions `n//2` and `n//2 - 1`. If the length is odd, the median is the element at position `n//2`.

This approach efficiently merges the two sorted arrays and calculates the median in linear time, O(n), where n is the total number of elements in both arrays. The space complexity is O(n) due to the storage required for the merged array.
