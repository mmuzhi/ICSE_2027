=== RAW ANSWER ===

To solve this problem, we need to move all zeros in an array to the end while maintaining the relative order of the non-zero elements. The solution uses a two-pointer technique where one pointer (slow) tracks the position where the next non-zero element should be placed, and another pointer (fast) traverses the array to find non-zero elements.

### Approach
1. **Initialization**: Start with the `slow` pointer at the beginning of the array (index 0).
2. **Traversal**: Use the `fast` pointer to traverse the array from start to end.
3. **Swapping Condition**: For each element encountered by the `fast` pointer, if it is non-zero and the `slow` pointer points to a zero, swap the elements at the `slow` and `fast` positions. This ensures that non-zero elements are moved forward while zeros are left behind.
4. **Increment Slow**: After processing each element with the `fast` pointer, if the element is non-zero, increment the `slow` pointer. This moves the `slow` pointer forward to the next position where a non-zero element should be placed.
5. **Termination**: The loop continues until all elements are processed, resulting in all zeros being moved to the end of the array while non-zero elements retain their original order.

### Solution Code
```python
class Solution:
    def moveZeroes(self, nums: list) -> None:
        slow = 0
        for fast in range(len(nums)):
            if nums[fast] != 0 and nums[slow] == 0:
                nums[slow], nums[fast] = nums[fast], nums[slow]
            if nums[fast] != 0:
                slow += 1
```

### Explanation
- **Initialization**: The `slow` pointer starts at 0, indicating the position where the next non-zero element should be placed.
- **Traversal**: The `fast` pointer iterates through each element in the array.
- **Swapping Condition**: If the `fast` pointer encounters a non-zero element and the `slow` pointer points to a zero, the elements at these positions are swapped. This moves the non-zero element to the `slow` position and the zero to the `fast` position.
- **Increment Slow**: If the `fast` pointer encounters a non-zero element, the `slow` pointer is incremented to move it forward, ensuring that the next non-zero element will be placed at the correct position.
- **Result**: After processing all elements, the array will have all zeros at the end, with non-zero elements in their original order.

This approach efficiently moves zeros to the end in a single pass through the array, making it optimal with a time complexity of O(n) and space complexity of O(1) as no additional space is used apart from the two pointers.

    def moveZeroes(self, nums: list) -> None:
