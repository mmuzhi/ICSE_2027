=== RAW ANSWER ===

To solve this problem, we need to determine if all the occurrences of elements in the given array are unique. This means that no two different elements should have the same frequency of occurrence.

### Approach
1. **Frequency Counting**: Use the `Counter` class from the `collections` module to count the occurrences of each element in the array.
2. **Check Uniqueness of Frequencies**: Convert the frequency counts into a set to eliminate duplicates. If the number of distinct frequencies matches the number of distinct elements in the array, then all frequencies are unique. Otherwise, there are duplicate frequencies.

### Solution Code
```python
from collections import Counter
class Solution:
    def uniqueOccurrences(self, arr: List[int]) -> bool:
        d = Counter(arr)
        if len(d) == len(set(d.values())):
            return True
        else:
            return False
```

### Explanation
1. **Frequency Counting**: The `Counter` class is used to count how many times each element appears in the array. This gives us a dictionary where keys are elements and values are their respective counts.
2. **Uniqueness Check**: By converting the values (frequencies) of the dictionary into a set, we can check if all frequencies are unique. The size of this set should be equal to the number of distinct elements in the array. If they are equal, it means all frequencies are unique, and we return `True`. Otherwise, we return `False`.

This approach efficiently checks for the uniqueness of frequencies by leveraging the properties of sets and dictionaries, ensuring optimal performance.
