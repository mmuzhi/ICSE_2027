To solve this problem, we need to find the median of two sorted arrays efficiently. The median is the middle value in an ordered list of numbers. If the combined length of the two arrays is even, the median is the average of the two middle numbers. 

### Approach
1. **Merge the Arrays**: We use two pointers to traverse both arrays simultaneously, merging them into a single sorted array. This is done by comparing elements from both arrays and appending the smaller element to the merged array.
2. **Handle Edge Cases**: If the merged array is empty (which happens when both input arrays are empty), we return 0.0 as the median.
3. **Calculate the Median**: 
   - For an even-length merged array, the median is the average of the two middle elements.
   - For an odd-length merged array, the median is the middle element.

### Solution Code
```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        p1 = 0
        p2 = 0
        new = []
        while p1 < len(nums1) and p2 < len(nums2):
            if nums1[p1] < nums2[p2]:
                new.append(nums1[p1])
                p1 += 1
            else:
                new.append(nums2[p2])
                p2 += 1
        while p1 < len(nums1):
            new.append(nums1[p1])
            p1 += 1
        while p2 < len(nums2):
            new.append(nums2[p2])
            p2 += 1

        if not new:
            return 0.0

        if len(new) % 2 == 0:
            index = len(new) // 2
            median = (new[index] + new[index-1]) / 2
        else:
            median = float(new[len(new)//2])
        return median
```
