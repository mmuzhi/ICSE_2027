To solve this problem, we need to construct a binary tree from given inorder and postorder traversals. The key insight is that the last element of the postorder traversal will always be the root of the current subtree. We then use the inorder traversal to determine the left and right subtrees of the root. The left subtree consists of elements before the root in the inorder traversal, and the right subtree consists of elements after the root.

### Approach
1. **Mapping Inorder Indices**: Create a dictionary to map each value in the inorder traversal to its index for quick lookup.
2. **Recursive Tree Construction**: Use a helper function that takes the current segment of the inorder and postorder traversals. The function:
   - **Base Case**: If the current segment is empty (left index exceeds right index), return `None`.
   - **Root Identification**: The root of the current subtree is the last element in the current postorder segment.
   - **Splitting Segments**: Using the root's position in the inorder traversal, split the inorder and postorder segments into left and right subtrees.
   - **Recursive Calls**: Recursively build the left and right subtrees using the corresponding segments of the inorder and postorder traversals.
3. **Initial Call**: Start the helper function with the entire inorder and postorder segments.

### Solution Code
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def buildTree(self, inorder: List[int], postorder: List[int]) -> Optional[TreeNode]:
        if not inorder or not postorder:
            return None
        
        inorder_map = {val: idx for idx, val in enumerate(inorder)}
        
        def treeHelper(left, right, left_post, right_post):
            if left > right or left_post > right_post:
                return None
            
            node_val = postorder[right_post]
            root = TreeNode(node_val)

            len_left = inorder_index - left
            root.left = treeHelper(left, inorder_index-1, left_post, left_post+len_left-1)
            
            len_right = right - inorder_index
            root.right = treeHelper(inorder_index+1, right, left_post+len_left, right_post-1)
            
            return root
        
        return treeHelper(0, len(inorder)-1, 0, len(postorder)-1)
```
