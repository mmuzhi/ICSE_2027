=== RAW ANSWER ===

To solve this problem, we need to find the maximum profit achievable by buying and selling stocks multiple times, given an array of stock prices. The key insight is that whenever the price increases from one day to the next, we can buy at the lower price and sell at the higher price, thereby capturing the profit. 

### Approach
The approach involves iterating through the array of stock prices and checking each consecutive day's price. If the price on the next day is higher than the current day, we add the difference to the total profit. This method efficiently captures all possible profits from consecutive price increases without the need for complex pointer manipulations or state tracking.

### Solution Code
```python
class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        profit = 0
        for i in range(len(prices)-1):
            if prices[i+1] > prices[i]:
                profit += prices[i+1] - prices[i]
        return profit
```

### Explanation
1. **Initialization**: We start with `profit` set to 0, which will accumulate the total profit.
2. **Iteration**: We loop through each day from 0 to the second last day. For each day, we compare its price with the next day's price.
3. **Profit Calculation**: If the next day's price is higher, we add the difference between the next day's price and the current day's price to the total profit. This difference represents the profit from buying on the current day and selling on the next day.
4. **Result**: After processing all consecutive days, the total profit is returned.

This approach ensures that we maximize the profit by taking advantage of every possible price increase, thus efficiently solving the problem in linear time with constant space complexity.
