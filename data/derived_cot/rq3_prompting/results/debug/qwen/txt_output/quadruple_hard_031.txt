   - **Forbidden Substring Check**: For each window, we check if any forbidden substring exists within it. If found, we adjust the left pointer to exclude the forbidden substring.

4. **Complexity Analysis**: 
   - **Time Complexity**: The algorithm involves iterating over each character in the word (O(n)) and, in the worst case, checking all forbidden substrings (O(m)) for each character.    - **Space Complexity**: The space complexity is O(1) as we only use a few additional variables for tracking the pointers and maximum length.

class Solution:
    def longestValidSubstring(self, word: str, forbidden: List[str]) -> int:
