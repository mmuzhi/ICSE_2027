=== RAW ANSWER ===

To solve this problem, we need to find all starting indices in a given string `s` that can form a substring which is a concatenation of each word in the list `words` exactly once. The words in `words` can be in any order, but each word must appear exactly once in the substring.

### Approach
1. **Problem Analysis**: The task is to identify all starting positions in `s` where a substring of length `total_len` (sum of lengths of all words in `words`) can be formed by concatenating each word in `words` exactly once. The solution involves checking each possible starting position in `s` and verifying if the substring starting at that position contains exactly the words from `words` in any order.

2. **Intuition**: The key insight is to use a frequency counter to keep track of the occurrences of each word in the current substring. For each starting index, we can slide a window of length `total_len` over the string `s` and update the frequency counter by removing the outgoing word and adding the incoming word as the window moves. However, for simplicity and clarity, we can also check each possible starting index by building a frequency counter from scratch for each window.

3. **Algorithm Selection**: We iterate over each possible starting index in `s` (from 0 to `n - total_len`). For each starting index, we check the substring of length `total_len` by splitting it into words of length `wlen` and comparing the frequency of these words with the frequency of words in `words`.

4. **Complexity Analysis**: The algorithm involves iterating over each possible starting index (O(n) iterations) and for each iteration, checking `m` words (O(m) operations per iteration). Thus, the overall time complexity is O(n * m), where `n` is the length of `s` and `m` is the number of words. The space complexity is O(m) due to the storage of the frequency counter.

### Solution Code
```python
import collections
from typing import List

class Solution:
    def findSubstring(self, s: str, words: List[str]) -> List[int]:
        if not s or not words:
            return []
        wlen = len(words[0])
        n = len(s)
        m = len(words)
        total_len = wlen * m
        if n < total_len:
            return []
        words_freq = collections.Counter(words)
        res = []
        
        for start_index in range(0, n - total_len + 1):
            freq = collections.Counter()
            for j in range(start_index, start_index + total_len, wlen):
                word = s[j:j+wlen]
                freq[word] += 1
            if freq == words_freq:
                res.append(start_index)
        return res
```

2. **Frequency Counter Setup**: The frequency of each word in `words` is stored in a `Counter` for quick comparison.

3. **Iterate Over Start Indices**: For each possible starting index in `s` (from 0 to `n - total_len`), the solution initializes a temporary frequency counter. It then checks the substring starting at this index by splitting it into words of length `wlen` and updates the temporary frequency counter accordingly.

4. **Comparison and Result Collection**: If the temporary frequency counter matches the frequency counter of `words`, the starting index is added to the result list.

5. **Return Result**: After processing all possible starting indices, the result list containing all valid starting indices is returned.
