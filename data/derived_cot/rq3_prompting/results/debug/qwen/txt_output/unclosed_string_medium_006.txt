To solve this problem, we need to determine the number of plates between the first and last occurrence of a candle (denoted by '|') in each query substring of a given string. The solution involves efficiently processing multiple queries by leveraging binary search to locate the relevant candles in the string.

### Approach
1. **Identify Candle Positions**: First, we collect all the indices in the string where candles occur. This helps in quickly accessing the positions of candles for any given query.
2. **Binary Search for Efficient Lookup**: We use a binary search function to find the first candle index that is greater than or equal to a given position (for the start of the query) and the last candle index that is less than or equal to a given position (for the end of the query).
3. **Process Queries**: For each query, we determine the first and last candles within the specified substring. If no such candles exist, the result is zero. Otherwise, the number of plates is calculated by subtracting the number of candles between the first and last candle from the distance between these two candles.

### Solution Code
```python
class Solution:
    def platesBetweenCandles(self, s: str, queries: List[List[int]]) -> List[int]:
        candies = [i for i, c in enumerate(s) if c == '|']
        
        def bns(x: int) -> int:
            l, r = 0, len(candies) - 1
            while l <= r:
                m = (l + r) // 2
                if candies[m] < x: 
                    l = m + 1
                else: 
                    r = m - 1
            return l

        ans = []
        for a, b in queries:
            l_index = bns(a)
            if l_index == len(candies):
                ans.append(0)
            else:
                r_index = bns(b + 1) - 1
                if r_index < 0 or candies[r_index] < a:
                    ans.append(0)
                else:
                    if l_index < r_index:
                        ans.append(candies[r_index] - candies[l_index] - (r_index - l_index))
                    else:
                        ans.append(0)
        return ans
```
