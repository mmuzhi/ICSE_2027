=== RAW ANSWER ===

To solve this problem, we need to find the longest harmonious subsequence in an array of integers. A harmonious subsequence is defined as a subsequence where the maximum difference between any two elements is exactly 1. This means the subsequence can include only two types of elements: a number `x` and the number `x+1`.

### Approach
1. **Frequency Map Construction**: First, we construct a frequency map (dictionary) where each key is a unique number from the input array, and the value is the count of occurrences of that number in the array.
2. **Iterate and Check Consecutive Numbers**: For each number in the frequency map, we check if the next consecutive number (current number + 1) exists in the map. If it does, the sum of their frequencies gives the length of a harmonious subsequence consisting of these two numbers.
3. **Track Maximum Length**: We keep track of the maximum length encountered during the iteration. If no such pair is found, the result remains 0, indicating no harmonious subsequence exists.

### Solution Code
```python
class Solution:
    def findLHS(self, nums: List[int]) -> int:
        count_map = {}
        for num in nums:
            count_map[num] = count_map.get(num, 0) + 1
        result = 0
        for num, count in count_map.items():
            if num + 1 in count_map:
                result = max(count + count_map[num + 1], result)
        return result
```

### Explanation
1. **Frequency Map Construction**: The loop iterates through each number in the input array, updating the frequency map to count occurrences of each number. For example, if the input is `[1, 2, 2, 3]`, the frequency map will be `{1: 1, 2: 2, 3: 1}`.
2. **Checking Consecutive Numbers**: For each number in the frequency map, the code checks if the next consecutive number exists. For instance, when processing the number `1`, it checks for `2` (which exists), and the sum of their frequencies (1 + 2) is 3, which becomes the initial result.
3. **Tracking Maximum Length**: The code continues to check all other numbers. When processing `2`, it checks for `3` (which exists), and the sum of their frequencies (2 + 1) is 3, which does not exceed the current result. The maximum length found is returned, which in this case is 3.

This approach efficiently checks all possible harmonious subsequences by leveraging the frequency map and ensures the solution is optimal with a time complexity of O(n), where n is the number of elements in the input array.
