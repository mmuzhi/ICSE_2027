To solve this problem, we need to find a valid arrangement of edges in a directed graph such that the arrangement forms an Eulerian trail. An Eulerian trail is a path that traverses each edge exactly once. The solution involves checking the graph's structure to determine if an Eulerian trail exists and then using a depth-first search (DFS) to construct the trail.

### Approach
1. **Graph Construction**: Build a directed graph from the given list of edges and compute the net out-degree for each node. The net out-degree is calculated as the number of outgoing edges minus the number of incoming edges.
2. **Identify Start Node**: The start node of the Eulerian trail is the node with a net out-degree of 1. If no such node exists, the graph might be an Eulerian circuit (all nodes have net degree 0), so any node can be chosen as the start.
3. **DFS Traversal**: Perform a DFS starting from the identified node. During traversal, remove each edge from the graph to ensure it is only traversed once. The DFS continues until all edges are traversed, and the nodes are recorded in reverse order.
4. **Construct Result**: Reverse the recorded nodes to get the correct order of the trail and then form the result by pairing consecutive nodes.

### Solution Code
```python
from collections import defaultdict
from typing import List

class Solution:
    def validArrangement(self, pairs: List[List[int]]) -> List[List[int]]:
        graph = defaultdict(list)
        degree = defaultdict(int)
        for x, y in pairs:
            graph[x].append(y)
            degree[x] += 1
            degree[y] -= 1
        
        x = None
        for k in degree:
            if degree[k] == 1:
                x = k
                break
        else:
            if graph:
                x = next(iter(graph.keys()))
            else:
                return []
        
        ans = []
        
        def fn(x):
            while graph[x]:
                y = graph[x].pop()
                fn(y)
            ans.append(x)
        
        fn(x)
        ans.reverse()
        return [[ans[i], ans[i+1]] for i in range(len(ans)-1)]
```

3. **DFS Traversal**: The DFS function (`fn`) traverses the graph starting from the identified node. It removes each edge from the graph as it is traversed to ensure no edge is used more than once. The nodes are recorded in reverse order during traversal.
4. **Result Construction**: The recorded nodes are reversed to form the correct order of the trail. The result is constructed by pairing consecutive nodes in the reversed list, forming the edges of the trail.
