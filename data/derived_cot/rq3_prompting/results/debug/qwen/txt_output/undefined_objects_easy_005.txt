=== RAW ANSWER ===

To solve this problem, we need to find the mode(s) (most frequent value(s)) in a binary tree. The approach involves traversing the tree to count the frequency of each node's value and then identifying the values with the highest frequency.

### Approach
1. **Tree Traversal**: We use a depth-first search (DFS) approach to traverse the tree. This involves visiting each node recursively, starting from the root, then moving to the left subtree, and finally the right subtree.
2. **Frequency Counting**: As we visit each node, we increment its value's count in a dictionary.
3. **Finding Modes**: After traversing the entire tree, we determine the maximum frequency from the dictionary values. Any node value(s) with this maximum frequency are considered modes.

### Solution Code
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right

class Solution:
    def findMode(self, root: Optional[TreeNode]) -> List[int]:
        if root is None:
            return []
        
        def dfs(freq):
            stack = [root]
            while stack:
                curr = stack.pop()
                freq.setdefault(curr.val, 0)
                freq[curr.val] += 1
                if curr.left:
                    stack.append(curr.left)
                if curr.right:
                    stack.append(curr.right)
        
        freq = {}
        dfs(freq)
        if not freq:
            return []
        max_cnt = max(freq.values())
        ans = [num for num, cnt in freq.items() if cnt == max_cnt]
        return ans
```

### Explanation
1. **Tree Traversal**: The DFS function starts from the root node and uses a stack to keep track of nodes to visit. Each node is processed by popping it from the stack, and its value is added to the frequency dictionary.
2. **Frequency Counting**: The dictionary `freq` keeps track of how many times each node's value appears in the tree. The `setdefault` method ensures that any new value is initialized to zero before incrementing.
3. **Finding Modes**: After processing all nodes, the maximum frequency value (`max_cnt`) is found using the `max` function. The modes are then collected by filtering the dictionary entries where the count matches `max_cnt`.

This approach efficiently counts the frequencies and identifies the modes in linear time relative to the number of nodes, making it suitable for large trees.
