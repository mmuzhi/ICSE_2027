=== RAW ANSWER ===

To solve this problem, we need to determine the maximum depth of a binary tree. The maximum depth is defined as the number of nodes along the longest path from the root node down to the farthest leaf node. 

### Approach
The approach involves using a Depth-First Search (DFS) traversal of the binary tree. We utilize a stack to keep track of nodes and their respective depths. The stack starts with the root node and an initial depth of 1. For each node popped from the stack, we update the maximum depth encountered so far. If the node has children, we push them onto the stack with an incremented depth. This ensures that we explore all paths from the root to the leaves, keeping track of the deepest level reached.

### Solution Code
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def maxDepth(self, root: Optional[TreeNode]) -> int:
        if not root:
            return 0
        stack = [[root, 1]]
        res = 0

        while stack:
            node, depth = stack.pop()
            if node:
                res = max(res, depth)
                stack.append([node.left, depth + 1])
                stack.append([node.right, depth + 1])
        return res
```

### Explanation
1. **Initialization**: The stack is initialized with the root node and depth 1. The variable `res` is initialized to 0 to keep track of the maximum depth encountered.
2. **DFS Traversal**: The loop continues as long as the stack is not empty. For each node popped from the stack, if the node is not `None`, the current depth is compared with `res` and updated if necessary.
3. **Child Processing**: The left and right children of the current node are pushed onto the stack with an incremented depth. This ensures that all nodes are processed in a depth-first manner.
4. **Termination**: The loop exits when all nodes have been processed, and the maximum depth (`res`) is returned.

This approach efficiently explores all paths from the root to the leaves, ensuring the maximum depth is accurately determined. The use of a stack allows for straightforward DFS traversal without recursion, making the solution both intuitive and efficient.
