=== RAW ANSWER ===

To solve this problem, we need to remove duplicates from a sorted array such that each element appears at most twice. The solution involves traversing the array with two pointers and maintaining a count of occurrences for each element to ensure no more than two duplicates are kept.

### Approach
1. **Problem Analysis**: The task is to modify the array in-place so that each element appears at most twice. The array is given in non-decreasing order, which allows us to use a two-pointer technique efficiently.
2. **Intuition**: By using two pointers, we can keep track of the current element and its count. The first pointer (`i`) marks the end of the allowed duplicates, and the second pointer (`j`) traverses the array to find new elements or additional allowed duplicates.
3. **Algorithm Selection**: We use a single loop to traverse the array starting from the second element. For each element, we check if it matches the element at pointer `i`. If it does, we increment the count. If the count exceeds 2, we skip further duplicates of that element. Otherwise, we move the pointer `i` and update the count.
4. **Complexity Analysis**: The algorithm runs in O(n) time, where n is the length of the array, as each element is processed exactly once. The space complexity is O(1) since we only use a few extra variables for tracking.

### Solution Code
```python
class Solution:
    def removeDuplicates(self, nums: List[int]) -> int:
        n = len(nums)
        if n <= 2:
            return n
        i = 1
        count = 1
        for j in range(1, n):
            if nums[j] == nums[i-1] and count < 2:
                nums[i] = nums[j]
                i += 1
                count += 1
            elif nums[j] == nums[i-1] and count == 2:
                continue
            else:
                nums[i] = nums[j]
                i += 1
                count = 1
        return i
```

### Explanation
1. **Initialization**: The function starts by checking if the array length is 2 or less, in which case no duplicates need removal, and the array length is returned as-is.
2. **Two Pointers**: The pointer `i` starts at 1 (indicating the first element is fixed), and `count` starts at 1 to track the occurrences of the current element.
3. **Traversal**: The loop from index 1 to the end of the array processes each element:
   - If the current element matches the element at `i-1` and the count is less than 2, it moves the element to position `i`, increments `i` and the count.
   - If the count reaches 2, further duplicates are skipped.
   - For new elements (different from `i-1`), it resets the count and moves the element to the next position.
4. **Result**: The value of `i` gives the new length of the array after removing extra duplicates, which is returned as the result.

This approach efficiently ensures that each element appears at most twice while maintaining the sorted order of the array.
