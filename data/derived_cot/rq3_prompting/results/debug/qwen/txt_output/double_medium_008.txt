To solve this problem, we need to determine the number of booked seats for each flight from 1 to n based on a list of bookings. Each booking is a list [from, to, seats], indicating that 'seats' number of seats are booked for a flight from 'from' to 'to'. The solution involves efficiently calculating the total seats for each flight using a difference array technique.

### Approach
1. **Problem Analysis**: The problem requires us to compute the total booked seats for each flight from 1 to n. Each booking affects a range of flights from 'from' to 'to' (inclusive). A naive approach would involve iterating over each booking and updating the count for each flight in the range, which would be inefficient for large n.

2. **Intuition**: The difference array technique allows us to efficiently handle range updates. For each booking [from, to, seats], we add 'seats' to the start of the range (flight 'from') and subtract 'seats' from the end+1 of the range (flight 'to'+1). This way, when we compute the prefix sum of the difference array, we get the total seats for each flight.

3. **Algorithm Selection**: We use a difference array (an array of zeros) of length n+1. For each booking, we update the difference array at the start and end+1 positions. Then, we compute the prefix sum of this array to get the total seats for each flight.

4. **Complexity Analysis**: The algorithm processes each booking in O(1) time for the updates and O(n) time for the prefix sum calculation. Thus, the overall time complexity is O(n + m), where m is the number of bookings, which is efficient.

### Solution Code
```python
from typing import List
from itertools import accumulate

class Solution:
    def corpFlightBookings(self, bookings: List[List[int]], n: int) -> List[int]:
        arr = [0] * (n + 1)
        for lv, ar, seats in bookings:
            arr[lv] += seats
            if ar + 1 <= n:
                arr[ar + 1] -= seats
        return list(accumulate(arr))[1:]
```

### Explanation
1. **Initialization**: We initialize an array `arr` of size n+1 with zeros. This array will be used to store the difference values.

2. **Processing Bookings**: For each booking [from, to, seats]:
   - We add 'seats' to the 'from' index (indicating the start of the range).
   - We subtract 'seats' from the 'to+1' index (if it is within bounds), indicating the end of the range.

3. **Prefix Sum Calculation**: Using the `accumulate` function from the `itertools` module, we compute the prefix sum of the difference array. The prefix sum gives the total seats booked up to each flight. We then return the prefix sum from index 1 to the end (since the first element is zero and not part of the flight counts).

This approach efficiently computes the total seats for each flight by leveraging the difference array technique, ensuring optimal performance even for large inputs.
