### Approach
1. **Tree Construction**: First, we construct an adjacency list representation of the tree from the given edges.
2. **DFS Traversal**: We perform a post-order DFS traversal starting from the root node (node 0). During the traversal, for each node, we sum the values of the node and its children (excluding the parent to avoid cycles).
3. **Component Check**: During the traversal, if the sum of a subtree equals the candidate sum (`cand`), we return 0 to indicate that the subtree can be split off as a separate component. Otherwise, we return the sum of the subtree.
4. **Finding Maximum Components**: We iterate over all possible candidate sums (`cand`) from 1 to half of the total sum of all node values. If both conditions are met, we return the number of components (total sum divided by candidate). If no valid candidate is found, we return 0.

class Solution:
    def componentValue(self, nums: List[int], edges: List[List[int]]) -> int:
