To solve this problem, we need to compute the median of every sliding window in a given array of integers. The sliding window moves one element at a time, and the window size is specified by the integer `k`. The median should be calculated efficiently for each window and stored in a result list.

### Approach
The approach involves using a balanced binary search tree (BST) to maintain the elements of the current window. This allows efficient insertion, deletion, and median retrieval operations. The BST operations are implemented using a Node class and helper functions (`insert`, `remove`, `size`, and `get`). 

1. **BST Operations**: The `insert` function adds a new element to the BST while maintaining balance. The `remove` function deletes a specified element from the BST. The `size` function returns the number of nodes in the BST. The `get` function retrieves the element at a specific order in the BST (e.g., the k-th smallest element).

2. **Sliding Window Handling**: For each element in the array, we insert it into the BST. If the size of the BST exceeds `k`, we remove the leftmost element of the previous window. After each insertion and potential removal, if the BST size is exactly `k`, we compute the median. For odd `k`, the median is the middle element. For even `k`, the median is the average of the two middle elements.

```python
class Solution:
    def medianSlidingWindow(self, nums: List[int], k: int) -> List[float]:
        tree = None
        ans = []
        for i, x in enumerate(nums):
            tree = insert(tree, x)
            if size(tree) > k:
                tree = remove(tree, nums[i - k + 1])
            if size(tree) == k:
                if k % 2 == 1:
                    ans.append(get(tree, k // 2 + 1))
                else:
                    ans.append((get(tree, k // 2) + get(tree, k // 2 + 1)) / 2)
        return ans

class Node:
    __slots__ = ('val', 'left', 'right', 'size')
    def __init__(self, val):
        self.val = val
        self.left = None
        self.right = None
        self.size = 1

def size(node):
    if node is None:
        return 0
    return node.size

def insert(node, val):
    if node is None:
        return Node(val)
    if val < node.val:
        node.left = insert(node.left, val)
    else:
        node.right = insert(node.right, val)
    node.size = size(node.left) + size(node.right) + 1
    return node

def remove(node, val):
    if node is None:
        return None
    if val < node.val:
        node.left = remove(node.left, val)
    elif val > node.val:
        node.right = remove(node.right, val)
    else:
        if node.left is None:
            return node.right
        elif node.right is None:
            return node.left
        else:
            temp = min_node(node.right)
            node.val = temp.val
            node.right = remove(node.right, temp.val)
    node.size = size(node.left) + size(node.right) + 1
    return node

def min_node(node):
    while node.left is not None:
        node = node.left
    return node

def get(node, rank):
    while node:
        left_size = size(node.left)
        if left_size + 1 == rank:
            return node.val
        elif left_size >= rank:
            node = node.left
        else:
            node = node.right
            rank -= left_size + 1
    return None
```
