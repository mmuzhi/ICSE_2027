To solve this problem, we need to perform a level-order traversal of a binary tree from bottom to top. The level-order traversal involves visiting each node level by level, starting from the root (top) and moving down to the deepest level. The challenge is to reverse this traversal so that the deepest level appears first.

### Approach
1. **Base Case Handling**: If the tree is empty (root is None), return an empty list.
2. **Breadth-First Search (BFS)**: Use a queue to traverse the tree level by level. For each level, collect the values of all nodes in that level. This is done by processing each node in the queue, recording its value, and adding its children to the queue.
3. **Reverse the Levels**: After collecting all levels from top to bottom, reverse the list of levels to present them from bottom to top. This is achieved by swapping elements from the start and end of the list until the middle is reached.

3. **Level Reversal**: After collecting all levels from top to bottom, the `checkReverse` function reverses the list of levels.
