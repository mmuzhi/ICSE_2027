The encoded text is the concatenation of the columns in the matrix. Our task is to reconstruct the original message by reading the matrix in row-major order.

### Approach
1. **Problem Analysis**: The encoded text is formed by reading the matrix column by column. Given the number of rows, we can determine the number of columns by dividing the length of the encoded text by the number of rows. The original message is reconstructed by reading the matrix row by row.
2. **Intuition**: The solution involves traversing the encoded text in column-major order to reconstruct the original matrix. Once the matrix is reconstructed, we read it in row-major order to get the original message.
3. **Algorithm Selection**: We use nested loops to traverse the encoded text. The outer loop iterates over each column, and the inner loop iterates over each row within the column. The elements are collected in the order they appear in the encoded text, which corresponds to column-major order. The collected elements are then joined to form the original message.
4. **Complexity Analysis**: The algorithm processes each character in the encoded text exactly once, resulting in a time complexity of O(N), where N is the length of the encoded text. The space complexity is also O(N) to store the original message.

### Solution Code
```python
class Solution:
    def decodeCiphertext(self, encoded_text: str, rows: int) -> str:
        if rows == 1:
            return encoded_text
        
        N = len(encoded_text)
        cols = N // rows
        original_text = []
        
        for j in range(cols):
            for i in range(rows):
                original_text.append(encoded_text[j * rows + i])
                
        return ''.join(original_text)
```

2. **Determine Columns**: Calculate the number of columns by dividing the length of the encoded text by the number of rows.
3. **Reconstruct Original Text**: Use nested loops to traverse the encoded text. The outer loop iterates over each column, and the inner loop iterates over each row within the column. The elements are collected in the order they appear in the encoded text, which corresponds to column-major order.
4. **Return Result**: Join the collected elements into a string to form the original message and return it.

This approach efficiently reconstructs the original message by leveraging the column-major order of the encoded text, ensuring correctness and optimal performance.

        original_text = []
        
        for j in range(cols):
            for i in range(rows):
                original_text.append(encoded_text[j * rows + i])
                
        return ''.join(original_text)
