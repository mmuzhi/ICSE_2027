    def cdist(self, it, distArray, distArrayIndex, edges):
        rdist = 0 
        nodes = []
        # We start with the given node 'it'
        while it != -1 and distArray[it][distArrayIndex] > rdist:
            distArray[it][distArrayIndex] = rdist
            nodes.append(it)
            it = edges[it]
            rdist += 1
        return nodes
            
    def closestMeetingNode(self, edges: List[int], node1: int, node2: int) -> int:
        INF = float('inf')

To solve this problem, we need to find the closest meeting node in an undirected graph where two nodes (node1 and node2) start traversing the graph level by level. The goal is to determine the node where the maximum of the distances from node1 and node2 is minimized. If multiple such nodes exist, we return the smallest one. If no such node exists, we return -1.

### Approach
1. **Breadth-First Search (BFS)**: We perform BFS from both node1 and node2 to compute the distances from each node to these starting points. For each node, we store two distances: one from node1 and another from node2.
2. **Combining Results**: After performing BFS from both nodes, we combine the lists of nodes visited by each BFS. This combined list contains all nodes reachable from either node1 or node2.
3. **Finding the Closest Node**: For each node in the combined list, we compute the maximum of its distances from node1 and node2. The node with the smallest such maximum distance is our answer. If multiple nodes have the same minimum maximum distance, we select the smallest node index.

### Solution Code
```python
from typing import List
from collections import defaultdict

class Solution:
    def cdist(self, it, distArray, distArrayIndex, edges):
        rdist = 0 
        nodes = []
        while it != -1 and distArray[it][distArrayIndex] > rdist:
            distArray[it][distArrayIndex] = rdist
            nodes.append(it)
            it = edges[it]
            rdist += 1
        return nodes
            
    def closestMeetingNode(self, edges: List[int], node1: int, node2: int) -> int:
        INF = float('inf')
        dist = defaultdict(lambda: [INF, INF])
        mmin, ans = INF, INF
        
        nodes1 = self.cdist(node1, dist, 0, edges)
        nodes2 = self.cdist(node2, dist, 1, edges)
        n = nodes1 + nodes2
        
        for k in n:
            d1, d2 = dist[k]
            m = max(d1, d2)
            if m != INF:
                if m < mmin:
                    mmin = m
                    ans = k
                elif m == mmin:
                    ans = min(ans, k)
                    
        return ans if ans != INF else -1
```
