The solution involves efficiently matching the characters of `s2` in the concatenated string without explicitly constructing it, leveraging cycle detection to optimize performance.

### Approach
1. **Initial Checks**: Verify if all characters of `s2` are present in `s1`. If not, return 0 as no `s2` can be formed.
2. **Position Tracking**: For each character in `s2`, precompute the positions where it appears in `s1`. This helps in quickly locating the next occurrence during the matching process.
3. **Cycle Detection**: While matching characters of `s2` in the repeated `s1`, track the positions and counts to detect cycles. This allows us to compute the number of repetitions without simulating each character every time.
4. **Efficient Matching**: Use binary search (via `bisect`) to find the next occurrence of each character in `s1` efficiently, considering the repeated structure of `s1`.
5. **Count Calculation**: After detecting a cycle, use the cycle's properties to compute the total number of `s2` repetitions, adjusting for any remaining characters outside the cycle.

import bisect
from collections import defaultdict

class Solution:
    def getMaxRepetitions(self, s1: str, n1: int, s2: str, n2: int) -> int:
        total_length = n1 * len(s1)
        if not set(s2).issubset(set(s1)):
            return 0
        
        pos_in_s1 = {}
        for char in set(s2):
            pos_in_s1[char] = [i for i, c in enumerate(s1) if c == char]
        
        rec = [0]
        track = defaultdict(int)
        ptr = 0
        start = 0
        ct = 0
        
        while True:
            for char in s2:
                found = False
                while start < total_length:
                    copy_index = start // len(s1)
                    pos_in_current = start % len(s1)
                    arr = pos_in_s1[char]
                    if not arr:
                        break

                    if idx < len(arr):
                        start = copy_index * len(s1) + arr[idx] + 1
                        found = True
                        break
                if not found:
                    ct += 1
                    start = 0
                    if start >= total_length:
                        break
            rec.append(ct + len(rec) - 1)
            
            if rec[-1] > total_length:
                return (len(rec) - 2) // n2
            
            if start not in track:
                track[start] = len(rec) - 1
            else:
                cycle_start = rec[track[start]]
                cycle_length = rec[-1] - cycle_start
                total_reps = (total_length - cycle_start) // cycle_length
