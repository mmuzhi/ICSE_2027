To solve this problem, we need to sort a given string by first outputting all its distinct characters in ascending order, then in descending order, and repeating this process until all characters are exhausted. The key insight is to handle the frequency of each character and efficiently manage the output process to avoid any premature deletion of characters from the frequency dictionary.

### Approach
1. **Frequency Dictionary**: We start by building a frequency dictionary where each key is a distinct character from the input string, and the value is the count of that character.
2. **Sorted Unique Characters**: We create a list of the unique characters in the string, sorted in ascending order.
3. **Output Process**: Using a while loop, we repeatedly perform two passes over the sorted unique characters:
   - **Ascending Pass**: Output each character in ascending order if its frequency is greater than zero, decrementing its count each time it's used.
   - **Descending Pass**: Output each character in descending order (reverse of the sorted list) if its frequency is still greater than zero after the ascending pass.
4. **Termination Condition**: The loop continues until all characters' frequencies are exhausted (i.e., the dictionary becomes empty).

### Solution Code
```python
class Solution:
    def sortString(self, s: str) -> str:
        freq = {}
        letters = sorted(set(s))
        res = ""
        for i in s:
            if i in freq:
                freq[i] += 1
            else:
                freq[i] = 1
        while freq:
            for i in letters:
                if i in freq and freq[i] > 0:
                    res += i
                    freq[i] -= 1
                    if freq[i] == 0:
                        del freq[i]
            for i in reversed(letters):
                if i in freq and freq[i] > 0:
                    res += i
                    freq[i] -= 1
                    if freq[i] == 0:
                        del freq[i]
        return res
```

### Explanation
1. **Frequency Dictionary Construction**: The code initializes a dictionary `freq` to count occurrences of each character in the input string `s`.
2. **Unique and Sorted Characters**: The list `letters` is created by first converting `s` to a set (to get unique characters), sorting them, and then using them for the output process.
3. **Loop Until Exhausted**: The while loop runs as long as there are characters left in the frequency dictionary.
4. **Ascending Pass**: In the first for-loop, characters are processed in ascending order. Each character is added to the result string, its frequency is decremented, and if the frequency drops to zero, the character is removed from the dictionary.
5. **Descending Pass**: In the second for-loop, characters are processed in descending order (reverse of `letters`). Similar to the ascending pass, each character is added to the result string, its frequency is decremented, and it is removed from the dictionary if its frequency becomes zero.
6. **Result Construction**: The result string is built by concatenating characters during each pass, ensuring the output alternates between ascending and descending order of distinct characters until all characters are used up.

This approach efficiently handles the problem by leveraging a frequency dictionary and two passes per iteration, ensuring optimal performance and correctness.

class Solution:
    def sortString(self, s: str) -> str:
        freq = {}
        letters = sorted(set(s))
        res = ""
        for i in s:
            if i in freq:
                freq[i] += 1
            else:
                freq[i] = 1
        while freq:
            for i in letters:
                if i in freq and freq[i] > 0:
                    res += i
                    freq[i] -= 1
                    if freq[i] == 0:
                        del freq[i]
            for i in reversed(letters):
                if i in freq and freq[i] > 0:
                    res += i
                    freq[i] -= 1
                    if freq[i] == 0:
                        del freq[i]
        return res
