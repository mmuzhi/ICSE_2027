Each query asks for the maximum value of `nums1[i] + nums2[i]` where `nums1[i] >= x` and `nums2[i] >= y` for given query parameters `x` and `y`.

### Approach
1. **Sorting and Preprocessing**: 
   - First, we sort the points from `nums1` and `nums2` based on their values. This allows us to efficiently process queries using binary search.
   - We then compute a suffix maximum array `max_y` where each element `max_y[i]` represents the maximum `nums2` value from index `i` to the end of the sorted array. This helps in quickly determining if there exists a point in a segment that meets the `nums2 >= y` condition.

2. **Sparse Table for Range Maximum Queries**:
   - We build a sparse table on the array `_sum`, which contains the sums of corresponding elements from `nums1` and `nums2`. This sparse table allows us to answer range maximum queries in logarithmic time, which is crucial for handling multiple queries efficiently.

3. **Query Processing**:
   - For each query, we first locate the starting index in the sorted array where `nums1` values are at least `x` using binary search.
   - Using the suffix maximum array, we then determine the segment where `nums2` values are at least `y`.
   - Finally, we use the sparse table to find the maximum sum within the identified segment, providing the answer for the query.

import math
from typing import List
import bisect

class Solution:
    def maximumSumQueries(self, nums1: List[int], nums2: List[int], queries: List[List[int]]) -> List[int]:
        def buildSparseTable(arr, n):
            k = math.floor(math.log2(n)) + 1 if n > 0 else 0
            if n == 0:
                return None
            st = [[0] * n for _ in range(k)]
            for i in range(n):
                st[0][i] = arr[i]
            j = 1
            while (1 << j) <= n:
                i = 0
                while i + (1 << j) - 1 < n:
                    st[j][i] = max(st[j-1][i], st[j-1][i + (1 << (j-1))])
                    i += 1
                j += 1
            return st
        
        def getMax(st, n, l, r):
            if l > r:
                return 0
            length = r - l + 1
            k = math.floor(math.log2(length))
            return max(st[k][l], st[k][r - (1 << k) + 1])
        
        n = len(nums1)
        a = sorted([[nums1[i], nums2[i]] for i in range(n)])
        max_y = [0] * n
        for i in range(n):
            max_y[i] = a[i][1]
        for i in range(n-2, -1, -1):
            max_y[i] = max(max_y[i], max_y[i+1])
        _sum = [v[0] + v[1] for v in a]
        st_max_sum = buildSparseTable(_sum, n) if n > 0 else None
        res = []
        for x, y in queries:
            i = bisect.bisect_left(a, [x, -10**9])
            if i == n or (n > 0 and max_y[i] < y):
                res.append(-1)
                continue
            left, right = i, n-1
            while left < right:
                mid = (left + right + 1) // 2
                if max_y[mid] >= y:
                    left = mid
                else:
                    right = mid - 1
            j = left
            if n > 0:
                ans = getMax(st_max_sum, n, i, j)
                res.append(ans)
            else:
                res.append(-1)
        return res
