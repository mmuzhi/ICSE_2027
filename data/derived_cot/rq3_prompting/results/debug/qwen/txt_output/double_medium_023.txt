=== RAW ANSWER ===

To solve this problem, we need to construct a quad-tree from a given grid. The quad-tree is a tree data structure where each node can have up to four children, representing the four quadrants of the node's area. The goal is to build this tree recursively, ensuring that leaf nodes (nodes with no children) contain uniform values within their area.

### Approach
1. **Base Case Handling**: If the grid size is 1x1, the node is a leaf node containing the single value from the grid.
2. **Recursive Splitting**: For larger grids, split the current node into four children (top-left, top-right, bottom-left, bottom-right) recursively. Each child covers a quadrant of the parent's area.
3. **Leaf Node Check**: After building the four children, check if all children are leaves and have the same value. If so, the current node can be a leaf with that value. Otherwise, the current node becomes an internal node with the four children.

### Solution Code
```python
class Solution:
    def construct(self, grid: List[List[int]]) -> 'Node':
        level = len(grid)
        if level == 0:
            return None
        root = None
        if level >= 1:
            root = self.buildTree(grid, 0, level, 0, level, level)
        return root
        
    def buildTree(self, grid, rs, re, cs, ce, level):
        if level == 1:
            return Node(grid[rs][cs], True, None, None, None, None)
        
        half = level // 2
        tl = self.buildTree(grid, rs, rs+half, cs, cs+half, half)
        tr = self.buildTree(grid, rs, rs+half, cs+half, cs+level, half)
        bl = self.buildTree(grid, rs+half, re, cs, cs+half, half)
        br = self.buildTree(grid, rs+half, re, cs+half, cs+level, half)
        
        if tl.isLeaf and tr.isLeaf and bl.isLeaf and br.isLeaf:
            if tl.val == tr.val == bl.val == br.val:
                new_node = Node(tl.val, True, None, None, None, None)
            else:
                new_node = Node(tl.val, False, tl, tr, bl, br)
        else:
            new_node = Node(tl.val, False, tl, tr, bl, br)
            
        return new_node
```

1. **Base Case Handling**: The `buildTree` function first checks if the current grid size is 1x1. If so, it returns a leaf node with the single value from the grid.
2. **Recursive Splitting**: For larger grids, the function splits the grid into four quadrants. Each quadrant is processed recursively to build the corresponding child node.
3. **Leaf Node Check**: After building the four children, the function checks if all children are leaves and have the same value. If true, the current node is a leaf with that value. If not, the current node is an internal node with the four children.
4. **Edge Cases**: The code handles grids of any size by recursively splitting until the base case is reached. The indices for each quadrant are calculated using integer division to ensure proper splitting.

This approach efficiently constructs the quad-tree by leveraging recursion and ensures the tree structure adheres to the quad-tree properties, with leaf nodes representing uniform values within their area.
