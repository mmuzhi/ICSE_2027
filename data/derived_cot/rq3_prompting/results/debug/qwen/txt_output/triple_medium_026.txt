To solve this problem, we need to determine the minimum number of insertions required to transform string `s` into an anagram of string `t` by only inserting characters. The solution involves comparing the character frequencies in both strings and calculating the difference in counts for each character present in `t`.

### Approach
1. **Frequency Counting**: Use the `Counter` class from the `collections` module to count the occurrences of each character in both strings `s` and `t`.
2. **Difference Calculation**: For each character in `t`, compute the difference between its count in `t` and its count in `s`. If the character is not present in `s`, the difference is simply the count in `t`.
3. **Summing Differences**: Sum all the positive differences obtained in the previous step. This sum represents the minimum number of insertions needed to make `s` an anagram of `t`.

### Solution Code
```python
from collections import Counter

class Solution:
    def minSteps(self, s: str, t: str) -> int:
        cnt_s = Counter(s)
        cnt_t = Counter(t)
        ans = 0
        for char in cnt_t:
            if char in cnt_s:
                ans += max(0, cnt_t[char] - cnt_s[char])
            else:
                ans += cnt_t[char]
        return ans
```

### Explanation
1. **Frequency Counting**: The `Counter` class is used to create frequency dictionaries for both strings `s` and `t`. This allows us to easily access the count of each character in both strings.
2. **Difference Calculation**: For each character in `t`, we check if it exists in `s`. If it does, we calculate the difference between the count in `t` and the count in `s` (if positive). If the character does not exist in `s`, we simply add the count from `t` to the answer.
3. **Summing Differences**: The total number of insertions required is the sum of all these differences, which is returned as the result.

This approach efficiently computes the minimal insertions by leveraging character frequency counts and ensures that the solution is both optimal and easy to understand.
