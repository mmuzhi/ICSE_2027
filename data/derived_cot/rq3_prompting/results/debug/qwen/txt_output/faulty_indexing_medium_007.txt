### Solution Code
```python
class Solution:
    def pivotArray(self, nums: List[int], pivot: int) -> List[int]:
        lst = []      # for numbers less than pivot
        val = []      # for numbers equal to pivot
        tmp = []      # for numbers greater than pivot
        for i in nums:
            if i == pivot:
                val.append(i)
            elif i < pivot:
                lst.append(i)
            else:
                tmp.append(i)
        return lst + val + tmp
```

### Explanation
- **Initialization**: Three empty lists (`lst`, `val`, `tmp`) are initialized to store elements less than, equal to, and greater than the pivot, respectively.
- **Loop Through Elements**: For each element in the input array, the code checks its value relative to the pivot:
  - If the element equals the pivot, it is added to `val`.
  - If the element is less than the pivot, it is added to `lst`.
  - If the element is greater than the pivot, it is added to `tmp`.
- **Concatenate Lists**: The final result is obtained by concatenating `lst`, `val`, and `tmp` in that order, ensuring the pivot array is correctly partitioned.

This approach efficiently partitions the array in linear time, making it optimal for the given problem. The solution avoids any index-related errors by directly comparing elements with the pivot value.
