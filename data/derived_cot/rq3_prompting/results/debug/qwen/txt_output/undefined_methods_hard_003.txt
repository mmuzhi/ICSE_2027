To solve this problem, we need to find the longest substring within a given word that does not contain any forbidden substrings. The approach involves using a trie data structure to efficiently check for forbidden substrings and a two-pointer technique to determine the longest valid substring.

### Approach
1. **Trie Construction**: Build a trie from all forbidden substrings. This allows us to efficiently check if any substring in the given word matches a forbidden pattern.
2. **Forbidden Substring Check**: For each position in the word, check if any substring starting from that position contains a forbidden substring by traversing the trie. This involves scanning all possible starting positions within the current substring to ensure no forbidden substring is present.
3. **Two-Pointer Technique**: Use two pointers, `i` and `j`, to represent the current substring being considered. The pointer `j` starts at the end of the word and moves leftwards. For each `i`, the substring from `i` to `j-1` is checked for forbidden substrings. If a forbidden substring is found, `j` is adjusted to exclude it. The maximum valid substring length encountered during this process is recorded.

### Solution Code
```python
class Solution:
    def longestValidSubstring(self, word: str, forbidden: List[str]) -> int:
        trie = {}
        for f in forbidden:
            t = trie
            for c in f:
                if c not in t:
                    t[c] = {}
                t = t[c]
            t["end"] = True
        
        def isForbidden(s):
            n = len(s)
            if n == 0:
                return False
            for start in range(n):
                temp = trie
                for pos in range(start, n):
                    c = s[pos]
                    if c not in temp:
                        break
                    temp = temp[c]
                    if "end" in temp:
                        return True
            return False
        
        res = 0
        j = len(word)
        for i in range(len(word) - 1, -1, -1):
            if isForbidden(word[i:j]):
                j = i + (isForbidden(word[i:j]) - 1)
            else:
                res = max(res, j - i)
        return res
```
