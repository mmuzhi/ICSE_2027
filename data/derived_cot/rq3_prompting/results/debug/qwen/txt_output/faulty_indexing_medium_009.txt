To solve this problem, we need to compute the sum of the minimum values of all contiguous subarrays in a given array. The key insight is to efficiently determine, for each element in the array, how many subarrays it is the minimum of, and then multiply this count by the element's value and accumulate the results.

### Approach
1. **Problem Analysis**: The problem requires summing the minimum values of all possible contiguous subarrays. A brute-force approach would involve checking every subarray, which is inefficient for large arrays. Instead, we use a more efficient method involving the concept of "next smaller elements" to determine the range where each element is the minimum.

2. **Intuition**: For each element in the array, the number of subarrays where it is the minimum can be determined by finding the nearest smaller elements to its left and right. Specifically:
   - The left boundary is the index of the previous smaller element.
   - The right boundary is the index of the next smaller element.
   The number of subarrays where the current element is the minimum is then calculated as `(i - left_bound[i]) * (right_bound[i] - i)`, where `i` is the current index.

3. **Algorithm Selection**: We use a monotonic stack to efficiently compute the left and right boundaries for each element in linear time. This involves two passes over the array:
   - **Left Pass**: Traverse the array from left to right to compute the previous smaller element for each index.
   - **Right Pass**: Traverse the array from right to left to compute the next smaller element for each index.

4. **Complexity Analysis**: Both passes over the array are O(n), where n is the number of elements in the array. The space complexity is O(n) due to the storage of the left and right boundaries. This approach is efficient and handles the constraints effectively.

### Solution Code
```python
class Solution:
    def sumSubarrayMins(self, arr: List[int]) -> int:
        n = len(arr)
        mod = 10**9 + 7
        left_bound = [-1] * n
        right_bound = [n] * n
        
        stack = []
        for i in range(n):
            while stack and arr[stack[-1]] > arr[i]:
                stack.pop()
            left_bound[i] = stack[-1] if stack else -1
            stack.append(i)
        
        stack = []
        for i in range(n-1, -1, -1):
            while stack and arr[stack[-1]] > arr[i]:
                stack.pop()
            right_bound[i] = stack[-1] if stack else n
            stack.append(i)
        
        res = 0
        for i in range(n):
            res = (res + arr[i] * (i - left_bound[i]) * (right_bound[i] - i)) % mod
        
        return res
```

### Explanation
1. **Initialization**: We initialize two arrays, `left_bound` and `right_bound`, to store the indices of the previous and next smaller elements for each position in the array. These arrays are initialized with default values (-1 for `left_bound` and `n` for `right_bound`).

2. **Left Pass**: Using a stack, we traverse the array from left to right. For each element, we pop elements from the stack that are greater than the current element until the stack is empty or the top element is smaller. The index of the top element becomes the `left_bound` for the current element.

3. **Right Pass**: Similarly, we traverse the array from right to left. For each element, we pop elements from the stack that are greater than the current element until the stack is empty or the top element is smaller. The index of the top element becomes the `right_bound` for the current element.

4. **Result Calculation**: For each element, the number of subarrays where it is the minimum is calculated using its `left_bound` and `right_bound`. The result is accumulated by multiplying the element's value by this count, taking modulo \(10^9 + 7\) to handle large numbers.

This approach efficiently computes the desired sum by leveraging stack operations to determine the boundaries of subarrays where each element is the minimum, ensuring optimal performance.
