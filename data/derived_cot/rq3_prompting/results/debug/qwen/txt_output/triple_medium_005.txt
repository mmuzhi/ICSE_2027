To solve this problem, we need to compute the sum of the minimum values of all contiguous subarrays in a given array. The challenge lies in efficiently calculating this sum without resorting to a brute-force approach, which would be computationally expensive for large arrays.

### Approach
The approach involves using a monotonic increasing stack to efficiently find the next smaller elements to the left and right for each element in the array. This allows us to determine the number of subarrays where each element is the minimum. The key steps are:

1. **Padding the Array**: We start by padding the original array with `float('-inf')` at both ends. This helps in handling edge cases where elements are the smallest in their subarrays.
2. **Monotonic Stack**: We use a stack to keep track of indices in a non-decreasing order. For each element, we pop elements from the stack that are greater than the current element. This helps in identifying the nearest smaller elements to the left.
3. **Calculating Contributions**: For each popped element, we calculate its contribution to the sum. The contribution is based on the number of subarrays where this element is the minimum, which is determined by the distances to the next smaller elements on both sides.
4. **Modulo Operation**: Since the result can be very large, we return the sum modulo \(10^9 + 7\).

### Solution Code
```python
class Solution:
    def sumSubarrayMins(self, arr: List[int]) -> int:
        n = len(arr)
        arr = [float('-inf')] + arr + [float('-inf')]
        res = 0
        stack = []
        for i in range(len(arr)):
            while stack and arr[stack[-1]] > arr[i]:
                cur = stack.pop()
                left_bound = stack[-1] if stack else -1
                res += arr[cur] * (cur - left_bound) * (i - cur)
            stack.append(i)
        return res % (10**9 + 7)
```
