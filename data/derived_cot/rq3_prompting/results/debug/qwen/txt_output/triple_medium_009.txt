=== RAW ANSWER ===

To solve this problem, we need to find the closest meeting node for two given nodes in a directed graph. The meeting node should be the one that minimizes the maximum distance from both given nodes. If no such node exists, we return -1.

### Approach
1. **Problem Analysis**: The problem involves traversing a directed graph from two given nodes (node1 and node2) and finding a node that is reachable from both nodes with the smallest maximum distance. The graph is represented by an array `edges` where `edges[i]` is the neighbor of node `i`, or -1 if there is no neighbor.

2. **Intuition**: We can use Breadth-First Search (BFS) to compute the shortest distances from each of the two nodes to all reachable nodes. For each node, we then determine the maximum of the two distances (from node1 and node2). The node with the smallest maximum distance is our answer. If there are multiple such nodes, we return the smallest node index.

3. **Algorithm Selection**: BFS is chosen because it efficiently computes shortest paths in unweighted graphs. We perform BFS twice: once from node1 and once from node2. We maintain a dictionary to store the distances for each node from both BFS traversals.

4. **Complexity Analysis**: The BFS traversal for each node is O(N + E), where N is the number of nodes and E is the number of edges. Since we perform BFS twice, the overall complexity is O(N + E).

### Solution Code
```python
from collections import defaultdict
from typing import List

class Solution:
    def cdist(self, it, distArray, distArrayIndex, edges):
        from collections import deque
        q = deque()
        q.append(it)
        distArray[it][distArrayIndex] = 0
        nodes = [it]
        while q:
            current = q.popleft()
            neighbor = edges[current]
            if neighbor == -1:
                continue
            if distArray[neighbor][distArrayIndex] == float('inf'):
                distArray[neighbor][distArrayIndex] = distArray[current][distArrayIndex] + 1
                q.append(neighbor)
                nodes.append(neighbor)
        return nodes

    def closestMeetingNode(self, edges: List[int], node1: int, node2: int) -> int:
        INF = float('inf')
        dist = defaultdict(lambda: [INF, INF])
        mmin, ans = INF, INF
        
        n1 = self.cdist(node1, dist, 0, edges)
        n2 = self.cdist(node2, dist, 1, edges)
        combined = n1 + n2
        
        for k in combined:
            d1, d2 = dist[k]
            if d1 == INF or d2 == INF:
                continue
            m = max(d1, d2)
            if m < mmin:
                mmin = m
                ans = k
            elif m == mmin:
                ans = min(ans, k)
                
        return ans if ans != INF else -1
```

### Explanation
1. **BFS Traversal**: The `cdist` method performs BFS starting from a given node (`it`) and updates the distance for each node in the graph. It uses a queue to explore nodes level by level, ensuring the shortest path is recorded for each node.

2. **Distance Calculation**: In the `closestMeetingNode` method, we initialize a dictionary `dist` to store distances from both nodes. We then call `cdist` twice, once for each starting node, to populate the distances.

3. **Finding the Meeting Node**: After computing the distances, we iterate over all nodes visited by either BFS. For each node, if both distances are finite, we compute the maximum of the two distances. The node with the smallest maximum distance is our answer. If multiple nodes have the same minimum maximum distance, we return the smallest node index.

This approach efficiently computes the closest meeting node by leveraging BFS for shortest path calculation and then evaluating the results to find the optimal node.

        combined = n1 + n2
        
        for k in combined:
            d1, d2 = dist[k]
            if d1 == INF or d2 == INF:
                continue
            m = max(d1, d2)
            if m < mmin:
                mmin = m
                ans = k
            elif m == mmin:
                ans = min(ans, k)
                
        return ans if ans != INF else -1
