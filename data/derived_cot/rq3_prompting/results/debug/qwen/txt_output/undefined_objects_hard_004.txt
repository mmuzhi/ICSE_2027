### Explanation
1. **BFS Initialization**: The BFS starts with the initial board configuration and zero moves.
2. **Neighbor Generation**: For each board state, the function `get_neighbors` identifies the position of the empty space (0) and generates all valid neighboring states by moving the empty space in four directions (if within the grid bounds).
3. **State Tracking**: Each new state is checked against a set of visited states to prevent revisiting and cycles. If a state hasn't been seen before, it is added to the queue with an incremented move count.
4. **Goal Check**: Each time a state is dequeued, it is checked against the goal state. If matched, the current move count is returned immediately, ensuring the shortest path is found.

This approach efficiently explores all possible states level by level, leveraging BFS properties to guarantee the shortest path is found if the goal state is reachable. The use of a set for visited states ensures optimal performance by avoiding redundant state processing.

class Solution:
    def slidingPuzzle(self, board: List[List[int]]) -> int:
        def get_neighbors(board):
            neighbors = []
            r, c = 0, 0
            for i in range(2):
                for j in range(3):
                    if board[i][j] == 0:
                        r, c = i, j
            directions = [(0, 1), (1, 0), (0, -1), (-1, 0)]
            for dr, dc in directions:
                new_r, new_c = r + dr, c + dc
                if 0 <= new_r < 2 and 0 <= new_c < 3:
                    new_board = [row[:] for row in board]
                    new_board[r][c], new_board[new_r][new_c] = new_board[new_r][new_c], new_board[r][c]
                    neighbors.append(new_board)
            return neighbors

        queue.append((board, 0))
