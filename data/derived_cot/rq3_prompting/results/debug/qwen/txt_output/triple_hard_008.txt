=== RAW ANSWER ===

To solve this problem, we need to find the median of two sorted arrays efficiently. The median is the middle value in an ordered list of numbers. If the combined length of the two arrays is even, the median is the average of the two middle numbers. 

### Approach
The approach involves merging the two sorted arrays into a single sorted array and then finding the median from this merged array. The key steps are:
1. **Merge the two sorted arrays**: We use two pointers to traverse both arrays simultaneously, appending the smaller element from either array to a new merged list.
2. **Handle remaining elements**: After one of the arrays is exhausted, append the remaining elements from the other array to the merged list.
3. **Calculate the median**: Depending on whether the merged array has an odd or even number of elements, compute the median accordingly.

### Solution Code
```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        new = []
        p1 = 0
        p2 = 0
        
        while p1 < len(nums1) and p2 < len(nums2):
            if nums1[p1] < nums2[p2]:
                new.append(nums1[p1])
                p1 += 1
            else:
                new.append(nums2[p2])
                p2 += 1
                
        while p1 < len(nums1):
            new.append(nums1[p1])
            p1 += 1
            
        while p2 < len(nums2):
            new.append(nums2[p2])
            p2 += 1
            
        n = len(new)
        if n % 2 == 0:
            index = n // 2
            median = (new[index] + new[index-1]) / 2
        else:
            median = float(new[n//2])
        return median
```

### Explanation
1. **Initialization**: We start by initializing an empty list `new` to store the merged array and two pointers `p1` and `p2` to traverse `nums1` and `nums2` respectively.
2. **Merging Arrays**: Using a while loop, we compare elements from both arrays and append the smaller element to `new`. The pointers are incremented accordingly.
3. **Remaining Elements**: After one array is fully traversed, the remaining elements from the other array are appended to `new`.
4. **Median Calculation**: The median is calculated based on the length of `new`. If the length is even, the median is the average of the two middle elements. If the length is odd, the median is the middle element.

This approach ensures that we efficiently merge the two sorted arrays and compute the median in linear time, O(n + m), where n and m are the lengths of the two arrays. The space complexity is O(n + m) to store the merged array.
