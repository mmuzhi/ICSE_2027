=== RAW ANSWER ===

To solve this problem, we need to find the median of two sorted arrays. The median is the middle value in an ordered list of numbers. If the combined length of the two arrays is even, the median is the average of the two middle numbers; if it's odd, the median is the middle number.

### Approach
1. **Merge the Two Arrays**: We merge the two sorted arrays into a single sorted array by comparing elements from both arrays using two pointers. This ensures that the merged array is in sorted order.
2. **Calculate the Median**: Once the arrays are merged, we determine the median based on the length of the merged array. If the length is even, the median is the average of the two middle elements; if odd, it's the middle element.

### Solution Code
```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        p1 = 0
        p2 = 0
        new = []
        while p1 < len(nums1) and p2 < len(nums2):
            if nums1[p1] < nums2[p2]:
                new.append(nums1[p1])
                p1 += 1
            else:
                new.append(nums2[p2])
                p2 += 1
        while p1 < len(nums1):
            new.append(nums1[p1])
            p1 += 1
        while p2 < len(nums2):
            new.append(nums2[p2])
            p2 += 1

        if len(new) % 2 == 0:
            index = len(new) // 2
            median = (new[index] + new[index-1]) / 2
        else:
            median = float(new[len(new)//2])
        return median
```

### Explanation
1. **Merging Process**: The algorithm uses two pointers, `p1` and `p2`, to traverse `nums1` and `nums2` respectively. By comparing elements at these pointers, the smaller element is appended to the merged array `new`. The corresponding pointer is then incremented. This process continues until one of the arrays is exhausted.
2. **Appending Remaining Elements**: After one array is exhausted, the remaining elements of the other array are appended to `new` in their original order.
3. **Finding the Median**: The median is calculated based on the length of `new`. If the length is even, the median is the average of the two middle elements. If the length is odd, the median is the middle element.

This approach efficiently merges the two sorted arrays and computes the median in linear time, O(n + m), where n and m are the lengths of the two arrays, respectively. The space complexity is O(n + m) due to the storage required for the merged array.
