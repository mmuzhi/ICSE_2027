### Solution Code
```python
class Solution:
    def longestString(self, x: int, y: int, z: int) -> int:
        if x > y:
            x, y = y, x
        return min(x + y, 2 * x + 1) - 2 * z
```

1. **Swap Handling**: The code first ensures that `x` is the smaller of the two values by swapping them if necessary. This step is crucial because the formula for the maximum string length depends on the smaller count.
2. **Max Length Calculation**: The maximum length of the string is computed as `min(x + y, 2 * x + 1)`. This formula effectively calculates the longest string that can be formed without adjacent characters being the same, considering the counts of both characters.
3. **Adjustment for Constraint**: The computed maximum length is then reduced by `2 * z`, where `z` is a parameter that specifies how many pairs of characters can be removed from the string. This adjustment ensures the solution adheres to the problem's constraints.
