=== RAW ANSWER ===

To solve this problem, we need to find the number of all possible distinct solutions for placing N queens on an N×N chessboard such that no two queens attack each other. The solution involves using backtracking to explore all possible configurations of queens on the board while ensuring that no two queens share the same row, column, or diagonal.

### Approach
1. **Backtracking with Pruning**: The approach involves placing queens one by one, starting from the first column. For each column, we try placing a queen in every row, ensuring that the placement does not conflict with any previously placed queens (i.e., no two queens share the same row, upper diagonal, or lower diagonal).
2. **Tracking Availability**: We use three arrays to keep track of available rows (`low`), upper diagonals (`upper`), and lower diagonals (`lower`). This helps in efficiently checking if a queen can be placed in a particular row and diagonal without conflicts.
3. **Base Case**: When all N queens are successfully placed (i.e., `col == n`), we record the solution by converting the current board configuration into a list of column positions.
4. **Recursion**: The function recursively attempts to place queens in subsequent columns, backtracking whenever a placement leads to an invalid configuration.

### Solution Code
```python
class Solution:
    def totalNQueens(self, n: int) -> int:
        def addans(board, ans):
            temp = []
            for row in board:
                for j in range(len(row)):
                    if row[j] == "Q":
                        temp.append(j + 1)
            ans.append(temp)
        
        def solve(col, board, low, upper, lower, ans, n):
            if col == n:
                addans(board, ans)
                return
            for row in range(n):
                if low[row] == 0 and upper[n - 1 + col - row] == 0 and lower[row + col] == 0:
                    board[row][col] = "Q"
                    low[row] = 1
                    upper[n - 1 + col - row] = 1
                    lower[row + col] = 1
                    solve(col + 1, board, low, upper, lower, ans, n)
                    low[row] = 0
                    upper[n - 1 + col - row] = 0
                    lower[row + col] = 0
        
        ans = []
        board = [[0] * n for _ in range(n)]
        low = [0] * n
        upper = [0] * (2 * n - 1)
        lower = [0] * (2 * n - 1)
        solve(0, board, low, upper, lower, ans, n)
        return len(ans)
```

### Explanation
1. **addans Function**: This helper function converts the current board configuration into a list of column positions where queens are placed. It appends this list to the solutions list (`ans`).
2. **solve Function**: This recursive function attempts to place a queen in the current column (`col`). For each row, it checks if the row and both diagonals (upper and lower) are available. If so, it places a queen and marks the corresponding row and diagonals as occupied. It then proceeds to the next column. If all columns are processed (`col == n`), it records the solution.
3. **Initialization**: The main function initializes the board, tracking arrays (`low`, `upper`, `lower`), and starts the backtracking process from the first column (column 0).
4. **Result**: The function returns the number of valid solutions found by returning the length of the solutions list (`ans`).

This approach efficiently explores all valid configurations using backtracking and pruning, ensuring optimal performance by avoiding invalid placements early in the process.
