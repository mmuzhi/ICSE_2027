The solution involves using a trie to efficiently store and search for forbidden substrings. The approach involves iterating over the word from the end to the beginning, checking substrings starting at each position to see if they contain any forbidden substrings. If a forbidden substring is found, the substring is truncated to exclude the forbidden part. The length of the longest valid substring is then recorded.

### Approach
1. **Trie Construction**: Build a trie from all forbidden substrings. Each node in the trie represents a character, and an "end" marker indicates the end of a forbidden substring.
2. **Sliding Window with Backward Traversal**: Iterate over the word from the end to the beginning. For each starting position, check the substring starting at that position and extending to the current end of the substring. If any forbidden substring is found within this substring, adjust the end position to exclude the forbidden part.
3. **Forbidden Substring Check**: For each substring, use the trie to check if it contains any forbidden substring. This involves traversing the trie character by character. If a forbidden substring is found, return its length; otherwise, return zero.

### Solution Code
```python
class Solution:
    def longestValidSubstring(self, word: str, forbidden: List[str]) -> int:
        trie = {}
        for f in forbidden:
            t = trie
            for c in f:
                if c not in t:
                    t[c] = {}
                t = t[c]
            t["end"] = True
        
        def isForbidden(s):
            n = len(s)
            for start in range(n):
                t = trie
                for pos in range(start, n):
                    c = s[pos]
                    if c in t:
                        t = t[c]
                        if "end" in t:
                            return pos - start + 1
                    else:
                        break
            return 0
        
        res = 0
        j = len(word) + 1
        for i in range(len(word) - 1, -1, -1):

            if truc:

        return res
```
