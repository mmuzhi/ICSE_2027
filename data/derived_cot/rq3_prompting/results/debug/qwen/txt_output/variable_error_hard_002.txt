To solve this problem, we need to find for each element in the array the value of the second greater element to its right. The second greater element is defined as the second element (from the current element) that is strictly greater than the current element. If no such element exists, we should return -1 for that position.

### Approach
1. **First Greater Element Identification**: We first identify the index of the first element to the right of each element that is strictly greater than it. This is done using a stack to efficiently compute the next greater element for each position in one pass from right to left.
2. **Second Greater Element Identification**: For each element that has a first greater element, we then find the next element to the right of the first greater element that is strictly greater than the original element. This is done by iterating from the position right after the first greater element until we find the required element or reach the end of the array.

### Solution Code
```python
class Solution:
    def secondGreaterElement(self, nums: List[int]) -> List[int]:
        n = len(nums)
        first_greater_index = [-1] * n
        stack = []
        for i in range(n-1, -1, -1):
            while stack and nums[stack[-1]] <= nums[i]:
                stack.pop()
            if stack:
                first_greater_index[i] = stack[-1]
            else:
                first_greater_index[i] = -1
            stack.append(i)
        
        ans = [-1] * n
        for i in range(n):
            if first_greater_index[i] == -1:
                continue
            j = first_greater_index[i] + 1
            while j < n and nums[j] <= nums[i]:
                j += 1
            if j < n:
                ans[i] = nums[j]
            else:
                ans[i] = -1
        return ans
```

### Explanation
1. **First Pass (Right to Left)**: We use a stack to keep track of indices. For each element, we pop elements from the stack that are less than or equal to the current element. The remaining top of the stack gives the index of the first greater element. This helps us in efficiently determining the first greater element for each position in O(n) time.
2. **Second Pass (Left to Right)**: For each element that has a first greater element, we start searching from the position right after the first greater element. We continue searching until we find an element strictly greater than the original element or reach the end of the array. The value of this element is the second greater element for the original position. If no such element is found, we return -1.

This approach ensures that we efficiently compute the second greater element for each position in the array with a time complexity of O(n^2) in the worst case, but with a practical average case performance due to the use of stacks for the first pass. The space complexity is O(n) for storing the indices and results.
