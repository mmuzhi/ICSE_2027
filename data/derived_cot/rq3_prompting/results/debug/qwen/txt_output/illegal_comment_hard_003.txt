=== RAW ANSWER ===

To solve this problem, we need to find the second greater element for each element in an array. The second greater element for an element at index `i` is the smallest element greater than `nums[i]` that appears after the first greater element of `nums[i]`. If no such element exists, the result should be `-1`.

### Approach
1. **Problem Analysis**: The task involves finding elements that are greater than a given element and appear after its first greater element. This requires two passes: one to find the first greater element and another to find the second greater element.
2. **Intuition**: We can use a stack-based approach where we maintain two stacks:
   - **First Stack (`stack`)**: Tracks elements waiting for their first greater element.
   - **Second Stack (`second_stack`)**: Tracks elements that have found their first greater element and are waiting for their second greater element.
3. **Algorithm Selection**: 
   - **Forward Pass**: For each element, pop from `second_stack` elements whose value is less than the current element and set their second greater element to the current element. Then, pop from `stack` elements whose value is less than the current element, record their first greater element, and move them to `second_stack`.
4. **Complexity Analysis**: The algorithm processes each element exactly twice (once for each stack operation), resulting in an O(n) time complexity, where n is the number of elements in the array. The space complexity is O(n) due to the stacks used.

### Solution Code
```python
class Solution:
    def secondGreaterElement(self, nums: List[int]) -> List[int]:
        n = len(nums)
        ans = [-1] * n
        stack = []
        second_stack = []
        
        for i in range(n):
            while second_stack and nums[second_stack[-1][0]] < nums[i] and second_stack[-1][1] < i:
                j = second_stack.pop()[0]
                ans[j] = nums[i]
            
            buff = []
            while stack and nums[stack[-1]] < nums[i]:
                k = stack.pop()
                buff.append((k, i))
            
            for (k, first_index) in buff:
                second_stack.append((k, first_index))
            
            stack.append(i)
        
        return ans
```

   - **First Stack Handling**: Elements from `stack` that are smaller than the current element are moved to `second_stack` with their first greater element marked as the current index.
3. **Result Construction**: The answer array is populated as we process each element, ensuring the second greater element is correctly identified and stored. The final result is returned after processing all elements.
