To solve this problem, we need to generate all numbers within a given range [low, high] that consist of consecutive increasing digits (sequential digits). The approach involves using a breadth-first search (BFS) to explore all possible numbers formed by appending the next digit to the current number until the last digit reaches 9.

### Approach
1. **Initialization**: Start with digits from 1 to 9 (since 10 is not a sequential digit and should be excluded).
2. **BFS Exploration**: For each number dequeued, check if it lies within the specified range [low, high]. If it does, add it to the result list.
3. **Extension**: For each number, compute the next sequential digit by appending the next digit (current last digit + 1) to form a new number. This is only possible if the last digit is less than 9.
4. **Termination**: The process continues until the queue is empty, ensuring all possible sequential digits are generated and checked against the range.

### Solution Code
```python
from collections import deque

class Solution:
    def sequentialDigits(self, low, high):
        out = []
        queue = deque(range(1, 10))
        while queue:
            elem = queue.popleft()
            if low <= elem <= high:
                out.append(elem)
            last = elem % 10
            if last < 9:
                queue.append(elem * 10 + last + 1)
        return out
```

### Explanation
1. **Initialization**: The queue starts with digits 1 through 9, as these are the only possible starting points for sequential digits.
2. **BFS Loop**: The loop processes each number in the queue. If the number is within the range [low, high], it is added to the result list.
3. **Digit Extension**: For each number, the last digit is extracted. If this digit is less than 9, the next sequential digit (last digit + 1) is appended to form a new number, which is then enqueued.
4. **Result Compilation**: The loop continues until all possible sequential digits are generated and checked. The result list, containing all valid numbers, is returned.

This approach efficiently explores all possible sequential digits using BFS, ensuring correctness and completeness within the given range.
