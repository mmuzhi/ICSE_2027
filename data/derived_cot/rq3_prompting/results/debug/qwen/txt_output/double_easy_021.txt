=== RAW ANSWER ===

To solve this problem, we need to determine the maximum depth of a binary tree. The maximum depth is the number of nodes along the longest path from the root node down to the farthest leaf node. 

### Approach
The approach involves using a Depth-First Search (DFS) with a stack to traverse the tree. Each node is pushed onto the stack along with its current depth. As we pop each node from the stack, we check if it is not None. If it is not None, we update the maximum depth encountered so far and push its left and right children onto the stack with an incremented depth. This ensures that we explore all paths from the root to the leaves, keeping track of the maximum depth encountered.

### Solution Code
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def maxDepth(self, root: Optional[TreeNode]) -> int:
        stack = [[root, 1]]
        res = 0

        while stack:
            node, depth = stack.pop()
            if node:
                res = max(res, depth)
                stack.append([node.left, depth + 1])
                stack.append([node.right, depth + 1])
        
        return res
```

### Explanation
1. **Initialization**: We start by initializing a stack with the root node and a depth of 1. The variable `res` is initialized to 0 to keep track of the maximum depth found.
2. **Stack Processing**: While the stack is not empty, we pop the top element, which contains a node and its current depth. If the node is not None, we update `res` to the maximum of its current value and the node's depth.
3. **Child Nodes**: We then push the left and right children of the current node onto the stack with their respective depths incremented by 1.
4. **Termination**: After processing all nodes, the variable `res` will hold the maximum depth of the tree, which is returned as the result.

This approach efficiently explores all paths from the root to the leaves using DFS, ensuring that the maximum depth is accurately recorded. The use of a stack allows us to manage the traversal iteratively, which is suitable for environments where recursion depth might be a concern.
