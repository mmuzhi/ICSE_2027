To solve this problem, we need to calculate the total score based on a series of operations performed on a stack of scores. The operations include pushing a new score, popping the last score, doubling the last score, and adding the last two scores. The key challenge is to correctly handle each operation and compute the final sum of all scores in the stack.

### Approach
1. **Initialize a Stack**: We start with an empty stack to keep track of the scores.
2. **Process Each Operation**: For each operation in the input list:
   - **'D' Operation**: Double the last score in the stack and push the result onto the stack.
   - **'C' Operation**: Remove the last score from the stack.
   - **'+' Operation**: Add the last two scores in the stack and push the result onto the stack.
   - **Digit String**: Convert the string to an integer and push it onto the stack.
3. **Sum the Stack**: After processing all operations, return the sum of all elements in the stack.

### Solution Code
```python
class Solution:
    def calPoints(self, operations: List[str]) -> int:
        stack = []
        for op in operations:
            if op == 'D':
                stack.append(2 * stack[-1])
            elif op == 'C':
                stack.pop()
            elif op == '+':
                stack.append(stack[-1] + stack[-2])
            else:
                stack.append(int(op))
        return sum(stack)
```

### Explanation
- **Stack Initialization**: The stack is initialized as an empty list to store the scores.
- **Processing Operations**:
  - **'D' Operation**: The last score in the stack is doubled and pushed onto the stack.
  - **'C' Operation**: The last score is removed from the stack.
  - **'+' Operation**: The sum of the last two scores is computed and pushed onto the stack.
  - **Digit String**: The string is converted to an integer and pushed onto the stack.
- **Summing the Stack**: After processing all operations, the sum of all elements in the stack is returned, which represents the total score.

This approach efficiently handles each operation in sequence, ensuring that the stack operations are performed correctly and the final result is computed accurately. The solution assumes that the input operations are valid, meaning that operations like '+' and 'D' will only be encountered when the stack has the required number of elements.
