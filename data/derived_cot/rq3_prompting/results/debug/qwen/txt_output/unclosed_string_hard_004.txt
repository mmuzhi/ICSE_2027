2. **Intuition**: We use three auxiliary arrays to keep track of available rows, upper diagonals, and lower diagonals:
   - `low` array tracks which rows are already occupied.
   - `upper` array tracks the upper diagonals (from top-right to bottom-left).
   - `lower` array tracks the lower diagonals (from bottom-left to top-right).
3. **Algorithm Selection**: The backtracking algorithm (`solve` function) recursively tries to place a queen in each column, checking the availability of the corresponding row and diagonals. If a valid position is found, it proceeds to the next column; otherwise, it backtracks.
4. **Complexity Analysis**: The time complexity is `O(n!)` in the worst case, as the backtracking explores all permutations of queen placements. The space complexity is `O(n)` due to the storage required for the auxiliary arrays and the recursion stack.

### Solution Code
```python
class Solution:
    def totalNQueens(self, n: int) -> int:
        def addans(board, ans):
            temp = []
            for row in board:
                for j in range(len(row)):
                    if row[j] == 1:
                        temp.append(j+1)
            ans.append(temp)
        
        def solve(col, board, low, upper, lower, ans, n):
            if col == n:
                addans(board, ans)
                return
            for row in range(n):
                if low[row] == 0 and upper[n-1+col-row] == 0 and lower[row+col] == 0:
                    board[row][col] = 1
                    low[row] = 1
                    upper[n-1+col-row] = 1
                    lower[row+col] = 1
                    solve(col+1, board, low, upper, lower, ans, n)
                    low[row] = 0
                    upper[n-1+col-row] = 0
                    lower[row+col] = 0
        
        ans = []
        board = [[0]*n for _ in range(n)]
        low = [0]*n
        upper = [0]*(2*n-1)
        lower = [0]*(2*n-1)
        solve(0, board, low, upper, lower, ans, n)
        return len(ans)
```

### Explanation
1. **Initialization**: The `totalNQueens` method initializes the chessboard, tracking arrays (`low`, `upper`, `lower`), and the answer list (`ans`).
2. **Backtracking (`solve` function)**: This function attempts to place a queen in each column (`col`). For each row (`row`), it checks if the row and diagonals are free. If so, it places the queen (sets `board[row][col]` to 1) and updates the tracking arrays. It then recursively processes the next column. If no valid placement is found, it backtracks by resetting the changes.
3. **Solution Collection (`addans` function)**: This helper function converts the current board configuration into a list of column indices where queens are placed, appends this configuration to `ans`, and is called whenever a complete solution is found (`col == n`).
4. **Result**: After exploring all possible configurations, the method returns the count of valid solutions (`len(ans)`).
