4. **Traverse the Array**: We traverse the array while maintaining a running sum. Whenever the running sum equals the target sum, we reset the running sum and increment the part count. If we reach three parts during the traversal, we return True.
5. **Final Check**: After traversing the entire array, if we have exactly three parts, return True; otherwise, return False.

### Solution Code

class Solution:
    def canThreePartsEqualSum(self, arr: List[int]) -> bool:
