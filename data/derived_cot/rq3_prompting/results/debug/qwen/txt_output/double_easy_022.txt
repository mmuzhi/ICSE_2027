### Approach
1. **Problem Analysis**: The task is to determine the largest positive number in the array where none of its negative counterparts are present. The solution involves checking each positive number in the array to ensure its negative is not present. If any positive number's negative is found, the answer is -1. Otherwise, the largest positive number is the answer.

2. **Intuition**: By using a set for O(1) lookups, we can efficiently check the presence of negative counterparts. Iterating through the array once allows us to both verify the condition and find the maximum positive number.

3. **Algorithm Selection**: 
   - **Set for Lookup**: Convert the array into a set for constant-time lookups.
   - **Single Pass**: Traverse the array to check each positive number's negative presence and simultaneously track the maximum positive number.

4. **Complexity Analysis**: 
   - **Time Complexity**: O(n) for iterating through the array and O(1) for set operations, leading to an overall O(n) time complexity.
   - **Space Complexity**: O(n) for storing the set.

### Solution Code
```python
class Solution:
    def findMaxK(self, nums: List[int]) -> int:
        s = set(nums)
        max_val = -1
        for x in nums:
            if x > 0:
                if -x in s:
                    return -1
                else:
                    if x > max_val:
                        max_val = x
        return max_val
```

### Explanation
1. **Set Creation**: The set `s` is created from the input array to allow O(1) lookups for checking the presence of negative numbers.
2. **Iteration and Check**: For each element in the array:
   - If the element is positive, check if its negative exists in the set. If it does, return -1 immediately as the condition fails.
   - If the element is positive and its negative is not found, update the maximum positive value encountered.
3. **Result**: After processing all elements, the largest positive number without its negative counterpart is returned. If no such number exists, -1 is returned.

This approach efficiently checks the required conditions and ensures optimal performance with linear time complexity.
