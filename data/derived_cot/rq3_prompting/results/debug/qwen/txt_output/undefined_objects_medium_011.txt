To solve this problem, we need to split a given linked list into `k` parts, where each part has as close to the same number of nodes as possible. The solution involves calculating the number of nodes each part should contain and then traversing the list to split it accordingly.

### Approach
1. **Calculate the Length of the Linked List**: First, we determine the total number of nodes in the linked list by traversing from the head node to the end.
2. **Determine the Number of Nodes per Part**: Using the total length and the number of parts `k`, we calculate how many nodes each part should have. The first `remainder` parts (where `remainder` is the remainder of the division of the length by `k`) will have `base + 1` nodes, and the remaining parts will have `base` nodes, where `base` is the quotient of the division.
3. **Split the Linked List**: We then traverse the linked list again, this time splitting it into the required parts. For each part, we move the appropriate number of nodes and adjust the `next` pointer of the last node in the part to `None` to separate it from the rest of the list.

### Solution Code
```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def splitListToParts(self, head: Optional[ListNode], k: int) -> List[Optional[ListNode]]:
        length = 0
        ptr = head
        while ptr:
            length += 1
            ptr = ptr.next
        
        base = length // k
        remainder = length % k
        arrange = [base + 1] * remainder + [base] * (k - remainder)
        
        parts = []
        if length == 0:
            return [None] * k
        
        current = head
        for i in range(k):
            if arrange[i] == 0:
                parts.append(None)
                continue
            
            part_head = current
            for _ in range(arrange[i] - 1):
                current = current.next
            
            next_node = current.next
            current.next = None
            parts.append(part_head)
            current = next_node
        
        return parts
```

1. **Calculate Length**: The code first traverses the linked list to count the total number of nodes (`length`).
2. **Determine Part Sizes**: Using the length and `k`, it calculates how many nodes each part should contain. The first `remainder` parts will have `base + 1` nodes, and the remaining parts will have `base` nodes.
3. **Split the List**: The code then traverses the list again, splitting it into `k` parts. For each part, it moves the appropriate number of nodes from the current position, sets the `next` pointer of the last node in the part to `None`, and records the head of the part. If a part should have zero nodes, it appends `None` to the result list.
4. **Handle Edge Cases**: If the list is empty, it returns a list of `k` `None` values. Otherwise, it processes each part as described.
