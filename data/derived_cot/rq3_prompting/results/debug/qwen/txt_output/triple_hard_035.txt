To solve this problem, we need to determine the maximum number of times we can repeat the string `s2` (each repetition being `n2` times) by concatenating the string `s1` `n1` times and then extracting non-overlapping occurrences of `s2`. The solution involves efficiently counting these occurrences without explicitly constructing the entire concatenated string, which is crucial for handling large values of `n1`.

### Approach
1. **Initial Check**: If `s2` contains any character not present in `s1`, return 0 immediately since no occurrences of `s2` can be formed.
2. **Cycle Detection**: The key insight is to detect cycles in the matching process to avoid redundant computations. By tracking the start positions during each matching of `s2`, we can identify when the matching process starts repeating (a cycle is detected).
3. **Simulation and Counting**: For each occurrence of `s2`, we simulate the matching process by scanning `s1` from the current position. The total characters matched so far are recorded. If a cycle is detected, we use the cycle's properties (number of characters and occurrences per cycle) to project the total occurrences for the entire string.
4. **Final Calculation**: The total number of `s2` occurrences is divided by `n2` (using integer division) to get the result.

### Solution Code
```python
from collections import defaultdict

class Solution:
    def getMaxRepetitions(self, s1: str, n1: int, s2: str, n2: int) -> int:
        if not set(s2).issubset(set(s1)):
            return 0
        
        rec = [0]
        track = defaultdict(int)
        ct = 0
        start = 0
        
        while True:
            found = True
            for char in s2:
                pos = s1.find(char, start)
                if pos == -1:
                    found = False
                    break
                start = pos + 1
                ct += 1
            
            rec.append(ct)
            
            if ct > n1 * len(s1):
                break
            
            if start in track:
                cycleStart = rec[track[start]]
                cycle1 = ct - cycleStart
                cycle2 = len(rec) - 1 - track[start]
                
                remaining_chars = n1 * len(s1) - cycleStart
                full_cycles = remaining_chars // cycle1
                additional_s2 = full_cycles * cycle2

                return total_s2 // n2
```
