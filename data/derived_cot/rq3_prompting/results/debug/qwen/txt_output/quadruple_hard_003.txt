### Approach
1. **Trie Construction**: Build a trie from the list of forbidden substrings. Each node in the trie represents a character, and an "end" marker indicates the completion of a forbidden substring.
2. **Forbidden Substring Check**: For any substring, traverse the trie to check if it contains any forbidden substring. If a forbidden substring is found, return its length; otherwise, return False.
3. **Two-Pointer Technique**: Use two pointers, `i` and `j`, to represent the current substring being checked. Start with `j` at the end of the word and move `i` from the end to the beginning. For each position of `i`, check the substring `word[i:j]` for forbidden substrings. If a forbidden substring is found, adjust `j` to the position right after the forbidden substring. The length of the valid substring is then `j - i`, and we keep track of the maximum length encountered.
