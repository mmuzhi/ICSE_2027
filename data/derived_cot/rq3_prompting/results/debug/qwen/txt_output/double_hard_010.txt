=== RAW ANSWER ===

To solve this problem, we need to find the median of two sorted arrays. The median is the middle value in an ordered list of numbers. If the combined length of the two arrays is even, the median is the average of the two middle numbers. 

### Approach
1. **Merge the Arrays**: We start by merging the two sorted arrays into a single sorted array. This is done by comparing elements from both arrays and appending the smaller element to the result array until one of the arrays is exhausted. The remaining elements from the non-exhausted array are then appended to the result.
2. **Calculate the Median**: After merging, we check the length of the resulting array. If the length is odd, the median is the middle element. If the length is even, the median is the average of the two middle elements.

### Solution Code
```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        p1 = 0
        p2 = 0
        new = []
        while p1 < len(nums1) and p2 < len(nums2):
            if nums1[p1] < nums2[p2]:
                new.append(nums1[p1])
                p1 += 1
            else:
                new.append(nums2[p2])
                p2 += 1
        while p1 < len(nums1):
            new.append(nums1[p1])
            p1 += 1
        while p2 < len(nums2):
            new.append(nums2[p2])
            p2 += 1

        n = len(new)
        if n % 2 == 0:
            index = n // 2
            median = (new[index] + new[index-1]) / 2
        else:
            median = float(new[n//2])
        return median
```

### Explanation
1. **Merging Arrays**: The code merges two sorted arrays by iterating through both arrays simultaneously, comparing elements, and appending the smaller element to the result array. This ensures the merged array remains sorted.
2. **Finding the Median**: After merging, the length of the merged array is checked. If the length is even, the median is calculated as the average of the two middle elements. If the length is odd, the median is simply the middle element. The result is returned as a float to handle both cases appropriately.

This approach efficiently combines the two sorted arrays and computes the median in linear time, making it optimal for large input sizes.
