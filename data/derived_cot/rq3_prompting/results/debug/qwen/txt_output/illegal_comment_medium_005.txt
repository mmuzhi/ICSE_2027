To solve this problem, we need to find the k-th smallest factor of a given integer `n`. The approach involves iterating through all numbers from 1 to `n` and checking if each number is a factor of `n`. We count each factor encountered until we reach the k-th factor, which we then return. If there are fewer than `k` factors, we return -1.

### Approach
1. **Initialization**: Start with a counter `j` initialized to 0 to keep track of the number of factors found.
2. **Iteration**: Loop through each number `i` from 1 to `n` (inclusive).
3. **Factor Check**: For each `i`, check if `i` is a factor of `n` (i.e., `n % i == 0`).
4. **Count and Break**: If `i` is a factor, increment the counter `j`. If `j` equals `k`, break out of the loop and return `i` immediately.
5. **Return Result**: After the loop completes, if `j` equals `k`, return the last factor found (which is the k-th factor). Otherwise, return -1 indicating there are fewer than `k` factors.

### Solution Code
```python
class Solution:
    def kthFactor(self, n: int, k: int) -> int:
        j = 0
        for i in range(1, n + 1):
            if n % i == 0:
                j += 1
                if j == k:
                    return i
        return -1
```
