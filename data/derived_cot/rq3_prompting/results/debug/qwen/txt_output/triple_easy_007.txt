The operations include doubling the last score ('D'), removing the last score ('C'), adding the last two scores ('+'), and pushing a new score. ### Approach
1. **Stack Utilization**: We use a stack to keep track of the scores. Each operation modifies the stack in a specific way:
   - **'D' Operation**: Double the last score and push it onto the stack.
   - **'C' Operation**: Remove the last score from the stack.
   - **'+' Operation**: Add the last two scores and push the result onto the stack.
   - **Numeric Operation**: Convert the operation string to an integer and push it onto the stack.
2. **Sum Calculation**: After processing all operations, the total score is the sum of all elements in the stack.

### Solution Code
```python
class Solution:
    def calPoints(self, operations: List[str]) -> int:
        stack = []
        for op in operations:
            if op == 'D':
                stack.append(2 * stack[-1])
            elif op == 'C':
                stack.pop()
            elif op == '+':
                stack.append(stack[-1] + stack[-2])
            else:
                stack.append(int(op))
        return sum(stack)
```

### Explanation
1. **Initialization**: We start with an empty stack.
2. **Processing Operations**:
   - For each operation in the input list:
     - If the operation is 'D', we double the top element of the stack and push the result onto the stack.
     - If the operation is 'C', we remove the top element from the stack.
     - If the operation is '+', we add the top two elements of the stack and push the result onto the stack.
     - For any other operation, we convert the string to an integer and push it onto the stack.
3. **Summing the Stack**: After processing all operations, the total score is the sum of all elements in the stack, which is returned as the result.

This approach efficiently handles each operation by leveraging stack operations, ensuring correctness and optimal performance. The use of a stack allows us to manage the dynamic nature of the operations, such as adding, removing, and combining scores as specified.
