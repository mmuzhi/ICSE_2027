To solve this problem, we need to find all the starting indices of substrings in a given string `s` that are a concatenation of each word in `words` exactly once, without any intervening characters. The solution involves efficiently checking all possible substrings of `s` that match the concatenation of the words.

### Approach
1. **Problem Analysis**: The task is to find all starting positions in `s` where a substring of length equal to the total length of all words concatenated together can be formed by concatenating each word exactly once in order. The challenge is to do this efficiently without checking every possible substring individually.

2. **Intuition**: The solution leverages a sliding window approach. For each possible starting offset (0 to `wlen-1` where `wlen` is the length of each word), we slide a window of length `slen` (total length of all words concatenated) over the string `s`. We maintain a frequency dictionary of words in the current window and compare it with the frequency dictionary of all words. If they match, the starting index of the window is recorded.

3. **Algorithm Selection**: 
   - **Frequency Dictionary**: Use `collections.Counter` to store the frequency of each word in `words`.
   - **Sliding Window**: For each starting offset `k`, build the initial window by processing words in `s` starting from `k` with step `wlen`. Then, slide the window by `wlen` each time, updating the frequency dictionary by removing the leftmost word and adding the new rightmost word.
   - **Validation**: After each slide, check if the current frequency dictionary matches the target frequency dictionary.

4. **Complexity Analysis**: The algorithm processes each starting offset (O(n) where `n` is the word length) and for each offset, processes a window of length `slen` (O(m) where `m` is the total length of all words). Thus, the overall complexity is O(n * m), which is efficient for the given constraints.

import collections
from typing import List

class Solution:
    def findSubstring(self, s: str, words: List[str]) -> List[int]:
        if not words:
