=== RAW ANSWER ===

To solve this problem, we need to restore all possible valid IP addresses from a given string. An IP address consists of four decimal numbers separated by dots, each ranging from 0 to 255. The solution involves generating all possible combinations of four parts from the string and checking if each part is a valid number.

### Approach
1. **Backtracking**: The solution uses a backtracking approach to explore all possible ways to split the string into four parts. Each part must be a number between 0 and 255.
2. **Base Case**: If the entire string is processed and exactly four parts are formed, the IP address is added to the result list.
3. **Recursive Case**: For each position in the string, the algorithm considers parts of length 1, 2, or 3 digits (since numbers longer than three digits cannot be less than 256). It checks if the substring forms a valid number (0-255) and proceeds recursively.
4. **Leading Zeros**: The algorithm allows parts with leading zeros (e.g., "01" is valid) as long as the integer value is within the valid range.

### Solution Code
```python
class Solution():
    def restoreIpAddresses(self, s: str) -> List[str]:
        res = []
        cur = []
        
        def backtrack(i):
            if i == len(s) and len(cur) == 4:
                res.append(".".join(cur))
                return
            if len(cur) > 4 or i > len(s):
                return
            j = 0
            while j < 3 and i + j < len(s):
                part = s[i:i+j+1]
                if int(part) < 256:
                    cur.append(part)
                    backtrack(i+j+1)
                    cur.pop()
                j += 1
        
        backtrack(0)
        return res
```

### Explanation
1. **Initialization**: The `res` list stores all valid IP addresses, and `cur` list keeps track of the current parts being formed.
2. **Backtracking Function**: The `backtrack` function is defined to explore all possible splits:
   - **Base Case**: If the entire string is processed (`i == len(s)`) and four parts are formed (`len(cur) == 4`), the current parts are joined into an IP address and added to `res`.
   - **Invalid Cases**: If more than four parts are formed or the index `i` exceeds the string length, the function returns without further processing.
3. **Part Generation**: The loop generates parts of length 1 to 3 digits. For each part, it checks if the integer value is less than 256. Valid parts are added to `cur`, and the function proceeds recursively. After returning from the recursion, the part is removed from `cur` to backtrack and explore other possibilities.
4. **Result Compilation**: After the backtracking completes, all valid IP addresses are returned in `res`.

This approach efficiently explores all possible valid IP addresses by leveraging backtracking and careful validation of each part's numeric range.
