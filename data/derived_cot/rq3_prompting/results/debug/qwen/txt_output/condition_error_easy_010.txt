=== RAW ANSWER ===

To solve this problem, we need to convert a given positive integer to its corresponding Excel column title. For example, 1 should be "A", 26 should be "Z", 27 should be "AA", and so on. The key insight here is that the conversion resembles converting a number to base-26, but with a few adjustments since there is no zero in the Excel column title system.

### Approach
1. **Problem Analysis**: The problem requires converting a number to a string where each digit represents a letter in the alphabet. The challenge is to handle numbers larger than 26 by breaking them down into their respective base-26 components, similar to how we convert numbers to different bases.
2. **Insight**: The solution involves repeatedly dividing the number by 26, adjusting for the fact that there is no zero in the system. Each division step gives a remainder that corresponds to a letter in the alphabet. The adjustment involves subtracting 1 before division to handle the zero case properly.
3. **Algorithm Selection**: The algorithm uses a loop to process each digit of the number. In each iteration, the number is reduced by subtracting 1, then divided by 26. The remainder from this division determines the current letter, which is prepended to the result string. This process continues until the number becomes zero.
4. **Complexity Analysis**: The time complexity is O(log n) because each iteration reduces the number by a factor of 26. The space complexity is O(log n) to store the result string.

### Solution Code
```python
class Solution:
    def convertToTitle(self, n: int) -> str:
        s = ""
        while n > 0:
            n, r = divmod(n - 1, 26)
            s = chr(65 + r) + s
        return s
```

### Explanation
1. **Initialization**: Start with an empty string `s` to build the result.
2. **Loop Until n is Zero**: Continue processing while `n` is greater than zero.
3. **Division and Remainder**: For each iteration, compute `divmod(n - 1, 26)`. The `n - 1` adjustment ensures that the remainder correctly maps to letters A-Z (0 to 25). The quotient from the division reduces `n` for the next iteration.
4. **Character Conversion**: Convert the remainder `r` to the corresponding uppercase letter by adding it to the ASCII value of 'A' (65). This letter is prepended to the result string `s`.
5. **Termination**: When `n` becomes zero, the loop exits, and the fully constructed string `s` is returned.

This approach efficiently handles all positive integers by leveraging base-26 conversion with proper adjustments for the absence of a zero digit in the Excel column title system.
