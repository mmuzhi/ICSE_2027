### Approach
1. **Backtracking Function**: The core of the solution is a recursive backtracking function that builds the IP address parts one by one. The function checks if the current part is valid (either a single '0' or a number between 1 and 255) before proceeding to the next part.
2. **Base Case**: If all four parts are formed and the entire string is consumed, the current IP address is added to the result list.
3. **Pruning Invalid Paths**: The backtracking function prunes paths where the number of parts exceeds four or the string is exhausted without forming four parts. It also handles cases where a part starts with '0' by only allowing single-digit '0's.
4. **Digit Limit**: Each part can have up to three digits (to stay within the 0-255 range), which is enforced by the loop condition in the backtracking function.

```python
class Solution:
    def restoreIpAddresses(self, s: str) -> List[str]:
        res = []
        cur = []
        
        def backtrack(i):
            if i == len(s) and len(cur) == 4:
                res.append(".".join(cur))
                return
            if len(cur) > 4 or i > len(s):
                return
            if s[i] == '0':
                cur.append('0')
                backtrack(i + 1)
                cur.pop()
                return

            while j < 4 and i + j < len(s):
                num = int(s[i:i+j+1])
                if num < 256:
                    cur.append(s[i:i+j+1])
                    backtrack(i + j + 1)
                    cur.pop()
                j += 1
        
        backtrack(0)
        return res
```
