=== RAW ANSWER ===

To solve this problem, we need to find the median of two sorted arrays efficiently. The median is the middle value in an ordered list of numbers. If the combined length of the two arrays is even, the median is the average of the two middle numbers. 

### Approach
The approach involves merging the two sorted arrays into a single sorted array and then determining the median based on the length of this merged array. The key steps are:
1. **Merge the Arrays**: Use two pointers to traverse both arrays simultaneously, appending the smaller element from either array to a new merged array.
2. **Handle Remaining Elements**: After one array is exhausted, append the remaining elements from the other array.
3. **Calculate the Median**: If the merged array has an even length, the median is the average of the two middle elements. For an odd length, it is the middle element.

### Solution Code
```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        p1 = 0
        p2 = 0
        new = []
        
        while p1 < len(nums1) and p2 < len(nums2):
            if nums1[p1] < nums2[p2]:
                new.append(nums1[p1])
                p1 += 1
            else:
                new.append(nums2[p2])
                p2 += 1
                
        while p1 < len(nums1):
            new.append(nums1[p1])
            p1 += 1
            
        while p2 < len(nums2):
            new.append(nums2[p2])
            p2 += 1
            
        if len(new) % 2 == 0:
            index = len(new) // 2
            median = (new[index] + new[index-1]) / 2
        else:
            median = float(new[len(new)//2])
        return median
```

### Explanation
1. **Merging Arrays**: The code uses two pointers (`p1` and `p2`) to traverse `nums1` and `nums2` respectively. The smaller element between the current pointers is appended to the new merged array, and the corresponding pointer is incremented.
2. **Remaining Elements**: Once one array is fully traversed, the remaining elements of the other array are appended to the merged array.
3. **Finding Median**: The median is calculated by checking the length of the merged array. If the length is even, the median is the average of the two middle elements. If the length is odd, the median is the middle element.

This approach efficiently merges the two sorted arrays and computes the median in linear time, O(n + m), where n and m are the lengths of the two arrays, respectively. The space complexity is O(n + m) to store the merged array.
