To solve this problem, we need to determine the number of seats booked for each flight from 1 to n based on a list of bookings. Each booking specifies a start flight, an end flight, and the number of seats booked. The challenge is to efficiently compute the total seats booked for each flight without resorting to a brute-force approach.

### Approach
1. **Difference Array Technique**: The solution leverages the difference array technique to efficiently compute the number of seats booked for each flight. This technique involves:
   - **Initialization**: Create an array `arr` of size `n+1` initialized to zero. This array will help in marking the start and end of each booking's effect.
   - **Marking Bookings**: For each booking `[lv, ar, seats]`, increment the start flight (`lv-1` in 0-based indexing) by `seats` and decrement the flight right after the end (`ar` in 0-based indexing) by `seats`. This marks the beginning and end of the booking's effect.
   - **Prefix Sum Calculation**: The prefix sum of the `arr` array (excluding the last element) gives the total seats booked for each flight from 1 to n. The prefix sum is computed using `itertools.accumulate`.

### Solution Code
```python
from typing import List
import itertools

class Solution:
    def corpFlightBookings(self, bookings: List[List[int]], n: int) -> List[int]:
        arr = [0] * (n + 1)
        for lv, ar, seats in bookings:
            arr[lv - 1] += seats
            arr[ar] -= seats
        return list(itertools.accumulate(arr[:-1]))
```

### Explanation
1. **Initialization**: The array `arr` is initialized to store the difference of bookings. The size `n+1` ensures we can handle all flights from 1 to n.
2. **Processing Bookings**: For each booking, the start flight (`lv-1` in 0-based indexing) is incremented by the number of seats (`seats`), and the flight right after the end (`ar` in 0-based indexing) is decremented by `seats`. This marks the start and end of the booking's effect.
3. **Prefix Sum Calculation**: The prefix sum of the `arr` array (excluding the last element) is computed using `itertools.accumulate`. This gives the total seats booked for each flight from 1 to n. The result is returned as a list.

This approach efficiently computes the result in O(n + m) time complexity, where m is the number of bookings, by leveraging the difference array technique and prefix sums. The space complexity is O(n) due to the storage of the difference array.
