To solve this problem, we need to compute the sum of the minimum values of all contiguous subarrays in a given array. The challenge lies in efficiently calculating this sum without resorting to a brute-force approach, which would be computationally expensive.

### Approach
The efficient approach involves using a monotonic stack to determine the boundaries for each element where it is the minimum. Here's the step-by-step breakdown:

1. **Left Boundaries Calculation**: For each element in the array, find the nearest previous element that is smaller than the current element. This defines the left boundary for the current element. The number of elements to the left until this boundary is the left span.
2. **Right Boundaries Calculation**: Similarly, for each element, find the nearest next element that is smaller than the current element. This defines the right boundary. The number of elements to the right until this boundary is the right span.
3. **Sum Calculation**: For each element, multiply its value by the product of its left and right spans. This gives the sum of all subarrays where the element is the minimum. Summing these products for all elements gives the final result.

This approach efficiently computes the sum in O(n) time using two passes over the array with a stack, ensuring optimal performance.

### Solution Code
```python
class Solution:
    def sumSubarrayMins(self, arr: List[int]) -> int:
        n = len(arr)
        mod = 10**9 + 7
        left_bound = [-1] * n
        stack = []
        for i in range(n):
            while stack and arr[stack[-1]] > arr[i]:
                stack.pop()
            left_bound[i] = stack[-1] if stack else -1
            stack.append(i)
        
        stack = []
        right_bound = [n] * n
        for i in range(n-1, -1, -1):
            while stack and arr[stack[-1]] > arr[i]:
                stack.pop()
            right_bound[i] = stack[-1] if stack else n
            stack.append(i)
        
        res = 0
        for i in range(n):
            left_span = i - left_bound[i]
            right_span = right_bound[i] - i
            res = (res + arr[i] * left_span * right_span) % mod
        return res
```

The left boundary for the current element is the index of the top element in the stack (if any) or -1 if the stack is empty.
2. **Right Boundaries Calculation**: The second loop processes each element from right to left. Similarly, it uses a stack to keep track of indices in increasing order. For each element, it pops elements from the stack that are greater than the current element, updating the right boundary for those elements. The right boundary for the current element is the index of the top element in the stack (if any) or the array length if the stack is empty.
3. **Sum Calculation**: For each element, the number of subarrays where it is the minimum is given by the product of its left and right spans. The value of the element is multiplied by this product, and the result is accumulated modulo \(10^9 + 7\).

This method efficiently computes the desired sum by leveraging stack operations to determine the boundaries for each element, ensuring optimal performance with linear time complexity.

class Solution:
    def sumSubarrayMins(self, arr: List[int]) -> int:
        n = len(arr)
        mod = 10**9 + 7
        left_bound = [-1] * n
        stack = []
        for i in range(n):
            while stack and arr[stack[-1]] > arr[i]:
                stack.pop()
            left_bound[i] = stack[-1] if stack else -1
            stack.append(i)
        
        stack = []
        right_bound = [n] * n
        for i in range(n-1, -1, -1):
            while stack and arr[stack[-1]] > arr[i]:
                stack.pop()
            right_bound[i] = stack[-1] if stack else n
            stack.append(i)
        
        res = 0
        for i in range(n):
            left_span = i - left_bound[i]
            right_span = right_bound[i] - i
            res = (res + arr[i] * left_span * right_span) % mod
        return res
