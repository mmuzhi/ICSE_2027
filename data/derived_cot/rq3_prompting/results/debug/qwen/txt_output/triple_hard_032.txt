### Approach
1. **Digit Grouping**: The solution involves grouping numbers by their digit length. For example, all 1-digit numbers come first, followed by all 2-digit numbers, and so on.
2. **Counting Non-decreasing Numbers**: For each digit length, count how many numbers in that group have digits in non-decreasing order. This is done using a helper function `fn(x)` that counts the numbers from `x` to `n` with the same number of digits as `x` and digits in non-decreasing order.
3. **Binary Search-like Adjustment**: The main loop adjusts the current number `x` and the count `k` to narrow down to the k-th number. If `k` is larger than the count of numbers in the current digit group, we move to the next digit group. Otherwise, we decrement `k` and adjust `x` within the current digit group.

### Solution Code
```python
class Solution:
    def findKthNumber(self, n: int, k: int) -> int:
        def fn(x):
            if x == 0:
                return 0
            s = str(x)
            d = len(s)
            low = 10**(d-1)
            high = min(n, 10**d - 1)
            if x < low:
                x = low
            if x > high:
                return 0
            
            def is_valid(num):
                s_num = str(num)
                for i in range(1, len(s_num)):
                    if s_num[i] < s_num[i-1]:
                        return False
                return True
            
            count = 0
            for num in range(x, high + 1):
                if is_valid(num):
                    count += 1
                    if count == k:
                        return num
            return x
        
        x = 1
        while k > 1:
            cnt = fn(x)
            if k > cnt:
                k -= cnt
                x += 1
            else:
                k -= 1
                x += 1
        return x
```
