```python
class Solution:
    def sortString(self, s: str) -> str:
        freq = {}
        letters = sorted(set(s))
        res = ""
        for i in s:
            if i in freq:
                freq[i] += 1
            else:
                freq[i] = 1
        
        while freq:
```

2. **Unique Letters**: Extract the unique characters from the string and sort them in ascending order.
3. **Processing Loop**: Use a while loop to continue processing until all characters are used up:
   - **First Pass (Increasing Order)**: Iterate over the sorted unique letters, appending each character to the result string and decrementing its frequency. If the frequency drops to zero, remove the character from the dictionary.
   - **Second Pass (Decreasing Order)**: Iterate over the reversed sorted unique letters, appending each character to the result string and decrementing its frequency. Again, remove the character if its frequency reaches zero.
4. **Termination**: The loop terminates when the dictionary is empty, indicating all characters have been processed.

### Solution Code
```python
class Solution:
    def sortString(self, s: str) -> str:
        freq = {}
        letters = sorted(set(s))
        res = ""
        for char in s:
            freq[char] = freq.get(char, 0) + 1
        
        while freq:
            for i in letters:
                if i in freq and freq[i] > 0:
                    res += i
                    freq[i] -= 1
                    if freq[i] == 0:
                        del freq[i]

                if i in freq and freq[i] > 0:
                    res += i
                    freq[i] -= 1
                    if freq[i] == 0:
                        del freq[i]
        return res
```
