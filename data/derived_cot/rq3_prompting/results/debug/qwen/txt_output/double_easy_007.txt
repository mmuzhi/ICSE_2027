=== RAW ANSWER ===

To solve this problem, we need to find the maximum distance between two balls of different colors in a list of balls. Each ball is represented by an integer color, and the balls are positioned from 0 to n-1 (left to right). The distance between two balls at indices i and j is given by |j - i|.

### Approach
The approach involves iterating through the list of balls and for each ball, checking if it is a different color from the first or last ball in the list. If it is, we calculate the distance from the current ball to the first or last ball and keep track of the maximum distance encountered. The key insight here is that the maximum distance between two balls of different colors will always be either the distance from the first ball to the last ball (if they are different) or the maximum distance between any ball and either the first or last ball (if they are the same).

1. **Initialization**: Start with `ans = 0` to keep track of the maximum distance found.
2. **Iteration**: For each ball at index `i` with color `x`, check if `x` is different from the first ball's color. If it is, update `ans` with the distance `i` (distance from the current ball to the first ball). Similarly, check if `x` is different from the last ball's color and update `ans` with the distance `n - 1 - i` (distance from the current ball to the last ball).
3. **Result**: After processing all balls, `ans` will hold the maximum distance between two balls of different colors.

### Solution Code
```python
from typing import List

class Solution:
    def maxDistance(self, colors: List[int]) -> int:
        ans = 0 
        for i, x in enumerate(colors): 
            if x != colors[0]: 
                ans = max(ans, i)
            if x != colors[-1]: 
                ans = max(ans, len(colors) - 1 - i)
        return ans
```

### Explanation
- **Initialization**: The variable `ans` is initialized to 0, which will store the maximum distance found.
- **Iteration**: For each ball at index `i`, the code checks if the ball's color is different from the first ball's color. If it is, the distance from this ball to the first ball (which is `i`) is considered. Similarly, if the ball's color is different from the last ball's color, the distance from this ball to the last ball (`len(colors) - 1 - i`) is considered. The maximum of these distances is stored in `ans`.
- **Result**: The loop continues until all balls are processed, and the final value of `ans` is returned, representing the maximum distance between two balls of different colors.

This approach efficiently checks all possible pairs of balls by leveraging the positions of the first and last balls, ensuring optimal performance with a time complexity of O(n), where n is the number of balls.
