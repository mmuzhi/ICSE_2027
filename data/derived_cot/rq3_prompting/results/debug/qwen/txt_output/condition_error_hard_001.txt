To solve this problem, we need to find all pairs of indices (i, j) such that the concatenation of words[i] and words[j] forms a palindrome. The solution involves efficiently checking all possible pairs by leveraging a dictionary to store reversed words for quick lookup.

### Approach
1. **Dictionary Construction**: We first build a dictionary where the keys are the reversed versions of each word in the input list, and the values are the indices of the original words. This allows us to quickly check if a reversed word exists in the list.
2. **Palindrome Check**: For each word in the list, we consider all possible ways to split it into a prefix and a suffix. 
   - **Case 1**: If the prefix is a palindrome and the suffix exists in the dictionary (meaning its reverse is in the list), we form a valid pair.
   - **Case 2**: If the suffix is a palindrome and the prefix exists in the dictionary (meaning its reverse is in the list), we form a valid pair.
3. **Avoiding Self-Pairs**: We ensure that we do not include pairs where the indices are the same unless explicitly required by the problem, but in this solution, we naturally avoid them by checking indices during pair formation.

### Solution Code
```python
class Solution:
    def palindromePairs(self, words: List[str]) -> List[List[int]]:
        # Build a dictionary that maps the reversed word to the list of indices
        dic = {}
        for i, word in enumerate(words):
            rev = word[::-1]
            if rev not in dic:
                dic[rev] = []
            dic[rev].append(i)
        
        res = []
        n = len(words)
        
        for i, word in enumerate(words):
            for j in range(len(word) + 1):
                prefix = word[:j]
                suffix = word[j:]
                
                # Case 1: prefix is a palindrome and suffix is in the dictionary
                if prefix == prefix[::-1]:
                    if suffix in dic:
                        for index in dic[suffix]:
                            if index != i:
                                res.append([index, i])
                # Case 2: suffix is a palindrome and prefix is in the dictionary
                if suffix == suffix[::-1]:
                    if prefix in dic:
                        for index in dic[prefix]:
                            if index != i:
                                res.append([i, index])
        
        return res
```

   - **Case 1**: If the prefix is a palindrome and the suffix is found in the dictionary, we append the pair [index of suffix, current index] to the result, ensuring the indices are distinct.
   - **Case 2**: If the suffix is a palindrome and the prefix is found in the dictionary, we append the pair [current index, index of prefix] to the result, again ensuring distinct indices.
