The solution involves using a two-pointer technique to efficiently track and place elements while ensuring no element appears more than twice.

3. **Two-Pointer Technique**: Iterate through the array starting from the third element. For each element, compare it with the element two positions before the current write pointer (`ans-2`). ### Solution Code
```python
class Solution:
    def removeDuplicates(self, nums: List[int]) -> int:
        n = len(nums)
        if n < 2:
            return n
        ans = 2
        for i in range(2, n):
            if nums[i] != nums[ans-2]:
                nums[ans] = nums[i]
                ans += 1
        return ans
```

1. **Edge Case Handling**: The function first checks if the array length is less than 2. If so, it returns the length immediately since no duplicates can exist.

3. **Iteration**: Starting from the third element (index 2), the function checks if the current element is different from the element two positions before the current write pointer (`ans-2`). If they differ, the current element is placed at the position `ans`, and `ans` is incremented.
4. **Result**: The final value of `ans` represents the new length of the array after removing duplicates, ensuring each element appears at most twice.
