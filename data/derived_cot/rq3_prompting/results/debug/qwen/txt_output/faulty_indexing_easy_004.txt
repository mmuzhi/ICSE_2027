=== RAW ANSWER ===

To solve this problem, we need to add 1 to a number represented as a list of its digits. The challenge lies in handling the carry-over when the digits are all 9s, which increases the number of digits in the result.

### Approach
1. **Convert the List to a String**: The list of digits is converted into a string to form a numerical string.
2. **Increment the Number**: The numerical string is converted to an integer, incremented by 1, and then converted back to a string.
3. **Convert Back to List**: The resulting string is split into individual characters, each converted back to an integer, forming the final list of digits.
4. **Return the Result**: The list of digits is returned, which now represents the original number plus one.

This approach efficiently handles all cases, including those where the number of digits increases due to carry-over (e.g., [9] becomes [1, 0]).

### Solution Code
```python
class Solution:
    def plusOne(self, digits: List[int]) -> List[int]:
        s = ''.join(map(str, digits))
        i = int(s) + 1
        li = list(map(int, str(i)))
        return li
```

### Explanation
1. **String Conversion**: The input list of digits is joined into a single string. For example, [1, 2, 3] becomes "123".
2. **Increment**: The string is converted to an integer and incremented by 1. For "123", this results in 124.
3. **Back to List**: The incremented integer is converted back to a string and then split into individual characters, each converted to an integer. For 124, this results in [1, 2, 4].
4. **Result**: The list of digits is returned, which now correctly represents the original number plus one. This approach efficiently handles all edge cases, including carry-over scenarios.
