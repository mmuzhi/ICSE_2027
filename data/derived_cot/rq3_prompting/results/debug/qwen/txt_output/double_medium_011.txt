    def getAncestors(self, n: int, edges: List[List[int]]) -> List[List[int]]:

To solve this problem, we need to determine all ancestors of each node in a directed acyclic graph (DAG). An ancestor of a node is any node that can reach it through directed edges. The solution involves traversing the graph in topological order to efficiently compute the ancestors for each node.

### Approach
1. **Graph Representation**: Represent the graph using an adjacency list where each node points to its neighbors.
2. **In-Degree Calculation**: Calculate the in-degree of each node, which is the number of edges pointing to it.
3. **Topological Sort (Kahn's Algorithm)**: Use a queue to process nodes starting from those with zero in-degree (sources). For each node processed, update the ancestor sets of its neighbors by adding the current node and all its ancestors.
4. **Ancestor Set Update**: For each neighbor of the current node, add the current node to its ancestor set and propagate all ancestors of the current node to the neighbor.
5. **Result Conversion**: After processing all nodes, convert each set of ancestors into a sorted list and return the result.

### Solution Code

### Explanation
1. **Graph Representation**: The graph is built using an adjacency list where each index represents a node, and the list at each index contains all nodes reachable directly from it.
2. **In-Degree Calculation**: The in-degree array keeps track of the number of incoming edges for each node, initialized to zero for all nodes.
3. **Topological Sort**: Nodes with zero in-degree are processed first. These nodes are added to a queue. For each node processed, its neighbors' in-degrees are decremented. If a neighbor's in-degree reaches zero, it is added to the queue.
4. **Ancestor Set Update**: As each node is processed, its ancestors are propagated to all its neighbors. The current node is added to the neighbor's ancestor set, and all ancestors of the current node are also added to the neighbor's set.
5. **Result Conversion**: After processing all nodes, each set of ancestors is converted into a sorted list to meet the output requirements. The final result is a list of lists where each inner list contains the sorted ancestors of the corresponding node. 

This approach efficiently computes all ancestors for each node in the DAG by leveraging topological sorting and propagating ancestor information through the graph. The complexity is linear with respect to the number of edges, making it suitable for large graphs.
