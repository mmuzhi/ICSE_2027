To solve this problem, we need to determine the number of ways to split a binary string into three non-empty parts such that each part contains exactly one-third of the total number of '1's in the string. If the total number of '1's is not divisible by 3, it's impossible to split the string as required, so we return 0. If there are no '1's, the solution involves counting the ways to split the string into three non-empty parts without any '1's.

### Approach
1. **Check for Zero '1's**: If the string contains no '1's, the number of ways to split the string is given by the combination formula \( \binom{N-1}{2} \), which is \((N-1) \times (N-2) // 2 \mod 10^9 + 7\).
2. **Check Divisibility by 3**: If the total number of '1's is not divisible by 3, return 0 immediately since it's impossible to split the string into three parts with equal '1's.
3. **Find Positions of '1's**: If the conditions are met, collect the positions of all '1's in the string.
4. **Determine Split Points**: The first split must occur between the \(c\)-th and \((c+1)\)-th '1', and the second split must occur between the \((2c+1)\)-th and \((2c+2)\)-th '1', where \(c\) is one-third of the total '1's. The number of valid splits for each part is derived from the positions of these '1's.
5. **Calculate Result**: Multiply the number of valid splits for the first and second parts to get the total number of ways to split the string, modulo \(10^9 + 7\).

### Solution Code
```python
class Solution:
    def numWays(self, s: str) -> int:
        MOD = 10**9 + 7
        N = len(s)
        c1 = s.count('1')
        if c1 == 0:
            return ((N-1)*(N-2)//2) % MOD
        if c1 % 3 != 0:
            return 0
        c = c1 // 3
        ones = []
        for i, char in enumerate(s):
            if char == '1':
                ones.append(i)
        a = ones[c-1]
        b = ones[c]
        d = ones[2*c]
        e = ones[2*c+1]
        return (b - a) * (e - d) % MOD
```
