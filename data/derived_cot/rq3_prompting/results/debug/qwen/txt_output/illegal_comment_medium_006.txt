To solve this problem, we need to count the number of 3x3 magic squares present inside a given grid. A 3x3 magic square is defined as a grid where the numbers 1 through 9 are used exactly once, and the sum of each row, column, and the two main diagonals is 15.

### Approach
1. **Grid Validation**: Iterate over all possible 3x3 subgrids within the given grid. The starting indices (i, j) must satisfy i ≤ M-3 and j ≤ N-3 to ensure the subgrid fits within the grid dimensions.
2. **Magic Square Check**: For each 3x3 subgrid:
   - **Check Distinct Numbers**: Ensure the subgrid contains all numbers from 1 to 9 exactly once.
   - **Check Row Sums**: Verify that the sum of each row is 15.
   - **Check Column Sums**: Verify that the sum of each column is 15.
   - **Check Diagonal Sums**: Verify that the sum of both main diagonals is 15.
3. **Count Valid Squares**: If all conditions are met, increment the count of magic squares found.

### Solution Code
```python
class Solution:
    def numMagicSquaresInside(self, grid: List[List[int]]) -> int:
        M, N = len(grid), len(grid[0])
        res = 0
        
        for i in range(M - 2):
            for j in range(N - 2):
                arr = []
                for k in range(3):
                    arr.append(tuple(grid[i + k][j:j + 3]))
                
                nums = set()
                for row in arr:
                    for num in row:
                        nums.add(num)
                if nums != set(range(1, 10)):
                    continue
                
                if sum(arr[0]) != 15 or sum(arr[1]) != 15 or sum(arr[2]) != 15:
                    continue
                
                if arr[0][0] + arr[1][1] + arr[2][2] != 15 or \
                   arr[0][2] + arr[1][1] + arr[2][0] != 15:
                    continue
                
                if arr[0][0] + arr[1][0] + arr[2][0] != 15 or \
                   arr[0][1] + arr[1][1] + arr[2][1] != 15 or \
                   arr[0][2] + arr[1][2] + arr[2][2] != 15:
                    continue
                
                res += 1
        
        return res
```

1. **Grid Validation**: The outer loops iterate over each possible starting position (i, j) for a 3x3 subgrid. The inner loop collects the three rows of the subgrid.
2. **Distinct Numbers Check**: The set `nums` collects all elements in the subgrid. If `nums` does not match the set {1, 2, ..., 9}, the subgrid is skipped.
3. **Row Sums Check**: The sum of each row must be 15. If any row sum is not 15, the subgrid is skipped.
4. **Column Sums Check**: The sum of each column must be 15. This is verified by summing elements at positions (i, j), (i+1, j), (i+2, j) for each column j.
5. **Diagonal Sums Check**: The sums of the two main diagonals must be 15. The first diagonal is from (i, j) to (i+2, j+2), and the second from (i, j+2) to (i+2, j).
6. **Result Calculation**: If all checks pass, the subgrid is a magic square, and the result counter is incremented.

This approach efficiently checks each possible 3x3 subgrid for the magic square properties, ensuring correctness while maintaining clarity and simplicity.
