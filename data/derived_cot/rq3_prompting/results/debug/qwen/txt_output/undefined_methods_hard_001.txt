3. **Cycle Detection**: Use a dictionary to track the start of potential cycles in the sequence of character matches. This helps in breaking the problem into a cycle part and a non-cycle part.

class Solution:
    def getMaxRepetitions(self, s1: str, n1: int, s2: str, n2: int) -> int:

                start = ptr + 1

                return (len(rec) - 2) // n2

        cycle1 = ct + 1 - cycleStart
