### Approach
1. **Count Frequencies**: Use a dictionary to count the occurrences of each number in the input list.
2. **Sort Distinct Numbers**: Extract the distinct numbers from the dictionary and sort them to facilitate binary search operations.
3. **Prefix Sum Array**: Compute a prefix sum array where each entry at index `i` represents the cumulative count of numbers less than or equal to the `i-th` distinct number.
4. **Binary Search**: For each number in the input list, use binary search to find the position where this number would be inserted in the sorted distinct numbers list. The count of numbers less than the current number is then derived from the prefix sum array at the position just before the insertion point.
