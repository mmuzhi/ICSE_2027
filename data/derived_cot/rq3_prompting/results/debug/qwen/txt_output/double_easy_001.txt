=== RAW ANSWER ===

To solve this problem, we need to determine if an array can be split into three contiguous non-empty parts such that the sum of each part is equal. The solution involves checking if the total sum of the array is divisible by 3 and then verifying if there exist two split points that divide the array into three parts each summing to one-third of the total sum.

### Approach
1. **Check Divisibility by 3**: The first step is to check if the total sum of the array is divisible by 3. If not, it's impossible to split the array into three parts with equal sums, so we return False.
2. **Minimum Array Length Check**: The array must have at least three elements to form three non-empty parts. If the array length is less than 3, return False.
3. **Track Sums and Count Parts**: Iterate through the array while accumulating the sum. Each time the accumulated sum equals one-third of the total sum, increment the count of parts found and reset the accumulator. If we find two such parts before reaching the second last element (ensuring the third part is non-empty), return True. If we finish the iteration without finding two parts, return False.

### Solution Code
```python
class Solution:
    def canThreePartsEqualSum(self, arr: List[int]) -> bool:
        total = sum(arr)
        if total % 3 != 0:
            return False
        each_sum = total // 3
        if len(arr) < 3:
            return False
        
        count = 0
        current_sum = 0
        for i in range(len(arr)-1):
            current_sum += arr[i]
            if current_sum == each_sum:
                count += 1
                current_sum = 0
            if count == 2:
                return True
        return False
```

### Explanation
1. **Divisibility Check**: The code first checks if the total sum of the array is divisible by 3. If not, it immediately returns False because equal division into three parts is impossible.
2. **Minimum Length Check**: The array must have at least three elements to form three non-empty parts. If not, the function returns False.
3. **Iterating Through the Array**: The code then iterates through the array (up to the second last element to ensure the third part is non-empty). During iteration, it accumulates the sum of elements. Whenever the accumulated sum equals one-third of the total sum, it increments the part count and resets the accumulator.
4. **Early Termination**: If two parts are found during the iteration, the function returns True immediately, as the third part will automatically sum to the required value due to the total sum being divisible by 3. If the loop completes without finding two parts, the function returns False.

This approach efficiently checks the feasibility of splitting the array into three parts with equal sums while ensuring all parts are non-empty.
