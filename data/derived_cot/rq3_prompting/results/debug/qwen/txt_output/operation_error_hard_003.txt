To solve this problem, we need to find all starting indices of substrings in a given string `s` that are a concatenation of each word in `words` exactly once and without any intervening characters. The solution involves efficiently checking possible substrings by leveraging a sliding window technique and a frequency dictionary to track word counts.

### Approach
1. **Problem Analysis**: The problem requires checking if any substring in `s` is formed by concatenating all words in `words` exactly once. The challenge is to do this efficiently without resorting to a brute-force approach, which would be computationally expensive.

2. **Intuition**: The solution involves using a sliding window approach where the window size is fixed to the total length of all words concatenated together. For each possible starting position of the window, we maintain a frequency dictionary of words within the window and compare it with the frequency dictionary of all words. If they match, the starting position is a valid solution.

3. **Algorithm Selection**: 
   - **Sliding Window**: The window slides by the length of one word at a time, adjusting the frequency dictionary by removing the leftmost word and adding the new rightmost word.
   - **Frequency Dictionary**: A dictionary (`occ`) keeps track of the frequency of each word in `words`. Another dictionary (`track`) maintains the current window's word frequencies. The helper function `test` checks if `track` matches `occ`.

4. **Optimization**: By initializing the window for each possible starting offset (0 to `wlen-1`), we reduce the number of checks needed. The window slides by one word at a time, updating the frequency dictionary efficiently.

5. **Complexity Analysis**: The algorithm runs in O(n * m) time, where `n` is the length of `s` and `m` is the number of words. This is efficient because each character in `s` is processed a constant number of times.

### Solution Code
```python
import collections

class Solution:
    def findSubstring(self, s: str, words: List[str]) -> List[int]:
        if not words:
            return []
        wlen = len(words[0])
        slen = wlen * len(words)
        if slen > len(s):
            return []
        
        occ = collections.Counter(words)
        res = []
        
        def test():
            for key in occ:
                if track[key] != occ[key]:
                    return False
            return True
        
        for k in range(wlen):
            if k + slen > len(s):
                continue
            track = {}
            for word in words:
                track[word] = 0
            
            for i in range(k, k + slen, wlen):
                word = s[i:i+wlen]
                if word in track:
                    track[word] += 1
            
            if test():
                res.append(k)
            
            for i in range(wlen + k, len(s) - slen + 1, wlen):
                leaving_word = s[i - wlen:i]
                entering_word = s[i + slen:i + slen + wlen]
                
                if leaving_word in track:
                    track[leaving_word] -= 1
                if entering_word in track:
                    track[entering_word] += 1
                
                if test():
                    res.append(i - wlen)
        return res
```
