=== RAW ANSWER ===

To solve this problem, we need to find all the possible ways to place `n` queens on an `n x n` chessboard such that no two queens attack each other. The solution involves using backtracking to explore all possible configurations while ensuring that no two queens share the same row, diagonal, or anti-diagonal.

### Approach
1. **Backtracking**: The algorithm explores all possible positions for queens column by column. For each column, it tries placing a queen in each row, ensuring that the placement does not conflict with any previously placed queens.
2. **Tracking Conflicts**: Three arrays are used to track conflicts:
   - `low` array tracks which rows are already occupied.
   - `upper` array tracks conflicts on the diagonal (slope 1).
   - `lower` array tracks conflicts on the anti-diagonal (slope -1).
3. **Base Case**: If all columns are filled (i.e., `col == n`), the current configuration is added to the result list.
4. **Recursive Backtracking**: For each valid placement, the algorithm recursively tries to fill the next column. If no valid placement is found, it backtracks by resetting the conflict markers.

### Solution Code
```python
class Solution:
    def totalNQueens(self, n: int) -> int:
        def addans(board, ans):
            temp = []
            for row in board:
                for j in range(len(row)):
                    if row[j] == "Q":
                        temp.append(j + 1)
            ans.append(temp)
        
        def solve(col, board, low, upper, lower, ans, n):
            if col == n:
                addans(board, ans)
                return
            for row in range(n):
                if low[row] == 0 and upper[row - col + n - 1] == 0 and lower[row + col] == 0:
                    board[row][col] = "Q"
                    low[row] = 1
                    upper[row - col + n - 1] = 1
                    lower[row + col] = 1
                    solve(col + 1, board, low, upper, lower, ans, n)
                    board[row][col] = "X"
                    low[row] = 0
                    upper[row - col + n - 1] = 0
                    lower[row + col] = 0
        
        ans = []
        board = [["X"] * n for _ in range(n)]
        low = [0] * n
        upper = [0] * (2 * n - 1)
        lower = [0] * (2 * n - 1)
        solve(0, board, low, upper, lower, ans, n)
        return len(ans)
```

### Explanation
1. **Initialization**: The `addans` function collects the positions of queens in the current configuration and appends them to the result list.
2. **Backtracking Function (`solve`)**:
   - **Base Case**: If all columns (`col == n`) are processed, the current configuration is added to the result.
   - **Recursive Case**: For each row in the current column, check if placing a queen is valid (no conflicts in row, diagonal, or anti-diagonal). If valid, place the queen, mark the conflicts, and proceed to the next column. After backtracking, reset the board and conflict markers.
3. **Main Function**:
   - Initialize the chessboard, conflict arrays (`low`, `upper`, `lower`), and start the backtracking from the first column (`col = 0`).
   - The result is the number of valid configurations found.

This approach efficiently explores all possible configurations using backtracking while ensuring no conflicts, leveraging auxiliary arrays to track valid positions dynamically.
