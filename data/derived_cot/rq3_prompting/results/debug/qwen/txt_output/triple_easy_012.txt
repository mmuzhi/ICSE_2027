=== RAW ANSWER ===

To solve this problem, we need to compute the absolute alternating sum of the digits of a given integer `n`. The alternating sum is defined such that digits at even indices (0-indexed) are added, and digits at odd indices are subtracted. The result is then taken as its absolute value.

### Approach
1. **Convert the integer to a string**: This allows us to process each digit individually.
2. **Handle negative numbers**: If the input number is negative, we first convert it to its absolute value to ensure we only process digits (ignoring the minus sign).
3. **Compute the alternating sum**: Iterate over each digit in the string representation of the number. For each digit at an even index, add its integer value to the sum. For each digit at an odd index, subtract its integer value from the sum.
4. **Return the absolute value**: The result of the alternating sum is taken as its absolute value to ensure the output is non-negative.

### Solution Code
```python
class Solution:
    def alternateDigitSum(self, n: int) -> int:
        s = str(abs(n))
        alt_sum = 0
        for i, char in enumerate(s):
            if i % 2 == 0:
                alt_sum += int(char)
            else:
                alt_sum -= int(char)
        return abs(alt_sum)
```

### Explanation
1. **Conversion to String**: The integer `n` is converted to a string using `str(abs(n))` to handle negative numbers by taking their absolute value first, ensuring only digits are processed.
2. **Alternating Sum Calculation**: The loop iterates over each character in the string. For each character at an even index, its integer value is added to `alt_sum`; for an odd index, its integer value is subtracted from `alt_sum`.
3. **Absolute Value**: The final result is the absolute value of `alt_sum`, ensuring the output is always non-negative, as required by the problem's definition of "absolute alternating sum".

This approach efficiently processes each digit of the number, alternates between addition and subtraction based on the digit's position, and ensures the result is correctly formatted as an absolute value.

class Solution:
    def alternateDigitSum(self, n: int) -> int:
        s = str(abs(n))
        alt_sum = 0
        for i, char in enumerate(s):
            if i % 2 == 0:
                alt_sum += int(char)
            else:
                alt_sum -= int(char)
        return abs(alt_sum)
