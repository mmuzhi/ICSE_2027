=== RAW ANSWER ===

To solve this problem, we need to find the maximum integer `k` in the given array such that the negative of `k` is not present in the array. The solution involves sorting the array in descending order and checking each element to see if its negative counterpart exists in the array. The first element (which is the largest due to descending order) that does not have its negative counterpart will be the answer. If no such element is found, we return -1.

### Approach
1. **Sort the Array**: By sorting the array in descending order, we ensure that we check the largest elements first. This allows us to return the first valid element encountered, which is the maximum possible `k`.
2. **Use a Set for Lookup**: Convert the array into a set for O(1) average-time complexity lookups to check if the negative of the current element exists in the array.
3. **Iterate and Check**: For each element in the sorted array, check if its negative is not present in the set. The first element that satisfies this condition is the answer. If no such element is found after checking all elements, return -1.

### Solution Code
```python
class Solution:
    def findMaxK(self, nums: List[int]) -> int:
        nums_sorted = sorted(nums, reverse=True)
        s = set(nums_sorted)
        for num in nums_sorted:
            if -num not in s:
                return num
        return -1
```

### Explanation
1. **Sorting the Array**: The array is sorted in descending order to prioritize checking larger elements first. This ensures that the first valid element found is the maximum possible `k`.
2. **Set Lookup**: By converting the array into a set, we enable efficient O(1) average-time complexity lookups to check if the negative of the current element exists in the array.
3. **Iteration and Check**: The loop iterates over each element in the sorted array. For each element, it checks if the negative of the element is not present in the set. If such an element is found, it is returned immediately as it is the maximum `k` satisfying the condition. If no such element is found after the loop completes, the function returns -1.

class Solution:
    def findMaxK(self, nums: List[int]) -> int:
        nums_sorted = sorted(nums, reverse=True)
        s = set(nums_sorted)
        for num in nums_sorted:
            if -num not in s:
                return num
        return -1
