To solve this problem, we need to calculate the total points from a series of operations performed on a stack of scores. The operations include doubling the last score, removing the last score, adding the last two scores, or pushing a new score onto the stack. The challenge is to correctly handle each operation and compute the final sum of all scores in the stack.

### Approach
1. **Initialize a stack**: We use a stack to keep track of the scores as we process each operation.
2. **Process each operation**: For each operation in the input list:
   - **'D'**: Double the last score in the stack and push the result onto the stack.
   - **'C'**: Remove the last score from the stack.
   - **'+'**: Add the last two scores in the stack and push the result onto the stack.
   - **Any other string**: Convert the string to an integer and push it onto the stack.
3. **Sum the stack**: After processing all operations, sum all the elements in the stack to get the total points.

### Solution Code
```python
class Solution:
    def calPoints(self, operations: List[str]) -> int:
        stack = []
        for op in operations:
            if op == 'D':
                stack.append(2 * stack[-1])
            elif op == 'C':
                stack.pop()
            elif op == '+':
                stack.append(stack[-1] + stack[-2])
            else:
                stack.append(int(op))
        return sum(stack)
```

### Explanation
- **Stack Initialization**: The stack starts empty and will store all the scores as they are processed.
- **Processing Operations**:
  - **'D'**: The last score is doubled and pushed onto the stack.
  - **'C'**: The last score is removed from the stack.
  - **'+'**: The sum of the last two scores is computed and pushed onto the stack.
  - **Other Strings**: The string is converted to an integer and pushed onto the stack.
- **Summing the Stack**: After processing all operations, the sum of all elements in the stack is returned, which gives the total points.

This approach efficiently handles each operation in sequence, ensuring the stack always reflects the current state of scores. The final sum is computed directly from the stack, leveraging the properties of the stack to maintain order and allow quick access to the necessary elements.
