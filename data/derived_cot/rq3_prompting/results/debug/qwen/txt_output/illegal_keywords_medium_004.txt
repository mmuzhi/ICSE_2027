=== RAW ANSWER ===

To solve this problem, we need to determine the minimum number of deletions required to make all elements in an array distinct. The solution involves counting the number of distinct elements in the array and then subtracting this count from the total number of elements to find the minimum deletions needed.

### Approach
1. **Problem Analysis**: The problem requires us to find the minimum number of deletions such that all remaining elements in the array are distinct. This can be efficiently solved by counting the number of unique elements in the array and subtracting this count from the total number of elements.
2. **Intuition**: By converting the array into a set (which inherently contains only unique elements), we can easily determine the number of distinct elements. The difference between the total elements and the distinct elements gives the minimum deletions required.
3. **Algorithm Selection**: The algorithm involves:
   - Converting the input array into a set to eliminate duplicates.
   - Calculating the number of distinct elements.
   - Subtracting the number of distinct elements from the total elements to get the result.
4. **Complexity Analysis**: The time complexity is O(n) where n is the number of elements in the array, primarily due to the set conversion. The space complexity is O(n) for storing the set.

### Solution Code
```python
from typing import List

class Solution:
    def minimumDeletions(self, nums: List[int]) -> int:
        distinct = len(set(nums))
        return len(nums) - distinct
```

### Explanation
- **Problem Analysis**: The problem is to find the minimum deletions needed to make all elements in the array unique.
- **Intuition**: The solution leverages the fact that the number of deletions required is simply the total elements minus the number of unique elements.
- **Algorithm Selection**: The algorithm uses a set to count unique elements efficiently. The result is derived by subtracting the size of the set from the original array length.
- **Complexity Analysis**: The operations involved (converting the array to a set and calculating lengths) are linear in time and space, making the solution efficient for the given problem constraints.
