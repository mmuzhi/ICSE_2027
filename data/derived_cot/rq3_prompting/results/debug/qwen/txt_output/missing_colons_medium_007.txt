=== RAW ANSWER ===

To solve this problem, we need to perform a level-order traversal of a binary tree from bottom to top. The level-order traversal (BFS) processes nodes level by level, starting from the root. The challenge is to reverse the order of the levels to achieve the bottom-up traversal.

### Approach
1. **Problem Analysis**: The task involves traversing the binary tree level by level and then reversing the order of the levels. The root node is at level 0, its children at level 1, and so on. The goal is to return the node values from the bottom-most level upwards.
2. **Intuition**: We can use a Breadth-First Search (BFS) approach to traverse the tree level by level. Each level's nodes are processed in order, and their values are stored in a list. After processing all levels, we reverse the list of levels to get the bottom-up order.
3. **Algorithm Selection**: 
   - **BFS**: We use a queue to keep track of nodes to visit. Starting from the root, we process each node, enqueue its children (if any), and record the node's value in the current level's list.
   - **Reversing Levels**: After collecting all levels from top to bottom, we reverse the list of levels to present them from bottom to top.
4. **Complexity Analysis**: 
   - **Time Complexity**: O(n), where n is the number of nodes in the tree. Each node is processed exactly once.
   - **Space Complexity**: O(n), required to store the levels and the queue during traversal.

### Solution Code
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def levelOrderBottom(self, root: Optional[TreeNode]) -> List[List[int]]:
        def checkReverse(ans):
            l, r = 0, len(ans)-1
            while l <= r:
                ans[l], ans[r] = ans[r], ans[l]
                l += 1
                r -= 1
            return ans
        
        if not root:
            return []
        q, ans = [root], []
        while q:
            n = len(q)
            level_list = []
            for i in range(n):
                curr = q.pop(0)
                level_list.append(curr.val)
                if curr.left:
                    q.append(curr.left)
                if curr.right:
                    q.append(curr.right)
            ans.append(level_list)
        return checkReverse(ans)
```

### Explanation
1. **Initialization**: The `checkReverse` function reverses the list of levels using two pointers (left and right) to swap elements in-place.
2. **Edge Case Handling**: If the root is `None`, return an empty list as there are no nodes to process.
3. **BFS Traversal**: 
   - Initialize a queue with the root node.
   - For each level, process all nodes in the queue, record their values in a temporary list (`level_list`), and enqueue their children.
   - Append the `level_list` to the result list (`ans`) after processing all nodes in the current level.
4. **Reversing Levels**: After BFS completes, the `checkReverse` function is called to reverse the order of levels in `ans`, providing the bottom-up level-order traversal.

This approach efficiently traverses the tree level by level and ensures the result is ordered from bottom to top by reversing the collected levels.
