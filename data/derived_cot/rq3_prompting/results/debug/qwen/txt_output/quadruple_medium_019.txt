=== RAW ANSWER ===

To solve this problem, we need to construct a quadtree from a given grid. The quadtree is a hierarchical data structure where each node can have up to four children, representing the four quadrants of the current node. The solution involves recursively dividing the grid into smaller quadrants until each leaf node represents a single cell in the grid.

### Approach
1. **Base Case Handling**: If the current grid being processed is a single cell (i.e., level 1), then the node is a leaf node with the value of that cell.
2. **Recursive Division**: For grids larger than 1x1, divide the grid into four quadrants (top-left, top-right, bottom-left, bottom-right) and recursively build the quadtree for each quadrant.
3. **Leaf Node Check**: If all four quadrants are leaf nodes and have the same value, the current node is also a leaf node with that value. Otherwise, the current node is an internal node with the four children.

### Solution Code
```python
class Solution:
    def construct(self, grid: List[List[int]]) -> 'Node':
        level = len(grid)
        root = None
        if level >= 1:
            root = self.buildTree(grid, 0, level, 0, level, level)
        return root
        
    def buildTree(self, grid, rs, re, cs, ce, level):
        if level == 1:
            return Node(grid[rs][cs], True, None, None, None, None)
        
        next_level = level // 2
        tl = self.buildTree(grid, rs, re - next_level, cs, ce - next_level, next_level)
        tr = self.buildTree(grid, rs, re - next_level, cs + next_level, ce, next_level)
        bl = self.buildTree(grid, re - next_level, re, cs, ce - next_level, next_level)
        br = self.buildTree(grid, re - next_level, re, cs + next_level, ce, next_level)
        
        if tl.isLeaf and tr.isLeaf and bl.isLeaf and br.isLeaf and tl.val == tr.val == bl.val == br.val:
            new_node = Node(tl.val, True, None, None, None, None)
        else:
            new_node = Node(tl.val, False, tl, tr, bl, br)
                
        return new_node
```

2. **Recursive Division**: For larger grids, the function divides the grid into four quadrants by adjusting the row and column start and end indices. Each quadrant is processed recursively to build its quadtree.
3. **Leaf Node Check**: After processing all four quadrants, the function checks if all are leaf nodes and have the same value. If true, the current node is a leaf node; otherwise, it is an internal node with the four children.
