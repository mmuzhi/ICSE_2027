        dir = ((), (0,1), (0,-1), (1,0), (-1,0))
        
        visited = [[False] * n for _ in range(m)]
        queue = deque()
        queue.append((0, 0))
        visited[0][0] = True
        cost = 0
        
        while queue:
            size = len(queue)
            for _ in range(size):
                x, y = queue.popleft()
                if x == m-1 and y == n-1:
                    return cost

                if d == 0:
                    continue

                    queue.append((nx, ny))

### Explanation
1. **Initialization**: The grid dimensions are checked, and a direction tuple is defined to map cell values to movement directions.
2. **BFS Setup**: A visited matrix tracks visited cells, and a queue starts from the initial position (0,0).
3. **BFS Execution**: For each level (cost increment), the algorithm processes all cells in the queue. If the destination is reached, the current cost is returned. Otherwise, the next cell is determined by the current cell's value and added to the queue if it's within bounds and unvisited.
4. **Termination**: The loop exits when the destination is found, returning the accumulated cost. If the destination is unreachable, the function returns the final cost after processing all levels.
