To solve this problem, we need to find the total number of solutions to the N-Queens problem, where N queens are placed on an NĂ—N chessboard such that no two queens attack each other. The solution involves using backtracking to explore all possible configurations of the board and count the valid configurations.

### Approach
1. **Backtracking with Pruning**: The approach uses backtracking to place queens column by column. For each column, it tries to place a queen in each row, ensuring that no two queens share the same row, or the same diagonal (both main and anti-diagonals).
2. **Tracking Occupied Rows and Diagonals**: Three arrays (`low`, `upper`, and `lower`) are used to keep track of occupied rows and diagonals:
   - `low` tracks rows (0 to N-1).
   - `upper` tracks the diagonals from top-right to bottom-left (indexed as `col - row + N - 1`).
   - `lower` tracks the diagonals from top-left to bottom-right (indexed as `row + col`).
3. **Base Case**: When all N columns are filled (i.e., `col == N`), the current configuration is added to the list of solutions.
4. **Recursive Backtracking**: For each valid placement of a queen, the function recursively attempts to fill the next column. After exploring all possibilities, the queen is removed to backtrack.

### Solution Code
```python
class Solution:
    def totalNQueens(self, n: int) -> int:
        def addans(board, ans):
            temp = []
            for row in board:
                for j in range(len(row)):
                    if row[j] == "Q":
                        temp.append(j + 1)
            ans.append(temp)
        
        def solve(col, board, low, upper, lower, ans, n):
            if col == n:
                addans(board, ans)
                return
            for row in range(n):
                if low[row] == 0 and upper[n - 1 + col - row] == 0 and lower[row + col] == 0:
                    board[row][col] = "Q"
                    low[row] = 1
                    upper[n - 1 + col - row] = 1
                    lower[row + col] = 1
                    solve(col + 1, board, low, upper, lower, ans, n)
                    low[row] = 0
                    upper[n - 1 + col - row] = 0
                    lower[row + col] = 0
        
        ans = []
        board = [["."] * n for _ in range(n)]
        low = [0] * n
        upper = [0] * (2 * n - 1)
        lower = [0] * (2 * n - 1)
        solve(0, board, low, upper, lower, ans, n)
        return len(ans)
```

### Explanation
1. **Initialization**: The `addans` function collects the column positions of queens in each row, converting them to 1-indexed values.
2. **Backtracking Function (`solve`)**: This function attempts to place a queen in the current column (`col`). For each row in the column, it checks if the row and both diagonals are free. If so, it places the queen and marks the row and diagonals as occupied.
3. **Base Case**: When all columns are filled (`col == n`), the current board configuration is added to the solutions list.
4. **Recursion and Backtracking**: The function recursively calls itself for the next column. After exploring all possibilities, it unmarks the row and diagonals to allow other configurations.
5. **Result**: The total number of valid configurations is returned by counting the solutions stored in `ans`.

This approach efficiently explores all valid configurations using backtracking with pruning, ensuring optimal performance for the N-Queens problem.
