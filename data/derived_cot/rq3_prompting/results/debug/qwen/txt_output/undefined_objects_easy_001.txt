### Approach

### Solution Code
```python
class Solution:
    def findComplement(self, num: int) -> int:
        if num == 0:
            return 0
        com = ''
        while num > 0:
            if num % 2 == 1:
                com += '0'
            else:
                com += '1'
            num = num // 2
        return int(com[::-1], 2)
```

### Explanation
1. **Initial Check**: If the input number is 0, the complement is trivially 0, so we return 0 immediately.
2. **Bit Processing**: For non-zero numbers, we enter a loop that processes each bit from the least significant bit (LSB) to the most significant bit (MSB). 
3. **Bit Flipping**: For each bit, if it is 1, we append '0' to the result string; if it is 0, we append '1'. This effectively flips each bit.
4. **String Reversal**: After processing all bits, the result string is reversed to convert it from LSB-first to MSB-first order.
5. **Conversion**: The reversed string is then converted from a binary string to an integer using base 2.

This approach efficiently computes the bitwise complement by leveraging bit manipulation and string operations, ensuring correctness for all non-negative integers.
