### Approach

2. **Iterate through Rod Indices**: The rod indices are located at odd indices (1, 3, 5, etc.) in the string. For each rod index encountered, we check if it has been processed before.
3. **Collect Colors for Each Rod**: For each unprocessed rod, we scan the entire string to collect all colors attached to that rod. Colors are found at the positions immediately preceding each rod index.
4. **Check for All Three Colors**: If the collected colors for a rod include all three (R, G, B), we increment our count.
5. **Return the Result**: After processing all rods, the count holds the number of rods with all three colors.

### Solution Code
```python
class Solution:
    def countPoints(self, rings: str) -> int:
        lst = set()
        count = 0
```
