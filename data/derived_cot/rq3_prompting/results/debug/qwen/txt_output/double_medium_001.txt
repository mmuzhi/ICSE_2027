To solve this problem, we need to determine if a given string containing parentheses '(', ')', and wildcards '*' is valid. The wildcards can be treated as any of the three characters: '(', ')', or an empty string. The solution involves using two stacks to keep track of the positions of open parentheses and wildcards, and then ensuring that all open parentheses can be matched with closing parentheses or wildcards.

### Approach
1. **Stack Initialization**: Use two stacks, `st` for open parentheses '(' and `sr` for wildcards '*'.
2. **Traversal**: Iterate through each character in the string:
   - For '(', push its index onto `st`.
   - For '*', push its index onto `sr`.
   - For ')', first try to match with the top of `st` (if available). If not, try to match with the top of `sr` (if available). If neither is available, the string is invalid.
3. **Final Matching**: After processing all characters, use the remaining wildcards in `sr` to match any remaining open parentheses in `st`. If there are not enough wildcards, the string is invalid.

### Solution Code
```python
class Solution:
    def checkValidString(self, s):
        st = []
        sr = []
        for i in range(len(s)):
            if s[i]=='(':
                st.append(i)
            elif s[i]== '*':
                sr.append(i)
            else:
                if st:
                    st.pop()
                elif sr:
                    sr.pop()
```

3. **Final Check**: After processing all characters, any remaining open parentheses in `st` must be matched with wildcards in `sr`. If there are not enough wildcards, the string is invalid. The function returns `True` if all parentheses and wildcards are properly matched, otherwise `False`.
