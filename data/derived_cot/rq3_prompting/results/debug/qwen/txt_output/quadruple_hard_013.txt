=== RAW ANSWER ===

To solve this problem, we need to find the median of two sorted arrays efficiently. The median is the middle value in an ordered list of numbers. If the combined length of the two arrays is even, the median is the average of the two middle numbers.

### Approach
1. **Merge the Arrays**: We merge the two sorted arrays into a single sorted array. This is done using two pointers to traverse both arrays simultaneously, comparing elements and appending the smaller one to the merged array.
2. **Handle Remaining Elements**: After one of the arrays is exhausted, the remaining elements of the other array are appended to the merged array.
3. **Calculate the Median**: The median is then determined based on the length of the merged array. If the length is odd, the median is the middle element. If the length is even, the median is the average of the two middle elements.

### Solution Code
```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        p1 = 0
        p2 = 0
        new = []
        
        while p1 < len(nums1) and p2 < len(nums2):
            if nums1[p1] < nums2[p2]:
                new.append(nums1[p1])
                p1 += 1
            else:
                new.append(nums2[p2])
                p2 += 1
        
        while p1 < len(nums1):
            new.append(nums1[p1])
            p1 += 1
        
        while p2 < len(nums2):
            new.append(nums2[p2])
            p2 += 1
        
        n = len(new)
        if n % 2 == 0:
            index = n // 2
            median = (new[index] + new[index-1]) / 2
        else:
            median = float(new[n//2])
        return median
```

### Explanation
1. **Merging Arrays**: The first loop merges elements from both arrays by comparing elements at the current pointers (`p1` and `p2`). The smaller element is added to the merged array (`new`), and the corresponding pointer is incremented.
2. **Appending Remaining Elements**: After one array is fully traversed, the remaining elements of the other array are appended to the merged array.
3. **Finding Median**: The median is calculated by checking the length of the merged array. If the length is even, the median is the average of the two middle elements. If the length is odd, the median is the middle element. The result is returned as a float to ensure correct division.

This approach efficiently merges the two sorted arrays and calculates the median in linear time, O(m + n), where m and n are the lengths of the two arrays. The space complexity is O(m + n) to store the merged array.
