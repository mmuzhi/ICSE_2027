4. **Complexity Analysis**: The time complexity is O(n^2) in the worst case, where `n` is the length of the string. This is because for each starting index, we might process up to `n` characters. The space complexity is O(n) due to memoization storage.

```python
from functools import lru_cache

class Solution:
    def numberOfArrays(self, s: str, k: int) -> int:
        n = len(s)
        mod = 10**9 + 7

        def rec(i=0):

                if int(curr) > k:
```
