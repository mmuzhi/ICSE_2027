To solve this problem, we need to find the minimum number of obstacles to remove so that there is a path from the top-left corner (0, 0) to the bottom-right corner (m-1, n-1) of a grid. The grid consists of cells that are either free (0) or blocked by obstacles (1). Removing an obstacle allows movement through that cell, and the goal is to minimize the total number of obstacles removed.

### Approach
The problem can be modeled as a shortest path problem where each cell's cost is the number of obstacles it contains (0 or 1). The solution involves using Dijkstra's algorithm to find the path with the minimum total cost from the start to the destination. 

1. **Initialization**: We initialize a distance matrix with infinity for all cells except the starting cell (0, 0), which is set to the value of the starting cell (grid[0][0]).
2. **Priority Queue**: We use a priority queue (min-heap) to explore cells in order of increasing cost. The queue starts with the initial cell (0, 0) and its cost.
3. **Processing Cells**: For each cell processed, we check its four neighbors. For each neighbor, if moving to it results in a lower total cost, we update the neighbor's distance and push it into the priority queue.
4. **Termination**: The algorithm terminates when the destination cell (m-1, n-1) is reached, returning its distance value, which represents the minimum obstacles removed.

import heapq
from typing import List

class Solution:
    def minimumObstacles(self, grid: List[List[int]]) -> int:
