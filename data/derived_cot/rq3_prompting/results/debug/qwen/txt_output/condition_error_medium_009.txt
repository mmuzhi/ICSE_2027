=== RAW ANSWER ===

To solve this problem, we need to generate all numbers between a given range [low, high] that consist of sequential digits. Sequential digits are numbers where each digit is exactly one more than the previous digit, such as 12, 123, etc. 

### Approach
1. **Problem Analysis**: The task involves generating numbers where each digit follows the previous digit by exactly one. For example, starting from a digit 1, the next number would be 12, then 123, and so on until the digits reach 9. The challenge is to efficiently generate all such numbers within a specified range [low, high].

2. **Intuition**: A Breadth-First Search (BFS) approach is suitable here because it allows us to explore all possible numbers level by level. Starting from each digit from 1 to 9, we can generate subsequent numbers by appending the next digit (current last digit + 1) until we cannot (when the last digit is 9).

3. **Algorithm Selection**: We use a queue to manage the BFS process. The initial queue contains digits 1 through 9. For each number dequeued, we check if it lies within the range [low, high]. If it does, we add it to the result list. Then, we check if the last digit of the number is less than 9; if so, we generate the next number by appending the next digit and enqueue it.

4. **Complexity Analysis**: The time complexity is O(N), where N is the number of sequential digits within the range [low, high]. The space complexity is also O(N) due to the queue storing intermediate numbers.

### Solution Code
```python
from collections import deque

class Solution:
    def sequentialDigits(self, low: int, high: int) -> List[int]:
        out = []
        queue = deque(range(1, 10))
        while queue:
            elem = queue.popleft()
            if low <= elem <= high:
                out.append(elem)
            last_digit = elem % 10
            if last_digit < 9:
                next_num = elem * 10 + last_digit + 1
                queue.append(next_num)
        return out
```

### Explanation
1. **Initialization**: The queue starts with digits 1 through 9. These are the base cases from which we will build larger sequential numbers.
2. **Processing**: For each number dequeued, we check if it falls within the specified range [low, high]. If it does, we add it to the result list.
3. **Generating Next Numbers**: For each number, we determine its last digit. If this digit is less than 9, we generate the next sequential number by appending the next digit (last digit + 1) and enqueue it. This ensures we only generate valid sequential numbers.
4. **Termination**: The loop continues until the queue is empty, ensuring all possible sequential numbers within the range are generated and collected.

This approach efficiently explores all possible sequential numbers using BFS, ensuring correctness and optimal performance.
