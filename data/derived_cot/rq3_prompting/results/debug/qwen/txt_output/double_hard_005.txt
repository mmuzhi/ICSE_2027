=== RAW ANSWER ===

To solve this problem, we need to find the second greater element for each element in the given array. The second greater element for an element at index `i` is defined as the first element to the right of `i` that is greater than the element at `i` and has at least one element between them that is also greater than the element at `i`.

### Approach
1. **First Greater Element (NGE) Calculation**: We first compute the first greater element (NGE) for each element in the array. This is done by iterating from right to left and using a stack to keep track of elements' indices. For each element, we pop elements from the stack that are less than or equal to the current element, then the top of the stack gives the NGE.
2. **Second Greater Element Calculation**: Using the NGE results, we then compute the second greater element for each element. We iterate from right to left again, using another stack to keep track of the NGE values. For each element, we push its NGE value into the stack and then find the smallest value in the stack that is greater than the current element, which becomes the second greater element.

```python
                ans[i] = stack2[-1][0]
            else:
                ans[i] = -1
        
        return ans
```

### Explanation
1. **First Pass (NGE Calculation)**: We iterate from right to left through the array. For each element, we use a stack to keep track of indices of elements for which we are yet to find the first greater element. By popping elements from the stack that are less than or equal to the current element, we ensure that the top of the stack contains the first greater element for the current element.
2. **Second Pass (Second Greater Element Calculation)**: Using the results from the first pass, we again iterate from right to left. For each element, if it has a first greater element, we push its value into a second stack. We then find the smallest value in this stack that is greater than the current element, which becomes the second greater element. If no such value exists, we leave the answer as -1.

This approach efficiently computes the second greater element for each element in the array by leveraging two passes with stack operations, ensuring optimal performance and correctness.

class Solution:
    def secondGreaterElement(self, nums: List[int]) -> List[int]:
        n = len(nums)
        ans = [-1] * n
        nge1 = [-1] * n
        stack = []
        for i in range(n-1, -1, -1):
            while stack and nums[stack[-1]] <= nums[i]:
                stack.pop()
            if stack:
                nge1[i] = nums[stack[-1]]
            else:
                nge1[i] = -1
            stack.append(i)
        
        stack2 = []
        for i in range(n-1, -1, -1):
            if nge1[i] != -1:
                stack2.append((nge1[i], nums[i]))
            while stack2 and stack2[-1][0] <= nums[i]:
                stack2.pop()
            if stack2:
                ans[i] = stack2[-1][0]
            else:
                ans[i] = -1
