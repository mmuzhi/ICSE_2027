- The algorithm iterates `i` from right to left, maintaining a window `word[i:j]`.
- `isForbidden` checks if any prefix of `word[i:j]` matches a forbidden string (since extending leftward, only new forbidden strings starting at `i` need checking).
- If a forbidden prefix of length `truc` is found, `j = i + truc - 1` shrinks the window to exclude it.
- When `c not in t`, no forbidden string can match the current prefix, so we must `return False` immediately.
