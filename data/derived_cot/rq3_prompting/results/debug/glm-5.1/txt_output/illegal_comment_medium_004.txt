**Bugs fixed:**

1. **`IndexError` when stack is empty after pop**: The original code accesses `stack[-1]` after popping, which crashes when the stack becomes empty. Fixed by using `left = stack[-1] if stack else -1` (treating the virtual left boundary as index `-1`).

2. **Unprocessed elements remaining in stack**: After the main loop, elements still in the stack have no smaller element to their right. A second `while` loop processes them with `n` (array length) as the right boundary, consistent with the monotonic stack approach.
