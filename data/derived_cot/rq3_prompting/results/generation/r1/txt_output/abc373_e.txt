You will be given a competitive programming problem.
An election is being held with N candidates numbered 1, 2, \ldots, N. There are K votes, some of which have been counted so far.
Up until now, candidate i has received A_i votes.
After all ballots are counted, candidate i (1 \leq i \leq N) will be elected if and only if the number of candidates who have received more votes than them is less than M.  There may be multiple candidates elected.
For each candidate, find the minimum number of additional votes they need from the remaining ballots to guarantee their victory regardless of how the other candidates receive votes.
Formally, solve the following problem for each i = 1,2,\ldots,N.
Determine if there is a non-negative integer X not exceeding K - \displaystyle{\sum_{i=1}^{N}} A_i satisfying the following condition.  If it exists, find the minimum possible such integer.

- If candidate i receives X additional votes, then candidate i will always be elected.

Input

The input is given from Standard Input in the following format:
N M K
A_1 A_2 \ldots A_N

Output

Let C_i be the minimum number of additional votes candidate i needs from the remaining ballots to guarantee their victory regardless of how other candidates receive votes. Print C_1, C_2, \ldots, C_N separated by spaces.
If candidate i has already secured their victory, then let C_i = 0. If candidate i cannot secure their victory under any circumstances, then let C_i = -1.

Constraints


- 1 \leq M \leq N \leq 2 \times 10^5
- 1 \leq K \leq 10^{12}
- 0 \leq A_i \leq 10^{12}
- \displaystyle{\sum_{i=1}^{N} A_i} \leq K
- All input values are integers.

3 1 4 1 5

Sample Output 1

- Candidate 1 can secure their victory by obtaining 2 more votes, while not by obtaining 1 more vote.  Thus, C_1 = 2.
- Candidate 2 can never (even if they obtain 2 more votes) secure their victory, so C_2 = -1.

import sys
from bisect import bisect_left

def solve():
    data = sys.stdin.read().strip().split()
    if not data:
        return
    it = iter(data)
    N = int(next(it))
    M = int(next(it))
    K = int(next(it))
    A = [int(next(it)) for _ in range(N)]
    total = sum(A)
    R = K - total
    # If M >= N, all always elected
    if M >= N:
        print(' '.join('0' for _ in range(N)))
        return
    # Build sorted descending list with indices
    pairs = sorted([(A[i], i) for i in range(N)], reverse=True)
    # Determine top M indices
    is_topM = [False] * N
    for idx in range(M):
        val, i = pairs[idx]
        is_topM[i] = True
    # Extract top M and top M+1 values
    topM_vals = [v for v,_ in pairs[:M]]
    topM1_vals = [v for v,_ in pairs[:M+1]]
    # Sort ascending
    topM_asc = sorted(topM_vals)
    topM1_asc = sorted(topM1_vals)
    # Prefix sums
    prefM = [0]
    for v in topM_asc:
        prefM.append(prefM[-1] + v)
    prefM1 = [0]
    for v in topM1_asc:
        prefM1.append(prefM1[-1] + v)
    
    ans = []
    # For each candidate
    for i in range(N):
        Ai = A[i]
        if is_topM[i]:
            asc = topM1_asc
            pref = prefM1
            own_val = Ai
        else:
            asc = topM_asc
            pref = prefM
            own_val = None  # not needed
        
        # Function to check if x works
        def check(x):
            T = Ai + x + 1
            cnt_full = bisect_left(asc, T)
            sum_full = pref[cnt_full]
            if is_topM[i]:
                if own_val < T:
                    cnt_full -= 1
                    sum_full -= own_val
            # Now cnt_full is number of top M others with A_j < T, sum_full sum of their A_j
            # g = cnt_full * T - sum_full + x  (since g = cnt*T - sum + x)
            # But we have T = Ai+x+1, so g = cnt_full*(Ai+x+1) - sum_full + x
            g = cnt_full * (Ai + x + 1) - sum_full + x
            return g > R
        
        # Determine if possible at max x
        if not check(R):
            ans.append(-1)
            continue
        # If 0 works
        if check(0):
            ans.append(0)
            continue
        # Binary search
        lo, hi = 0, R
        while lo < hi:
            mid = (lo + hi) // 2
            if check(mid):
                hi = mid
            else:
                lo = mid + 1
        # lo is the answer
        ans.append(lo)
    print(' '.join(map(str, ans)))

if __name__ == "__main__":
    solve()

import sys
from bisect import bisect_left

def solve() -> None:
    data = sys.stdin.buffer.read().split()
    if not data:
        return
    it = iter(data)
    N = int(next(it))
    M = int(next(it))
    K = int(next(it))
    A = [int(next(it)) for _ in range(N)]

    total = sum(A)
    R = K - total

    # If every candidate is automatically elected
    if M >= N:
        sys.stdout.write(' '.join('0' for _ in range(N)))
        return

    # Sort candidates by votes descending
    pairs = sorted(((A[i], i) for i in range(N)), reverse=True)
    # Mark the M candidates with highest A
    topM_idx = [False] * N
    for pos in range(M):
        _, idx = pairs[pos]
        topM_idx[idx] = True

    # Values of the top M and top M+1 candidates
    topM_vals = [v for v, _ in pairs[:M]]
    topM1_vals = [v for v, _ in pairs[:M+1]]

    # Work with ascending order for binary search
    topM_asc = sorted(topM_vals)
    topM1_asc = sorted(topM1_vals)

    # Prefix sums
    prefM = [0]
    for v in topM_asc:
        prefM.append(prefM[-1] + v)
    prefM1 = [0]
    for v in topM1_asc:
        prefM1.append(prefM1[-1] + v)

    out = []
    for i in range(N):
        Ai = A[i]
        if topM_idx[i]:
            asc = topM1_asc
            pref = prefM1
            own = Ai
        else:
            asc = topM_asc
            pref = prefM
            own = None   # not used

        # Helper to evaluate the condition for a given x
        def condition(x: int) -> bool:
            T = Ai + x + 1
            cnt = bisect_left(asc, T)
            s = pref[cnt]
            if topM_idx[i] and own < T:
                cnt -= 1
                s -= own
            # g(x) = cnt * (Ai+x+1) - s + x
            return cnt * (Ai + x + 1) - s + x > R

        # Impossible even with all remaining votes
        if not condition(R):
            out.append(-1)
            continue
        # Already secure
        if condition(0):
            out.append(0)
            continue

        # Binary search for the minimal x
        lo, hi = 0, R
        while lo < hi:
            mid = (lo + hi) // 2
            if condition(mid):
                hi = mid
            else:
                lo = mid + 1
        out.append(lo)

    sys.stdout.write(' '.join(map(str, out)))
