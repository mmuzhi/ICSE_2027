=== ANSWER ===
```python
from typing import List

class Solution:
    def transformArray(self, nums: List[int]) -> List[int]:
        # Replace each number by 0 if even, 1 if odd
        transformed = [0 if x % 2 == 0 else 1 for x in nums]
        # Sort in non-decreasing order
        transformed.sort()
        return transformed
```
