You will be given a competitive programming problem.
Takahashi has 10^{100} flower pots. Initially, he is not growing any plants.
You are given Q queries to process in order.
There are three types of queries as follows.

- 1: Prepare one empty flower pot and put a plant in it. Here, the plant's height is 0.
- 2 T: Wait for T days. During this time, the height of every existing plants increases by T.
- 3 H: Harvest all plants with a height of at least H, and output the number of plants harvested. The harvested plants are removed from their flower pots.

Assume that performing queries of the first and third types takes zero time.

Input

The input is given from Standard Input in the following format:

Let there be K queries of the third type, and print K lines.
The i-th line (1 \leq i \leq K) should contain the answer to the i-th query of type 3.

Constraints


- 1 \leq Q \leq 2 \times 10^{5}
- 1 \leq T,H \leq 10^{9}
- There is at least one query of the third type.
- All input values are integers.

So harvesting can be done by counting plants with start <= threshold. And we remove them. So we need a data structure that supports:

- Insert a value (the start time) when a new plant is added.
- Query and remove all entries with value <= some threshold (which is cum - H). Count them.

This is like a multiset where we can pop all elements <= X efficiently. Since we only need to pop them, we can use a priority queue (min-heap) where we maintain the start times. When we need to harvest, we pop from the heap while the smallest start time <= threshold, count them, and output count. This works because we only remove all plants that meet the condition, and they are removed permanently. Since each plant is added once and removed at most once, total operations O(Q log Q). import sys
import heapq

def solve():
    data = sys.stdin.read().split()
    it = iter(data)
    Q = int(next(it))
    cum = 0
    heap = []
    out = []
    for _ in range(Q):
        typ = next(it)
        if typ == '1':
            heapq.heappush(heap, cum)
        elif typ == '2':
            T = int(next(it))
            cum += T
        else: # typ == '3'
            H = int(next(it))
            threshold = cum - H
            cnt = 0
            while heap and heap[0] <= threshold:
                heapq.heappop(heap)
                cnt += 1
            out.append(str(cnt))
    sys.stdout.write("\n".join(out))

=== ANSWER ===
```python
import sys
import heapq

def solve() -> None:
    data = sys.stdin.read().split()
    it = iter(data)
    Q = int(next(it))

    cum = 0               # total growth applied so far
    heap = []             # min-heap of start times of current plants
    answers = []

    for _ in range(Q):
        typ = next(it)
        if typ == '1':
            heapq.heappush(heap, cum)
        elif typ == '2':
            T = int(next(it))
            cum += T
        else:             # typ == '3'
            H = int(next(it))
            threshold = cum - H
            cnt = 0
            while heap and heap[0] <= threshold:
                heapq.heappop(heap)
                cnt += 1
            answers.append(str(cnt))

    sys.stdout.write("\n".join(answers))

if __name__ == "__main__":
    solve()
```
