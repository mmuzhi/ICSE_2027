- For every vertex v with a degree of at least 1, the total XOR of the numbers written on its adjacent vertices (excluding v itself) is 0.


What is XOR?

The XOR of two non-negative integers A and B, denoted as A \oplus B, is defined as follows:


- In the binary representation of A \oplus B, the bit at position 2^k \, (k \geq 0) is 1 if and only if exactly one of the bits at position 2^k in the binary representations of A and B is 1. Otherwise, it is 0.


For example, 3 \oplus 5 = 6 (in binary: 011 \oplus 101 = 110).

In general, the bitwise XOR of k integers p_1, \dots, p_k is defined as (\cdots ((p_1 \oplus p_2) \oplus p_3) \oplus \cdots \oplus p_k).  It can be proved that this is independent of the order of p_1, \dots, p_k.

        rows[v] ^= 1 << u

    # Gaussian elimination to reduced row echelon form (over GF(2))
    pivot_col = [-1] * N      # which row is pivot for each column
    rank = 0
    for col in range(N):
        # find a row with a 1 in this column, starting from rank
        sel = -1
        for i in range(rank, N):
            if (rows[i] >> col) & 1:
                sel = i
                break
        if sel == -1:
            continue
        # swap current row with found row
        rows[rank], rows[sel] = rows[sel], rows[rank]
        # eliminate this column from all other rows
        for i in range(N):
            if i != rank and ((rows[i] >> col) & 1):
                rows[i] ^= rows[rank]
        pivot_col[col] = rank
        rank += 1

    # free columns (no pivot)
    free_mask = 0
    for col in range(N):
        if pivot_col[col] == -1:
            free_mask |= 1 << col

    # check if any vertex is forced to 0 in all solutions
    for col in range(N):
        if pivot_col[col] != -1:
            r = pivot_col[col]
            if (rows[r] & free_mask) == 0:
                print("No")
                return

    # For each vertex v, build a nullspace vector with a 1 in coordinate v
    w_list = [0] * N      # each entry is an N‑bit mask
    for v in range(N):
        if pivot_col[v] == -1:                # v is a free variable
            free_vars = 1 << v
            pivots = 0
            for p in range(N):
                if pivot_col[p] != -1:
                    r = pivot_col[p]
                    if (rows[r] >> v) & 1:
                        pivots |= 1 << p
            w = free_vars | pivots
            w_list[v] = w
        else:                                 # v is a pivot variable
            r = pivot_col[v]
            row_v = rows[r]
            # find a free column f with coefficient 1 in this row
            f = None
            free_cols = free_mask
            while free_cols:
                lsb = free_cols & -free_cols
                fbit = (lsb.bit_length() - 1)
                if (row_v >> fbit) & 1:
                    f = fbit
                    break
                free_cols ^= lsb
            # such f exists because forced-zero check passed
            free_vars = 1 << f
            pivots = 0
            for p in range(N):
                if pivot_col[p] != -1:
                    rp = pivot_col[p]
                    if (rows[rp] >> f) & 1:
                        pivots |= 1 << p
            w = free_vars | pivots
            w_list[v] = w

    # Build the final numbers X_i
    X = [0] * N
    for i in range(N):
        xi = 0
        for v in range(N):
            if (w_list[v] >> i) & 1:
                xi |= (1 << v)
        X[i] = xi

    print("Yes")
    print(' '.join(str(x) for x in X))

if __name__ == "__main__":
    solve()
