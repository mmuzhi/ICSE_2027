You will be given a competitive programming problem.
There are N foods, each containing exactly one of vitamins 1, 2, and 3.
Specifically, eating the i-th food gives you A_i units of vitamin V_i, and C_i calories.
Takahashi can choose any subset of these N foods as long as the total calorie consumption does not exceed X.
Find the maximum possible value of this: the minimum intake among vitamins 1, 2, and 3.

Input

The input is given from Standard Input in the following format:
N X
V_1 A_1 C_1
V_2 A_2 C_2
\vdots
V_N A_N C_N

Output

Print the maximum possible value of "the minimum intake among vitamins 1, 2, and 3" when the total calories consumed is at most X.

Constraints


- 1 \leq N \leq 5000
- 1 \leq X \leq 5000
- 1 \leq V_i \leq 3
- 1 \leq A_i \leq 2 \times 10^5
- 1 \leq C_i \leq X
- All input values are integers.

Sample Input 1

5 25
1 8 5
2 3 5
2 7 10
3 2 5
3 3 10

Sample Output 1

3

Each food provides the following if eaten:

- 1st food: 8 units of vitamin 1, and 5 calories
- 2nd food: 3 units of vitamin 2, and 5 calories
- 3rd food: 7 units of vitamin 2, and 10 calories
- 4th food: 2 units of vitamin 3, and 5 calories
- 5th food: 3 units of vitamin 3, and 10 calories

Eating the 1st, 2nd, 4th, and 5th foods gives 8 units of vitamin 1, 3 units of vitamin 2, 5 units of vitamin 3, and 25 calories.
In this case, the minimum among the three vitamin intakes is 3 (vitamin 2).
It is impossible to get 4 or more units of each vitamin without exceeding 25 calories, so the answer is 3.

2 200000 1

Sample Output 2

import sys

def solve():

    if not input_data: return
    it = iter(input_data)
    N = int(next(it))
    X = int(next(it))
    groups = [[] for _ in range(3)]

        V = int(next(it))
        A = int(next(it))
        C = int(next(it))

            if dp[i] > cur:

            best[i] = cur

```python
import sys
import bisect

def solve() -> None:
    data = sys.stdin.read().strip().split()
    if not data:
        return
    it = iter(data)
    N = int(next(it))
    X = int(next(it))

    # group foods by vitamin (1-indexed -> 0,1,2)
    groups = [[] for _ in range(3)]
    for _ in range(N):
        v = int(next(it))
        a = int(next(it))
        c = int(next(it))
        groups[v - 1].append((a, c))

    best_per_vitamin = []   # list of best[c] arrays for each vitamin

    for items in groups:
        # knapsack: dp[w] = max intake with exact weight w
        dp = [-1] * (X + 1)
        dp[0] = 0
        for a, c in items:
            for w in range(X, c - 1, -1):
                if dp[w - c] != -1:
                    val = dp[w - c] + a
                    if val > dp[w]:
                        dp[w] = val
        # prefix max: best[w] = max intake with at most w calories
        best = [0] * (X + 1)
        cur = 0
        for i in range(X + 1):
            cur = max(cur, dp[i])
            best[i] = cur
        best_per_vitamin.append(best)

    # upper bound: the best we could possibly get for each vitamin alone
    hi = min(best_per_vitamin[i][X] for i in range(3))
    lo = 0

    while lo < hi:
        mid = (lo + hi + 1) // 2
        total_cal = 0
        feasible = True
        for best in best_per_vitamin:
            # smallest c such that best[c] >= mid
            c = bisect.bisect_left(best, mid)
            if c > X:          # should not happen because mid <= hi <= best[X]
                feasible = False
                break
            total_cal += c
        if feasible and total_cal <= X:
            lo = mid
        else:
            hi = mid - 1

    print(lo)

if __name__ == "__main__":
    solve()
```
