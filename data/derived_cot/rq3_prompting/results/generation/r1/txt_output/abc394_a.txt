=== ANSWER ===
```python
import sys

def main() -> None:
    S = sys.stdin.readline().strip()
    # Filter only '2' characters
    result = ''.join(ch for ch in S if ch == '2')
    print(result)

if __name__ == "__main__":
    main()
```
