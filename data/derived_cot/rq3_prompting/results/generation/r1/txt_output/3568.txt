The key of num1, num2, and num3 is defined as a four-digit number such that:

The i^th digit (1 <= i <= 4) of the key is generated by taking the smallest digit among the i^th digits of num1, num2, and num3.

Return the key of the three numbers without leading zeros (if any).
 
Example 1:

Input: num1 = 1, num2 = 10, num3 = 1000
Output: 0
Explanation:

The 1^st digit of the key is min(0, 0, 1).
The 2^nd digit of the key is min(0, 0, 0).
The 3^rd digit of the key is min(0, 1, 0).
The 4^th digit of the key is min(1, 0, 0).

Hence, the key is "0000", i.e. 0.

Output: 777

=== ANSWER ===
```python
class Solution:
    def generateKey(self, num1: int, num2: int, num3: int) -> int:
        # Pad each number to 4 digits with leading zeros
        s1 = str(num1).zfill(4)
        s2 = str(num2).zfill(4)
        s3 = str(num3).zfill(4)
        
        # Build key by taking minimum digit at each position
        key_digits = []
        for i in range(4):
            d = min(int(s1[i]), int(s2[i]), int(s3[i]))
            key_digits.append(str(d))
        
        key_str = ''.join(key_digits)
        return int(key_str)
```
