=== ANSWER ===
```python
class Solution:
    def reverseDegree(self, s: str) -> int:
        total = 0
        for i, ch in enumerate(s):
            # 1-indexed position
            pos = i + 1
            # reversed alphabet index: 'a' -> 26, 'b' -> 25, ..., 'z' -> 1
            rev_idx = 26 - (ord(ch) - ord('a'))
            total += rev_idx * pos
        return total
```
