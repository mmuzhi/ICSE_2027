Let's step back and think about the problem more abstractly. For two binary matrices A and B with same row and column sums, they are "similar". The fixed elements are those that are invariant across all similar matrices. So essentially, we want to know, for a given matrix A, what entries are determined solely by its row and column sums. This is similar to the concept of "structural zeros/ones" in contingency tables. There is a known characterization using the concept of "convexity" of the polytope: For any cell, its value is not determined if and only if there exists a 2×2 submatrix with a specific pattern (alternating) that allows swapping.
