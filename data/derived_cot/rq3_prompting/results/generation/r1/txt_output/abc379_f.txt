You will be given a competitive programming problem.
There are N buildings, building 1, building 2, \ldots, building N, arranged in this order in a straight line from west to east. Building 1 is the westernmost, and building N is the easternmost. The height of building i\ (1\leq i\leq N) is H_i.
For a pair of integers (i,j)\ (1\leq i\lt j\leq N), building j can be seen from building i if the following condition is satisfied.

- There is no building taller than building j between buildings i and j. In other words, there is no integer k\ (i\lt k\lt j) such that H_k > H_j.

You are given Q queries. In the i-th query, given a pair of integers (l_i,r_i)\ (l_i\lt r_i), find the number of buildings to the east of building r_i (that is, buildings r_i + 1, r_i + 2, \ldots, N) that can be seen from both buildings l_i and r_i.

Input

The input is given from Standard Input in the following format:
N Q
H_1 H_2 \ldots H_N
l_1 r_1
l_2 r_2
\vdots
l_Q r_Q

Output

Constraints


- 2 \leq N \leq 2 \times 10^5
- 1 \leq Q \leq 2 \times 10^5
- 1 \leq H_i \leq N
- H_i\neq H_j\ (i\neq j)
- 1 \leq l_i < r_i \leq N
- All input values are integers.

import sys

def solve() -> None:
    input = sys.stdin.readline
    N, Q = map(int, input().split())
    H = [0] + list(map(int, input().split()))          # 1‑indexed

    # next greater element to the right (N+1 if none)
    nxt = [0] * (N + 2)
    stack = []
    for i in range(N, 0, -1):
        while stack and H[stack[-1]] < H[i]:
            stack.pop()
        nxt[i] = stack[-1] if stack else N + 1
        stack.append(i)

    M = N + 1                      # dummy root
    parent = [0] * (M + 1)
    for i in range(1, N + 1):
        parent[i] = nxt[i]
    parent[M] = M                  # root points to itself

    # depth = number of real buildings from this node to the root (excluding root)
    depth = [0] * (M + 1)
    for i in range(N, 0, -1):
        depth[i] = depth[parent[i]] + 1

    # binary lifting table
    LOG = (M).bit_length() + 1
    up = [[0] * (M + 1) for _ in range(LOG)]
    for i in range(1, M + 1):
        up[0][i] = parent[i]
    for k in range(1, LOG):
        for i in range(1, M + 1):
            up[k][i] = up[k-1][up[k-1][i]]

    def lca(u: int, v: int) -> int:
        if depth[u] < depth[v]:
            u, v = v, u
        # lift u to depth of v
        diff = depth[u] - depth[v]
        bit = 0
        while diff:
            if diff & 1:
                u = up[bit][u]
            diff >>= 1
            bit += 1
        if u == v:
            return u
        for k in range(LOG - 1, -1, -1):
            if up[k][u] != up[k][v]:
                u = up[k][u]
                v = up[k][v]
        return up[0][u]

    out = []
    for _ in range(Q):
        l, r = map(int, input().split())
        s1 = l + 1
        s2 = r + 1
        anc = lca(s1, s2)
        out.append(str(depth[anc]))
    sys.stdout.write("\n".join(out))
