=== ANSWER ===
```python
class Solution:
    def canAliceWin(self, n: int) -> bool:
        turn = 0          # 0 for Alice, 1 for Bob
        remove = 10       # stones to remove on current turn

        while n >= remove:
            n -= remove
            remove -= 1
            turn = 1 - turn

        # If it's Bob's turn and he cannot move, Alice wins; otherwise Alice loses
        return turn == 1
```
