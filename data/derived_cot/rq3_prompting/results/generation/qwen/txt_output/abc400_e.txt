To solve this problem, we need to find the largest "400 number" not exceeding a given integer \( A \) for multiple queries. A 400 number is defined as a positive integer \( N \) that has exactly two distinct prime factors, and for each prime factor \( p \) of \( N \), \( p \) divides \( N \) an even number of times. 

### Approach
1. **Understanding 400 Numbers**: A 400 number \( N \) must be of the form \( M^2 \), where \( M \) is a number with exactly two distinct prime factors. This means \( N \) is the square of a number that is a product of two distinct primes, each raised to any positive integer exponent. For example, \( 400 = 2^4 \times 5^2 \) is a 400 number because \( 400 = (2^2 \times 5)^2 \).

2. **Precomputation**: 
   - **Smallest Prime Factor (SPF) Sieve**: We precompute the smallest prime factor for every number up to \( 10^6 \) (since \( \sqrt{10^{12}} = 10^6 \)) using the Sieve of Eratosthenes. This helps in efficiently factorizing numbers later.
   - **Distinct Prime Factors Count (DP Array)**: Using the SPF array, we compute an array `dp` where `dp[i]` is the number of distinct prime factors of `i`. This is done by dynamic programming: if `i` is a prime, `dp[i] = 1`; otherwise, `dp[i] = dp[i // spf[i]] + 1` if the smallest prime factor of `i // spf[i]` is different from `spf[i]`.

3. **Two-Prime Numbers List**: We create a list of all numbers up to \( 10^6 \) that have exactly two distinct prime factors (i.e., `dp[i] == 2`). This list is sorted for efficient querying.

4. **Query Processing**: For each query, compute the integer square root \( X \) of \( A \). Use binary search to find the largest number \( M \leq X \) in the precomputed list of two-prime numbers. The answer is \( M^2 \).

### Solution Code
```python
import math
from bisect import bisect_right
import sys

def main():
    MAX = 1000000
    spf = list(range(MAX + 1))
    for i in range(2, int(math.isqrt(MAX)) + 1):
        if spf[i] == i:
            for j in range(i * i, MAX + 1, i):
                if spf[j] == j:
                    spf[j] = i

    dp = [0] * (MAX + 1)
    dp[1] = 0
    for i in range(2, MAX + 1):
        if spf[i] == i:
            dp[i] = 1
        else:
            p = spf[i]
            i0 = i // p
            if spf[i0] == p:
                dp[i] = dp[i0]
            else:
                dp[i] = dp[i0] + 1

    two_prime_list = [i for i in range(2, MAX + 1) if dp[i] == 2]
    two_prime_list.sort()

    data = sys.stdin.read().split()
    q = int(data[0])
    index = 1
    results = []
    for _ in range(q):
        A = int(data[index])
        index += 1
        X = math.isqrt(A)
        pos = bisect_right(two_prime_list, X) - 1
        if pos < 0:
            candidate = 36
        else:
            candidate = two_prime_list[pos]
        results.append(str(candidate * candidate))
    
    sys.stdout.write("\n".join(results))

if __name__ == "__main__":
    main()
```
