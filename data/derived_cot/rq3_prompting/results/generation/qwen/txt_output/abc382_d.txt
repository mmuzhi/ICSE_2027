-  (S_1, S_2, \ldots, S_{i-1}) = (T_1, T_2, \ldots, T_{i-1})

To solve this problem, we need to generate all integer sequences of length N that satisfy specific conditions, sorted in lexicographical order. The conditions are: each element must be at least 1, each subsequent element must be at least 10 more than the previous one, and the last element must not exceed M.

### Approach
1. **Problem Analysis**: The problem requires generating sequences where each element after the first is at least 10 more than the previous element, and the entire sequence must end at or below M. The sequences must be listed in lexicographical order.
2. **Key Insight**: The sequences can be constructed by starting with the first element (x1) ranging from 1 to M - 10*(N-1). For each x1, the subsequent elements are determined by adding increments (gaps) that are at least 10 each. The gaps must be such that the sum of the gaps plus the base value (10*(N-1)) does not exceed M - x1.
3. **Algorithm Selection**: We use a recursive generator function to explore all possible gap sequences (non-negative integers) that sum up to a value not exceeding the allowed maximum (T = M - x1 - 10*(N-1)). The sequences are built by iteratively adding these gaps to the initial element, ensuring the conditions are met.
4. **Complexity Analysis**: The number of sequences is determined by combinatorial mathematics, specifically the number of ways to distribute gaps under the constraints. The solution efficiently generates these sequences using recursion and checks each sequence against the conditions.

### Solution Code
```python
import sys
sys.setrecursionlimit(10000)

def iterate_gaps_lex(depth, max_sum):
    if depth == 0:
        yield []
        return
    for first in range(0, max_sum + 1):
        for tail in iterate_gaps_lex(depth - 1, max_sum - first):
            yield [first] + tail

def main():
    data = sys.stdin.read().split()
    if not data:
        return
    N = int(data[0])
    M = int(data[1])
    base = 10 * (N - 1)
    sequences = []
    for x1 in range(1, M - base + 1):
        T = M - x1 - base
        if T < 0:
            break
        for gap_seq in iterate_gaps_lex(N - 1, T):
            seq = [x1]
            for i in range(1, N):
                add_val = 10 + (gap_seq[i - 1] if i - 1 < len(gap_seq) else 0)
                next_val = seq[-1] + add_val
                seq.append(next_val)
            sequences.append(seq)
    
    print(len(sequences))
    for seq in sequences:
        print(" ".join(map(str, seq)))

if __name__ == '__main__':
    main()
```
