=== ANSWER ===

To solve this problem, we need to convert a given date in the Gregorian calendar format (yyyy-mm-dd) into a binary representation where each component (year, month, day) is converted to its binary form without leading zeros. The solution involves parsing the input string, converting each numeric component to binary, and then combining them back into the required format.

### Approach
1. **Parse the Input String**: The input string is split into three parts (year, month, day) using the '-' delimiter.
2. **Convert to Integers**: Each part of the split string is converted from a string to an integer.
3. **Binary Conversion**: Each integer is then converted to its binary representation. The built-in Python function `bin()` is used for this purpose, and the '0b' prefix is removed from the binary string.
4. **Reconstruct the Result**: The binary strings for the year, month, and day are combined into the final result string in the same yyyy-mm-dd format but with binary digits.

### Solution Code
```python
class Solution:
    def convertDateToBinary(self, date: str) -> str:
        parts = date.split('-')
        year_bin = bin(int(parts[0]))[2:]
        month_bin = bin(int(parts[1]))[2:]
        day_bin = bin(int(parts[2]))[2:]
        return f"{year_bin}-{month_bin}-{day_bin}"
```

### Explanation
1. **Parsing the Input**: The input string is split into three components (year, month, day) using the `split('-')` method. This results in a list where the first element is the year, the second is the month, and the third is the day.
2. **Conversion to Integers**: Each component from the split list is converted from a string to an integer using `int()`. This is necessary because the `bin()` function requires integer input.
3. **Binary Conversion**: The `bin()` function converts each integer to its binary string representation. The `[2:]` slice is used to remove the '0b' prefix that `bin()` adds to the result.
4. **Result Construction**: The binary strings for the year, month, and day are formatted back into a single string with hyphens separating them, matching the original input format but with binary digits.

This approach efficiently handles the conversion by leveraging Python's built-in functions for simplicity and clarity, ensuring correctness for all valid dates within the specified range.
