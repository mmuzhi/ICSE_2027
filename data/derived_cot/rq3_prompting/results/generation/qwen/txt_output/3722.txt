To solve this problem, we need to find the maximum sum of `k` non-overlapping subarrays in an integer array `nums`, where each subarray has a length of at least `m`. The solution involves dynamic programming to efficiently compute the maximum sum by considering all possible subarrays and their combinations.

### Approach
1. **Problem Analysis**: The problem requires selecting `k` non-overlapping subarrays, each of length at least `m`, such that their combined sum is maximized. The challenge is to efficiently explore all possible subarrays without resorting to a brute-force approach, which would be computationally expensive.

2. **Dynamic Programming Setup**: We use a dynamic programming approach where `dp[j][i]` represents the maximum sum of `j` non-overlapping subarrays ending at index `i` in the array. The solution involves iterating over the array and for each position, considering all possible subarrays ending at that position with length at least `m`.

3. **Prefix Sum Array**: To efficiently compute the sum of any subarray, we precompute a prefix sum array. This allows us to calculate the sum of any subarray in constant time.

4. **Sliding Window for Optimization**: For each position `i` in the array, we maintain a sliding window that tracks the best sums from previous subarrays. This helps in reducing the time complexity by avoiding recalculations for each subarray.

5. **Combining Results**: After processing all positions for each subarray count (from 1 to `k`), the solution is derived from the maximum value obtained when exactly `k` subarrays are considered.

### Solution Code
```python
class Solution:
    def maxSum(self, nums: List[int], k: int, m: int) -> int:
        n = len(nums)
        if n == 0:
            return 0
        prefix = [0] * n
        prefix[0] = nums[0]
        for i in range(1, n):
            prefix[i] = prefix[i-1] + nums[i]
        if k == 0:
            return 0
        
        NEG_INF = -10**15
        dp_prev = [0] * (n + 1)
        
        for j in range(1, k + 1):
            B = [0] * (n + 1)
            bestB = [0] * (n + 1)
            if j == 1:
                B[0] = 0
                bestB[0] = 0
                for t_index in range(1, n + 1):
                    B[t_index] = dp_prev[t_index] - (prefix[t_index - 1] if t_index - 1 >= 0 else 0)
                    bestB[t_index] = max(bestB[t_index - 1], B[t_index])
            else:
                B[0] = NEG_INF
                bestB[0] = NEG_INF
                for t_index in range(1, n + 1):
                    B[t_index] = dp_prev[t_index] - (prefix[t_index - 1] if t_index - 1 >= 0 else 0)
                    bestB[t_index] = max(bestB[t_index - 1], B[t_index])
            
            dp_curr = [NEG_INF] * n
            for i in range(n):
                if i < m - 1:
                    dp_curr[i] = NEG_INF
                else:
                    t_index_max = i - m + 1
                    if t_index_max < 0:
                        candidate_val = NEG_INF
                    else:
                        if j == 1:
                            candidate_val = prefix[i] + bestB[t_index_max]
                        else:
                            candidate_val = prefix[i] + bestB[t_index_max]
                    if i == 0:
                        dp_curr[i] = candidate_val
                    else:
                        dp_curr[i] = max(dp_curr[i - 1], candidate_val)
            dp_prev = [0] * (n + 1)
            for i in range(n):
                dp_prev[i + 1] = dp_curr[i]
            dp_prev[0] = 0
        
        return max(dp_curr) if n > 0 else 0
```

### Explanation
1. **Prefix Sum Calculation**: The prefix sum array is computed to allow efficient sum calculation of any subarray in constant time.
2. **Dynamic Programming Initialization**: The `dp_prev` array is initialized to store the best sums for the previous count of subarrays (initially zero subarrays).
3. **Iterating Over Subarray Counts**: For each count `j` from 1 to `k`, the algorithm computes the best possible sums for subarrays ending at each position in the array. This involves:
   - **Sliding Window (`B` and `bestB` arrays)**: These arrays help in efficiently tracking the best sums from previous subarrays by maintaining a window of relevant indices.
   - **Updating Current State (`dp_curr`)**: For each position, the algorithm checks if a subarray of sufficient length can end at that position and updates the current state accordingly.
4. **Result Extraction**: After processing all subarray counts, the maximum value in the `dp_curr` array (which now contains the best sums for `k` subarrays) is returned as the solution.

This approach efficiently narrows down the problem using dynamic programming and sliding window techniques, ensuring optimal performance even for larger input sizes.
