To solve this problem, we need to compute the inversion number of a permutation after performing a series of operations. Each operation involves a left-to-right pass through the permutation, swapping adjacent elements if they are out of order. The challenge is to efficiently compute the inversion number after each operation in the sequence, especially given the constraints where both the permutation length and the number of operations can be as large as 200,000.

### Approach
1. **Initial Inversion Count Calculation**: The initial inversion count of the permutation is computed using a straightforward double-loop approach. This is feasible for small input sizes but would be inefficient for larger ones.
2. **Operation Simulation**: For each operation in the sequence, simulate the left-to-right pass. During each pass, swap adjacent elements if they are out of order (i.e., the left element is greater than the right element).
3. **Inversion Count Update**: After each operation, recompute the inversion count by counting all pairs (i, j) where i < j and the element at position i is greater than the element at position j. This involves another double-loop, which is efficient enough for small input sizes but not for larger ones.

The solution provided here is optimized for small input sizes (N and M up to 200). For larger inputs, a more sophisticated data structure like a Fenwick tree or segment tree would be necessary to handle the inversion counts efficiently.

### Solution Code
```python
import sys

def main():
    data = sys.stdin.read().strip().split()
    if not data:
        return
    
    it = iter(data)
    N = int(next(it))
    P = [int(next(it)) for _ in range(N)]
    M = int(next(it))
    A = [int(next(it)) for _ in range(M)]
    
    arr = P[:]
    results = []
    
    for k in A:
        for i in range(k-1):
            if arr[i] > arr[i+1]:
                arr[i], arr[i+1] = arr[i+1], arr[i]
                
        inv_count = 0
        for i in range(N):
            for j in range(i+1, N):
                if arr[i] > arr[j]:
                    inv_count += 1
                    
        results.append(str(inv_count))
    
    print("\n".join(results))

if __name__ == '__main__':
    main()
```
