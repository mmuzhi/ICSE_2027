=== ANSWER ===

To solve this problem, we need to determine whether a given 6-digit number satisfies specific conditions regarding the counts of its digits. The conditions are that the digit '1' must appear exactly once, the digit '2' exactly twice, and the digit '3' exactly three times. 

### Approach
1. **Problem Analysis**: The problem requires checking the frequency of three specific digits ('1', '2', and '3') in a 6-digit number. The total number of digits is fixed at 6, and the sum of the required counts (1 + 2 + 3) is exactly 6. This means the number must consist solely of these three digits with no other digits present.
2. **Intuition**: By counting the occurrences of '1', '2', and '3' in the number, we can directly verify if they meet the specified counts. If any other digits are present, the counts will not sum to 6, making the conditions impossible to satisfy.
3. **Algorithm Selection**: The solution involves converting the input number into a string and using the `count` method to tally the occurrences of '1', '2', and '3'. If these counts match the required values (1, 2, and 3 respectively), the answer is "Yes"; otherwise, it is "No".

### Solution Code
```python
def main():
    s = input().strip()
    if s.count('1') == 1 and s.count('2') == 2 and s.count('3') == 3:
        print("Yes")
    else:
        print("No")

if __name__ == '__main__':
    main()
```

### Explanation
1. **Reading Input**: The input is read as a string to facilitate digit counting.
2. **Counting Digits**: The `count` method is used to determine how many times each of the digits '1', '2', and '3' appears in the string.
3. **Condition Check**: The solution checks if the counts of these digits are exactly 1, 2, and 3, respectively. If all conditions are met, it prints "Yes"; otherwise, it prints "No".
4. **Efficiency**: The approach efficiently checks the conditions in linear time relative to the number of digits (6 digits in this case), making it optimal for the given constraints. The solution leverages simple string operations and direct comparison, ensuring clarity and correctness.
