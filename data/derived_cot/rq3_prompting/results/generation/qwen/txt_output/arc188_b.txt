To solve this problem, we need to determine whether all points on a circle can be colored black by Alice and Bob, who take turns coloring points while adhering to symmetry conditions after each move. The solution involves analyzing the symmetry conditions and the fixed points around the axes defined by Alice's and Bob's starting positions.

### Approach
1. **Problem Analysis**: The problem involves a circle with N equally spaced points. Alice starts at point 0, and Bob at point K. They alternate turns, coloring one white point black each time, ensuring the coloring remains symmetric with respect to the line connecting the operator (Alice or Bob) to the circle's center. The goal is to check if all points can be colored black under optimal cooperative play.

2. **Key Insight**: The solution hinges on the observation that the fixed points (points invariant under reflection over the operator's axis) determine the game's outcome. Specifically, if the sets of fixed points for Alice's axis (0) and Bob's axis (K) are identical, it is impossible to color all points; otherwise, it is possible.

3. **Mathematical Formulation**: The fixed points for an axis are solutions to the equation \(2x \equiv 0 \mod N\) for Alice's axis (0) and \(2x \equiv 2K \mod N\) for Bob's axis (K). The number of solutions (fixed points) is given by the greatest common divisor (gcd) of 2 and N. The sets of fixed points are equal if K is divisible by \(N \div \text{gcd}(2, N)\).

4. **Algorithm Selection**: For each test case, compute the gcd of 2 and N. The period is \(N \div \text{gcd}(2, N)\). If K is divisible by this period, output "No"; otherwise, output "Yes".

### Solution Code
```python
import math
import sys

def main():
    data = sys.stdin.read().split()
    t = int(data[0])
    index = 1
    results = []
    for _ in range(t):
        N = int(data[index])
        K = int(data[index + 1])
        index += 2
        d = math.gcd(2, N)
        period = N // d
        if K % period == 0:
            results.append("No")
        else:
            results.append("Yes")
```
