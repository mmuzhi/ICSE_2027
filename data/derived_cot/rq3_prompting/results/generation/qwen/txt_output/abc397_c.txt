To solve this problem, we need to find the maximum sum of the counts of distinct integers in two non-empty contiguous subarrays obtained by splitting the given array at any single position. The solution involves efficiently computing the distinct counts for all possible prefixes and suffixes of the array and then combining these counts to find the optimal split.

### Approach
1. **Prefix Distinct Counts Calculation**: We traverse the array from left to right, maintaining a frequency array to keep track of the distinct elements encountered so far. For each position, we update the frequency array and record the number of distinct elements up to that point in a prefix array.
2. **Suffix Distinct Counts Calculation**: Similarly, we traverse the array from right to left, updating a frequency array to track distinct elements encountered. For each position, we record the number of distinct elements from that position to the end of the array in a suffix array.
3. **Combining Prefix and Suffix Counts**: For each possible split position, we combine the corresponding prefix and suffix distinct counts. The maximum value obtained from these combinations is our answer.

This approach efficiently computes the required values in linear time, making it suitable for large input sizes as specified in the problem constraints.

### Solution Code
```python
import sys

def main():
    data = sys.stdin.read().split()
    n = int(data[0])
    A = list(map(int, data[1:1+n]))
    
    freq1 = [0] * (n + 1)
    prefix_distinct = [0] * (n + 1)
    distinct_count = 0
    for i in range(n):
        num = A[i]
        if freq1[num] == 0:
            distinct_count += 1
        freq1[num] += 1
        prefix_distinct[i + 1] = distinct_count
        
    freq2 = [0] * (n + 1)
    suffix_distinct = [0] * n
    distinct_count = 0
    for i in range(n - 1, -1, -1):
        num = A[i]
        if freq2[num] == 0:
            distinct_count += 1
        freq2[num] += 1
        suffix_distinct[i] = distinct_count
        
    max_sum = 0
    for i in range(1, n):
        total = prefix_distinct[i] + suffix_distinct[i]
        if total > max_sum:
            max_sum = total
            
    print(max_sum)

if __name__ == "__main__":
    main()
```

3. **Suffix Calculation**: The array is traversed from right to left. Another frequency array (`freq2`) keeps track of distinct elements encountered. The `suffix_distinct` array records the distinct elements from each position to the end of the array.
