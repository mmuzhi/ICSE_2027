To solve this problem, we need to determine whether it's possible to transform sequences A and B such that, after performing any number of operations (filling missing values and rearranging A), all elements in A and B are non-negative and the sum of corresponding elements in A and B is the same for every pair.

### Approach
1. **Problem Analysis**: The problem involves two sequences, A and B, with some elements possibly missing (marked as -1). We can fill these missing elements with any non-negative integers and rearrange sequence A in any order. The goal is to ensure that after these operations, all corresponding elements in A and B sum to the same value S for every pair.

2. **Key Insights**:
   - **Fixed Pairs**: If there are pairs where both elements are not -1, their sums must be the same. If they differ, it's impossible to satisfy the condition.
   - **S0 Determination**: If there are fixed pairs, S0 is their common sum. Otherwise, S0 is the maximum of the largest elements in A and B (or 0 if no elements are present).
   - **Non-Negative Check**: All fixed elements in A and B must be ≤ S0 to ensure the missing elements can be filled appropriately.
   - **Missing Elements Calculation**: For each fixed element in B, we check if the required complement (S0 - B) exists in A. If not, we need to fill missing elements in A to cover the deficit.

3. **Algorithm Selection**:
   - **Case 1**: At least one fixed pair exists. Verify all fixed pairs have the same sum. Then, ensure all fixed elements ≤ S0. Calculate the number of missing elements needed in A to cover the complements of fixed B elements.
   - **Case 2**: No fixed pairs exist. Determine S0 as the maximum of the largest elements in A and B (or 0). Ensure all fixed elements ≤ S0. Calculate the number of missing elements needed in A similarly.

4. **Complexity Analysis**: The algorithm processes each element once to collect fixed values and sums, then uses a counter for frequency checks. The overall complexity is O(N), which is efficient for the given constraints (N ≤ 2000).

import sys
from collections import Counter

def main():
    data = sys.stdin.read().strip().split()
    if not data:
        print("No")
        return

    n = int(data[0])
    A = list(map(int, data[1:1+n]))
    B = list(map(int, data[1+n:1+2*n]))

    fixed_sums = []
    fixed_A_all = []
    fixed_B_all = []

    for i in range(n):
        a = A[i]
        b = B[i]
        if a != -1 and b != -1:
            fixed_sums.append(a + b)
            fixed_A_all.append(a)
            fixed_B_all.append(b)
        else:
            if a != -1:
                fixed_A_all.append(a)
            if b != -1:
                fixed_B_all.append(b)

    if fixed_sums:
        if len(set(fixed_sums)) > 1:
            print("No")
            return
        S0 = fixed_sums[0]
    else:
        S0 = None

    fixed_A = fixed_A_all
    fixed_B = fixed_B_all
    total_missing_A = n - len(fixed_A)
    total_missing_B = n - len(fixed_B)

    if S0 is not None:
        for a in fixed_A:
            if a > S0:
                print("No")
                return
        for b in fixed_B:
            if b > S0:
                print("No")
                return

        count_fixed_A = Counter(fixed_A)
        count_fixed_B = Counter(fixed_B)
        missing_A_needed = 0
        for b_val, count_b in count_fixed_B.items():
            needed_val = S0 - b_val
            count_needed = count_fixed_A.get(needed_val, 0)
            if count_b > count_needed:
                missing_A_needed += (count_b - count_needed)

        if missing_A_needed <= total_missing_A:
            print("Yes")
        else:
            print("No")
    else:
        if fixed_A or fixed_B:
            S0_min = max(max(fixed_A) if fixed_A else 0, max(fixed_B) if fixed_B else 0)
        else:
            S0_min = 0

        for a in fixed_A:
            if a > S0_min:
                print("No")
                return
        for b in fixed_B:
            if b > S0_min:
                print("No")
                return

        count_fixed_A = Counter(fixed_A)
        count_fixed_B = Counter(fixed_B)
        missing_A_needed = 0
        for b_val, count_b in count_fixed_B.items():
            needed_val = S0_min - b_val
            count_needed = count_fixed_A.get(needed_val, 0)
            if count_b > count_needed:
                missing_A_needed += (count_b - count_needed)

        if missing_A_needed <= total_missing_A:
