=== ANSWER ===

To solve this problem, we need to determine if a given number of rows \( A \) can form a rectangular arrangement with exactly 400 people. The arrangement must have \( A \) rows and \( B \) columns such that \( A \times B = 400 \). If such a \( B \) exists, we print it; otherwise, we print -1.

### Approach
1. **Problem Analysis**: The problem requires finding an integer \( B \) such that when multiplied by \( A \), the result is exactly 400. This translates to checking if 400 is divisible by \( A \). If it is, \( B \) is simply \( 400 \) divided by \( A \); otherwise, no such \( B \) exists, and we return -1.
2. **Key Insight**: The solution hinges on the divisibility check of 400 by \( A \). If \( A \) is a divisor of 400, then \( B \) is the corresponding quotient; otherwise, the answer is -1.
3. **Algorithm Selection**: The algorithm involves a straightforward divisibility check. Given \( A \) is between 1 and 400, we can directly compute \( 400 \% A \). If the remainder is zero, \( B \) is \( 400 // A \); otherwise, return -1.
4. **Complexity Analysis**: The solution involves a constant-time operation (division and modulus check), making it \( O(1) \) in time and space complexity, which is optimal for the given constraints.

### Solution Code
```python
A = int(input().strip())
if 400 % A == 0:
    print(400 // A)
else:
    print(-1)
```

### Explanation
- **Input Handling**: The code reads an integer \( A \) from standard input.
- **Divisibility Check**: It checks if 400 is divisible by \( A \) using the modulus operator. If \( 400 \% A \) equals zero, \( A \) is a divisor of 400.
- **Output Result**: If \( A \) divides 400, the quotient \( 400 // A \) is printed; otherwise, -1 is printed indicating no valid \( B \) exists.
This approach efficiently leverages basic arithmetic operations to solve the problem within the constraints, ensuring correctness and optimal performance.
