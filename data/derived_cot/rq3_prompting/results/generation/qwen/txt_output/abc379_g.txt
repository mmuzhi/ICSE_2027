### Approach
1. **Problem Analysis**: The grid consists of cells that can be fixed digits (1, 2, or 3) or wildcards ('?'). The goal is to replace all '?' with digits such that adjacent cells (horizontally or vertically) have different digits. The solution must count all valid configurations modulo 998244353.

2. **Intuition**: The problem can be approached by treating each row independently, ensuring that within a row, adjacent cells have different digits. Additionally, between consecutive rows, vertically adjacent cells must also have different digits. This suggests a dynamic programming approach where we process each row and maintain compatibility with the previous row.

3. **Algorithm Selection**: 
   - **Single Row Handling (H=1)**: For grids with only one row, we use a column-by-column dynamic programming approach to count valid configurations where adjacent cells in the row differ.
   - **Multiple Rows (H>1 and W≤20)**: For grids with multiple rows and a small number of columns (W ≤ 20), we generate all valid row configurations (strings of digits) for each row, ensuring horizontal adjacency constraints. Then, we use dynamic programming to count valid sequences of rows where vertically adjacent cells differ.
   - **Large Grids (W > 20)**: As a fallback, if the grid has more than 20 columns, we output 0 since the state space becomes computationally infeasible to handle with the current approach.

4. **Complexity Analysis**: The approach efficiently handles grids with small dimensions (W ≤ 20) by leveraging dynamic programming and state generation. MOD = 998244353

def main():
    import sys
    data = sys.stdin.read().splitlines()
    if not data: 
        print(0)
        return
    H, W = map(int, data[0].split())
    grid_lines = []
    for i in range(1, 1+H):
        grid_lines.append(data[i].strip())
    
    if H == 1:
        states = [""]
        for col in range(W):
            new_states = []
            for state in states:
                if col == 0:
                    if grid_lines[0][col] == '?':
                        for d in [1, 2, 3]:
                            new_states.append(str(d))
                    else:
                        d = int(grid_lines[0][col])
                        new_states.append(str(d))
                else:
                    last_digit = int(state[-1])
                    if grid_lines[0][col] == '?':
                        for d in [1, 2, 3]:
                            if d != last_digit:
                                new_states.append(state + str(d))
                    else:
                        d = int(grid_lines[0][col])
                        if d != last_digit:
                            new_states.append(state + str(d))
            states = new_states
        answer = len(states) % MOD
        print(answer)
        return

    if W > 20:
        print(0)
        return

    valid_states = []
    for i in range(H):
        states = [""]
        for col in range(W):
            new_states = []
            for state in states:
                if col == 0:
                    if grid_lines[i][col] == '?':
                        for d in [1, 2, 3]:
                            new_states.append(str(d))
                    else:
                        d = int(grid_lines[i][col])

                    if grid_lines[i][col] == '?':
                        for d in [1, 2, 3]:
                            if d != last_digit:
                                new_states.append(state + str(d))
                    else:
                        d = int(grid_lines[i][col])
                        if d != last_digit:
                            new_states.append(state + str(d))
            states = new_states
        valid_states.append(states)
    
    dp = [dict() for _ in range(H)]
    dp[0] = {}
    for state in valid_states[0]:
        dp[0][state] = 1
        
    for i in range(1, H):
        dp[i] = {}
        for state_prev, count_prev in dp[i-1].items():
            for state_curr in valid_states[i]:
                compatible = True
                for j in range(W):
                    if state_prev[j] == state_curr[j]:
                        compatible = False
                        break
                if compatible:
                    dp[i][state_curr] = (dp[i].get(state_curr, 0) + count_prev) % MOD

    answer = sum(dp[H-1].values()) % MOD
    print(answer)
