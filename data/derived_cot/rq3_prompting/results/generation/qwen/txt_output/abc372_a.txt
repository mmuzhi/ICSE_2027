=== ANSWER ===

To solve this problem, we need to remove all occurrences of the character '.' from a given string. The solution involves processing the input string and constructing a new string that excludes all '.' characters. 

### Approach
The approach is straightforward due to the problem's simplicity. The key steps are:
1. **Read the input string**: The input is provided as a string `S` which may contain lowercase English letters and '.' characters.
2. **Remove all '.' characters**: We can efficiently remove all '.' characters by replacing them with an empty string. This can be done using the `replace` method available in Python for strings.
3. **Output the result**: The resulting string, after removing all '.', is printed as the output.

This approach is optimal because it leverages Python's built-in string operations, which are both efficient and easy to understand. The `replace` method ensures that all occurrences of '.' are removed in constant time relative to the length of the string.

### Solution Code
```python
def main():
    S = input().strip()
    result = S.replace('.', '')
    print(result)

if __name__ == '__main__':
    main()
```

### Explanation
1. **Reading Input**: The `input()` function reads the entire input string from standard input. The `strip()` method is used to remove any leading or trailing whitespace, though the problem constraints ensure the input is a string without spaces.
2. **Replacing '.' Characters**: The `replace` method is called on the input string with the argument '.' and empty string. This method scans the string and replaces every occurrence of '.' with nothing, effectively removing them.
3. **Printing the Result**: The modified string, now free of all '.', is printed to standard output.

This solution efficiently handles all specified cases, including strings with no '.', strings consisting entirely of '.', and strings with a mix of letters and '.' characters. The use of Python's built-in string operations ensures clarity and performance.
