To solve this problem, we need to determine if we can select five cards from a given set of seven cards such that the selected cards form a full house. A full house is defined as three cards of one integer and two cards of another distinct integer.

### Approach
1. **Problem Analysis**: The problem requires checking if there exists a pair of distinct integers in the given seven cards where one integer appears at least three times and the other appears at least twice. The solution involves counting the occurrences of each integer in the input and then verifying the existence of such a pair.
2. **Intuition**: By counting the frequency of each integer, we can identify integers that meet the criteria for a full house. Specifically, we need at least one integer with a frequency of 3 or more and another distinct integer with a frequency of 2 or more.
3. **Algorithm Selection**: We use a frequency counter (like Python's `Counter` from the `collections` module) to count occurrences of each integer. We then separate these integers into two sets: one for integers with a count of 3 or more (set B) and another for integers with a count of 2 or more (set C). Finally, we check if there exists any integer in set B that is different from any integer in set C.
4. **Complexity Analysis**: The algorithm involves counting frequencies (O(n)) and checking for the existence of a valid pair (O(k^2) where k is the number of distinct integers, which is at most 7 in this case). Thus, the overall complexity is efficient and suitable for the problem constraints.

### Solution Code
```python
from collections import Counter

def main():
    data = input().split()
    if len(data) < 7:
        nums = list(map(int, data))
    else:
        nums = list(map(int, data[:7]))
    
    counts = Counter(nums)
    
    B = set()
    C = set()
    for num, cnt in counts.items():
        if cnt >= 3:
            B.add(num)
        if cnt >= 2:
            C.add(num)
            
    for x in B:
        for y in C:
            if x != y:
                print("Yes")
                return
                
    print("No")

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The input is read as a list of strings, which are then converted to integers. If the input contains fewer than seven elements, we use all available; otherwise, we take the first seven elements.
2. **Frequency Counting**: Using the `Counter` from the `collections` module, we count the occurrences of each integer in the input list.
3. **Set Construction**: We construct two sets, `B` for integers appearing at least three times and `C` for integers appearing at least twice.
4. **Checking for Valid Pair**: We iterate through all integers in set `B` and check if there exists any integer in set `C` that is distinct from the current integer in `B`. If such a pair is found, we print "Yes" and exit. If no such pair is found after all iterations, we print "No".

This approach efficiently checks the necessary conditions for forming a full house by leveraging frequency counting and set operations, ensuring optimal performance for the given problem constraints.
