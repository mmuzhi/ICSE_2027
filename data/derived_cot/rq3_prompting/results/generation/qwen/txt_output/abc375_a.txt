=== ANSWER ===

To solve this problem, we need to count the number of valid positions in a row of seats where two specific conditions are met. The conditions are that the seats at positions i and i+2 are occupied (represented by '#'), and the seat at position i+1 is unoccupied (represented by '.'). 

### Approach
1. **Problem Analysis**: The problem requires checking each triplet of consecutive seats in the row. Specifically, for each position i (1-indexed), we need to check if seats i and i+2 are occupied and seat i+1 is unoccupied. The solution involves iterating through the string representing the seats and counting all such valid triplets.
2. **Intuition**: By iterating through the string from the first seat to the third last seat (since we need to check two seats ahead), we can efficiently check each triplet in constant time. This approach ensures that we cover all possible valid positions without any unnecessary computations.
3. **Algorithm Selection**: A linear scan through the string is chosen due to its simplicity and efficiency. The algorithm checks each triplet starting at position i, ensuring that the conditions are met for the seats at positions i, i+1, and i+2.
4. **Complexity Analysis**: The algorithm runs in O(N) time, where N is the number of seats, because each seat is checked exactly once. The space complexity is O(1) as we only use a few variables for counting and indexing.

### Solution Code
```python
n = int(input().strip())
s = input().strip()

count = 0
for i in range(n - 2):
    if s[i] == '#' and s[i+1] == '.' and s[i+2] == '#':
        count += 1

print(count)
```

### Explanation
1. **Reading Input**: The code starts by reading the number of seats (N) and the string (S) that represents the occupancy of each seat.
2. **Initialization**: A counter variable `count` is initialized to zero to keep track of the number of valid triplets found.
3. **Iteration and Check**: The loop runs from the first seat to the third last seat (i.e., from index 0 to N-3). For each seat at position i, it checks if the current seat (i), the next seat (i+1), and the seat after that (i+2) meet the conditions (occupied, unoccupied, occupied).
4. **Counting Valid Triplets**: If the conditions are satisfied, the counter is incremented.
5. **Output Result**: After processing all possible triplets, the total count of valid positions is printed.

This approach efficiently checks all possible valid positions in linear time, making it optimal for the given problem constraints.
