=== ANSWER ===

To solve this problem, we need to find the sum of all integers in a 9x9 multiplication table that are not equal to a given integer X. The multiplication table is defined such that each cell (i, j) contains the product of its row and column indices (i * j). 

### Approach
1. **Total Sum Calculation**: The sum of all elements in the 9x9 multiplication table can be derived by recognizing that each row i contains the numbers i*1, i*2, ..., i*9. The sum of each row is i multiplied by the sum of the first 9 natural numbers (45). Therefore, the total sum of the table is the sum of the first 9 natural numbers squared (45^2 = 2025).

2. **Counting Occurrences of X**: For the given integer X, we need to count how many times it appears in the multiplication table. This involves checking all possible pairs (i, j) where i and j range from 1 to 9, and verifying if i*j equals X. 

3. **Adjusting the Total Sum**: Once the count of X is determined, the desired sum is obtained by subtracting the product of X and its count from the total sum of the table.

### Solution Code
```python
def main():
    X = int(input().strip())
    total_sum = 45 * 45  # Since (1+2+...+9)^2 = 45^2 = 2025
    count = 0
    for i in range(1, 10):
        if X % i == 0:
            j = X // i
            if 1 <= j <= 9:
                count += 1
    result = total_sum - count * X
    print(result)

if __name__ == '__main__':
    main()
```

### Explanation
1. **Reading Input**: The input is read as an integer X, which represents the value to be excluded from the sum.
2. **Total Sum Calculation**: The total sum of all elements in the 9x9 multiplication table is calculated as 45^2, which equals 2025. This is because the sum of the first 9 natural numbers is 45, and the table's sum is the product of the sums of rows and columns.
3. **Counting Occurrences**: The code iterates over each integer i from 1 to 9. For each i, it checks if X is divisible by i. If it is, the corresponding j (X divided by i) is checked to ensure it lies within the range 1 to 9. Each valid pair (i, j) increments the count.
4. **Result Calculation**: The result is computed by subtracting the total occurrences of X (count * X) from the total sum of the table. This gives the sum of all elements in the table that are not equal to X.
5. **Output**: The result is printed, which is the desired sum of all integers in the table excluding X.

This approach efficiently leverages mathematical insights to minimize computation, ensuring optimal performance even for the upper constraint values.
