To solve this problem, we need to determine the maximum number of kagamimochi that can be formed from a given segment of mochi. A kagamimochi is formed by stacking one mochi on top of another, where the top mochi's size is at most half of the bottom mochi's size. The solution involves efficiently processing each query to find the optimal number of pairs in the specified segment.

### Approach
1. **Problem Analysis**: The problem requires us to process multiple queries on a sorted array of mochi sizes. For each query, we need to find the maximum number of pairs (kagamimochi) that can be formed from a contiguous segment of the array, where each pair (a, b) satisfies the condition a ≤ b/2.
2. **Intuition**: Given the array is sorted, we can use a two-pointer technique to efficiently count valid pairs. The idea is to traverse the segment with two pointers: one for the top mochi (smaller size) and one for the bottom mochi (larger size). For each top mochi, we find the smallest bottom mochi that is at least twice its size, ensuring optimal use of available mochi.
3. **Algorithm Selection**: The two-pointer approach is chosen for its efficiency (O(n) per query) and simplicity. The algorithm involves:
   - Initializing two pointers, `i` (for the top mochi) and `j` (for the bottom mochi), starting from the beginning of the segment.
   - Moving the `j` pointer forward until a valid bottom mochi (≥ 2 * top mochi) is found.
   - Incrementing both pointers and counting a valid pair when the condition is met.
4. **Complexity Analysis**: For each query, the algorithm processes the segment in O(m) time, where m is the length of the segment. In the worst case, if all queries are for the entire array, the total time complexity is O(n*q), which is feasible given the constraints (n and q up to 2×10^5).

import sys

def main():
    data = sys.stdin.read().split()
    if not data:
        return
    
    n = int(data[0])
    A = list(map(int, data[1:1+n]))
    q = int(data[1+n])
    
    index = 1+n+1
    out_lines = []
    for _ in range(q):
        L = int(data[index])
        R = int(data[index+1])
        index += 2
        L0 = L-1
        R0 = R-1
        
        i_ptr = L0
        j_ptr = L0
        count = 0
        
        while i_ptr <= R0 and j_ptr <= R0:
            if A[j_ptr] >= 2 * A[i_ptr]:
                count += 1
                i_ptr += 1
                j_ptr += 1
            else:
                j_ptr += 1
                
        out_lines.append(str(count))
    
    sys.stdout.write("\n".join(out_lines))
