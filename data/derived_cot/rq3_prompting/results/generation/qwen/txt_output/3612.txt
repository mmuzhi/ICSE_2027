To solve this problem, we need to determine if there exist two adjacent subarrays of a given length `k` in an array `nums` such that both subarrays are strictly increasing. The subarrays must be adjacent, meaning the second subarray starts exactly `k` indices after the start of the first subarray.

### Approach
1. **Problem Analysis**: The task involves checking for two non-overlapping subarrays of length `k` that are adjacent in the array. The first subarray starts at index `a`, and the second subarray starts at index `a + k`. Both subarrays must be strictly increasing, i.e., each element in the subarray must be less than the next element.

2. **Intuition**: For each possible starting index `i` of a subarray of length `k`, we check if the subarray starting at `i` is strictly increasing. We then check if there exists another subarray starting at `i + k` that is also strictly increasing. If such a pair of subarrays exists, we return `True`; otherwise, we return `False`.

3. **Algorithm Selection**: 
   - **Precompute Valid Subarrays**: Create an array `inc` where `inc[i]` is `True` if the subarray starting at index `i` of length `k` is strictly increasing.
   - **Check Adjacent Subarrays**: Iterate through the `inc` array to find any index `i` such that both `inc[i]` and `inc[i + k]` are `True`. If such an index is found, return `True`; otherwise, return `False`.

4. **Complexity Analysis**: 
   - **Time Complexity**: O(n * k), where `n` is the length of the array. For each starting index `i`, we check up to `k` elements to determine if the subarray is strictly increasing.
   - **Space Complexity**: O(n), as we use an additional array `inc` of size `n - k + 1` to store the validity of each subarray.

### Solution Code
```python
class Solution:
    def hasIncreasingSubarrays(self, nums: List[int], k: int) -> bool:
        n = len(nums)
        inc = [False] * (n - k + 1)
        for i in range(n - k + 1):
            flag = True
            for j in range(i, i + k - 1):
                if nums[j] >= nums[j + 1]:
                    flag = False
                    break
            inc[i] = flag
        
        for i in range(n - 2 * k + 1):
            if inc[i] and inc[i + k]:
                return True
        return False
```

### Explanation
1. **Precompute Valid Subarrays**: The loop iterates over each possible starting index `i` of a subarray of length `k`. For each `i`, it checks if the subarray from `i` to `i + k - 1` is strictly increasing by comparing each element with the next one. The result is stored in the `inc` array.
2. **Check Adjacent Subarrays**: The second loop checks for any index `i` in the range `[0, n - 2*k]` (ensuring `i + k` is a valid starting index) where both the subarray starting at `i` and the subarray starting at `i + k` are strictly increasing (as indicated by `inc[i]` and `inc[i + k]` being `True`).
3. **Result**: If such an index `i` is found, the function returns `True`; otherwise, after checking all possible indices, it returns `False`.

This approach efficiently checks for the required condition by leveraging precomputation and a straightforward iteration, ensuring optimal performance for the given constraints.
