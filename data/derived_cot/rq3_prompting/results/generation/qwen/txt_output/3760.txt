To solve this problem, we need to assign an element from a given array to each group in another array based on specific divisibility conditions. The key challenge is to efficiently determine the smallest index element that satisfies the condition for each group without resorting to a brute-force approach, which would be computationally expensive.

### Approach
1. **Precompute Divisors**: For each number up to the maximum value in the groups array (100,000), precompute all its divisors. This allows quick access to the divisors of any group value during the assignment phase.
2. **Track Minimum Indices**: For each element in the elements array, record the smallest index where each element value appears. This helps in quickly identifying the earliest occurrence of any divisor during the assignment process.
3. **Assign Elements to Groups**: For each group value, iterate through its precomputed divisors. For each divisor present in the elements array, check if it appears at a smaller index than the current candidate. The smallest index encountered is chosen for the group; if no such divisor exists, assign -1.

### Solution Code
```python
class Solution:
    def assignElements(self, groups: List[int], elements: List[int]) -> List[int]:
        max_val = 100000
        divisors_list = [[] for _ in range(max_val + 1)]
        for i in range(1, max_val + 1):
            for j in range(i, max_val + 1, i):
                divisors_list[j].append(i)
        
        min_index = {}
        element_set = set()
        for idx, val in enumerate(elements):
            element_set.add(val)
            if val not in min_index:
                min_index[val] = idx
        
        res = [-1] * len(groups)
        
        for idx, n in enumerate(groups):
            candidate = float('inf')
            for d in divisors_list[n]:
                if d in element_set:
                    if min_index[d] < candidate:
                        candidate = min_index[d]
            if candidate != float('inf'):
                res[idx] = candidate
        
        return res
```

### Explanation
1. **Precompute Divisors**: The code first initializes a list of lists (`divisors_list`) where each entry `divisors_list[i]` contains all divisors of `i`. This is done using a sieve-like approach, iterating through each number and its multiples to populate the divisors list efficiently.
2. **Track Minimum Indices**: The code then processes the elements array to build two dictionaries: `min_index` (mapping each element value to its first occurrence index) and `element_set` (a set of all unique element values for quick lookup).
3. **Assign Elements to Groups**: For each group value in the groups array, the code checks all its divisors. For each divisor found in the `element_set`, it checks if the divisor's first occurrence index is smaller than the current candidate. The smallest index found is recorded for the group. If no valid divisor is found, the group is assigned -1.

This approach efficiently leverages precomputation and set operations to minimize the number of checks required per group, ensuring optimal performance even for large input sizes.
