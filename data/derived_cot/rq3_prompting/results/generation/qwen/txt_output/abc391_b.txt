=== ANSWER ===

To solve this problem, we need to find a subgrid within a given N x N grid S that exactly matches a given M x M grid T. The solution involves checking all possible positions in grid S where the M x M subgrid could start and verifying if the subgrid matches T.

### Approach
1. **Problem Analysis**: The problem requires identifying a top-left position (a, b) in grid S such that the subgrid starting at (a, b) and of size M x M matches grid T exactly. The grids are composed of '.' (white) and '#' (black) cells.
2. **Intuition**: Since the constraints are small (N and M up to 50), a brute-force approach is feasible. We can iterate over all possible starting positions in S for the M x M subgrid and check each position for a match with T.
3. **Algorithm Selection**: The algorithm involves:
   - Reading the dimensions N and M.
   - Reading the N x N grid S and M x M grid T.
   - Iterating over each possible top-left position (a, b) in S where a subgrid of size M x M can fit (i.e., from (0, 0) to (N - M, N - M) in 0-indexed terms).
   - For each position, comparing each cell in the subgrid of S with the corresponding cell in T. If all cells match, the position (a + 1, b + 1) (converted to 1-indexed) is the solution.
4. **Complexity Analysis**: The algorithm checks (N - M + 1) x (N - M + 1) positions, each requiring M x M comparisons. Thus, the worst-case time complexity is O((N - M + 1)^2 * M^2), which is efficient given the constraints (N and M are at most 50).

### Solution Code
```python
def main():
    import sys
    data = sys.stdin.read().splitlines()
    n, m = map(int, data[0].split())
    S = []
    index = 1
    for _ in range(n):
        S.append(data[index].strip())
        index += 1
    T = []
    for _ in range(m):
        T.append(data[index].strip())
        index += 1

    for a in range(n - m + 1):
        for b in range(n - m + 1):
            match = True
            for i in range(m):
                for j in range(m):
                    if S[a + i][b + j] != T[i][j]:
                        match = False
                        break
                if not match:
                    break
            if match:
                print(f"{a + 1} {b + 1}")
                return

if __name__ == '__main__':
    main()
```

### Explanation
1. **Reading Input**: The input is read from standard input. The first line contains N and M, the dimensions of grids S and T. The next N lines define grid S, and the following M lines define grid T.
2. **Grid Processing**: Grid S is stored as a list of strings, each representing a row. Similarly, grid T is stored as a list of strings.
3. **Brute-force Search**: The algorithm iterates over each possible starting position (a, b) for the M x M subgrid in S. For each position, it checks if the corresponding subgrid in S matches T by comparing each cell.
4. **Output**: Upon finding the first valid position, the algorithm prints the 1-indexed coordinates (a + 1, b + 1) and exits. The problem guarantees exactly one such position exists.

This approach efficiently checks all possible subgrids in S, ensuring correctness while adhering to the problem constraints.
