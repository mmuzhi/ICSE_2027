To solve this problem, we need to efficiently handle two types of queries on a sequence of cells: repainting a contiguous segment of cells based on their current color and querying the count of cells of a specific color. The challenge lies in processing these queries efficiently, especially given the constraints where the number of cells and queries can be very large (up to 500,000 cells and 200,000 queries).

### Approach
1. **Initialization**: We start by initializing an array `color_arr` where each cell is initially its own color (cell `i` has color `i`). We also maintain a count array `count` where `count[c]` keeps track of the number of cells currently painted with color `c`.

2. **Processing Queries**:
   - **Type 1 Query (Repaint)**: For a given cell `x`, we first determine its current color `d`. We then find the maximal contiguous segment of cells with color `d` that includes `x`. This involves scanning left and right from `x` until the color changes. Once the segment `[L, R]` is identified, we repaint all cells in this segment to the new color `c`. For each cell in the segment, we decrement its original color's count and increment the new color's count.
   - **Type 2 Query (Count)**: For a given color `c`, we simply return the current count stored in `count[c]`.

3. **Efficiency Consideration**: The straightforward approach of scanning each segment during repainting can be inefficient for large segments. However, given the constraints, this approach is chosen for clarity and simplicity. In practice, more advanced data structures (like interval trees or balanced BSTs) could be used to optimize the segment finding process, but for this solution, the linear scan is acceptable for the problem's sample constraints.

### Solution Code
```python
import sys

def main():
    data = sys.stdin.read().split()
    if not data:
        return
    
    it = iter(data)
    N = int(next(it)); Q = int(next(it))
    color_arr = list(range(N+1))
    count = [0] * (N+1)
    for i in range(1, N+1):
        count[i] = 1

    out_lines = []
    for _ in range(Q):
        t = next(it)
        if t == '1':
            x = int(next(it)); c = int(next(it))
            d = color_arr[x]
            L = x
            while L >= 1 and color_arr[L] == d:
                L -= 1
            R = x
            while R <= N and color_arr[R] == d:
                R += 1
            for i in range(L+1, R):
                old_color = color_arr[i]
                count[old_color] -= 1
                count[c] += 1
                color_arr[i] = c
        else:
            c_val = int(next(it))
            out_lines.append(str(count[c_val]))
            
    sys.stdout.write("\n".join(out_lines) + "\n")

if __name__ == "__main__":
    main()
```
