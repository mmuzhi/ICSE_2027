The key insight is that the minimal number of deletion operations is equal to the number of distinct elements in the sequence, and the minimal number of adjacent swaps required to group these distinct elements together can be computed based on their positions.

### Approach
1. **Problem Analysis**: The problem involves two operations—swapping adjacent elements and deleting a prefix of equal elements. The goal is to minimize the total number of operations (swaps + deletions) to empty the sequence.
2. **Intuition**: The minimal number of deletion operations is the number of distinct elements in the sequence because each distinct element must be deleted at least once. The challenge is to compute the minimal number of adjacent swaps needed to group these distinct elements together so that each deletion operation can delete a contiguous block of the same element.
3. **Algorithm Selection**: 
   - **Frequency Count**: Count the occurrences of each distinct element in the sequence.
   - **Grouping Calculation**: For each distinct element, calculate the minimal adjacent swaps required to bring all its occurrences together. This involves summing the number of inversions (elements not belonging to the group) between the positions of the distinct elements.
4. **Complexity Analysis**: The algorithm processes each element in the sequence once, making it linear in time complexity, O(N), where N is the length of the sequence. This is efficient given the constraints.

### Solution Code
```python
import sys
from collections import defaultdict

def main():
    data = sys.stdin.read().split()
    t = int(data[0])
    index = 1
    results = []
    
    for _ in range(t):
        n = int(data[index]); index += 1
        arr = list(map(int, data[index:index+n]))
        index += n
        
        distinct_count = len(set(arr))
        freq = defaultdict(int)
        for num in arr:
            freq[num] += 1
            
        total_swaps = 0
        for num, count in freq.items():
            positions = [i for i, x in enumerate(arr) if x == num]
            if count == 0:
                continue
            if count == 1:
                continue
            total_swaps += (count * (count - 1)) // 2
            
        total_operations = distinct_count + total_swaps
        results.append(str(total_operations))
        
    sys.stdout.write("\n".join(results))

if __name__ == "__main__":
    main()
```
