class Solution:
    def maxRectangleArea(self, points: List[List[int]]) -> int:
        # Create a set of points for O(1) lookups
        point_set = set(tuple(point) for point in points)
        
        # Get distinct x and y coordinates
        xs = sorted(set(point[0] for point in points))
        ys = sorted(set(point[1] for point in points))
        
        max_area = 0
        
        # Iterate over all pairs of x's and y's
        n = len(xs)
        m = len(ys)
        for i in range(n):
            for j in range(i+1, n):
                x1, x2 = xs[i], xs[j]
                for k in range(m):
                    for l in range(k+1, m):
                        y1, y2 = ys[k], ys[l]
                        # Check the four corners
                        corners = [(x1, y1), (x1, y2), (x2, y1), (x2, y2)]
                        valid_corners = True
                        for corner in corners:
                            if corner not in point_set:
                                valid_corners = False
                                break
                        if not valid_corners:
                            continue
                            
                        # Check other points: they must not be in the rectangle (including borders)
                        invalid = False
                        for point in points:
                            pt = tuple(point)
                            if pt in corners:
                                continue
                            px, py = point
                            if x1 <= px <= x2 and y1 <= py <= y2:
                                invalid = True
                                break
                                
                        if not invalid:
                            area = (x2 - x1) * (y2 - y1)
                            if area > max_area:
                                max_area = area
                                
        return max_area if max_area != 0 else -1

To solve this problem, we need to find the maximum area of a rectangle that can be formed using four points from a given list of points on a 2D plane. The rectangle must have edges parallel to the axes, and it should not contain any other point inside or on its border. 

### Approach
1. **Problem Analysis**: The rectangle's edges must be parallel to the axes, meaning its sides are aligned with the x and y axes. The four corners of the rectangle will be determined by two distinct x-coordinates and two distinct y-coordinates. The key insight is that for any rectangle defined by these coordinates, the four corners must be present in the given points, and no other point should lie within or on the rectangle's borders.

2. **Intuition**: The solution involves checking all possible rectangles formed by pairs of distinct x-coordinates and pairs of distinct y-coordinates. For each such rectangle, we verify if all four corners are present in the given points and that no other point lies within or on the rectangle's borders.

3. **Algorithm Selection**: 
   - **Set for Lookup**: Convert the list of points into a set for O(1) membership checks.
   - **Distinct Coordinates**: Extract all distinct x and y coordinates from the points, sorted for consistent iteration.
   - **Brute-force Check**: Iterate over all pairs of x-coordinates and y-coordinates to form potential rectangles. For each rectangle, check if all four corners are present in the set. Then, verify that no other point lies within or on the rectangle's borders.

4. **Complexity Analysis**: Given the constraints (up to 10 points), the approach is feasible. The number of distinct x and y coordinates is at most 10, leading to O(n^4) operations in the worst case (where n is the number of distinct coordinates). However, with n ≤ 10, this is manageable (up to 2025 operations).

### Solution Code
```python
class Solution:
    def maxRectangleArea(self, points: List[List[int]]) -> int:
        point_set = set(tuple(point) for point in points)
        
        xs = sorted(set(point[0] for point in points))
        ys = sorted(set(point[1] for point in points))
        
        max_area = 0
        
        for i in range(len(xs)):
            for j in range(i+1, len(xs)):
                x1, x2 = xs[i], xs[j]
                for k in range(len(ys)):
                    for l in range(k+1, len(ys)):
                        y1, y2 = ys[k], ys[l]
                        corners = [(x1, y1), (x1, y2), (x2, y1), (x2, y2)]
                        if all(corner in point_set for corner in corners):
                            invalid = False
                            for point in points:
                                pt_tuple = tuple(point)
                                if pt_tuple in corners:
                                    continue
                                px, py = point
                                if x1 <= px <= x2 and y1 <= py <= y2:
                                    invalid = True
                                    break
                            if not invalid:
                                area = (x2 - x1) * (y2 - y1)
                                if area > max_area:
                                    max_area = area
        
        return max_area if max_area != 0 else -1
```

### Explanation
1. **Set Creation**: The given points are converted into a set for efficient lookup.
2. **Coordinate Extraction**: Distinct x and y coordinates are extracted and sorted to facilitate iteration over all possible pairs.
3. **Rectangle Validation**: For each pair of x-coordinates and y-coordinates, the four corners of the rectangle are checked for presence in the set. If all corners are present, the algorithm then checks if any other point lies within or on the rectangle's borders.
4. **Area Calculation**: The area of valid rectangles is computed, and the maximum area encountered is tracked.
5. **Result Handling**: If no valid rectangle is found, the function returns -1; otherwise, it returns the maximum area found.

This approach efficiently checks all possible rectangles formed by the given points, ensuring the solution meets the problem's requirements.
