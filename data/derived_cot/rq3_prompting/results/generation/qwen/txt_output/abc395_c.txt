=== ANSWER ===

To solve this problem, we need to find the shortest contiguous subarray in a given sequence where at least one value appears more than once in the entire array. The solution involves identifying all such values that appear at least twice, then determining the minimal subarray length that contains two occurrences of any such value.

### Approach
1. **Frequency Dictionary**: First, we create a dictionary where each key is a unique number from the array, and the value is a list of indices where this number appears. This helps in efficiently tracking the positions of each number.
2. **Check for Duplicates**: For each number that appears at least twice, we examine the list of its indices to find the minimal gap between any two consecutive occurrences. The minimal gap plus one gives the length of the shortest subarray containing two occurrences of that number.
3. **Early Termination**: If at any point we find a subarray of length 2 (the minimal possible), we immediately terminate the search and return 2, as no shorter subarray can exist.
4. **Result Determination**: After processing all numbers, if no such subarray is found, we return -1; otherwise, we return the minimal length found.

### Solution Code
```python
n = int(input().strip())
A = list(map(int, input().split()))

from collections import defaultdict
pos_dict = defaultdict(list)
for i, num in enumerate(A):
    pos_dict[num].append(i)

ans = float('inf')

for num, positions in pos_dict.items():
    if len(positions) < 2:
        continue
    min_gap = positions[1] - positions[0]
    for i in range(1, len(positions)):
        gap = positions[i] - positions[i-1]
        if gap < min_gap:
            min_gap = gap
        if min_gap == 1:
            break
    candidate = min_gap + 1
    if candidate < ans:
        ans = candidate
    if ans == 2:
        break

print(-1 if ans == float('inf') else ans)
```

### Explanation
1. **Frequency Dictionary Construction**: The code reads the input array and constructs a dictionary where each number maps to a list of its indices. This allows efficient access to all positions of each number.
2. **Finding Minimal Subarray**: For each number appearing at least twice, the code calculates the minimal gap between consecutive occurrences. The minimal gap plus one gives the length of the shortest subarray containing two of that number.
3. **Early Termination**: If the minimal gap is 1 (indicating two consecutive occurrences), the subarray length is 2, the smallest possible. The loop breaks early to return this result immediately.
4. **Result Handling**: After processing all relevant numbers, if no subarray was found (i.e., no number appeared more than once), the code outputs -1. Otherwise, it outputs the minimal subarray length found.

This approach efficiently narrows down the problem to checking each number's occurrences and leveraging early termination to optimize performance, especially for large input sizes.
