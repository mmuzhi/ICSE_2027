To solve this problem, we need to generate an N x N grid pattern based on specific operations. The operations involve coloring rectangular regions of the grid in alternating colors (black and white) for certain rows. The key observation is that each cell's color is determined by the largest row number (1-indexed) that satisfies certain conditions related to its position in the grid.

### Approach
1. **Problem Analysis**: The problem requires constructing an N x N grid where each cell's color is determined by the last operation that affects it. The operations are performed for rows 1 to N (1-indexed), but only if the row number is less than or equal to a calculated column number. Each operation colors a square region starting from the current row and column to a calculated bottom-right cell. The color alternates between black (odd rows) and white (even rows).

2. **Insight**: For each cell (i, j) (0-indexed), the largest row number (1-indexed) that can affect it is the minimum of four values: (i+1), (j+1), (N-i), and (N-j). This value must also not exceed half the grid size (rounded down). The parity (odd or even) of this largest row number determines the cell's color.

3. **Algorithm Selection**: We iterate over each cell in the grid, compute the largest row number that affects it, and determine its color based on the row's parity. This approach efficiently computes the grid without explicitly performing each operation, leveraging mathematical conditions to determine the final color of each cell.

4. **Complexity Analysis**: The algorithm runs in O(N^2) time, which is optimal for the given constraints (N ≤ 50). Each cell is processed once, and the operations per cell are constant time.

### Solution Code
```python
def main():
    n = int(input().strip())
    grid = [['' for _ in range(n)] for __ in range(n)]
    
    for i in range(n):
        for j in range(n):
            m = min(i+1, j+1, n-i, n-j)
            i0 = min(m, (n + 1) // 2)
            if i0 % 2 == 1:
                grid[i][j] = '#'
            else:
                grid[i][j] = '.'
                
    for row in grid:
        print(''.join(row))

if __name__ == '__main__':
    main()
```

This minimum value, `m`, represents the largest row number (1-indexed) that can affect the cell. The actual row number used is the minimum of `m` and half the grid size (rounded down).
4. **Color Determination**: If the computed row number is odd, the cell is colored black ('#'); otherwise, it is colored white ('.').
5. **Output**: The grid is printed row by row, with each row being a string of the appropriate length.

This approach efficiently determines the final color of each cell by leveraging mathematical conditions, ensuring correctness and optimal performance.
