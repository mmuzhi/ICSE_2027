To solve this problem, we need to maximize the integer value of a string S after performing a series of operations where each operation involves replacing a character in S with a corresponding character from another string T. The operations must be performed in order, and we can choose any index in S for each operation.

### Approach
1. **Problem Analysis**: The key observation is that the final value of the string S is determined by the last operation performed on each index. To maximize the string, we should prioritize setting the leftmost characters to the highest possible digits from T, ensuring that these digits are not overwritten by lower digits in subsequent operations.

2. **Intuition**: The leftmost characters have the most significant impact on the integer value. Therefore, we should first set the leftmost characters to the highest available digits in T and then ensure that these digits are not overwritten by any subsequent operations.

3. **Algorithm Selection**: 
   - **Greedy Algorithm**: We process the string S from left to right. For each position, we find the highest digit available in T that can be used to potentially improve the current digit. We then reserve this digit for the current position and remove it from the pool of available digits. This ensures that higher digits are used for more significant positions first.
   - **Handling Remaining Operations**: After setting the leftmost characters, the remaining operations must be applied to the rightmost characters. These operations can overwrite the digits of the rightmost characters, but since the leftmost characters are already set to their highest possible values, the rightmost characters can be set to the highest available digits in the remaining operations.

4. **Complexity Analysis**: The algorithm processes each character in S once and each character in T once, resulting in a time complexity of O(N + M), where N is the length of S and M is the length of T. This is efficient given the constraints (N and M up to 10^6).

import sys
from collections import defaultdict, deque

def main():
    data = sys.stdin.read().splitlines()

    n_m_line = data[0].split()

    S = data[1].strip()
    T_str = data[2].strip()
    
    freq = defaultdict(int)
    for char in T_str:
        freq[char] += 1
        
    res = list(S)

    for i in range(N):

                if freq[digit] > 0:

            if best_digit > res[i]:
                res[i] = best_digit
