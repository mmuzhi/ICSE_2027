To solve this problem, we need to count the number of ways to divide a given sequence of integers into contiguous subsequences such that no subsequence has a sum equal to a given integer K. The solution involves dynamic programming and efficient handling of prefix sums to avoid a brute-force approach.

### Approach
1. **Dynamic Programming (DP) Setup**: We use a DP array where `dp[i]` represents the number of valid ways to split the first `i` elements of the sequence. The base case is `dp[0] = 1`, representing an empty prefix.
2. **Prefix Sum Calculation**: We compute prefix sums `P[i]` for the sequence to quickly determine the sum of any contiguous subsequence. The sum from index `j` to `i-1` is `P[i] - P[j]`.
3. **Dictionary for Efficient Lookup**: A dictionary is used to store the sum of `dp[j]` values for each prefix sum encountered. This allows us to efficiently compute the number of invalid segments (those summing to K) and subtract them from the total valid segments.
4. **Modulo Arithmetic**: Given the constraints, all operations are performed modulo 998244353 to handle large numbers and prevent overflow.

The key insight is that for each position `i`, the number of valid ways to split the sequence up to `i` is the total valid ways up to `i-1` minus the ways where the last segment sums to K. This is efficiently computed using prefix sums and a dictionary for quick lookups.

### Solution Code
```python
mod = 998244353

import sys

def main():
    data = sys.stdin.read().split()
    if not data:
        print(0)
        return
        
    n = int(data[0])
    K = int(data[1])
    A = list(map(int, data[2:2+n]))
    
    dp = [0] * (n+1)
    P = [0] * (n+1)
    
    dp[0] = 1
    P[0] = 0
    total_dp = 1
    dict_p = {0: 1}
    
    for i in range(1, n+1):
        P[i] = P[i-1] + A[i-1]
        x = P[i] - K
        
        subtract = dict_p.get(x, 0)
        dp[i] = (total_dp - subtract) % mod
        
        total_dp = (total_dp + dp[i]) % mod
        
        if P[i] in dict_p:
            dict_p[P[i]] = (dict_p[P[i]] + dp[i]) % mod
        else:
            dict_p[P[i]] = dp[i] % mod
            
    print(dp[n] % mod)

if __name__ == "__main__":
    main()
```

### Explanation
1. **Initialization**: The code reads input values and initializes the DP array, prefix sum array, and a dictionary to keep track of prefix sums and their corresponding DP values.
2. **Prefix Sum Calculation**: For each element in the sequence, the prefix sum `P[i]` is computed as the cumulative sum up to that element.
3. **Dynamic Programming Update**: For each position `i`, the code checks if the sum of the segment ending at `i` (computed as `P[i] - K`) exists in the dictionary. If it does, those segments are subtracted from the total valid ways (`total_dp`).
4. **Dictionary Update**: After updating `dp[i]`, the dictionary is updated with the new prefix sum `P[i]` and its corresponding DP value to be used in subsequent iterations.
5. **Output**: The result is the value of `dp[n]` modulo 998244353, representing the number of valid ways to split the entire sequence without any segment summing to K.

This approach efficiently counts the valid segmentations by leveraging dynamic programming and prefix sums, ensuring optimal performance even for large input sizes.
