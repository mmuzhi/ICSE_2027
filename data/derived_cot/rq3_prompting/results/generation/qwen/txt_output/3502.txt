To solve this problem, we need to count the number of substrings in a given string where at least one character appears at least `k` times. The solution involves efficiently traversing the string and using a sliding window technique to keep track of character frequencies within the current window.

### Approach
1. **Problem Analysis**: The task is to find all contiguous substrings where at least one character's frequency is at least `k`. A brute-force approach would check every possible substring, but that would be too slow for large strings (up to 3000 characters). Instead, we use a sliding window approach to optimize the process.
2. **Sliding Window Technique**: We maintain a window defined by `left` and `right` pointers. As we expand the window by moving the `right` pointer, we keep track of the frequency of each character in the current window. If any character's frequency reaches `k`, we note that the current window meets the condition.
3. **Counting Valid Substrings**: For each position of the `left` pointer, once the condition is met (i.e., at least one character in the window has frequency `k`), all substrings starting at `left` and ending at any position from `right-1` to the end of the string are valid. The number of such substrings is `n - right + 1`.
4. **Adjusting the Window**: After processing each `left` position, we move the `left` pointer forward by one, adjusting the character frequencies and the count of characters that meet or exceed `k`.

### Solution Code
```python
class Solution:
    def numberOfSubstrings(self, s: str, k: int) -> int:
        n = len(s)
        freq = [0] * 26
        count_ge = 0
        ans = 0
        right = 0
        for left in range(n):
            while right < n and count_ge == 0:
                idx = ord(s[right]) - ord('a')
                freq[idx] += 1
                if freq[idx] == k:
                    count_ge += 1
                right += 1
            if count_ge > 0:
                ans += n - right + 1
            idx = ord(s[left]) - ord('a')
            freq[idx] -= 1
            if freq[idx] == k - 1:
                count_ge -= 1
        return ans
```

### Explanation
1. **Initialization**: We initialize an array `freq` to keep track of character frequencies (size 26 for lowercase letters), a counter `count_ge` to track characters that meet or exceed `k` frequency, and `ans` to store the result.
2. **Sliding Window**: The `right` pointer expands the window by including new characters. For each new character, we update its frequency and check if it meets or exceeds `k`. If so, `count_ge` is incremented.
3. **Valid Substrings Calculation**: Once `count_ge` is positive, all substrings starting at the current `left` and ending at any position from `right-1` to the end of the string are valid. The count of these substrings is `n - right + 1`, which is added to `ans`.
4. **Window Adjustment**: The `left` pointer moves forward, and the corresponding character's frequency is decremented. If this decrement reduces the frequency to `k-1`, `count_ge` is decremented.
5. **Result**: After processing all `left` positions, `ans` contains the total number of valid substrings.

This approach efficiently counts all valid substrings in O(n) time by leveraging the sliding window technique and maintaining character frequencies, ensuring optimal performance even for the upper constraint limits.
