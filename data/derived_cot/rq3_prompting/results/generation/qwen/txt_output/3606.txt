### Solution Code
```python
class Solution:
    def minElement(self, nums: List[int]) -> int:
        min_val = float('inf')
        for num in nums:
            total = 0
            n = num
            while n:
                total += n % 10
                n //= 10
            if total < min_val:
                min_val = total
        return min_val
```

### Explanation
1. **Initialization**: We start by initializing `min_val` to a very large number (using `float('inf')`) to ensure any computed digit sum will be smaller.
2. **Iteration**: For each number in the array, we decompose it into its digits. This is done by repeatedly taking the modulo 10 of the number (to get the last digit) and then performing integer division by 10 (to remove the last digit).
3. **Sum Calculation**: During each iteration of the inner loop, the last digit is added to a running total (`total`), which represents the sum of the digits of the current number.
4. **Tracking Minimum**: After computing the digit sum for a number, we compare it with `min_val`. If the current sum is smaller, we update `min_val` to this new value.
5. **Result**: After processing all numbers, `min_val` holds the smallest digit sum found, which is returned as the result.

This approach efficiently computes the required minimum value by leveraging simple arithmetic operations and a single pass through the array, ensuring optimal performance for the given problem constraints.
