=== ANSWER ===

To solve this problem, we need to determine if there exists a substring of length exactly `k` in the given string `s` that meets three specific conditions: all characters in the substring must be the same, the character immediately before the substring (if it exists) must be different from the substring's character, and the character immediately after the substring (if it exists) must also be different from the substring's character.

### Approach
1. **Iterate through all possible substrings of length `k`**: For each starting index `i` in the string, extract the substring `s[i:i+k]`.
2. **Check for uniform characters**: Verify that all characters in the substring are identical. This can be done efficiently by converting the substring into a set and checking if the set size is 1.
3. **Check adjacent characters**: If the substring meets the uniform character condition, then check the characters immediately before and after the substring. If either adjacent character (if they exist) is the same as the substring's character, skip further checks for this substring. Otherwise, return `True`.
4. **Return False if no valid substring found**: If no substring of length `k` satisfies all conditions after checking all possible substrings, return `False`.

### Solution Code
```python
class Solution:
    def hasSpecialSubstring(self, s: str, k: int) -> bool:
        n = len(s)
        for i in range(n - k + 1):
            substr = s[i:i+k]
            if len(set(substr)) != 1:
                continue
            if i > 0 and s[i-1] == substr[0]:
                continue
            if i + k < n and s[i+k] == substr[0]:
                continue
            return True
        return False
```

### Explanation
1. **Iteration through substrings**: The loop runs from the start of the string to the position where a substring of length `k` can still be formed.
2. **Uniform character check**: For each substring, the code checks if all characters are the same by converting the substring into a set. If the set size is not 1, the substring is skipped.
3. **Adjacent character checks**: The code checks the character before the substring (if it exists) and ensures it is different from the substring's character. Similarly, it checks the character after the substring (if it exists) and ensures it is different.
4. **Early termination**: If a valid substring is found, the function immediately returns `True`. If no valid substring is found after all iterations, the function returns `False`.

This approach efficiently checks all possible substrings of length `k` while leveraging set operations for uniform character verification and simple index checks for adjacent characters, ensuring optimal performance for the given constraints.
