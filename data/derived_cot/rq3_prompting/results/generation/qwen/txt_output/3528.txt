=== ANSWER ===

To solve this problem, we need to find the maximum score obtained by starting at the first index of an array and jumping to indices greater than the current index, with the score for each jump calculated as (j - i) * nums[i]. The goal is to reach the last index with the highest possible score.

### Approach
1. **Problem Analysis**: The problem requires maximizing the score by strategically choosing jumps from each index to a higher index. The score for a jump from index `i` to index `j` is `(j - i) * nums[i]`. The challenge is to efficiently compute the maximum score without resorting to a brute-force O(n^2) approach, especially given the constraint that the array length can be up to 10^5.

2. **Insight**: The key observation is that the score for reaching any index `i` can be derived from the maximum score of any previous index `j` plus the score from jumping from `j` to `i`, i.e., `dp[i] = max_{j < i} { dp[j] + (i - j) * nums[j] }`. This can be rewritten as `dp[i] = max_{j < i} { i * nums[j] + (dp[j] - j * nums[j]) }`, which resembles a set of linear functions evaluated at `i`.

3. **Algorithm Selection**: To handle this efficiently, we use a Li Chao Tree, which is a data structure designed to manage a set of lines and answer queries for the maximum line value at a given point in logarithmic time. Each line represents the function `f_j(x) = x * nums[j] + (dp[j] - j * nums[j])`, and the tree helps in finding the maximum value among these lines at any query point `x`.

4. **Complexity Analysis**: The Li Chao Tree allows each update and query operation to be performed in O(log n) time. Since we perform `n` updates and `n` queries, the total time complexity is O(n log n), which is efficient for the given constraints.

### Solution Code

class Solution:
    def findMaximumScore(self, nums):
        n = len(nums)
        if n == 0:
            return 0
        if n == 1:
            return 0
        
        class Line:
            __slots__ = ('a', 'b')
            def __init__(self, a, b):
                self.a = a
                self.b = b
            def eval(self, x):
                return self.a * x + self.b
        
        class LiChaoTree:
            __slots__ = ('x_low', 'x_high', 'size', 'tree')
            def __init__(self, x_low, x_high):
                self.x_low = x_low
                self.x_high = x_high
                self.size = 1
                while self.size < x_high - x_low + 1:
                    self.size *= 2
                self.tree = [None] * (4 * self.size)
            
            def update(self, a, b):
                self._update(0, self.x_low, self.x_high, a, b)
            
            def _update(self, node, l, r, a, b):
                mid = (l + r) // 2
                if self.tree[node] is None or a * mid + b > self.tree[node].eval(mid):
                    self.tree[node] = Line(a, b)
                if l == r:
                    return
                if a * l + b > self.tree[node].eval(l):
                    self._update(node*2, l, mid, self.tree[node].a, self.tree[node].b)
                    self.tree[node] = Line(a, b)
                elif a * r + b > self.tree[node].eval(r):
                    self._update(node*2+1, mid+1, r, self.tree[node].a, self.tree[node].b)
                    self.tree[node] = Line(a, b)
            
            def query(self, x):
                return self._query(0, self.x_low, self.x_high, x)
            
            def _query(self, node, l, r, x):
                if self.tree[node] is None:
                    return -10**18
                res = self.tree[node].eval(x)
                if l == r:
                    return res
                mid = (l + r) // 2
                if x <= mid:
                    res = max(res, self._query(node*2, l, mid, x))
                else:
                    res = max(res, self._query(node*2+1, mid+1, r, x))
                return res
        
        tree = LiChaoTree(0, n-1)
        dp = [0] * n
        dp[0] = 0
        tree.update(nums[0], 0)
        for i in range(1, n):
            dp[i] = tree.query(i)
            tree.update(nums[i], dp[i] - i * nums[i])
        return dp[n-1]

1. **Initialization**: The `Line` class represents a linear function with slope `a` and intercept `b`. The `LiChaoTree` class is initialized with a range `[x_low, x_high]` and uses a segment tree to store lines for efficient maximum queries.

2. **Update Operation**: The `update` method inserts a new line into the tree. The `_update` method recursively ensures the new line is placed in the appropriate segments of the tree.

3. **Query Operation**: The `query` method retrieves the maximum value from all lines at a specific `x` coordinate. The `_query` method recursively checks the relevant segments of the tree.

4. **Dynamic Programming**: The `dp` array stores the maximum score for each index. For each index `i`, the score is computed by querying the tree for the best previous line, then updating the tree with the current line.

This approach efficiently handles the problem constraints by leveraging the Li Chao Tree to reduce the time complexity from O(n^2) to O(n log n).
