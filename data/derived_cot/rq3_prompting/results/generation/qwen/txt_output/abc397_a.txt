=== ANSWER ===

To solve this problem, we need to classify Takahashi's body temperature into one of three categories based on its value. The categories are "High fever", "Fever", and "Normal", each corresponding to a specific range of temperatures. The solution involves reading the input temperature, comparing it against predefined thresholds, and outputting the appropriate classification.

### Approach
1. **Input Handling**: Read the input temperature as a string and convert it to a floating-point number for precise comparison.
2. **Temperature Comparison**: 
   - If the temperature is 38.0 degrees Celsius or higher, classify it as "High fever" (output 1).
   - If the temperature is between 37.5 and 38.0 degrees Celsius, classify it as "Fever" (output 2).
   - If the temperature is below 37.5 degrees Celsius, classify it as "Normal" (output 3).
3. **Output Result**: Based on the comparison, print the corresponding integer value.

### Solution Code
```python
def main():
    X = float(input().strip())
    if X >= 38.0:
        print(1)
    elif X >= 37.5:
        print(2)
    else:
        print(3)

if __name__ == '__main__':
    main()
```

### Explanation
1. **Reading Input**: The input is read as a string and converted to a float to handle the decimal value accurately.
2. **Condition Checks**: The code checks the temperature against the thresholds in descending order. 
   - The first condition checks if the temperature is at least 38.0°C, which directly outputs 1 for "High fever".
   - The second condition checks if the temperature is between 37.5°C and 38.0°C, outputting 2 for "Fever".
   - The remaining temperatures (below 37.5°C) output 3 for "Normal".
3. **Output**: The result is printed immediately after the appropriate condition is met, ensuring efficient and straightforward classification.

This approach efficiently categorizes the temperature with minimal computational steps, leveraging simple conditional checks to determine the correct classification.
