The solution involves efficiently searching for such a triple without resorting to a brute-force approach, which would be computationally expensive for large input sizes.

### Approach
1. **Problem Analysis**: The problem requires finding three distinct elements in an array that sum to a given value X, with their original indices in increasing order. Given the constraints (N up to 10^6), a brute-force approach (O(N^3)) is infeasible. Instead, we use a sorting-based approach combined with a two-pointer technique to reduce the time complexity to O(N^2) in the worst case, which is more manageable for large N.

2. **Intuition**: By sorting the array while keeping track of original indices, we can leverage the two-pointer technique to efficiently find triplets that sum to X. For each element in the sorted array, we use two pointers to explore pairs of elements that, when added to the current element, sum to X. We then verify if the original indices of these elements are in increasing order.

3. **Algorithm Selection**: 
   - **Sorting**: The array is sorted by values while preserving the original indices. This allows us to use the two-pointer technique effectively.
   - **Two-pointer Technique**: For each element in the sorted array, we maintain two pointers (left and right) that start from the next element and the end of the array, respectively. These pointers help in finding pairs of elements that sum to the required value (X - current element).

4. **Complexity Analysis**: 
   - **Time Complexity**: The sorting step takes O(N log N) time. The two-pointer traversal for each element takes O(N) time per element, leading to an overall O(N^2) time complexity in the worst case. This is acceptable for the given constraints as it is significantly better than O(N^3).

### Solution Code
```python
import sys

def main():
    data = sys.stdin.read().split()
    if not data:
        print(-1)
        return
    
    n = int(data[0])
    X = int(data[1])
    A = list(map(int, data[2:2+n]))
    
    if n < 3:
        print(-1)
        return
        
    arr = [(A[i], i) for i in range(n)]
    arr.sort(key=lambda x: x[0])
    
    for i in range(n-2):
        left = i+1
        right = n-1
        while left < right:
            total = arr[i][0] + arr[left][0] + arr[right][0]
            if total < X:
                left += 1
            elif total > X:
                right -= 1
            else:
                if arr[i][1] < arr[left][1] < arr[right][1]:
                    print(f"{arr[i][1]+1} {arr[left][1]+1} {arr[right][1]+1}")
                    return
                    
    print(-1)

if __name__ == "__main__":
    main()
```
