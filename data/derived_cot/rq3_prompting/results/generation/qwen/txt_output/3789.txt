To solve this problem, we need to maximize the number of valid subarrays in an array of numbers from 1 to n after removing exactly one conflicting pair. A subarray is considered valid if it does not contain both elements of any remaining conflicting pair. 

### Approach
1. **Problem Analysis**: The problem requires counting the number of non-empty subarrays that do not contain any conflicting pairs after removing one such pair. The key insight is that removing a conflicting pair allows more subarrays to be considered valid, as those subarrays were previously invalidated by this pair.

2. **Intuition**: The total number of subarrays in an array of size n is \( \frac{n(n+1)}{2} \). For each conflicting pair, we can compute the number of subarrays that would become valid upon its removal. The optimal solution involves selecting the pair whose removal maximizes the count of such valid subarrays.

3. **Algorithm Selection**: 
   - **Precompute Forbidden Subarrays**: For each starting index, determine the minimal ending index such that the subarray contains at least one conflicting pair. This helps in efficiently calculating the number of invalid subarrays.
   - **Count Valid Subarrays After Removal**: For each conflicting pair, calculate the number of subarrays that contain this pair but no other conflicting pairs. The difference between the total subarrays and the invalid subarrays gives the count of valid subarrays after removal.

4. **Complexity Analysis**: The approach involves preprocessing to compute forbidden subarrays in \( O(n) \) time and then evaluating each conflicting pair in \( O(n) \) time. With \( m \) conflicting pairs, the total complexity is \( O(n \cdot m) \), which is feasible given the constraints (n up to \( 10^5 \) and m up to \( 2 \cdot n \)).

### Solution Code
```python
class Solution:
    def maxSubarrays(self, n: int, conflictingPairs: List[List[int]]) -> int:
        total_sub = n * (n + 1) // 2
        
        min_val = [10**9] * n
        for a, b in conflictingPairs:
            idx = a - 1
            if idx < n and b - 1 < min_val[idx]:
                min_val[idx] = b - 1
        
        ans = [10**9] * (n + 1)
        ans[n] = 10**9
        for s in range(n - 1, -1, -1):
            ans[s] = min(min_val[s], ans[s + 1])
        
        invalid_before = 0
        for s in range(n):
            if ans[s] != 10**9:
                invalid_before += (n - ans[s])
        
        best_count = 0
        for a, b in conflictingPairs:
            count_e = 0
            for l in range(a):
                if ans[l] == 10**9:
                    count_e += (n - b + 1)
                else:
                    if b - 1 < ans[l]:
                        count_e += (ans[l] - b + 1)
            best_count = max(best_count, count_e)
        
        return total_sub - invalid_before + best_count
```

### Explanation
1. **Precompute Forbidden Subarrays**: The `min_val` array is initialized to store the minimal ending index for each starting index such that the subarray contains a conflicting pair. This is updated for each conflicting pair.
2. **Calculate Minimal Ending Indices**: The `ans` array is computed to store the minimal ending index for each starting index, considering all conflicting pairs. This helps in determining the earliest point at which a subarray becomes invalid.
3. **Count Invalid Subarrays**: The `invalid_before` variable accumulates the count of subarrays that contain at least one conflicting pair before any removal.
4. **Evaluate Each Conflicting Pair**: For each conflicting pair, the algorithm calculates the number of subarrays that would become valid upon its removal. This involves checking all subarrays containing the pair and ensuring they do not contain any other conflicting pairs.
5. **Compute Result**: The result is derived by adjusting the total subarrays count with the invalid subarrays count and the best count of valid subarrays obtained by removing the optimal conflicting pair.

This approach efficiently narrows down the optimal conflicting pair to remove, maximizing the number of valid subarrays while adhering to the problem constraints.
