            n, K = map(int, input().split())

                new_freq = freq.copy()   # states from -1 to i-1

                # Determine allowed_letters
                if S[i] == '?':
                    letters = ['A','B','C']
                else:
                    letters = [S[i]]

                for state, total_dict in dp.items():
                    for letter in letters:
                        # Get the vector for the letter
                        if letter == 'A':
                            vec = (1,0,0)
                        elif letter == 'B':
                            vec = (0,1,0)
                        else: # 'C'
                            vec = (0,0,1)

                        new_state = (state[0]^vec[0], state[1]^vec[1], state[2]^vec[2])

                        # Calculate the good substrings ending at i for this new_state

                            if new_state not in new_dp:

                # Now, update the frequency for the new_state at i.
                # First, aggregate the counts for each state in new_dp.
                state_count_i = {}
                for state, total_dict in new_dp.items():
                    for count_val in total_dict:
                        state_count_i[state] = state_count_i.get(state,0) + total_dict[count_val]

                # Update new_freq with state_count_i
                for state, cnt in state_count_i.items():
                    new_freq[state] = new_freq.get(state,0) + cnt

                # Set dp = new_dp and freq = new_freq for the next iteration.
                dp = new_dp
                freq = new_freq

            # Now, after processing the entire string, sum the ways for all states and for all total_good_count >= K.
            ans = 0
            for state, total_dict in dp.items():
                for count_val, ways in total_dict.items():
                    if count_val >= K:

=== ANSWER ===

To solve this problem, we need to count the number of ways to replace each '?' in a given string with 'A', 'B', or 'C' such that the resulting string contains at least K contiguous good substrings. A good substring is defined as a substring that can be reduced to an empty string using two operations: deleting two identical characters or deleting one 'A', one 'B', and one 'C'.

### Approach
1. **Problem Analysis**: The key observation is that a substring is good if the counts of 'A', 'B', and 'C' in it have the same parity (all even or all odd). This simplifies the problem to checking the parity of character counts in every possible substring.
2. **Dynamic Programming (DP) Setup**: We use a DP approach where the state tracks the parity of counts of 'A', 'B', and 'C' encountered so far. For each position in the string, we consider all possible replacements (if the character is '?') and update the DP state accordingly.
3. **Good Substring Counting**: For each new character added, we count how many previous states (from the DP) would result in a good substring ending at the current position. This is done by checking if the current state matches any previous state or its complement (in terms of parity).
4. **Frequency Tracking**: We maintain a frequency dictionary to keep track of how many times each state (parity combination) has been encountered. This helps in efficiently calculating the number of good substrings ending at each position.
5. **Result Calculation**: After processing the entire string, we sum the number of ways to achieve at least K good substrings across all possible states.

### Solution Code
```python
MOD = 998244353

def main():
    import sys
    from collections import defaultdict
    data = sys.stdin.read().splitlines()
    if not data:
        return
    n, K = map(int, data[0].split())
    S = data[1].strip()
    
    dp = {(0, 0, 0): {0: 1}}
    freq = {(0, 0, 0): 1}
    
    for i in range(n):
        new_dp = {}
        new_freq = freq.copy()
        
        if S[i] == '?':
            letters = ['A', 'B', 'C']
        else:
            letters = [S[i]]
            
        for state, total_dict in dp.items():
            for letter in letters:
                if letter == 'A':
                    vec = (1, 0, 0)
                elif letter == 'B':
                    vec = (0, 1, 0)
                else:
                    vec = (0, 0, 1)
                    
                new_state = (state[0] ^ vec[0], state[1] ^ vec[1], state[2] ^ vec[2])
                comp = (1 - new_state[0], 1 - new_state[1], 1 - new_state[2])
                count_here = new_freq.get(new_state, 0) + new_freq.get(comp, 0)
                
                for total_count, ways in total_dict.items():
                    new_total = total_count + count_here
                    if new_state not in new_dp:
                        new_dp[new_state] = {}
                    new_dp[new_state][new_total] = (new_dp[new_state].get(new_total, 0) + ways) % MOD
        
        state_count_i = defaultdict(int)
        for state, total_dict in new_dp.items():
            for count_val in total_dict:
                state_count_i[state] += total_dict[count_val]
                
        for state, cnt in state_count_i.items():
            new_freq[state] = new_freq.get(state, 0) + cnt
            
        dp = new_dp
        freq = new_freq
        
    ans = 0
    for state, total_dict in dp.items():
        for count_val, ways in total_dict.items():
            if count_val >= K:
                ans = (ans + ways) % MOD
                
    print(ans % MOD)

if __name__ == '__main__':
    main()
```

### Explanation
1. **Initialization**: The DP starts with the empty prefix state `(0, 0, 0)` and a count of 0 good substrings.
2. **Processing Each Character**: For each character in the string, if it's '?', we consider all three possible letters ('A', 'B', 'C'). For each letter, we update the current state by XORing the letter's vector (which represents its parity contribution).
3. **Counting Good Substrings**: For each new state, we count how many previous states (including the complement) would result in a good substring ending at the current position. This count is added to the total good substrings for each way to reach the new state.
4. **Updating Frequencies**: After processing each position, we update the frequency dictionary to include the new states encountered, ensuring future positions can efficiently compute good substrings.
5. **Result Calculation**: After processing the entire string, we sum the number of ways all states have at least K good substrings, providing the answer modulo 998244353.

This approach efficiently leverages dynamic programming and parity checks to count valid configurations without explicitly enumerating all possible strings, making it feasible for the given constraints.
