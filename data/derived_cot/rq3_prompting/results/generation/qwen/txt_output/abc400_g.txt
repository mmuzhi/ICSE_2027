To solve this problem, we need to maximize the total price of forming K pairs of cakes, where the price of each pair is the maximum of the sums of the corresponding attributes (beauty, tastiness, popularity) of the two cakes in the pair. The challenge is to efficiently find the optimal pairs without resorting to brute-force methods, especially given the constraints where the number of cakes can be as large as 100,000.

### Approach
1. **Problem Analysis**: The problem requires forming K disjoint pairs of cakes such that the sum of the maximum values of the three attribute sums (beauty, tastiness, popularity) for each pair is maximized. Each cake can be used in at most one pair.
2. **Intuition**: The key observation is that the maximum price for any pair is determined by the highest sum achievable in any of the three attributes. For each attribute, the highest sum is the sum of the two largest values in that attribute. The optimal strategy involves greedily selecting pairs that achieve the highest possible sums first.
3. **Algorithm Selection**:
   - **Small N (N ≤ 100)**: For small datasets, generate all possible pairs, compute their prices, sort them in descending order, and select the top K pairs greedily.
   - **Large N (N > 100)**: For larger datasets, use a heuristic approach. For each of the K pairs, find the highest achievable sum (T) among the remaining cakes by considering the top two values in each attribute. Select a pair that achieves this sum (if available) and remove those cakes from further consideration.
4. **Complexity Analysis**: 
   - **Small N**: The number of pairs is O(N²), which is manageable for N ≤ 100 (4,950 pairs). Sorting and greedy selection is O(N² log N).
   - **Large N**: The heuristic approach processes each pair in O(N) time per step, making it feasible for large N under the constraint that the total number of cakes across all test cases is 100,000.

import sys

def main():
    data = sys.stdin.read().strip().split()
    if not data:
        return
    
    t = int(data[0])
    index = 1
    results = []
    
    for _ in range(t):
        N = int(data[index]); index += 1
        K = int(data[index]); index += 1
        cakes = []
        for i in range(N):
            x = int(data[index]); y = int(data[index+1]); z = int(data[index+2]); 
            index += 3
            cakes.append((x, y, z))
            
        if N <= 100:
            pairs = []
            for i in range(N):
                for j in range(i+1, N):
                    x1, y1, z1 = cakes[i]
                    x2, y2, z2 = cakes[j]
                    s1 = x1 + x2
                    s2 = y1 + y2
                    s3 = z1 + z2
                    price = max(s1, s2, s3)
                    pairs.append((price, i, j))
                    
            pairs.sort(key=lambda x: x[0], reverse=True)
            selected = [False] * N
            total_price = 0
            count = 0
            for (price, i, j) in pairs:
                if not selected[i] and not selected[j]:
                    selected[i] = True
                    selected[j] = True
                    total_price += price
                    count += 1
                    if count == K:
                        break
            results.append(str(total_price))
        else:
            remaining = set(range(N))
            total_price = 0
            for _ in range(K):
                if not remaining:
                    break
                X_vals = [cakes[i][0] for i in remaining]
                Y_vals = [cakes[i][1] for i in remaining]
                Z_vals = [cakes[i][2] for i in remaining]
                
                X_vals.sort(reverse=True)
                Y_vals.sort(reverse=True)
                Z_vals.sort(reverse=True)
                
                S_A = X_vals[0] + X_vals[1] if len(X_vals) >= 2 else 0
                S_B = Y_vals[0] + Y_vals[1] if len(Y_vals) >= 2 else 0
                S_C = Z_vals[0] + Z_vals[1] if len(Z_vals) >= 2 else 0
                T = max(S_A, S_B, S_C)
                
                candidate_pair = None
                if S_A == T and len(X_vals) >= 2:
                    set1 = [i for i in remaining if cakes[i][0] == X_vals[0]]
                    set2 = [i for i in remaining if cakes[i][0] == X_vals[1]]
                    if X_vals[0] == X_vals[1]:
                        if len(set1) >= 2:
                            candidate_pair = (set1[0], set1[1])
                    else:
                        if set1 and set2:
                            candidate_pair = (set1[0], set2[0])
                
                if candidate_pair is None and S_B == T and len(Y_vals) >= 2:
                    set1 = [i for i in remaining if cakes[i][1] == Y_vals[0]]
                    set2 = [i for i in remaining if cakes[i][1] == Y_vals[1]]
                    if Y_vals[0] == Y_vals[1]:
                        if len(set1) >= 2:
                            candidate_pair = (set1[0], set1[1])
                    else:
                        if set1 and set2:
                            candidate_pair = (set1[0], set2[0])
                
                if candidate_pair is None and S_C == T and len(Z_vals) >= 2:
                    set1 = [i for i in remaining if cakes[i][2] == Z_vals[0]]
                    set2 = [i for i in remaining if cakes[i][2] == Z_vals[1]]
                    if Z_vals[0] == Z_vals[1]:
                        if len(set1) >= 2:
                            candidate_pair = (set1[0], set1[1])
                    else:
                        if set1 and set2:
                            candidate_pair = (set1[0], set2[0])
                
                if candidate_pair is not None:
                    i1, i2 = candidate_pair
                    remaining.discard(i1)
                    remaining.discard(i2)
                    total_price += T
                else:
                    break
                    
            results.append(str(total_price))
            
    sys.stdout.write("\n".join(results))

if __name__ == "__main__":
    main()
