To solve this problem, we need to determine the number of stones each alien will have after N years, considering the rules of stone distribution when aliens become adults. The key challenge is efficiently calculating the number of stones each alien receives and gives without resorting to a brute-force simulation, which would be too slow for large inputs.

### Approach
1. **Problem Analysis**: Each alien becomes an adult at a specific year and receives stones from all previous adults who have at least one stone at that time. The stone count of each alien changes dynamically as they receive and give stones over the years.
2. **Insight**: The solution involves tracking the number of previous aliens (adults) who have stones available to give. This is done using a Fenwick Tree (or Binary Indexed Tree) to efficiently query and update the count of aliens who can give stones.
3. **Algorithm Selection**: 
   - **Fenwick Tree**: Used to maintain the running out years of each alien (the year by which they run out of stones). This allows efficient range queries to determine how many previous aliens can give stones at any given year.
   - **Dynamic Calculation**: For each alien, compute the number of stones they receive (based on previous aliens' stone availability) and the number of stones they give (based on their own stone count and the remaining years).
4. **Complexity Analysis**: The algorithm processes each alien once, performing logarithmic-time operations (O(log M)) for each query and update, where M is the maximum possible value (1,500,000). This results in an overall complexity of O(N log M), which is efficient for the given constraints.

### Solution Code
```python
import sys

class Fenw:
    def __init__(self, size):
        self.n = size
        self.tree = [0] * (self.n + 1)
    
    def update(self, index, delta):
        i = index
        while i <= self.n:
            self.tree[i] += delta
            i += i & -i
            
    def prefix_query(self, index):
        if index <= 0:
            return 0
        s = 0
        i = index
        while i:
            s += self.tree[i]
            i -= i & -i
        return s

def main():
    data = sys.stdin.read().split()
    n = int(data[0])
    A = list(map(int, data[1:1 + n]))
    
    max_val = 1500000
    fenw = Fenw(max_val)
    
    R = [0] * (n + 1)
    y = [0] * (n + 1)
    
    for i in range(1, n + 1):
        count_prev = (i - 1) - fenw.prefix_query(i - 1)
        R[i] = count_prev
        x_i = A[i - 1] + R[i]
        y_i = i + x_i
        fenw.update(y_i, 1)
    
    B = []
    for i in range(1, n + 1):
        x_i = A[i - 1] + R[i]
        G_i = min(x_i, n - i)
        B_i = A[i - 1] + R[i] - G_i
        B.append(str(B_i))
    
    print(" ".join(B))

if __name__ == "__main__":
    main()
```
