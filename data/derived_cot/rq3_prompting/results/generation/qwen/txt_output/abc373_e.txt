An election is being held with N candidates numbered 1, 2, \ldots, N. There are K votes, some of which have been counted so far.
Up until now, candidate i has received A_i votes.

N M K

- 1 \leq K \leq 10^{12}
- 0 \leq A_i \leq 10^{12}
- \displaystyle{\sum_{i=1}^{N} A_i} \leq K
- All input values are integers.

To solve this problem, we need to determine the minimum number of additional votes required for each candidate to guarantee their victory in an election, considering that victory is defined by having fewer than M candidates with more votes than them. The solution involves analyzing each candidate's potential to win under various scenarios of vote distribution.

### Approach
1. **Problem Analysis**: The problem requires checking for each candidate the minimal number of additional votes needed such that, no matter how the remaining votes are distributed, the candidate will be among the top M-1 candidates with the most votes. The key insight is that the worst-case scenario for a candidate is when other candidates receive votes in a way that maximizes the number of candidates with more votes than them.

2. **Binary Search**: For each candidate, we use binary search to determine the minimal number of additional votes (X) required. The binary search is performed over the range [0, total_remaining_votes], where total_remaining_votes is the sum of all remaining votes.

3. **Condition Check**: For each midpoint (X) during the binary search, we:
   - Calculate the candidate's total votes (V = A_i + X).
   - Count the number of other candidates already having more votes than V (count0).
   - Determine the maximum number of other candidates that can be made to have more votes than V (k_max) using the remaining votes (T_rem = total_remaining - X). This involves sorting the relevant candidates' votes in descending order and using prefix sums to efficiently compute the minimal votes needed.

4. **Result Determination**: If the sum of count0 and k_max is less than M, the candidate can be elected with X votes; otherwise, we continue the binary search. If no such X is found, the candidate cannot be elected (-1), and if X is zero, the candidate is already elected (0).

```python
import bisect

def main():
    import sys
    data = sys.stdin.read().strip().split()
    if not data:
        return
    
    it = iter(data)
    N = int(next(it)); M = int(next(it)); K = int(next(it))
    A = [int(next(it)) for _ in range(N)]
    
    total_votes = sum(A)
    total_remaining = K - total_votes
    
    T_all = sorted(A)
    
    results = []
    for i in range(N):
        low = 0
        high = total_remaining + 1
        ans = -1
        
        while low < high:
            mid = (low + high) // 2
            V = A[i] + mid
            T_rem = total_remaining - mid
            
            idx = bisect.bisect_right(T_all, V)
            count0 = N - idx
            if A[i] > V:
                count0 -= 1
            
            candidates = []
            for j in range(N):
                if j == i:
                    continue
                if A[j] <= V:
                    candidates.append(A[j])
            
            candidates.sort(reverse=True)
            prefix = [0]
            for val in candidates:
                prefix.append(prefix[-1] + val)
            
            lo_k = 0
            hi_k = len(candidates)
            best_k = 0
            while lo_k <= hi_k:
                mid_k = (lo_k + hi_k) // 2
                if mid_k == 0:
                    cost = 0
                else:
                    cost = mid_k * (V + 1) - prefix[mid_k]
                if cost <= T_rem:
                    best_k = mid_k
                    lo_k = mid_k + 1
                else:
                    hi_k = mid_k - 1
            
            if count0 + best_k < M:
                ans = mid
                high = mid
            else:
                low = mid + 1
        
        if ans == -1:
            results.append("-1")
        else:
            results.append(str(ans))
    
    print(" ".join(results))

if __name__ == '__main__':
    main()
```
