=== ANSWER ===

To solve this problem, we need to find the sum of all good subsequences in an integer array. A good subsequence is defined as one where the absolute difference between any two consecutive elements is exactly 1. Single-element subsequences are considered good by default. Given the constraints, a brute-force approach is infeasible, so we use dynamic programming to efficiently compute the solution.

### Approach
1. **Dynamic Programming Setup**: We use two arrays, `dp_count` and `dp_sum`, where:
   - `dp_count[x]` stores the number of good subsequences ending with the number `x`.
   - `dp_sum[x]` stores the sum of all elements in these subsequences.

2. **Processing Each Element**: For each element `x` in the array:
   - **Base Case**: The single-element subsequence `[x]` contributes `1` to `dp_count[x]` and `x` to `dp_sum[x]`.
   - **Extension Case**: For subsequences ending with `x`, we can extend subsequences ending with `x-1` or `x+1` (if they exist). The count of new subsequences is `dp_count[x-1] + dp_count[x+1]`, and their sum is `dp_sum[x-1] + dp_sum[x+1] + x * (dp_count[x-1] + dp_count[x+1])`.

3. **Modulo Operation**: Since the result can be very large, all operations are performed modulo \(10^9 + 7\).

4. **Summing Results**: After processing all elements, the total sum is obtained by summing up all values in `dp_sum`.

### Solution Code
```python
class Solution:
    def sumOfGoodSubsequences(self, nums: List[int]) -> int:
        mod = 10**9 + 7
        max_val = 10**5
        
        dp_count = [0] * (max_val + 1)
        dp_sum = [0] * (max_val + 1)
        
        for x in nums:
            base_count = 1
            base_sum = x
            
            count_prev = 0
            sum_prev = 0
            
            if x - 1 >= 0:
                count_prev = (count_prev + dp_count[x - 1]) % mod
                sum_prev = (sum_prev + dp_sum[x - 1]) % mod
            if x + 1 <= max_val:
                count_prev = (count_prev + dp_count[x + 1]) % mod
                sum_prev = (sum_prev + dp_sum[x + 1]) % mod
            
            new_count = (base_count + count_prev) % mod
            new_sum = (base_sum + sum_prev + x * count_prev) % mod
            
            dp_count[x] = (dp_count[x] + new_count) % mod
            dp_sum[x] = (dp_sum[x] + new_sum) % mod
        
        total = 0
        for i in range(max_val + 1):
            total = (total + dp_sum[i]) % mod
        
        return total
```

### Explanation
- **Initialization**: We initialize `dp_count` and `dp_sum` arrays to store counts and sums of good subsequences ending at each number up to \(10^5\).
- **Processing Elements**: For each number `x` in the input array, we consider it as the end of a new subsequence (base case) and extend existing subsequences ending with `x-1` or `x+1` (extension case).
- **Modulo Handling**: All additions and multiplications are performed modulo \(10^9 + 7\) to prevent integer overflow and adhere to problem constraints.
