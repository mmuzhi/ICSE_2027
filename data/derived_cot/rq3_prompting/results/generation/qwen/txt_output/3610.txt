=== ANSWER ===

To solve this problem, we need to compute the x-sum for every contiguous subarray of length `k` in the given array `nums`. The x-sum is defined as the sum of the top `x` most frequent elements in the subarray, with ties broken by selecting the larger element. If the subarray has fewer than `x` distinct elements, the x-sum is simply the sum of all elements in the subarray.

### Approach
1. **Sliding Window Technique**: We use a sliding window approach to efficiently process each subarray of length `k`. The window starts at the beginning of the array and moves one element at a time to the right.
2. **Frequency Dictionary**: For each window, maintain a frequency dictionary that counts occurrences of each element within the window.
3. **Sorting for Top Elements**: For each window, convert the frequency dictionary into a list of tuples (element, frequency) and sort this list primarily by frequency in descending order and secondarily by element value in descending order (to handle ties).
4. **Summing Top Elements**: After sorting, select the top `x` elements (or all elements if there are fewer than `x` distinct elements) and compute their sum, where each element is multiplied by its frequency.

### Solution Code
```python
class Solution:
    def findXSum(self, nums: List[int], k: int, x: int) -> List[int]:
        n = len(nums)
        from collections import defaultdict
        freq = defaultdict(int)
        for j in range(k):
            freq[nums[j]] += 1
        
        ans = []
        for i in range(n - k + 1):
            arr = [(num, count) for num, count in freq.items()]
            arr.sort(key=lambda x: (-x[1], -x[0]))
            total = 0
            for j in range(min(x, len(arr))):
                total += arr[j][0] * arr[j][1]
            ans.append(total)
            
            if i == n - k:
                break
            
            num_out = nums[i]
            freq[num_out] -= 1
            if freq[num_out] == 0:
                del freq[num_out]
            
            num_in = nums[i + k]
            freq[num_in] += 1
        
        return ans
```

### Explanation
1. **Initialization**: The frequency dictionary `freq` is initialized to count occurrences of each element in the first window (from index `0` to `k-1`).
2. **Processing Each Window**: For each window starting at index `i`:
   - **Frequency Dictionary to List**: Convert the frequency dictionary into a list of tuples (element, frequency).
   - **Sorting**: Sort the list by frequency in descending order and by element value in descending order for ties.
   - **Sum Calculation**: Sum the product of the top `x` elements' values and their frequencies.
3. **Sliding the Window**: After processing each window, remove the leftmost element (at index `i`) from the frequency dictionary (decrementing its count or removing it if the count drops to zero) and add the next element (at index `i + k`) to the dictionary.
4. **Termination**: The loop terminates after processing all windows, and the results are returned as a list.

This approach efficiently processes each subarray using a sliding window and maintains the necessary frequency counts, leveraging sorting to determine the top `x` elements for each window. The complexity is manageable due to the constraints, ensuring optimal performance.
