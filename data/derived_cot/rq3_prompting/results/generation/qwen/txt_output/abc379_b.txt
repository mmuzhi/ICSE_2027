To solve this problem, we need to determine the maximum number of strawberries Takahashi can eat, where each strawberry requires K consecutive healthy teeth (represented by 'O'). After eating a strawberry, those K teeth become unhealthy (change to 'X'). The challenge is to find the optimal sequence of removals to maximize the number of strawberries eaten.

### Approach
1. **Problem Analysis**: The problem involves processing a string of 'O's and 'X's, where each 'O' represents a healthy tooth. The goal is to find the maximum number of non-overlapping segments of length K consisting entirely of 'O's. Each time such a segment is found, it is replaced with 'X's, and the count of strawberries eaten is incremented.

2. **Intuition**: The optimal strategy is to always remove the leftmost possible segment of K consecutive 'O's. This approach ensures that we maximize the number of segments by efficiently utilizing the string from left to right, leaving the right part of the string for potential future segments.

3. **Algorithm Selection**: The algorithm involves scanning the string from left to right. For each position, it checks if the next K characters form a valid segment (all 'O's). If found, the segment is replaced with 'X's, and the count is incremented. The scan continues from the next position onwards, repeating the process until no more valid segments are found.

4. **Complexity Analysis**: The algorithm processes each character in the string at most once, leading to a time complexity of O(N*K), where N is the length of the string and K is the segment length. Given the constraints (N and K up to 100), this approach is efficient.

### Solution Code
```python
def main():
    import sys
    data = sys.stdin.read().splitlines()
    if not data:
        print(0)
        return
    first_line = data[0].split()
    N = int(first_line[0])
    K = int(first_line[1])
    S = data[1].strip()
    
    s_list = list(S)
    count = 0
    i = 0
    while i < N:
        if i + K <= N and all(s_list[j] == 'O' for j in range(i, i + K)):
            for j in range(i, i + K):
                s_list[j] = 'X'
            count += 1
            i += 1
        else:
            i += 1
            
    print(count)

if __name__ == "__main__":
    main()
```

4. **Output**: The total count of strawberries eaten is printed once the scan completes.
