To solve this problem, we need to determine whether we can transform sequence A into sequence B using a series of operations where each operation allows us to copy the value from one element to another within a specified distance K. The solution involves checking two main conditions to ensure the transformation is possible.

### Approach
1. **Check Distinct Values**: The first condition is that all distinct values in sequence B must be present in sequence A. If any value in B is not found in A, it's impossible to transform A into B, so the answer is "No".

2. **Check Feasibility**: The second condition is more nuanced. For each value in B, we need to ensure that there exists at least one occurrence of that value in A within the K-length neighborhood of every index where the target value in B differs from the initial value in A. This ensures that we can propagate the necessary values through the sequence using the allowed operations.

### Solution Code
```python
import sys
from collections import defaultdict

def main():
    data = sys.stdin.read().split()
    t = int(data[0])
    index = 1
    results = []
    
    for _ in range(t):
        N = int(data[index]); index += 1
        K = int(data[index]); index += 1
        A = list(map(int, data[index:index+N])); index += N
        B = list(map(int, data[index:index+N])); index += N
        
        distinct_A = set(A)
        distinct_B = set(B)
        
        if not distinct_B.issubset(distinct_A):
            results.append("No")
            continue
            
        possible = True
        for i in range(N):
            if B[i] != A[i]:
                found = False
                for j in range(max(0, i-K), min(N, i+K+1)):
                    if A[j] == B[i]:
                        found = True
                        break
                if not found:
                    possible = False
                    break
                    
        results.append("Yes" if possible else "No")
        
    for res in results:
        print(res)

if __name__ == "__main__":
    main()
```
