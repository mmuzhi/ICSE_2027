To solve this problem, we need to determine if a given string ends with the substring "san". The solution involves checking the last three characters of the string to see if they match "san". 

### Approach
1. **Problem Analysis**: The problem requires checking if the input string ends with the specific substring "san". The string length is constrained to be between 4 and 30 characters, ensuring that the string is long enough to have at least three characters at the end.
2. **Intuition**: The solution leverages the string method `endswith()` which efficiently checks if the string ends with the specified suffix. This method is straightforward and optimal for this problem.
3. **Algorithm Selection**: The chosen algorithm is a direct check using the `endswith()` method, which is both time and space efficient with a constant time complexity O(1) since it only checks the last three characters.
4. **Complexity Analysis**: The `endswith()` method operates in O(1) time and space, making it highly efficient for the given constraints.

### Solution Code
```python
def main():
    S = input().strip()
    if S.endswith('san'):
        print("Yes")
    else:
        print("No")

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The input string `S` is read from standard input and stripped of any leading or trailing whitespace.
2. **Checking Suffix**: The `endswith('san')` method checks if the string `S` ends with the substring "san". If it does, the program prints "Yes"; otherwise, it prints "No".
3. **Efficiency**: The solution efficiently checks the required condition in constant time, making it suitable for the given problem constraints. The use of the `endswith()` method ensures clarity and simplicity in the code.
