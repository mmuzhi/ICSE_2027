To solve this problem, we need to determine the minimum total cost required to adjust the lengths of Takahashi's teeth such that they fit together well. The teeth are considered to fit together well if two conditions are met: there exists a common sum H for all upper and lower teeth pairs, and the lengths of the upper teeth form a sequence where consecutive teeth differ by at most X. The cost is the sum of all reductions applied to the teeth.

### Approach
1. **Problem Analysis**: The problem requires finding the maximum integer H such that for each tooth, the sum of the upper and lower teeth lengths equals H, and the sequence of upper teeth lengths has consecutive differences not exceeding X. The cost is minimized by maximizing H, as the total cost is given by the sum of all original tooth lengths minus N times H.

2. **Key Insight**: The valid values of H are contiguous from 0 to the minimum possible sum of any tooth pair (min_sum). We can use binary search to efficiently find the maximum H for which the sequence conditions are satisfied.

3. **Feasibility Check**: For a given H, we check if there exists a sequence of upper teeth lengths (within their respective intervals) such that consecutive teeth differ by at most X. This involves:
   - **Interval Calculation**: For each tooth, the interval [L_i, R_i] is derived from H, where L_i is the maximum of 0 and (H - D_i), and R_i is the minimum of U_i and H.
   - **Two-Pass Validation**: A forward and backward pass ensure that the sequence of intervals allows consecutive teeth to differ by at most X.

4. **Binary Search**: We perform binary search on H from 0 to min_sum to find the highest H for which the feasibility check passes. The minimal cost is then calculated as the total sum of all tooth lengths minus N times the found H.

### Solution Code
```python
import sys

def main():
    data = sys.stdin.read().strip().split()
    if not data:
        return
    it = iter(data)
    N = int(next(it))
    X = int(next(it))
    U = []
    D = []
    for _ in range(N):
        u = int(next(it))
        d = int(next(it))
        U.append(u)
        D.append(d)
    
    min_sum = min(U[i] + D[i] for i in range(N))
    total_sum = sum(U[i] + D[i] for i in range(N))
    
    def feasible(H):
        L = [0] * N
        R = [0] * N
        for i in range(N):
            L[i] = max(0, H - D[i])
            R[i] = min(U[i], H)
        for i in range(N):
            if L[i] > R[i]:
                return False
        
        for i in range(1, N):
            L[i] = max(L[i], R[i-1] - X)
            R[i] = min(R[i], L[i-1] + X)
            if L[i] > R[i]:
                return False
        
        for i in range(N-2, -1, -1):
            L[i] = max(L[i], R[i+1] - X)
            R[i] = min(R[i], L[i+1] + X)
            if L[i] > R[i]:
                return False
        
        return True

    lo = 0
    hi = min_sum
    ans = 0
    while lo <= hi:
        mid = (lo + hi) // 2
        if feasible(mid):
            ans = mid
            lo = mid + 1
        else:
            hi = mid - 1
    
    print(total_sum - N * ans)

if __name__ == "__main__":
    main()
```
