   - **Base Case Handling**: If the array has only one element, the factor score is simply the square of that element.
   - **Entire Array Calculation**: Compute the factor score for the entire array without any removal.
   - **Single Element Removal**: For each element in the array, remove it and compute the factor score of the remaining array. Track the maximum factor score encountered during this process.
4. **Complexity Analysis**: The algorithm involves iterating over each element once and for each removal, computing the GCD and LCM of the remaining elements. The GCD and LCM computations are efficient for small numbers (as specified in constraints), making the overall complexity O(n^2) where n is the number of elements in the array. Given the constraints (n ≤ 100), this approach is feasible.

### Solution Code
```python
from functools import reduce
import math

class Solution:
    def maxScore(self, nums: List[int]) -> int:
        n = len(nums)
        if n == 1:
            return nums[0] * nums[0]
        
        def lcm(a, b):
            return a * b // math.gcd(a, b)
        
        gcd_all = reduce(math.gcd, nums)
        lcm_all = reduce(lcm, nums)
        best_score = gcd_all * lcm_all
        
        for i in range(n):
            new_nums = nums[:i] + nums[i+1:]
            gcd_val = reduce(math.gcd, new_nums)
            lcm_val = reduce(lcm, new_nums)
            score = gcd_val * lcm_val
            if score > best_score:
                best_score = score
        
        return best_score
```

### Explanation
1. **Base Case Handling**: The solution first checks if the array has only one element. In this case, the factor score is the square of that element.
2. **GCD and LCM Calculation**: The helper function `lcm` computes the LCM of two numbers using their GCD. The `reduce` function is used to compute the GCD and LCM for the entire array.
3. **Factor Score Calculation**: The initial factor score for the entire array is computed and stored as `best_score`.
4. **Single Element Removal**: For each element in the array, the code removes the element, computes the GCD and LCM of the remaining array, and updates `best_score` if the resulting factor score is higher.
5. **Result**: After evaluating all possible single-element removals, the maximum factor score encountered is returned.

This approach efficiently checks all possible scenarios (no removal or exactly one removal) to determine the maximum factor score, leveraging the associative properties of GCD and LCM for optimal performance.
