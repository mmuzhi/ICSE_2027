```python
                        if source[k] == pattern[j] and k in target_set:
                            found = k
                            break
                        k += 1
                    if found is not None:
                        j += 1
                        i = found + 1
                        kept += 1
                    else:
                        break
            else:
                i += 1
        
        return len(target_set) - kept
```

### Explanation
1. **Initialization**: We start by converting `targetIndices` into a set for O(1) lookups. We initialize pointers `j` for `pattern` and `i` for `source`, along with a counter `kept` to track the number of target characters used.

2. **Matching Loop**: The loop continues until either `pattern` is fully matched or we reach the end of `source`. For each character in `pattern`:
   - **Non-target Matching**: We search for the next occurrence of `pattern[j]` in `source` that is not in `target_set`. If found, we increment `j` and move `i` past the found character.
   - **Target Matching**: If no non-target character is found, we search for the earliest occurrence in `target_set`. If found, we increment `j`, move `i` past the found character, and increment `kept`.

3. **Result Calculation**: The result is the total number of target characters minus the number of target characters used (`kept`). This gives the maximum number of characters that can be removed while keeping `pattern` as a subsequence.

This approach efficiently balances between using non-target characters to maximize removals and falling back to target characters when necessary, ensuring optimal performance and correctness.
