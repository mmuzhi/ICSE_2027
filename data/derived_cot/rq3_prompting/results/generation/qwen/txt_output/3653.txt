To solve this problem, we need to find the maximum sum of a contiguous subarray in an array of integers, where the length of the subarray is divisible by a given integer `k`. The solution involves efficiently scanning the array while keeping track of prefix sums and their residues modulo `k` to identify valid subarrays.

### Approach
1. **Prefix Sum Calculation**: Compute the prefix sums of the array to facilitate quick calculation of subarray sums. The prefix array `prefix` is constructed such that `prefix[i]` is the sum of the first `i` elements of the array.
2. **Tracking Minimum Prefix Sums**: For each position `j` in the array, compute the residue `r = j % k`. The goal is to find a previous position `i` (where `i < j`) such that `i % k == r` and the subarray from `i` to `j-1` has a sum that is as large as possible. This is achieved by maintaining an array `min_val` where `min_val[r]` stores the smallest prefix sum encountered for any index with residue `r`.
3. **Maximizing Subarray Sum**: For each position `j`, the candidate subarray sum is calculated as `prefix[j] - min_val[r]`. The maximum of all such candidates across the array is the desired result.

### Solution Code
```python
class Solution:
    def maxSubarraySum(self, nums: List[int], k: int) -> int:
        n = len(nums)
        prefix = [0] * (n + 1)
        for i in range(1, n + 1):
            prefix[i] = prefix[i - 1] + nums[i - 1]
        
        min_val = [float('inf')] * k
        best = float('-inf')
        
        for j in range(n + 1):
            r = j % k
            candidate = prefix[j] - min_val[r]
            if candidate > best:
                best = candidate
            if prefix[j] < min_val[r]:
                min_val[r] = prefix[j]
        
        return best
```

### Explanation
1. **Prefix Array Construction**: The prefix array is built by iterating through the input array and accumulating the sums. This allows the sum of any subarray to be computed in constant time as the difference between two elements in the prefix array.
2. **Modulo Tracking**: For each index `j`, the residue `r = j % k` is computed. This residue helps in identifying valid starting indices `i` (where `i % k == r`) that form subarrays of length divisible by `k`.
3. **Dynamic Tracking of Minimum Prefix Sums**: The `min_val` array keeps track of the smallest prefix sum encountered for each residue. This ensures that when calculating the candidate subarray sum (`prefix[j] - min_val[r]`), the difference is maximized by minimizing the subtracted value.
4. **Result Calculation**: The algorithm iterates through the prefix array, updating the best candidate subarray sum whenever a larger valid sum is found. The final result is the maximum sum encountered during this iteration.

This approach efficiently narrows down the search for valid subarrays using modulo arithmetic and dynamic tracking of prefix sums, ensuring optimal performance with a linear time complexity.
