        # If grid is empty
        if not grid or not grid[0]:

        row_sets = [set(row) for row in grid]
        available = []

            # Only consider values in distinct_vals that are in the row
            avail = [val for val in distinct_vals if val in row_sets[i]]
            available.append(avail)

        from functools import lru_cache
        
        @lru_cache(maxsize=None)
        def dfs(i, used):
            # i: current row index
            # used: frozenset of used distinct_vals
            if i == len(grid):

                if value not in used:

                    best = max(best, value + dfs(i+1, new_used))

To solve this problem, we need to select one or more cells from a given 2D grid such that no two selected cells are in the same row, and all selected cells have unique values. The goal is to maximize the sum of the values of the selected cells.

### Approach
1. **Problem Analysis**: The problem requires selecting cells from different rows with unique values to maximize the sum. The grid can be up to 10x10, and cell values range from 1 to 100. The solution involves exploring all possible combinations of cells from different rows while ensuring uniqueness of values.
2. **Intuition**: The optimal solution will likely involve selecting the largest possible values first, as larger values contribute more to the sum. By limiting the consideration to the top 10 distinct values in the grid, we reduce the state space for dynamic programming, making the solution feasible.
3. **Algorithm Selection**: We use a depth-first search (DFS) approach with memoization. The state in the DFS is defined by the current row index and a set of used values. For each row, we consider all available values (from the top 10 distinct values) that haven't been used yet, and recursively explore the solution for the next row.
4. **Optimization**: By limiting the distinct values to the top 10, we ensure the state space remains manageable (10 * 2^10 = 10,240 states), which is feasible for a DFS with memoization.

from functools import lru_cache
from typing import List

class Solution:
    def maxScore(self, grid: List[List[int]]) -> int:
        if not grid or not grid[0]:
            return 0
        
        all_values = []
        for row in grid:
            all_values.extend(row)
        
        distinct_vals = sorted(set(all_values), reverse=True)[:10]
        if not distinct_vals:
            return 0
        
        row_sets = [set(row) for row in grid]
        available = []
        for i in range(len(grid)):
            avail = [val for val in distinct_vals if val in row_sets[i]]
            available.append(avail)
        
        @lru_cache(maxsize=None)
        def dfs(i, used):
            if i == len(grid):
                return 0
            best = dfs(i+1, used)
            for value in available[i]:
                if value not in used:
                    new_used = used | {value}

2. **Distinct Values Selection**: It then selects the top 10 distinct values from the collected values, sorted in descending order.
3. **Preprocessing**: For each row, it precomputes a list of available values (from the top 10 distinct values) that are present in the row.
4. **DFS with Memoization**: The DFS function explores each row, selecting an available value if it hasn't been used before. The memoization cache stores results of subproblems (state defined by current row and used values) to avoid redundant calculations.
