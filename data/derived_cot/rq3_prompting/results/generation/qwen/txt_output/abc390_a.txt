=== ANSWER ===

To solve this problem, we need to determine whether a given sequence of integers (a permutation of 1 to 5) can be sorted in ascending order by performing exactly one swap of two adjacent elements. The solution involves checking if the sequence is already sorted (in which case the answer is "No" because no operation is allowed) or if any single adjacent swap can transform the sequence into the sorted sequence [1, 2, 3, 4, 5].

### Approach
1. **Check if the sequence is already sorted**: If the sequence matches [1, 2, 3, 4, 5], output "No" because no operation is performed.
2. **Generate all possible sequences by swapping adjacent elements**: For each adjacent pair in the sequence, create a new sequence by swapping the pair and check if the new sequence is sorted.
3. **Determine the result**: If any of the generated sequences matches the sorted sequence, output "Yes"; otherwise, output "No".

This approach efficiently checks all possible single adjacent swaps in a sequence of fixed length (5 elements), ensuring correctness with minimal computational overhead.

### Solution Code
```python
def main():
    arr = list(map(int, input().split()))
    sorted_arr = [1, 2, 3, 4, 5]
    
    if arr == sorted_arr:
        print("No")
        return
        
    n = len(arr)
    found = False
    for i in range(n - 1):
        candidate = arr[:i] + [arr[i+1], arr[i]] + arr[i+2:]
        if candidate == sorted_arr:
            found = True
            break
            
    print("Yes" if found else "No")

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The input is read as a list of integers.
2. **Check for Sorted Sequence**: If the input sequence is already sorted, the program immediately outputs "No" since no operation is allowed.
3. **Generating Candidates**: For each adjacent pair in the sequence, the program generates a candidate sequence by swapping the pair. Each candidate is checked against the sorted sequence.
4. **Result Determination**: If any candidate matches the sorted sequence, the program outputs "Yes". If none do, it outputs "No".

This method ensures that all possible single adjacent swaps are considered, providing an efficient and correct solution to the problem.
