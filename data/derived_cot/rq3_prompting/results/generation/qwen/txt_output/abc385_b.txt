You will be given a competitive programming problem.
There is a grid with H rows and W columns. Let (i,j) denote the cell at the i-th row from the top and the j-th column from the left.
If S_{i,j} is #, the cell (i,j) is impassable; if it is ., the cell is passable and contains no house; if it is @, the cell is passable and contains a house.
Initially, Santa Claus is in cell (X,Y). He will act according to the string T as follows.

- Let |T| be the length of the string T. For i=1,2,\ldots,|T|, he moves as follows.
- Let (x,y) be the cell he is currently in.
- If T_i is U and cell (x-1,y) is passable, move to cell (x-1,y).
- If T_i is D and cell (x+1,y) is passable, move to cell (x+1,y).
- If T_i is L and cell (x,y-1) is passable, move to cell (x,y-1).
- If T_i is R and cell (x,y+1) is passable, move to cell (x,y+1).
- Otherwise, stay in cell (x,y).





Find the cell where he is after completing all actions, and the number of distinct houses that he passed through or arrived at during his actions. If the same house is passed multiple times, it is only counted once.

Input

The Input is given from Standard Input in the following format:
H W X Y
S_{1,1}S_{1,2}\ldots S_{1,W}
\dots
S_{H,1}S_{H,2}\ldots S_{H,W}
T

Output

Let (X,Y) be the cell where he is after completing all actions, and C be the number of distinct houses he passed through or arrived at during his actions. Print X,Y,C in this order separated by spaces.

Constraints


- 3 \leq H,W \leq 100
- 1 \leq X \leq H
- 1 \leq Y \leq W
- All given numbers are integers.
- Each S_{i,j} is one of #, ., @.
- S_{i,1} and S_{i,W} are # for every 1 \leq i \leq H.
- S_{1,j} and S_{H,j} are # for every 1 \leq j \leq W.
- S_{X,Y}= .
- T is a string of length at least 1 and at most 10^4, consisting of U, D, L, R.

To solve this problem, we need to simulate Santa Claus's movement on a grid according to a given sequence of directions, while keeping track of the distinct houses he visits or passes through. The grid contains impassable walls ('#'), passable empty cells ('.'), and houses ('@'). Santa moves according to the directions provided in the string, but only if the next cell is passable and within bounds. The solution involves parsing the grid and the movement string, simulating each move, and counting distinct houses encountered.

### Approach
1. **Parse Input**: Read the grid dimensions (H rows and W columns), starting position (X, Y), the grid itself, and the movement string T.
2. **Initialize Position and Tracking Set**: Convert the starting position to 0-indexed for easier array manipulation. Use a set to keep track of distinct house positions visited.
3. **Simulate Movement**: For each character in the movement string T:
   - Determine the next cell based on the current direction (U, D, L, R).
   - Check if the next cell is within bounds and not a wall ('#'). If valid, move to the next cell.
   - After each move (or stay), check if the current cell contains a house ('@'). If so, add its position to the set of visited houses.
4. **Output Result**: After processing all moves, output the final position (converted back to 1-indexed) and the count of distinct houses visited.

### Solution Code
```python
def main():
    import sys
    data = sys.stdin.read().splitlines()
    if not data:
        return
    
    first_line = data[0].split()
    H = int(first_line[0])
    W = int(first_line[1])
    X = int(first_line[2])
    Y = int(first_line[3])
    
    grid = []
    for i in range(1, 1 + H):
        grid.append(data[i].strip())
    
    T = data[1 + H].strip()
    
    x = X - 1
    y = Y - 1
    visited_houses = set()
    
    for move in T:
        dx, dy = 0, 0
        if move == 'U':
            dx = -1
        elif move == 'D':
            dx = 1
        elif move == 'L':
            dy = -1
        elif move == 'R':
            dy = 1
            
        nx = x + dx
        ny = y + dy
        
        if 0 <= nx < H and 0 <= ny < W and grid[nx][ny] != '#':
            x, y = nx, ny
            
        if grid[x][y] == '@':
            visited_houses.add((x, y))
            
    print(f"{x + 1} {y + 1} {len(visited_houses)}")

if __name__ == "__main__":
    main()
```

3. **Movement Simulation**: For each direction in the movement string:
   - The next cell is calculated based on the direction (U: up, D: down, L: left, R: right).
   - If the next cell is within bounds and not a wall, Santa moves to that cell.
   - After each move (or stay), if the current cell contains a house ('@'), its position is added to the set of visited houses. The set ensures each house is counted only once, regardless of multiple visits.
