To solve this problem, we need to find the minimum value of the expression \((\max_{i \in S} A_i) \times (\sum_{i \in S} B_i)\) for a subset \(S\) of size \(K\) from the given sequences \(A\) and \(B\). The solution involves efficiently selecting elements from the sequences to minimize this expression.

### Approach
1. **Problem Analysis**: The key observation is that the maximum element in the subset \(S\) determines the first part of the expression, while the sum of the corresponding \(B_i\) values determines the second part. To minimize the product, we need to balance between a small maximum \(A_i\) and a small sum of \(B_i\) values in the subset.

2. **Intuition**: By sorting the elements based on \(A_i\) (and \(B_i\) in case of ties), we can consider each element as the potential maximum in the subset. For each element, we need to select the smallest \(K-1\) \(B_i\) values from elements with \(A_i\) less than or equal to the current element's \(A_i\) (excluding the current element itself).

3. **Algorithm Selection**: We use a max-heap to keep track of the largest \(B_i\) values in the smallest \(K-1\) elements encountered so far. This allows us to efficiently maintain the sum of the smallest \(K-1\) \(B_i\) values. For each element, if there are at least \(K-1\) elements before it in the sorted list, we compute the candidate value and update the heap if necessary.

4. **Complexity Analysis**: The algorithm involves sorting the elements, which takes \(O(N \log N)\) time, and processing each element with heap operations, each taking \(O(\log K)\) time. Thus, the overall time complexity is \(O(N \log N)\) per test case, which is efficient given the constraints.

### Solution Code
```python
import sys
import heapq

def main():
    data = sys.stdin.read().split()
    t = int(data[0])
    index = 1
    results = []
    
    for _ in range(t):
        N = int(data[index])
        index += 1
        K = int(data[index])
        index += 1
        A = list(map(int, data[index:index+N]))
        index += N
        B = list(map(int, data[index:index+N]))
        index += N
        
        arr = [(A[i], B[i]) for i in range(N)]
        arr.sort(key=lambda x: (x[0], x[1]))
        
        heap = []
        total = 0
        count = 0
        ans = float('inf')
        
        for a, b in arr:
            if count >= K-1:
                candidate = a * (total + b)
                if candidate < ans:
                    ans = candidate
                    
            if count < K-1:
                heapq.heappush(heap, -b)
                total += b
                count += 1
            else:
                if heap and b < -heap[0]:
                    popped = -heapq.heappop(heap)
                    total = total - popped + b
                    heapq.heappush(heap, -b)
                    
        results.append(str(ans))
        
    print("\n".join(results))

if __name__ == '__main__':
    main()
```
