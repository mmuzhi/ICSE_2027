class Solution:
    def maxScore(self, n: int, k: int, stayScore: List[List[int]], travelScore: List[List[int]]) -> int:
        # dp_prev: previous day's dp array, length n
        dp_prev = [stayScore[0][j] for j in range(n)]
        
        # If k is 1, we don't need to loop.
        for day in range(1, k):
            dp_curr = [0] * n
            # For each city j, compute the best way to end up in j on day 'day'
            for j in range(n):
                # Option 1: staying in j
                option1 = dp_prev[j] + stayScore[day][j]
                # Option 2: moving to j from some other city m
                option2 = -10**18  # a very small number
                for m in range(n):
                    if m == j:
                        continue
                    # From city m to city j, the points from travelScore is travelScore[m][j]
                    candidate = dp_prev[m] + travelScore[m][j]
                    if candidate > option2:
                        option2 = candidate
                dp_curr[j] = max(option1, option2)
            dp_prev = dp_curr  # move to next day
        
        return max(dp_prev)

To solve this problem, we need to maximize the points a tourist can earn over exactly `k` days by either staying in the current city or moving to another city each day. The solution involves dynamic programming to keep track of the maximum points achievable at the end of each day for each possible city.

### Approach
1. **Problem Analysis**: The tourist starts on day 0 in any city. Each subsequent day, they can either stay in the current city, earning points from `stayScore`, or move to another city, earning points from `travelScore`. The goal is to maximize the total points over `k` days.

2. **Dynamic Programming Setup**: We use a dynamic programming array `dp_prev` where `dp_prev[j]` represents the maximum points achievable by ending day 0 in city `j`. This is initialized by considering both staying in city `j` (using `stayScore[0][j]`) or moving into city `j` from any other city (using `travelScore[m][j]` for all `m != j`).

3. **Iterative DP Update**: For each subsequent day (from 1 to `k-1`), we update the DP array (`dp_curr`) by considering two options for each city `j`:
   - **Stay Option**: Add `stayScore[day][j]` to the points from the previous day's ending in city `j`.
   - **Move Option**: Add `travelScore[m][j]` to the points from the previous day's ending in any other city `m` (where `m != j`).

4. **Result Extraction**: After processing all `k` days, the result is the maximum value in the final DP array, representing the highest points achievable by ending in any city after `k` days.

class Solution:
    def maxScore(self, n: int, k: int, stayScore: List[List[int]], travelScore: List[List[int]]) -> int:
        dp_prev = [0] * n
        for j in range(n):
            option1 = stayScore[0][j]

                candidate = travelScore[m][j]
                if candidate > option2:
                    option2 = candidate
            dp_prev[j] = max(option1, option2)
        
        if k == 1:
            return max(dp_prev)

2. **Day Processing**: For each subsequent day, the algorithm computes the maximum points achievable for each city by either staying (adding the day's `stayScore`) or moving from any other city (adding the corresponding `travelScore`). 3. **Result Calculation**: After processing all `k` days, the maximum value in the `dp_prev` array gives the highest possible points the tourist can earn, considering all possible paths and actions over the `k` days.
