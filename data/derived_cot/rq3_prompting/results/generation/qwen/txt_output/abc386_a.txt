To solve this problem, we need to determine whether adding one card to a given set of four cards can form a Full House. A Full House is defined as a set of five cards where three cards have the same integer and the remaining two cards have another integer.

### Approach
1. **Problem Analysis**: The problem requires checking if the four given cards can be extended to a Full House by adding exactly one card. The Full House condition mandates that the five cards must consist of three cards of one integer and two cards of another distinct integer.
2. **Key Insight**: The solution hinges on analyzing the frequency distribution of the four given cards. If there are exactly two distinct integers in the four cards, then the counts of these integers must either be (3, 1) or (2, 2). In the (3, 1) case, adding the missing integer (the one with count 1) will form a Full House. In the (2, 2) case, adding either of the two integers will form a Full House.
3. **Algorithm Selection**: The algorithm involves:
   - Reading the four integers.
   - Checking if there are exactly two distinct integers in the set.
   - If there are two distinct integers, verifying if their counts are either (1, 3) or (2, 2).
   - Outputting "Yes" if the condition is met, otherwise "No".

### Solution Code
```python
def main():
    A, B, C, D = map(int, input().split())
    cards = [A, B, C, D]
    distinct_count = len(set(cards))
    
    if distinct_count == 2:
        freq = {}
        for card in cards:
            freq[card] = freq.get(card, 0) + 1
        counts = sorted(freq.values())
        if counts == [1, 3] or counts == [2, 2]:
            print("Yes")
        else:
            print("No")
    else:
        print("No")

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The four integers are read from the input and stored in a list.
2. **Distinct Count Check**: The number of distinct integers in the list is determined. If there are not exactly two distinct integers, it's impossible to form a Full House by adding one card, so the answer is "No".
3. **Frequency Calculation**: If there are two distinct integers, their frequencies are counted. The sorted list of these frequencies is checked to see if it matches [1, 3] or [2, 2]. 
4. **Result Determination**: If the frequencies match either [1, 3] or [2, 2], adding the appropriate card will form a Full House, so the answer is "Yes". Otherwise, the answer is "No".

This approach efficiently checks the necessary conditions to determine if a Full House can be formed with the given four cards by adding one card, leveraging simple frequency analysis and set operations.
