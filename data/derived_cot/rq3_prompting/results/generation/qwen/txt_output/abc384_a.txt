=== ANSWER ===

To solve this problem, we need to replace every character in a given string that is not a specified character `c_1` with another specified character `c_2`. The solution involves reading the input values, processing the string according to the given criteria, and outputting the modified string.

### Approach
1. **Read Input**: The first line of input contains an integer `N` (the length of the string) and two lowercase letters `c_1` and `c_2`. The second line contains the string `S` of length `N`.
2. **Process String**: Iterate over each character in the string `S`. For each character, if it is not equal to `c_1`, replace it with `c_2`; otherwise, leave it unchanged.
3. **Output Result**: Construct the resulting string by joining the processed characters and print it.

This approach efficiently processes each character in the string in linear time, making it optimal for the given constraints (string length up to 100).

### Solution Code
```python
def main():
    data = input().split()
    N = int(data[0])
    c1 = data[1]
    c2 = data[2]
    S = input().strip()
    
    result = ''.join(c2 if char != c1 else char for char in S)
    print(result)

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The first line of input is split into three parts: `N` (the length of the string), `c_1` (the character to keep), and `c_2` (the character to replace others with). The second line is read as the string `S`.
2. **String Processing**: Using a list comprehension, each character in `S` is checked. If the character is not `c_1`, it is replaced with `c_2`; otherwise, it remains unchanged. The processed characters are joined into a single string.
3. **Output**: The resulting string is printed, which is the original string with all non-`c_1` characters replaced by `c_2`.

This solution efficiently handles the problem requirements by leveraging simple string operations and ensures correctness by directly processing each character according to the given conditions.
