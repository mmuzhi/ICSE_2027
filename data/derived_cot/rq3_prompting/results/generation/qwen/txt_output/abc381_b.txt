=== ANSWER ===

To solve this problem, we need to determine if a given string meets all the specified conditions to be classified as a 1122 string. The conditions are:
1. The string's length must be even.
2. For each pair of characters at positions (2i-1) and 2i (0-indexed), the characters must be the same.
3. Each character in the string must appear exactly zero or two times.

### Approach
1. **Check String Length**: The first condition requires the string length to be even. If the length is odd, the string immediately fails the check.
2. **Check Consecutive Pairs**: The second condition requires that every two consecutive characters (at positions 0-1, 2-3, etc.) must be the same. If any pair fails this check, the string is invalid.
3. **Check Character Frequencies**: The third condition mandates that each character in the string must appear exactly twice. If any character appears a different number of times (not zero or two), the string is invalid.

### Solution Code
```python
S = input().strip()

n = len(S)
if n % 2 != 0:
    print("No")
    exit()

for i in range(0, n, 2):
    if S[i] != S[i+1]:
        print("No")
        exit()

from collections import Counter
freq = Counter(S)
for count in freq.values():
    if count != 2:
        print("No")
        exit()

print("Yes")
```

### Explanation
1. **Input Handling**: The input string is read and stripped of any leading or trailing whitespace.
2. **Even Length Check**: The code first checks if the string length is even. If not, it prints "No" and exits.
3. **Consecutive Pairs Check**: The code then verifies that every two consecutive characters (at even indices) are the same. If any pair fails this check, it prints "No" and exits.
4. **Character Frequency Check**: Using the `Counter` from the `collections` module, the code counts the frequency of each character in the string. If any character's count is not exactly two, it prints "No" and exits.
5. **Output Result**: If all checks pass, the code prints "Yes", confirming the string is a 1122 string.

This approach efficiently verifies all three conditions in sequence, ensuring the solution is both correct and optimal for the given constraints.
