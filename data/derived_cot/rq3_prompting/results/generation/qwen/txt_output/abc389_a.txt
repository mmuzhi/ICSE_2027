=== ANSWER ===

To solve this problem, we need to extract two digits from a given 3-character string where the first and third characters are digits, and the second character is 'x'. The task is to multiply these two digits and output the result.

### Approach
1. **Problem Analysis**: The input string is guaranteed to be exactly 3 characters long. The first and third characters are digits (from '1' to '9'), and the second character is always 'x'. Our goal is to extract these two digits, convert them into integers, and compute their product.
2. **Intuition**: By leveraging string indexing, we can directly access the first and third characters of the string. These characters are then converted from strings to integers using the `int()` function. The product of these two integers is the solution.
3. **Algorithm Selection**: The algorithm involves simple string manipulation and arithmetic operations. No complex data structures or algorithms are needed here.
4. **Complexity Analysis**: The solution involves constant-time operations (indexing and conversion) making it O(1) in both time and space complexity.

### Solution Code
```python
def main():
    S = input().strip()
    first_digit = int(S[0])
    second_digit = int(S[2])
    result = first_digit * second_digit
    print(result)

if __name__ == '__main__':
    main()
```

### Explanation
1. **Reading Input**: The input string `S` is read from standard input.
2. **Extracting Digits**: The first character (`S[0]`) and the third character (`S[2]`) of the string are extracted.
3. **Conversion to Integers**: These characters are converted from strings to integers using the `int()` function.
4. **Multiplication**: The two integers are multiplied together.
5. **Output Result**: The result of the multiplication is printed.

This approach efficiently handles the problem by directly accessing and converting the relevant parts of the input string, ensuring correctness and simplicity.
