=== ANSWER ===

To solve this problem, we need to generate a four-digit key by taking the smallest digit from each corresponding position of three given numbers after padding them with leading zeros to ensure they all have four digits. The key is then formed by combining these smallest digits and returned as an integer without any leading zeros.

### Approach
1. **Padding Numbers**: Convert each of the three numbers into strings and pad them with leading zeros to make each string exactly four characters long. This ensures that each number has the same length for digit-wise comparison.
2. **Digit-wise Comparison**: For each of the four positions (from left to right), compare the digits at that position in all three strings and select the smallest one.
3. **Forming the Key**: Combine the selected smallest digits into a four-digit string. This string may contain leading zeros, which are acceptable during formation but should be removed when converting to an integer.
4. **Conversion to Integer**: Convert the combined four-digit string into an integer to eliminate any leading zeros and return the result.

### Solution Code
```python
class Solution:
    def generateKey(self, num1: int, num2: int, num3: int) -> int:
        s1 = str(num1).zfill(4)
        s2 = str(num2).zfill(4)
        s3 = str(num3).zfill(4)
        
        key_digits = []
        for i in range(4):
            min_digit = min(s1[i], s2[i], s3[i])
            key_digits.append(min_digit)
        
        key_str = ''.join(key_digits)
        return int(key_str)
```

### Explanation
1. **Padding Numbers**: The `zfill(4)` method is used to pad each number to four digits with leading zeros. For example, the number 1 becomes "0001".
2. **Digit-wise Comparison**: For each position (0 to 3), the smallest digit among the three numbers at that position is selected. For instance, if the digits at position 0 are '0', '0', and '1', the smallest digit '0' is chosen.
3. **Forming the Key**: The selected digits are combined into a string. For example, digits '0', '0', '0', '0' form the string "0000".
4. **Conversion to Integer**: The combined string is converted to an integer, which automatically removes any leading zeros. For example, "0000" becomes 0, and "0001" becomes 1.

This approach efficiently handles the problem by leveraging string manipulation and digit-wise comparison, ensuring the key is generated correctly according to the given specifications.
