To solve this problem, we need to count the number of ordered pairs of rest areas (s, t) such that the minimum number of steps to walk clockwise from s to t is a multiple of M. The rest areas are arranged in a circle, and the steps between consecutive rest areas are given. The challenge is to efficiently compute these pairs without resorting to a brute-force approach, which would be computationally expensive for large N.

### Approach
1. **Prefix Sum Calculation**: Compute the prefix sums of the steps between consecutive rest areas. This helps in quickly determining the clockwise distance between any two rest areas.
2. **Modulo Handling**: For each rest area, compute the prefix sum modulo M. This allows us to use modular arithmetic to check divisibility conditions efficiently.
3. **Frequency Arrays**: Use frequency arrays to count the number of valid pairs:
   - **Case 1**: For pairs where the starting rest area is before the destination (s < t), the condition is that the difference in their prefix sums modulo M is zero.
   - **Case 2**: For pairs where the starting rest area is after the destination (s > t), the condition involves the total steps modulo M and the prefix sums of both rest areas.
4. **Efficient Counting**: By traversing the rest areas and maintaining frequency counts, we can efficiently count valid pairs in linear time, leveraging the properties of modular arithmetic.

### Solution Code
```python
import sys

def main():
    data = sys.stdin.read().split()
    if not data:
        print(0)
        return
    N = int(data[0])
    M = int(data[1])
    A = list(map(int, data[2:2+N]))
    
    S = [0] * (N+1)
    for i in range(1, N+1):
        S[i] = S[i-1] + A[i-1]
        
    T = S[N] % M
    
    R = [0] * N
    for i in range(N):
        R[i] = S[i] % M
        
    if M == 0:
        print(0)
        return
        
    freq_case1 = [0] * M
    for i in range(N):
        freq_case1[R[i]] += 1
        
    count1 = 0
    for i in range(N):
        r = R[i]
        freq_case1[r] -= 1
        count1 += freq_case1[r]
        
    freq_case2 = [0] * M
    count2 = 0
    for i in range(N):
        required = (R[i] - T + M) % M
        count2 += freq_case2[required]
        freq_case2[R[i]] += 1
        
    print(count1 + count2)

if __name__ == '__main__':
    main()
```

### Explanation
1. **Prefix Sum Calculation**: The array `S` is computed where `S[i]` represents the sum of steps from the first rest area up to the (i-1)th rest area. This helps in determining the clockwise distance between any two rest areas.
2. **Modulo Handling**: The array `R` stores each prefix sum modulo M, simplifying the divisibility checks.
3. **Frequency Arrays**:
   - **Case 1**: The frequency array `freq_case1` counts how many times each residue appears. For each rest area, we decrement its frequency and count how many subsequent rest areas share the same residue, which gives the number of valid pairs where the starting rest area is before the destination.
   - **Case 2**: The frequency array `freq_case2` counts valid pairs where the starting rest area is after the destination. For each rest area, we compute the required residue that would make the clockwise distance a multiple of M and count how many previous rest areas have this residue.
4. **Efficiency**: The algorithm efficiently counts all valid pairs in O(N + M) time, which is optimal given the constraints. The use of modular arithmetic ensures that the solution handles large numbers and avoids overflow issues.
