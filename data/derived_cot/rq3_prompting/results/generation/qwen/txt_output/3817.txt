To solve this problem, we need to determine the maximum number of containers that can be loaded onto an n x n cargo deck without exceeding the ship's weight capacity. Each container has a fixed weight, and the deck has a limited number of cells. The solution involves finding the minimum between the total number of cells available on the deck and the maximum number of containers that can be carried without exceeding the weight limit.

### Approach
1. **Problem Analysis**: The problem requires us to maximize the number of containers loaded onto the deck, considering both the physical constraints (the deck size) and the weight constraints (the ship's capacity). Each container weighs exactly `w`, and the total weight of all containers must not exceed `maxWeight`.
2. **Key Insight**: The solution hinges on two main constraints:
   - The number of containers cannot exceed the total number of cells on the deck, which is `n * n`.
   - The total weight of the containers must not exceed `maxWeight`, so the maximum number of containers by weight is `maxWeight // w`.
3. **Algorithm Selection**: The solution involves calculating both constraints and returning the smaller of the two values, as this represents the maximum number of containers that can be loaded without violating either constraint.
4. **Complexity Analysis**: The approach involves basic arithmetic operations (multiplication and division) which run in constant time, O(1), making it highly efficient even for the upper constraint limits.

### Solution Code
```python
class Solution:
    def maxContainers(self, n: int, w: int, maxWeight: int) -> int:
        total_cells = n * n
        max_by_weight = maxWeight // w
        return min(total_cells, max_by_weight)
```

### Explanation
1. **Total Cells Calculation**: The variable `total_cells` is computed as `n * n`, representing the maximum number of containers that can physically fit on the deck.
2. **Weight Constraint Calculation**: The variable `max_by_weight` is calculated as `maxWeight // w`, which gives the maximum number of containers that can be carried without exceeding the ship's weight capacity.
3. **Result Determination**: The result is the minimum of `total_cells` and `max_by_weight`, ensuring that the solution adheres to both the physical and weight constraints. This approach efficiently leverages simple arithmetic to derive the optimal solution.
