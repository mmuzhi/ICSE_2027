To solve this problem, we need to calculate the reverse degree of a given string. The reverse degree is defined as the sum of the products of each character's position in the reversed alphabet and its 1-indexed position in the string. 

### Approach
1. **Understanding the Reversed Alphabet**: The reversed alphabet assigns 'a' a value of 26, 'b' a value of 25, and so on, down to 'z' which has a value of 1. This can be derived by subtracting the character's position in the normal alphabet (where 'a' is 1, 'b' is 2, ..., 'z' is 26) from 27. Alternatively, we can compute the reversed value directly using the formula `26 - (ord(char) - ord('a'))`.

2. **Iterating Through the String**: For each character in the string, we calculate its reversed alphabet value and multiply it by its 1-indexed position (i.e., its position in the string starting from 1). We then accumulate these products to get the total reverse degree.

3. **Efficiency**: The algorithm processes each character exactly once, making it linear in time complexity, O(n), where n is the length of the string. This is efficient given the constraint that the string length does not exceed 1000.

### Solution Code
```python
class Solution:
    def reverseDegree(self, s: str) -> int:
        total = 0
        for i, char in enumerate(s):
            reversed_val = 26 - (ord(char) - ord('a'))
            total += reversed_val * (i + 1)
        return total
```

### Explanation
1. **Initialization**: We start by initializing a variable `total` to 0, which will hold the cumulative sum of the products of each character's reversed value and its position.
2. **Iteration and Calculation**: For each character in the string, we compute its reversed alphabet value using the formula `26 - (ord(char) - ord('a'))`. This formula effectively maps 'a' to 26, 'b' to 25, ..., and 'z' to 1.
3. **Accumulation**: The reversed value of each character is multiplied by its 1-indexed position (i.e., `i + 1` where `i` is the 0-based index) and added to `total`.
4. **Result**: After processing all characters, `total` contains the reverse degree of the string, which is returned as the result.

This approach efficiently computes the desired result by leveraging simple arithmetic operations and linear iteration through the string, ensuring optimal performance and correctness.
