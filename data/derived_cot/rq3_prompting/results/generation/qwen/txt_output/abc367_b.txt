### Approach
1. **Input Handling**: Read the input string and strip any leading or trailing whitespace.
2. **Check for Decimal Point**: Determine if the input string contains a decimal point.
3. **Process Decimal Input**: If a decimal point is present:
   - Split the string into integer and fractional parts.
   - Remove any leading zeros from the integer part to ensure it is in standard form.
   - Remove any trailing zeros from the fractional part.
   - If the fractional part becomes empty after removing trailing zeros, output only the integer part. Otherwise, output the integer part followed by the fractional part without trailing zeros.
4. **Process Non-Decimal Input**: If there is no decimal point, simply output the integer part after removing any leading zeros.

### Solution Code
```python
s = input().strip()
if '.' in s:
    integer_part, fractional_part = s.split('.', 1)
    integer_part = integer_part.lstrip('0') or '0'
    fractional_part = fractional_part.rstrip('0')
    if fractional_part:
        print(integer_part + '.' + fractional_part)
    else:
        print(integer_part)
else:
    integer_part = s.lstrip('0') or '0'
    print(integer_part)
```

### Explanation
1. **Input Handling**: The input string is read and any surrounding whitespace is stripped to ensure clean processing.
2. **Decimal Check**: The presence of a decimal point is checked to determine if the number has a fractional part.
3. **Splitting Parts**: If a decimal point is found, the string is split into integer and fractional parts.
4. **Leading Zero Removal**: The integer part is processed to remove any leading zeros. If the integer part becomes empty after stripping zeros, it defaults to '0'.
5. **Trailing Zero Removal**: The fractional part is processed to remove any trailing zeros. If the fractional part becomes empty after this operation, only the integer part is printed. Otherwise, the integer part and the processed fractional part are combined and printed.
6. **Non-Decimal Handling**: If there is no decimal point, the input is treated as an integer, with leading zeros removed, and printed.
