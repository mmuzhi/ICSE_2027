=== ANSWER ===

To solve this problem, we need to determine the number of empty squares on an 8x8 grid where a new piece can be placed such that it cannot be captured by any existing pieces. The capture condition is defined such that a piece placed on a square (i, j) can be captured if there is any existing piece in the same row or column.

### Approach
1. **Problem Analysis**: The grid consists of 8 rows and 8 columns. Each square is either empty ('.') or contains a piece ('#'). The goal is to count all empty squares where placing a piece would not result in it being captured by any existing piece. A piece is captured if there is any existing piece in the same row or column.

2. **Intuition**: For a square (i, j) to be a valid placement, two conditions must be met:
   - There should be no existing piece in row i.
   - There should be no existing piece in column j.
   - The square (i, j) must be empty (though the first two conditions imply this).

3. **Algorithm Selection**:
   - **Precompute Row and Column Presence**: For each row, check if there is at least one piece ('#'). Similarly, for each column, check if there is at least one piece.
   - **Count Valid Squares**: Iterate over each cell in the grid. If the row and column of the cell have no pieces, then the cell is a valid placement.

4. **Complexity Analysis**: The algorithm involves two passes over the grid: one to compute row and column presence (O(8*8) = O(64) operations) and another to count valid squares (also O(64) operations). Thus, the overall complexity is O(1) since the grid size is fixed (8x8).

### Solution Code
```python
def main():
    grid = []
    for _ in range(8):
        grid.append(input().strip())
    
    row_has_piece = [False] * 8
    for i in range(8):
        if '#' in grid[i]:
            row_has_piece[i] = True
            
    col_has_piece = [False] * 8
    for j in range(8):
        for i in range(8):
            if grid[i][j] == '#':
                col_has_piece[j] = True
                break
                
    count = 0
    for i in range(8):
        if row_has_piece[i]:
            continue
        for j in range(8):
            if col_has_piece[j]:
                continue
            count += 1
            
    print(count)

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The grid is read as 8 strings, each representing a row.
2. **Row Check**: For each row, we check if there is any '#' character. If found, the row is marked as having a piece.
3. **Column Check**: For each column, we check all rows for any '#' character. If found, the column is marked as having a piece.
4. **Counting Valid Squares**: For each cell (i, j), if neither row i nor column j has any piece, the cell is counted as a valid placement.
5. **Output**: The total count of valid squares is printed.

This approach efficiently checks all possible placements by leveraging precomputed row and column information, ensuring optimal performance for the given problem constraints.
