### Approach

Given the periodic nature, the solution involves leveraging the periodic property to break down the problem into manageable parts.
2. **Key Insights**: 
   - **Case 1**: The contiguous subsequence is entirely contained within two periods (i.e., its length is at most 2*N). We use a sliding window (two-pointer) technique on a duplicated array (two periods) to check if any contiguous subsequence sums to S.
   - **Case 2**: The contiguous subsequence consists of k full periods (each summing to T, the total sum of one period) plus a remainder (a contiguous part of the base period). Here, we check if S can be expressed as k*T + X, where X is the sum of a contiguous part in the base period.
   - **Special Case**: If S is a multiple of T and the number of full periods (k) is at least 3, then such a subsequence exists (as it would be k full periods).
3. **Algorithm Selection**: 
   - **Sliding Window (Two-pointer)**: Efficiently checks for contiguous subsequences summing to S within a bounded array (two periods).
   - **Modular Arithmetic**: Used to determine if S can be decomposed into full periods and a remainder.
4. **Complexity Analysis**: The algorithm processes the duplicated array (size 2*N) twice (once for Case 1 and once for Case 2), resulting in a linear time complexity O(N), which is efficient for the given constraints (N ≤ 2×10^5).

import sys
