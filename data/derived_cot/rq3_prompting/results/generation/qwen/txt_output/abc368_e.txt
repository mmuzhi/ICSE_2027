### Approach
1. **Problem Analysis**: The problem requires setting delays for trains such that for any two trains where the arrival city of the first matches the departure city of the second and the arrival time of the first is less than or equal to the departure time of the second, the delayed arrival time of the first train must be less than or equal to the delayed departure time of the second train. This translates to a system of constraints where each train's delay must satisfy certain inequalities relative to other trains' delays.

2. **Graph Construction**: We model the problem as a directed graph where each node represents a train. An edge from train `i` to train `j` exists if train `i` arrives at the same city where train `j` departs and the arrival time of `i` is less than or equal to the departure time of `j`. The weight of the edge is `T_i - S_j`, representing the minimal delay required for `j` relative to `i`.

3. **Bellman-Ford Algorithm**: Using the Bellman-Ford algorithm, we compute the minimal delays for each train (except the first one) that satisfy all the constraints. The algorithm relaxes all edges in each iteration, updating the minimal delays until no further improvements can be made.

4. **Output Preparation**: After computing the delays, we ensure that all delays are non-negative. If a train is not involved in any constraints, its delay is set to zero. The delays for all trains except the first one are then output in the required format.

### Solution Code
```python
import sys
import bisect

def main():
    data = sys.stdin.read().strip().split()
    if not data:
        return
    it = iter(data)
    N = int(next(it)); M = int(next(it)); X1 = int(next(it))
    trains = []
    for i in range(M):
        a = int(next(it)); b = int(next(it)); s = int(next(it)); t = int(next(it))
        trains.append((a, b, s, t))
    
    departures = [[] for _ in range(N+1)]
    for j in range(M):
        a, b, s, t = trains[j]
        if a <= N:
            departures[a].append((s, j))
    
    for i in range(1, N+1):
        departures[i].sort(key=lambda x: x[0])
    
    graph = [[] for _ in range(M)]
    for i in range(M):
        a, b, s, t = trains[i]
        if b <= N:
            deps = departures[b]
            if not deps:
                continue
            idx = bisect.bisect_left(deps, (t, -1))
            for dep in deps[idx:]:
                s_j = dep[0]
                j_index = dep[1]
                weight = t - s_j
                graph[i].append((j_index, weight))
    
    d = [10**18] * M
    d[0] = X1
    for _ in range(M-1):
        updated = False
        for u in range(M):
            if d[u] == 10**18:
                continue
            for (v, weight) in graph[u]:
                if d[u] + weight < d[v]:
                    d[v] = d[u] + weight
                    updated = True
        if not updated:
            break
    
    res = []
    for i in range(1, M):
        if d[i] == 10**18:
            res.append('0')
        else:
            x_val = d[i] if d[i] >= 0 else 0
            res.append(str(int(x_val)))
    
    print(" ".join(res))

if __name__ == '__main__':
    main()
```

### Explanation
1. **Input Parsing**: The input is read and parsed into the number of cities (N), the number of trains (M), the initial delay for the first train (X1), and the details of each train (departure city, arrival city, departure time, and arrival time).

2. **Graph Construction**: For each city, we collect all trains departing from that city and sort them by their departure time. This allows efficient lookup of trains that can be reached from a given train's arrival.

3. **Edge Creation**: For each train, we determine all trains that can be reached from it (i.e., trains departing from the arrival city of the current train with a departure time greater than or equal to the current train's arrival time). An edge is added from the current train to each reachable train with a weight calculated as the difference between the current train's arrival time and the reachable train's departure time.

4. **Bellman-Ford Algorithm**: The algorithm initializes the delay for the first train as given (X1) and all other trains as infinity. It then iteratively relaxes all edges, updating the minimal delays for each train until no further updates are possible.

5. **Output Preparation**: For each train (except the first), if its computed delay is infinity, it is set to zero. Otherwise, the delay is adjusted to ensure it is non-negative. The results are printed in the required format.

This approach efficiently models the problem using graph theory and solves it with the Bellman-Ford algorithm, ensuring minimal delays while satisfying all transfer constraints.
