To solve this problem, we need to determine the middle brother (the second oldest) among three brothers A, B, and C based on given age relationships. The relationships are provided as three characters indicating whether one brother is older or younger than another. The solution involves analyzing these relationships to identify the oldest and youngest brothers, and then deducing the middle brother from the remaining one.

### Approach
1. **Problem Analysis**: The problem provides three characters (S_AB, S_AC, S_BC) that describe the age relationships between the brothers. Each character is either '<' or '>', indicating whether the first brother mentioned is younger or older than the second, respectively.
2. **Key Insight**: The middle brother is the one who is neither the oldest nor the youngest. To find the oldest and youngest, we use the given relationships to check specific conditions for each brother:
   - **Maximum Condition (Oldest Brother)**:
     - A is the oldest if S_AB and S_AC are both '>'.
     - B is the oldest if S_AB is '<' and S_BC is '>'.
     - C is the oldest if S_AC and S_BC are both '<'.
   - **Minimum Condition (Youngest Brother)**:
     - A is the youngest if S_AB and S_AC are both '<'.
     - B is the youngest if S_AB is '>' and S_BC is '<'.
     - C is the youngest if S_AC and S_BC are both '>'.
3. **Algorithm Selection**: The solution involves checking the above conditions in sequence. Once the oldest and youngest brothers are identified, the remaining brother is the middle one.
4. **Complexity Analysis**: The solution involves simple conditional checks and operates in constant time O(1) since the number of conditions is fixed and does not depend on input size.

### Solution Code
```python
def main():
    S_AB, S_AC, S_BC = input().split()
    
    if S_AB == '>' and S_AC == '>':
        max_bro = 'A'
    elif S_AB == '<' and S_BC == '>':
        max_bro = 'B'
    elif S_AC == '<' and S_BC == '<':
        max_bro = 'C'
    else:
        max_bro = None
        
    if S_AB == '<' and S_AC == '<':
        min_bro = 'A'
    elif S_AB == '>' and S_BC == '<':
        min_bro = 'B'
    elif S_AC == '>' and S_BC == '>':
        min_bro = 'C'
    else:
        min_bro = None
        
    brothers = ['A', 'B', 'C']
    for bro in brothers:
        if bro != max_bro and bro != min_bro:
            print(bro)
            break

if __name__ == "__main__":
    main()
```
