To solve this problem, we need to calculate the minimal total traveled distance of a finger moving across a keyboard to type the letters from A to Z in order. The keyboard is represented by a string where each character's position is fixed, and the finger starts at the coordinate of 'A'. The movement between consecutive letters is measured by the absolute difference in their coordinates.

### Approach
1. **Problem Analysis**: The keyboard is given as a permutation of the alphabet, where each character's position is its index in the string plus one. The task is to compute the sum of absolute differences between the coordinates of consecutive letters in the sequence from A to Z.
2. **Intuition**: The solution involves mapping each letter to its corresponding coordinate in the keyboard. Then, by iterating through the sequence of letters from A to Z, we calculate the total distance traveled by summing the absolute differences between each consecutive pair of coordinates.
3. **Algorithm Selection**: The algorithm involves:
   - Creating a dictionary to map each letter to its coordinate (index + 1 in the input string).
   - Generating a list of coordinates for each letter in the sequence A to Z.
   - Summing the absolute differences between consecutive coordinates in this list.
4. **Complexity Analysis**: The algorithm runs in O(n) time, where n is the length of the input string (26 letters). The space complexity is O(n) to store the coordinate mappings and the sequence of coordinates.

### Solution Code
```python
def main():
    s = input().strip()
    mapping = {}
    for idx, char in enumerate(s):
        mapping[char] = idx + 1
        
    sequence = "ABCDEFGHIJKLMNOPQRSTUVWXYZ"
    coords = [mapping[letter] for letter in sequence]
    
    total_distance = 0
    for i in range(1, len(coords)):
        total_distance += abs(coords[i] - coords[i-1])
        
    print(total_distance)

if __name__ == '__main__':
    main()
```

1. **Input Handling**: The input string is read and stripped of any extraneous whitespace.
2. **Coordinate Mapping**: A dictionary is created where each character from the input string is mapped to its 1-based index (coordinate) in the string.
3. **Sequence Processing**: The sequence of letters from A to Z is generated, and their corresponding coordinates are retrieved from the dictionary.
4. **Distance Calculation**: The total distance is computed by iterating through the list of coordinates and summing the absolute differences between each consecutive pair of coordinates.
5. **Output**: The computed total distance is printed.

This approach efficiently calculates the minimal total traveled distance by leveraging direct coordinate lookups and a straightforward summation of absolute differences, ensuring optimal performance for the given problem constraints.
