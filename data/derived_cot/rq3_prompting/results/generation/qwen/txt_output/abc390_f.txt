To solve this problem, we need to compute the sum of the minimum number of operations required to erase all integers from each contiguous subarray of a given sequence. The operation involves erasing all occurrences of a consecutive range of integers that are present in the subarray. The key insight is recognizing that the number of operations for a subarray is equal to the number of connected components in the set of distinct integers present in the subarray. 

### Approach
1. **Problem Analysis**: For each contiguous subarray, the distinct integers form a set. The number of operations needed to erase all integers is determined by the number of maximal consecutive segments (connected components) in this set. For example, the set {1, 3, 4} has two connected components: [1] and [3, 4], requiring two operations.

2. **Intuition**: The solution involves breaking down the problem into two parts:
   - **Part A**: Count the number of subarrays that contain the integer 1. This is derived by calculating the total subarrays minus those subarrays that do not contain 1.
   - **Part B**: For each integer `i` from 1 to N-1, count the number of subarrays that contain `i+1` but do not contain `i`. This is done by identifying gaps where `i` is absent and then counting subarrays within these gaps that include `i+1`.

3. **Algorithm Selection**: 
   - **Preprocessing**: Store the positions of each integer in the array for quick access.
   - **Gap Analysis**: For each integer, determine the segments (gaps) where it does not appear. Within these gaps, count subarrays that include the next integer (`i+1`).
   - **Summation**: Combine the results from Part A and Part B to get the total sum.

4. **Complexity Analysis**: The algorithm efficiently processes each integer and its gaps using binary search and linear scans, resulting in an overall time complexity of O(N log N) due to the use of bisect for range queries. The space complexity is O(N) for storing positions of integers.

### Solution Code
```python
import bisect

def main():
    import sys
    data = sys.stdin.read().strip().split()
    if not data:
        return
    n = int(data[0])
    A = list(map(int, data[1:1+n]))
    
    pos = [[] for _ in range(n+1)]
    for i in range(n):
        num = A[i]
        pos[num].append(i+1)
    
    total_subarrays = n * (n + 1) // 2
    
    if n == 0:
        print(0)
        return
        
    PartA_sum = 0
    if not pos[1]:
        PartA_sum = 0
    else:
        gaps = []
        if pos[1][0] > 1:
            gaps.append((1, pos[1][0]-1))
        for j in range(len(pos[1])-1):
            if pos[1][j] + 1 <= pos[1][j+1] - 1:
                gaps.append((pos[1][j]+1, pos[1][j+1]-1))
        if pos[1][-1] < n:
            gaps.append((pos[1][-1]+1, n))
            
        not_contain_1 = 0
        for (a, b) in gaps:
            L = b - a + 1
            not_contain_1 += L * (L + 1) // 2
            
        PartA_sum = total_subarrays - not_contain_1
        
    PartB_sum = 0
    for i in range(1, n):
        q = i + 1
        if not pos[i] or not pos[q]:
            count_i = 0
        else:
            gaps_i = []
            if pos[i][0] > 1:
                gaps_i.append((1, pos[i][0]-1))
            for j in range(len(pos[i])-1):
                if pos[i][j] + 1 <= pos[i][j+1] - 1:
                    gaps_i.append((pos[i][j]+1, pos[i][j+1]-1))
            if pos[i][-1] < n:
                gaps_i.append((pos[i][-1]+1, n))
                
            count_i = 0
            for gap in gaps_i:
                a, b = gap
                L_gap = b - a + 1
                total_in_gap = L_gap * (L_gap + 1) // 2
                
                left_idx = bisect.bisect_left(pos[q], a)
                right_idx = bisect.bisect_right(pos[q], b) - 1
                
                if left_idx > right_idx:
                    count_for_gap = 0
                else:
                    segments = []
                    if left_idx == 0:
                        start_seg0 = a
                        end_seg0 = pos[q][left_idx] - 1
                        if start_seg0 <= end_seg0:
                            segments.append((start_seg0, end_seg0))
                    else:
                        start_seg0 = a
                        end_seg0 = pos[q][left_idx] - 1
                        if start_seg0 <= end_seg0:
                            segments.append((start_seg0, end_seg0))
                    
                    for j in range(left_idx, right_idx):
                        start_seg = pos[q][j] + 1
                        end_seg = pos[q][j+1] - 1
                        if start_seg <= end_seg:
                            segments.append((start_seg, end_seg))
                    
                    start_seg_last = pos[q][right_idx] + 1
                    end_seg_last = b
                    if start_seg_last <= end_seg_last:
                        segments.append((start_seg_last, end_seg_last))
                    
                    avoiding = 0
                    for (s, e) in segments:
                        L_seg = e - s + 1
                        avoiding += L_seg * (L_seg + 1) // 2
                    
                    count_for_gap = total_in_gap - avoiding
                    
                count_i += count_for_gap
                
        PartB_sum += count_i
        
    answer = PartA_sum + PartB_sum
    print(answer)

if __name__ == '__main__':
    main()
```

### Explanation
1. **Reading Input**: The input is read and parsed into an integer `n` (length of the array) and the array `A`.
2. **Position Tracking**: For each integer in the array, store its 1-based positions in a list of lists `pos`.
3. **Total Subarrays Calculation**: Compute the total number of contiguous subarrays as `n*(n+1)/2`.
4. **PartA Calculation**: Determine the number of subarrays containing the integer 1 by identifying gaps where 1 is absent and subtracting the count of subarrays in these gaps from the total subarrays.
5. **PartB Calculation**: For each integer `i` from 1 to `n-1`, count subarrays that contain `i+1` but not `i` by analyzing gaps where `i` is absent and using binary search to find occurrences of `i+1` within these gaps.
6. **Result Computation**: The final result is the sum of PartA and PartB, printed as the answer.

This approach efficiently breaks down the problem into manageable parts, leveraging gap analysis and binary search to handle large input sizes within feasible time complexity.
