=== ANSWER ===

To solve this problem, we need to determine if a given string of length 3 can be rearranged to form the string "ABC". The solution involves checking whether the characters in the input string match the characters in "ABC" in any order.

### Approach
1. **Problem Analysis**: The problem requires us to check if the input string can be rearranged to exactly match the string "ABC". Since both strings are of length 3, we can compare the sorted versions of both strings. If they are identical, then it is possible to rearrange the input string to form "ABC"; otherwise, it is not.
2. **Intuition**: By sorting both the input string and the target string "ABC", we can easily compare them. If the sorted versions match, it means the input string contains exactly the same characters as "ABC" in any order, making rearrangement possible.
3. **Algorithm Selection**: The algorithm involves sorting both strings and then comparing them. This approach is efficient with a constant time complexity O(1) since the strings are of fixed length (3 characters).
4. **Complexity Analysis**: Sorting strings of fixed length 3 is a constant time operation, making the overall time complexity O(1). The space complexity is also O(1) as we only store a few characters during comparison.

### Solution Code
```python
def main():
    S = input().strip()
    if sorted(S) == sorted("ABC"):
        print("Yes")
    else:
        print("No")

if __name__ == "__main__":
    main()
```

### Explanation
1. **Input Handling**: The input string `S` is read from standard input.
2. **Sorting and Comparison**: The input string `S` and the target string "ABC" are sorted. If the sorted versions of both strings are identical, it means the input string can be rearranged to form "ABC", and the output is "Yes". Otherwise, the output is "No".
3. **Output**: Based on the comparison, the program prints "Yes" or "No".

This approach efficiently checks for the possibility of rearranging the input string to match "ABC" by leveraging simple sorting and comparison, ensuring correctness with minimal computational overhead.
