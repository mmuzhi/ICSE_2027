To solve this problem, we need to efficiently compute the sum of elements in a 3D array for multiple queries. Each query specifies a rectangular region in the 3D array, and we need to return the sum of all elements within that region. Given the constraints, a brute-force approach for each query would be too slow, so we use a 3D prefix sum array to answer each query in constant time.

### Approach
1. **Reading Input:** The input consists of a 3D array of size \(N \times N \times N\) and a series of queries. Each query specifies a rectangular region in the 3D array.
2. **Building 3D Prefix Sum Array:** We construct a 3D prefix sum array where each element at \((x, y, z)\) stores the sum of all elements from \((1, 1, 1)\) to \((x, y, z)\) in the original array. This allows us to compute the sum of any rectangular region using inclusion-exclusion principles.
3. **Processing Queries:** For each query, we use the prefix sum array to compute the sum of the specified region by applying the inclusion-exclusion formula, which involves subtracting and adding overlapping regions to avoid double-counting.

import sys

def main():

        return
