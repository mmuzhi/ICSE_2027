To solve this problem, we need to count the number of substrings in a given string of digits that are divisible by their non-zero last digit. The solution involves efficiently checking each substring's divisibility by leveraging mathematical properties and dynamic programming to handle large input sizes.

### Approach
1. **Direct Count for Simple Cases**: For digits 1, 2, 4, and 5, any substring ending with these digits is automatically divisible by them. Thus, we can directly count all substrings ending at each occurrence of these digits.
2. **Prefix Sum for Digits 3 and 6**: For digits 3 and 6, a substring is divisible if the sum of its digits is divisible by 3 (since 6 requires divisibility by both 2 and 3, but the last digit ensures evenness). We use a prefix sum array and a frequency dictionary to efficiently count valid substrings.
3. **Dynamic Programming for Digits 7 and 9**: For digits 7 and 9, we use a dynamic programming approach to keep track of substring remainders. The remainder of a substring ending at the current position is derived from the remainder of substrings ending at the previous position, adjusted by the current digit.

### Solution Code
```python
class Solution:
    def countSubstrings(self, s: str) -> int:
        n = len(s)
        ans = 0
        freq = [0] * 3
        freq[0] = 1
        cur_prefix = 0
        dp = None
        
        for j in range(n):
            if s[j] == '0':
                continue
            elif s[j] in '1245':
                ans += j + 1
            elif s[j] in '36':
                cur_prefix = (cur_prefix + int(s[j])) % 3
                ans += freq[cur_prefix]
                freq[cur_prefix] += 1
            elif s[j] in '79':
                if dp is None:
                    mod = 7 if s[j] == '7' else 9
                    dp = [0] * mod
                new_dp = [0] * len(dp)
                r_single = int(s[j]) % len(dp)
                new_dp[r_single] += 1
                if j > 0:
                    for r_prev in range(len(dp)):
                        if dp[r_prev] > 0:
                            new_r = (r_prev * 10 + int(s[j])) % len(dp)
                            new_dp[new_r] += dp[r_prev]
                ans += new_dp[0]
                dp = new_dp
        
        return ans
```

### Explanation
1. **Initialization**: We initialize variables to keep track of the answer, a frequency array for prefix sums (mod 3), and a dynamic programming array for digits 7 and 9.
2. **Iterate Through String**: For each character in the string:
   - **Skip '0'**: As substrings ending with '0' are not considered.
   - **Digits 1, 2, 4, 5**: Directly add all substrings ending at the current position to the answer.
   - **Digits 3 and 6**: Use the prefix sum and frequency array to count valid substrings efficiently.
   - **Digits 7 and 9**: Use dynamic programming to compute the number of valid substrings by extending previous substrings' remainders.
3. **Output**: After processing all characters, the answer contains the total count of valid substrings.

This approach efficiently handles the constraints by leveraging mathematical properties and dynamic programming to avoid the O(n^2) complexity of a naive solution.
