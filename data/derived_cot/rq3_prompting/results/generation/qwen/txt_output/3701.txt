To solve this problem, we need to transform a given string into a "good caption" using the minimum number of operations. A good caption is defined as a string where every character appears in groups of at least 3 consecutive occurrences. The allowed operations are changing a character to its immediate predecessor or successor in the alphabet, except for 'a' and 'z'. The solution involves dynamic programming to find the minimal cost transformation and then backtracking to construct the lexicographically smallest string.

### Approach
1. **Problem Analysis**: The problem requires converting each character in the string to form contiguous runs of at least 3 identical characters. The cost is the sum of absolute differences between the original and target characters. The challenge is to find the minimal cost transformation and, among all such transformations, the lexicographically smallest string.

2. **Dynamic Programming (DP) Setup**: We use a 3D DP array where `dp[i][c][l]` represents the minimal cost to transform the substring up to index `i` into a run ending with character `c` of length state `l` (0 for length 1, 1 for length 2, and 2 for length ≥3). We also maintain two auxiliary 3D arrays, `prev_char` and `prev_state`, to store the previous character and the length of the previous run for backtracking.

3. **DP Initialization**: For each character at position 0, initialize the DP state for a run of length 1. For positions beyond 0, update the DP states by either extending the current run or starting a new run (only if the previous run was of length ≥3).

4. **Backtracking**: After computing the DP table, find the minimal cost and the corresponding character and state at the last position. Then, backtrack from the end to the start using the auxiliary arrays to construct the resulting string, ensuring it is lexicographically smallest by choosing the smallest character at each step when multiple options have the same cost.

### Solution Code
```python
class Solution:
    def minCostGoodCaption(self, caption: str) -> str:
        n = len(caption)
        if n < 3:
            return ""
        
        inf = 10**9
        dp = [[[inf] * 3 for _ in range(26)] for __ in range(n)]
        prev_char = [[[[-1] * 26 for _ in range(3)] for __ in range(n)]
        prev_state = [[[0] * 3 for _ in range(26)] for __ in range(n)]
        
        for c in range(26):
            cost0 = abs(ord(caption[0]) - ord('a') + c)
            dp[0][c][0] = cost0
        
        for i in range(1, n):
            for c in range(26):
                for l in range(3):
                    if dp[i-1][c][l] != inf:
                        new_cost = dp[i-1][c][l] + abs(ord(caption[i]) - (ord('a') + c))
                        if l == 0:
                            new_state = 1
                        else:
                            new_state = 2
                        if new_cost < dp[i][c][new_state]:
                            dp[i][c][new_state] = new_cost
                            prev_char[i][c][new_state] = c
                            prev_state[i][c][new_state] = l
                
                for prev_c in range(26):
                    if prev_c == c:
                        continue
                    if dp[i-1][prev_c][2] != inf:
                        new_cost = dp[i-1][prev_c][2] + abs(ord(caption[i]) - (ord('a') + c))
                        if new_cost < dp[i][c][0]:
                            dp[i][c][0] = new_cost
                            prev_char[i][c][0] = prev_c
                            prev_state[i][c][0] = 2
        
        min_cost = inf
        best_c = -1
        best_state = -1
        for c in range(26):
            for state in range(3):
                if dp[n-1][c][state] < min_cost:
                    min_cost = dp[n-1][c][state]
                    best_c = c
                    best_state = state
                elif dp[n-1][c][state] == min_cost and best_c == -1:
                    if chr(ord('a') + c) < chr(ord('a') + best_c):
                        best_c = c
                        best_state = state
        
        if min_cost == inf:
            return ""
        
        res = []
        i = n - 1
        c_val = best_c
        state_val = best_state
        while i >= 0:
            res.append(chr(ord('a') + c_val))
            if i == 0:
                break
            c_val = prev_char[i][c_val][state_val]
            state_val = prev_state[i][c_val][state_val]
            i -= 1
        
        res.reverse()
        return ''.join(res)
```

### Explanation
1. **Initialization**: The solution starts by checking if the string length is less than 3, in which case it's impossible to form a good caption. Otherwise, it initializes the DP table and auxiliary arrays.

2. **DP Table Population**: For each character in the string, the DP table is updated by considering both extending the current run or starting a new run (if the previous run was of sufficient length). The cost for each operation is the absolute difference between the original and target characters.

3. **Backtracking**: After computing the minimal cost, the solution backtracks from the end of the string to the beginning using the auxiliary arrays to construct the resulting string. The lexicographically smallest string is chosen by preferring smaller characters when multiple options have the same cost.

This approach efficiently computes the minimal cost transformation and ensures the result is lexicographically smallest by leveraging dynamic programming and careful backtracking.
