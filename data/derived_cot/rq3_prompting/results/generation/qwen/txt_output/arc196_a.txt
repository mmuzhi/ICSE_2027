To solve this problem, we need to maximize the total score obtained by repeatedly removing two adjacent numbers from a sequence until the sequence has at most one number. The score for each removal operation is the absolute difference between the two chosen numbers. 

### Approach
1. **Problem Analysis**: The key observation here is that the maximum total score is achieved by pairing the smallest and largest numbers in the sequence, the next smallest and next largest, and so on. This approach ensures that the largest possible differences are considered first, maximizing the total score.
2. **Intuition**: By sorting the sequence, we can efficiently pair the smallest and largest elements, which will yield the highest absolute differences. This method works because the sorted array allows us to systematically pair elements from the ends towards the center, ensuring optimal pairings.
3. **Algorithm Selection**: The algorithm involves sorting the array and then using two pointers—one starting at the beginning (smallest elements) and the other at the end (largest elements)—to pair elements and compute their differences. This approach efficiently reduces the problem to a linear pass after sorting, which dominates the time complexity.
4. **Complexity Analysis**: Sorting the array takes O(n log n) time, and the subsequent linear pass takes O(n) time. Thus, the overall time complexity is O(n log n), which is efficient for the given constraints (n ≤ 3 × 10^5).

import sys

def main():
    data = sys.stdin.read().split()
