To solve this problem, we need to compute the sum of distinct values for all contiguous subsequences (subarrays) of a given sequence of integers. The challenge is to efficiently calculate this sum without resorting to a brute-force approach, which would be computationally infeasible for large sequences.

### Approach
1. **Problem Analysis**: The problem requires summing the number of distinct elements in every possible contiguous subarray of the given sequence. A brute-force approach would involve checking every subarray, which is O(N^2) and impractical for large N (up to 200,000).

2. **Key Insight**: Instead of processing each subarray individually, we can leverage the observation that each distinct number's contribution to the total sum is the number of subarrays that contain it at least once. This transforms the problem into calculating, for each number, how many subarrays include at least one occurrence of that number.

3. **Algorithm Selection**: For each number in the sequence, we determine the subarrays that do not contain it. The number of subarrays containing the number is then the total number of subarrays minus the number of subarrays that exclude it. The total number of subarrays is N*(N+1)/2. To find subarrays excluding a number, we identify gaps between its occurrences and calculate the subarrays within these gaps.

4. **Complexity Analysis**: The algorithm processes each number's occurrences once. For each number, it processes its positions in O(F) time, where F is the frequency of the number. Since the sum of all frequencies is N, the overall complexity is O(N), which is efficient for large N.

### Solution Code
```python
import sys

def main():
    data = sys.stdin.read().split()
    n = int(data[0])
    A = list(map(int, data[1:1+n]))
    
    total_subarrays = n * (n + 1) // 2
    positions = [[] for _ in range(n + 1)]
    
    for i, num in enumerate(A):
        if 1 <= num <= n:
            positions[num].append(i)
    
    ans = 0
    for x in range(1, n + 1):
        if not positions[x]:
            continue
        pos = positions[x]
        total_gap = 0
        
        left_gap_length = pos[0]
        total_gap += left_gap_length * (left_gap_length + 1) // 2
        
        for i in range(1, len(pos)):
            gap_length = pos[i] - pos[i - 1] - 1
            total_gap += gap_length * (gap_length + 1) // 2
        
        right_gap_length = n - 1 - pos[-1]
        total_gap += right_gap_length * (right_gap_length + 1) // 2
        
        ans += (total_subarrays - total_gap)
    
    print(ans)

if __name__ == '__main__':
    main()
```

4. **Gap Calculation**: For each number, we calculate the number of subarrays that do not contain it by summing the subarrays within the gaps between its occurrences. The gaps include the segments before the first occurrence, between consecutive occurrences, and after the last occurrence.

5. **Result Calculation**: The result for each number is derived by subtracting the count of subarrays excluding it from the total subarrays. Summing these results for all numbers gives the final answer.
