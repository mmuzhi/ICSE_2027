To solve this problem, we need to count the number of good integers containing exactly `n` digits. A good integer is defined as one that can be rearranged to form a k-palindromic integer. A k-palindromic integer must be a palindrome and divisible by `k`. Additionally, the integer must not have leading zeros before or after rearrangement.

### Approach
1. **Generate Multisets**: First, we generate all possible multisets (frequency distributions) of digits (0-9) that can form a palindrome of length `n`. For even `n`, all digit counts must be even. For odd `n`, exactly one digit count must be odd, and the rest even.
2. **Check Non-zero Digits**: For each generated multiset, ensure it contains at least one non-zero digit to avoid leading zeros in the palindrome.
3. **DFS for Palindrome Divisibility**: For each valid multiset, use a Depth-First Search (DFS) approach to check if there exists a palindrome arrangement (using the multiset digits) that is divisible by `k`. The DFS explores all possible first halves of the palindrome (since the second half is determined by the first half in a palindrome) and checks if the resulting palindrome is divisible by `k`.
4. **Count Valid Integers**: For each multiset that allows a valid palindrome arrangement divisible by `k`, calculate the number of distinct `n`-digit numbers (without leading zeros) that can be formed from the multiset. This is done by considering all permutations of the multiset and adjusting for leading zeros.

### Solution Code
```python
import itertools
from functools import lru_cache
from math import comb

class Solution:
    def countGoodIntegers(self, n: int, k: int) -> int:
        @staticmethod
        def yield_solutions(t, num_vars=10):
            if t == 0:
                yield [0] * num_vars
            else:
                for counts in itertools.product(range(t + 1), repeat=num_vars):
                    if sum(counts) == t:
                        yield counts
        
        total = 0
        m = (n + 1) // 2
        
        if n % 2 == 0:
            t_val = n // 2
            for counts in Solution.yield_solutions(t_val, 10):
                freq = [2 * counts[i] for i in range(10)]
                if not any(freq[i] > 0 for i in range(1, 10)):
                    continue
                if Solution.dfs(m, k, freq.copy()):
                    count_non_zero = n - freq[0]
                    count_zero = freq[0]
                    total_permutations = comb(n, freq)
                    count_without_leading_zero = total_permutations * (1 - count_zero / n)
                    total += int(count_without_leading_zero)
        else:
            t_val = (n - 1) // 2
            for d in range(10):
                for counts in Solution.yield_solutions(t_val, 10):
                    freq = [0] * 10
                    for i in range(10):
                        if i == d:
                            freq[i] = 2 * counts[i] + 1
                        else:
                            freq[i] = 2 * counts[i]
                    if not any(freq[i] > 0 for i in range(1, 10)):
                        continue
                    if Solution.dfs(m, k, freq.copy()):
                        count_non_zero = n - freq[0]
                        count_zero = freq[0]
                        total_permutations = comb(n, freq)
                        count_without_leading_zero = total_permutations * (1 - count_zero / n)
                        total += int(count_without_leading_zero)
        return total

    @staticmethod
    def dfs(m, k, freq):
        @lru_cache(maxsize=None)
        def _dfs(pos, rem, freq_tuple):
            if pos == m:
                return rem % k == 0
            freq_list = list(freq_tuple)
            for digit in range(10):
                if freq_list[digit] > 0:
                    new_freq = freq_list.copy()
                    new_freq[digit] -= 1
                    if _dfs(pos + 1, rem * 10 + digit, tuple(new_freq)):
                        return True
            return False
        return _dfs(0, 0, tuple(freq))
```
