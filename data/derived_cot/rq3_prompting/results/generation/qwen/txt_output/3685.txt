To solve this problem, we need to count the number of subarrays of length 3 in a given integer array where the sum of the first and third elements equals exactly half of the second element. 

### Approach
1. **Problem Analysis**: The task involves checking all possible subarrays of length 3 in the array. For each subarray, we verify if the sum of the first and third elements is equal to half of the second element. This translates to checking if `2*(first + third) == second` for each triplet.
2. **Intuition**: By iterating through the array, we can examine each consecutive triplet of elements. For each triplet, we apply the condition mentioned above. If the condition is satisfied, we increment our count.
3. **Algorithm Selection**: The straightforward approach is to use a single loop that runs from the start of the array to the third last element. For each element at index `i`, we check the triplet `nums[i]`, `nums[i+1]`, and `nums[i+2]`.
4. **Complexity Analysis**: The algorithm runs in O(n) time, where n is the length of the array. This is efficient given the constraint `3 <= nums.length <= 100`.

### Solution Code
```python
class Solution:
    def countSubarrays(self, nums: List[int]) -> int:
        count = 0
        n = len(nums)
        for i in range(n - 2):
            if 2 * (nums[i] + nums[i+2]) == nums[i+1]:
                count += 1
        return count
```

### Explanation
1. **Initialization**: We start by initializing a counter `count` to zero.
2. **Iteration**: We loop through the array from the first element to the third last element (inclusive). For each element at index `i`, we consider the triplet `nums[i]`, `nums[i+1]`, and `nums[i+2]`.
3. **Condition Check**: For each triplet, we check if twice the sum of the first and third elements equals the second element. If this condition is met, we increment the counter.
4. **Result**: After processing all possible triplets, the counter holds the total number of valid subarrays, which is returned as the result.

This approach efficiently checks all possible subarrays of length 3 and counts those that meet the specified condition, leveraging simple iteration and arithmetic comparison.
