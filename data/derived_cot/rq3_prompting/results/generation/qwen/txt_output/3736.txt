To solve this problem, we need to find the first valid adjacent pair of digits in a given string where each digit in the pair appears exactly as many times as its numeric value in the entire string. The solution involves checking each adjacent pair from left to right and verifying the specified conditions.

### Approach
1. **Frequency Count**: First, we count the occurrences of each digit in the string using a frequency dictionary (or counter). This helps us quickly check the condition for any digit in the string.
2. **Adjacent Pair Check**: Traverse the string from left to right, checking each adjacent pair of digits. For each pair:
   - Ensure the digits are different.
   - Verify that the first digit appears exactly `int(first_digit)` times in the entire string.
   - Verify that the second digit appears exactly `int(second_digit)` times in the entire string.
3. **Return Result**: The first valid pair encountered during the traversal is returned immediately. If no such pair is found after checking all adjacent pairs, return an empty string.

### Solution Code
```python
from collections import Counter

class Solution:
    def findValidPair(self, s: str) -> str:
        cnt = Counter(s)
        n = len(s)
        for i in range(n-1):
            a, b = s[i], s[i+1]
            if a == b:
                continue
            if cnt[a] == int(a) and cnt[b] == int(b):
                return a + b
        return ""
```

### Explanation
1. **Frequency Count**: The `Counter` from the `collections` module is used to count occurrences of each digit in the string. This allows us to efficiently check the required conditions for any digit in constant time.
2. **Adjacent Pair Check**: The loop iterates through each adjacent pair in the string. For each pair, it checks if the digits are different and if each digit's count in the string matches its numeric value.
3. **Early Termination**: The loop returns the first valid pair found, ensuring optimal performance by avoiding unnecessary checks once the solution is found.
4. **Result Handling**: If no valid pair is found after checking all adjacent pairs, the function returns an empty string, indicating the absence of a valid pair.

This approach efficiently checks each possible pair in linear time, leveraging the frequency count for quick validation, making it both optimal and easy to understand.
