=== ANSWER ===

To solve this problem, we need to move the K-th 1-block in a binary string to immediately after the (K-1)-th 1-block. A 1-block is defined as a contiguous segment of '1's bounded by '0's or the start/end of the string. The solution involves identifying the positions of the (K-1)-th and K-th 1-blocks, then constructing the resulting string by rearranging these blocks according to the specified operation.

### Approach
1. **Identify 1-Blocks**: Traverse the input string to identify all contiguous segments of '1's (1-blocks). Each 1-block is recorded with its start and end indices.
2. **Locate Target Blocks**: Once all 1-blocks are identified, locate the (K-1)-th and K-th 1-blocks from the list.
3. **Construct Result String**: The result string is constructed by:
   - Keeping the substring up to the end of the (K-1)-th 1-block.
   - Appending the K-th 1-block as a sequence of '1's.
   - Inserting zeros in the position where the K-th 1-block originally started, for the length of the gap between the end of the (K-1)-th block and the start of the K-th block.
   - Appending the remaining substring after the K-th 1-block.

### Solution Code
```python
def main():
    import sys
    data = sys.stdin.read().splitlines()
    if not data:
        return
    
    first_line = data[0].split()
    if not first_line:
        return
        
    try:
        N = int(first_line[0])
        K = int(first_line[1])
    except Exception:
        return

    if len(data) < 2:
        S = ""
    else:
        S = data[1].strip()
    
    n = len(S)
    blocks = []
    i = 0
    while i < n and len(blocks) < K:
        if S[i] == '1':
            start = i
            while i < n and S[i] == '1':
                i += 1
            end = i - 1
            blocks.append((start, end))
        else:
            i += 1
            
    if len(blocks) < K:
        print(S)
        return
        
    block_prev = blocks[K-2]
    block_curr = blocks[K-1]
    
    part1 = S[:block_prev[1] + 1]
    ones_str = '1' * (block_curr[1] - block_curr[0] + 1)
    zeros_str = '0' * (block_curr[0] - block_prev[1] - 1)
    part4 = S[block_curr[1] + 1:]
    
    T = part1 + ones_str + zeros_str + part4
    print(T)

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The input is read from standard input, which includes the length of the string (N), the block number (K), and the binary string (S).
2. **Finding 1-Blocks**: The code scans the string to identify all contiguous '1' segments (1-blocks), recording their start and end indices. The loop stops once K blocks are found to optimize performance.
3. **Constructing Result String**: The result string is built by:
   - **Part1**: The substring from the start of the string up to the end of the (K-1)-th 1-block.
   - **Ones String**: The K-th 1-block represented as a sequence of '1's.
   - **Zeros String**: A sequence of zeros whose length corresponds to the gap between the (K-1)-th and K-th blocks.
   - **Part4**: The substring from the end of the K-th block to the end of the original string.
4. **Output**: The constructed string is printed, which is the original string with the K-th 1-block moved to the position immediately after the (K-1)-th 1-block.
