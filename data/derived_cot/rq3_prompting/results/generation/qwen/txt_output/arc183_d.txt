=== ANSWER ===

To solve this problem, we need to remove two leaves from a tree in each operation while ensuring the remaining tree always has a perfect matching. The goal is to maximize the total score, which is the sum of the distances between the two leaves removed in each operation. 

### Approach
1. **Tree Construction**: Read the input to construct the tree using an adjacency list.
2. **Leaf Identification**: Identify all leaves (vertices with degree 1) in the tree.
3. **Distance Calculation**: For each operation, find the two leaves with the maximum distance in the current tree. This is done using BFS to compute the shortest path (which is the only path in a tree) between all pairs of leaves.
4. **Removal and Update**: Remove the two leaves with the maximum distance, update the tree structure, and identify new leaves formed by the removal of these vertices.
5. **Output**: After processing all operations, output the sequence of pairs removed in each operation.
