=== ANSWER ===

To solve this problem, we need to determine if a given string of digits is balanced. A string is considered balanced if the sum of the digits at even indices (0-indexed) is equal to the sum of the digits at odd indices. 

### Approach
1. **Problem Analysis**: The problem requires checking two separate sums from the string. The even indices (0, 2, 4, etc.) contribute to one sum, while the odd indices (1, 3, 5, etc.) contribute to another. The solution involves calculating these two sums and comparing them.
2. **Intuition**: By iterating through the string, we can separate the digits into even and odd indexed positions. For each digit, we convert it to an integer and add it to the respective sum based on its index.
3. **Algorithm Selection**: The algorithm involves a single pass through the string. For each character at index `i`, if `i` is even, add its integer value to `sum_even`; otherwise, add it to `sum_odd`. After processing all characters, compare the two sums.
4. **Complexity Analysis**: The algorithm runs in O(n) time, where n is the length of the string, as it processes each character exactly once. The space complexity is O(1) since we only use a few extra variables for storing the sums.

### Solution Code
```python
class Solution:
    def isBalanced(self, num: str) -> bool:
        sum_even = sum(int(num[i]) for i in range(0, len(num), 2))
        sum_odd = sum(int(num[i]) for i in range(1, len(num), 2))
        return sum_even == sum_odd
```

### Explanation
1. **Initialization**: We initialize two variables, `sum_even` and `sum_odd`, to zero. These will hold the sums of digits at even and odd indices, respectively.
2. **Sum Calculation**: Using list comprehensions, we iterate over the string. For even indices (0, 2, 4, etc.), we add the integer value of the digit to `sum_even`. For odd indices (1, 3, 5, etc.), we add the integer value of the digit to `sum_odd`.
3. **Comparison**: After processing all digits, we check if `sum_even` equals `sum_odd`. If they are equal, the function returns `True`, indicating the string is balanced; otherwise, it returns `False`.

This approach efficiently checks the balance condition by leveraging simple iteration and arithmetic operations, ensuring optimal performance and correctness.
