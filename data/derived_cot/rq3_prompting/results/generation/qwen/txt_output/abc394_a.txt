=== ANSWER ===

To solve this problem, we need to process a given string consisting of digits and extract all occurrences of the digit '2'. The solution involves iterating through each character in the string, checking if it is a '2', and if so, including it in the result. The result is formed by concatenating all the '2's in their original order.

### Approach
1. **Problem Analysis**: The task requires us to filter out all characters from the input string except for the digit '2'. The remaining characters should be concatenated in the order they appear in the original string.
2. **Intuition**: By iterating through each character of the string and checking if it matches '2', we can efficiently collect all required characters. This approach ensures that we maintain the original order of the '2's.
3. **Algorithm Selection**: A simple linear scan of the string is sufficient. For each character, we check if it is '2' and, if true, append it to the result string. This approach has a time complexity of O(n), where n is the length of the string, making it efficient for the given constraints.
4. **Optimization**: The solution is already optimal as it processes each character exactly once. No further optimizations are necessary.
5. **Complexity Analysis**: The algorithm runs in O(n) time, where n is the length of the input string. The space complexity is O(m), where m is the number of '2's in the input string, as we store each '2' in the result.

### Solution Code
```python
def main():
    S = input().strip()
    result = ''.join(char for char in S if char == '2')
    print(result)

if __name__ == '__main__':
    main()
```

### Explanation
1. **Reading Input**: The input string is read from standard input.
2. **Filtering Characters**: Using a generator expression, each character in the string is checked. If the character is '2', it is included in the generator.
3. **Joining Characters**: The generator is joined into a single string, which is then printed as the output.
This approach efficiently processes the input string in linear time, ensuring that all '2's are collected in their original order, providing the correct output as required.
