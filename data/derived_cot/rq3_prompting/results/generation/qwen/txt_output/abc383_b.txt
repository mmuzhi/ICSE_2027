To solve this problem, we need to determine the maximum number of floor cells that can be humidified by placing two distinct humidifiers on two different floor cells in a grid. A cell is considered humidified if it is a floor cell and lies within a Manhattan distance D from at least one of the humidifiers.

### Approach
1. **Problem Analysis**: The problem involves a grid with floor cells (.) and desk cells (#). We need to select two distinct floor cells to place humidifiers such that the number of floor cells within Manhattan distance D from either humidifier is maximized. The Manhattan distance between two cells (i, j) and (k, l) is |i - k| + |j - l|.

2. **Intuition**: The solution involves checking all possible pairs of floor cells, calculating the union of floor cells within distance D from each pair, and then selecting the pair that covers the maximum number of unique floor cells.

3. **Algorithm Selection**: 
   - **Preprocessing**: Collect all floor cell coordinates from the grid.
   - **Distance Calculation**: For each floor cell, precompute the set of all floor cells within Manhattan distance D. This helps in efficiently determining the coverage of each potential humidifier placement.
   - **Pair Evaluation**: For each pair of distinct floor cells, compute the union of their coverage sets. The size of this union gives the number of humidified floor cells for that pair. The maximum size across all pairs is the answer.

4. **Complexity Analysis**: 
   - **Time Complexity**: O(n^2), where n is the number of floor cells. For each of the O(n^2) pairs, we compute the union of their coverage sets, which involves O(n) operations in the worst case (since each set can contain up to n elements).
   - **Space Complexity**: O(n^2), due to storing the coverage sets for each floor cell.

### Solution Code
```python
def main():
    import sys
    data = sys.stdin.read().splitlines()
    if not data:
        return
    
    H, W, D = map(int, data[0].split())
    grid = []
    for i in range(1, 1 + H):
        grid.append(data[i].strip())
    
    F = []
    for i in range(H):
        for j in range(W):
            if grid[i][j] == '.':
                F.append((i, j))
                
    n = len(F)
    if n < 2:
        print(0)
        return
        
    sets = [set() for _ in range(n)]
    for i in range(n):
        for j in range(n):
            d = abs(F[i][0] - F[j][0]) + abs(F[i][1] - F[j][1])
            if d <= D:
                sets[i].add(j)
                
    ans = 0
    for i in range(n):
        for j in range(i + 1, n):
            union_set = sets[i] | sets[j]
            if len(union_set) > ans:
                ans = len(union_set)
                
    print(ans)

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The input is read from standard input, including grid dimensions (H, W), the Manhattan distance D, and the grid itself.
2. **Floor Cell Collection**: All coordinates of floor cells (.) are collected into a list `F`.
3. **Coverage Sets**: For each floor cell, a set is precomputed containing indices of all floor cells within Manhattan distance D. This allows efficient union operations later.
4. **Pair Evaluation**: For each unique pair of floor cells, the union of their coverage sets is computed. The size of this union gives the number of floor cells humidified by the pair. The maximum size across all pairs is printed as the result.

This approach efficiently checks all possible pairs of humidifier placements and leverages set operations to determine the optimal solution. The preprocessing step ensures that each pair's coverage is computed in constant time during the evaluation phase.
