To solve this problem, we need to determine whether Snuke is following Takahashi's instructions correctly when raising his hands. The instructions specify that Snuke should raise only his left hand if he wants to eat takoyaki, and only his right hand otherwise. The solution involves checking the given input values (L and R) to decide the appropriate output based on the problem's conditions.

### Approach
1. **Problem Analysis**: The problem requires us to evaluate two integers, L and R, each being either 0 or 1. These integers represent the state of Snuke's hands (0 for not raised, 1 for raised). The key conditions are:
   - If Snuke raises only his left hand (L=1 and R=0), he wants to eat takoyaki, so we print "Yes".
   - If Snuke raises only his right hand (L=0 and R=1), he does not want to eat takoyaki, so we print "No".
   - If Snuke raises both hands (L=1 and R=1) or neither hand (L=0 and R=0), the action is invalid, so we print "Invalid".

2. **Algorithm Selection**: The solution involves straightforward conditional checks based on the input values. The algorithm checks each possible combination of L and R to determine the correct output.

3. **Complexity Analysis**: The solution involves a constant-time check (O(1)) since it only requires evaluating two conditions, making it highly efficient for the given constraints.

### Solution Code
```python
def main():
    data = input().split()
    L = int(data[0])
    R = int(data[1])
    
    if L == 1 and R == 0:
        print("Yes")
    elif L == 0 and R == 1:
        print("No")
    else:
        print("Invalid")

if __name__ == "__main__":
    main()
```

### Explanation
- **Input Handling**: The code reads two integers, L and R, from the input. These values represent the state of Snuke's left and right hands, respectively.
- **Condition Checks**: 
  - If L is 1 and R is 0, Snuke is raising only his left hand, indicating he wants to eat takoyaki, so the output is "Yes".
  - If L is 0 and R is 1, Snuke is raising only his right hand, indicating he does not want to eat takoyaki, so the output is "No".
  - For any other combination (both hands raised or no hands raised), the output is "Invalid" as Snuke is not following the instructions correctly.
- **Output**: The result is printed based on the condition checks, providing the correct response for each scenario. This approach efficiently handles all possible input cases with minimal computational overhead.
