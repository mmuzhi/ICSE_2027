```python
def main():
    import sys
    data = sys.stdin.read().split()
    n = int(data[0])
    A = list(map(int, data[1:1+n]))
    
    count = 0
    while True:
        A.sort(reverse=True)
        if A[0] > 0:
            A[0] -= 1
        if len(A) > 1 and A[1] > 0:
            A[1] -= 1
        count += 1
        
        positive_count = sum(1 for x in A if x > 0)
        if positive_count <= 1:
            break
            
    print(count)

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The input is read from standard input, where the first value is the number of elements, followed by the sequence of integers.
2. **Initialization**: The count of operations is initialized to zero.
3. **Loop Until Condition Met**: The loop continues indefinitely until the condition is met.
   - **Sorting**: The sequence is sorted in descending order to ensure the largest elements are at the front.
   - **Decrementing**: The first two elements are decremented by 1 if they are positive.
   - **Operation Count**: Each iteration increments the operation count.
   - **Condition Check**: After each operation, the number of positive elements is counted. If it is one or fewer, the loop breaks.
4. **Output**: The total number of operations performed is printed.

This approach efficiently simulates each operation, ensuring the problem constraints are respected and the solution is both correct and optimal for the given problem.
