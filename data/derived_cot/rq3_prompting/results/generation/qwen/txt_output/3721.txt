2. **User Status Tracking**: Each user's status is tracked using an array `offlineUntil`, which stores the timestamp until they will remain offline. Initially, all users are online (i.e., `offlineUntil` is set to -1).
3. **Mention Counting**: For each MESSAGE event:
   - **Individual Mentions**: If the token starts with "id", the corresponding user's mention count is incremented regardless of their online status.
   - **ALL Mentions**: If the token is "ALL", all users' mention counts are incremented by 1.
   - **HERE Mentions**: If the token is "HERE", only online users (those whose `offlineUntil` timestamp is less than or equal to the current event's timestamp) are incremented.

class Solution:
