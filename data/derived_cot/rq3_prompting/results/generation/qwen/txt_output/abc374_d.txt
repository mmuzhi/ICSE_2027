To solve this problem, we need to determine the minimum time required to print all given line segments on an xy-plane using a printing machine that starts at the origin (0, 0). The machine can move the laser either by printing a line segment (emitting the laser) or by moving without printing (non-emitting movement). The key is to find the optimal sequence of printing segments and the direction for each segment (starting from either endpoint) to minimize the total time, which includes both printing and non-printing movement times.

### Approach
1. **Problem Analysis**: The problem involves printing N line segments, each defined by two endpoints. The machine starts at (0, 0). For each segment, the laser must move to one endpoint (non-printing) and then print the entire segment to the other endpoint (printing). The total time is the sum of the printing times (segment length divided by T) and the non-printing movement times (distance between endpoints divided by S).

2. **Intuition**: The solution requires exploring all possible sequences of printing segments and directions (starting from either endpoint) to find the sequence with the minimal total time. Given the constraints (N ≤ 6), a brute-force approach with dynamic programming is feasible.

3. **Dynamic Programming State**: The state is defined by a bitmask representing which segments have been printed and the current endpoint position. The DP dictionary stores the minimum time to reach each state.

4. **State Transitions**: For each state, iterate over all unprinted segments and both possible directions (start endpoints). For each transition, compute the non-printing movement time from the current endpoint to the segment's start endpoint and the printing time for the segment. Update the DP state for the new bitmask and endpoint.

5. **Initialization**: Start at (0, 0) with an empty bitmask (no segments printed) and zero time.

6. **Result Extraction**: After processing all states, the result is the minimum time from the state where all segments are printed (full bitmask).

### Solution Code
```python
import math
import sys

def main():
    data = sys.stdin.read().strip().split()
    if not data:
        return
    
    it = iter(data)
    N = int(next(it)); S = int(next(it)); T = int(next(it))
    segments = []
    for i in range(N):
        A = int(next(it)); B = int(next(it)); C = int(next(it)); D = int(next(it))
        p1 = (A, B)
        p2 = (C, D)
        length = math.sqrt((C-A)**2 + (D-B)**2)
        segments.append( (p1, p2, length) )
        
    full_mask = (1 << N) - 1
    dp = [dict() for _ in range(full_mask+1)]
    dp[0][(0,0)] = 0.0

    if N == 0:
        print(0.0)
        return

    for mask in range(0, full_mask+1):
        for (x,y) in list(dp[mask].keys()):
            for j in range(N):
                if mask & (1 << j):
                    continue
                p1, p2, length_val = segments[j]
                for direction in [0,1]:
                    if direction == 0:
                        start = p1
                        end = p2
                    else:
                        start = p2
                        end = p1
                    dx = start[0] - x
                    dy = start[1] - y
                    non_printing_time = math.sqrt(dx*dx + dy*dy) / S
                    printing_time = length_val / T
                    new_mask = mask | (1 << j)
                    new_pos = end
                    new_time = dp[mask][(x,y)] + non_printing_time + printing_time
                    if new_pos not in dp[new_mask] or dp[new_mask][new_pos] > new_time:
                        dp[new_mask][new_pos] = new_time

    if full_mask not in dp or not dp[full_mask]:
        print(0.0)
    else:
        ans = min(dp[full_mask].values())
        print("{:.15f}".format(ans))

if __name__ == '__main__':
    main()
```

3. **Processing States**: For each bitmask (representing printed segments), and each endpoint, the algorithm checks all unprinted segments.
