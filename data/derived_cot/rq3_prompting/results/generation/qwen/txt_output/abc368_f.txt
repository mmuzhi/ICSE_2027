To solve this problem, we need to determine the winner of a game played between Anna and Bruno, who take turns replacing elements in a sequence with one of their proper divisors. The game ends when no more moves can be made, and the player who cannot move loses. The solution involves analyzing the game using the concept of Grundy numbers (or nimbers) from combinatorial game theory, where each element in the sequence is treated as an independent pile in a game of Nim.

### Approach
1. **Problem Analysis**: The game is an impartial game where each player can replace any element in the sequence with one of its proper divisors. The player who cannot make a move loses. The solution requires determining the winner assuming both players play optimally.

2. **Key Insight**: The game can be broken down into independent subgames for each element in the sequence. Each element's state transitions to its proper divisors, and the Grundy number for each element is computed based on the mex (minimum excludant) of the Grundy numbers of its proper divisors.

3. **Grundy Numbers Calculation**: For each number from 2 to 100,000, compute its Grundy number by considering all its proper divisors. The Grundy number for a number is the smallest non-negative integer not present in the set of Grundy numbers of its proper divisors.

4. **Nim Sum**: The overall game state is determined by the XOR (nim sum) of the Grundy numbers of all elements in the sequence. If the nim sum is non-zero, the first player (Anna) can force a win; otherwise, the second player (Bruno) wins.

5. **Efficiency**: Precompute the divisors for all numbers up to 100,000 using a sieve-like approach. Then, compute the Grundy numbers in increasing order, leveraging the precomputed divisors for efficiency.

### Solution Code
```python
import sys

def main():
    max_val = 100000
    divisors = [[] for _ in range(max_val + 1)]
    
    for j in range(2, max_val + 1):
        divisors[j].append(1)
        
    for i in range(2, max_val + 1):
        for j in range(i * 2, max_val + 1, i):
            divisors[j].append(i)
            
    g = [0] * (max_val + 1)
    g[1] = 0
    for n in range(2, max_val + 1):
        s = set()
        for d in divisors[n]:
            s.add(g[d])
        mex_val = 0
        while mex_val in s:
            mex_val += 1
        g[n] = mex_val
        
    data = sys.stdin.read().split()
    if not data:
        print("Bruno")
        return
        
    n_val = int(data[0])
    arr = list(map(int, data[1:1 + n_val]))
    
    nim = 0
    for num in arr:
        nim ^= g[num]
        
    print("Anna" if nim != 0 else "Bruno")

if __name__ == "__main__":
    main()
```
