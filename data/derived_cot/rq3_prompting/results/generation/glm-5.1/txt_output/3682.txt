The problem asks for the number of arrays of size $n$ with elements in $[1, m]$ such that exactly $k$ indices $i$ ($1 \le i < n$) satisfy $arr[i-1] == arr[i]$.

Let $C$ be the number of contiguous segments of equal elements. If there are $C$ segments, then there are $n - C$ indices where $arr[i-1] \neq arr[i]$.
The number of indices where $arr[i-1] == arr[i]$ is $n - C$.
Wait, if there are $C$ segments, the number of boundaries between segments is $C - 1$.
The total number of adjacent pairs is $n - 1$.
The number of pairs with $arr[i-1] == arr[i]$ is $k$.
The number of pairs with $arr[i-1] \neq arr[i]$ is $(n - 1) - k$.
The number of segments $C$ is exactly $1$ (for the first segment) plus the number of boundaries, which is $(n - 1) - k$.
So $C = n - k$.

We need to choose the lengths of these $C$ segments. The sum of their lengths is $n$, and each length is at least 1.
The number of ways to choose the lengths is the number of compositions of $n$ into $C$ positive integers, which is $\binom{n-1}{C-1}$.
For each segment, we can choose its value from $[1, m]$. The first segment has $m$ choices.
Each subsequent segment must have a value different from the previous segment, so it has $m - 1$ choices.
The total number of choices for the values is $m \cdot (m - 1)^{C - 1}$.

Therefore, the number of good arrays is:
$\binom{n-1}{C-1} \cdot m \cdot (m - 1)^{C - 1}$
Substitute $C = n - k$:
$\binom{n-1}{n - k - 1} \cdot m \cdot (m - 1)^{n - k - 1} = \binom{n-1}{k} \cdot m \cdot (m - 1)^{n - k - 1}$.

```python
MOD = 10**9 + 7

class Solution:
    def countGoodArrays(self, n: int, m: int, k: int) -> int:
        if k > n - 1:
            return 0
        fact = [1] * (n + 1)
        inv_fact = [1] * (n + 1)
        for i in range(1, n + 1):
            fact[i] = fact[i-1] * i % MOD
        inv_fact[n] = pow(fact[n], MOD - 2, MOD)
        for i in range(n, 0, -1):
            inv_fact[i-1] = inv_fact[i] * i % MOD
        
        def comb(n, k):
            if k < 0 or k > n:
                return 0
            return fact[n] * inv_fact[k] % MOD * inv_fact[n-k] % MOD
            
        C = n - k
        ans = comb(n - 1, C - 1) * m % MOD
        ans = ans * pow(m - 1, C - 1, MOD) % MOD
        return ans
```
