Looking at this problem, I need to find minimum weight paths in a graph where vertices are connected if their intervals don't overlap.

**Key observations:**
1. Vertex i connects to vertex j if R_i < L_j or R_j < L_i (intervals don't overlap)
2. For minimum weight path, I can use Dijkstra with vertex weights
3. Neighbors of vertex u split into "right neighbors" (L_v > R_u) and "left neighbors" (R_v < L_u)
4. If I sort by L and R, right neighbors form a suffix, left neighbors form a prefix
5. Using DSU-like "next pointer" technique, I can efficiently skip already-processed vertices
6. Each vertex is found as a neighbor at most once, giving O(N) total neighbor iterations per Dijkstra

**Algorithm:**
- Sort vertices by L and by R separately
- For each query, run Dijkstra with DSU-optimized neighbor finding
- When finding right neighbors (L_v > R_u), iterate through L-sorted list using binary search + DSU skip
- When finding left neighbors (R_v < L_u), iterate through R-sorted list similarly
- Remove vertices from both DSU structures once added to queue (first addition gives shortest path)

**Optimization:** Group queries by source to avoid redundant Dijkstra runs.

```python
import heapq, bisect, sys
from collections import defaultdict

def solve():
    input = sys.stdin.readline
    N = int(input())
    W = list(map(int, input().split()))
    L, R = [0]*N, [0]*N
    for i in range(N):
        L[i], R[i] = map(int, input().split())
    
    Q = int(input())
    qbs = defaultdict(list)
    for i in range(Q):
        s, t = map(int, input().split())
        qbs[s-1].append((t-1, i))
    
    by_L = sorted(range(N), key=lambda i: L[i])
    by_R = sorted(range(N), key=lambda i: R[i])
    L_s = [L[i] for i in by_L]
    R_s = [R[i] for i in by_R]
    pL, pR = [0]*N, [0]*N
    for j, i in enumerate(by_L): pL[i] = j
    for j, i in enumerate(by_R): pR[i] = j
    
    res = [0]*Q
    
    for s, tgts in qbs.items():
        dist = [float('inf')]*N
        dist[s] = W[s]
        pq = [(W[s], s)]
        prL = list(range(N+1))
        prR = list(range(N+1))
        
        def fL(x):
            r = x
            while prL[r] != r: r = prL[r]
            while prL[x] != r: prL[x], x = r, prL[x]
            return r
        
        def fR(x):
            r = x
            while prR[r] != r: r = prR[r]
            while prR[x] != r: prR[x], x = r, prR[x]
            return r
        
        prL[pL[s]] = pL[s]+1
        prR[pR[s]] = pR[s]+1
        tfind = set(t for t,_ in tgts)
        
        while pq and tfind:
            d, u = heapq.heappop(pq)
            if d > dist[u]: continue
            if u in tfind: tfind.discard(u)
            
            idx = bisect.bisect_right(L_s, R[u])
            j = fL(idx)
            while j < N:
                v = by_L[j]
                nd = d + W[v]
                if nd < dist[v]:
                    dist[v] = nd
                    heapq.heappush(pq, (nd, v))
                prL[j] = j+1
                prR[pR[v]] = pR[v]+1
                j = fL(j)
            
            idx = bisect.bisect_left(R_s, L[u])
            j = fR(0)
            while j < idx:
                v = by_R[j]
                nd = d + W[v]
                if nd < dist[v]:
                    dist[v] = nd
                    heapq.heappush(pq, (nd, v))
                prR[j] = j+1
                prL[pL[v]] = pL[v]+1
                j = fR(j)
        
        for t, i in tgts:
            res[i] = -1 if dist[t] == float('inf') else dist[t]
    
    print('\n'.join(map(str, res)))
```
