**Key Insight:** The optimal strategy is to search subtrees in order of decreasing density (sum of probabilities / size). This is because if we have two subtrees T1 and T2 with densities d1 > d2, searching T1 first minimizes the expected cost.

**Algorithm:**
1. Build the tree and compute subtree sizes and sums using post-order DFS
2. Sort children of each vertex by subtree density (decreasing)
3. Compute search order using DFS with sorted children
4. Calculate expected cost as Σ(aᵢ × pos(i)) / Σ(aᵢ)

For the density comparison, I compare `sub_sum[c1] * size[c2]` vs `sub_sum[c2] * size[c1]` to avoid floating point issues.

        for i in range(1, N + 1):
            children[p[i]].append(i)
        
        size = [0] * (N + 1)
        sub_sum = [0] * (N + 1)
        
        stack = [(0, False)]
        while stack:
            v, visited = stack.pop()
            if visited:
                size[v] = 1
                sub_sum[v] = a[v]
                for c in children[v]:
                    size[v] += size[c]
                    sub_sum[v] += sub_sum[c]
            else:
                stack.append((v, True))
                for c in children[v]:
                    stack.append((c, False))
        
        def compare(c1, c2):
            val1 = sub_sum[c1] * size[c2]
            val2 = sub_sum[c2] * size[c1]
            if val1 > val2:
                return -1
            elif val1 < val2:
                return 1
            else:
                return 0
        
        for v in range(N + 1):
            children[v].sort(key=cmp_to_key(compare))
        
        pos = [0] * (N + 1)
        current_pos = 1
        
        stack = [0]
        while stack:
            v = stack.pop()
            if v >= 1:
                pos[v] = current_pos
                current_pos += 1
            for c in reversed(children[v]):
                stack.append(c)
        
        S = sum(a[1:N+1])
        E = 0
        for v in range(1, N + 1):
            E += a[v] * pos[v]
        
        E_mod = E % MOD
        S_mod = S % MOD
        S_inv = pow(S_mod, MOD - 2, MOD)
        ans = E_mod * S_inv % MOD
        print(ans)
