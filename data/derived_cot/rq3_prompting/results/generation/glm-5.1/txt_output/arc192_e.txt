We can compute $A$ by summing over all possible end points $(X,Y)$.
For a fixed end point $(X,Y) \in S$, the number of paths ending at $(X,Y)$ is $h(X,Y)$.
So $A = \sum_{(X,Y) \in S} h(X,Y)$.
This is symmetric, so it doesn't help directly.

Let's use the complement.
Total paths in full grid starting in $S$ is $\sum_{(x,y) \in S} F(x,y)$.
We want to subtract the paths that visit $Ob$.
A path visits $Ob$ if and only if it intersects $[L, R] \times [D, U]$.
By inclusion-exclusion, the number of paths that visit $Ob$ is:
$\sum_{x=L}^R \sum_{y=D}^U (\text{paths that have } (x,y) \text{ as the first point in } Ob)$.
Wait, the first point in $Ob$ must be on the left or bottom boundary of $Ob$.
So it must be $(L, y)$ for $D \le y \le U$ or $(x, D)$ for $L \le x \le R$.
Let $B$ be the set of these boundary points.
For each $(x,y) \in B$, the number of paths that have $(x,y)$ as the first point in $Ob$ is:
(Number of paths from any $(u,v) \in S$ to $(x,y)$ avoiding $Ob$) $\times F(x,y)$.
The number of paths to $(x,y)$ avoiding $Ob$ is exactly the number of paths to $(x-1, y)$ avoiding $Ob$ plus paths to $(x, y-1)$ avoiding $Ob$.
Let $P(x,y)$ be the number of paths to $(x,y)$ avoiding $Ob$.
Then $P(x,y) = P(x-1, y) + P(x, y-1)$ for $(x,y) \in S$, and $P(x,y) = 0$ for $(x,y) \in Ob$.
We need $P(x,y)$ for $(x,y) \in B$.
Since $(x,y) \in B$, either $x=L$ or $y=D$.
If $x=L$, $P(L, y) = P(L-1, y) + P(L, y-1)$.
Since $L-1 < L$, $(L-1, y)$ is not in $Ob$.
So $P(L-1, y)$ is just the number of paths to $(L-1, y)$ in the full grid.
Let $G(x,y)$ be the number of paths to $(x,y)$ in the full grid.
$G(x,y) = \binom{x+y}{x}$.
So for $x < L$ or $y < D$, $P(x,y) = G(x,y)$.
Thus, for $(L, y)$ with $D \le y \le U$:
$P(L, y) = G(L-1, y) + P(L, y-1)$.
Since $P(L, D-1) = G(L, D-1)$, we can compute $P(L, y)$ for all $D \le y \le U$.
Similarly, for $(x, D)$ with $L \le x \le R$:
$P(x, D) = P(x-1, D) + G(x, D-1)$.
Since $P(L-1, D) = G(L-1, D)$, we can compute $P(x, D)$ for all $L \le x \le R$.
This is amazing! The number of paths to the boundary of $Ob$ avoiding $Ob$ can be computed in $O(R-L + U-D)$ time!
Let's double check this.
For $(L, y)$ with $y \ge D$:
$P(L, y) = P(L-1, y) + P(L, y-1)$.
Since $(L-1, y)$ is outside $Ob$, $P(L-1, y) = G(L-1, y)$.
So $P(L, y) = G(L-1, y) + P(L, y-1)$.
This is a simple 1D DP!
Let's verify.
$P(L, D) = G(L-1, D) + P(L, D-1) = G(L-1, D) + G(L, D-1)$.
This is exactly $G(L, D)$ because $G(L, D) = G(L-1, D) + G(L, D-1)$.
Wait, if $P(L, D) = G(L, D)$, then the paths to $(L, D)$ avoiding $Ob$ is the same as all paths to $(L, D)$.
Is this true? Yes, because $(L, D)$ is the first point in $Ob$, so no path to $(L, D)$ can visit $Ob$ before $(L, D)$.
So $P(L, D) = G(L, D)$.
Then $P(L, D+1) = G(L-1, D+1) + P(L, D)$.
This is correct, because any path to $(L, D+1)$ avoiding $Ob$ must come from $(L-1, D+1)$ or $(L, D)$.
Both $(L-1, D+1)$ and $(L, D)$ are either outside $Ob$ or on the boundary, and since we are computing paths AVOIDING $Ob$, the paths to $(L, D)$ avoiding $Ob$ is $P(L, D)$.
So $P(L, y) = G(L-1, y) + P(L, y-1)$ is perfectly correct!
Similarly, $P(x, D) = P(x-1, D) + G(x, D-1)$ is perfectly correct!

So we can compute $P(x,y)$ for all $(x,y) \in B$.
The set of first points in $Ob$ is $B = \{ (L, y) \mid D \le y \le U \} \cup \{ (x, D) \mid L \le x \le R \}$.
Wait, $(L, D)$ is in both sets, but we only count it once.
For each $(x,y) \in B$, the number of paths that have $(x,y)$ as the FIRST point in $Ob$ is $P(x,y) \times F(x,y)$.
Wait, is it?
A path that has $(x,y)$ as the first point in $Ob$ consists of:
1. A prefix from some start point to $(x,y)$ avoiding $Ob$.
2. A suffix from $(x,y)$ to some end point.
The number of such prefixes is exactly $P(x,y)$.
The number of such suffixes is exactly $F(x,y)$.
So the number of such paths is $P(x,y) \times F(x,y)$.
This is incredibly simple and beautiful!
Let's check if there's any overlap.
Can a path have two different first points in $Ob$?
No, the first point in $Ob$ is unique.
So the sets of paths with different first points in $Ob$ are disjoint.
Therefore, the total number of bad paths is $\sum_{(x,y) \in B} P(x,y) F(x,y)$.
Wait, we need to be careful. The start point of the path must be in $S$.
Does $P(x,y)$ count paths starting in $S$?
$P(x,y)$ is the number of paths to $(x,y)$ avoiding $Ob$.
Since the path avoids $Ob$ and ends at $(x,y) \in B \subset Ob$, the only point in $Ob$ it visits is $(x,y)$, which is the end point of the prefix.
So the prefix is entirely in $S$.
Thus, all paths counted by $P(x,y)$ start in $S$.
So the number of bad paths is exactly $\sum_{(x,y) \in B} P(x,y) F(x,y)$.
Wait, what if a path starts in $Ob$?
The problem says: "First, he chooses one block and stands there. Then he performs the following operation... the point after moving must also have a block."
So the path must be entirely in $S$.
The bad paths are those that visit $Ob$.
Since the path must start in $S$, it cannot start in $Ob$.
So any path that visits $Ob$ must have a first point in $Ob$, which is exactly what we counted.
So the number of valid paths is:
$\text{Answer} = \sum_{(x,y) \in S} F(x,y) - \sum_{(x,y) \in B} P(x,y) F(x,y)$.

$F(x,y) = \binom{W-x+H-y+2}{W-x+1} - 1$.

Then $F(x,y) = \binom{u+v+2}{u+1} - 1$.

$\sum_{u=W-B}^{W-A} \sum_{v=H-D}^{H-C} \binom{u+v+2}{u+1} - (B-A+1)(D-C+1)$.

So $\sum_{v=V_1}^{V_2} \binom{u+v+2}{u+1} = \binom{u+V_2+3}{u+2} - \binom{u+V_1+2}{u+2}$.
Then we sum this over $u$:
$\sum_{u=U_1}^{U_2} \left( \binom{u+V_2+3}{u+2} - \binom{u+V_1+2}{u+2} \right)$.
Let $S(V) = \sum_{u=U_1}^{U_2} \binom{u+V+3}{u+2}$.
We know $\binom{u+V+3}{u+2} = \binom{u+V+3}{V+1}$.
So $S(V) = \sum_{u=U_1}^{U_2} \binom{u+V+3}{V+1} = \binom{U_2+V+4}{V+2} - \binom{U_1+V+3}{V+2}$.
This is extremely simple!

$\sum_{u=0}^{U} \binom{u+V+3}{V+1} = \binom{U+V+4}{V+2}$.
Yes, by Hockey Stick Identity.

Let's formalize this.
We want to compute $\text{SumF}(x_1, x_2, y_1, y_2) = \sum_{x=x_1}^{x_2} \sum_{y=y_1}^{y_2} F(x,y)$.
Let $u_1 = W - x_2$, $u_2 = W - x_1$.
Let $v_1 = H - y_2$, $v_2 = H - y_1$.
$\text{SumF} = \sum_{u=u_1}^{u_2} \sum_{v=v_1}^{v_2} \left( \binom{u+v+2}{u+1} - 1 \right)$.
The $-1$ part gives $- (u_2 - u_1 + 1)(v_2 - v_1 + 1)$.
The binomial part:
$\sum_{u=u_1}^{u_2} \left( \binom{u+v_2+3}{u+2} - \binom{u+v_1+2}{u+2} \right)$.

Wait, $\sum_{v=v_1}^{v_2} \binom{u+v+2}{u+1} = \binom{u+v_2+3}{u+2} - \binom{u+v_1+2}{u+2}$.

$\sum_{u=u_1}^{u_2} \binom{u+V+3}{V+1} = \binom{u_2+V+4}{V+2} - \binom{u_1+V+3}{V+2}$.

```python
def sum_F(x1, x2, y1, y2):
    if x1 > x2 or y1 > y2: return 0
    u1 = W - x2
    u2 = W - x1
    v1 = H - y2
    v2 = H - y1
    ans = 0
    # term 1: sum_{u=u1}^{u2} C(u+v2+3, u+2)
    ans += C(u2 + v2 + 4, u2 + 2) - C(u1 + v2 + 3, u1 + 1)
    # term 2: sum_{u=u1}^{u2} C(u+v1+2, u+2)
    ans -= C(u2 + v1 + 3, u2 + 2) - C(u1 + v1 + 2, u1 + 1)
    # term 3: - (x2 - x1 + 1) * (y2 - y1 + 1)
    ans -= (x2 - x1 + 1) * (y2 - y1 + 1)
    return ans % MOD
```
