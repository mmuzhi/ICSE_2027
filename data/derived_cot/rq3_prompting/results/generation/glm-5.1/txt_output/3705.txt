Is there any edge case?

k = 1. Each window is a single element. The number of windows an element appears in is just its frequency in the array. So we want elements that appear exactly once in the array.

Implementation:
```python
class Solution:
    def largestInteger(self, nums: List[int], k: int) -> int:
        from collections import defaultdict
        freq = defaultdict(int)
        n = len(nums)
        for i in range(n - k + 1):
            window = set(nums[i:i+k])
            for x in window:
                freq[x] += 1
        
        ans = -1
        for x, count in freq.items():
            if count == 1:
                ans = max(ans, x)
        
        return ans
```
