Looking at this problem, I need to find the maximum value of composing K distinct linear functions applied to 1.

**Key Insight**: For a fixed set of functions, the optimal ordering can be determined. Comparing two functions f_i and f_j:
- f_j should be outside f_i if: A_j·B_i + B_j > A_i·B_j + B_i
- This simplifies to: B_i·(A_j - 1) > B_j·(A_i - 1)
- Equivalently: B_i/(A_i - 1) > B_j/(A_j - 1)

So functions with **higher B/(A-1)** should be **inside** (applied first). Functions with A=1 have "infinite" B/(A-1), so they go inside.

**Algorithm**:
1. Sort functions by B/(A-1) in decreasing order (innermost first)
2. Use DP where dp[k] = maximum composition value using k functions
3. Process functions in sorted order, adding each as the outermost layer

When adding function f_i outside current composition with value V:
- New value = A_i·V + B_i

**DP Update**: dp[k] = max(dp[k], A_i·dp[k-1] + B_i)

Processing in reverse order (K down to 1) ensures we don't reuse a function in the same iteration.
