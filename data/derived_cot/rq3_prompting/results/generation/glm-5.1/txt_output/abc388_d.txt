    input = sys.stdin.readline
    N = int(input())
    A = list(map(int, input().split()))
    
    size = N + 3
    bit = [0] * size
    
    def bit_add(pos, val):
        i = pos + 1
        while i < size:
            bit[i] += val
            i += i & (-i)
    
    def bit_sum(pos):
        if pos < 0:
            return 0
        i = pos + 1
        s = 0
        while i > 0:
            s += bit[i]
            i -= i & (-i)
        return s
    
    result = []
    
    for j in range(N):
        alien_num = j + 1
        gain = j - bit_sum(alien_num - 1)
        R_j = A[j] + gain
        T_j = R_j + alien_num
        if T_j > N:
            T_j = N
        bit_add(T_j, 1)
        result.append(max(0, R_j - (N - alien_num)))
    
    print(' '.join(map(str, result)))

main()

Looking at this problem, I need to track stone transfers when aliens become adults.

**Key observations:**
- Alien i becomes adult at year i
- When alien i becomes adult, they receive 1 stone from each existing adult with ≥1 stone
- After becoming adult, alien i gives 1 stone to each later alien as long as they have stones

**Deriving the formula:**
Let R_i = stones alien i has right after becoming adult (before giving any away)
- R_i = A_i + gain_i, where gain_i = stones received when becoming adult

Let T_k = R_k + k (the "reach" of alien k - the last alien they'll give a stone to)

Then gain_i = |{k < i : T_k ≥ i}| (count of previous aliens whose reach extends to alien i)

Final stones: B_i = max(0, R_i - (N - i))

**Algorithm:**
Process aliens in order. For each alien j:
1. Query: gain = j - (count of T_k < alien_num) = j - bit_sum(alien_num - 1)
2. Compute R_j = A_j + gain
3. Compute T_j = R_j + alien_num (cap at N since we only query up to N)
4. Insert T_j into BIT
5. Compute B_j = max(0, R_j - (N - alien_num))

**Verification with Sample 1:** N=4, A=[5,0,9,3]
- j=0: gain=0, R=5, T=6→4, B=max(0,5-3)=2 ✓
- j=1: gain=1-bit_sum(1)=1, R=1, T=3, B=max(0,1-2)=0 ✓  
- j=2: gain=2-bit_sum(2)=2, R=11, T=14→4, B=max(0,11-1)=10 ✓
- j=3: gain=3-bit_sum(3)=2, R=5, T=9→4, B=max(0,5-0)=5 ✓
