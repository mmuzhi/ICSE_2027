Wait, what if `min_val[c]` is `inf`?
Then `M_c` will be `inf`, so it won't be in `best_c_list`.
This is handled correctly.

One small detail: `S_hist[i-3][c]` when $i=3$.
`S_hist[0][c]` is 0. This is correct.

Let's double check `SP` calculation.
`SP[k] = (SP[k-1] * P + 1) % MOD`.
`SP[0] = 0`.
`SP[1] = 1`.
`SP[2] = P + 1`.
`SP[3] = P^2 + P + 1`.
This is exactly $\sum_{j=0}^{k-1} P^j$.
So `SP[i] - SP[prev[i]]` is $\sum_{j=prev[i]}^{i-1} P^j$.
This is correct.

What about `H[i]`?
`H[i] = (H[prev[i]] + char[i] * (SP[i] - SP[prev[i]])) % MOD`.
This is exactly the polynomial hash of $Res[i]$.
This is correct.

What about `get_hash_candidate`?
`u = get_node(j, k_len - 1)`
Returns `(H[prev[u]] + char[u] * (SP[k_len] - SP[prev[u]])) % MOD`.
This is the hash of the prefix of length `k_len` of $Res[u]$.
Since `k_len <= j`, and $u$ is the node covering `k_len - 1`, the prefix of length `k_len` of $Res[j]$ is exactly the prefix of length `k_len` of $Res[u]$.
This is correct.

What if `k_len > j`?
Returns `(H[j] + c * (SP[k_len] - SP[j])) % MOD`.
This is the hash of $Res[j] + c^{k\_len - j}$.
This is correct.

The logic seems solid. Time complexity is well within limits, space complexity is $O(n \log n)$ which is around $5 \times 10^4 \times 17 \times 8$ bytes $\approx 7$ MB, perfectly fine.

        n = len(caption)
        if n < 3:
            return ""
        
        MOD = 10**18 + 9
        P = 31
        MAX_B = 16
        
        dp = [float('inf')] * (n + 1)
        dp[0] = 0
        prev = [-1] * (n + 1)
        char = [-1] * (n + 1)
        depth = [0] * (n + 1)
        up = [[-1] * (MAX_B + 1) for _ in range(n + 1)]
        H = [0] * (n + 1)
        SP = [0] * (n + 2)
        for k in range(1, n + 2):
            SP[k] = (SP[k-1] * P + 1) % MOD
            
        S_hist = [[0] * 26 for _ in range(n + 1)]
        for i in range(1, n + 1):
            for c in range(26):
                S_hist[i][c] = S_hist[i-1][c] + abs(ord(caption[i-1]) - ord('a') - c)
                
        min_val = [float('inf')] * 26
        best_j = [-1] * 26
        
        def lca(u, v):
            if depth[u] < depth[v]:
                u, v = v, u
            diff = depth[u] - depth[v]
            for b in range(MAX_B + 1):
                if (diff >> b) & 1:
                    u = up[u][b]
            if u == v:
                return u
            for b in range(MAX_B, -1, -1):
                if up[u][b] != up[v][b]:
                    u = up[u][b]
                    v = up[v][b]
            return up[u][0]
            
        def get_node(j, k):

                return u
            for b in range(MAX_B, -1, -1):
                if up[u][b] != -1 and prev[up[u][b]] > k:

                if hash_A == hash_B and char_A_mid == char_B_mid:
                    low = mid + 1
                else:
                    diff_idx = mid
                    high = mid - 1
                    
            if diff_idx == i:
                return True

            char_B = get_char_candidate(B_j, B_c, i, diff_idx)
            return char_A < char_B

        for i in range(1, n + 1):
            if i >= 3:
                for c in range(26):

                    if val < min_val[c]:
