Looking at this problem, I need to determine if a tree with NK vertices can be decomposed into N paths, each of length K (K vertices).

**Key observations:**
- Each vertex must be an endpoint (degree 1 in path) or internal vertex (degree 2 in path) of exactly one path
- A leaf must be an endpoint
- I'll use tree DP, tracking possible "open path lengths" extending from each node toward its parent

**DP definition:** `dp[v]` = set of possible path lengths from v toward parent, where:
- 0 means all paths in subtree are complete (no open path)
- l (1 ≤ l ≤ K-1) means there's an open path of length l that needs extending

**Transitions at node v:**
- If more than 2 children must connect (don't have 0 in their dp), impossible
- If exactly 2 must-connect children: pair them if l1 + l2 + 1 = K
- If exactly 1 must-connect child: extend path (add l+1) or complete if l+1=K
- If 0 must-connect children: can start new path (add 1), extend one child's path, or pair two children

For the root, answer is "Yes" if 0 ∈ dp[root].

import sys

def solve():
    input = sys.stdin.readline
    N, K = map(int, input().split())
    n = N * K
    adj = [[] for _ in range(n + 1)]
    for _ in range(n - 1):
        u, v = map(int, input().split())
        adj[u].append(v)
        adj[v].append(u)
    
    if K == 1:
        print("Yes")
        return
    
    parent = [0] * (n + 1)
    children = [[] for _ in range(n + 1)]
    stack = [1]
    parent[1] = -1
    order = []
    while stack:
        v = stack.pop()
        order.append(v)
        for u in adj[v]:
            if u != parent[v]:
                parent[u] = v
                children[v].append(u)
                stack.append(u)
    
    dp = [set() for _ in range(n + 1)]
    
    for v in reversed(order):
        if not children[v]:
            dp[v].add(1)
        else:
            must_connect = [c for c in children[v] if 0 not in dp[c]]
            flexible = [c for c in children[v] if 0 in dp[c]]
            
            if len(must_connect) > 2:
                continue
            
            if len(must_connect) == 2:
                ci, cj = must_connect
                li = set(l for l in dp[ci] if 1 <= l <= K - 1)
                lj = set(l for l in dp[cj] if 1 <= l <= K - 1)
                for l1 in li:
                    if K - 1 - l1 in lj:
                        dp[v].add(0)
                        break
            elif len(must_connect) == 1:
                ci = must_connect[0]
                for l in dp[ci]:
                    if 1 <= l <= K - 1:
                        if l + 1 <= K - 1:
                            dp[v].add(l + 1)
                        elif l + 1 == K:
                            dp[v].add(0)
            else:
                dp[v].add(1)
                cl = {}
                for ci in flexible:
                    for l in dp[ci]:
                        if 1 <= l <= K - 1:
                            if l + 1 <= K - 1:
                                dp[v].add(l + 1)
                            elif l + 1 == K:
                                dp[v].add(0)
                            cl.setdefault(l, []).append(ci)
                
                can_pair = False
                for l1 in cl:
                    l2 = K - 1 - l1

                    if l2 in cl:
                        if l1 == l2:
                            if len(cl[l1]) >= 2:
                                can_pair = True
                                break
                        else:
                            if len(cl[l1]) >= 2 or len(cl[l2]) >= 2 or cl[l1][0] != cl[l2][0]:
                                can_pair = True
                                break
                if can_pair:
                    dp[v].add(0)
    
    print("Yes" if 0 in dp[1] else "No")
