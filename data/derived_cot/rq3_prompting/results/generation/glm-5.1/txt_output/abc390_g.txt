Looking at this problem, I need to sum f(P) over all permutations P of (1,2,...,N), where f(P) concatenates the numbers.

**Key Insight**: For number k at position i in a permutation, its contribution is k × 10^(sum of digits of numbers after k). The contribution depends only on k's digit count.

For numbers with the same digit count d, they share the same "structural" contribution C_d. So the total sum = Σ_d C_d × T_d, where T_d = sum of all d-digit numbers.

**Computing C_d**: 
- Let y_j = 10^(d_j) where d_j is digit count of j
- C_d = Σ_{t=0}^{N-1} (N-1-t)! × t! × e_t({y_j : j≠k}) for any k with d_k=d
- The polynomial P(x) = Π_d (1 + 10^d·x)^(n_d) gives us e_t
- P_d(x) = P(x)/(1 + 10^d·x) can be computed via recurrence: [x^t]P_d = P[t] - 10^d·[x^(t-1)]P_d

**Algorithm**:
1. Compute P(x) using NTT-based polynomial multiplication (divide-and-conquer)
2. For each digit count d, compute P_d via recurrence
3. Compute C_d and accumulate total sum

    for dc in digit_counts:
        yd = y_d[dc]
        Pd = [0] * N
        Pd[0] = 1
        for t in range(1, N):
            Pd[t] = (P[t] - yd * Pd[t - 1]) % MOD
        Cd = 0
        for t in range(N):
            Cd = (Cd + fact[N - 1 - t] * fact[t] % MOD * Pd[t]) % MOD
        total_sum = (total_sum + Cd * T_d[dc]) % MOD
    
    print(total_sum)
