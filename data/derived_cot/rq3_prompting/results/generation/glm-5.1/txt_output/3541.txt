Need to check if at least 2 words in `message` exactly match any word in `bannedWords`.

Complexity: O(N + M) time, O(M) space where M is size of bannedWords.

Implementation: set(bannedWords), count = 0, for w in message: if w in banned_set: count += 1, if count == 2: return True. return False.
Risks: None.

=== ANSWER ===
```python
class Solution:
    def reportSpam(self, message: List[str], bannedWords: List[str]) -> bool:
        banned = set(bannedWords)
        count = 0
        for word in message:
            if word in banned:
                count += 1
                if count == 2:
                    return True
        return False
```
