- **Constraints**: N, D <= 100. T_i, L_i <= 100.
- **Algorithm**: For each k from 1 to D, iterate through all N snakes, compute the weight T_i * (L_i + k), and find the maximum. Since N and D are at most 100, the total operations are 100 * 100 = 10,000, which is extremely fast.
- **Feasibility Screen**: Complexity is O(N * D), which is well within the limits. No special data structures needed. No edge cases to worry about since N >= 1, D >= 1, and T_i, L_i >= 1.
- **Implementation Ledger**:

    - Read T and L arrays.
    - Loop k from 1 to D.
    - Inside the loop, loop i from 0 to N-1, compute T[i] * (L[i] + k), find max.
    - Print max for each k.

=== ANSWER ===
```python
import sys

def solve():
    input = sys.stdin.read
    data = input().split()
    if not data:
        return
    N = int(data[0])
    D = int(data[1])
    
    T = []
    L = []
    idx = 2
    for _ in range(N):
        T.append(int(data[idx]))
        L.append(int(data[idx+1]))
        idx += 2
        
    for k in range(1, D + 1):
        max_weight = 0
        for i in range(N):
            weight = T[i] * (L[i] + k)
            if weight > max_weight:
                max_weight = weight
        print(max_weight)

if __name__ == '__main__':
    solve()
```
