Problem: Maximize sum of weights of remaining edges in a tree such that the degree of each node is at most $k$.
This is a tree DP problem.
For each node $u$, we want to keep some edges to its children and possibly its parent.
Let $dp(u, 0)$ be the maximum weight sum in the subtree of $u$, given that the edge between $u$ and its parent is NOT selected.
Let $dp(u, 1)$ be the maximum weight sum in the subtree of $u$, given that the edge between $u$ and its parent IS selected.

If the edge to the parent is selected, $u$ can select at most $k-1$ edges to its children.
If the edge to the parent is not selected, $u$ can select at most $k$ edges to its children.

For a node $u$, let its children be $v_1, v_2, \ldots, v_m$.
The edge to $v_i$ has weight $w_i$.
If we select the edge to $v_i$, we gain $w_i + dp(v_i, 1)$.
If we don't select the edge to $v_i$, we gain $dp(v_i, 0)$.
The difference is $\Delta_i = (w_i + dp(v_i, 1)) - dp(v_i, 0)$.
We should select the edges with the largest positive $\Delta_i$.
To compute $dp(u, 0)$, we can select up to $k$ edges to children.
To compute $dp(u, 1)$, we can select up to $k-1$ edges to children.

```python
import sys
from typing import List

class Solution:
    def maximizeSumOfWeights(self, edges: List[List[int]], k: int) -> int:
        sys.setrecursionlimit(300000)
        n = len(edges) + 1
        adj = [[] for _ in range(n)]
        for u, v, w in edges:
            adj[u].append((v, w))
            adj[v].append((u, w))
            
        def dfs(u, p):
            base_sum = 0
            diffs = []
            for v, w in adj[u]:
                if v == p:
                    continue
                dp0, dp1 = dfs(v, u)
                base_sum += dp0
                diffs.append(w + dp1 - dp0)
            
            diffs.sort(reverse=True)
            
            dp0 = base_sum
            dp1 = base_sum
            
            for i in range(len(diffs)):
                if diffs[i] <= 0:
                    break
                if i < k:
                    dp0 += diffs[i]
                if i < k - 1:
                    dp1 += diffs[i]
                    
            return dp0, dp1

        ans, _ = dfs(0, -1)
        return ans
```
