You will be given a competitive programming problem.
You are given integers N, M and three integer sequences of length M: X = (X_1, X_2, \ldots, X_M), Y = (Y_1, Y_2, \ldots, Y_M), and Z = (Z_1, Z_2, \ldots, Z_M). It is guaranteed that all elements of X and Y are between 1 and N, inclusive.
We call a length-N sequence of non-negative integers A = (A_1, A_2, \ldots, A_N) a good sequence if and only if it satisfies the following condition:

- For every integer i with 1 \le i \le M, the XOR of A_{X_i} and A_{Y_i} is Z_i.

Determine whether a good sequence A=(A_1,A_2,\ldots,A_N) exists, and if it exists, find one good sequence that minimizes the sum of its elements \displaystyle \sum_{i=1}^N A_i.

Notes on XOR

For non-negative integers A and B, their XOR A \oplus B is defined as follows:


- In the binary representation of A \oplus B, the digit in the place corresponding to 2^k \,(k \ge 0) is 1 if and only if exactly one of the digits in the same place of A and B is 1; otherwise, it is 0.


For example, 3 \oplus 5 = 6 (in binary: 011 \oplus 101 = 110).

Input

The input is given from Standard Input in the following format:
N M
X_1 Y_1 Z_1
X_2 Y_2 Z_2
\vdots
X_M Y_M Z_M

Output

If no good sequence exists, print -1.
If a good sequence exists, print one good sequence that minimizes the sum of its elements, separated by spaces.
If there are multiple good sequences with the same minimum sum, printing any of them is accepted.

Constraints


- 1 \le N \le 2\times 10^5
- 0 \le M \le 10^5
- 1 \le X_i, Y_i \le N
- 0 \le Z_i \le 10^9
- All input values are integers.

The problem asks us to find a sequence $A$ of length $N$ such that $A_{X_i} \oplus A_{Y_i} = Z_i$ for all $i$. If it exists, we want to minimize $\sum_{i=1}^N A_i$.

Let's consider the constraints $A_u \oplus A_v = Z$.
This means $A_u = A_v \oplus Z$.
We can model this as a graph where each node is an index $1 \dots N$, and each edge is a constraint $A_u \oplus A_v = Z$.
The graph might have multiple connected components. For each connected component, we can pick a root node $r$ and assign a value to $A_r$. Then all other nodes in the component will have their values determined as $A_v = A_r \oplus P_v$, where $P_v$ is the XOR sum of edge weights along the path from $r$ to $v$.
First, we need to check if the constraints are consistent. For any cycle in the graph, the XOR sum of the edge weights must be 0. If it's not 0, there is no valid sequence, and we should output -1.
We can do this using a Disjoint Set Union (DSU) with XOR path weights, or just BFS/DFS.
Since $N \le 2\times 10^5$ and $M \le 10^5$, BFS/DFS is perfectly fine and takes $O(N+M)$ time.

```python
import sys
from collections import deque

def solve():
    input = sys.stdin.read
    data = input().split()
    if not data:
        return
    N = int(data[0])
    M = int(data[1])
    
    adj = [[] for _ in range(N + 1)]
    idx = 2
    for _ in range(M):
        x = int(data[idx])
        y = int(data[idx+1])
        z = int(data[idx+2])
        adj[x].append((y, z))
        adj[y].append((x, z))
        idx += 3
        
    visited = [False] * (N + 1)
    P = [0] * (N + 1)
    ans = [0] * (N + 1)
    
    for i in range(1, N + 1):
        if not visited[i]:
            visited[i] = True
            P[i] = 0
            comp = [i]
            q = deque([i])
            
            is_consistent = True
            while q:
                u = q.popleft()
                for v, z in adj[u]:
                    if not visited[v]:
                        visited[v] = True
                        P[v] = P[u] ^ z
                        comp.append(v)
                        q.append(v)
                    else:
                        if P[v] != P[u] ^ z:
                            is_consistent = False
                            break
                if not is_consistent:
                    break
            
            if not is_consistent:
                print(-1)
                return
            
            base_val = 0
            comp_len = len(comp)
            for bit in range(31):
                cnt1 = 0
                for v in comp:
                    if (P[v] >> bit) & 1:
                        cnt1 += 1
                if cnt1 > comp_len - cnt1:
                    base_val |= (1 << bit)
            
            for v in comp:
                ans[v] = base_val ^ P[v]
                
    print(*(ans[1:]))

solve()
```
