- loop: for x in nums: s = sum(int(c) for c in str(x)); min_val = min(min_val, s)

=== ANSWER ===
```python
class Solution:
    def minElement(self, nums: List[int]) -> int:
        return min(sum(int(c) for c in str(x)) for x in nums)
```
