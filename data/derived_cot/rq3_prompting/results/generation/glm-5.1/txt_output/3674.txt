Let's summarize the operations:
We process `L` from `n-1` down to `0`.
We maintain `R` (the right bound of the window).
We maintain a stack of `(val, count)`.
We maintain `sum_val_count = sum(val * count)` for the stack.
We maintain `sum_nums = sum(nums[L...R])`.
The cost is `sum_val_count - sum_nums`.

When `L` decreases to `L-1`:
1. We add `nums[L-1]` to the left.
2. `sum_nums += nums[L-1]`.
3. We pop elements from the stack with `val <= nums[L-1]`.
   For each popped `(val, count)`, we subtract `val * count` from `sum_val_count`, and add `count` to a `total_count`.
4. We push `(nums[L-1], total_count + 1)` to the stack.
   We add `nums[L-1] * (total_count + 1)` to `sum_val_count`.
5. Now we might need to shrink `R` because the cost increased.
   While `cost > k` and `R >= L-1`:
      a. We remove `nums[R]` from the right.
      b. `sum_nums -= nums[R]`.
      c. We decrement the count of the top element of the stack.
         `sum_val_count -= stack[-1].val`.
         `stack[-1].count -= 1`.
         If `stack[-1].count == 0`, we pop it.
      d. `R -= 1`.
6. After shrinking, the valid subarrays starting at `L-1` are those ending at `L-1, ..., R`.
   So we add `R - (L-1) + 1` to the answer.

from collections import deque
from typing import List

class Solution:
    def countNonDecreasingSubarrays(self, nums: List[int], k: int) -> int:
        n = len(nums)
        ans = 0
        R = n - 1
        stack = deque()
        sum_val_count = 0
        sum_nums = 0
        
        for L in range(n - 1, -1, -1):
            sum_nums += nums[L]
            total_count = 0
            while stack and stack[-1][0] <= nums[L]:
                val, count = stack.pop()
                sum_val_count -= val * count
                total_count += count
            
            stack.append((nums[L], total_count + 1))
            sum_val_count += nums[L] * (total_count + 1)
            
            while R >= L and sum_val_count - sum_nums > k:
                sum_nums -= nums[R]
                val, count = stack[0]
                sum_val_count -= val
                if count == 1:
                    stack.popleft()
                else:
                    stack[0] = (val, count - 1)
                R -= 1
            
            if R >= L:
                ans += R - L + 1
                
        return ans

from collections import deque
from typing import List

class Solution:
    def countNonDecreasingSubarrays(self, nums: List[int], k: int) -> int:
        n = len(nums)
        ans = 0
        R = n - 1
        stack = deque()
        sum_val_count = 0
        sum_nums = 0
        
        for L in range(n - 1, -1, -1):
            # Add nums[L] to the left of the window
            sum_nums += nums[L]
            total_count = 0
            
            # Pop elements from stack that are dominated by nums[L]
            while stack and stack[-1][0] <= nums[L]:
                val, count = stack.pop()
                sum_val_count -= val * count
                total_count += count
            
            # Push the new dominant element
            stack.append((nums[L], total_count + 1))
            sum_val_count += nums[L] * (total_count + 1)
            
            # Shrink window from the right if cost exceeds k
            while R >= L and sum_val_count - sum_nums > k:
                sum_nums -= nums[R]
                val, count = stack[0]
                sum_val_count -= val
                
                if count == 1:
                    stack.popleft()
                else:
                    stack[0] = (val, count - 1)
                    
                R -= 1
            
            # Add the number of valid subarrays starting at L
            if R >= L:
                ans += R - L + 1
                
        return ans
