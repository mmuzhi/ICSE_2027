We need to solve: given a simple undirected graph with N vertices and M edges, find the minimum number of edges to delete to make it a forest (i.e., acyclic). Equivalent to find the number of edges that must be removed to eliminate all cycles. In any undirected graph, the minimum number of edges to delete to make it a forest is equal to the number of edges beyond a spanning forest. A forest on N vertices can have at most N - c edges, where c is the number of connected components in the forest. Initially, the graph has some number of connected components (let's call it comps). A forest on the same vertex set with the same components would have exactly N - comps edges (since each component tree has vertices_i - 1 edges). So the minimum number of edges to delete is M - (N - comps). Because we can keep a spanning tree in each component, delete all extra edges. That is the minimum because any forest on those vertices can have at most N - comps edges, and we can achieve that by keeping a spanning tree per component. So answer = M - (N - number_of_connected_components). However, careful: The graph may already be a forest, then M <= N - comps, answer 0. So we just compute number of connected components (including isolated vertices). Since N up to 2e5, M up to 2e5, we can do DFS/BFS or union-find.

Thus solution: Use DSU to count components. Initialize each vertex as its own component. For each edge, union u and v. After processing all edges, count number of distinct roots. Then answer = M - (N - components). Equivalent: answer = M - N + components.

Edge case: M=0, components = N, answer = 0 - N + N = 0. Good.

import sys

def solve() -> None:
    data = sys.stdin.read().split()
    if not data:
        return
    it = iter(data)
    N = int(next(it))
    M = int(next(it))

    parent = list(range(N + 1))
    size = [1] * (N + 1)

    def find(x: int) -> int:
        while parent[x] != x:
            parent[x] = parent[parent[x]]
            x = parent[x]
        return x

    def union(a: int, b: int) -> None:
        ra = find(a)
        rb = find(b)
        if ra == rb:
            return
        if size[ra] < size[rb]:
            ra, rb = rb, ra
        parent[rb] = ra
        size[ra] += size[rb]

    for _ in range(M):
        u = int(next(it))
        v = int(next(it))
        union(u, v)

    # Count connected components
    comps = sum(1 for i in range(1, N + 1) if find(i) == i)

    # Minimum edges to delete = M - (N - comps)
    ans = M - N + comps
    print(ans)

if __name__ == "__main__":
    solve()
