- Create an array pos_of_val: for each value v (1..N), store its initial index (1-based).
- Initialize BIT of size N with all 1's.
- total_cost = 0
- For v from N down to 1:
    - original_index = pos_of_val[v]
    - current_position = BIT.prefix_sum(original_index)   // number of present elements up to original_index
    - current_length = BIT.prefix_sum(N)   // total number of remaining elements
    - If current_position < current_length:   // need to move v to the end
        - // cost = sum of k from current_position to current_length-1
        - // sum = (current_position + (current_length-1)) * (current_length - current_position) / 2
        - cost = (current_position + current_length - 1) * (current_length - current_position) // 2
        - total_cost += cost
    - // Remove v: set BIT at original_index to 0
    - BIT.add(original_index, -1)
