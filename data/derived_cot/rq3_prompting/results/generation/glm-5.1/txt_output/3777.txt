You are given an integer array nums and two integers, k and limit. Your task is to find a non-empty subsequence of nums that:

Has an alternating sum equal to k.
Maximizes the product of all its numbers without the product exceeding limit.

Return the product of the numbers in such a subsequence. If no subsequence satisfies the requirements, return -1.
The alternating sum of a 0-indexed array is defined as the sum of the elements at even indices minus the sum of the elements at odd indices.
 
Example 1:

Input: nums = [1,2,3], k = 2, limit = 10
Output: 6
Explanation:
The subsequences with an alternating sum of 2 are:

[1, 2, 3]

Alternating Sum: 1 - 2 + 3 = 2
Product: 1 * 2 * 3 = 6


[2]

Alternating Sum: 2
Product: 2



The maximum product within the limit is 6.

Example 2:

Input: nums = [0,2,3], k = -5, limit = 12
Output: -1
Explanation:
A subsequence with an alternating sum of exactly -5 does not exist.

Example 3:

Input: nums = [2,2,3,3], k = 0, limit = 9
Output: 9

[2, 2]

Product: 2 * 2 = 4

Alternating Sum: 3 - 3 = 0

Product: 2 * 2 * 3 * 3 = 36



The subsequence [2, 2, 3, 3] has the greatest product with an alternating sum equal to k, but 36 > 9. The next greatest product is 9, which is within the limit.

```python
    def maxProduct(self, nums: List[int], k: int, limit: int) -> int:
```
```python
        if k > 1800 or k < -1800:
            return -1
        
        offset = 1800
        dp = [[0, 0] for _ in range(3601)]
        
        for x in nums:
            next_dp = [row[:] for row in dp]
            
            # Start new subsequence with x
            if x <= limit:
                next_dp[offset + x][1] |= 1 << x
            
            for s in range(3601):
                for par in range(2):
                    M = dp[s][par]
                    if M == 0:
                        continue
                    
                    # Multiply mask
                    if x == 0:
                        new_M = 1
                    else:
                        new_M = 0
                        temp_M = M
                        while temp_M:
                            lowest_bit = temp_M & -temp_M
                            i = lowest_bit.bit_length() - 1
                            if i * x <= limit:
                                new_M |= 1 << (i * x)
                            temp_M ^= lowest_bit
                    
                    if new_M:
                        if par == 0:
                            new_s = s + x
                        else:
                            new_s = s - x
                        
                        if 0 <= new_s <= 3600:
                            next_dp[new_s][1 - par] |= new_M
            
            dp = next_dp
        
        target_s = offset + k
        if not (0 <= target_s <= 3600):
            return -1
            
        ans = -1
        for par in range(2):
            M = dp[target_s][par]
            if M:
                # find max set bit
                ans = max(ans, M.bit_length() - 1)
        
        return ans
```

Wait, if we iterate `s` from 3600 down to 0, and `par` from 1 down to 0.
If we include `x`, `new_s` can be `s + x` or `s - x`.
If `par == 0`, `new_s = s + x`. Since we iterate `s` downwards, `new_s > s`, so we won't re-process it in the same element's step!
If `par == 1`, `new_s = s - x`. Since we iterate `s` downwards, `new_s < s`, so we MIGHT re-process it!
Wait, if we update in place, and `par == 1`, `new_s = s - x`.
If we iterate `s` downwards, `new_s` is smaller, so we will encounter it later in the loop.
When we encounter it, `M` will have the newly added bits.
Then we might extend it AGAIN with the same `x`!
This would mean using `x` multiple times, which is incorrect.
So we CANNOT update in place simply by iterating backwards.
We must use `next_dp`.
However, we can optimize the copy:
```python
next_dp = [[dp[s][0], dp[s][1]] for s in range(3601)]
```
This is basically the same as `row[:]`.
What if we use a 1D array of tuples?
`dp = [(0, 0)] * 3601`
`next_dp = list(dp)`
Then `next_dp[s] = (next_dp[s][0], next_dp[s][1] | new_M)`
This avoids creating 3601 inner lists!

```python
class Solution:
    def maxProduct(self, nums: List[int], k: int, limit: int) -> int:
        if k > 1800 or k < -1800:
            return -1
        
        offset = 1800
        dp = [(0, 0)] * 3601
        
        for x in nums:
            next_dp = list(dp)
            
            # Start new subsequence with x
            if x <= limit:
                val = next_dp[offset + x]
                next_dp[offset + x] = (val[0], val[1] | (1 << x))
            
            for s in range(3601):
                m0, m1 = dp[s]
                
                if m0:
                    if x == 0:
                        new_M = 1
                    else:
                        new_M = 0
                        temp_M = m0
                        while temp_M:
                            lowest_bit = temp_M & -temp_M
                            i = lowest_bit.bit_length() - 1
                            if i * x <= limit:
                                new_M |= 1 << (i * x)
                            temp_M ^= lowest_bit
                    
                    if new_M:
                        new_s = s + x
                        if 0 <= new_s <= 3600:
                            val = next_dp[new_s]
                            next_dp[new_s] = (val[0], val[1] | new_M)
                            
                if m1:
                    if x == 0:
                        new_M = 1
                    else:
                        new_M = 0
                        temp_M = m1
                        while temp_M:
                            lowest_bit = temp_M & -temp_M
                            i = lowest_bit.bit_length() - 1
                            if i * x <= limit:
                                new_M |= 1 << (i * x)
                            temp_M ^= lowest_bit
                    
                    if new_M:
                        new_s = s - x
                        if 0 <= new_s <= 3600:
                            val = next_dp[new_s]
                            next_dp[new_s] = (val[0] | new_M, val[1])
            
            dp = next_dp
        
        target_s = offset + k
        if not (0 <= target_s <= 3600):
            return -1
            
        ans = -1
        m0, m1 = dp[target_s]
        if m0:
            ans = max(ans, m0.bit_length() - 1)
        if m1:
            ans = max(ans, m1.bit_length() - 1)
        
        return ans
```
