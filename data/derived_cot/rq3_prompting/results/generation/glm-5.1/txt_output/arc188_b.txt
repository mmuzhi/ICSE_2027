- g = 2 AND N/2 is odd

Verification with examples:
- N=6, K=2: g=gcd(6,4)=2, N/2=3 (odd) → Yes ✓
- N=6, K=3: g=gcd(6,6)=6 → No ✓  
- N=6, K=1: g=gcd(6,2)=2, N/2=3 (odd) → Yes ✓
- N=200000, K=100000: g=200000 → No ✓

```python
import sys
from math import gcd

def solve():
    input = sys.stdin.readline
    T = int(input())
    for _ in range(T):
        N, K = map(int, input().split())
        g = gcd(N, 2 * K)
        if g == 1 or (g == 2 and (N // 2) % 2 == 1):
            print("Yes")
        else:
            print("No")

solve()
```
