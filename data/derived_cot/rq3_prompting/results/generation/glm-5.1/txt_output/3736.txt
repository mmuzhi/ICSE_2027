- Problem: Find the first valid adjacent pair in a string `s` of digits.
- Valid pair: Two adjacent digits `s[i]` and `s[i+1]` such that:

  2. Count of `s[i]` in `s` equals `int(s[i])`
  3. Count of `s[i+1]` in `s` equals `int(s[i+1])`
- Return the first such pair, or "" if none.
- Constraints: `s.length <= 100`, digits '1'-'9'.
- Approach:
  - Count frequencies of each digit in `s`. Since only '1'-'9', can use `collections.Counter` or an array of size 10.
  - Iterate `i` from 0 to `len(s) - 2`.
  - Check if `s[i] != s[i+1]`.
  - Check if `count[s[i]] == int(s[i])` and `count[s[i+1]] == int(s[i+1])`.
  - If both true, return `s[i:i+2]`.
  - If loop finishes, return `""`.
- Feasibility: `O(N)` time, `O(1)` space. Trivially fits constraints.
- Edge cases: No valid pair, pair at the end, same digits adjacent.
- Implementation:
  - `counts = [0] * 10`
  - `for c in s: counts[int(c)] += 1`
  - `for i in range(len(s) - 1):`
  - `  a, b = int(s[i]), int(s[i+1])`
  - `  if a != b and counts[a] == a and counts[b] == b: return s[i:i+2]`
  - `return ""`

=== ANSWER ===
```python
class Solution:
    def findValidPair(self, s: str) -> str:
        counts = [0] * 10
        for c in s:
            counts[int(c)] += 1
            
        for i in range(len(s) - 1):
            a, b = int(s[i]), int(s[i+1])
            if a != b and counts[a] == a and counts[b] == b:
                return s[i:i+2]
                
        return ""
```
