Problem: Count number of valid `copy` arrays.
Difference array of `copy` must match difference array of `original`.
So `copy[i] = copy[0] + (original[i] - original[0])`.
Let `diff[i] = original[i] - original[0]`. Then `copy[i] = copy[0] + diff[i]`.
For each `i`, we must have `bounds[i][0] <= copy[0] + diff[i] <= bounds[i][1]`.
This means `bounds[i][0] - diff[i] <= copy[0] <= bounds[i][1] - diff[i]`.
Let `L_i = bounds[i][0] - diff[i]` and `R_i = bounds[i][1] - diff[i]`.
We need to find the number of integers `copy[0]` such that `max(L_i) <= copy[0] <= min(R_i)`.
The answer is `max(0, min(R_i) - max(L_i) + 1)`.

Let's check Example 1:

- `min_R = infinity`
- `diff = 0`
- Loop `i` from 0 to n-1:

=== ANSWER ===
```python
class Solution:
    def countArrays(self, original: List[int], bounds: List[List[int]]) -> int:
        max_L = -float('inf')
        min_R = float('inf')
        diff = 0
        n = len(original)
        
        for i in range(n):
            max_L = max(max_L, bounds[i][0] - diff)
            min_R = min(min_R, bounds[i][1] - diff)
            if i < n - 1:
                diff += original[i+1] - original[i]
                
        return max(0, min_R - max_L + 1)
```
