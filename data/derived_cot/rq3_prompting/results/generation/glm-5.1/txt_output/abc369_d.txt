You will be given a competitive programming problem.
Takahashi will encounter N monsters in order. The i-th monster (1\leq i\leq N) has a strength of A_i.
For each monster, he can choose to either let it go or defeat it.
Each action awards him experience points as follows:

- If he lets a monster go, he gains 0 experience points.
- If he defeats a monster with strength X, he gains X experience points.
  If it is an even-numbered defeated monster (2nd, 4th, ...), he gains an additional X experience points.

Find the maximum total experience points he can gain from the N monsters.

Input

The input is given from Standard Input in the following format:
N
A_1 A_2 \ldots A_N

Output

Print the maximum total experience points he can gain from the N monsters as an integer.

Constraints


- 1\leq N\leq 2\times 10^5
- 1\leq A_i\leq 10^9
- All input values are integers.

Sample Input 1

5
1 5 3 2 7

Sample Output 1

28

If Takahashi defeats the 1st, 2nd, 3rd, and 5th monsters, and lets the 4th monster go, he gains experience points as follows:

- Defeats a monster with strength A_1=1. He gains 1 experience point.
- Defeats a monster with strength A_2=5. He gains 5 experience points. As it is the 2nd defeated monster, he gains an additional 5 points.
- Defeats a monster with strength A_3=3. He gains 3 experience points.
- Lets the 4th monster go. Takahashi gains no experience points.
- Defeats a monster with strength A_5=7. He gains 7 experience points. As it is the 4th defeated monster, he gains an additional 7 points.

Therefore, in this case, he gains 1+(5+5)+3+0+(7+7)=28 experience points.
Note that even if he encounters a monster, if he lets it go, it does not count as defeated.
He can gain at most 28 experience points no matter how he acts, so print 28.
As a side note, if he defeats all monsters in this case, he would gain 1+(5+5)+3+(2+2)+7=25 experience points.

Each monster $i$ has strength $A_i$.
He can let it go (0 points) or defeat it ($A_i$ points).
If it's an even-numbered defeated monster (2nd, 4th, ...), he gains an additional $A_i$ points.
So the 1st defeated monster gives $A_i$, 2nd gives $2A_i$, 3rd gives $A_i$, 4th gives $2A_i$, etc.
This is equivalent to: for each defeated monster, he gets $A_i$. For every pair of defeated monsters, the second one gives an additional $A_i$.
Let $S$ be the set of indices of defeated monsters. The total points is $\sum_{i \in S} A_i + \sum_{j=1}^{\lfloor |S|/2 \rfloor} A_{S_{2j}}$.
Wait, the order of defeated monsters is the order of their indices.
So if he defeats monsters at indices $i_1 < i_2 < i_3 < \dots < i_k$, he gets:
$A_{i_1} + 2A_{i_2} + A_{i_3} + 2A_{i_4} + \dots$
This means for every pair of defeated monsters, the second one in the pair gives an extra $A_i$.
To maximize the sum, for any pair of defeated monsters, we want the second one to have a larger strength than the first one, or at least we want to pair them up such that the second ones are as large as possible.
Actually, we can choose any subset of monsters to defeat.
Let's use dynamic programming.
Let $dp[i][j]$ be the maximum experience points considering the first $i$ monsters, where $j$ is the number of defeated monsters modulo 2.
$dp[i][0]$: max points considering first $i$ monsters, and the number of defeated monsters is even.
$dp[i][1]$: max points considering first $i$ monsters, and the number of defeated monsters is odd.

Transitions:
For $dp[i][0]$:
- Let monster $i$ go: $dp[i-1][0]$
- Defeat monster $i$: it's an even-numbered defeated monster, so we get $2A_i$. The previous state must have an odd number of defeated monsters. So $dp[i-1][1] + 2A_i$.
Thus, $dp[i][0] = \max(dp[i-1][0], dp[i-1][1] + 2A_i)$.

For $dp[i][1]$:

Thus, $dp[i][1] = \max(dp[i-1][1], dp[i-1][0] + A_i)$.

$dp[0][0] = 0$
$dp[0][1] = -\infty$ (or a very small number, since 0 monsters defeated means even, odd is impossible).

import sys

def solve():
    input = sys.stdin.read
    data = input().split()
    if not data:
        return
    N = int(data[0])
    A = [int(x) for x in data[1:N+1]]
    
    INF = 10**18
    dp0 = 0
    dp1 = -INF
    
    for x in A:
        new_dp0 = max(dp0, dp1 + 2 * x)
        new_dp1 = max(dp1, dp0 + x)
        dp0, dp1 = new_dp0, new_dp1
        
    print(max(dp0, dp1))

solve()
