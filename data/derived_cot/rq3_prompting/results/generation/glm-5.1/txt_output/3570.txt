Let's review the sliding window logic for `at_most(k)`:
We maintain:
- `left_con`: the smallest `left` such that `word[left..right]` has at most `k` consonants.
- `left_vow`: the smallest `left` such that `word[left..right]` does NOT have all 5 vowels.
  (Wait, if `word[left..right]` does not have all 5 vowels, then `left` is NOT a valid left. The valid lefts are those `< left_vow`.)
  So valid lefts are in `[left_con, left_vow - 1]`.
  Number of valid lefts is `max(0, left_vow - left_con)`.
  This matches `valid = left_vow - left_con`.

Let's trace `left_vow` carefully.
`left_vow` starts at 0.
When we move `right` by 1, we add `word[right]` to the window.
Then we try to increment `left_vow` as much as possible while `word[left_vow..right]` still has all 5 vowels.
Wait, if `word[left_vow..right]` has all 5 vowels, we can increment `left_vow` to `left_vow + 1`.
We keep incrementing `left_vow` until `word[left_vow..right]` DOES NOT have all 5 vowels.
So `left_vow` will be the smallest `left` such that the window lacks a vowel.
This means `left_vow - 1` is the LARGEST `left` such that the window has all 5 vowels.
So valid lefts are `<= left_vow - 1`.
And valid lefts are `>= left_con`.
So the number of valid lefts is `max(0, left_vow - left_con)`.

Let's trace `left_con`.
`left_con` starts at 0.
When we move `right` by 1, we add `word[right]` to the window.
If the number of consonants in `word[left_con..right]` exceeds `k`, we increment `left_con` until the number of consonants is `<= k`.
So `left_con` is the SMALLEST `left` such that the window has at most `k` consonants.
So valid lefts are `>= left_con`.

This means the intersection is `[left_con, left_vow - 1]`.
The count is `max(0, left_vow - left_con)`.

Let's write a function `at_most(k)`:

So we CANNOT share the counts between `left_con` and `left_vow`.

def at_most(k):
    if k < 0: return 0
    last_occ = {v: -1 for v in 'aeiou'}
    left_con = 0
    cons_count = 0
    total = 0
    
    for right in range(len(word)):
        char = word[right]
        if char in last_occ:
            last_occ[char] = right
        else:
            cons_count += 1
            
        while cons_count > k and left_con <= right:
            char_left = word[left_con]
            if char_left not in last_occ:
                cons_count -= 1
            left_con += 1
            
        min_last_occ = min(last_occ.values())
        if min_last_occ >= left_con:
            total += min_last_occ - left_con + 1
            
    return total

class Solution:
    def countOfSubstrings(self, word: str, k: int) -> int:
        def at_most(k):
            if k < 0:
                return 0
            last_occ = {v: -1 for v in 'aeiou'}
            left_con = 0
            cons_count = 0
            total = 0
            
            for right in range(len(word)):
                char = word[right]
                if char in last_occ:
                    last_occ[char] = right
                else:
                    cons_count += 1
                    
                while cons_count > k and left_con <= right:
                    char_left = word[left_con]
                    if char_left not in last_occ:
                        cons_count -= 1
                    left_con += 1
                    
                min_last_occ = min(last_occ.values())
                if min_last_occ >= left_con:
                    total += min_last_occ - left_con + 1
                    
            return total
            
        return at_most(k) - at_most(k - 1)
