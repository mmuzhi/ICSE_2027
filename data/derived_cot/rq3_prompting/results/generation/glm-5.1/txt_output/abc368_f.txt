Looking at this problem, I need to find the Grundy value for each number and XOR them together.

**Key insight:** The Grundy value of a number n equals Ω(n) — the total count of prime factors (with multiplicity).

**Verification:**
- G(1) = 0 (terminal), G(2) = mex{0} = 1, G(3) = mex{0} = 1, G(4) = mex{0,1} = 2, G(6) = mex{0,1,1} = 2, G(8) = mex{0,1,2} = 3, G(12) = mex{0,1,1,2,2} = 3

This matches Ω(n) for all cases. The proof: from n with Ω(n)=k, you can reach any divisor d with Ω(d) ranging from 0 to k-1 (since any proper divisor has fewer prime factors), so mex gives exactly k.

**Samples:**
- Sample 1: Ω(2)=1, Ω(3)=1, Ω(4)=2 → XOR = 1⊕1⊕2 = 2 ≠ 0 → Anna ✓
- Sample 2: Ω(2)=1, Ω(3)=1, Ω(4)=2, Ω(6)=2 → XOR = 1⊕1⊕2⊕2 = 0 → Bruno ✓

**Implementation:** Sieve for smallest prime factor, then count prime factors for each number.
