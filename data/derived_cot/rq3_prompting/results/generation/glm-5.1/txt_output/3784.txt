So the count of $u$ in the REMAINING array is $\ge k$.
If $u \notin P$, then the count of $u$ in the original array is the same as in the remaining array, so it's $\ge k$.
Thus $u \in S$, and $depth(u) \le d_{off}$.
If $u \in P$, then the count of $u$ in the remaining array is $count(u) - 1 \ge k$.
So $count(u) \ge k + 1 > k$.
Thus $u \in P$ and $count(u) > k$, so $depth(u) \le d_{on}$.
In either case, $d \le \max(d_{on}, d_{off})$.
So the answer is EXACTLY $\max(d_{on}, d_{off})$.
The proof is solid!

Let's summarize the algorithm and data structures.
1. Build a Trie of all words.
   - Each node needs: `count`, `depth`, `children` (dict or array of 26).
   - Since total characters is $\le 10^5$, we can use a list of dicts or a list of arrays.
   - `Trie = [{'children': {}, 'count': 0}]`
   - For each word, insert into Trie, incrementing `count` for each node visited.
2. Find all nodes in $S$ (nodes with `count` $\ge k$).
   - We can just iterate through all nodes in the Trie and collect those with `count` $\ge k$.
3. Find $M_1$ and $M_2$.
   - $M_1$ is the node in $S$ with the maximum `depth`.
   - $M_2$ is the node in $S$ with the maximum `depth` that is NOT on the path to $M_1$.
   - To check if a node is on the path to $M_1$, we can just trace the path of the word that created $M_1$ (or any word passing through $M_1$).
   - Wait, $M_1$ is a node. We can store the path of nodes from root to $M_1$ in a set `path_M1`.
   - Then $M_2$ is the node in $S$ with max `depth` not in `path_M1`.
4. For each word $w$:
   - Find $d_{on}$: max `depth` of node on the path of $w$ with `count` $> k$.
   - Find $d_{off}$:
     - If the path of $w$ contains $M_1$, then $d_{off} = depth(M_2)$ (or 0 if $M_2$ doesn't exist).
     - If the path of $w$ does not contain $M_1$, then $d_{off} = depth(M_1)$.
   - Answer for $w$ is $\max(d_{on}, d_{off})$.

Let's trace this carefully.
Is it possible that $w$ passes through $M_1$, and $M_2$ is on the path of $w$?
No, because $M_2$ is not on the path to $M_1$.
If $w$ passes through $M_1$, its path is exactly the path to $M_1$ (plus descendants, but $M_1$ is the deepest in $S$, so no descendants in $S$).
So the path of $w$ (restricted to $S$) is exactly the path to $M_1$ restricted to $S$.
Since $M_2$ is not on the path to $M_1$, $M_2$ is not on the path of $w$.
So $d_{off} = depth(M_2)$ is correct.

Wait, what if there are multiple nodes with the same maximum depth in $S$?
Let $M_{1a}$ and $M_{1b}$ be two nodes in $S$ with the same maximum depth.
Then $M_1$ can be either of them.
If we pick $M_{1a}$ as $M_1$, then $M_{1b}$ is not on the path to $M_{1a}$ (since they have the same depth, they can't be ancestors of each other).
So $M_2$ will be $M_{1b}$, and $depth(M_2) = depth(M_1)$.
So $d_{off}$ will be $depth(M_1)$ for all $w$.
This is correct, because for any $w$, it can pass through at most one of $M_{1a}$ and $M_{1b}$.
So the other one is not on the path of $w$, and $d_{off} = depth(M_1)$.
So the logic holds perfectly!

```python
class Solution:
    def longestCommonPrefix(self, words: List[str], k: int) -> List[int]:
        if len(words) <= k:
            return [0] * len(words)
        
        # Trie node: [count, depth, children_dict]
        # To save memory and time, we can use a list of dicts for children, or just a list of size 26.
        # Since sum of lengths is 10^5, list of dicts is fine.
        trie = [{'count': 0, 'depth': 0, 'children': {}}]
        
        def add_word(word):
            node = 0
            for ch in word:
                if ch not in trie[node]['children']:
                    trie.append({'count': 0, 'depth': trie[node]['depth'] + 1, 'children': {}})
                    trie[node]['children'][ch] = len(trie) - 1
                node = trie[node]['children'][ch]
                trie[node]['count'] += 1
                
        for word in words:
            add_word(word)
            
        # Find M1 and M2
        # M1 is the node with max depth and count >= k
        # M2 is the node with max depth and count >= k, not on path to M1
        
        # We need to find the node with max depth in S
        # Since we want max depth, we can just iterate over all nodes
        max_depth = -1
        M1 = -1
        for i in range(1, len(trie)):
            if trie[i]['count'] >= k:
                if trie[i]['depth'] > max_depth:
                    max_depth = trie[i]['depth']
                    M1 = i
                    
        if M1 == -1:
            return [0] * len(words)
            
        # Find path to M1
        # We can trace back from M1, but we don't have parent pointers.
        # So we just find M2 by checking all nodes in S.
        # But wait, how to check if a node is on the path to M1?
        # We can just find the word that passes through M1.
        # Actually, we don't need to trace back. We just need to find M2.
        # M2 is the node in S with max depth not on path to M1.
        # To do this, we can just find the top 2 deepest nodes in S.
        # Wait, if M2 is on the path to M1, it's an ancestor of M1.
        # So we need the deepest node in S that is NOT an ancestor of M1.
        # How to find ancestors of M1?
        # We can just search for any word that reaches M1, and record its path.
        
        # Let's find a word that passes through M1.
        # Actually, we can just find the path to M1 by DFS or by checking words.
        # Since we need to compute d_on for each word, we will traverse the Trie for each word anyway.
        # So we can just find the path of M1 by doing a DFS from root and finding M1.
        
        # Let's find the path to M1.
        path_M1 = set()
        # DFS to find M1 and record path
        stack = [(0, [0])] # (node, path)
        found_M1_path = None
        while stack:
            u, path = stack.pop()
            if u == M1:
                found_M1_path = path
                break
            for ch, v in trie[u]['children'].items():
                stack.append((v, path + [v]))
                
        path_M1 = set(found_M1_path)
        
        # Find M2
        max_depth_M2 = -1
        M2 = -1
        for i in range(1, len(trie)):
            if trie[i]['count'] >= k and i not in path_M1:
                if trie[i]['depth'] > max_depth_M2:
                    max_depth_M2 = trie[i]['depth']
                    M2 = i
                    
        # Now compute answers
        ans = []
        for word in words:
            d_on = 0
            d_off = 0
            node = 0
            passes_M1 = False
            for ch in word:
                node = trie[node]['children'][ch]
                if trie[node]['count'] > k:
                    d_on = max(d_on, trie[node]['depth'])
                if node == M1:
                    passes_M1 = True
            
            if passes_M1:
                if M2 != -1:
                    d_off = trie[M2]['depth']
            else:
                d_off = trie[M1]['depth']
                
            ans.append(max(d_on, d_off))
            
        return ans
```

```python
class Solution:
    def longestCommonPrefix(self, words: List[str], k: int) -> List[int]:
        if len(words) <= k:
            return [0] * len(words)
            
        # Pre-allocate lists. Max nodes = sum(len(w) for w in words) + 1
        max_nodes = sum(len(w) for w in words) + 1
        count = [0] * max_nodes
        depth = [0] * max_nodes
        parent = [-1] * max_nodes
        children = [{} for _ in range(max_nodes)]
        
        node_count = 1
        
        for word in words:
            node = 0
            for ch in word:
                if ch not in children[node]:
                    children[node][ch] = node_count
                    parent[node_count] = node
                    depth[node_count] = depth[node] + 1
                    node_count += 1
                node = children[node][ch]
                count[node] += 1
                
        max_depth = -1
        M1 = -1
        for i in range(1, node_count):
            if count[i] >= k:
                if depth[i] > max_depth:
                    max_depth = depth[i]
                    M1 = i
                    
        if M1 == -1:
            return [0] * len(words)
            
        path_M1 = set()
        curr = M1
        while curr != -1:
            path_M1.add(curr)
            curr = parent[curr]
            
        max_depth_M2 = -1
        M2 = -1
        for i in range(1, node_count):
            if count[i] >= k and i not in path_M1:
                if depth[i] > max_depth_M2:
                    max_depth_M2 = depth[i]
                    M2 = i
                    
        ans = []
        for word in words:
            d_on = 0
            d_off = 0
            node = 0
            passes_M1 = False
            for ch in word:
                node = children[node][ch]
                if count[node] > k:
                    d_on = max(d_on, depth[node])
                if node == M1:
                    passes_M1 = True
                    
            if passes_M1:
                if M2 != -1:
                    d_off = depth[M2]
            else:
                d_off = depth[M1]
                
            ans.append(max(d_on, d_off))
            
        return ans
```

Space complexity: $O(\sum |words[i]|)$.

```python
class Solution:
    def longestCommonPrefix(self, words: List[str], k: int) -> List[int]:
        if len(words) <= k:
            return [0] * len(words)
            
        total_len = sum(len(w) for w in words)
        max_nodes = total_len + 1
        
        count = [0] * max_nodes
        depth = [0] * max_nodes
        parent = [-1] * max_nodes
        children = [{} for _ in range(max_nodes)]
        
        node_count = 1
        
        for word in words:
            node = 0
            for ch in word:
                if ch not in children[node]:
                    children[node][ch] = node_count
                    parent[node_count] = node
                    depth[node_count] = depth[node] + 1
                    node_count += 1
                node = children[node][ch]
                count[node] += 1
                
        max_depth = -1
        M1 = -1
        for i in range(1, node_count):
            if count[i] >= k:
                if depth[i] > max_depth:
                    max_depth = depth[i]
                    M1 = i
                    
        if M1 == -1:
            return [0] * len(words)
            
        path_M1 = set()
        curr = M1
        while curr != -1:
            path_M1.add(curr)
            curr = parent[curr]
            
        max_depth_M2 = -1
        M2 = -1
        for i in range(1, node_count):
            if count[i] >= k and i not in path_M1:
                if depth[i] > max_depth_M2:
                    max_depth_M2 = depth[i]
                    M2 = i
                    
        ans = []
        for word in words:
            d_on = 0
            d_off = 0
            node = 0
            passes_M1 = False
            for ch in word:
                node = children[node][ch]
                if count[node] > k:
                    d_on = depth[node]
                if node == M1:
                    passes_M1 = True
                    
            if passes_M1:
                if M2 != -1:
                    d_off = depth[M2]
            else:
                d_off = depth[M1]
                
            ans.append(max(d_on, d_off))
            
        return ans
```
