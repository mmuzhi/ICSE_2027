- Problem: Convert a date string "yyyy-mm-dd" to its binary representation "year_bin-month_bin-day_bin" without leading zeros.
- Constraints: Date is valid, between 1900 and 2100.
- Approach: Split the string by '-', convert each part to integer, then to binary string using `bin()`, and slice off the '0b' prefix. Join with '-'.
- Feasibility: `bin(int(x))[2:]` works perfectly. Time complexity O(1), Space O(1).
- Implementation: `return "-".join(bin(int(x))[2:] for x in date.split("-"))`
- Edge cases: None, `bin()` handles positive integers correctly without leading zeros.
- Draft: Just write the one-liner.

=== ANSWER ===
```python
class Solution:
    def convertDateToBinary(self, date: str) -> str:
        return "-".join(bin(int(x))[2:] for x in date.split("-"))
```
