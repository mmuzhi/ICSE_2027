4. Target runs must alternate values starting with 1

For each target run with value v spanning positions [start, end], I compute m_i = (end-start)/2 + 1, representing how many "V-runs" of the target value are in that span.

The number of ways to merge 2m-1 initial runs into 1 run is the odd double factorial (2m-3)!! = 1×3×5×...×(2m-3).

Since operations for different target runs are independent, the total count is:
- Multinomial coefficient for interleaving: S! / ∏(m_i-1)!
- Times product of ways for each run: ∏(2m_i-3)!!

Where S = ∑(m_i-1) is total operations needed.
