- **Chosen Candidate**: Coordinate compression / Prefix Sums + Binary Search.
  - Since X is sorted, we can build a prefix sum array `S` where `S[i]` is the sum of P_1...P_i.
  - For a query [L, R], find the leftmost index `l` such that X_l >= L, and the rightmost index `r` such that X_r <= R.

  - If `l <= r`, the answer is `S[r+1] - S[l]`. Otherwise, 0.
