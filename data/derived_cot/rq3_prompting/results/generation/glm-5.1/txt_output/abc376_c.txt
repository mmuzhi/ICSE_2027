There are N toys numbered from 1 to N, and N-1 boxes numbered from 1 to N-1.
Toy i\ (1 \leq i \leq N) has a size of A_i, and box i\ (1 \leq i \leq N-1) has a size of B_i.

- Place each of the N toys into one of the N boxes (the N-1 existing boxes plus the newly purchased box).

Determine whether there exists a value of x such that he can execute step 2, and if it exists, find the minimum such x.

N

Output

Constraints


- 2 \leq N \leq 2 \times 10^5
- 1 \leq A_i, B_i \leq 10^9
- All input values are integers.

Sample Output 1

3

def solve():
    import sys
    input = sys.stdin.readline
    N = int(input())
    A = list(map(int, input().split()))
    B = list(map(int, input().split()))
    A.sort()
    B.sort()
    # necessary condition
    for i in range(N-1):
        if A[i] > B[i]:
            print(-1)
            return
    lo = 1
    hi = max(A[-1], B[-1])
    # define feasible
    def feasible(x):
        i = 0  # index in B
        inserted = False
        for j in range(N):
            if not inserted:
                if i == N-1 or x <= B[i]:
                    # take x
                    if A[j] > x:
                        return False
                    inserted = True
                else:
                    # take B[i]
                    if A[j] > B[i]:
                        return False
                    i += 1
            else:
                # take B[i]
                if A[j] > B[i]:
                    return False
                i += 1
        return True

    while lo < hi:
        mid = (lo + hi) // 2
        if feasible(mid):
            hi = mid
        else:
            lo = mid + 1
    print(lo)
