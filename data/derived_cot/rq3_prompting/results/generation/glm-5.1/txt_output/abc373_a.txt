- Feasibility: Very simple, no edge cases to worry about.
- Implementation:

  - For i in 1..12:
    - read S
    - if len(S) == i: count += 1

=== ANSWER ===
```python

import sys

def main():
    count = 0
    # Read exactly 12 lines from standard input
    for i in range(1, 13):
        line = sys.stdin.readline().strip()
        if len(line) == i:
            count += 1
    print(count)

if __name__ == "__main__":
    main()
```
