Here, \operatorname{floor}(x) represents the largest integer not exceeding x.
 What is lexicographical order for sequences?
A sequence S = (S_1,S_2,\ldots,S_{|S|}) is lexicographically smaller than a sequence T = (T_1,T_2,\ldots,T_{|T|}) if either 1. or 2. below holds.
Here, |S| and |T| represent the lengths of S and T, respectively.

-  |S| \lt |T| and (S_1,S_2,\ldots,S_{|S|}) = (T_1,T_2,\ldots,T_{|S|}). 
-  There exists an integer 1 \leq i \leq \min\lbrace |S|, |T| \rbrace such that both of the following hold:

-  (S_1,S_2,\ldots,S_{i-1}) = (T_1,T_2,\ldots,T_{i-1})
-  S_i is (numerically) smaller than T_i.

    primes = []
    is_prime = [True] * (M + 1)
    for i in range(2, M + 1):

            primes.append(i)
            for j in range(i * i, M + 1, i):
                is_prime[j] = False

    exps = [0] * len(primes)
    for idx, p in enumerate(primes):
        # Exponent in M!
        m = M
        while m > 0:
            m //= p
            exps[idx] += m
        # Exponent in (K!)^N
        k = K
        while k > 0:
            k //= p
            exps[idx] -= k * N
            
    # Compute product using divide and conquer

        if l > r:
            return 1
        if l == r:
            return primes[l] ** exps[l]
        mid = (l + r) // 2
        return product(l, mid) * product(mid + 1, r)
    
    return product(0, len(primes) - 1)

        self.bit = [0] * (n + 1)
        
    def add(self, idx, delta):
        idx += 1
        while idx <= self.n:
            self.bit[idx] += delta
            idx += idx & (-idx)
            
    def query(self, idx):
        # sum of [0, idx)
        res = 0
        while idx > 0:
            res += self.bit[idx]
            idx -= idx & (-idx)
        return res
        
    def find_kth(self, k):
        # find smallest idx such that sum [0, idx] >= k
        idx = 0
        bit_mask = 1
        while bit_mask <= self.n:
            bit_mask <<= 1
        bit_mask >>= 1
            
        for i in range(bit_mask, 0, bit_mask >> 1):
            if idx + i <= self.n and self.bit[idx + i] < k:
                k -= self.bit[idx + i]
                idx += i
                
        return idx # 0-based index
