Implementation:
```python
class Solution:
    def isZeroArray(self, nums: List[int], queries: List[List[int]]) -> bool:
        n = len(nums)
        diff = [0] * (n + 1)
        for l, r in queries:
            diff[l] += 1
            diff[r + 1] -= 1
        
        curr = 0
        for i in range(n):
            curr += diff[i]
            if curr < nums[i]:
                return False
        return True
```

Complexity:
Time: O(N + Q)
Space: O(N)

```python
class Solution:
    def isZeroArray(self, nums: List[int], queries: List[List[int]]) -> bool:
        n = len(nums)
        diff = [0] * (n + 1)
        
        for l, r in queries:
            diff[l] += 1
            diff[r + 1] -= 1
            
        curr_coverage = 0
        for i in range(n):
            curr_coverage += diff[i]
            if curr_coverage < nums[i]:
                return False
                
        return True
```
