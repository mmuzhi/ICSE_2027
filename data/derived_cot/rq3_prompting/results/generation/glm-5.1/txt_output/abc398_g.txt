Looking at this problem, I need to analyze a game on a bipartite graph where players add edges while keeping it bipartite.

**Key observations:**
1. A graph is bipartite iff it has no odd cycles. Adding an edge creates an odd cycle iff both endpoints are in the same partition.
2. The game ends when the graph becomes a complete bipartite graph with partitions of sizes A and B.
3. Total moves = A×B - M (where M is initial edges).
4. First player wins if total moves is odd.

**Parity analysis:**
- If N is odd: A and B have different parities, so A×B is always even. Thus A×B - M has same parity as M.
- If N is even: A×B is odd iff A is odd. So A×B - M is odd iff A and M have different parities.

**For N even, can players control A's parity?**
- When merging two components, we can flip one component's coloring
- Flipping a component with odd size changes A's parity
- If there are components with odd size (k_odd > 0), the first player can control A's parity
- If k_odd = 0, A's parity is fixed

**Verification with examples:**
- Example 1: N=4 (even), M=3, one component (2,2), k_odd=0, A=2 (even), M=3 (odd) → different parity → Aoki wins ✓
- Example 2: N=4 (even), M=2, two components (1,1) each, k_odd=0, A=2 (even), M=2 (even) → same parity → Takahashi wins ✓  
- Example 3: N=9 (odd), M=5 (odd) → Aoki wins ✓

**Algorithm:**
1. BFS to find components and their bipartitions
2. If N is odd: first player wins iff M is odd
3. If N is even:
   - If k_odd > 0: first player wins (can control parity)
   - If k_odd = 0: first player wins iff A and M have different parities

```python
import sys
from collections import deque

def solve():
    input = sys.stdin.readline
    N, M = map(int, input().split())
    adj = [[] for _ in range(N)]
    for _ in range(M):
        u, v = map(int, input().split())
        u -= 1
        v -= 1
        adj[u].append(v)
        adj[v].append(u)
    
    color = [-1] * N
    k_odd = 0
    total_a = 0
    
    for start in range(N):
        if color[start] != -1:
            continue
        # BFS to find component and bipartition
        queue = deque([start])
        color[start] = 0
        cnt = [0, 0]
        while queue:
            u = queue.popleft()
            cnt[color[u]] += 1
            for v in adj[u]:
                if color[v] == -1:
                    color[v] = 1 - color[u]
                    queue.append(v)
        if (cnt[0] + cnt[1]) % 2 == 1:
            k_odd += 1
        total_a += cnt[0]
    
    if N % 2 == 1:
        # A*B is always even, so A*B - M has same parity as M
        if M % 2 == 1:
            print("Aoki")
        else:
            print("Takahashi")
    else:
        if k_odd > 0:
            # First player can control parity of A
            print("Aoki")
        else:
            # Parity of A is fixed
            if (total_a % 2) != (M % 2):
                print("Aoki")
            else:
                print("Takahashi")

solve()
```
