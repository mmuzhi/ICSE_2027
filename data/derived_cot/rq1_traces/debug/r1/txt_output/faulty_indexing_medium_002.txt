=== RAW ANSWER ===
To solve this problem, we need to generate all numbers with sequential digits within a given range [low, high]. A sequential digit number is one where each subsequent digit is one more than the previous digit. For example, 123, 234, and 345 are sequential digit numbers.

### Approach
1. **Problem Analysis**: The task is to find all numbers between `low` and `high` where each digit in the number is exactly one more than the previous digit. The numbers must be in increasing order and within the specified range.
2. **Intuition**: We can generate these numbers using a breadth-first search (BFS) approach. Start with all single-digit numbers (1 through 9). For each number, check if it lies within the range [low, high]. If it does, add it to the result. Then, generate the next sequential digit number by appending the next digit (current last digit + 1) to the number, provided the last digit is less than 9.
3. **Algorithm Selection**: Using a queue (deque for efficient popping from the front), we start with numbers 1 to 9. For each number popped from the queue, if it lies within [low, high], it is added to the result. The last digit of the number is checked; if it is less than 9, the next sequential digit number is generated by appending (last digit + 1) to the current number. This new number is then added to the queue for further processing.
4. **Complexity Analysis**: The algorithm efficiently generates all possible sequential digit numbers without redundancy. The total number of such numbers is limited (as the maximum length is 9 digits), making the approach feasible even for the upper constraint (high = 10^9).

### Solution Code
```python
from collections import deque

class Solution:
    def sequentialDigits(self, low, high):
        out = []
        queue = deque(range(1, 10))
        while queue:
            elem = queue.popleft()
            if low <= elem <= high:
                out.append(elem)
            last = elem % 10
            if last < 9:
                new_num = elem * 10 + (last + 1)
                queue.append(new_num)
        return out
```

### Explanation
1. **Initialization**: The queue is initialized with numbers 1 through 9, representing the starting digits for sequential numbers.
2. **Processing Queue**: For each number dequeued:
   - If the number lies within the range [low, high], it is added to the result list.
   - The last digit of the number is checked. If it is less than 9, the next sequential digit number is formed by appending (last digit + 1) to the current number. This new number is enqueued for further processing.
3. **Termination**: The loop continues until the queue is empty, ensuring all possible sequential digit numbers within the range are generated.
4. **Output**: The result list, containing all valid sequential digit numbers in increasing order, is returned.

This approach efficiently generates the required numbers using BFS, ensuring correctness and optimal performance given the problem constraints.
