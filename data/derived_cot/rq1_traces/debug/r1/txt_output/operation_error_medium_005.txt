=== RAW ANSWER ===

To solve this problem, we need to find the longest word from a given dictionary that can be formed by deleting some characters of a given string. If there are multiple words of the same length, the lexicographically smallest word should be returned.

### Approach
1. **Check Subsequence**: For each word in the dictionary, check if it is a subsequence of the given string. A word is a subsequence if we can form it by deleting some characters of the string without changing the order of the remaining characters.
2. **Track Longest and Lexicographically Smallest Word**: Maintain a result string that keeps track of the longest valid word found. If multiple words have the same length, the lexicographically smallest one is chosen.

### Solution Code
```python
class Solution:
    def findLongestWord(self, s: str, d: List[str]) -> str:
        res = ''
        for cand in d:
            if self.is_subsequence(s, cand) and (len(cand) > len(res) or (len(cand) == len(res) and cand < res)):
                res = cand
        return res
    
    def is_subsequence(self, s: str, d: str) -> bool:
        i, j = 0, 0
        while i < len(s) and j < len(d):
            if s[i] == d[j]:
                j += 1
            i += 1
        return j == len(d)
```

### Explanation
1. **Subsequence Check**: The `is_subsequence` method uses two pointers to traverse both the input string `s` and the candidate word `d`. If characters at both pointers match, the pointer for `d` is moved forward. The pointer for `s` is always moved forward. If the pointer for `d` reaches the end, it means `d` is a subsequence of `s`.
2. **Updating Result**: For each candidate word in the dictionary, if it is a valid subsequence, we check if it is longer than the current result or if it is the same length but lexicographically smaller. The result is updated accordingly.

This approach ensures that we efficiently check each candidate word and maintain the longest and lexicographically smallest valid word. The time complexity is O(N * M), where N is the number of words in the dictionary and M is the length of the input string `s`.
