The state of our recursive function is defined by the current position in the sequence and the last value used in the permutation.
2. **Handling DI Sequences**: For each position in the sequence:
   - If the current character is 'D', the next number must be smaller than the current value.
   - If the current character is 'I', the next number must be larger than the current value.
3. **Modulo Operation**: Since the result can be very large, we use modulo \(10^9 + 7\) to keep the numbers manageable and prevent overflow.

### Solution Code
```python
from collections import defaultdict

class Solution:
    def numPermsDISequence(self, s: str) -> int:
        MOD = 10**9 + 7
        mem = defaultdict(int)
        
        def dfs(i, val=0):
            if i == len(s):
                return 1
            if (i, val) in mem:
                return mem[(i, val)]
            p = 0
            if s[i] == "D":
                for ind in range(0, val + 1):
                    p += dfs(i + 1, ind) % MOD
```
