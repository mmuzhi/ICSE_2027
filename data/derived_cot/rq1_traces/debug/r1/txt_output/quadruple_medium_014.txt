### Approach

2. **Recursive Exploration with Memoization**: We use a recursive function to explore all possible combinations of toppings. Memoization is applied to optimize repeated calculations.
3. **Handling Base Costs**: For each base cost, we calculate the closest possible total cost by adding the base cost to the best combination of toppings that brings the sum closest to the target value.
4. **Key Comparisons**: When comparing possible sums, we prioritize the absolute difference from the target value. In case of ties, the smaller sum is preferred.

from typing import List
from functools import cache

class Solution:
    def closestCost(self, baseCosts: List[int], toppingCosts: List[int], target: int) -> int:
        toppingCosts = toppingCosts * 2  # Allow each topping to be used up to two times

        @cache
        def fn(i, x):
            """Return the sum of subsequence of toppingCosts[i:] closest to x."""
            if x < 0 or i == len(toppingCosts):
                return 0

            take = toppingCosts[i] + fn(i + 1, x - toppingCosts[i])
