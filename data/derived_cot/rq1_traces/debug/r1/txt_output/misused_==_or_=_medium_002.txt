To solve this problem, we need to determine if it is possible to cut a path in a grid such that there is no valid path from the top-left corner (0, 0) to the bottom-right corner (m-1, n-1). The solution involves using Depth-First Search (DFS) to mark cells that can reach the bottom-right corner and then using Breadth-First Search (BFS) to check for a bottleneck node that, if removed, would block all paths.

### Approach
1. **DFS from the Bottom-Right Corner**: We start from the bottom-right corner (m-1, n-1) and use DFS to mark all cells that can reach this corner by moving up or left. These cells are marked with a value of 2.
2. **BFS from the Top-Left Corner**: We then perform a BFS starting from the top-left corner (0, 0), moving only right or down through cells marked by the DFS. During the BFS, we check if there is a level (or layer) where there is exactly one node. If such a node exists and it is not the target node (m-1, n-1), it indicates a bottleneck, meaning we can cut this node to block all paths.

### Solution Code
```python
from typing import List
import collections

class Solution:
    def isPossibleToCutPath(self, grid: List[List[int]]) -> bool:
        m, n = len(grid), len(grid[0])
        dirs = [(-1, 0), (0, -1)]
        
        def dfs(i: int, j: int) -> None:
            grid[i][j] = 2
            for di, dj in dirs:
                ni, nj = i + di, j + dj
                if 0 <= ni < m and 0 <= nj < n and grid[ni][nj] == 1:
                    dfs(ni, nj)
        
        dfs(m - 1, n - 1)
        
        dq = collections.deque([(0, 0)])
        grid[0][0] = 0
        dirs = [(1, 0), (0, 1)]
        while dq:
            level_size = len(dq)
            for _ in range(level_size):
                i, j = dq.popleft()
                if i == m - 1 and j == n - 1:
                    return False
                for di, dj in dirs:
                    ni, nj = i + di, j + dj
                    if 0 <= ni < m and 0 <= nj < n and grid[ni][nj] == 2:
                        dq.append((ni, nj))
                        grid[ni][nj] = 0
            if len(dq) == 1 and dq[0] != (m - 1, n - 1):
                return True
        return True
```

### Explanation
1. **DFS Marking**: The DFS starts from the bottom-right corner and marks all reachable cells (moving up or left) with a value of 2. This helps identify cells that can reach the end.
2. **BFS for Bottleneck Check**: The BFS starts from the top-left corner and moves right or down through cells marked as 2. By processing each level of the BFS, we check if there is a level with exactly one node (a bottleneck). If such a node exists and it is not the target node, we return True, indicating that cutting this node will block all paths. If the BFS reaches the target node, we return False, indicating no such bottleneck exists.

This approach efficiently combines DFS and BFS to determine the existence of a critical node that can block all paths from the start to the end of the grid.
