### Approach

2. **Intuition**: For each element in the array, we determine how many subarrays have that element as the minimum. This is done by finding the previous smaller element (left boundary) and the next smaller element (right boundary). The number of such subarrays is the product of the distances to these boundaries.

class Solution:
    def sumSubarrayMins(self, arr: List[int]) -> int:
