2. **Optimize `s1`**: Filter `s1` to include only characters present in `s2` to reduce unnecessary processing.
3. **Track Cycles**: Use a dictionary to track positions in `s1` where cycles start. This helps in detecting repeating patterns which can be used to optimize the calculation.
4. **Cycle Detection**: When a cycle is detected, calculate the number of times the cycle can be repeated within the remaining repetitions of `s1` and compute the result based on the cycle and the remaining part.

from collections import defaultdict

class Solution:
    def getMaxRepetitions(self, s1: str, n1: int, s2: str, n2: int) -> int:

                start = ptr + 1

        cycle2 = len(rec) - 1 - cycle_start
