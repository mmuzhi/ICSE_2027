To solve this problem, we need to find the longest valid substring of a given word that does not contain any of the forbidden substrings. The approach involves using a sliding window technique combined with a trie data structure to efficiently check for forbidden substrings.

### Approach
1. **Trie Construction**: Build a trie from the list of forbidden substrings. This allows efficient checking of substrings within the main word.
2. **Sliding Window Technique**: Use a sliding window to maintain the longest valid substring. For each position in the word, check all possible substrings ending at that position to see if they are forbidden. Adjust the window's left boundary to exclude any forbidden substrings found.
3. **Max Forbidden Length**: Precompute the maximum length of forbidden substrings to limit the range of substring checks, improving efficiency.

from typing import List

class Solution:
    def longestValidSubstring(self, word: str, forbidden: List[str]) -> int:

### Explanation
1. **Trie Construction**: The trie is built by iterating over each forbidden substring and inserting each character into the trie. The end of a forbidden substring is marked with a special character (`#`).
2. **Sliding Window**: The sliding window is defined by `left` and `right` pointers. For each `right` pointer, we check all possible substrings ending at `right` with lengths up to the maximum forbidden substring length. This is done by traversing the trie from the current character backwards.
3. **Adjusting the Window**: If a forbidden substring is found, the `left` pointer is adjusted to exclude the forbidden substring, ensuring the window remains valid.
4. **Max Length Calculation**: The maximum length of valid substrings is updated continuously as the window slides through the word.

This approach efficiently checks for forbidden substrings using the trie and maintains the longest valid substring using the sliding window technique, resulting in an optimal solution.
