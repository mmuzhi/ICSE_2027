from typing import List

class Solution:
    def removeDuplicates(self, nums: List[int]) -> int:
        if len(nums) < 3:
            return len(nums)

        p1, p2 = 1, 2

                    p2 += 1

            nums[p1] = nums[p2]
            p2 += 1

2. **Pointers Initialization**: `p1` starts at index 1 and `p2` starts at index 2 to check the third element initially.
3. **Main Loop**: Traverse the array with `p2`. For each element at `p2`, check if it forms a third consecutive duplicate with the elements at `p1` and `p1 - 1`.
