To solve this problem, we need to generate all possible valid IP addresses from a given string. A valid IP address consists of four decimal numbers (each ranging from 0 to 255) separated by dots. Each number must not have leading zeros unless it is exactly 0.

### Approach
The approach to solve this problem involves using backtracking to explore all possible ways to split the string into four valid segments. Here are the key steps:

1. **Backtracking Function**: We define a recursive function `backtrack` that builds possible IP address segments and checks their validity.
2. **Base Cases**:
   - If we reach the end of the string and have exactly four segments, we add the valid IP address to the result list.
   - If we exceed the string length or have more than four segments, we backtrack.
3. **Handling Leading Zeros**: If a segment starts with '0', it must be a single '0'. We handle this case separately to avoid invalid segments like "01".
4. **Valid Segment Check**: For each segment, we check if it is a valid number (0-255) and recursively explore further segments.

### Solution Code
```python
from typing import List

class Solution:
    def restoreIpAddresses(self, s: str) -> List[str]:
        res = []
        cur = []
        
        def backtrack(i):
            if i == len(s) and len(cur) == 4:
                res.append(".".join(cur))
                return
            if len(cur) > 4 or i > len(s):
                return
            if i < len(s) and s[i] == '0':
                cur.append('0')
                backtrack(i + 1)
                cur.pop()
                return
            j = 0
            while j < 3 and i + j < len(s):
                num_str = s[i:i+j+1]
                if int(num_str) < 256:
                    cur.append(num_str)
                    backtrack(i + j + 1)
                    cur.pop()
                j += 1
        
        backtrack(0)
        return res
```

### Explanation
1. **Backtracking Function**: The `backtrack` function is called recursively to explore all possible segments starting from index `i`.
2. **Base Cases**:
   - When `i` reaches the end of the string and we have exactly four segments, we join the segments with dots and add the result to `res`.
   - If we have more than four segments or exceed the string length, we backtrack.
3. **Handling Leading Zeros**: If the current character is '0', we immediately form a segment with '0' and proceed to the next index, ensuring no leading zeros in subsequent segments.
4. **Valid Segment Check**: For each possible segment length (1 to 3 digits), we check if the segment is a valid number (less than 256) and continue exploring further segments recursively.

This approach efficiently explores all valid IP address combinations using backtracking, ensuring correctness by checking segment validity and handling leading zeros appropriately.
