=== RAW ANSWER ===

To solve this problem, we need to construct a quad tree from a given binary grid. A quad tree is a tree data structure where each internal node has exactly four children. Each node in the quad tree represents a subgrid of the original grid. The goal is to recursively divide the grid into four subgrids and check if all values in a subgrid are the same. If they are, the node becomes a leaf node; otherwise, it becomes an internal node with four children.

### Approach
1. **Recursive Division**: The grid is recursively divided into four quadrants (top-left, top-right, bottom-left, bottom-right) until each subgrid is a single cell (leaf node) or can no longer be divided.
2. **Check for Uniformity**: For each subgrid, check if all values are the same. If they are, create a leaf node. If not, create an internal node with four children.
3. **Conversion to Boolean**: Convert the grid values (0 or 1) to boolean values (False or True) as required by the problem statement.

class Node:
    def __init__(self, val, isLeaf, topLeft, topRight, bottomLeft, bottomRight):
        self.val = val
        self.isLeaf = isLeaf
        self.topLeft = topLeft
        self.topRight = topRight
        self.bottomLeft = bottomLeft
        self.bottomRight = bottomRight

class Solution:
    def construct(self, grid: List[List[int]]) -> 'Node':
        level = len(grid)
        if level == 0:
            return None
        return self.buildTree(grid, 0, level, 0, level, level)
        
    def buildTree(self, grid, rs, re, cs, ce, level):
        if level == 1:
            return Node(bool(grid[rs][cs]), True, None, None, None, None)
        
        next_level = level // 2
        tl = self.buildTree(grid, rs, rs + next_level, cs, cs + next_level, next_level)
        tr = self.buildTree(grid, rs, rs + next_level, cs + next_level, ce, next_level)
        bl = self.buildTree(grid, rs + next_level, re, cs, cs + next_level, next_level)
        br = self.buildTree(grid, rs + next_level, re, cs + next_level, ce, next_level)

1. **Base Case**: When the subgrid size is 1x1 (level 1), create a leaf node with the boolean value of the grid cell.
2. **Recursive Division**: For larger grids, divide the grid into four quadrants. The `buildTree` method is called recursively for each quadrant with updated boundaries.
3. **Check Uniformity**: After constructing the four children, check if all are leaf nodes and have the same value. If true, merge them into a single leaf node. Otherwise, create an internal node with the four children.
