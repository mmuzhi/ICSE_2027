To solve this problem, we need to find the minimum cost to travel from the top-left corner (0, 0) to the bottom-right corner (m-1, n-1) of a grid where each cell contains an arrow pointing in one of four directions (right, left, down, up). Moving in the direction of the arrow incurs no cost, but changing direction (using a different arrow) costs 1. The solution involves using a combination of Depth-First Search (DFS) to explore paths without cost and Breadth-First Search (BFS) to explore paths with cost increments when changing direction.

### Approach
1. **Initialization**: Start by initializing the grid dimensions, a queue for BFS, and the directions corresponding to each arrow (1: right, 2: left, 3: down, 4: up).
2. **DFS for Zero-Cost Paths**: Use DFS to follow the arrows from the starting cell (0, 0) without any cost. Mark visited cells by setting them to `None` and add them to the queue for subsequent BFS processing.
3. **BFS for Cost-Based Paths**: Process the queue in levels. For each level (representing paths that require a cost increment), check if the target cell (m-1, n-1) has been visited. If so, return the current cost. Otherwise, increment the cost and explore all four directions from each cell in the current level. For each new direction, use DFS to follow the arrow path without additional cost and add visited cells to the queue for the next level.
4. **Termination**: Since the problem guarantees a path exists, the loop terminates when the target cell is visited, returning the minimum cost.

### Solution Code
```python
from collections import deque
from typing import List

class Solution:
    def minCost(self, grid: List[List[int]]) -> int:
        m, n = len(grid), len(grid[0])
        cost = 0
        queue = deque()
        dirs = ((), (0, 1), (0, -1), (1, 0), (-1, 0))
        
        seen = lambda x, y: not (0 <= x < m and 0 <= y < n) or grid[x][y] is None
        
        def dfs(x, y):
            while not seen(x, y):
                dx, dy = dirs[grid[x][y]]
                grid[x][y] = None
                queue.append((x, y))
                x, y = x + dx, y + dy
        
        dfs(0, 0)
        
        while queue:
            if grid[m-1][n-1] is None:
                return cost
            cost += 1
            q_len = len(queue)
            for _ in range(q_len):
                x, y = queue.popleft()
                for dx, dy in dirs[1:]:
                    dfs(x + dx, y + dy)
        
        return cost
```

3. **BFS Processing**: The BFS processes each level of the queue, where each level represents paths that require an additional cost. Before processing each level, it checks if the target cell has been visited (grid[m-1][n-1] is `None`), returning the current cost if true.
4. **Cost Increment**: For each cell in the current level, all four directions are explored. Each new direction initiates a DFS to follow the arrow path without additional cost, adding visited cells to the queue for the next level.
