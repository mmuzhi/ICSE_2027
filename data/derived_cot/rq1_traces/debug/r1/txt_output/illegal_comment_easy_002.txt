=== RAW ANSWER ===
To solve this problem, we need to compute the alternate digit sum of a given integer. The alternate digit sum is calculated by adding the digits at even indices (0-based) and subtracting the digits at odd indices. 

### Approach
1. **Convert the integer to a string**: This allows us to easily iterate over each digit by index.
2. **Iterate over each digit**: For each digit in the string representation of the integer:
   - If the index is even (0, 2, 4, etc.), add the digit to the running sum.
   - If the index is odd (1, 3, 5, etc.), subtract the digit from the running sum.
3. **Return the computed sum**: After processing all digits, the sum is returned as the result.

### Solution Code
```python
class Solution:
    def alternateDigitSum(self, n: int) -> int:
        s = str(n)
        count = 0
        for i in range(len(s)):
            if i % 2 == 0:
                count += int(s[i])
            else:
                count -= int(s[i])
        return count
```

### Explanation
1. **Conversion to String**: The integer `n` is converted to a string `s` to facilitate indexing and iteration over each digit.
2. **Initialization**: A variable `count` is initialized to 0 to accumulate the alternate digit sum.
3. **Iteration and Sum Calculation**: For each character in the string `s`:
   - If the index `i` is even, the corresponding digit is converted to an integer and added to `count`.
   - If the index `i` is odd, the corresponding digit is converted to an integer and subtracted from `count`.
4. **Result**: The accumulated value in `count` is returned, representing the alternate digit sum of the input integer.

This approach efficiently processes each digit in the integer, applying the appropriate sign based on its position, and computes the required sum. The solution handles all edge cases, including single-digit numbers and numbers with leading zeros (which are not present in integer string representations).
