### Solution Code
```python
class Solution:
    def longestValidSubstring(self, word: str, forbidden: List[str]) -> int:
        trie = {}
        max_len_forbidden = 0
        for f in forbidden:
            max_len_forbidden = max(max_len_forbidden, len(f))
            node = trie

                node = node[c]

            for k in range(max_len_forbidden):
```

3. **Efficient Checking**: By using the trie, we efficiently check each substring in reverse order. If a forbidden substring is found, the `left` pointer is updated to ensure the window remains valid, and the maximum length of valid substrings is tracked.

This approach ensures that we efficiently check for forbidden substrings and maintain the longest valid substring using a sliding window, resulting in an optimal solution.
