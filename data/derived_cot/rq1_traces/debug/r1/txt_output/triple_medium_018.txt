```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def levelOrderBottom(self, root: Optional[TreeNode]) -> List[List[int]]:
        def checkReverse(ans):
            l, r = 0, len(ans) - 1
            while l < r:
                ans[l], ans[r] = ans[r], ans[l]
                l += 1
                r -= 1
            return ans

        if not root:
            return []
        q = [root]
        ans = []
        while q:
            n = len(q)
            level = []
            for _ in range(n):
                curr = q.pop(0)
                level.append(curr.val)
                if curr.left:
                    q.append(curr.left)
                if curr.right:
                    q.append(curr.right)
            ans.append(level)
        return checkReverse(ans)
```

### Explanation

1. **checkReverse Function**: This function reverses the list of levels. It uses two pointers starting from the beginning and end of the list, swapping elements until they meet in the middle.
2. **Handling Empty Tree**: If the input `root` is `None`, the function immediately returns an empty list.
3. **Queue Management**: The queue `q` is initialized with the root node. For each level, the current size of the queue is stored in `n` to process all nodes at that level.
4. **Processing Nodes**: For each node in the current level, its value is added to the current level's list. Non-null left and right children are enqueued for subsequent levels.
5. **Reversing Levels**: After collecting all levels in a top-down manner, the `checkReverse` function reverses the list to produce the bottom-up level order traversal.
