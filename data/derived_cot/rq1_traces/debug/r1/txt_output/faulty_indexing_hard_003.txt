=== RAW ANSWER ===
To solve this problem, we need to find the length of the longest valid substring in a given string `word` such that no forbidden substring appears within it. The forbidden substrings are provided as a list of strings. 

### Approach
1. **Trie Construction**: We construct a trie (prefix tree) from the forbidden words. Each node in the trie represents a character, and the end of a forbidden word is marked with a special key (e.g., "end").
2. **Checking Forbidden Substrings**: For each position in the string `word`, we check if any substring starting at that position (and extending up to 10 characters, as forbidden words are at most 10 characters long) matches any forbidden word in the trie. This is done by traversing the trie character by character.
3. **Sliding Window with Two Pointers**: We use a sliding window approach with two pointers, `i` and `j`, where `i` starts from the end of the string and moves backward. The pointer `j` marks the exclusive right boundary of the current valid substring. If a forbidden word is found starting at `i`, we adjust `j` to exclude the forbidden word, ensuring the substring remains valid.
4. **Maximizing Valid Substring Length**: During each iteration, we calculate the length of the current valid substring (from `i` to `j-1`) and update the maximum length found so far.

### Solution Code
```python
class Solution:
    def longestValidSubstring(self, word: str, forbidden: List[str]) -> int:
        trie = {}
        for f in forbidden:
            t = trie
            for c in f:
                if c not in t:
                    t[c] = {}
                t = t[c]
            t["end"] = True
        
        def isForbidden(s):
            t = trie
            for i in range(min(len(s), 10)):
                c = s[i]
                if c not in t:
                    return False
                t = t[c]
                if "end" in t:
                    return i+1
            return False
        
        res = 0
        j = len(word)
        for i in range(len(word)-1, -1, -1):
            truc = isForbidden(word[i:j])
            if truc:
                j = i + truc - 1
            res = max(res, j - i)
        return res
```

### Explanation
1. **Trie Construction**: The trie is built by iterating over each forbidden word. For each character in the word, we traverse the trie, creating nodes as needed, and mark the end of the word with "end".
2. **Forbidden Check**: The `isForbidden` function checks if any substring starting at the given position (up to 10 characters) is in the trie. It returns the length of the forbidden word found or `False` if none.
3. **Sliding Window**: Starting from the end of the string, we move `i` backward. For each `i`, we check the substring `word[i:j]` for forbidden words. If found, we adjust `j` to `i + length - 1` to exclude the forbidden word.
4. **Result Calculation**: The length of the current valid substring (`j - i`) is compared with the maximum length found so far (`res`), updating `res` if the current substring is longer. This ensures we track the longest valid substring throughout the iteration.

This approach efficiently checks for forbidden substrings using a trie and maintains a valid sliding window to maximize the substring length, providing an optimal solution.
