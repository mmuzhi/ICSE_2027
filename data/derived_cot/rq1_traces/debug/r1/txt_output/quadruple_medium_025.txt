=== RAW ANSWER ===

To solve this problem, we need to generate all possible valid IP addresses from a given string of digits. A valid IP address consists of four segments separated by dots, where each segment is a number between 0 and 255. Additionally, no segment can have leading zeros unless it is exactly "0".

### Approach
The approach to solve this problem involves using backtracking to explore all possible ways to split the string into four valid segments. Here are the key steps:

1. **Base Cases**:
   - If we reach the end of the string and have exactly four segments, we add the current combination to the result list.
   - If we exceed the string length or have more than four segments, we backtrack.

2. **Handling Leading Zeros**:
   - If a segment starts with '0', it must be exactly "0". We handle this case separately to avoid leading zeros in other segments.

3. **Generating Segments**:
   - For each position in the string, generate segments of length 1, 2, or 3 digits, ensuring they form a valid number less than 256.

### Solution Code
```python
class Solution:
    def restoreIpAddresses(self, s: str) -> List[str]:
        res = []
        cur = []
        
        def backtrack(i):
            if i == len(s) and len(cur) == 4:
                res.append(".".join(cur))
                return
            if len(cur) > 4 or i > len(s):
                return
            if i < len(s) and s[i] == '0':
                cur.append('0')
                backtrack(i + 1)
                cur.pop()
                return
            j = 0
            while j < 3 and i + j < len(s):
                num_str = s[i:i+j+1]
                if int(num_str) < 256:
                    cur.append(num_str)
                    backtrack(i + j + 1)
                    cur.pop()
                j += 1
        
        backtrack(0)
        return res
```

### Explanation
1. **Backtracking Function**:
   - The function `backtrack(i)` starts at index `i` of the string and tries to form valid segments.
   - If the current index `i` reaches the end of the string and exactly four segments are formed, the combination is added to the result list.

2. **Handling Leading Zeros**:
   - If the current character is '0', it is immediately added as a segment, and the function proceeds to the next index. This ensures segments like "0" are allowed but prevents segments like "01".

3. **Generating Valid Segments**:
   - For each position, the loop generates segments of length 1, 2, or 3 digits. Each segment is checked to ensure it is a valid number less than 256 before proceeding further.

This approach efficiently explores all valid combinations using backtracking, ensuring that all constraints for valid IP addresses are met.
