=== RAW ANSWER ===

To solve this problem, we need to generate all possible valid IP addresses from a given string. A valid IP address consists of four decimal numbers (each ranging from 0 to 255) separated by dots. Each number must not have leading zeros unless it is exactly 0.

### Approach
The approach to solve this problem involves using backtracking to explore all possible ways to split the string into four valid parts. Here are the key steps:

1. **Backtracking Function**: We define a recursive function that builds possible IP address parts by iterating through the string.
2. **Base Cases**:
   - If we reach the end of the string and have exactly four parts, we add the current combination to the result list.
   - If we exceed the string length or have more than four parts, we backtrack.
3. **Handling Leading Zeros**: If a part starts with '0', it must be a single '0'. We handle this case separately to avoid invalid combinations.
4. **Valid Number Check**: For each part, we check if it is a valid number (0-255) and recursively explore further splits.

### Solution Code
```python
from typing import List

class Solution:
    def restoreIpAddresses(self, s: str) -> List[str]:
        res = []
        cur = []
        
        def backtrack(i):
            if i == len(s) and len(cur) == 4:
                res.append(".".join(cur))
                return
            if len(cur) > 4 or i > len(s):
                return
            if i < len(s) and s[i] == '0':
                cur.append('0')
                backtrack(i + 1)
                cur.pop()
                return
            j = 0
            while j < 3 and i + j < len(s):
                num_str = s[i:i+j+1]
                if int(num_str) < 256:
                    cur.append(num_str)
                    backtrack(i + j + 1)
                    cur.pop()
                j += 1
        
        backtrack(0)
        return res
```

### Explanation
1. **Initialization**: We initialize the result list `res` and the current combination list `cur`.
2. **Backtracking Function**: The `backtrack` function starts at index `i` of the string.
3. **Base Cases**:
   - If `i` reaches the end of the string and `cur` has exactly four parts, we join and add the combination to `res`.
   - If `cur` exceeds four parts or `i` goes beyond the string length, we backtrack.
4. **Leading Zero Handling**: If the current character is '0', we add it to `cur` and move to the next index, ensuring no leading zeros in subsequent parts.
5. **Valid Number Check**: For each possible part length (1 to 3 digits), we check if the substring forms a valid number (less than 256). If valid, we recursively explore further splits.

This approach efficiently explores all valid IP address combinations using backtracking, ensuring correctness by checking for leading zeros and valid number ranges.
