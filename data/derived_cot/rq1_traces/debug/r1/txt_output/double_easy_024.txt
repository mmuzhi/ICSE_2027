=== RAW ANSWER ===
To solve this problem, we need to sort a given string in a specific alternating manner: first by picking the smallest available character, then the next smallest character larger than the last picked character, and so on until no more characters can be picked. Then, we reverse the direction and pick the largest available character, followed by the next largest character smaller than the last picked character, repeating this process until all characters are used.

### Approach
1. **Frequency Dictionary**: We first create a frequency dictionary to count the occurrences of each character in the input string.
2. **Unique Sorted Letters**: We extract the unique characters from the string and sort them in ascending order.
3. **Alternating Passes**: 
   - **Forward Pass**: Iterate through the sorted unique letters. For each character present in the frequency dictionary, append it to the result, decrement its count, and remove it from the dictionary if its count reaches zero.
   - **Backward Pass**: Iterate through the sorted unique letters in reverse order. Similarly, append each present character to the result, decrement its count, and remove it if the count becomes zero.
4. **Repeat Until Empty**: Repeat the forward and backward passes until the frequency dictionary is empty, indicating all characters have been used.

### Solution Code
```python
class Solution:
    def sortString(self, s: str) -> str:
        freq = {}
        letters = sorted(set(s))
        res = ""
        for char in s:
            freq[char] = freq.get(char, 0) + 1
        while freq:
            for char in letters:
                if char in freq:
                    res += char
                    freq[char] -= 1
                    if freq[char] == 0:
                        del freq[char]
            for char in letters[::-1]:
                if char in freq:
                    res += char
                    freq[char] -= 1
                    if freq[char] == 0:
                        del freq[char]
        return res
```

### Explanation
1. **Frequency Dictionary**: We initialize a dictionary `freq` to count how many times each character appears in the string `s`.
2. **Unique Sorted Letters**: The variable `letters` holds the sorted list of unique characters from `s`.
3. **Forward Pass**: For each character in `letters`, if it exists in `freq`, we append it to the result string `res`, decrement its count in `freq`, and delete the character from `freq` if its count drops to zero.
4. **Backward Pass**: Similarly, we iterate over `letters` in reverse order, appending each existing character to `res`, decrementing its count, and removing it if the count becomes zero.
5. **Loop Until Empty**: The while loop continues until `freq` is empty. Each iteration consists of a forward pass followed by a backward pass, ensuring characters are picked in the required alternating order until all characters are exhausted.

This approach efficiently processes the string by leveraging sorted unique characters and alternating traversal directions, ensuring the result meets the specified sorting criteria.
