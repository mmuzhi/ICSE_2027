To solve this problem, we need to determine the number of valid configurations for placing N queens on an N×N chessboard such that no two queens threaten each other. The solution uses backtracking with optimizations to efficiently check for conflicts using row and diagonal tracking arrays.

### Approach
1. **Backtracking**: We place queens column by column, starting from the first column. For each column, we try placing a queen in each row and check if it is a valid position.
2. **Conflict Checking**: We use three arrays to track conflicts:
   - `low` array to track occupied rows.
   - `upper` array to track occupied anti-diagonals.
   - `lower` array to track occupied main diagonals.
3. **Recursive Placement**: For each valid position found, we recursively place queens in subsequent columns. If we reach the last column, we record the valid configuration.
4. **Reset State**: After backtracking from a recursive call, we reset the board and tracking arrays to their previous state to explore other configurations.

### Solution Code
```python
class Solution:
    def totalNQueens(self, n: int) -> int:
        def addans(board, ans):
            temp = []
            for row in board:
                for j in range(len(row)):
                    if row[j] == "Q":
                        temp.append(j + 1)
            ans.append(temp)
        
        def solve(col, board, low, upper, lower, ans, n):
            if col == n:
                addans(board, ans)
                return
            for row in range(n):
                if low[row] == 0 and upper[n - 1 + col - row] == 0 and lower[row + col] == 0:
                    board[row][col] = "Q"
                    low[row] = 1
                    upper[n - 1 + col - row] = 1
                    lower[row + col] = 1
                    solve(col + 1, board, low, upper, lower, ans, n)
                    board[row][col] = 0  # Reset the board after backtracking
                    low[row] = 0
                    upper[n - 1 + col - row] = 0
                    lower[row + col] = 0
        
        ans = []
        board = [[0] * n for _ in range(n)]
        low = [0] * n
        upper = [0] * (2 * n - 1)
        lower = [0] * (2 * n - 1)
        solve(0, board, low, upper, lower, ans, n)
        return len(ans)
```

1. **addans Function**: This function collects the column positions of queens for each valid configuration. Each position is adjusted to 1-based indexing for consistency with traditional problem outputs.

3. **Initialization and Execution**: The main function initializes the board and tracking arrays, then starts the recursive placement from the first column. The result is the count of all valid configurations found.
