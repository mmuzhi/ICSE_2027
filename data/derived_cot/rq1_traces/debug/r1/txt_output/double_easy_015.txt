=== RAW ANSWER ===

To solve this problem, we need to determine if a given matrix is a valid n x n matrix where each row and column contains all numbers from 1 to n exactly once. 

### Approach
The approach involves checking each row and each column of the matrix to ensure they contain all integers from 1 to n exactly once. This can be efficiently done using sets to verify the presence of each required number. Here's the step-by-step approach:

1. **Check Rows**: For each row in the matrix, convert the row into a set and compare it with the expected set of integers from 1 to n. If any row does not match the expected set, the matrix is invalid.
2. **Check Columns**: Transpose the matrix to convert columns into rows, then check each transposed column (original column) in the same way as the rows. If any column does not match the expected set, the matrix is invalid.

### Solution Code
```python
from typing import List

class Solution:
    def checkValid(self, matrix: List[List[int]]) -> bool:
        n = len(matrix)
        expected = set(range(1, n + 1))
        
        # Check each row
        for row in matrix:
            if set(row) != expected:
                return False
        
        # Check each column by transposing the matrix
        for col in zip(*matrix):
            if set(col) != expected:
                return False
        
        return True
```

### Explanation
1. **Initialization**: We first determine the size of the matrix `n` and create an expected set of integers from 1 to n.
2. **Row Check**: Iterate through each row of the matrix. Convert each row into a set and compare it with the expected set. If any row does not match, return `False`.
3. **Column Check**: Transpose the matrix using `zip(*matrix)` to convert columns into rows. Check each transposed column in the same way as the rows. If any column does not match, return `False`.
4. **Return Result**: If all rows and columns pass the check, return `True`.

This approach ensures that both rows and columns are checked efficiently using set operations, which simplifies the verification process and ensures correctness. The time complexity is O(n^2) due to iterating through each element in the matrix twice (once for rows and once for columns), which is optimal for this problem.
