=== RAW ANSWER ===

To solve this problem, we need to determine the next greater element for each element in the array `nums1` based on their positions in the array `nums2`. The next greater element of an element `x` in `nums2` is the first element to the right of `x` that is greater than `x`. If no such element exists, the result should be `-1`.

### Approach
1. **Identify the Position**: For each element in `nums1`, find its position in `nums2`.
2. **Search for Next Greater Element**: From the identified position in `nums2`, look to the right to find the first element that is greater than the current element.
3. **Store Results**: If a greater element is found, store it; otherwise, store `-1`.

### Solution Code
```python
class Solution:
    def nextGreaterElement(self, nums1: List[int], nums2: List[int]) -> List[int]:
        result = []
        for num in nums1:
            index_in_nums2 = nums2.index(num)
            next_greater = -1
            for k in range(index_in_nums2 + 1, len(nums2)):
                if nums2[k] > num:
                    next_greater = nums2[k]
                    break
            result.append(next_greater)
        return result
```

### Explanation
1. **Identify Position**: For each element `num` in `nums1`, we use `nums2.index(num)` to find its position in `nums2`. This is efficient given the problem constraints where all elements are unique.
2. **Search for Greater Element**: Starting from the position immediately after the found index in `nums2`, we iterate through the remaining elements. The first element greater than `num` is recorded as the next greater element.
3. **Result Compilation**: If no greater element is found during the iteration, `-1` is appended to the result list. Otherwise, the found element is appended.

This approach ensures that each element in `nums1` is processed in linear time relative to its position in `nums2`, leading to an overall time complexity of O(n*m) where n is the length of `nums1` and m is the length of `nums2`. This is efficient for the problem constraints and ensures correctness by directly addressing each element's position and subsequent elements.
