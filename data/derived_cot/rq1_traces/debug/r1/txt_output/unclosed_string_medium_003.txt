=== RAW ANSWER ===

To solve this problem, we need to generate all possible valid IP addresses from a given string. A valid IP address consists of four segments separated by dots, where each segment is a number between 0 and 255. Additionally, no segment can have leading zeros unless it is exactly "0".

### Approach
The approach to solve this problem involves using backtracking to explore all possible ways to split the string into four valid segments. Here are the key steps:

1. **Backtracking Function**: We define a recursive function that builds potential IP segments and checks their validity.
2. **Base Cases**: 
   - If we reach the end of the string and have exactly four segments, we add the valid IP to the result list.
   - If we exceed the string length or have more than four segments, we backtrack.
3. **Handling Leading Zeros**: If a segment starts with '0', it must be exactly "0". We handle this case separately to avoid invalid segments.
4. **Valid Segment Check**: For segments that do not start with '0', we check all possible substrings of length 1, 2, or 3 to see if they form a valid number less than 256.

### Solution Code
```python
class Solution:
    def restoreIpAddresses(self, s: str) -> List[str]:
        res = []
        cur = []
        
        def backtrack(i):
            if i == len(s) and len(cur) == 4:
                res.append(".".join(cur))
                return
            if len(cur) > 4 or i >= len(s):
                return
            if s[i] == '0':
                cur.append(s[i])
                backtrack(i + 1)
                cur.pop()
                return
            j = 0
            while j < 3 and i + j < len(s):
                num = s[i:i + j + 1]
                if int(num) < 256:
                    cur.append(num)
                    backtrack(i + j + 1)
                    cur.pop()
                j += 1
        
        backtrack(0)
        return res
```

### Explanation
1. **Initialization**: We initialize `res` to store valid IP addresses and `cur` to keep track of the current segments being formed.
2. **Backtracking Function**: The function `backtrack(i)` starts at index `i` of the string and tries to form valid segments.
3. **Base Cases**:
   - If we reach the end of the string with exactly four segments, we join them with dots and add to `res`.
   - If we exceed the string length or have more than four segments, we backtrack.
4. **Handling '0'**: If the current character is '0', we immediately form a segment "0" and proceed to the next character.
5. **Valid Segments**: For non-zero starting characters, we check substrings of length 1, 2, and 3. If the substring forms a valid number (less than 256), we proceed recursively.
6. **Recursive Exploration**: For each valid segment, we recursively explore further segments, backtracking by removing the last segment to try other possibilities.

This approach efficiently explores all valid IP addresses using backtracking, ensuring that all constraints are met for each segment.
