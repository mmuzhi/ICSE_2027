### Approach
1. **Delete both elements from the front**: This requires deleting up to the rightmost of the two elements. The number of deletions needed is the maximum of the positions (1-based index) of the minimum and maximum elements.
2. **Delete both elements from the end**: This requires deleting from the end up to the leftmost of the two elements. The number of deletions needed is the maximum of the distances from the end of the array to the positions of the minimum and maximum elements.
3. **Delete one element from the front and the other from the end**: This involves two possibilities:
   - Delete the minimum element from the front and the maximum element from the end.
   - Delete the maximum element from the front and the minimum element from the end.
   The number of deletions needed is the sum of the steps required for the optimal of these two possibilities.

from typing import List

class Solution:
    def minimumDeletions(self, nums: List[int]) -> int:
        if len(nums) == 0:
            return 0
        min_val = min(nums)
        max_val = max(nums)
