```python
class Solution:
    def restoreIpAddresses(self, s: str) -> List[str]:
        res = []
        cur = []
        def backtrack(i):
            if i == len(s) and len(cur) == 4:
                res.append(".".join(cur))
                return
            if len(cur) > 4 or i > len(s):
                return
            if i < len(s) and s[i] == '0':
                cur.append(s[i])
                backtrack(i + 1)
                cur.pop()
                return
            j = 0
            while j < 3 and i + j < len(s):
                num = s[i:i+j+1]
                if int(num) < 256:
                    cur.append(num)
                    backtrack(i + j + 1)
                    cur.pop()
                j += 1
        backtrack(0)
        return res
```

**Explanation of Fixes:**
1. **Base Case Condition:** Changed `i = len(s)` to `i == len(s)` to correctly check if the end of the string is reached.
2. **Loop Bounds Adjustment:** Modified the loop condition from `j < 4` to `j < 3` to ensure each IP segment is 1 to 3 digits long, preventing invalid 4-digit segments.
3. **Handling Leading Zeros:** Added a check `i < len(s)` before accessing `s[i]` to avoid index errors when the entire string has been processed.
