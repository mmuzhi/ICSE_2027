=== RAW ANSWER ===
To solve this problem, we need to maximize the profit from buying and selling stocks multiple times, where each transaction involves buying one share and selling one share. The key insight is that we can capture all the positive price differences between consecutive days, as each positive difference represents a potential profit from buying at the lower price and selling at the higher price.

### Approach
1. **Initial Check**: If the list of prices has fewer than two elements, return 0 because no transactions can be made.
2. **Iterate Through Prices**: For each day starting from the second day, check if the current price is higher than the previous day's price.
3. **Accumulate Profits**: If the current price is higher, add the difference between the current and previous day's prices to the total profit. This approach effectively captures all possible profits from multiple non-overlapping transactions.

### Solution Code
```python
class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        if len(prices) < 2:
            return 0
        profit = 0
        for i in range(1, len(prices)):
            if prices[i] > prices[i-1]:
                profit += prices[i] - prices[i-1]
        return profit
```

### Explanation
- **Initial Check**: The first condition checks if there are fewer than two days of prices, in which case no transactions are possible, so the profit is zero.
- **Profit Calculation**: The loop starts from the second day (index 1) and compares each day's price with the previous day's price. If the current price is higher, the difference is added to the profit. This method works because it sums all the positive price movements, which corresponds to buying at the start of each upward trend and selling at the peak before any downward movement. This ensures the maximum possible profit from multiple transactions without any overlaps.
