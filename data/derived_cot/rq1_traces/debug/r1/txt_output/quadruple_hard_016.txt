2. **Merge Arrays Correctly**: Use two pointers to traverse both input arrays. Compare elements at the current pointers and append the smaller element to the merged list, incrementing the respective pointer.

### Solution Code
```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        p1 = 0
        p2 = 0
        new = []
        
        while p1 < len(nums1) and p2 < len(nums2):
            if nums1[p1] < nums2[p2]:
                new.append(nums1[p1])
                p1 += 1
            else:
                new.append(nums2[p2])
                p2 += 1
        while p1 < len(nums1):
            new.append(nums1[p1])
            p1 += 1
        while p2 < len(nums2):
            new.append(nums2[p2])
            p2 += 1
        
        if len(new) % 2 == 0:
            index = len(new) // 2
            median = (new[index] + new[index - 1]) / 2
        else:
            median = float(new[len(new) // 2])
        return median
```

### Explanation
1. **Initialization**: The `new` list is initialized to store the merged elements.
2. **Merging Process**: Using two pointers `p1` and `p2`, elements from `nums1` and `nums2` are compared and the smaller element is appended to `new`, with the respective pointer incremented.
3. **Appending Remaining Elements**: After the main loop, any remaining elements from either `nums1` or `nums2` are appended to `new`.
4. **Median Calculation**: The median is calculated based on the length of the merged list. For an even-length list, the average of the two middle elements is taken. For an odd-length list, the middle element is the median.

This approach ensures that the arrays are merged correctly and efficiently, leading to an accurate median calculation.
