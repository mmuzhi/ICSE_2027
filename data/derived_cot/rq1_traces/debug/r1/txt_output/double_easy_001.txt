To solve this problem, we need to determine if an array can be split into three non-empty parts such that each part has the same sum. 

### Approach
1. **Check Divisibility**: First, check if the total sum of the array is divisible by 3. If not, it's impossible to split the array into three parts with equal sums, so we return False.
2. **Calculate Target Sum**: If the total sum is divisible by 3, compute the target sum for each part as one-third of the total sum.
3. **Track Cumulative Sum**: Iterate through the array while tracking the cumulative sum. Look for the first split where the cumulative sum matches the target sum.
4. **Find Second Split**: Continue iterating to find the second split where the cumulative sum reaches twice the target sum. Ensure that this second split is not at the last element of the array to guarantee a non-empty third part.

### Solution Code
```python
from typing import List

class Solution:
    def canThreePartsEqualSum(self, arr: List[int]) -> bool:
        total = sum(arr)
        if total % 3 != 0:
            return False
        each_sum = total // 3
        cum_sum = 0
        first_part = False
        for i in range(len(arr)):
            cum_sum += arr[i]
            if not first_part and cum_sum == each_sum:
                first_part = True
            elif first_part and cum_sum == 2 * each_sum:
                if i != len(arr) - 1:
                    return True
        return False
```

### Explanation
1. **Divisibility Check**: The total sum of the array is checked for divisibility by 3. If not divisible, return False immediately.
2. **Target Sum Calculation**: The target sum for each part is calculated as one-third of the total sum.
3. **First Split Detection**: As we iterate through the array, we track the cumulative sum. When the cumulative sum matches the target sum, we mark the first split.
4. **Second Split Detection**: Continue tracking the cumulative sum. When it reaches twice the target sum, check if this occurs before the last element. If so, return True as we can split the array into three non-empty parts. If the loop completes without finding valid splits, return False.

This approach ensures that we efficiently check for the required splits in linear time, making the solution both optimal and easy to understand.
