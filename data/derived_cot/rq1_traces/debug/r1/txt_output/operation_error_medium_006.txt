To solve this problem, we need to determine which words from the list `words1` are universal with respect to the list `words2`. A word is considered universal if it contains at least as many occurrences of each character as required by any word in `words2`. 

### Approach
1. **Understanding the Requirement**: For a word in `words1` to be universal, it must meet or exceed the maximum count of each character required by any word in `words2`. This means we need to compute the maximum frequency of each character across all words in `words2`.
2. **Compute Maximum Frequencies**: Iterate through each word in `words2` and track the maximum occurrence of each character using a Counter.
3. **Check Universal Words**: For each word in `words1`, check if its character counts meet or exceed the maximum counts derived from `words2`.

### Solution Code
```python
from collections import Counter
from typing import List

class Solution:
    def wordSubsets(self, words1: List[str], words2: List[str]) -> List[str]:
        max_counts = Counter()
        for word in words2:
            cnt = Counter(word)
            for char, count in cnt.items():
                max_counts[char] = max(max_counts[char], count)
        return [w1 for w1 in words1 if Counter(w1) >= max_counts]
```

### Explanation
1. **Compute Maximum Character Counts**: We iterate through each word in `words2` and use a Counter to track the maximum frequency of each character encountered. This ensures that we capture the highest requirement for each character across all words in `words2`.
2. **Check Each Word in words1**: For each word in `words1`, we convert it into a Counter and check if it meets or exceeds the maximum character counts stored in `max_counts`. The `>=` operator for Counters ensures that all character requirements are satisfied.

This approach efficiently combines the character requirements from `words2` and checks each word in `words1` against these requirements, resulting in an optimal solution.
