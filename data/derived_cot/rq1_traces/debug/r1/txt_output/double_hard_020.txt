=== RAW ANSWER ===

To solve this problem, we need to find the median of two sorted arrays. The median is the middle value of a sorted list of numbers. If the list has an even number of elements, the median is the average of the two middle numbers. The approach involves merging the two sorted arrays into a single sorted array and then computing the median from this merged array.

### Approach
1. **Merge the Arrays**: Use two pointers to traverse both input arrays simultaneously. Compare elements at the current positions of both pointers and append the smaller element to the merged array, moving the corresponding pointer forward. Continue this until one of the arrays is fully traversed.
2. **Handle Remaining Elements**: After one array is fully traversed, append the remaining elements of the other array to the merged array.
3. **Compute the Median**: Once the merged array is formed, check its length. If the length is even, the median is the average of the two middle elements. If the length is odd, the median is the middle element.

### Solution Code
```python
from typing import List

class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        p1 = 0
        p2 = 0
        new = []
        while p1 < len(nums1) and p2 < len(nums2):
            if nums1[p1] < nums2[p2]:
                new.append(nums1[p1])
                p1 += 1
            else:
                new.append(nums2[p2])
                p2 += 1
        while p1 < len(nums1):
            new.append(nums1[p1])
            p1 += 1
        while p2 < len(nums2):
            new.append(nums2[p2])
            p2 += 1
        
        if len(new) % 2 == 0:
            index = len(new) // 2
            median = (new[index] + new[index - 1]) / 2
        else:
            median = float(new[len(new) // 2])
        return median
```

### Explanation
1. **Merging Arrays**: The code uses two pointers (`p1` and `p2`) to traverse `nums1` and `nums2`. The elements are compared, and the smaller element is added to the merged array `new`, with the corresponding pointer incremented. This ensures the merged array is sorted.
2. **Appending Remaining Elements**: After one array is exhausted, the remaining elements of the other array are appended to `new` using separate loops, each correctly incrementing their respective pointers.
3. **Median Calculation**: The length of the merged array determines if the median is the middle element (odd length) or the average of the two middle elements (even length).

This approach efficiently merges the two arrays in linear time, O(n + m), where n and m are the lengths of the input arrays, and then computes the median in constant time.
