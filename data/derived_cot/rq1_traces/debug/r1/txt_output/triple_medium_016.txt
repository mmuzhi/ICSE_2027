=== RAW ANSWER ===

To solve this problem, we need to form the largest possible integer by replacing some digits of the given number string based on a provided change array. The replacement should start from the first digit where the change value is higher than the original digit and continue contiguously until a digit cannot be replaced with a higher value.

### Approach
1. **Initialization**: Convert the input number string into a list to allow in-place modifications and create a copy of the original digits to process.
2. **Replacement Logic**: Iterate through each digit of the original number:
   - If we haven't started replacing digits yet (`on` is False), check if the current digit can be replaced with a higher value from the change array. If yes, replace the digit and mark the start of replacement (`on` becomes True).
   - Once replacement has started (`on` is True), continue replacing subsequent digits as long as their change value is higher. If a digit cannot be replaced with a higher value, stop the replacement process.
3. **Result Construction**: Convert the modified list back into a string and return it.

### Solution Code
```python
class Solution:
    def maximumNumber(self, num: str, change: List[int]) -> str:
        original = list(num)
        num_list = list(num)
        on = False
        for i, ch in enumerate(original):
            x = int(ch)
            if not on:
                if change[x] > x:
                    num_list[i] = str(change[x])
                    on = True
            else:
                if change[x] > x:
                    num_list[i] = str(change[x])
                else:
                    break
        return "".join(num_list)
```

### Explanation
- **Initialization**: The input number string is converted into a list (`num_list`) for modification, and a copy of the original digits (`original`) is maintained to ensure we process the original values.
- **Replacement Logic**: 
  - The loop iterates through each digit of the original number. For each digit, we check if it can be replaced with a higher value from the change array.
  - If replacement hasn't started (`on` is False), we check if the current digit's change value is higher. If so, we replace the digit and start the replacement process.
  - Once replacement starts (`on` is True), we continue replacing subsequent digits until we encounter a digit that cannot be replaced with a higher value, at which point we break out of the loop.
- **Result Construction**: The modified list is joined back into a string and returned as the result.

This approach ensures that we replace digits contiguously starting from the first valid replacement and stop when no further valid replacements can be made, thus forming the largest possible integer.
