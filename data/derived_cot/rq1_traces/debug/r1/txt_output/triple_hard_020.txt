=== RAW ANSWER ===

To solve this problem, we need to find the median of two sorted arrays. The approach involves merging the two sorted arrays into a single sorted array and then computing the median of the merged array. 

### Approach
1. **Merge the Arrays**: Use two pointers to traverse both input arrays simultaneously, comparing elements at each pointer and appending the smaller element to a new list. This process continues until one of the arrays is fully traversed. Then, append the remaining elements of the other array to the new list.
2. **Compute the Median**: Once the merged array is formed, check its length. If the length is even, the median is the average of the two middle elements. If the length is odd, the median is the middle element.

### Solution Code
```python
from typing import List

class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        p1 = 0
        p2 = 0
        new = []
        
        while p1 < len(nums1) and p2 < len(nums2):
            if nums1[p1] < nums2[p2]:
                new.append(nums1[p1])
                p1 += 1
            else:
                new.append(nums2[p2])
                p2 += 1
        while p1 < len(nums1):
            new.append(nums1[p1])
            p1 += 1
        while p2 < len(nums2):
            new.append(nums2[p2])
            p2 += 1
        
        if len(new) % 2 == 0:
            index = len(new) // 2
            median = (new[index] + new[index - 1]) / 2
        else:
            median = float(new[len(new) // 2])
        
        return median
```

### Explanation
1. **Initialization**: Two pointers `p1` and `p2` are initialized to 0 to traverse `nums1` and `nums2` respectively. An empty list `new` is initialized to store the merged array.
2. **Merging Process**: 
   - While both pointers are within their respective array bounds, compare elements at `p1` and `p2`, append the smaller element to `new`, and increment the corresponding pointer.
   - After one array is fully traversed, append the remaining elements of the other array to `new`.
3. **Median Calculation**: 
   - If the length of `new` is even, the median is the average of the two middle elements.
   - If the length is odd, the median is the middle element.

This approach ensures that the arrays are merged in linear time, making the overall time complexity O(m + n), where m and n are the lengths of the input arrays. The space complexity is O(m + n) due to the merged array.
