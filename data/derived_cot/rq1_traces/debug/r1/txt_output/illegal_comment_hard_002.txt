=== RAW ANSWER ===
To solve this problem, we need to find the longest valid substring of a given string `word` such that it does not contain any of the forbidden substrings. The solution involves using a trie data structure to efficiently check for forbidden substrings and a sliding window approach to determine the longest valid substring.

### Approach
1. **Trie Construction**: Construct a trie from the list of forbidden words. Each node in the trie represents a character, and the end of a forbidden word is marked with a special key (e.g., "end").
2. **Checking Forbidden Substrings**: For any substring, we traverse the trie to check if it starts with any forbidden word. If a forbidden word is found as a prefix of the substring, we return its length; otherwise, we return `False`.
3. **Sliding Window**: We use a sliding window approach starting from the end of the string and move backwards. For each starting index `i`, we check the substring from `i` to `j` (where `j` is initially the end of the string). If a forbidden substring is found starting at `i`, we adjust `j` to exclude the forbidden part. The maximum valid substring length encountered during this process is recorded.

### Solution Code
```python
from typing import List

class Solution:
    def longestValidSubstring(self, word: str, forbidden: List[str]) -> int:
        trie = {}
        for f in forbidden:
            t = trie
            for c in f:
                if c not in t:
                    t[c] = {}
                t = t[c]
            t["end"] = True
        
        def isForbidden(s):
            t = trie
            counter = 0
            for c in s:
                if c not in t:
                    break
                t = t[c]
                counter += 1
                if "end" in t:
                    return counter
            return False
        
        res = 0
        j = len(word)
        for i in range(len(word) - 1, -1, -1):
            truc = isForbidden(word[i:j])
            if truc:
                j = i + truc - 1
            res = max(res, j - i)
        return res
```

### Explanation
1. **Trie Construction**: The trie is built by iterating over each forbidden word. For each character in the word, we traverse the trie, creating new nodes as needed. The end of a word is marked with an "end" key in the trie node.
2. **Forbidden Check**: The `isForbidden` function checks if a given string starts with any forbidden word by traversing the trie. If a forbidden word is found (indicated by an "end" key), it returns the length of that word; otherwise, it returns `False`.
3. **Sliding Window**: The main loop processes the string from right to left. For each starting index `i`, the substring from `i` to `j` is checked. If a forbidden word is found at the start of this substring, `j` is adjusted to `i + length - 1` to exclude the forbidden part. The maximum valid substring length encountered during this process is stored in `res` and returned.

This approach efficiently checks for forbidden substrings using the trie and dynamically adjusts the sliding window to find the longest valid substring. The complexity is linear with respect to the input size, making it optimal for large inputs.
