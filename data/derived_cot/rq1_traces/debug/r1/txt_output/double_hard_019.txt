### Solution Code
```python
class Solution:
    def numPermsDISequence(self, s: str) -> int:
        from collections import defaultdict
        mem = defaultdict(int)
        mod = 10**9 + 7
        
        def dfs(i, val):
            if i == len(s):
                return 1
            if (i, val) in mem:
                return mem[(i, val)]
            p = 0
            if s[i] == "D":
                for ind in range(val):
                    p += dfs(i + 1, ind)
                    p %= mod
            else:
                for ind in range(val + 1, len(s) + 1):
                    p += dfs(i + 1, ind)
                    p %= mod
            mem[(i, val)] = p % mod
            return mem[(i, val)]
        
        total = 0
        for start in range(len(s) + 1):
            total += dfs(0, start)
            total %= mod
        return total
```

### Explanation
1. **Memoization (mem)**: A dictionary to store already computed results for each state `(i, val)` to avoid redundant calculations.
2. **DFS Function**: The recursive function `dfs(i, val)` computes the number of valid permutations starting from index `i` with the current value `val`.
3. **Base Case**: When `i` reaches the length of the sequence `s`, there's exactly one valid permutation (return 1).
4. **Loop Adjustments**: 
   - For "D", iterate over possible values less than `val`.
   - For "I", iterate over possible values greater than `val` up to the maximum value allowed (length of the sequence).
5. **Initial Call**: Sum the results of all possible starting values for the first element to ensure all valid permutations are considered.

This approach efficiently computes the result using dynamic programming with memoization, ensuring correctness and optimal performance.
