=== RAW ANSWER ===

To solve this problem, we need to find the median of two sorted arrays. The median is the middle value of a sorted list of numbers. If the list has an even number of elements, the median is the average of the two middle numbers. The approach involves merging the two sorted arrays into a single sorted array and then computing the median from this merged array.

### Approach
1. **Merge the Arrays**: Use a two-pointer technique to merge the two sorted arrays into one sorted array. This is similar to the merge process in merge sort.
2. **Handle Remaining Elements**: After the initial merge, append any remaining elements from either of the two arrays.
3. **Compute the Median**: Check the length of the merged array. If the length is even, the median is the average of the two middle elements. If the length is odd, the median is the middle element.

### Solution Code
```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        p1 = 0
        p2 = 0
        new = []
        
        while p1 < len(nums1) and p2 < len(nums2):
            if nums1[p1] < nums2[p2]:
                new.append(nums1[p1])
                p1 += 1
            else:
                new.append(nums2[p2])
                p2 += 1
        
        while p1 < len(nums1):
            new.append(nums1[p1])
            p1 += 1
        
        while p2 < len(nums2):
            new.append(nums2[p2])
            p2 += 1
        
        if len(new) % 2 == 0:
            index = len(new) // 2
            median = (new[index] + new[index - 1]) / 2
        else:
            median = float(new[len(new) // 2])
        
        return median
```

### Explanation
1. **Initialization**: We initialize two pointers `p1` and `p2` to traverse `nums1` and `nums2`, respectively. We also initialize an empty list `new` to store the merged result.
2. **Merging Process**: Using a while loop, we compare elements at the current positions of `p1` and `p2`, appending the smaller element to `new` and moving the corresponding pointer forward.
3. **Appending Remaining Elements**: After the initial merge, any remaining elements in `nums1` or `nums2` are appended to `new` using separate while loops.
4. **Median Calculation**: The length of the merged array determines whether the median is the middle element (for odd length) or the average of the two middle elements (for even length).

This approach efficiently merges the two arrays in linear time, O(n + m), where n and m are the lengths of the input arrays, and then computes the median in constant time.
