=== RAW ANSWER ===

To solve this problem, we need to perform a level order traversal of a binary tree from the bottom up. The goal is to return the nodes' values level by level, starting from the deepest level and ending at the root level.

### Approach
1. **Check for an empty tree**: If the root is `None`, return an empty list immediately.
2. **Use a queue for BFS traversal**: A queue helps in processing nodes level by level. We start by adding the root node to the queue.
3. **Process each level**: For each level, determine the number of nodes (level size) and process each node in this level. Collect their values and enqueue their children (left first, then right) if they exist.
4. **Collect levels in reverse order**: After processing all levels, reverse the list of levels to get the bottom-up order.

### Solution Code
```python
from collections import deque
from typing import Optional, List

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right

class Solution:
    def levelOrderBottom(self, root: Optional[TreeNode]) -> List[List[int]]:
        if not root:
            return []
        queue = deque([root])
        lst = []
        while queue:
            level_size = len(queue)
            current_level = []
            for _ in range(level_size):
                node = queue.popleft()
                current_level.append(node.val)
                if node.left:
                    queue.append(node.left)
                if node.right:
                    queue.append(node.right)
            lst.append(current_level)
        return lst[::-1]
```

### Explanation
1. **Initial Check**: The function first checks if the root is `None` and returns an empty list if true.
2. **Queue Initialization**: The root node is added to a deque to start the BFS traversal.
3. **Level Processing**: For each level, the number of nodes (level size) is determined. Each node in this level is processed by dequeuing, collecting its value, and enqueuing its children (left first, then right) if they exist.
4. **Collect and Reverse Levels**: After processing all levels, the collected levels are reversed to achieve the bottom-up order required by the problem.

This approach ensures that each level is processed correctly and efficiently, using BFS to traverse the tree level by level and then reversing the result to meet the problem's requirements.
