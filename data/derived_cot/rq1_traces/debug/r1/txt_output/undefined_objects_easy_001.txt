=== RAW ANSWER ===

To solve this problem, we need to find the complement of a given positive integer. The complement of a number is obtained by flipping each bit of its binary representation. For example, the complement of 5 (binary 101) is 2 (binary 010).

### Approach
1. **Binary Conversion and Flipping**: Convert the given number to its binary form and flip each bit. This can be done by iterating through each bit of the number and appending the flipped bit to a string.
2. **Reversing the Result**: Since we process the bits from the least significant to the most significant, we need to reverse the resulting string to get the correct binary representation of the complement.
3. **Conversion to Integer**: Convert the reversed binary string back to an integer to get the final result.

### Solution Code
```python
class Solution:
    def findComplement(self, num: int) -> int:
        com = ''
        while num > 0:
            if num % 2 == 1:
                com += '0'
            else:
                com += '1'
            num = num // 2
        return int(com[::-1], 2) if com else 0
```

### Explanation
1. **Initialization**: We start with an empty string `com` to accumulate the flipped bits.
2. **Loop Through Bits**: Using a while loop, we process each bit of the number by checking the remainder when divided by 2. If the remainder is 1 (bit is set), we append '0' to `com`; otherwise, we append '1'.
3. **Update Number**: After processing each bit, we update the number by integer division by 2 to move to the next bit.
4. **Reverse and Convert**: After processing all bits, we reverse the string `com` to correct the order and convert it back to an integer using base 2. If the input number is 0, the result is 0 as well.

This approach efficiently processes each bit of the number in linear time, resulting in a time complexity of O(log n), where n is the given number. The space complexity is O(log n) due to the storage of the binary string.
