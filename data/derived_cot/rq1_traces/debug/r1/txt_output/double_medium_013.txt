```python
class Solution:
    def findKthBit(self, N, K, R=True):
        if K == 1:
            return '0' if R else '1'
        mid = 1 << (N - 1)
        if K < mid:
            return self.findKthBit(N - 1, K, R)
        if K == mid:
            return '1' if R else '0'
        return self.findKthBit(N - 1, 2 * mid - K, not R)
```

### Explanation
1. **Base Case**: When K is 1, the bit is '0' if the current segment is not inverted (R=True) and '1' if it is inverted (R=False).
2. **Mid Position**: The middle bit of the string for any N is always '1'. If K is at this position, return '1' or '0' based on the inversion flag R.
3. **First Half Handling**: If K is in the first half, recursively process the same K in the previous string (N-1) without changing the inversion flag.
4. **Second Half Handling**: If K is in the second half, recursively process the mirrored position (2*mid - K) in the previous string (N-1) and toggle the inversion flag (not R) to account for the inversion in the second half.

This approach efficiently narrows down the problem using recursion and mirroring positions, avoiding the need to construct the entire string explicitly. The time complexity is O(N) as each recursive call reduces N by 1.
