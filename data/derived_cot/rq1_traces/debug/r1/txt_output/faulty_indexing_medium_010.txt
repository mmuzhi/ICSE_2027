=== RAW ANSWER ===
To solve this problem, we need to construct a quad tree from a given n x n binary grid. The quad tree is constructed by recursively dividing the grid into four quadrants until each quadrant contains a single cell (a leaf node) or can be merged into a single leaf node if all cells in the quadrant have the same value.

### Approach
1. **Base Case Handling**: If the current segment of the grid is a single cell (1x1), create a leaf node with the value of that cell.
2. **Recursive Division**: For larger segments, divide the grid into four quadrants (top-left, top-right, bottom-left, bottom-right) and recursively build the quad tree for each quadrant.
3. **Merging Quadrants**: After building the four subtrees, check if all four subtrees are leaf nodes and have the same value. If so, merge them into a single leaf node. Otherwise, create an internal node with the four subtrees as children.

### Solution Code
```python
class Solution:
    def construct(self, grid: List[List[int]]) -> 'Node':
        n = len(grid)
        if n == 0:
            return None
        return self.buildTree(grid, 0, n, 0, n, n)
        
    def buildTree(self, grid, rs, re, cs, ce, level):
        if level == 1:
            return Node(grid[rs][cs], True, None, None, None, None)
        
        next_level = level // 2
        mid_row = rs + next_level
        mid_col = cs + next_level
        tl = self.buildTree(grid, rs, mid_row, cs, mid_col, next_level)
        tr = self.buildTree(grid, rs, mid_row, mid_col, ce, next_level)
        bl = self.buildTree(grid, mid_row, re, cs, mid_col, next_level)
        br = self.buildTree(grid, mid_row, re, mid_col, ce, next_level)
        
        if tl.isLeaf and tr.isLeaf and bl.isLeaf and br.isLeaf and tl.val == tr.val == bl.val == br.val:
            return Node(tl.val, True, None, None, None, None)
        else:
            return Node(tl.val, False, tl, tr, bl, br)
```

1. **Initial Check**: The `construct` method first checks if the grid is empty (n = 0) and returns `None` in such cases.
2. **Base Case**: In the `buildTree` method, when the segment size is 1x1 (level = 1), it creates a leaf node with the value of the single cell at `(rs, cs)`.
3. **Recursive Division**: For larger segments, the method calculates the midpoint for rows (`mid_row`) and columns (`mid_col`). It then recursively builds the quad tree for each of the four quadrants defined by these midpoints.
4. **Merging Quadrants**: After building the four subtrees, it checks if all are leaf nodes with the same value.
