=== RAW ANSWER ===
To solve this problem, we need to determine if the number of occurrences of each value in an array is unique. The solution involves counting the occurrences of each element in the array and then checking if all the counts are distinct.

### Approach
1. **Count Occurrences**: Use the `Counter` class from the `collections` module to count the occurrences of each element in the input array.
2. **Extract Counts**: Retrieve the counts of each element from the counter and store them in a list.
3. **Check Uniqueness**: Compare the length of the list of counts with the length of the set of these counts. If they are the same, it means all counts are unique, so we return `True`. Otherwise, we return `False`.

### Solution Code
```python
from collections import Counter
from typing import List

class Solution:
    def uniqueOccurrences(self, arr: List[int]) -> bool:
        d = Counter(arr)
        l = list(d.values())
        if len(l) == len(set(l)):
            return True
        else:
            return False
```

### Explanation
1. **Counting Occurrences**: The `Counter` class efficiently counts the occurrences of each element in the array. For example, if the input array is `[1,2,2,3,3,3]`, the counter will produce `{1: 1, 2: 2, 3: 3}`.
2. **Extracting Counts**: The values from the counter (the counts) are converted into a list. In the example above, this list would be `[1, 2, 3]`.
3. **Checking for Unique Counts**: By converting the list of counts into a set, we eliminate duplicates. If the length of the set is the same as the length of the list, all counts are unique. In our example, the lengths match (both are 3), so the function returns `True`. If there were duplicate counts, the set would be smaller, and the function would return `False`.

This approach efficiently checks for unique occurrences using the properties of sets and counters, ensuring optimal performance and correctness.
