=== RAW ANSWER ===

To solve this problem, we need to find the closest meeting node between two given nodes in a directed graph represented by an array of edges. The goal is to determine the node that can be reached by both starting nodes such that the maximum distance from each starting node to this meeting node is minimized. If there are multiple such nodes, we return the smallest index.

### Approach
1. **Graph Traversal**: We traverse the graph starting from each of the two given nodes (node1 and node2) using a helper function `cdist`. This function updates the distances from the starting node to all reachable nodes and collects the nodes visited during the traversal.
2. **Distance Tracking**: We maintain a distance array where each node has two entries: one for the distance from node1 and another for the distance from node2.
3. **Determine Meeting Node**: After traversing the graph from both nodes, we check each node that was visited during either traversal. For each node, we compute the maximum of the two distances (from node1 and node2). The node with the smallest maximum distance is our answer. If there are ties, we choose the smallest node index.

### Solution Code
```python
from typing import List
from collections import defaultdict

class Solution:
    def cdist(self, it, distArray, distArrayIndex, edges):
        rdist = 0 
        nodes = []
        while it != -1 and distArray[it][distArrayIndex] > rdist:
            distArray[it][distArrayIndex] = rdist
            nodes.append(it)
            it = edges[it]
            rdist += 1
        return nodes
            
    def closestMeetingNode(self, edges: List[int], node1: int, node2: int) -> int:
        INF = float('inf')
        dist = defaultdict(lambda: [INF, INF])
        mmin, ans = INF, INF
        
        n = self.cdist(node1, dist, 0, edges)
        n += self.cdist(node2, dist, 1, edges)
                
        for k in n:
            current_dist = dist[k]
            m = max(current_dist)
            if m != INF:
                if m < mmin:
                    mmin = m
                    ans = k
                elif m == mmin:
                    ans = min(ans, k)
                    
        return ans if ans != INF else -1
```

### Explanation
1. **Graph Traversal (cdist function)**: This function starts from a given node and follows the edges, updating the distance for each node encountered. It stops when it encounters a node that has already been visited with a shorter distance or when it reaches a node with no outgoing edge (-1).
2. **Distance Tracking**: The `dist` array keeps track of distances from both starting nodes. Each entry in `dist` is a list where the first element is the distance from node1 and the second element is the distance from node2.
3. **Finding the Closest Meeting Node**: After traversing the graph from both nodes, we iterate over all nodes visited during either traversal. For each node, we check if it is reachable from both starting nodes (i.e., both distances are not infinity). We then find the node with the smallest maximum distance. If there are multiple such nodes, we choose the smallest index.

This approach ensures that we efficiently find the closest meeting node by leveraging graph traversal and distance tracking, resulting in an optimal solution.

from typing import List
from collections import defaultdict

class Solution:
    def cdist(self, it, distArray, distArrayIndex, edges):
        rdist = 0 
        nodes = []
        while it != -1 and distArray[it][distArrayIndex] > rdist:
            distArray[it][distArrayIndex] = rdist
            nodes.append(it)
            it = edges[it]
            rdist += 1
        return nodes
            
    def closestMeetingNode(self, edges: List[int], node1: int, node2: int) -> int:
        INF = float('inf')
        dist = defaultdict(lambda: [INF, INF])
        mmin, ans = INF, INF
        
        n = self.cdist(node1, dist, 0, edges)
        n += self.cdist(node2, dist, 1, edges)
                
        for k in n:
            current_dist = dist[k]
            m = max(current_dist)
            if m != INF:
                if m < mmin:
                    mmin = m
                    ans = k
                elif m == mmin:
                    ans = min(ans, k)
                    
        return ans if ans != INF else -1
