=== RAW ANSWER ===

To solve this problem, we need to identify self-dividing numbers between a given range [left, right]. A self-dividing number is a number that is divisible by every digit it contains. Note that a self-dividing number must not contain the digit zero.

### Approach
1. **Check Each Number in Range**: Iterate through each number from left to right (inclusive).
2. **Check Each Digit**: For each number, convert it to a string to examine each digit.
3. **Zero Check**: If any digit is zero, the number cannot be self-dividing.
4. **Divisibility Check**: Ensure the number is divisible by each of its digits.
5. **Collect Valid Numbers**: If a number passes both checks, add it to the result list.

### Solution Code
```python
class Solution:
    def selfDividingNumbers(self, left: int, right: int) -> List[int]:
        res = []
        for i in range(left, right + 1):
            is_self_dividing = True
            for j in str(i):
                if j == '0' or i % int(j) != 0:
                    is_self_dividing = False
                    break
            if is_self_dividing:
                res.append(i)
        return res
```

### Explanation
1. **Range Handling**: The loop runs from `left` to `right + 1` to include the upper bound.
2. **Digit Conversion and Check**: Each number is converted to a string to iterate over its digits. For each digit:
   - If the digit is '0', the number is immediately invalid.
   - If the number is not divisible by the digit, it is invalid.
3. **Result Collection**: Valid numbers are collected in the result list and returned.

This approach efficiently checks each number and its digits, ensuring correctness by handling edge cases like zero digits and proper range inclusion.
