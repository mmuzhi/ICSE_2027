To solve this problem, we need to determine the maximum profit that can be obtained from multiple transactions in the stock market, where each transaction involves buying a stock and selling it at a higher price. The key insight is to accumulate profits from all consecutive price increases.

### Approach
The optimal approach to solve this problem is to sum up all the positive differences between consecutive days. This works because buying at a lower price and selling at a higher price on consecutive days will yield the same total profit as buying at the lowest price and selling at the highest price in a continuous upward trend. For example, if the prices are [1, 2, 3, 4], buying at 1 and selling at 4 gives a profit of 3, which is the same as summing the differences (2-1) + (3-2) + (4-3) = 3.

### Solution Code
```python
from typing import List

class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        profit = 0
        for i in range(1, len(prices)):
            if prices[i] > prices[i-1]:
                profit += prices[i] - prices[i-1]
        return profit
```

### Explanation
1. **Initialization**: We start with a profit of 0.
2. **Iterate through Prices**: We loop through the list of prices starting from the second day (index 1) to the end.
3. **Check for Price Increase**: For each day, we check if the price is higher than the previous day. If it is, we add the difference between the current day's price and the previous day's price to the profit.
4. **Return Total Profit**: After processing all days, the accumulated profit is returned.

This approach ensures that we capture all possible profits from consecutive price increases efficiently with a time complexity of O(n), where n is the number of days, and a space complexity of O(1) as we only use a few variables.
