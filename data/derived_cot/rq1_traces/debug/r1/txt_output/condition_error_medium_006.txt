=== RAW ANSWER ===

To solve this problem, we need to find the bottom-left value of a binary tree. This value is the leftmost node's value in the deepest level of the tree. The approach involves performing a level order traversal (BFS) to track the leftmost node at each level and return the leftmost node of the deepest level.

### Approach
1. **Level Order Traversal (BFS)**: We use a queue to traverse the tree level by level. This ensures that we process each node in the order they appear from left to right at each level.
2. **Track Leftmost Node**: For each level, we track the leftmost node (the first node encountered at that level). This is done by checking if the current node is the first node in the level (i.e., when the index `i` is 0).
3. **Update Result**: As we traverse each level, we update our result to be the leftmost node of the current level. By the end of the traversal, the result will hold the leftmost node of the deepest level.

### Solution Code
```python
from collections import deque
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def findBottomLeftValue(self, root: Optional[TreeNode]) -> int:
        queue = deque([root])
        leftmost_value = root.val
        while queue:
            level_size = len(queue)
            for i in range(level_size):
                node = queue.popleft()
                if i == 0:
                    leftmost_value = node.val
                if node.left:
                    queue.append(node.left)
                if node.right:
                    queue.append(node.right)
        return leftmost_value
```

### Explanation
1. **Initialization**: We start by initializing a queue with the root node. We also initialize `leftmost_value` with the root's value to handle the case where the tree has only one node.
2. **Traversal Loop**: The outer `while` loop processes each level of the tree. The inner `for` loop iterates over all nodes in the current level.
3. **Track Leftmost Node**: For each level, the first node (when `i == 0`) is recorded as the current level's leftmost node.
4. **Update Queue**: For each node processed, its left and right children (if present) are added to the queue, ensuring the next level's nodes are processed in subsequent iterations.
5. **Return Result**: After processing all levels, the `leftmost_value` holds the value of the leftmost node in the deepest level, which is returned as the result.

This approach efficiently traverses the tree in O(n) time complexity, where n is the number of nodes, and uses O(n) space complexity for the queue.
