import collections
from typing import List

class Solution:
    def findSubstring(self, s: str, words: List[str]) -> List[int]:
        if not words or not s:
            return []
        wlen = len(words[0])
        slen = wlen * len(words)
        occ = collections.Counter(words)
        
        # If the entire s length is less than slen, return empty
        if len(s) < slen:
            return []
        
        res = []
        
        # We'll iterate over each starting offset mod wlen
        for k in range(wlen):
            # If the initial window doesn't fit, skip
            if k + slen > len(s):
                continue
                
            # Initialize track with zeros for each word in occ
            track = {word: 0 for word in occ}
            
            # Build the initial window
            start_index = k
            # We are going to take words from k, k+wlen, k+2*wlen, ... until we have len(words) words
            for i in range(start_index, start_index + slen, wlen):
                w = s[i:i+wlen]
                if w in occ:
                    track[w] += 1
            # Check if the initial window matches
            if track == occ:  # Instead of test(), we can directly compare dictionaries
                res.append(k)
            
            # Now slide the window: the next window starts at k + wlen, then k+2*wlen, etc.
            # We'll iterate i from k+wlen to the last possible start (which is len(s)-slen) in steps of wlen
            for i in range(k + wlen, len(s) - slen + 1, wlen):
                # The word to remove: the one that was at the beginning of the previous window
                pw = s[i - wlen: i]
                # The word to add: the one at the end of the new window
                nw = s[i + slen - wlen: i + slen]
                
                if pw in occ:
                    track[pw] -= 1
                if nw in occ:
                    track[nw] += 1
                    
                # Check if the current window matches
                if track == occ:
                    res.append(i)  # because i is the start of the current window
                    
        return res

=== RAW ANSWER ===
To solve this problem, we need to find all starting indices of substrings in a given string `s` that are formed by concatenating all the words from a given list `words` exactly once, without any intervening characters. The solution involves using a sliding window approach combined with a frequency counter to efficiently check for valid substrings.

### Approach
1. **Initial Checks**: Handle edge cases where either the input string `s` or the list `words` is empty, or where the combined length of all words in `words` exceeds the length of `s`.
2. **Frequency Counter**: Use a counter to keep track of the frequency of each word in the list `words`.
3. **Sliding Window**: For each possible starting position (modulo the length of the words), initialize a sliding window of the same length as the combined words. Maintain a tracker dictionary to count occurrences of each word within the current window.
4. **Window Validation**: After building the initial window, check if the tracker matches the frequency counter. If it does, record the starting index.
5. **Slide the Window**: Move the window by the length of one word at a time. Update the tracker by removing the outgoing word (leftmost word of the previous window) and adding the incoming word (rightmost word of the new window). After each update, check if the tracker matches the frequency counter and record the new starting index if it does.
6. **Result Compilation**: Collect all valid starting indices and return them.

### Solution Code
```python
class Solution:
    def findSubstring(self, s: str, words: List[str]) -> List[int]:
        if not words or not s:
            return []
        wlen = len(words[0])
        total_length = wlen * len(words)
        if len(s) < total_length:
            return []
        
        occ = collections.Counter(words)
        res = []
        
        for k in range(wlen):
            if k + total_length > len(s):
                continue
            track = {word: 0 for word in occ}
            
            for i in range(k, k + total_length, wlen):
                word_segment = s[i:i + wlen]
                if word_segment in occ:
                    track[word_segment] += 1
            
            def test():
                for key, val in track.items():
                    if val != occ[key]:
                        return False
                return True
            
            if test():
                res.append(k)
            
            for i in range(k + wlen, len(s) - total_length + 1, wlen):
                prev_word = s[i - wlen:i]
                next_word = s[i + total_length - wlen:i + total_length]
                
                if prev_word in occ:
                    track[prev_word] -= 1
                if next_word in occ:
                    track[next_word] += 1
                
                if test():
                    res.append(i)
        
        return res
```

### Explanation
1. **Initial Checks**: The code first checks for edge cases such as empty input strings or word lists. It also verifies if the combined length of all words exceeds the length of the string `s`, in which case no valid substrings can exist.
2. **Frequency Counter**: The `collections.Counter` is used to create a frequency dictionary (`occ`) of the words in the input list.
3. **Sliding Window Initialization**: For each possible starting offset `k` (from 0 to the length of a word minus one), the code initializes a tracker dictionary (`track`) to zero for each distinct word.
4. **Building Initial Window**: The initial window of length `total_length` (combined length of all words) is processed. Each segment of length `wlen` (length of each word) within this window is checked against the frequency counter. If a segment matches a word in `words`, its count in the tracker is incremented.
5. **Validation**: After processing the initial window, the tracker is compared to the frequency counter. If they match, the starting index `k` is recorded.
6. **Sliding the Window**: The window is then slid by one word length (`wlen`). The outgoing word (leftmost word of the previous window) is decremented in the tracker, and the incoming word (rightmost word of the new window) is incremented. After each slide, the tracker is again compared to the frequency counter, and valid starting indices are recorded.
7. **Result Compilation**: All valid starting indices collected during the process are returned as the result.

This approach efficiently checks all possible valid substrings by leveraging a sliding window and frequency counting, ensuring optimal performance and correctness.
