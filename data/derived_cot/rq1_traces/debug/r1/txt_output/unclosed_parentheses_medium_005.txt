To solve this problem, we need to determine if an array can be partitioned into valid subarrays according to specific conditions. The valid subarrays can be either two consecutive equal elements, three consecutive equal elements, or three consecutive elements in increasing order by 1.

### Approach
The approach uses dynamic programming to keep track of the validity of partitions up to the current element. We maintain a tuple `checks` where each element represents the validity of the partition ending at the previous positions. The tuple is updated iteratively for each element in the array, considering the three valid conditions for forming subarrays.

1. **Initialization**: Start with the initial checks for the first two elements.
2. **Iteration**: For each subsequent element, update the checks based on whether the current element can form a valid subarray with the previous elements.
3. **Conditions Check**: For each element, check if it can form a valid pair or triplet with the previous elements, considering the validity of partitions ending at earlier positions.

```python
from typing import List

class Solution:
    def validPartition(self, nums: List[int]) -> bool:
        if len(nums) < 2:
            return False
        
        checks = (True, False, nums[0] == nums[1])
        
        for curr, prev1, prev2 in zip(nums[2:], nums[1:], nums):
            new_third = (
                (checks[1] and curr == prev1) or
                (checks[0] and curr == prev1 == prev2) or
                (checks[0] and curr == prev1 + 1 == prev2 + 2)
            )
            checks = (checks[1], checks[2], new_third)
        
        return checks[2]
```

### Explanation
1. **Initialization**: The initial checks tuple is set to `(True, False, nums[0] == nums[1])`, where the third element checks if the first two elements are equal.
2. **Iteration**: For each element starting from the third, we check if it can form a valid subarray with the previous elements. The tuple `checks` is updated to shift the previous valid states and compute the new state.
3. **Conditions Check**:
   - **Condition 1**: The current element forms a valid pair with the previous element, and the partition up to two elements before was valid.
   - **Condition 2**: The current element forms a valid triplet of equal elements with the previous two elements, and the partition up to three elements before was valid.
   - **Condition 3**: The current element forms a valid triplet of consecutive elements with the previous two elements, and the partition up to three elements before was valid.

By iteratively updating the checks tuple, we efficiently determine if the entire array can be partitioned into valid subarrays, returning the final result from the last check.

from typing import List

class Solution:
    def validPartition(self, nums: List[int]) -> bool:
        if len(nums) < 2:
            return False
        
        checks = (True, False, nums[0] == nums[1])
        
        for curr, prev1, prev2 in zip(nums[2:], nums[1:], nums):
            new_third = (
                (checks[1] and curr == prev1) or
                (checks[0] and curr == prev1 == prev2) or
                (checks[0] and curr == prev1 + 1 == prev2 + 2)
            )
            checks = (checks[1], checks[2], new_third)
        
        return checks[2]
