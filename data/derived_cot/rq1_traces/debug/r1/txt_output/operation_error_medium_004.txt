1. **Tree Representation**: Represent the cities and roads using an adjacency list to form a tree structure.
2. **DFS Traversal**: Use DFS to traverse the tree starting from the capital city's neighbors. This helps in calculating the number of people traveling from each subtree to the capital.
3. **Fuel Calculation**: For each subtree, calculate the number of cars required based on the number of people and available seats. The fuel cost for each edge in the tree is determined by the number of cars needed to transport people from that subtree to their parent city.

### Solution Code
```python
class Solution:
    def minimumFuelCost(self, roads: List[List[int]], seats: int) -> int:
        adjacencyList = [[] for _ in range(len(roads) + 1)]
        res = 0
        for a, b in roads:
            adjacencyList[a].append(b)
            adjacencyList[b].append(a)
        visited = [False] * (len(roads) + 1)
        visited[0] = True
        
        def dfs(node):
            nonlocal res
            total_people = 1
            for neighbor in adjacencyList[node]:
                if not visited[neighbor]:
                    visited[neighbor] = True
                    total_people += dfs(neighbor)
            if node != 0:
                res += (total_people + seats - 1) // seats
            return total_people
        
        for neighbor in adjacencyList[0]:
            if not visited[neighbor]:
                visited[neighbor] = True
                dfs(neighbor)
        
        return res
```

### Explanation
1. **Tree Representation**: The adjacency list is built to represent each city and its connected cities (roads). This helps in efficiently traversing the tree.
2. **DFS Traversal**: Starting from the capital's neighbors, the DFS function recursively visits each city, marking it as visited to avoid cycles. For each city, it calculates the total number of people traveling from its subtree.
3. **Fuel Calculation**: For each city (except the capital), the number of cars required to transport people to the parent city is calculated using ceiling division. This is done by `(total_people + seats - 1) // seats`, which ensures that any remainder in division results in an additional car. The total fuel cost is accumulated in the result variable `res`.

This approach efficiently computes the minimum fuel cost by leveraging the tree structure and DFS traversal, ensuring that each edge's contribution to the fuel cost is accurately calculated.
