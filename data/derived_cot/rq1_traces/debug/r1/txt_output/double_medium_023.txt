=== RAW ANSWER ===
To solve this problem, we need to construct a quad-tree from a given binary grid. The quad-tree is built recursively by dividing the grid into four equal quadrants until each quadrant contains a single cell or all cells in a quadrant have the same value. Each node in the quad-tree can be either a leaf node (if all cells in its quadrant have the same value) or an internal node (if the cells have different values, in which case it has four children corresponding to the four quadrants).

### Approach
1. **Base Case Handling**: When the current quadrant size is 1x1 (i.e., `level == 1`), we create a leaf node with the value of that single cell.
2. **Recursive Division**: For larger quadrants, we split the current grid into four equal quadrants (top-left, top-right, bottom-left, and bottom-right). Each quadrant is processed recursively to build its subtree.
3. **Merging Quadrants**: After processing all four quadrants, we check if all four resulting nodes are leaves and have the same value. If so, we merge them into a single leaf node. Otherwise, we create an internal node with these four nodes as children.
4. **Tree Construction**: The construction starts from the entire grid and recursively processes each quadrant until the base case is met.

### Solution Code
```python
class Solution:
    def construct(self, grid: List[List[int]]) -> 'Node':
        n = len(grid)
        if n == 0:
            return None
        return self.buildTree(grid, 0, n, 0, n, n)
        
    def buildTree(self, grid, rs, re, cs, ce, level):
        if level == 1:
            return Node(grid[rs][cs], True, None, None, None, None)
        
        next_level = level // 2
        mid_row = rs + next_level
        mid_col = cs + next_level
```

### Explanation
1. **Base Case Handling**: When the quadrant size is 1x1 (`level == 1`), the function creates a leaf node with the value of the single cell at position `(rs, cs)`.
2. **Recursive Division**: For larger quadrants, the function calculates the midpoint for rows (`mid_row`) and columns (`mid_col`). It then recursively processes the four quadrants:
   - **Top-left (tl)**: Rows from `rs` to `mid_row`, columns from `cs` to `mid_col`.
   - **Top-right (tr)**: Rows from `rs` to `mid_row`, columns from `mid_col` to `ce`.
   - **Bottom-left (bl)**: Rows from `mid_row` to `re`, columns from `cs` to `mid_col`.
   - **Bottom-right (br)**: Rows from `mid_row` to `re`, columns from `mid_col` to `ce`.
3. **Merging Quadrants**: After processing all four quadrants, if all four nodes are leaves and have the same value, they are merged into a single leaf node. Otherwise, an internal node is created with these four nodes as children. The value for internal nodes is set to `1` (though any value can be used since it is ignored for non-leaf nodes).
4. **Tree Construction**: The construction starts with the entire grid (from `(0, 0)` to `(n, n)`) and recursively processes each quadrant, building the quad-tree from the bottom up.
