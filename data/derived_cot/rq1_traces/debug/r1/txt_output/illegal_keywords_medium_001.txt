=== RAW ANSWER ===

To solve this problem, we need to determine the number of ways to split an array into subarrays such that each subarray contains exactly one '1'. The solution involves analyzing the segments of zeros between consecutive '1's and calculating the product of possible splits for each segment.

### Approach
1. **Check for Presence of '1'**: If there are no '1's in the array, return 0 as no valid splits are possible.
2. **String Conversion and Trimming**: Convert the array to a string and remove leading and trailing zeros to focus on the segments between '1's.
3. **Split on '1's**: Split the trimmed string on '1's to get segments of zeros between consecutive '1's.
4. **Calculate Possible Splits**: For each segment of zeros between '1's, the number of possible splits is the length of the segment plus one. This is because each zero can be a potential split point.
5. **Product of Splits**: Compute the product of all possible splits for each segment to get the total number of valid splits.

### Solution Code
```python
from functools import reduce
from typing import List

class Solution:
    def numberOfGoodSubarraySplits(self, nums: List[int]) -> int:
        if 1 not in nums:
            return 0
        nums_str = ''.join(map(str, nums)).strip('0')
        parts = nums_str.split('1')
        factors = [1 + len(part) for part in parts]
        return reduce(lambda a, b: a * b, factors) % 1000000007
```

### Explanation
1. **Check for '1'**: The code first checks if there is at least one '1' in the array. If not, it returns 0 immediately.
2. **String Conversion**: The array is converted to a string to facilitate trimming of leading and trailing zeros using `strip('0')`.
3. **Splitting on '1's**: The trimmed string is split on '1's to isolate the segments of zeros between consecutive '1's.
4. **Calculate Factors**: For each segment of zeros, the number of possible splits is calculated as the length of the segment plus one. This is stored in a list of factors.
5. **Product Calculation**: The product of all factors is computed using `reduce` with a lambda function for multiplication, giving the total number of valid splits. The result is taken modulo \(10^9 + 7\) to handle large numbers.

This approach efficiently processes the array and leverages string operations to simplify the analysis of zero segments, ensuring an optimal solution.
