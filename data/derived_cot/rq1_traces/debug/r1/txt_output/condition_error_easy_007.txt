To solve this problem, we need to generate Pascal's triangle up to a specified number of rows. Pascal's triangle is a triangular array where each number is the sum of the two numbers directly above it. The challenge is to correct the given recursive implementation to ensure it generates the correct number of rows starting from the first row [1].

### Approach
The approach involves using recursion to build each row based on the previous row. The key steps are:
1. **Base Case Handling**: If the number of rows (numRows) is 0, return an empty list as there are no rows to generate.
2. **Recursive Generation**: For numRows greater than 0, recursively generate the triangle up to numRows-1.
3. **First Row Handling**: If the result of the recursive call is empty (indicating we are generating the first row), initialize it with the first row [1].
4. **Subsequent Rows**: For each subsequent row, compute each element as the sum of the two elements from the previous row, and append the new row to the result.

### Solution Code
```python
class Solution:
    def generate(self, numRows: int) -> List[List[int]]:
        if numRows == 0:
            return []
        prev = self.generate(numRows - 1)
        if not prev:
            prev = [[1]]
        else:
            fin = prev[-1]
            now = [1]
            for i in range(len(fin) - 1):
                now.append(fin[i] + fin[i + 1])
            now.append(1)
            prev.append(now)
        return prev
```

### Explanation
1. **Base Case**: When `numRows` is 0, the function returns an empty list as there are no rows to generate.
2. **Recursive Call**: The function calls itself with `numRows - 1` to generate the triangle up to the previous row.
3. **First Row Check**: If the result of the recursive call is empty, it initializes the result with the first row [1].
4. **Row Construction**: For each subsequent row, starting from the second row, each element (except the first and last, which are always 1) is computed as the sum of the two elements directly above it from the previous row. This new row is then appended to the result.

This approach ensures that each row is built correctly based on the previous row, and handles the base case appropriately to avoid off-by-one errors. The recursion efficiently builds each row step-by-step, ensuring the correct structure of Pascal's triangle.
