### Approach
1. **Create a Dictionary for List2:** This helps in O(1) look-up time for elements in list2.
2. **Identify Common Elements:** Iterate through list1 and check if each element exists in the dictionary created from list2. If it does, calculate the sum of indices from both lists and store these sums along with the element.
3. **Sort the Results:** Sort the collected results based on the sum of indices. This ensures that the smallest sum comes first.
4. **Collect All Elements with Minimum Sum:** After sorting, the first element has the minimum sum. Collect all subsequent elements that have the same sum as the first element.

### Solution Code
```python
class Solution:
    def findRestaurant(self, list1: List[str], list2: List[str]) -> List[str]:
        d2 = {}
        l = []
        for i in range(len(list2)):
            d2[list2[i]] = i
        for i in range(len(list1)):
            if list1[i] in d2:
                l.append([i + d2[list1[i]], list1[i]])
        l1 = []
        l.sort()
        if not l:
            return []
        l1.append(l[0][1])
        for i in range(1, len(l)):
            if l[i][0] == l[i-1][0]:
                l1.append(l[i][1])
            else:
                break
        return l1
```

### Explanation
1. **Dictionary Creation:** The dictionary `d2` maps each element in list2 to its index, allowing quick look-up.
2. **Common Elements Check:** For each element in list1, if it exists in `d2`, the sum of its indices from both lists is computed and stored in a list `l` along with the element.
3. **Sorting:** The list `l` is sorted based on the sum of indices. This ensures that elements with the smallest sum come first.
4. **Result Collection:** After sorting, the first element's sum is the minimum. All consecutive elements with the same sum are collected into the result list `l1`.

This approach efficiently finds the common elements with the minimum index sum using dictionary look-up and sorting, ensuring correctness and optimal performance.
