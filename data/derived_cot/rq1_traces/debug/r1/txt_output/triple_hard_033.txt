=== RAW ANSWER ===

To solve this problem, we need to find all starting indices in a string `s` where a concatenation of all words from a given list `words` exists as a substring. Each word in the list is of the same length, and the solution must efficiently check all possible starting positions.

### Approach
1. **Understand the Problem Requirements**: We need to find all starting indices in `s` such that the substring starting at that index is a concatenation of all words in `words` in any order. Each word in `words` has the same length.

2. **Sliding Window Technique**: Use a sliding window approach to check all possible starting positions. The window size is the total length of all words combined. For each starting position, maintain a count of words to check if they match the required counts.

3. **Counter for Words**: Use a counter to keep track of the frequency of each word in `words`. This helps in quickly comparing the current window's word counts with the required counts.

4. **Initial Window Setup**: For each possible starting position modulo the word length, build the initial window and check if it matches the required word counts.

5. **Slide the Window**: After setting up the initial window, slide the window one word at a time, adjusting the counts by removing the word that is left behind and adding the new word that enters the window. Check if the current window matches the required counts.

### Solution Code
```python
import collections
from typing import List

class Solution:
    def findSubstring(self, s: str, words: List[str]) -> List[int]:
        if not words:
            return []
        word_length = len(words[0])
        total_words = len(words)
        substring_length = word_length * total_words
        word_count = collections.Counter(words)
        result = []
        
        def is_valid():
            for word, count in word_count.items():
                if current_counts.get(word, 0) != count:
                    return False
            return True
        
        for start_offset in range(word_length):
            current_counts = {word: 0 for word in word_count}
            # Initialize the current window
            for i in range(start_offset, start_offset + substring_length, word_length):
                if i + word_length > len(s):
                    break
                current_word = s[i:i + word_length]
                if current_word in word_count:
                    current_counts[current_word] += 1
            if is_valid():
                result.append(start_offset)
            # Slide the window
            for new_start in range(start_offset + word_length, len(s) - substring_length + 1, word_length):
                # Remove the outgoing word
                outgoing_word = s[new_start - word_length:new_start]
                if outgoing_word in word_count:
                    current_counts[outgoing_word] -= 1
                # Add the incoming word
                incoming_word = s[new_start + substring_length - word_length:new_start + substring_length]
                if incoming_word in word_count:
                    current_counts[incoming_word] += 1
                if is_valid():
                    result.append(new_start)
        
        return result
```

### Explanation
1. **Initialization**: We first check if the input list `words` is empty. If it is, we return an empty list. We then determine the length of each word, the total number of words, and the total length of the concatenated substring.

2. **Counter Setup**: We use a counter to keep track of the frequency of each word in `words`.

3. **Validation Function**: The `is_valid` function checks if the current window's word counts match the required counts from the counter.

4. **Sliding Window Initialization**: For each possible starting offset (modulo the word length), we initialize the current window and populate the counts of words in that window.

5. **Sliding the Window**: After checking the initial window, we slide the window one word at a time. For each new position, we adjust the counts by removing the outgoing word (leftmost word of the previous window) and adding the incoming word (rightmost word of the new window). We then check if the current window is valid and record the starting index if it is.

This approach efficiently checks all possible starting positions using a sliding window technique, ensuring that we only adjust counts incrementally, leading to an optimal solution.
