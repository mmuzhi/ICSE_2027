=== RAW ANSWER ===
To solve this problem, we need to perform a level-order traversal of a binary tree starting from the bottom level and moving upwards. The solution involves traversing the tree level by level, collecting node values for each level, and then reversing the order of the levels to achieve the bottom-up traversal.

### Approach
1. **Problem Analysis**: The task is to traverse the binary tree in a level-order manner but return the levels from the bottom to the top. This means the deepest level should be the first in the result list, and the root level should be the last.
2. **Intuition**: We can use a breadth-first search (BFS) approach to traverse the tree level by level. During the BFS, we collect all node values for each level into a list. Once all levels are collected, we reverse the list of levels to achieve the desired bottom-up order.
3. **Algorithm Selection**: 
   - **BFS Traversal**: We use a queue to manage the nodes during traversal. For each level, we process all nodes currently in the queue (which represent the current level), collect their values, and enqueue their children for the next level.
   - **Reversing Levels**: After collecting all levels in top-down order, we reverse the list of levels to get the bottom-up order.
4. **Optimization**: To efficiently reverse the list of levels, we use Python's built-in `reverse()` method, which reverses the list in place. This avoids the need for a custom reversal function and ensures clarity and efficiency.
5. **Edge Case Handling**: If the tree is empty (root is `None`), we return an empty list immediately.

### Solution Code
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def levelOrderBottom(self, root: Optional[TreeNode]) -> List[List[int]]:
        if not root: 
            return []
        q, ans = [root], []
        while q:
            n, l = len(q), []
            for i in range(n):
                curr = q.pop(0)
                l.append(curr.val)
                if curr.left:
                    q.append(curr.left)
                if curr.right:
                    q.append(curr.right)
            ans.append(l)
        ans.reverse()
        return ans
```

### Explanation
1. **Initialization**: We start by checking if the root is `None`. If so, we return an empty list. Otherwise, we initialize a queue with the root node and an empty list `ans` to store the levels.
2. **BFS Traversal**: While the queue is not empty, we determine the number of nodes (`n`) in the current level. For each node in the current level:
   - Dequeue the node (using `pop(0)`).
   - Append its value to the current level list `l`.
   - Enqueue its left and right children if they exist.
3. **Collect Levels**: After processing all nodes in the current level, we append the list `l` to `ans`.
4. **Reverse Levels**: After processing all levels, we reverse the list `ans` using `ans.reverse()` to change the order from top-down to bottom-up.
5. **Return Result**: The reversed list `ans` is returned as the final result.

This approach efficiently collects the levels in top-down order and then reverses them to achieve the desired bottom-up traversal, ensuring clarity and correctness.
