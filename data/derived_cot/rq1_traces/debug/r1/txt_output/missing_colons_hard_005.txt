To solve this problem, we need to find the longest valid substring of a given word such that it does not contain any forbidden substrings. The solution involves using a sliding window approach combined with a trie data structure to efficiently check for forbidden substrings.

### Approach
1. **Trie Construction**: Build a trie from the list of forbidden words. This allows efficient checking of substrings against the forbidden list.
2. **Sliding Window Technique**: Use two pointers, `left` and `right`, to represent the current window of the word being checked. The `right` pointer expands the window by moving to the right, while the `left` pointer adjusts to ensure no forbidden substrings are included within the window.
3. **Forbidden Substring Check**: For each position of the `right` pointer, check all possible substrings ending at `right` with lengths up to the maximum length of forbidden words. If a forbidden substring is found, adjust the `left` pointer to exclude it from the current window.

### Solution Code
```python
from typing import List

class Solution:
    def longestValidSubstring(self, word: str, forbidden: List[str]) -> int:
        trie = {}
        max_len = 0
        for f in forbidden:
            max_len = max(max_len, len(f))
            t = trie
            for c in f:
                if c not in t:
                    t[c] = {}
                t = t[c]
            t["end"] = True
        
        def is_forbidden(s):
            t = trie
            for c in s:
                if c not in t:
                    return False
                t = t[c]
            return "end" in t
        
        left = 0
        max_length = 0
        for right in range(len(word)):
            # Check all possible substrings ending at 'right' with length up to max_len
            for k in range(0, max_len):
                start = right - k
                if start < 0:
                    break
                substring = word[start:right+1]
                if is_forbidden(substring):
                    left = max(left, start + 1)
            current_length = right - left + 1
            if current_length > max_length:
                max_length = current_length
        return max_length
```

### Explanation
1. **Trie Construction**: The trie is built by iterating over each forbidden word and inserting each character into the trie. The end of a word is marked with a special key "end".
2. **Forbidden Check Function**: The `is_forbidden` function checks if a given substring exists in the trie by traversing the trie character by character. If the traversal reaches an "end" marker after processing all characters of the substring, the substring is forbidden.
3. **Sliding Window**: The main loop iterates over each character in the word using the `right` pointer. For each position of `right`, it checks all possible substrings ending at `right` with lengths up to the maximum forbidden length. If a forbidden substring is found, the `left` pointer is adjusted to exclude it. The maximum length of valid substrings is updated accordingly.

This approach ensures that we efficiently check for forbidden substrings and maintain the longest valid window using the sliding window technique, resulting in an optimal solution.
