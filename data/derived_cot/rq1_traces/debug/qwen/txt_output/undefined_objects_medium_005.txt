To solve this problem, we need to determine the minimum number of steps required to convert string `s` into string `t` by only adding characters. Each step allows adding one character to `s`. The solution involves comparing the frequency of each character in both strings and calculating the difference in counts for characters present in `t` but not sufficiently covered in `s`.

### Approach
1. **Frequency Counting**: Use the `Counter` class from the `collections` module to count the occurrences of each character in both strings `s` and `t`.
2. **Difference Calculation**: For each character in `t`, if its count in `s` is less than its count in `t`, the difference between these counts is added to the total steps. This difference represents the number of additional characters needed to match the frequency in `t`.
3. **Summing Differences**: Sum all the differences calculated in the previous step to get the total minimum steps required.

### Solution Code
```python
from collections import Counter

class Solution:
    def minSteps(self, s: str, t: str) -> int:
        cnt1 = Counter(s)
        cnt2 = Counter(t)
        sm = 0
        for char in cnt2:
            if cnt2[char] > cnt1.get(char, 0):
                sm += (cnt2[char] - cnt1.get(char, 0))
        return sm
```

from collections import Counter

class Solution:
    def minSteps(self, s: str, t: str) -> int:
        cnt1 = Counter(s)
        cnt2 = Counter(t)
        sm = 0
        for char in cnt2:
            if cnt2[char] > cnt1.get(char, 0):
                sm += (cnt2[char] - cnt1.get(char, 0))
        return sm
