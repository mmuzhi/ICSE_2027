The approach involves using a prefix sum array to enable efficient range sum queries. The prefix sum array is constructed such that each element at index `i` contains the sum of all elements from the start of the original array up to, but not including, the element at index `i` in the original array. from itertools import accumulate

class NumArray:

        return self.nums[right+1] - self.nums[left]
