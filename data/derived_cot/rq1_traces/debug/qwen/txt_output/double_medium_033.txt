=== RAW ANSWER ===

To solve this problem, we need to find the maximum profit achievable by buying and selling a stock at most once. The solution involves iterating through the list of stock prices to keep track of the minimum purchase price and the maximum profit difference encountered during the traversal.

### Approach
1. **Problem Analysis**: The problem requires finding the best time to buy and sell a stock to maximize profit with only one transaction allowed. The solution must efficiently determine the maximum difference between any two days where the buy day occurs before the sell day.
2. **Intuition**: By maintaining the minimum purchase price encountered so far, we can calculate the potential profit for each subsequent day. The maximum profit is updated whenever a higher profit is found.
3. **Algorithm Selection**: A single-pass algorithm is used where we iterate through the list of prices once. For each price, we compare it with the current minimum purchase price and update the maximum profit if selling at the current price yields a better profit.
4. **Complexity Analysis**: The algorithm runs in O(n) time, where n is the number of days, as it processes each price exactly once. The space complexity is O(1) since only a few variables are used.

### Solution Code
```python
class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        if not prices:
            return 0
        min_price = prices[0]
        max_profit = 0
        for i in range(1, len(prices)):
            if prices[i] < min_price:
                min_price = prices[i]
            else:
                max_profit = max(max_profit, prices[i] - min_price)
        return max_profit
```

### Explanation
1. **Initialization**: The function starts by checking if the input list is empty, returning 0 if true. Otherwise, it initializes `min_price` to the first price in the list and `max_profit` to 0.
2. **Iteration**: The loop processes each subsequent price in the list. For each price, it checks if it is lower than the current `min_price`. If so, it updates `min_price`. Otherwise, it calculates the potential profit by subtracting `min_price` from the current price and updates `max_profit` if this profit is higher than the current maximum.
3. **Result**: After processing all prices, the function returns `max_profit`, which holds the highest profit achievable with one transaction. This approach efficiently finds the solution in linear time with constant space complexity.
