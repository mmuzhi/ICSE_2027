To solve this problem, we need to find the largest number that can be formed by rearranging the digits of a given integer. The solution involves using two heaps to efficiently retrieve the largest available digits during the rearrangement process.

### Approach
1. **Digit Separation**: Separate the digits of the input number into even and odd lists.
2. **Heap Initialization**: Convert these lists into max-heaps by storing negative values, allowing the use of Python's min-heap functions to simulate max-heaps.
3. **Digit Selection**: For each position in the resulting number, compare the largest available digits from both heaps (even and odd) and select the larger one. This ensures the largest possible digit is placed first, leading to the largest number overall.
4. **Result Construction**: Construct the result by combining the selected digits in the order they are chosen, converting them back to positive values, and then forming the final number by joining these digits.

                    result.append(-heapq.heappop(even))
                else:
                    result.append(-heapq.heappop(odd))
        
        return int(''.join(str(x) for x in result))

### Explanation
1. **Digit Separation**: The input number is converted into a list of its digits. These digits are then split into two lists: one containing even digits and the other containing odd digits.
2. **Heap Initialization**: The even and odd digit lists are converted into max-heaps by storing their negative values. This allows the use of Python's `heapq` module, which only supports min-heaps, to efficiently retrieve the largest digits.
3. **Digit Selection**: For each digit position in the result, the largest digit from either the even or odd heap is selected. The selection is based on comparing the top elements of both heaps (converted back to positive values) to ensure the largest available digit is chosen.
4. **Result Construction**: The selected digits are collected into a list, which is then joined into a string and converted back to an integer to form the largest possible number from the rearranged digits.

This approach efficiently leverages heap data structures to maximize the digit selection process, ensuring the largest number is formed with optimal performance.
