=== RAW ANSWER ===

To solve this problem, we need to find the maximum depth of a binary tree. The maximum depth is defined as the number of nodes along the longest path from the root node down to the farthest leaf node. 

### Approach
The approach involves using a Depth-First Search (DFS) approach to traverse the tree. We use a stack to keep track of nodes and their respective depths. The algorithm starts by initializing the stack with the root node and a depth of 1. For each node popped from the stack, we update the maximum depth encountered so far. We then push the left and right children of the current node onto the stack with an incremented depth. This ensures that we explore all paths from the root to the leaves, keeping track of the deepest path encountered.

### Solution Code
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def maxDepth(self, root: Optional[TreeNode]) -> int:
        if not root:
            return 0
        
        stack = [[root, 1]]
        res = 0
        
        while stack:
            node, depth = stack.pop()
            if node:
                res = max(res, depth)
                if node.left:
                    stack.append([node.left, depth + 1])
                if node.right:
                    stack.append([node.right, depth + 1])
                    
        return res
```

### Explanation
1. **Initialization**: The stack is initialized with the root node and a depth of 1. The variable `res` is initialized to 0 to keep track of the maximum depth encountered.
2. **Stack Processing**: The while loop processes each node in the stack. For each node, if it is not None, the current depth is compared with `res` to update the maximum depth.
3. **Child Nodes Processing**: The left and right children of the current node are pushed onto the stack with an incremented depth if they exist. This ensures that all paths from the root to the leaves are explored.
4. **Result**: After processing all nodes, the maximum depth found (`res`) is returned.

This approach efficiently explores all paths in the tree using DFS, ensuring that the deepest path is correctly identified. The use of a stack allows for systematic traversal of each node, updating the maximum depth as needed.
