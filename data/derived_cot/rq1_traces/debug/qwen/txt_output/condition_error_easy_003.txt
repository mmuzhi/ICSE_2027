### Solution Code
```python
class Solution:
    def search(self, nums: List[int], target: int) -> int:
        left, right = 0, len(nums) - 1
        
        while left <= right:
            mid = (left + right) // 2
            if nums[mid] == target:
                return mid
            elif nums[mid] < target:
                left = mid + 1
            else:
                right = mid - 1
        
        return -1
```

### Explanation
1. **Initialization**: The binary search starts with `left` set to 0 and `right` set to `len(nums)-1`, ensuring the initial bounds are valid.
2. **Mid Calculation**: The middle index `mid` is calculated as the average of `left` and `right`.
3. **Comparison**:
   - If `nums[mid]` equals the target, the index `mid` is returned immediately.
   - If `nums[mid]` is less than the target, the left pointer is moved to `mid + 1` to search the right half.
   - If `nums[mid]` is greater than the target, the right pointer is moved to `mid - 1` to search the left half.
4. **Termination**: If the loop exits without finding the target, the function returns `-1`, indicating the target is not present in the array.

This approach ensures efficient O(log n) time complexity and correct handling of all edge cases, including the absence of the target value in the array.
