=== RAW ANSWER ===

To solve this problem, we need to determine the minimum number of operations required to turn all elements of an array into zeros using a specific operation. The operation allows choosing an index and a number, then setting the element at that index to the result of a bitwise XOR between the current element and the chosen number. 

### Approach
The key observation here is that each non-zero element in the array requires exactly one operation to turn it into zero. This is because for any non-zero element `num`, we can choose `x = num` such that `num ^ num = 0`. Therefore, the minimum number of operations needed is simply the count of non-zero elements in the array.

### Solution Code
```python
class Solution:
    def minimumOperations(self, nums: List[int]) -> int:
        count = 0
        for num in nums:
            if num != 0:
                count += 1
        return count
```

### Explanation
1. **Initialization**: We start by initializing a counter `count` to zero. This counter will keep track of the number of non-zero elements in the array.
2. **Iteration**: For each element in the array, we check if it is non-zero. If it is, we increment the counter.
3. **Result**: The value of the counter after processing all elements gives the minimum number of operations required, as each non-zero element needs exactly one operation to be turned into zero.

This approach efficiently counts the non-zero elements, ensuring optimal performance with a time complexity of O(n), where n is the number of elements in the array. The space complexity is O(1) as we only use a single counter variable.
