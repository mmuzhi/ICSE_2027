=== RAW ANSWER ===

To solve this problem, we need to determine all ancestors of each node in a directed acyclic graph (DAG). An ancestor of a node is any node that has a path leading to it. The solution involves using a modified Kahn's algorithm for topological sorting to efficiently compute the ancestors for each node.

### Approach
1. **Graph Construction**: Build an adjacency list representation of the graph and initialize an indegree array to keep track of the number of incoming edges for each node.
2. **Topological Sorting**: Use a queue to process nodes with no incoming edges (indegree zero). These nodes are the starting points in the topological sort.
3. **Ancestor Propagation**: For each node processed, update the ancestors of its neighbors by including the current node and all its ancestors. Decrement the indegree of the neighbors and add them to the queue if their indegree becomes zero.
4. **Result Compilation**: After processing all nodes, convert each node's ancestor set into a sorted list and return the result.

### Solution Code
```python
from collections import deque

class Solution:
    def getAncestors(self, n: int, edges: List[List[int]]) -> List[List[int]]:
        graph = [[] for _ in range(n)]
        indegrees = [0] * n
        
        for edge in edges:
            src, dest = edge
            graph[src].append(dest)
            indegrees[dest] += 1
        
        queue = deque()
        ans = [set() for _ in range(n)]
        
        for i in range(n):
            if indegrees[i] == 0:
                queue.append(i)
                
        while queue:
            cur = queue.popleft()
            for neighbor in graph[cur]:
                ans[neighbor].add(cur)
                ans[neighbor].update(ans[cur])
                indegrees[neighbor] -= 1
                if indegrees[neighbor] == 0:
                    queue.append(neighbor)
                    
        return [sorted(list(s)) for s in ans]
```

### Explanation
1. **Graph Construction**: The graph is built using an adjacency list where each node points to its neighbors. The indegree array is initialized to zero for all nodes and updated based on the edges provided.
2. **Topological Sorting**: Nodes with no incoming edges (indegree zero) are added to the queue first. These nodes are processed first as they have no ancestors.
3. **Ancestor Propagation**: For each node processed, its neighbors' ancestor sets are updated by adding the current node and all its ancestors. The indegree of each neighbor is decremented, and if it reaches zero, the neighbor is added to the queue for further processing.
4. **Result Compilation**: After processing all nodes, each node's ancestor set is converted into a sorted list. The result is a list of lists where each sublist contains the ancestors of the corresponding node in sorted order.

This approach efficiently computes all ancestors for each node in linear time relative to the number of edges and nodes, leveraging topological sorting to ensure correct propagation of ancestor information.
