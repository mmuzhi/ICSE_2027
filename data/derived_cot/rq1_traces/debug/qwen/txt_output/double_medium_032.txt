To solve this problem, we need to determine the number of plates (represented by '*' characters) between candles (represented by '|' characters) within a specified substring of a given string. The solution involves efficiently processing multiple queries on the string to find the number of plates between the first and last candles within each query's substring.

### Approach
1. **Identify Candle Positions**: First, we collect all the indices of the candles (('|' characters) in the input string. This helps in quickly locating the candles within any substring.
2. **Binary Search for Efficient Lookup**: For each query, we use binary search to find the first candle to the right of the left query boundary and the last candle to the left of the right query boundary. This is done using a helper function that performs a lower bound search (first position where the candle index is greater than or equal to the query boundary).
3. **Calculate Plates Between Candles**: Once the first and last candles within the query boundaries are identified, the number of plates between them is calculated by subtracting the number of candles (including the boundaries) from the total length of the substring between these candles.

### Solution Code
```python
class Solution:
    def platesBetweenCandles(self, s: str, queries: List[List[int]]) -> List[int]:
        candies = [i for i, c in enumerate(s) if c == '|']
        
        def bns(x: int) -> int:
            l, r = 0, len(candies)
            while l < r:
                m = (l + r) // 2
                if candies[m] < x:
                    l = m + 1
                else:
                    r = m
            return l
        
        ans = []
        for a, b in queries:
            l_index = bns(a)
            r_index = bns(b + 1) - 1
            if l_index < r_index:
                ans.append(candies[r_index] - candies[l_index] - (r_index - l_index))
            else:
                ans.append(0)
        return ans
```

### Explanation
1. **Candle Position Identification**: The code starts by collecting all indices where the character is '|' (candle) into a list called `candies`.
2. **Binary Search Function**: The `bns` function performs a binary search to find the first candle index that is greater than or equal to a given value `x`. This helps in efficiently locating the boundaries for each query.
3. **Query Processing**: For each query `[a, b]`, the code finds the first candle to the right of `a` (using `bns(a)`) and the last candle to the left of `b` (using `bns(b + 1) - 1`). If there are at least two candles within the query boundaries, the number of plates is calculated by subtracting the number of candles (found as `r_index - l_index + 1`) from the total characters between these candles (found as `candies[r_index] - candies[l_index] + 1`). The result is the difference between these two values, giving the number of plates. If there are fewer than two candles, the result is 0.
4. **Efficiency**: The binary search ensures that each query is processed in logarithmic time relative to the number of candles, making the solution efficient even for large input sizes. The overall complexity is O(n + q log n), where n is the length of the string and q is the number of queries.
