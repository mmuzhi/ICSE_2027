=== RAW ANSWER ===

To solve this problem, we need to find the median of two sorted arrays efficiently. The median is the middle value in an ordered list of numbers. If the combined length of the two arrays is even, the median is the average of the two middle numbers; otherwise, it is the middle number itself.

### Approach
1. **Merge the Arrays**: Combine the two sorted arrays into a single sorted array. This can be done efficiently by using two pointers to traverse both arrays and selecting the smaller element at each step.
2. **Calculate the Median**: Once the arrays are merged into a single sorted array, determine the median based on the length of the merged array. If the length is odd, the median is the middle element. If the length is even, the median is the average of the two middle elements.

### Solution Code
```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        merged = []
        p1 = 0
        p2 = 0
        
        while p1 < len(nums1) and p2 < len(nums2):
            if nums1[p1] < nums2[p2]:
                merged.append(nums1[p1])
                p1 += 1
            else:
                merged.append(nums2[p2])
                p2 += 1
                
        while p1 < len(nums1):
            merged.append(nums1[p1])
            p1 += 1
            
        while p2 < len(nums2):
            merged.append(nums2[p2])
            p2 += 1
            
        n = len(merged)
        if n % 2 == 1:
            return float(merged[n//2])
        else:
            return (merged[n//2 - 1] + merged[n//2]) / 2.0
```

### Explanation
1. **Merging Arrays**: The code uses two pointers, `p1` and `p2`, to traverse `nums1` and `nums2` respectively. At each step, it compares the current elements of both arrays and appends the smaller element to the merged array, then moves the corresponding pointer forward. This ensures the merged array is sorted.
2. **Handling Remaining Elements**: After one of the arrays is exhausted, the remaining elements of the other array are appended to the merged array since they are already in sorted order.
3. **Finding the Median**: The median is calculated based on the length of the merged array. If the length is odd, the middle element is the median. If even, the average of the two middle elements is taken.

This approach efficiently combines the two sorted arrays and computes the median in linear time, O(n + m), where n and m are the lengths of the two input arrays.
