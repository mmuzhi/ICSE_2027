=== RAW ANSWER ===

To solve this problem, we need to compute the alternating digit sum of a given positive integer. The alternating digit sum is defined as the sum of the digits at even indices (0-indexed) minus the sum of the digits at odd indices. The solution involves converting the integer into a string to process each digit individually, then iterating through each digit, adding or subtracting it based on its position, and finally returning the result.

### Approach
1. **Convert the integer to a string**: This allows us to easily access each digit by its index.
2. **Initialize a total sum variable**: This variable will hold the cumulative sum of the digits, alternating between addition and subtraction based on the digit's index.
3. **Iterate through each digit**: For each digit, check if its index is even or odd. If the index is even, add the digit's value to the total; if odd, subtract it.
4. **Return the total sum**: The result is the alternating sum of the digits.

### Solution Code
```python
class Solution:
    def alternateDigitSum(self, n: int) -> int:
        s = str(n)
        total = 0
        for i in range(len(s)):
            if i % 2 == 0:
                total += int(s[i])
            else:
                total -= int(s[i])
        return total
```

### Explanation
1. **String Conversion**: The integer `n` is converted into a string `s` to facilitate digit-by-digit processing.
2. **Initialization**: The variable `total` is initialized to 0 to accumulate the sum of the digits.

class Solution:
    def alternateDigitSum(self, n: int) -> int:
        s = str(n)
        total = 0
        for i in range(len(s)):
            if i % 2 == 0:
                total += int(s[i])
            else:
                total -= int(s[i])
        return total
