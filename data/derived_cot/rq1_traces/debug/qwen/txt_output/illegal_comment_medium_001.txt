2. **Cycle Detection**: We use Depth-First Search (DFS) to detect cycles in the graph. During the DFS traversal, we maintain two sets: one to keep track of all visited nodes (`visited`), and another to keep track of nodes currently in the recursion stack (`rec_stack`). If we encounter a node that is already in the recursion stack, a cycle is detected.
3. **DFS Traversal**: For each course, if it hasn't been visited, we initiate DFS. During DFS, we mark the current node as visited and add it to the recursion stack. For each prerequisite of the current node, if it hasn't been visited, we recursively check it. If it has been visited and is still in the recursion stack, a cycle is detected, and we return false. After processing all prerequisites, we remove the node from the recursion stack.
