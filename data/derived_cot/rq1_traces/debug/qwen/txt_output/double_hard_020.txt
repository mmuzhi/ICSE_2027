=== RAW ANSWER ===

To solve this problem, we need to find the median of two sorted arrays. The median is the middle value in an ordered list of numbers. If the combined length of the two arrays is even, the median is the average of the two middle numbers; if it's odd, the median is the middle number.

### Approach
1. **Merge the Arrays**: Since both input arrays are sorted, we can merge them into a single sorted array by comparing elements from both arrays and appending the smaller element to the result array. This ensures the merged array remains sorted.
2. **Calculate the Median**: After merging, the median can be determined based on the length of the merged array. If the length is even, the median is the average of the two middle elements; if odd, it's the middle element.

### Solution Code
```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        p1 = 0
        p2 = 0
        new = []
        while p1 < len(nums1) and p2 < len(nums2):
            if nums1[p1] < nums2[p2]:
                new.append(nums1[p1])
                p1 += 1
            else:
                new.append(nums2[p2])
                p2 += 1
        while p1 < len(nums1):
            new.append(nums1[p1])
            p1 += 1
        while p2 < len(nums2):
            new.append(nums2[p2])
            p2 += 1
        
        n = len(new)
        if n % 2 == 1:
            return float(new[n//2])
        else:
            return (new[n//2 - 1] + new[n//2]) / 2.0
```

### Explanation
1. **Merging Arrays**: The code uses two pointers, `p1` and `p2`, to traverse `nums1` and `nums2` respectively. The smaller element from either array is appended to the merged array `new`, and the corresponding pointer is incremented.
2. **Handling Remaining Elements**: After one array is exhausted, the remaining elements from the other array are appended to `new` in order.
3. **Finding the Median**: The median is calculated based on the length of `new`. If the length is odd, the median is the middle element. If even, it's the average of the two middle elements. The result is returned as a float to handle both cases appropriately.

This approach efficiently merges the two sorted arrays and computes the median in linear time relative to the combined length of the input arrays, ensuring optimal performance.
