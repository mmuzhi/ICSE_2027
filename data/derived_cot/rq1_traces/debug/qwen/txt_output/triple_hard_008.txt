=== RAW ANSWER ===

To solve this problem, we need to find the median of two sorted arrays. The median is the middle value in an ordered list of numbers. If the total number of elements is even, the median is the average of the two middle numbers.

### Approach
1. **Merge the Arrays**: We will merge the two sorted arrays into a single sorted array. This is done by comparing elements from both arrays and appending the smaller element to the result array until one of the arrays is exhausted. The remaining elements of the other array are then appended to the result.
2. **Calculate the Median**: After merging, if the length of the merged array is odd, the median is the middle element. If the length is even, the median is the average of the two middle elements.

### Solution Code
```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        p1 = 0
        p2 = 0
        new = []
        
        while p1 < len(nums1) and p2 < len(nums2):
            if nums1[p1] < nums2[p2]:
                new.append(nums1[p1])
                p1 += 1
            else:
                new.append(nums2[p2])
                p2 += 1
                
        while p1 < len(nums1):
            new.append(nums1[p1])
            p1 += 1
            
        while p2 < len(nums2):
            new.append(nums2[p2])
            p2 += 1
            
        n = len(new)
        if n % 2 == 0:
            index1 = n // 2 - 1
            index2 = n // 2
            median = (new[index1] + new[index2]) / 2.0
        else:
            median = float(new[n // 2])
            
        return median
```

### Explanation
1. **Merging Arrays**: The code uses two pointers, `p1` and `p2`, to traverse the two input arrays (`nums1` and `nums2`). In each iteration, the smaller of the two current elements is appended to the result array `new`, and the corresponding pointer is incremented. This continues until one of the arrays is fully traversed. The remaining elements of the other array are then appended to `new`.
2. **Finding the Median**: The merged array `new` is now sorted. The median is determined based on the length of `new`. If the length is odd, the median is the middle element. If the length is even, the median is the average of the two middle elements. The result is returned as a float to ensure correct handling in both cases.

This approach efficiently merges the two sorted arrays and calculates the median in linear time, O(n + m), where n and m are the lengths of the two input arrays. The space complexity is O(n + m) to store the merged array.
