=== RAW ANSWER ===

To solve this problem, we need to generate all numbers within a given range [low, high] that consist of consecutive increasing digits. The solution involves using a breadth-first search (BFS) approach to explore all possible numbers formed by appending digits to the current number, ensuring that each subsequent digit is exactly one more than the previous digit.

### Approach
1. **Initialization**: Start with a queue containing all single-digit numbers from 1 to 9.
2. **BFS Traversal**: For each number dequeued, check if it lies within the specified range [low, high]. If it does, add it to the result list.
3. **Digit Extension**: For each number, determine its last digit. If the last digit is less than 9, extend the number by appending the next digit (last digit + 1) and enqueue the new number.
4. **Termination**: The BFS continues until the queue is empty, ensuring all possible sequential digit numbers are considered.

This approach efficiently generates all valid numbers by leveraging BFS to explore numbers of increasing length, ensuring that each number meets the sequential digit criteria.

### Solution Code
```python
from collections import deque

class Solution:
    def sequentialDigits(self, low, high):
        out = []
        queue = deque(range(1, 10))
        while queue:
            elem = queue.popleft()
            if low <= elem <= high:
                out.append(elem)
            last_digit = elem % 10
            if last_digit < 9:
                next_num = elem * 10 + (last_digit + 1)
                queue.append(next_num)
        return out
```

### Explanation
1. **Initialization**: The queue starts with numbers 1 through 9, representing all possible starting points for sequential digit numbers.
2. **BFS Loop**: The loop processes each number in the queue:
   - **Range Check**: If the current number is within the range [low, high], it is added to the result list.
   - **Digit Extension**: The last digit of the current number is calculated. If this digit is less than 9, the next number is formed by appending the next digit (last digit + 1) and added to the queue for further processing.
3. **Result Compilation**: The loop continues until all numbers in the queue are processed, ensuring all valid sequential digit numbers within the range are collected and returned.

This method efficiently explores all possible sequential digit numbers by leveraging BFS, ensuring correctness and completeness in the solution.
