To solve this problem, we need to find all starting indices of substrings in a given string `s` that are concatenations of each word in a list `words` exactly once and consecutively. The solution involves using a sliding window technique combined with a frequency dictionary to efficiently check for valid substrings.

### Approach
1. **Problem Analysis**: The task is to find all substrings in `s` that can be formed by concatenating each word in `words` exactly once without any intervening characters. The words can appear in any order, but each must be used exactly once.
2. **Intuition**: We can use a sliding window approach where the window size is the total length of all words combined (i.e., `len(words[0]) * len(words)`). For each starting position in the string that aligns with the beginning of a word (i.e., positions 0 to `len(words[0])-1`), we check if the substring formed by the window matches the frequency of words in `words`.
3. **Algorithm Selection**: 
   - **Frequency Dictionary**: We first create a frequency dictionary (`occ`) for all words in `words`.
   - **Sliding Window**: For each starting position `k` (0 to `len(words[0])-1`), we initialize a frequency dictionary (`cur`) and populate it with the words in the initial window (from `k` to `k + total_length`).
   - **Validation**: If the initial window's frequency matches `occ`, the starting index `k` is added to the result. Then, we slide the window by one word length (`wlen`), updating the frequency dictionary by removing the first word of the previous window and adding the new word at the end. If the updated frequency matches `occ`, the new starting index is added to the result.
4. **Complexity Analysis**: The algorithm efficiently checks each possible starting position and slides the window in linear time, making it suitable for larger strings. The time complexity is O(n * m), where `n` is the length of `s` and `m` is the total length of all words combined.

import collections
from typing import List

class Solution:
    def findSubstring(self, s: str, words: List[str]) -> List[int]:
