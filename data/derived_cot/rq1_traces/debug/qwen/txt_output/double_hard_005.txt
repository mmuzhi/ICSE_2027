=== RAW ANSWER ===

To solve this problem, we need to find for each element in the array the second element to its right that is greater than it. If no such element exists, we should return -1 for that position. 

### Approach
1. **First Greater Element Identification**: We first identify the first element to the right of each element that is greater than it. This is done using a stack where we traverse the array from left to right, popping elements from the stack that are smaller than the current element and recording their first greater element index.
2. **Next Greater Element Identification**: Next, we identify the next greater element for each element in the array by traversing from right to left. This helps us determine the immediate next element that is greater than the current element.
3. **Second Greater Element Calculation**: For each element, if there exists a first greater element, we then find the next greater element of that first greater element. This next greater element will be the second greater element for the original element, provided it exists.

### Solution Code
```python
class Solution:
    def secondGreaterElement(self, nums: List[int]) -> List[int]:
        n = len(nums)
        ans = [-1] * n
        first_greater = [-1] * n
        stack = []
        for i in range(n):
            while stack and nums[stack[-1]] < nums[i]:
                idx = stack.pop()
                first_greater[idx] = i
            stack.append(i)
        
        next_greater = [-1] * n
        stack = []
        for i in range(n-1, -1, -1):
            while stack and nums[stack[-1]] <= nums[i]:
                stack.pop()
            if stack:
                next_greater[i] = stack[-1]
            else:
                next_greater[i] = -1
            stack.append(i)
        
        for i in range(n):
            if first_greater[i] != -1 and next_greater[first_greater[i]] != -1:
                ans[i] = nums[next_greater[first_greater[i]]]
        return ans
```

1. **First Greater Element Identification**: The first loop processes each element from left to right. For each element, it uses a stack to keep track of indices. Elements in the stack that are smaller than the current element are popped, and their first greater element index is recorded as the current index.
2. **Next Greater Element Identification**: The second loop processes each element from right to left. It uses a stack to find the immediate next greater element for each element. Elements in the stack that are less than or equal to the current element are popped, and the top of the stack is recorded as the next greater element index.
3. **Second Greater Element Calculation**: The third loop checks for each element if there exists a first greater element. If so, it checks if there is a next greater element for that first greater element. If both exist, the value at the next greater element's index is recorded as the second greater element for the original element.

This approach efficiently computes the second greater element for each position in linear time, making it suitable for large input sizes.
