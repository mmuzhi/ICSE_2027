=== RAW ANSWER ===

To solve this problem, we need to find the median of two sorted arrays. The median is the middle value in an ordered list of numbers. If the combined length of the two arrays is even, the median is the average of the two middle numbers; if it's odd, the median is the middle number.

### Approach
1. **Merge the Arrays**: Since both input arrays are sorted, we can merge them into a single sorted array by comparing elements from both arrays using two pointers. This ensures that the merged array is sorted and we can efficiently find the median.
2. **Calculate the Median**: After merging the arrays, we determine if the total number of elements is even or odd. For an even count, the median is the average of the two middle elements; for an odd count, it's the middle element.

### Solution Code
```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        p1 = 0
        p2 = 0
        new = []
        while p1 < len(nums1) and p2 < len(nums2):
            if nums1[p1] <= nums2[p2]:
                new.append(nums1[p1])
                p1 += 1
            else:
                new.append(nums2[p2])
                p2 += 1
        while p1 < len(nums1):
            new.append(nums1[p1])
            p1 += 1
        while p2 < len(nums2):
            new.append(nums2[p2])
            p2 += 1
        
        n = len(new)
        if n % 2 == 0:
            index = n // 2
            median = (new[index] + new[index-1]) / 2
        else:
            median = float(new[n//2])
        return median
```

### Explanation
1. **Merging Arrays**: The algorithm uses two pointers, `p1` and `p2`, to traverse `nums1` and `nums2` respectively. It compares elements from both arrays and appends the smaller element to the merged array `new`. This process continues until one of the arrays is exhausted.
2. **Appending Remaining Elements**: After one array is fully traversed, the remaining elements of the other array are appended to `new` since they are already in sorted order.
3. **Finding the Median**: The median is calculated based on the length of the merged array. If the length is even, the median is the average of the elements at positions `n//2` and `n//2 - 1`. If the length is odd, the median is the element at position `n//2`.

This approach efficiently merges the two sorted arrays and computes the median in linear time, O(n), where n is the total number of elements in both arrays. The space complexity is also O(n) due to the storage required for the merged array.
