3. **Position Tracking**: For each character in `s`, we find its corresponding leftmost position in `t` and store these positions in an array `a`.

4. **Binary Indexed Tree (BIT)**: We use a BIT to keep track of the number of characters placed so far. The BIT helps in efficiently computing the prefix sums and updating counts.

class BIT():
    def __init__(self, n):
        self.n = n
        self.tree = [0] * (n + 1)
