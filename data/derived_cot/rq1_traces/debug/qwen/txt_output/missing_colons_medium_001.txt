To solve this problem, we need to remove duplicates from a sorted array such that each element appears at most twice. The solution should modify the array in-place and return the new length of the array after removing duplicates.

### Approach
1. **Problem Analysis**: The problem requires us to process a sorted array and ensure that each element appears no more than twice. The solution must efficiently modify the array in-place without using extra space.
2. **Key Insight**: Since the array is sorted, all duplicates of a number will be consecutive. We can use a pointer `ans` to track the end of the modified array. The first two elements are always kept. For subsequent elements, we check if they are duplicates of the last two elements in the modified array. If they are, we skip them; otherwise, we include them.
3. **Algorithm Selection**: We iterate through the array starting from the third element. For each element, we compare it with the last two elements in the modified array. If it matches both, it's a third occurrence and is skipped. Otherwise, it's added to the modified array by incrementing the `ans` pointer.
4. **Complexity Analysis**: The algorithm runs in O(n) time, where n is the length of the array, as it processes each element exactly once. The space complexity is O(1) since we only use a few extra variables.

### Solution Code
```python
class Solution:
    def removeDuplicates(self, nums: List[int]) -> int:
        if len(nums) == 0:
            return 0
        if len(nums) == 1:
            return 1
        ans = 2
        for i in range(2, len(nums)):
            if nums[i] == nums[ans-1] and nums[i] == nums[ans-2]:
                continue
            else:
                nums[ans] = nums[i]
                ans += 1
        return ans
```

### Explanation
1. **Initial Checks**: The function first checks if the array is empty or has only one element, returning 0 or 1 respectively.
2. **Initialization**: The variable `ans` is initialized to 2, indicating that the first two elements are kept.
3. **Iteration**: Starting from the third element, each element is checked against the last two elements in the modified array (pointed to by `ans-1` and `ans-2`). If the current element matches both, it is skipped. Otherwise, it is placed at the position `ans`, and `ans` is incremented.
4. **Result**: The function returns `ans`, which represents the new length of the modified array with at most two occurrences of each element.

This approach efficiently processes the array in linear time with constant space, adhering to the problem constraints and requirements.
