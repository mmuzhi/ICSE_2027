=== RAW ANSWER ===

To solve this problem, we need to find the median of two sorted arrays. The median is the middle value in an ordered list of numbers. If the combined length of the arrays is even, the median is the average of the two middle numbers. 

### Approach
1. **Merge the Arrays**: Since both input arrays are sorted, we can merge them into a single sorted array by comparing elements from both arrays and appending the smaller element to the result array.
2. **Calculate the Median**: After merging, if the length of the merged array is odd, the median is the middle element. If the length is even, the median is the average of the two middle elements.

### Solution Code
```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        p1 = 0
        p2 = 0
        new = []
        while p1 < len(nums1) and p2 < len(nums2):
            if nums1[p1] < nums2[p2]:
                new.append(nums1[p1])
                p1 += 1
            else:
                new.append(nums2[p2])
                p2 += 1
        while p1 < len(nums1):
            new.append(nums1[p1])
            p1 += 1
        while p2 < len(nums2):
            new.append(nums2[p2])
            p2 += 1
        
        n = len(new)
        if n % 2 == 1:
            median = float(new[n // 2])
        else:
            median = (new[n // 2 - 1] + new[n // 2]) / 2.0
        return median
```

### Explanation
1. **Merging Arrays**: The code starts by initializing two pointers, `p1` and `p2`, to the start of each array. It then iterates through both arrays, comparing elements at the current pointers and appending the smaller element to the result array `new`. The corresponding pointer is incremented after each comparison.
2. **Appending Remaining Elements**: After one of the arrays is exhausted, the remaining elements from the other array are appended to the result array.
3. **Finding the Median**: The length of the merged array is checked. If it's odd, the median is the middle element. If it's even, the median is the average of the two middle elements. The result is returned as a float to handle both integer and average cases appropriately. 

This approach efficiently merges the two sorted arrays and calculates the median in linear time, O(n), where n is the total number of elements in both arrays.
