=== RAW ANSWER ===

To solve this problem, we need to find all starting indices of substrings in a given string `s` that are exactly the concatenation of each word in a list `words` exactly once. The words in the list are of the same length, and the order of words in the substring does not matter.

### Approach
1. **Problem Analysis**: The task is to identify all substrings in `s` that can be formed by concatenating each word in `words` exactly once. The substring must be contiguous and exactly of length `n * wlen`, where `n` is the number of words and `wlen` is the length of each word.

2. **Intuition**: We can use a sliding window approach combined with a frequency dictionary to efficiently check for valid substrings. The sliding window will move one word length (`wlen`) at a time, updating the frequency count of words in the current window. The frequency dictionary (`occ`) will keep track of the required word counts from `words`.

3. **Algorithm Selection**: 
   - **Frequency Dictionary**: We first create a frequency dictionary (`occ`) for all words in `words`.
   - **Sliding Window**: For each starting index `k` (from 0 to `wlen-1`), we build an initial window of length `total_len` (i.e., `n * wlen`). We then slide this window by `wlen` each time, updating the frequency count by removing the leftmost word and adding the new rightmost word.
   - **Validation**: After each update, we check if the current window's word frequencies match `occ`. If they do, the starting index of the window is added to the result list.

4. **Complexity Analysis**: 
   - **Time Complexity**: O(N * M), where N is the length of `s` and M is the number of words. Each window slide involves O(M) operations for updating and checking the frequency dictionary.
   - **Space Complexity**: O(M), where M is the number of words, due to the storage required for the frequency dictionary and the sliding window.

### Solution Code
```python
import collections
from typing import List

class Solution:
    def findSubstring(self, s: str, words: List[str]) -> List[int]:
        if not words:
            return []
        wlen = len(words[0])
        n = len(words)
        total_len = wlen * n
        if total_len > len(s):
            return []
        from collections import defaultdict, Counter
        occ = Counter(words)

        def test(track):
            for key in occ:
                if key not in track or track[key] != occ[key]:
                    return False
            return True

        res = []
        for k in range(wlen):
            if k + total_len > len(s):
                continue
            track = defaultdict(int)
            for i in range(k, k + total_len, wlen):
                if i + wlen > len(s):
                    break
                word = s[i:i+wlen]
                if word in occ:
                    track[word] += 1
            if test(track):
                res.append(k)
                
            for i in range(k + wlen, len(s) - total_len + 1, wlen):
                word = s[i - wlen:i - wlen + wlen]
                if word in occ:
                    track[word] -= 1
                word = s[i + total_len - wlen:i + total_len]
                if word in occ:
                    track[word] += 1
                if test(track):
                    res.append(i)
        return res
```

### Explanation
1. **Initial Checks**: The code first checks if the list of words is empty or if the combined length of all words exceeds the length of `s`, in which case it returns an empty list.
2. **Frequency Dictionary**: The `occ` dictionary is built using `collections.Counter` to store the frequency of each word in `words`.
3. **Sliding Window Setup**: For each starting index `k` (from 0 to `wlen-1`), the code initializes a sliding window and checks if the initial window (from `k` to `k + total_len`) contains all words from `words` with the correct frequencies.
4. **Window Sliding**: The window is then slid by `wlen` each time. The leftmost word of the previous window is removed, and the new rightmost word is added. The frequency dictionary is updated accordingly.
5. **Validation**: After each update, the code checks if the current window's word frequencies match `occ`. If they do, the starting index of the window is added to the result list.
6. **Result Compilation**: The result list, containing all valid starting indices, is returned after processing all possible starting indices.

This approach efficiently checks all possible substrings of the required length by leveraging a sliding window and a frequency dictionary, ensuring optimal performance and correctness.
