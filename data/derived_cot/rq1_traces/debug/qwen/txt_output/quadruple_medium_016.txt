=== RAW ANSWER ===

To solve this problem, we need to restore all possible valid IP addresses from a given string. An IP address consists of four parts separated by dots, where each part is a number between 0 and 255. The solution involves using a backtracking approach to explore all possible ways to split the string into four valid parts.

### Approach
1. **Backtracking with Pruning**: The solution uses a backtracking method to explore all possible ways to split the string into four parts. Each part must be a number between 0 and 255.
2. **Handling Leading Zeros**: If a part starts with '0', it can only be a single zero (e.g., "0" is valid, but "01" is not).
3. **Base Case**: When the entire string is processed and exactly four parts are formed, the current combination is added to the result.
4. **Recursive Case**: For each position in the string, the algorithm tries to form parts of length 1, 2, or 3 digits (as long as the number is less than 256 and does not exceed the string length).

### Solution Code
```python
class Solution():
    def restoreIpAddresses(self, s: str) -> List[str]:
        res = []
        cur = []
        
        def backtrack(i):
            if i == len(s) and len(cur) == 4:
                res.append(".".join(cur))
                return
            if len(cur) > 4 or i >= len(s):
                return
            if s[i] == '0':
                cur.append('0')
                backtrack(i+1)
                cur.pop()
                return
            j = 0
            while j < 3 and i + j < len(s):
                num = int(s[i:i+j+1])
                if num < 256:
                    cur.append(str(num))
                    backtrack(i+j+1)
                    cur.pop()
                j += 1
        
        backtrack(0)
        return res
```

1. **Initialization**: The function starts by initializing an empty list `res` to store the valid IP addresses and an empty list `cur` to keep track of the current parts being formed.
2. **Backtracking Function**: The nested function `backtrack(i)` processes the string starting from index `i`:
   - **Base Case**: If the entire string is processed (`i == len(s)`) and exactly four parts are formed (`len(cur) == 4`), the current parts are joined into a string and added to `res`.
   - **Pruning**: If more than four parts are formed or the index `i` is beyond the string length, the function returns without further processing.
   - **Zero Handling**: If the current character is '0', it is added as a single part, and the function proceeds recursively.
   - **Non-Zero Handling**: For non-zero characters, the function tries parts of length 1, 2, or 3 digits (as long as the number is less than 256 and within bounds). Each valid part is added to `cur`, and the function proceeds recursively.
3. **Result Compilation**: After the backtracking completes, the function returns the list of all valid IP addresses found.
