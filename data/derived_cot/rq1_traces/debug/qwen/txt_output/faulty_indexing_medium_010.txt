If the region is entirely composed of the same value (0 or 1), it is represented as a leaf node. Otherwise, it is split into four quadrants, each represented by child nodes.

### Approach
1. **Base Case Handling**: If the current region is of size 1x1, return a leaf node with the value of that cell.
2. **Recursive Division**: For larger regions, divide the grid into four quadrants (top-left, top-right, bottom-left, bottom-right) of equal size.
3. **Node Construction**: For each quadrant, recursively build the quad-tree. After building the child nodes, check if all four quadrants are leaf nodes with the same value. If so, merge them into a single leaf node. Otherwise, create a non-leaf node with the values from the child nodes.
