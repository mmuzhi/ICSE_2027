=== RAW ANSWER ===

To solve this problem, we need to sort a string by alternating between ascending and descending order of its characters until all characters are used. The approach involves counting the frequency of each character, then repeatedly outputting characters in ascending order followed by descending order until no characters remain.

### Approach
1. **Frequency Counting**: First, we count the frequency of each character in the input string using a dictionary.
2. **Distinct Characters**: We then sort the distinct characters of the string to get the initial order for ascending sorting.
3. **Alternating Output**: While there are characters left to output, we perform two passes:
   - **Ascending Pass**: Output each remaining character in ascending order, reducing their frequency each time they are used.
   - **Descending Pass**: Output each remaining character in descending order, again reducing their frequency each time they are used.
4. **Termination**: The process continues until all characters' frequencies are exhausted.

### Solution Code
```python
class Solution:
    def sortString(self, s: str) -> str:
        freq = {}
        for i in s:
            freq[i] = freq.get(i, 0) + 1
        letters = sorted(freq.keys())
        res = ""
        while freq:
            for i in letters:
                if freq[i] > 0:
                    res += i
                    freq[i] -= 1
            for i in letters[::-1]:
                if freq[i] > 0:
                    res += i
                    freq[i] -= 1
        return res
```

### Explanation
1. **Frequency Counting**: The code starts by initializing a dictionary `freq` to count occurrences of each character in the input string `s`.
2. **Distinct Characters**: The keys of the dictionary (distinct characters) are sorted to form the list `letters`, which determines the order for ascending and descending passes.
3. **Alternating Output**: The `while` loop continues as long as there are characters left (i.e., `freq` is not empty). In each iteration:
   - **Ascending Pass**: The code iterates over `letters` in ascending order, appending each character to the result string `res` and decrementing its frequency.
   - **Descending Pass**: The code iterates over `letters` in reverse (descending) order, appending each character to `res` and decrementing its frequency if it's still available.
4. **Result Construction**: The process repeats until all characters are used up, resulting in a string that alternates between ascending and descending orders of characters based on their remaining frequencies.

This approach efficiently handles the problem by leveraging a frequency dictionary and alternating passes to construct the desired output string.
