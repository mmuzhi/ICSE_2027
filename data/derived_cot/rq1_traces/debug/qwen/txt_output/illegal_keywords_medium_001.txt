To solve this problem, we need to count the number of ways to split a given array of integers into contiguous subarrays such that each subarray starts and ends with the same first and last elements of the entire array. The solution involves identifying valid splits based on the positions of the first and last elements and calculating the number of ways to split the array between these elements.

### Approach
1. **Check for Valid Start and End Elements**: The array must start and end with the same element; otherwise, there are no valid splits, and we return 0.
2. **Transform the Array**: Convert the array into a string and remove any leading or trailing zeros. Then, split the string by the target element (1 in this case) to isolate the segments between the target elements.
3. **Calculate Splits**: For each segment between the target elements, the number of ways to split is determined by the length of the segment plus one. Multiply these values for all segments to get the total number of valid splits.
4. **Modulo Operation**: Since the result can be very large, return the result modulo 1000000007.

### Solution Code
```python
from functools import reduce

class Solution:
    def numberOfGoodSubarraySplits(self, nums: List[int]) -> int:
        if nums[0] != nums[-1] or nums[0] not in nums:
            return 0
        
        s = ''.join(map(str, nums)).strip('0')
        parts = s.split(str(nums[0]))
        
        if not parts[0]:
            parts = parts[1:]
        if not parts[-1]:
            parts = parts[:-1]
        
        ways = [len(part) + 1 for part in parts]
        
        result = reduce(lambda a, b: a * b, ways, 1) % 1000000007
        return result
```

3. **Handle Edge Cases**: If the first or last part is empty (due to leading/trailing zeros being stripped), these parts are removed from the list.
4. **Calculate Splits**: For each part between the target elements, the number of ways to split is `len(part) + 1`. These values are multiplied together to get the total number of valid splits.
