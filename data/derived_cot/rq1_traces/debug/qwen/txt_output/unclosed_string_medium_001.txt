=== RAW ANSWER ===

To solve this problem, we need to construct a quad-tree from a given grid. The quad-tree is a hierarchical data structure where each node represents a 2x2 quadrant of the grid. If all four quadrants of a node have the same value and are leaves, then the node can be merged into a single leaf node. Otherwise, the node remains an internal node with four children.

### Approach
1. **Base Case Handling**: If the grid is a single cell (level 1), return a leaf node with that cell's value.
2. **Recursive Division**: For larger grids, divide the grid into four quadrants and recursively build the quad-tree for each quadrant.
3. **Merge Condition**: After building the four child nodes, check if all four are leaves and have the same value. If so, merge them into a single leaf node. Otherwise, create an internal node with the four child nodes.

### Solution Code
```python
class Solution:
    def construct(self, grid: List[List[int]]) -> 'Node':
        level = len(grid)
        root = None
        if level >= 1:
            root = self.buildTree(grid, 0, level, 0, level, level)
        return root
        
    def buildTree(self, grid, rs, re, cs, ce, level):
        if level == 1:
            return Node(grid[rs][cs], True, None, None, None, None)
        
        next_level = level // 2
        tl = self.buildTree(grid, rs, re - next_level, cs, ce - next_level, next_level)
        tr = self.buildTree(grid, rs, re - next_level, ce - next_level, ce, next_level)
        bl = self.buildTree(grid, re - next_level, re, cs, ce - next_level, next_level)
        br = self.buildTree(grid, re - next_level, re, ce - next_level, ce, next_level)
        
        if tl.isLeaf and tr.isLeaf and bl.isLeaf and br.isLeaf and tl.val == tr.val == bl.val == br.val:
            new_node = Node(tl.val, True, None, None, None, None)
        else:
            new_node = Node(tl.val, False, tl, tr, bl, br)
            
        return new_node
```

### Explanation
1. **Base Case Handling**: The `construct` method initializes the root node by calling `buildTree` with the entire grid. If the grid is empty (level 0), it returns `None`. Otherwise, it starts the recursive process.
2. **Recursive Division**: The `buildTree` method checks if the current grid size is 1. If so, it returns a leaf node. Otherwise, it divides the grid into four quadrants and recursively builds the quad-tree for each quadrant.
3. **Merge Condition**: After building the four child nodes, the method checks if all are leaves and have the same value. If true, it creates a new leaf node; otherwise, it creates an internal node with the four children. This ensures the quad-tree is built efficiently by merging uniform quadrants into single nodes.
