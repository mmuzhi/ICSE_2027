3. **Algorithm Selection**: We use Dijkstra's algorithm with a priority queue, but since the weights are only 0 or 1, a 0-1 BFS using a deque is more efficient. The deque allows us to prioritize 0-cost moves, ensuring optimal pathfinding.

from collections import deque

class Solution:
    def minimumObstacles(self, grid: List[List[int]]) -> int:
        m = len(grid)
