=== RAW ANSWER ===

To solve this problem, we need to compute the alternating digit sum of a given positive integer. The alternating digit sum is defined as the sum of digits at even indices (0-indexed) minus the sum of digits at odd indices. 

### Approach
1. **Convert the integer to a string**: This allows us to easily access each digit by its index.
2. **Iterate over each digit**: For each digit, check if its index is even or odd.
3. **Sum the digits accordingly**: Add the digit's value to the total if its index is even, and subtract it if the index is odd.
4. **Return the result**: After processing all digits, the accumulated total is the desired alternating digit sum.

### Solution Code
```python
class Solution:
    def alternateDigitSum(self, n: int) -> int:
        count = 0
        n_str = str(n)
        for i in range(len(n_str)):
            if i % 2 == 0:
                count += int(n_str[i])
            else:
                count -= int(n_str[i])
        return count
```

### Explanation
1. **Conversion to String**: The integer `n` is converted to a string to facilitate individual digit access.
2. **Iteration and Index Check**: For each digit in the string, the index is checked. If the index is even (i.e., divisible by 2), the digit's value is added to the total. If the index is odd, the digit's value is subtracted from the total.
3. **Result Calculation**: The loop processes all digits, adjusting the total based on the digit's position. The final result is returned after processing all digits.

This approach efficiently computes the alternating digit sum by leveraging string manipulation for simplicity and clarity, ensuring correctness for both even and odd-length numbers.
