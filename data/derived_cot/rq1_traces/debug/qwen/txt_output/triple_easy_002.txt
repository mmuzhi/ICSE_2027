=== RAW ANSWER ===

To solve this problem, we need to determine if all the frequencies of the numbers in an array are unique. The frequencies should not repeat; each number in the array should appear a distinct number of times.

### Approach
1. **Count Frequencies**: Use the `Counter` class from the `collections` module to count the occurrences of each number in the array.
2. **Check Uniqueness**: Convert the frequency counts into a set to eliminate duplicates. If the number of unique frequencies matches the total number of distinct numbers (keys) in the frequency dictionary, then all frequencies are unique.

### Solution Code
```python
from collections import Counter

class Solution:
    def uniqueOccurrences(self, arr: List[int]) -> bool:
        d = Counter(arr)
        return len(set(d.values())) == len(d)
```

### Explanation
1. **Counting Frequencies**: The `Counter` class efficiently counts how many times each element appears in the input array. For example, given the array `[1, 2, 2, 3, 3, 3]`, the `Counter` will produce `{1:1, 2:2, 3:3}`.
2. **Checking Uniqueness**: By converting the values (frequencies) of the `Counter` into a set, we remove any duplicates. If the size of this set matches the number of distinct elements (keys) in the `Counter`, it means all frequencies are unique. For instance, in the example above, the set of frequencies `{1, 2, 3}` has a size of 3, which matches the number of distinct elements, so the function returns `True`.

This approach efficiently checks for unique frequencies by leveraging the properties of sets and dictionaries, ensuring optimal performance with a time complexity of O(n), where n is the number of elements in the input array.
