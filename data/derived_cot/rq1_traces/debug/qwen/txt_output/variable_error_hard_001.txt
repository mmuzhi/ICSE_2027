=== RAW ANSWER ===

To solve this problem, we need to find the median of two sorted arrays. The median is the middle value in an ordered list of numbers. If the combined length of the two arrays is even, the median is the average of the two middle numbers; if it's odd, the median is the middle number.

### Approach
1. **Merge the Arrays**: We use two pointers to traverse both arrays simultaneously, comparing elements from each array and appending the smaller element to a new merged array. This ensures the merged array remains sorted.
2. **Handle Remaining Elements**: After one of the arrays is exhausted, the remaining elements of the other array are appended to the merged array in their natural order since both original arrays are sorted.
3. **Calculate the Median**: Depending on whether the total number of elements in the merged array is even or odd, we compute the median. For an even count, the median is the average of the two middle elements; for an odd count, it's the middle element.

### Solution Code
```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        p1 = 0
        p2 = 0
        new = []
        while p1 < len(nums1) and p2 < len(nums2):
            if nums1[p1] < nums2[p2]:
                new.append(nums1[p1])
                p1 += 1
            else:
                new.append(nums2[p2])
                p2 += 1
        while p1 < len(nums1):
            new.append(nums1[p1])
            p1 += 1
        while p2 < len(nums2):
            new.append(nums2[p2])
            p2 += 1
        
        if len(new) % 2 == 0:
            index = len(new) // 2
            median = (new[index] + new[index-1]) / 2
        else:
            median = float(new[len(new)//2])
        return median
```

### Explanation
1. **Merging Arrays**: The first loop merges elements from both arrays by comparing them and appending the smaller element to the new array. This continues until one of the arrays is exhausted.
2. **Appending Remaining Elements**: The subsequent loops append the remaining elements of the non-exhausted array, ensuring the merged array is fully sorted.
3. **Finding the Median**: The median is calculated based on the length of the merged array. If the length is even, the median is the average of the two middle elements. If the length is odd, the median is the middle element. The result is returned as a float to handle both cases appropriately.

This approach efficiently merges the two sorted arrays and computes the median in linear time, making it optimal for large input sizes.
