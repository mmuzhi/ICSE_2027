=== RAW ANSWER ===

To solve this problem, we need to determine if a singly-linked list has a cycle. The approach involves using two pointers moving at different speeds through the list. The faster pointer moves two nodes at a time, while the slower pointer moves one node at a time. If the faster pointer eventually catches up to the slower pointer, a cycle exists in the list.

### Approach
1. **Initialization**: Start both pointers (`slow` and `fast`) at the head of the list.
2. **Traversal**: Move the `slow` pointer one node at a time and the `fast` pointer two nodes at a time in each iteration.
3. **Cycle Detection**: If at any point the `fast` pointer catches up to the `slow` pointer, a cycle is detected. If the `fast` pointer reaches the end of the list (i.e., `fast` or `fast.next` is `None`), the list does not contain a cycle.

### Solution Code
```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, x):
#         self.val = x
#         self.next = None

class Solution:
    def hasCycle(self, head: Optional[ListNode]) -> bool:
        slow = fast = head
        while fast and fast.next:
            slow = slow.next
            fast = fast.next.next
            if slow == fast:
                return True
        return False
```

### Explanation
1. **Initialization**: Both `slow` and `fast` pointers start at the head of the list.
2. **Loop Condition**: The loop continues as long as `fast` and `fast.next` are not `None`. This ensures that the `fast` pointer can move two nodes without causing a `None` reference error.
3. **Pointer Movement**: In each iteration, the `slow` pointer moves one node forward, and the `fast` pointer moves two nodes forward.
4. **Cycle Check**: If at any point `slow` and `fast` point to the same node, the function returns `True`, indicating a cycle exists. If the loop completes without the pointers meeting, the function returns `False`, indicating no cycle.

This approach efficiently detects cycles in linear time (O(n)) and constant space (O(1)), leveraging the properties of two pointers to traverse the list optimally.
