=== RAW ANSWER ===

To solve this problem, we need to find the median of two sorted arrays. The median is the middle value in an ordered list of numbers. If the combined length of the two arrays is even, the median is the average of the two middle numbers; if it's odd, the median is the middle number itself.

### Approach
The approach involves merging the two sorted arrays into a single sorted array and then finding the median based on the length of this merged array. The steps are as follows:

1. **Merge the Arrays**: Traverse both arrays simultaneously, comparing elements and appending the smaller element to a new array until one of the arrays is exhausted. Then, append the remaining elements of the non-exhausted array to the new array.
2. **Calculate the Median**: 
   - If the length of the merged array is even, the median is the average of the elements at positions `len(new)//2 - 1` and `len(new)//2`.
   - If the length is odd, the median is the element at position `len(new)//2`.

### Solution Code
```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        p1 = 0
        p2 = 0
        new = []
        while p1 < len(nums1) and p2 < len(nums2):
            if nums1[p1] < nums2[p2]:
                new.append(nums1[p1])
                p1 += 1
            else:
                new.append(nums2[p2])
                p2 += 1
        while p1 < len(nums1):
            new.append(nums1[p1])
            p1 += 1
        while p2 < len(nums2):
            new.append(nums2[p2])
            p2 += 1

        if len(new) % 2 == 0:
            index = len(new) // 2
            median = (new[index - 1] + new[index]) / 2.0
        else:
            median = float(new[len(new) // 2])
        return median
```

### Explanation
1. **Merging Arrays**: The code starts by initializing pointers for both arrays (`p1` and `p2`). It then iterates through both arrays, appending the smaller of the two current elements to the merged array `new` and moving the corresponding pointer forward. Once one array is exhausted, the remaining elements of the other array are appended to `new`.
2. **Finding the Median**: The merged array `new` is checked for its length. If the length is even, the median is calculated as the average of the two middle elements. If the length is odd, the median is the middle element directly. The result is returned as a float to ensure correct handling of both even and odd lengths.

This approach efficiently combines the two sorted arrays and computes the median in linear time relative to the combined length of the input arrays.
