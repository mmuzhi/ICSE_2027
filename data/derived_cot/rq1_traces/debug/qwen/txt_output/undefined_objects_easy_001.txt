=== RAW ANSWER ===

To solve this problem, we need to find the complement of a given number. The complement of a number is obtained by flipping all the bits in its binary representation. The key insight here is to process each bit of the number from the least significant bit to the most significant bit, flipping each bit (changing 1 to 0 and 0 to 1), and then converting the resulting binary string back to an integer.

### Approach
1. **Bit Processing**: Iterate through each bit of the number from the least significant bit (LSB) to the most significant bit (MSB). For each bit, if it is 1, append '0' to the result string; if it is 0, append '1'.
2. **Handling Zero**: Special case when the input number is 0. The complement of 0 should be 1, as flipping the single bit (0) results in 1.
3. **String Reversal**: After processing all bits, reverse the constructed string to get the correct binary representation and convert it to an integer.

### Solution Code
```python
class Solution:
    def findComplement(self, num: int) -> int:
        com = ''
        while num > 0:
            if num % 2 == 1:
                com += '0'
            else:
                com += '1'
            num = num // 2
        if com == '':
            return 1
        else:
            return int(com[::-1], 2)
```

### Explanation
1. **Bit Processing**: The loop processes each bit of the number by checking if the current bit is set (1) or not (0). For each bit, it appends the flipped bit to the result string `com`.
2. **Zero Handling**: If the input number is 0, the loop does not execute, and `com` remains an empty string. In this case, the function returns 1, which is the complement of 0.
3. **String Reversal and Conversion**: After processing all bits, the result string `com` is reversed (since bits were processed from LSB to MSB) and converted from a binary string to an integer. This integer is the complement of the input number.

This approach efficiently computes the complement by leveraging bit manipulation and string operations, ensuring correctness for all non-negative input values.
