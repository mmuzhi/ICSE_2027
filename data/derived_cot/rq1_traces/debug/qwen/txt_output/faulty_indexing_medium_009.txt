To solve this problem, we need to compute the sum of the minimum values of all contiguous subarrays in a given array. The challenge lies in efficiently calculating this sum without resorting to a brute-force approach, which would be too slow for large arrays.

### Approach
1. **Problem Analysis**: The task requires summing the minimum values of every possible contiguous subarray. A brute-force approach would involve checking all possible subarrays, leading to a time complexity of O(n^2), which is inefficient for large arrays (n can be up to 10^5).

2. **Intuition**: For each element in the array, determine how many subarrays it is the minimum element. This can be done by finding the nearest smaller elements to the left and right of each element. The number of subarrays where the current element is the minimum is given by the product of the distances to these nearest smaller elements.

3. **Algorithm Selection**: 
   - **Forward Pass**: Traverse the array from left to right to compute the left boundary for each element (the nearest smaller element to the left).
   - **Backward Pass**: Traverse the array from right to left to compute the right boundary for each element (the nearest smaller element to the right).
   - **Sum Calculation**: For each element, multiply its value by the product of the distances to its left and right boundaries, then sum these products. Use modulo operation to handle large numbers.

4. **Complexity Analysis**: Both passes (forward and backward) run in O(n) time each, and the final sum calculation is O(n). The overall time complexity is O(n), which is optimal for this problem. The space complexity is O(n) due to the storage of boundaries.

### Solution Code
```python
class Solution:
    def sumSubarrayMins(self, arr: List[int]) -> int:
        n = len(arr)
        left_bound = [-1] * n
        right_bound = [n] * n
        
        stack = []
        for i in range(n):
            while stack and arr[stack[-1]] > arr[i]:
                stack.pop()
            if stack:
                left_bound[i] = stack[-1]
            else:
                left_bound[i] = -1
            stack.append(i)
        
        stack = []
        for i in range(n-1, -1, -1):
            while stack and arr[stack[-1]] > arr[i]:
                stack.pop()
            if stack:
                right_bound[i] = stack[-1]
            else:
                right_bound[i] = n
            stack.append(i)
        
        mod = 10**9 + 7
        res = 0
        for i in range(n):
            count_left = i - left_bound[i]
            count_right = right_bound[i] - i
            res = (res + arr[i] * count_left * count_right) % mod
        
        return res
```

3. **Sum Calculation**: For each element, the number of subarrays where it is the minimum is calculated by multiplying the distance to the left boundary and the distance to the right boundary.
