=== RAW ANSWER ===

To solve this problem, we need to compute the median of every sliding window of size `k` in an array of integers `nums`. The median is the middle value in an ordered list of numbers. If the window size `k` is odd, the median is the middle element. If `k` is even, the median is the average of the two middle elements.

### Approach
1. **Binary Search Tree (BST) Construction**: We use a BST to maintain the current elements in the sliding window. This allows efficient insertion, deletion, and retrieval of the k-th smallest element.
2. **Insertion and Deletion**: For each new element entering the window, we insert it into the BST. When the window size exceeds `k`, we remove the oldest element from the BST.
3. **Median Calculation**: After each insertion and deletion, if the window size is `k`, we compute the median. For an odd `k`, the median is the (k//2 + 1)-th smallest element in the BST. For an even `k`, the median is the average of the k//2-th and (k//2 + 1)-th smallest elements.

### Solution Code
```python
class Node:
    def __init__(self, val):
        self.val = val
        self.left = None
        self.right = None

def insert(root, val):
    if root is None:
        return Node(val)
    if val < root.val:
        root.left = insert(root.left, val)
    else:
        root.right = insert(root.right, val)
    return root

def remove(root, val):
    if root is None:
        return None
    if val < root.val:
        root.left = remove(root.left, val)
    elif val > root.val:
        root.right = remove(root.right, val)
    else:
        if root.left is None and root.right is None:
            return None
        elif root.left is None:
            return root.right
        elif root.right is None:
            return root.left
        else:
            temp = root.right
            while temp.left:
                temp = temp.left
            root.val = temp.val
            root.right = remove(root.right, temp.val)
    return root

def size(node):
    if node is None:
        return 0
    return 1 + size(node.left) + size(node.right)

def get(root, k):
    if root is None:
        return None
    left_size = size(root.left) if root.left else 0
    if left_size + 1 == k:
        return root.val
    elif k <= left_size:
        return get(root.left, k)
    else:
        return get(root.right, k - left_size - 1)

class Solution:
    def medianSlidingWindow(self, nums: List[int], k: int) -> List[float]:
        tree = None
        ans = []
        for i, x in enumerate(nums):
            tree = insert(tree, x)
            if i >= k:
                tree = remove(tree, nums[i - k])
            if i >= k - 1:
                if k % 2 == 1:
                    ans.append(get(tree, k // 2 + 1))
                else:
                    ans.append((get(tree, k // 2) + get(tree, k // 2 + 1)) / 2)
        return ans
```

1. **Node Class**: Represents each node in the BST with a value and pointers to left and right children.
2. **Insert Function**: Adds a new value to the BST, maintaining the BST properties.
3. **Remove Function**: Removes a specified value from the BST, handling cases where the node has zero, one, or two children.
4. **Size Function**: Computes the number of nodes in the BST subtree rooted at the given node.
5. **Get Function**: Retrieves the k-th smallest element in the BST by performing an inorder traversal and counting nodes.
6. **MedianSlidingWindow Method**: Processes each element in the array, inserts it into the BST, removes the oldest element if the window size exceeds `k`, and calculates the median for the current window. The median is computed based on whether `k` is odd or even.

This approach efficiently maintains the sliding window using a BST, allowing for quick median calculations by leveraging the BST's properties. The complexity is O(n log k) for each insertion and deletion, where `n` is the number of elements in the array and `k` is the window size.
