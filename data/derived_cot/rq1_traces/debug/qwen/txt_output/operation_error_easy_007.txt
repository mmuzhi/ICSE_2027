### Solution Code
```python
class Solution:
    def findComplement(self, num: int) -> int:
        if num == 0:
            return 0
        n = num.bit_length()
        s = bin(num)[2:]
        comp_str = ''.join('1' if bit == '0' else '0' for bit in s)
```

4. **Bit Flipping**: Each bit in the binary string is flipped using a simple conditional check. '0' becomes '1' and '1' becomes '0'.
