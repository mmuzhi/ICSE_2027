### Approach
1. **Problem Analysis**: The task is to find all contiguous subarrays of length at least 3 where the difference between consecutive elements remains the same throughout the subarray. The challenge is to efficiently count these subarrays without resorting to a brute-force approach, which would be computationally expensive.

2. **Intuition**: The key observation is that for any subarray ending at index `i`, if the difference between the current element and the previous element is the same as the difference between the previous element and the element before it, then we can extend the arithmetic sequence ending at `i-1` to include `i`. This allows us to use dynamic programming to keep track of valid subarrays ending at each position.

3. **Algorithm Selection**: We use a dynamic programming array `dp` where `dp[i]` represents the number of valid arithmetic subarrays ending at index `i`. For each index `i` starting from 2, we check if the difference between `nums[i]` and `nums[i-1]` matches the difference between `nums[i-1]` and `nums[i-2]`. If it does, we increment `dp[i]` by `dp[i-1] + 1`; otherwise, we reset `dp[i]` to 0. The total number of valid subarrays is the sum of all `dp[i]` values from index 2 to the end of the array.

4. **Complexity Analysis**: The algorithm runs in O(n) time, where n is the length of the array, because each element is processed exactly once. The space complexity is O(n) due to the storage of the `dp` array.
