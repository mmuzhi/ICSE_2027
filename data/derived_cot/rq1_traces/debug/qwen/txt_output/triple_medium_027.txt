To solve this problem, we need to generate all valid IP addresses from a given string. The solution involves checking all possible combinations of substrings that form valid IP addresses, ensuring each part is between 0 and 255 and does not have leading zeros unless the part is exactly "0".

### Approach
1. **Backtracking with Depth-First Search (DFS)**: The approach involves recursively exploring all possible ways to split the string into four parts, each representing an octet of an IP address.
2. **Base Case Handling**: When the entire string is processed and exactly four parts are formed, the current combination of parts is added to the result.
3. **Validation Checks**: 
   - If a part starts with '0', it must be exactly "0" (handled separately to avoid leading zeros).
   - For other parts, ensure the substring is between 1 and 3 digits and its integer value is less than 256.
4. **Pruning Invalid Paths**: Stop further exploration if more than four parts are formed or the string is fully processed without forming exactly four parts.

### Solution Code
```python
class Solution:
    def restoreIpAddresses(self, s: str) -> List[str]:
        res = []
        cur = []
        
        def backtrack(i):
            if i == len(s) and len(cur) == 4:
                res.append(".".join(cur))
                return
            if len(cur) > 4 or i >= len(s):
                return
            
            if s[i] == '0':
                cur.append('0')
                backtrack(i + 1)
                cur.pop()
                return
            
            j = 0
            while j < 3 and i + j < len(s):
                num = int(s[i:i+j+1])
                if num < 256:
                    cur.append(s[i:i+j+1])
                    backtrack(i+j+1)
                    cur.pop()
                j += 1
        
        backtrack(0)
        return res
```

### Explanation
1. **Initialization**: The `res` list stores all valid IP addresses, and `cur` holds the current parts being formed during recursion.
2. **Backtracking Function**: The `backtrack` function processes the string starting from index `i`:
   - **Base Case**: If the entire string is processed and four parts are formed, the current parts are joined into an IP address and added to `res`.
   - **Leading Zero Handling**: If the current character is '0', it is added as a part, and the function recurses to the next index.
   - **Non-Zero Handling**: For other characters, the function checks all possible 1 to 3-digit parts, ensuring they are less than 256. Valid parts are added to `cur`, and the function recurses.
3. **Result Compilation**: After exploring all valid combinations, the `res` list is returned, containing all valid IP addresses.

This approach efficiently explores all possible valid IP addresses using backtracking, ensuring correctness through careful validation checks and pruning of invalid paths.
