=== RAW ANSWER ===

To solve this problem, we need to determine the maximum profit achievable by buying and selling a stock at most once over a given period. The solution involves finding the optimal buy and sell points where the sell price is higher than the buy price.

### Approach
The approach to solve this problem efficiently is to iterate through the list of stock prices and keep track of the minimum price encountered so far. For each subsequent price, if it is higher than the minimum price, the difference between the current price and the minimum price is added to the profit. This approach ensures that we only consider profitable transactions and maximize the profit by selling at the highest possible price after buying at the lowest.

### Solution Code
```python
class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        if len(prices) < 2:
            return 0
        min_price = prices[0]
        max_profit = 0
        for price in prices:
            if price < min_price:
                min_price = price
            else:
                if price - min_price > max_profit:
                    max_profit = price - min_price
        return max_profit
```

### Explanation
1. **Initialization**: We start by checking if the list of prices has at least two elements. If not, we return 0 since no transaction can be made.
2. **Tracking Minimum Price**: We initialize `min_price` to the first price in the list and `max_profit` to 0.
3. **Iterating Through Prices**: For each price in the list:
   - If the current price is lower than `min_price`, we update `min_price` to this lower value.
   - If the current price is higher than `min_price`, we calculate the potential profit (`price - min_price`) and update `max_profit` if this potential profit is greater than the current `max_profit`.
4. **Result**: After processing all prices, `max_profit` holds the maximum profit achievable from a single transaction, which is returned as the result.

This approach efficiently finds the best buy and sell points in a single pass through the list, resulting in an optimal solution with a time complexity of O(n) and a space complexity of O(1).
