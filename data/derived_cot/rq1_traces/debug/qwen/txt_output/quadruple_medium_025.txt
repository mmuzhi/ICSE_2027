=== RAW ANSWER ===

To solve this problem, we need to generate all valid IP addresses from a given string by splitting it into four parts, each ranging from 0 to 255. The solution involves using a backtracking approach to explore all possible valid splits.

### Approach
1. **Backtracking**: The algorithm uses recursion to explore all possible ways to split the string into four parts. Each recursive call processes the string starting from a specific index, appending valid parts to a temporary list.
2. **Base Case**: When the entire string is processed (i.e., the current index reaches the end of the string) and exactly four parts are formed, the current combination of parts is added to the result list.
3. **Validation Checks**: 
   - If the current character is '0', only one digit is considered to avoid leading zeros in subsequent parts.
   - For non-zero characters, up to three digits are considered to ensure the part does not exceed 255.
4. **Pruning**: The algorithm prunes invalid paths early by checking the length of the parts and ensuring the current part does not exceed 255.

### Solution Code
```python
class Solution:
    def restoreIpAddresses(self, s: str) -> List[str]:
        res = []
        cur = []
        
        def backtrack(i):
            if i == len(s) and len(cur) == 4:
                res.append(".".join(cur))
                return
            if len(cur) > 4 or i > len(s):
                return
            if s[i] == '0':
                cur.append(s[i])
                backtrack(i + 1)
                cur.pop()
                return
            j = 0
            while j < 3 and i + j < len(s):
                num_str = s[i:i+j+1]
                if int(num_str) < 256:
                    cur.append(num_str)
                    backtrack(i + j + 1)
                    cur.pop()
                j += 1
        
        backtrack(0)
        return res
```

### Explanation
1. **Initialization**: The `res` list stores all valid IP addresses, and `cur` is a temporary list that holds the current parts being formed.
2. **Backtracking Function**: The `backtrack` function starts from index `i` and processes the string recursively:
   - **Base Case**: If the entire string is processed (`i == len(s)`) and four parts are formed (`len(cur) == 4`), the current parts are joined into an IP address and added to `res`.
   - **Pruning**: If more than four parts are formed or the index exceeds the string length, the function returns immediately.
   - **Handling '0'**: If the current character is '0', only one digit is taken to avoid invalid leading zeros.
   - **Non-zero Handling**: For non-zero characters, the algorithm checks up to three digits (to stay within the 0-255 range) and appends valid parts, then proceeds recursively.
3. **Result Compilation**: After the backtracking completes, `res` contains all valid IP addresses, which are returned as the result.

This approach efficiently explores all valid splits using backtracking while pruning invalid paths early, ensuring optimal performance and correctness.
