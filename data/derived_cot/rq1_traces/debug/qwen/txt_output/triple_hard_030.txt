To solve this problem, we need to find a valid arrangement of directed edges in a graph such that all edges are traversed exactly once, forming an Eulerian trail or circuit. The solution involves constructing the graph, determining the starting node based on net out-degrees, and using Hierholzer's algorithm to traverse the graph.

### Approach
1. **Graph Construction**: Build the graph from the given pairs of edges, where each node points to a list of its outgoing edges.
2. **Degree Calculation**: Compute the net out-degree for each node (out-degree minus in-degree). This helps identify the starting node for the Eulerian trail.
3. **Starting Node Selection**: The starting node is the one with a net out-degree of 1. If no such node exists, the graph is an Eulerian circuit, and any node can be the start.
4. **Hierholzer's Algorithm**: Perform a depth-first search (DFS) to traverse the graph, popping edges from the graph as they are used. The traversal continues until all edges are exhausted, and the nodes are recorded in reverse order to form the Eulerian trail.

```python
from collections import defaultdict

class Solution:
    def validArrangement(self, pairs: List[List[int]]) -> List[List[int]]:
        graph = defaultdict(list)
        degree = defaultdict(int)
        for x, y in pairs:
            graph[x].append(y)
            degree[x] += 1
            degree[y] -= 1
        
        start = None
        for node in degree:
            if degree[node] == 1:
                start = node
                break
        else:
            if degree:
                start = next(iter(degree.keys()))
            else:
                return []
        
        ans = []
        def dfs(x):
            while graph[x]:
                neighbor = graph[x].pop()
                dfs(neighbor)
            ans.append(x)
        
        dfs(start)
        ans.reverse()
        return [[ans[i], ans[i+1]] for i in range(len(ans)-1)]
```

### Explanation
1. **Graph Construction**: The graph is built using a defaultdict where each node maps to a list of its adjacent nodes (outgoing edges). The degree dictionary keeps track of the net out-degree for each node.
2. **Starting Node Selection**: The loop checks for a node with a net out-degree of 1, which is the start of the Eulerian trail. If no such node is found, the first node in the graph is chosen as the start.
3. **DFS Traversal**: The `dfs` function recursively traverses the graph, popping edges from the graph as they are used. Nodes are appended to the answer list when all outgoing edges are exhausted.
4. **Result Formation**: The answer list is reversed to present the traversal in the correct order, and consecutive nodes are paired to form the final arrangement of edges.

This approach efficiently checks for the existence of an Eulerian trail or circuit and constructs the arrangement using DFS, ensuring all edges are traversed exactly once.
