To solve this problem, we need to construct a quad-tree from a given grid. The quad-tree is a hierarchical data structure where each node represents a region of the grid. If a region contains all the same values, it is represented as a leaf node; otherwise, it is split into four quadrants, each represented by child nodes.

### Approach
1. **Node Class Definition**: Define a `Node` class to represent each node in the quad-tree. Each node has a value (`val`), a boolean indicating if it is a leaf (`isLeaf`), and four child nodes (top-left, top-right, bottom-left, bottom-right).

2. **Base Case Handling**: When the region size is 1x1 (i.e., `level == 1`), the node is a leaf node with the value from the grid cell.

3. **Recursive Division**: For larger regions, divide the grid into four quadrants. Recursively build the quad-tree for each quadrant. The top-left quadrant spans from `(rs, re - next_level)` to `(cs, ce - next_level)`, the top-right from `(rs, re - next_level)` to `(cs + next_level, ce)`, the bottom-left from `(re - next_level, re)` to `(cs, ce - next_level)`, and the bottom-right from `(re - next_level, re)` to `(cs + next_level, ce)`.

4. **Leaf Node Check**: After building the child nodes, check if all four children are leaves and have the same value. If so, the current node is a leaf node with that value. Otherwise, it is an internal node with the four children.

class Node:
    def __init__(self, val, isLeaf, topLeft, topRight, bottomLeft, bottomRight):
        self.val = val
        self.isLeaf = isLeaf
        self.topLeft = topLeft
        self.topRight = topRight
        self.bottomLeft = bottomLeft
        self.bottomRight = bottomRight

class Solution:
    def construct(self, grid: List[List[int]]) -> 'Node':
        level = len(grid)
        root = None
        if level >= 1:

    def buildTree(self, grid, rs, re, cs, ce, level):

        tr = self.buildTree(grid, rs, re - next_level, cs + next_level, ce, next_level)
