=== RAW ANSWER ===

To solve this problem, we need to find for each element in `nums1` the next element in `nums2` that is greater than it. The solution involves iterating through each element in `nums1`, locating its first occurrence in `nums2`, and then searching to the right of this position for a greater element. If no such element is found, we append -1 to the result list.

### Approach
1. **Problem Analysis**: The task is to find the next greater element for each element in `nums1` within `nums2`. The next greater element must be the first element in `nums2` (to the right of the current element's position) that is larger than the current element.
2. **Intuition**: For each element in `nums1`, we need to:
   - Locate its first occurrence in `nums2`.
   - If the element is not found, append -1 to the result.
   - If found, search from the next position onward in `nums2` for the first element greater than the current element. If found, append it; otherwise, append -1.
3. **Algorithm Selection**: We use nested loops. The outer loop iterates over each element in `nums1`, and the inner loop searches `nums2` for the first occurrence of the current element. Once found, another loop checks the remaining elements in `nums2` for a greater value.
4. **Complexity Analysis**: The time complexity is O(n*m), where n is the length of `nums1` and m is the length of `nums2`. This is because for each element in `nums1`, we might scan all of `nums2` in the worst case. The space complexity is O(n) for the result list.

### Solution Code
```python
class Solution:
    def nextGreaterElement(self, nums1: List[int], nums2: List[int]) -> List[int]:
        l = []
        for i in nums1:
            found_index = None
            for idx, j in enumerate(nums2):
                if j == i:
                    found_index = idx
                    break
            if found_index is None:
                l.append(-1)
            else:
                found = False
                for k in range(found_index + 1, len(nums2)):
                    if nums2[k] > i:
                        l.append(nums2[k])
                        found = True
                        break
                if not found:
                    l.append(-1)
        return l
```

### Explanation
1. **Initialization**: We start with an empty list `l` to store the results.
2. **Outer Loop**: For each element `i` in `nums1`, we search `nums2` to find its first occurrence.
3. **Finding Position**: If `i` is found in `nums2`, we note its index (`found_index`) and break out of the search loop.
4. **Checking for Greater Element**: Using `found_index`, we then search the remaining elements in `nums2` (from `found_index + 1` to the end) for the first element greater than `i`.
5. **Result Handling**: If a greater element is found, it is added to the result list; otherwise, -1 is appended.
6. **Return Result**: After processing all elements in `nums1`, the result list `l` is returned.

This approach efficiently checks each element in `nums1` against `nums2` to determine the next greater element, ensuring correctness while maintaining clarity in the solution.
