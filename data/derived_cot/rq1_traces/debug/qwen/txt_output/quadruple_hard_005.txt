=== RAW ANSWER ===

To solve this problem, we need to find the median of two sorted arrays efficiently. The median is the middle value in an ordered list of numbers. If the combined length of the two arrays is even, the median is the average of the two middle numbers; otherwise, it is the middle number.

### Approach
1. **Merge the Arrays**: Combine the two sorted arrays into a single sorted array by comparing elements from both arrays and appending the smaller element to the result array. This ensures the merged array remains sorted.
2. **Calculate the Median**: Once the arrays are merged into a single sorted array, determine the median by checking the length of the array. If the length is odd, the median is the middle element. If the length is even, the median is the average of the two middle elements.

### Solution Code
```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        p1 = 0
        p2 = 0
        new = []
        
        while p1 < len(nums1) and p2 < len(nums2):
            if nums1[p1] < nums2[p2]:
                new.append(nums1[p1])
                p1 += 1
            else:
                new.append(nums2[p2])
                p2 += 1
        
        while p1 < len(nums1):
            new.append(nums1[p1])
            p1 += 1
        
        while p2 < len(nums2):
            new.append(nums2[p2])
            p2 += 1
        
        n = len(new)
        if n % 2 == 1:
            median = float(new[n // 2])
        else:
            median = (new[n // 2 - 1] + new[n // 2]) / 2.0
        
        return median
```

### Explanation
1. **Merging Arrays**: The code uses two pointers, `p1` and `p2`, to traverse `nums1` and `nums2` respectively. By comparing elements at these pointers, the smaller element is appended to the merged array `new`. This ensures the merged array remains sorted.
2. **Handling Remaining Elements**: After one of the arrays is exhausted, the remaining elements from the other array are appended to `new`.
3. **Finding the Median**: The median is calculated based on the length of `new`. If the length is odd, the median is the middle element. If even, it is the average of the two middle elements. The result is returned as a float to handle both cases appropriately.

This approach efficiently merges the arrays and computes the median in linear time, O(n + m), where n and m are the lengths of the two arrays, making it optimal for the given problem constraints.
