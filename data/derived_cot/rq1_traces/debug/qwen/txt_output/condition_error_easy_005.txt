=== RAW ANSWER ===

To solve this problem, we need to add 1 to a number represented as a list of its digits and return the result as a list of digits. The challenge is to handle cases where adding 1 might cause a carry-over, such as when the number is 999, which becomes 1000.

### Approach
1. **Convert the List of Digits to a String**: The digits in the input list are converted into a string to form the complete number.
2. **Increment the Number**: The string is converted to an integer, incremented by 1, and then converted back to a string.
3. **Convert the Result Back to a List of Digits**: The resulting string is split into individual digits, each converted back to an integer, and returned as a list.

This approach efficiently handles carry-overs by leveraging Python's ability to convert between strings and integers seamlessly. It simplifies the process of adding 1 to a multi-digit number and ensures the result is returned in the required format.

### Solution Code
```python
class Solution:
    def plusOne(self, digits: List[int]) -> List[int]:
        s = ''.join(map(str, digits))
        num = int(s) + 1
        return list(map(int, str(num)))
```

### Explanation
1. **String Conversion**: The input list of digits is joined into a single string. For example, `[1, 2, 3]` becomes `"123"`.
2. **Incrementing the Number**: The string is converted to an integer, incremented by 1, and then converted back to a string. For example, `"123"` becomes `124` after incrementing.
3. **Digit Conversion**: The resulting string is split into individual characters, each converted back to an integer, and collected into a list. For example, `"124"` becomes `[1, 2, 4]`.

This method efficiently handles all cases, including those with carry-overs, by utilizing simple string manipulation and integer conversion. The solution is both concise and effective for the given problem constraints.
