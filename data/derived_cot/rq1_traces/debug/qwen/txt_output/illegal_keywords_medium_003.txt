The meeting node is a node that both traversals reach, and it should be the node with the smallest maximum distance from both starting nodes. If no such node exists, we return -1.

### Approach
1. **Breadth-First Search (BFS)**: We use BFS to compute the shortest distances from both starting nodes (node1 and node2) to all reachable nodes. BFS is suitable here because it explores all nodes level by level, ensuring the shortest path is found first.
2. **Distance Tracking**: For each node, we maintain two distances: one from node1 and one from node2. This is stored in a dictionary where each key is a node, and the value is a list of two elements representing the distances from node1 and node2, respectively.
3. **Finding the Meeting Node**: After computing the distances, we iterate through all nodes to find those reachable from both starting nodes. For each such node, we compute the maximum of the two distances. The node with the smallest maximum distance is our answer. If multiple nodes have the same maximum distance, we choose the smallest node index.

### Solution Code
```python
from collections import deque
from typing import List, Dict, DefaultDict
from collections import defaultdict

class Solution:
    def cdist(self, it, distArray, distArrayIndex, edges):
        q = deque([it])
        distArray[it][distArrayIndex] = 0
        while q:
            node = q.popleft()
            neighbor = edges[node]
            if neighbor == -1:
                continue
            if distArray[neighbor][distArrayIndex] == float('inf'):
                distArray[neighbor][distArrayIndex] = distArray[node][distArrayIndex] + 1
                q.append(neighbor)
    
    def closestMeetingNode(self, edges: List[int], node1: int, node2: int) -> int:
        INF = float('inf')
        dist = defaultdict(lambda: [INF, INF])
        mmin, ans = INF, -1
        
        self.cdist(node1, dist, 0, edges)
        self.cdist(node2, dist, 1, edges)
        
        for node in dist:
            d1, d2 = dist[node]
            if d1 != INF and d2 != INF:
                max_dist = max(d1, d2)
                if max_dist < mmin:
                    mmin = max_dist
                    ans = node
                elif max_dist == mmin:
                    if node < ans:
                        ans = node
        
        return ans if ans != -1 else -1
```

### Explanation
1. **BFS Implementation**: The `cdist` function performs a BFS starting from a given node (either node1 or node2). It updates the distance for each node in the direction of traversal, storing the shortest path distances in a dictionary.
2. **Distance Combination**: The `closestMeetingNode` function initializes a dictionary to store distances from both starting nodes. It then calls `cdist` twice—once for each starting node—to populate the distances.
3. **Finding the Optimal Node**: The function iterates over all nodes, checking if they are reachable from both starting nodes. For each valid node, it calculates the maximum distance from the two starting nodes. The node with the smallest maximum distance is selected. If multiple nodes have the same maximum distance, the smallest node index is chosen.
4. **Result Handling**: If no meeting node is found, the function returns -1; otherwise, it returns the index of the closest meeting node.

This approach efficiently computes the solution by leveraging BFS to explore the graph and then combining the results to find the optimal meeting node.
