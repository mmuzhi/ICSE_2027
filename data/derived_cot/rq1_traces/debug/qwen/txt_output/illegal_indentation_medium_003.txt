To solve this problem, we need to generate all numbers within a given range [low, high] that are composed of sequential digits. A sequential digit number is one where each digit is exactly one more than the previous digit. For example, 123 is a sequential digit number because 1, 2, and 3 are consecutive digits.

### Approach
The approach involves using a breadth-first search (BFS) strategy to generate all possible sequential digit numbers. We start with digits from 1 to 9 and then iteratively build larger numbers by appending the next consecutive digit. For each generated number, we check if it lies within the specified range [low, high]. If it does, we include it in our result list. The BFS ensures that we explore all possible sequential digit numbers systematically.

1. **Initialization**: Start with digits 1 through 9 using a queue (deque for efficient popping from the front).
2. **BFS Processing**: For each number dequeued, check if it is within the range [low, high]. If it is, add it to the result list.
3. **Next Number Generation**: For each number, determine the last digit. If this digit is less than 9, generate the next number by appending the next digit (last digit + 1) to the current number and enqueue it.
4. **Termination**: The process continues until the queue is empty, ensuring all possible sequential digit numbers are considered.

### Solution Code
```python
from collections import deque

class Solution:
    def sequentialDigits(self, low, high):
        out = []
        queue = deque(range(1, 10))
        while queue:
            elem = queue.popleft()
            if low <= elem <= high:
                out.append(elem)
            last = elem % 10
            if last < 9:
                queue.append(elem * 10 + last + 1)
        return out
```

### Explanation
1. **Initialization**: The queue is initialized with digits 1 through 9, as these are the smallest sequential digit numbers.
2. **Processing Each Number**: The main loop processes each number in the queue. For each number, it checks if the number is within the range [low, high]. If it is, the number is added to the result list.
3. **Generating Next Numbers**: For each number, the last digit is found using `elem % 10`. If this digit is less than 9, the next sequential digit number is generated by multiplying the current number by 10 and adding the next digit (last digit + 1). This new number is then enqueued for further processing.
4. **Result Compilation**: The loop continues until all possible sequential digit numbers are generated and checked against the range. The result list, which contains all valid numbers, is returned at the end.

This approach efficiently explores all possible sequential digit numbers using BFS, ensuring that all numbers within the specified range are found. The use of a deque allows for efficient queue operations, optimizing the BFS process.
