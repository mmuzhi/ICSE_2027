To solve this problem, we need to determine the maximum number of times a given string `s2` can appear as a subsequence in `n1` repetitions of another string `s1`. The result should then be divided by `n2` using integer division. The challenge is to efficiently compute this without explicitly constructing the concatenated string, especially since `n1` can be very large.

### Approach
1. **Initial Checks**: First, we check if `s2` is empty or if it contains characters not present in `s1`. If either condition is true, the answer is 0 because `s2` cannot form any subsequences in `s1`.
2. **Cycle Detection**: We use a two-pointer technique to traverse the concatenated string (simulated by repeating `s1`) and count the occurrences of `s2` as a subsequence. To handle large `n1`, we detect cycles in the state (current position in `s1`, current matched position in `s2`). Once a cycle is detected, we calculate the number of complete cycles remaining and multiply the count of `s2` in one cycle by the number of remaining cycles.
3. **Simulation**: For each character in `s1`, we check if it matches the next character in `s2`. If it does, we advance the pointer in `s2`. If the entire `s2` is matched, we increment the count and reset the pointer in `s2`.
4. **Cycle Handling**: When a state (pointer positions) repeats, we calculate the cycle's length and the number of `s2` occurrences per cycle. Using this information, we compute the total occurrences without explicitly processing all characters.

### Solution Code
```python
class Solution:
    def getMaxRepetitions(self, s1: str, n1: int, s2: str, n2: int) -> int:
        if not s2 or not s1:
            return 0
        if not set(s2).issubset(set(s1)):
            return 0
        
        j = 0
        count = 0
        i = 0
        visited = {}
        total_chars = n1 * len(s1)
        
        for copy in range(n1):
            while i < len(s1):
                if j < len(s2) and s1[i] == s2[j]:
                    j += 1
                    if j == len(s2):
                        count += 1
                        j = 0
                state = (j, i)
                if state in visited:
                    prev_count, prev_total = visited[state]
                    cycle_length = (copy * len(s1) + i) - prev_total
                    cycle_count = count - prev_count
                    remaining_chars = total_chars - (copy * len(s1) + i)
                    complete_cycles = remaining_chars // cycle_length
                    count += complete_cycles * cycle_count
                    break
                visited[state] = (count, copy * len(s1) + i)
                i += 1
            else:
                continue
            break
        
        return count // n2
```

1. **Initial Checks**: The code first checks if `s2` is empty or contains characters not in `s1`. If so, it returns 0 immediately.
2. **Cycle Detection**: The main loop iterates over each character in `s1` for `n1` times. For each character, it checks if it matches the current character in `s2`. If it does, the pointer in `s2` is advanced. If the entire `s2` is matched, the count is incremented and the pointer in `s2` is reset.
3. **State Tracking**: The code records the state (current position in `s1`, current position in `s2`) along with the count of `s2` matched so far. If a state repeats, it indicates a cycle.
4. **Cycle Handling**: Upon detecting a cycle, the code calculates the number of complete cycles remaining and updates the count by multiplying the occurrences per cycle by the number of remaining cycles. This avoids processing all characters explicitly, significantly optimizing the solution.
5. **Result Calculation**: The final result is obtained by integer division of the total count by `n2`.
