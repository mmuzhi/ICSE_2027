=== RAW ANSWER ===

To solve this problem, we need to convert a given positive integer into its corresponding Excel column title. The Excel column titles follow a pattern similar to a base-26 numbering system, but without the use of a zero digit. For example, 1 becomes "A", 2 becomes "B", ..., 26 becomes "Z", 27 becomes "AA", and so on.

### Approach
1. **Problem Analysis**: The challenge is to convert a number into a string where each digit represents a letter in the alphabet. The key difference here is that there is no '0' in the Excel column title system. Therefore, when the remainder of the division by 26 is zero, it should map to 'Z' and the quotient should be adjusted accordingly.
2. **Insight**: The solution involves repeatedly dividing the number by 26, adjusting the remainder when it is zero to map to 'Z', and building the result string from right to left. This adjustment ensures that numbers like 26 (which should be "Z") are handled correctly.
3. **Algorithm Selection**: The algorithm uses a loop to process each digit of the number. In each iteration, the number is decremented by 1 to handle the zero case correctly, the remainder when divided by 26 is calculated, and the corresponding character is appended to the result string. The number is then divided by 26 for the next iteration.
4. **Complexity Analysis**: The time complexity is O(log n) because each iteration reduces the number by a factor of 26. The space complexity is O(log n) to store the result string.

### Solution Code
```python
class Solution:
    def convertToTitle(self, n: int) -> str:
        s = ""
        while n > 0:
            n -= 1
            r = n % 26
            s = chr(65 + r) + s
            n //= 26
        return s
```

### Explanation
1. **Initialization**: Start with an empty string `s` to build the result.
2. **Loop**: While the number `n` is greater than zero:
   - **Adjustment**: Decrement `n` by 1 to handle the zero case properly.
   - **Remainder Calculation**: Compute the remainder `r` when `n` is divided by 26. This remainder determines the current character.
   - **Character Mapping**: Convert the remainder to the corresponding uppercase letter using ASCII values (65 is the ASCII for 'A').
   - **String Building**: Prepend the character to the result string `s`.
   - **Division**: Divide `n` by 26 to process the next higher digit in the next iteration.
3. **Return Result**: The loop exits when `n` becomes zero, and the constructed string `s` is returned.

This approach efficiently converts the number into the correct Excel column title by handling each digit appropriately and adjusting for the absence of a zero digit in the system.
