Inequalities must then be checked to ensure that variables marked as different are not in the same group in the DSU.
2. **Intuition**: The DSU data structure helps manage the equivalence classes of variables. By processing equalities, we merge the sets of related variables. For inequalities, we verify that the variables are not in the same set, which would contradict the equation.
3. **Algorithm Selection**: The DSU with path compression and union by rank is used to efficiently manage and query the sets. Path compression ensures that future queries are faster, and union by rank keeps the tree shallow.
4. **Complexity Analysis**: The DSU operations (find and union) are nearly constant time due to path compression and union by rank.     def __init__(self):
        self.rank = [0] * 26
        self.parent = [i for i in range(26)]
    
    def finduPar(self, node):

            return node
        self.parent[node] = self.finduPar(self.parent[node])
        return self.parent[node]

        if ulp_u == ulp_v:

            self.rank[ulp_u] += 1
