To solve this problem, we need to determine if an array can be split into three contiguous parts such that the sum of each part is equal. The solution involves checking if the total sum of the array is divisible by 3 and then verifying if there exists a valid split point for the first two parts, ensuring the third part is also valid.

### Approach
1. **Check Total Sum Divisibility**: The first step is to check if the total sum of the array is divisible by 3. If not, it's impossible to split the array into three parts with equal sums, so we return False.
2. **Prefix Sum Calculation**: Compute the prefix sums of the array. This helps in efficiently checking the sum of any contiguous subarray.
3. **Find First Valid Split**: Locate the first index where the prefix sum equals one-third of the total sum. This index marks the end of the first part.
4. **Check for Second Valid Split**: From the first valid split, find another index where the prefix sum equals two-thirds of the total sum. This index marks the end of the second part, ensuring the third part (the remainder of the array) also sums to one-third of the total sum.
5. **Edge Cases**: Handle cases where the array length is less than 3 or no valid splits are found.

### Solution Code
```python
class Solution:
    def canThreePartsEqualSum(self, arr: List[int]) -> bool:
        total = sum(arr)
        if total % 3 != 0:
            return False
        target = total // 3
        n = len(arr)
        if n < 3:
            return False
        
        prefix = [0] * n
        prefix[0] = arr[0]
        for i in range(1, n):
            prefix[i] = prefix[i-1] + arr[i]
        
        i_index = -1
        for i in range(n):
            if prefix[i] == target:
                i_index = i
                break
        
        if i_index == -1 or i_index >= n-1:
            return False
        
        for j in range(i_index+1, n-1):
            if prefix[j] == 2 * target:
                return True
        
        return False
```

### Explanation
1. **Total Sum Check**: The code first checks if the total sum of the array is divisible by 3. If not, it immediately returns False since equal splitting is impossible.
2. **Prefix Sum Array**: The prefix sum array is constructed to store cumulative sums up to each index. This allows efficient range sum queries.
3. **First Split Index**: The loop searches for the first index where the prefix sum equals one-third of the total sum. This index signifies the end of the first part.
4. **Second Split Index**: Another loop checks if there exists an index after the first split where the prefix sum equals two-thirds of the total sum. This ensures the second part ends correctly, leaving the third part to sum to the remaining one-third.
5. **Edge Handling**: If the array length is less than 3 or no valid splits are found, the function returns False. Otherwise, it returns True upon finding valid splits for all three parts.

This approach efficiently checks for the required conditions using prefix sums, ensuring optimal performance with a linear pass through the array.
