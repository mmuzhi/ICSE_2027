=== RAW ANSWER ===

To solve this problem, we need to generate a list of numbers from 1 to n in lexicographical order. Lexicographical order for numbers is determined by comparing their string representations. For example, the number 1 comes before 10 because '1' is lexicographically less than '10'.

### Approach
1. **Problem Analysis**: The task requires generating numbers from 1 to n and sorting them lexicographically. The key observation here is that lexicographical order for numbers can be achieved by sorting their string representations.
2. **Intuition**: By converting each number to a string and then sorting the strings, we can obtain the desired lexicographical order. This approach works because string comparison in Python naturally orders numbers in the required way (e.g., '1' < '10' < '2').
3. **Algorithm Selection**: The algorithm involves:
   - Generating a list of strings from 1 to n.
   - Sorting this list lexicographically.
   - Converting the sorted strings back to integers.
4. **Complexity Analysis**: The algorithm involves generating a list of n elements, sorting them, which takes O(n log n) time, and converting each element from string to integer, which is O(n). Thus, the overall time complexity is O(n log n). The space complexity is O(n) to store the list of strings and integers.

### Solution Code
```python
class Solution:
    def lexicalOrder(self, n: int) -> List[int]:
        lst = [str(i) for i in range(1, n+1)]
        lst.sort()
        return [int(x) for x in lst]
```

### Explanation
1. **Generating Strings**: The list comprehension `[str(i) for i in range(1, n+1)]` creates a list of string representations of numbers from 1 to n. This ensures that numbers are compared as strings.
2. **Sorting**: The `sort()` method sorts the list of strings lexicographically, which corresponds to the desired lexicographical order for numbers.
3. **Converting Back to Integers**: The list comprehension `[int(x) for x in lst]` converts each sorted string back to an integer, resulting in the final list of numbers in lexicographical order.

This approach efficiently leverages Python's string comparison to achieve the required lexicographical ordering of numbers.
