To solve this problem, we need to find all starting indices of substrings in a given string `s` that are formed by concatenating each word in the list `words` exactly once and without any intervening characters. The solution involves checking every possible substring of the required length and verifying if it matches the concatenation of the words in `words` (each appearing exactly once).

### Approach
1. **Problem Analysis**: The problem requires us to find all starting positions in `s` where a substring of length equal to the total length of all words (when concatenated) contains each word from `words` exactly once. The words in the substring must appear in any order, but each word must appear exactly once.
2. **Intuition**: For each possible starting index in `s`, we can extract a substring of length equal to the sum of all word lengths. We then break this substring into segments of length equal to the individual word lengths and check if these segments exactly match the words in `words`.
3. **Algorithm Selection**: We use a sliding window approach combined with a frequency map (or counter) to efficiently check each substring. For each starting index, we generate the substring, split it into words of the given length, and compare the frequency of these words with the frequency of words in `words`.
4. **Complexity Analysis**: The algorithm involves iterating over each possible starting index in `s` (O(n) where n is the length of `s`). For each starting index, we process a substring of fixed length (O(m) where m is the total length of all words combined). Thus, the overall complexity is O(n * m), which is efficient for typical input sizes.

### Solution Code
```python
import collections
from typing import List

class Solution:
    def findSubstring(self, s: str, words: List[str]) -> List[int]:
        if not s or not words:
            return []
        word_len = len(words[0])
        total_len = word_len * len(words)
        n = len(s)
        if n < total_len:
            return []
        occ = collections.Counter(words)
        res = []
        for start in range(0, n - total_len + 1):
            window_words = []
            for i in range(len(words)):
                end_index = start + i * word_len + word_len
                word = s[start + i * word_len:end_index]
                window_words.append(word)
            window_freq = collections.Counter(window_words)
            if window_freq == occ:
                res.append(start)
        return res
```

4. **Frequency Map Construction**: A frequency map (`occ`) is created for the words in `words` using `collections.Counter`.
5. **Sliding Window Check**: For each starting index in `s`, the substring of length `total_len` is extracted. This substring is split into individual words of length `word_len`. The frequency of these words is compared with `occ`. If they match, the starting index is added to the result list.

This approach efficiently checks all possible substrings by leveraging a sliding window and frequency comparison, ensuring correctness while maintaining optimal performance.
