To solve this problem, we need to compute the sum of the minimum values of all contiguous subarrays in a given array. The challenge lies in efficiently calculating this sum without resorting to a brute-force approach, which would be computationally expensive.

### Approach
1. **Problem Analysis**: The task requires summing the minimum values of every possible contiguous subarray in the array. A brute-force approach would involve checking every possible subarray, which is inefficient for large arrays. Instead, we use a more efficient approach leveraging monotonic stacks to find the left and right boundaries for each element where it is the minimum.

2. **Intuition**: For each element in the array, determine the number of subarrays where it is the minimum. This can be done by finding the nearest greater elements to the left and right. The number of such subarrays for an element at index `i` is given by `(i - left_bound[i]) * (right_bound[i] - i)`, where `left_bound[i]` is the index of the previous element greater than or equal to `arr[i]`, and `right_bound[i]` is the index of the next element greater than `arr[i]`.

3. **Algorithm Selection**: We use two passes with a monotonic stack:
   - **Left Boundaries**: Traverse the array from left to right, maintaining a stack of indices. For each element, pop elements from the stack that are greater than the current element to determine the left boundary.
   - **Right Boundaries**: Traverse the array from right to left, similarly using a stack to determine the right boundary for each element.
   - **Sum Calculation**: For each element, compute its contribution to the total sum using its left and right boundaries.

4. **Complexity Analysis**: The algorithm runs in O(n) time for both passes (left and right boundaries) and uses O(n) space for the boundaries and stack. This is efficient compared to the brute-force O(n^2) approach.

### Solution Code
```python
class Solution:
    def sumSubarrayMins(self, arr: List[int]) -> int:
        MOD = 10**9 + 7
        n = len(arr)
        left_bound = [-1] * n
        right_bound = [n] * n
        stack = []
        
        # Compute left boundaries
        for i in range(n):
            while stack and arr[stack[-1]] > arr[i]:
                idx = stack.pop()
                left_bound[idx] = i
            stack.append(i)
        
        stack = []
        # Compute right boundaries
        for i in range(n-1, -1, -1):
            while stack and arr[stack[-1]] >= arr[i]:
                idx = stack.pop()
                right_bound[idx] = i
            stack.append(i)
        
        res = 0
        for i in range(n):
            res = (res + arr[i] * (i - left_bound[i]) * (right_bound[i] - i)) % MOD
        
        return res
```

1. **Initialization**: We initialize `left_bound` and `right_bound` arrays to store the indices of the boundaries for each element. The `MOD` constant ensures the result is within the required range.
2. **Left Boundaries Calculation**: Using a stack, we traverse the array from left to right. For each element, we pop elements from the stack that are greater than the current element, setting their left boundary to the current index. This helps in determining how many elements to the left are greater, thus defining the subarrays where the current element is the minimum.
3. **Right Boundaries Calculation**: Similarly, we traverse the array from right to left, popping elements from the stack that are greater than or equal to the current element. This sets the right boundary for each element, indicating how many elements to the right are greater, thus defining the subarrays where the current element is the minimum.
4. **Sum Calculation**: For each element, we compute its contribution to the total sum by multiplying its value with the number of subarrays where it is the minimum (determined by its left and right boundaries). The result is accumulated modulo \(10^9 + 7\).

This approach efficiently computes the desired sum by leveraging stack operations to avoid the O(n^2) complexity of a brute-force solution.
