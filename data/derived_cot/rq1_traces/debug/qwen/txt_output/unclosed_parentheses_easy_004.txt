To solve this problem, we need to generate Pascal's Triangle up to a specified number of rows. Pascal's Triangle is a triangular array of binomial coefficients where each number is the sum of the two numbers directly above it. The first row is `[1]`, and each subsequent row is built by adding adjacent elements from the previous row.

### Approach
1. **Base Case Handling**: If the number of rows requested is 1, return a list containing just the first row `[1]`.
2. **Recursive Construction**: For more than one row, recursively generate the triangle for `numRows - 1` rows. The last row of this smaller triangle is used to construct the next row.
3. **Row Construction**: The next row starts and ends with 1. Each intermediate element is the sum of the two elements directly above it in the previous row.
4. **Appending Rows**: The newly constructed row is appended to the list of rows from the recursive call, and the updated list is returned.

### Solution Code
```python
class Solution:
    def generate(self, numRows: int) -> List[List[int]]:
        if numRows == 1:
            return [[1]]
        prev = self.generate(numRows - 1)
        fin = prev[-1]
        now = [1]
        for i in range(len(fin) - 1):
            now.append(fin[i] + fin[i+1])
        now.append(1)
        prev.append(now)
        return prev
```

### Explanation
1. **Base Case**: The function checks if `numRows` is 1. If so, it returns a list containing the first row `[1]`.
2. **Recursive Call**: For `numRows > 1`, the function calls itself with `numRows - 1` to get the triangle up to the previous row.
3. **Building the Current Row**: The last row from the recursive call (`prev[-1]`) is used to construct the current row. The new row starts with 1, and each subsequent element is the sum of adjacent elements from the previous row. The row ends with another 1.
4. **Appending and Returning**: The newly constructed row is appended to the list of rows from the recursive call, and this updated list is returned as the result.

This approach efficiently builds Pascal's Triangle by leveraging recursion and simple arithmetic operations, ensuring correctness and clarity.
