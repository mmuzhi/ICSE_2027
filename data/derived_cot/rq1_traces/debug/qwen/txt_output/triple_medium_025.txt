The solution involves traversing the graph from both nodes to compute their distances to all reachable nodes, then determining the node with the smallest maximum distance from both nodes.

### Approach
1. **Graph Traversal**: We perform two separate traversals starting from node1 and node2. During each traversal, we use a BFS-like approach to compute the shortest distances from the starting node to all reachable nodes. The graph is represented by an array where each entry points to the next node or -1 if there is no outgoing edge.
2. **Distance Calculation**: For each node, we maintain two distances—one from node1 and one from node2. After both traversals, we check each node to see if it is reachable from both nodes.
3. **Finding the Closest Node**: For each node reachable from both nodes, we compute the maximum of the two distances. The node with the smallest such maximum distance is our answer. If multiple nodes have the same maximum distance, we choose the smallest node index.

class Solution:
    def cdist(self, it, distArray, distArrayIndex, edges):
        rdist = 0
        while it != -1 and distArray[it][distArrayIndex] > rdist:

            it = edges[it]
            rdist += 1
            
    def closestMeetingNode(self, edges: List[int], node1: int, node2: int) -> int:
