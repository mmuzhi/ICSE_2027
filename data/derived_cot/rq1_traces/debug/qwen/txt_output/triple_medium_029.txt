To solve this problem, we need to restore all possible valid IP addresses from a given string by inserting dots into the string to form four parts. Each part must be a number between 0 and 255, and parts starting with '0' must be exactly '0' (no leading zeros allowed).

### Approach
1. **Backtracking with Pruning**: We use a backtracking approach to explore all possible ways to split the string into four parts. The algorithm checks each possible split and ensures that the parts are valid (i.e., they are numbers between 0 and 255 and do not have leading zeros unless the part is '0').
2. **Base Case Handling**: The base case is when the entire string is processed (i.e., the current index reaches the end of the string) and exactly four parts have been formed. At this point, the valid IP address is added to the result.
3. **Pruning Invalid Paths**: The algorithm prunes paths where more than four parts are formed or the current index exceeds the string length, ensuring efficiency and correctness.

### Solution Code
```python
class Solution:
    def restoreIpAddresses(self, s: str) -> List[str]:
        res = []
        cur = []
        
        def backtrack(i):
            if i == len(s) and len(cur) == 4:
                res.append(".".join(cur))
                return
            if len(cur) > 4 or i > len(s):
                return
            if s[i] == '0':
                cur.append('0')
                backtrack(i + 1)
                cur.pop()
                return
            j = 0
            while j < 3 and i + j < len(s):
                num = int(s[i:i+j+1])
                if num < 256:
                    cur.append(s[i:i+j+1])
                    backtrack(i + j + 1)
                    cur.pop()
                j += 1
        
        backtrack(0)
        return res
```
