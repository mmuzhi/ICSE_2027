To solve this problem, we need to find the longest substring within a given string that does not contain any of the forbidden substrings. The solution involves using a trie data structure to efficiently check for the presence of forbidden substrings within any given substring of the input string.

### Approach
1. **Trie Construction**: We first construct a trie from the list of forbidden substrings. Each node in the trie represents a character, and an "end" marker is used to denote the completion of a forbidden substring.
2. **Helper Function**: We create a helper function `containsForbidden` that checks if a given substring contains any forbidden substring. This function iterates over each starting position in the substring and uses the trie to see if any forbidden substring starts at that position.
3. **Sliding Window**: For each starting index in the input string, we extend the window to the right as much as possible without including any forbidden substring. The maximum length of such valid substrings is tracked throughout the process.

### Solution Code
```python
class Solution:
    def longestValidSubstring(self, word: str, forbidden: List[str]) -> int:
        if not word:
            return 0
        
        trie = {}
        for f in forbidden:
            t = trie
            for c in f:
                if c not in t:
                    t[c] = {}
                t = t[c]
            t["end"] = True
        
        def containsForbidden(s):
            n = len(s)
            if n == 0:
                return False
            for start in range(n):
                t = trie
                matched = 0
                for i in range(start, n):
                    c = s[i]
                    if c not in t:
                        break
                    t = t[c]
                    matched += 1
                    if "end" in t:
                        return True
            return False
        
        n = len(word)
        max_len = 0
        for i in range(n):
            j = i
            while j < n:
                if not containsForbidden(word[i:j+1]):
                    j += 1
                else:
                    break
            max_len = max(max_len, j - i)
        return max_len
```
