To solve this problem, we need to compute the sum of the minimum values of all contiguous subarrays in a given array. The challenge lies in efficiently calculating this sum without resorting to a brute-force approach, which would be computationally expensive.

### Approach
1. **Problem Analysis**: The problem requires summing the minimum values of all possible contiguous subarrays in an array. A brute-force approach would involve checking every possible subarray, which is O(n^2) time complexity and not feasible for large arrays.

2. **Intuition**: The key insight is to use a monotonic increasing stack to efficiently compute the left and right boundaries for each element where it is the minimum. For each element in the array, the number of subarrays where it is the minimum can be determined by its nearest smaller elements on both sides.

3. **Algorithm Selection**: We use two passes over the array:
   - **First Pass**: Compute the left boundary (previous smaller element) for each element using a monotonic increasing stack.
   - **Second Pass**: Compute the right boundary (next smaller element) for each element by traversing the array from right to left.
   - **Sum Calculation**: For each element, multiply its value by the number of subarrays where it is the minimum (calculated using the left and right boundaries) and accumulate the result.

4. **Complexity Analysis**: The algorithm runs in O(n) time for each pass, resulting in an overall O(n) time complexity. The space complexity is O(n) due to the stack usage.

### Solution Code
```python
class Solution:
    def sumSubarrayMins(self, arr: List[int]) -> int:
        MOD = 10**9 + 7
        n = len(arr)
        left_bound = [-1] * n
        right_bound = [n] * n
        
        stack = []
        for i in range(n):
            while stack and arr[stack[-1]] > arr[i]:
                stack.pop()
            if stack:
                left_bound[i] = stack[-1]
            else:
                left_bound[i] = -1
            stack.append(i)
        
        stack = []
        for i in range(n-1, -1, -1):
            while stack and arr[stack[-1]] > arr[i]:
                stack.pop()
            if stack:
                right_bound[i] = stack[-1]
            else:
                right_bound[i] = n
            stack.append(i)
        
        res = 0
        for i in range(n):
            res = (res + arr[i] * (i - left_bound[i]) * (right_bound[i] - i)) % MOD
        
        return res
```

3. **Right Boundary Calculation**: Similarly, we traverse the array from right to left, popping elements from the stack that are greater than the current element to find the nearest smaller element on the right.
