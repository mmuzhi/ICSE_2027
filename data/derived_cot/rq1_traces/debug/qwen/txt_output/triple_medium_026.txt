=== RAW ANSWER ===

To solve this problem, we need to determine the minimum number of steps required to transform string `s` into string `t` by only adding characters. Each addition of a character counts as one step. The solution involves comparing the character frequencies in both strings and calculating the deficit of characters in `s` compared to `t`.

### Approach
1. **Frequency Counting**: Use the `Counter` class from the `collections` module to count the occurrences of each character in both strings `s` and `t`.
2. **Deficit Calculation**: For each character in `t`, compute the difference between its count in `t` and its count in `s`. If `s` has fewer occurrences of a character than `t`, the deficit is the difference; otherwise, the deficit is zero.
3. **Summing Deficits**: Sum all the deficits across all characters to get the total number of steps required. This sum represents the number of characters that need to be added to `s` to match the frequency of characters in `t`.

### Solution Code
```python
from collections import Counter

class Solution:
    def minSteps(self, s: str, t: str) -> int:
        cnt1 = Counter(s)
        cnt2 = Counter(t)
        return sum((cnt2 - cnt1).values())
```

### Explanation
1. **Frequency Counting**: The `Counter` objects `cnt1` and `cnt2` store the counts of each character in strings `s` and `t`, respectively.
2. **Deficit Calculation**: The expression `(cnt2 - cnt1)` computes the difference in character counts between `t` and `s`. For each character in `t`, if `s` has fewer occurrences, the difference is positive; otherwise, it is zero or negative (in which case it is treated as zero in the subtraction).
3. **Summing Deficits**: The `sum` function calculates the total number of characters that need to be added to `s` to match the frequency of characters in `t`, which is the result returned by the method.

This approach efficiently computes the minimum steps by leveraging the `Counter` class to handle frequency counts and arithmetic operations on these counts, ensuring optimal performance and correctness.
