2. **Intuition**: For each row, we can check if the length of the row equals the length of the set of its elements. If they are not equal, duplicates exist. Similarly, for each column, we collect all elements in that column and check for duplicates by comparing the length of the column with the length of the set of its elements.
3. **Algorithm Selection**: The solution involves iterating over each row and column, using sets to check for uniqueness. Sets automatically remove duplicates, so comparing the length of the original list with the length of the set will reveal any duplicates.
4. **Complexity Analysis**: The algorithm runs in O(n*m) time, where n is the number of rows and m is the number of columns. This is because we process each element once for rows and once for columns. The space complexity is O(m) for rows and O(n) for columns, but since these are temporary sets, the overall space complexity is O(m + n).

### Solution Code
```python
class Solution:
    def checkValid(self, matrix: List[List[int]]) -> bool:
        n = len(matrix)
        m = len(matrix[0])
        
        for i in range(n):
            if len(set(matrix[i])) != m:
                return False
        
        for j in range(m):
            col = [matrix[i][j] for i in range(n)]
            if len(set(col)) != n:
                return False
        
        return True
```

### Explanation
1. **Row Check**: For each row in the matrix, we convert the row into a set. If the length of this set is less than the number of columns, it means there are duplicate elements in that row, and we return `False`.
2. **Column Check**: For each column, we collect all elements in that column into a list. We then convert this list into a set. If the length of the set is less than the number of rows, it indicates duplicate elements in that column, leading us to return `False`.
3. **Result**: If all rows and columns pass the uniqueness check, the function returns `True`, indicating the matrix is valid according to the specified conditions.

This approach efficiently checks for uniqueness in both rows and columns using set operations, ensuring correctness while maintaining optimal performance.
