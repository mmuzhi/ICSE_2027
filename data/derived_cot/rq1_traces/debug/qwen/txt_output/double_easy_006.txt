=== RAW ANSWER ===

To solve this problem, we need to determine if all the elements in an array occur with distinct frequencies. The solution involves counting the occurrences of each element in the array and then checking if these counts are all unique.

### Approach
1. **Count Frequencies**: Use the `Counter` class from the `collections` module to count the occurrences of each element in the array.
2. **Check Uniqueness of Frequencies**: Convert the values (frequencies) from the `Counter` object into a set. If the length of the set is equal to the length of the original frequency list, all frequencies are unique. Otherwise, there are duplicates.

### Solution Code
```python
from collections import Counter

class Solution:
    def uniqueOccurrences(self, arr: List[int]) -> bool:
        d = Counter(arr)
        freqs = list(d.values())
        return len(freqs) == len(set(freqs))
```

### Explanation
1. **Counting Frequencies**: The `Counter` class efficiently counts how many times each element appears in the array. For example, if the input array is `[1, 2, 2, 3, 3, 3]`, the `Counter` will produce `{1:1, 2:2, 3:3}`.
2. **Checking Uniqueness**: By converting the values of the `Counter` into a set, we eliminate any duplicate counts. If the length of this set matches the length of the original frequency list, it means all frequencies are unique. For instance, in the example above, the set of frequencies `{1, 2, 3}` has the same length as the list `[1, 2, 3]`, so the function returns `True`. If there were duplicate frequencies, such as `[1, 1, 2, 2]` which gives frequencies `[2, 2]`, the set would be `{2}` with length 1, which is less than the original list length 2, so the function returns `False`.

This approach efficiently checks for unique frequencies using basic Python operations, ensuring correctness and simplicity.
