To solve this problem, we need to find the second greater element for each element in the given array. The second greater element for an element at index `i` is defined as the first element to the right of `i` that is greater than the element at `i`, and then the first element to the right of that element that is greater than it. If no such element exists, the result for that index should be -1.

### Approach
1. **Problem Analysis**: The problem requires us to find, for each element in the array, the second element to the right that is greater than it. This can be broken down into two steps:
   - First, find the immediate next greater element (NGE) for each element.
   - Then, for each element, find the NGE of its immediate next greater element to get the second greater element.

2. **Intuition**: The solution involves two passes over the array. In the first pass, we compute the immediate next greater element for each element using a stack. In the second pass, we use the results from the first pass to determine the second greater element for each element.

3. **Algorithm Selection**: 
   - **First Pass**: Use a stack to keep track of indices of elements for which we haven't found the immediate next greater element yet. For each element, pop elements from the stack that are smaller than the current element and record their immediate next greater element as the current element's index.
   - **Second Pass**: For each element, if it has an immediate next greater element, check if that greater element also has a next greater element. If so, the second greater element is the value of that next greater element; otherwise, it remains -1.

4. **Complexity Analysis**: The algorithm processes each element twice (once in each pass), resulting in a time complexity of O(n). The space complexity is O(n) due to the storage required for the stack and the result array.

    def secondGreaterElement(self, nums: List[int]) -> List[int]:
