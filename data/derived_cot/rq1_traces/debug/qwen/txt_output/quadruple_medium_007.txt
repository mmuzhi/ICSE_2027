=== RAW ANSWER ===

To solve this problem, we need to perform a level-order traversal (BFS) of a binary tree and return the nodes' values in a bottom-up manner. This means the deepest level should appear first in the result, followed by the next level, and so on. 

### Approach
1. **Problem Analysis**: The task involves traversing each level of the binary tree from left to right and collecting the node values. The challenge is to reverse the order of these levels so that the deepest level appears first.
2. **Intuition**: We can use a queue to facilitate the BFS traversal. For each level, we process all nodes in the queue, collect their values, and then append these values to the result list. After traversing all levels, we reverse the result list to achieve the bottom-up order.
3. **Algorithm Selection**: 
   - **BFS Traversal**: We start from the root node and process each level by dequeuing all nodes in the current level, collecting their values, and enqueuing their children (if they exist) for the next level.
   - **Reversing Levels**: After collecting all levels in the order they are traversed (top to bottom), we reverse the entire list of levels to present them from bottom to top.
4. **Complexity Analysis**: The time complexity is O(n) where n is the number of nodes in the tree, as each node is processed exactly once. The space complexity is also O(n) due to the storage required for the queue and the result list.

### Solution Code
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right

class Solution:
    def levelOrderBottom(self, root: Optional[TreeNode]) -> List[List[int]]:
        def checkReverse(ans):
            l, r = 0, len(ans) - 1
            while l < r:
                ans[l], ans[r] = ans[r], ans[l]
                l += 1
                r -= 1
            return ans
        
        if not root:
            return []
        
        q, ans = [root], []
        while q:
            n = len(q)
            level = []
            for i in range(n):
                curr = q.pop(0)
                level.append(curr.val)
                if curr.left is not None:
                    q.append(curr.left)
                if curr.right is not None:
                    q.append(curr.right)
            ans.append(level)
        
        return self.checkReverse(ans)
```

### Explanation
1. **Initial Check**: The function first checks if the root is `None`, returning an empty list if true.
2. **BFS Initialization**: A queue is initialized with the root node, and an empty list `ans` is prepared to store each level's node values.
3. **Level Processing**: For each level, the number of nodes is determined by the queue's length. Each node is dequeued, its value is added to the current level list, and its children (if they exist) are enqueued for processing in the next level.
4. **Reversing Levels**: After processing all levels, the `checkReverse` function reverses the order of the levels in `ans` to present the deepest level first.
5. **Result Return**: The reversed list of levels is returned, providing the bottom-up level-order traversal of the binary tree. 

This approach efficiently handles the BFS traversal and ensures the levels are presented in the required bottom-up order.
