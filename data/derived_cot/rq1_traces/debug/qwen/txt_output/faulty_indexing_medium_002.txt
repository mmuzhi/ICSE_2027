=== RAW ANSWER ===

To solve this problem, we need to generate all numbers within a given range [low, high] that consist of sequential digits. Sequential digits mean each digit in the number is exactly one more than the previous digit. For example, 123 and 56789 are valid, but 102 and 111 are not.

### Approach
The approach involves using a breadth-first search (BFS) strategy to generate all possible sequential digit numbers. We start with each digit from 1 to 9 and iteratively build longer numbers by appending the next digit (current last digit + 1) until the last digit is 9, which cannot be incremented further. Each generated number is checked to see if it falls within the specified range [low, high]. If it does, it is added to the result list.

### Solution Code
```python
from collections import deque

class Solution:
    def sequentialDigits(self, low: int, high: int) -> List[int]:
        out = []
        queue = deque(range(1, 10))
        while queue:
            elem = queue.popleft()
            if low <= elem <= high:
                out.append(elem)
            last_digit = elem % 10
            if last_digit < 9:
                new_elem = elem * 10 + (last_digit + 1)
                queue.append(new_elem)
        return out
```

### Explanation
1. **Initialization**: We start by initializing a queue with digits from 1 to 9 (range(1, 10)), ensuring we begin with single-digit numbers.
2. **BFS Processing**: For each number dequeued, we check if it lies within the range [low, high]. If it does, the number is added to the result list.
3. **Generating Next Numbers**: For each number, we determine its last digit. If this digit is less than 9, we generate the next sequential number by appending the next digit (last digit + 1) to the current number. This new number is then enqueued for further processing.
4. **Termination**: The process continues until the queue is empty, ensuring all possible sequential digit numbers within the range are generated and collected.

This approach efficiently explores all possible sequential digit numbers using BFS, ensuring that all valid numbers within the specified range are found. The use of a queue ensures that numbers are processed in the order they are generated, maintaining the correct sequence in the output.
