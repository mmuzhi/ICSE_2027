To solve this problem, we need to determine the maximum number of times we can repeat the string `s2` by using characters from the string `s1` repeated `n1` times, while preserving the order of characters in `s2`. The solution involves efficiently counting the occurrences of `s2` as a subsequence in the concatenated string `s1` repeated `n1` times without explicitly constructing the entire concatenated string, which could be very large.

### Approach
1. **Initial Checks**: 
   - If `s2` contains any characters not present in `s1`, it's impossible to form `s2`, so return 0.
   - Filter `s1` to include only characters present in `s2` to simplify the matching process.
   - If the filtered `s1` is empty and `s2` is not empty, return 0 since no valid `s2` can be formed.

2. **Precompute Positions**:
   - For each character in `s2`, precompute the list of positions where it appears in the filtered `s1`. This helps in quickly finding the next occurrence of each character during the subsequence matching.

3. **State Machine with Jump Function**:
   - Use a state machine to traverse the concatenated string (conceptually `s1` repeated `n1` times) without explicitly constructing it.
   - For each character in `s2`, use binary search to find the next occurrence in the remaining part of the string. This ensures efficient jumping through the string, avoiding linear scans.
   - Count the number of complete `s2` sequences formed by tracking the progress through `s2` and resetting when a complete sequence is found.

### Solution Code
```python
import bisect
from collections import defaultdict

class Solution:
    def getMaxRepetitions(self, s1: str, n1: int, s2: str, n2: int) -> int:
        if not s2:
            return 0
        
        if not set(s2).issubset(set(s1)):
            return 0
        
        s1_filtered = ''.join(char for char in s1 if char in set(s2))
        if not s1_filtered:
            return 0
        
        L = len(s1_filtered)
        total_length = n1 * L
        
        positions = {}
        for char in set(s2):
            positions[char] = [i for i, c in enumerate(s1_filtered) if c == char]
        
        j = 0
        count = 0
        current_position = 0
        
        while current_position < total_length:
            if j < len(s2):
                c = s2[j]
                if c not in positions:
                    break
                arr = positions[c]
                copy_index = current_position // L
                index_in_copy = current_position % L
                
                if copy_index >= n1:
                    break
                
                idx = bisect.bisect_left(arr, index_in_copy)
                if idx < len(arr):
                    next_pos = copy_index * L + arr[idx]
                    current_position = next_pos + 1
                    j += 1
                else:
                    if copy_index == n1 - 1:
                        break
                    next_pos = (copy_index + 1) * L + arr[0]
                    current_position = next_pos + 1
                    j += 1
            else:
                count += 1
                j = 0
```
