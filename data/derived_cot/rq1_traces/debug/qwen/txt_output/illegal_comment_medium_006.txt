To solve this problem, we need to count the number of 3x3 magic squares present inside a given grid. A 3x3 magic square is defined as a 3x3 grid where the numbers 1 through 9 are used exactly once, and the sums of the numbers in each row, column, and both main diagonals are all equal to 15.

### Approach
1. **Problem Analysis**: The task involves checking all possible 3x3 subgrids within the given grid to determine if they form a magic square. A magic square must meet two conditions:
   - All numbers in the subgrid must be distinct and range from 1 to 9.
   - The sum of each row, column, and the two main diagonals must be 15.

2. **Intuition**: For each possible starting position (i, j) in the grid where a 3x3 subgrid can be formed, we extract the subgrid and verify if it meets the magic square conditions. This involves:
   - Checking that all numbers in the subgrid are unique and within the range 1-9.
   - Verifying the sums of all rows, columns, and diagonals are 15.

3. **Algorithm Selection**: The solution involves iterating over each possible top-left corner of a 3x3 subgrid. For each subgrid, we:
   - Collect all elements into a list.
   - Check if the elements are exactly the numbers 1 through 9.
   - Verify the sum of each row, column, and diagonal is 15.

4. **Complexity Analysis**: The algorithm checks all possible 3x3 subgrids in the grid. If the grid has dimensions M x N, the number of subgrids is (M - 2) * (N - 2). For each subgrid, the checks involve O(1) operations (since the subgrid size is fixed). Thus, the overall complexity is O(M*N), which is efficient for typical grid sizes.

### Solution Code
```python
class Solution:
    def numMagicSquaresInside(self, grid: List[List[int]]) -> int:
        M = len(grid)
        N = len(grid[0])
        res = 0
        
        if M < 3 or N < 3:
            return 0
        
        for i in range(M - 2):
            for j in range(N - 2):
                square = []
                for x in range(3):
                    square.append(grid[i + x][j:j + 3])
                
                nums = []
                for row in square:
                    nums.extend(row)
                
                if set(nums) != set(range(1, 10)):
                    continue
                
                if sum(square[0]) != 15 or sum(square[1]) != 15 or sum(square[2]) != 15:
                    continue
                
                if square[0][0] + square[1][0] + square[2][0] != 15:
                    continue
                if square[0][1] + square[1][1] + square[2][1] != 15:
                    continue
                if square[0][2] + square[1][2] + square[2][2] != 15:
                    continue
                
                if square[0][0] + square[1][1] + square[2][2] != 15:
                    continue
                if square[0][2] + square[1][1] + square[2][0] != 15:
                    continue
                
                res += 1
        
        return res
```
