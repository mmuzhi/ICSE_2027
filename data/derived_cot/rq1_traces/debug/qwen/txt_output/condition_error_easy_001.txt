=== RAW ANSWER ===

To solve this problem, we need to find the third maximum distinct number in a given list of integers. If there are fewer than three distinct numbers, we should return the maximum number from the list. 

### Approach
1. **Problem Analysis**: The task is to identify the third largest unique number in the list. If there are fewer than three unique numbers, the solution should return the largest number in the list.
2. **Intuition**: By first removing duplicates from the list, we can focus on the distinct numbers. Sorting these distinct numbers in descending order allows us to easily access the third largest number if it exists. If there are fewer than three distinct numbers, the largest number (the first in the sorted list) is returned.
3. **Algorithm Selection**: The algorithm involves converting the list to a set to eliminate duplicates, sorting the set in descending order, and then checking the length of the sorted set. If the length is less than 3, return the first element; otherwise, return the third element.
4. **Complexity Analysis**: The time complexity is dominated by the sorting step, which is O(n log n) where n is the number of distinct elements in the list. The space complexity is O(n) for storing the distinct elements.

### Solution Code
```python
class Solution:
    def thirdMax(self, nums: List[int]) -> int:
        distinct_nums = sorted(set(nums), reverse=True)
        if len(distinct_nums) < 3:
            return distinct_nums[0]
        else:
            return distinct_nums[2]
```

### Explanation
1. **Removing Duplicates**: The set conversion ensures that only unique numbers are considered.
2. **Sorting**: The distinct numbers are sorted in descending order to easily access the largest, second largest, and third largest numbers.
3. **Condition Check**: If there are fewer than three distinct numbers, the largest number (first element in the sorted list) is returned. Otherwise, the third largest number (third element in the sorted list) is returned.
This approach efficiently handles both scenarios where there are three or more distinct numbers and cases with fewer than three distinct numbers, providing the correct result in each case.
