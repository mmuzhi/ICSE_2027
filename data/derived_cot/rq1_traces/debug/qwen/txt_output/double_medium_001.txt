To solve this problem, we need to determine if a given string containing parentheses '(', ')', and wildcards '*' is valid. The wildcards can be used as either '(' or ')' or as an empty string. The solution involves using two stacks to keep track of the indices of '(' and '*' respectively, ensuring that all closing parentheses are properly matched.

### Approach
1. **Initialization**: We use two stacks, `st` for open parentheses '(' and `sr` for wildcards '*'.
2. **First Pass**: Traverse the string:
   - For each '(', push its index onto `st`.
   - For each '*', push its index onto `sr`.
   - For each ')', try to match it with the most recent '(' (pop from `st`). If not available, match it with the most recent wildcard (pop from `sr`). If neither is available, return `False`.
3. **Second Pass**: After the first pass, if there are unmatched '(' in `st`, match each with a wildcard in `sr` that appears after it. This is done by comparing the indices of the top elements of both stacks. If a wildcard is found after the corresponding '(', they are matched; otherwise, the string is invalid.

class Solution:
    def checkValidString(self, s):
        st = []

   - ')' is matched with the top of `st` if available; otherwise, with the top of `sr`. If neither is available, the string is invalid.
3. **Second Pass**: The loop checks if any unmatched '(' can be matched with a wildcard that appears later in the string. If a wildcard is found after the corresponding '(', they are matched; otherwise, the string is invalid.
4. **Result**: If all parentheses and wildcards are properly matched, the function returns `True`; otherwise, it returns `False`.
