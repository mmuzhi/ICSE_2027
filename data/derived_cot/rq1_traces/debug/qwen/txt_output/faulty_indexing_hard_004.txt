### Approach
1. **Dynamic Programming Setup**: We use a dynamic programming array `dp` where `dp[i]` represents the number of arithmetic slices ending at index `i`. This is initialized to zero for all elements.
2. **Base Case Handling**: If the array has fewer than two elements, return zero since no arithmetic slices can exist.
3. **Iterate Through the Array**: For each index `i` starting from 2, check if the difference between the current element and the previous element is the same as the difference between the previous element and the element before it. If true, it means the current element can extend the arithmetic slices ending at the previous index. The value of `dp[i]` is then set to `dp[i-1] + 1`, where `dp[i-1]` accounts for all slices ending at the previous index, and `+1` accounts for the new two-element slice ending at the current index.
4. **Summing Up**: The total number of arithmetic slices is the sum of all values in the `dp` array from index 2 onwards.

class Solution:
    def numberOfArithmeticSlices(self, nums: List[int]) -> int:
