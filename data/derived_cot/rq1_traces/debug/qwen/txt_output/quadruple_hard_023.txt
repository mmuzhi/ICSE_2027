### Approach
1. **Trie Construction**: Build a trie from all forbidden substrings. Each node in the trie represents a character, and an "end" marker is used to denote the end of a forbidden substring.
2. **Forbidden Substring Check**: For each substring of the word starting at index `i` and ending at index `j-1`, use a helper function `isForbidden` to check if any forbidden substring exists within it. The helper function traverses the trie character by character. If it encounters an "end" marker during traversal, it returns the length of the forbidden substring found. If no forbidden substring is found, it returns `False`.
3. **Sliding Window**: Iterate over the word from right to left, adjusting the right boundary `j` whenever a forbidden substring is detected. The goal is to find the longest valid substring by ensuring that no forbidden substrings are present within the current window `[i, j-1]`.
