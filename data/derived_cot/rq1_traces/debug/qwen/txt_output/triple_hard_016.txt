=== RAW ANSWER ===

To solve this problem, we need to count the number of ways to place `n` queens on an `n x n` chessboard such that no two queens attack each other. The solution involves using a backtracking approach with three auxiliary arrays to keep track of the positions of queens to avoid conflicts.

### Approach
1. **Problem Analysis**: The problem requires placing `n` queens on an `n x n` chessboard such that no two queens share the same row, column, or diagonal. The challenge is to efficiently explore all possible configurations without conflicts.
2. **Intuition**: The backtracking approach is suitable here because it systematically explores all possible placements of queens, placing one queen at a time and backtracking if a conflict is found. Three arrays are used to track:
   - **Rows**: `low` array checks if a row is already occupied.
   - **Main Diagonals**: `upper` array checks the main diagonals (top-left to bottom-right) using the formula `n-1 + col - row`.
   - **Anti-Diagonals**: `lower` array checks the anti-diagonals (top-right to bottom-left) using the formula `row + col`.
3. **Algorithm Selection**: The backtracking algorithm recursively tries to place a queen in each column, checking for conflicts using the auxiliary arrays. If a valid position is found, it proceeds to the next column; otherwise, it backtracks.
4. **Complexity Analysis**: The time complexity is O(n!) in the worst case, as the algorithm explores all permutations of queen placements. The space complexity is O(n) for the auxiliary arrays and the recursion stack.

### Solution Code
```python
class Solution:
    def totalNQueens(self, n: int) -> int:
        def addans(board, ans):
            temp = []
            for row in board:
                for j in range(len(row)):
                    if row[j] == "Q":
                        temp.append(j + 1)
            ans.append(temp)
        
        def solve(col, board, low, upper, lower, ans, n):
            if col == n:
                addans(board, ans)
                return
            for row in range(n):
                if low[row] == 0 and upper[n-1+col-row] == 0 and lower[row+col] == 0:
                    board[row][col] = "Q"
                    low[row] = 1
                    upper[n-1+col-row] = 1
                    lower[row+col] = 1
                    solve(col + 1, board, low, upper, lower, ans, n)
                    low[row] = 0
                    upper[n-1+col-row] = 0
                    lower[row+col] = 0
        
        ans = []
        board = [["" for _ in range(n)] for _ in range(n)]
        low = [0] * n
        upper = [0] * (2 * n - 1)
        lower = [0] * (2 * n - 1)
        solve(0, board, low, upper, lower, ans, n)
        return len(ans)
```
