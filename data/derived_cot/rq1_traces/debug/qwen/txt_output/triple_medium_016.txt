The approach involves traversing the digits from left to right and replacing each digit if certain conditions are met, while ensuring the process stops when further replacements are no longer beneficial.

2. **Intuition**: The solution involves a greedy approach where we process each digit from left to right. 4. **Complexity Analysis**: The algorithm processes each digit in the string exactly once, leading to a time complexity of O(n), where n is the length of the input string. The space complexity is O(n) due to the storage required for the list of digits.

### Solution Code
```python
class Solution:
    def maximumNumber(self, num: str, change: List[int]) -> str:
        num_list = list(num)
        on = False
        for i in range(len(num_list)):
            x = int(num_list[i])
            if x < change[x]:
                num_list[i] = str(change[x])
                on = True
            elif on and x >= change[x]:
                break
        return "".join(num_list)
```

class Solution:
    def maximumNumber(self, num: str, change: List[int]) -> str:
        num_list = list(num)
        on = False
        for i in range(len(num_list)):
