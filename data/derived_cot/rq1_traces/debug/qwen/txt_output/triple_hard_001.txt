=== RAW ANSWER ===

To solve this problem, we need to find the median of two sorted arrays efficiently. The median is the middle value in an ordered list of numbers. If the combined length of the two arrays is even, the median is the average of the two middle numbers; otherwise, it is the middle number itself.

### Approach
1. **Merge the Arrays**: Since both input arrays are sorted, we can merge them into a single sorted array by comparing elements from both arrays using two pointers.
2. **Calculate the Median**: Once the merged array is obtained, we determine the median by checking if the length of the merged array is odd or even. For an odd length, the median is the middle element; for an even length, it is the average of the two middle elements.

### Solution Code
```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        new = []
        p1 = 0
        p2 = 0
        
        while p1 < len(nums1) and p2 < len(nums2):
            if nums1[p1] < nums2[p2]:
                new.append(nums1[p1])
                p1 += 1
            else:
                new.append(nums2[p2])
                p2 += 1
        
        while p1 < len(nums1):
            new.append(nums1[p1])
            p1 += 1
            
        while p2 < len(nums2):
            new.append(nums2[p2])
            p2 += 1
            
        n = len(new)
        if n % 2 == 1:
            median = float(new[n // 2])
        else:
            median = (new[n // 2 - 1] + new[n // 2]) / 2.0
            
        return median
```

### Explanation
1. **Merging Arrays**: The code initializes two pointers, `p1` and `p2`, to start from the beginning of each array. It then iterates through both arrays, comparing elements at the current pointers and appending the smaller element to the merged array `new`. This process continues until one of the arrays is exhausted.
2. **Appending Remaining Elements**: After one array is exhausted, the code appends the remaining elements from the other array to `new`.
3. **Finding the Median**: The length of the merged array `new` is checked. If the length is odd, the median is the middle element of `new`. If the length is even, the median is the average of the two middle elements. The result is returned as a float to ensure correct handling of both integer and fractional results.

This approach efficiently merges the two sorted arrays and calculates the median in linear time, O(m + n), where m and n are the lengths of the two input arrays.
