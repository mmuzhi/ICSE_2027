=== RAW ANSWER ===

To solve this problem, we need to determine if a singly-linked list contains a cycle. The approach involves using two pointers moving at different speeds to detect if they meet, indicating a cycle exists.

### Approach
The solution employs the Floyd's Tortoise and Hare algorithm, which uses two pointers to detect cycles in a linked list. One pointer (slow) moves one step at a time, while the other pointer (fast) moves two steps. If there is a cycle, the fast pointer will eventually meet the slow pointer. If the fast pointer reaches the end of the list (i.e., encounters a `None` value), the list is not cyclic.

### Solution Code
```python
# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, x):
#         self.val = x
#         self.next = None

class Solution:
    def hasCycle(self, head: Optional[ListNode]) -> bool:
        slow = head
        fast = head
        while slow and fast.next:
            slow = slow.next
            fast = fast.next.next
            if slow == fast:
                return True
        return False
```

### Explanation
1. **Initialization**: Both pointers (`slow` and `fast`) start at the head of the linked list.
2. **Loop Condition**: The loop continues as long as `slow` is not `None` and `fast.next` is not `None`. This ensures that both pointers can move without causing a runtime error.
3. **Pointer Movement**: In each iteration, `slow` moves one step to the next node, while `fast` moves two steps (one step to `fast.next` and then another to `fast.next.next`).
4. **Cycle Detection**: If at any point `slow` and `fast` point to the same node, the function returns `True`, indicating a cycle exists. If the loop exits without the pointers meeting, the function returns `False`, indicating no cycle.

This approach efficiently checks for cycles in linear time (O(n)) and constant space (O(1)), making it optimal for this problem.
