=== RAW ANSWER ===

To solve this problem, we need to find the longest harmonious subsequence in an array of integers. A harmonious subsequence is defined as a subsequence where the absolute difference between the maximum and minimum elements is exactly 1. The solution involves efficiently checking all possible pairs of consecutive numbers and determining the maximum length of the subsequence that can be formed by combining their occurrences.

### Approach
1. **Frequency Map Construction**: First, we construct a frequency map (dictionary) where each key is a unique number from the input array, and the corresponding value is the count of that number's occurrences in the array.
2. **Check Consecutive Numbers**: For each number in the frequency map, we check if either the number itself or the number plus one exists in the map. This ensures we only consider pairs of consecutive numbers.
3. **Calculate Maximum Length**: For each valid pair of consecutive numbers, we calculate the sum of their frequencies. The maximum value obtained from these sums across all valid pairs is our answer.

### Solution Code
```python
class Solution:
    def findLHS(self, nums: List[int]) -> int:
        from collections import defaultdict
        freq = defaultdict(int)
        for num in nums:
            freq[num] += 1
        
        result = 0
        for num in freq:
            if num + 1 in freq:
                result = max(result, freq[num] + freq[num + 1])
            if num - 1 in freq:
                result = max(result, freq[num] + freq[num - 1])
        return result
```

### Explanation
1. **Frequency Map Construction**: The code starts by building a frequency map using a defaultdict from the collections module. This map keeps track of how many times each number appears in the input array.
2. **Checking Consecutive Numbers**: For each number in the frequency map, the code checks if the number plus one or the number minus one exists in the map. This step ensures we only consider pairs of consecutive numbers (e.g., 1 and 2, or 2 and 3).
3. **Calculating Maximum Length**: For each valid consecutive pair found, the code calculates the total occurrences of both numbers by summing their frequencies. The maximum value obtained from these sums is stored in the result variable, which is returned at the end.

This approach efficiently checks all possible harmonious subsequences by leveraging a frequency map and ensures optimal performance with a time complexity of O(n), where n is the number of elements in the input array. The space complexity is also O(n) due to the storage requirements of the frequency map.
