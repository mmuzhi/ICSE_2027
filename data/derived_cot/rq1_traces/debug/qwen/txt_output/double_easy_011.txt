2. **Traversal**: We iterate through the array using the `fast` pointer. For each element:

   - After swapping (if necessary), we check if the element at `slow` is non-zero. If it is, we increment `slow` to move the pointer forward, indicating that the position is now occupied by a non-zero element.
3. **Result**: After processing all elements, all zeros will be moved to the end of the array, while non-zero elements retain their original order.
