=== RAW ANSWER ===

To solve this problem, we need to efficiently compute the median of all possible sliding windows of size `k` in an array of integers. The challenge lies in maintaining the sliding window and quickly accessing the median values, which requires a data structure that can dynamically insert and remove elements while allowing efficient median queries.

### Approach
The approach involves using a balanced binary search tree (BST) that supports order statistic operations. The BST nodes store the count of elements and the total size of the subtree rooted at each node. This allows us to efficiently insert, remove, and query the k-th smallest element in the current window. The key operations are:
1. **Insertion**: Add a new element to the BST and update the sizes of affected nodes.
2. **Removal**: Remove an element from the BST and update the sizes of affected nodes.
3. **Querying the k-th Smallest Element**: Traverse the BST to find the k-th smallest element based on subtree sizes.

The sliding window is maintained by inserting each new element and removing the oldest element when the window size exceeds `k`. The median is then computed based on whether the window size `k` is odd or even.

### Solution Code
```python
class Solution:
    class Node:
        __slots__ = ('val', 'count', 'left', 'right', 'size')
        def __init__(self, val):
            self.val = val
            self.count = 1
            self.left = None
            self.right = None
            self.size = 1

    def insert(self, root, val):
        if root is None:
            return self.Node(val)
        if val == root.val:
            root.count += 1
            root.size += 1
            return root
        if val < root.val:
            root.left = self.insert(root.left, val)
        else:
            root.right = self.insert(root.right, val)
        root.size = 1 + (root.left.size if root.left else 0) + (root.right.size if root.right else 0)
        return root

    def remove(self, root, val):
        if root is None:
            return None
        if val == root.val:
            root.count -= 1
            root.size -= 1
            return root
        if val < root.val:
            root.left = self.remove(root.left, val)
        else:
            root.right = self.remove(root.right, val)
        root.size = 1 + (root.left.size if root.left else 0) + (root.right.size if root.right else 0)
        return root

    def size(self, root):
        if root is None:
            return 0
        return root.size

    def get(self, root, k):
        if root is None:
            return None
        left_size = root.left.size if root.left else 0
        if k <= left_size:
            return self.get(root.left, k)
        elif k == left_size + 1:
            return root.val
        else:
            return self.get(root.right, k - left_size - 1)

    def medianSlidingWindow(self, nums: List[int], k: int) -> List[float]:
        tree = None
        ans = []
        n = len(nums)
        for i, x in enumerate(nums):
            tree = self.insert(tree, x)
            if i >= k:
                tree = self.remove(tree, nums[i - k])
            if i >= k - 1:
                if k % 2 == 1:
                    median = self.get(tree, (k // 2) + 1)
                    ans.append(median)
                else:
                    left_val = self.get(tree, k // 2)
                    right_val = self.get(tree, k // 2 + 1)
                    ans.append((left_val + right_val) / 2.0)
        return ans
```

1. **Node Class**: Each node in the BST stores the value, a count of occurrences, left and right children, and the total size of the subtree rooted at that node.
2. **Insert Method**: Inserts a value into the BST, incrementing the count and size of the node if the value exists, otherwise creating a new node.
3. **Remove Method**: Removes a value from the BST by decrementing the count and size of the node, effectively removing one occurrence.
4. **Size Method**: Returns the total number of nodes in the subtree rooted at the given node.
5. **Get Method**: Retrieves the k-th smallest element in the BST by traversing based on subtree sizes.
6. **MedianSlidingWindow Method**: Processes each element in the input array, maintaining a sliding window of size `k` by inserting new elements and removing old ones. The median for each window is computed using the BST's order statistic operations, ensuring efficient median queries.

This approach efficiently handles the sliding window median problem with a time complexity of O(n log k) for each element processed, leveraging the properties of the BST to maintain and query the median dynamically.
