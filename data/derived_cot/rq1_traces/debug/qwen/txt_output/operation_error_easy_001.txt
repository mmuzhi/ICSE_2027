=== RAW ANSWER ===

To solve this problem, we need to determine if all occurrences of elements in the given array have unique frequencies. The approach involves counting the frequency of each element in the array and then checking if these frequencies are all distinct.

### Approach
1. **Count Frequencies**: Use the `Counter` class from the `collections` module to count the occurrences of each element in the array. This gives us a dictionary where keys are the distinct elements and values are their respective counts.
2. **Check Unique Frequencies**: Extract the values (frequencies) from the dictionary and check if all these values are unique. This is done by comparing the length of the list of frequencies with the size of the set created from these frequencies. If the lengths are the same, all frequencies are unique; otherwise, there are duplicates.

### Solution Code
```python
from collections import Counter

class Solution:
    def uniqueOccurrences(self, arr: List[int]) -> bool:
        d = Counter(arr)
        freqs = list(d.values())
        return len(freqs) == len(set(freqs))
```

### Explanation
1. **Counting Frequencies**: The `Counter` class efficiently counts how many times each element appears in the array. For example, if the input array is `[1, 2, 2, 3]`, the `Counter` will produce a dictionary `{1:1, 2:2, 3:1}`.
2. **Checking Uniqueness**: The values from the dictionary (frequencies) are converted into a list. The length of this list is compared to the size of the set created from these values. If all frequencies are unique, the set will have the same number of elements as the list. If there are duplicate frequencies, the set will be smaller than the list, indicating the function should return `False`.

This approach ensures that we efficiently check for unique frequencies using the properties of sets and dictionaries, providing an optimal solution with a time complexity of O(n), where n is the number of elements in the input array.
