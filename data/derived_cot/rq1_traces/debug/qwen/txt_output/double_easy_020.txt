=== RAW ANSWER ===

To solve this problem, we need to find the third maximum distinct value in a given list of numbers. If there are fewer than three distinct values, we return the maximum value among them. The approach involves removing duplicates, sorting the unique values, and then determining the third largest value.

### Approach
1. **Remove Duplicates**: Convert the input list into a set to eliminate duplicate values.
2. **Check Distinct Values**: If the number of distinct values is less than three, return the maximum value from the set.
3. **Sort and Find Third Maximum**: If there are at least three distinct values, sort them in descending order and return the third element in this sorted list. This element is the third largest value in the original list.

### Solution Code
```python
class Solution:
    def thirdMax(self, nums) -> int:
        unique = set(nums)
        if len(unique) < 3:
            return max(unique)
        else:
            sorted_unique = sorted(unique, reverse=True)
            return sorted_unique[2]
```

### Explanation
1. **Removing Duplicates**: The input list `nums` is converted into a set `unique` to ensure all elements are distinct.
2. **Handling Fewer Than Three Distinct Values**: If the size of the set `unique` is less than three, the maximum value in the set is returned directly.
3. **Finding the Third Maximum**: For sets with three or more distinct values, the elements are sorted in descending order. The third element in this sorted list (index 2) is the third largest value, which is then returned.

This approach efficiently handles both scenarios (fewer than three distinct values and three or more) and ensures the correct third maximum value is returned. The use of sorting in descending order simplifies the retrieval of the third largest element.
