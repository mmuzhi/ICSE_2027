=== RAW ANSWER ===

To solve this problem, we need to construct a quad-tree from a given 2D grid of 0s and 1s. The quad-tree is a hierarchical data structure where each node can have four children representing the four quadrants of the grid. The goal is to build this tree recursively, merging nodes whenever all four quadrants of a node have the same value.

### Approach
1. **Problem Analysis**: The problem involves constructing a quad-tree from a grid where each node represents a quadrant. If all four quadrants of a node have the same value, the node is a leaf; otherwise, it is an internal node with four children.
2. **Intuition**: The solution uses a divide-and-conquer approach. The grid is recursively divided into four quadrants until each quadrant is a single cell (leaf node). If all four quadrants of a node have the same value, they are merged into a single leaf node.
3. **Algorithm Selection**: The algorithm recursively processes each quadrant of the grid. For each node, it checks if all four children are leaves with the same value. If they are, it creates a leaf node; otherwise, it creates an internal node.
4. **Complexity Analysis**: The algorithm processes each cell in the grid exactly once, leading to a time complexity of O(n^2) where n is the grid size. The space complexity is also O(n^2) due to the recursive calls and storage of the quad-tree nodes.

### Solution Code
```python
class Solution:
    def construct(self, grid: List[List[int]]) -> 'Node':
        level = len(grid)
        root = None
        if level >= 1:
            root = self.buildTree(grid, 0, level, 0, level, level)
        return root
        
    def buildTree(self, grid, rs, re, cs, ce, level):
        if level == 1:
            return Node(grid[rs][cs], True, None, None, None, None)

        bl = self.buildTree(grid, re - next_level, re, cs, ce - next_level, next_level)
        br = self.buildTree(grid, re - next_level, re, ce - next_level, ce, next_level)
        
        if tl.isLeaf and tr.isLeaf and bl.isLeaf and br.isLeaf:
            if tl.val == tr.val == bl.val == br.val:
                new_node = Node(tl.val, True, None, None, None, None)
            else:
                new_node = Node(tl.val, False, tl, tr, bl, br)

            new_node = Node(tl.val, False, tl, tr, bl, br)
            
        return new_node
```

        tr = self.buildTree(grid, rs, re - next_level, ce - next_level, ce, next_level)
        bl = self.buildTree(grid, re - next_level, re, cs, ce - next_level, next_level)
        br = self.buildTree(grid, re - next_level, re, ce - next_level, ce, next_level)
        
        if tl.isLeaf and tr.isLeaf and bl.isLeaf and br.isLeaf:
            if tl.val == tr.val == bl.val == br.val:
                new_node = Node(tl.val, True, None, None, None, None)
            else:
                new_node = Node(tl.val, False, tl, tr, bl, br)
        else:
            new_node = Node(tl.val, False, tl, tr, bl, br)
            
        return new_node
