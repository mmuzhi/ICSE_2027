3. **Algorithm Selection**: The 0-1 BFS algorithm is chosen because it efficiently handles the grid's binary weights (0 or 1) with a time complexity of O(m*n), where m and n are the grid dimensions.
4. **Optimization**: By using a deque, we ensure that cells with 0 weight are processed first (added to the front) and those with 1 weight are processed later (added to the back), mimicking a priority queue without the overhead of a heap.

from collections import deque

class Solution:
    def minimumObstacles(self, grid: List[List[int]]) -> int:
        m, n = len(grid), len(grid[0])
