To solve this problem, we need to compute the sum of the minimum values of all contiguous subarrays in a given array. The challenge lies in efficiently determining the minimum value for each subarray without resorting to a brute-force approach, which would be computationally expensive.

### Approach
1. **Problem Analysis**: The problem requires summing the minimum values of all possible contiguous subarrays in an array. A brute-force approach would involve checking every possible subarray, which is O(n^2) and inefficient for large arrays. Instead, we use a more efficient method involving monotonic stacks to precompute boundaries where each element is the minimum.

2. **Intuition**: For each element in the array, we determine the left and right boundaries within which it is the minimum. The left boundary is the first element to the left that is greater than the current element, and the right boundary is the first element to the right that is greater than or equal to the current element. The number of subarrays where the current element is the minimum is then calculated based on these boundaries.

3. **Algorithm Selection**: 
   - **Monotonic Stack**: We use a stack to keep track of elements in decreasing order to efficiently find the next greater elements to the left and right for each element.
   - **Boundary Calculation**: For each element, the left boundary is the index of the next greater element to the left plus one, or zero if no such element exists. The right boundary is the index of the next greater or equal element to the right, or the end of the array if none exists.
   - **Sum Calculation**: For each element, multiply its value by the number of subarrays where it is the minimum (derived from the boundaries) and accumulate the result.

4. **Complexity Analysis**: The algorithm processes each element twice (once for left and once for right boundaries) using a stack, resulting in an O(n) time complexity for each pass. The overall time complexity is O(n), and the space complexity is O(n) due to the stack and boundary arrays.

### Solution Code
```python
class Solution:
    def sumSubarrayMins(self, arr: List[int]) -> int:
        n = len(arr)
        left = [0] * n
        right = [0] * n
        stack = []
        for i in range(n):
            while stack and arr[stack[-1]] > arr[i]:
                stack.pop()
            left[i] = stack[-1] + 1 if stack else 0
            stack.append(i)
        
        stack = []
        for i in range(n-1, -1, -1):
            while stack and arr[stack[-1]] >= arr[i]:
                stack.pop()
            right[i] = stack[-1] if stack else n-1
            stack.append(i)
        
        ans = 0
        mod = 10**9 + 7
        for i in range(n):
            ans = (ans + arr[i] * (left[i] - i + right[i] - i + 1)) % mod
        return ans
```

3. **Right Boundaries Calculation**: Similarly, we traverse the array from right to left, using a stack to find the right boundaries. Elements greater than or equal to the current element are popped, and the top of the stack provides the right boundary.
4. **Sum Calculation**: For each element, the number of subarrays where it is the minimum is calculated using the boundaries. The result is accumulated by multiplying the element's value by this count, taking modulo 10^9 + 7 to handle large numbers.
