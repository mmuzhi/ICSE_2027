### Approach
The approach involves iterating through each number from 1 to `n` and checking divisibility conditions. The key steps are:
1. **Loop through numbers 1 to n**: This ensures we start from 1, as the original code mistakenly started from 0.
2. **Check divisibility conditions**: The conditions are checked in order: first for divisibility by both 3 and 5, then by 3, then by 5, and finally for numbers not divisible by either.

### Solution Code
```python
class Solution:
    def fizzBuzz(self, n: int) -> List[str]:
        arr = []
        for i in range(1, n+1):
            if i % 3 == 0 and i % 5 == 0:
                arr.append("FizzBuzz")
            elif i % 3 == 0:
                arr.append("Fizz")
            elif i % 5 == 0:
                arr.append("Buzz")
            else:
                arr.append(str(i))
        return arr
```

### Explanation
- **Loop from 1 to n**: The loop starts at 1 and goes up to `n`, ensuring we process each number in the specified range.
- **Divisibility checks**: The conditions are checked in a specific order to handle all cases correctly. If a number is divisible by both 3 and 5, "FizzBuzz" is added. Otherwise, if divisible by 3, "Fizz" is added; if divisible by 5, "Buzz" is added. For all other numbers, the number itself is converted to a string and added.
- **Efficiency**: The solution efficiently checks each number in linear time, O(n), with each check involving constant time operations (modulo operations and string appends). This approach is optimal for the given problem constraints.
