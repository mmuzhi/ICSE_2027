To solve this problem, we need to find the longest consecutive sequence in an array of integers, where the length of the sequence is defined as the total frequency of all numbers in the consecutive sequence. The consecutive sequence must be contiguous, meaning each number in the sequence is exactly one more than the previous number.

### Approach
1. **Frequency Map Construction**: First, we build a frequency map (dictionary) where each key is a unique number from the input array, and the value is the count of occurrences of that number in the array.
2. **Sorting Keys**: We then sort the keys of the frequency map to process numbers in ascending order. This allows us to efficiently check for consecutive numbers.
3. **Dynamic Programming (DP) Setup**: We use a dynamic programming approach where `dp[key]` represents the total frequency of the longest consecutive sequence ending at `key`. If the previous number (`key - 1`) exists in the frequency map, we extend the sequence by adding the frequency of `key` to the sequence ending at `key - 1`. Otherwise, the sequence starts anew at `key`.
4. **Result Calculation**: As we iterate through the sorted keys, we keep track of the maximum value in the DP array, which gives us the total frequency of the longest consecutive sequence.

### Solution Code
```python
from collections import defaultdict

class Solution:
    def findLHS(self, nums: List[int]) -> int:
        count_map = defaultdict(int)
        for num in nums:
            count_map[num] += 1
        
        if not count_map:
            return 0
        
        keys = sorted(count_map.keys())
        dp = {}
        result = 0
        
        for key in keys:
            if key - 1 in count_map:
                dp[key] = dp[key - 1] + count_map[key]
            else:
                dp[key] = count_map[key]
            if dp[key] > result:
                result = dp[key]
        
        return result
```
