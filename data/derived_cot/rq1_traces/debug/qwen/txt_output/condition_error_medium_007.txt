    def corpFlightBookings(self, bookings: List[List[int]], n: int) -> List[int]:

        arr = [0]*(n+1)
        for lv, ar, seats in bookings:
            arr[lv-1]+= seats
            arr[ar+1]-= seats

### Approach

class Solution:
    def corpFlightBookings(self, bookings: List[List[int]], n: int) -> List[int]:
        arr = [0] * (n + 2)

            arr[e + 1] -= seats

4. **Result Extraction**: The result is the first `n` elements of the prefix sum array, which corresponds to the total seats booked for each flight from airport 1 to airport `n`.
