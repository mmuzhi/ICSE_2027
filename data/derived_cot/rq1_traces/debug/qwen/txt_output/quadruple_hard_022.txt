=== RAW ANSWER ===

To solve this problem, we need to find a valid arrangement (Eulerian trail) in a directed graph that covers all the edges exactly once. The solution involves checking the graph's Eulerian trail conditions and then performing a depth-first search (DFS) to construct the trail.

### Approach
1. **Graph Construction and Degree Calculation**: 
   - Build a directed graph from the given pairs of nodes.
   - Calculate the net degree for each node, which is the difference between its out-degree and in-degree.

2. **Check Eulerian Trail Conditions**:
   - For an Eulerian trail (not necessarily a circuit), the graph must have exactly two nodes with non-zero net degrees: one with a net degree of +1 (start node) and one with a net degree of -1 (end node). If the graph is an Eulerian circuit, all nodes must have a net degree of 0, and we can start from any node.

3. **Depth-First Search (DFS)**:
   - Perform a DFS starting from the identified start node, traversing edges and popping them from the graph as we go. This ensures each edge is used exactly once.
   - The DFS records nodes in post-order (after all their children are processed). The recorded nodes are then reversed to form the Eulerian trail from start to end.

### Solution Code
```python
from collections import defaultdict
from typing import List

class Solution:
    def validArrangement(self, pairs: List[List[int]]) -> List[List[int]]:
        graph = defaultdict(list)
        degree = defaultdict(int)
        
        for x, y in pairs:
            graph[x].append(y)
            degree[x] += 1
            degree[y] -= 1
        
        non_zero = [k for k, d in degree.items() if d != 0]
        
        if len(non_zero) == 0:
            start = next(iter(degree.keys()))
        elif len(non_zero) == 2:
            if set(degree[k] for k in non_zero) != {1, -1}:
                return []
            start = non_zero[0] if degree[non_zero[0]] == 1 else non_zero[1]
        else:
            return []
        
        ans = []
        
        def fn(x):
            while graph[x]:
                y = graph[x].pop()
                fn(y)
            ans.append(x)
        
        fn(start)
        ans.reverse()
        
        result = []
        for i in range(len(ans) - 1):
            result.append([ans[i], ans[i+1]])
        
        return result
```

### Explanation
1. **Graph Construction**: The graph is built using a dictionary where each node maps to a list of its outgoing edges. The `degree` dictionary keeps track of the net degree (out-degree minus in-degree) for each node.
2. **Eulerian Trail Check**: The code checks if the graph meets the conditions for an Eulerian trail (exactly two nodes with net degrees +1 and -1) or an Eulerian circuit (all nodes have net degree 0). If the conditions are not met, it returns an empty list.
3. **DFS Traversal**: The DFS function starts from the identified start node and traverses each edge, popping it from the graph to ensure it's used only once. Nodes are recorded in post-order during traversal.
4. **Result Construction**: The recorded nodes are reversed to form the trail from start to end. Consecutive nodes in the reversed list are paired to form the final result, which is a list of edges covering all nodes in the graph exactly once.
