To solve this problem, we need to find the minimum total price for all trips in a tree structure, where each city's price can be discounted at most once if it is visited twice or more. The discount is applied by halving the city's price (using floor division) for all its visits.

### Approach
1. **Tree Construction**: Represent the tree using an adjacency list from the given edges.
2. **Frequency Calculation**: For each trip (start and destination), compute the frequency of each node being visited by all trips. This is done using a Depth-First Search (DFS) to determine levels and parent relationships, followed by finding the Lowest Common Ancestor (LCA) for each trip to count node visits.
3. **Dynamic Programming (DP)**: Use a DP approach to determine the optimal city to discount. The DP function calculates the minimum total price for the subtree rooted at each node, considering whether to discount the current node or not. The solution involves:
   - **DP0[i]**: The minimum total price for the subtree rooted at node `i` without discounting any node.
   - **DP1[i]**: The minimum total price for the subtree rooted at node `i` if exactly one node is discounted in the subtree.
4. **Result Calculation**: The final result is derived from the DP values at the root node, choosing between the full price or the best discounted price.

class Solution:
    def minimumTotalPrice(self, n: int, edges: List[List[int]], price: List[int], trips: List[List[int]]) -> int:
        g = [[] for _ in range(n)]
        for i, j in edges:
            g[i].append(j)
            g[j].append(i)
            
        freq = [0] * n
        level = [0] * n
        parent = [0] * n
        
        def dfs(i, l, p):
            level[i] = l
            parent[i] = p
            for j in g[i]:
                if j != p:
                    dfs(j, l + 1, i)
        
        def LCA(a, b):
            if level[a] < level[b]:
                a, b = b, a
            d = level[a] - level[b]
            while d:
                a = parent[a]
                d -= 1
            if a == b:
                return a
            while a != b:
                a = parent[a]
                b = parent[b]
            return a
        
        dfs(0, 0, -1)
        for i, j in trips:
            lca = LCA(i, j)
            path = []
            while i != lca:
                freq[i] += 1
                i = parent[i]
            freq[lca] += 1
            while j != lca:
                freq[j] += 1
                j = parent[j]
