To solve this problem, we need to remove duplicates from a sorted array such that each element appears at most twice. The solution involves modifying the array in place and returning the new length of the array after removing the extra duplicates.

### Approach
1. **Initial Check**: If the array has fewer than 3 elements, return its length immediately since no duplicates can be removed beyond the allowed limit.
2. **Two Pointers Technique**: Use two pointers, `p1` and `p2`, where `p1` is the pointer for the last allowed element, and `p2` is the pointer for the current element being considered.
3. **Duplicate Detection**: For each element, check if it is part of three consecutive duplicates. If so, move `p2` forward until a different element is found.
4. **Update Pointers**: If no three consecutive duplicates are found, increment `p1` and update the array at `p1` with the value from `p2`, then increment `p2`.
5. **Termination**: If `p2` reaches the end of the array, break out of the loop.
6. **Return Result**: The new length is `p1 + 1` since `p1` points to the last allowed element.

### Solution Code
```python
class Solution:
    def removeDuplicates(self, nums: List[int]) -> int:
        if len(nums) < 3:
            return len(nums)
        
        p1, p2 = 1, 2
        
        while p2 < len(nums):
            if nums[p1] == nums[p1-1] and nums[p2] == nums[p2-1] == nums[p1-2]:
                while p2 < len(nums) and nums[p2] == nums[p2-1]:
                    p2 += 1
                if p2 == len(nums):
                    break
            p1 += 1
            nums[p1] = nums[p2]
            p2 += 1
            
        return p1 + 1
```

### Explanation
- **Initial Check**: The solution first checks if the array length is less than 3, in which case it returns the array length immediately since no duplicates can exceed the allowed limit.
- **Two Pointers**: The pointers `p1` and `p2` start at indices 1 and 2, respectively. `p1` marks the position of the last allowed element, and `p2` is used to find the next valid element.
- **Duplicate Handling**: The condition checks if the current elements at `p1` and `p2` form three consecutive duplicates. If so, `p2` is advanced until a different element is found.
- **Pointer Update**: If no three consecutive duplicates are found, `p1` is incremented, and the value at `p1` is updated with the value from `p2`. `p2` is then incremented to continue processing.
- **Termination**: If `p2` reaches the end of the array, the loop breaks, and the new length is `p1 + 1`, which is returned.
