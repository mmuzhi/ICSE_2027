=== RAW ANSWER ===

To solve this problem, we need to sort a given string in a specific pattern where the characters are first arranged in increasing order, then in decreasing order, and this pattern repeats until all characters are used. The key insight is to alternate between these two orders while consuming each character from the string exactly once.

### Approach
1. **Frequency Dictionary**: First, we build a frequency dictionary to keep track of how many times each character appears in the input string.
2. **Unique Characters**: We then create a sorted list of unique characters from the string to determine the order in which characters should be processed.
3. **Alternating Passes**: The main loop continues until all characters are used up (i.e., the frequency dictionary is empty). In each iteration:
   - **Increasing Order Pass**: We process each character in the sorted order, appending it to the result string and decrementing its count. If the count reaches zero, we remove the character from the frequency dictionary.
   - **Decreasing Order Pass**: We process each character in reverse sorted order (decreasing order), appending it to the result string and decrementing its count. Again, if the count reaches zero, we remove the character from the frequency dictionary.
4. **Termination**: The loop terminates when the frequency dictionary is empty, meaning all characters have been processed.

### Solution Code
```python
class Solution:
    def sortString(self, s: str) -> str:
        freq = {}
        for char in s:
            freq[char] = freq.get(char, 0) + 1
        letters = sorted(set(s))
        res = ""
        while freq:
            for i in letters:
                if i in freq and freq[i] > 0:
                    res += i
                    freq[i] -= 1
                    if freq[i] == 0:
                        del freq[i]
            for i in letters[::-1]:
                if i in freq and freq[i] > 0:
                    res += i
                    freq[i] -= 1
                    if freq[i] == 0:
                        del freq[i]
        return res
```

2. **Unique Characters**: The unique characters from the string are sorted to determine the order in which they should be processed in the increasing and decreasing passes.
3. **Processing Loop**: The loop continues as long as there are characters left to process (i.e., the frequency dictionary is not empty). In each iteration:
   - **Increasing Pass**: The code processes each character in the sorted order. For each character present in the frequency dictionary with a positive count, it appends the character to the result string, decrements its count, and removes it from the dictionary if the count drops to zero.
   - **Decreasing Pass**: Similarly, the code processes each character in reverse sorted order, appending them to the result string and decrementing their counts, removing them if counts reach zero.
4. **Result Construction**: The result string is built by alternating between these two passes until all characters are exhausted. The final result is the string formed by concatenating characters in the specified alternating order.
