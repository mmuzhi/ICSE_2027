### Approach
1. **BFS Traversal**: Use a queue to process each node level by level. Start from the root node and process each node in the current level, appending their values to a temporary list.
2. **Appending Children**: For each node, append its left child followed by its right child to the queue. This ensures that nodes are processed in the correct order (left to right) within each level.
3. **Reversing Levels**: After processing all nodes at the current level, append the temporary list to the result. Since we process levels from top to bottom, reversing the result list gives us the bottom-up order.

### Solution Code

from collections import deque

class Solution:
    def levelOrderBottom(self, root: Optional[TreeNode]) -> List[List[int]]:
