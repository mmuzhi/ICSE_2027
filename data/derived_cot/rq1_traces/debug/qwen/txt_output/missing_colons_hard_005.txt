To solve this problem, we need to find the longest substring within a given word that does not contain any forbidden substrings. The solution involves using a trie data structure to efficiently check for forbidden substrings and a two-pointer approach to determine the longest valid substring.

### Approach
1. **Trie Construction**: Build a trie from the list of forbidden substrings. Each node in the trie represents a character, and an "end" marker indicates the completion of a forbidden substring.
2. **Forbidden Substring Check**: For each substring starting at position `i` and ending at position `j-1`, check if it contains any forbidden substring using the trie. The function `isForbidden` traverses the trie while checking each character in the substring. If a forbidden substring is found, it returns the length of that substring.
3. **Two-Pointer Technique**: Traverse the word from right to left, adjusting the end pointer `j` whenever a forbidden substring is detected. The goal is to find the longest substring ending at `j-1` that does not contain any forbidden substrings.

### Solution Code
```python
class Solution:
    def longestValidSubstring(self, word: str, forbidden: List[str]) -> int:
        if not forbidden:
            return len(word)
        
        trie = {}
        for f in forbidden:
            t = trie
            for c in f:
                if c not in t:
                    t[c] = {}
                t = t[c]
            t["end"] = True
        
        def isForbidden(s):
            t = trie
            counter = 0
            for c in s:
                if c not in t:
                    return False
                t = t[c]
                counter += 1
                if "end" in t:
                    return counter
            return False
        
        res = 0
        j = len(word)
        for i in range(len(word)-1, -1, -1):
            truc = isForbidden(word[i:j])
            if truc:
                j = i + truc
            res = max(res, j - i - 1)
        return res
```

1. **Trie Construction**: The trie is built by iterating over each forbidden substring and adding each character to the trie. An "end" marker is added at the completion of each forbidden substring to denote its presence.
2. **Forbidden Substring Check**: The `isForbidden` function checks if a given substring contains any forbidden substring by traversing the trie. If a forbidden substring is found, it returns its length; otherwise, it returns False.
3. **Two-Pointer Technique**: The main loop starts from the end of the word and moves leftwards. If a forbidden substring is found, `j` is adjusted to the position right after the start of the forbidden substring.
