=== RAW ANSWER ===

To solve this problem, we need to find the median of two sorted arrays. The median is the middle value in an ordered list of numbers. If the combined length of the arrays is even, the median is the average of the two middle numbers. 

### Approach
1. **Merge the Arrays**: Combine the two sorted arrays into a single sorted array. This is done using a two-pointer technique where we compare elements from both arrays and append the smaller one to the result list.
2. **Calculate the Median**: After merging, if the total number of elements is odd, the median is the middle element. If the total number is even, the median is the average of the two middle elements.

### Solution Code
```python
class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        new = []
        p1 = 0
        p2 = 0
        
        while p1 < len(nums1) and p2 < len(nums2):
            if nums1[p1] < nums2[p2]:
                new.append(nums1[p1])
                p1 += 1
            else:
                new.append(nums2[p2])
                p2 += 1
        
        while p1 < len(nums1):
            new.append(nums1[p1])
            p1 += 1
            
        while p2 < len(nums2):
            new.append(nums2[p2])
            p2 += 1
            
        n = len(new)
        if n % 2 == 1:
            median = new[n // 2]
        else:
            median = (new[n // 2 - 1] + new[n // 2]) / 2.0
            
        return median
```

### Explanation
1. **Merging Arrays**: The code starts by initializing an empty list `new` and two pointers `p1` and `p2` to traverse `nums1` and `nums2` respectively. The while loop continues comparing elements from both arrays, appending the smaller element to `new` and moving the corresponding pointer forward.
2. **Appending Remaining Elements**: After one of the arrays is exhausted, the remaining elements from the other array are appended to `new`.
3. **Finding the Median**: The median is determined based on the length of `new`. If the length is odd, the median is the middle element. If even, it is the average of the two middle elements. The result is returned as a float.

This approach efficiently merges the two sorted arrays and computes the median in linear time, O(n), where n is the total number of elements in both arrays.
