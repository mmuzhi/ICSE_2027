### Approach
The approach is straightforward:
1. **Concatenate Strings**: Use the `join` method to concatenate all strings in each list. The `join` method is used with an empty string as the separator to ensure that the strings are concatenated without any additional characters between them.
2. **Compare Concatenated Strings**: Compare the concatenated strings from both lists. If they are identical, return `True`; otherwise, return `False`.

### Solution Code
```python
class Solution:
    def arrayStringsAreEqual(self, word1: List[str], word2: List[str]) -> bool:
        word3 = ''.join(word1)
        word4 = ''.join(word2)
        return word3 == word4
```

### Explanation
1. **Concatenation**: The `join` method is used with an empty string (`''`) as the separator. This ensures that all strings in each list are concatenated into a single string without any intervening characters. For example, if `word1` is `["abc", "def"]`, then `''.join(word1)` results in `"abcdef"`.
2. **Comparison**: The concatenated strings from both lists (`word3` and `word4`) are compared. If they match exactly, the function returns `True`, indicating that the strings formed by concatenating the respective lists are equal. Otherwise, it returns `False`.

This approach efficiently checks for the equality of the concatenated strings by leveraging Python's built-in string methods, ensuring clarity and correctness.
