=== RAW ANSWER ===

To solve this problem, we need to find the complement of a given integer. The complement is obtained by flipping all the bits in the binary representation of the number. The key insight is to process each bit of the number from the least significant bit (LSB) to the most significant bit (MSB), flipping each bit (changing 1 to 0 and vice versa), and then constructing the resulting binary number from the flipped bits.

### Approach
1. **Bit Processing**: Iterate through each bit of the number starting from the LSB. For each bit, check if it is 1 or 0.
2. **Flipping Bits**: If the bit is 1, append 0 to the result string; if it is 0, append 1.
3. **Constructing Result**: After processing all bits, reverse the constructed string to get the correct binary representation of the complement.
4. **Conversion to Integer**: Convert the reversed binary string back to an integer.

### Solution Code
```python
class Solution:
    def findComplement(self, num: int) -> int:
        com = ''
        while num > 0:
            if num % 2 == 1:
                com += '0'
            else:
                com += '1'
            num = num // 2
        return int(com[::-1], 2)
```

### Explanation
1. **Initialization**: Start with an empty string `com` to accumulate the complement bits.
2. **Loop Until Zero**: While the number is greater than zero, process each bit:
   - **Check Bit**: Use modulo 2 to determine the least significant bit (0 or 1). If it is 1, append '0' to `com` (flipping to 0); if it is 0, append '1' (flipping to 1).
   - **Update Number**: Perform integer division by 2 to shift right and process the next bit.
3. **Reverse and Convert**: The bits are collected from LSB to MSB, so reverse the string to get the correct binary order. Convert this binary string to an integer using base 2.

This approach efficiently flips each bit of the input number and constructs the complement by processing each bit individually, ensuring correctness and simplicity.
