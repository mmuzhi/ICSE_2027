To solve this problem, we need to find a valid arrangement of given directed edges such that the arrangement forms an Eulerian trail or circuit. The solution involves checking the graph's degrees to determine if an Eulerian trail exists and then using Hierholzer's algorithm to construct the trail.

### Approach
1. **Graph Construction and Degree Calculation**: We first build a directed graph from the given pairs and calculate the net out-degree for each node (out-degree minus in-degree). This helps in identifying the start and end nodes for the Eulerian trail.
2. **Start and End Node Identification**: The start node is identified as the node with a net out-degree of 1, and the end node is the node with a net out-degree of -1. If no such nodes exist, the graph is either a circuit (all nodes have net out-degree 0) or invalid (other degrees present).
3. **Hierholzer's Algorithm**: Using a depth-first search (DFS), we traverse the graph, removing edges as we go, until no edges remain. The traversal starts from the identified start node and builds the trail by appending nodes once all their outgoing edges are exhausted.
4. **Result Construction**: The nodes collected during DFS are reversed to form the correct trail order. The trail is then converted into pairs of consecutive nodes.

### Solution Code
```python
from collections import defaultdict

class Solution:
    def validArrangement(self, pairs: List[List[int]]) -> List[List[int]]:
        graph = defaultdict(list)
        degree = defaultdict(int)
        for x, y in pairs:
            graph[x].append(y)
            degree[x] += 1
            degree[y] -= 1
        
        start = None
        end = None
        for node, d in degree.items():
            if d == 1:
                start = node
            elif d == -1:
                end = node
            elif d != 0:
                return []
        
        if start is None and end is None:
            start = next(iter(degree.keys()))
        elif start is None:
            return []
        elif end is None:
            return []
        
        ans = []
        
        def dfs(x):
            while graph[x]:
                y = graph[x].pop()
                dfs(y)
            ans.append(x)
        
        dfs(start)
        ans.reverse()
        return [[ans[i], ans[i+1]] for i in range(len(ans)-1)]
```

1. **Graph Construction**: The graph is built using a dictionary where each key is a node, and the value is a list of its neighbors (outgoing edges). The degree dictionary keeps track of the net out-degree for each node.
2. **Degree Check**: The algorithm checks if the graph can form an Eulerian trail by verifying the net out-degrees of nodes. The start node must have a net out-degree of 1, the end node must have -1, and all other nodes must have 0.
3. **DFS Traversal**: The DFS function starts from the identified start node, traversing edges and removing them as it goes. Nodes are appended to the result list once all their outgoing edges are processed.
4. **Result Construction**: The collected nodes are reversed to form the correct trail order. The trail is then split into pairs of consecutive nodes to produce the final arrangement.
