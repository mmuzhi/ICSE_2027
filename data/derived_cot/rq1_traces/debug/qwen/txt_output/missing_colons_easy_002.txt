=== RAW ANSWER ===

To solve this problem, we need to determine if all the frequencies of elements in an array are unique. The solution involves counting the occurrences of each element and then checking if these counts are all distinct.

### Approach
1. **Count Frequencies**: Use the `Counter` class from the `collections` module to count the occurrences of each element in the array. This gives us a dictionary where keys are the elements and values are their respective counts.
2. **Check Uniqueness of Counts**: Convert the values (counts) of the dictionary into a list. Then, check if the length of this list is equal to the length of the set created from the list. If they are equal, it means all counts are unique; otherwise, there are duplicate counts.

### Solution Code
```python
from collections import Counter

class Solution:
    def uniqueOccurrences(self, arr):
        d = Counter(arr)
        l = list(d.values())
        if len(l) == len(set(l)):
            return True
        else:
            return False
```

### Explanation
1. **Counting Elements**: The `Counter` class efficiently counts the occurrences of each element in the input array. For example, if the input array is `[1, 2, 2, 3]`, the `Counter` will produce a dictionary `{1: 1, 2: 2, 3: 1}`.
2. **Checking Uniqueness**: The values from the dictionary (counts) are extracted into a list. By converting this list into a set, we remove any duplicate counts. If the length of the original list and the set are the same, it means all counts are unique, and the function returns `True`. Otherwise, it returns `False`.

This approach ensures that we efficiently check for unique frequencies by leveraging the properties of sets to eliminate duplicates and comparing the lengths of the original list and the set. The solution is both concise and optimal for the given problem.
