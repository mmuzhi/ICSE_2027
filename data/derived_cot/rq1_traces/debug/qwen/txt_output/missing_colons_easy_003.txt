=== RAW ANSWER ===

To solve this problem, we need to find the mode(s) (most frequent elements) in a binary search tree (BST). The approach involves traversing the tree to count the frequency of each node's value and then determining which values have the highest frequency.

### Approach
1. **Tree Traversal**: We use a depth-first search (DFS) approach to traverse the tree. This involves using a stack to keep track of nodes to visit. Starting from the root, we process each node, pushing its children onto the stack.
2. **Frequency Counting**: As we visit each node, we increment its value in a frequency dictionary.
3. **Finding Modes**: After processing all nodes, we determine the maximum frequency from the dictionary. Any value(s) with this maximum frequency are considered the mode(s) of the BST.

### Solution Code
```python
class Solution:
    def findMode(self, root: Optional[TreeNode]) -> List[int]:
        if root is None:
            return []
        
        freq = {}
        stack = [root]
        while stack:
            curr = stack.pop()
            freq[curr.val] = freq.get(curr.val, 0) + 1
            if curr.right:
                stack.append(curr.right)
            if curr.left:
                stack.append(curr.left)
                
        max_cnt = 0
        ans = []
        for num, cnt in freq.items():
            if cnt > max_cnt:
                max_cnt = cnt
                ans = [num]
            elif cnt == max_cnt:
                ans.append(num)
        return ans
```

### Explanation
1. **Tree Traversal**: The code uses a stack to implement DFS. Starting from the root, each node is processed by popping it from the stack. The node's children are pushed onto the stack in a right-to-left order to ensure correct traversal.
2. **Frequency Counting**: The frequency dictionary (`freq`) keeps track of how many times each node's value appears in the tree. For each node visited, its value is incremented in the dictionary.
3. **Finding Modes**: After traversing the entire tree, the code checks the frequency dictionary to find the highest frequency. All values with this highest frequency are collected into the result list (`ans`), which is then returned.

This approach efficiently counts the frequencies of all node values and determines the modes by leveraging a simple stack-based DFS traversal and dictionary operations. The solution handles edge cases, such as an empty tree, by immediately returning an empty list.
