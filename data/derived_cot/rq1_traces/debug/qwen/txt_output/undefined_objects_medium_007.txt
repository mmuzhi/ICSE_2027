To solve this problem, we need to construct a quad-tree from a given grid. The quad-tree is a hierarchical data structure where each node represents a square region of the grid. The node can be a leaf (if all values in its region are the same) or a non-leaf node (if the region contains mixed values, which are then divided into four smaller quadrants).

### Approach
1. **Problem Analysis**: The goal is to build a quad-tree from a square grid where each cell is either 0 or 1. The quad-tree should recursively divide the grid into four quadrants until each leaf node's region contains only one type of value.
2. **Intuition**: The solution involves recursively splitting the grid into four quadrants. If all values in a quadrant are the same, it becomes a leaf node. Otherwise, it is split into four smaller quadrants, and the process repeats.
3. **Algorithm Selection**: The algorithm uses a divide-and-conquer approach. The grid is divided into four equal parts recursively. Each part is processed to determine if it forms a leaf node or needs further division.
4. **Complexity Analysis**: The time complexity is O(n^2) where n is the side length of the grid, as each cell is processed once. The space complexity is O(log n) due to the recursion depth.

class Solution:
    def construct(self, grid: List[List[int]]) -> 'Node':
        level = len(grid)
