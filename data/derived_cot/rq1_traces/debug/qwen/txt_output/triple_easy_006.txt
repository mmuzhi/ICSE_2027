=== RAW ANSWER ===

To solve this problem, we need to calculate the total score of all rounds based on a series of operations applied to a stack of scores. The operations include doubling the last score, removing the last score, adding the last two scores, and directly inputting a new score. The challenge lies in correctly interpreting each operation and maintaining the stack of scores efficiently.

### Approach
1. **Stack Initialization**: Use a stack to keep track of the scores as they are modified by each operation.
2. **Operation Handling**:
   - **'D' Operation**: Double the last score in the stack and push the result onto the stack.
   - **'C' Operation**: Remove the last score from the stack.
   - **'+' Operation**: Add the last two scores in the stack and push the result onto the stack.
   - **Integer Operation**: Convert the string to an integer and push it onto the stack.
3. **Result Calculation**: After processing all operations, sum all the elements in the stack to get the total score.

### Solution Code
```python
class Solution:
    def calPoints(self, operations: List[str]) -> int:
        stack = []
        for op in operations:
            if op == 'D':
                stack.append(2 * stack[-1])
            elif op == 'C':
                stack.pop()
            elif op == '+':
                stack.append(stack[-1] + stack[-2])
            else:
                stack.append(int(op))
        return sum(stack)
```

### Explanation
- **Stack Initialization**: The stack is initialized as an empty list to store the scores.
- **Processing Operations**:
  - For the 'D' operation, the last score in the stack is doubled and the result is pushed onto the stack.
  - For the 'C' operation, the last score is removed from the stack.
  - For the '+' operation, the sum of the last two scores is computed and pushed onto the stack.
  - For any other operation, the string is converted to an integer and pushed onto the stack.
- **Result Calculation**: The total score is obtained by summing all elements in the stack after processing all operations.

This approach efficiently handles each operation by directly manipulating the stack, ensuring correctness and simplicity. The time complexity is O(n) where n is the number of operations, as each operation is processed in constant time. The space complexity is O(n) to store the stack of scores.
