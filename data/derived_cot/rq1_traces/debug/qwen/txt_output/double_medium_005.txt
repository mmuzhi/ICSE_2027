To solve this problem, we need to determine if a given string containing '(', ')', and '*' characters is valid. The validity is defined by the ability to use '*' as either '(' or ')' to ensure all parentheses are properly matched and balanced.

### Approach
1. **Problem Analysis**: The problem requires checking if all parentheses in the string can be matched correctly, considering that '*' can act as both opening and closing parentheses. The key challenge is to handle the flexibility of '*' while ensuring the parentheses are balanced in a valid manner.

2. **Intuition**: We can use two stacks to keep track of the indices of unmatched '(' and unmatched '*'. As we iterate through the string:
   - For '(', push its index onto the '(' stack.
   - For '*', push its index onto the '*' stack.
   - For ')', try to pop from the '(' stack first. If not possible, pop from the '*' stack (treating '*' as a closing parenthesis). If neither is available, the string is invalid.

3. **Algorithm Selection**: After processing all characters, any remaining unmatched '(' must be matched with unmatched '*' that appear after them. This is done by checking if for each unmatched '(', there exists a '*' with a higher index (i.e., appearing later in the string). This ensures the '*' can be used to close the '(' without violating the order of characters.

4. **Complexity Analysis**: The algorithm processes each character in the string exactly once, making it O(n) time complexity, where n is the length of the string. The space complexity is O(n) in the worst case, as we might store all indices in the stacks.

### Solution Code
```python
class Solution:
    def checkValidString(self, s):
        st = []
        sr = []
        n = len(s)
        for i in range(n): 
            if s[i] == '(':
                st.append(i)
            elif s[i] == '*':
                sr.append(i)
            else:
                if st:
                    st.pop()
                elif sr:
                    sr.pop()
                else:
                    return False
        j = 0
        for i in range(len(st)):
            while j < len(sr) and sr[j] <= st[i]:
                j += 1
            if j >= len(sr):
                return False
            j += 1
        return True
```

4. **Result**: If all '(' can be matched with appropriate '*', the string is valid; otherwise, it is invalid.
