=== ANSWER ===

To solve this problem, we need to determine if a given 6-digit number meets specific conditions regarding the counts of its digits. The conditions are that the digit '1' must appear exactly once, the digit '2' must appear exactly twice, and the digit '3' must appear exactly three times. 

### Approach
1. **Input Handling**: Read the input number as a string to easily access each digit.
2. **Digit Counting**: Count the occurrences of the digits '1', '2', and '3' in the input string.
3. **Condition Check**: Verify if the counts of '1', '2', and '3' match the required conditions (exactly one '1', exactly two '2's, and exactly three '3's).
4. **Output Result**: Print "Yes" if all conditions are satisfied, otherwise print "No".

The solution leverages the fact that the total number of digits is 6, and the sum of the required counts (1 + 2 + 3) is exactly 6. This ensures that any digit not '1', '2', or '3' would violate the conditions, but the problem does not explicitly require checking for other digits. The conditions are sufficient to ensure the solution is correct.

### Solution Code
```python
def main():
    s = input().strip()
    count_1 = s.count('1')
    count_2 = s.count('2')
    count_3 = s.count('3')
    
    if count_1 == 1 and count_2 == 2 and count_3 == 3:
        print("Yes")
    else:
        print("No")

if __name__ == '__main__':
    main()
```

### Explanation
1. **Input Handling**: The input is read as a string to facilitate digit counting.
2. **Digit Counting**: The `count` method is used to tally the occurrences of '1', '2', and '3' in the input string.
3. **Condition Check**: The solution checks if the counts of '1', '2', and '3' are exactly 1, 2, and 3, respectively. If all conditions are met, it prints "Yes"; otherwise, it prints "No".
4. **Efficiency**: The approach efficiently checks the conditions in linear time relative to the number of digits, which is optimal for this problem given the fixed 6-digit constraint. The solution is straightforward and leverages built-in string methods for simplicity and clarity.
