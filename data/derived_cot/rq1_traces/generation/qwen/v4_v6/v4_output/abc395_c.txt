To solve this problem, we need to find the shortest contiguous subarray in a given sequence that contains at least one repeated value. The repeated value must occur at least twice in the entire array. The solution involves identifying all numbers that appear more than once in the array and then determining the minimal window length that contains at least two occurrences of any such number.

### Approach
1. **Problem Analysis**: The task is to find the smallest contiguous subarray where at least one number repeats. The key observation is that for any number appearing at least twice in the array, the minimal window containing two of its occurrences is determined by the smallest gap between any two consecutive occurrences of that number.
2. **Intuition**: For each number that appears multiple times, we can record the indices of its occurrences. The minimal window length for that number is the smallest difference between consecutive indices plus one (to include both occurrences). The overall solution is the smallest such window length across all numbers.
3. **Algorithm Selection**: We use a dictionary to map each number to the list of its indices in the array. For each number with at least two occurrences, we iterate through its list of indices to find the smallest gap between consecutive occurrences. The minimal window length from all such gaps is our answer.
4. **Optimization**: The algorithm efficiently processes the array in O(N) time by leveraging the dictionary to store indices and then scanning these indices once. The space complexity is O(N) to store the indices for each number.
5. **Complexity Analysis**: The algorithm runs in O(N) time, where N is the length of the array, as each element is processed once to build the dictionary and once more to find the minimal window. The space complexity is O(N) due to the storage of indices.

### Solution Code
```python
import sys
from collections import defaultdict

def main():
    data = sys.stdin.read().split()
    if not data:
        print(-1)
        return
    n = int(data[0])
    A = list(map(int, data[1:1+n]))
    
    indices_dict = defaultdict(list)
    for idx, num in enumerate(A):
        indices_dict[num].append(idx)
        
    min_window = float('inf')
    for key, lst in indices_dict.items():
        if len(lst) < 2:
            continue
        for i in range(1, len(lst)):
            window_length = lst[i] - lst[i-1] + 1
            if window_length < min_window:
                min_window = window_length
                if min_window == 2:
                    print(2)
                    return
                    
    if min_window == float('inf'):
        print(-1)
    else:
        print(min_window)

if __name__ == '__main__':
    main()
```

### Explanation
1. **Reading Input**: The input is read from standard input, which includes the array length and the array elements.
2. **Dictionary Construction**: A dictionary is built where each key is a number from the array, and the value is a list of indices where that number appears.
3. **Finding Minimal Window**: For each number with at least two occurrences, the algorithm checks the gaps between consecutive indices. The smallest gap (plus one) gives the minimal window length for that number. If any window length is 2 (the smallest possible), the algorithm immediately returns 2.
4. **Output Result**: If no such window is found, the algorithm outputs -1. Otherwise, it outputs the smallest window length found.

This approach efficiently narrows down the problem to checking each number's occurrences and their gaps, ensuring optimal performance and correctness.
