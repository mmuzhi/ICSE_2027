The Manhattan distance between two cells (x1, y1) and (x2, y2) is defined as |x1 - x2| + |y1 - y2|. The solution involves leveraging combinatorial mathematics and modular arithmetic to efficiently compute the required sum.

### Approach
1. **Problem Analysis**: The problem requires summing the Manhattan distances for all pairs of distinct cells in the grid, considering each possible arrangement of k pieces. The key insight is that the total sum can be broken down into the sum of row differences and column differences separately, then combined.
2. **Combinatorial Mathematics**: The number of valid arrangements is given by the combination C(m*n - 2, k - 2), which counts the ways to choose k-2 cells from the remaining m*n - 2 cells after fixing two specific cells. This combination is computed modulo 10^9 + 7.
3. **Sum of Manhattan Distances**: The sum of Manhattan distances for all unordered pairs of distinct cells is derived by summing the absolute differences in row indices and column indices separately. The row sum is computed using the formula for the sum of squares and linear sums up to m-1, and similarly for columns.
4. **Modular Arithmetic**: Given the large numbers involved, all calculations are performed modulo 10^9 + 7. The final result is obtained by multiplying the combination value with the computed sum of Manhattan distances, adjusted for modular arithmetic.

```python
mod = 10**9 + 7
mod2 = 2 * mod

class Solution:
    def distanceSum(self, m: int, n: int, k: int) -> int:
        total = m * n
        if total < k:
            return 0
        
        max_val = 100000
        fact = [1] * (max_val + 1)
        inv_fact = [1] * (max_val + 1)
        
        for i in range(1, max_val + 1):
            fact[i] = fact[i - 1] * i % mod
        
        inv_fact[max_val] = pow(fact[max_val], mod - 2, mod)
        for i in range(max_val, 0, -1):
            inv_fact[i - 1] = inv_fact[i] * i % mod
        
        def nCr(n, r):
            if r < 0 or r > n:
                return 0
            return fact[n] * inv_fact[r] % mod * inv_fact[n - r] % mod
        
        def sum_sq(n):
```
