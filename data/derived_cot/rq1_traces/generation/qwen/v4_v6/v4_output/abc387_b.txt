=== ANSWER ===

To solve this problem, we need to find the sum of all integers in a 9x9 multiplication table that are not equal to a given integer X. The multiplication table consists of all products of integers from 1 to 9, resulting in 81 distinct integers. The solution involves calculating the total sum of all these integers and then subtracting the sum of all occurrences of X in the table.

### Approach
1. **Calculate Total Sum**: The sum of all integers in the 9x9 multiplication table can be derived using the formula for the sum of the first n natural numbers. Specifically, the sum is the square of the sum of the first 9 natural numbers (since each row's sum is a multiple of the row number). The sum of the first 9 natural numbers is 45, so the total sum is 45² = 2025.
2. **Count Occurrences of X**: For each integer i from 1 to 9, check if X is divisible by i. If it is, compute j = X / i. If j is also between 1 and 9, then the pair (i, j) is a valid occurrence of X in the table. The count of such valid pairs gives the number of times X appears in the table.
3. **Compute Result**: Subtract the product of the count of X and X itself from the total sum to get the desired result. If X does not appear in the table, the count will be zero, and the result remains the total sum.

### Solution Code
```python
X = int(input().strip())
total_sum = 2025
count = 0
for i in range(1, 10):
    if X % i == 0:
        j = X // i
        if 1 <= j <= 9:
            count += 1
result = total_sum - count * X
print(result)
```

### Explanation
1. **Input Handling**: The integer X is read from standard input.
2. **Total Sum Calculation**: The total sum of all integers in the 9x9 multiplication table is known to be 2025, as calculated by the formula (sum of first 9 natural numbers)².
3. **Counting Occurrences**: For each integer i from 1 to 9, the code checks if X is divisible by i. If it is, it calculates j as X divided by i. If j is within the range [1, 9], it increments the count of occurrences.
4. **Result Calculation**: The result is obtained by subtracting the sum of all occurrences of X (count * X) from the total sum. This gives the sum of all integers in the table that are not X.
5. **Output**: The result is printed.

This approach efficiently computes the desired sum by leveraging mathematical properties of the multiplication table and straightforward iteration to count occurrences of X. The solution handles all edge cases, including when X does not appear in the table, by simply returning the total sum.
