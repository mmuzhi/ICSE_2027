=== ANSWER ===

To solve this problem, we need to rearrange the digits of a given three-digit integer into two different orders and then print the resulting integers. The first arrangement involves swapping the hundreds and tens digits, while the second arrangement involves swapping the hundreds and ones digits.

### Approach
1. **Input Handling**: Read the input as a string to easily access each digit.
2. **Digit Extraction**: Extract the hundreds, tens, and ones digits from the input string.
3. **First Arrangement**: Form the first integer by placing the tens digit first, followed by the ones digit, and then the hundreds digit.
4. **Second Arrangement**: Form the second integer by placing the ones digit first, followed by the hundreds digit, and then the tens digit.
5. **Output**: Print both integers separated by a space.

### Solution Code
```python
def main():
    s = input().strip()
    a, b, c = s[0], s[1], s[2]
    num1 = int(b + c + a)
    num2 = int(c + a + b)
    print(f"{num1} {num2}")

if __name__ == "__main__":
    main()
```

### Explanation
1. **Input Handling**: The input is read as a string to facilitate digit extraction. For example, if the input is "379", the string `s` will be "379".
2. **Digit Extraction**: The digits are extracted into variables `a`, `b`, and `c` representing the hundreds, tens, and ones digits respectively. For "379", `a` is '3', `b` is '7', and `c` is '9'.
3. **First Arrangement**: The first integer is formed by concatenating `b`, `c`, and `a` (i.e., '7', '9', '3') resulting in 793.
4. **Second Arrangement**: The second integer is formed by concatenating `c`, `a`, and `b` (i.e., '9', '3', '7') resulting in 937.
5. **Output**: The two integers are printed in the specified format, separated by a space. For the input "379", the output will be "793 937".

This approach efficiently rearranges the digits and converts them into integers for the required output, leveraging string manipulation for simplicity and clarity.
