To solve this problem, we need to count the number of contiguous subsequences (subarrays) in a given sequence of integers that form an arithmetic progression. An arithmetic progression is defined as a sequence where the difference between consecutive elements is constant. 

### Approach
1. **Problem Analysis**: The problem requires counting all contiguous subarrays where the difference between consecutive elements is the same throughout the subarray. Single-element subarrays are always valid, and two-element subarrays are also valid since any two numbers form an arithmetic progression. For longer subarrays, the difference between each pair of consecutive elements must be the same.
2. **Intuition**: For each starting index in the array, we can determine the longest subarray starting at that index which forms an arithmetic progression. The number of valid subarrays starting at that index is equal to the length of this longest subarray. This is because any contiguous part of this subarray will also be an arithmetic progression.
3. **Algorithm Selection**: We use a two-pointer technique where for each starting index, we extend the subarray as long as the difference between consecutive elements remains constant. The difference is determined by the first two elements of the subarray. The algorithm efficiently checks each possible extension in linear time by moving a pointer forward until the condition breaks.
4. **Complexity Analysis**: The algorithm processes each element once, and the while loop inside the main loop runs in linear time overall (each element is visited at most once). Thus, the total time complexity is O(N), where N is the number of elements in the array.

import sys

def main():

        print(0)
