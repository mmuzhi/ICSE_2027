        freq = [0] * (max_val+1)
        for num in nums:
            if num <= max_val:
                freq[num] += 1

        # But we can do: for d from 1 to max_val, then for multiples
        # Alternatively, we can iterate over numbers and mark divisors, but the above is standard
        for d in range(1, max_val+1):

            for j in range(d, max_val+1, d):

To solve this problem, we need to find the GCD of all possible pairs in an array, sort these GCD values, and then answer multiple queries about the elements at specific indices in the sorted list. Given the constraints, a brute-force approach is infeasible, so we use a mathematical approach involving frequency arrays and inclusion-exclusion principles.

### Approach
1. **Frequency Array**: First, we create a frequency array to count occurrences of each number in the input array.
2. **Count Array**: For each divisor `d`, we count how many numbers in the array are divisible by `d`. This helps in determining how many pairs can potentially have a GCD divisible by `d`.
3. **Exact GCD Count**: Using inclusion-exclusion, we compute the exact number of pairs for each divisor `d` where the GCD is exactly `d`. This involves subtracting pairs that have GCDs which are multiples of `d`.
4. **Cumulative Count**: We build a cumulative array where each entry at index `d` holds the total number of pairs with GCD less than or equal to `d`.
5. **Query Processing**: For each query, we use binary search on the cumulative array to find the smallest GCD value such that the cumulative count of pairs up to that GCD is greater than or equal to the query index plus one.

### Solution Code
```python
class Solution:
    def gcdValues(self, nums: List[int], queries: List[int]) -> List[int]:
        if not nums:
            return []
        max_val = max(nums)
        n = len(nums)
        freq = [0] * (max_val + 1)
        for num in nums:
            if num <= max_val:
                freq[num] += 1
        
        count = [0] * (max_val + 1)
        for d in range(1, max_val + 1):
            total = 0
            for j in range(d, max_val + 1, d):
                total += freq[j]
            count[d] = total
        
        exact = [0] * (max_val + 1)
        for d in range(max_val, 0, -1):
            exact[d] = count[d] * (count[d] - 1) // 2
            for k in range(2, (max_val // d) + 1):
                if k * d <= max_val:
                    exact[d] -= exact[k * d]
        
        cumulative = [0] * (max_val + 1)
        if max_val >= 1:
            cumulative[1] = exact[1]
        for d in range(2, max_val + 1):
            cumulative[d] = cumulative[d - 1] + exact[d]
        
        res = []
        for k in queries:
            low, high = 1, max_val
            while low < high:
                mid = (low + high) // 2
                if cumulative[mid] >= k + 1:
                    high = mid
                else:
                    low = mid + 1
            res.append(low)
        return res
```

### Explanation
1. **Frequency Array**: The `freq` array counts how many times each number appears in the input array.
2. **Count Array**: The `count` array for each divisor `d` is populated by summing frequencies of all numbers divisible by `d`.
3. **Exact GCD Count**: The `exact` array calculates the number of pairs with GCD exactly `d` by subtracting pairs that have GCDs which are multiples of `d` from the total pairs divisible by `d`.
4. **Cumulative Count**: The `cumulative` array stores the prefix sum of the `exact` array, allowing us to quickly determine how many pairs have GCDs less than or equal to any given `d`.
5. **Query Processing**: For each query, binary search is used on the `cumulative` array to find the smallest GCD value such that the cumulative count of pairs up to that GCD meets or exceeds the query index plus one, ensuring efficient query resolution even for large input sizes.
