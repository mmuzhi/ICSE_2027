=== ANSWER ===

To solve this problem, we need to determine the number of empty squares in an 8x8 grid where placing a piece would not result in it being captured by any existing pieces. The capturing condition is defined such that a piece captures any other piece in the same row or column. 

### Approach
1. **Problem Analysis**: The grid consists of 64 squares, each either empty ('.') or occupied by a piece ('#'). The goal is to count all empty squares where placing a piece would not be captured by any existing piece. A piece is captured if there is any existing piece in the same row or column.
2. **Intuition**: For a square to be safe, two conditions must be met:
   - The entire row of the square must not contain any existing pieces.
   - The entire column of the square must not contain any existing pieces.
3. **Algorithm Selection**: 
   - **Precompute Row and Column Conditions**: For each row and column, check if there is at least one '#' (existing piece). If a row or column contains any '#', it is marked as unsafe.
   - **Count Safe Squares**: For each empty square ('.'), check if its row and column are both marked as safe (i.e., no existing pieces in them). If both conditions are satisfied, count the square as safe.
4. **Complexity Analysis**: The algorithm involves two passes over the grid: one to determine row and column conditions (O(64) operations) and another to count safe squares (O(64) operations). Thus, the overall complexity is O(n^2) where n is the grid size (8), making it efficient for the given constraints.

### Solution Code
```python
def main():
    grid = []
    for _ in range(8):
        grid.append(input().strip())
    
    row_has_piece = [False] * 8
    for i in range(8):
        if '#' in grid[i]:
            row_has_piece[i] = True
            
    col_has_piece = [False] * 8
    for j in range(8):
        for i in range(8):
            if grid[i][j] == '#':
                col_has_piece[j] = True
                break
                
    count = 0
    for i in range(8):
        for j in range(8):
            if grid[i][j] == '.' and not row_has_piece[i] and not col_has_piece[j]:
                count += 1
                
    print(count)

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The grid is read as 8 strings of length 8 each.
2. **Row Check**: For each row, we check if there is any '#' present. If found, the row is marked as unsafe (`row_has_piece[i] = True`).
3. **Column Check**: For each column, we check all rows to see if there is any '#' in that column. If found, the column is marked as unsafe (`col_has_piece[j] = True`).
4. **Counting Safe Squares**: For each empty square ('.'), we check if its row and column are both safe (no '#'). If both conditions are met, the square is counted as safe.
5. **Output**: The total count of safe squares is printed.

This approach efficiently checks all possible positions for placing a piece while ensuring it remains uncaptured by any existing pieces, leveraging precomputed row and column conditions for optimal performance.
