=== ANSWER ===

To solve this problem, we need to determine the validity of Snuke's hand-raising behavior based on the given inputs L and R. The solution involves checking specific conditions to decide whether Snuke is following the instructions correctly, partially following them, or not following them at all.

### Approach
1. **Problem Analysis**: The problem requires us to check two binary inputs, L and R, each representing whether Snuke is raising his left or right hand (1 for raising, 0 for not raising). The conditions to consider are:
   - If Snuke is raising only one hand (either left or right), print "Yes".
   - If Snuke is raising both hands or none, print "Invalid".

2. **Key Insight**: The solution hinges on evaluating the combination of L and R values. Specifically:
   - The combination (1, 0) means only the left hand is raised, so the output should be "Yes".
   - The combination (0, 1) means only the right hand is raised, so the output should be "Yes".
   - The combinations (1, 1) and (0, 0) indicate both hands are raised or none are raised, respectively, so the output should be "Invalid".

3. **Algorithm Selection**: The solution involves straightforward conditional checks to determine the correct output based on the input values. No complex data structures or algorithms are needed, just simple if-else logic.

### Solution Code
```python
def main():
    data = input().split()
    L = int(data[0])
    R = int(data[1])
    
    if L == 1 and R == 0:
        print("Yes")
    elif L == 0 and R == 1:
        print("Yes")
    else:
        print("Invalid")

if __name__ == "__main__":
    main()
```

### Explanation
1. **Input Handling**: The code reads two integers, L and R, from the input. These integers represent whether Snuke is raising his left or right hand.
2. **Condition Checks**: The code checks if the combination of L and R matches the valid conditions (either (1, 0) or (0, 1)). If so, it prints "Yes". For any other combination (either both hands raised or none raised), it prints "Invalid".
3. **Output**: Based on the conditions, the code outputs the appropriate response ("Yes" or "Invalid") to the console.

This approach efficiently handles all possible input combinations by leveraging simple conditional checks, ensuring correctness with minimal computational overhead.
