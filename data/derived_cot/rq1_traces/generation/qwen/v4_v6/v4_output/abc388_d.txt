To solve this problem, we need to determine the number of stones each alien will have after N years, considering that each alien becomes an adult at a specific year and receives stones from other adults at that time. The key challenge is efficiently calculating the number of stones each alien has after all the gift exchanges over the years.

### Approach
1. **Problem Analysis**: The problem involves N aliens, each becoming an adult at year i (for the i-th alien). When an alien becomes an adult, every other adult (from previous years) with at least one stone gives one stone to the new adult. The goal is to compute the final number of stones each alien has after N years.

2. **Insight**: The solution hinges on determining for each alien the number of gifts it receives before its birthday (from other adults) and the number of gifts it gives after its birthday. The final stone count for each alien is derived from its initial stones plus received gifts minus the gifts it gives.

3. **Algorithm Selection**: We use a Fenwick Tree (Binary Indexed Tree) to efficiently count the number of aliens who have not run out of stones by each year. This allows us to compute the number of gifts received by each alien in linear time.

4. **Complexity Analysis**: The algorithm processes each alien once, performing a query and an update operation on the Fenwick Tree for each. Each operation is O(log M), where M is the maximum value (set to 1,500,000 to cover all possible values). Thus, the overall complexity is O(N log M), which is efficient for the given constraints (N ≤ 500,000).

### Solution Code
```python
import sys

def main():
    data = sys.stdin.read().split()
    if not data:
        return
    n = int(data[0])
    A = list(map(int, data[1:1+n]))
    
    MAX_VAL = 1500000
    tree = [0] * (MAX_VAL + 2)
    
    def update(x, delta):
        idx = x + 1
        while idx <= MAX_VAL + 1:
            tree[idx] += delta
            idx += idx & -idx

    def query(x):
        if x < 0:
            return 0
        s = 0
        idx = x + 1
        while idx:
            s += tree[idx]
            idx -= idx & -idx
        return s

    R = [0] * n
    X = [0] * n
    
    for i in range(n):
        if i == 0:
            count = 0
        else:
            total = query(MAX_VAL)
            if i >= 0:
                count = total - query(i)
            else:
                count = total
        R[i] = count
        X[i] = i + A[i] + R[i]
        if X[i] <= MAX_VAL:
            update(X[i], 1)
        else:
            pass

    B = [0] * n
    for i in range(n):
        initial_stones = A[i]
        received = R[i]
        total_stones = initial_stones + received
        gifts_given = min(total_stones, n - i - 1)
        B[i] = total_stones - gifts_given

    print(" ".join(map(str, B)))

if __name__ == "__main__":
    main()
```
