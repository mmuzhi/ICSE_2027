### Approach
1. **Prime Factorization Insight**: The number of divisors of a number is determined by its prime factorization. For a number \( X = \prod_{p} p^{e_p} \), the number of divisors is \( \prod_{p} (e_p + 1) \). Here, \( e_p \) is the exponent of prime \( p \) in the factorization of \( X \).

3. **Geometric Series**: The sum of geometric series \( \sum_{k=1}^{N} r^k = r \frac{r^N - 1}{r - 1} \) (for \( r \neq 1 \)) allows us to compute the sum efficiently even for large \( N \).

4. **Modular Arithmetic**: Given the result needs to be modulo 998244353, we use modular exponentiation and modular inverses to handle large numbers and divisions.

MOD = 998244353

def main():
    import sys
    data = sys.stdin.read().split()
    N = int(data[0])
    M = int(data[1])
    
    if M == 1:
        print(N % MOD)
        return
        
    primes = [2, 3, 5, 7, 11, 13]
    exp_vectors = []
    for num in range(1, M + 1):
        vec = []
        temp = num
        for p in primes:
            cnt = 0
            while temp % p == 0:
                cnt += 1
                temp //= p
            vec.append(cnt)
        exp_vectors.append(vec)
    
    total_primes = len(primes)
    all_subsets = 1 << total_primes
    U_S = [0] * all_subsets
    
    for s in range(all_subsets):
        prod = 1
        for j in range(total_primes):
            if (s >> j) & 1:
                prod = (prod * exp_vectors[0][j]) % MOD
        U_S[s] = prod
    
    total = 0
    for s in range(all_subsets):
        U_val = U_S[s]
        if U_val == 0:
            continue
        if U_val == 1:
            total = (total + N) % MOD
        else:
            numerator = U_val * (pow(U_val, N, MOD) - 1) % MOD
            denom_inv = pow(U_val - 1, MOD - 2, MOD)
            geom_sum = numerator * denom_inv % MOD
            total = (total + geom_sum) % MOD
            
    print(total % MOD)

if __name__ == '__main__':
    main()
