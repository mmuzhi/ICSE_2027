To solve this problem, we need to find the sum of all good subsequences in an integer array, where a good subsequence is defined as a subsequence where the absolute difference between any two consecutive elements is exactly 1. Single-element subsequences are considered good by default.

### Approach
1. **Problem Analysis**: The problem requires us to consider all possible subsequences of the given array where each consecutive pair of elements in the subsequence differs by exactly 1. The challenge is to efficiently compute the sum of all elements in these subsequences without enumerating all possible subsequences explicitly, which would be computationally infeasible for large arrays.

2. **Dynamic Programming (DP) Setup**: We use two DP arrays, `dp_count` and `dp_sum`, where:
   - `dp_count[x]` stores the number of good subsequences ending with the number `x`.
   - `dp_sum[x]` stores the sum of all elements in these subsequences ending with `x`.

3. **Processing Each Element**: For each element `x` in the array:
   - **New Subsequence**: Start a subsequence with `x` alone, contributing `1` to `dp_count[x]` and `x` to `dp_sum[x]`.
   - **Extending Subsequences**: Extend existing subsequences ending with `x-1` or `x+1` (if they exist) by appending `x` to them. For each such subsequence, the count increases by `dp_count[x-1]` or `dp_count[x+1]`, and the sum increases by `dp_sum[x-1] + x * dp_count[x-1]` or `dp_sum[x+1] + x * dp_count[x+1]` respectively.

4. **Modulo Operation**: Since the result can be very large, all operations are performed modulo \(10^9 + 7\).

5. **Result Calculation**: After processing all elements, the total sum is obtained by summing up all values in `dp_sum` across all possible numbers (from 0 to \(10^5\)).
