To solve this problem, we need to compute the sum of distinct values in every contiguous subsequence of a given sequence of integers. The challenge is to efficiently calculate this sum without resorting to a brute-force approach, which would be computationally expensive for large sequences.

### Approach
1. **Problem Analysis**: The problem requires summing the distinct values of all contiguous subsequences. A brute-force approach would involve checking every possible subsequence, which is infeasible for large sequences (up to 200,000 elements). Instead, we leverage the insight that the sum can be broken down into counting the occurrences of each distinct number across all relevant subsequences.

2. **Key Insight**: For each distinct number in the sequence, count how many contiguous subsequences contain at least one occurrence of that number. The total sum is the sum of these counts for all distinct numbers. This transforms the problem into efficiently calculating, for each number, the number of subsequences that include it.

3. **Algorithm Selection**: 
   - **Total Subsequences Calculation**: The total number of contiguous subsequences in a sequence of length \( N \) is \( N(N+1)/2 \).
   - **Gap Analysis**: For each distinct number, determine the segments (gaps) in the sequence where the number does not appear. The number of subsequences not containing the number is the sum of the triangular numbers of the lengths of these gaps. Subtract this from the total subsequences to get the count of subsequences containing the number.
   - **Efficiency**: By processing each number's occurrences and their gaps, we efficiently compute the required counts in linear time relative to the input size.

### Solution Code
```python
import sys
from collections import defaultdict

def main():
    data = sys.stdin.read().split()
    n = int(data[0])
    A = list(map(int, data[1:1+n]))
    
    total_subsequences = n * (n + 1) // 2
    
    d = defaultdict(list)
    for i, num in enumerate(A):
        d[num].append(i)
    
    ans = 0
    for num, positions in d.items():
        gaps = []
        gaps.append(positions[0])
        for i in range(1, len(positions)):
            gaps.append(positions[i] - positions[i-1] - 1)
        gaps.append(n - 1 - positions[-1])
        
        total_gap = 0
        for L in gaps:
            total_gap += L * (L + 1) // 2
        
        count_for_num = total_subsequences - total_gap
        ans += count_for_num
    
    print(ans)

if __name__ == "__main__":
    main()
```

1. **Reading Input**: The input is read from standard input, which includes the sequence length \( N \) and the sequence \( A \).
2. **Total Subsequences Calculation**: The total number of contiguous subsequences in \( A \) is computed as \( N(N+1)/2 \).
3. **Dictionary Construction**: A dictionary maps each distinct number in \( A \) to a list of its indices (0-based) in the sequence.
4. **Gap Analysis for Each Number**: For each distinct number, the gaps (segments where the number does not appear) are identified. The number of subsequences not containing the number is the sum of triangular numbers of these gap lengths.
5. **Result Calculation**: For each number, the count of subsequences containing it is derived by subtracting the total gap subsequences from the total subsequences. The sum of these counts for all distinct numbers gives the final answer.

This approach efficiently breaks down the problem into manageable parts, leveraging mathematical insights to avoid brute-force computation, ensuring optimal performance even for large sequences.
