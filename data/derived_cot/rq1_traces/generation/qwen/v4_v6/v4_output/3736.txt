To solve this problem, we need to find the first valid adjacent pair of digits in a given string where each digit in the pair appears exactly as many times as its numeric value. The solution involves checking each adjacent pair in the string from left to right and verifying the conditions for both digits in the pair.

### Approach
1. **Frequency Calculation**: First, we calculate the frequency of each digit in the entire string. This helps us determine if a digit meets the condition of appearing exactly as many times as its numeric value.
2. **Adjacent Pair Check**: We then traverse the string from left to right, checking each adjacent pair of digits. For each pair, we ensure the digits are different and that both digits individually satisfy the frequency condition.
3. **Early Termination**: The first valid pair encountered during the traversal is returned immediately. If no such pair is found after checking all possible adjacent pairs, we return an empty string.

### Solution Code
```python
class Solution:
    def findValidPair(self, s: str) -> str:
        freq = [0] * 10
        for char in s:
            digit = int(char)
            freq[digit] += 1
        
        n = len(s)
        for i in range(n - 1):
            a_char = s[i]
            b_char = s[i+1]
            if a_char != b_char:
                a = int(a_char)
                b = int(b_char)
                if freq[a] == a and freq[b] == b:
                    return a_char + b_char
        return ""
```

### Explanation
1. **Frequency Calculation**: The code initializes a frequency array of size 10 (indices 0 to 9) to zero. It then iterates through each character in the string, converting each character to its integer value and incrementing the corresponding index in the frequency array.
2. **Adjacent Pair Check**: The code then iterates through the string, checking each adjacent pair of characters. For each pair, it checks if the digits are different. If they are, it converts both digits to integers and checks if their frequencies in the string match their numeric values.
3. **Early Termination**: If a valid pair is found (both digits meet the frequency condition), the pair is returned immediately. If no valid pair is found after checking all adjacent pairs, the function returns an empty string.

This approach efficiently checks each adjacent pair in the string while leveraging the precomputed frequency array to quickly verify the conditions for both digits in the pair. The solution ensures optimal performance with a time complexity of O(n), where n is the length of the string, as it involves two linear passes through the string.
