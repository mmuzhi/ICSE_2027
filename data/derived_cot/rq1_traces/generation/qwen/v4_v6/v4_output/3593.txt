=== ANSWER ===

To solve this problem, we need to find the maximum factor score of an array after removing at most one element. The factor score is defined as the product of the greatest common divisor (GCD) and the least common multiple (LCM) of all elements in the array. 

### Approach
1. **Problem Analysis**: The problem requires us to compute the factor score for the original array and for each possible array obtained by removing one element. The factor score for an empty array is zero. The key insight is that the factor score can be efficiently computed for any subarray by leveraging prefix and suffix arrays to store GCD and LCM values, allowing us to compute the score for any subarray in constant time after preprocessing.

2. **Intuition**: By precomputing prefix and suffix arrays for both GCD and LCM, we can quickly determine the GCD and LCM of any subarray obtained by removing one element. The GCD of the subarray is the GCD of the left and right parts around the removed element. Similarly, the LCM is the LCM of the left and right parts. 

3. **Algorithm Selection**: 
   - **Prefix and Suffix Arrays**: These arrays store cumulative GCD and LCM values from the start and end of the array, respectively. This allows efficient computation of GCD and LCM for any subarray.
   - **Iterative Calculation**: For each element removal, compute the GCD and LCM of the resulting subarray by combining the relevant prefix and suffix values. The factor score is then the product of these values.

4. **Complexity Analysis**: 
   - **Time Complexity**: O(n^2) in the worst case, where n is the number of elements. This is because we process each element once and compute GCD and LCM for each subarray in constant time after preprocessing.
   - **Space Complexity**: O(n) for storing the prefix and suffix arrays.

### Solution Code
```python
class Solution:
    def maxScore(self, nums: List[int]) -> int:
        n = len(nums)
        
        def gcd(a, b):
            while b:
                a, b = b, a % b
            return a
        
        def lcm(a, b):
            if a == 0 or b == 0:
                return 0
            return a * b // gcd(a, b)
        
        total_gcd = nums[0]
        total_lcm = nums[0]
        for i in range(1, n):
            total_gcd = gcd(total_gcd, nums[i])
            total_lcm = lcm(total_lcm, nums[i])
        ans = total_gcd * total_lcm
        
        prefix_gcd = [0] * n
        suffix_gcd = [0] * n
        prefix_lcm = [0] * n
        suffix_lcm = [0] * n
        
        prefix_gcd[0] = nums[0]
        prefix_lcm[0] = nums[0]
        for i in range(1, n):
            prefix_gcd[i] = gcd(prefix_gcd[i-1], nums[i])
            prefix_lcm[i] = lcm(prefix_lcm[i-1], nums[i])
        
        suffix_gcd[n-1] = nums[n-1]
        suffix_lcm[n-1] = nums[n-1]
        for i in range(n-2, -1, -1):
            suffix_gcd[i] = gcd(suffix_gcd[i+1], nums[i])
            suffix_lcm[i] = lcm(suffix_lcm[i+1], nums[i])
        
        for i in range(n):
            left_gcd = prefix_gcd[i-1] if i-1 >= 0 else 0
            right_gcd = suffix_gcd[i+1] if i+1 < n else 0
            g_val = gcd(left_gcd, right_gcd)
            
            left_lcm_val = prefix_lcm[i-1] if i-1 >= 0 else 1
            right_lcm_val = suffix_lcm[i+1] if i+1 < n else 1
            l_val = lcm(left_lcm_val, right_lcm_val)
            
            score = g_val * l_val
            if score > ans:
                ans = score
        
        return ans
```
