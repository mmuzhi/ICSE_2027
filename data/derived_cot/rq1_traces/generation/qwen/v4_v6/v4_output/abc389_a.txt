=== ANSWER ===

To solve this problem, we need to extract two digits from a given 3-character string where the first and third characters are digits, and the second character is 'x'. The goal is to multiply these two digits and output the result.

### Approach
1. **Extract the First Digit**: The first character of the string is a digit. Convert this character to an integer.
2. **Extract the Third Digit**: The third character of the string is another digit. Convert this character to an integer.
3. **Multiply the Digits**: Multiply the two extracted integers.
4. **Output the Result**: Print the product of the two digits.

This approach efficiently leverages string indexing and basic arithmetic operations to solve the problem. The solution is straightforward given the constraints that the string length is exactly 3 characters and the digits are single characters between '1' and '9'.

### Solution Code
```python
def main():
    S = input().strip()
    a = int(S[0])
    b = int(S[2])
    result = a * b
    print(result)

if __name__ == "__main__":
    main()
```

### Explanation
1. **Input Handling**: The input string `S` is read and stripped of any leading or trailing whitespace.
2. **Digit Extraction**: The first character (at index 0) and the third character (at index 2) of the string are converted from characters to integers using the `int()` function.
3. **Multiplication**: The two integers obtained from the string are multiplied together.
4. **Output**: The result of the multiplication is printed.

This solution efficiently processes the input string by focusing on the relevant characters and performing the necessary arithmetic operation to produce the desired output. The simplicity of the approach ensures clarity and correctness within the given constraints.
