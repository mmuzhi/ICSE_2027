=== ANSWER ===

To solve this problem, we need to process a given string consisting of digits and extract all the occurrences of the digit '2'. The solution involves iterating through each character in the string, checking if it is a '2', and collecting these characters to form a new string. The resulting string should contain all the '2's in their original order.

### Approach
1. **Problem Analysis**: The problem requires us to filter out all characters from the input string except for the digit '2'. The remaining characters should be concatenated in their original order to form the output string.
2. **Intuition**: By iterating through each character in the string and checking if it is '2', we can efficiently collect all the required characters. The order of collection ensures that the resulting string maintains the original sequence of '2's.
3. **Algorithm Selection**: A simple loop or list comprehension can be used to traverse the string and filter out the '2's. The collected characters are then joined into a single string.
4. **Complexity Analysis**: The algorithm processes each character in the string exactly once, resulting in a linear time complexity, O(n), where n is the length of the string. This is optimal for the given problem constraints.

### Solution Code
```python
def main():
    S = input().strip()
    result = ''.join(char for char in S if char == '2')
    print(result)

if __name__ == '__main__':
    main()
```

### Explanation
1. **Reading Input**: The input string `S` is read from standard input.
2. **Filtering Characters**: Using a list comprehension, each character in `S` is checked. If the character is '2', it is included in the resulting list.
3. **Joining Characters**: The collected '2's are joined into a single string using the `join` method.
4. **Output**: The resulting string is printed.

This approach efficiently processes the input string in linear time, ensuring optimal performance while maintaining the order of '2's in the original string. The solution is straightforward and leverages Python's built-in functions for simplicity and clarity.
