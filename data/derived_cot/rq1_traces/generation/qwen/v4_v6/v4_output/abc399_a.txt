=== ANSWER ===

To solve this problem, we need to find the Hamming distance between two strings, which is the number of positions at which the corresponding characters differ. The solution involves reading two strings of equal length and comparing each character at the same position to count the differences.

### Approach
1. **Read Input:** The first line of input is an integer \( N \), which represents the length of the strings. The next two lines are the strings \( S \) and \( T \) of length \( N \).
2. **Compare Characters:** Iterate through each character position in the strings \( S \) and \( T \). For each position, if the characters differ, increment a counter.
3. **Output Result:** The counter holds the Hamming distance, which is printed as the result.

This approach efficiently checks each corresponding character in the two strings, ensuring an accurate count of differing positions. The time complexity is \( O(N) \), where \( N \) is the length of the strings, as each character is compared exactly once.

### Solution Code
```python
def main():
    N = int(input().strip())
    S = input().strip()
    T = input().strip()
    
    count = 0
    for i in range(N):
        if S[i] != T[i]:
            count += 1
            
    print(count)

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input:** The integer \( N \) is read first, followed by the two strings \( S \) and \( T \). The `strip()` method is used to remove any leading or trailing whitespace.
2. **Character Comparison:** A loop runs from 0 to \( N-1 \), comparing each character of \( S \) with the corresponding character in \( T \). If they differ, the counter `count` is incremented.
3. **Output:** After processing all characters, the value of `count` is printed, which represents the Hamming distance between the two strings.

This method ensures that we accurately count all positions where the strings differ, providing the correct Hamming distance efficiently.
