=== ANSWER ===

To solve this problem, we need to determine how many of the given 12 strings have a length equal to their respective index (1-based). The solution involves reading each string, checking if its length matches its position in the list, and counting how many such strings exist.

### Approach
1. **Read Input:** The input consists of 12 strings, each on a separate line. These strings are read into a list.
2. **Check Lengths:** For each string in the list, compare its length with its 1-based index (i.e., the first string's index is 1, the second is 2, etc.).
3. **Count Matches:** Increment a counter each time a string's length matches its index.
4. **Output Result:** After processing all strings, print the count of such matches.

### Solution Code
```python
def main():
    strings = []
    for _ in range(12):
        strings.append(input().strip())
    
    count = 0
    for idx, s in enumerate(strings, start=1):
        if len(s) == idx:
            count += 1
            
    print(count)

if __name__ == '__main__':
    main()
```

### Explanation
1. **Reading Input:** The code reads 12 strings from standard input and stores them in a list called `strings`.
2. **Processing Each String:** Using a loop with `enumerate`, the code checks each string's length against its 1-based index. The `enumerate` function starts counting from 1 to facilitate this comparison.
3. **Counting Matches:** For each string, if its length matches its index, the counter `count` is incremented.
4. **Output:** Finally, the code prints the total count of strings that meet the length condition.

This approach efficiently checks each string once, ensuring optimal performance with a time complexity of O(n), where n is the number of strings (12 in this case). The solution is straightforward and leverages basic string operations and iteration.
