To solve this problem, we need to determine if there exists a permutation \( R \) such that when we connect each point \( P_i \) to \( Q_{R_i} \), no two segments intersect. If such a permutation exists, we need to find one; otherwise, we return -1.

### Approach
1. **Problem Analysis**: The problem requires matching each point from set \( P \) to a point in set \( Q \) such that the segments connecting them do not intersect. The key observation here is that the segments will not intersect if the matching is done in a non-crossing manner. 

2. **Intuition**: The solution involves sorting both sets of points by their x-coordinates (and y-coordinates if x-coordinates are the same) to facilitate a greedy matching approach. By processing points in increasing x-order and ensuring each new segment does not intersect any previously matched segments, we can construct a valid permutation \( R \).

3. **Algorithm Selection**: The algorithm uses a greedy approach where we sort both \( P \) and \( Q \) by their x-coordinates. For each point in the sorted \( P \) list, we try to match it with an available point in the sorted Q list, checking that the new segment does not intersect any already matched segments. The intersection check is performed using the orientation method from computational geometry, which efficiently determines if two segments cross each other.

4. **Complexity Analysis**: The algorithm involves sorting \( O(N \log N) \) time. The main loop runs in \( O(N^3) \) time in the worst case, where \( N \) can be up to 300. This is feasible because \( 300^3 = 27,000,000 \) operations, which is manageable in Python for typical programming competition constraints.

import sys

def main():
    data = sys.stdin.read().strip().split()
    if not data:
        return
    n = int(data[0])
    P = []
    index = 1
    for i in range(n):
        x = int(data[index]); y = int(data[index+1]); index += 2
        P.append((x, y, i))
    Q = []
    for i in range(n):
        x = int(data[index]); y = int(data[index+1]); index += 2
        Q.append((x, y, i))
    
    if n == 0:
        print()
        return
        
    P_sorted = sorted(P, key=lambda x: (x[0], x[1]))
    Q_sorted = sorted(Q, key=lambda x: (x[0], x[1]))
    
    usedQ = [False] * n
    match = [-1] * n

    def orientation(a, b, c):
        ax, ay = a
        bx, by = b
        cx, cy = c
        return (bx - ax) * (cy - ay) - (by - ay) * (cx - ax)
    
    def do_segments_intersect(a, b, c, d):
        o1 = orientation(a, b, c)
        o2 = orientation(a, b, d)
        o3 = orientation(c, d, a)
        o4 = orientation(c, d, b)
        if o1 * o2 < 0 and o3 * o4 < 0:
            return True
        return False

    for i in range(n):
        found = False
        for j in range(n):
            if usedQ[j]:
                continue
            valid = True
            for k in range(i):
                if match[k] == -1:
                    continue
                if do_segments_intersect(P_sorted[i], Q_sorted[j], P_sorted[k], Q_sorted[match[k]]):
                    valid = False
                    break
            if valid:
                usedQ[j] = True
                match[i] = j
                found = True
                break
        if not found:
            print(-1)
            return

    ans = [0] * n
    for i in range(n):
        orig_p = P_sorted[i][2]
        q_index = match[i]
        orig_q = Q_sorted[q_index][2]
        ans[orig_p] = orig_q + 1

    print(" ".join(map(str, ans)))

if __name__ == '__main__':
    main()
