You will be given a competitive programming problem.
There are N dice.
The i-th die has K_i faces, with the numbers A_{i,1}, A_{i,2}, \ldots, A_{i,K_i} written on them.
When you roll this die, each face appears with probability \frac{1}{K_i}.
You choose two dice from the N dice and roll them.
Determine the maximum probability that the two dice show the same number, when the dice are chosen optimally.

Input

The input is given from Standard Input in the following format:
N
K_1 A_{1,1} A_{1,2} \dots A_{1,K_1}
\vdots
K_N A_{N,1} A_{N,2} \dots A_{N,K_N}

Output

Print the answer.
Your answer is considered correct if the absolute or relative error from the true solution does not exceed 10^{-8}.

Constraints


- 2 \leq N \leq 100
- 1 \leq K_i
- K_1 + K_2 + \dots + K_N \leq 10^5
- 1 \leq A_{i,j} \leq 10^5
- All input values are integers.

Sample Input 1

3
3 1 2 3
4 1 2 2 1
6 1 2 3 4 5 6

Sample Output 1

0.333333333333333


- When choosing the 1st and 2nd dice, the probability that the outcomes are the same is \frac{1}{3}.
- When choosing the 1st and 3rd dice, the probability is \frac{1}{6}.
- When choosing the 2nd and 3rd dice, the probability is \frac{1}{6}.

Therefore, the maximum probability is \frac{1}{3} = 0.3333333333\ldots.

Sample Input 2

3
5 1 1 1 1 1
4 2 2 2 2
3 1 1 2

Sample Output 2

To solve this problem, we need to determine the maximum probability that two dice chosen from a given set of dice will show the same number when rolled. The dice can have different numbers of faces, and each face has a specific number written on it. The solution involves calculating the probability for every possible pair of dice and selecting the maximum probability.

### Approach
1. **Problem Analysis**: The problem requires evaluating all pairs of dice to find the pair with the highest probability of showing the same number when rolled. For each die, the probability of rolling a specific number is the count of that number on the die divided by the total number of faces. The combined probability for two dice to show the same number is the sum of the products of the counts of each common number divided by the product of the total faces of both dice.

2. **Intuition**: For each pair of dice, we need to find all numbers common to both dice, compute the product of their counts in each die, and then sum these products. The sum is then divided by the product of the total faces of the two dice to get the probability for that pair. The maximum probability across all pairs is the solution.

3. **Algorithm Selection**: 
   - **Data Structures**: For each die, we store a set of distinct numbers and a frequency dictionary mapping each number to its count on the die. This allows efficient computation of common numbers and their counts between any two dice.
   - **Probability Calculation**: For each pair of dice, compute the intersection of their distinct numbers. For each common number, multiply its counts in both dice and sum these products. Divide this sum by the product of the total faces of the two dice to get the probability for that pair.

4. **Complexity Analysis**: The algorithm involves checking all pairs of dice, which is O(N^2) where N is the number of dice (up to 100). For each pair, the intersection of distinct numbers is computed, which in the worst case is O(K) where K is the number of distinct numbers per die (up to 10^5). However, given the constraints, the total operations are manageable (around 500 million in the worst case), making the solution feasible.

### Solution Code
```python
import sys
from collections import defaultdict

def main():
    data = sys.stdin.read().split()
    if not data:
        return
    
    n = int(data[0])
    dice = []
    index = 1
    for _ in range(n):
        k = int(data[index])
        index += 1
        faces = list(map(int, data[index:index+k]))
        index += k
        
        freq_dict = defaultdict(int)
        distinct_set = set()
        for num in faces:
            freq_dict[num] += 1
            distinct_set.add(num)
            
        dice.append((k, distinct_set, freq_dict))
    
    max_prob = 0.0
    for i in range(n):
        for j in range(i+1, n):
            k1, set1, dict1 = dice[i]
            k2, set2, dict2 = dice[j]
            common = set1 & set2
            total_common = 0.0
            for num in common:
                total_common += dict1[num] * dict2[num]
            prob = total_common / (k1 * k2)
            if prob > max_prob:
                max_prob = prob
                
    print("{:.15f}".format(max_prob))

if __name__ == "__main__":
    main()
```
