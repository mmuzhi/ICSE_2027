To solve this problem, we need to count the number of n-digit integers (without leading zeros) that can be rearranged to form a k-palindromic integer. A k-palindromic integer is a palindrome divisible by k. The solution involves generating all possible n-digit palindromic numbers divisible by k, then determining the count of n-digit numbers (without leading zeros) that can be formed from the digit multisets of these palindromes.

### Approach
1. **Generate Palindromic Numbers**: For given n and k, generate all n-digit palindromic numbers (without leading zeros) that are divisible by k. This involves constructing palindromes by mirroring the first half of the number. For even n, the palindrome is formed by mirroring the entire first half. For odd n, the palindrome is formed by mirroring the first half excluding the middle digit.
2. **Digit Multiset Extraction**: For each generated palindrome, compute the multiset (count of each digit) of its digits.
3. **Count Valid Numbers**: For each unique multiset, compute the number of n-digit numbers (without leading zeros) that can be formed from it. This involves calculating the total permutations of the digits (including those with leading zeros) and subtracting those permutations that start with zero.

### Solution Code
```python
import math
from math import factorial
from itertools import product
from collections import Counter

class Solution:
    def countGoodIntegers(self, n: int, k: int) -> int:
        def multinomial(n_val, counts):
            if sum(counts) != n_val:
                return 0
            num = factorial(n_val)
            den = 1
            for count in counts:
                den *= factorial(count)
            return num // den
        
        if n % 2 == 0:
            m = n // 2
            palindromes_multiset = set()
            for first_half in product(range(10), repeat=m):
                if first_half[0] == 0:
                    continue
                s = ''.join(str(d) for d in first_half) + ''.join(str(d) for d in reversed(first_half))
                num = int(s)
                if num % k == 0:
                    cnt = [0]*10
                    for char in s:
                        cnt[int(char)] += 1
                    palindromes_multiset.add(tuple(cnt))
        else:
            m = (n+1) // 2
            palindromes_multiset = set()
            for first_half in product(range(10), repeat=m):
                if first_half[0] == 0:
                    continue
                s = ''.join(str(d) for d in first_half) + ''.join(str(d) for d in first_half[-2::-1])
                num = int(s)
                if num % k == 0:
                    cnt = [0]*10
                    for char in s:
                        cnt[int(char)] += 1
                    palindromes_multiset.add(tuple(cnt))
        
        total = 0
        for ms in palindromes_multiset:
            T = multinomial(n, ms)
            T0 = 0
            if ms[0] > 0:
                ms0 = list(ms)
                ms0[0] -= 1
                T0 = multinomial(n-1, ms0)
            total += T - T0
        
        return total
```
