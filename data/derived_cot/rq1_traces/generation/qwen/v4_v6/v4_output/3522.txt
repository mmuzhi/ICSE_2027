=== ANSWER ===

To solve this problem, we need to determine the power of all subarrays of a given array `nums` of length `k`. The power of a subarray is defined as the maximum element in the subarray if all elements are consecutive integers and sorted in ascending order; otherwise, it is -1.

### Approach
1. **Problem Analysis**: The task involves checking each contiguous subarray of length `k` in the given array. For each subarray, we need to verify two conditions:
   - **Consecutive Condition**: Each element in the subarray must be exactly one more than the previous element (i.e., the subarray forms a sequence of consecutive integers).
   - **Sorted Condition**: The subarray must be in ascending order. However, the consecutive condition inherently ensures the sorted condition because each subsequent element is larger by exactly one.

2. **Intuition**: For each subarray of length `k`, iterate through the elements to check if each adjacent pair satisfies the consecutive condition (i.e., `nums[i+j] == nums[i+j-1] + 1`). If all adjacent pairs meet this condition, the subarray is valid, and its power is the last element. Otherwise, the power is -1.

3. **Algorithm Selection**: Use a sliding window approach to process each subarray of length `k` one by one. For each window, check the consecutive condition in a single pass. If the condition is satisfied, append the last element of the window to the results; otherwise, append -1.

4. **Complexity Analysis**: The algorithm processes each of the `n - k + 1` subarrays, and for each subarray, it performs up to `k - 1` comparisons. Thus, the time complexity is O(n*k), which is efficient given the constraints (n ≤ 500 and k ≤ n).

### Solution Code
```python
class Solution:
    def resultsArray(self, nums: List[int], k: int) -> List[int]:
        n = len(nums)
        results = []
        for i in range(n - k + 1):
            is_consecutive = True
            for j in range(1, k):
                if nums[i+j] != nums[i+j-1] + 1:
                    is_consecutive = False
                    break
            if is_consecutive:
                results.append(nums[i+k-1])
            else:
                results.append(-1)
        return results
```

### Explanation
1. **Initialization**: The method starts by determining the length of the input array `nums` and initializes an empty list `results` to store the power values for each valid subarray.
2. **Sliding Window Processing**: For each starting index `i` of a subarray of length `k`, the algorithm checks if the subarray from `i` to `i + k - 1` meets the consecutive condition.
3. **Consecutive Condition Check**: For each subarray, the algorithm verifies that each element is exactly one more than the previous element. If any element fails this check, the subarray is invalid, and the result for that subarray is -1.
4. **Result Compilation**: If the subarray meets the consecutive condition, the last element of the subarray (which is the maximum due to the ascending order) is appended to the results list.
5. **Output**: After processing all possible subarrays, the method returns the results list containing the power values for each subarray.

This approach efficiently checks each subarray for the required conditions and compiles the results in linear time relative to the number of subarrays, making it suitable for the given problem constraints.
