To solve this problem, we need to maximize the sum of elements in a sequence S after performing a series of operations where each element from the input array is either appended to S or used to pop the last element of S (if S is not empty). The challenge is to determine the optimal sequence of operations to achieve the maximum possible sum at the end.

### Approach
1. **Problem Analysis**: The problem requires processing each element of the input array in order, deciding at each step whether to append it to the sequence S or to pop the last element of S (if S is not empty). The goal is to maximize the sum of elements in S after all operations. The key observation is that we can only pop the last element of S, which means the elements in S must maintain their order of appearance in the input array.

2. **Intuition**: The optimal strategy involves keeping positive elements in S whenever possible, as they contribute to increasing the sum. For negative elements, we decide to append them only if they do not reduce the sum below a certain threshold. However, if a negative element is appended, it can be popped later if a positive element follows, provided it is at the top of S.

3. **Algorithm Selection**: We use a stack to simulate the sequence S. For each element in the input array:
   - If the element is positive, we push it onto the stack (since it increases the sum).
   - If the element is negative, we push it onto the stack only if the current sum plus this element is greater than the current sum (i.e., it doesn't make the sum worse). Otherwise, we pop the top element of the stack (if it's non-negative) to potentially improve the sum for future positive elements.

4. **Complexity Analysis**: The algorithm processes each element exactly once, performing a constant amount of work per element (push or pop operation on the stack). Thus, the time complexity is O(N), where N is the number of elements in the input array. The space complexity is O(N) in the worst case, as the stack may store all elements if they are all positive.

def main():
    import sys
    data = sys.stdin.read().split()
    n = int(data[0])
    A = list(map(int, data[1:1+n]))
    
    stack = []
    current_sum = 0
    
    for a in A:
        if not stack:
            stack.append(a)
            current_sum += a
        else:
            if a > 0 or (current_sum + a > current_sum):
                stack.append(a)
                current_sum += a
            else:
                if stack[-1] >= 0:
                    stack.pop()
                    current_sum -= stack.pop() if stack else 0
                else:
                    stack.append(a)
                    current_sum += a
    
    print(current_sum)
