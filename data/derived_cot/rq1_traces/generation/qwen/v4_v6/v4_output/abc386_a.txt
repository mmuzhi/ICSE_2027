To solve this problem, we need to determine if adding one more card (an integer between 1 and 13) to the given four cards can form a Full House. A Full House is defined as a set of five cards where there are three cards of one number and two cards of another number.

### Approach
1. **Problem Analysis**: The key observation is that a Full House requires exactly two distinct numbers in the five-card set, with one number appearing three times and the other appearing two times. Given four cards, we need to check if adding one more card can meet this condition.
2. **Intuition**: The solution involves counting the frequency of each distinct number in the four cards. If there are not exactly two distinct numbers, it's impossible to form a Full House by adding one card. If there are two distinct numbers, their frequencies must either be (3, 1) or (2, 2). This means:
   - If the frequencies are (3, 1), adding the card to the number with frequency 1 will make it (3, 2), forming a Full House.
   - If the frequencies are (2, 2), adding a card to either number will make it (3, 2) or (2, 3), both of which form a Full House.
3. **Algorithm Selection**: We use a frequency counter (like Python's `collections.Counter`) to count occurrences of each number. We then check the number of distinct numbers and their frequencies to determine if a Full House can be formed.
4. **Complexity Analysis**: The algorithm involves counting frequencies and sorting, which is efficient with a constant time complexity O(1) since the input size is fixed (four numbers).

### Solution Code
```python
from collections import Counter

def main():
    data = list(map(int, input().split()))
    cnt = Counter(data)
    distinct_count = len(cnt)
    
    if distinct_count != 2:
        print("No")
    else:
        freqs = sorted(cnt.values(), reverse=True)
        if freqs == [3, 1] or freqs == [2, 2]:
            print("Yes")
        else:
            print("No")

if __name__ == "__main__":
    main()
```
