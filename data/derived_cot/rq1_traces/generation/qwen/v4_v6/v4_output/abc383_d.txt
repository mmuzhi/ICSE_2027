To solve this problem, we need to count the number of positive integers not greater than a given number \( N \) that have exactly 9 positive divisors. The solution involves understanding the mathematical properties of numbers with exactly 9 divisors and efficiently counting such numbers without exceeding computational limits for large \( N \).

### Approach
1. **Mathematical Insight**: A number has exactly 9 divisors if it can be expressed in one of two forms:
   - \( p^8 \), where \( p \) is a prime number. This is because the number of divisors is \( (8 + 1) = 9 \).
   - \( p^2 \times q^2 \), where \( p \) and \( q \) are distinct primes. Here, the number of divisors is \( (2 + 1) \times (2 + 1) = 9 \).

2. **First Form Handling**: For numbers of the form \( p^8 \), we generate all primes up to a certain limit (1000 in our case) and check if \( p^8 \leq N \). Since \( p^8 \) grows rapidly, only a few primes will satisfy this condition for large \( N \).

3. **Second Form Handling**: For numbers of the form \( (p \times q)^2 \), we generate all primes up to \( \sqrt{N} \) using the Sieve of Eratosthenes. For each prime \( p \), we count the number of primes \( q \) (where \( q > p \)) such that \( p \times q \leq \sqrt{N} \). This ensures \( (p \times q)^2 \leq N \).

4. **Efficiency**: The Sieve of Eratosthenes efficiently generates primes up to \( \sqrt{N} \), which is feasible even for \( N \) up to \( 4 \times 10^{12} \) (where \( \sqrt{N} \) is \( 2 \times 10^6 \)). Binary search is used to quickly count valid primes \( q \) for each prime \( p \).

### Solution Code
```python
import math
from bisect import bisect_right

def generate_primes_up_to(n):
    if n < 2:
        return []
    sieve = [True] * (n + 1)
    sieve[0] = sieve[1] = False
    for i in range(2, int(math.sqrt(n)) + 1):
        if sieve[i]:
            for j in range(i * i, n + 1, i):
                sieve[j] = False
    return [i for i, flag in enumerate(sieve) if flag]

def main():
    import sys
    data = sys.stdin.read().split()
    if not data:
        print(0)
        return
    try:
        N = int(data[0])
    except Exception:
        print(0)
        return

    if N < 256:
        count_first = 0
    else:
        primes1 = generate_primes_up_to(1000)
        count_first = 0
        for p in primes1:
            if p**8 > N:
                break
            count_first += 1

    M = math.isqrt(N)
    if M < 2:
        count_second = 0
    else:
        primes2 = generate_primes_up_to(M)
        count_second = 0
        for i, p in enumerate(primes2):
            if p > M // p:
                break
            idx = bisect_right(primes2, M // p)
            count_second += idx - i - 1

    total = count_first + count_second
    print(total)

if __name__ == "__main__":
    main()
```
