To solve this problem, we need to determine if it's possible to transform the initial configuration of pieces (given by string A) into a target configuration (given by string B) using the minimum number of operations. Each operation involves choosing a center and moving all pieces simultaneously closer to that center. The target configuration requires that every square with '1' in B must have at least one piece, and every square with '0' must have no pieces.

### Approach
1. **Initial Checks**: 
   - If the number of '1's in A is less than the number of '1's in B, it's impossible to meet the target configuration, so we return -1.
   
2. **Position Analysis**:
   - For each piece in the initial configuration (positions where A has '1'), calculate the minimal number of operations required to move it to any target position (positions where B has '1'). The minimal operations for a piece at position `x` to reach any target position `y` is simply the Manhattan distance `|x - y|`, as each operation moves the piece one step closer to the chosen center.

3. **Determine Minimum Operations**:
   - The answer is the maximum of these minimal operations across all pieces. This is because all pieces must be moved to their respective target positions within the same number of operations, and the longest individual movement dictates the total operations needed.

### Solution Code
```python
import bisect

t = int(input().strip())
results = []

for _ in range(t):
    n = int(input().strip())
    A = input().strip()
    B = input().strip()
    
    count_A = A.count('1')
    count_B = B.count('1')
    
    if count_A < count_B:
        results.append("-1")
        continue
        
    positions_A = [i for i, char in enumerate(A) if char == '1']
    positions_B = [i for i, char in enumerate(B) if char == '1']
    
    if not positions_B:
        results.append("0")
        continue
        
    positions_B_sorted = sorted(positions_B)
    
    def min_distance(x):
        idx = bisect.bisect_left(positions_B_sorted, x)
        candidates = []
        if idx > 0:
            candidates.append(positions_B_sorted[idx-1])
        if idx < len(positions_B_sorted):
            candidates.append(positions_B_sorted[idx])
        return min(abs(x - y) for y in candidates)
    
    ans = 0
    for x in positions_A:
        d = min_distance(x)
        if d > ans:
            ans = d
            
    results.append(str(ans))

for res in results:
    print(res)
```
