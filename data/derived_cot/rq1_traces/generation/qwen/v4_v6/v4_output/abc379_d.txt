To solve this problem, we need to efficiently process a series of queries involving planting plants, waiting for them to grow, and harvesting plants based on their height. The challenge lies in handling large input sizes (up to 200,000 queries) and large values (up to 10^9) efficiently.

### Approach
1. **Problem Analysis**: The problem involves three types of queries:
   - **Type 1**: Plant a new plant with initial height 0.
   - **Type 2**: Increase the height of all existing plants by a given value T.
   - **Type 3**: Harvest all plants with height at least H and output the count of such plants.

2. **Key Insight**: Each plant's current height is determined by the total time (sum of all previous Type 2 queries) minus the time when it was planted. To efficiently handle Type 3 queries, we need a way to quickly find and remove all plants whose planting time is less than or equal to the current total time minus H.

3. **Data Structure**: We use a sorted list to keep track of the planting times (bases) of active plants. For each Type 3 query, we calculate the threshold (current total time - H) and use binary search to find all plants planted at or before this threshold. These plants are then removed from the list.

4. **Complexity Analysis**: 
   - **Type 1 Queries**: Insertion into the sorted list is O(n) per operation, but since we use `bisect.insort`, it's O(n) per insertion.
   - **Type 2 Queries**: Simple arithmetic, O(1).
   - **Type 3 Queries**: Binary search (O(log n)) to find the threshold, and removal of elements (O(k + n) for slicing the list). In the worst case, this can lead to O(n^2) operations, which might be slow for large inputs.

### Solution Code
```python
import bisect

def main():
    import sys
    data = sys.stdin.read().splitlines()
    if not data:
        return
    Q = int(data[0].strip())
    total_wait = 0
    active_bases = []
    output_lines = []
    index = 1
    for _ in range(Q):
        parts = data[index].split()
        index += 1
        if not parts:
            continue
        if parts[0] == '1':
            base = total_wait
            bisect.insort(active_bases, base)
        elif parts[0] == '2':
            T = int(parts[1])
            total_wait += T
        elif parts[0] == '3':
            H_val = int(parts[1])
            X = total_wait - H_val
            if X < 0:
                output_lines.append("0")
            else:
                pos = bisect.bisect_right(active_bases, X)
                if pos == 0:
                    output_lines.append("0")
                else:
                    removed_count = pos
                    active_bases[:] = active_bases[pos:]
                    output_lines.append(str(removed_count))
    for line in output_lines:
        print(line)

if __name__ == "__main__":
    main()
```
