### Approach
1. **Initial Check**: If the starting cell (0, 0) is the destination (m-1, n-1), immediately check if the initial health (adjusted for the starting cell's value) is at least 1.
2. **Health Calculation**: Compute the initial health after accounting for the starting cell's value. If this value is non-positive, return false since further movement is impossible.
3. **BFS Initialization**: Use a queue to manage the BFS traversal, starting from (0, 0). Maintain a 2D array `dp` to store the maximum health remaining at each cell.
4. **Traversal**: For each cell dequeued, explore its adjacent cells. For each adjacent cell, calculate the new health after accounting for its value. If the new health is positive and better than the previously recorded value for that cell, update the `dp` array and enqueue the cell.
5. **Destination Check**: During traversal, if the destination cell (m-1, n-1) is reached with a health value of at least 1, return true immediately.
6. **Termination**: If the queue is exhausted without reaching the destination with sufficient health, return false.

```python
from collections import deque
from typing import List

class Solution:
    def findSafeWalk(self, grid: List[List[int]], health: int) -> bool:
        m = len(grid)
        n = len(grid[0])
        if 0 == m - 1 and 0 == n - 1:
            init_health = health - (1 if grid[0][0] == 1 else 0)
            return init_health >= 1
        
        init_health = health - (1 if grid[0][0] == 1 else 0)
        if init_health <= 0:
            return False
        
        dp = [[-1] * n for _ in range(m)]
        dp[0][0] = init_health
        
        q = deque([(0, 0)])
        dirs = [(1, 0), (-1, 0), (0, 1), (0, -1)]
        
        while q:
            i, j = q.popleft()
            for dx, dy in dirs:
                ni, nj = i + dx, j + dy
                if 0 <= ni < m and 0 <= nj < n:
                    new_health = dp[i][j] - (1 if grid[ni][nj] == 1 else 0)
                    if new_health > 0 and new_health > dp[ni][nj]:
                        dp[ni][nj] = new_health
                        if ni == m - 1 and nj == n - 1:
                            return True
                        q.append((ni, nj))
        return False
```

### Explanation
1. **Initial Check**: The code first checks if the starting cell is the destination. If so, it verifies if the initial health (adjusted for the starting cell's value) is sufficient.
2. **Health Calculation**: The initial health is adjusted by subtracting 1 if the starting cell is unsafe (value 1). If the adjusted health is non-positive, the function returns false immediately.
3. **BFS Initialization**: A queue is initialized with the starting cell (0, 0), and a `dp` array is set up to keep track of the maximum health remaining at each cell.
4. **Traversal**: The BFS processes each cell by exploring its neighbors. For each neighbor, the new health is calculated by subtracting 1 if the neighbor is unsafe. If the new health is positive and better than the recorded value, the `dp` array is updated, and the neighbor is enqueued for further exploration.
5. **Destination Check**: If during traversal the destination cell is reached with a health value of at least 1, the function returns true immediately.
6. **Termination**: If the BFS completes without finding a valid path to the destination with sufficient health, the function returns false.

This approach efficiently explores all possible paths while maximizing the health remaining at each cell, ensuring optimal performance even for larger grids.
