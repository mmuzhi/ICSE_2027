=== ANSWER ===

To solve this problem, we need to determine if a given number of rows (A) can form a rectangular arrangement with exactly 400 people, meaning the number of columns (B) must be a positive integer such that A multiplied by B equals 400. 

### Approach
1. **Problem Analysis**: The problem requires checking if 400 can be divided evenly by the given integer A. If it can, then B is the quotient (400 / A), otherwise, there is no valid B, and we return -1.
2. **Key Insight**: The solution hinges on checking divisibility. Specifically, we need to verify if 400 is divisible by A without any remainder. If it is, then B is simply 400 divided by A; otherwise, no valid B exists.
3. **Algorithm Selection**: The algorithm involves a straightforward divisibility check. Given the constraints (A is between 1 and 400), the solution is efficient with a constant time complexity O(1).

### Solution Code
```python
def main():
    A = int(input().strip())
    if 400 % A == 0:
        B = 400 // A
        print(B)
    else:
        print(-1)

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The code reads the integer A from standard input.
2. **Divisibility Check**: It checks if 400 is divisible by A using the modulus operator. If the remainder is zero, it means A is a divisor of 400.
3. **Output Result**: If A divides 400 evenly, the code calculates B as 400 divided by A and prints B. Otherwise, it prints -1 indicating no valid B exists.
This approach efficiently solves the problem by leveraging basic arithmetic operations and conditional checks.
