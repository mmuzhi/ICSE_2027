You will be given a competitive programming problem.
AtCoder Regular Contest (ARC) is divided into two divisions.

- In ARC Div. 1, participants whose rating at the start of the contest is between 1600 and 2799, inclusive, are subject to rating updates.
- In ARC Div. 2, participants whose rating at the start of the contest is between 1200 and 2399, inclusive, are subject to rating updates.

Takahashi decided to participate in N ARCs.
Initially, his rating is R.
The i-th (1\leq i\leq N) ARC is Div. D _ i, and his performance in that contest is represented by an integer A _ i.

Rating updates for ARCs are performed immediately after the contest ends, and whether he is subject to rating updates in the next contest is determined based on his rating after the update.
Find his rating after finishing the N ARCs.
He does not participate in any contests other than these N ARCs, and his rating does not change in other ways.

- -1000\leq A _ i\leq 1000\ (1\leq i\leq N)
- All input values are integers.

=== ANSWER ===

To solve this problem, we need to determine Takahashi's rating after participating in N ARCs, considering the rating updates based on the division of each contest and his performance. The key is to check the current rating before each contest to decide if an update should be applied.

### Approach
1. **Initialization**: Start with the initial rating R.
2. **Iterate through each contest**: For each contest, check its division (D_i) and the current rating to determine if an update is applicable.
3. **Update Condition**:
   - For Division 1 (D_i = 1), if the current rating is between 1600 and 2799 (inclusive), add the performance value (A_i) to the current rating.
   - For Division 2 (D_i = 2), if the current rating is between 1200 and 2399 (inclusive), add the performance value (A_i) to the current rating.
4. **Output the final rating**: After processing all contests, the final rating is the result.

### Solution Code
```python
def main():
    import sys
    data = sys.stdin.read().split()
    n = int(data[0])
    initial_rating = int(data[1])
    index = 2
    current_rating = initial_rating
    for _ in range(n):
        d = int(data[index])
        a = int(data[index + 1])
        index += 2
        if d == 1:
            if 1600 <= current_rating <= 2799:
                current_rating += a
        else:
            if 1200 <= current_rating <= 2399:
                current_rating += a
    print(current_rating)

if __name__ == "__main__":
    main()
```

3. **Output**: After processing all contests, the final rating is printed. The solution efficiently handles each contest in sequence, updating the rating only when conditions are met, and outputs the result after all updates are processed. 

This approach ensures that the rating is updated correctly based on the contest division and performance, adhering to the specified conditions for each contest.
