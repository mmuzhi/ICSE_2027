=== ANSWER ===

To solve this problem, we need to determine the maximum number of operations we can perform where each operation involves selecting two balls of the same color and discarding them. The solution involves counting the frequency of each color in the given list of four balls and then calculating how many pairs of each color can be formed. The sum of these pairs gives the total number of operations possible.

### Approach
1. **Problem Analysis**: The problem requires us to process four integers, each representing the color of a ball. The goal is to find the maximum number of times we can remove two balls of the same color. The key observation here is that each operation removes two balls of the same color, and the order of operations does not matter. The solution hinges on counting how many pairs of each color can be formed from the four balls.
2. **Intuition**: For each color, the number of operations possible is the floor division of the count of that color by 2. Summing these values across all colors gives the total operations.
3. **Algorithm Selection**: We use a frequency array to count occurrences of each color (1 to 4). Then, for each color, we compute the number of pairs (using integer division by 2) and sum these values.
4. **Complexity Analysis**: The algorithm involves a single pass through the input to count frequencies (O(1) since there are only four elements) and another pass through the frequency array (O(1) since there are only four colors). Thus, the overall complexity is O(1), which is optimal for the given constraints.

### Solution Code
```python
def main():
    arr = list(map(int, input().split()))
    freq = [0] * 5
    for num in arr:
        freq[num] += 1
    result = 0
    for i in range(1, 5):
        result += freq[i] // 2
    print(result)

if __name__ == '__main__':
    main()
```

### Explanation
1. **Reading Input**: The input consists of four integers representing the colors of the balls. These integers are read and stored in a list.
2. **Frequency Counting**: A frequency array of size 5 (to cover colors 1 to 4) is initialized to zero. Each element in the input list increments the corresponding index in the frequency array.
3. **Calculating Operations**: For each color (1 to 4), the number of operations is calculated by integer division of the frequency by 2. These values are summed to get the total operations.
4. **Output Result**: The total number of operations is printed.

This approach efficiently counts the possible operations by leveraging simple arithmetic on the frequency counts of each color, ensuring optimal performance for the problem constraints.
