To solve this problem, we need to count the number of integer pairs (i, j) such that all four corners of the square centered at (i, j) are inside or on the boundary of a circle of radius R centered at the origin. The key insight is that the entire square is contained within the circle if the farthest corner of the square from the origin is within the circle. 

### Approach
1. **Problem Analysis**: The problem requires checking all integer coordinates (i, j) where the square defined by the corners (iÂ±0.5, jÂ±0.5) lies entirely within the circle of radius R. The condition for the square to be inside the circle is that the maximum distance from the origin to any of its corners is â‰¤ R. This translates to the condition \(i^2 + j^2 + |i| + |j| + 0.5 \leq R^2\).

2. **Mathematical Transformation**: To avoid floating-point operations, we transform the condition into an integer form by multiplying by 2: \(2(i^2 + j^2 + |i| + |j|) + 1 \leq 2R^2\). This allows us to use integer arithmetic for precise calculations.

3. **Symmetry Utilization**: The problem is symmetric with respect to all four quadrants. We can iterate over non-negative integers (x, y) where x = |i| and y = |j|, and then account for the symmetry by multiplying the count appropriately. For example, if both x and y are positive, there are four corresponding points in the plane (i, j), (i, -j), (-i, j), and (-i, -j).

4. **Efficient Calculation**: For each x (from 0 to R), compute the maximum y such that the transformed condition holds. The number of valid y values for each x is determined using the quadratic solution derived from the condition. The solution involves calculating the integer square root to find the maximum y efficiently.

5. **Complexity**: The algorithm iterates over x from 0 to R, and for each x, computes y_max using mathematical operations. The overall complexity is O(R), which is feasible since R can be up to \(10^6\).

### Solution Code
```python
import math

def main():
    R = int(input().strip())
    total = 0
    for x in range(0, R + 1):
        C = 2 * R * R - 1 - 2 * x * (x + 1)
        if C < 0:
            break
        n = 1 + 2 * C
        try:
            root = math.isqrt(n)
        except AttributeError:
            low, high = 0, n
            while low <= high:
                mid = (low + high) // 2
                mid2 = mid * mid
                if mid2 <= n:
                    root = mid
                    low = mid + 1
                else:
                    high = mid - 1
        y_max = (root - 1) // 2
        if x == 0:
            total += 1 + 2 * y_max
        else:
            total += 2 + 4 * y_max
    print(total)

if __name__ == '__main__':
    main()
```

4. **Early Termination**: If C becomes negative, the loop breaks early since further x values will not yield valid y values.
5. **Integer Square Root Calculation**: The code computes the integer square root of `n = 1 + 2*C` to determine the maximum y value (y_max) that satisfies the condition. This is done using `math.isqrt` if available, or a binary search fallback otherwise.
