### Approach
1. **Problem Analysis**: The task is to find all substrings that include all five vowels ('a', 'e', 'i', 'o', 'u') at least once and exactly `k` consonants. The solution requires examining every possible substring of the input string and verifying these conditions.
2. **Intuition**: For each starting index in the string, we expand the substring to include subsequent characters, maintaining a set of encountered vowels and a count of consonants. If at any point the set contains all vowels and the consonant count matches `k`, the substring is counted. The loop breaks early if the consonant count exceeds `k` while all vowels are present, as further expansion cannot satisfy the condition.
3. **Algorithm Selection**: The algorithm uses a nested loop where the outer loop fixes the starting index, and the inner loop expands the substring by moving the ending index. A set tracks vowels, and a counter tracks consonants. The inner loop breaks early if the vowel set is complete and the consonant count exceeds `k`.
4. **Complexity Analysis**: The algorithm has a time complexity of O(n^2) where `n` is the length of the string. This is because for each starting index, we potentially examine up to `n` ending indices. Given the constraint `n ≤ 250`, this approach is efficient enough.

### Solution Code
```python
class Solution:
    def countOfSubstrings(self, word: str, k: int) -> int:
        n = len(word)
        vowels = "aeiou"
        count = 0
        for start in range(n):
            vowel_set = set()
            cons_count = 0
            for end in range(start, n):
                char = word[end]
                if char in vowels:
                    vowel_set.add(char)
                else:
                    cons_count += 1
                if len(vowel_set) == 5 and cons_count == k:
                    count += 1
                if len(vowel_set) == 5 and cons_count > k:
                    break
        return count
```

### Explanation
1. **Initialization**: The code initializes the length of the string, a set of vowels, and a counter for valid substrings.
2. **Outer Loop**: For each starting index in the string, it initializes an empty set to track vowels and a counter for consonants.
3. **Inner Loop**: For each ending index starting from the current starting index, it checks the character:
   - If the character is a vowel, it adds it to the set.
   - If the character is a consonant, it increments the consonant count.
4. **Condition Check**: After each addition, it checks if all vowels are present and the consonant count matches `k`. If so, it increments the valid substring count.
5. **Early Termination**: If all vowels are found and the consonant count exceeds `k`, the inner loop breaks early since further expansion cannot yield a valid substring.
6. **Result**: After processing all starting indices, the code returns the total count of valid substrings.

This approach efficiently checks all possible substrings while leveraging early termination to optimize performance, ensuring it handles the problem constraints effectively.
