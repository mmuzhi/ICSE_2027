To solve this problem, we need to determine whether all points on a circle can be colored black by two players, Alice and Bob, who take turns coloring points under specific symmetry conditions. The key observation is that the diameters (lines connecting the players to the circle's center) must be distinct for all points to be colored. If the diameters are the same, only certain cases allow complete coloring, primarily when the circle has exactly two points.

### Approach
1. **Problem Analysis**: The problem involves coloring points on a circle such that after each move, the coloring remains symmetric with respect to the current player's diameter. The players alternate turns, starting with Alice at point 0 and Bob at point K. The goal is to check if all points can be colored black under these conditions.
2. **Key Insight**: The diameters (lines through the center and players' positions) must be distinct for all points to be colored. If the diameters are the same (which happens when N is even and K is exactly N/2), then not all points can be colored, except in the special case where N is 2.
3. **Algorithm Selection**: For each test case, we check if N is even and K equals N/2. If so, the answer is "No" (except when N=2). Otherwise, the answer is "Yes". This simple check efficiently determines the solution without complex computations.

### Solution Code
```python
import sys

def main():
    data = sys.stdin.read().split()
    t = int(data[0])
    index = 1
    results = []
    for _ in range(t):
        N = int(data[index])
        K = int(data[index + 1])
        index += 2
        if N == 2:
            results.append("Yes")
        else:
            if N % 2 == 0 and K == N // 2:
                results.append("No")
            else:
                results.append("Yes")
    for res in results:
        print(res)

if __name__ == '__main__':
    main()
```

### Explanation
1. **Reading Input**: The input is read all at once for efficiency, especially given the constraints (T can be as large as 100,000).
2. **Processing Each Test Case**: For each test case, we check if N is 2. If so, the answer is "Yes" because both points can be colored regardless of K.
3. **Checking Diameter Condition**: For other values of N, if N is even and K is exactly half of N (N/2), the diameters are the same, and the answer is "No". Otherwise, the answer is "Yes".
4. **Output**: The results for all test cases are collected in a list and printed at the end for efficiency.

This approach efficiently checks the necessary conditions to determine if all points can be colored, leveraging simple arithmetic checks to avoid complex symmetry calculations.
