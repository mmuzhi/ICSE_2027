=== ANSWER ===

To solve this problem, we need to determine if a given string ends with the substring "san". The solution involves checking the last three characters of the string to see if they match "san". 

### Approach
1. **Problem Analysis**: The problem requires checking if the input string ends with the exact substring "san". The constraints specify that the string length is between 4 and 30 characters, ensuring that the string is long enough to have at least three characters, making the substring check feasible.
2. **Intuition**: By examining the last three characters of the string, we can directly verify if they form the substring "san". This approach is efficient and straightforward.
3. **Algorithm Selection**: The solution involves a simple string slicing operation to extract the last three characters and then comparing them with "san".
4. **Complexity Analysis**: The time complexity is O(1) since we only need to check a fixed number of characters (three) regardless of the string's length. The space complexity is also O(1) as we are not using any additional data structures.

### Solution Code
```python
def main():
    S = input().strip()
    if S.endswith('san'):
        print('Yes')
    else:
        print('No')

if __name__ == '__main__':
    main()
```

### Explanation
1. **Input Handling**: The input string `S` is read from standard input and stripped of any leading or trailing whitespace (though the problem guarantees lowercase letters, this step ensures robustness).
2. **Substring Check**: The `endswith` method is used to check if the string ends with "san". This method efficiently checks the last three characters.
3. **Output Decision**: Based on the result of the `endswith` check, the program prints "Yes" if the condition is met, otherwise it prints "No".

This approach is concise and leverages Python's built-in string methods for an efficient solution.
