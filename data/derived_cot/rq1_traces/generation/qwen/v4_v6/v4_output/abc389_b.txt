=== ANSWER ===

To solve this problem, we need to find the positive integer \( N \) such that \( N! \) (factorial of \( N \)) equals the given integer \( X \). The solution involves iteratively computing the factorial of increasing integers until we reach the value \( X \).

### Approach
1. **Initialization**: Start with \( N = 2 \) and compute \( 2! = 2 \).
2. **Iterative Multiplication**: For each subsequent integer \( N \), multiply the current factorial value by \( N \) to compute \( (N)! \).
3. **Comparison**: After each multiplication, check if the resulting factorial equals \( X \). If it does, \( N \) is the answer. If the factorial exceeds \( X \), the loop terminates, and \( N \) is the correct value since factorials grow strictly and the problem guarantees exactly one solution.

This approach efficiently narrows down the possible values of \( N \) by leveraging the rapid growth of factorials, ensuring minimal iterations even for the upper constraint \( X \leq 3 \times 10^{18} \).

### Solution Code
```python
X = int(input().strip())
n = 2
current = 2
while current < X:
    n += 1
    current *= n
print(n)
```

### Explanation
1. **Initialization**: The variable \( X \) is read from the input, representing the target factorial value. We initialize \( n \) (the candidate integer) to 2 and compute its factorial (\( 2! = 2 \)) into the variable `current`.
2. **Loop**: The loop continues as long as `current` is less than \( X \). In each iteration, \( n \) is incremented by 1, and `current` is updated to \( n! \) by multiplying it with \( n \).
3. **Termination**: The loop exits when `current` equals or exceeds \( X \). Since the problem guarantees exactly one solution, the last value of \( n \) for which `current` was less than \( X \) is the correct answer. This is printed immediately after the loop terminates.

This method efficiently finds \( N \) by leveraging the properties of factorials, ensuring correctness with minimal computational steps.
