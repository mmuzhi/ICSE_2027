Here is a more precise description.

- Let S_{l\ldots r} denote the substring of S from the l-th character through the r-th character.
- We define a substring S_{l\ldots r} of S to be a 1-block if it satisfies all of the following conditions:
- S_l = S_{l+1} = \cdots = S_r =  1
- l = 1 or S_{l-1} =  0
- r = N or S_{r+1} =  0

- T_i =  0 for r_{K-1} + (r_K - l_K) + 2 \leq i \leq r_K
- T_i = S_i for r_K + 1 \leq i \leq N

To solve this problem, we need to move the K-th 1-block in a binary string to immediately after the (K-1)-th 1-block. A 1-block is defined as a contiguous segment of '1's bounded by '0's or the edges of the string. 

### Approach
1. **Identify 1-Blocks**: Traverse the string to identify all contiguous segments of '1's that meet the 1-block conditions (bounded by '0's or string edges). This involves checking each segment of '1's to ensure it starts and ends with a '0' or the string boundaries.
2. **Locate Target Blocks**: Once all 1-blocks are identified, locate the (K-1)-th and K-th 1-blocks in the list of identified blocks.
3. **Construct Result String**: The resulting string is constructed by:
   - Keeping all characters up to the end of the (K-1)-th block.
   - Appending the K-th block.
   - Including the substring between the (K-1)-th block and the K-th block.
   - Appending the remaining part of the string after the K-th block.

### Solution Code
```python
import sys

def main():
    data = sys.stdin.read().splitlines()
    if not data:
        return
    
    parts = data[0].split()
    if not parts:
        return
        
    N = int(parts[0])
    K = int(parts[1])
    S = data[1].strip()
    
    i = 0
    blocks = []
    n = len(S)
    while i < n:
        if S[i] == '1':
            start = i
            while i < n and S[i] == '1':
                i += 1
            end = i - 1
            if (start == 0 or S[start - 1] == '0') and (end == n - 1 or S[end + 1] == '0'):
                blocks.append((start, end))
        else:
            i += 1
            
    if K - 1 >= len(blocks):
        print(S)
        return
        
    r_prev = blocks[K - 2][1]
    l_curr = blocks[K - 1][0]
    r_curr = blocks[K - 1][1]
    
    part1 = S[:r_prev + 1]
    part2 = S[l_curr:r_curr + 1]
    part3 = S[r_prev + 1:l_curr]
    part4 = S[r_curr + 1:]
    
    result = part1 + part2 + part3 + part4
    print(result)

if __name__ == '__main__':
    main()
```

1. **Reading Input**: The input is read from standard input, which includes the string length `N`, the integer `K`, and the binary string `S`.
2. **Identifying 1-Blocks**: The code scans the string to find all contiguous segments of '1's that are bounded by '0's or the string edges. Each valid segment (1-block) is recorded with its start and end indices.
3. **Handling Edge Cases**: If there are fewer than `K` 1-blocks (though the problem guarantees at least `K`), the original string is printed.
4. **Constructing Result**: The result string is built by:
   - **Part1**: Characters from the start of the string up to the end of the (K-1)-th block.
   - **Part2**: The K-th block itself.
   - **Part3**: The substring between the (K-1)-th and K-th blocks.
   - **Part4**: The remaining part of the string after the K-th block.
5. **Output**: The constructed result string is printed.

This approach efficiently processes the string in linear time, ensuring optimal performance even for large input sizes.
