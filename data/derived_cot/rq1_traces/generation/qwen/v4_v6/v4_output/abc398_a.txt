To solve this problem, we need to generate a unique palindrome string of length N that meets specific conditions regarding the characters '=' and '-'. The string must be a palindrome, contain exactly one or exactly two '=' characters, and if there are two '=' characters, they must be adjacent.

### Approach
1. **Problem Analysis**: The key observation is that the string must be a palindrome, meaning the characters must mirror around the center. For odd-length strings, the center character is a single '=', while for even-length strings, the two central characters must be '=' and adjacent. The rest of the characters are '-'.
2. **Intuition**: 
   - For odd N, the '=' is placed exactly at the center, with '-' characters on both sides.
   - For even N, the two central characters are '=', and the remaining characters are '-'.
3. **Algorithm Selection**: The solution involves constructing the string by placing the '=' characters in the appropriate positions based on whether N is odd or even. The rest of the positions are filled with '-' characters.
4. **Complexity Analysis**: The algorithm runs in O(N) time to construct the string, which is optimal given the constraints (N ≤ 100). The space complexity is O(N) to store the result.

### Solution Code
```python
def main():
    N = int(input().strip())
    if N % 2 == 1:
        s = '-' * (N // 2) + '=' + '-' * (N // 2)
    else:
        s = '-' * (N // 2 - 1) + '==' + '-' * (N // 2 - 1)
    print(s)

if __name__ == "__main__":
    main()
```

- **Even Length Handling**: If N is even, the string is constructed by placing two adjacent '=' characters in the center, with '-' characters on both sides. The number of '-' characters on each side is (N//2) - 1.
