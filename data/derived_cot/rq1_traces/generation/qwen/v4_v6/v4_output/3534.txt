      class Solution:
          def countPairs(self, nums: List[int]) -> int:
              from collections import Counter
              freq = Counter(nums)
              distinct_nums = sorted(freq.keys())
              T = {}
              for x in distinct_nums:
                  s = str(x)
                  n = len(s)
                  t_set = set()
                  t_set.add(x)
                  if n > 1:
                      # Generate all swaps
                      for i in range(n):
                          for j in range(i+1, n):
                              lst = list(s)
                              lst[i], lst[j] = lst[j], lst[i]
                              new_s = ''.join(lst)
                              # Convert to int to remove leading zeros
                              num = int(new_s)
                              t_set.add(num)
                  T[x] = t_set

              result = 0
              # Count distinct pairs (x, y) with x < y and (y in T[x] or x in T[y])
              n_distinct = len(distinct_nums)
              for i in range(n_distinct):
                  for j in range(i+1, n_distinct):
                      x = distinct_nums[i]
                      y = distinct_nums[j]
                      if y in T[x] or x in T[y]:
                          result += freq[x] * freq[y]

              # Count same number pairs
              for x in distinct_nums:
                  count = freq[x]
                  result += count*(count-1)//2

              return result

To solve this problem, we need to count the number of pairs of indices (i, j) in an array where i < j and the elements at these indices are almost equal. Two integers are considered almost equal if they can be made equal by performing at most one digit swap operation on one of the integers. 

### Approach
1. **Frequency Map**: First, we create a frequency map to count occurrences of each distinct number in the array.
2. **Generate Possible Values**: For each distinct number, we generate all possible numbers that can be formed by swapping any two digits (including the original number itself). This helps us determine all numbers that can be transformed into the current number with at most one swap.
3. **Count Valid Pairs**: We then iterate over all pairs of distinct numbers (x, y) where x < y. For each pair, we check if y can be formed from x or vice versa by checking if y is in the set of values generated from x or if x is in the set of values generated from y. If so, we add the product of their frequencies to the result.
4. **Count Same Number Pairs**: Additionally, we count all pairs of the same number by using the combination formula C(n, 2) for each number's frequency, where n is the count of that number.

### Solution Code
```python
from collections import Counter

class Solution:
    def countPairs(self, nums: List[int]) -> int:
        freq = Counter(nums)
        distinct_nums = sorted(freq.keys())
        T = {}
        for x in distinct_nums:
            s = str(x)
            n = len(s)
            t_set = set()
            t_set.add(x)
            if n > 1:
                for i in range(n):
                    for j in range(i+1, n):
                        lst = list(s)
                        lst[i], lst[j] = lst[j], lst[i]
                        new_s = ''.join(lst)
                        num = int(new_s)
                        t_set.add(num)
            T[x] = t_set
        
        result = 0
        n_distinct = len(distinct_nums)
        for i in range(n_distinct):
            for j in range(i+1, n_distinct):
                x = distinct_nums[i]
                y = distinct_nums[j]
                if y in T[x] or x in T[y]:
                    result += freq[x] * freq[y]
        
        for x in distinct_nums:
            count = freq[x]
            result += count * (count - 1) // 2
        
        return result
```

### Explanation
1. **Frequency Map**: The `Counter` from the `collections` module is used to count how many times each number appears in the input array.
2. **Generating Possible Values**: For each distinct number, we convert it to a string and generate all possible numbers by swapping any two digits. Each generated number is converted back to an integer and stored in a set to avoid duplicates.
3. **Counting Valid Pairs**: By iterating over all pairs of distinct numbers (x, y) where x < y, we check if y can be formed from x or vice versa using the precomputed sets. If they can, the product of their frequencies is added to the result.
4. **Same Number Pairs**: For each distinct number, the number of ways to choose two indices (i, j) where i < j and the elements are the same is calculated using the combination formula C(n, 2) = n*(n-1)/2, where n is the frequency of the number. This value is added to the result.

This approach efficiently counts all valid pairs by leveraging digit swapping and frequency analysis, ensuring optimal performance even for the upper constraint limits.
