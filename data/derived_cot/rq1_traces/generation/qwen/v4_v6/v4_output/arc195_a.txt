Subsequence
A subsequence of A is a sequence obtained by removing zero or more elements from A and leaving the remaining elements in their original order.

Input

The input is given from Standard Input in the following format:
N M
A_1 A_2 \ldots A_N
B_1 B_2 \ldots B_M

Output

If there are at least two subsequences of A that match B, print Yes. Otherwise, print No.

Constraints


- 1 \leq M \leq N \leq 2\times 10^5
- 1 \leq A_i \leq 10^9
- 1 \leq B_i \leq 10^9
- All input values are integers.

To solve this problem, we need to determine whether there are at least two distinct subsequences in a given sequence A that match another sequence B. The subsequences are considered distinct if they are formed from different positions in A, even if the sequences themselves are identical.

### Approach
1. **Check Subsequence Existence**: First, verify if B is a subsequence of A. This involves checking if all elements of B appear in A in the same order, though not necessarily consecutively.
2. **Minimal Matching**: Compute the leftmost (minimal) indices in A that form the subsequence B. This gives the earliest possible positions where B can be found in A.
3. **Maximal Matching**: Compute the rightmost (maximal) indices in A that form the subsequence B. This gives the latest possible positions where B can be found in A.
4. **Comparison**: If the minimal and maximal matchings are different, it means there are at least two distinct subsequences of A that match B. Otherwise, there is only one such subsequence.

import sys

    if not data:

        return

    B = list(map(int, data[2+n:2+n+m]))

        if A[j] == B[i]:

        return

    while i >= 0 and j >= 0:

2. **Subsequence Check**: The algorithm checks if B is a subsequence of A by traversing both sequences. If B is not found, it immediately outputs "No".
3. **Minimal Matching**: The leftmost indices of A that form B are recorded. This is done by traversing A from left to right and matching elements of B in order.
4. **Maximal Matching**: The rightmost indices of A that form B are recorded by traversing A from right to left and matching elements of B in reverse order.
5. **Comparison**: The minimal and maximal matchings are compared. If they differ, it means there are at least two distinct subsequences, so "Yes" is printed. Otherwise, "No" is printed, indicating only one subsequence exists.

This approach efficiently checks for the existence of multiple distinct subsequences by leveraging simple traversal and comparison, ensuring optimal performance even for large input sizes.
