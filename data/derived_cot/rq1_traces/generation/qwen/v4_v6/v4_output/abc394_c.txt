The challenge is to perform this operation efficiently, especially given the constraints where the string can be very long (up to 300,000 characters).

### Approach
1. **Problem Analysis**: The problem requires scanning the string from left to right to find the leftmost "WA" substring and replacing it with "AC". After each replacement, the modified string must be re-scanned from the beginning to check for any new "WA" substrings that might have been created by the replacement. The key observation here is that each replacement does not change the string length but can potentially create new "WA" substrings to the left of the replacement position if the character before the replacement is 'W'.

2. **Intuition**: By processing the string character by character and using a pointer to track the current position, we can efficiently handle replacements. After replacing a "WA" with "AC", we need to check the position immediately before the replacement (if it exists) to see if it forms a new "WA" with the newly placed 'A'. This ensures that all possible "WA" substrings are considered in each iteration.

3. **Algorithm Selection**: The algorithm involves traversing the string with a pointer. For each character, if it forms "WA" with the next character, replace it with "AC" and adjust the pointer to check the previous character. Otherwise, move the pointer forward. This approach ensures that all "WA" substrings are replaced in a left-to-right manner, and the string is processed in linear time relative to its length.

4. **Complexity Analysis**: The algorithm processes each character in the string at most once, except for the characters involved in replacements. Each replacement operation is O(1), and the total number of replacements is bounded by the number of 'W's followed by 'A's in the original string. Thus, the overall time complexity is O(n), where n is the length of the string, making it efficient for large inputs.

### Solution Code
```python
def main():
    S = input().strip()
    s = list(S)
    i = 0
    n = len(s)
    while i < n:
        if i + 1 < n and s[i] == 'W' and s[i+1] == 'A':
            s[i] = 'A'
            s[i+1] = 'C'
            i -= 1
        else:
            i += 1
    print(''.join(s))

if __name__ == '__main__':
    main()
```

1. **Initialization**: The input string is converted into a list of characters for easier manipulation. A pointer `i` is initialized to 0 to start traversing the string from the beginning.

3. **Replacement Handling**: If "WA" is found, it is replaced with "AC". The pointer `i` is decremented to check if the replacement creates a new "WA" substring starting from the previous character.
4. **Pointer Adjustment**: If no "WA" is found at the current position, the pointer `i` is incremented to move forward in the string.
5. **Result Construction**: After processing the entire string, the modified list of characters is joined back into a string and printed.
