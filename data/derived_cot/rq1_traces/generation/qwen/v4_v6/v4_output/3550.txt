To solve this problem, we need to place three rooks on an m x n chessboard such that they do not attack each other (i.e., no two rooks share the same row or column) and the sum of the values of the cells they occupy is maximized. 

### Approach
1. **Problem Analysis**: The problem requires selecting three distinct cells, each in different rows and columns, such that their values sum to the maximum possible. The constraints ensure the board is at least 3x3, so we can always find such cells.
2. **Intuition**: For each row, the top three highest-value cells are considered because the optimal solution likely involves these cells. By focusing on these top cells, we reduce the search space while still covering potential optimal solutions.
3. **Algorithm Selection**: 
   - **Preprocessing**: For each row, compute the top three cells (by value) along with their column indices.
   - **Combination Check**: For every combination of three distinct rows, check all possible assignments of their top three columns, ensuring columns are distinct. The maximum sum from these valid assignments is recorded.
4. **Complexity Analysis**: The algorithm processes each row to find top three cells in O(m*n log n) time due to sorting. Then, it checks all combinations of three rows (C(m,3) combinations) and for each, up to 3x3x3 = 27 column assignments. Given m and n are up to 100, the total operations are manageable (around 3 million), making the approach efficient.

### Solution Code
```python
import itertools
from typing import List

class Solution:
    def maximumValueSum(self, board: List[List[int]]) -> int:
        m = len(board)
        n = len(board[0])
        top3s = []
        for i in range(m):
            arr = [(board[i][j], j) for j in range(n)]
            arr.sort(key=lambda x: x[0], reverse=True)
            top3s.append(arr[:3])
        
        ans = float('-inf')
        for i, j, k in itertools.combinations(range(m), 3):
            list0 = top3s[i]
            list1 = top3s[j]
            list2 = top3s[k]
            for a_val, a_col in list0:
                for b_val, b_col in list1:
                    if b_col == a_col:
                        continue
                    for c_val, c_col in list2:
                        if c_col == a_col or c_col == b_col:
                            continue
                        total = a_val + b_val + c_val
                        if total > ans:
                            ans = total
        return ans
```

### Explanation
1. **Preprocessing**: For each row in the board, we sort the cell values in descending order and keep the top three cells (each cell's value and column index).
2. **Combination Check**: Using combinations of three distinct rows, we check all possible assignments of their top three columns. For each assignment, we ensure the columns are distinct (to prevent rook attacks) and calculate the sum of the cell values.
3. **Result Calculation**: The maximum sum encountered during these checks is returned as the solution. The use of combinations ensures all possible valid placements are considered efficiently, leveraging the preprocessing step to limit the search space. This approach efficiently narrows down the potential candidates for optimal solutions without exhaustive checks.
