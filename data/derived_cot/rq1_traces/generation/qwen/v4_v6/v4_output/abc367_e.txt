=== ANSWER ===

To solve this problem, we need to perform a series of operations on an array K times, where each operation replaces the array with another array formed by taking elements from the current array based on fixed indices provided in another array. Given the constraints where K can be as large as 10^18, a brute-force approach is infeasible. Instead, we use a functional graph approach combined with binary lifting to efficiently compute the result after K operations.

### Approach
1. **Problem Analysis**: The operation involves transforming the array by replacing each element with the element at a specific index (given by another array) from the current array. This operation is applied K times. The challenge is to compute the result efficiently for very large K.

2. **Key Insight**: Each operation can be viewed as applying a function f, where f(i) is the index from which the new value is taken. After K operations, the value at position i in the resulting array is the initial value at position f^K(i), where f^K denotes the function f applied K times.

3. **Algorithm Selection**: 
   - **Functional Graph**: The function f defines a directed graph where each node points to one other node. This graph consists of cycles and trees leading into these cycles.
   - **Binary Lifting**: To handle large K, we preprocess the graph using binary lifting. This allows us to jump K steps in logarithmic time by decomposing K into its binary representation and using precomputed steps (powers of two).

4. **Complexity Analysis**: 
   - **Preprocessing**: Building the binary lifting table takes O(N log K) time and space, where N is the array length and K is the number of operations.
   - **Query Processing**: For each element in the array, computing the result takes O(log K) time. Thus, the total time is O(N log K), which is feasible given the constraints.

### Solution Code
```python
import sys

def main():
    data = sys.stdin.read().strip().split()
    if not data:
        return
    n = int(data[0])
    k = int(data[1])
    X = list(map(int, data[2:2+n]))
    A = list(map(int, data[2+n:2+2*n]))
    
    if k == 0:
        print(" ".join(map(str, A)))
        return
    
    f = [x-1 for x in X]
    N = n
    LOG = 60
    
    next_table = [[0] * LOG for _ in range(N)]
    
    for i in range(N):
        next_table[i][0] = f[i]
    
    for j in range(1, LOG):
        for i in range(N):
            prev_node = next_table[i][j-1]
            next_table[i][j] = next_table[prev_node][j-1]
    
    result = []
    for i in range(N):
        node = i
        for j in range(LOG):
            if (k >> j) & 1:
                node = next_table[node][j]
        result.append(str(A[node]))
    
    print(" ".join(result))

if __name__ == "__main__":
    main()
````
