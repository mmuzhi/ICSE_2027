    def countSubstrings(self, s: str) -> int:

To solve this problem, we need to count the number of substrings in a given string of digits that are divisible by their non-zero last digit. The key insight is that for each substring ending with a non-zero digit, the divisibility condition can be transformed into a problem of checking divisibility by a derived value based on the last digit. 

### Approach
1. **Problem Analysis**: The problem requires checking all possible substrings of the given string to see if they are divisible by their last non-zero digit. A brute-force approach would be too slow for large strings (up to 10^5 characters), so we need an optimized solution.
2. **Key Insight**: For a substring ending with digit `d` (non-zero), the entire substring must be divisible by `d`. This can be rephrased as the substring formed by excluding the last digit must be divisible by `d_prime = d // gcd(d, 10)`. This transformation leverages mathematical properties of divisibility.
3. **Dynamic Programming**: For each possible divisor `d_prime` (values 1, 2, 3, 4, 5, 6, 7, 9), we precompute the number of valid substrings ending at each position. This involves maintaining a dictionary of remainders and their counts as we process each digit in the string.
4. **Efficiency**: By precomputing remainders for each divisor, we reduce the problem to checking each digit's contribution in constant time, leading to an overall time complexity of O(n), where n is the length of the string.

### Solution Code
```python
import math

class Solution:
    def countSubstrings(self, s: str) -> int:
        n = len(s)
        divisors = [1, 2, 3, 4, 5, 6, 7, 9]
        dp_count = {d: [0] * n for d in divisors}
        
        for d_prime in divisors:
            dp_prev = {}
            for i in range(n):
                digit_val = int(s[i])
                new_r = digit_val % d_prime
                count_new = 1
                
                new_dp = {}
                for r, cnt in dp_prev.items():
                    new_r_ext = (r * 10 + digit_val) % d_prime
                    new_dp[new_r_ext] = new_dp.get(new_r_ext, 0) + cnt
                
                new_dp[new_r] = new_dp.get(new_r, 0) + count_new
                dp_count[d_prime][i] = new_dp.get(0, 0)
                dp_prev = new_dp
        
        res = 0
        for i in range(n):
            if s[i] == '0':
                continue
            d_val = int(s[i])
            g = math.gcd(d_val, 10)
            d_prime_val = d_val // g
            if i == 0:
                res += 1
            else:
                res += dp_count[d_prime_val][i-1] + 1
        
        return res
```

### Explanation
1. **Initialization**: We start by defining the divisors (1 through 9) and initializing a dictionary `dp_count` to store the number of valid substrings for each divisor at each position in the string.
2. **Dynamic Programming Pass**: For each divisor, we iterate through the string. For each digit, we compute new remainders by extending previous substrings and starting new substrings. The count of valid substrings (those divisible by the divisor) is updated for each position.
3. **Result Calculation**: For each digit in the string (skipping zeros), we determine the appropriate divisor `d_prime` using the digit's greatest common divisor with 10. The result is incremented by the count of valid substrings ending at the previous position plus one (accounting for the empty substring case).
4. **Efficiency**: The algorithm efficiently processes each digit in constant time per divisor, leveraging dynamic programming to avoid the O(n^2) complexity of a brute-force solution. The final result is the sum of valid substrings for all non-zero last digits in the string.
