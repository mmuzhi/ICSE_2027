=== ANSWER ===

To solve this problem, we need to determine the minimum number of operations required to transform a given permutation into the identity permutation where each element \( P_i = i \). The allowed operations involve selecting an integer \( k \) and sorting specific segments of the permutation around the \( k \)-th element.

### Approach
1. **Check for Identity Permutation**: First, we check if the permutation is already the identity permutation. If it is, the answer is 0.
2. **Prefix and Suffix Arrays**: For each test case, we precompute prefix arrays (`min_prefix` and `max_prefix`) to store the minimum and maximum values of the permutation up to each index. Similarly, we compute suffix arrays (`suffix_min` and `suffix_max`) for the minimum and maximum values from each index to the end.
3. **Validate One-Move Operations**: We then check if there exists any \( k \) (from 2 to \( N-1 \)) such that:
   - The \( k \)-th element is \( k \).
   - The left segment (elements before \( k \)) contains all integers from 1 to \( k-1 \).
   - The right segment (elements after \( k \)) contains all integers from \( k+1 \) to \( N \).
   If such a \( k \) is found, the answer is 1.
4. **Default to Two Operations**: If no such \( k \) is found, the answer is 2, as it can always be achieved in two operations for any permutation under the given constraints.

### Solution Code
```python
import sys

def main():
    data = sys.stdin.read().split()
    t = int(data[0])
    index = 1
    results = []
    
    for _ in range(t):
        N = int(data[index])
        index += 1
        P = list(map(int, data[index:index+N]))
        index += N
        
        is_identity = True
        for i in range(N):
            if P[i] != i+1:
                is_identity = False
                break
        if is_identity:
            results.append("0")
            continue
            
        min_prefix = [0] * N
        max_prefix = [0] * N
        min_prefix[0] = P[0]
        max_prefix[0] = P[0]
        for i in range(1, N):
            min_prefix[i] = min(min_prefix[i-1], P[i])
            max_prefix[i] = max(max_prefix[i-1], P[i])
            
        suffix_min = [0] * N
        suffix_max = [0] * N
        suffix_min[N-1] = P[N-1]
        suffix_max[N-1] = P[N-1]
        for i in range(N-2, -1, -1):
            suffix_min[i] = min(suffix_min[i+1], P[i])
            suffix_max[i] = max(suffix_max[i+1], P[i])
            
        found = False
        for k in range(2, N):
            if P[k-1] != k:
                continue
            L = k - 1
            if L > 0:
                left_min = min_prefix[L-1]
                left_max = max_prefix[L-1]
            else:
                left_min = float('inf')
                left_max = float('-inf')
                
            right_min = suffix_min[k]
            right_max = suffix_max[k]
            
            if left_min == 1 and left_max == L and right_min == k+1 and right_max == N:
                found = True
                break
                
        if found:
            results.append("1")
        else:
            results.append("2")
            
    for res in results:
        print(res)

if __name__ == '__main__':
    main()
```
