To solve this problem, we need to count the number of pairs of positive integers (x, y) that satisfy the condition \(A_i \times x + B_i \times y < C_i\) for all given constraints \(1 \leq i \leq N\). The solution involves iterating over possible values of x and, for each x, determining the maximum valid y that satisfies all constraints.

### Approach
1. **Initial Checks**: For each test case, first check if any constraint \(C_i \leq B_i\). If so, no valid pairs exist, and the answer is 0.
2. **Determine Maximum x (X_max)**: Compute the maximum possible x value (X_max) such that for all constraints, \(A_i \times x < C_i - B_i\). This is derived by taking the minimum of \((C_i - B_i - 1) // A_i\) across all constraints.
3. **Iterate Over x Values**: For each x from 1 to X_max (or a reasonable limit if X_max is very large), compute the minimum value of \(\lfloor (C_i - A_i \times x - 1) / B_i \rfloor\) across all constraints. This minimum value gives the maximum y for the current x.
4. **Count Valid Pairs**: For each x, if the computed minimum y value is positive, add this value to the total count of valid pairs.
