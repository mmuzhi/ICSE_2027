=== ANSWER ===

To solve this problem, we need to remove all occurrences of the character '.' from a given string. The solution involves reading the input string, processing it to eliminate all '.' characters, and then outputting the resulting string. 

### Approach
The approach is straightforward:
1. **Read the Input String**: The input is read from standard input.
2. **Remove All '.' Characters**: Using the string replacement method, all instances of '.' are replaced with an empty string, effectively removing them from the original string.
3. **Output the Result**: The modified string, which now contains no '.' characters, is printed.

This approach leverages Python's built-in string method `replace()`, which efficiently handles the removal of all specified characters in linear time relative to the length of the string.

### Solution Code
```python
s = input().strip()
result = s.replace('.', '')
print(result)
```

### Explanation
1. **Reading Input**: The `input().strip()` function reads the input string and removes any leading or trailing whitespace, though the problem constraints specify that the input consists of lowercase letters and '.', so whitespace handling is not strictly necessary but included for robustness.
2. **Replacing '.'**: The `replace('.', '')` method is called on the input string. This method scans the string and replaces every occurrence of '.' with an empty string, thus removing all '.' characters.
3. **Output**: The resulting string, now devoid of any '.' characters, is printed. 

This solution efficiently addresses the problem requirements with minimal computational overhead, ensuring clarity and correctness.
