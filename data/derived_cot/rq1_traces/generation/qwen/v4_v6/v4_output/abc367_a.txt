=== ANSWER ===

To solve this problem, we need to determine if Takahashi can shout his love for takoyaki at A o'clock every day, given that he goes to bed at B o'clock and wakes up at C o'clock. The key insight is to check whether the time A falls within Takahashi's awake periods during the day.

### Approach
1. **Problem Analysis**: 
   - Takahashi's awake periods depend on his sleep schedule. If he goes to bed at B and wakes up at C, his sleep period is from B to C. However, if B is greater than C, his sleep period wraps around midnight, meaning he is asleep from B to 24 and from 0 to C the next day.
   - The solution involves checking if the time A is outside his sleep period. For cases where B ≤ C, the sleep period is [B, C), so A must be either less than B or greater than or equal to C to be within his awake periods. For cases where B > C, the sleep period is [0, C) and [B, 24), so A must be within [C, B) to be awake.

2. **Algorithm Selection**:
   - The solution involves a simple conditional check based on the relationship between B and C.
   - If B ≤ C, check if A is not in [B, C). If A is outside this interval, print "Yes"; otherwise, print "No".
   - If B > C, check if A is within [C, B). If it is, print "Yes"; otherwise, print "No".

3. **Complexity Analysis**:
   - The solution involves a constant number of comparisons, making it O(1) in time and space complexity. This is efficient for the given constraints.

### Solution Code
```python
def main():
    A, B, C = map(int, input().split())
    if B <= C:
        if A < B or A >= C:
            print("Yes")
        else:
            print("No")
    else:
        if C <= A < B:
            print("Yes")
        else:
            print("No")

if __name__ == "__main__":
    main()
```

### Explanation
- **Input Handling**: The code reads three integers A, B, and C from the input, representing the time Takahashi wants to shout, his bedtime, and his wake-up time, respectively.
- **Condition Checks**:
  - **Case 1 (B ≤ C)**: If Takahashi's bedtime is earlier than or equal to his wake-up time, he is awake from 0 to B and from C to 23. The code checks if A falls outside the sleep period [B, C). If it does, he can shout ("Yes"); otherwise, he cannot ("No").
  - **Case 2 (B > C)**: Here, Takahashi's sleep period wraps around midnight. He is awake from C to B (the middle segment of the day). The code checks if A lies within this interval [C, B). If it does, he can shout ("Yes"); otherwise, he cannot ("No").
- **Output**: Based on the conditions, the code prints "Yes" or "No" indicating whether Takahashi can shout at the specified time every day. 

This approach efficiently checks the required conditions to determine Takahashi's ability to shout at the given time, considering his sleep schedule.
