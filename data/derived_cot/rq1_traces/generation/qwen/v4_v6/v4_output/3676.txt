To solve this problem, we need to find the smallest number \( x \) that is greater than or equal to a given positive number \( n \), such that the binary representation of \( x \) consists only of set bits (i.e., all bits are 1). 

### Approach
1. **Problem Analysis**: The key observation here is that numbers with only set bits in their binary representation are of the form \( 2^k - 1 \). For example, 1 (binary "1"), 3 (binary "11"), 7 (binary "111"), 15 (binary "1111"), etc. These numbers are known as Mersenne numbers. 
2. **Insight**: The smallest number \( x \) greater than or equal to \( n \) with only set bits can be found by determining the smallest \( k \) such that \( 2^k - 1 \geq n \). The value of \( k \) is essentially the number of bits required to represent \( n \) in binary. This is derived from the fact that the largest number with \( k \) bits is \( 2^k - 1 \).
3. **Algorithm Selection**: The algorithm involves calculating the bit length of \( n \), which gives the number of bits required. The solution is then \( 2^k - 1 \), where \( k \) is the bit length of \( n \). This works because \( 2^k - 1 \) is the smallest number with \( k \) bits that consists entirely of set bits and is greater than or equal to \( n \).

### Solution Code
```python
class Solution:
    def smallestNumber(self, n: int) -> int:
        k = n.bit_length()
        return (1 << k) - 1
```

### Explanation
1. **Bit Length Calculation**: The method `n.bit_length()` returns the number of bits required to represent \( n \) in binary. For example, if \( n = 5 \) (binary "101"), the bit length is 3.
2. **Power of Two Calculation**: Once the bit length \( k \) is known, the smallest number with \( k \) bits that consists entirely of set bits is \( 2^k - 1 \). This is because \( 2^k \) is the smallest number that requires \( k+1 \) bits, so \( 2^k - 1 \) is the largest number that can be represented with \( k \) bits, and it has all bits set.
3. **Efficiency**: The solution efficiently computes the result in constant time \( O(1) \) using bit manipulation, leveraging the `bit_length()` method and bit shifting to find the required number. This approach is optimal given the constraints and ensures the solution is both correct and efficient.
