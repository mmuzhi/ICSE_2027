To solve this problem, we need to find the maximum possible number that can be formed by concatenating the binary representations of three given integers in some order. The key insight here is that the order of concatenation significantly affects the resulting number, so we need to explore all possible permutations of the given numbers to determine which combination yields the highest value.

### Approach
1. **Problem Analysis**: The problem requires us to concatenate the binary representations of three integers in different orders and then convert the resulting binary string back to a decimal number. The challenge is to find the permutation of the three numbers that produces the highest decimal value when their binary strings are concatenated.
2. **Intuition**: Since there are only three numbers, we can generate all possible permutations (6 in total) and evaluate each one. For each permutation, we convert each number to its binary string (without leading zeros), concatenate these strings, and then convert the concatenated string to a decimal number. The maximum value among all these conversions is our answer.
3. **Algorithm Selection**: We use itertools.permutations to generate all possible orderings of the three numbers. For each permutation, we process the numbers as described above. This approach efficiently checks all possible combinations without redundancy.
4. **Complexity Analysis**: With three numbers, there are 3! = 6 permutations. Each permutation involves converting three numbers to binary and then combining them, which is a constant-time operation. Thus, the overall complexity is O(1) since the input size is fixed.

### Solution Code
```python
import itertools

class Solution:
    def maxGoodNumber(self, nums: List[int]) -> int:
        max_val = 0
        for perm in itertools.permutations(nums):
            binary_str = ''.join(bin(num)[2:] for num in perm)
            current_val = int(binary_str, 2)
            if current_val > max_val:
                max_val = current_val
        return max_val
```

### Explanation
1. **Generating Permutations**: The code uses `itertools.permutations` to generate all possible orderings of the three input numbers. This ensures that every possible concatenation order is considered.
2. **Binary Conversion and Concatenation**: For each permutation, each number is converted to its binary string representation (stripping the '0b' prefix). These binary strings are then concatenated into a single string.
3. **Decimal Conversion**: The concatenated binary string is converted back to a decimal number using Python's built-in `int` function with base 2.
4. **Tracking Maximum Value**: The maximum value encountered during the iteration over all permutations is stored and returned as the result.

This approach efficiently explores all possible concatenation orders and ensures the highest possible number is found by leveraging Python's standard libraries for permutations and binary conversions.
