To solve this problem, we need to determine whether it's possible to redistribute all the balls from N boxes such that only the X-th box contains balls, and if possible, find the minimum number of operations required. Each operation involves selecting a box, moving all its balls to two different boxes based on given permutations, and emptying the selected box.

### Approach
1. **Problem Analysis**: The problem involves redistributing balls in boxes using operations defined by permutations. The goal is to ensure all balls end up in the X-th box. The key insight is that for any box containing initial balls (either red or blue), the permutation paths must allow routing these balls to the X-th box. If any box with initial balls cannot reach the X-th box through its permutation paths, the task is impossible.

2. **Intuition**: The solution hinges on checking if all initial balls can be routed to the X-th box via the given permutations. If routing is possible, the minimal number of operations is the count of boxes with initial balls, as each such box must be operated on at least once to move its balls.

3. **Algorithm Selection**: 
   - **Cycle Detection**: For each permutation (P for red balls and Q for blue balls), decompose into cycles. This helps determine if a box's cycle includes the X-th box.
   - **Feasibility Check**: For each box with initial balls, verify that its cycle in both permutations includes the X-th box. If any box fails this check, the task is impossible.
   - **Operation Count**: If all checks pass, the minimal operations required is the number of boxes with initial balls, as each such box must be operated on at least once.

4. **Complexity Analysis**: The algorithm involves cycle detection in permutations, which is O(N) for each permutation. The overall complexity is O(N) due to the linear passes through the permutations and initial ball counts.

### Solution Code
```python
def main():
    import sys
    data = sys.stdin.read().split()
    if not data:
        return
    
    n = int(data[0])
    x = int(data[1]) - 1
    
    A = list(map(int, data[2:2+n]))
    B = list(map(int, data[2+n:4+n]))
    
    P = list(map(int, data[4+n:4+n+n]))
    Q = list(map(int, data[4+n+n:4+n+n+n]))
    
    for i in range(n):
        P[i] -= 1
        Q[i] -= 1
        
    visited_red = [False] * n
    cycles_red = []
    for i in range(n):
        if not visited_red[i]:
            cycle = []
            cur = i
            while not visited_red[cur]:
                visited_red[cur] = True
                cycle.append(cur)
                cur = P[cur]
            cycles_red.append(cycle)
            
    visited_blue = [False] * n
    cycles_blue = []
    for i in range(n):
        if not visited_blue[i]:
            cycle = []
            cur = i
            while not visited_blue[cur]:
                visited_blue[cur] = True
                cycle.append(cur)
                cur = Q[cur]
            cycles_blue.append(cycle)
            
    def in_cycle(cycle, target):
        return target in cycle
        
    for i in range(n):
        if A[i] > 0:
            found = False
            for cycle in cycles_red:
                if in_cycle(cycle, x):
                    found = True
                    break
            if not found:
                print(-1)
                return
                
    for i in range(n):
        if B[i] > 0:
            found = False
            for cycle in cycles_blue:
                if in_cycle(cycle, x):
                    found = True
                    break
            if not found:
                print(-1)
                return
                
    count = 0
    for i in range(n):
        if A[i] > 0 or B[i] > 0:
            count += 1
            
    print(count)

if __name__ == '__main__':
    main()
```
