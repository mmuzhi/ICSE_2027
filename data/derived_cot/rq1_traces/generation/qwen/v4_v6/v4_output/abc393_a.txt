To solve this problem, we need to determine which type of oyster causes stomach trouble based on the given information about whether Takahashi and Aoki got sick after eating specific oysters. The solution involves analyzing the input strings to deduce the culprit oyster type.

### Approach
1. **Problem Analysis**: The problem involves four types of oysters (1, 2, 3, 4), with exactly one causing stomach trouble. Takahashi ate oysters 1 and 2, and Aoki ate oysters 1 and 3. The input consists of two strings, S_1 and S_2, indicating whether each person got sick ("sick") or not ("fine").
2. **Key Insight**: By examining the combination of S_1 and S_2, we can map each scenario to a specific oyster type:
   - If both are "sick", oyster 1 is the culprit (since both ate it).
   - If S_1 is "sick" and S_2 is "fine", oyster 2 is the culprit (Takahashi ate it but Aoki didn't).
   - If S_1 is "fine" and S_2 is "sick", oyster 3 is the culprit (Aoki ate it but Takahashi didn't).
   - If both are "fine", oyster 4 is the culprit (neither ate it).
3. **Algorithm Selection**: The solution uses conditional checks to directly map the input strings to the corresponding oyster type based on the above insights.

```python
    S2 = data[1]
    
    if S1 == "sick" and S2 == "sick":
        print(1)
    elif S1 == "sick" and S2 == "fine":
        print(2)
    elif S1 == "fine" and S2 == "sick":
        print(3)
    elif S1 == "fine" and S2 == "fine":
        print(4)

if __name__ == '__main__':
    main()
```
