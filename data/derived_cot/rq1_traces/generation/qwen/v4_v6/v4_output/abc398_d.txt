To solve this problem, we need to determine if smoke exists at a specific cell (R, C) at time t+0.5 for each integer t from 1 to N in a grid where smoke moves according to wind directions given by a string S. The solution involves tracking the movement of smoke and checking for its presence at the specified cell at each time step.

### Approach
1. **Problem Analysis**: The problem involves simulating the movement of smoke on a grid based on wind directions. The smoke starts at (0, 0) at time 0. At each time step t, the wind blows, moving all existing smoke according to the t-th character of S. If after the wind blows, (0, 0) has no smoke, new smoke is generated there. The task is to check if the cell (R, C) has smoke at each time t+0.5.

2. **Key Insight**: The smoke at any time t+0.5 can either be:
   - The original smoke (generated at time 0) moved by the first t winds.
   - Smoke generated at some earlier time i (0 ≤ i < t) and moved by the subsequent t - i - 1 winds.

3. **Algorithm Selection**:
   - **Precompute Displacements**: Calculate the net displacement (X, Y) after each prefix of the wind string S. This helps in determining the position of the original smoke and any generated smoke at any time t.
   - **Track Generations**: For each time step i, determine if a generation occurred (i.e., (0, 0) was empty after the wind blew). This is done by checking if the current displacement matches any previous displacement or is (0, 0).
   - **Check Smoke Presence**: For each time t, check if the original smoke or any generated smoke (from earlier times) ends up at (R, C) after t winds.

4. **Complexity Analysis**: The algorithm processes each character of S once to compute displacements and checks each time step once, resulting in an O(N) time complexity, which is efficient given the constraints (N ≤ 200,000).

### Solution Code
```python
import sys

def main():
    data = sys.stdin.read().splitlines()
    if not data: 
        return
    parts = data[0].split()
    N = int(parts[0])
    R = int(parts[1])
    C = int(parts[2])
    S = data[1].strip()
    
    dx_map = {'N': -1, 'S': 1, 'E': 0, 'W': 0}
    dy_map = {'N': 0, 'S': 0, 'E': 1, 'W': -1}
    
    X = [0] * (N + 1)
    Y = [0] * (N + 1)
    
    for i in range(N):
        d = S[i]
        X[i + 1] = X[i] + dx_map[d]
        Y[i + 1] = Y[i] + dy_map[d]
    
    G = [0] * N
    seen = set()
    for i in range(N):
        pos = (X[i + 1], Y[i + 1])
        if pos == (0, 0) or pos in seen:
            G[i] = 0
        else:
            G[i] = 1
            seen.add(pos)
            
    first_occurrence = {}
    for i in range(N):
        if G[i] == 1:
            pos = (X[i + 1], Y[i + 1])
            if pos not in first_occurrence:
                first_occurrence[pos] = i
                
    res = []
    for t in range(1, N + 1):
        if (X[t], Y[t]) == (R, C):
            res.append('1')
        else:
            target = (X[t] - R, Y[t] - C)
            if target in first_occurrence and first_occurrence[target] < t:
                res.append('1')
            else:
                res.append('0')
                
    print(''.join(res))

if __name__ == "__main__":
    main()
```

### Explanation
1. **Initialization**: The input is read and parsed into N (length of string S), R and C (target cell coordinates), and S (wind directions).
2. **Displacement Calculation**: Arrays X and Y are computed to store the net displacement after each prefix of S. Each character in S modifies the current displacement based on predefined dx and dy maps.
3. **Generation Tracking**: For each time step i, it checks if (0, 0) is empty after the wind. If not, a generation occurs, and the displacement is recorded.
4. **Smoke Presence Check**: For each time t, it checks if the original smoke or any generated smoke (from earlier times) is at (R, C) after t winds. The result is built as a string of '0's and '1's indicating smoke presence at each time step.
5. **Output**: The result string is printed, showing smoke presence at (R, C) for each time t+0.5 from 1 to N. 

This approach efficiently tracks smoke movement and generation, leveraging prefix sums and set operations to determine smoke presence at the target cell for each time step.
