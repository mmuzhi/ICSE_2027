=== ANSWER ===

To solve this problem, we need to determine if there exists any sequence of three or more consecutive identical elements in a given integer sequence. The solution involves checking each possible triplet of consecutive elements in the sequence to see if all three are the same. If such a triplet is found, we immediately return "Yes"; otherwise, after checking all possible triplets, we return "No".

### Approach
1. **Input Reading**: Read the integer sequence length `N` and the sequence itself from standard input.
2. **Triplet Check**: Iterate through the sequence from the first element to the third last element. For each element at position `i`, check if it is equal to the elements at positions `i+1` and `i+2`.
3. **Early Termination**: If any such triplet is found, print "Yes" and terminate the program immediately. If no such triplet is found after checking all possible positions, print "No".

This approach efficiently checks for the presence of three consecutive identical elements by leveraging a simple linear scan through the sequence, ensuring optimal performance with a time complexity of O(N), where N is the length of the sequence.

### Solution Code
```python
def main():
    n = int(input().strip())
    arr = list(map(int, input().split()))
    
    for i in range(n - 2):
        if arr[i] == arr[i+1] == arr[i+2]:
            print("Yes")
            return
            
    print("No")

if __name__ == "__main__":
    main()
```

### Explanation
1. **Input Handling**: The code reads the sequence length `N` and the sequence of integers from standard input.
2. **Triplet Check Loop**: The loop runs from the first element to the third last element (i.e., `i` ranges from `0` to `N-3`). For each element at position `i`, it checks if the elements at positions `i`, `i+1`, and `i+2` are all equal.
3. **Result Determination**: If any triplet of equal elements is found, the program prints "Yes" and exits. If the loop completes without finding any such triplet, it prints "No".

This approach ensures that the solution is both efficient and straightforward, leveraging a simple loop and conditional check to solve the problem within linear time complexity.
