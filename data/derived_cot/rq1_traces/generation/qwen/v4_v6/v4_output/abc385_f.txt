To solve this problem, we need to determine the maximum height at coordinate 0 from which it is not possible to see all buildings. If it is possible to see all buildings from height 0, we should return -1. The solution involves checking for each building the minimal height required to see it and then finding the maximum of these minimal heights. 

### Approach
1. **Problem Analysis**: The problem requires checking visibility of each building from a point (0, h) where h is a non-negative height. A building is visible if the line from (0, h) to its top does not intersect any other building. The key insight is that for each building, the minimal height h must be such that the line from (0, h) to the building's top clears all buildings to its left.
2. **Intuition**: For each building i, the minimal height h_i is determined by the most constraining building j to its left. The constraint is derived from ensuring the line from (0, h) to (X_i, H_i) passes above (X_j, H_j). This translates to solving for h in the inequality derived from the line's equation.
3. **Algorithm Selection**: For each building i, compute the minimal height h_i by considering all buildings j to its left. The minimal h_i is the maximum of the values (H_j * X_i - H_i * X_j) / (X_i - X_j) for all j < i, ensuring the line condition is met. The overall solution is the maximum of all h_i values. If this maximum is 0 and all buildings are visible at height 0, return -1.
4. **Complexity Analysis**: The algorithm involves nested loops: for each building, it checks all previous buildings. With N buildings, this results in O(N^2) time complexity. Given the constraints (N ≤ 2 × 10^5), this approach may not be efficient for large inputs but is implemented for clarity and correctness.

### Solution Code
```python
import sys

def main():
    data = sys.stdin.read().strip().split()
    if not data:
        return
    n = int(data[0])
    buildings = []
    index = 1
    for i in range(n):
        x = int(data[index])
        h = int(data[index+1])
        index += 2
        buildings.append((x, h))
    
    candidates = [0] * n
    for i in range(n):
        x_i, h_i = buildings[i]
        candidate_val = -10**18
        for j in range(i):
            x_j, h_j = buildings[j]
            numerator = h_j * x_i - h_i * x_j
            denominator = x_i - x_j
            value = numerator / denominator
            if value > candidate_val:
                candidate_val = value
        candidates[i] = candidate_val

    min_h_i = [candidates[i] if candidates[i] > 0 else 0 for i in range(n)]
    h_min = max(min_h_i) if min_h_i else 0

    if h_min == 0:
        all_negative = True
        for i in range(n):
            if candidates[i] >= 0:
                all_negative = False
                break
        if all_negative:
            print(-1)
        else:
            print(h_min)
    else:
        print(h_min)

if __name__ == '__main__':
    main()
```

2. **Candidate Calculation**: For each building i, compute the minimal height h_i required to see it by evaluating the line conditions against all buildings j to its left. This involves solving the inequality derived from the line's equation to ensure visibility.
3. **Determine Minimal Height**: For each building, if the computed minimal height is positive, it is retained; otherwise, 0 is used. The overall minimal height is the maximum of these values.
4. **Output Result**: If the overall minimal height is 0 and all buildings are visible at height 0 (checked by ensuring all candidates are negative), output -1. Otherwise, output the computed minimal height.

This approach efficiently checks the visibility condition for each building and determines the critical height from which all buildings can be seen, providing the solution as required.
