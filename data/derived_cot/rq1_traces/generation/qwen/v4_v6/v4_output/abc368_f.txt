To solve this problem, we need to determine the winner of a game played between Anna and Bruno, who take turns replacing integers in a sequence with one of their divisors (excluding the number itself). The player who cannot make a move loses. The solution involves analyzing the game using combinatorial game theory, specifically the concept of Grundy numbers (or Nimbers), which helps in determining the optimal moves for both players.

### Approach
1. **Grundy Numbers Calculation**: The Grundy number for each integer in the sequence is defined as the total number of prime factors (counted with multiplicity). This is because each move from a number `n` to one of its divisors `d` effectively reduces the number of prime factors, and the game's outcome depends on the XOR (nim-sum) of these Grundy numbers.
2. **Precomputation**: We precompute the smallest prime factors (SPF) for all numbers up to the maximum value in the input (100,000) using a sieve method. This allows efficient factorization of any number in the range.
3. **Factor Count Array**: Using the SPF array, we compute an array `factor_count` where each entry at index `i` holds the total number of prime factors (with multiplicity) of `i`.
4. **Game Outcome Determination**: The overall game state's Grundy number is the XOR of the Grundy numbers of all integers in the sequence. If this XOR is non-zero, Anna (the first player) can force a win; otherwise, Bruno wins.

### Solution Code

import sys

def main():
    data = sys.stdin.read().split()
    if not data:
        return
    n = int(data[0])
    A = list(map(int, data[1:1+n]))
    
    max_val = 100000
    spf = list(range(max_val+1))
    for i in range(2, int(max_val**0.5) + 1):
        if spf[i] == i:
            for j in range(i*i, max_val+1, i):
                if spf[j] == j:

        f[i] = count + f[temp]
