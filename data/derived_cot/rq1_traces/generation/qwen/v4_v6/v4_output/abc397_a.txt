=== ANSWER ===

To solve this problem, we need to classify Takahashi's body temperature into one of three categories based on given temperature ranges. The classification is straightforward: high fever for temperatures 38.0°C and above, fever for temperatures between 37.5°C and below 38.0°C, and normal for temperatures below 37.5°C. 

### Approach
1. **Input Reading**: Read the input temperature value from standard input.
2. **Temperature Comparison**: Compare the input temperature against the predefined thresholds to determine the appropriate classification.
3. **Output Result**: Based on the comparison, output the corresponding integer value (1 for high fever, 2 for fever, and 3 for normal).

### Solution Code
```python
def main():
    X = float(input().strip())
    if X >= 38.0:
        print(1)
    elif X >= 37.5:
        print(2)
    else:
        print(3)

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The code reads the input temperature as a floating-point number to handle the one decimal place precision.
2. **Condition Checks**: The code checks if the temperature is 38.0°C or higher, printing 1 for "High fever". If not, it checks if the temperature is 37.5°C or higher but less than 38.0°C, printing 2 for "Fever". If neither condition is met, it prints 3 for "Normal".
3. **Efficiency**: The solution efficiently uses conditional statements to determine the correct output in constant time, O(1), making it optimal for the given constraints. The solution handles all edge cases, such as exactly 38.0°C or 37.5°C, correctly by leveraging the comparison operators.
