To solve this problem, we need to find the minimum cost to travel from vertex 1 to vertex N in a directed graph, where we can either move along an edge (costing 1) or reverse all edges (costing X). The reversal operation flips the direction of all edges, allowing us to traverse edges in the opposite direction. 

### Approach
1. **Problem Analysis**: The problem involves navigating a directed graph with the option to reverse all edges at a cost. The key insight is to model the problem using two graphs: the original graph (state 0) and its reversed version (state 1). Each vertex can be in one of these two states, and transitions between states (via edge reversal) incur a cost X.

2. **Graph Representation**: We represent the graph using two adjacency lists:
   - `adj0` for the original graph (edges as given).
   - `adj1` for the reversed graph (edges flipped from the original).

3. **Dijkstra's Algorithm**: We use Dijkstra's algorithm on a graph where each node is represented as (vertex, state). The algorithm explores the shortest path from vertex 1 (state 0) to vertex N (either state). The priority queue processes nodes by cost, updating distances for both states when moving or reversing edges.

4. **Complexity Analysis**: The algorithm processes up to 2N nodes (vertices × states) and O(M) edges (original and reversed). The time complexity is O((2N + M) log (2N)), which is efficient given the constraints (N and M up to 2×10^5).

### Solution Code
```python

import sys

def main():
    data = sys.stdin.read().split()
    if not data:
        return
    
    it = iter(data)
    n = int(next(it)); m = int(next(it)); X = int(next(it))
    
    adj0 = [[] for _ in range(n+1)]
    adj1 = [[] for _ in range(n+1)]
    
    for _ in range(m):
        u = int(next(it)); v = int(next(it))
        adj0[u].append(v)
        adj1[v].append(u)
    
    dist0 = [10**18] * (n+1)
    dist1 = [10**18] * (n+1)
    
    heap = []
    dist0[1] = 0
    heapq.heappush(heap, (0, 1, 0))
    
    while heap:
        cost, node, state = heapq.heappop(heap)
        if state == 0 and cost != dist0[node]:
            continue
        if state == 1 and cost != dist1[node]:
            continue
            
        if state == 0:
            for neighbor in adj0[node]:
                new_cost = cost + 1
                if new_cost < dist0[neighbor]:
                    dist0[neighbor] = new_cost
                    heapq.heappush(heap, (new_cost, neighbor, 0))
            new_cost = cost + X
            if new_cost < dist1[node]:
                dist1[node] = new_cost
                heapq.heappush(heap, (new_cost, node, 1))
        else:
            for neighbor in adj1[node]:
                new_cost = cost + 1
                if new_cost < dist1[neighbor]:
                    dist1[neighbor] = new_cost
                    heapq.heappush(heap, (new_cost, neighbor, 1))
            new_cost = cost + X
            if new_cost < dist0[node]:
                dist0[node] = new_cost
                heapq.heappush(heap, (new_cost, node, 0))
                
    answer = min(dist0[n], dist1[n])
    print(answer)

if __name__ == '__main__':
    main()
```
