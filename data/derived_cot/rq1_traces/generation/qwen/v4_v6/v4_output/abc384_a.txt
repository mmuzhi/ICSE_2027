=== ANSWER ===

To solve this problem, we need to transform a given string by replacing every character that is not a specified character (c_1) with another specified character (c_2). The solution involves reading the input values, processing each character of the string, and constructing the result string based on the given conditions.

### Approach
1. **Read Input:** The first line of input contains an integer N (the length of the string) and two lowercase English letters c_1 and c_2. The second line is the string S of length N.
2. **Process String:** For each character in the string S, check if it matches c_1. If it does, retain the character; otherwise, replace it with c_2.
3. **Output Result:** Construct the resulting string by appending the processed characters and print it.

This approach efficiently processes each character in linear time, making it optimal for the given constraints.

### Solution Code
```python
def main():
    data = input().split()
    n = int(data[0])
    c1 = data[1]
    c2 = data[2]
    s = input().strip()
    
    result = ''.join(c2 if char != c1 else char for char in s)
    print(result)

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input:** The first line is split into three parts: the integer N (though not used directly in processing), and the two characters c_1 and c_2. The second line is read as the string S.
2. **String Processing:** The string S is processed character by character. Each character is checked against c_1. If it matches, the character remains unchanged; otherwise, it is replaced with c_2.
3. **Output:** The processed characters are joined into a single string, which is then printed as the result.

This method ensures that all characters not equal to c_1 are replaced with c_2, while those that are c_1 remain unchanged, efficiently producing the desired output. The solution handles all edge cases, including when the input string is identical to the output string (as in Sample Input 2).
