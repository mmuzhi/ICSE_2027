To solve this problem, we need to find the shortest substring in a given string `s` that matches a pattern string `p` containing exactly two '*' characters. The '*' characters can match any sequence of zero or more characters. The solution involves breaking down the pattern into three parts (A, B, C) separated by the two '*' characters and then efficiently searching for the shortest substring in `s` that starts with A, contains B, and ends with C.

### Approach
1. **Pattern Decomposition**: The pattern `p` is split into three parts (A, B, C) based on the positions of the two '*' characters. A is the substring before the first '*', B is the substring between the two '*' characters, and C is the substring after the second '*'.
2. **Special Cases Handling**: 
   - If the pattern is "**", the answer is 0 since the empty substring matches.
   - If only one of A, B, or C is non-empty, the solution involves checking if that part exists in `s` and returning its length if found.
3. **General Case Handling**: For patterns with all three parts (A, B, C) non-empty, we:
   - Find all occurrences of A in `s`.
   - For each occurrence of A, find the smallest occurrence of C such that the substring between A and C contains B.
   - The length of the valid substring is calculated as the distance from the start of A to the end of C plus the length of C.
   - The minimal length across all valid substrings is the answer.

### Solution Code
```python
class Solution:
    def shortestMatchingSubstring(self, s: str, p: str) -> int:
        if p == "**":
            return 0
        
        i1 = p.find('*')
        i2 = p.find('*', i1 + 1)
        if i1 == -1 or i2 == -1:
            return -1
        
        A = p[:i1]
        B = p[i1+1:i2]
        C = p[i2+1:]
        
        if A == "" and B == "":
            return len(C) if C in s else -1
        elif A == "" and C == "":
            return len(B) if B in s else -1
        elif B == "" and C == "":
            return len(A) if A in s else -1
        elif A == "" and B != "" and C != "":
            return -1
        elif A != "" and B == "" and C == "":
            return -1
        elif A != "" and B == "" and C != "":
            return -1
        elif A == "" and B != "" and C == "":
            return -1
        elif A != "" and B != "" and C == "":
            return -1
        else:
            if A == "" or B == "" or C == "":
                return -1
            n = len(s)
            lenA = len(A)
            lenB = len(B)
            lenC = len(C)
            if lenA + lenB + lenC > n:
                return -1
            
            ans = float('inf')
            for i in range(n - lenA - lenB - lenC + 1):
                if s[i:i+lenA] != A:
                    continue
                for j in range(i + lenA, n - lenC + 1):
                    if s[j:j+lenC] != C:
                        continue
                    if B in s[i:j+lenC]:
                        ans = min(ans, j - i + lenC)
            return ans if ans != float('inf') else -1
```
