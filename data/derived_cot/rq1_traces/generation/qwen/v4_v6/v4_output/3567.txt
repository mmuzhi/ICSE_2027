=== ANSWER ===

To solve this problem, we need to convert a given Gregorian calendar date in the format "yyyy-mm-dd" into its binary representation. The binary representation should be formed by converting each component of the date (year, month, and day) into its binary form without leading zeros, and then combining them in the same "yyyy-mm-dd" format but with the binary strings.

### Approach
1. **Split the Input Date**: The input string is split into three parts using the hyphens ('-') as delimiters. This separates the year, month, and day components.
2. **Convert to Integers**: Each part of the split string is converted from a string to an integer. This allows us to perform the binary conversion accurately.
3. **Binary Conversion**: Each integer component is converted to its binary representation. Python's built-in `bin()` function is used for this purpose, and the '0b' prefix is stripped from the resulting string.
4. **Combine the Results**: The binary strings of the year, month, and day are combined into a single string, separated by hyphens, to form the final output.

### Solution Code
```python
class Solution:
    def convertDateToBinary(self, date: str) -> str:
        parts = date.split('-')
        year, month, day = map(int, parts)
        bin_year = bin(year)[2:]
        bin_month = bin(month)[2:]
        bin_day = bin(day)[2:]
        return f"{bin_year}-{bin_month}-{bin_day}"
```

### Explanation
1. **Splitting the Date**: The input string is split into three parts using the `split('-')` method. For example, "2080-02-29" is split into ["2080", "02", "29"].
2. **Conversion to Integers**: Each part of the split string is converted to an integer using `map(int, parts)`. This step ensures that each component is treated as a numerical value for binary conversion.
3. **Binary Conversion**: The `bin()` function converts each integer to its binary representation. For instance, `bin(2080)[2:]` converts 2080 to its binary string "100000100000" by removing the '0b' prefix.
4. **Combining Results**: The binary strings of the year, month, and day are combined into a single string using an f-string, resulting in the final output in the required format. For example, the input "2080-02-29" is converted to "100000100000-10-11101".

This approach efficiently handles the conversion and formatting required by the problem, ensuring clarity and correctness.
