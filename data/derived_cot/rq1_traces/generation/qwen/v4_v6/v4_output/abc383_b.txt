You will be given a competitive programming problem.
The AtCoder company office can be represented as a grid of H rows and W columns. Let (i, j) denote the cell at the i-th row from the top and j-th column from the left.
The state of each cell is represented by a character S_{i,j}. If S_{i,j} is #, that cell contains a desk; if S_{i,j} is ., that cell is a floor. It is guaranteed that there are at least two floor cells.
You will choose two distinct floor cells and place a humidifier on each.
After placing the humidifiers, a cell (i,j) is humidified if and only if it is within a Manhattan distance D from at least one of the humidifier cells (i',j'). The Manhattan distance between (i,j) and (i',j') is defined as |i - i'| + |j - j'|.
Note that any floor cell on which a humidifier is placed is always humidified.
Find the maximum possible number of humidified floor cells.

Input

The input is given from Standard Input in the following format:
H W D
S_{1,1}S_{1,2}\cdotsS_{1,W}
S_{2,1}S_{2,2}\cdotsS_{2,W}
\vdots
S_{H,1}S_{H,2}\cdotsS_{H,W}

Output

Print the answer.

Constraints


- 1 \leq H \leq 10
- 1 \leq W \leq 10
- 2 \leq H \times W
- 0 \leq D \leq H+W-2
- H,W,D are integers.
- S_{i,j} is # or .. (1 \leq i \leq H, 1 \leq j \leq W)
- There are at least two floor cells.

.###.
.#.##

Sample Output 1

3

In total, three cells are humidified. No configuration can humidify four or more floor cells, so the answer is 3.

=== ANSWER ===

To solve this problem, we need to maximize the number of floor cells in a grid that are humidified by placing two distinct humidifiers. A cell is considered humidified if it is within a Manhattan distance D from at least one of the humidifiers. The solution involves checking all possible pairs of floor cells to determine which pair covers the maximum number of floor cells when their humidification ranges overlap.

### Approach
1. **Problem Analysis**: The grid consists of floor cells (.) and desk cells (#). We need to select two distinct floor cells to place humidifiers. Each humidifier covers all floor cells within a Manhattan distance D. The goal is to find the pair of floor cells such that the union of their coverage includes the maximum number of floor cells.
2. **Intuition**: Since the grid dimensions are small (up to 10x10), a brute-force approach is feasible. We can iterate over all possible pairs of floor cells, compute the number of floor cells covered by each pair, and keep track of the maximum count.
3. **Algorithm Selection**: For each pair of floor cells, calculate the Manhattan distance from every other floor cell to both humidifiers. If the distance to either humidifier is ≤ D, the floor cell is counted. The pair with the highest count of such floor cells is the solution.
4. **Complexity Analysis**: The number of floor cells can be up to 100 (in a 10x10 grid). The number of pairs is C(100, 2) = 4950. For each pair, we check up to 100 floor cells, resulting in 4950 * 100 = 495,000 operations, which is computationally feasible for the given constraints.

### Solution Code
```python
def main():
    import sys
    data = sys.stdin.read().splitlines()
    if not data:
        return
    
    H, W, D = map(int, data[0].split())
    grid = []
    for i in range(1, 1+H):
        grid.append(data[i].strip())
    
    F = []
    for i in range(H):
        for j in range(W):
            if grid[i][j] == '.':
                F.append((i, j))
                
    n = len(F)
    if n < 2:
        print(0)
        return
        
    max_count = 0
    for i in range(n):
        for j in range(i+1, n):
            x1, y1 = F[i]
            x2, y2 = F[j]
            count = 0
            for k in range(n):
                a, b = F[k]
                d1 = abs(a - x1) + abs(b - y1)
                d2 = abs(a - x2) + abs(b - y2)
                if d1 <= D or d2 <= D:
                    count += 1
            if count > max_count:
                max_count = count
                
    print(max_count)

if __name__ == '__main__':
    main()
```

2. **Floor Cell Collection**: We collect all coordinates of floor cells (.) into a list F.
3. **Brute-Force Pair Check**: For each unique pair of floor cells in F, we calculate the Manhattan distance from every floor cell to both humidifiers. If a floor cell is within distance D of either humidifier, it is counted.
4. **Tracking Maximum Count**: The maximum count of humidified floor cells across all pairs is tracked and printed as the result.

This approach efficiently checks all possible pairs of floor cells to determine the optimal placement of humidifiers, ensuring the maximum number of floor cells are humidified within the given Manhattan distance constraint.
