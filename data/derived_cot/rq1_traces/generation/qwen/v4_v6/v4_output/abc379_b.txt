To solve this problem, we need to determine the maximum number of strawberries Takahashi can eat given a string representing his teeth and the requirement of K consecutive healthy teeth (represented by 'O') to eat one strawberry. Each time a strawberry is eaten, the K teeth used become unhealthy ('X'), and they cannot be reused.

### Approach
1. **Problem Analysis**: The problem requires counting the maximum number of non-overlapping segments of K consecutive 'O's in the string. Each segment allows Takahashi to eat one strawberry. The key observation is that each segment of 'O's can contribute floor(length_of_segment / K) strawberries. 

2. **Intuition**: By splitting the string into contiguous segments of 'O's separated by 'X's, we can independently calculate the number of strawberries each segment can produce. The sum of these values across all segments gives the total maximum strawberries Takahashi can eat.

3. **Algorithm Selection**: The solution involves:
   - Splitting the input string by 'X' to isolate contiguous segments of 'O's.
   - For each segment, compute the number of strawberries as the floor division of the segment's length by K.
   - Sum these values to get the total number of strawberries.

4. **Complexity Analysis**: The algorithm processes the string in linear time, O(N), where N is the length of the string. This is efficient given the constraints (N ≤ 100).

### Solution Code
```python
n, k = map(int, input().split())
s = input().strip()

segments = s.split('X')
total = 0
for seg in segments:
    total += len(seg) // k

print(total)
```

### Explanation
1. **Reading Input**: The first line of input contains integers N (length of the string) and K (required consecutive 'O's for a strawberry). The second line is the string S representing Takahashi's teeth.
2. **Splitting Segments**: The string S is split into segments of contiguous 'O's using 'X' as the delimiter. For example, "OOXOOOO" splits into ["OO", "OOOO"].
3. **Counting Strawberries**: For each segment, the number of strawberries is calculated as the floor division of the segment's length by K. For instance, a segment "OOOO" (length 4) with K=3 yields 1 strawberry (4 // 3 = 1).
4. **Output Result**: The total strawberries from all segments is printed, which is the solution to the problem.

This approach efficiently counts the maximum strawberries by leveraging simple string manipulation and arithmetic operations, ensuring optimal performance for the given constraints.
