To solve this problem, we need to compute the sum of the maximum and minimum elements of all subsequences of a given array with at most k elements. The solution involves a combinatorial approach where we efficiently calculate the contributions of each element in the sorted array as the minimum or maximum in various subsequences.

### Approach
1. **Sorting the Array**: The first step is to sort the array to easily determine the number of elements less than or greater than any given element.
2. **Precompute Modular Inverses**: Since we need to compute binomial coefficients modulo \(10^9 + 7\), we precompute the modular inverses for numbers 1 to 70 (as k is at most 70) using Fermat's Little Theorem.
3. **Binomial Coefficient Sums**: For each possible count of elements (from 0 to n-1), precompute the sum of binomial coefficients \(C(n, j)\) for \(j\) from 0 to k-1. This helps in quickly determining the number of valid subsequences for any given element.
4. **Calculate Contributions**: For each element in the sorted array, determine the number of subsequences where it is the minimum or maximum. This involves counting elements less than or greater than the current element and using the precomputed binomial sums to find the valid subsequences.
5. **Summing Contributions**: Multiply each element's value by its contributions as minimum and maximum, and sum these values modulo \(10^9 + 7\) to get the final result.

### Solution Code
```python
class Solution:
    def minMaxSums(self, nums: List[int], k: int) -> int:
        mod = 10**9 + 7
        
        # Precompute modular inverses for 1 to 70
        max_inv = 70
        inv = [0] * (max_inv + 1)
        for i in range(1, max_inv + 1):
            inv[i] = pow(i, mod - 2, mod)
        
        n = len(nums)
        nums_sorted = sorted(nums)
        
        # Precompute binom_sums_arr for count_val from 0 to n-1
        binom_sums_arr = [0] * n
        
        for count_val in range(0, n):
            res = [0] * k
            if k > 0:
                res[0] = 1
            for j in range(1, k):
                if j > count_val:
                    res[j] = 0
                else:
                    res[j] = res[j - 1] * (count_val - j + 1) % mod * inv[j] % mod
            binom_sums_arr[count_val] = sum(res) % mod
        
        total = 0
        for x in nums:
            count_less = bisect.bisect_left(nums_sorted, x)
            count_ge = n - count_less
            count_less_and_equal = count_less + (n - count_ge)
            min_count = count_ge - 1
            max_count = count_less_and_equal - 1
            min_val = binom_sums_arr[min_count] if min_count >= 0 and min_count < n else 0
            max_val = binom_sums_arr[max_count] if max_count >= 0 and max_count < n else 0
            total = (total + x * (min_val + max_val)) % mod
        
        return total % mod
```
