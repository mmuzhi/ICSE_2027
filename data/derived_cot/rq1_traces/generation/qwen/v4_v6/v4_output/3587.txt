=== ANSWER ===

To solve this problem, we need to maximize the points a tourist can earn over exactly k days by either staying in the current city or moving to another city each day. The solution involves dynamic programming to efficiently compute the maximum points by considering all possible city transitions and actions (stay or move) for each day.

### Approach
1. **Problem Analysis**: The tourist starts in any city and has two choices each day: stay in the current city to earn points from `stayScore`, or move to another city to earn points from `travelScore`. The goal is to maximize the total points over k days.
2. **Dynamic Programming Setup**: We use a dynamic programming (DP) array `dp` where `dp[j]` represents the maximum points achievable after processing the previous day and ending in city `j`.
3. **Initialization (Day 0)**: For each city `j`, the initial value is the maximum between staying in `j` (using `stayScore[0][j]`) or moving into `j` from any other city (using `travelScore[i][j]` for all `i != j`).
4. **Iterate Over Days**: For each subsequent day, update the DP array by considering both staying in the current city (adding `stayScore[day][j]` to the previous day's value for city `j`) or moving into `j` from any other city (adding `travelScore[i][j]` to the previous day's value for city `i`).
5. **Result Extraction**: After processing all k days, the maximum value in the DP array gives the answer.

### Solution Code
```python
class Solution:
    def maxScore(self, n: int, k: int, stayScore: List[List[int]], travelScore: List[List[int]]) -> int:
        dp = [0] * n
        for j in range(n):
            stay_val = stayScore[0][j]
            move_val = -10**18
            for i in range(n):
                if i != j:
                    if travelScore[i][j] > move_val:
                        move_val = travelScore[i][j]
            dp[j] = max(stay_val, move_val)
        
        for day in range(1, k):
            new_dp = [0] * n
            for j in range(n):
                stay_val = dp[j] + stayScore[day][j]
                move_val = -10**18
                for i in range(n):
                    if i != j:
                        candidate = dp[i] + travelScore[i][j]
                        if candidate > move_val:
                            move_val = candidate
                new_dp[j] = max(stay_val, move_val)
            dp = new_dp
        
        return max(dp)
```

### Explanation
1. **Initialization**: The DP array `dp` is initialized for day 0. For each city `j`, the value is set to the maximum points obtained by either staying in `j` or moving into `j` from any other city.
2. **Day Processing**: For each subsequent day, a new DP array `new_dp` is computed. For each city `j`, the value is updated by considering both staying in `j` (adding the day's `stayScore[j]` to the previous day's `dp[j]`) or moving into `j` (adding `travelScore[i][j]` to the previous day's `dp[i]` for all `i != j`).
3. **Result**: After processing all k days, the maximum value in the DP array is returned, representing the highest possible points the tourist can earn by optimally choosing to stay or move each day.

This approach efficiently explores all possible paths using dynamic programming, leveraging the fact that each day's decision only depends on the previous day's results, thus optimizing both time and space complexity.
