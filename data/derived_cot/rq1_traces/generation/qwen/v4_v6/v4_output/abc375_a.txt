=== ANSWER ===

To solve this problem, we need to count the number of valid triplets in a row of seats where the first and third seats are occupied ('#') and the second seat is unoccupied ('.'). The seats are represented by a string where each character corresponds to a seat's state.

### Approach
1. **Problem Analysis**: The problem requires checking each possible triplet of consecutive seats in the given string. For each triplet starting at position `i` (0-indexed), we need to verify if the seats at positions `i`, `i+1`, and `i+2` satisfy the condition: the first seat is '#', the second is '.', and the third is '#'.
2. **Intuition**: By iterating through the string from the start to the third last seat, we can check each triplet in constant time. The solution efficiently counts all such valid triplets by leveraging a simple loop.
3. **Algorithm Selection**: The algorithm involves a single pass through the string, checking each triplet of consecutive seats. This approach ensures optimal performance with a time complexity of O(N), where N is the number of seats, as each seat is checked at most once.
4. **Complexity Analysis**: The algorithm runs in linear time, O(N), which is efficient for the given constraints (N ≤ 2 × 10^5). The space complexity is O(1) since we only use a few extra variables for counting and indexing.

### Solution Code
```python
n = int(input().strip())
s = input().strip()

count = 0
for i in range(n - 2):
    if s[i] == '#' and s[i + 1] == '.' and s[i + 2] == '#':
        count += 1

print(count)
```
