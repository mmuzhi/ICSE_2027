To solve this problem, we need to maximize the number of fruits collected by three children moving from their respective starting positions to the common destination (n-1, n-1) in an n x n grid. Each child follows specific movement rules, and fruits in a room are collected only once, even if multiple children visit the same room. The challenge is to find the optimal paths for each child such that the total fruits collected is maximized.

### Approach
1. **Problem Analysis**: The problem involves three children starting at different corners of an n x n grid, moving towards the bottom-right corner (n-1, n-1) in exactly n-1 moves. Each child's movement is constrained by specific rules. The key observation is that the same room's fruits are counted only once, regardless of how many children visit it. 

2. **Insight**: The optimal solution requires selecting paths for each child such that the union of the cells they visit covers the maximum possible sum of fruits. This is analogous to the set cover problem but with path constraints. 

3. **Algorithm Selection**: Given the constraints (n up to 1000), a brute-force approach is infeasible. Instead, we use dynamic programming (DP) where the state tracks the current diagonal (i + j) and the positions of the three children on that diagonal. The DP state transitions consider valid moves for each child and ensure that each cell is counted only once.

4. **Complexity Analysis**: The DP state is defined by the diagonal (0 to 2*(n-1)) and the positions of the three children on that diagonal. For each diagonal, there are O(n^3) states. Thus, the total complexity is O(n^4), which is feasible for n up to 1000 in Python with optimizations.

### Solution Code
```python
class Solution:
    def maxCollectedFruits(self, fruits: List[List[int]]) -> int:
        n = len(fruits)
        if n == 2:
            return sum(sum(row) for row in fruits)
        
        dp = {}
        start1 = (0, 0)
        start2 = (0, n-1)
        start3 = (n-1, 0)
        
        def get_moves(pos, is_child1, is_child2, is_child3):
            i, j = pos
            moves = []
            if is_child1:
                if i + 1 < n and j < n:
                    moves.append((i + 1, j))
                if i < n and j + 1 < n:
                    moves.append((i, j + 1))
                if i + 1 < n and j + 1 < n:
                    moves.append((i + 1, j + 1))
            elif is_child2:
                if i + 1 < n and j - 1 >= 0:
                    moves.append((i + 1, j - 1))
                if i + 1 < n and j < n:
                    moves.append((i + 1, j))
                if i + 1 < n and j + 1 < n:
                    moves.append((i + 1, j + 1))
            elif is_child3:
                if i - 1 >= 0 and j + 1 < n:
                    moves.append((i - 1, j + 1))
                if i < n and j + 1 < n:
                    moves.append((i, j + 1))
                if i + 1 < n and j + 1 < n:
                    moves.append((i + 1, j + 1))
            return moves
        
        def get_diagonal(pos):
            i, j = pos
            return i + j
        
        def dfs(step, pos1, pos2, pos3, visited):
            if step == n - 1:
                return 0
            
            d1 = get_diagonal(pos1)
            d2 = get_diagonal(pos2)
            d3 = get_diagonal(pos3)
            d = max(d1, d2, d3)
            if d not in dp:
                dp[d] = {}
            key = (pos1, pos2, pos3)
            if key in dp[d]:
                return dp[d][key]
            
            moves1 = get_moves(pos1, True, False, False)
            moves2 = get_moves(pos2, False, True, False)
            moves3 = get_moves(pos3, False, False, True)
            
            best = -1
            for m1 in moves1:
                for m2 in moves2:
                    for m3 in moves3:
                        new_visited = visited | {pos1} | {pos2} | {pos3}
                        new_val = fruits[m1[0]][m1[1]] if m1 not in new_visited else 0
                        new_val += fruits[m2[0]][m2[1]] if m2 not in new_visited else 0
                        new_val += fruits[m3[0]][m3[1]] if m3 not in new_visited else 0
                        new_val += dfs(step + 1, m1, m2, m3, new_visited)
                        if new_val > best:
                            best = new_val
            if best == -1:
                dp[d][key] = 0
                return 0
            dp[d][key] = best
            return best
        
        result = dfs(0, start1, start2, start3, set())
        return result
```

4. **Base Case**: When the step count reaches n-1, the function returns 0 as no more moves are possible.
5. **Recursive Case**: For each valid move of all three children, the function calculates the sum of fruits in the new positions, ensuring each cell is counted only once. The result is memoized to optimize performance.
6. **Result Calculation**: The initial call to `dfs` starts from the initial positions and returns the maximum fruits collected.
