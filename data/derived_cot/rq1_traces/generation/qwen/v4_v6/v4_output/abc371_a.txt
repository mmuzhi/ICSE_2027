=== ANSWER ===

To solve this problem, we need to determine the middle brother (the second oldest) among three brothers A, B, and C based on given age relationships. The relationships are provided as three characters indicating whether one brother is older or younger than another. 

### Approach
1. **Problem Analysis**: The problem involves interpreting three given relationships (S_AB, S_AC, S_BC) which specify whether A is younger or older than B, A than C, and B than C, respectively. The goal is to find the brother who is the second oldest (middle brother) based on these relationships.
2. **Intuition**: The solution involves checking all possible permutations of the three brothers (A, B, C) to see which permutation satisfies all given relationships. Each permutation represents a potential order from oldest to youngest. For each permutation, we verify if the positions of the brothers match the given relationships.
3. **Algorithm Selection**: We use itertools.permutations to generate all possible orders of the three brothers. For each permutation, we check if the conditions derived from the input relationships hold true. The first permutation that satisfies all conditions will have the middle brother as the one at the second position in the permutation.
4. **Complexity Analysis**: The algorithm checks all 6 possible permutations of three elements, which is a constant time operation O(1) since the number of permutations is fixed (3! = 6). This makes the solution efficient and straightforward.

### Solution Code
```python
import itertools

def main():
    s_ab, s_ac, s_bc = input().split()
    brothers = ['A', 'B', 'C']
    for perm in itertools.permutations(brothers):
        pos = {brother: idx for idx, brother in enumerate(perm)}
        cond1 = (s_ab == '>') == (pos['A'] < pos['B'])
        cond2 = (s_ac == '>') == (pos['A'] < pos['C'])
        cond3 = (s_bc == '>') == (pos['B'] < pos['C'])
        if cond1 and cond2 and cond3:
            print(perm[1])
            return

if __name__ == '__main__':
    main()
```

### Explanation
1. **Input Handling**: The input consists of three characters (S_AB, S_AC, S_BC) which are read and split into individual variables.
2. **Permutation Generation**: Using itertools.permutations, all possible orders of the three brothers (A, B, C) are generated. Each permutation represents a potential age order from oldest to youngest.
3. **Condition Checking**: For each permutation, the positions of A, B, and C are determined. The conditions derived from the input relationships (S_AB, S_AC, S_BC) are checked. Specifically:
   - S_AB must be '>' if A is older than B (i.e., A appears before B in the permutation) or '<' if A is younger.
   - Similarly, S_AC and S_BC are checked based on their respective positions.
4. **Output**: The first permutation that satisfies all conditions is selected, and the middle brother (the second element in the permutation) is printed. The loop terminates immediately after finding the correct permutation due to the problem's guarantee of a unique solution without contradictions. 

This approach efficiently narrows down the correct age order by leveraging permutation checks and ensures the solution is both correct and optimal.
