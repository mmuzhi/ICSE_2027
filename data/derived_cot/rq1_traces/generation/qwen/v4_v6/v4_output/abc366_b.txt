=== ANSWER ===

To solve this problem, we need to convert a given set of strings into a vertical format by columns, filling any missing positions with asterisks (*). The key insight is to arrange each character from the input strings into specific positions in the output strings based on their original positions and the length of their respective strings.

### Approach
1. **Problem Analysis**: The problem requires transforming a list of strings into a grid of M rows and M columns (where M is the maximum length among the strings). Each character from the i-th string should be placed in the (N-i+1)-th position of the j-th output string, where N is the number of strings. Positions not specified by any string should be filled with asterisks (*), and no output string should end with an asterisk.

2. **Intuition**: The solution involves creating a grid where each row corresponds to an output string. For each string in the input, its characters are placed in specific columns of the grid, determined by the string's index. The grid is then filled with asterisks in any remaining positions.

3. **Algorithm Selection**: 
   - Read the number of strings (N) and the strings themselves.
   - Determine M as the maximum length of the strings.
   - Initialize a grid with M rows and M columns, filled with asterisks.
   - For each string, place its characters in the grid at the calculated column positions.
   - Output each row of the grid as a string.

4. **Complexity Analysis**: The algorithm processes each character in the input strings once, leading to a time complexity of O(N*M), where N is the number of strings and M is the maximum string length. The space complexity is O(M^2) due to the grid storage.

### Solution Code
```python
import sys

def main():
    data = sys.stdin.read().splitlines()
    if not data:
        return
    n = int(data[0].strip())
    strings = []
    for i in range(1, 1 + n):
        strings.append(data[i].strip())
    
    M = max(len(s) for s in strings) if strings else 0
    
    if M == 0:
        for _ in range(n):
            print("")
        return
            
    grid = [['*'] * M for _ in range(M)]
    
    for i in range(n):
        s = strings[i]
        for j in range(len(s)):
            col = n - i - 1
            if 0 <= col < M and j < M:
                grid[j][col] = s[j]
                
    for row in grid:
        print(''.join(row))

if __name__ == "__main__":
    main()
```
