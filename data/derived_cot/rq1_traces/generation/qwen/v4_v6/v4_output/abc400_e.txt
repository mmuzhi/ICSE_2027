To solve this problem, we need to find the largest 400 number (a number with exactly two distinct prime factors, each raised to an even exponent) that is less than or equal to a given integer A. The solution involves precomputing all such numbers up to a certain limit and then efficiently answering each query by binary searching the precomputed list.

### Approach
1. **Precomputation**:
   - **Smallest Prime Factor (SPF) Array**: We first build an SPF array up to \(10^6\) (since \( \sqrt{10^{12}} = 10^6 \)) using a sieve method. This array helps in quickly factorizing any number by its smallest prime factor.
   - **Count Distinct Prime Factors**: Using the SPF array, we factorize each number from 2 to \(10^6\) and count the number of distinct prime factors. Numbers with exactly two distinct prime factors are stored in a list.
   - **Sort the List**: The list of numbers with exactly two distinct prime factors is sorted for efficient binary search during query processing.

2. **Query Processing**:
   - For each query, compute the integer square root of A (max_M), which is the largest possible M such that \(M^2 \leq A\).
   - Use binary search to find the largest number in the precomputed list that is less than or equal to max_M. The square of this number is the answer.

### Solution Code
```python
import math
import bisect

def main():
    max_val = 10**6
    spf = list(range(max_val + 1))
    sqrt_max_val = int(math.isqrt(max_val)) + 1
    for i in range(2, sqrt_max_val):
        if spf[i] == i:
            for j in range(i * i, max_val + 1, i):
                if spf[j] == j:
                    spf[j] = i
    
    count_arr = [0] * (max_val + 1)
    for num in range(2, max_val + 1):
        if spf[num] == num:
            count_arr[num] = 1
        else:
            factors = set()
            n = num
            while n > 1:
                p = spf[n]
                factors.add(p)
                n //= p
            count_arr[num] = len(factors)
    
    M_list = []
    for num in range(6, max_val + 1):
        if count_arr[num] == 2:
            M_list.append(num)
    
    M_list.sort()
    
    import sys
    data = sys.stdin.read().split()
    Q = int(data[0])
    index = 1
    out_lines = []
    for _ in range(Q):
        A = int(data[index])
        index += 1
        max_M = math.isqrt(A)
        pos = bisect.bisect_right(M_list, max_M) - 1
        if pos < 0:
            M0 = 6
        else:
            M0 = M_list[pos]
        out_lines.append(str(M0 * M0))
    
    sys.stdout.write("\n".join(out_lines))

if __name__ == "__main__":
    main()
```

1. **Precomputation**:
   - The SPF array is built using a sieve method, marking the smallest prime factor for each number up to \(10^6\). This allows efficient factorization of any number in the range.
   - Each number from 2 to \(10^6\) is factorized using the SPF array, and the count of distinct prime factors is stored. Numbers with exactly two distinct prime factors are collected into a list and sorted.

2. **Query Processing**:
   - For each query, the integer square root of A is computed. Using binary search, the largest number in the precomputed list that is less than or equal to this square root is found. The square of this number is the answer, ensuring it meets the 400 number criteria (exactly two distinct prime factors, each raised to an even exponent).
