=== PROMPT ===
You will be given a competitive programming problem.
There are N people, labeled 1 to N. Person i has an integer A_i.
Among the people who satisfy the condition "None of the other N-1 people has the same integer as themselves," find the one with the greatest integer, and print that person's label.
If no person satisfies the condition, report that fact instead.

Input

The input is given from Standard Input in the following format:
N
A_1 A_2 \ldots A_N

Output

If no person satisfies the condition "None of the other N-1 people has the same integer as themselves," print -1.
Otherwise, among those who satisfy it, print the label of the person whose integer is the largest.

Constraints


- 1 \leq N \leq 3\times 10^5
- 1 \leq A_i \leq 10^9
- All input values are integers.

Sample Input 1

9
2 9 9 7 9 2 4 5 8

Sample Output 1

9

Those who satisfy the condition are the persons labeled 4, 7, 8, and 9.
Their integers are 7, 4, 5, and 8, respectively, and the person with the largest integer is the person labeled 9.
Thus, the answer is 9.

4
1000000000 1000000000 998244353 998244353

Sample Output 2

To solve this problem, we need to identify the person with the greatest integer among those who satisfy the condition that no other person has the same integer. The solution involves counting the frequency of each integer in the list and then determining the largest integer that appears exactly once. If no such integer exists, we return -1.

### Approach
1. **Frequency Counting**: We first traverse the list of integers to count how many times each integer appears. This helps us identify which integers are unique (appear exactly once).
2. **Tracking First Occurrence**: While counting frequencies, we also record the first occurrence of each integer. This is crucial because the answer requires the label (1-based index) of the unique integer with the highest value.
3. **Finding the Maximum Unique Integer**: After processing all integers, we iterate through the frequency dictionary to find the largest integer that appears exactly once.
4. **Result Determination**: If such an integer exists, we output its first occurrence's 1-based index. If no unique integers are found, we output -1.

### Solution Code
```python
import sys

def main():
    data = sys.stdin.read().split()
    if not data:
        print(-1)
        return
    n = int(data[0])
    A = list(map(int, data[1:1+n]))
    
    count = {}
    first_occurrence = {}
    for i, num in enumerate(A):
        if num not in count:
            count[num] = 1
            first_occurrence[num] = i
        else:
            count[num] += 1
            
    max_val = None
    for num, cnt in count.items():
        if cnt == 1:
            if max_val is None or num > max_val:
                max_val = num
                
    if max_val is None:
        print(-1)
    else:
        print(first_occurrence[max_val] + 1)

if __name__ == '__main__':
    main()
```

4. **Finding the Maximum Unique Integer**: Among all unique integers, we find the one with the highest value.
