=== ANSWER ===

To solve this problem, we need to determine if a given string of three uppercase English letters can be rearranged to form the string "ABC". The solution involves checking whether the characters in the input string match the required characters exactly, in any order.

### Approach
1. **Problem Analysis**: The task is to check if the input string contains exactly one 'A', one 'B', and one 'C'. Since the string length is fixed at 3, any permutation of these three distinct characters will suffice. 
2. **Intuition**: By sorting the input string and comparing it to the sorted string "ABC" (which is ['A', 'B', 'C']), we can efficiently determine if the characters match. If the sorted versions are identical, the answer is "Yes"; otherwise, it's "No".
3. **Algorithm Selection**: The chosen algorithm involves sorting the input string and comparing it to the sorted target string. This approach leverages the fact that sorting provides a consistent way to compare multisets of characters.
4. **Complexity Analysis**: Sorting a string of length 3 is an O(1) operation, making the solution very efficient with constant time complexity.

### Solution Code
```python
def main():
    S = input().strip()
    if sorted(S) == ['A', 'B', 'C']:
        print("Yes")
    else:
        print("No")

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The input string `S` is read from standard input.
2. **Sorting and Comparison**: The string `S` is sorted lexicographically. For example, "BAC" sorted becomes ['A', 'B', 'C'], which matches the sorted target string. If the sorted input matches the sorted target, the output is "Yes"; otherwise, it's "No".
3. **Output Result**: Based on the comparison, the program prints "Yes" if the characters can be rearranged to form "ABC", otherwise it prints "No".

This approach efficiently checks for the required characters by leveraging sorting, ensuring correctness with minimal computational overhead.
