### Approach
1. **Problem Analysis**: The task involves generating all possible three-digit numbers from the given digits, ensuring that each digit is used only once per number, the number is even (last digit is even), and the number does not start with a zero. The solution requires counting distinct numerical values of these valid numbers.

2. **Intuition**: The approach involves generating all permutations of three digits from the input list. For each permutation, we check if the first digit is not zero and the last digit is even. Valid permutations are converted into integers and stored in a set to ensure only distinct numbers are counted.

3. **Algorithm Selection**: Using itertools.permutations to generate all possible three-digit combinations from the input list. For each combination, we verify the conditions (no leading zero and even last digit) and use a set to track distinct numbers formed.

4. **Complexity Analysis**: The algorithm's time complexity is O(n^3) where n is the number of digits (up to 10). This is manageable because 10^3 is 1000, and with the constraints (n ≤ 10), the maximum number of permutations is 720, which is efficient for the problem constraints.

### Solution Code
```python
from itertools import permutations
from typing import List

class Solution:
    def totalNumbers(self, digits: List[int]) -> int:
        perms = permutations(digits, 3)
        seen = set()
        for p in perms:
            if p[0] == 0:
                continue
            if p[2] % 2 == 0:
                num = p[0]*100 + p[1]*10 + p[2]
                seen.add(num)
        return len(seen)
```

### Explanation
