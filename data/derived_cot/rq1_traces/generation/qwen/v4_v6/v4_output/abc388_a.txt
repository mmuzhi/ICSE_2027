To solve this problem, we need to generate a new string by concatenating the first character of the input string with the substring "UPC". The input string is guaranteed to start with an uppercase letter followed by lowercase letters. 

### Approach
1. **Input Handling**: Read the input string from standard input.
2. **Extract First Character**: The first character of the input string is isolated.
3. **Concatenation**: Combine the first character with the string "UPC" to form the result.

### Solution Code
```python

    S = input().strip()
    result = S[0] + "UPC"
    print(result)

if __name__ == '__main__':
    main()
```
