=== ANSWER ===

To solve this problem, we need to determine if a given string meets the criteria of being an 11/22 string. The criteria are:
1. The string's length must be odd.
2. The first half of the string (excluding the middle character) must consist entirely of '1's.
3. The middle character must be '/'.
4. The second half of the string (excluding the middle character) must consist entirely of '2's.

### Approach
1. **Check for Odd Length**: The first step is to verify if the string length is odd. If not, it immediately fails the criteria.
2. **Determine Middle Position**: Calculate the middle position of the string using integer division. This helps in splitting the string into three parts: the leading '1's, the middle '/', and the trailing '2's.
3. **Validate Leading Characters**: Ensure all characters before the middle position are '1's.
4. **Validate Middle Character**: Confirm that the middle character is '/'.
5. **Validate Trailing Characters**: Ensure all characters after the middle position are '2's.

### Solution Code
```python
n = int(input().strip())
s = input().strip()

if n % 2 == 0:
    print("No")
else:
    mid = (n + 1) // 2
    if any(c != '1' for c in s[:mid-1]):
        print("No")
    elif s[mid-1] != '/':
        print("No")
    elif any(c != '2' for c in s[mid:]):
        print("No")
    else:
        print("Yes")
```

### Explanation
1. **Odd Length Check**: The solution first checks if the string length is even. If it is, the answer is immediately "No" because the length must be odd.
2. **Middle Position Calculation**: The middle position is calculated as `(n + 1) // 2`, which correctly identifies the position of the '/' in the string.
3. **Leading Characters Check**: The solution checks if all characters before the middle position are '1's using the `any` function combined with a condition that checks for any non-'1' character.
4. **Middle Character Check**: The character at the middle position must be '/'.
5. **Trailing Characters Check**: Similarly, the solution checks if all characters after the middle position are '2's using the same approach as the leading characters.

If all checks pass, the string is an 11/22 string, and the solution outputs "Yes"; otherwise, it outputs "No". This approach efficiently verifies all conditions with clear, concise steps.
