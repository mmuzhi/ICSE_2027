2. **Intuition**: For each diagonal (defined by d = i - j), we collect all elements. For diagonals in the bottom-left triangle (d ≥ 0), we sort the elements in non-increasing order.    - **Step 2**: For each diagonal, retrieve the original values from the matrix, sort them according to the specified order (non-increasing for d ≥ 0, non-decreasing for d < 0), and then reassign these sorted values back to their original positions.
4. **Complexity Analysis**: The algorithm processes each element of the matrix exactly once to build the dictionary and once more to reassign the sorted values. The sorting step is O(k log k) for each diagonal, where k is the number of elements in the diagonal. ### Solution Code
```python
from collections import defaultdict

class Solution:
    def sortMatrix(self, grid: List[List[int]]) -> List[List[int]]:
        n = len(grid)
        original_grid = [row[:] for row in grid]
        diagonals = defaultdict(list)
        for i in range(n):
            for j in range(n):
                d = i - j
                diagonals[d].append((i, j))
        for d, positions in diagonals.items():
            values = [original_grid[i][j] for (i, j) in positions]
            if d >= 0:
                values.sort(reverse=True)
            else:
                values.sort()
            for idx, (i, j) in enumerate(positions):
                grid[i][j] = values[idx]
        return grid
```

   - The sorted values are then reassigned back to their original positions in the matrix.
4. **Result**: The modified matrix is returned, where all specified diagonals are sorted according to the given conditions.
