=== ANSWER ===

To solve this problem, we need to determine if there exists a permutation of the given three integers such that the product of the first two elements equals the third element. 

### Approach
1. **Problem Analysis**: The problem requires checking all possible permutations of the three given integers to see if any permutation satisfies the condition \(B_1 \times B_2 = B_3\). Given the small input size (only three numbers), we can directly check all possible conditions without generating all permutations explicitly.
2. **Key Insight**: For the three integers \(a\), \(b\), and \(c\), we need to verify if any two of them multiply to the third. This can be done by checking three conditions:
   - \(a \times b = c\)
   - \(a \times c = b\)
   - \(b \times c = a\)
3. **Algorithm Selection**: The solution involves reading the three integers and checking the above three conditions. If any of the conditions hold true, the answer is "Yes"; otherwise, it's "No".

### Solution Code
```python
def main():
    a, b, c = map(int, input().split())
    if a * b == c or a * c == b or b * c == a:
        print("Yes")
    else:
        print("No")

if __name__ == '__main__':
    main()
```

### Explanation
1. **Reading Input**: The three integers are read from standard input.
2. **Condition Check**: The code checks if any two of the integers multiply to the third. Specifically, it checks:
   - The product of the first and second integers equals the third.
   - The product of the first and third integers equals the second.
   - The product of the second and third integers equals the first.
3. **Output Result**: If any of the conditions is satisfied, the code prints "Yes"; otherwise, it prints "No". This approach efficiently checks all possible scenarios without the need for generating all permutations, leveraging the small input size for optimal performance.
