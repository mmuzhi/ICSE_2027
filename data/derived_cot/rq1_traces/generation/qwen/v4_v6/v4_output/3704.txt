To solve this problem, we need to determine the number of partitions in an array such that the difference between the sum of the left and right subarrays is even. The key observation here is that the difference between the sums of the two subarrays is even if and only if the total sum of the entire array is even. 

### Approach
1. **Problem Analysis**: The problem requires counting partitions where the difference between the left and right subarray sums is even. The difference is defined as left_sum - right_sum. The solution hinges on recognizing that this difference is even if and only if the total sum of the array is even. This is because the difference can be rewritten as `2*left_sum - total_sum`, and since `2*left_sum` is always even, the parity of the difference depends solely on the parity of the total sum.
2. **Insight**: If the total sum of the array is odd, no partition will satisfy the condition because the difference will always be odd. Conversely, if the total sum is even, every possible partition will satisfy the condition, and the number of such partitions is simply the number of ways to split the array, which is `n-1` (where `n` is the length of the array).
3. **Algorithm Selection**: The algorithm involves calculating the total sum of the array. If the total sum is odd, return 0. Otherwise, return `n-1` as the number of valid partitions.
4. **Complexity Analysis**: The algorithm runs in O(n) time due to the single pass through the array to compute the sum. The space complexity is O(1) as no additional space is used apart from a few variables.

### Solution Code
```python
class Solution:
    def countPartitions(self, nums: List[int]) -> int:
        total = sum(nums)
        if total % 2 != 0:
            return 0
        return len(nums) - 1
```

### Explanation
1. **Total Sum Calculation**: The code first calculates the total sum of all elements in the array.
2. **Parity Check**: If the total sum is odd, the function returns 0 because no partition can satisfy the condition (the difference will always be odd).
3. **Valid Partitions Count**: If the total sum is even, the function returns `n-1`, which is the number of valid partitions. This is because each partition (there are `n-1` possible partitions for an array of length `n`) will inherently have an even difference between the left and right subarray sums when the total sum is even.
This approach efficiently leverages the mathematical property that the difference between two numbers is even if their total sum is even, simplifying the problem to a straightforward check of the array's total sum.
