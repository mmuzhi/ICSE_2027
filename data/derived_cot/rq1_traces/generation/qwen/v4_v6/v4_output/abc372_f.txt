The graph is structured such that each vertex i (1 ≤ i ≤ N) has a directed edge to vertex i+1 (with vertex N+1 being vertex 1). Additionally, there are M edges that connect arbitrary vertices.

### Approach
1. **Problem Analysis**: The problem involves counting the number of walks of length K starting from vertex 1 in a directed graph. The graph is a combination of a cycle (each vertex i has an edge to i+1) and M additional edges. The challenge is to efficiently compute this count given the constraints (N and K can be as large as 2×10^5).

2. **Intuition**: The solution leverages dynamic programming (DP) where `dp[k][v]` represents the number of ways to reach vertex v in exactly k steps. The recurrence relation involves two parts:
   - **Cycle Edge Contribution**: For each vertex v, the number of ways to reach it via the cycle edge is `dp[k-1][prev_cycle(v)]`, where `prev_cycle(v)` is the predecessor of v in the cycle.
   - **Additional Edge Contribution**: For each vertex v, the number of ways to reach it via additional edges is the sum of `dp[k-1][a]` for all edges (a → v).

3. **Algorithm Selection**: The algorithm uses a DP array that is updated iteratively for each step from 1 to K. For each step, it computes the contributions from both cycle edges and additional edges. The complexity is O(K*(N + M)), which is feasible for moderate values of K and N, but may be borderline for the upper constraints.

4. **Optimization**: The solution optimizes by using modulo arithmetic to handle large numbers and by precomputing the list of additional edges for each vertex to efficiently calculate contributions during each iteration.

### Solution Code
```python
import sys

mod = 998244353

def main():
    data = sys.stdin.read().split()
    if not data:
        return
    
    it = iter(data)
    n = int(next(it))
    m = int(next(it))
    k_steps = int(next(it))
    
    edges = []
    for _ in range(m):
        u = int(next(it)) - 1
        v = int(next(it)) - 1
        edges.append((u, v))
    
    add_edges = [[] for _ in range(n)]
    for u, v in edges:
        add_edges[v].append(u)
    
    dp = [0] * n
    dp[0] = 1
    
    if k_steps == 0:
        print(1)
        return
    
    for _ in range(1, k_steps + 1):
        shifted = [dp[(i - 1) % n] for i in range(n)]
        additional = [0] * n
        for u, v in edges:
            additional[v] = (additional[v] + dp[u]) % mod
        new_dp = [ (shifted[i] + additional[i]) % mod for i in range(n) ]
        dp = new_dp
    
    total = sum(dp) % mod
    print(total)

if __name__ == '__main__':
    main()
```
