To solve this problem, we need to determine if we can transform string `S` into string `T` using at most one operation, where the operation can be an insertion, deletion, or replacement of a single character. 

### Approach
1. **Initial Check**: If `S` is already equal to `T`, no operations are needed, so the answer is "Yes".
2. **Length Comparison**:
   - **Equal Lengths**: If the lengths of `S` and `T` are the same, we check if there is exactly one mismatched character between them. If so, a single replacement operation suffices.
   - **S is Longer by One Character**: This implies a deletion operation is needed. We check if `T` can be obtained by deleting exactly one character from `S`. This involves verifying that after aligning the strings, the remaining characters in `S` match those in `T` when one character is skipped.
   - **S is Shorter by One Character**: This implies an insertion operation is needed. We check if `T` can be obtained by inserting one character into `S`. Similar to the deletion case, but in reverse.
3. **Final Check**: If none of the above conditions are met, the answer is "No".

### Solution Code
```python
def main():
    import sys
    data = sys.stdin.read().splitlines()
    K = int(data[0].strip())
    S = data[1].strip()
    T = data[2].strip()
    
    n = len(S)
    m = len(T)
    
    if S == T:
        print("Yes")
        return
        
    if n == m:
        count = 0
        for a, b in zip(S, T):
            if a != b:
                count += 1
        if count == 1:
            print("Yes")
        else:
            print("No")
        return
            
    elif n == m + 1:
        i = 0
        j = 0
        while i < n and j < m and S[i] == T[j]:
            i += 1
            j += 1
        if j == m and i < n and n - i == 1:
            print("Yes")
        elif i < n and j < m and S[i] != T[j]:
            if S[i+1:] == T[j:]:
                print("Yes")
            else:
                print("No")
        else:
            print("No")
        return
            
    elif n == m - 1:
        i = 0
        j = 0
        while i < n and j < m and S[i] == T[j]:
            i += 1
            j += 1
        if i == n and j < m and m - j == 1:
            print("Yes")
        elif i < n and j < m and S[i] != T[j]:
            if S[i:] == T[j+1:]:
                print("Yes")
            else:
                print("No")
        else:
            print("No")
        return
            
    else:
        print("No")
        return

if __name__ == '__main__':
    main()
```

3. **S is Longer by One Character**: The code checks if `T` can be formed by deleting one character from `S`. This is done by aligning the strings and verifying that skipping one character in `S` results in a match with `T`.
4. **S is Shorter by One Character**: Similarly, the code checks if `T` can be formed by inserting one character into `S` by aligning the strings and verifying that inserting one character into `S` results in a match with `T`.
