You are given a simple connected undirected graph with N vertices numbered 1 through N and M edges numbered 1 through M. Edge i connects vertices u_i and v_i, and has a label w_i.
Among all simple paths (paths that do not pass through the same vertex more than once) from vertex 1 to vertex N, find the minimum XOR of the labels of the edges on the path.

Notes on XOR

For non-negative integers A and B, their XOR A \oplus B is defined as follows:


- In the binary representation of A \oplus B, the digit in the place corresponding to 2^k \,(k \ge 0) is 1 if and only if exactly one of the digits in the same place of A and B is 1; otherwise, it is 0.


For example, 3 \oplus 5 = 6 (in binary: 011 \oplus 101 = 110).

In general, the XOR of k integers p_1, \dots, p_k is defined as (\cdots ((p_1 \oplus p_2) \oplus p_3) \oplus \cdots \oplus p_k). It can be proved that it does not depend on the order of p_1, \dots, p_k.

Input

The input is given from Standard Input in the following format:
N M
u_1 v_1 w_1
u_2 v_2 w_2
\vdots
u_M v_M w_M

Output

Print the answer.

Constraints


- 2 \leq N \leq 10
- N-1 \leq M \leq \frac{N(N-1)}{2}
- 1 \leq u_i < v_i \leq N
- 0 \leq w_i < 2^{60}
- The given graph is a simple connected undirected graph.
- All input values are integers.

The solution involves exploring all possible simple paths from the start vertex to the target vertex and computing the XOR of edge labels for each path, then selecting the minimum XOR value.

### Approach
1. **Graph Representation**: Represent the graph using an adjacency list where each node points to its neighbors along with the edge labels.
2. **Dynamic Programming with Bitmasking**: Use a dynamic programming approach where the state is defined by a bitmask representing the set of visited nodes and the current node. The DP state `dp[mask][node]` stores the minimum XOR value achievable to reach `node` with the set of visited nodes represented by `mask`.
3. **State Transitions**: For each state `(mask, node)`, update the state by considering all unvisited neighbors. The new state is `(mask | (1 << neighbor), neighbor)`, and the new XOR value is the current XOR value XORed with the edge label between `node` and `neighbor`.
4. **Initialization and Base Case**: Start from the initial node (vertex 1) with a bitmask indicating only that node is visited and an initial XOR value of 0.
5. **Result Extraction**: After processing all possible states, the answer is the minimum XOR value found among all states that include the target node (vertex N).

import sys

def main():
    data = sys.stdin.read().strip().split()
    if not data:
        return
    it = iter(data)
    n = int(next(it))
    m = int(next(it))
    graph = [[] for _ in range(n)]
    for _ in range(m):
        u = int(next(it)) - 1
        v = int(next(it)) - 1
        w = int(next(it))
        graph[u].append((v, w))
        graph[v].append((u, w))
    
    adj = [[10**18] * n for _ in range(n)]
    for i in range(n):
        for (v, w) in graph[i]:
            adj[i][v] = w
    
    total_masks = 1 << n
    INF = 10**18
    dp = [[INF] * n for _ in range(total_masks)]
    dp[1 << 0][0] = 0
    
    for mask in range(total_masks):
        for node in range(n):
            if not (mask & (1 << node)):
                continue
            prev_mask = mask ^ (1 << node)
            min_val = INF
            for x in range(n):
                if x == node:
                    continue
                if prev_mask & (1 << x):
                    if adj[x][node] != 10**18 and dp[prev_mask][x] != INF:
                        candidate = dp[prev_mask][x] ^ adj[x][node]
                        if candidate < min_val:
                            min_val = candidate
            if min_val != INF:
                dp[mask][node] = min_val
            if dp[mask][node] != INF:
                for neighbor in range(n):
                    if mask & (1 << neighbor):
                        continue
                    if adj[node][neighbor] != 10**18:
                        new_mask = mask | (1 << neighbor)
                        new_xor = dp[mask][node] ^ adj[node][neighbor]
                        if new_xor < dp[new_mask][neighbor]:
                            dp[new_mask][neighbor] = new_xor
    
    ans = INF
    for mask in range(total_masks):
        if dp[mask][n-1] < ans:
            ans = dp[mask][n-1]
    print(ans)

if __name__ == "__main__":
