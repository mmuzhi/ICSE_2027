=== PROMPT ===
You will be given a competitive programming problem.
There are n employees in a company, numbered from 0 to n - 1. Each employee i has worked for hours[i] hours in the company.
The company requires each employee to work for at least target hours.
You are given a 0-indexed array of non-negative integers hours of length n and a non-negative integer target.
Return the integer denoting the number of employees who worked at least target hours.
 
Example 1:

Input: hours = [0,1,2,3,4], target = 2
Output: 3
Explanation: The company wants each employee to work for at least 2 hours.
- Employee 0 worked for 0 hours and didn't meet the target.
- Employee 1 worked for 1 hours and didn't meet the target.
- Employee 2 worked for 2 hours and met the target.
- Employee 3 worked for 3 hours and met the target.
- Employee 4 worked for 4 hours and met the target.
There are 3 employees who met the target.

Example 2:

Input: hours = [5,1,4,2,2], target = 6
Output: 0
Explanation: The company wants each employee to work for at least 6 hours.
There are 0 employees who met the target.

 
Constraints:

1 <= n == hours.length <= 50
0 <= hours[i], target <= 10^5

=== ANSWER ===

To solve this problem, we need to count the number of employees who have worked at least the target number of hours. The solution involves iterating through each employee's hours and checking if their hours meet or exceed the target. 

### Approach
1. **Problem Analysis**: The problem requires us to count how many employees in a list have worked at least a specified number of hours. The solution involves a straightforward check for each employee's hours against the target.
2. **Intuition**: By iterating through each element in the hours array, we can compare each employee's hours with the target. If an employee's hours are greater than or equal to the target, we increment our count.
3. **Algorithm Selection**: The algorithm involves a single pass through the hours array, making it efficient with a time complexity of O(n), where n is the number of employees. This is optimal for the given constraints.
4. **Optimization**: No further optimization is needed beyond the linear scan since the problem size is small (up to 50 employees).

### Solution Code
```python
class Solution:
    def numberOfEmployeesWhoMetTarget(self, hours: List[int], target: int) -> int:
        count = 0
        for h in hours:
            if h >= target:
                count += 1
        return count
```

### Explanation
1. **Initialization**: We start by initializing a counter `count` to zero. This counter will keep track of the number of employees who meet or exceed the target hours.
2. **Iteration**: For each employee's hours in the `hours` array, we check if the employee's hours (`h`) are greater than or equal to the target.
3. **Counting**: If the condition is met, we increment the `count` by one.
4. **Return Result**: After processing all employees, the `count` variable holds the total number of employees who worked at least the target hours, which is then returned.

This approach efficiently checks each employee's hours in linear time, ensuring optimal performance for the given problem constraints.
