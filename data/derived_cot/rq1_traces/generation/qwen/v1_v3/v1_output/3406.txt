The solution involves dynamic programming to efficiently count valid sequences while adhering to the constraints.

### Approach

2. **Dynamic Programming Setup**: We use two 3D arrays, `dp0` and `dp1`, where:
   - `dp0[z][o][r]` represents the number of valid sequences with `z` zeros, `o` ones, ending with a zero, and a current run of `r` consecutive zeros.

4. **Recurrence Relations**:
   - For sequences ending with a zero (`dp0`), the last digit can either start a new run (run length 1) or continue a previous run (run length > 1).
