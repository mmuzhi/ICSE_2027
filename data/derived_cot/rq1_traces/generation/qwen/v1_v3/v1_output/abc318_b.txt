To solve this problem, we need to compute the total area covered by one or more rectangular sheets on a coordinate plane. Each sheet is defined by its bottom-left and top-right coordinates, and the sheets can overlap. The solution involves efficiently calculating the union of all these rectangles.

### Approach
1. **Grid Method**: The approach involves dividing the plane into unit squares (1x1) and marking each square that lies within at least one of the given rectangles. The total area is then the count of these marked squares.
2. **Efficient Marking**: For each rectangle, we iterate over all the unit squares it covers and mark them as part of the union. This is done using a 100x100 grid (since the coordinates range from 0 to 100), where each cell represents a unit square.
3. **Counting Marked Squares**: After processing all rectangles, we count all the marked cells in the grid, which gives the total area covered by the union of all rectangles.

### Solution Code
```python
N = int(input().strip())
grid = [[False] * 100 for _ in range(100)]

for _ in range(N):
    a, b, c, d = map(int, input().split())
    for x in range(a, b):
        for y in range(c, d):
            grid[x][y] = True

total = 0
for i in range(100):
    for j in range(100):
        if grid[i][j]:
            total += 1

print(total)
```

### Explanation
1. **Initialization**: We start by reading the number of rectangles, `N`, and initialize a 100x100 grid with all cells set to `False`.
2. **Processing Rectangles**: For each rectangle, we read its coordinates (A, B, C, D). We then iterate over each unit square within the rectangle's boundaries (from A to B-1 in the x-direction and C to D-1 in the y-direction) and mark these squares as `True` in the grid.
3. **Counting Union Area**: After processing all rectangles, we count all the `True` cells in the grid. Each `True` cell represents a unit square covered by at least one rectangle, and the total count gives the area of the union of all rectangles.
4. **Output**: The total count is printed as the result.

This approach efficiently handles the problem by leveraging the grid method, which simplifies the calculation of the union area by focusing on individual unit squares. The solution is optimal given the constraints, with a time complexity of O(N * (B - A) * (D - C)), which is feasible since N is at most 100 and the grid dimensions are fixed at 100x100.
