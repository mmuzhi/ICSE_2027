To solve this problem, we need to find the maximum possible total weight of edges chosen from a complete undirected graph such that no two edges share a common vertex. This is essentially the problem of finding a maximum matching in a complete graph, where the matching can be any subset of edges without common vertices.

### Approach
1. **Problem Analysis**: The problem requires selecting edges from a complete graph such that no two edges share a vertex, and the sum of their weights is maximized. The graph is complete, meaning every pair of distinct vertices is connected by a unique edge. The challenge is to efficiently explore all possible matchings without exceeding computational limits, especially since the number of vertices (N) can be up to 16.

2. **Intuition**: Given the constraints (N ≤ 16), a brute-force approach is infeasible. Instead, we use dynamic programming (DP) with bitmasking to efficiently explore all possible matchings. Each bitmask represents a subset of vertices that have been matched so far. The DP state tracks the maximum weight achievable for each subset of matched vertices.

3. **Algorithm Selection**: We use a bitmask DP approach where:
   - **State Representation**: A bitmask `mask` represents the set of vertices already included in the matching.
   - **State Transition**: For each state `mask`, we identify the smallest unmatched vertex and pair it with every other unmatched vertex, updating the DP state for the new bitmask that includes both vertices.
   - **Optimization**: By iterating over all possible masks in increasing order, we ensure that each state is built upon previously computed states, leveraging the optimal substructure property of dynamic programming.

4. **Complexity Analysis**: The algorithm's time complexity is O(2^N * N^2), where N is the number of vertices. For N=16, this results in approximately 7.8 million operations, which is feasible given the constraints.

def main():
    import sys
    data = sys.stdin.read().strip().split()
    if not data:

    for i in range(n-1):
        count = n - i - 1
        for j in range(i+1, n):
            D[i][j] = int(data[index])
            D[j][i] = int(data[index])
            index += 1

    dp = [-10**18] * (1 << n)

        unmatched = []
