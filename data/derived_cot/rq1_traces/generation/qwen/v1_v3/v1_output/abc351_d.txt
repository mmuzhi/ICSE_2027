Output

Constraints

- H and W are integers.

To solve this problem, we need to determine the maximum degree of freedom for cells without magnets in a grid. The degree of freedom for a cell is the number of cells Takahashi can reach from that cell by moving according to the given rules. The movement rules specify that Takahashi can move from a cell if none of its adjacent cells contain a magnet; otherwise, he cannot move at all.

### Approach
1. **Problem Analysis**: The grid consists of cells that may contain magnets ('#') or be empty ('.'). For each empty cell, if any adjacent cell contains a magnet, the degree of freedom is 1 (only the cell itself). Otherwise, Takahashi can move to any adjacent cell (including those with magnets), and from there, he can continue moving if the current cell allows it (i.e., no adjacent magnets).

2. **Key Insight**: The problem can be modeled as a directed graph where nodes are grid cells, and edges exist from an active empty cell (no adjacent magnets) to its four adjacent cells. The degree of freedom for an active empty cell is the size of the reachable set from that cell in this graph.

3. **Algorithm Selection**: For each active empty cell, perform a BFS to explore all reachable cells. The BFS starts from the cell and explores its neighbors, continuing from active empty cells only. The size of the reachable set from each active empty cell is recorded, and the maximum value among all empty cells is the answer.

4. **Optimization**: If there are no magnets in the grid, all empty cells are active, and the reachable set for any cell is the entire grid, so the answer is simply the grid's total cells.

from collections import deque
import sys

def main():
    data = sys.stdin.read().splitlines()
    if not data:
        return
    H, W = map(int, data[0].split())
    grid = []
    for i in range(1, 1+H):

        has_magnet = False
        for i in range(H):
            if '#' in grid[i]:
                has_magnet = True
                break
        if not has_magnet:
            print(H * W)
            return

    for i in range(H):
        for j in range(W):
            if grid[i][j] == '.':
                has_magnet = False
                for dx, dy in [(0,1), (0,-1), (1,0), (-1,0)]:
                    ni, nj = i+dx, j+dy
                    if 0 <= ni < H and 0 <= nj < W:
                        if grid[ni][nj] == '#':

                active_empty[i][j] = not has_magnet

    for i in range(H):

            if grid[i][j] == '.' and active_empty[i][j]:

4. **BFS for Active Cells**: For each active empty cell, perform a BFS to explore all reachable cells. The BFS explores neighbors of active cells only, marking visited cells to avoid reprocessing.
5. **Tracking Maximum Degree**: The size of the reachable set for each active cell is recorded, and the maximum value among all empty cells is printed as the result.

This approach efficiently computes the maximum degree of freedom by leveraging BFS for each active cell, ensuring optimal exploration of reachable cells while adhering to movement constraints.
