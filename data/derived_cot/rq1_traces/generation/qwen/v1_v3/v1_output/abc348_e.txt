1. **Tree Representation**: Represent the tree using an adjacency list for efficient traversal.
2. **BFS for Depth and Parent Calculation**: Perform a BFS starting from an arbitrary root (node 0) to compute the depth of each node and its parent in the tree.
3. **Subtree Sum Calculation**: Using a post-order traversal, compute the sum of weights \( C_i \) for each subtree rooted at every node. This helps in efficiently calculating the contribution of each node to the function \( f(v) \).
4. **Initial Root Value Calculation**: Compute the initial value of \( f(v) \) for the root node (node 0) by summing \( C_i \times \text{depth}[i] \) for all nodes.
5. **Rerooting Technique**: Using the rerooting technique, compute \( f(v) \) for all other nodes by leveraging the parent's value and the subtree sums. The key insight here is that moving from a parent to a child node adjusts the total distance based on the weights of nodes in the child's subtree.
6. **Result Extraction**: The minimum value of \( f(v) \) across all nodes is the answer.

### Solution Code
```python
import sys
from collections import deque

def main():
    data = sys.stdin.read().split()
    if not data:
        return
    n = int(data[0])
    graph = [[] for _ in range(n)]
    index = 1
    for _ in range(n-1):
        u = int(data[index])
        v = int(data[index+1])
        index += 2
        u -= 1
        v -= 1
        graph[u].append(v)
        graph[v].append(u)
    
    c_list = list(map(int, data[index:index+n]))
    index += n

    total = sum(c_list)

    depth = [-1] * n
    parent = [-1] * n
    q = deque([0])
    depth[0] = 0
    parent[0] = -1
    while q:
        u = q.popleft()
        for v in graph[u]:
            if depth[v] == -1:
                depth[v] = depth[u] + 1
                parent[v] = u
                q.append(v)
                
    children = [[] for _ in range(n)]
    for i in range(1, n):
        children[parent[i]].append(i)
        
    stack = [0]
    order = []
    while stack:
        u = stack.pop()
        order.append(u)
        for v in children[u]:
            stack.append(v)
            
    S = [0] * n
    for node in reversed(order):
        S[node] = c_list[node]
        for child in children[node]:
            S[node] += S[child]
            
    d0 = 0
    for i in range(n):
        d0 += c_list[i] * depth[i]
        
    d_vals = [0] * n
    d_vals[0] = d0
    q = deque([0])
    while q:
        u = q.popleft()
        for v in graph[u]:
            if v == parent[u]:
                continue
            d_vals[v] = d_vals[u] + total - 2 * S[v]
            q.append(v)
            
    ans = min(d_vals)
    print(ans)

if __name__ == '__main__':
    main()
```

### Explanation
1. **Tree Representation**: The tree is built using an adjacency list from the input edges.
2. **BFS for Depth and Parent**: Starting from node 0, BFS computes the depth of each node and its parent, which helps in structuring the tree for further processing.
3. **Subtree Sum Calculation**: A post-order traversal (using a stack) processes nodes from leaves to root, accumulating the sum of weights \( C_i \) in each subtree. This sum is crucial for adjusting the function values during rerooting.
4. **Initial Root Value**: The initial value \( f(0) \) is computed by summing \( C_i \times \text{depth}[i] \) for all nodes, leveraging the depth values obtained from BFS.
5. **Rerooting**: For each node, the function value is derived from its parent's value, adjusted by the total weight of the tree and the subtree sum of the node. This adjustment efficiently computes the function value for all nodes without recalculating from scratch.
6. **Result Extraction**: The minimum value among all computed function values is printed as the result, providing the solution to the problem. This approach efficiently handles the constraints by leveraging tree properties and dynamic programming techniques.
