=== ANSWER ===

To solve this problem, we need to determine if Alice can win a game by selecting either all single-digit numbers or all double-digit numbers from a given array, such that the sum of her selected numbers is strictly greater than the sum of the remaining numbers (Bob's numbers).

### Approach
1. **Problem Analysis**: The problem requires checking two scenarios:
   - **Scenario 1**: Alice selects all single-digit numbers (1-9). The sum of these numbers is compared against the sum of the remaining double-digit numbers (10-99). If the sum of single-digit numbers is greater, Alice wins.
   - **Scenario 2**: Alice selects all double-digit numbers (10-99). The sum of these numbers is compared against the sum of the remaining single-digit numbers. If the sum of double-digit numbers is greater, Alice wins.
2. **Intuition**: The solution involves calculating the total sums of single-digit and double-digit numbers in the array. By comparing these sums in both scenarios, we can determine if Alice can win the game.
3. **Algorithm Selection**: The algorithm iterates through the array once, categorizing each number into single-digit or double-digit, and accumulating their sums. After processing all numbers, it checks the two conditions mentioned above.
4. **Complexity Analysis**: The algorithm runs in O(n) time, where n is the number of elements in the array, as it processes each element exactly once. The space complexity is O(1) since we only use two additional variables to store the sums.

### Solution Code
```python
class Solution:
    def canAliceWin(self, nums: List[int]) -> bool:
        s1 = 0  # sum for single-digit numbers
        s2 = 0  # sum for double-digit numbers
        for num in nums:
            if num < 10:
                s1 += num
            else:
                s2 += num
        return s1 > s2 or s2 > s1
```

### Explanation
1. **Initialization**: Two variables, `s1` and `s2`, are initialized to zero. These will hold the sums of single-digit and double-digit numbers, respectively.
2. **Iteration and Categorization**: The code iterates over each number in the input array. If a number is single-digit (less than 10), it is added to `s1`. Otherwise, it is added to `s2`.
3. **Condition Check**: After processing all numbers, the code checks if either `s1` is greater than `s2` (Alice wins by choosing single-digit numbers) or `s2` is greater than `s1` (Alice wins by choosing double-digit numbers). If either condition is true, the function returns `True`; otherwise, it returns `False`.

This approach efficiently checks both possible winning conditions for Alice by leveraging simple iteration and arithmetic comparisons, ensuring optimal performance and correctness.
