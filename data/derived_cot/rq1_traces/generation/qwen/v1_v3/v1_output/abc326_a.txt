To solve this problem, we need to determine whether Takahashi uses the stairs or the elevator to move from floor X to floor Y in a building with 100 floors. The stairs are used for moving up two floors or less or moving down three floors or less. Otherwise, the elevator is used.

### Approach
1. **Problem Analysis**: The key observation here is that the entire journey from floor X to floor Y must be evaluated based on the net change in floors (d = Y - X). 
2. **Intuition**: If the net change (d) is positive and ≤ 2, Takahashi uses the stairs (up to 2 floors). If the net change is negative and ≥ -3, he uses the stairs (down to 3 floors). For any other net change, the elevator is used.
3. **Algorithm Selection**: The solution involves a straightforward conditional check on the value of d. The algorithm checks if d falls within the specified ranges for stairs and outputs "Yes" if it does; otherwise, it outputs "No".

### Solution Code
```python
X, Y = map(int, input().split())
d = Y - X

if d > 0 and d <= 2:
    print("Yes")
elif d < 0 and d >= -3:
    print("Yes")
else:
    print("No")
```

### Explanation
1. **Input Handling**: The code reads two integers, X and Y, representing the starting and destination floors.
2. **Net Change Calculation**: The variable `d` is computed as the difference between Y and X (d = Y - X).
3. **Condition Checks**: 
   - If `d` is positive and ≤ 2, it means Takahashi is moving up 2 floors or less, so the stairs are used ("Yes").
   - If `d` is negative and ≥ -3, it means Takahashi is moving down 3 floors or less, so the stairs are used ("Yes").
   - For any other value of `d`, the elevator is used ("No").
4. **Output**: Based on the conditions, the code prints "Yes" or "No" indicating whether the stairs or elevator is used for the move.

This approach efficiently checks the net change in floors and determines the appropriate mode of movement according to the given constraints.
