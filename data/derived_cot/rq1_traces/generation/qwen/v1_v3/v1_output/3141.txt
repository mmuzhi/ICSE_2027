To solve this problem, we need to find the shortest contiguous subarray in an infinitely repeated array that sums to a given target. The solution involves leveraging the periodic nature of the array to efficiently search for the subarray without explicitly constructing the infinite array.

### Approach
1. **Problem Analysis**: The problem requires finding a contiguous subarray in an infinitely repeated array that sums to a target. The key observation is that any such subarray must either be entirely within one or two consecutive copies of the original array. This allows us to limit our search to two copies of the original array, significantly reducing the problem size.

2. **Intuition**: If the target sum is greater than the sum of the original array, we can break it down into full cycles (each summing to the total of the original array) and a remainder. The minimal subarray for the remainder is found within two copies of the original array. If the remainder is zero, the solution is simply the number of full cycles multiplied by the length of the original array.

3. **Algorithm Selection**: 
   - **Sliding Window (Two-pointer Technique)**: Used to find the minimal length subarray summing to the remainder (if target > total) or the target (if target ≤ total) within two copies of the original array. This technique efficiently searches for the subarray by expanding and contracting the window while maintaining the sum constraint.

4. **Complexity Analysis**: The algorithm processes two copies of the array (O(n) space) and uses a linear scan with two pointers (O(n) time), making the overall complexity O(n), which is efficient for the given constraints.

class Solution:
    def minSizeSubarray(self, nums: List[int], target: int) -> int:
        total = sum(nums)
        n = len(nums)
        
        if target > total:
            k = target // total
            r = target % total
            if r == 0:
                return k * n
            else:
                arr = nums * 2
                left = 0
                s = 0
                min_len = float('inf')
                for right in range(len(arr)):
                    s += arr[right]
                    while s > r and left <= right:
                        s -= arr[left]
                        left += 1
                    if s == r:
                        min_len = min(min_len, right - left + 1)
                if min_len == float('inf'):
                    return -1
                else:
                    return min_len + k * n
        else:
            arr = nums * 2
            left = 0
            s = 0
            min_len = float('inf')
            for right in range(len(arr)):
                s += arr[right]
                while s > target and left <= right:
                    s -= arr[left]
                    left += 1
                if s == target:
                    min_len = min(min_len, right - left + 1)
            if min_len == float('inf'):
                return -1
            else:
                return min_len

### Explanation
1. **Initial Setup**: Calculate the total sum of the original array and its length.
2. **Handling Large Targets**: If the target exceeds the total sum of the array, decompose it into full cycles (each summing to the total) and a remainder. If the remainder is zero, the solution is the product of the number of full cycles and the array length. Otherwise, find the minimal subarray summing to the remainder within two copies of the array and add the full cycles' length to this minimal length.
3. **Sliding Window Technique**: For targets not exceeding the array's total sum, use a sliding window approach on two copies of the array to find the shortest contiguous subarray summing exactly to the target. The window expands by moving the right pointer and contracts from the left when the sum exceeds the target.
4. **Result Determination**: If no valid subarray is found, return -1; otherwise, return the length of the minimal subarray.

This approach efficiently narrows down the problem size by leveraging the array's periodic nature and uses a linear scan with a sliding window to find the solution, ensuring optimal performance.
