To solve this problem, we need to find the position of the unique character in a string where all other characters are the same. The string consists of two distinct lowercase English letters, and exactly one character differs from the rest. 

### Approach
1. **Initial Check**: The first two characters of the string are checked. If they are the same, the majority character is this character. We then scan the string from the beginning to find the first character that differs from the majority character. The position of this character is our answer.
2. **Different First Two Characters**: If the first two characters are different, the third character is checked. If the third character matches the first character, the second character is the unique one (position 2). Otherwise, the first character is unique (position 1).

This approach efficiently narrows down the unique character's position by leveraging the problem constraints, ensuring optimal performance with minimal comparisons.

### Solution Code
```python
S = input().strip()
if S[0] == S[1]:
    for i in range(len(S)):
        if S[i] != S[0]:
            print(i+1)
            break
else:
    if S[0] == S[2]:
        print(2)
    else:
        print(1)
```

### Explanation
1. **Initial Check**: The code first checks if the first two characters are the same. If they are, it scans the string from the start to find the first character that differs from the first character. The position of this character (1-indexed) is printed.
2. **Different First Two Characters**: If the first two characters differ, the code checks the third character. If the third character matches the first character, the second character is unique (position 2). If not, the first character is unique (position 1).

This method efficiently identifies the unique character's position by leveraging the problem's constraints, ensuring correctness with minimal computational overhead. The solution handles all edge cases within the given problem constraints.
