=== ANSWER ===

To solve this problem, we need to determine the minimum cost for Takahashi to buy one pen, given that he dislikes a specific color and cannot purchase pens of that color. The solution involves reading the costs of the three available pens and the disliked color, then selecting the cheapest available pen from the two colors he can buy.

### Approach
1. **Read Input Values**: The first line of input contains three integers representing the costs of red, green, and blue pens. The second line contains a string indicating the color Takahashi dislikes.
2. **Determine Allowed Colors**: Based on the disliked color, exclude that color from consideration and evaluate the minimum cost between the remaining two colors.
3. **Output the Result**: The minimum cost among the allowed colors is the answer.

### Solution Code
```python
def main():
    data = input().split()
    R = int(data[0])
    G = int(data[1])
    B = int(data[2])
    C = input().strip()
    
    if C == "Red":
        print(min(G, B))
    elif C == "Green":
        print(min(R, B))
    else:
        print(min(R, G))

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The first line of input is split into three integers (R, G, B) representing the costs of the red, green, and blue pens, respectively. The second line is read as a string (C) indicating the disliked color.
2. **Condition Handling**: 
   - If the disliked color is "Red", the solution calculates the minimum cost between green and blue pens.
   - If the disliked color is "Green", the solution calculates the minimum cost between red and blue pens.
   - If the disliked color is "Blue", the solution calculates the minimum cost between red and green pens.
3. **Output**: The result is printed, which is the minimum cost of the allowed pens.

This approach efficiently narrows down the problem to selecting the cheapest available pen from the two colors not disliked by Takahashi, ensuring optimal performance with constant time complexity O(1).
