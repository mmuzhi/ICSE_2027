=== ANSWER ===

To solve this problem, we need to calculate the total number of steps Takahashi walked each week based on the given daily steps over N weeks. The input consists of N followed by a list of 7*N integers, where each week's steps are recorded consecutively.

### Approach
1. **Read Input:** The first line of input is the number of weeks, N. The second line contains the steps for each day, totaling 7*N integers.
2. **Process Data:** For each week, sum the steps from the corresponding 7 days. The first week's steps are the first 7 days, the next week's steps are the next 7 days, and so on.
3. **Output Results:** Print the weekly step totals separated by spaces.

The solution involves straightforward list processing. We iterate over the list of daily steps in chunks of 7, sum each chunk, and collect these sums into a result list. The result list is then printed as space-separated values.

### Solution Code
```python
def main():
    N = int(input().strip())
    data = list(map(int, input().split()))
    
    result = []
    for i in range(N):
        start_index = i * 7
        week_steps = data[start_index:start_index+7]
        total_steps = sum(week_steps)
        result.append(str(total_steps))
    
    print(" ".join(result))

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input:** The first line reads the number of weeks, N. The second line reads the list of daily steps as integers.
2. **Processing Data:** The loop runs N times. For each iteration, it calculates the start index of the current week's data (i*7), then extracts the next 7 elements (a week's worth of steps). The sum of these elements is computed and stored as a string in the result list.
3. **Output:** After processing all weeks, the result list is joined into a single string with spaces separating the weekly step counts, which is then printed.

This approach efficiently processes the input in linear time, making it optimal for the given constraints. The solution is simple and leverages basic list operations to achieve the desired result.
