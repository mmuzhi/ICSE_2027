The solution involves dynamic programming to efficiently count the number of favorable outcomes without enumerating all possible outcomes, which would be computationally infeasible for large dice ranges.

### Approach
1. **Dynamic Programming (DP) Setup**: We use a bitmask DP approach where each state represents the set of achievable subset sums (from 0 to 10) using the dice outcomes processed so far. The bitmask is a 11-bit number where each bit indicates whether a particular sum (0 to 10) can be formed.
2. **State Transitions**: For each die, we consider all possible outcomes (from 1 to the die's maximum value, but limited to values up to 10 for efficiency). For each outcome, we update the current state by shifting the existing sums by the outcome value and combining them with the original sums.
3. **Modular Arithmetic**: Since the result needs to be reported modulo 998244353, we handle large numbers using modular arithmetic. The final probability is computed as the ratio of favorable outcomes to total outcomes, using modular inversion for division.
4. **Efficiency**: By limiting the considered outcomes to a maximum of 10 per die (since sums beyond 10 are irrelevant), the solution efficiently handles large dice ranges (up to 10^6) with a manageable state space (2048 possible masks for sums 0-10).

### Solution Code

MOD = 998244353

def main():
    import sys
    data = sys.stdin.read().split()
    if not data:
        return
    n = int(data[0])
    A = list(map(int, data[1:1+n]))
    
    dp = [0] * 2048
    dp[1] = 1
    
    D_mod = 1
    for a in A:
        D_mod = (D_mod * a) % MOD
        
    for i in range(n):
        new_dp = [0] * 2048
        A_i = A[i]
        low = min(A_i, 10)
        for mask in range(2048):
            count = dp[mask]
            if count == 0:

            count_dict = {}

                shifted = (mask << x) & 0x7FF
                new_mask_val = mask | shifted
                count_dict[new_mask_val] = count_dict.get(new_mask_val, 0) + 1
            for new_mask_val, cnt in count_dict.items():
                new_dp[new_mask_val] = (new_dp[new_mask_val] + count * cnt) % MOD
            if A_i > 10:
                new_dp[mask] = (new_dp[mask] + count * (A_i - low)) % MOD
        dp = new_dp
