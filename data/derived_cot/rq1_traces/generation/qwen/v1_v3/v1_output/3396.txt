=== ANSWER ===

To solve this problem, we need to determine if a given word meets specific validity criteria. The word must be at least 3 characters long, contain only digits or English letters, include at least one vowel, and include at least one consonant. 

### Approach
1. **Check Length**: The word must have a minimum of 3 characters. If not, it's immediately invalid.
2. **Check Character Validity**: Each character in the word must be either a digit (0-9) or an English letter (uppercase or lowercase). Any other character makes the word invalid.
3. **Check for Vowels and Consonants**: The word must contain at least one vowel (a, e, i, o, u, or their uppercase counterparts) and at least one consonant (an English letter that is not a vowel). 

The solution involves iterating through each character in the word to verify these conditions. The presence of vowels and consonants is tracked during this iteration to ensure all criteria are met.

### Solution Code
```python
class Solution:
    def isValid(self, word: str) -> bool:
        if len(word) < 3:
            return False
        
        vowels = "aeiouAEIOU"
        has_vowel = False
        has_consonant = False
        
        for char in word:
            if not (char.isdigit() or char.isalpha()):
                return False
            if char in vowels:
                has_vowel = True
            if char.isalpha() and char not in vowels:
                has_consonant = True
        
        return has_vowel and has_consonant
```

### Explanation
1. **Length Check**: The function first checks if the word length is less than 3. If so, it returns `False` since the word is too short.
2. **Character Validation**: For each character in the word, the function checks if it is a digit or an English letter. If any character fails this check, the function returns `False`.
3. **Vowel and Consonant Check**: During the iteration, the function tracks whether any vowel or consonant has been encountered. A vowel is any character in "aeiouAEIOU", and a consonant is any other English letter.
4. **Final Check**: After processing all characters, the function returns `True` only if both a vowel and a consonant were found; otherwise, it returns `False`.

This approach efficiently verifies all the given conditions in a single pass through the word, ensuring optimal performance with a time complexity of O(n), where n is the length of the word.
