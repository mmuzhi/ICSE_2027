=== ANSWER ===

To solve this problem, we need to maximize the amount of currency in the last country (country N) by performing a series of operations that transfer currency from left to right across the countries. Each operation on a country i transfers S_i units of currency from country i to T_i units in country i+1. The challenge is to determine the optimal number of operations for each edge to maximize the final amount in country N.

### Approach
1. **Problem Analysis**: The problem involves a sequence of countries where each country can perform operations to transfer currency to the next country. The operations are unidirectional and can be performed any number of times, provided the required currency is available in the current country. The goal is to maximize the currency in the last country by strategically performing these operations.

2. **Key Insight**: The optimal strategy involves processing each country from left to right, using as many operations as possible on each edge to maximize the currency transferred to the next country. The number of operations on each edge is determined by the available currency in the current country, which includes the initial amount plus any currency received from previous operations.

3. **Algorithm Selection**: We use a greedy approach where we process each edge in sequence, calculating the maximum number of operations possible based on the available currency (initial plus incoming transfers). The result for the last country is then computed by adding the initial amount to the total currency transferred from the last operation.

4. **Complexity Analysis**: The algorithm processes each edge exactly once, making it linear in time complexity, O(N), where N is the number of countries. This is efficient given the constraints (N ≤ 2 × 10^5).

### Solution Code
```python
import sys

def main():
    data = sys.stdin.read().split()
    n = int(data[0])
    A = list(map(int, data[1:1+n]))
    edges = []
    idx = 1 + n
    for i in range(n-1):
        s = int(data[idx])
        t = int(data[idx+1])
        idx += 2
        edges.append((s, t))
    
    k = [0] * (n-1)
    if n >= 2:
        k[0] = A[0] // edges[0][0]
        for i in range(1, n-1):
            available = A[i] + k[i-1] * edges[i-1][1]
            k[i] = available // edges[i][0]
        ans = A[n-1] + k[n-2] * edges[n-2][1]
        print(ans)

if __name__ == '__main__':
    main()
```
