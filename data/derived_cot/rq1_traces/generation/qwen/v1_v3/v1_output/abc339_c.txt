To solve this problem, we need to determine the minimum possible current number of passengers on a bus that adheres to the given constraints. The bus starts with a non-negative number of passengers and stops at N points, where at each stop, the number of passengers changes by a given value (which can be positive or negative). The key is to ensure that after each stop, the number of passengers remains non-negative throughout the journey.

### Approach
1. **Problem Analysis**: The bus starts with an initial number of passengers, say `x` (which must be non-negative). At each of the N stops, the number of passengers changes by the given value `A_i`. The challenge is to find the smallest `x` such that after each stop, the number of passengers never drops below zero. The final number of passengers will be `x` plus the sum of all changes (`A_1 + A_2 + ... + A_N`).

2. **Key Insight**: The minimal initial number of passengers `x` must be large enough to counteract the most negative cumulative change encountered during the journey. Specifically, `x` should be the maximum of 0 and the negative of the minimum cumulative sum of the changes (including the initial state).

3. **Algorithm Selection**: 
   - Compute the cumulative sum of the changes (`A_i`) as we iterate through the list.
   - Track the minimum cumulative sum encountered during this iteration.
   - The minimal initial passengers `x` is `max(0, -min_cumulative_sum)`.
   - The final number of passengers is `x` plus the total sum of all changes.

4. **Complexity Analysis**: The algorithm involves a single pass through the list of changes, making it linear in time complexity, O(N), which is efficient given the constraints (N ≤ 2 × 10^5).

### Solution Code
```python
n = int(input().strip())
A = list(map(int, input().split()))

total = 0
current = 0
min_val = 0

for a in A:
    total += a
    current += a
    if current < min_val:
        min_val = current

x_min = max(0, -min_val)
print(x_min + total)
```

### Explanation
1. **Reading Input**: The first line reads the number of stops `N`, and the second line reads the list of changes `A` at each stop.
2. **Tracking Cumulative Sum**: We initialize `total` to accumulate the sum of all changes, `current` to keep track of the cumulative sum at each step, and `min_val` to record the minimum cumulative sum encountered.
3. **Determining Minimum Initial Passengers**: After processing all stops, `min_val` holds the most negative cumulative sum. The minimal initial passengers `x_min` is set to the maximum of 0 and the negative of `min_val` to ensure all intermediate passenger counts remain non-negative.
4. **Calculating Final Passengers**: The final number of passengers is the sum of `x_min` and the total sum of all changes (`total`), which is printed as the result.

This approach efficiently ensures that the bus's passenger count never drops below zero during its journey, providing the minimal possible final passenger count.
