To solve this problem, we need to generate a string of length \(N+1\) where each character at position \(i\) is determined based on specific conditions involving divisors of \(N\). The conditions require checking if \(i\) is a multiple of \(N/j\) for each divisor \(j\) of \(N\) between 1 and 9. The smallest such \(j\) that satisfies the condition determines the character at position \(i\); if no such \(j\) exists, the character is '-'.

### Approach
1. **Identify Divisors**: First, we find all divisors of \(N\) that lie between 1 and 9. These divisors are stored in a list.
2. **Generate the String**: For each position \(i\) from 0 to \(N\), we check each divisor \(j\) in the list (in increasing order) to see if \(i\) is divisible by \(N/j\). The first \(j\) that satisfies this condition is used to determine the character at position \(i\) (the digit representing \(j\)). If no such \(j\) is found, the character is '-'.
3. **Output the Result**: The characters collected for each position \(i\) are concatenated into a string and printed.

### Solution Code
```python
n = int(input().strip())
divisors = [j for j in range(1, 10) if n % j == 0]

s = []
for i in range(n + 1):
    for d in divisors:
        if i % (n // d) == 0:
            s.append(str(d))
            break
    else:
        s.append('-')

print(''.join(s))
```

### Explanation
1. **Reading Input**: The input integer \(N\) is read from standard input.
2. **Finding Divisors**: We generate a list of all divisors of \(N\) that are between 1 and 9. This is done by iterating from 1 to 9 and checking divisibility.
3. **Building the String**: For each index \(i\) from 0 to \(N\), we check each divisor \(d\) in the list. If \(i\) is divisible by \(N/d\), we append the digit \(d\) to the result string and break out of the loop. If no divisor satisfies the condition, we append '-'.
4. **Output**: The constructed string is printed, which combines all characters determined for each position \(i\).

This approach efficiently checks the required conditions for each position in the string, ensuring the correct character is placed based on the smallest valid divisor or '-' if no divisor meets the criteria. The solution handles all edge cases, including when \(N\) is 1 or other values within the given constraints.
