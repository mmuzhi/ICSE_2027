N M D

To solve this problem, we need to find two gifts, one for Aoki and one for Snuke, such that the absolute difference in their values is at most D. The goal is to maximize the sum of their values if such a pair exists; otherwise, return -1.

### Approach
1. **Sorting**: First, we sort both arrays of gift values for Aoki and Snuke. Sorting helps in efficiently searching for valid pairs using a two-pointer technique.
2. **Two-pointer Technique**: We use two pointers to traverse the sorted array of Snuke's gifts (B) while iterating through Aoki's sorted gifts (A). For each value in A, we determine the range of values in B that satisfy the condition |a - b| ≤ D. This translates to finding values in B that lie within [a - D, a + D].
3. **Efficient Range Search**: For each element in A, we move the pointers in B to find the start (j1) and end (j2) of the valid range. The start pointer (j1) is adjusted to the first element in B that is ≥ (a - D), and the end pointer (j2) is adjusted to the first element in B that is > (a + D). The valid elements in B are from j1 to j2-1.
4. **Maximizing the Sum**: For each valid range in B, the maximum value in the range (which is B[j2-1] due to the sorted order) is used to compute the maximum possible sum (a + B[j2-1]). We keep track of the maximum sum encountered during the iteration.

### Solution Code
```python
import sys

def main():
    data = sys.stdin.read().split()
    n = int(data[0])
    m = int(data[1])
    D = int(data[2])
    A = list(map(int, data[3:3+n]))
    B = list(map(int, data[3+n:3+n+m]))
    
    A.sort()
    B.sort()
    
    j1 = 0
    j2 = 0
    ans = -1
    
    for a in A:
        while j1 < m and B[j1] < a - D:
            j1 += 1
        while j2 < m and B[j2] <= a + D:
            j2 += 1
            
        if j1 < j2:
            candidate = a + B[j2-1]
            if candidate > ans:
                ans = candidate
                
    print(ans if ans != -1 else -1)

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The input is read and parsed into the number of gifts for Aoki (n), Snuke (m), the maximum allowed difference (D), and the respective lists of gift values (A and B).
2. **Sorting**: Both lists A and B are sorted to facilitate the two-pointer approach.
3. **Two-pointer Initialization**: Two pointers, j1 and j2, are initialized to 0. These pointers will traverse the sorted list B to find valid ranges for each element in A.
4. **Iterating Through A**: For each element `a` in A:
   - **Adjust j1**: Move j1 to the first element in B that is ≥ (a - D). This ensures we start from the lower bound of the valid range.
   - **Adjust j2**: Move j2 to the first element in B that is > (a + D). This marks the end of the valid range.
   - **Check Valid Range**: If there exists any element in B between j1 and j2-1 (i.e., j1 < j2), the maximum value in this range (B[j2-1]) is used to compute the sum (a + B[j2-1]). The maximum sum encountered is stored in `ans`.
5. **Output Result**: After processing all elements in A, if a valid pair was found, the maximum sum is printed; otherwise, -1 is printed.

This approach efficiently narrows down the valid pairs using sorting and two-pointer traversal, ensuring optimal performance even for large input sizes.
