The success rate for each person is defined as the ratio of heads to total tosses (A_i / (A_i + B_i)). If two people have the same success rate, they should be sorted in ascending order of their assigned numbers.

### Approach
1. **Problem Analysis**: The challenge is to compare two fractions (success rates) without floating-point precision issues. Direct comparison using floating-point division can lead to inaccuracies due to the limitations of floating-point arithmetic. Instead, we use cross-multiplication to compare fractions exactly.
2. **Key Insight**: To compare two fractions \( \frac{A_i}{A_i + B_i} \) and \( \frac{A_j}{A_j + B_j} \), we can compare \( A_i \times (A_j + B_j) \) and \( A_j \times (A_i + B_i) \). This avoids division and ensures exact comparison.
3. **Algorithm Selection**: We use a custom comparator function that performs cross-multiplication to compare the success rates of two people. The comparator returns -1 if the first person has a higher success rate, 1 if the second person has a higher success rate, and 0 if their success rates are equal. In case of a tie, the comparator compares their assigned numbers to determine the order.
4. **Complexity Analysis**: The solution involves reading N inputs, storing them, and sorting them using a custom comparator. The sorting step has a time complexity of \( O(N \log N) \), where each comparison in the sort takes \( O(1) \) time (since it involves simple arithmetic operations). This is efficient given the constraints (N ≤ 2×10^5).

### Solution Code
```python
import sys
from functools import cmp_to_key

def main():
    data = sys.stdin.read().split()
    if not data:
        return
    n = int(data[0])
    people = []
    index = 1
    for i in range(n):
        a = int(data[1 + 2*i])
        b = int(data[1 + 2*i + 1])
        people.append((index, a, b))
        index += 1

    def cmp_func(p1, p2):
        a1, b1 = p1[1], p1[2]
        a2, b2 = p2[1], p2[2]
        product1 = a1 * (a2 + b2)
        product2 = a2 * (a1 + b1)
        if product1 > product2:
            return -1
        elif product1 < product2:
            return 1
        else:
            if p1[0] < p2[0]:
                return -1
            elif p1[0] > p2[0]:
                return 1
            else:
                return 0

    sorted_people = sorted(people, key=cmp_to_key(cmp_func))
    result = [str(p[0]) for p in sorted_people]
    print(" ".join(result))

if __name__ == "__main__":
    main()
```
