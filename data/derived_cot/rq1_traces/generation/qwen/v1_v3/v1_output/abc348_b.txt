=== ANSWER ===

To solve this problem, we need to determine for each point on a 2D plane the farthest point from it, considering Euclidean distance. If multiple points are equally far, we select the one with the smallest ID. 

### Approach
1. **Problem Analysis**: The problem requires calculating the Euclidean distance between each point and every other point. The challenge is to efficiently find the farthest point for each given point, handling ties by selecting the smallest ID.
2. **Intuition**: For each point, compute the squared Euclidean distance to all other points. Since the square root function is monotonically increasing, comparing squared distances is equivalent to comparing actual Euclidean distances. This avoids floating-point precision issues and simplifies calculations.
3. **Algorithm Selection**: For each point, iterate through all other points, compute the squared distance, and keep track of the maximum distance encountered. If multiple points have the same maximum distance, select the one with the smallest ID.
4. **Complexity Analysis**: The algorithm involves a nested loop where each of the N points is compared with every other point. This results in O(N^2) time complexity, which is feasible given the constraint N ≤ 100.

### Solution Code
```python
n = int(input().strip())
points = []
for i in range(n):
    data = input().split()
    x = int(data[0])
    y = int(data[1])
    points.append((i+1, x, y))

for i in range(n):
    max_sq = -1
    best_id = None
    for j in range(n):
        if i == j:
            continue
        x_i, y_i = points[i][1], points[i][2]
        x_j, y_j = points[j][1], points[j][2]
        dx = x_i - x_j
        dy = y_i - y_j
        d_sq = dx*dx + dy*dy
        if d_sq > max_sq:
            max_sq = d_sq
            best_id = points[j][0]
        elif d_sq == max_sq:
            if points[j][0] < best_id:
                best_id = points[j][0]
    print(best_id)
```

### Explanation
1. **Reading Input**: The input consists of an integer N followed by N lines, each containing the coordinates of a point. Each point is stored with its ID (1-based index) and coordinates in a list.
2. **Processing Each Point**: For each point, initialize variables to track the maximum squared distance encountered and the corresponding smallest ID.
3. **Comparing Distances**: For each other point, compute the squared Euclidean distance. Update the maximum distance and the best ID if a larger distance is found. In case of a tie (equal distance), update the best ID only if the current point's ID is smaller.
4. **Output Result**: After processing all points, the best ID (farthest point) for the current point is printed.

This approach efficiently handles the problem by leveraging squared distances for comparison and ensures correctness by carefully managing tie-breakers through ID comparisons.
