To solve this problem, we need to determine the minimum cost required to purchase at least N eggs from a supermarket that sells packs of 6, 8, and 12 eggs. The packs have different costs, and we can buy any number of each pack. The goal is to find the optimal combination of these packs that meets or exceeds the required number of eggs at the lowest cost.

### Approach
1. **Problem Analysis**: The problem involves selecting combinations of three different pack sizes (6, 8, and 12 eggs) to meet or exceed a given number of eggs (N) at the minimum cost. The challenge is to explore all possible combinations of these packs efficiently.
2. **Intuition**: Since N can be up to 100, a brute-force approach is feasible. We iterate over all possible quantities of 12-egg and 8-egg packs, then calculate the remaining eggs needed and determine the number of 6-egg packs required to cover the deficit. The total cost for each combination is computed, and the minimum cost across all combinations is selected.
3. **Algorithm Selection**: The solution uses nested loops to iterate over possible counts of 12-egg and 8-egg packs. For each combination, if the total eggs from these packs meet or exceed N, the cost is simply the sum of the costs of the selected packs. Otherwise, the remaining eggs are covered by the cheapest 6-egg packs (using ceiling division to determine the minimal number needed).
4. **Complexity Analysis**: The algorithm's complexity is O((N/12 + 3) * (N/8 + 3)), which simplifies to O(N^2) in the worst case. Given that N is at most 100, this approach is efficient and feasible.

### Solution Code
```python
def main():
    import sys
    N, S, M, L = map(int, sys.stdin.readline().split())
    min_cost = float('inf')
    max_x = (N // 12) + 3
    max_y = (N // 8) + 3
    
    for x in range(0, max_x):
        for y in range(0, max_y):
            eggs = 12 * x + 8 * y
            if eggs >= N:
                total_cost = x * L + y * M
                if total_cost < min_cost:
                    min_cost = total_cost
            else:
                remaining_eggs = N - eggs
                z = (remaining_eggs + 5) // 6
                total_cost = x * L + y * M + z * S
                if total_cost < min_cost:
                    min_cost = total_cost
    print(min_cost)

if __name__ == '__main__':
    main()
```
