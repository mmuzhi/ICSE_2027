=== ANSWER ===

To solve this problem, we need to identify the single cell in two given grids where the characters differ. The grids are provided as input, and each grid has exactly N rows and N columns. The solution involves comparing each corresponding cell in both grids until the differing cell is found.

### Approach
1. **Read Input:** The first line of input is the integer N, which specifies the dimensions of the grids. The next N lines represent the first grid (grid A), followed by another N lines for the second grid (grid B).
2. **Compare Grids:** Iterate through each cell of both grids. For each cell (i, j), compare the character from grid A with the corresponding character in grid B.
3. **Output Result:** Once a differing cell is found, output its 1-based indices (i+1, j+1) and terminate the program. The problem guarantees exactly one differing cell, so no further checks are needed after the first difference is found.

### Solution Code
```python
n = int(input().strip())
gridA = [input().strip() for _ in range(n)]
gridB = [input().strip() for _ in range(n)]

for i in range(n):
    for j in range(n):
        if gridA[i][j] != gridB[i][j]:
            print(i+1, j+1)
            exit()
```

### Explanation
1. **Reading Input:** The code starts by reading the integer N, which determines the size of the grids. It then reads the next N lines into `gridA` and the following N lines into `gridB`.
2. **Grid Comparison:** The nested loops iterate over each cell in the grids. The outer loop runs from 0 to N-1 (rows), and the inner loop runs from 0 to N-1 (columns).
3. **Finding Difference:** For each cell (i, j), the character from `gridA` is compared with the corresponding character in `gridB`. If they differ, the code prints the 1-based indices (i+1, j+1) and exits immediately, as the problem guarantees only one differing cell.

This approach efficiently locates the differing cell by leveraging direct comparison and early termination upon finding the first difference, ensuring optimal performance.
