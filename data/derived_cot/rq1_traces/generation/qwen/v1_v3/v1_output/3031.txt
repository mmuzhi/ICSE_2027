### Approach

2. **Prefix Product Array (Left)**: Compute an array where each element at index `i` is the product of all elements to the left of `i` in the flattened array (including the first element as 1).
3. **Suffix Product Array (Right)**: Compute an array where each element at index `i` is the product of all elements to the right of `i` in the flattened array (including the last element as 1).
4. **Combine Results**: For each element in the flattened array, the product except itself is the product of the corresponding left and right prefix products, modulo 12345.
5. **Reshape the Result**: Convert the resulting 1D array back into the original 2D grid dimensions.

This approach efficiently computes the product matrix by leveraging prefix and suffix products, ensuring optimal performance with a time complexity of O(n*m), where n and m are the dimensions of the grid.

class Solution:
