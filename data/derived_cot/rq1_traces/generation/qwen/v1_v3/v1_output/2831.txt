You will be given a competitive programming problem.
You are given a 0-indexed integer array nums. A pair of indices i, j where 0 <= i < j < nums.length is called beautiful if the first digit of nums[i] and the last digit of nums[j] are coprime.
Return the total number of beautiful pairs in nums.
Two integers x and y are coprime if there is no integer greater than 1 that divides both of them. In other words, x and y are coprime if gcd(x, y) == 1, where gcd(x, y) is the greatest common divisor of x and y.
 
Example 1:

Input: nums = [2,5,1,4]
Output: 5
Explanation: There are 5 beautiful pairs in nums:
When i = 0 and j = 1: the first digit of nums[0] is 2, and the last digit of nums[1] is 5. We can confirm that 2 and 5 are coprime, since gcd(2,5) == 1.
When i = 0 and j = 2: the first digit of nums[0] is 2, and the last digit of nums[2] is 1. Indeed, gcd(2,1) == 1.
When i = 1 and j = 2: the first digit of nums[1] is 5, and the last digit of nums[2] is 1. Indeed, gcd(5,1) == 1.
When i = 1 and j = 3: the first digit of nums[1] is 5, and the last digit of nums[3] is 4. Indeed, gcd(5,4) == 1.
When i = 2 and j = 3: the first digit of nums[2] is 1, and the last digit of nums[3] is 4. Indeed, gcd(1,4) == 1.
Thus, we return 5.

=== ANSWER ===

To solve this problem, we need to count the number of beautiful pairs in an array of integers. A pair (i, j) is considered beautiful if the first digit of the number at index i and the last digit of the number at index j are coprime. Two numbers are coprime if their greatest common divisor (gcd) is 1.

### Approach
1. **Extract First and Last Digits**: For each number in the array, extract the first digit and the last digit. The first digit is obtained by converting the number to a string and taking the first character, then converting it back to an integer. The last digit is obtained using the modulo operation (number % 10).
2. **Check Coprimality**: For each pair of indices (i, j) where i < j, check if the first digit of the number at index i and the last digit of the number at index j are coprime. This is done using the gcd function.
3. **Count Valid Pairs**: Iterate through all possible pairs (i, j) with i < j and count those pairs where the gcd of the first digit of nums[i] and the last digit of nums[j] is 1.

### Solution Code
```python
class Solution:
    def countBeautifulPairs(self, nums: List[int]) -> int:
        n = len(nums)
        first_digits = [int(str(x)[0]) for x in nums]
        last_digits = [x % 10 for x in nums]
        
        def gcd(a, b):
            while b:
                a, b = b, a % b
            return a
        
        count = 0
        for i in range(n):
            for j in range(i+1, n):
                if gcd(first_digits[i], last_digits[j]) == 1:
                    count += 1
        return count
```

### Explanation
1. **Extracting Digits**: The first digit of each number is extracted by converting the number to a string and taking the first character. The last digit is obtained using the modulo operation, which efficiently gives the last digit of any number.
2. **Checking Coprimality**: The gcd function uses the Euclidean algorithm to determine if two numbers are coprime. This algorithm repeatedly replaces the larger number by its remainder when divided by the smaller number until one of the numbers becomes zero. The non-zero number at this point is the gcd.
3. **Counting Pairs**: The solution iterates over all pairs (i, j) where i < j. For each pair, it checks if the first digit of nums[i] and the last digit of nums[j] are coprime using the gcd function. If they are, the pair is counted as beautiful.
4. **Efficiency**: The algorithm efficiently checks all possible pairs in O(n^2) time, which is feasible given the constraint that the array length is at most 100. The space complexity is O(n) for storing the first and last digits of each number.
