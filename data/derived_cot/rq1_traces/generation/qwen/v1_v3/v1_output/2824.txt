=== ANSWER ===

To solve this problem, we need to determine if a given three-digit number is fascinating. A number is considered fascinating if, when concatenated with its double and triple, the resulting number contains all digits from 1 to 9 exactly once and does not contain any zeros. 

### Approach
1. **Concatenation**: First, we concatenate the given number `n`, `2 * n`, and `3 * n` into a single string.
2. **Length Check**: The concatenated string must have exactly 9 digits. If the length is not 9, the number cannot be fascinating, so we return `False`.
3. **Digit Check**: The concatenated string must contain all digits from 1 to 9 exactly once. This is verified by checking if the set of digits in the string matches the set of digits from 1 to 9. If they match, the number is fascinating; otherwise, it is not.

### Solution Code
```python
class Solution:
    def isFascinating(self, n: int) -> bool:
        s = str(n) + str(2 * n) + str(3 * n)
        if len(s) != 9:
            return False
        return set(s) == set('123456789')
```

### Explanation
1. **Concatenation**: The given number `n` is converted to a string along with its double (`2 * n`) and triple (`3 * n`), forming a single concatenated string `s`.
2. **Length Check**: The length of `s` is checked. If it is not 9, the function immediately returns `False` because a valid fascinating number must have exactly 9 digits.
3. **Digit Check**: The function checks if the set of characters in `s` is identical to the set of characters from '1' to '9'. This ensures that all digits from 1 to 9 are present exactly once and no other digits (like zeros) are present. If the sets match, the function returns `True`; otherwise, it returns `False`.

This approach efficiently checks the necessary conditions for a number to be fascinating by leveraging string manipulation and set operations for digit validation.
