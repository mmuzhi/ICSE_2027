=== ANSWER ===

To solve this problem, we need to find the first occurrence of the substring "ABC" in a given string S. The solution involves checking each possible 3-character substring in S to see if it matches "ABC". The position of the first occurrence is returned in 1-based indexing, or -1 if the substring is not found.

### Approach
1. **Problem Analysis**: The problem requires scanning the string S from left to right to find the smallest starting index where the substring "ABC" appears. The solution must efficiently check each possible 3-character substring in S.
2. **Algorithm Selection**: The straightforward approach is to iterate through each character in S, checking if the next two characters form "ABC". The iteration starts from the first character (index 0) and continues until the third last character (index N-3) to ensure there are enough characters to form a 3-character substring.
3. **Complexity Analysis**: The algorithm runs in O(N) time, where N is the length of the string. This is efficient because each character is checked exactly once, and the solution stops as soon as the first occurrence is found.
4. **Optimization**: No further optimization is needed beyond the linear scan, as the problem constraints (N ≤ 100) make even a brute-force approach efficient.

### Solution Code
```python
def main():
    n = int(input().strip())
    s = input().strip()
    for i in range(n - 2):
        if s[i] == 'A' and s[i+1] == 'B' and s[i+2] == 'C':
            print(i + 1)
            return
    print(-1)

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The code reads the integer N (length of the string) and the string S itself.
2. **Iterating Through the String**: The loop runs from the first character (index 0) to the third last character (index N-3). For each character at position i, it checks if the substring starting at i is "ABC".
3. **Checking Substring**: If the characters at positions i, i+1, and i+2 are 'A', 'B', and 'C' respectively, the code prints the 1-based index (i+1) and exits.
4. **Handling No Occurrence**: If the loop completes without finding "ABC", the code prints -1, indicating the substring is not present in S.

This approach efficiently checks each possible 3-character substring in linear time, ensuring optimal performance for the given problem constraints.
