You are given a 1-indexed integer array nums of length n.
An element nums[i] of nums is called special if i divides n, i.e. n % i == 0.
Return the sum of the squares of all special elements of nums.
 
Example 1:

Input: nums = [1,2,3,4]
Output: 21
Explanation: There are exactly 3 special elements in nums: nums[1] since 1 divides 4, nums[2] since 2 divides 4, and nums[4] since 4 divides 4. 
Hence, the sum of the squares of all special elements of nums is nums[1] * nums[1] + nums[2] * nums[2] + nums[4] * nums[4] = 1 * 1 + 2 * 2 + 4 * 4 = 21.  

Example 2:

Input: nums = [2,7,1,19,18,3]
Output: 63
Explanation: There are exactly 4 special elements in nums: nums[1] since 1 divides 6, nums[2] since 2 divides 6, nums[3] since 3 divides 6, and nums[6] since 6 divides 6. 
Hence, the sum of the squares of all special elements of nums is nums[1] * nums[1] + nums[2] * nums[2] + nums[3] * nums[3] + nums[6] * nums[6] = 2 * 2 + 7 * 7 + 1 * 1 + 3 * 3 = 63.

=== ANSWER ===

To solve this problem, we need to find the sum of the squares of all special elements in a given 1-indexed array. An element is considered special if its 1-based index divides the length of the array. 

### Approach
1. **Problem Analysis**: The task involves iterating through each element of the array, checking if its 1-based index is a divisor of the array length, and if so, including its square in the sum.
2. **Intuition**: For each index from 1 to the length of the array, we check if the index divides the array length without a remainder. If it does, the element at that index (adjusted for 0-based indexing in Python) contributes its square to the result.
3. **Algorithm Selection**: The solution involves a simple iteration over each index, checking the divisibility condition, and accumulating the sum of squares of qualifying elements.
4. **Complexity Analysis**: The algorithm runs in O(n) time, where n is the length of the array, as it checks each element exactly once. The space complexity is O(1) since we only use a single variable to accumulate the sum.

### Solution Code
```python
class Solution:
    def sumOfSquares(self, nums: List[int]) -> int:
        n = len(nums)
        total = 0
        for i in range(1, n + 1):
            if n % i == 0:
                total += nums[i - 1] ** 2
        return total
```

### Explanation
1. **Initialization**: The method starts by determining the length of the input array `nums` and initializing a variable `total` to 0 to keep track of the sum of squares.
2. **Iteration**: The loop runs from 1 to `n` (inclusive), where `n` is the length of the array. For each index `i`, it checks if `n` is divisible by `i`.
3. **Condition Check**: If the condition `n % i == 0` is true, the square of the element at position `i-1` (adjusting for 0-based indexing in Python) is added to `total`.
4. **Result**: After processing all elements, the method returns the accumulated sum `total`.

This approach efficiently checks each element's eligibility to be part of the sum and computes the result in linear time, making it optimal for the given constraints.
