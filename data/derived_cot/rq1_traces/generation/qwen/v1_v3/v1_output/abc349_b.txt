=== ANSWER ===

To solve this problem, we need to determine if a given string is a "good string" based on specific conditions. A string is considered good if, for every integer \( i \) (where \( i \geq 1 \)), there are either zero or exactly two distinct letters that appear exactly \( i \) times in the string.

### Approach
1. **Frequency Counting**: First, we count the frequency of each character in the string using a dictionary or a counter.
2. **Frequency of Frequencies**: We then create another dictionary where the keys are the frequencies of the characters, and the values are the counts of how many distinct characters have those frequencies.
3. **Check Conditions**: For each integer \( i \) from 1 up to the maximum frequency found in the string, we check if the number of distinct characters with frequency \( i \) is either 0 or exactly 2. If any \( i \) fails this condition, the string is not good.
4. **Output Result**: If all \( i \) satisfy the condition, the string is good; otherwise, it is not.

### Solution Code
```python
from collections import Counter

def main():
    s = input().strip()
    if s == "":
        print("Yes")
    else:
        c = Counter(s)
        max_freq = max(c.values())
        freq_of_freq = {}
        for count in c.values():
            freq_of_freq[count] = freq_of_freq.get(count, 0) + 1
        
        for i in range(1, max_freq + 1):
            count_i = freq_of_freq.get(i, 0)
            if count_i != 0 and count_i != 2:
                print("No")
                return
        
        print("Yes")

if __name__ == "__main__":
    main()
```

### Explanation
1. **Frequency Counting**: The `Counter` from the `collections` module is used to count occurrences of each character in the string. For example, for the input "commencement", the counts would be `{'c': 2, 'o': 1, 'm': 3, 'e': 3, 'n': 2, 't': 1, 's': 0}` (though 's' is not present, so it's omitted).
2. **Frequency of Frequencies**: We then process these counts to determine how many distinct characters have each specific frequency. For "commencement", the frequencies are 1, 2, and 3, with counts of 2, 2, and 2 respectively for these frequencies.
3. **Check Conditions**: We iterate from 1 to the maximum frequency (3 in this case). For each \( i \), we check if the number of characters with that frequency is 0 or 2. In "commencement", \( i = 1 \) has 2 characters, \( i = 2 \) has 2 characters, and \( i = 3 \) has 2 characters, all satisfying the condition, so the output is "Yes".
4. **Output Result**: If all frequencies meet the condition, the string is good, and we print "Yes"; otherwise, we print "No". For instance, "banana" has counts `{'b': 1, 'a': 3, 'n': 2}`, leading to \( i = 1 \) having 1 character (not 0 or 2), so it outputs "No". 

This approach efficiently checks the given conditions by leveraging frequency counting and a straightforward iteration over possible frequencies, ensuring correctness and optimal performance.
