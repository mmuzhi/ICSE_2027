=== ANSWER ===

To solve this problem, we need to determine the total frequencies of elements in an array that have the maximum frequency. The frequency of an element is the number of times it appears in the array. The solution involves counting the occurrences of each element, identifying the maximum frequency, and then summing the frequencies of all elements that match this maximum frequency.

### Approach
1. **Count Frequencies**: Use a dictionary or a counter to keep track of how many times each element appears in the array.
2. **Find Maximum Frequency**: Iterate through the frequency counts to find the highest frequency present in the array.
3. **Sum Matching Frequencies**: Sum the frequencies of all elements that have this maximum frequency. This sum represents the total occurrences of elements that appear most frequently in the array.

### Solution Code
```python
from collections import Counter

class Solution:
    def maxFrequencyElements(self, nums: List[int]) -> int:
        freq_counter = Counter(nums)
        max_freq = max(freq_counter.values())
        return sum(count for count in freq_counter.values() if count == max_freq)
```

### Explanation
1. **Count Frequencies**: The `Counter` from the `collections` module is used to count the occurrences of each element in the input array. This gives a dictionary where keys are elements and values are their respective counts.
2. **Find Maximum Frequency**: The maximum value in the frequency dictionary is identified, which represents the highest frequency of any element in the array.
3. **Sum Matching Frequencies**: The solution then sums all the frequencies that match the maximum frequency. For example, if the array is `[1,2,2,3,1,4]`, the frequencies are `1:2`, `2:2`, `3:1`, `4:1`. The maximum frequency is `2`, and the sum of frequencies for elements with this frequency is `2 + 2 = 4`.

This approach efficiently computes the desired result by leveraging dictionary operations and simple arithmetic, ensuring optimal performance even for the upper constraint limits.
