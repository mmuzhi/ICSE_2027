Explanation: There are no vowels in s (all characters in s are consonants), so we return "lYmpH".

=== ANSWER ===

To solve this problem, we need to permute a given string such that all consonants remain in their original positions while the vowels are sorted in non-decreasing order of their ASCII values. 

### Approach
1. **Identify Vowels**: First, we define which characters are vowels (both uppercase and lowercase).
2. **Collect and Sort Vowels**: We collect all vowels from the input string and sort them in ascending order based on their ASCII values.
3. **Replace Vowels in Original String**: We traverse the original string, replacing each vowel with the next vowel from the sorted list. Consonants are left unchanged in their original positions.

### Solution Code
```python
class Solution:
    def sortVowels(self, s: str) -> str:
        vowels_set = set("aeiouAEIOU")
        vowels_list = [char for char in s if char in vowels_set]
        vowels_list.sort()
        sorted_vowels = iter(vowels_list)
        result = []
        for char in s:
            if char in vowels_set:
                result.append(next(sorted_vowels))
            else:
                result.append(char)
        return "".join(result)
```

### Explanation
1. **Vowel Identification**: The code checks each character in the input string to determine if it is a vowel (either lowercase or uppercase).
2. **Sorting Vowels**: All identified vowels are collected into a list, which is then sorted in ascending order based on their ASCII values.
3. **String Reconstruction**: The original string is processed character by character. If a character is a vowel, it is replaced with the next vowel from the sorted list. Non-vowel characters (consonants) are added to the result as-is. This ensures that vowels are placed in the correct positions in the sorted order while consonants remain in their original positions.

This approach efficiently handles the constraints by leveraging sorting and iteration, ensuring optimal performance even for large input sizes.
