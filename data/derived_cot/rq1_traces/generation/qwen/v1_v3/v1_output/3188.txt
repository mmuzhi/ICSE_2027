To solve this problem, we need to determine the champion team in a tournament where each team's strength is defined by a given grid. The champion is the team that no other team can defeat. 

### Approach
1. **Problem Analysis**: The problem involves analyzing a grid where each entry `grid[i][j]` indicates whether team `i` is stronger than team `j` (1) or not (0). The champion is the team that cannot be defeated by any other team, meaning for every other team `b`, `grid[b][a]` must be 0 (indicating `b` is not stronger than `a`).

2. **Insight**: The solution hinges on checking each team to see if all other teams are weaker than it. This translates to verifying that in the column corresponding to the team (excluding its own entry), all entries are 0. 

3. **Algorithm Selection**: For each team `a`, iterate through all other teams `b` and check if `grid[b][a]` is 1 (indicating `b` is stronger than `a`). If any such `b` is found, team `a` cannot be the champion. If no such `b` is found, team `a` is the champion.

4. **Complexity Analysis**: The algorithm checks each team against all other teams, resulting in a time complexity of O(n^2), where `n` is the number of teams. Given the constraint `n <= 100`, this approach is efficient.

### Solution Code
```python
class Solution:
    def findChampion(self, grid: List[List[int]]) -> int:
        n = len(grid)
        for a in range(n):
            is_champion = True
            for b in range(n):
                if b == a:
                    continue
                if grid[b][a] == 1:
                    is_champion = False
                    break
            if is_champion:
                return a
```

3. **Champion Identification**: If no stronger team is found for `a` during the check, `a` is declared the champion, and its index is returned immediately.
4. **Termination**: The loop terminates as soon as the first champion is found, leveraging the problem's condition that there is exactly one champion in a total order scenario.
