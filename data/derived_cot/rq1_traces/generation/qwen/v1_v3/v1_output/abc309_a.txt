=== ANSWER ===

To solve this problem, we need to determine if two given numbers on a standard 3x3 grid are adjacent horizontally. The grid is arranged in row-major order with numbers 1 through 9. 

### Approach
1. **Problem Analysis**: The problem requires checking if two numbers, A and B (where A < B), are located in the same row and adjacent columns on the grid. The grid is fixed as follows:
   ```
   1 2 3
   4 5 6
   7 8 9
   ```
   Horizontal adjacency means the two numbers must be in the same row and their columns must be consecutive (i.e., the column of B is exactly one more than the column of A).

2. **Intuition**: Since the grid is fixed, we can predefine all pairs of numbers that are horizontally adjacent. These pairs are (1,2), (2,3), (4,5), (5,6), (7,8), and (8,9). 

3. **Algorithm Selection**: The solution involves checking if the given pair (A, B) exists in the predefined list of horizontally adjacent pairs. If it does, the answer is "Yes"; otherwise, it's "No".

4. **Complexity Analysis**: The solution involves a simple lookup in a set (or list) of fixed pairs, which is an O(1) operation due to the small constant size of the grid.

### Solution Code
```python
A, B = map(int, input().split())
pairs = {(1, 2), (2, 3), (4, 5), (5, 6), (7, 8), (8, 9)}
print("Yes" if (A, B) in pairs else "No")
```

### Explanation
1. **Reading Input**: The input consists of two integers, A and B, where A is less than B.
2. **Checking Adjacency**: The solution checks if the pair (A, B) is present in a predefined set of horizontally adjacent pairs. The set contains all pairs of numbers that are adjacent in the same row of the grid.
3. **Output Result**: If the pair is found in the set, it prints "Yes"; otherwise, it prints "No". This approach efficiently leverages the fixed grid structure to determine adjacency without complex computations.
