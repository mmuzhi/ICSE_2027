=== ANSWER ===

To solve this problem, we need to calculate the final score obtained by repeatedly removing the largest number from each row of a given matrix until the matrix is empty. The score is incremented by the highest number removed in each operation.

### Approach
1. **Sort Each Row in Descending Order**: For each row in the matrix, sort the elements in descending order. This allows us to easily access the largest remaining element in each row during each operation.
2. **Determine Maximum Operations**: The number of operations required is equal to the length of the longest row in the matrix, as each operation removes one element from each row until all rows are exhausted.
3. **Iterate Over Each Operation**: For each operation, examine the current largest element (at the current index) of each row that still has elements. The highest element among these is added to the score.
4. **Sum the Scores**: Accumulate the highest elements from each operation to get the final score.

### Solution Code
```python
class Solution:
    def matrixSum(self, nums: List[List[int]]) -> int:
        for row in nums:
            row.sort(reverse=True)
        max_ops = max(len(row) for row in nums)
        score = 0
        for op in range(max_ops):
            current_max = 0
            for row in nums:
                if op < len(row) and row[op] > current_max:
                    current_max = row[op]
            score += current_max
        return score
```

### Explanation
1. **Sorting Rows**: Each row is sorted in descending order to facilitate easy access to the largest remaining elements during each operation. For example, the row `[7, 2, 1]` becomes `[7, 2, 1]` after sorting.
2. **Operations Calculation**: The number of operations is determined by the longest row in the matrix. For instance, if the longest row has 3 elements, there will be 3 operations.
3. **Processing Each Operation**: For each operation, the algorithm checks the current largest element (at the current index) of each row that still has elements. The highest element among these is added to the score. For example, in the first operation, the elements `[7, 6, 6, 3]` are considered, and the highest is `7`.
4. **Score Accumulation**: The highest elements from each operation are summed up to produce the final score. In the example, the scores from operations are `7`, `5`, and `3`, summing to `15`.

This approach efficiently handles the problem by leveraging sorting and iterative processing, ensuring optimal performance even for the upper constraint limits.
