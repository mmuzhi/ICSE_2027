=== ANSWER ===

To solve this problem, we need to determine whether the plate of rice ('R') is positioned to the left of the plate of miso soup ('M') in a given string of length 3. The string consists of exactly one 'R', one 'M', and one 'S'. 

### Approach
1. **Problem Analysis**: The problem requires checking the relative positions of 'R' and 'M' in the string. Since the string length is fixed at 3 and contains each character exactly once, we can directly compare the indices of 'R' and 'M'.
2. **Key Insight**: If the index of 'R' is less than the index of 'M', then 'R' is to the left of 'M'. Otherwise, it is not.
3. **Algorithm Selection**: The solution involves finding the positions of 'R' and 'M' in the string and comparing their indices. This is a straightforward comparison after locating the characters' positions.

### Solution Code
```python
def main():
    S = input().strip()
    idx_rice = S.index('R')
    idx_miso = S.index('M')
    if idx_rice < idx_miso:
        print("Yes")
    else:
        print("No")

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The input string `S` is read from standard input.
2. **Finding Indices**: The `index` method is used to find the positions of 'R' and 'M' in the string. This method returns the first occurrence of the specified character.
3. **Comparison**: The indices of 'R' and 'M' are compared. If the index of 'R' is smaller, it means 'R' appears before 'M' in the string, and "Yes" is printed. Otherwise, "No" is printed.
4. **Output**: The result is printed based on the comparison.

This approach efficiently checks the relative positions of the two characters by leveraging simple string operations and comparison, ensuring correctness with minimal computational overhead.
