You will be given a competitive programming problem.
You are given a 1-indexed array of distinct integers nums of length n.
You need to distribute all the elements of nums between two arrays arr1 and arr2 using n operations. In the first operation, append nums[1] to arr1. In the second operation, append nums[2] to arr2. Afterwards, in the i^th operation:

If the last element of arr1 is greater than the last element of arr2, append nums[i] to arr1. Otherwise, append nums[i] to arr2.

The array result is formed by concatenating the arrays arr1 and arr2. For example, if arr1 == [1,2,3] and arr2 == [4,5,6], then result = [1,2,3,4,5,6].
Return the array result.
 
Example 1:

Input: nums = [2,1,3]
Output: [2,3,1]
Explanation: After the first 2 operations, arr1 = [2] and arr2 = [1].
In the 3^rd operation, as the last element of arr1 is greater than the last element of arr2 (2 > 1), append nums[3] to arr1.
After 3 operations, arr1 = [2,3] and arr2 = [1].
Hence, the array result formed by concatenation is [2,3,1].

Example 2:

Input: nums = [5,4,3,8]
Output: [5,3,4,8]
Explanation: After the first 2 operations, arr1 = [5] and arr2 = [4].
In the 3^rd operation, as the last element of arr1 is greater than the last element of arr2 (5 > 4), append nums[3] to arr1, hence arr1 becomes [5,3].
In the 4^th operation, as the last element of arr2 is greater than the last element of arr1 (4 > 3), append nums[4] to arr2, hence arr2 becomes [4,8].
After 4 operations, arr1 = [5,3] and arr2 = [4,8].
Hence, the array result formed by concatenation is [5,3,4,8].

All elements in nums are distinct.

=== ANSWER ===

To solve this problem, we need to distribute elements from a given array into two separate arrays based on specific rules. The distribution is done through a series of operations where each subsequent element is appended to one of the arrays depending on the comparison of their last elements.

### Approach
1. **Initialization**: Start with two empty arrays, `arr1` and `arr2`.
2. **First Operation**: The first element of the input array is appended to `arr1`.
3. **Second Operation**: The second element of the input array is appended to `arr2`.
4. **Subsequent Operations**: For each remaining element in the input array, compare the last elements of `arr1` and `arr2`. If the last element of `arr1` is greater, append the current element to `arr1`; otherwise, append it to `arr2`.
5. **Result Construction**: After processing all elements, concatenate `arr1` and `arr2` to form the final result array.

### Solution Code
```python
class Solution:
    def resultArray(self, nums: List[int]) -> List[int]:
        arr1 = []
        arr2 = []
        arr1.append(nums[0])
        arr2.append(nums[1])
        
        for i in range(2, len(nums)):
            if arr1[-1] > arr2[-1]:
                arr1.append(nums[i])
            else:
                arr2.append(nums[i])
                
        return arr1 + arr2
```

### Explanation
1. **Initialization**: Two empty lists, `arr1` and `arr2`, are initialized to store the elements.
2. **First Operation**: The first element of `nums` (index 0) is added to `arr1`.
3. **Second Operation**: The second element of `nums` (index 1) is added to `arr2`.
4. **Subsequent Operations**: For each subsequent element (from index 2 onwards), the algorithm checks the last elements of both arrays. If the last element of `arr1` is greater, the current element is added to `arr1`; otherwise, it is added to `arr2`.
5. **Result Construction**: The final result is obtained by concatenating `arr1` and `arr2` in that order.

This approach efficiently distributes the elements according to the given rules, ensuring the correct order in the resulting array. The algorithm operates in O(n) time complexity, where n is the length of the input array, making it optimal for the given constraints.
