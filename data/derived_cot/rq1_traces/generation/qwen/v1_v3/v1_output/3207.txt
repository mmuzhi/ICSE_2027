To solve this problem, we need to determine the minimum number of operations required to make three given strings equal by repeatedly removing the rightmost character from any of the strings (as long as the string's length is at least 2). If it's not possible to make the strings equal, we should return -1.

### Approach
1. **Identify the Common Prefix**: The key observation is that the leftmost characters of the three strings must be the same for them to be made equal by removing characters from the right. The common prefix is the longest sequence of characters starting from the left that is identical in all three strings.
2. **Check for Feasibility**: If the common prefix length is zero, it means the first characters of the strings differ, making it impossible to make the strings equal by any number of operations. Hence, we return -1.
3. **Calculate Operations**: If a common prefix exists, the minimum number of operations required is the sum of the differences between each string's length and the common prefix length. This is derived from the fact that each string must be reduced to the common prefix by removing characters from the right.

### Solution Code
```python
class Solution:
    def findMinimumOperations(self, s1: str, s2: str, s3: str) -> int:
        min_len = min(len(s1), len(s2), len(s3))
        L = 0
        for i in range(min_len):
            if s1[i] == s2[i] == s3[i]:
                L += 1
            else:
                break
        if L == 0:
            return -1
        return (len(s1) + len(s2) + len(s3)) - 3 * L
```

### Explanation
1. **Finding the Common Prefix**: The loop iterates through each character position from the start of the strings up to the length of the shortest string. For each position, if all three strings have the same character, the common prefix length `L` is incremented. If any character differs, the loop breaks, and `L` remains the count of matching characters from the start.
2. **Feasibility Check**: If `L` is zero, the first characters of the strings differ, so it's impossible to make the strings equal by removing characters from the right. Hence, return -1.
3. **Calculating Operations**: The total operations needed is the sum of the characters to be removed from each string to reach the common prefix. This is computed as the sum of the lengths of all three strings minus three times the common prefix length (since each character in the common prefix is retained in all strings). 

This approach efficiently checks for the possibility of making the strings equal and calculates the minimal operations required by leveraging the common prefix concept.
