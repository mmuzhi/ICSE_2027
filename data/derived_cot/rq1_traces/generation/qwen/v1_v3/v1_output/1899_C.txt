To solve this problem, we need to find the maximum sum of a non-empty contiguous subarray where adjacent elements have alternating parities (i.e., one element is even and the next is odd, or vice versa). The solution involves efficiently traversing the array while keeping track of the maximum subarray sum that meets the given conditions.

### Approach
1. **Problem Analysis**: The problem requires checking each adjacent pair in the subarray to ensure they have different parities. The key observation is that a valid subarray can be extended if the current element and the previous element have alternating parities. Otherwise, the subarray must start anew from the current element.
2. **Dynamic Programming (DP) Setup**: We use a DP array where `dp[i]` represents the maximum sum of a valid subarray ending at index `i`. The value of `dp[i]` is either the element itself (if it starts a new subarray) or the sum of the current element and `dp[i-1]` (if the parities of the current and previous elements alternate).
3. **Efficiency Consideration**: To optimize space, we only keep track of the previous DP value (`dp_prev`) and the current maximum sum (`best`) during the traversal. This reduces the space complexity from O(n) to O(1) per test case.
4. **Parity Check**: For each element, we check its parity (even or odd) against the previous element's parity. If they alternate, we can extend the subarray; otherwise, we start a new subarray from the current element.

### Solution Code
```python
t = int(input().strip())
for _ in range(t):
    n = int(input().strip())
    a = list(map(int, input().split()))
    if n == 1:
        print(a[0])
        continue

    dp_prev = a[0]
    best = a[0]
    last_parity = a[0] % 2

    for i in range(1, n):
        current_parity = a[i] % 2
        if current_parity != last_parity:
            dp_current = max(a[i], dp_prev + a[i])
        else:
            dp_current = a[i]
        if dp_current > best:
            best = dp_current
        dp_prev = dp_current
        last_parity = current_parity

    print(best)
```

### Explanation
1. **Initialization**: For each test case, read the array length `n` and the array `a`. If the array length is 1, the result is simply the element itself.
2. **DP Setup**: Initialize `dp_prev` with the first element of the array, `best` with the same value, and `last_parity` with the parity of the first element.
3. **Traversal**: For each subsequent element:
   - **Parity Check**: Compare the current element's parity with the previous element's parity. If they alternate, update `dp_current` to be the maximum of the current element or the sum of the current element and `dp_prev`. Otherwise, set `dp_current` to the current element.
   - **Update Best**: If `dp_current` exceeds `best`, update `best`.
   - **Update Previous**: Set `dp_prev` to `dp_current` for the next iteration and update `last_parity` to the current element's parity.
4. **Output**: After processing all elements, print the maximum subarray sum (`best`) found.

This approach efficiently checks all possible valid subarrays in linear time with constant space, making it optimal for large input sizes.
