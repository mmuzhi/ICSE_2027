=== ANSWER ===

To solve this problem, we need to determine if a given integer is a Harshad number. A Harshad number is an integer divisible by the sum of its own digits. If it is, we return the sum of its digits; otherwise, we return -1.

### Approach
1. **Calculate the Sum of Digits**: For the given integer, compute the sum of its digits. This can be done by converting the integer to a string, splitting it into individual digits, converting each digit back to an integer, and summing them up.
2. **Check Harshad Condition**: Verify if the original integer is divisible by the sum of its digits. If it is, return the sum of the digits; otherwise, return -1.

### Solution Code
```python
class Solution:
    def sumOfTheDigitsOfHarshadNumber(self, x: int) -> int:
        s = sum(int(d) for d in str(x))
        if x % s == 0:
            return s
        else:
            return -1
```

### Explanation
1. **Sum Calculation**: The code first converts the integer `x` into a string to process each digit individually. Each character in the string is converted back to an integer and summed up to get the total digit sum.
2. **Divisibility Check**: The code checks if the original number `x` is divisible by the computed digit sum. If it is, the digit sum is returned; otherwise, -1 is returned, indicating that the number is not a Harshad number.

This approach efficiently checks the Harshad number condition by leveraging simple string manipulation and arithmetic operations, ensuring clarity and correctness within the given constraints.
