To solve this problem, we need to count the number of distinct non-empty substrings in a given string that consist of repetitions of a single character. The key observation here is that for each character in the string, the distinct substrings are formed by repeating that character from 1 up to the maximum consecutive occurrences of that character in the string. 

### Approach
1. **Problem Analysis**: The problem requires counting all distinct substrings that are repetitions of a single character. For example, in the string "aaabaa", the distinct substrings are "a", "aa", "aaa", and "b". The challenge is to efficiently compute these without enumerating all possible substrings, which would be computationally expensive for large strings.
2. **Insight**: For each character in the string, the maximum number of distinct substrings it can form is equal to the length of its longest contiguous run. For instance, a run of 'a' of length 3 contributes three distinct substrings: "a", "aa", and "aaa".
3. **Algorithm Selection**: We traverse the string once, keeping track of the current character and its consecutive count. Whenever the character changes, we update the maximum run length for the previous character and reset the count for the new character. After processing the entire string, we sum the maximum run lengths for all characters to get the total count of distinct substrings.
4. **Complexity Analysis**: The algorithm processes each character in the string exactly once, resulting in a time complexity of O(N), where N is the length of the string. This is optimal for the given constraints (N ≤ 2×10^5).

### Solution Code
```python
def main():
    n = int(input().strip())
    s = input().strip()

    if n == 0:
        print(0)
        return

    max_run_dict = {}
    current_char = s[0]
    count = 1

    for i in range(1, n):
        if s[i] == current_char:
            count += 1
        else:
            if current_char in max_run_dict:
                if count > max_run_dict[current_char]:
                    max_run_dict[current_char] = count
            else:
                max_run_dict[current_char] = count
            current_char = s[i]
            count = 1

    if current_char in max_run_dict:
        if count > max_run_dict[current_char]:
            max_run_dict[current_char] = count
    else:
        max_run_dict[current_char] = count

    total = sum(max_run_dict.values())
    print(total)

if __name__ == '__main__':
    main()
```

### Explanation
1. **Initialization**: We start by reading the input values: the length of the string (N) and the string itself (S). If the string is empty, we immediately return 0 as there are no substrings to count.
2. **Tracking Runs**: We initialize a dictionary `max_run_dict` to keep track of the maximum consecutive runs for each character. We start with the first character of the string and a count of 1.
3. **Traversal**: As we iterate through the string, we check if the current character matches the previous one. If it does, we increment the count. If not, we update the dictionary with the maximum run length for the previous character and reset the count for the new character.
4. **Final Update**: After the loop, we perform a final update to include the last character's run length in the dictionary.
5. **Result Calculation**: The total number of distinct substrings is the sum of all values in `max_run_dict`, which are the maximum run lengths for each character. This sum is printed as the result.

This approach efficiently counts all distinct single-character repeated substrings by leveraging the observation that each character's contribution is determined by its longest contiguous run in the string. The solution is both optimal and straightforward, handling the constraints effectively.
