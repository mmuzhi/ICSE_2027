To solve this problem, we need to determine the overall winner between Takahashi and Aoki based on the results of N games. The winner is decided by two criteria: the player who wins more games, or if both have the same number of wins, the player who reaches that number of wins first.

### Approach
1. **Problem Analysis**: The problem involves processing a string of game results where each character 'T' or 'A' indicates a win for Takahashi or Aoki, respectively. The solution requires counting the total wins for each player and then determining the winner based on the specified criteria.
2. **Key Insight**: If one player has more wins, they are the immediate winner. If both have the same number of wins, we need to find the earliest game at which each player reached that win count and declare the one who reached it first as the winner.
3. **Algorithm Selection**: 
   - **Counting Wins**: First, count the total wins for both players by iterating through the string.
   - **Determine Winner**: Compare the win counts. If one is higher, that player wins. If counts are equal, iterate through the string again to find the earliest positions where each player reached the tie-breaking win count (which is the same for both).
4. **Complexity Analysis**: The solution involves two passes through the string: one to count wins and another to determine the earliest win positions. Both passes are O(N), making the overall complexity O(N), which is efficient given the constraint N ≤ 100.

### Solution Code
```python
def main():
    n = int(input().strip())
    s = input().strip()
    
    total_t = s.count('T')
    total_a = s.count('A')
    
    if total_t > total_a:
        print('T')
    elif total_a > total_t:
        print('A')
    else:
        k = total_t
        t_count = 0
        a_count = 0
        t_reach_index = None
        a_reach_index = None
        
        for i, char in enumerate(s):
            if char == 'T':
                t_count += 1
                if t_count == k:
                    t_reach_index = i
            if char == 'A':
                a_count += 1
                if a_count == k:
                    a_reach_index = i
        
        if t_reach_index < a_reach_index:
            print('T')
        else:
            print('A')

if __name__ == '__main__':
    main()
```
