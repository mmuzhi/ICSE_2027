=== ANSWER ===

To solve this problem, we need to determine if all the integers in a given list are equal. The solution involves reading the input values, checking if all elements in the list are the same, and then printing "Yes" if they are all equal, otherwise "No".

### Approach
1. **Read Input**: The first line of input is an integer \( N \), which represents the number of integers to follow. The second line contains \( N \) integers separated by spaces.
2. **Check Uniformity**: Convert the second line of input into a list of integers. Then, check if all elements in the list are identical by comparing each element with the first element of the list.
3. **Output Result**: Based on the check, print "Yes" if all elements are the same, otherwise print "No".

### Solution Code
```python
def main():
    n = int(input().strip())
    arr = list(map(int, input().split()))
    
    if all(x == arr[0] for x in arr):
        print("Yes")
    else:
        print("No")

if __name__ == '__main__':
    main()
```

### Explanation
1. **Reading Input**: The first line is read as an integer \( N \), which is the number of elements in the list. The second line is split into individual strings, which are then converted to integers.
2. **Checking Uniformity**: The code uses a generator expression within the `all()` function to check if every element in the list matches the first element. If all elements are the same, the `all()` function returns `True`.
3. **Output**: Depending on the result of the uniformity check, the code prints "Yes" if all elements are identical, otherwise it prints "No".

This approach efficiently checks for uniformity in the list by leveraging the `all()` function, which short-circuits early if any element differs from the first, optimizing performance. The solution handles all edge cases within the given constraints, ensuring correctness for any valid input.
