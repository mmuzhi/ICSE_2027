=== ANSWER ===

To solve this problem, we need to determine if a given string of three distinct characters 'a', 'b', and 'c' can be transformed into the string "abc" with at most one swap operation. 

### Approach
1. **Problem Analysis**: The problem involves checking if the given string can be converted to "abc" by swapping at most two characters. Since there are only six possible permutations of three distinct characters, we can directly check if the string is one of the permutations that can be transformed into "abc" with zero or one swap.
2. **Key Insight**: The permutations that can be transformed into "abc" with at most one swap are "abc" itself (no swap needed), "acb" (swap 'c' and 'b'), "bac" (swap 'b' and 'a'), and "cba" (swap 'c' and 'a'). The remaining permutations, "bca" and "cab", require two swaps and thus cannot be achieved with at most one swap.
3. **Algorithm Selection**: The solution involves checking if the input string matches any of the four valid permutations ("abc", "acb", "bac", "cba"). If it does, the answer is "YES"; otherwise, it's "NO".

### Solution Code
```python
t = int(input().strip())
for _ in range(t):
    s = input().strip()
    if s in ['abc', 'acb', 'bac', 'cba']:
        print("YES")
    else:
        print("NO")
```

### Explanation
1. **Reading Input**: The first line reads the number of test cases, `t`.
2. **Processing Each Test Case**: For each test case, the string `s` is read.
3. **Checking Valid Permutations**: The solution checks if `s` is one of the four valid permutations ("abc", "acb", "bac", "cba"). If it is, it prints "YES"; otherwise, it prints "NO".
4. **Efficiency**: The solution efficiently checks each string in constant time, making it optimal for the given constraints (up to 6 test cases). The approach leverages the small input size to use a straightforward membership check, ensuring clarity and correctness.
