To solve this problem, we need to find all beautiful indices in a given string `s` based on the presence of two substrings `a` and `b` within a certain distance `k` from each other. An index `i` is considered beautiful if the substring starting at `i` matches `a`, and there exists another index `j` where the substring starting at `j` matches `b`, with the absolute difference between `i` and `j` being at most `k`.

### Approach
1. **Find Occurrences of Substrings**: First, we scan the string `s` to collect all starting indices where the substrings `a` and `b` occur. This involves iterating through the string and checking each possible starting position for a match with `a` and `b`.
2. **Check for Beautiful Indices**: For each occurrence of `a` (stored in `list_a`), we check if there exists any occurrence of `b` (stored in `list_b`) within the range `[i - k, i + k]`. This is efficiently done using binary search (via `bisect_left`) to find the first occurrence in `list_b` that is within the specified range.
3. **Return Result**: Collect all indices `i` from `list_a` that meet the condition and return them in sorted order.
