You will be given a competitive programming problem.
Takahashi, a young baseball enthusiast, has been a very good boy this year, so Santa has decided to give him a bat or a glove, whichever is more expensive.
If a bat costs B yen and a glove costs G yen (B\neq G), which one will Santa give to Takahashi?

Input

The input is given from Standard Input in the following format:
B G

Output

If Santa gives Takahashi a bat, print Bat; if Santa gives him a glove, print Glove.

Constraints


- B and G are different integers between 1 and 1000, inclusive.

Sample Input 1

300 100

Sample Output 1

Bat

The bat is more expensive than the glove, so Santa will give Takahashi the bat.

Sample Input 2

334 343

Sample Output 2

Glove

The glove is more expensive than the bat, so Santa will give Takahashi the glove.

=== ANSWER ===

To solve this problem, we need to determine which item—either a bat or a glove—is more expensive based on the given costs. Santa will give Takahashi the more expensive item between the two. 

### Approach
1. **Input Reading**: Read the input values which consist of two integers, B and G, representing the costs of the bat and glove respectively.
2. **Comparison**: Compare the two integers to determine which one is larger.
3. **Output Decision**: Based on the comparison, output "Bat" if the bat is more expensive, or "Glove" if the glove is more expensive.

### Solution Code
```python
def main():
    data = input().split()
    B = int(data[0])
    G = int(data[1])
    if B > G:
        print("Bat")
    else:
        print("Glove")

if __name__ == "__main__":
    main()
```

### Explanation
1. **Input Handling**: The input is read as a string and split into two parts. These parts are then converted into integers representing the costs of the bat (B) and glove (G).
2. **Comparison**: The integers B and G are compared using a simple conditional statement. If B is greater than G, the program prints "Bat". Otherwise, it prints "Glove".
3. **Output**: The result is printed based on the comparison, ensuring that Santa gives Takahashi the more expensive item.

This approach efficiently checks the costs and outputs the correct item with minimal computational steps, adhering to the problem constraints and requirements.
