=== ANSWER ===

To solve this problem, we need to determine if a given string meets specific conditions regarding the case of its characters. The first character must be uppercase, and all subsequent characters must be lowercase. 

### Approach
1. **Check the First Character**: The first character of the string must be uppercase. This can be verified using the `isupper()` method available for string characters.
2. **Check the Remaining Characters**: All characters from the second position to the end of the string must be lowercase. This can be verified using the `islower()` method on each character or by ensuring the substring from the second character to the end consists only of lowercase letters.

The solution involves two main checks:
1. The first character is uppercase.
2. The rest of the string (if any) consists only of lowercase letters.

### Solution Code
```python
def main():
    S = input().strip()
    if S[0].isupper() and S[1:].islower():
        print("Yes")
    else:
        print("No")

if __name__ == "__main__":
    main()
```

### Explanation
1. **Input Handling**: The input string `S` is read and stripped of any leading or trailing whitespace.
2. **First Character Check**: The first character of `S` is checked to ensure it is uppercase using the `isupper()` method.
3. **Remaining Characters Check**: The substring from the second character to the end of `S` is checked to ensure all characters are lowercase using the `islower()` method. The `islower()` method returns `True` for an empty string, which handles the edge case where the string has only one character.
4. **Output Result**: If both conditions are satisfied, "Yes" is printed; otherwise, "No" is printed.

This approach efficiently checks the required conditions with minimal computational overhead, leveraging Python's built-in string methods for case checking.
