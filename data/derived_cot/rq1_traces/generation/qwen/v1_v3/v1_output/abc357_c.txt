=== ANSWER ===

To solve this problem, we need to generate a level-N carpet according to the specified fractal pattern. The carpet is defined recursively, where each level builds upon the previous level by dividing the grid into nine smaller grids and replacing the central grid with white cells while keeping the other eight grids as the previous level's carpet.

### Approach
1. **Base Case Handling**: For level 0, the carpet is a single black cell ("#").
2. **Recursive Construction**: For levels greater than 0, the carpet is constructed by:
   - Dividing the grid into nine 3^(N-1) x 3^(N-1) blocks.
   - The central block is entirely white ('.').
   - The other eight blocks are level-(N-1) carpets.
3. **Grid Assembly**: The grid is assembled by combining the top, middle, and bottom parts:
   - **Top Part**: The first 3^(N-1) rows, each formed by horizontally concatenating three level-(N-1) carpets.
   - **Middle Part**: The next 3^(N-1) rows, each formed by horizontally concatenating a level-(N-1) carpet, a white block of the same size, and another level-(N-1) carpet.
   - **Bottom Part**: The last 3^(N-1) rows, identical to the top part.

### Solution Code
```python
def main():
    import sys
    n = int(sys.stdin.readline())

    def build_carpet(k):
        if k == 0:
            return ["#"]
        prev = build_carpet(k-1)
        size_prev = len(prev)
        white_block = ['.' * size_prev for _ in range(size_prev)]
        top = []
        for i in range(size_prev):
            top.append(prev[i] * 3)
        middle = []
        for i in range(size_prev):
            middle.append(prev[i] + white_block[i] + prev[i])
        bottom = []
        for i in range(size_prev):
            bottom.append(prev[i] * 3)
        return top + middle + bottom

    if n == 0:
        print("#")
    else:
        carpet = build_carpet(n)
        for row in carpet:
            print(row)

if __name__ == "__main__":
    main()
```

### Explanation
1. **Base Case**: The function `build_carpet` returns a single "#" for level 0.
2. **Recursive Step**: For higher levels, the function constructs the carpet by:
   - **Previous Level Carpet**: Calls `build_carpet(k-1)` to get the pattern for the previous level.
   - **White Block**: Creates a grid of the same size as the previous level's carpet filled entirely with '.'.
   - **Top and Bottom Parts**: These parts are formed by repeating each row of the previous level's carpet three times horizontally.
   - **Middle Part**: Each row here is formed by concatenating a row from the previous level's carpet, a row from the white block, and another row from the previous level's carpet.
3. **Output**: The final grid is printed row by row, with each row being a string of '#' and '.' characters as specified by the fractal pattern.

This approach efficiently builds the fractal pattern recursively, leveraging the self-similar nature of the carpet design to construct each level based on the previous one. The solution handles all levels from 0 to 6 within feasible computational limits due to the small maximum grid size (729x729 for level 6).
