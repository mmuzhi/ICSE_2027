=== ANSWER ===

To solve this problem, we need to count the number of index pairs (i, j) in a given list of words such that the word at index i is both a prefix and a suffix of the word at index j, where i < j. 

### Approach
1. **Problem Analysis**: The task involves checking each word pair (i, j) where i < j to determine if the word at index i is both a prefix and a suffix of the word at index j. The solution requires iterating through all possible pairs and verifying the prefix and suffix conditions for each pair.
2. **Intuition**: For each word at index i, we compare it with every subsequent word at index j (i < j). The word at index i must be a prefix (starting part) and a suffix (ending part) of the word at index j. The solution leverages Python's built-in string methods `startswith` and `endswith` to efficiently check these conditions.
3. **Algorithm Selection**: A nested loop approach is used where the outer loop runs from the first word to the last, and the inner loop runs from the next word to the end. For each pair, the conditions are checked using the string methods mentioned.
4. **Complexity Analysis**: Given the constraints (words length ≤ 50 and each word length ≤ 10), the solution involves O(n^2) operations where n is the number of words. Each operation checks the prefix and suffix conditions in O(1) time (since string checks in Python are efficient for small strings), making the overall complexity O(n^2), which is feasible for the given constraints.

### Solution Code
```python
class Solution:
    def countPrefixSuffixPairs(self, words: List[str]) -> int:
        count = 0
        n = len(words)
        for i in range(n):
            for j in range(i + 1, n):
                if words[j].startswith(words[i]) and words[j].endswith(words[i]):
                    count += 1
        return count
```

### Explanation
1. **Initialization**: The solution initializes a counter `count` to zero to keep track of valid pairs.
2. **Nested Loop**: The outer loop iterates over each word in the list (from index 0 to n-1). The inner loop iterates over each subsequent word (from i+1 to n-1).
3. **Condition Check**: For each pair of words (words[i], words[j]), the solution checks if `words[i]` is both a prefix and a suffix of `words[j]` using the `startswith` and `endswith` methods. If both conditions are met, the counter is incremented.
4. **Result**: After processing all pairs, the solution returns the total count of valid pairs.

This approach efficiently checks all possible pairs and leverages Python's string methods to determine the prefix and suffix conditions, ensuring correctness and simplicity.
