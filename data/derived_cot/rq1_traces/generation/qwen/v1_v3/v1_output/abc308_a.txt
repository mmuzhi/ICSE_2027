=== ANSWER ===

To solve this problem, we need to verify if a sequence of eight integers meets three specific conditions: being monotonically non-decreasing, all values falling within a specified range, and all values being multiples of 25. 

### Approach
1. **Monotonically Non-Decreasing Check**: Iterate through the sequence and ensure each subsequent element is greater than or equal to the previous one.
2. **Range Check**: Verify that every element in the sequence is between 100 and 675, inclusive.
3. **Multiples of 25 Check**: Confirm that each element in the sequence is divisible by 25.

If all three conditions are satisfied, the answer is "Yes"; otherwise, it's "No".

### Solution Code
```python
def main():
    data = input().split()
    numbers = list(map(int, data))
    
    if len(numbers) != 8:
        print("No")
        return
        
    for i in range(1, 8):
        if numbers[i] < numbers[i-1]:
            print("No")
            return
            
    for num in numbers:
        if num < 100 or num > 675:
            print("No")
            return
            
    for num in numbers:
        if num % 25 != 0:
            print("No")
            return
            
    print("Yes")

if __name__ == '__main__':
    main()
```

### Explanation
1. **Input Handling**: The input is read as a string and split into individual elements, which are then converted to integers.
2. **Sequence Length Check**: The code first checks if there are exactly eight elements. If not, it immediately outputs "No".
3. **Non-Decreasing Check**: The code verifies that each element is not less than the previous one. If any element violates this, it outputs "No".
4. **Range Check**: Each element is checked to ensure it lies between 100 and 675. If any element is outside this range, the output is "No".
5. **Multiples of 25 Check**: The code checks if each element is divisible by 25. If any element fails this check, the output is "No".
6. **Final Output**: If all checks pass, the code outputs "Yes".

This approach efficiently verifies all conditions in sequence, ensuring correctness with minimal computational overhead.
