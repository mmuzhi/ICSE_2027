To solve this problem, we need to minimize the total cost of moving items between boxes such that each box ends up with exactly one item. The cost of moving an item is its weight, and the goal is to find the minimal total cost required for these operations.

### Approach
1. **Problem Analysis**: The problem involves redistributing items across boxes so that each box contains exactly one item. The key observation is that the minimal cost is achieved by leaving as many items as possible in their original boxes (thereby avoiding moving costs) and moving the rest. The optimal strategy is to leave the heaviest item in each box that originally contains at least one item. This maximizes the savings (since moving heavier items incurs higher costs).

2. **Intuition**: For each box that has multiple items, we can leave the heaviest one unmoved. The sum of the weights of these heaviest items across all boxes gives the maximum possible savings. The total cost is then the sum of all item weights minus this savings.

3. **Algorithm Selection**: 
   - **Step 1**: Calculate the total weight of all items.
   - **Step 2**: For each box, find the maximum weight among the items initially in that box.
   - **Step 3**: Sum these maximum weights to get the total savings.
   - **Step 4**: The minimal total cost is the total weight minus the total savings.

4. **Complexity Analysis**: The algorithm processes each item exactly once to compute the maximum weight per box, resulting in a time complexity of O(N), where N is the number of items. This is efficient given the constraint N ≤ 10^5.
