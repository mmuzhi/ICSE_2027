Example 1:

Input: nums = [1,4,3,3,2]
Output: 2

The strictly increasing subarrays of nums are [1], [2], [3], [3], [4], and [1,4].
The strictly decreasing subarrays of nums are [1], [2], [3], [3], [4], [3,2], and [4,3].

Input: nums = [3,2,1]

The strictly increasing subarrays of nums are [3], [2], and [1].
The strictly decreasing subarrays of nums are [3], [2], [1], [3,2], [2,1], and [3,2,1].

To solve this problem, we need to find the length of the longest contiguous subarray in an array of integers that is either strictly increasing or strictly decreasing. The solution involves traversing the array once and keeping track of the current lengths of the longest increasing and decreasing subarrays ending at each position.

### Approach
1. **Initialization**: Start by initializing two variables, `cur_inc` and `cur_dec`, to 1. These variables will store the lengths of the current strictly increasing and decreasing subarrays ending at the current position, respectively.
2. **Traversal**: Iterate through the array starting from the second element. For each element, check if it continues the current increasing or decreasing sequence:
   - If the current element is greater than the previous one, increment `cur_inc`.
   - If the current element is less than the previous one, increment `cur_dec`.
   - If neither condition is met, reset both `cur_inc` and `cur_dec` to 1, as the sequence breaks.
3. **Tracking Maximums**: During each iteration, update the maximum lengths found so far (`max_inc` and `max_dec`) if the current sequence lengths exceed them.
4. **Result**: After processing all elements, the result is the maximum value between `max_inc` and `max_dec`.

This approach efficiently checks all possible strictly increasing and decreasing subarrays in a single pass through the array, making it optimal with a time complexity of O(n) and space complexity of O(1).

### Solution Code
```python
class Solution:
    def longestMonotonicSubarray(self, nums: List[int]) -> int:
        n = len(nums)
        if n == 0:
            return 0
        max_inc = 1
        max_dec = 1
        cur_inc = 1
        cur_dec = 1
        for i in range(1, n):
            if nums[i] > nums[i-1]:

            else:
                cur_inc = 1
            if nums[i] < nums[i-1]:

        return max(max_inc, max_dec)
```

### Explanation
- **Initialization**: The variables `max_inc` and `max_dec` are initialized to 1 because the minimum length of any subarray is 1.
- **Traversal**: The loop starts from the second element (index 1) to the end of the array. For each element, it checks the relationship with the previous element to determine if it continues an increasing or decreasing sequence.
- **Tracking Maximums**: The current lengths of the increasing and decreasing sequences are updated, and the maximum lengths encountered so far are stored in `max_inc` and `max_dec`.
- **Result**: The solution returns the longer of the two maximum lengths found during the traversal, representing the longest strictly increasing or decreasing contiguous subarray in the input array.

This approach efficiently captures the longest subarrays by leveraging a single pass through the array, ensuring optimal performance even for the upper constraint limits.
