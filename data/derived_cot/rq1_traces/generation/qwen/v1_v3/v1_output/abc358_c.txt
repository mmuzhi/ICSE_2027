=== PROMPT ===
You will be given a competitive programming problem.
In AtCoder Land, there are N popcorn stands numbered 1 to N. They have M different flavors of popcorn, labeled 1, 2, \dots, M, but not every stand sells all flavors of popcorn.
Takahashi has obtained information about which flavors of popcorn are sold at each stand. This information is represented by N strings S_1, S_2, \dots, S_N of length M. If the j-th character of S_i is o, it means that stand i sells flavor j of popcorn. If it is x, it means that stand i does not sell flavor j. Each stand sells at least one flavor of popcorn, and each flavor of popcorn is sold at least at one stand.
Takahashi wants to try all the flavors of popcorn but does not want to move around too much. Determine the minimum number of stands Takahashi needs to visit to buy all the flavors of popcorn.

Input

The input is given from Standard Input in the following format:
N M
S_1
S_2
\vdots
S_N

Output

Print the minimum number of stands Takahashi needs to visit to buy all the flavors of popcorn.

Constraints


- N and M are integers.
- 1 \leq N, M \leq 10
- Each S_i is a string of length M consisting of o and x.
- For every i (1 \leq i \leq N), there is at least one o in S_i.
- For every j (1 \leq j \leq M), there is at least one i such that the j-th character of S_i is o.

Sample Input 1

3 5
oooxx
xooox
xxooo

Sample Output 1

2

By visiting the 1st and 3rd stands, you can buy all the flavors of popcorn. It is impossible to buy all the flavors from a single stand, so the answer is 2.

oo
ox
xo

Sample Output 2

1

Sample Input 3

8 6
xxoxxo
xxoxxx
xoxxxx
xxxoxx
xxoooo
xxxxox
xoxxox
oxoxxo

Sample Output 3

3

=== ANSWER ===

To solve this problem, we need to determine the minimum number of popcorn stands Takahashi needs to visit to buy all the flavors of popcorn available. The solution involves checking all possible combinations of stands to find the smallest subset that collectively offers all flavors.

### Approach
1. **Problem Analysis**: The problem requires selecting the smallest subset of stands such that the union of their offered flavors covers all M flavors. Each stand is represented by a bitmask where each bit indicates the presence of a specific flavor.
2. **Bitmask Representation**: Convert each stand's string into an integer bitmask. For example, a string "oooxx" of length 5 would be converted to a bitmask where the first three bits are set (since 'o' represents availability), resulting in the bitmask `0b00011` (or 3 in decimal).
3. **Dynamic Programming (DP) Setup**: Use a DP array where each entry `dp[state]` represents the minimum number of stands needed to achieve the flavor combination represented by `state`. Initialize `dp[0]` to 0 (no stands selected) and all other entries to a large number.
4. **DP State Transition**: For each stand, update the DP array by considering the union of the current state with the stand's bitmask. The transition is `dp[new_state] = min(dp[new_state], dp[state] + 1)`, where `new_state = state | stand_mask`.
5. **Result Extraction**: The answer is found in `dp[full_mask]`, where `full_mask` is `(1 << M) - 1`, representing all flavors being covered.

### Solution Code
```python
import sys

def main():
    data = sys.stdin.read().splitlines()
    if not data:
        return
    
    n_m = data[0].split()
    N = int(n_m[0])
    M = int(n_m[1])
    strings = []
    for i in range(1, 1+N):
        s = data[i].strip()
        strings.append(s)
        
    full_mask = (1 << M) - 1
    flavors = []
    for s in strings:
        mask = 0
        for j in range(M):
            if s[j] == 'o':
                mask |= (1 << j)
        flavors.append(mask)
        
    dp = [10**9] * (1 << M)
    dp[0] = 0
    
    for i in range(N):
        mask_i = flavors[i]
        for state in range((1 << M) - 1, -1, -1):
            if dp[state] == 10**9:
                continue
            new_state = state | mask_i
            if dp[state] + 1 < dp[new_state]:
                dp[new_state] = dp[state] + 1
                
    print(dp[full_mask])

if __name__ == '__main__':
    main()
```

### Explanation
1. **Reading Input**: The input is read from standard input, where the first line contains the number of stands (N) and flavors (M), followed by N strings representing each stand's available flavors.
2. **Bitmask Conversion**: Each string is converted into a bitmask. For each character in the string, if it is 'o', the corresponding bit in the bitmask is set.
3. **DP Initialization**: The DP array is initialized with a large number (10^9) to represent an initially unreachable state, except for `dp[0]` which is set to 0 (no stands selected).
4. **DP State Updates**: For each stand, the DP array is updated in reverse order (from full mask to 0) to ensure each stand is only considered once. The new state is the union of the current state and the stand's bitmask, and the DP value is updated if a smaller number of stands is found.
5. **Result**: The value at `dp[full_mask]` gives the minimum number of stands needed to cover all flavors, which is printed as the result.

This approach efficiently checks all possible combinations of stands using bitmask operations and dynamic programming to find the optimal solution. The complexity is manageable due to the constraints (N and M up to 10), making it feasible for the given problem size.
