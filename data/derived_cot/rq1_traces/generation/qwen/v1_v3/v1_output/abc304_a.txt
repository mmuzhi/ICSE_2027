You will be given a competitive programming problem.
There are N people numbered 1, 2, \ldots, N, sitting in this clockwise order around a round table.
In particular, person 1 is sitting next to person N in the clockwise direction.
For each i = 1, 2, \ldots, N, person i has a name S_i and an age A_i.
Here, no two people have the same name or the same age.
Starting from the youngest person, print the names of all N people in the order of their seating positions in clockwise order.

Input

The input is given from Standard Input in the following format:
N
S_1 A_1
S_2 A_2
\vdots
S_N A_N

Output

Print N lines.
For each i = 1, 2, \ldots, N, the i-th line should contain the name of the person sitting in the i-th position clockwise from the youngest person.

Constraints


- 2 \leq N \leq 100
- N is an integer.
- S_i is a string of length between 1 and 10, consisting of lowercase English letters.
- i \neq j \implies S_i \neq S_j
- 0 \leq A_i \leq 10^9
- A_i is an integer.
- i \neq j \implies A_i \neq A_j

=== ANSWER ===

To solve this problem, we need to determine the seating order of people around a circular table starting from the youngest person and proceeding in a clockwise direction. The solution involves reading the input data, identifying the youngest person, and then printing the names in the order they appear clockwise from that person.

### Approach
1. **Read Input Data**: The first line of input specifies the number of people, N. The subsequent N lines each contain a person's name and age.
2. **Store Data**: Store each person's name and age in a list of tuples for easy access.
3. **Find the Youngest Person**: Iterate through the list to find the index of the person with the smallest age.
4. **Circular Traversal**: Starting from the youngest person's index, traverse the list in a circular manner (using modulo arithmetic) to print each subsequent person's name in clockwise order.

### Solution Code
```python
def main():
    import sys
    data = sys.stdin.read().splitlines()
    n = int(data[0].strip())
    people = []
    for i in range(1, n + 1):
        parts = data[i].split()
        name = parts[0]
        age = int(parts[1])
        people.append((name, age))
    
    min_index = 0
    for i in range(1, n):
        if people[i][1] < people[min_index][1]:
            min_index = i
            
    for i in range(n):
        current_index = (min_index + i) % n
        print(people[current_index][0])

if __name__ == '__main__':
    main()
```

### Explanation
1. **Reading Input**: The input is read all at once for efficiency, especially given the constraints. The first line is the number of people, N.
2. **Storing Data**: Each subsequent line is split into name and age, which are stored as tuples in a list called `people`.
3. **Finding the Youngest**: The loop iterates through the list to find the index of the person with the smallest age. This index is stored in `min_index`.
4. **Circular Printing**: Using modulo arithmetic, the code prints each person's name starting from `min_index` and moving clockwise. The modulo operation ensures the traversal wraps around to the beginning of the list after reaching the end, maintaining the circular order.

This approach efficiently handles the circular nature of the problem and ensures the output starts from the youngest person and proceeds in the correct order. The solution is optimal with a time complexity of O(N), which is suitable for the given constraints.
