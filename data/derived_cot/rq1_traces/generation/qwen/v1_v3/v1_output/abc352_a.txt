=== ANSWER ===

To solve this problem, we need to determine whether a train stops at a given station Z during a travel from station X to station Y on a railway line with N stations. The railway line has two types of trains: inbound trains that move from station 1 to station N in increasing order, and outbound trains that move from station N to station 1 in decreasing order. The solution involves checking if the station Z lies within the segment of the journey from X to Y based on the direction of travel.

### Approach
1. **Problem Analysis**: The key observation is that the direction of travel (inbound or outbound) depends on whether X is less than or greater than Y. If X ≤ Y, the train is inbound and moves from X to Y in increasing order. If X > Y, the train is outbound and moves from X to Y in decreasing order.
2. **Intuition**: For an inbound train (X ≤ Y), station Z must be between X and Y (inclusive) to be part of the journey. For an outbound train (X > Y), station Z must be between Y and X (inclusive) to be part of the journey.
3. **Algorithm Selection**: The solution involves a straightforward conditional check. If the travel is inbound (X ≤ Y), verify if Z is within the range [X, Y]. If the travel is outbound (X > Y), verify if Z is within the range [Y, X]. If either condition is met, print "Yes"; otherwise, print "No".

### Solution Code
```python
def main():
    N, X, Y, Z = map(int, input().split())
    
    if X <= Y:
        if Z >= X and Z <= Y:
            print("Yes")
        else:
            print("No")
    else:
        if Z >= Y and Z <= X:
            print("Yes")
        else:
            print("No")

if __name__ == "__main__":
    main()
```

### Explanation
- **Input Handling**: The code reads four integers N, X, Y, and Z, representing the number of stations, starting station, destination station, and the station to check, respectively.
- **Direction Check**: The code checks if the travel is inbound (X ≤ Y) or outbound (X > Y).
- **Range Check**: For an inbound train, it checks if Z is between X and Y (inclusive). For an outbound train, it checks if Z is between Y and X (inclusive).
- **Output**: Based on the range check, the code prints "Yes" if station Z is part of the journey, otherwise it prints "No".

This approach efficiently determines the presence of station Z in the travel path by leveraging simple conditional checks based on the direction of travel. The solution is optimal with a constant time complexity O(1) as it involves only basic arithmetic and logical operations.
