To solve this problem, we need to determine the number of possible infection sequences for a queue of children, where some are initially infected and the disease spreads to adjacent children in each second. The solution involves breaking down the problem into manageable parts by analyzing the gaps between infected children and calculating the number of ways to infect each gap, then combining these results using combinatorial mathematics.

### Approach
1. **Problem Analysis**: The problem involves a queue of `n` children, some of whom are initially infected. The disease spreads to adjacent children each second, and we need to count all possible sequences in which the remaining children get infected. The key observation is that the infection sequences can be divided into segments (gaps) between infected children. Each gap can be infected independently, and the sequences within each gap must follow specific rules based on whether the gap is at the boundary or in the middle.

2. **Gap Identification**: The healthy children form contiguous segments (gaps) between the infected children. These gaps can be:
   - **Left Gap**: Between the start of the queue and the first infected child.
   - **Middle Gaps**: Between two consecutive infected children.
   - **Right Gap**: Between the last infected child and the end of the queue.

3. **Infection Sequences per Gap**:
   - **Boundary Gaps (Left or Right)**: The infection must start from the only available end and proceed inward, resulting in only one possible sequence for such gaps.
   - **Middle Gaps**: The infection can start from either end, and subsequent infections can be chosen in any order, leading to \(2^{(L-1)}\) sequences for a gap of length \(L\).

4. **Combining Sequences**: The total number of infection sequences is the product of the sequences for each gap, multiplied by the multinomial coefficient that accounts for interleaving the infections from different gaps. The multinomial coefficient is calculated as \( \frac{\text{total\_healthy}!}{\prod (\text{gap\_lengths!})} \), where `total_healthy` is the total number of healthy children.

5. **Modulo Handling**: Since the result can be very large, all calculations are performed modulo \(10^9 + 7\).

class Solution:
    def numberOfSequence(self, n: int, sick: List[int]) -> int:
        mod = 10**9 + 7
