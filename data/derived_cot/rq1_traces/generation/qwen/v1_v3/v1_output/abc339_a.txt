=== ANSWER ===

To solve this problem, we need to find the longest suffix of a given string that does not contain any dots ('.'). The string is guaranteed to contain at least one dot and does not end with a dot. The solution involves splitting the string by dots and selecting the last part of the resulting list, which will be the desired suffix.

### Approach
1. **Problem Analysis**: The problem requires identifying the longest substring at the end of the input string that does not include any dots. The key observation here is that the longest such substring starts immediately after the last dot in the string and extends to the end of the string. 
2. **Insight**: By splitting the string using dots as delimiters, the last element of the resulting list will be the substring that starts right after the last dot and goes to the end of the string. This element is exactly the longest suffix without any dots.
3. **Algorithm Selection**: The solution involves splitting the string by dots and then selecting the last element from the split result. This approach efficiently isolates the required substring without needing complex string manipulation.
4. **Complexity Analysis**: The split operation in Python has a time complexity of O(n), where n is the length of the string. Given the constraints (string length up to 100), this approach is optimal and efficient.

### Solution Code
```python
def main():
    S = input().strip()
    parts = S.split('.')
    print(parts[-1])

if __name__ == '__main__':
    main()
```

### Explanation
1. **Input Handling**: The input string is read and stripped of any leading or trailing whitespace.
2. **Splitting the String**: The string is split into a list of substrings using the dot ('.') as the delimiter. This operation separates the string at every occurrence of a dot.
3. **Selecting the Last Part**: The last element of the resulting list is the substring that starts right after the last dot and extends to the end of the string. This element is printed as the output, which is the longest suffix without any dots.
4. **Example**: For the input "atcoder.jp", the split operation yields ['atcoder', 'jp'], and the last element 'jp' is printed. Similarly, for "translate.google.com", the last element 'com' is printed. For ".z", the last element 'z' is printed, and for "..........txt", the last element 'txt' is printed.

This approach efficiently leverages the string splitting operation to isolate the required substring with minimal computational overhead.
