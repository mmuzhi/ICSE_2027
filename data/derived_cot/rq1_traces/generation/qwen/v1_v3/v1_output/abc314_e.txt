To solve this problem, we need to determine the minimal expected amount of money Takahashi needs to pay to earn at least M points by playing N roulette wheels, each with different costs and outcomes. The solution involves dynamic programming to compute the expected cost for each possible state (current points) and choosing the optimal wheel to play next at each state to minimize the expected cost.

### Approach
1. **Problem Analysis**: The problem requires calculating the minimal expected cost to reach at least M points by strategically choosing which roulette wheel to play next after each play. Each wheel has a cost and a set of outcomes with equal probability. The key is to model this as a dynamic programming problem where the expected cost from each state (current points) is computed based on the outcomes of each wheel.

2. **Dynamic Programming Setup**: We define `dp[m]` as the minimal expected cost from state `m` (current points) to reach at least M points. The solution involves iterating from the highest state (M-1) down to 0, computing the expected cost for each state by considering all possible wheels and their outcomes.

3. **Wheel Processing**: For each wheel, we precompute the count of each outcome (from 0 to M). For each state `m`, we calculate the expected cost if we choose a particular wheel by considering the cost of playing the wheel and the expected cost from the next state (if the play doesn't reach M points). The minimal expected cost across all wheels is stored in `dp[m]`.

4. **Iterative Calculation**: Starting from the highest state (M-1), we compute `dp[m]` for each state `m` by evaluating all wheels. The calculation involves summing the contributions of all positive outcomes (those that don't immediately reach M points) and adjusting for the wheel's cost and probability distribution.

5. **Result Extraction**: The solution is found in `dp[0]`, which gives the minimal expected cost starting from 0 points.

### Solution Code
```python
import sys

def main():
    data = sys.stdin.read().split()
    if not data:
        return
    
    it = iter(data)
    n = int(next(it)); m = int(next(it))
    wheels = []
    for _ in range(n):
        c = int(next(it)); p = int(next(it))
        outcomes = [int(next(it)) for _ in range(p)]
        wheels.append((c, p, outcomes))
    
    if m == 0:
        print(0.0)
        return
        
    dp = [0.0] * m
    
    for state in range(m-1, -1, -1):
        s_max = m - state - 1
        best = float('inf')
        for (c, p, outcomes) in wheels:
            counts = [0] * (m+1)
            for out in outcomes:
                if out <= m:
                    counts[out] += 1
            counts0 = counts[0]
            total = c * p
            for s_val in range(1, s_max+1):
                if s_val <= m:
                    total += counts[s_val] * dp[state+s_val]
            denominator = p - counts0
            if denominator == 0:
                candidate = float('inf')
            else:
                candidate = total / denominator
            if candidate < best:
                best = candidate
        dp[state] = best
        
    print("{0:.15f}".format(dp[0]))

if __name__ == '__main__':
    main()
```
