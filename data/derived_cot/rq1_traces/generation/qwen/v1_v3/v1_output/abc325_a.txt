=== ANSWER ===

To solve this problem, we need to format a person's name by concatenating their surname, a space, and the honorific "san". The input consists of two strings: the surname and the first name, separated by a space. The solution involves reading these two strings and then outputting the surname followed by the honorific "san".

### Approach
1. **Read Input**: The input is read from standard input, which provides two strings separated by a space.
2. **Concatenate Strings**: The surname is combined with the string " san" to form the desired output.
3. **Output Result**: The concatenated string is printed, which includes the surname, a space, and the honorific "san".

This approach efficiently processes the input by splitting it into two parts and then combining them with the required honorific.

### Solution Code
```python
def main():
    data = input().split()
    surname = data[0]
    print(surname + " san")

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The `input()` function reads the entire line of input, and `split()` splits this line into a list of strings based on spaces. For example, if the input is "Takahashi Chokudai", `split()` returns `['Takahashi', 'Chokudai']`.
2. **Extracting Surname**: The first element of the list (index 0) is the surname, which is stored in the variable `surname`.
3. **Output**: The surname is concatenated with the string " san" and printed. For the sample input "Takahashi Chokudai", the output is "Takahashi san".

This solution efficiently handles the input and produces the required output by leveraging simple string operations. The constraints ensure that the input strings are well-formed, simplifying the processing logic.
