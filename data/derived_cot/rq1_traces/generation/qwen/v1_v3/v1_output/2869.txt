class Solution:
    def maxNonDecreasingLength(self, nums1: List[int], nums2: List[int]) -> int:
        n = len(nums1)
        # If the array has only one element, return 1
        if n == 0:
            return 0
        # Initialize for the first element
        prev1_len = 1
        prev1_val = nums1[0]
        prev2_len = 1
        prev2_val = nums2[0]
        max_len = 1
        
        # Iterate from index 1 to n-1
        for i in range(1, n):
            # For the current index, two choices
            curr1_len = 1
            # Check if we can extend from the previous choice of nums1[i-1]
            if nums1[i] >= prev1_val:
                curr1_len = max(curr1_len, prev1_len + 1)
            # Check if we can extend from the previous choice of nums2[i-1]
            if nums1[i] >= prev2_val:
                curr1_len = max(curr1_len, prev2_len + 1)
                
            curr2_len = 1
            if nums2[i] >= prev1_val:
                curr2_len = max(curr2_len, prev1_len + 1)
            if nums2[i] >= prev2_val:
                curr2_len = max(curr2_len, prev2_len + 1)
                
            # Update the max_len
            max_len = max(max_len, curr1_len, curr2_len)
            
            # Update the previous states for the next iteration
            prev1_len, prev1_val = curr1_len, nums1[i]
            prev2_len, prev2_val = curr2_len, nums2[i]
            
        return max_len

To solve this problem, we need to maximize the length of the longest non-decreasing subarray in an array `nums3` formed by choosing elements from either `nums1` or `nums2` at each index. The solution involves dynamic programming to efficiently track the longest non-decreasing subarray ending at each index for both possible choices (from `nums1` or `nums2`).

### Approach
1. **Initialization**: Start by initializing two variables for the first element: one for choosing `nums1[0]` and another for `nums2[0]`. Each variable stores the length of the subarray (initially 1) and the corresponding value.
2. **Iteration**: For each subsequent index, compute the maximum length of the non-decreasing subarray ending at that index for both choices (from `nums1` or `nums2`). This involves checking if the current value can extend the subarray ending at the previous index by comparing it with both previous choices.
3. **Update**: After computing the lengths for the current index, update the previous state variables to these new values for the next iteration.
4. **Tracking Maximum**: Keep track of the maximum length encountered during the iteration.

This approach efficiently computes the solution in O(n) time with O(1) space complexity, as it only needs to maintain the previous state's lengths and values for both choices.

### Solution Code
```python
class Solution:
    def maxNonDecreasingLength(self, nums1: List[int], nums2: List[int]) -> int:
        n = len(nums1)
        if n == 0:
            return 0
        
        prev1_len = 1
        prev1_val = nums1[0]
        prev2_len = 1
        prev2_val = nums2[0]
        max_len = 1
        
        for i in range(1, n):
            curr1_len = 1
            if nums1[i] >= prev1_val:
                curr1_len = max(curr1_len, prev1_len + 1)
            if nums1[i] >= prev2_val:
                curr1_len = max(curr1_len, prev2_len + 1)
            
            curr2_len = 1
            if nums2[i] >= prev1_val:
                curr2_len = max(curr2_len, prev1_len + 1)
            if nums2[i] >= prev2_val:
                curr2_len = max(curr2_len, prev2_len + 1)
            
            max_len = max(max_len, curr1_len, curr2_len)
            
            prev1_len, prev1_val = curr1_len, nums1[i]
            prev2_len, prev2_val = curr2_len, nums2[i]
        
        return max_len
```
