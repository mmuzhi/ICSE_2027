=== ANSWER ===

To solve this problem, we need to determine if a given string of length 6, where the first three characters are 'ABC' and the last three are digits, represents an abbreviation of a contest that was held and concluded on AtCoder before the start of the current contest. 

### Approach
1. **Problem Analysis**: The valid contest abbreviations are structured as ABC followed by three digits. These digits must form numbers from 1 to 314, or the specific numbers 315, 317, 318, ..., up to 349. Notably, the number 316 is excluded from the valid set.
2. **Key Insight**: The solution involves checking if the integer value derived from the last three digits of the input string falls within the specified ranges. The ranges are:
   - Numbers from 1 to 314 (inclusive).
   - The number 315.
   - Numbers from 317 to 349 (inclusive).
3. **Algorithm Selection**: The algorithm involves converting the last three digits of the input string to an integer and then checking if this integer lies within any of the predefined valid ranges. If it does, the answer is "Yes"; otherwise, it's "No".

### Solution Code
```python
def main():
    S = input().strip()
    n = int(S[3:])
    
    if 1 <= n <= 314:
        print("Yes")
    elif n == 315:
        print("Yes")
    elif 317 <= n <= 349:
        print("Yes")
    else:
        print("No")

if __name__ == "__main__":
    main()
```

### Explanation
1. **Input Handling**: The input string is read and stripped of any leading or trailing whitespace.
2. **Digit Extraction**: The last three characters of the string are converted to an integer. This integer represents the contest number.
3. **Range Check**: The integer is checked against the valid ranges:
   - If the number is between 1 and 314 (inclusive), it is valid.
   - If the number is 315, it is valid.
   - If the number is between 317 and 349 (inclusive), it is valid.
4. **Output Result**: Based on the range check, the program prints "Yes" if the number is valid, otherwise it prints "No".

This approach efficiently checks the validity of the contest abbreviation by leveraging simple range checks, ensuring correctness and optimal performance.
