=== ANSWER ===

To solve this problem, we need to reverse a subsequence within a given sequence of integers. The sequence is initially (1, 2, ..., N), and we need to reverse the elements from the L-th to the R-th positions (1-indexed). The solution involves creating the initial sequence, identifying the subsequence to reverse, and then constructing the final sequence with the reversed subsequence.

### Approach
1. **Problem Analysis**: The problem requires reversing a subsequence in a sequence of integers from 1 to N. The reversal operation is performed once on the specified subsequence. The key observation is that the reversal operation affects a contiguous part of the sequence from index L-1 to R-1 (0-indexed) in the initial sequence.
2. **Algorithm Selection**: The solution involves:
   - Generating the initial sequence from 1 to N.
   - Extracting the subsequence from L to R (1-indexed), which translates to indices (L-1) to (R-1) in 0-indexed.
   - Reversing the extracted subsequence.
   - Combining the parts of the sequence before L, the reversed subsequence, and the parts after R to form the final sequence.
3. **Complexity Analysis**: The algorithm runs in O(N) time, where N is the length of the sequence. This is because we generate the sequence, reverse a subsequence, and then combine the parts, all of which are linear operations relative to N.

### Solution Code
```python
def main():
    data = input().split()
    N = int(data[0])
    L = int(data[1])
    R = int(data[2])
    
    sequence = list(range(1, N+1))
    
    if L <= R:
        reversed_sub = sequence[L-1:R][::-1]
        result = sequence[:L-1] + reversed_sub + sequence[R-1+1:]
    else:
        result = sequence
    
    print(" ".join(map(str, result)))

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The input consists of three integers: N (length of the sequence), L (start of the subsequence), and R (end of the subsequence).
2. **Initial Sequence Generation**: The sequence is generated as a list of integers from 1 to N.
3. **Subsequence Reversal**: The subsequence from index L-1 to R-1 (0-indexed) is extracted, reversed, and then reinserted into the original sequence. The parts before L and after R remain unchanged.
4. **Output**: The resulting sequence is printed as a space-separated string.

This approach efficiently handles all edge cases, including when L equals R (no change in the sequence) or when the entire sequence is reversed (L=1 and R=N). The solution is straightforward and leverages Python's slicing and string manipulation capabilities for clarity and efficiency.
