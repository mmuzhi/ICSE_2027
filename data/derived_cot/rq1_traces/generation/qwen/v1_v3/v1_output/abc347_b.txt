=== ANSWER ===

To solve this problem, we need to count the number of distinct non-empty contiguous substrings in a given string. A substring is a contiguous sequence of characters within a string. The solution involves generating all possible substrings from the input string and then using a set to automatically handle duplicates, allowing us to count only unique substrings.

### Approach
1. **Problem Analysis**: The task is to find all unique contiguous substrings of the input string. The key observation here is that any substring can be uniquely identified by its starting and ending indices. By iterating over all possible starting and ending indices, we can generate all potential substrings.
2. **Intuition**: Using a set data structure ensures that only distinct substrings are counted. Each substring is generated by selecting a range of characters from the original string, and each substring is added to the set. The size of the set at the end gives the number of distinct substrings.
3. **Algorithm Selection**: A nested loop approach is used where the outer loop runs from the start of the string to just before the end, and the inner loop runs from the current position of the outer loop to the end of the string. This generates all possible substrings starting at each index.
4. **Complexity Analysis**: The time complexity is O(n^2) where n is the length of the string, since there are O(n^2) possible substrings. The space complexity is also O(n^2) in the worst case, as the set may store up to O(n^2) distinct substrings.

### Solution Code
```python
def main():
    S = input().strip()
    n = len(S)
    substrings = set()
    for i in range(n):
        for j in range(i + 1, n + 1):
            substrings.add(S[i:j])
    print(len(substrings))

if __name__ == '__main__':
    main()
```

### Explanation
1. **Reading Input**: The input string is read from standard input.
2. **Initialization**: A set named `substrings` is initialized to store unique substrings.
3. **Nested Loops**: The outer loop iterates over each character in the string (as the starting index). The inner loop extends the substring from the starting index to each subsequent character (as the ending index).
4. **Substring Generation**: For each pair of indices (i, j), the substring from index i to j-1 is added to the set. The set automatically handles duplicates, ensuring only unique substrings are stored.
5. **Output**: The size of the set, which represents the number of distinct substrings, is printed.

This approach efficiently generates all possible substrings and leverages the properties of a set to count only the unique ones, providing an optimal solution to the problem.
