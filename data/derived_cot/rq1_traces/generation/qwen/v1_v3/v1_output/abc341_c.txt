Output

Constraints

- S_i is a string of length W consisting of . and #.
- There is at least one cell that could be Takahashi's current position.

### Approach
1. **Problem Analysis**: The problem involves simulating the movement of Takahashi's spaceship on a grid for a given sequence of moves. The key observation is that the entire path from the starting cell to the final cell must consist of land cells. The challenge is to efficiently check all possible starting cells (interior land cells) and count the distinct final positions reached after following the move sequence without encountering any sea cells or going out of bounds.

2. **Intuition**: For each interior land cell, we simulate the given sequence of moves. During the simulation, if any move leads to a sea cell or out-of-bounds position, the starting cell is invalid. If the entire sequence is valid, the final cell is recorded. The solution involves checking all such valid starting cells and counting the distinct final positions.

3. **Algorithm Selection**: The algorithm involves iterating over each interior cell (not on the grid's perimeter) that is land. For each such cell, we simulate the move sequence step-by-step, checking at each step if the next cell is within bounds and is land. If all steps are valid, the final cell is added to a set to ensure distinctness.

4. **Complexity Analysis**: The worst-case time complexity is O(H * W * N), where H and W are grid dimensions, and N is the number of moves. Given the constraints (H, W ≤ 500 and N ≤ 500), the maximum operations are around 125 million, which is feasible with optimizations in Python.

### Solution Code
```python
import sys

def main():
    data = sys.stdin.read().splitlines()
    if not data: 
        return
    first_line = data[0].split()
    H = int(first_line[0])
    W = int(first_line[1])
    N = int(first_line[2])
    T = data[1].strip()
    grid = []
    index = 2
    for i in range(H):
        grid.append(data[index].strip())
        index += 1

    dx_map = {'U': -1, 'D': 1, 'L': 0, 'R': 0}
    dy_map = {'U': 0, 'D': 0, 'L': -1, 'R': 1}

    final_cells = set()

    for i in range(1, H-1):
        for j in range(1, W-1):
            if grid[i][j] != '.':
                continue
            x, y = i, j
            valid = True
            for move in T:
                x += dx_map[move]
                y += dy_map[move]
                if x < 0 or x >= H or y < 0 or y >= W or grid[x][y] != '.':
                    valid = False
                    break
            if valid:
                final_cells.add((x, y))

    print(len(final_cells))

if __name__ == '__main__':
    main()
```
