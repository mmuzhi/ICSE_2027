### Approach
1. **Problem Analysis**: The game involves Alice and Bob taking turns to pick flowers from either the clockwise or anti-clockwise direction. The player who picks the last flower wins. The total number of moves is x + y, where x and y are the number of flowers in the clockwise and anti-clockwise directions, respectively. Alice wins if the total number of moves (x + y) is odd.
2. **Insight**: The solution hinges on counting the number of pairs (x, y) where x is in [1, n], y is in [1, m], and x + y is odd. This can be efficiently computed by considering the parity (odd or even) of x and y.
3. **Algorithm Selection**: 
   - Calculate the number of odd and even values in the range [1, n] for x.
   - Similarly, calculate the number of odd and even values in the range [1, m] for y.
   - The number of valid pairs is the sum of the products of the count of odd x values with even y values and the count of even x values with odd y values. This is because an odd sum is achieved when one of the numbers is odd and the other is even.

### Solution Code
```python
class Solution:
    def flowerGame(self, n: int, m: int) -> int:
        x_odd = (n + 1) // 2
        x_even = n // 2
        y_odd = (m + 1) // 2
        y_even = m // 2
        return x_odd * y_even + x_even * y_odd
```

### Explanation
1. **Counting Odd and Even Values**: 
   - For x (from 1 to n), the number of odd values is given by `(n + 1) // 2` and the number of even values by `n // 2`.
   - Similarly, for y (from 1 to m), the number of odd values is `(m + 1) // 2` and the number of even values is `m // 2`.
2. **Calculating Valid Pairs**: 
   - The total number of valid pairs (x, y) where x + y is odd is the sum of:
     - The product of the count of odd x values and even y values.
     - The product of the count of even x values and odd y values.
3. **Efficiency**: The approach efficiently computes the result in constant time O(1) by leveraging simple arithmetic operations, making it suitable for large input sizes up to 10^5.
