You will be given a competitive programming problem.
On a blackboard, there are N sets S_1,S_2,\dots,S_N consisting of integers between 1 and M. Here, S_i = \lbrace S_{i,1},S_{i,2},\dots,S_{i,A_i} \rbrace.
You may perform the following operation any number of times (possibly zero):

- choose two sets X and Y with at least one common element.  Erase them from the blackboard, and write X\cup Y on the blackboard instead.

Here, X\cup Y denotes the set consisting of the elements contained in at least one of X and Y.
Determine if one can obtain a set containing both 1 and M.  If it is possible, find the minimum number of operations required to obtain it.

Input

The input is given from Standard Input in the following format:
N M
A_1
S_{1,1} S_{1,2} \dots S_{1,A_1}
A_2
S_{2,1} S_{2,2} \dots S_{2,A_2}
\vdots
A_N
S_{N,1} S_{N,2} \dots S_{N,A_N}

Output

If one can obtain a set containing both 1 and M, print the minimum number of operations required to obtain it; if it is impossible, print -1 instead.

Constraints


- 1 \le N \le 2 \times 10^5
- 2 \le M \le 2 \times 10^5
- 1 \le \sum_{i=1}^{N} A_i \le 5 \times 10^5
- 1 \le S_{i,j} \le M(1 \le i \le N,1 \le j \le A_i)
- S_{i,j} \neq S_{i,k}(1 \le j < k \le A_i)
- All values in the input are integers.

To solve this problem, we need to determine if we can obtain a set containing both 1 and M by merging sets that share at least one common element. The goal is to find the minimum number of operations required or return -1 if it's impossible.

### Approach
1. **Problem Analysis**: The problem involves merging sets that share common elements to form a set that includes both 1 and M. Each merge operation reduces the number of sets by one. The challenge is to find the shortest sequence of merges that connects 1 and M.

2. **Graph Representation**: We model the problem using a bipartite graph where one partition consists of numbers (1 to M) and the other partition consists of sets. An edge connects a number to a set if the number is present in the set. 

3. **BFS Traversal**: We perform a BFS starting from the number 1. For each number, we explore all sets that contain it. From each set, we explore all numbers contained in it. The BFS keeps track of the number of sets used to reach each number. The solution is derived from the shortest path in this graph, where the distance to M gives the number of sets used, and the operations required is this distance minus one.

4. **Complexity Analysis**: The BFS runs in O(N + M + total elements) time, where N is the number of sets and M is the range of numbers. The total elements is the sum of all set sizes, which is up to 500,000. This is efficient given the constraints.

### Solution Code
```python
import sys
from collections import deque

def main():
    data = sys.stdin.read().splitlines()
    if not data:
        print(-1)
        return
        
    parts = data[0].split()
    if not parts:
        print(-1)
        return
        
    n = int(parts[0])
    m = int(parts[1])
    
    sets = []
    index = 1
    for i in range(n):
        if index >= len(data):
            break
        a_line = data[index].strip()
        index += 1
        if a_line == '':
            continue
        a = int(a_line)
        s_line = data[index].split()
        index += 1
        s_list = list(map(int, s_line))
        sets.append(s_list)
        
    graph_num = [[] for _ in range(m+1)]
    
    for i, s in enumerate(sets):
        for num in s:
            if 1 <= num <= m:
                graph_num[num].append(i)
                
    if m < 1:
        print(-1)
        return
        
    dist_num = [-1] * (m+1)
    dist_set = [-1] * (n)
    
    q = deque()
    dist_num[1] = 0
    q.append(('num', 1))
    
    while q:
        type_node, node = q.popleft()
        if type_node == 'num':
            for set_index in graph_num[node]:
                if dist_set[set_index] == -1:
                    dist_set[set_index] = dist_num[node] + 1
                    q.append(('set', set_index))
        else:
            s = sets[node]
            for num in s:
                if num <= m and dist_num[num] == -1:
                    dist_num[num] = dist_set[node]
                    q.append(('num', num))
                    
    if dist_num[m] == -1:
        print(-1)
    else:
        print(dist_num[m] - 1)

if __name__ == '__main__':
    main()
```

3. **BFS Initialization**: The BFS starts from the number 1. For each number, all connected sets are processed, and for each set, all connected numbers are processed.
