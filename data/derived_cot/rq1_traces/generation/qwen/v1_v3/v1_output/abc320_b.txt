   There are two types of palindromes: odd and even.

   For each center (each index for odd, and each gap between i and i+1 for even), expand as long as the characters at the left and right are equal.

   Steps:

   n = len(S)

                left -= 1
                right += 1

                left -= 1
                right += 1

=== ANSWER ===

To solve this problem, we need to find the maximum length of a contiguous substring in a given string that is a palindrome. A palindrome is a string that reads the same forwards and backwards. The solution involves checking all possible substrings of the string to determine if they are palindromes and keeping track of the longest one found.

### Approach
1. **Problem Analysis**: The task is to find the longest contiguous substring that is a palindrome. The solution must efficiently check all possible substrings without resorting to a brute-force method that would be too slow for larger strings.
2. **Insight**: The solution leverages the fact that palindromes can be of odd or even length. For each character in the string (considering it as the center of a potential palindrome for odd-length palindromes), we expand outwards to check for the longest palindrome centered at that character. Similarly, for even-length palindromes, we consider the center between two identical characters and expand outwards.
3. **Algorithm Selection**: The chosen algorithm is the "expand around centers" method, which efficiently checks all possible palindromic substrings by expanding around each character and each pair of characters. This method has a time complexity of O(n^2) and a space complexity of O(1), which is optimal for the given constraints (string length up to 100).

### Solution Code
```python
S = input().strip()
n = len(S)
max_len = 1

# Odd length palindromes
for i in range(n):
    l, r = i, i
    while l >= 0 and r < n and S[l] == S[r]:
        if r - l + 1 > max_len:
            max_len = r - l + 1
        l -= 1
        r += 1

# Even length palindromes
for i in range(n-1):
    l, r = i, i+1
    while l >= 0 and r < n and S[l] == S[r]:
        if r - l + 1 > max_len:
            max_len = r - l + 1
        l -= 1
        r += 1

print(max_len)
```

### Explanation
1. **Initialization**: The input string is read, and the maximum length of the palindrome (`max_len`) is initialized to 1, as the smallest palindrome is a single character.
2. **Odd-Length Palindromes**: For each character in the string, treat it as the center of a potential palindrome and expand outwards (left and right) as long as the characters match. Update `max_len` if a longer palindrome is found.
3. **Even-Length Palindromes**: For each pair of adjacent characters, treat the space between them as the center of a potential even-length palindrome and expand outwards. Again, update `max_len` if a longer palindrome is found.
4. **Output**: After checking all possible centers and expansions, the maximum length of a palindromic substring found is printed.

This approach efficiently checks all possible palindromic substrings by leveraging the symmetry around each center, ensuring optimal performance for the given problem constraints.
