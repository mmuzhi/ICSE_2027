You are given a simple undirected graph consisting of N vertices and M edges.
For i = 1, 2, \ldots, M, the i-th edge connects vertices u_i and v_i.
Also, for i = 1, 2, \ldots, N, vertex i is assigned a positive integer W_i, and there are A_i pieces placed on it.
As long as there are pieces on the graph, repeat the following operation:

- First, choose and remove one piece from the graph, and let x be the vertex on which the piece was placed.

- 1 \leq M \leq \min \lbrace N(N-1)/2, 5000 \rbrace
- 1 \leq u_i, v_i \leq N
- u_i \neq v_i
- i \neq j \implies \lbrace u_i, v_i \rbrace \neq \lbrace u_j, v_j \rbrace
- 1 \leq W_i \leq 5000

To solve this problem, we need to determine the maximum number of operations that can be performed on a graph where each operation involves removing a piece from a vertex and distributing it to its adjacent vertices under specific weight constraints. The key insight is to model each vertex's breaking potential using dynamic programming and a knapsack approach to maximize the number of operations.

### Approach
1. **Problem Analysis**: The problem involves a graph where each vertex has a weight and a count of pieces. Each operation allows removing a piece from a vertex and distributing it to adjacent vertices if the sum of their weights is less than the vertex's weight. The goal is to maximize the number of such operations.

2. **Intuition**: Each vertex's breaking potential depends on its neighbors' breaking potentials. The optimal strategy involves breaking a vertex into the subset of neighbors that maximizes the total operations from the resulting pieces, subject to the weight constraint.

3. **Algorithm Selection**: We use a fixed-point iteration approach combined with a knapsack problem solution. For each vertex, we compute the maximum operations by considering all possible subsets of its neighbors that meet the weight constraint. The solution involves:
   - **Graph Construction**: Build an adjacency list for the graph.
   - **Initialization**: Start with each vertex having a base value of 1 (the operation itself).
   - **Fixed-Point Iteration**: For each vertex, compute the best subset of neighbors (using knapsack) to maximize operations, updating neighbors' values if improvements are found.
   - **Result Calculation**: Sum the product of initial piece counts and each vertex's final operation count.

4. **Complexity Analysis**: The algorithm processes each vertex and its neighbors, with knapsack operations proportional to the product of vertex weight and degree. Given constraints (N ≤ 5000, M ≤ 5000), the total operations are manageable due to the graph's sparsity.

### Solution Code
```python
import sys
from collections import deque

def main():
    data = sys.stdin.read().split()
    if not data:
        return
    it = iter(data)
    n = int(next(it)); m = int(next(it))
    graph = [[] for _ in range(n+1)]
    for _ in range(m):
        u = int(next(it)); v = int(next(it))
        graph[u].append(v)
        graph[v].append(u)
    
    weights = [0] * (n+1)
    for i in range(1, n+1):
        weights[i] = int(next(it))
    
    A = [0] * (n+1)
    for i in range(1, n+1):
        A[i] = int(next(it))
    
    if sum(A) == 0:
        print(0)
        return
        
    f = [0] * (n+1)
    for i in range(1, n+1):
        f[i] = 1
        
    q = deque()
    for i in range(1, n+1):
        q.append(i)
        
    while q:
        i = q.popleft()
        items = []
        for j in graph[i]:
            items.append((weights[j], f[j]))
            
        capacity = weights[i] - 1
        
        if capacity < 0:
            best_value = 0
        else:
            dp = [0] * (capacity+1)
            for w_val, v_val in items:
                for cap in range(capacity, w_val-1, -1):
                    if dp[cap] < dp[cap - w_val] + v_val:
                        dp[cap] = dp[cap - w_val] + v_val
            best_value = max(dp) if dp else 0
            
        new_f = 1 + best_value
        if new_f > f[i]:
            f[i] = new_f
            for j in graph[i]:
                q.append(j)
                
    total_operations = 0
    for i in range(1, n+1):
        total_operations += A[i] * f[i]
        
    print(total_operations)

if __name__ == '__main__':
    main()
```
