=== ANSWER ===

To solve this problem, we need to determine if a given string of length 3 matches any of the specified strings: "ACE", "BDF", "CEG", "DFA", "EGB", "FAC", or "GBD". The solution involves checking the input string against these predefined strings and printing "Yes" if there's a match, otherwise "No".

### Approach
1. **Problem Analysis**: The problem requires checking if the input string matches any of the seven specified strings. The solution can be efficiently implemented by using a set for quick membership testing.
2. **Intuition**: By storing all valid strings in a set, we can leverage the constant-time lookup property of sets to determine if the input string is one of the valid strings.
3. **Algorithm Selection**: The algorithm involves reading the input string, checking its presence in a predefined set of valid strings, and printing the result accordingly.
4. **Complexity Analysis**: The solution operates in O(1) time for the lookup in the set, making it very efficient for the given constraints.

### Solution Code
```python
valid_strings = {"ACE", "BDF", "CEG", "DFA", "EGB", "FAC", "GBD"}

S = input().strip()
if S in valid_strings:
    print("Yes")
else:
    print("No")
```

### Explanation
1. **Reading Input**: The input string is read from standard input and stripped of any leading or trailing whitespace.
2. **Set Membership Check**: The input string is checked against a set containing all valid strings. This set allows for O(1) time complexity for membership checks.
3. **Output Result**: Based on the membership check, "Yes" is printed if the input string is found in the set; otherwise, "No" is printed.

This approach efficiently solves the problem by leveraging set operations to quickly determine the validity of the input string.
