To solve this problem, we need to read a sequence of integers from standard input until we encounter a zero, which signifies the end of the input. The integers are read in the order they appear, and we need to output them in reverse order, starting from the last non-zero integer down to the first one, followed by the zero.

### Approach
1. **Reading Input**: The input consists of integers one per line, ending with a zero. We read each line until we encounter the zero or reach the end of the input.
2. **Storing Values**: We store each integer in a list until we read the zero. The zero is included in the list and marks the end of the sequence.
3. **Reversing the List**: Once the input is complete, we reverse the list to print the integers in the required order (from last to first).
4. **Output**: Each element of the reversed list is printed on a new line.

### Solution Code
```python
numbers = []
while True:
    try:
        data = input().strip()
        if data == '0':
            numbers.append(0)
            break
        else:
            numbers.append(int(data))
    except EOFError:
        break

for num in reversed(numbers):
    print(num)
```

### Explanation
1. **Reading Input**: The loop continuously reads each line of input. If the line is '0', it appends zero to the list and breaks the loop. For any other line, it converts the string to an integer and appends it to the list.
2. **Handling End of Input**: The loop also handles the end of file (EOF) scenario, which breaks the loop if no more input is available.
3. **Reversing the List**: The `reversed(numbers)` function reverses the list, allowing us to iterate from the last element to the first.
4. **Printing the Result**: Each element of the reversed list is printed on a new line, resulting in the desired output order.

This approach efficiently reads and processes the input, ensuring the correct order of output by reversing the list before printing. The solution handles all edge cases, including a single zero input, as specified in the problem constraints.
