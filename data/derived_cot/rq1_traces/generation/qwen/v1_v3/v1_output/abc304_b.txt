To solve this problem, we need to truncate a given integer N based on its magnitude. The truncation involves setting the last few digits to zero, depending on how many digits the number has. The approach varies based on the number of digits in N, from leaving the number unchanged if it has three or fewer digits, to truncating more digits if it has more digits.

### Approach
1. **Problem Analysis**: The problem requires truncating digits of N based on its digit count. For numbers with up to 3 digits, no truncation is needed. For numbers with 4 or more digits, the truncation involves setting the last (d-3) digits to zero, where d is the number of digits in N.
2. **Intuition**: The key observation is that the number of digits to truncate (set to zero) is determined by the total number of digits in N minus 3. For example, a 4-digit number requires truncating the last 1 digit, a 5-digit number requires truncating the last 2 digits, and so on.
3. **Algorithm Selection**: The algorithm involves:
   - Converting N to a string to determine its digit count.
   - If the digit count is 3 or fewer, output N as-is.
   - Otherwise, calculate the base as 10 raised to the power of (digit count - 3). The result is obtained by performing integer division of N by this base and then multiplying the quotient by the base, effectively setting the last (digit count - 3) digits to zero.
4. **Complexity Analysis**: The solution involves basic arithmetic operations and string manipulation, which are efficient with a time complexity of O(d), where d is the number of digits in N. This is optimal for the given constraints (N up to 10^9 - 1).

### Solution Code
```python
def main():
    N = int(input().strip())
    if N == 0:
        print(0)
    else:
        d = len(str(N))
        if d <= 3:
            print(N)
        else:
            base = 10 ** (d - 3)
            result = (N // base) * base
            print(result)

if __name__ == '__main__':
    main()
```
