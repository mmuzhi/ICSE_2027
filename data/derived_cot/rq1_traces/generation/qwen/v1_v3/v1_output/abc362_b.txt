=== ANSWER ===

To solve this problem, we need to determine whether a triangle formed by three given points in the xy-plane is a right triangle. The solution involves checking if any of the angles formed by the points is a right angle, which can be efficiently done using the Pythagorean theorem.

### Approach
1. **Problem Analysis**: The problem requires checking if the triangle formed by three non-collinear points is a right triangle. The key insight is that in a right triangle, the square of the length of the hypotenuse (the side opposite the right angle) is equal to the sum of the squares of the other two sides.
2. **Intuition**: For each vertex of the triangle, we can check if the sum of the squares of the two adjacent sides equals the square of the opposite side. If any of these conditions is true, the triangle is a right triangle.
3. **Algorithm Selection**: Calculate the squared distances between each pair of points. Then, verify if the sum of any two squared distances equals the third squared distance. If so, the triangle is right-angled.
4. **Complexity Analysis**: The solution involves basic arithmetic operations and comparisons, making it efficient with a constant time complexity O(1) since the number of operations is fixed regardless of input size.

### Solution Code
```python
def main():
    xA, yA = map(int, input().split())
    xB, yB = map(int, input().split())
    xC, yC = map(int, input().split())
    
    d1 = (xB - xA)**2 + (yB - yA)**2
    d2 = (xC - xB)**2 + (yC - yB)**2
    d3 = (xA - xC)**2 + (yA - yC)**2
    
    if (d1 + d2 == d3) or (d1 + d3 == d2) or (d2 + d3 == d1):
        print("Yes")
    else:
        print("No")

if __name__ == '__main__':
    main()
```

### Explanation
1. **Reading Input**: The code reads three lines of input, each containing the coordinates of one of the points.
2. **Squared Distance Calculation**: The squared distances between each pair of points are calculated. For example, `d1` is the squared distance between points A and B.
3. **Right Triangle Check**: The code checks if the sum of any two squared distances equals the third squared distance. If this condition is met, the triangle is right-angled, and the output is "Yes"; otherwise, it outputs "No".
4. **Output Result**: Based on the check, the result is printed.

This approach efficiently leverages the Pythagorean theorem to determine the presence of a right angle in the triangle, ensuring correctness with minimal computational overhead.
