To solve this problem, we need to count the number of days in a year where the date has a repdigit. A repdigit date is defined as a day where both the month and the day have all digits the same. 

### Approach
1. **Problem Analysis**: The problem requires checking each month and day in the year to determine if both the month and day are repdigits. A repdigit is a number where all digits are the same (e.g., 1, 22, 333). The solution involves:
   - Identifying all repdigit months (from 1 to N).
   - For each repdigit month, checking the corresponding days (from 1 to D_i) to see if they are repdigits with the same digit as the month.
   
2. **Intuition**: For each month, if the month number is a repdigit, we then count all days in that month that are also repdigits with the same digit. For example, if the month is 1 (digit '1'), we count days like 1 and 11 (if they exist in the month).

3. **Algorithm Selection**: 
   - **Precompute Repdigits**: For each digit from 1 to 9, precompute all repdigit numbers up to 100 (since days can be up to 100). This includes single-digit numbers (1-9) and two-digit numbers (11, 22, ..., 99).
   - **Check Months and Days**: For each month, check if it is a repdigit. If it is, use the precomputed set to count valid days (days that are repdigits with the same digit as the month and do not exceed the month's day limit).

4. **Complexity Analysis**: The algorithm efficiently checks each month and day by leveraging precomputed sets. The time complexity is O(N * M), where N is the number of months and M is the average number of days per month. Given the constraints (N ≤ 100 and days ≤ 100), this is optimal.

### Solution Code
```python
def main():
    N = int(input().strip())
    D_list = list(map(int, input().split()))
    
    repdigit_by_digit = {}
    for d in '123456789':
        num1 = int(d)
        num2 = int(d * 2)
        repdigit_by_digit[d] = {num1, num2}
    
    total = 0
    for i in range(N):
        month_num = i + 1
        s = str(month_num)
        if s.count(s[0]) != len(s):
            continue
        d_char = s[0]
        if d_char not in repdigit_by_digit:
            continue
        day_set = repdigit_by_digit[d_char]
        count_days = 0
        for day_val in day_set:
            if day_val <= D_list[i]:
                count_days += 1
        total += count_days
        
    print(total)

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The code reads the number of months (N) and the list of days in each month (D_list).
2. **Precompute Repdigits**: For each digit from 1 to 9, it stores the corresponding single-digit and two-digit repdigit numbers in a dictionary (`repdigit_by_digit`).
3. **Processing Each Month**: For each month, it checks if the month number is a repdigit by verifying all its digits are the same.
4. **Counting Valid Days**: If the month is a repdigit, it retrieves the set of repdigit days with the same digit and counts those days that are valid (i.e., within the month's day limit).
5. **Output Result**: The total count of repdigit dates is printed.

This approach efficiently checks each month and day, leveraging precomputation to minimize redundant checks, ensuring optimal performance for the given constraints.
