You will be given a competitive programming problem.
Takahashi has N teeth, one in each of the holes numbered 1, 2, \dots, N.
Dentist Aoki will perform Q treatments on these teeth and holes.
In the i-th treatment, hole T_i is treated as follows:

- If there is a tooth in hole T_i, remove the tooth from hole T_i.
- If there is no tooth in hole T_i (i.e., the hole is empty), grow a tooth in hole T_i.

After all treatments are completed, how many teeth does Takahashi have?

Input

The input is given from Standard Input in the following format:
N Q
T_1 T_2 \dots T_Q

Output

Print the number of teeth as an integer.

Constraints


- All input values are integers.
- 1 \le N, Q \le 1000
- 1 \le T_i \le N

Sample Input 1

30 6
2 9 18 27 18 9

Sample Output 1

28

The final count of teeth is 28.

=== ANSWER ===

To solve this problem, we need to simulate the process of dental treatments performed on Takahashi's teeth over a series of operations. The goal is to determine the number of teeth Takahashi has after all treatments are completed. 

### Approach
1. **Initialization**: Start with `N` teeth, meaning all holes from 1 to `N` initially contain a tooth.
2. **Processing Treatments**: For each treatment in the sequence:
   - If the specified hole contains a tooth, remove it (decrement the tooth count).
   - If the hole is empty, grow a tooth (increment the tooth count).
3. **Tracking State**: Use a boolean array to keep track of which holes currently have a tooth. This allows efficient checking and updating of the hole's state during each treatment.
4. **Result Calculation**: After processing all treatments, the count of teeth is simply the number of holes that still contain a tooth.

### Solution Code
```python
def main():
    import sys
    data = sys.stdin.read().split()
    n = int(data[0])
    q = int(data[1])
    treatments = list(map(int, data[2:2+q]))
    
    teeth = [True] * (n + 1)
    count = n
    
    for t in treatments:
        if teeth[t]:
            teeth[t] = False
            count -= 1
        else:
            teeth[t] = True
            count += 1
            
    print(count)

if __name__ == "__main__":
    main()
```

### Explanation
- **Initialization**: The code reads the initial number of teeth `N` and the number of treatments `Q`. It then reads the sequence of treatments.
- **Tracking Teeth**: A boolean array `teeth` is initialized to keep track of which holes have teeth. The initial count of teeth is `N`.
- **Processing Treatments**: For each treatment in the sequence:
  - If the hole specified by the treatment has a tooth (`teeth[t]` is `True`), the tooth is removed (`teeth[t]` is set to `False`), and the count is decremented.
  - If the hole is empty (`teeth[t]` is `False`), a tooth is grown (`teeth[t]` is set to `True`), and the count is incremented.
- **Output**: After processing all treatments, the final count of teeth is printed.

This approach efficiently tracks the state of each hole through each treatment, ensuring accurate results by maintaining a running count of teeth. The solution handles all edge cases, including repeated treatments on the same hole, by dynamically updating the hole's state and the tooth count.
