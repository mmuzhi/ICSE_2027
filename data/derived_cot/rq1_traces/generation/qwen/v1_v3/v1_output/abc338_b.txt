=== ANSWER ===

To solves this problem, we need to find the character that appears most frequently in a given string. If multiple characters have the same maximum frequency, we should select the one that comes first in alphabetical order.

### Approach
1. **Count Character Frequencies**: We first traverse the input string and count the occurrences of each character using a dictionary. This allows us to efficiently track how many times each character appears.
2. **Determine Maximum Frequency**: After counting all characters, we identify the highest frequency value from the dictionary values.
3. **Select Earliest Character**: Among all characters that have this maximum frequency, we select the one that is lexicographically smallest (i.e., comes first in alphabetical order). This is done by finding the minimum character in the list of characters that have the maximum frequency.

### Solution Code
```python
def main():
    S = input().strip()
    count_dict = {}
    for char in S:
        count_dict[char] = count_dict.get(char, 0) + 1
        
    max_count = max(count_dict.values())
    candidates = [char for char, count in count_dict.items() if count == max_count]
    answer = min(candidates)
    print(answer)

if __name__ == '__main__':
    main()
```

### Explanation
1. **Reading Input**: The input string is read from standard input and stripped of any leading or trailing whitespace.
2. **Counting Characters**: A dictionary is used to count the occurrences of each character in the string. For each character encountered, its count in the dictionary is incremented by 1.
3. **Finding Maximum Frequency**: The maximum value in the dictionary's values is determined, which represents the highest frequency of any character in the string.
4. **Selecting the Earliest Character**: All characters with this maximum frequency are collected into a list. The lexicographically smallest character from this list is selected using the `min` function, ensuring the correct character is chosen if there are ties.
5. **Output the Result**: The selected character is printed as the result.

This approach efficiently handles the problem by leveraging dictionary operations for counting and simple list operations to determine the result, ensuring optimal performance even for the upper constraint of the problem.
