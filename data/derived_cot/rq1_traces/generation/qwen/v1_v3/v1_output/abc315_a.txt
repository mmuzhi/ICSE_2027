=== ANSWER ===

To solve this problem, we need to remove all occurrences of the vowels 'a', 'e', 'i', 'o', 'u' from a given string and print the resulting string. The solution involves iterating through each character in the string and including only those characters that are not vowels in the output.

### Approach
1. **Problem Analysis**: The task is to filter out all vowels from the input string. The string consists of lowercase English letters, and there is at least one non-vowel character present. The solution requires processing each character in the string and constructing a new string that excludes all vowels.
2. **Intuition**: By checking each character against a predefined set of vowels, we can determine whether to include it in the result or not. This can be efficiently done using a loop or a list comprehension.
3. **Algorithm Selection**: A straightforward approach is to iterate through each character in the string and build a new string by including only those characters that are not vowels. This method ensures that all vowels are removed while preserving the order of the remaining characters.
4. **Complexity Analysis**: The algorithm processes each character exactly once, resulting in a time complexity of O(n), where n is the length of the input string. The space complexity is also O(n) for the output string.

### Solution Code
```python
vowels = "aeiou"
S = input().strip()
result = ''.join(char for char in S if char not in vowels)
print(result)
```

### Explanation
1. **Initialization**: The set of vowels "aeiou" is defined for easy checking.
2. **Input Handling**: The input string `S` is read from standard input and stripped of any leading or trailing whitespace (though the problem constraints ensure it's unnecessary).
3. **String Construction**: A list comprehension is used to iterate over each character in `S`. For each character, if it is not found in the vowels set, it is included in the new string. The `join` method combines these characters into a single string.
4. **Output**: The resulting string, which contains only consonants, is printed.

This approach efficiently removes all vowels from the input string by leveraging a simple filtering mechanism, ensuring clarity and optimal performance.
