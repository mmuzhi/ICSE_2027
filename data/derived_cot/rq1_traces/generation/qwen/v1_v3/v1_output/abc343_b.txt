=== PROMPT ===
You will be given a competitive programming problem.
There is a simple undirected graph G with N vertices labeled with numbers 1, 2, \ldots, N.
You are given the adjacency matrix (A_{i,j}) of G. That is, G has an edge connecting vertices i and j if and only if A_{i,j} = 1.
For each i = 1, 2, \ldots, N, print the numbers of the vertices directly connected to vertex i in ascending order.
Here, vertices i and j are said to be directly connected if and only if there is an edge connecting vertices i and j.

Input

The input is given from Standard Input in the following format:
N
A_{1,1} A_{1,2} \ldots A_{1,N}
A_{2,1} A_{2,2} \ldots A_{2,N}
\vdots
A_{N,1} A_{N,2} \ldots A_{N,N}

Output

Print N lines.
The i-th line should contain the numbers of the vertices directly connected to vertex i in ascending order, separated by a space.

Constraints


- 2 \leq N \leq 100
- A_{i,j} \in \lbrace 0,1 \rbrace
- A_{i,i} = 0
- A_{i,j} = A_{j,i}
- All input values are integers.

=== ANSWER ===

To solve this problem, we need to process an adjacency matrix of an undirected graph and output, for each vertex, its neighbors in ascending order. The adjacency matrix is given as input, where each entry indicates whether there is an edge between two vertices.

### Approach
1. **Read Input**: The first line of input is the number of vertices, N. The subsequent N lines represent the adjacency matrix.
2. **Process Each Vertex**: For each vertex (0-indexed), examine its corresponding row in the adjacency matrix. For each entry in the row that is 1, note the connected vertex (converting from 0-indexed to 1-indexed).
3. **Sort Neighbors**: Collect all connected vertices for the current vertex and sort them in ascending order.
4. **Output Results**: For each vertex, print its sorted list of neighbors. If a vertex has no neighbors, print an empty line.

### Solution Code
```python
n = int(input().strip())
graph = []
for _ in range(n):
    row = list(map(int, input().split()))
    graph.append(row)

for i in range(n):
    neighbors = []
    for j in range(n):
        if graph[i][j] == 1:
            neighbors.append(j + 1)
    neighbors.sort()
    if neighbors:
        print(' '.join(map(str, neighbors)))
    else:
        print()
```

### Explanation
1. **Reading Input**: The code starts by reading the number of vertices, N. Then, it reads the N lines of the adjacency matrix and stores them in a list of lists called `graph`.
2. **Processing Vertices**: For each vertex (from 0 to N-1), the code checks each entry in its corresponding row. If the entry is 1, it means there is an edge to the vertex represented by the column index (converted to 1-indexed).
3. **Sorting Neighbors**: The indices of the connected vertices are collected into a list and sorted in ascending order.
4. **Output**: The sorted list of neighbors is printed for each vertex. If no neighbors are found, an empty line is printed. This ensures the output format matches the problem requirements, handling all edge cases including graphs with no edges. 

This approach efficiently processes the adjacency matrix to generate the required output by leveraging simple loops and sorting, ensuring correctness and clarity.
