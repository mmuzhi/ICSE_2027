To solve this problem, we need to determine if a given airport code T can be derived from a string S using one of two specified methods. The first method involves taking a subsequence of three characters from S (not necessarily contiguous) and converting them to uppercase to form T. The second method involves taking a subsequence of two characters from S, converting them to uppercase, and appending 'X' to form T.

### Approach
1. **Problem Analysis**: The problem requires checking two distinct conditions based on the third character of T. If the third character is 'X', we check both methods; otherwise, we only check the first method.
2. **Key Insight**: For the first method, we need to verify if the lowercase version of T is a subsequence of S. For the second method, we need to verify if the first two characters of the lowercase version of T form a subsequence in S, and the third character is 'X'.
3. **Algorithm Selection**: The solution involves checking subsequences in S for the given pattern derived from T. The helper function `is_subsequence` efficiently checks if a string `t` is a subsequence of S by traversing S once.
4. **Complexity Analysis**: The algorithm processes S in linear time for each subsequence check. Since we perform at most two checks (when T's third character is 'X'), the overall complexity is O(n), where n is the length of S, which is optimal for the given constraints.

### Solution Code
```python
def is_subsequence(S, t):
    t_index = 0
    for char in S:
        if t_index < len(t) and char == t[t_index]:
            t_index += 1
    return t_index == len(t)

def main():
    S = input().strip()
    T_str = input().strip()
    
    if T_str[2] != 'X':
        t_lower = T_str.lower()
        if is_subsequence(S, t_lower):
            print("Yes")
        else:
            print("No")
    else:
        t_lower = T_str.lower()
        if is_subsequence(S, t_lower) or is_subsequence(S, t_lower[:2]):
            print("Yes")
        else:
            print("No")

if __name__ == "__main__":
    main()
```

1. **Helper Function `is_subsequence`**: This function checks if the string `t` is a subsequence of `S` by iterating through `S` and matching characters in `t` in order.
2. **Main Logic**: 
   - **Case 1 (T's third character is not 'X')**: Convert T to lowercase and check if it is a subsequence of S. If yes, print "Yes"; otherwise, "No".
   - **Case 2 (T's third character is 'X')**: Check if either the entire lowercase T or its first two characters form a subsequence in S. If either condition is met, print "Yes"; otherwise, "No".
3. **Efficiency**: The solution efficiently checks the required conditions in linear time, making it suitable for large input sizes as specified in the problem constraints.
