So the final algorithm:

1. Count the frequencies of each character (ignore characters not present).
2. Let max_freq = max(frequencies) (if there are no characters, then 0)
3. Initialize best = a large number (or the total length, which is the worst-case deletion)
4. For M from 0 to max_freq (inclusive):
   a. deletions = 0
   b. For each frequency f in the list of frequencies:
        if f < M - k:
            deletions += f  (delete all)
        else:
            deletions += max(0, f - M)  (keep and set to M, delete the excess)
   c. best = min(best, deletions)
5. Return best

But note: M - k might be negative, and that's okay.

Complexity: O(max_freq * number_of_distinct_characters). The max_freq can be up to 10^5 and distinct characters up to 26, so 10^5 * 26 = 2.6e6, which is acceptable in Python.

class Solution:
    def minimumDeletions(self, word: str, k: int) -> int:
        # Count frequencies
        from collections import Counter
        freqs = list(Counter(word).values())
        if not freqs:
            return 0
        max_freq = max(freqs)
        best = float('inf')
        # Iterate M from 0 to max_freq
        for M in range(0, max_freq+1):
            deletions = 0

To solve this problem, we need to determine the minimum number of character deletions required to make a given string k-special. A string is considered k-special if the absolute difference between the frequencies of any two characters in the string is at most k. 

### Approach
1. **Frequency Counting**: First, we count the frequency of each character in the string. This helps us understand how many times each character appears.
2. **Iterate Over Possible Maximum Frequencies**: For each possible maximum frequency (M) from 0 to the highest frequency of any character in the string, we calculate the number of deletions required to adjust the frequencies of all characters such that:
   - Characters with frequencies below (M - k) must be entirely removed (since their frequencies cannot be increased).
   - Characters with frequencies above M must have their frequencies reduced to M (to ensure no frequency exceeds M).
3. **Minimize Deletions**: For each possible M, we compute the total deletions required and keep track of the minimum deletions encountered during the iteration.

### Solution Code
```python
from collections import Counter

class Solution:
    def minimumDeletions(self, word: str, k: int) -> int:
        freqs = list(Counter(word).values())
        max_freq = max(freqs) if freqs else 0
        best = float('inf')
        for M in range(0, max_freq + 1):
            current = 0
            for f in freqs:
                if f < M - k:
                    current += f
                else:
                    current += max(0, f - M)
            best = min(best, current)
        return best
```

### Explanation
1. **Frequency Counting**: The `Counter` from the `collections` module is used to count the occurrences of each character in the string. This gives us a list of frequencies.
2. **Iterating Over M**: For each possible maximum frequency (M) from 0 to the highest frequency in the string, we check how many deletions are needed to adjust the frequencies of all characters:
   - If a character's frequency is less than (M - k), it must be entirely removed (adding its entire frequency to the deletion count).
   - Otherwise, the character's frequency is reduced to M (if it exceeds M), and the difference (f - M) is added to the deletion count.
3. **Tracking Minimum Deletions**: The minimum deletions encountered across all values of M is returned as the result. This ensures that the string meets the k-special condition with the least number of deletions.

This approach efficiently explores all possible ways to adjust character frequencies by leveraging the maximum frequency in the string, ensuring optimal performance even for large input sizes.
