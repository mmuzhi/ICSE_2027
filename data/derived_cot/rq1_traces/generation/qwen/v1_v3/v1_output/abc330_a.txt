=== ANSWER ===

To solve this problem, we need to determine the number of people who passed an exam based on their scores. The exam passing condition is that a person must score at least L points. 

### Approach
1. **Read Input**: The first line of input contains two integers, N (number of people) and L (minimum passing score). The second line contains N integers representing the scores of each person.
2. **Count Passes**: Initialize a counter to zero. Iterate through each score in the list. For each score that is greater than or equal to L, increment the counter.
3. **Output Result**: After processing all scores, print the counter which holds the number of people who passed the exam.

This approach efficiently checks each score against the passing threshold and counts the valid entries, leveraging a straightforward loop for simplicity and clarity.

### Solution Code
```python
def main():
    data = input().split()
    n = int(data[0])
    l = int(data[1])
    scores = list(map(int, input().split()))
    count = 0
    for score in scores:
        if score >= l:
            count += 1
    print(count)

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The first line is read and split into two parts: N (number of people) and L (passing score). The second line is read and split into individual scores, which are converted into integers.
2. **Counting Passes**: A loop iterates over each score. If a score meets or exceeds L, the counter is incremented.
3. **Output**: The final count of passing scores is printed, which is the solution to the problem.

This solution efficiently processes the input in linear time, O(N), where N is the number of people, making it optimal for the given constraints. The simplicity of the approach ensures clarity and correctness.
