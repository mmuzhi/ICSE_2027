Given an array of strings words and a string s, determine if s is an acronym of words.
The string s is considered an acronym of words if it can be formed by concatenating the first character of each string in words in order. For example, "ab" can be formed from ["apple", "banana"], but it can't be formed from ["bear", "aardvark"].
Return true if s is an acronym of words, and false otherwise. 
 
Example 1:

Input: words = ["alice","bob","charlie"], s = "abc"
Output: true
Explanation: The first character in the words "alice", "bob", and "charlie" are 'a', 'b', and 'c', respectively. Hence, s = "abc" is the acronym. 

Example 2:

Input: words = ["an","apple"], s = "a"
Output: false
Explanation: The first character in the words "an" and "apple" are 'a' and 'a', respectively. 
The acronym formed by concatenating these characters is "aa". 
Hence, s = "a" is not the acronym.

Example 3:

Input: words = ["never","gonna","give","up","on","you"], s = "ngguoy"
Output: true
Explanation: By concatenating the first character of the words in the array, we get the string "ngguoy". 
Hence, s = "ngguoy" is the acronym.

=== ANSWER ===

To solve this problem, we need to determine if a given string `s` can be formed by concatenating the first character of each string in the provided list `words` in the same order. 

### Approach
1. **Check Length Mismatch**: First, we check if the length of the string `s` matches the number of words in the list. If they don't match, it's immediately clear that `s` cannot be an acronym, so we return `False`.
2. **Character Matching**: For each word in the list, we extract its first character and compare it with the corresponding character in the string `s`. If any character does not match, we return `False`.
3. **Return Result**: If all corresponding characters match, we return `True`, indicating that `s` is indeed an acronym of the given words.

### Solution Code
```python
class Solution:
    def isAcronym(self, words: List[str], s: str) -> bool:
        if len(words) != len(s):
            return False
        for i in range(len(words)):
            if words[i][0] != s[i]:
                return False
        return True
```

### Explanation
1. **Initial Check**: The solution starts by verifying if the number of words matches the length of the string `s`. If not, it returns `False` because an acronym must have exactly one character per word.
2. **Character Comparison**: The solution then iterates over each word and its corresponding character in `s`. For each word, the first character is compared to the character in `s` at the same position. If any character does not match, the function returns `False` immediately.
3. **Result**: If all characters match their respective positions in `s`, the function returns `True`, confirming that `s` is an acronym formed by the first characters of each word in order.

This approach efficiently checks the necessary conditions with a time complexity of O(n), where n is the number of words (or the length of `s`), as each word's first character is checked exactly once. The space complexity is O(1) since no additional data structures are used.
