To solve this problem, we need to rearrange the bits of a given binary string such that the resulting binary number is the maximum possible odd number. The key observation here is that for a binary number to be odd, its last bit must be '1'. To maximize the number, we should place as many '1's as possible at the beginning of the string, followed by '0's, and ensure the last bit is '1'.

### Approach
1. **Count the number of '1's**: This helps in determining how many '1's are available to place in the binary number.
2. **Determine the last bit**: The last bit must be '1' to make the number odd. We reserve one '1' for this purpose.
3. **Construct the remaining bits**: The remaining bits (all except the last bit) should be filled with the remaining '1's followed by '0's. This ensures the largest possible number is formed by placing the highest value bits (1's) first.
4. **Combine the parts**: Combine the constructed remaining bits with the last bit '1' to form the final result.

### Solution Code
```python
class Solution:
    def maximumOddBinaryNumber(self, s: str) -> str:
        count_ones = s.count('1')
        n = len(s)
        zeros = n - count_ones
        return ('1' * (count_ones - 1) + '0' * zeros) + '1'
```

### Explanation
1. **Counting '1's**: The variable `count_ones` stores the number of '1's in the input string.
2. **Calculating zeros**: The variable `zeros` is derived by subtracting `count_ones` from the total length of the string, giving the number of '0's available.
3. **Constructing the binary number**: The first part of the result is formed by placing all remaining '1's (after reserving one for the last bit) followed by all available '0's. This part is then concatenated with a '1' at the end to ensure the number is odd and maximized.
4. **Result**: The final string is the combination of the constructed first part and the last '1', which forms the largest possible odd binary number from the given bits.

This approach efficiently leverages the properties of binary numbers (oddness determined by the last bit and maximization by placing higher value bits first) to solve the problem in linear time, making it optimal for the given constraints.
