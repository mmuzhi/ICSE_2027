To solve this problem, we need to determine if Takahashi can eat all the dishes without feeling sick, given that eating two sweet dishes consecutively makes him unable to eat any more dishes. The solution involves checking the sequence of dishes to ensure that no two consecutive sweet dishes occur except possibly at the end of the sequence.

### Approach
1. **Problem Analysis**: The key observation is that if two consecutive sweet dishes appear at any point in the sequence except the last two dishes, Takahashi will stop eating and cannot consume the remaining dishes. However, if the two consecutive sweet dishes are the last two in the sequence, he can still eat all dishes since there are no more dishes left to eat after that.
2. **Intuition**: We need to traverse the list of dishes and check each pair of consecutive dishes. If we encounter two consecutive sweet dishes before the last dish, we immediately conclude that Takahashi cannot eat all dishes. If the two consecutive sweet dishes are the last two, we allow him to eat all dishes.
3. **Algorithm Selection**: The algorithm involves a single pass through the list of dishes, checking each adjacent pair. For each pair, if both are sweet and not the last two dishes, we print "No" and exit. If the pair is the last two dishes, we break the loop and print "Yes" after processing all dishes.
4. **Complexity Analysis**: The algorithm runs in O(N) time, where N is the number of dishes, as it processes each dish exactly once. The space complexity is O(N) to store the list of dishes, which is optimal for the given constraints.

### Solution Code
```python
n = int(input().strip())
s = [input().strip() for _ in range(n)]

for i in range(1, n):
    if s[i] == 'sweet' and s[i-1] == 'sweet':
        if i < n-1:
            print("No")
            exit(0)
        else:
            break

print("Yes")
```

### Explanation
1. **Reading Input**: The code starts by reading the number of dishes `n` and then reads each dish's type (either "sweet" or "salty") into a list `s`.
2. **Checking Consecutive Sweet Dishes**: The loop iterates over each dish starting from the second one. For each dish, it checks if it and the previous dish are both sweet. If they are and this is not the last two dishes, it prints "No" and exits, as Takahashi cannot eat the remaining dishes.
3. **Handling Last Two Dishes**: If the two consecutive sweet dishes are the last two in the sequence, the loop breaks after processing them, and the code proceeds to print "Yes" since Takahashi can eat all dishes.
4. **Output Result**: If no invalid consecutive sweet dishes are found during the loop, the code prints "Yes", indicating Takahashi can eat all dishes without feeling sick. 

This approach efficiently checks the sequence of dishes to determine if Takahashi can eat all of them, adhering to the given constraints.
