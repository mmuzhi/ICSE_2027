=== ANSWER ===

To solve this problem, we need to take a string consisting of uppercase English letters and print each character separated by a space, ensuring there is no trailing space after the last character.

### Approach
The approach involves processing each character of the input string and formatting them with spaces between them. The key observation here is that we need to join all characters of the string with a single space between each character. Python's built-in `join` method is particularly suitable for this task as it efficiently concatenates all elements of an iterable (in this case, the characters of the string) into a single string, inserting the specified separator (space) between each element. 

### Solution Code
```python
def main():
    S = input().strip()
    print(' '.join(S))

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The input string `S` is read from standard input and stripped of any leading or trailing whitespace to ensure clean processing.
2. **Joining Characters**: The `join` method is applied to the string `S`, using a space as the separator. This method processes each character in the string and inserts a space between them, resulting in a new string where each character is separated by a space.
3. **Output**: The resulting string is printed, which meets the requirement of having each character separated by a space without any trailing space at the end.

This approach efficiently handles the problem by leveraging Python's string manipulation capabilities, ensuring clarity and correctness with minimal code. The solution is both concise and optimal, adhering to the problem constraints and requirements.
