We define Extended A strings, Extended B strings, Extended C strings, and Extended ABC strings as follows:

- A string S is an Extended A string if all characters in S are A.

=== ANSWER ===

To solve this problem, we need to determine if a given string is an Extended ABC string. An Extended ABC string is defined as a concatenation of three specific types of strings: an Extended A string (all characters are 'A'), followed by an Extended B string (all characters are 'B'), and finally an Extended C string (all characters are 'C'). The empty string is considered an Extended A, B, or C string.

### Approach
1. **Problem Analysis**: The key observation is that the string must be divided into three contiguous segments where the first segment consists only of 'A's, the second segment only of 'B's, and the third segment only of 'C's. The segments can be empty, but their order must be strictly followed.
2. **Algorithm Selection**: The solution involves traversing the string to check for these segments in order. The algorithm reads all consecutive 'A's from the start of the string, then all consecutive 'B's following the 'A's, and finally all consecutive 'C's following the 'B's. If the entire string is processed in this manner, it is an Extended ABC string; otherwise, it is not.
3. **Complexity Analysis**: The algorithm processes each character in the string exactly once, resulting in a linear time complexity of O(n), where n is the length of the string. This is optimal for the given constraints.

### Solution Code
```python
def main():
    S = input().strip()
    n = len(S)
    i = 0
    while i < n and S[i] == 'A':
        i += 1
        
    j = i
    while j < n and S[j] == 'B':
        j += 1
        
    k = j
    while k < n and S[k] == 'C':
        k += 1
        
    if k == n:
        print("Yes")
    else:
        print("No")

if __name__ == "__main__":
    main()
```

### Explanation
1. **Initialization**: The input string is read and its length is stored.
2. **Reading 'A's**: The first loop increments the index `i` until it encounters a non-'A' character or reaches the end of the string.
3. **Reading 'B's**: The second loop starts from the current index `i` and increments `j` until it encounters a non-'B' character or reaches the end.
4. **Reading 'C's**: The third loop starts from the current index `j` and increments `k` until it encounters a non-'C' character or reaches the end.
5. **Validation**: If the final index `k` equals the string length `n`, the entire string is processed as per the required segments, and "Yes" is printed. Otherwise, "No" is printed, indicating the string does not meet the criteria.

This approach efficiently checks the string structure by leveraging simple traversal and conditional checks, ensuring correctness with minimal computational overhead.
