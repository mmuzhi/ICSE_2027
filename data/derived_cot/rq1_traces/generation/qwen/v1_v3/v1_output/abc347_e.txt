To solve this problem, we need to efficiently handle a series of queries that toggle elements in a set and update an array based on the current size of the set. The challenge lies in performing these operations without resorting to a brute-force approach, which would be too slow for large inputs.

### Approach
1. **Problem Analysis**: The problem involves processing a sequence of queries where each query either adds or removes an element from a set S. After each toggle operation, we need to add the current size of S to every element currently in S. The goal is to determine the final state of the array A after all queries.

2. **Key Insight**: The final value of each element in array A is the sum of the sizes of S after each query where the element was present in S. Instead of tracking each element's presence in S for every query, we can leverage the fact that the size of S changes incrementally with each query.

3. **Algorithm Selection**:
   - **Tracking Set Size**: Maintain an array `size_arr` where each entry `size_arr[i]` represents the size of S after the i-th query.
   - **Toggle History**: For each element, record the query indices where it was toggled. This helps in determining the intervals where the element is present in S.
   - **Prefix Sum Array**: Construct a prefix sum array `P` from `size_arr` to quickly compute the sum of sizes over any interval of queries.

4. **Efficiency**: The approach efficiently computes the sum for each element by leveraging the prefix sum array, reducing the problem to summing intervals where the element is in S. The total operations are linear with respect to the number of queries and elements, making it suitable for large inputs.

### Solution Code
```python
import sys

def main():
    data = sys.stdin.read().split()
    if not data:
        return
    
    it = iter(data)
    N = int(next(it))
    Q = int(next(it))
    queries = [int(next(it)) for _ in range(Q)]
    
    size_arr = [0] * (Q + 1)
    state = [0] * (N + 1)
    toggles = [[] for _ in range(N + 1)]
    
    for i in range(1, Q + 1):
        x = queries[i - 1]
        if state[x] == 1:
            state[x] = 0
            size_arr[i] = size_arr[i - 1] - 1
        else:
            state[x] = 1
            size_arr[i] = size_arr[i - 1] + 1
        toggles[x].append(i)
    
    P = [0] * (Q + 1)
    for i in range(1, Q + 1):
        P[i] = P[i - 1] + size_arr[i]
    
    ans = [0] * (N + 1)
    
    for j in range(1, N + 1):
        T = toggles[j]
        T.sort()
        intervals_odd = []
        if not T:
            continue
            
        for k in range(0, len(T), 2):
            if k + 1 < len(T):
                L = T[k]
                R = T[k + 1] - 1
                if L <= R:
                    intervals_odd.append((L, R))
            else:
                if len(T) % 2 == 1:
                    L = T[-1]
                    R = Q
                    if L <= R:
                        intervals_odd.append((L, R))
        
        total = 0
        for L, R in intervals_odd:
            if L > Q:
                continue
            total += P[R] - P[L - 1]
        ans[j] = total
        
    result = []
    for i in range(1, N + 1):
        result.append(str(ans[i]))
    print(" ".join(result))

if __name__ == '__main__':
    main()
```
