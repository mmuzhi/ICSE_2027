=== ANSWER ===

To solve this problem, we need to determine the relationship between two strings, S and T, based on whether S is a prefix, suffix, both, or neither of T. The solution involves straightforward string comparisons to check these conditions.

### Approach
1. **Check Prefix Condition**: Compare the first N characters of T with S. If they match, S is a prefix of T.
2. **Check Suffix Condition**: Compare the last N characters of T with S. If they match, S is a suffix of T.
3. **Determine Result**: Based on the results of the above checks, output the corresponding value:
   - **0** if S is both a prefix and a suffix of T.
   - **1** if S is only a prefix.
   - **2** if S is only a suffix.
   - **3** if S is neither.

### Solution Code
```python

def main():
    import sys
    data = sys.stdin.read().splitlines()
    if not data: 
        print(3)
        return
    
    parts = data[0].split()
    N = int(parts[0])
    M = int(parts[1])
    S = data[1].strip()
    T = data[2].strip()
    
    is_prefix = S == T[:N]
    is_suffix = S == T[-N:]
    
    if is_prefix and is_suffix:
        print(0)
    elif is_prefix:
        print(1)
    elif is_suffix:
        print(2)
    else:
        print(3)

if __name__ == "__main__":
    main()
```

1. **Reading Input**: The input is read from standard input, which includes the lengths of strings S and T (N and M), followed by the strings themselves.
2. **Prefix Check**: The first N characters of T are compared with S. If they match, S is a prefix of T.
3. **Suffix Check**: The last N characters of T are compared with S. If they match, S is a suffix of T.
4. **Result Determination**: The results of the prefix and suffix checks are used to determine the output:
   - If both checks pass, the output is 0.
   - If only the prefix check passes, the output is 1.
   - If only the suffix check passes, the output is 2.
   - If neither check passes, the output is 3.

This approach efficiently checks the necessary conditions using simple string comparisons and outputs the result based on the checks. The solution handles all edge cases, including when S and T are identical, by leveraging the given conditions.
