=== ANSWER ===

To solve this problem, we need to determine how many aliens can successfully disinfect all their hands given a limited amount of disinfectant. The disinfectant can disinfect exactly M hands, and each alien requires a certain number of hands to be disinfected. The key is to process each alien in order, checking if the remaining disinfectant is sufficient for their needs. If it is, we subtract the required amount; otherwise, we use up the remaining disinfectant and stop counting further aliens since no more disinfectant will be left.

### Approach
1. **Initialization**: Read the input values for the number of aliens (N), the total disinfectant capacity (M), and the list of hands for each alien.
2. **Processing Each Alien**: For each alien in sequence:
   - Check if the remaining disinfectant is at least the number of hands the alien needs.
   - If yes, subtract the required hands from the remaining disinfectant and increment the count of successfully disinfected aliens.
   - If not, set the remaining disinfectant to zero (since the alien uses up all remaining disinfectant) and do not count this alien.
3. **Output the Result**: After processing all aliens, print the count of aliens who successfully disinfected all their hands.

### Solution Code
```python
def main():
    import sys
    data = sys.stdin.read().split()
    n = int(data[0])
    m = int(data[1])
    hands = list(map(int, data[2:2+n]))
    
    remaining = m
    count = 0
    for h in hands:
        if remaining >= h:
            remaining -= h
            count += 1
        else:
            remaining = 0
    
    print(count)

if __name__ == "__main__":
    main()
```

### Explanation
- **Initialization**: The code reads the input values for the number of aliens (N), the total disinfectant capacity (M), and the list of hands for each alien.
- **Processing Loop**: The loop iterates over each alien's hand count. For each alien, it checks if the remaining disinfectant is sufficient. If yes, it subtracts the required hands and increments the success count. If not, it sets the remaining disinfectant to zero, effectively stopping further processing since no more disinfectant will be available.
- **Output**: The total count of aliens who successfully disinfected all their hands is printed.

This approach efficiently processes each alien in sequence, ensuring optimal use of the disinfectant and accurately counting the number of successful disinfections. The solution handles all edge cases, including when the disinfectant is exhausted before all aliens have had their turn.
