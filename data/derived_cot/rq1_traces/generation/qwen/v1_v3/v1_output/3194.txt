=== ANSWER ===

To solve this problem, we need to find all indices in an array of strings where the given character `x` is present in any of the strings. The solution involves iterating through each string in the array and checking if the character `x` exists in the string. If it does, the index of that string is added to the result list.

### Approach
1. **Problem Analysis**: The task is straightforward—examine each string in the input array to determine if it contains the specified character `x`. The solution requires collecting the indices of all such strings.
2. **Intuition**: By iterating through each string and using a simple check for the presence of `x`, we can efficiently gather all relevant indices. The order of the indices in the result does not matter, as per the problem statement.
3. **Algorithm Selection**: The algorithm involves a linear scan of the array. For each string, we check if `x` is a substring of the string. If it is, the index is added to the result list.
4. **Complexity Analysis**: The time complexity is O(n * m), where `n` is the number of strings and `m` is the average length of the strings. This is because for each string, we potentially check every character. The space complexity is O(k), where `k` is the number of indices that contain `x`, as we store these indices in a list.

### Solution Code
```python
class Solution:
    def findWordsContaining(self, words: List[str], x: str) -> List[int]:
        result = []
        for idx, word in enumerate(words):
            if x in word:
                result.append(idx)
        return result
```

### Explanation
1. **Initialization**: We start with an empty list `result` to store the indices of strings that contain the character `x`.
2. **Iteration**: Using `enumerate`, we loop through each string in the input array `words`, keeping track of both the index and the string itself.
3. **Check for Character**: For each string, we check if `x` is a substring. This is done using the `in` keyword, which efficiently checks for the presence of `x` in the string.
4. **Collect Indices**: If `x` is found in the string, the index of that string is appended to the `result` list.
5. **Return Result**: After processing all strings, the `result` list, which contains all indices where `x` was found, is returned.

This approach ensures that we efficiently and correctly identify all strings containing the specified character, leveraging simple iteration and substring checks.
