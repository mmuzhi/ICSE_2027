To solve this problem, we need to compute the expected number of wins for each player in a tournament where teams merge based on specific probabilities. The tournament involves N players, each starting as a separate team, and N-1 matches where two teams merge based on their sizes. The goal is to determine, for each player, the expected number of times their team wins throughout the tournament, modulo 998244353.

### Approach
1. **Tree Construction**: The tournament can be modeled as a tree where each internal node represents a match between two teams. The leaves are the initial players, and each internal node has two children representing the teams that merged in that match. The first child is the team specified by the first player in the match, and the second child is the team specified by the second player.

2. **Size Calculation**: For each node in the tree, compute the size of its subtree (number of players it contains). This is done during the tree construction phase by summing the sizes of the children.

3. **Modular Inverses**: Precompute modular inverses for all possible sizes (up to N) to efficiently compute probabilities modulo 998244353.

4. **Path Traversal**: For each player (leaf), traverse the path from the leaf to the root of the tree. For each match (internal node) along this path, compute the probability that the player's team wins that match, which is the size of the player's team divided by the total size of the two teams in the match. Sum these probabilities to get the expected number of wins for the player.

MOD = 998244353

import sys

def main():
    data = sys.stdin.read().strip().split()
    if not data:
        return
    n = int(data[0])
    matches = []
    index = 1
    for i in range(n-1):
        p = int(data[index])
        q = int(data[index+1])
        index += 2
        matches.append((p, q))
    
    total_nodes = 2 * n - 1
    parent = [0] * (total_nodes + 1)
    children = [[] for _ in range(total_nodes + 1)]
    first_child = [0] * (total_nodes + 1)
    second_child = [0] * (total_nodes + 1)
    size = [0] * (total_nodes + 1)
    
    for i in range(1, n+1):
        parent[i] = i
        size[i] = 1
        
    next_node = n + 1
    
    for (p, q) in matches:
        root1 = p
        while parent[root1] != root1:
            root1 = parent[root1]
        root2 = q
        while parent[root2] != root2:
            root2 = parent[root2]
            
        if root1 == root2:
            continue
            
        u = next_node
        next_node += 1
        parent[u] = u
        children[u] = [root1, root2]
        first_child[u] = root1
        second_child[u] = root2
        size[u] = size[root1] + size[root2]
        parent[root1] = u
        parent[root2] = u
        
    max_size = n
    inv = [0] * (max_size + 1)
    for i in range(1, max_size + 1):
        inv[i] = pow(i, MOD-2, MOD)
        
    results = [0] * (n + 1)
    
    for i in range(1, n+1):
        current = i
        total = 0
        while parent[current] != current:
            u = parent[current]
            if current == first_child[u]:
                child = first_child[u]
            else:
                child = second_child[u]
            size_child = size[child]
            size_u = size[u]
            if size_u == 0:
                prob = 0
            else:
                prob = size_child * inv[size_u] % MOD
            total = (total + prob) % MOD
            current = parent[child]
        results[i] = total
