You will be given a competitive programming problem.
A pair of sequences of length M consisting of positive integers at most N, (S, T) = ((S_1, S_2, \dots, S_M), (T_1, T_2, \dots, T_M)), is said to be a good pair of sequences when (S, T) satisfies the following condition.

- There exists a sequence X = (X_1, X_2, \dots, X_N) of length N consisting of 0 and 1 that satisfies the following condition:
- X_{S_i} \neq X_{T_i} for each i=1, 2, \dots, M.



You are given a pair of sequences of length M consisting of positive integers at most N: (A, B) = ((A_1, A_2, \dots, A_M), (B_1, B_2, \dots, B_M)). If (A, B) is a good pair of sequences, print Yes; otherwise, print No.

Input

The input is given from Standard Input in the following format:
N M
A_1 A_2 \dots A_M
B_1 B_2 \dots B_M

Output

If (A, B) is a good pair of sequences, print Yes; otherwise, print No.

Constraints


- 1 \leq N, M \leq 2 \times 10^5
- 1 \leq A_i, B_i \leq N
- All input values are integers.

To solve this problem, we need to determine if there exists a binary sequence \( X \) of length \( N \) such that for each given pair of indices \( (A_i, B_i) \), the values \( X_{A_i} \) and \( X_{B_i} \) are different. This problem can be modeled as a graph problem where each node represents an element in the sequence, and each constraint \( (A_i, B_i) \) represents an edge between two nodes. The solution involves checking if the graph is bipartite, meaning it can be colored with two colors such that no two adjacent nodes have the same color.

### Approach
1. **Graph Construction**: Construct an undirected graph where each node is a number from 1 to \( N \). For each pair \( (A_i, B_i) \), add an edge between nodes \( A_i \) and \( B_i \). If any edge is a self-loop (i.e., \( A_i = B_i \)), immediately return "No" because it's impossible to satisfy the condition \( X_{A_i} \neq X_{B_i} \).

2. **Bipartite Check**: For each connected component in the graph, perform a BFS or DFS to check if the component is bipartite. During the traversal, assign colors (0 or 1) to nodes. If any adjacent nodes have the same color, the graph is not bipartite, and we return "No". If all components are bipartite, return "Yes".

### Solution Code
```python
import sys
from collections import deque

def main():
    data = sys.stdin.read().split()
    if not data:
        print("Yes")
        return
    it = iter(data)
    n = int(next(it))
    m = int(next(it))
    A = [int(next(it)) for _ in range(m)]
    B = [int(next(it)) for _ in range(m)]
    
    graph = [set() for _ in range(n+1)]
    for i in range(m):
        u = A[i]
        v = B[i]
        if u == v:
            print("No")
            return
        graph[u].add(v)
        graph[v].add(u)
    
    visited = [False] * (n+1)
    color = [-1] * (n+1)
    
    for i in range(1, n+1):
        if not visited[i]:
            dq = deque([i])
            visited[i] = True
            color[i] = 0
            while dq:
                cur = dq.popleft()
                for neighbor in graph[cur]:
                    if not visited[neighbor]:
                        visited[neighbor] = True
                        color[neighbor] = 1 - color[cur]
                        dq.append(neighbor)
                    else:
                        if color[neighbor] == color[cur]:
                            print("No")
                            return
    print("Yes")

if __name__ == "__main__":
    main()
```

2. **Bipartite Check**: Using BFS, each connected component is traversed. Nodes are colored alternately (0 and 1).
