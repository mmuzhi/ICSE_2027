To solve this problem, we need to identify a single spoiled bottle of juice among N bottles using the minimum number of friends. The solution involves a non-adaptive group testing strategy where each friend is given a subset of bottles based on a binary representation of the bottle indices. This approach ensures that the results from the friends can be combined to uniquely identify the spoiled bottle.

### Approach
1. **Problem Analysis**: The problem requires identifying one bad bottle among N bottles using the fewest friends possible. Each friend can be given any number of bottles, and the same bottle can be given to multiple friends. The key insight is that each friend's response (whether they get a stomach upset) corresponds to a bit in a binary number. By assigning each bottle a unique binary number (based on its index), we can determine the spoiled bottle by interpreting the friends' responses as a binary number.

2. **Intuition**: The minimum number of friends needed is the smallest integer M such that 2^M ≥ N. This is derived from the fact that each friend can provide one bit of information, and with M friends, we can distinguish up to 2^M bottles.

3. **Algorithm Selection**: We use a binary representation approach where each bottle is assigned a unique M-bit binary number. For each bit position (from least significant to most significant), we give the corresponding bottles to a specific friend. The pattern of upset friends (1s in the binary number) directly points to the spoiled bottle's index.

4. **Complexity Analysis**: The algorithm involves O(M*N) operations to generate the distributions for each friend, where M is ceil(log2(N)). Since N can be up to 100, M is at most 7, making the approach efficient with a constant number of operations per test case.

### Solution Code
```python
import sys

def main():
    N = int(sys.stdin.readline().strip())
    M = 0
    while (1 << M) < N:
        M += 1
        
    print(M)
    
    for j in range(M):
        bottles = []
        for i in range(N):
            if (i >> j) & 1:
                bottles.append(str(i + 1))
        print(str(len(bottles)) + " " + " ".join(bottles))
    
    S = sys.stdin.readline().strip()
    bottle_index = 0
    for j in range(M):
        if S[j] == '1':
            bottle_index |= (1 << j)
            
    print(bottle_index + 1)

    main()
```
