=== ANSWER ===

To solve this problem, we need to compute the product of consecutive elements in a given list of integers. The solution involves reading the input values, processing them to generate the required products, and then outputting the results in the specified format.

### Approach
1. **Read Input:** The first step is to read the number of integers, N, followed by the list of N integers.
2. **Compute Products:** For each element in the list from the first to the second-to-last element, compute the product of the current element and the next element.
3. **Output Results:** Convert each computed product into a string and join them with spaces to form the final output.

This approach efficiently processes the input in linear time, O(N), where N is the number of elements in the list. The solution leverages simple arithmetic operations and string manipulation to meet the problem requirements.

### Solution Code
```python
def main():
    n = int(input().strip())
    arr = list(map(int, input().split()))
    result = []
    for i in range(n-1):
        result.append(str(arr[i] * arr[i+1]))
    print(" ".join(result))

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input:** The code starts by reading the integer N, which denotes the number of elements in the list. The next line reads the list of integers.
2. **Processing Data:** Using a loop, the code iterates through each element from the first to the second-to-last element. For each element, it calculates the product with the next element and stores the result as a string in a list.
3. **Output:** The list of stringified products is then joined into a single string separated by spaces, which is printed as the final output.

This solution efficiently handles the problem constraints and ensures the output is formatted correctly, meeting the specified requirements.
