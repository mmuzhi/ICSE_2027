=== ANSWER ===

To solve this problem, we need to find the maximum sum of a pair of numbers from a given list such that the maximum digit in both numbers is equal. If no such pair exists, we should return -1.

### Approach
1. **Problem Analysis**: The task involves processing each number in the list to determine its maximum digit. Once we have this information, we need to group numbers by their maximum digit. For each group with at least two numbers, we find the two largest numbers and compute their sum. The solution involves efficiently grouping numbers and then checking these groups for the maximum sum.

2. **Intuition**: By grouping numbers based on their maximum digit, we can easily access all numbers that meet the pairing condition. For each group, sorting the numbers in descending order allows us to quickly identify the two largest numbers, which will yield the maximum sum for that group.

3. **Algorithm Selection**: The algorithm involves:
   - Converting each number to a string to find its maximum digit.
   - Using a dictionary to map each maximum digit to a list of numbers that share that digit.
   - For each digit group with at least two numbers, sorting the list in descending order and summing the top two numbers.
   - Tracking the maximum sum encountered across all digit groups.

4. **Complexity Analysis**: The algorithm processes each number once to determine its maximum digit, which is O(n) where n is the number of elements. Sorting each digit group takes O(m log m) for a group of size m. Given that the maximum group size is limited (since digits are from 0 to 9), the overall complexity is efficient for the given constraints.

### Solution Code
```python
class Solution:
    def maxSum(self, nums: List[int]) -> int:
        digit_group = {}
        for num in nums:
            num_str = str(num)
            max_digit = max(num_str)
            if max_digit not in digit_group:
                digit_group[max_digit] = []
            digit_group[max_digit].append(num)
        
        best = -1
        for key, group in digit_group.items():
            if len(group) < 2:
                continue
            group.sort(reverse=True)
            total = group[0] + group[1]
            if total > best:
                best = total
        
        return best
```

### Explanation
1. **Initialization**: We start by initializing a dictionary `digit_group` to store numbers grouped by their maximum digit.
2. **Processing Numbers**: For each number in the input list, convert it to a string, find its maximum digit, and add it to the corresponding group in the dictionary.
3. **Finding Maximum Sum**: For each digit group with at least two numbers, sort the numbers in descending order and compute the sum of the top two numbers. Track the maximum sum encountered during this process.
4. **Result**: After processing all groups, return the maximum sum found. If no valid pair exists, return -1.

This approach efficiently groups numbers by their maximum digit and checks each group for the highest possible sum, ensuring optimal performance and correctness.
