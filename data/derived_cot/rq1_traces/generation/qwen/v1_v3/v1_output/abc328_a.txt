=== ANSWER ===

To solve this problem, we need to calculate the total score of all problems in a programming contest where each problem's score is less than or equal to a given threshold X. The solution involves reading the input values, processing them to sum the scores that meet the condition, and then outputting the result.

### Approach
1. **Read Input:** The first line of input contains two integers, N (number of problems) and X (the threshold score). The second line contains N integers representing the scores of each problem.
2. **Process Data:** Iterate through each score in the list and check if it is less than or equal to X. If it is, include it in the sum.
3. **Output Result:** Print the total sum of all scores that meet the condition.

The approach is straightforward due to the constraints (N is small, between 4 and 8), making it efficient to use a simple loop without needing complex data structures or algorithms.

### Solution Code
```python
import sys

def main():
    data = sys.stdin.read().split()
    n = int(data[0])
    x = int(data[1])
    scores = list(map(int, data[2:2+n]))
    
    total = 0
    for score in scores:
        if score <= x:
            total += score
            
    print(total)

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input:** The input is read all at once and split into a list of strings. The first two elements are N and X, converted to integers. The remaining elements are the scores, converted to integers and stored in a list.
2. **Processing Scores:** The code initializes a total sum to zero. It then iterates over each score in the list. For each score, if it is less than or equal to X, it is added to the total sum.
3. **Output:** Finally, the total sum is printed. This approach efficiently checks each score and accumulates the sum of those meeting the condition, providing the correct result as demonstrated in the sample inputs.
