=== ANSWER ===

To solve this problem, we need to partition a given binary string into the minimum number of beautiful substrings. A beautiful substring is defined as a substring that does not contain leading zeros and is the binary representation of a number that is a power of 5. If it's impossible to partition the string into such substrings, we should return -1.

### Approach
1. **Initial Check**: If the binary string starts with a '0', it's immediately impossible to partition it into beautiful substrings because any substring starting with '0' is invalid. Hence, we return -1 in this case.
2. **Precompute Valid Binaries**: We generate all binary strings that represent powers of 5 (from 5^0 up to 5^6, since higher powers exceed the maximum 15-bit binary number). These strings are stored in a set for quick lookup.
3. **Dynamic Programming (DP) Setup**: We use a DP array where `dp[i]` represents the minimum number of beautiful substrings needed to partition the first `i` characters of the string. Initialize `dp[0]` to 0 (indicating an empty string requires 0 partitions) and the rest to a large number (indicating they are initially unreachable).
4. **DP Array Population**: For each position `i` in the string, we check all possible substrings ending at `i` (from `j` to `i-1`). If a substring starts with '0', it's skipped. Otherwise, if the substring is found in the set of valid binaries, we update `dp[i]` to be the minimum of its current value and `dp[j] + 1`.
5. **Result Extraction**: After processing the entire string, if `dp[n]` (where `n` is the string length) remains at its initial large value, return -1. Otherwise, return `dp[n]`.

### Solution Code
```python
class Solution:
    def minimumBeautifulSubstrings(self, s: str) -> int:
        if s[0] == '0':
            return -1
        
        powers = [5**i for i in range(0, 7)]
        valid_binaries = set()
        for p in powers:
            bin_str = bin(p)[2:]
            valid_binaries.add(bin_str)
        
        n = len(s)
        dp = [float('inf')] * (n + 1)
        dp[0] = 0
        
        for i in range(1, n + 1):
            for j in range(0, i):
                if s[j] == '0':
                    continue
                substr = s[j:i]
                if len(substr) > 15:
                    continue
                if substr in valid_binaries:
                    dp[i] = min(dp[i], dp[j] + 1)
        
        return dp[n] if dp[n] != float('inf') else -1
```

### Explanation
1. **Initial Check**: The solution starts by checking if the string begins with '0'. If it does, the answer is -1 because no valid partition can start with '0'.
2. **Valid Binaries Set**: The code precomputes all binary strings for powers of 5 up to 5^6 (since 5^7 exceeds the maximum 15-bit binary number). These strings are stored in a set for O(1) lookups.
3. **DP Array Initialization**: The DP array is initialized with `inf` (a very large number) to represent unreachable states, except for `dp[0]` which is 0 (the base case for an empty string).
4. **DP Array Population**: For each position `i` in the string, the code checks all possible substrings ending at `i`. If a substring starts with '0', it is skipped. Otherwise, if the substring is found in the set of valid binaries, the DP value at `i` is updated to reflect the minimum partitions needed.
5. **Result Extraction**: After processing the entire string, the solution checks if the last element of the DP array is `inf` (indicating no valid partition exists) and returns -1. Otherwise, it returns the minimum number of partitions stored in `dp[n]`.

This approach efficiently checks all possible partitions using dynamic programming and a precomputed set of valid binaries, ensuring optimal performance for the given constraints.
