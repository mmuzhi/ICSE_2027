=== ANSWER ===

To solve this problem, we need to calculate the score of a given string based on the sum of the absolute differences between the ASCII values of adjacent characters. 

### Approach
1. **Problem Analysis**: The problem requires us to process each adjacent pair of characters in the string, compute the absolute difference of their ASCII values, and sum all these differences. The key observation here is that each character in the string (except the last one) will be part of exactly one adjacent pair with the next character.
2. **Algorithm Selection**: We iterate through the string from the first character to the second last character. For each character at position `i`, we compute the absolute difference between its ASCII value and the ASCII value of the next character (at position `i+1`). The sum of all these differences gives the desired score.
3. **Complexity Analysis**: The algorithm involves a single pass through the string, making the time complexity O(n), where n is the length of the string. The space complexity is O(1) as we only use a few variables for computation.

### Solution Code
```python
class Solution:
    def scoreOfString(self, s: str) -> int:
        total_score = 0
        for i in range(len(s) - 1):
            total_score += abs(ord(s[i]) - ord(s[i+1]))
        return total_score
```

### Explanation
1. **Initialization**: We start by initializing `total_score` to 0, which will hold the cumulative sum of the absolute differences.
2. **Iteration**: We loop through each character in the string except the last one. For each character at index `i`, we compute the absolute difference between its ASCII value (obtained using `ord`) and the ASCII value of the next character (at index `i+1`).
3. **Summation**: The absolute difference for each adjacent pair is added to `total_score` during each iteration.
4. **Result**: After processing all adjacent pairs, `total_score` contains the sum of all absolute differences, which is returned as the result.

This approach efficiently computes the score by leveraging a straightforward iteration and arithmetic operations, ensuring optimal performance and correctness.
