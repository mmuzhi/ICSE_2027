=== ANSWER ===

To solve this problem, we need to determine the maximum number of games Takahashi could have won in a series of rock-paper-scissors games against Aoki, adhering to two conditions: Takahashi never loses, and consecutive moves are different. 

### Approach
1. **Problem Analysis**: The problem involves analyzing each game where Aoki's move is given, and Takahashi must choose a move that either wins or draws. The key constraints are:
   - Takahashi never loses, so he must choose moves that either win or draw against Aoki's moves.
   - Consecutive moves by Takahashi must be different.

2. **Insight**: For each game, Takahashi has two possible moves (either the winning move or the drawing move) based on Aoki's move. The challenge is to sequence these moves such that consecutive moves differ, maximizing the number of wins.

3. **Dynamic Programming (DP) Setup**: We use a DP approach where `dp[i][c]` represents the maximum wins up to the i-th game if the i-th move is `c` (0 for Rock, 1 for Paper, 2 for Scissors). The DP state transitions consider the previous game's moves, ensuring consecutive moves differ.

4. **Win Condition**: The winning move for Takahashi against Aoki's move `x` is `(x + 1) % 3`. For example, if Aoki plays Rock (0), Takahashi should play Paper (1) to win.

5. **Algorithm**:
   - **Initialization**: For the first game, initialize the DP states for both possible moves (win or draw).
   - **Iteration**: For each subsequent game, compute the maximum wins by considering the previous game's valid moves (excluding the current move to ensure consecutive moves differ) and adding the win score for the current move.
   - **Result Extraction**: The result is the maximum value from the DP states of the last game.

### Solution Code
```python
import sys

def main():
    data = sys.stdin.read().splitlines()
    if not data: 
        print(0)
        return
    try:
        N = int(data[0].strip())
        S = data[1].strip()
    except:
        print(0)
        return

    if N == 0:
        print(0)
        return

    AokiNum = []
    allowed = []
    for ch in S:
        if ch == 'R':
            AokiNum.append(0)
            allowed.append({0, 1})
        elif ch == 'P':
            AokiNum.append(1)
            allowed.append({1, 2})
        elif ch == 'S':
            AokiNum.append(2)
            allowed.append({2, 0})

    if N == 1:
        dp_prev = {}
        for c in allowed[0]:
            win_move = (AokiNum[0] + 1) % 3
            score = 1 if c == win_move else 0
            dp_prev[c] = score
        print(max(dp_prev.values()))
        return

    dp_prev = {}
    for c in allowed[0]:
        win_move = (AokiNum[0] + 1) % 3
        score = 1 if c == win_move else 0
        dp_prev[c] = score

    for i in range(1, N):
        dp_curr = {}
        for c in allowed[i]:
            candidates = []
            for d in allowed[i-1]:
                if d != c:
                    candidates.append(dp_prev[d])
            best_prev = max(candidates)
            win_move = (AokiNum[i] + 1) % 3
            score = 1 if c == win_move else 0
            dp_curr[c] = best_prev + score
        dp_prev = dp_curr

    print(max(dp_prev.values()))

if __name__ == '__main__':
    main()
```
