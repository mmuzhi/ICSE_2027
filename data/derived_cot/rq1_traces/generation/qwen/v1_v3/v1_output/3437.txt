To solve this problem, we need to maximize the total damage a magician can cast given that casting a spell with a certain damage prevents casting any spells with damages within ±2 of that spell's damage. The solution involves dynamic programming to efficiently compute the maximum possible total damage by considering the constraints.

### Approach
1. **Frequency Counting**: First, we count the frequency of each unique damage value in the input array. This helps in calculating the total damage for each unique value (damage multiplied by its frequency).
2. **Sorting Unique Values**: We then sort the unique damage values to process them in ascending order, which is crucial for the dynamic programming step.
3. **Dynamic Programming Setup**: We initialize a DP array where `dp[i]` represents the maximum total damage achievable considering the first `i+1` unique damage values (sorted).
4. **DP Array Calculation**: For each unique damage value, we determine the largest value in the sorted list that is at least 3 less than the current value. This ensures that selecting the current value does not conflict with any previously selected values. The DP value for the current index is updated by either including the current value (plus the best total damage from the values at least 3 less) or excluding it (carrying forward the previous best total damage).
5. **Result Extraction**: The final result is found in the last element of the DP array, which contains the maximum total damage achievable under the given constraints.

### Solution Code
```python
from collections import Counter
import bisect
from typing import List

class Solution:
    def maximumTotalDamage(self, power: List[int]) -> int:
        freq = Counter(power)
        vals = sorted(freq.keys())
        n = len(vals)
        total_damage = [val * freq[val] for val in vals]
        
        if n == 0:
            return 0
        
        dp = [0] * n
        dp[0] = total_damage[0]
        
        for i in range(1, n):
            x = vals[i] - 3
            j = bisect.bisect_right(vals, x) - 1
            dp[i] = max(dp[i-1], total_damage[i] + (dp[j] if j >= 0 else 0))
        
        return dp[-1]
```

### Explanation
1. **Frequency Counting**: The `Counter` from the `collections` module is used to count occurrences of each damage value, stored in `freq`.
2. **Sorting Unique Values**: The unique damage values are sorted to process them in ascending order, ensuring that we can efficiently find the largest value that is at least 3 less than the current value using binary search.
3. **Dynamic Programming Initialization**: The `dp` array is initialized with zeros, and the first element is set to the total damage of the smallest unique value.
4. **DP Array Calculation**: For each subsequent unique value, we calculate the largest value in the sorted list that is at least 3 less than the current value using `bisect_right`. The DP value for the current index is updated by comparing the best total damage without the current value (`dp[i-1]`) and the best total damage including the current value (total damage of the current value plus the best total damage from the values at least 3 less).
5. **Result Extraction**: The last element in the `dp` array contains the maximum total damage achievable, considering all constraints, which is returned as the result.

This approach efficiently computes the solution using dynamic programming and binary search, ensuring optimal performance even for large input sizes.
