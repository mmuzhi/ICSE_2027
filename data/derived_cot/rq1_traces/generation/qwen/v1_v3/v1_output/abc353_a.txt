=== ANSWER ===

To solve this problem, we need to determine if there exists any building taller than the first building from the left. If such a building exists, we should find the position of the leftmost such building. If no such building exists, we should return -1.

### Approach
1. **Read Input**: The first line of input is the number of buildings, N. The second line contains the heights of the N buildings.
2. **Initial Comparison**: The first building's height is compared with all subsequent buildings.
3. **Find Leftmost Taller Building**: Starting from the second building (index 1), we check each building's height. The first building encountered that is taller than the first building is the answer. If none is found after checking all buildings, return -1.

### Solution Code
```python
def main():
    n = int(input().strip())
    heights = list(map(int, input().split()))
    
    if n <= 1:
        print(-1)
        return
        
    first_height = heights[0]
    for i in range(1, n):
        if heights[i] > first_height:
            print(i + 1)
            return
            
    print(-1)

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The code starts by reading the number of buildings, `n`, and then the list of building heights.
2. **Edge Case Handling**: If there is only one building or no buildings, it immediately returns -1 since no other building can be taller than the first one.
3. **Comparison Loop**: The loop starts from the second building (index 1) and checks each subsequent building's height. The first building found with a height greater than the first building's height will print its 1-based index (i + 1) and exit.
4. **No Taller Building**: If the loop completes without finding any taller building, the code prints -1.

This approach efficiently checks each building in linear time, making it optimal for the given constraints. The solution is straightforward and leverages simple iteration and comparison.
