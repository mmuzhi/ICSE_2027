=== ANSWER ===

To solve this problem, we need to find the largest integer in a list that is not the largest integer in the list. The solution involves identifying the maximum value in the list and then finding the maximum value among the remaining integers that are strictly less than this maximum value.

### Approach
1. **Read Input:** The input consists of an integer `N` followed by a list of `N` integers.
2. **Find Maximum Value:** Determine the maximum value in the list.
3. **Filter and Find Second Maximum:** Create a new list containing all integers from the original list that are strictly less than the maximum value. The largest value in this new list is the answer.

The approach leverages the fact that the problem guarantees the existence of at least one integer not equal to the maximum value. By filtering out all instances of the maximum value and then finding the maximum of the remaining values, we efficiently solve the problem.

### Solution Code
```python
n = int(input().strip())
arr = list(map(int, input().split()))
max_val = max(arr)
result = max([x for x in arr if x < max_val])
print(result)
```

### Explanation
1. **Reading Input:** The first line of input is read as an integer `N`, which denotes the number of elements in the list. The second line is read and split into individual elements, which are converted into integers and stored in a list.
2. **Finding Maximum Value:** The `max()` function is used to find the largest integer in the list.
3. **Filtering and Finding Result:** A new list is created by including all integers from the original list that are less than the maximum value. The `max()` function is then applied to this new list to find the largest integer among them, which is printed as the result.

This approach efficiently narrows down the problem to finding the second largest distinct value in the list by leveraging simple list operations and built-in functions. The solution is optimal with a time complexity of O(N) due to the linear scans involved in finding the maximum and filtering the list.
