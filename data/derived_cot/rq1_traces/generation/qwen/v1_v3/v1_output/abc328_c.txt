Input

The input is given from Standard Input in the following format:

S

l_2 r_2
\vdots

Output

For i = 1, 2, \ldots, Q, the i-th line should contain the answer to the i-th query.

Constraints


- N and Q are integers.
- 1 \leq N, Q \leq 3 \times 10^5
- S is a string of length N consisting of lowercase English letters.
- l_i and r_i are integers.
- 1 \leq l_i \leq r_i \leq N

=== ANSWER ===

To solve this problem, we need to efficiently answer multiple queries about the number of consecutive duplicate characters in a substring of a given string. The key insight is that we can preprocess the string to mark positions where consecutive duplicates occur and then use a prefix sum array to quickly compute the answer for any query.

### Approach
1. **Preprocessing**:
   - **Mark Consecutive Duplicates**: Iterate through the string and create an array where each element indicates whether the current and next characters are the same (1 if they are, 0 otherwise).
   - **Prefix Sum Array**: Construct a prefix sum array from the marked array. This allows us to compute the sum of any subarray in constant time.

2. **Query Processing**:
   - For each query, determine if the substring length is at least 2 (since consecutive duplicates require at least two characters). If not, the answer is 0.
   - If the substring length is sufficient, use the prefix sum array to compute the number of consecutive duplicates in the specified range by subtracting the prefix sums at the boundaries of the range.

### Solution Code
```python
import sys

def main():
    data = sys.stdin.read().splitlines()
    if not data:
        return
    
    first_line = data[0].split()
    N = int(first_line[0])
    Q = int(first_line[1])
    S = data[1].strip()
    
    arr = []
    for i in range(N-1):
        if S[i] == S[i+1]:
            arr.append(1)
        else:
            arr.append(0)
            
    pref = [0] * (len(arr) + 1)
    for i in range(1, len(arr) + 1):
        pref[i] = pref[i-1] + arr[i-1]
        
    out_lines = []
    for _ in range(Q):
        parts = data[2 + _].split()
        if not parts:
            continue
        l = int(parts[0])
        r = int(parts[1])
        if r - 1 >= l:
            ans = pref[r-1] - pref[l-1]
            out_lines.append(str(ans))
        else:
            out_lines.append("0")
            
    sys.stdout.write("\n".join(out_lines))

if __name__ == '__main__':
    main()
```

### Explanation
1. **Reading Input**: The input is read all at once for efficiency, especially given the constraints. The first line contains the string length `N` and the number of queries `Q`. The second line is the string `S`.
2. **Marking Consecutive Duplicates**: The array `arr` is constructed by iterating through `S` and marking positions where consecutive characters are the same.
3. **Prefix Sum Array**: The `pref` array is built from `arr` to allow quick range sum queries. Each element `pref[i]` holds the sum of the first `i` elements of `arr`.
4. **Processing Queries**: For each query, the solution checks if the substring length is at least 2. If so, it calculates the number of consecutive duplicates in the substring using the prefix sum array. The result for each query is collected in a list and printed at the end for efficiency.

This approach ensures that each query is answered in constant time after an initial O(N) preprocessing step, making the solution efficient for large inputs.
