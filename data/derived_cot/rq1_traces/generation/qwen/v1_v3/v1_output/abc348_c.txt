=== ANSWER ===

To solve this problem, we need to determine the optimal color of beans to choose such that the minimum deliciousness of the bean eaten from that color is maximized. The solution involves grouping beans by their color and then finding the minimum deliciousness for each color. The goal is to select the color with the highest minimum deliciousness among all colors.

### Approach
1. **Problem Analysis**: The problem requires selecting a color of beans where, after choosing the color, we eat one bean from that color. The key insight is that for each color, the worst-case scenario (minimum deliciousness) is the smallest deliciousness value among all beans of that color. We need to find the color whose minimum deliciousness is the highest compared to all other colors.
2. **Intuition**: By grouping beans by their color and tracking the minimum deliciousness for each color, we can efficiently determine the optimal color. The solution involves processing each bean, updating the minimum deliciousness for its color, and then finding the maximum of these minimum values.
3. **Algorithm Selection**: We use a dictionary to map each color to its minimum deliciousness. For each bean, we update the dictionary entry for its color if the current bean's deliciousness is smaller than the stored value. After processing all beans, the solution is the maximum value in the dictionary.
4. **Complexity Analysis**: The algorithm processes each bean exactly once, making it O(N) time complexity, where N is the number of beans. The space complexity is O(M), where M is the number of distinct colors, which is optimal for this problem.

### Solution Code
```python
import sys

def main():
    data = sys.stdin.read().splitlines()
    n = int(data[0].strip())
    min_vals = {}
    for i in range(1, n + 1):
        parts = data[i].split()
        a = int(parts[0])
        c = int(parts[1])
        if c in min_vals:
            if a < min_vals[c]:
                min_vals[c] = a
        else:
            min_vals[c] = a
    print(max(min_vals.values()))

if __name__ == '__main__':
    main()
```

### Explanation
1. **Reading Input**: The input is read all at once for efficiency, especially given the constraints. The first line gives the number of beans, and subsequent lines provide each bean's deliciousness and color.
2. **Processing Beans**: For each bean, we check its color in the dictionary. If the color is already present, we update the minimum deliciousness if the current bean's deliciousness is smaller. If the color is not present, we add it to the dictionary with the current bean's deliciousness as the initial minimum.
3. **Finding the Result**: After processing all beans, the dictionary contains the minimum deliciousness for each color. The solution is the maximum value among these minimum values, which is printed as the result.

This approach efficiently groups the beans by color and computes the required minimum values, ensuring optimal performance even for large input sizes.
