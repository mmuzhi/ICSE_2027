=== ANSWER ===

To solve this problem, we need to determine the number of trailing zeros in the binary representation of a given positive integer \( N \). The trailing zeros are the consecutive zeros at the end of the binary form of \( N \). 

### Approach
The key insight here is recognizing that the number of trailing zeros in the binary representation of \( N \) is determined by the highest power of 2 that divides \( N \). For example, if \( N = 12 \) (which is \( 1100 \) in binary), it is divisible by \( 2^2 \), so there are two trailing zeros. 

To find this efficiently, we can use a bitwise operation. The expression \( N \& -N \) isolates the least significant bit set to 1 in \( N \). The number of bits required to represent this isolated bit (using bit_length()) minus one gives the count of trailing zeros. This method leverages the properties of two's complement representation and bitwise operations to achieve the result in constant time.

### Solution Code
```python
n = int(input().strip())
print((n & -n).bit_length() - 1)
```

### Explanation
1. **Reading Input**: The input is read as an integer \( N \).
2. **Bitwise Operation**: The expression \( N \& -N \) computes the two's complement of \( N \) and performs a bitwise AND with \( N \). This operation effectively isolates the least significant set bit in \( N \). For example, if \( N = 12 \) (binary \( 1100 \)), \( -N \) in two's complement is \( ...11110100 \), and the bitwise AND of \( 12 \) and \( -12 \) yields \( 4 \) (binary \( 100 \)).
3. **Bit Length Calculation**: The bit_length() method returns the number of bits required to represent the isolated bit. For the result from step 2, \( 4 \) has a bit length of 3. Subtracting 1 from this value gives the count of trailing zeros, which is 2 in this case.
4. **Output**: The result is printed, which is the count of trailing zeros in the binary representation of \( N \).

This approach efficiently computes the result using bitwise operations and properties of binary numbers, ensuring optimal performance even for large values of \( N \) up to \( 10^9 \).
