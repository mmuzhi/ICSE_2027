class Solution:
    def numberOfGoodSubarraySplits(self, nums: List[int]) -> int:
        mod = 10**9 + 7
        indices = [i for i, x in enumerate(nums) if x == 1]
        n = len(indices)
        if n == 0:
            return 0
        if n == 1:
            return 1
        result = 1

To solve this problem, we need to determine the number of ways to split a binary array into contiguous non-empty subarrays such that each subarray contains exactly one '1'. The solution involves analyzing the positions of the '1's in the array and calculating the product of the gaps between consecutive '1's.

### Approach
1. **Identify Positions of '1's**: First, we traverse the array to collect the indices of all elements that are '1'. This helps us understand where the splits can potentially occur.
2. **Handle Edge Cases**: 
   - If there are no '1's in the array, it's impossible to form any good subarrays, so the answer is 0.
   - If there is exactly one '1', the entire array itself is the only good subarray, so the answer is 1.
3. **Calculate the Number of Ways**: For arrays with more than one '1', the number of ways to split the array is the product of the differences between consecutive '1' indices. This is because each gap between two consecutive '1's can be split in multiple ways, and the total ways is the product of these individual choices.

### Solution Code
```python
class Solution:
    def numberOfGoodSubarraySplits(self, nums: List[int]) -> int:
        mod = 10**9 + 7
        indices = [i for i, x in enumerate(nums) if x == 1]
        n = len(indices)
        if n == 0:
            return 0
        if n == 1:
            return 1
        result = 1
        for i in range(1, n):
            result = (result * (indices[i] - indices[i-1])) % mod
        return result
```

### Explanation
1. **Edge Cases Handling**: The code first checks if there are no '1's in the array (returning 0) or exactly one '1' (returning 1), as these cases have straightforward solutions.
2. **Product Calculation**: For arrays with multiple '1's, the solution calculates the product of the differences between consecutive '1' indices. Each difference represents the number of ways to split the array between two consecutive '1's. The modulo operation ensures the result stays within the required constraints.
3. **Efficiency**: The algorithm efficiently processes the array in linear time, O(n), where n is the length of the array, by traversing it once to collect '1' indices and then processing these indices in a single pass. The space complexity is O(m), where m is the number of '1's, due to storing their indices. This approach is optimal for the given problem constraints.
