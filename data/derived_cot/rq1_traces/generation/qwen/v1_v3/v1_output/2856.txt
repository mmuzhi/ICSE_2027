class Solution:
    def countCompleteSubarrays(self, nums: List[int]) -> int:
        n = len(nums)
        total_distinct = len(set(nums))
        # Helper function to count subarrays with at most k distinct
        def count_at_most(k):
            if k == 0:
                return 0
            freq = {}
            left = 0
            distinct = 0
            count = 0
            for right in range(n):
                # Add the current element
                freq[nums[right]] = freq.get(nums[right], 0) + 1
                if freq[nums[right]] == 1:
                    distinct += 1
                # Shrink the window from the left if distinct count exceeds k
                while distinct > k and left <= right:
                    freq[nums[left]] -= 1
                    if freq[nums[left]] == 0:
                        distinct -= 1
                    left += 1
                # Now, the window [left:right] has <=k distinct
                # All subarrays ending at right and starting from left to right are valid
                count += (right - left + 1)
            return count
        
        # If the entire array has only one distinct, then k=1, then we return total number of subarrays (n*(n+1)//2)
        # But the helper function will handle that.
        return count_at_most(total_distinct) - count_at_most(total_distinct - 1)

### Approach

For example, if the entire array has 3 distinct elements, any subarray must also have exactly those 3 distinct elements to be considered complete.

2. **Intuition**: The solution involves counting all subarrays that have exactly the same number of distinct elements as the entire array. This can be efficiently computed by using a helper function that counts subarrays with at most `k` distinct elements. By calculating the number of subarrays with at most `k` distinct elements and subtracting those with at most `k-1` distinct elements, we obtain the count of subarrays with exactly `k` distinct elements.

3. **Algorithm Selection**: The helper function uses a sliding window (two-pointer) technique to count subarrays with at most `k` distinct elements. The window expands by moving the right pointer and contracts from the left when the number of distinct elements exceeds `k`. For each position of the right pointer, the number of valid subarrays ending at that position is `right - left + 1`.

4. **Complexity Analysis**: The algorithm processes the array twice (once for `k` distinct elements and once for `k-1` distinct elements), each time in O(n) time, where `n` is the length of the array. Thus, the overall time complexity is O(n), which is efficient for the given constraints (n ≤ 1000).

class Solution:
    def countCompleteSubarrays(self, nums: List[int]) -> int:
        n = len(nums)
        total_distinct = len(set(nums))
        
        def count_at_most(k):
            if k == 0:
                return 0

            distinct = 0
            count = 0

                if freq[nums[right]] == 1:

                    if freq[nums[left]] == 0:
                        distinct -= 1
                    left += 1
                count += (right - left + 1)
            return count

3. **Result Calculation**: The result is obtained by subtracting the count of subarrays with at most `k-1` distinct elements from those with at most `k` distinct elements, yielding the number of subarrays with exactly `k` distinct elements, which are the complete subarrays.
