=== ANSWER ===

To solve this problem, we need to remove all contiguous substrings from a given string that start with '(', end with ')', and contain no other parentheses in between. The solution involves identifying all such substrings and removing them from the original string. The key insight is that these substrings must be matched in a nested manner, meaning innermost substrings are removed first before outer ones.

### Approach
1. **Stack Matching**: We use a stack to keep track of the positions of '(' characters. As we iterate through the string, whenever we encounter a ')', we check if there is a matching '(' at the top of the stack. If found, we record the positions of this matched pair.
2. **Difference Array**: To efficiently mark all characters within these matched pairs for removal, we use a difference array. This array helps in counting how many times each character is covered by any matched pair.
3. **Prefix Sum**: After processing all matched pairs, we compute the prefix sum of the difference array to determine which characters are part of any matched pair (i.e., covered by at least one matched pair).
4. **Construct Result**: Finally, we construct the result string by including only those characters not covered by any matched pair.

### Solution Code
```python
import sys

def main():
    data = sys.stdin.read().splitlines()
    n = int(data[0].strip())
    S = data[1].strip()
    
    stack = []
    matched = []
    for i, char in enumerate(S):
        if char == '(':
            stack.append(i)
        elif char == ')':
            if stack:
                j = stack.pop()
                matched.append((j, i))
                
    diff = [0] * (n + 1)
    for (j, i) in matched:
        diff[j] += 1
        if i + 1 <= n:
            diff[i + 1] -= 1
            
    for i in range(1, n):
        diff[i] += diff[i - 1]
        
    result = []
    for i in range(n):
        if diff[i] == 0:
            result.append(S[i])
            
    print(''.join(result))

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The input is read from standard input, which includes the length of the string and the string itself.
2. **Stack Processing**: We iterate through the string, pushing the indices of '(' onto a stack. When a ')' is encountered, if the stack is not empty, we pop the top index (a matching '(') and record the pair.
3. **Difference Array**: For each matched pair, we update the difference array to mark the start and end of the substring to be removed. The difference array helps in efficiently calculating which characters fall within any matched pair.
4. **Prefix Sum Calculation**: The difference array is converted into a coverage array using prefix sums. This array indicates how many matched pairs cover each character in the string.
5. **Result Construction**: Characters not covered by any matched pair (i.e., those with a coverage count of zero) are added to the result string, which is then printed.

This approach efficiently processes the string in linear time, making it suitable for large input sizes up to 200,000 characters. The use of a stack and difference array ensures optimal performance and correctness.
