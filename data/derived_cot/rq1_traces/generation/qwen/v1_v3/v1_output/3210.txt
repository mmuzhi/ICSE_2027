class Solution:
    def beautifulSubstrings(self, s: str, k: int) -> int:
        n = len(s)
        # Define vowels
        vowels = "aeiou"
        # Build prefix array
        prefix = [0]*(n+1)
        for i in range(1, n+1):
            prefix[i] = prefix[i-1] + (1 if s[i-1] in vowels else 0)
        
        # Get valid even lengths
        valid_lengths = []
        # We'll iterate over even lengths from 2 to n
        for L in range(2, n+1, 2):
            half = L // 2
            # Condition2: (half * half) % k == 0
            if (half * half) % k == 0:
                valid_lengths.append(L)
        
        count = 0
        # For each valid length, count substrings with exactly half vowels
        for L in valid_lengths:
            half = L // 2
            # Iterate over starting indices
            for i in range(0, n - L + 1):
                # The substring is from i to i+L-1
                # Number of vowels is prefix[i+L] - prefix[i]
                if prefix[i+L] - prefix[i] == half:
                    count += 1
        
        return count

=== ANSWER ===

To solve this problem, we need to count the number of non-empty beautiful substrings in a given string. A substring is considered beautiful if it meets two conditions: the number of vowels equals the number of consonants, and the product of the number of vowels and consonants is divisible by a given integer \( k \).

### Approach
1. **Problem Analysis**: The key observation is that a beautiful substring must have an even length because the number of vowels must equal the number of consonants. For each even-length substring, we need to check if the number of vowels (and hence consonants) is exactly half the substring's length and if the square of this number is divisible by \( k \).

2. **Prefix Sum Array**: We first precompute a prefix sum array where each entry at index \( i \) represents the number of vowels from the start of the string up to (but not including) the character at position \( i \). This allows us to quickly calculate the number of vowels in any substring.

3. **Valid Lengths**: We determine all even lengths \( L \) (from 2 to the length of the string) for which the square of half the length is divisible by \( k \). These lengths are stored in a list of valid lengths.

4. **Counting Beautiful Substrings**: For each valid length \( L \), we count all substrings of length \( L \) that contain exactly \( L/2 \) vowels. This is done by checking if the difference between the prefix sums at positions \( i \) and \( i + L \) equals \( L/2 \).

### Solution Code
```python
class Solution:
    def beautifulSubstrings(self, s: str, k: int) -> int:
        n = len(s)
        vowels = "aeiou"
        prefix = [0] * (n + 1)
        for i in range(1, n + 1):
            prefix[i] = prefix[i - 1] + (1 if s[i - 1] in vowels else 0)
        
        valid_lengths = []
        for L in range(2, n + 1, 2):
            half = L // 2
            if (half * half) % k == 0:
                valid_lengths.append(L)
        
        count = 0
        for L in valid_lengths:
            half = L // 2
            for i in range(0, n - L + 1):
                if prefix[i + L] - prefix[i] == half:
                    count += 1
        
        return count
```

### Explanation
1. **Prefix Sum Array**: The prefix array is built by iterating through the string. Each entry `prefix[i]` holds the cumulative count of vowels up to the \( i \)-th character (0-based). This allows efficient vowel count queries for any substring.

2. **Valid Lengths**: For each even length \( L \) from 2 to the string length, we check if the square of half the length is divisible by \( k \). If it is, \( L \) is added to the list of valid lengths.

3. **Counting Substrings**: For each valid length \( L \), we slide a window of size \( L \) across the string. For each window starting at index \( i \), the number of vowels is found by subtracting the prefix sum at \( i \) from the prefix sum at \( i + L \). If this difference equals \( L/2 \), the substring is beautiful, and we increment our count.

This approach efficiently narrows down the problem to checking only relevant substring lengths and leverages the prefix sum array for quick vowel count calculations, ensuring optimal performance.
