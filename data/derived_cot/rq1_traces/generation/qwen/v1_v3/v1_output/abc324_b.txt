=== ANSWER ===

To solve this problem, we need to determine if a given positive integer \( N \) can be expressed as \( 2^x \times 3^y \) where \( x \) and \( y \) are non-negative integers. The solution involves checking if \( N \) can be completely factorized into powers of 2 and 3 only.

### Approach
1. **Problem Analysis**: The problem requires us to verify if \( N \) is a product of some powers of 2 and 3. This means after removing all factors of 2 and 3 from \( N \), the remaining number should be 1. If it is, then \( N \) can be expressed in the required form; otherwise, it cannot.
2. **Intuition**: By repeatedly dividing \( N \) by 2 and 3, we can strip away all factors of these primes. If the remaining number is 1, it confirms that \( N \) is composed solely of these primes.
3. **Algorithm Selection**: The algorithm involves two main steps:
   - **Remove all factors of 2** from \( N \) by continuous division.
   - **Remove all factors of 3** from the resulting number.
   - After these steps, if the remaining number is 1, output "Yes"; otherwise, output "No".
4. **Complexity Analysis**: The algorithm efficiently processes \( N \) by leveraging division operations. The number of divisions is logarithmic in \( N \) (base 2 and base 3), making it suitable for large \( N \) (up to \( 10^{18} \)).

### Solution Code
```python
import sys

def main():
    N = int(sys.stdin.readline().strip())
    temp = N
    while temp % 2 == 0:
        temp //= 2
    while temp % 3 == 0:
        temp //= 3
    print("Yes" if temp == 1 else "No")

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The input is read from standard input as an integer \( N \).
2. **Processing \( N \)**:
   - **Divide by 2**: The loop continues dividing \( N \) by 2 as long as the remainder is zero, effectively removing all factors of 2.
   - **Divide by 3**: Similarly, the loop continues dividing the resulting number by 3 until it is no longer divisible.
3. **Check Result**: After removing all factors of 2 and 3, if the remaining number is 1, \( N \) is a product of powers of 2 and 3, so "Yes" is printed. Otherwise, "No" is printed.
This approach efficiently checks the required condition by leveraging simple arithmetic operations, ensuring optimal performance even for large values of \( N \).
