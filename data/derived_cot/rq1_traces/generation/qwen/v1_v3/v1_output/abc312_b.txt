To solve this problem, we need to identify all 9x9 regions within a given grid that satisfy the conditions of a Tak Code. The conditions are that the top-left 3x3 and bottom-right 3x3 regions must be entirely black ('#'), and all cells adjacent (horizontally, vertically, or diagonally) to these two regions must be white ('.'). 

### Approach
1. **Problem Analysis**: The task involves scanning the grid for every possible 9x9 region. For each such region, we need to check two main conditions:
   - The top-left 3x3 and bottom-right 3x3 regions must consist entirely of '#' characters.
   - The 14 adjacent cells (one step in any direction from the two 3x3 regions) must be entirely '.' characters.

2. **Intuition**: For each candidate 9x9 region, we first verify the black regions. If they pass, we then check the adjacent cells to ensure they are white. The adjacent cells are determined by considering all eight possible directions from each cell in the two 3x3 regions, but only those cells that lie within the 9x9 region and are not part of the two 3x3 regions.

3. **Algorithm Selection**: The solution involves nested loops to iterate over all possible top-left corners of 9x9 regions within the grid. For each region, we check the two 3x3 regions for '#' and then verify the adjacent cells are '.'. The adjacent cells are collected into a set to avoid duplicates and ensure exactly 14 cells are checked.

4. **Complexity Analysis**: The algorithm's time complexity is O((N-8)*(M-8)*constant), where N and M are the grid dimensions. Given the constraints (N and M up to 100), this approach is efficient with a maximum of around 8,464 iterations, each involving a constant amount of work.

### Solution Code
```python
import sys

def main():
    data = sys.stdin.read().splitlines()
    if not data:
        return
    
    N, M = map(int, data[0].split())
    grid = []
    for i in range(1, 1+N):
        grid.append(data[i].strip())
    
    results = []
    for i in range(N-8):
        for j in range(M-8):
            valid = True
            # Check top-left 3x3
            for x in range(i, i+3):
                for y in range(j, j+3):
                    if grid[x][y] != '#':
                        valid = False
                        break
                if not valid:
                    break
            if not valid:
                continue
                
            # Check bottom-right 3x3
            for x in range(i+6, i+9):
                for y in range(j+6, j+9):
                    if grid[x][y] != '#':
                        valid = False
                        break
                if not valid:
                    break
            if not valid:
                continue
                
            adj_set = set()
            # For top-left 3x3
            for x in range(i, i+3):
                for y in range(j, j+3):
                    for dx in [-1,0,1]:
                        for dy in [-1,0,1]:
                            if dx == 0 and dy == 0:
                                continue
                            nx, ny = x+dx, y+dy
                            if i <= nx <= i+8 and j <= ny <= j+8:
                                if not (i <= nx <= i+2 and j <= ny <= j+2) and not (i+6 <= nx <= i+8 and j+6 <= ny <= j+8):
                                    adj_set.add((nx, ny))
            
            # For bottom-right 3x3
            for x in range(i+6, i+9):
                for y in range(j+6, j+9):
                    for dx in [-1,0,1]:
                        for dy in [-1,0,1]:
                            if dx == 0 and dy == 0:
                                continue
                            nx, ny = x+dx, y+dy
                            if i <= nx <= i+8 and j <= ny <= j+8:
                                if not (i <= nx <= i+2 and j <= ny <= j+2) and not (i+6 <= nx <= i+8 and j+6 <= ny <= j+8):
                                    adj_set.add((nx, ny))
            
            if len(adj_set) != 14:
                continue
                
            valid_adj = True
            for (x, y) in adj_set:
                if grid[x][y] != '.':
                    valid_adj = False
                    break
                    
            if valid_adj:
                results.append((i+1, j+1))
                
    results.sort(key=lambda x: (x[0], x[1]))
    for (a, b) in results:
        print(f"{a} {b}")
```
