You will be given a competitive programming problem.
There are N people numbered 1, 2, \ldots, N on a two-dimensional plane, and person i is at the point represented by the coordinates (X_i,Y_i).
Person 1 has been infected with a virus. The virus spreads to people within a distance of D from an infected person.
Here, the distance is defined as the Euclidean distance, that is, for two points (a_1, a_2) and (b_1, b_2), the distance between these two points is \sqrt {(a_1-b_1)^2 + (a_2-b_2)^2}.
After a sufficient amount of time has passed, that is, when all people within a distance of D from person i are infected with the virus if person i is infected, determine whether person i is infected with the virus for each i.

Input

The input is given from Standard Input in the following format:
N D
X_1 Y_1
X_2 Y_2
\vdots
X_N Y_N

Output

Print N lines. The i-th line should contain Yes if person i is infected with the virus, and No otherwise.

Constraints


- 1 \leq N, D \leq 2000
- -1000 \leq X_i, Y_i \leq 1000
- (X_i, Y_i) \neq (X_j, Y_j) if i \neq j.
- All input values are integers.

To solve this problem, we need to determine which individuals on a 2D plane are infected with a virus that spreads from an initially infected person (Person 1) to all others within a certain Euclidean distance. The infection spreads transitively, meaning if an infected person is within the given distance of another person, that person gets infected, and this process continues until no more infections can occur.

### Approach
1. **Problem Analysis**: The problem involves checking the Euclidean distance between each pair of points. If the distance between two points is less than or equal to the given threshold D, they are considered connected. The infection spreads from Person 1 to all connected points, and this process repeats for all newly infected individuals until no more infections are possible.
2. **Graph Representation**: We model the problem as an undirected graph where each node represents a person, and edges exist between nodes if the Euclidean distance between their coordinates is ≤ D. This graph helps in efficiently determining the spread of infection.
3. **BFS for Infection Spread**: Using Breadth-First Search (BFS), we start from Person 1 (node 0) and explore all reachable nodes. Each node reached during BFS is marked as infected. The BFS ensures that all possible infections are propagated in the shortest number of steps.
4. **Complexity Analysis**: The graph construction involves checking all pairs of points, resulting in O(N^2) time complexity, where N is the number of people (up to 2000). The BFS operates in O(N + E) time, where E is the number of edges, which in the worst case is also O(N^2). Given the constraints, this approach is efficient.

### Solution Code
```python
import sys
from collections import deque

def main():
    data = sys.stdin.read().split()
    n = int(data[0])
    d_val = int(data[1])
    points = []
    index = 2
    for i in range(n):
        x = int(data[index])
        y = int(data[index+1])
        index += 2
        points.append((x, y))
    
    graph = [[] for _ in range(n)]
    for i in range(n):
        for j in range(i+1, n):
            x1, y1 = points[i]
            x2, y2 = points[j]
            dx = x1 - x2
            dy = y1 - y2
            if dx*dx + dy*dy <= d_val*d_val:
                graph[i].append(j)
                graph[j].append(i)
                
    visited = [False] * n
    q = deque()
    visited[0] = True
    q.append(0)
    
    while q:
        node = q.popleft()
        for neighbor in graph[node]:
            if not visited[neighbor]:
                visited[neighbor] = True
                q.append(neighbor)
                
    for i in range(n):
        if visited[i]:
            print("Yes")
        else:
            print("No")

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The input is read in one go for efficiency, especially given the constraints. The first two values are the number of people (N) and the infection radius (D). The subsequent lines contain the coordinates of each person.
2. **Graph Construction**: For each pair of points, the squared Euclidean distance is computed. If this distance is ≤ D², an undirected edge is added between the corresponding nodes in the graph.
3. **BFS Initialization**: Starting from Person 1 (node 0), the BFS explores all reachable nodes. Each node visited during BFS is marked as infected (visited).
4. **Output**: After BFS completes, the solution checks each node's visited status and prints "Yes" if infected, otherwise "No".

This approach efficiently models the problem using graph theory and leverages BFS to propagate the infection, ensuring optimal performance even for the upper constraint limits.
