To solve this problem, we need to determine if an array of even length can be split into two parts of equal length, where each part contains distinct elements. The solution involves checking the frequency of each element in the array. 

### Approach
1. **Frequency Analysis**: The key observation is that if any element in the array appears more than twice, it is impossible to split the array into two parts where each part has distinct elements. This is because each part can include at most one occurrence of any element. If an element appears three or more times, at least one part will have to include it more than once, violating the distinctness condition.
2. **Check Frequencies**: Using a frequency counter, we check if all elements in the array appear no more than twice. If any element appears more than twice, the answer is immediately `False`. Otherwise, it is possible to split the array as required, so the answer is `True`.

### Solution Code
```python
from collections import Counter

class Solution:
    def isPossibleToSplit(self, nums: List[int]) -> bool:
        freq = Counter(nums)
        for count in freq.values():
            if count > 2:
                return False
        return True
```

### Explanation
1. **Frequency Counter**: The `Counter` from the `collections` module is used to count the occurrences of each element in the array.
2. **Check for Valid Frequencies**: For each element in the frequency dictionary, if the count exceeds 2, the function returns `False` because it's impossible to split the array under the given constraints.
3. **Return Result**: If all elements have counts of 2 or less, the function returns `True`, indicating that the array can be split into two parts, each containing distinct elements.

This approach efficiently checks the feasibility of splitting the array by leveraging frequency analysis, ensuring optimal performance with a time complexity of O(n), where n is the number of elements in the array. The space complexity is also O(n) due to the storage required for the frequency dictionary.
