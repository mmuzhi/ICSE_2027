To solve this problem, we need to count the number of arithmetic subsequences of each length from 1 to N in a given sequence. An arithmetic subsequence is a sequence where the difference between consecutive elements is constant. The solution involves dynamic programming to efficiently count these subsequences for all lengths.

### Approach
1. **Problem Analysis**: The problem requires counting all arithmetic subsequences in a sequence, where subsequences can be of any length from 1 to N. For lengths 1 and 2, every subsequence is arithmetic by definition. For lengths 3 and above, we need to ensure the subsequence has a constant difference between consecutive elements.

2. **Dynamic Programming Setup**: We use a dynamic programming approach where `dp[i][d]` is a dictionary that maps lengths of subsequences ending at index `i` with common difference `d` to their counts. This allows us to efficiently extend subsequences by adding elements that maintain the common difference.

3. **Initialization**: For each element in the sequence, we initialize the two-element subsequences (base case) and then extend these subsequences by considering previous elements that allow maintaining the common difference.

4. **Aggregation**: After processing all elements, we aggregate the counts of subsequences for each length from 3 to N. The counts for lengths 1 and 2 are straightforward as they are derived from combinatorial formulas.

5. **Modulo Operation**: Since the results can be large, all counts are computed modulo 998244353.

### Solution Code
```python
import sys

mod = 998244353

def main():
    data = sys.stdin.read().split()
    if not data:
        return
    n = int(data[0])
    A = list(map(int, data[1:1+n]))
    
    if n == 1:
        print(1)
        return
        
    ans = [0] * (n + 1)
    ans[1] = n % mod
    ans[2] = (n * (n - 1) // 2) % mod

    dp = [dict() for _ in range(n)]
    
    for i in range(n):
        for j in range(i):
            d_val = A[i] - A[j]
            if d_val not in dp[i]:
                dp[i][d_val] = {}
            if 2 not in dp[i][d_val]:
                dp[i][d_val][2] = 0
            dp[i][d_val][2] = (dp[i][d_val].get(2, 0) + 1) % mod
            
            if d_val in dp[j]:
                for length, count_val in dp[j][d_val].items():
                    new_length = length + 1
                    if new_length > n:
                        continue
                    dp[i][d_val][new_length] = (dp[i][d_val].get(new_length, 0) + count_val) % mod

    for length in range(3, n + 1):
        total = 0
        for i in range(n):
            if length in dp[i]:
                for d_val in dp[i]:
                    if length in dp[i][d_val]:
                        total = (total + dp[i][d_val][length]) % mod
        ans[length] = total

    print(" ".join(str(x) for x in ans[1:n+1]))

if __name__ == '__main__':
    main()
```
