=== ANSWER ===

To solve this problem, we need to count the number of substrings in a given string that start and end with a specified character. The key observation here is that any substring starting and ending with the character `c` can be uniquely determined by selecting two positions (start and end) where the character `c` occurs in the string, with the start position being less than or equal to the end position.

### Approach
1. **Problem Analysis**: The problem requires counting all substrings of the given string that start and end with the character `c`. A substring is defined by its start and end indices. For a substring to start and end with `c`, both the start and end characters must be `c`.
2. **Intuition**: The number of valid substrings is determined by the number of ways we can choose two positions (allowing the same position twice) from all occurrences of `c` in the string. This is a combination problem where the number of substrings is given by the formula \( n \times (n + 1) / 2 \), where \( n \) is the count of `c` in the string.
3. **Algorithm Selection**: The algorithm involves:
   - Counting the number of times `c` appears in the string.
   - Applying the formula \( n \times (n + 1) / 2 \) to compute the total number of valid substrings.
4. **Complexity Analysis**: The algorithm runs in \( O(n) \) time, where \( n \) is the length of the string, because it requires a single pass through the string to count the occurrences of `c`. The space complexity is \( O(1) \) as no additional data structures are used.

### Solution Code
```python
class Solution:
    def countSubstrings(self, s: str, c: str) -> int:
        n = s.count(c)
        return n * (n + 1) // 2
```

### Explanation
- **Problem Analysis**: The problem is to count all substrings in a string that start and end with a specific character. The solution leverages the observation that each valid substring is defined by two positions (start and end) where the character `c` occurs.
- **Intuition**: By counting the occurrences of `c` in the string, we can determine how many valid substrings exist. Each pair of positions (including the same position twice) forms a valid substring.
- **Algorithm**: The algorithm counts the number of `c` characters in the string and uses the formula \( n \times (n + 1) / 2 \) to compute the total substrings. This formula accounts for all possible pairs of start and end positions (including single-character substrings).
- **Complexity**: The solution efficiently computes the result in linear time with constant space, making it optimal for large input sizes as specified in the constraints.
