=== ANSWER ===

To solve this problem, we need to determine the minimal time required to type a given string using a keyboard with three keys: 'a', Shift, and Caps Lock. The Caps Lock key toggles the case of the letters typed. The challenge is to find the optimal sequence of key presses to type the string with minimal time, considering the costs associated with each key press.

### Approach
1. **Problem Analysis**: The problem involves typing a string consisting of 'A' and 'a' using three actions: pressing 'a' (cost X), pressing 'a' and Shift (cost Y), or pressing Caps Lock (cost Z). The state of the Caps Lock (on or off) affects which action produces the desired character. The goal is to minimize the total time spent.

2. **Dynamic Programming (DP) Setup**: We use a DP approach where `dp0` and `dp1` represent the minimal time to be in state 0 (Caps Lock off) or state 1 (Caps Lock on) before processing each character in the string. The initial state is Caps Lock off (`dp0 = 0`), and the cost to switch to Caps Lock on is `Z` (`dp1 = Z`).

3. **State Transitions**: For each character in the string, we calculate the minimal time required to type it from both states (Caps Lock on or off). For each state, we consider two options:
   - **Without toggling Caps Lock**: Use the current state to determine the cost of typing the character (either action 1 or 2).
   - **With toggling Caps Lock**: Switch the state (cost Z) and then type the character using the appropriate action.

4. **Update DP States**: After processing each character, update `dp0` and `dp1` based on the minimal time required to reach each state from the previous states.

5. **Result Extraction**: The answer is the minimal time between the two states after processing all characters (`min(dp0, dp1)`).

### Solution Code
```python
X, Y, Z = map(int, input().split())
S = input().strip()

if S == "":
    print(0)
    exit(0)

dp0 = 0
dp1 = Z

for c in S:
    if c == 'a':
        cost0_0 = X
        cost0_1 = Z + Y
        cost1_0 = Y
        cost1_1 = Z + X
    else:
        cost0_0 = Y
        cost0_1 = Z + X
        cost1_0 = X
        cost1_1 = Z + Y

    new_dp0 = min(dp0 + cost0_0, dp1 + cost1_1)
    new_dp1 = min(dp0 + cost0_1, dp1 + cost1_0)
    
    dp0, dp1 = new_dp0, new_dp1

print(min(dp0, dp1))
```
