Calculate the sum \displaystyle\sum_{i=1}^{N-1}\displaystyle\sum_{j=i+1}^N \text{dist}(P_i, P_j).

Constraints


- 2 \leq N \leq 2 \times 10^5
- 0 \leq X_i, Y_i \leq 10^8
- For i \neq j, (X_i, Y_i) \neq (X_j, Y_j)
- All input values are integers.

To solve this problem, we need to calculate the sum of the minimal number of jumps required for a rabbit to move from one point to another on a 2D plane, where each jump can move the rabbit in one of four diagonal directions. The distance between two points is defined as the minimal number of such jumps required to reach the destination, or zero if it's impossible.

### Approach
1. **Problem Analysis**: The rabbit can move in four diagonal directions, changing both coordinates by ±1 in each jump. The key observation is that the minimal number of jumps between two points (x1, y1) and (x2, y2) is determined by the parity of the differences in their coordinates. Specifically, the sum (x2 - x1 + y2 - y1) must be even for a path to exist. If this condition is met, the minimal number of jumps is the maximum of the absolute differences in the x and y coordinates.

2. **Insight**: By transforming the problem into considering the sum and difference of the coordinates (u = x + y, v = x - y), we can simplify the distance calculation. The minimal jumps required is (|u2 - u1| + |v2 - v1|) / 2, provided the parity condition holds.

3. **Algorithm Selection**: 
   - **Grouping by Parity**: Points are grouped based on the parity of their (x + y) value. Only points within the same group can have a valid path.
   - **Sum of Absolute Differences**: For each group, compute the sum of absolute differences for all pairs of u and v values. The total distance for the group is derived from these sums.

4. **Complexity Analysis**: The algorithm involves sorting the u and v values for each group, which has a time complexity of O(N log N) due to the sorting step. Given the constraints (N ≤ 2 × 10^5), this approach is efficient and feasible.

### Solution Code
```python
import sys

def main():
    data = sys.stdin.read().strip().split()
    if not data:
        return
    
    n = int(data[0])
    points = []
    index = 1
    for _ in range(n):
        x = int(data[index])
        y = int(data[index + 1])
        index += 2
        points.append((x, y))
    
    group0 = []
    group1 = []
    
    for x, y in points:
        u = x + y
        v = x - y
        if u % 2 == 0:
            group0.append((u, v))
        else:
            group1.append((u, v))
            
    def sum_abs_diff(arr):
        arr.sort()
        total = 0
        prefix = 0
        for i, val in enumerate(arr):
            total += val * i - prefix
            prefix += val
        return total

    total_ans = 0
    
    for group in [group0, group1]:
        if len(group) < 2:
            continue
        u_list = [u for u, v in group]
        v_list = [v for u, v in group]
        sum_u = sum_abs_diff(u_list)
        sum_v = sum_abs_diff(v_list)
        total_ans += (sum_u + sum_v) // 2
        
    print(total_ans)

if __name__ == "__main__":
    main()
```
