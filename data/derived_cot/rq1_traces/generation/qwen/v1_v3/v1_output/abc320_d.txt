Input

The input is given from Standard Input in the following format:
N M
A_1 B_1 X_1 Y_1

- 1 \leq N \leq 2\times 10^5

- 1\leq A_i,  B_i \leq N
- A_i \neq B_i
- -10^9 \leq X_i,Y_i \leq 10^9
- All input values are integers.
- The given information is consistent.

2. **BFS Traversal**: Starting from Person 1 at the origin (0, 0), we perform a Breadth-First Search (BFS) to explore all reachable nodes. For each node visited, we compute its coordinates based on the coordinates of its predecessor and the edge weight. If a node is revisited, we ensure its coordinates match those derived from the new path (though the problem guarantees consistency, this step is theoretically important).
