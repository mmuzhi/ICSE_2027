To solve this problem, we need to find the maximum length of all alternating subarrays present in a given array. A subarray is considered alternating if it meets specific conditions: it must be of length greater than 1, and the elements alternate between increasing and decreasing by 1 in a specific pattern. 

### Approach
1. **Problem Analysis**: The key observation is that an alternating subarray must start with a sequence where each subsequent element alternates between adding 1 and subtracting 1 from the previous element. For example, the subarray [3, 4, 3, 4] alternates between 3 and 4, starting with an increase by 1 and then a decrease by 1, and so on.

2. **Intuition**: For each element in the array, we can check how far we can extend the subarray starting at that element while maintaining the alternating pattern. The pattern requires that the difference between consecutive elements alternates between +1 and -1, starting with +1.

3. **Algorithm Selection**: We use a nested loop approach where the outer loop runs from each starting index, and the inner loop extends the subarray from the starting index, checking the alternating condition for each subsequent element. If the condition is met, we update the maximum length found so far. If the condition fails, we break the inner loop early since further elements cannot form a valid alternating subarray starting from the current index.

4. **Complexity Analysis**: The algorithm has a time complexity of O(n^2) in the worst case, where n is the length of the array. This is efficient given the constraint that the array length is at most 100, making the solution feasible.

### Solution Code
```python
class Solution:
    def alternatingSubarray(self, nums: List[int]) -> int:
        n = len(nums)
        max_len = 0
        for i in range(n):
            for j in range(i + 1, n):
                step = j - i
                if (step - 1) % 2 == 0:
                    expected = 1
                else:
                    expected = -1
                if nums[j] - nums[j - 1] == expected:
                    current_length = step + 1
                    if current_length > max_len:
                        max_len = current_length
                else:
                    break
        return max_len if max_len != 0 else -1
```

### Explanation
1. **Initialization**: We start by initializing `max_len` to 0, which will keep track of the longest alternating subarray found.
2. **Outer Loop**: The outer loop iterates over each element in the array, treating it as the starting element of a potential alternating subarray.
3. **Inner Loop**: For each starting element, the inner loop checks subsequent elements to see if they form an alternating sequence. The variable `step` tracks the position in the subarray relative to the start.
4. **Condition Check**: The expected difference between consecutive elements alternates between +1 and -1 based on the `step` value. If the actual difference matches the expected value, the subarray length is updated; otherwise, the inner loop breaks early.
5. **Result Determination**: After processing all possible subarrays, if `max_len` remains 0, it means no valid alternating subarray was found, and we return -1. Otherwise, we return the length of the longest alternating subarray found.

This approach efficiently checks all possible alternating subarrays by leveraging early termination when the condition fails, ensuring optimal performance for the given constraints.
