To solve this problem, we need to calculate the sum of the squares of the distinct counts of all possible contiguous subarrays of a given array. The distinct count of a subarray is the number of unique elements in that subarray. 

### Approach
1. **Problem Analysis**: The problem requires iterating over all possible contiguous subarrays of the input array, computing the distinct count for each subarray, squaring it, and summing these squares. Given the constraints (array length up to 100), a straightforward approach is feasible.
2. **Intuition**: For each starting index of the subarray, we can extend the subarray to include subsequent elements, maintaining a set of encountered elements to track distinct values. The distinct count is updated dynamically as new elements are added, and the square of this count is added to the result.
3. **Algorithm Selection**: We use nested loops where the outer loop fixes the starting index of the subarray, and the inner loop extends the subarray to include elements up to the current ending index. A set is used to keep track of distinct elements encountered in the current subarray. The distinct count is updated incrementally, and its square is added to the result for each subarray.
4. **Complexity Analysis**: The algorithm involves two nested loops, resulting in O(n^2) time complexity, where n is the length of the array. Each inner loop operation (checking and updating the set) is O(1) on average, making the overall complexity acceptable for the given constraints.

### Solution Code
```python
class Solution:
    def sumCounts(self, nums: List[int]) -> int:
        n = len(nums)
        result = 0
        for i in range(n):
            s = set()
            distinct_count = 0
            for j in range(i, n):
                if nums[j] not in s:
                    s.add(nums[j])
                    distinct_count += 1
                result += distinct_count * distinct_count
        return result
```

### Explanation
1. **Initialization**: The solution starts by initializing the result variable to zero, which will accumulate the sum of squares of distinct counts.
2. **Outer Loop**: The outer loop iterates over each possible starting index `i` of the subarray.
3. **Inner Loop**: For each starting index `i`, the inner loop extends the subarray from `i` to `j` (inclusive), where `j` ranges from `i` to the end of the array.
4. **Tracking Distinct Elements**: A set is used to track elements encountered in the current subarray. For each new element `nums[j]`, if it is not already in the set, it is added, and the distinct count is incremented.
5. **Result Update**: For each subarray defined by indices `i` and `j`, the square of the distinct count is added to the result.
6. **Return Result**: After processing all subarrays, the accumulated result is returned.

This approach efficiently computes the desired sum by leveraging dynamic updates of the distinct count and incremental processing, ensuring optimal performance for the given problem constraints.
