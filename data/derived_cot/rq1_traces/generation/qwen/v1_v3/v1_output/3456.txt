To solve this problem, we need to find the maximum length of a subsequence in an array where there are at most `k` indices where consecutive elements differ. The solution involves dynamic programming to efficiently track the longest subsequences ending with specific numbers and within given change constraints.

### Approach
1. **Dynamic Programming Setup**: We use a dynamic programming (DP) approach where `dp[j]` is a dictionary that maps each number to the length of the longest subsequence ending with that number and having exactly `j` changes.
2. **Iterate Through Elements**: For each element in the array, we update the DP tables for all possible change counts (from `k` down to 0). This involves two main steps:
   - **Same Number Extension**: Extend subsequences ending with the same number without increasing the change count.
   - **Different Number Extension**: Extend subsequences ending with a different number, increasing the change count by 1 if within the allowed limit.
3. **Track Maximum Length**: After processing all elements, the answer is the maximum value found in any of the DP tables (`dp[0]` to `dp[k]`).

### Solution Code
```python
class Solution:
    def maximumLength(self, nums: List[int], k: int) -> int:
        dp = [dict() for _ in range(k+1)]
        
        for num in nums:
            prev_dp = [dict(d) for d in dp]
            for j in range(k, -1, -1):
                candidate_same = 0
                if num in prev_dp[j]:
                    candidate_same = prev_dp[j][num] + 1
                
                candidate_diff = 0
                if j > 0:
                    best_from_prev = 0
                    for num_prev, val_prev in prev_dp[j-1].items():
                        if num_prev != num and val_prev > best_from_prev:
                            best_from_prev = val_prev
                    candidate_diff = best_from_prev + 1
                
                best_candidate = max(candidate_same, candidate_diff)
                if best_candidate == 0:
                    best_candidate = 1
                
                dp[j][num] = best_candidate
        
        ans = 0
        for j in range(k+1):
            if not dp[j]:
                continue
            ans = max(ans, max(dp[j].values()))
        return ans
```

### Explanation
1. **Initialization**: The DP array `dp` is initialized to store dictionaries for each possible change count from 0 to `k`.
2. **Processing Each Element**: For each number in the array:
   - **Previous State Copy**: The state of the DP tables from the previous iteration is saved to avoid overwriting during the current update.
   - **Same Number Extension**: For each change count `j`, if the current number exists in the previous state, the subsequence length is incremented by 1 without changing `j`.
   - **Different Number Extension**: For each change count `j` (greater than 0), the algorithm checks all numbers in the previous state for `j-1` to find the longest subsequence that can be extended by the current number, increasing `j` by 1.
3. **Result Calculation**: After processing all elements, the maximum subsequence length is found by examining all DP tables (`dp[0]` to `dp[k]`).

This approach efficiently tracks the longest valid subsequences by leveraging dynamic programming and careful state management, ensuring optimal performance even for the upper constraint limits.
