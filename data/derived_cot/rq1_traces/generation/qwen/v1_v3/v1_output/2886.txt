Since the fourth character is an 'i', the text gets reversed and becomes "rts".
After the fifth character, the text is "rtsn". 
After the sixth character, the text is "rtsng". 
Therefore, we return "rtsng".

After the sixth character, the text is "pont". 
After the seventh character, the text is "ponte". 
After the eighth character, the text is "ponter". 
Therefore, we return "ponter".

=== ANSWER ===

To solve this problem, we need to simulate the process of typing a string on a faulty keyboard where each 'i' character reverses the current string. The solution involves processing each character of the input string and modifying the current string accordingly. Non-'i' characters are appended directly to the current string, while 'i' characters trigger a reversal of the current string.

### Approach
1. **Initialization**: Start with an empty string to represent the current state of the typed text.
2. **Iterate through each character** in the input string:
   - For each character, if it is not 'i', append it to the current string.
   - If the character is 'i', reverse the current string.
3. **Return the final string** after processing all characters.

This approach efficiently handles each character in sequence, updating the current string either by appending the character or reversing it. The solution leverages the fact that reversing a string is a straightforward operation, especially given the constraint that the input string length does not exceed 100 characters.

### Solution Code
```python
class Solution:
    def finalString(self, s: str) -> str:
        current = ""
        for char in s:
            if char != 'i':
                current += char
            else:
                current = current[::-1]
        return current
```

### Explanation
- **Initialization**: The variable `current` is initialized as an empty string to start building the result.
- **Processing Characters**: For each character in the input string `s`:
  - If the character is not 'i', it is directly appended to `current`.
  - If the character is 'i', the current string is reversed using slicing (`current[::-1]`).
- **Result**: After processing all characters, the `current` string is returned, which now contains the final result after all reversals and appends.

This approach ensures that each operation is performed in linear time relative to the input size, making it efficient and easy to understand. The use of string slicing for reversal is both concise and effective for the given problem constraints.
