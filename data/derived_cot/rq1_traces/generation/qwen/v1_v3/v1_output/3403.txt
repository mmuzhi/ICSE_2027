To solve this problem, we need to partition a given string into the minimum number of balanced substrings. A balanced substring is defined as a substring where each character appears the same number of times. The solution involves using dynamic programming to efficiently explore all possible partitions and determine the minimal number of balanced substrings required.

### Approach
1. **Dynamic Programming Setup**: We use a dynamic programming array `dp` where `dp[i]` represents the minimum number of partitions needed for the substring `s[0:i]` (the first `i` characters of the string). The base case is `dp[0] = 0`, indicating an empty string requires zero partitions.

2. **Iterate Through the String**: For each position `i` in the string, we consider all possible substrings ending at `i-1` (from `j = i-1` down to `0`). For each substring `s[j:i]`, we check if it is balanced by maintaining a frequency array of characters in the substring.

3. **Check for Balanced Substring**: For each substring `s[j:i]`, we update the frequency of each character. We then determine the minimum and maximum frequencies of the characters in the substring. If these frequencies are equal, the substring is balanced, and we update `dp[i]` to be the minimum of its current value or `dp[j] + 1`.

4. **Early Termination**: If at any point the entire substring `s[0:i]` is found to be balanced (i.e., `dp[i]` becomes 1), we can terminate the inner loop early since this is the minimal possible partition count.

### Solution Code
```python
class Solution:
    def minimumSubstringsInPartition(self, s: str) -> int:
        n = len(s)
        dp = [float('inf')] * (n + 1)
        dp[0] = 0

            freq = [0] * 26

                idx = ord(s[j]) - ord('a')

```

### Explanation
1. **Initialization**: The `dp` array is initialized with `inf` to represent that initially, no partitions are known for any substring except the empty string (`dp[0] = 0`).

2. **Dynamic Programming Loop**: For each position `i` from 1 to `n`, we consider all possible substrings ending at `i-1` by iterating `j` from `i-1` down to `0`. The frequency array `freq` is updated for each character in the substring `s[j:i]`.

3. **Balanced Substring Check**: For each substring, the minimum and maximum frequencies of characters are computed. If these frequencies are equal, the substring is balanced, and `dp[i]` is updated to reflect the minimal partitions required up to `i`.

4. **Early Termination**: If the entire substring `s[0:i]` is balanced (i.e., `dp[i]` becomes 1), the inner loop breaks early as no further partitions can yield a better result.

This approach efficiently explores all possible partitions using dynamic programming and frequency checks, ensuring the minimal number of balanced substrings is found while optimizing for early termination when possible.
