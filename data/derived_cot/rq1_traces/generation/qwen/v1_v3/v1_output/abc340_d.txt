=== ANSWER ===

To solve this problem, we need to find the minimum time required to reach stage N from stage 1 in a game where each stage allows two possible actions: progressing to the next stage or jumping to another stage, each with a specific time cost. The solution involves modeling the problem as a directed graph where nodes represent stages, and edges represent the actions with their respective time costs. We then use Dijkstra's algorithm to find the shortest path from stage 1 to stage N in this graph.

### Approach
1. **Graph Construction**: The game's stages are represented as nodes in a graph. Each node (except the last) has two outgoing edges: one to the next stage (node i+1) with cost A_i, and another to stage X_i (node X_i - 1) with cost B_i.
2. **Dijkstra's Algorithm**: We use Dijkstra's algorithm to find the shortest path from node 0 (stage 1) to node N-1 (stage N). The algorithm efficiently explores the graph by always expanding the least costly path first, ensuring the first time we reach node N-1 is via the shortest path.
3. **Priority Queue**: A priority queue (min-heap) is used to manage the nodes to be explored, prioritized by the current minimum time to reach each node. This ensures that once a node is processed, the shortest path to it has been found.

### Solution Code
```python
import sys
import heapq

def main():
    data = sys.stdin.read().splitlines()
    if not data:
        return
    
    n = int(data[0].strip())
    graph = [[] for _ in range(n)]
    
    for i in range(1, n):
        parts = data[i].split()
        if not parts:
            continue
        a = int(parts[0])
        b = int(parts[1])
        x = int(parts[2])
        graph[i-1].append((i, a))
        graph[i-1].append((x-1, b))
    
    INF = 10**18
    dist = [INF] * n
    dist[0] = 0
    heap = [(0, 0)]
    
    while heap:
        d, node = heapq.heappop(heap)
        if d != dist[node]:
            continue
        if node == n-1:
            print(d)
            return
        for neighbor, weight in graph[node]:
            new_d = d + weight
            if new_d < dist[neighbor]:
                dist[neighbor] = new_d
                heapq.heappush(heap, (new_d, neighbor))
    
    print(dist[n-1] if n > 1 else 0)

if __name__ == '__main__':
    main()
```

### Explanation
1. **Graph Construction**: The graph is built by reading each stage's data (A_i, B_i, X_i) and adding edges from the current stage to the next stage (i+1) and to the stage specified by X_i (converted to 0-based index).
2. **Dijkstra's Algorithm**: The algorithm starts from node 0 (stage 1) with an initial time of 0. It processes each node by exploring its neighbors, updating their shortest path times if a shorter path is found.
3. **Priority Queue**: The priority queue ensures that nodes are processed in order of increasing path cost. Once the target node (N-1) is reached, the algorithm outputs the minimum time and terminates, as Dijkstra's algorithm guarantees the shortest path upon first arrival at the target node.

This approach efficiently handles the problem by leveraging graph theory and Dijkstra's algorithm, ensuring optimal performance even for large input sizes.
