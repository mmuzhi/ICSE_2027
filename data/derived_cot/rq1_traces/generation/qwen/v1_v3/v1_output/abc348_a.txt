Takahashi will have N penalty kicks in a soccer match.
For the i-th penalty kick, he will fail if i is a multiple of 3, and succeed otherwise.
Print the results of his penalty kicks.

Input

The input is given from Standard Input in the following format:
N

Output

Print a string of length N representing the results of Takahashi's penalty kicks. The i-th character (1 \leq i \leq N) should be o if Takahashi succeeds in the i-th penalty kick, and x if he fails.

Constraints


- 1 \leq N \leq 100
- All inputs are integers.

Sample Input 1

7

Sample Output 1

ooxooxo

Takahashi fails the third and sixth penalty kicks, so the third and sixth characters will be x.

=== ANSWER ===

To solve this problem, we need to determine the success or failure of Takahashi's penalty kicks based on their position in the sequence. The key observation is that Takahashi fails a penalty kick if its position is a multiple of 3; otherwise, he succeeds. 

### Approach
1. **Problem Analysis**: The problem requires processing each penalty kick from 1 to N. For each position i, we check if i is divisible by 3. If it is, the result is 'x'; otherwise, it's 'o'.
2. **Algorithm Selection**: We use a loop to iterate through each number from 1 to N. For each number, we check divisibility by 3 and append the corresponding character ('o' or 'x') to a result string.
3. **Complexity Analysis**: The algorithm runs in O(N) time, where N is the number of penalty kicks, as it processes each kick exactly once. The space complexity is O(N) to store the result string.

### Solution Code
```python
def main():
    N = int(input().strip())
    result = []
    for i in range(1, N + 1):
        if i % 3 == 0:
            result.append('x')
        else:
            result.append('o')
    print(''.join(result))

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The input is read as an integer N, which represents the number of penalty kicks.
2. **Loop Through Positions**: For each position from 1 to N, we check if the position is a multiple of 3 using the modulus operator. If it is, we append 'x' to the result list; otherwise, we append 'o'.
3. **Output Result**: After processing all positions, the result list is joined into a single string and printed. This string consists of 'o's and 'x's corresponding to the success or failure of each penalty kick in sequence.

This approach efficiently checks each position and constructs the result string in linear time, making it optimal for the given constraints.
