=== ANSWER ===

To solve this problem, we need to extract all elements from a given sequence that are multiples of a specified integer K, divide each of these multiples by K, and then print the resulting quotients in ascending order. 

### Approach
1. **Input Reading**: Read the integers N (length of the sequence) and K (the divisor) from the first line of input. Then, read the sequence of integers from the second line.
2. **Filtering Multiples**: Iterate through each number in the sequence. For each number, check if it is a multiple of K (i.e., the remainder when divided by K is zero). If it is, divide the number by K and store the quotient in a result list.
3. **Sorting Results**: Since the problem requires the quotients to be printed in ascending order, sort the result list. Although the original sequence is given in ascending order, sorting ensures the quotients are in the required order regardless of the extraction order.
4. **Output**: Convert each quotient in the sorted result list to a string and join them with spaces to form the final output string.

### Solution Code
```python
def main():
    n, k = map(int, input().split())
    arr = list(map(int, input().split()))
    
    result = []
    for num in arr:
        if num % k == 0:
            result.append(num // k)
            
    result.sort()
    
    print(" ".join(map(str, result)))

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The first line of input is read to get the values of N (number of elements in the sequence) and K (the divisor). The second line is read to obtain the sequence of integers.
2. **Processing Elements**: For each number in the sequence, the code checks if it is divisible by K. If it is, the number is divided by K, and the quotient is added to the result list.
3. **Sorting and Output**: The result list, containing the quotients, is sorted to ensure ascending order. The sorted quotients are then converted to strings and joined into a single string separated by spaces, which is printed as the final output.

This approach efficiently processes the input, filters the relevant elements, and ensures the output meets the specified requirements. The solution handles all edge cases, including sequences where all elements are multiples of K, by leveraging sorting to maintain ascending order.
