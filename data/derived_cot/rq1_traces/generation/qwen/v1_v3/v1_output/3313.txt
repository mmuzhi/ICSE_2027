The strength of the selected subarrays is defined as an alternating sum where each subarray's contribution is multiplied by a decreasing coefficient and an alternating sign. 

### Approach
1. **Problem Analysis**: The problem requires selecting k disjoint contiguous subarrays from an array to maximize their strength. The strength is calculated by summing each subarray's elements, multiplying by a coefficient that decreases from k to 1 and alternates in sign. The challenge is to choose these subarrays optimally to maximize the total strength.

2. **Key Insight**: The optimal assignment of subarrays to coefficients involves sorting both the subarray sums and the coefficients in the same order (both ascending or both descending) and then computing the dot product. This ensures that the largest subarray sums are paired with the largest coefficients (when positive) and the smallest subarray sums with the smallest coefficients (when negative).

3. **Dynamic Programming (DP) Setup**: We use a DP approach where `dp[i][j]` represents the maximum strength achievable from the first `i` elements with `j` subarrays selected. The DP state transitions involve either including the current element in a new subarray or extending an existing subarray.

4. **Coefficient Handling**: The coefficients are precomputed based on their position (1-based index) and sorted in descending order. The sorted coefficients are then used to compute the dot product with the sorted subarray sums.

5. **Complexity**: The algorithm runs in O(n*k) time and space, which is feasible given the constraints (n*k ≤ 10^6).

### Solution Code
```python
class Solution:
    def maximumStrength(self, nums: List[int], k: int) -> int:
        n = len(nums)
        # Calculate the coefficients for positions 1 to k
        coef = []
        for i in range(k):
            sign = 1 if (i % 2 == 0) else -1
            coef.append(sign * (k - i))
        coef_sorted = sorted(coef, reverse=True)
        
        # If k is 0, return 0
        if k == 0:
            return 0
        
        # If k is greater than n, we can't select k subarrays, but problem states k<=n
        # Initialize DP table: dp[i][j] will store the maximum strength from first i elements with j subarrays
        # We use -10**18 as negative infinity for unreachable states
        dp = [[-10**18] * (k+1) for _ in range(n+1)]
        dp[0][0] = 0
        
        # Iterate over the array
        for i in range(1, n+1):
            for j in range(0, k+1):
                # Option 1: not use nums[i-1]
                dp[i][j] = dp[i-1][j]
                
                # Option 2: use nums[i-1] as a new subarray
                if j > 0:
                    new_val = dp[i-1][j-1] + nums[i-1]
                    if new_val > dp[i][j]:
                        dp[i][j] = new_val
                
                # Option 3: extend the previous subarray (if any)
                if j > 0 and i > 1:
                    extend_val = dp[i-1][j] + nums[i-1]
                    if extend_val > dp[i][j]:
                        dp[i][j] = extend_val
        
        # After processing all elements, extract the best j subarrays
        best_sums = []
        for j in range(1, k+1):
            best_sums.append(dp[n][j])
        
        # Sort the best_sums in descending order
        best_sums.sort(reverse=True)
        strength = 0
        for i in range(k):
            strength += best_sums[i] * coef_sorted[i]
        
        return strength
```
