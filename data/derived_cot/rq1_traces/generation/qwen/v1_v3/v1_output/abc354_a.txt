=== ANSWER ===

To solve this problem, we need to determine the first day when a plant's height exceeds Takahashi's given height, H. The plant's height grows by doubling each day, starting from 0 cm on day 0. Specifically, on each day i, the plant's height increases by \(2^i\) cm during the night. The morning height after day d is the sum of \(2^0 + 2^1 + \ldots + 2^{d-1}\), which simplifies to \(2^d - 1\) cm.

### Approach
1. **Problem Analysis**: The plant's height on day d is \(2^d - 1\) cm. We need the smallest day d such that \(2^d - 1 > H\). This translates to finding the smallest integer d where \(2^d > H + 1\).
2. **Insight**: The smallest d satisfying \(2^d > H + 1\) can be found using the bit length of \(H + 1\). The bit length of a number \(n\) is the number of bits required to represent \(n\) in binary. For \(n = H + 1\), the bit length directly gives the smallest d such that \(2^d > H + 1\).
3. **Algorithm Selection**: The solution leverages the bit length property of integers. By computing \((H + 1).bit\_length()\), we efficiently determine the required day d without resorting to loops or logarithmic calculations, ensuring optimal performance even for large H (up to \(10^9\)).

### Solution Code
```python
H = int(input())
print((H + 1).bit_length())
```

### Explanation
1. **Input Handling**: The input is read as an integer H, representing Takahashi's height.
2. **Bit Length Calculation**: The expression \((H + 1).bit\_length()\)) computes the number of bits required to represent \(H + 1\) in binary. This value is the smallest integer d such that \(2^d > H + 1\), which ensures \(2^d - 1 > H\).
3. **Output**: The result is printed, which is the first day d where the plant's height strictly exceeds H.

This approach efficiently leverages binary representation properties to solve the problem in constant time, making it optimal for large input sizes.
