=== ANSWER ===

To solve this problem, we need to determine the number of pairs of people who never stood next to each other in any of the given photos. The solution involves analyzing all possible pairs of people and checking if they appeared as adjacent in any photo. 

### Approach
1. **Problem Analysis**: The problem requires counting all unordered pairs of people (x, y) where x and y never stood next to each other in any photo. The key observation is that if a pair (x, y) appears as adjacent in any photo, they are not in a bad mood. Otherwise, they are.
2. **Intuition**: The total number of possible pairs is given by the combination formula C(N, 2) = N*(N-1)/2. For each photo, we can extract all adjacent pairs (considering both orders as the same by storing them in a canonical form, i.e., (min(x, y), max(x, y))).
3. **Algorithm Selection**: 
   - Read the input values for the number of people (N) and photos (M).
   - For each photo, read the sequence of people and extract all adjacent pairs, storing them in a set to avoid duplicates.
   - The number of bad mood pairs is the total possible pairs minus the number of distinct adjacent pairs found in all photos.
4. **Complexity Analysis**: The algorithm processes each photo once, and for each photo, it processes N-1 adjacent pairs. With N and M both up to 50, the total operations are manageable (50*49 = 2450), making the solution efficient for the given constraints.

### Solution Code
```python
def main():
    import sys
    input = sys.stdin.read().split()
    n = int(input[0])
    m = int(input[1])
    photos = []
    index = 2
    for _ in range(m):
        row = list(map(int, input[index:index+n]))
        photos.append(row)
        index += n
    
    total_pairs = n * (n - 1) // 2
    adjacent_set = set()
    
    for photo in photos:
        for i in range(len(photo) - 1):
            a = photo[i]
            b = photo[i + 1]
            key = (min(a, b), max(a, b))
            adjacent_set.add(key)
    
    bad_mood_pairs = total_pairs - len(adjacent_set)
    print(bad_mood_pairs)

if __name__ == '__main__':
    main()
```

### Explanation
1. **Reading Input**: The input is read in one go for efficiency, especially given the constraints. The first two values are N (number of people) and M (number of photos).
2. **Processing Photos**: For each photo, the sequence of people is processed to extract all adjacent pairs. Each pair is stored as a tuple with the smaller number first to ensure consistency (e.g., (1, 2) instead of (2, 1)).
3. **Counting Pairs**: The total number of possible pairs is calculated using the combination formula. The number of distinct adjacent pairs from all photos is subtracted from this total to get the number of pairs that never stood next to each other.
4. **Output**: The result, which is the count of bad mood pairs, is printed.

This approach efficiently checks all possible pairs and leverages set operations to avoid counting duplicates, ensuring optimal performance for the problem constraints.
