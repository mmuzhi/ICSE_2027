To solve this problem, we need to count the number of pairs of indices (i, j) where i < j such that the combination of two notes (b_i, b_j) equals the combination (b_j, b_i). Here, each note b_i is defined as 2 raised to the power of a_i. The key observation is that the condition b_i^{b_j} = b_j^{b_i} simplifies to a_i * 2^{a_j} = a_j * 2^{a_i}. 

### Approach
1. **Problem Analysis**: The problem requires us to find pairs (i, j) where the combination of notes (b_i, b_j) is the same as (b_j, b_i). This translates to the equation a_i * 2^{a_j} = a_j * 2^{a_i}. 
2. **Key Insight**: The equation holds true if either:
   - a_i equals a_j (since any number raised to the same power is the same).
   - The pair (a_i, a_j) is (1, 2) or (2, 1) (since 1 * 2^2 = 2 * 2^1 = 4).
3. **Algorithm Selection**: 
   - **Frequency Dictionary**: We use a dictionary to count the occurrences of each number in the array.
   - **Identical Pairs**: For each number, the number of valid pairs is the combination of its count taken two at a time (i.e., count * (count - 1) / 2).
   - **Cross Pairs**: The number of valid pairs between 1 and 2 is the product of their respective counts.
4. **Complexity Analysis**: The algorithm processes each element in the array once to build the frequency dictionary, which takes O(n) time per test case. The subsequent calculations are O(1) per distinct number, making the overall complexity O(n) per test case, which is efficient given the constraints.

### Solution Code
```python
t = int(input().strip())
results = []

for _ in range(t):
    n = int(input().strip())
    arr = list(map(int, input().split()))
    from collections import defaultdict
    freq = defaultdict(int)
    for num in arr:
        freq[num] += 1
        
    identical_pairs = 0
    for count in freq.values():
        identical_pairs += count * (count - 1) // 2
        
    cross_pairs = freq.get(1, 0) * freq.get(2, 0)
    
    results.append(identical_pairs + cross_pairs)

for res in results:
    print(res)
```

### Explanation
1. **Reading Input**: The number of test cases `t` is read first. For each test case, the length of the array `n` and the array `arr` are read.
2. **Frequency Dictionary**: A dictionary `freq` is used to count occurrences of each number in the array.
3. **Identical Pairs Calculation**: For each number in the dictionary, the number of pairs where both elements are the same is calculated using the combination formula C(count, 2) = count * (count - 1) / 2.
4. **Cross Pairs Calculation**: The number of valid pairs between the numbers 1 and 2 is the product of their counts.
5. **Output**: The results for all test cases are collected in a list and printed at the end.

This approach efficiently counts all valid pairs by leveraging mathematical insights and dictionary operations, ensuring optimal performance even for large input sizes.
