=== ANSWER ===

To solve this problem, we need to find the length of the longest special substring (a substring consisting of the same character) that occurs at least three times in the given string. If no such substring exists, we return -1.

### Approach
1. **Group Consecutive Characters**: First, we traverse the string to group consecutive identical characters. For each character, we record the lengths of all its consecutive runs.
2. **Check for Valid Substrings**: For each character, we determine the maximum length `k` such that the total number of occurrences of the substring (character repeated `k` times) in the string is at least three. This is done by summing the occurrences from all runs of that character. The number of occurrences for a run of length `L` and substring length `k` is `L - k + 1` if `L >= k`.
3. **Early Termination**: Since the total occurrences decrease as `k` increases, we can break early once the condition fails for a particular `k`.
4. **Result Calculation**: The maximum `k` across all characters that meets the condition is our answer. If no such `k` exists, return -1.

### Solution Code
```python
from collections import defaultdict

class Solution:
    def maximumLength(self, s: str) -> int:
        n = len(s)
        runs = defaultdict(list)
        current_char = s[0]
        count = 1
        for i in range(1, n):
            if s[i] == s[i-1]:
                count += 1
            else:
                runs[current_char].append(count)
                current_char = s[i]
                count = 1
        runs[current_char].append(count)
        
        max_ans = 0
        for char, runs_list in runs.items():
            if not runs_list:
                continue
            max_run_char = max(runs_list)
            max_k_char = 0
            for k in range(1, max_run_char + 1):
                total_occurrences = 0
                for run in runs_list:
                    if run >= k:
                        total_occurrences += (run - k + 1)
                if total_occurrences >= 3:
                    max_k_char = k
                else:
                    break
            max_ans = max(max_ans, max_k_char)
        
        return max_ans if max_ans != 0 else -1
```

### Explanation
1. **Grouping Characters**: The code first processes the string to group consecutive identical characters. For example, the string "aaabbb" is grouped into runs of lengths [3, 3] for 'a' and 'b'.
2. **Checking Substring Occurrences**: For each character, the code checks all possible substring lengths (from 1 up to the maximum run length of that character). For each length `k`, it calculates the total occurrences of the substring (character repeated `k` times) by summing `(run_length - k + 1)` for all runs where the run length is at least `k`.
3. **Early Termination**: If for any `k` the total occurrences drop below three, the loop breaks early since further increases in `k` will only decrease the total occurrences.
4. **Result Determination**: The maximum valid `k` across all characters is returned. If no valid `k` is found, the function returns -1, indicating no special substring occurs at least three times. 

This approach efficiently narrows down the solution by leveraging the properties of consecutive character runs and early termination to optimize performance.
