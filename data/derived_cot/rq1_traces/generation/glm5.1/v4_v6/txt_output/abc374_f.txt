You will be given a competitive programming problem.
KEYENCE is famous for quick delivery.

In this problem, the calendar proceeds as Day 1, Day 2, Day 3, \dots.
There are orders 1,2,\dots,N, and it is known that order i will be placed on Day T_i.
For these orders, shipping is carried out according to the following rules.

- At most K orders can be shipped together.
- Order i can only be shipped on Day T_i or later.
- Once a shipment is made, the next shipment cannot be made until X days later.
- That is, if a shipment is made on Day a, the next shipment can be made on Day a+X.



For each day that passes from order placement to shipping, dissatisfaction accumulates by 1 per day.
That is, if order i is shipped on Day S_i, the dissatisfaction accumulated for that order is (S_i - T_i).
Find the minimum possible total dissatisfaction accumulated over all orders when you optimally schedule the shipping dates.

Input

The input is given from Standard Input in the following format:
N K X
T_1 T_2 \dots T_N

Output

Print the answer as an integer.

Constraints


- All input values are integers.
- 1 \le K \le N \le 100
- 1 \le X \le 10^9
- 1 \le T_1 \le T_2 \le \dots \le T_N \le 10^{12}

Sample Input 1

5 2 3
1 5 6 10 12

Sample Output 1

2

import sys

def solve() -> None:
    input = sys.stdin.readline
    N, K, X = map(int, input().split())
    T = list(map(int, input().split()))
    total_T = sum(T)

    # ---- generate all possible values of last_day ----
    cand_set = set()
    cand_set.add(-X)                     # initial dummy value
    for ti in T:
        for d in range(N + 1):           # d = 0 .. N
            cand_set.add(ti + d * X)
    cand = sorted(cand_set)
    M = len(cand)
    idx_of = {v: i for i, v in enumerate(cand)}

    # ---- M_for_j : the maximal T in a group ending at j ----
    M_for_j = [0] * (N + 1)              # 1‑based
    for j in range(1, N + 1):
        M_for_j[j] = T[j - 1]

    # ---- precompute transition indices ----
    # trans[idx][j] = index of the new last_day after adding a group ending at j
    trans = [[0] * (N + 1) for _ in range(M)]
    for idx, last in enumerate(cand):
        last_plus_X = last + X
        for j in range(1, N + 1):
            M_val = M_for_j[j]
            if last_plus_X <= M_val:
                S = M_val
            else:
                S = last_plus_X
            trans[idx][j] = idx_of[S]

    INF = 10 ** 18
    dp = [[INF] * M for _ in range(N + 1)]
    start_idx = idx_of[-X]
    dp[0][start_idx] = 0

    # ---- DP ----
    for i in range(N):
        dp_i = dp[i]
        for idx in range(M):
            cost = dp_i[idx]
            if cost == INF:
                continue
            max_j = i + K
            if max_j > N:
                max_j = N
            trans_idx = trans[idx]
            for j in range(i + 1, max_j + 1):
                size = j - i
                nxt_idx = trans_idx[j]
                S = cand[nxt_idx]
                new_cost = cost + size * S
                if new_cost < dp[j][nxt_idx]:
                    dp[j][nxt_idx] = new_cost

    min_cost = min(dp[N])
    print(min_cost - total_T)

if __name__ == "__main__":
    solve()
