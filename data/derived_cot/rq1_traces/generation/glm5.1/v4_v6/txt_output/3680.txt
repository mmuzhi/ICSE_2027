    def countComponents(self, nums: List[int], threshold: int) -> int:
        n = len(nums)
        parent = list(range(n))
        rank = [0] * n
        
        def find(x):
            if parent[x] != x:
                parent[x] = find(parent[x])
            return parent[x]
        
        def union(x, y):
            px, py = find(x), find(y)
            if px == py:
                return
            if rank[px] < rank[py]:
                px, py = py, px
            parent[py] = px
            if rank[px] == rank[py]:
                rank[px] += 1
        
        # For each m from 1 to threshold, maintain a list of indices of elements that divide m
        multiples = [[] for _ in range(threshold + 1)]

                for m in range(v, threshold + 1, v):

                for idx in multiples[m][1:]:
                    union(first, idx)
        
        # Count connected components
        roots = set(find(i) for i in range(n))
        return len(roots)

        n = len(nums)
        parent = list(range(n))
        rank = [0] * n
        
        def find(x):
            if parent[x] != x:
                parent[x] = find(parent[x])
            return parent[x]

            px, py = find(x), find(y)
            if px == py:
                return
            if rank[px] < rank[py]:
                px, py = py, px
            parent[py] = px
            if rank[px] == rank[py]:
                rank[px] += 1
        
        # Create a mapping from value to index
        val_to_idx = {v: i for i, v in enumerate(nums)}

        for m in range(1, threshold + 1):
            divisors_in_nums = []
            # Find all divisors of m
            d = 1
            while d * d <= m:
                if m % d == 0:
                    if d in val_to_idx:

                    if d != m // d and m // d in val_to_idx:

                for idx in divisors_in_nums[1:]:
                    union(first, idx)

Looking at this problem, I need to find connected components where nodes are connected if their LCM is ≤ threshold.

**Key Observations:**
1. If `a > threshold`, then `lcm(a, b) ≥ a > threshold` for any b, so elements > threshold form singleton components
2. For elements ≤ threshold: `lcm(a, b) ≤ threshold` iff there exists some `m ≤ threshold` where both `a | m` and `b | m` (since `lcm(a,b)` divides any common multiple)

**Algorithm:**
- Use Union-Find to group connected elements
- For each number `v` in nums where `v ≤ threshold`, add its index to lists for all multiples of `v` up to threshold
- For each value `m` from 1 to threshold, union all elements that divide `m` (they share a common multiple ≤ threshold, so their LCM ≤ threshold)

**Complexity Analysis:**
- Building multiples lists: O(threshold × log(n)) total additions
- Union operations: O(threshold × log(n)) with nearly O(1) per union
- Space: O(threshold × log(n))
