There are N islands and M bidirectional bridges connecting two islands. The islands and bridges are numbered 1, 2, \ldots, N and 1, 2, \ldots, M, respectively.
Bridge i connects islands U_i and V_i, and the time it takes to cross it in either direction is T_i.
No bridge connects an island to itself, but it is possible for two islands to be directly connected by more than one bridge.
One can travel between any two islands using some bridges.
You are given Q queries, so answer each of them. The i-th query is as follows:

You are given K_i distinct bridges: bridges B_{i,1}, B_{i,2}, \ldots, B_{i,K_i}.
Find the minimum time required to travel from island 1 to island N using each of these bridges at least once.
Only consider the time spent crossing bridges.
You can cross the given bridges in any order and in any direction.

Input

The input is given from Standard Input in the following format:

B_{2,1} B_{2,2} \cdots B_{2,{K_2}}
\vdots
K_Q

Output

Print Q lines. The i-th line (1 \leq i \leq Q) should contain the answer to the i-th query as an integer.

Constraints


- 2 \leq N \leq 400
- N-1 \leq M \leq 2 \times 10^5
- 1 \leq U_i < V_i \leq N

- 1 \leq Q \leq 3000
- 1 \leq K_i \leq 5
- 1 \leq B_{i,1} < B_{i,2} < \cdots < B_{i,K_i} \leq M
- All input values are integers.
- It is possible to travel between any two islands using some bridges.

Thus, the algorithm is:
1. Read N, M.
2. Store edges: for each edge i, store U_i, V_i, T_i.
3. Compute all-pairs shortest paths. Since N <= 400, we can use Floyd-Warshall. Initialize dist[n][n] with inf, dist[i][i]=0. For each edge, dist[u][v] = min(dist[u][v], T), dist[v][u] = min(dist[v][u], T). Then run Floyd.
4. Read Q.
5. For each query:
   a. Read K, then K bridge indices.
   b. Retrieve the K edges: their endpoints and weights.
   c. Initialize ans = infinity.
   d. Generate all permutations of the K edges. For each permutation:
      e. For each of the 2^K direction choices:
         f. Compute the cost: start at 1.
         g. cost = 0
         h. prev = 1
         i. For each edge in the permutation:
             if direction is 0: u = U, v = V
             else: u = V, v = U
             cost += dist[prev][u] + T
             prev = v
         j. cost += dist[prev][N]
         k. ans = min(ans, cost)
   l. Print ans.

  dist = [[inf]*N for _ in range(N)]
  for i in range(N): dist[i][i] = 0
  for each edge: u,v,w: dist[u-1][v-1] = min(dist[u-1][v-1], w); same for v-1,u-1.

      dk = dist[k]
      for i in range(N):
          di = dist[i]
          dik = di[k]
          if dik == inf: continue
          for j in range(N):
              if di[j] > dik + dk[j]:
                  di[j] = dik + dk[j]
  This is standard and fast in Python.

def solve():

            dist[u][v] = t
            dist[v][u] = t

        dk = dist[k]
        for i in range(N):
            di = dist[i]

            for j in range(N):

    Q = int(next(it))

        K = int(next(it))

def solve():
    import sys
    data = sys.stdin.buffer.read().split()
    if not data:
        return
    it = iter(data)
    N = int(next(it))
    M = int(next(it))
    edges = []
    INF = 10**18
    dist = [[INF]*N for _ in range(N)]
    for i in range(N):
        dist[i][i] = 0
    for _ in range(M):
        u = int(next(it)) - 1
        v = int(next(it)) - 1
        t = int(next(it))
        edges.append((u, v, t))
        if t < dist[u][v]:
            dist[u][v] = t
            dist[v][u] = t
    
    # Floyd-Warshall
    for k in range(N):
        dk = dist[k]
        for i in range(N):
            di = dist[i]
            dik = di[k]
            if dik == INF:
                continue
            # inner loop
            for j in range(N):
                # di[j] = min(di[j], dik + dk[j])
                new_dist = dik + dk[j]
                if new_dist < di[j]:
                    di[j] = new_dist
    
    Q = int(next(it))
    out = []
    for _ in range(Q):
        K = int(next(it))
        req_idx = [int(next(it)) - 1 for _ in range(K)]
        req_edges = [edges[idx] for idx in req_idx]

            imp_set.add(u)
            imp_set.add(v)

        node_to_idx = {node: i for i, node in enumerate(imp_list)}
        S_len = len(imp_list)
        dp = [[INF] * S_len for _ in range(1 << K)]
        start_idx = node_to_idx[0]
        dp[0][start_idx] = 0
        for mask in range(1 << K):
            for i in range(S_len):
                cur_cost = dp[mask][i]
                if cur_cost == INF:
                    continue
                cur_node = imp_list[i]
                for e in range(K):
                    if (mask & (1 << e)) == 0:
                        u, v, t = req_edges[e]
