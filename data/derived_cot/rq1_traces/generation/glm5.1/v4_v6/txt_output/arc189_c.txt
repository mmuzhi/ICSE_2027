Then we find the longest increasing subsequence (LIS) of this sequence of indices. That gives the length of LCS. Since elements are distinct, this works.

Step 1: Find the cycles containing X.

red_tokens_indices = [index of v in red_path for v where A_v > 0 and v != X]
blue_tokens_indices = [index of u in blue_path for u where B_u > 0 and u != X]

If there is any red ball not in C_R (i.e., not in the red_path mapping), then impossible. Similarly for blue balls not in C_B.

If impossible, output -1.

Else, let min_red = min(red_tokens_indices) if not empty else None.
min_blue = min(blue_tokens_indices) if not empty else None.

Active red sequence: if min_red is not None, it's red_path[min_red : L_R-1] (excluding X). Else empty.
Active blue sequence: similarly.

Now we have two sequences A and B. We need LCS(A, B). Since elements are distinct within each sequence, we can compute LCS by:
- Create a dict mapping each element in A to its index (0 to len(A)-1).
- For each element in B (in order), if it is in A, append its index to a list.
- Compute LIS of this list. The length of LIS is the LCS.

Then answer = len(A) + len(B) - LIS_length.

    # Find the earliest (farthest from X) red token
    min_red = len(red_path)
    for i in range(N):
        if A[i] > 0 and i != X:
            idx = red_pos[i]
            if idx < min_red:
                min_red = idx

    # Find the earliest blue token
    min_blue = len(blue_path)
    for i in range(N):
        if B[i] > 0 and i != X:
            idx = blue_pos[i]
            if idx < min_blue:
                min_blue = idx

    active_red = red_path[min_red:] if min_red < len(red_path) else []
    active_blue = blue_path[min_blue:] if min_blue < len(blue_path) else []

    # Compute LCS of active_red and active_blue
    # Since both have distinct elements, LCS can be found via LIS
    pos_in_red = {v: i for i, v in enumerate(active_red)}
    seq = []
    for v in active_blue:
        if v in pos_in_red:
            seq.append(pos_in_red[v])

    tails = []
    for x in seq:
        i = bisect.bisect_left(tails, x)
        if i == len(tails):
            tails.append(x)
        else:
            tails[i] = x
    lcs = len(tails)

    ans = len(active_red) + len(active_blue) - lcs
    print(ans)

    solve()
