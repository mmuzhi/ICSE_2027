Looking at this problem, I need to:
1. Track path sums from (1,1) to (H,W) where paths only go right or down
2. Handle updates efficiently when cells change

The key insight is that when updating cell (h,w), the change in the final answer can be computed using:
- `dp[h][w]` = sum of products along all paths from (1,1) to (h,w)
- `rev_dp[h][w]` = sum of products along all paths from (h,w) to (H,W)

When changing A[h][w] from old to new value, the change is:
`delta = (dp[h][w] / old_val) * (rev_dp[h][w] / old_val) * (new_val - old_val)`

After each update, I need to recompute the affected portions of dp and rev_dp.

Since HW ≤ 200000, the grid is manageable. For each update at (sh,sw):
- dp needs updating for cells (h,w) where h ≥ sh and w ≥ sw
- rev_dp needs updating for cells (h,w) where h ≤ sh and w ≤ sw

import sys
input = sys.stdin.readline

MOD = 998244353

def solve():
    H, W = map(int, input().split())
    A = [[0] * (W + 2) for _ in range(H + 2)]
    for h in range(1, H + 1):
        row = list(map(int, input().split()))
        for w in range(1, W + 1):
            A[h][w] = row[w - 1]
    
    Q, sh, sw = map(int, input().split())
    
    dp = [[0] * (W + 2) for _ in range(H + 2)]
    rev_dp = [[0] * (W + 2) for _ in range(H + 2)]
    
    def compute_dp():
        dp[1][1] = A[1][1]
        for h in range(1, H + 1):
            for w in range(1, W + 1):
                if h == 1 and w == 1:
                    continue
                dp[h][w] = A[h][w] * ((dp[h-1][w] if h > 1 else 0) + (dp[h][w-1] if w > 1 else 0)) % MOD
    
    def compute_rev_dp():
        rev_dp[H][W] = A[H][W]
        for h in range(H, 0, -1):
            for w in range(W, 0, -1):
                if h == H and w == W:
                    continue
                rev_dp[h][w] = A[h][w] * ((rev_dp[h+1][w] if h < H else 0) + (rev_dp[h][w+1] if w < W else 0)) % MOD
    
    compute_dp()
    compute_rev_dp()
    
    for _ in range(Q):
        d, a = input().split()
        a = int(a)
        
        if d == 'L':
            sw -= 1
        elif d == 'R':
            sw += 1
        elif d == 'U':
            sh -= 1
        elif d == 'D':
            sh += 1
        
        old_val = A[sh][sw]
        new_val = a
        
        if old_val != 0:
            contrib = dp[sh][sw] * pow(old_val, MOD - 2, MOD) % MOD * rev_dp[sh][sw] % MOD * pow(old_val, MOD - 2, MOD) % MOD
            delta = contrib * (new_val - old_val) % MOD
            dp[H][W] = (dp[H][W] + delta) % MOD
        
        A[sh][sw] = new_val
        
        for h in range(sh, H + 1):
            for w in range(sw, W + 1):
