You will be given a competitive programming problem.
There are N dice.
The i-th die has K_i faces, with the numbers A_{i,1}, A_{i,2}, \ldots, A_{i,K_i} written on them.
When you roll this die, each face appears with probability \frac{1}{K_i}.
You choose two dice from the N dice and roll them.
Determine the maximum probability that the two dice show the same number, when the dice are chosen optimally.

Input

The input is given from Standard Input in the following format:
N
K_1 A_{1,1} A_{1,2} \dots A_{1,K_1}
\vdots
K_N A_{N,1} A_{N,2} \dots A_{N,K_N}

Output

Print the answer.
Your answer is considered correct if the absolute or relative error from the true solution does not exceed 10^{-8}.

Constraints


- 2 \leq N \leq 100
- 1 \leq K_i
- K_1 + K_2 + \dots + K_N \leq 10^5
- 1 \leq A_{i,j} \leq 10^5
- All input values are integers.

Sample Input 1

3
3 1 2 3
4 1 2 2 1
6 1 2 3 4 5 6

Sample Output 1

0.333333333333333

import sys
from collections import Counter

def main() -> None:
    data = list(map(int, sys.stdin.read().split()))
    it = iter(data)
    N = next(it)
    dice = []                     # each element: (K, Counter)

    for _ in range(N):
        K = next(it)
        faces = [next(it) for _ in range(K)]
        cnt = Counter(faces)
        dice.append((K, cnt))

    max_prob = 0.0

    for i in range(N):
        K_i, cnt_i = dice[i]
        for j in range(i + 1, N):
            K_j, cnt_j = dice[j]

            # iterate over the smaller counter
            if len(cnt_i) > len(cnt_j):
                small, large = cnt_j, cnt_i
            else:
                small, large = cnt_i, cnt_j

            dot = 0
            for num, c_small in small.items():
                c_large = large.get(num)
                if c_large is not None:
                    dot += c_small * c_large

            prob = dot / (K_i * K_j)
            if prob > max_prob:
                max_prob = prob

    # print with sufficient precision
    print("{:.15f}".format(max_prob))

if __name__ == "__main__":
    main()
