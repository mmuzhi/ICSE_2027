=== ANSWER ===
```python

class Solution:
    def countPartitions(self, nums: List[int]) -> int:
        total = sum(nums)
        # If total sum is even, every partition yields an even difference.
        # Otherwise, no partition yields an even difference.
        return len(nums) - 1 if total % 2 == 0 else 0

```
