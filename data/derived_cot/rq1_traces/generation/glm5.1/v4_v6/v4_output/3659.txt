You will be given a competitive programming problem.
You are given a 2D integer array grid with size m x n. You are also given an integer k.
Your task is to calculate the number of paths you can take from the top-left cell (0, 0) to the bottom-right cell (m - 1, n - 1) satisfying the following constraints:

You can either move to the right or down. Formally, from the cell (i, j) you may move to the cell (i, j + 1) or to the cell (i + 1, j) if the target cell exists.
The XOR of all the numbers on the path must be equal to k.

Return the total number of such paths.
Since the answer can be very large, return the result modulo 10^9 + 7.
 
Example 1:

Input: grid = [[2, 1, 5], [7, 10, 0], [12, 6, 4]], k = 11
Output: 3
Explanation: 
The 3 paths are:

(0, 0) → (1, 0) → (2, 0) → (2, 1) → (2, 2)
(0, 0) → (1, 0) → (1, 1) → (1, 2) → (2, 2)
(0, 0) → (0, 1) → (1, 1) → (2, 1) → (2, 2)


Example 2:

Input: grid = [[1, 3, 3, 3], [0, 3, 3, 2], [3, 0, 1, 1]], k = 2
Output: 5
Explanation:
The 5 paths are:

(0, 0) → (1, 0) → (2, 0) → (2, 1) → (2, 2) → (2, 3)
(0, 0) → (1, 0) → (1, 1) → (2, 1) → (2, 2) → (2, 3)

from typing import List

class Solution:
    def countPathsWithXorValue(self, grid: List[List[int]], k: int) -> int:
        MOD = 10**9 + 7
        m, n = len(grid), len(grid[0])

        # DP for the first row (only moves to the right)
        cur = [[0] * 16 for _ in range(n)]
        cur[0][grid[0][0]] = 1
        for j in range(1, n):
            val = grid[0][j]
            # cur[j][y] = cur[j-1][y ^ val]   (only left neighbour)
            cur[j] = [cur[j-1][y ^ val] for y in range(16)]

        prev = cur   # dp for the previous row

        # Process remaining rows
        for i in range(1, m):
            new_cur = [[0] * 16 for _ in range(n)]

            # first column: only top neighbour
            val = grid[i][0]
            for y in range(16):
                new_cur[0][y] = prev[0][y ^ val]

            # other columns: top + left
            for j in range(1, n):
                val = grid[i][j]
                for y in range(16):
                    top = prev[j][y ^ val]
                    left = new_cur[j-1][y ^ val]
                    new_cur[j][y] = (top + left) % MOD

            prev = new_cur

        return prev[n-1][k] % MOD
