You will be given a competitive programming problem.
You are given an integer array nums and two integers, k and m.
Return the maximum sum of k non-overlapping subarrays of nums, where each subarray has a length of at least m.
 
Example 1:

Input: nums = [1,2,-1,3,3,4], k = 2, m = 2
Output: 13
Explanation:
The optimal choice is:

Subarray nums[3..5] with sum 3 + 3 + 4 = 10 (length is 3 >= m).
Subarray nums[0..1] with sum 1 + 2 = 3 (length is 2 >= m).

The total sum is 10 + 3 = 13.

Example 2:

Input: nums = [-10,3,-1,-2], k = 4, m = 1
Output: -10
Explanation:
The optimal choice is choosing each element as a subarray. The output is (-10) + 3 + (-1) + (-2) = -10.

    def maxSum(self, nums: List[int], k: int, m: int) -> int:
        n = len(nums)
        
        # Compute prefix sums
        prefix = [0] * (n + 1)
        for i in range(n):
            prefix[i + 1] = prefix[i] + nums[i]
        
        # dp[i][j] = max sum of j subarrays from nums[0..i-1]
        # Initialize with -infinity
        INF = float('-inf')
        dp = [[INF] * (k + 1) for _ in range(n + 1)]
        dp[0][0] = 0

            # max_val = max over s from 0 to some limit of (dp[s][j-1] - prefix[s])

            for i in range(m, n + 1):
                # When i = m, s can range from 0 to 0 (i.e., s = 0)
                # When i = m+1, s can range from 0 to 1
                # ...
                # When i, s can range from 0 to i-m

Looking at this problem, I need to find the maximum sum of k non-overlapping subarrays, each with length at least m.

**Approach: Dynamic Programming with Optimization**

Let me define `dp[i][j]` = maximum sum of j non-overlapping subarrays from `nums[0..i-1]`, where each subarray has length ≥ m.

**Transition:**
- Option 1: Don't include `nums[i-1]` in any subarray → `dp[i-1][j]`
- Option 2: Include `nums[i-1]` as end of j-th subarray starting at position s → `dp[s][j-1] + sum(nums[s..i-1])` where `s ≤ i-m`

So: `dp[i][j] = max(dp[i-1][j], max over s from 0 to i-m of (dp[s][j-1] + prefix[i] - prefix[s]))`

**Key Optimization:** Rewrite as:
`dp[i][j] = max(dp[i-1][j], prefix[i] + max over s from 0 to i-m of (dp[s][j-1] - prefix[s]))`

We can maintain `max_val = max(dp[s][j-1] - prefix[s])` incrementally as i increases, making each transition O(1).

**Verification with Example 1:** `nums = [1,2,-1,3,3,4], k = 2, m = 2`
- prefix = [0,1,3,2,5,8,12]
- For j=2, i=6: max_val = max(dp[2][1]-3, dp[3][1]-2, dp[4][1]-5) = max(0, 1, 0) = 1
- dp[6][2] = max(dp[5][2], 12 + 1) = max(9, 13) = 13 ✓

**Verification with Example 2:** `nums = [-10,3,-1,-2], k = 4, m = 1`
- Each element must be its own subarray
- Sum = -10 + 3 + (-1) + (-2) = -10 ✓

**Complexity:** O(n × k) time, O(n × k) space

```python
class Solution:
    def maxSum(self, nums: List[int], k: int, m: int) -> int:
        n = len(nums)
        
        # Compute prefix sums
        prefix = [0] * (n + 1)
        for i in range(n):
            prefix[i + 1] = prefix[i] + nums[i]
        
        # dp[i][j] = max sum of j non-overlapping subarrays from nums[0..i-1],
        # where each subarray has length at least m
        INF = float('-inf')
        dp = [[INF] * (k + 1) for _ in range(n + 1)]
        
        # Base case: dp[i][0] = 0 for all i (no subarrays selected)
        for i in range(n + 1):
            dp[i][0] = 0
        
        # Fill the dp table
        for j in range(1, k + 1):
            # max_val maintains max of (dp[s][j-1] - prefix[s]) for s from 0 to i-m
            max_val = INF
            for i in range(m, n + 1):
                # Add dp[i-m][j-1] - prefix[i-m] to the running maximum
                candidate = dp[i - m][j - 1] - prefix[i - m]
                if candidate > max_val:
                    max_val = candidate
                
                # dp[i][j] = max(dp[i-1][j], prefix[i] + max_val)
                # - dp[i-1][j]: don't include nums[i-1] in any subarray
                # - prefix[i] + max_val: include nums[i-1] as end of j-th subarray
                dp[i][j] = max(dp[i - 1][j], prefix[i] + max_val)
        
        return dp[n][k]
```
