An array is said to be non-decreasing if each element is greater than or equal to its previous element, if it exists.

Output: 17

Looking at this problem, I need to count subarrays that can be made non-decreasing with at most k increment operations.

**Key Insight:** For a subarray `[l, r]`, the minimum cost to make it non-decreasing is the sum of `(max(nums[l..i]) - nums[i])` for each position `i`. This is because we can only increment elements, so each element must be raised to at least the maximum of all elements before it.

**Approach:** Use a two-pointer technique where for each ending position `r`, I find the minimum starting position `l` such that `cost(l, r) ≤ k`. Since cost increases as we extend the subarray to the right, and decreases as we shrink from the left, the two-pointer approach works.

To efficiently track the cost, I use a monotonic stack storing `(value, count)` pairs where `value` is a "peak" (local maximum in the running max sequence) and `count` is how many positions have that value as their prefix maximum.

When adding an element: pop all peaks ≤ new element, push new peak with accumulated count.

When removing from left: decrement count of front element. If count reaches 0, rebuild the stack for remaining elements.

```python
class Solution:
    def countNonDecreasingSubarrays(self, nums: List[int], k: int) -> int:
        n = len(nums)
        total = 0
        l = 0
        cost = 0
        stack = []  # (value, count)
        
        for r in range(n):
            # Add nums[r] to the window
            popped_count = 0
            while stack and stack[-1][0] <= nums[r]:
                v, c = stack.pop()
                cost -= v * c
                popped_count += c
            stack.append((nums[r], popped_count + 1))
            cost += nums[r] * (popped_count + 1)
            cost -= nums[r]  # subtract original value
            
            # Shrink window from left while cost > k
            while l <= r and cost > k:
                cost += nums[l]  # add back original value
                
                v, c = stack[0]
                cost -= v  # subtract max value for this position
                
                if c > 1:
                    stack[0] = (v, c - 1)
                else:
                    stack.pop(0)
                    # Rebuild stack for [l+1, r]
                    new_stack = []
                    new_cost = 0
                    for i in range(l + 1, r + 1):
                        pc = 0
                        while new_stack and new_stack[-1][0] <= nums[i]:
                            vv, cc = new_stack.pop()
                            new_cost -= vv * cc
                            pc += cc
                        new_stack.append((nums[i], pc + 1))
                        new_cost += nums[i] * (pc + 1)
                        new_cost -= nums[i]
                    stack = new_stack
                    cost = new_cost
                
                l += 1
            
            total += r - l + 1
        
        return total
```
