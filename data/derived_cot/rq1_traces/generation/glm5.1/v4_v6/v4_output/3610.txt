```python
from typing import List
from collections import Counter

class Solution:
    def findXSum(self, nums: List[int], k: int, x: int) -> List[int]:
        n = len(nums)
        ans = []
        for i in range(n - k + 1):
            sub = nums[i:i+k]
            freq = Counter(sub)
            if len(freq) <= x:
                ans.append(sum(sub))
            else:
                items = list(freq.items())
                items.sort(key=lambda item: (-item[1], -item[0]))
                current_sum = sum(val * count for val, count in items[:x])
                ans.append(current_sum)
        return ans
```

=== ANSWER ===
An elegant and optimal solution in Python to solve the "Find X-Sum of All K-Long Subarrays" problem.

### Method Explanation
The problem asks us to find the `x-sum` for every subarray of length `k` in the given array `nums`. Since the maximum length of `nums` is only 50, we can easily solve this using a straightforward simulation without needing complex data structures like a sliding window with balanced binary search trees. 

For each subarray of length `k`:
1. We count the frequencies of each element using a hash map (or `Counter`).
2. If the number of distinct elements in the subarray is less than or equal to `x`, the `x-sum` is simply the sum of all elements in the subarray.
3. Otherwise, we need to keep only the top `x` most frequent elements. We can achieve this by sorting the distinct elements based on their frequencies in descending order. If two elements have the same frequency, we break the tie by sorting their values in descending order (as per the problem description).
4. We take the top `x` elements from this sorted list, multiply each element by its frequency, and sum these products to get the `x-sum` for the current subarray.

### Python 3 Source Code

```python
from typing import List
from collections import Counter

class Solution:
    def findXSum(self, nums: List[int], k: int, x: int) -> List[int]:
        n = len(nums)
        ans = []
        
        for i in range(n - k + 1):
            # Extract the current subarray of length k
            sub = nums[i:i + k]
            
            # Count the frequency of each element in the subarray
            freq = Counter(sub)
            
            # If there are x or fewer distinct elements, the x-sum is the total sum of the subarray
            if len(freq) <= x:
                ans.append(sum(sub))
            else:
                # Sort elements by frequency (descending) and then by value (descending) for ties
                # item is a tuple of (element, frequency)
                items = list(freq.items())
                items.sort(key=lambda item: (-item[1], -item[0]))
                
                # Sum the occurrences of the top x most frequent elements
                current_sum = sum(val * count for val, count in items[:x])
                ans.append(current_sum)
                
        return ans
```
