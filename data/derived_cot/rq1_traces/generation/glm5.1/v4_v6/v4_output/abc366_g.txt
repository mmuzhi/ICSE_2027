You will be given a competitive programming problem.
You are given a simple undirected graph with N vertices and M edges. The i-th edge connects vertices u_i and v_i bidirectionally.
Determine if there exists a way to write an integer between 1 and 2^{60} - 1, inclusive, on each vertex of this graph so that the following condition is satisfied:

- For every vertex v with a degree of at least 1, the total XOR of the numbers written on its adjacent vertices (excluding v itself) is 0.


What is XOR?

The XOR of two non-negative integers A and B, denoted as A \oplus B, is defined as follows:


- In the binary representation of A \oplus B, the bit at position 2^k \, (k \geq 0) is 1 if and only if exactly one of the bits at position 2^k in the binary representations of A and B is 1. Otherwise, it is 0.


For example, 3 \oplus 5 = 6 (in binary: 011 \oplus 101 = 110).

In general, the bitwise XOR of k integers p_1, \dots, p_k is defined as (\cdots ((p_1 \oplus p_2) \oplus p_3) \oplus \cdots \oplus p_k).  It can be proved that this is independent of the order of p_1, \dots, p_k.

Input

The input is given from Standard Input in the following format:
N M
u_1 v_1
u_2 v_2
\vdots
u_M v_M

Output

If there is no way to write integers satisfying the condition, print No.
Otherwise, let X_v be the integer written on vertex v, and print your solution in the following format. If multiple solutions exist, any of them will be accepted.
Yes
X_1 X_2 \dots X_N

Constraints


- 1 \leq N \leq 60
- 0 \leq M \leq N(N-1)/2
- 1 \leq u_i < v_i \leq N
- (u_i, v_i) \neq (u_j, v_j) for i \neq j.
- All input values are integers.

def gauss_nullspace(A, N):
    # A: list of ints length N, each row bitmask
    # returns: list of basis vectors for nullspace (as ints, bitmask length N)
    # also returns rank, pivot columns etc.

    rows = A[:]  # copy
    pivot_row = [-1]*N  # for each column, which row is pivot
    pivot_col_for_row = [-1]*N  # for each row, which column is pivot (if any)

    row = 0
    for col in range(N):
        # find a row with bit col set, from row onward
        sel = -1
        for i in range(row, N):
            if (rows[i] >> col) & 1:
                sel = i
                break
        if sel == -1:
            continue
        # swap rows row and sel
        rows[row], rows[sel] = rows[sel], rows[row]
        pivot_row[col] = row
        pivot_col_for_row[row] = col
        # eliminate this column from other rows
        for i in range(N):
            if i != row and ((rows[i] >> col) & 1):
                rows[i] ^= rows[row]
        row += 1
        if row == N:
            break

    rank = row
    # Now rows[0..rank-1] are pivot rows in reduced row echelon form (since we eliminated above and below? Actually we eliminated all other rows, so it's RREF)
    # Identify free columns: columns without pivot
    free_cols = [c for c in range(N) if pivot_row[c] == -1]

    # For each free column, construct a nullspace vector:
    null_basis = []
    for fc in free_cols:
        # set x[fc] = 1, and other free cols = 0
        # determine pivot variables: for each pivot row, x[pivot_col] = sum of (row entries in free columns) because we need A x = 0.
        # Since rows are in RREF, each pivot row has equation: x[pivot_col] + sum_{j in free_cols} A[row][j] * x[j] = 0.
        # So x[pivot_col] = sum_{j in free_cols} A[row][j] * x[j].
        # For this particular free column fc, set x[fc]=1, others free=0.
        # Then compute x[pivot_col] = A[row][fc] (since only fc contributes).
        vec = 0
        # set free column fc bit
        vec |= (1 << fc)
        # For each pivot row, if the row has a 1 in column fc, then set the pivot column bit.
        for r in range(rank):
            pc = pivot_col_for_row[r]  # pivot column for this row
            if (rows[r] >> fc) & 1:
                vec |= (1 << pc)
        null_basis.append(vec)

    return null_basis, rank
