The number of 0's in the string is at most k.
The number of 1's in the string is at most k.

Example 1:

Input: s = "10101", k = 1
Output: 12
Explanation:
Every substring of s except the substrings "1010", "10101", and "0101" satisfies the k-constraint.

Output: 25
Explanation:
Every substring of s except the substrings with a length greater than 5 satisfies the k-constraint.

Example 3:

Input: s = "11111", k = 1
Output: 15
Explanation:
All substrings of s satisfy the k-constraint.

 
Constraints:

1 <= s.length <= 50 
1 <= k <= s.length
s[i] is either '0' or '1'.

=== ANSWER ===
Looking at this problem, I need to count substrings where either the count of 0's ≤ k OR the count of 1's ≤ k.

Given the small constraint (s.length ≤ 50), a brute force approach checking all substrings works well.

**Approach:**
1. For each starting index `i`, iterate through all ending indices `j >= i`
2. Track the running count of 0s and 1s as we extend the substring
3. If either count ≤ k, the substring satisfies the constraint

**Verification with Example 1:** s = "10101", k = 1
- Length 1: 5 substrings (all satisfy - at most 1 of each)
- Length 2: 4 substrings (all satisfy - one 0 and one 1)
- Length 3: 3 substrings (all satisfy - one type has count 1)
- Length 4: "1010" (2 zeros, 2 ones) - fails; "0101" (2 zeros, 2 ones) - fails
- Length 5: "10101" (2 zeros, 3 ones) - fails

Total: 5 + 4 + 3 = 12 ✓

**Verification with Example 3:** s = "11111", k = 1
- All substrings have 0 zeros (≤ 1), so all satisfy
- Total = 5×6/2 = 15 ✓

```python
class Solution:
    def countKConstraintSubstrings(self, s: str, k: int) -> int:
        n = len(s)
        count = 0
        for i in range(n):
            zeros = 0
            ones = 0
            for j in range(i, n):
                if s[j] == '0':
                    zeros += 1
                else:
                    ones += 1
                if zeros <= k or ones <= k:
                    count += 1
        return count
```

**Complexity:**
- Time: O(n²) where n is the length of s
- Space: O(1)
