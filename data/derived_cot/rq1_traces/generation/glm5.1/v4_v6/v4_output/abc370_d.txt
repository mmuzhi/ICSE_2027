You will be given a competitive programming problem.
There is a grid with H rows and W columns. Let (i, j) denote the cell at the i-th row from the top and j-th column from the left.
Initially, there is one wall in each cell.
After processing Q queries explained below in the order they are given, find the number of remaining walls.
In the q-th query, you are given two integers R_q and C_q.
You place a bomb at (R_q, C_q) to destroy walls. As a result, the following process occurs.

- If there is a wall at (R_q, C_q), destroy that wall and end the process.
- If there is no wall at (R_q, C_q), destroy the first walls that appear when looking up, down, left, and right from (R_q, C_q). More precisely, the following four processes occur simultaneously:
- If there exists an i \lt R_q such that a wall exists at (i, C_q) and no wall exists at (k, C_q) for all i \lt k \lt R_q, destroy the wall at (i, C_q).
- If there exists an i \gt R_q such that a wall exists at (i, C_q) and no wall exists at (k, C_q) for all R_q \lt k \lt i, destroy the wall at (i, C_q).
- If there exists a j \lt C_q such that a wall exists at (R_q, j) and no wall exists at (R_q, k) for all j \lt k \lt C_q, destroy the wall at (R_q, j).
- If there exists a j \gt C_q such that a wall exists at (R_q, j) and no wall exists at (R_q, k) for all C_q \lt k \lt j, destroy the wall at (R_q, j).

Input

The input is given from Standard Input in the following format:
H W Q
R_1 C_1
R_2 C_2
\vdots
R_Q C_Q

Output

Print the number of remaining walls after processing all queries.

Constraints


- 1 \leq H, W
- H \times W \leq 4 \times 10^5
- 1 \leq Q \leq 2 \times 10^5
- 1 \leq R_q \leq H
- 1 \leq C_q \leq W
- All input values are integers.

1 3

After processing all queries, there are two remaining walls, at (2, 1) and (2, 4).

import sys
from array import array

def solve():
    import sys
    data = sys.stdin.buffer.read().split()
    it = iter(data)
    H = int(next(it))
    W = int(next(it))
    Q = int(next(it))

    total = H * W
    alive = bytearray(b'\x01') * total

    W2 = W + 2
    H2 = H + 2

    # row DSUs
    row_template = array('i', range(W2))
    right_row = array('i')
    left_row = array('i')
    for _ in range(H):
        right_row.extend(row_template)
        left_row.extend(row_template)

    # column DSUs
    col_template = array('i', range(H2))
    down_col = array('i')
    up_col = array('i')
    for _ in range(W):
        down_col.extend(col_template)
        up_col.extend(col_template)

    # define find functions
    def find_right(r, c):
        base = (r-1) * W2
        # collect path
        path = []
        while True:
            idx = base + c
            parent = right_row[idx]
            if parent == c:
                break
            path.append(c)
            c = parent
        # compress
        for col in path:
            right_row[base + col] = c
        return c

    def find_left(r, c):
        base = (r-1) * W2
        path = []
        while True:
            idx = base + c
            parent = left_row[idx]
            if parent == c:
                break
            path.append(c)
            c = parent
        for col in path:
            left_row[base + col] = c
        return c

    def find_down(col, r):
        base = (col-1) * H2
        path = []
        while True:
            idx = base + r
            parent = down_col[idx]
            if parent == r:
                break
            path.append(r)
            r = parent
        for row in path:
            down_col[base + row] = r
        return r

    def find_up(col, r):
        base = (col-1) * H2
        path = []
        while True:
            idx = base + r
            parent = up_col[idx]
            if parent == r:
                break
            path.append(r)
            r = parent
        for row in path:
            up_col[base + row] = r
        return r

    def destroy(r, c):
        idx = (r-1)*W + (c-1)
        if not alive[idx]:
            return
        alive[idx] = 0

        base_r = (r-1) * W2
        # right
        next_col = find_right(r, c+1)
        right_row[base_r + c] = next_col
        # left
        prev_col = find_left(r, c-1)
        left_row[base_r + c] = prev_col

        base_c = (c-1) * H2
        # down
        next_row = find_down(c, r+1)
        down_col[base_c + r] = next_row
        # up
        prev_row = find_up(c, r-1)
        up_col[base_c + r] = prev_row

    # process queries
    for _ in range(Q):
        R = int(next(it))
        C = int(next(it))
        if alive[(R-1)*W + (C-1)]:
            destroy(R, C)
        else:
            # up
            up_r = find_up(C, R-1)
            if up_r >= 1:
                destroy(up_r, C)
            # down
            down_r = find_down(C, R+1)
            if down_r <= H:
                destroy(down_r, C)
            # left
            left_c = find_left(R, C-1)
            if left_c >= 1:
                destroy(R, left_c)
            # right
            right_c = find_right(R, C+1)
