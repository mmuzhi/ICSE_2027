=== ANSWER ===
```python
class Solution:
    def checkTwoChessboards(self, coordinate1: str, coordinate2: str) -> bool:
        # The color of a square can be determined by the parity of the sum of its column index (0-based) and row number.
        # If both squares have the same parity, they are the same color.
        def color(c: str) -> int:
            return (ord(c[0]) - ord('a') + int(c[1])) % 2
        
        return color(coordinate1) == color(coordinate2)
```
