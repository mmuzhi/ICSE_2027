=== ANSWER ===
Looking at this problem, I need to find the sum of (minimum + maximum) for all subsequences with at most k elements.

**Key Insight:** Instead of enumerating all subsequences, I'll count how many times each element contributes as a minimum and as a maximum.

**Approach:**
1. Sort the array
2. For element at index i (after sorting):
   - As a **minimum**: It's the minimum when all other elements have index ≥ i. For subsequences of size s (1 ≤ s ≤ k), we pick s-1 elements from the n-1-i elements with index > i. Count = Σ C(n-1-i, j) for j from 0 to k-1
   - As a **maximum**: It's the maximum when all other elements have index ≤ i-1. Count = Σ C(i, j) for j from 0 to k-1

3. Define f(m) = Σ C(m, j) for j from 0 to k-1
4. Total contribution of nums[i] = nums[i] × (f(n-1-i) + f(i))

**Computing f(m) efficiently:**
Using Pascal's identity: C(m, j) = C(m-1, j-1) + C(m-1, j)

I maintain an array C[j] representing C(current_m, j) and update it iteratively from m=0 to n-1. Update from j=k-1 down to j=1 to avoid overwriting values needed for the update.

**Verification with Example 1:** nums=[1,2,3], k=2
- f(0)=1, f(1)=2, f(2)=3
- i=0: 1×(3+1)=4, i=1: 2×(2+2)=8, i=2: 3×(1+3)=12
- Total: 24 ✓

**Verification with Example 2:** nums=[5,0,6], k=1
- After sorting: [0,5,6], f(m)=1 for all m
- i=0: 0×2=0, i=1: 5×2=10, i=2: 6×2=12
- Total: 22 ✓

**Verification with Example 3:** nums=[1,1,1], k=2
- f(0)=1, f(1)=2, f(2)=3

```python
class Solution:
    def minMaxSums(self, nums: List[int], k: int) -> int:
        MOD = 10**9 + 7
        n = len(nums)
        nums.sort()
        
        # Compute f(m) = sum(C(m, j) for j in 0..k-1) for m from 0 to n-1
        f = [0] * n
        C = [0] * k
        C[0] = 1
        f[0] = 1
        total = 1  # running sum of C
        
        for m in range(1, n):
            # Update C using Pascal's identity, from high j to low j
            for j in range(min(k-1, m), 0, -1):
                total = (total + C[j-1]) % MOD
                C[j] = (C[j] + C[j-1]) % MOD
            f[m] = total
        
        # Compute the answer
        ans = 0
        for i in range(n):
            s_min = f[n - 1 - i]  # count as minimum
            s_max = f[i]           # count as maximum
            ans = (ans + nums[i] * (s_min + s_max)) % MOD
        
        return ans
```

**Time Complexity:** O(n × k) for computing f, O(n) for the answer - total O(n × k)
**Space Complexity:** O(n) for f array, O(k) for C array - total O(n)
