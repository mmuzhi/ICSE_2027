1. For x = 1: f(1) = 0 (already 1), so it's k-reducible for any k ≥ 0

1. Precompute f(c) for c from 1 to L (length of binary string)
2. Find set C = {c : f(c) ≤ k-1}

4. Handle x=1 separately for k=0 (since f(1)=0 but 1∉C when k=0)

- For tight=0: can place 0 or 1 (stay not tight)
