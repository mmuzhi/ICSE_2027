You will be given a competitive programming problem.
You are given two integer arrays, nums1 and nums2, both of length n, along with a positive integer k.
For each index i from 0 to n - 1, perform the following:

Find all indices j where nums1[j] is less than nums1[i].
Choose at most k values of nums2[j] at these indices to maximize the total sum.

Return an array answer of size n, where answer[i] represents the result for the corresponding index i.
 
Example 1:

Input: nums1 = [4,2,1,5,3], nums2 = [10,20,30,40,50], k = 2
Output: [80,30,0,80,50]
Explanation:

For i = 0: Select the 2 largest values from nums2 at indices [1, 2, 4] where nums1[j] < nums1[0], resulting in 50 + 30 = 80.
For i = 1: Select the 2 largest values from nums2 at index [2] where nums1[j] < nums1[1], resulting in 30.
For i = 2: No indices satisfy nums1[j] < nums1[2], resulting in 0.
For i = 3: Select the 2 largest values from nums2 at indices [0, 1, 2, 4] where nums1[j] < nums1[3], resulting in 50 + 30 = 80.
For i = 4: Select the 2 largest values from nums2 at indices [1, 2] where nums1[j] < nums1[4], resulting in 30 + 20 = 50.

Output: [0,0,0,0]
Explanation:
Since all elements in nums1 are equal, no indices satisfy the condition nums1[j] < nums1[i] for any i, resulting in 0 for all positions.

Pseudo:

def findMaxSum(self, nums1, nums2, k):
    n = len(nums1)
    arr = [(nums1[i], nums2[i], i) for i in range(n)]
    arr.sort(key=lambda x: x[0])  # sort by nums1
    ans = [0]*n
    import heapq
    heap = []  # min-heap of top k values among processed
    total = 0
    i = 0
    while i < n:
        # start of group
        val = arr[i][0]
        # compute current top k sum for this group
        group_sum = total  # sum of top k in heap
        # assign answer to all indices in this group
        j = i
        while j < n and arr[j][0] == val:
            idx = arr[j][2]
            ans[idx] = group_sum
            j += 1
        # now add nums2 of this group to heap
        for t in range(i, j):
            num2 = arr[t][1]
            heapq.heappush(heap, num2)
            total += num2
            if len(heap) > k:
                smallest = heapq.heappop(heap)
                total -= smallest
        i = j
    return ans

class Solution:

        ans = [0] * n
        heap = []          # min-heap of the current k largest nums2
        total = 0          # sum of elements in the heap

        i = 0
        while i < n:

            # answer for all indices with this value is the current top‑k sum
            cur_sum = total
            j = i

                ans[idx] = cur_sum
                j += 1

            # now insert the nums2 values of this group into the heap
            for t in range(i, j):

                heapq.heappush(heap, num2)
                total += num2
                if len(heap) > k:
                    smallest = heapq.heappop(heap)
                    total -= smallest

            i = j   # move to next group

        return ans
