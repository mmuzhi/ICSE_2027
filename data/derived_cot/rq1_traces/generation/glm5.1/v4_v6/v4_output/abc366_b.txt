You will be given a competitive programming problem.
You are given a horizontally written text. Convert it to vertical writing, filling spaces with *.

You are given N strings S_1, S_2, \dots, S_N consisting of lowercase English letters. Let M be the maximum length of these strings.
Print M strings T_1, T_2, \dots, T_M that satisfy the following conditions:

- Each T_i consists of lowercase English letters and *.
- Each T_i does not end with *.
- For each 1 \leq i \leq N, the following holds:
- For each 1 \leq j \leq |S_i|, the (N-i+1)-th character of T_j exists, and the concatenation of the (N-i+1)-th characters of T_1, T_2, \dots, T_{|S_i|} in this order equals S_i.

Here, |S_i| denotes the length of the string S_i.

The input is given from Standard Input in the following format:
N

S_N

Output

Print the answer in the following format:

Constraints


- N is an integer between 1 and 100, inclusive.
- Each S_i is a string of lowercase English letters with length between 1 and 100, inclusive.

def solve():
    N = int(input())
    strings = [input() for _ in range(N)]
    M = max(len(s) for s in strings)
    
    # Create grid
    grid = [['' for _ in range(N)] for _ in range(M)]
    
    # Fill grid
    for i in range(N):
        for j in range(len(strings[i])):
            grid[j][N-1-i] = strings[i][j]
    
    # Fill empty cells with * if needed
    for j in range(M):
        rightmost = -1
        for k in range(N):
            if grid[j][k] != '':
                rightmost = k
        for k in range(rightmost):
            if grid[j][k] == '':
                grid[j][k] = '*'
    
    # Construct output
    for j in range(M):
        print(''.join(grid[j]))

**Understanding the problem:**
- For string S_i, the (N-i+1)-th character of T_j equals S_i[j] for j from 1 to |S_i|
- This means S_1 fills the rightmost column, S_2 fills the second-rightmost, etc.
- Empty positions before a letter (in the same row) must be filled with *
- No T_i can end with *

**Approach:**
1. Create a grid with M rows and N columns (M = max string length)
2. Fill the grid: for string S_i, place its characters in column (N-1-i)
3. For each row, find the rightmost letter and fill any gaps before it with *
4. Output each row (empty cells after the rightmost letter are ignored)
