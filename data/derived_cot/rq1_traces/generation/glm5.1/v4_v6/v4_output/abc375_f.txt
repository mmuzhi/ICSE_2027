In the nation of AtCoder, there are N cities numbered 1 to N, and M roads numbered 1 to M.
Road i connects cities A_i and B_i bidirectionally and has a length of C_i.
You are given Q queries to process in order. The queries are of the following two types.

- 1 i: Road i becomes closed.
- 2 x y: Print the shortest distance from city x to city y, using only roads that are not closed. If city y cannot be reached from city x, print -1 instead.

It is guaranteed that each test case contains at most 300 queries of the first type.

Input

The input is given from Standard Input in the following format:
N M Q
A_1 B_1 C_1

\mathrm{query}_1
\vdots
\mathrm{query}_Q

Each query is in one of the following two formats:
1 i

2 x y

Output

Process the queries in order.

Constraints


- 2 \leq N \leq 300
- 0 \leq M \leq \frac{N(N-1)}{2}
- 1 \leq A_i < B_i \leq N
- All pairs (A_i, B_i) are distinct.
- 1 \leq C_i \leq 10^9
- 1 \leq Q \leq 2 \times 10^5
- In the queries of the first type, 1 \leq i \leq M.
- The road given in a query of the first type is not already closed at that time.
- The number of queries of the first type is at most 300.
- In the queries of the second type, 1 \leq x < y \leq N.
- All input values are integers.

import sys

def main():
    import sys
    data = sys.stdin.buffer.read().split()
    it = iter(data)
    N = int(next(it))
    M = int(next(it))
    Q = int(next(it))
    A = [0]*(M+1)
    B = [0]*(M+1)
    C = [0]*(M+1)
    for i in range(1, M+1):
        A[i] = int(next(it))
        B[i] = int(next(it))
        C[i] = int(next(it))
    queries = []
    closed_set = set()
    for _ in range(Q):
        t = int(next(it))
        if t == 1:
            i = int(next(it))
            queries.append((1, i))
            closed_set.add(i)
        else:
            x = int(next(it)); y = int(next(it))
            queries.append((2, x, y))
    INF = 10**18
    # build initial distance matrix for final state (edges not closed)
    dist = [[INF]*N for _ in range(N)]
    for i in range(N):
        dist[i][i] = 0
    for idx in range(1, M+1):
        if idx not in closed_set:
            u = A[idx]-1
            v = B[idx]-1
            w = C[idx]
            dist[u][v] = w
            dist[v][u] = w
    # Floyd-Warshall
    for k in range(N):
        dk = dist[k]
        for i in range(N):
            di = dist[i]
            dik = di[k]
            if dik == INF:
                continue
            # inner loop
            for j in range(N):
                nd = dik + dk[j]
                if nd < di[j]:
                    di[j] = nd
    # process queries in reverse
    answers = []
    for q in reversed(queries):
        if q[0] == 2:
            _, x, y = q
            d = dist[x-1][y-1]
            answers.append(d if d != INF else -1)
        else:
            _, idx = q
            u = A[idx]-1
            v = B[idx]-1
            w = C[idx]
            # add edge (u,v,w)
            # store old rows/cols
            row_u = dist[u][:]  # copy
            row_v = dist[v][:]
            col_u = [dist[i][u] for i in range(N)]
            col_v = [dist[i][v] for i in range(N)]
            # update
            for i in range(N):
                base1 = col_u[i] + w
                base2 = col_v[i] + w
                di = dist[i]
                for j in range(N):
                    cand = base1 + row_v[j]
                    if cand < di[j]:
                        di[j] = cand
                    cand = base2 + row_u[j]
                    if cand < di[j]:
                        di[j] = cand
    # reverse answers
    answers.reverse()
    out = '\n'.join(map(str, answers))
    sys.stdout.write(out)

if __name__ == "__main__":
    main()
