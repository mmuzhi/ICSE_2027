=== ANSWER ===
Looking at this problem, I need to determine when adding one card to four existing cards can form a Full House (3 of one value + 2 of another value, where the values are different).

Let me analyze the possible frequency patterns of the 4 cards:

1. **Four of a kind** (e.g., 8,8,8,8): Adding same value → 5 of a kind (not Full House). Adding different value → 4+1 (not Full House). **Result: No**

2. **Three of a kind + one different** (e.g., 7,7,7,1): Adding the value that appears once → 3+2 (Full House!). **Result: Yes**

3. **Two pairs** (e.g., 3,3,5,5): Adding either of the pair values → 3+2 (Full House!). **Result: Yes**

4. **One pair + two different singles** (e.g., 1,3,4,1): Adding the pair value → 3+1+1 (not Full House). Adding anything else → at most 2+1+1+1 (not Full House). **Result: No**

5. **All different** (e.g., 13,12,11,10): Can't make 3+2 with just one more card. **Result: No**

So the answer is "Yes" only when the frequency pattern is [3,1] or [2,2].

- 8,8,8,8 → freq [4] → No ✓
