The manufacturing of a certain product requires N processes numbered 1,2,\dots,N.
For each process i, there are two types of machines S_i and T_i available for purchase to handle it.

Suppose that process i can handle W_i products per day as a result of introducing machines.
Here, we define the production capacity as the minimum of W, that is, \displaystyle \min^{N}_{i=1} W_i.
Given a total budget of X yen, find the maximum achievable production capacity.

The input is given from Standard Input in the following format:
N X
A_1 P_1 B_1 Q_1
A_2 P_2 B_2 Q_2
\vdots
A_N P_N B_N Q_N

Output

Print the answer as an integer.

Constraints


- All input values are integers.
- 1 \le N \le 100
- 1 \le A_i,B_i \le 100
- 1 \le P_i,Q_i,X \le 10^7

3 22
2 5 3 6
1 1 3 3
1 3 2 4

4

import sys

def min_cost(A, P, B, Q, m):
    """
    Minimum cost to achieve at least m capacity for one process
    using machines (A, P) and (B, Q).
    """
    best = 10**30          # sufficiently large

    # Case: number of S machines (x) is less than B
    for x in range(B):
        cap = x * A
        if cap >= m:
            cost = x * P
        else:
            y = (m - cap + B - 1) // B   # ceil division
            cost = x * P + y * Q
        if cost < best:
            best = cost

    # Case: number of T machines (y) is less than A
    for y in range(A):
        cap = y * B
        if cap >= m:
            cost = y * Q
        else:
            x = (m - cap + A - 1) // A   # ceil division
            cost = y * Q + x * P
        if cost < best:
            best = cost

    return best


def solve() -> None:
    input = sys.stdin.readline
    N, X = map(int, input().split())
    A, P, B, Q = [0]*N, [0]*N, [0]*N, [0]*N
    for i in range(N):
        a, p, b, q = map(int, input().split())
        A[i], P[i], B[i], Q[i] = a, p, b, q

    # binary search on the answer
    lo, hi = 0, 10**9 + 1   # hi is exclusive and certainly infeasible
    while hi - lo > 1:
        mid = (lo + hi) // 2
        total = 0
        feasible = True
        for i in range(N):
            cost = min_cost(A[i], P[i], B[i], Q[i], mid)
            total += cost
            if total > X:
                feasible = False
                break
        if feasible:
            lo = mid
        else:
            hi = mid

    print(lo)


if __name__ == "__main__":
    solve()
