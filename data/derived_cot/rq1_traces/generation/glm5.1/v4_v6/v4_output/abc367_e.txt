You will be given a competitive programming problem.
You are given a sequence X of length N where each element is between 1 and N, inclusive, and a sequence A of length N.
Print the result of performing the following operation K times on A.

- Replace A with B such that B_i = A_{X_i}.

Input

The input is given from Standard Input in the following format:
N K
X_1 X_2 \dots X_N
A_1 A_2 \dots A_N

Output

Let A' be the sequence A after the operations. Print it in the following format:
A'_1 A'_2 \dots A'_N

Constraints


- All input values are integers.
- 1 \le N \le 2 \times 10^5
- 0 \le K \le 10^{18}
- 1 \le X_i \le N
- 1 \le A_i \le 2 \times 10^5

Sample Input 1

5 2 6 3 1 4 6
1 2 3 5 7 9 11

- After one operation, the sequence is (7,2,9,3,1,5,9).
- After two operations, the sequence is (1,2,5,9,7,3,5).
- After three operations, the sequence is (7,2,3,5,1,9,3).

There may be cases where no operations are performed.

import sys

def solve() -> None:
    input = sys.stdin.readline

    N, K = map(int, input().split())
    X = list(map(int, input().split()))
    A = list(map(int, input().split()))

    # No operation
    if K == 0:
        print(' '.join(map(str, A)))
        return

    # 0‑based indices
    X = [x - 1 for x in X]

    # cur[i] = f^{current}(i)   (initially identity)
    cur = list(range(N))
    # up[i] = f^{2^j}(i) for the current j (initially j=0)
    up = X[:]

    max_bit = K.bit_length() - 1   # highest set bit

    for j in range(max_bit + 1):
        # if the j‑th bit of K is set, apply f^{2^j}
        if (K >> j) & 1:
            cur = [up[c] for c in cur]   # cur = up ∘ cur

        # prepare up for the next power (unless this is the last bit)
        if j < max_bit:
            up = [up[u] for u in up]      # up = up ∘ up

    # final answer
    ans = [A[cur[i]] for i in range(N)]
    print(' '.join(map(str, ans)))

if __name__ == "__main__":
    solve()
