=== ANSWER ===
Looking at this problem, I need to determine if Takahashi is awake at time A, given that he sleeps from B (bedtime) to C (wake time).

**Key insight:** The sleep schedule wraps around midnight differently depending on whether B < C or B > C:

**Case 1: B < C** (e.g., sleeps at 8, wakes at 14)
- He's asleep from B to C
- He's awake from C to B (wrapping around midnight)
- So he's awake at A if: `A >= C` or `A < B`

**Case 2: B > C** (e.g., sleeps at 21, wakes at 7)  
- He's asleep from B to C the next day (wrapping around midnight)
- He's awake from C to B (no wrapping)
- So he's awake at A if: `C <= A < B`

**Verification with examples:**

- Example 1: A=21, B=8, C=14 → B < C, so check `21 >= 14` → True → **Yes** ✓
- Example 2: A=0, B=21, C=7 → B > C, so check `7 <= 0 < 21` → False → **No** ✓
- Example 3: A=10, B=7, C=17 → B < C, so check `10 >= 17` or `10 < 7` → both False → **No** ✓

```python
A, B, C = map(int, input().split())

if B < C:
    # Awake from C to B (wrapping around midnight)
    if A >= C or A < B:
        print("Yes")
    else:
        print("No")
else:
    # Awake from C to B (no wrapping)
    if C <= A < B:
        print("Yes")
    else:
        print("No")
```
