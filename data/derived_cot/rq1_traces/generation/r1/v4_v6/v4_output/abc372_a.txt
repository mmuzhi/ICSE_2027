=== ANSWER ===
To solve this problem, we need to remove all occurrences of the '.' character from a given string and then print the resulting string. The solution involves a straightforward string manipulation where we filter out the '.' characters.

### Approach
1. **Problem Analysis**: The problem requires processing an input string that may contain lowercase English letters and '.' characters. The goal is to produce a new string where all '.' characters are removed.
2. **Intuition**: The solution can be efficiently achieved by replacing each '.' character in the string with an empty string, effectively removing them.
3. **Algorithm Selection**: The `replace` method of Python's string class is suitable here. It allows us to replace all occurrences of a specified substring (in this case, '.') with another substring (an empty string).
4. **Complexity Analysis**: The `replace` method operates in linear time, O(n), where n is the length of the string. This is efficient given the problem constraints (string length up to 100 characters).

### Solution Code
```python
def main():
    S = input().strip()
    result = S.replace('.', '')
    print(result)

if __name__ == '__main__':
    main()
```

### Explanation
1. **Reading Input**: The input string `S` is read from standard input.
2. **Processing**: The `replace` method is called on `S` to replace every '.' character with an empty string, effectively removing all '.' characters.
3. **Output**: The resulting string, which is the original string without any '.' characters, is printed.

This approach efficiently handles all possible cases, including strings that contain no '.' characters, strings consisting entirely of '.' characters (resulting in an empty string), and any combination in between. The solution is optimal for the given constraints.
