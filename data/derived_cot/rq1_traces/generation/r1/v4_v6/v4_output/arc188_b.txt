To solve this problem, we need to determine whether all points on a circle can be colored black when two players, Alice and Bob, alternately color points under specific symmetry constraints. The key insight revolves around understanding the symmetry conditions and how the players' choices affect the possibility of coloring all points.

### Approach
1. **Problem Analysis**: The circle has N points equally spaced. Alice starts at point 0 and Bob at point K. The players must color a white point such that after coloring, the entire coloring is symmetric with respect to the line connecting the current player (Alice or Bob) to the center of the circle. The goal is to check if all points can be colored black under optimal play.

2. **Symmetry Condition**: For a coloring to be symmetric with respect to the line through a player's position (say point `x`), for every colored point `y`, its reflection over the line through `x` and the center must also be colored. The reflection of a point `y` over the line through `x` can be computed as `(2*x - y) mod N`.

3. **Key Insight**: The problem can be reduced to checking if the circle can be partitioned into cycles where each cycle is closed under reflections over the lines through the players' initial positions and their subsequent moves. The main observation is that the entire coloring process is possible only if the group generated by the reflections (which form a dihedral group) can act transitively on the points. However, a simpler condition suffices:

   - Let `g = gcd(N, K)`. The points can be partitioned into `g` cycles, each of size `N / g`. Within each cycle, the coloring must be consistent. 
   - The players alternately color points, and each move must respect the symmetry condition. The symmetry lines change after each move because the players' positions change. However, the initial symmetry conditions (from Alice at 0 and Bob at K) and the subsequent moves must allow all points to be colored.

4. **Cycle Analysis**: Each cycle must be colorable without violating the symmetry conditions. The key is to note that the symmetry conditions imply that if a point `x` is colored, then so are `x`, `(2*0 - x) mod N`, `(2*K - x) mod N`, and so on. The group generated by the reflections over the lines through 0 and K is the same as the dihedral group generated by the reflections over the lines through multiples of `g` (where `g = gcd(N, K)`). The orbits (cycles) under this group action have size `N / g`. However, the coloring process requires that each move colors an entire orbit (or more precisely, a set that is closed under the reflections).

5. **Condition for Success**: The problem reduces to checking whether `N / g` is even. If `N / g` is even, then the two players can alternately color entire cycles (since each cycle has even size) without conflict, leading to all points being colored. If `N / g` is odd, then one cycle will be left uncolored because both players would need to color the same cycle at some point, which is not possible under the symmetry constraints.

import math
import sys

def main():
    data = sys.stdin.read().split()
    t = int(data[0])
    results = []
    index = 1
    for _ in range(t):
        N = int(data[index])
        K = int(data[index + 1])
        index += 2
        g = math.gcd(N, K)
        if (N // g) % 2 == 0:
            results.append("Yes")
        else:
