=== ANSWER ===
To solve this problem, we need to process a given string of digits by removing all characters except '2' and then concatenating the remaining '2's in their original order. The solution involves a straightforward filtering of the input string to retain only the '2's and then outputting the resulting string.

### Approach
1. **Input Reading**: Read the input string from standard input.
2. **Filtering Characters**: Iterate through each character in the string and keep only those characters that are '2'.
3. **Concatenation**: The retained '2's are concatenated in the order they appear in the original string.
4. **Output**: Print the resulting string of concatenated '2's.

This approach efficiently processes the string in linear time, O(n), where n is the length of the input string, by examining each character exactly once. The space complexity is also O(n) in the worst case, as we store the resulting string which could be up to the length of the input string if all characters are '2's.

### Solution Code
```python
def main():
    S = input().strip()
    result = ''.join(char for char in S if char == '2')
    print(result)

if __name__ == '__main__':
    main()
```

### Explanation
1. **Reading Input**: The input string `S` is read from standard input and stripped of any leading or trailing whitespace.
2. **Filtering Characters**: A comprehension is used to iterate over each character in `S`. For each character, it checks if the character is '2'. If it is, the character is included in the resulting list.
3. **Joining Characters**: The list of '2' characters is joined into a single string without any separators.
4. **Output**: The resulting string, which contains only '2's in their original order, is printed.

This solution efficiently meets the problem requirements by leveraging simple string operations and comprehensions for clarity and conciseness. The constraints ensure that the solution handles all edge cases, including strings of maximum length 100 and strings that contain at least one '2'.
