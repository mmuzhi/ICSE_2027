=== ANSWER ===
To solve this problem, we need to determine whether a given string is an 11/22 string based on specific conditions. An 11/22 string must meet the following criteria:
1. The length of the string must be odd.
2. The first half of the string (excluding the middle character) must consist entirely of '1's.
3. The middle character must be a '/'.
4. The second half of the string (excluding the middle character) must consist entirely of '2's.

### Approach
1. **Check String Length**: The first step is to verify if the length of the string is odd. If the length is even, the string cannot be an 11/22 string, and we immediately return "No".
2. **Find Middle Character**: For strings with an odd length, calculate the position of the middle character. The middle character is located at index `(N + 1) // 2 - 1` in a 0-based index system.
3. **Validate Middle Character**: Check if the middle character is '/'. If not, the string is invalid.
4. **Validate First Half**: Ensure all characters before the middle character are '1's.
5. **Validate Second Half**: Ensure all characters after the middle character are '2's.
6. **Output Result**: If all conditions are satisfied, output "Yes"; otherwise, output "No".

### Solution Code
```python
def main():
    N = int(input().strip())
    S = input().strip()
    
    if N % 2 == 0:
        print("No")
        return
        
    mid_index = (N + 1) // 2 - 1
    if S[mid_index] != '/':
        print("No")
        return
        
    if S[:mid_index] != '1' * mid_index:
        print("No")
        return
        
    if S[mid_index+1:] != '2' * (N - mid_index - 1):
        print("No")
        return
        
    print("Yes")

if __name__ == "__main__":
    main()
```

### Explanation
1. **Input Handling**: The code reads the integer `N` (length of the string) and the string `S` itself.
2. **Odd Length Check**: The first condition checks if `N` is even. If it is, the string cannot be an 11/22 string, so "No" is printed.
3. **Middle Character Check**: The middle character's position is calculated as `(N + 1) // 2 - 1`. If this character is not '/', the string is invalid.
4. **First Half Validation**: The substring before the middle character is checked to ensure it consists entirely of '1's. This is done by comparing it to a string of '1's of the same length.
5. **Second Half Validation**: The substring after the middle character is checked to ensure it consists entirely of '2's, similar to the first half validation.
6. **Result Output**: If all checks pass, "Yes" is printed; otherwise, "No" is printed at the first failure encountered.

This approach efficiently verifies all conditions required for the string to be classified as an 11/22 string, ensuring correctness with minimal computational overhead.
