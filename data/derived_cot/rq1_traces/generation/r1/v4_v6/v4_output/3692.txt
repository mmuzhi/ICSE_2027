You will be given a competitive programming problem.
You are given a string s and a pattern string p, where p contains exactly two '*' characters.
The '*' in p matches any sequence of zero or more characters.
Return the length of the shortest substring in s that matches p. If there is no such substring, return -1.
Note: The empty substring is considered valid.
 
Example 1:

Input: s = "abaacbaecebce", p = "ba*c*ce"
Output: 8
Explanation:
The shortest matching substring of p in s is "baecebce".

Example 2:

Input: s = "baccbaadbc", p = "cc*baa*adb"
Output: -1
Explanation:
There is no matching substring in s.

Example 3:

Input: s = "a", p = "**"
Output: 0
Explanation:
The empty substring is the shortest matching substring.

Example 4:

Input: s = "madlogic", p = "*adlogi*"
Output: 6
Explanation:
The shortest matching substring of p in s is "adlogi".

 
Constraints:

1 <= s.length <= 10^5
2 <= p.length <= 10^5
s contains only lowercase English letters.
p contains only lowercase English letters and exactly two '*'.

              # Split p into three parts
              part1 = p[:stars[0]]
              part2 = p[stars[0]+1:stars[1]]
              part3 = p[stars[1]+1:]

              def kmp_search(text, pattern):

                  m = len(pattern)

                  pi = [0] * m

                      if pattern[i] == pattern[j]:
                          j += 1
                      else:
                          j = 0
                      pi[i] = j

                  res = []

                          j = pi[j-1]

=== ANSWER ===
To solve this problem, we need to find the shortest substring in a given string `s` that matches a pattern `p` containing exactly two asterisks (`*`). The asterisks in the pattern can match any sequence of zero or more characters. The solution involves breaking the pattern into three parts separated by the asterisks, then efficiently searching for these parts in the string `s` while minimizing the length of the matching substring.

### Approach
1. **Pattern Splitting**: Split the pattern `p` into three parts: `part1` (before the first asterisk), `part2` (between the two asterisks), and `part3` (after the second asterisk).
2. **KMP Search**: For each non-empty part, use the Knuth-Morris-Pratt (KMP) algorithm to find all starting indices of occurrences in `s`. This ensures efficient searching even for large strings.
3. **Handling Empty Parts**: Special cases are handled where any of the parts might be empty (e.g., if the pattern starts or ends with an asterisk).
4. **Finding Valid Substrings**:
   - **Non-empty `part2`**: Iterate over each occurrence of `part2`. For each occurrence, find the largest occurrence of `part1` that ends before `part2` starts and the smallest occurrence of `part3` that starts after `part2` ends. The candidate substring length is calculated as the distance from the start of `part1` to the end of `part3`.
   - **Empty `part2`**: Directly check for valid occurrences of `part1` and `part3` such that `part3` starts after `part1` ends. The minimal substring length is derived from these occurrences or the lengths of `part1` or `part3` if the other is empty.
5. **Result Calculation**: The minimal valid substring length is returned, or `-1` if no valid substring exists.

### Solution Code
```python
import bisect

class Solution:
    def shortestMatchingSubstring(self, s: str, p: str) -> int:
        stars = []
        for i, ch in enumerate(p):
            if ch == '*':
                stars.append(i)
        part1 = p[:stars[0]]
        part2 = p[stars[0]+1:stars[1]]
        part3 = p[stars[1]+1:]
        
        def kmp_search(text, pattern):
            if not pattern:
                return None
            n = len(text)
            m = len(pattern)
            pi = [0] * m
            j = 0
            for i in range(1, m):
                while j > 0 and pattern[i] != pattern[j]:
                    j = pi[j-1]
                if pattern[i] == pattern[j]:
                    j += 1
                else:
                    j = 0
                pi[i] = j
            
            res = []
            j = 0
            for i in range(n):
                while j > 0 and text[i] != pattern[j]:
                    j = pi[j-1]
                if text[i] == pattern[j]:
                    j += 1
                if j == m:
                    res.append(i - m + 1)
                    j = pi[j-1]
            return res
        
        list1 = kmp_search(s, part1) if part1 else None
        list2 = kmp_search(s, part2) if part2 else None
        list3 = kmp_search(s, part3) if part3 else None
        
        INF = 10**9
        
        if part2:
            if list2 is None or len(list2) == 0:
                return -1
            if list1 is not None:
                list1.sort()
            if list3 is not None:
                list3.sort()
            
            ans = INF
            for i2 in list2:
                if part1:
                    if list1 is None or len(list1) == 0:
                        continue
                    pos = bisect.bisect_right(list1, i2 - len(part1)) - 1
                    if pos < 0:
                        continue
                    i1 = list1[pos]
                else:
                    i1 = i2
                
                if part3:
                    if list3 is None or len(list3) == 0:
                        continue
                    pos = bisect.bisect_left(list3, i2 + len(part2))
                    if pos == len(list3):
                        continue
                    i3 = list3[pos]
                else:
                    i3 = i2 + len(part2)
                
                candidate = i3 + len(part3) - i1
                if candidate < ans:
                    ans = candidate
            return ans if ans != INF else -1
        else:
            if not part1 and not part3:
                return 0
            ans = INF
            if part1 and part3:
                if list1 is None or len(list1) == 0 or list3 is None or len(list3) == 0:
                    return -1
                for i3 in list3:
                    pos = bisect.bisect_right(list1, i3 - len(part1)) - 1
                    if pos < 0:
                        continue
                    i1 = list1[pos]
                    candidate = i3 + len(part3) - i1
                    if candidate < ans:
                        ans = candidate
            elif part1:
                if list1 is None or len(list1) == 0:
                    return -1
                ans = len(part1)
            elif part3:
                if list3 is None or len(list3) == 0:
                    return -1
                ans = len(part3)
            return ans if ans != INF else -1
```

### Explanation
1. **Pattern Splitting**: The pattern `p` is split into three parts (`part1`, `part2`, `part3`) using the positions of the two asterisks.
2. **KMP Search**: The KMP algorithm efficiently finds all starting indices of each non-empty part in the string `s`, handling large input sizes optimally.
3. **Substring Validation**:
   - For non-empty `part2`, each occurrence is considered. The largest valid `part1` occurrence ending before `part2` starts and the smallest valid `part3` occurrence starting after `part2` ends are found using binary search. The minimal substring length is updated accordingly.
   - For empty `part2`, direct checks between `part1` and `part3` occurrences are performed. The minimal substring length is derived from the lengths of `part1` or `part3` if one is absent, or the combined length if both are present.
4. **Result**: The minimal valid substring length is returned, or `-1` if no valid substring exists. This approach efficiently handles all edge cases and large inputs within feasible time complexity.
