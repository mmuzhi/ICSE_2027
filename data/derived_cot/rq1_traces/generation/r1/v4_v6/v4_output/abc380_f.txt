To solve this problem, we need to determine the winner of a card game between Takahashi and Aoki, where both players play optimally. The game involves players taking turns to play cards from their hands onto the table, optionally taking a card from the table if it has a lower value than the one they just played. The player who cannot make a move loses.

### Approach
1. **Problem Analysis**: The game is a turn-based game with perfect information (both players know all cards). Each player starts with a set of cards, and there are additional cards on the table. Players alternate turns, starting with Takahashi. On each turn, a player must play one card from their hand onto the table. After playing, they may take one card from the table that has a lower value than the card they just played. The game ends when a player cannot make a move (i.e., they have no cards left to play), and that player loses.

2. **State Representation**: The state of the game can be represented by:
   - **Turn**: Indicates whose turn it is (Takahashi or Aoki).
   - **Hands**: The sets of cards each player currently holds.
   - **Table**: The set of cards on the table, which can be derived as the remaining cards not in either player's hand.

3. **Game Simulation**: Using a recursive approach with memoization, we simulate each possible move from the current state. For each move, we consider both options: taking a card from the table (if possible) or not taking any card. The outcome of each move is determined recursively by checking if the opposing player can win from the resulting state.

4. **Memoization**: To optimize the recursive solution, we memoize the results of previously computed states to avoid redundant calculations. The memoization key is a tuple containing the current turn and the frozensets representing each player's hand.

5. **Base Case**: If a player has no cards to play during their turn, they lose, and the opposing player wins.

### Solution Code
```python
import sys
sys.setrecursionlimit(1100000)

def main():
    data = sys.stdin.read().split()
    if not data:
        return
    n = int(data[0])
    m = int(data[1])
    l_val = int(data[2])
    A = list(map(int, data[3:3+n]))
    B = list(map(int, data[3+n:3+n+m]))
    C = list(map(int, data[3+n+m:3+n+m+l_val]))
    
    values = A + B + C
    total_cards = n + m + l_val
    U = set(range(total_cards))
    t0 = frozenset(range(n))
    a0 = frozenset(range(n, n+m))
    
    memo = {}
    
    def can_win(turn, t, a):
        t = frozenset(t)
        a = frozenset(a)
        key = (turn, t, a)
        if key in memo:
            return memo[key]
            
        table = U - t - a
        
        if turn == 0:
            if not t:
                memo[key] = False
                return False
            for x in t:
                new_t = t - {x}
                if not can_win(1, new_t, a):
                    memo[key] = True
                    return True
                for y in table:
                    if values[y] < values[x]:
                        new_t2 = (t - {x}) | {y}
                        if not can_win(1, new_t2, a):
                            memo[key] = True
                            return True
            memo[key] = False
            return False
        else:
            if not a:
                memo[key] = False
                return False
            for x in a:
                new_a = a - {x}
                if not can_win(0, t, new_a):
                    memo[key] = True
                    return True
                for y in table:
                    if values[y] < values[x]:
                        new_a2 = (a - {x}) | {y}
                        if not can_win(0, t, new_a2):
                            memo[key] = True
                            return True
            memo[key] = False
            return False
            
    result = can_win(0, t0, a0)
    print("Takahashi" if result else "Aoki")

if __name__ == "__main__":
    main()
```
