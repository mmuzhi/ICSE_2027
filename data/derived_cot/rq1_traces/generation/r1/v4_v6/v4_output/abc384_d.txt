To solve this problem, we need to determine if there exists a non-empty contiguous subsequence in an infinite periodic sequence (with period N) whose sum equals a given integer S. The sequence repeats every N elements, so any contiguous subsequence can span multiple periods. 

### Approach
1. **Problem Analysis**: The sequence is periodic with period N, meaning it repeats every N elements. We need to check for any contiguous subsequence (which can wrap around multiple periods) that sums to S. The challenge is to efficiently handle the infinite sequence by leveraging the periodic nature and constraints on the subsequence length.
2. **Key Insight**: Any contiguous subsequence can be decomposed into:
   - A "residue" part: a contiguous segment from one period (length between 1 and N).
   - Zero or more full periods (each summing to T, the total of one period).
3. **Conditions**:
   - **Condition 1**: The subsequence lies entirely within one or two consecutive periods (length ≤ 2N). We check if S is the sum of such a segment.
   - **Condition 2**: For longer subsequences (length > 2N), we express S as \( S = k \times T + B \), where \( k \geq 1 \) and \( B \) is the sum of a residue segment (1 ≤ B ≤ T). We check if such a \( B \) exists.
4. **Efficiency**: Using prefix sums and a sliding window technique, we efficiently check for the existence of residue segments (both for Condition 1 and Condition 2). The sliding window ensures we only consider segments of length ≤ N, making the solution feasible within O(N) time per check.

### Solution Code
```python
import sys
from collections import deque

def main():
    data = sys.stdin.read().split()
    n = int(data[0])
    s = int(data[1])
    A = list(map(int, data[2:2+n]))
    
    T = sum(A)
    D = A + A
    
    P = [0] * (2*n + 1)
    for i in range(1, 2*n + 1):
        P[i] = P[i-1] + D[i-1]
    
    def exists(x):
        count = {}
        queue = deque()
        count[P[0]] = 1
        queue.append(0)
        
        for j in range(0, 2*n):
            low_index = max(0, j + 1 - n)
            while queue and queue[0] < low_index:
                idx_out = queue.popleft()
                p_out = P[idx_out]
                count[p_out] -= 1
                if count[p_out] == 0:
                    del count[p_out]
            if (P[j+1] - x) in count:
                return True
            queue.append(j+1)
            count[P[j+1]] = count.get(P[j+1], 0) + 1
        return False

    if exists(s):
        print("Yes")
    else:
        if s > T:
            k0 = (s - 1) // T
            B = s - k0 * T
            if 1 <= B <= T and exists(B):
                print("Yes")
            else:
                print("No")
        else:
            print("No")

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The input consists of N (period length), S (target sum), and the first N terms of the sequence.
2. **Total Calculation**: Compute the total sum T of one period.
3. **Prefix Sum Array**: Construct a prefix sum array for two periods (2N elements) to handle circular segments.
4. **Function `exists(x)`**: Checks if there exists a contiguous segment of length ≤ N in the circular array that sums to x. It uses a sliding window approach with a deque to maintain valid start indices and a dictionary to track prefix sums efficiently.
5. **Condition Checks**:
   - **Condition 1**: Directly checks if S exists as a residue segment.
   - **Condition 2**: For S > T, computes \( B = S - k \times T \) (where \( k \) is derived from floor division) and checks if B is a valid residue segment.
6. **Output**: Prints "Yes" if either condition is satisfied, otherwise "No".

This approach efficiently handles the infinite sequence by leveraging periodicity and sliding window techniques, ensuring optimal performance for large inputs.
