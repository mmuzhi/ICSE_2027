=== ANSWER ===
To solve this problem, we need to determine the maximum number of containers that can be loaded onto an n x n cargo deck without exceeding the ship's maximum weight capacity. Each container has a fixed weight, and the deck has a limited number of cells. The solution involves considering both the physical space on the deck and the weight constraint.

### Approach
1. **Problem Analysis**: The problem requires finding the maximum number of containers that can be placed on an n x n grid such that the total weight of these containers does not exceed the ship's maximum weight capacity. Each container weighs exactly `w` units.
2. **Key Insight**: The maximum number of containers is constrained by two factors:
   - The physical space available on the deck, which is `n * n` cells.
   - The weight capacity, which allows up to `maxWeight // w` containers (since each container weighs `w` units).
3. **Algorithm Selection**: The solution involves computing the minimum of these two constraints. This ensures that we do not exceed either the deck's capacity or the weight limit.

### Solution Code
```python
class Solution:
    def maxContainers(self, n: int, w: int, maxWeight: int) -> int:
        total_cells = n * n
        max_by_weight = maxWeight // w
        return min(total_cells, max_by_weight)
```

### Explanation
1. **Total Cells Calculation**: The total number of cells available on the deck is calculated as `n * n` since the deck is an n x n grid.
2. **Weight Constraint Calculation**: The maximum number of containers allowed by the weight constraint is determined by integer division of `maxWeight` by `w` (i.e., `maxWeight // w`). This gives the number of containers whose combined weight does not exceed `maxWeight`.
3. **Result Determination**: The solution returns the minimum of the two values (`total_cells` and `max_by_weight`), ensuring that we adhere to both the physical space and weight constraints.

This approach efficiently combines both constraints to provide the optimal number of containers that can be loaded onto the ship's deck. The algorithm operates in constant time, O(1), making it highly efficient even for large values of `n` and `maxWeight`.
