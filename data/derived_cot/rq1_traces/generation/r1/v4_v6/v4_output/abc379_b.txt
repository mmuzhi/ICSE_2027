=== ANSWER ===
To solve this problem, we need to determine the maximum number of strawberries Takahashi can eat given a sequence of teeth, each either healthy (O) or having a cavity (X). The key constraint is that he can only eat a strawberry when he has K consecutive healthy teeth, and after eating, those teeth develop cavities and become unhealthy.

### Approach
1. **Problem Analysis**: The problem requires us to find non-overlapping segments of K consecutive healthy teeth (O's) in the string. Each time such a segment is found, it contributes one strawberry, and the teeth in that segment are marked as cavities (X), meaning they cannot be reused.
2. **Intuition**: The solution involves scanning the string to identify contiguous segments of healthy teeth (O's) separated by cavities (X's). For each contiguous segment of O's, the maximum number of non-overlapping K-length segments is simply the length of the segment divided by K (using integer division).
3. **Algorithm Selection**: The algorithm splits the input string into segments of consecutive O's separated by X's. For each segment, it calculates how many strawberries can be derived by dividing the segment length by K. The sum of these values across all segments gives the total number of strawberries Takahashi can eat.
4. **Complexity Analysis**: The algorithm processes the string once to split it into segments and then processes each segment once. The time complexity is O(N), where N is the length of the string, which is efficient given the constraints (N ≤ 100).

### Solution Code
```python
import sys

def main():
    data = sys.stdin.read().splitlines()
    n, k = map(int, data[0].split())
    s = data[1].strip()
    segments = s.split('X')
    total_strawberries = 0
    for seg in segments:
        total_strawberries += len(seg) // k
    print(total_strawberries)

if __name__ == '__main__':
    main()
```

### Explanation
1. **Reading Input**: The input is read from standard input, where the first line contains integers N (number of teeth) and K (required consecutive healthy teeth), and the second line is the string S representing the teeth conditions.
2. **Splitting Segments**: The string S is split into segments of consecutive O's using 'X' as the delimiter. This results in a list where each element is a substring of O's (or empty strings if no O's are present between X's).
3. **Calculating Strawberries**: For each segment of O's, the number of strawberries that can be derived is the length of the segment divided by K (using integer division). This value is accumulated into a total count.
4. **Output Result**: The total count of strawberries is printed as the result.

This approach efficiently counts the maximum number of strawberries by leveraging contiguous segments of healthy teeth and processing each segment independently, ensuring optimal performance even for the upper constraint limits.
