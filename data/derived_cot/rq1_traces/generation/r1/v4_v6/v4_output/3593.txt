=== ANSWER ===
To solve this problem, we need to find the maximum factor score of an array after removing at most one element. The factor score of an array is defined as the product of the Least Common Multiple (LCM) and the Greatest Common Divisor (GCD) of all elements in the array. 

### Approach
1. **Problem Analysis**: The problem requires us to compute the factor score for the array with the possibility of removing zero or one element. The factor score is the product of the LCM and GCD of the array elements. The key challenge is efficiently computing the LCM and GCD for the array and all possible subarrays formed by removing one element at a time.
2. **Intuition**: Since the array size is small (up to 100 elements), we can afford to compute the factor score for the original array and for each subarray obtained by removing one element individually. The LCM and GCD computations can be done sequentially: GCD by iteratively applying the GCD function pairwise, and LCM similarly using the formula LCM(a, b) = (a * b) / GCD(a, b).
3. **Algorithm Selection**: For each possible subarray (original array and each subarray with one element removed), compute the GCD and LCM. The factor score is then the product of these two values. The maximum score across all these subarrays is the solution.
4. **Complexity Analysis**: Given the constraints (n ≤ 100), the approach involves O(n^2) operations. For each of the n subarrays (including the original array), we compute the GCD and LCM in O(n) time, leading to a total of O(n^2) time complexity, which is efficient for the given constraints.

### Solution Code
```python
import math
from typing import List

class Solution:
    def maxScore(self, nums: List[int]) -> int:
        n = len(nums)
        if n == 0:
            return 0
        
        def lcm(a: int, b: int) -> int:
            return a * b // math.gcd(a, b)
        
        def compute_score(arr: List[int]) -> int:
            if not arr:
                return 0
            g = arr[0]
            m = arr[0]
            for i in range(1, len(arr)):
                g = math.gcd(g, arr[i])
                m = lcm(m, arr[i])
            return g * m
        
        best = compute_score(nums)
        for i in range(n):
            new_arr = nums[:i] + nums[i+1:]
            score = compute_score(new_arr)
            if score > best:
                best = score
        
        return best
```

### Explanation
1. **Initialization**: The solution starts by checking if the array is empty, in which case it returns 0. Otherwise, it proceeds to define helper functions for LCM and factor score computation.
2. **Helper Functions**: 
   - `lcm(a, b)`: Computes the LCM of two numbers using their GCD.
   - `compute_score(arr)`: Computes the factor score of an array by iteratively calculating the GCD and LCM of all elements in the array.
3. **Main Logic**: The initial factor score is computed for the entire array. Then, for each element in the array, a new subarray is formed by removing that element, and the factor score of this subarray is computed. The maximum score encountered during these computations is returned as the result.
4. **Efficiency**: The approach efficiently handles all possible subarrays by leveraging sequential GCD and LCM computations, ensuring optimal performance within the problem constraints.
