=== ANSWER ===
To solve this problem, we need to determine whether the outcome of a mayoral election is already decided based on the current vote counts. The election involves two candidates, Takahashi and Aoki, and there are N total votes, with T votes for Takahashi and A votes for Aoki so far. The remaining votes (N - T - A) are yet to be counted. The outcome is decided if one candidate will win regardless of how the remaining votes are distributed.

### Approach
1. **Problem Analysis**: The key observation is that the outcome is decided if one candidate already has more than half of the total votes (N/2) because even if all remaining votes go to the other candidate, the leading candidate will still have more votes. Since N is odd, half of N is not an integer, so having more than N/2 votes means having at least (N/2 + 0.5) votes, which translates to checking if 2*T > N or 2*A > N.
2. **Intuition**: If either candidate's current votes (T or A) multiplied by 2 exceeds the total votes N, then that candidate has an insurmountable lead. This is because the worst-case scenario (all remaining votes going to the opponent) still leaves the leading candidate with more votes.
3. **Algorithm Selection**: The solution involves a straightforward check: if either 2*T > N or 2*A > N, the outcome is decided ("Yes"); otherwise, it is not ("No").

### Solution Code
```python
def main():
    N, T, A = map(int, input().split())
    if 2 * T > N or 2 * A > N:
        print("Yes")
    else:
        print("No")

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The input consists of three integers: N (total votes), T (votes for Takahashi), and A (votes for Aoki).
2. **Condition Check**: The solution checks if either candidate's votes, when doubled, exceed the total votes N. This condition ensures that the candidate has more than half the total votes, making their lead unbeatable even if all remaining votes go to the opponent.
3. **Output Result**: If either condition is met, the program prints "Yes" indicating the outcome is decided. Otherwise, it prints "No" indicating the outcome depends on the remaining votes.

This approach efficiently determines the election outcome by leveraging simple arithmetic checks, ensuring optimal performance with constant time complexity O(1).
