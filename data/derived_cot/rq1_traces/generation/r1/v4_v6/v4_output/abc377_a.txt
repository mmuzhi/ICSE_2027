=== ANSWER ===
To solve this problem, we need to determine whether a given string of length 3 can be rearranged to match the string "ABC". The solution involves checking if the characters in the input string can be rearranged to form the exact sequence "A", "B", and "C" in order. 

### Approach
1. **Problem Analysis**: The problem requires checking if the input string can be rearranged to "ABC". Since the string length is fixed at 3, the solution is straightforward. The key observation is that the input string must contain exactly one 'A', one 'B', and one 'C' in any order. 
2. **Intuition**: By sorting the input string, we can directly compare it to the sorted version of "ABC", which is also "ABC". If the sorted input string matches "ABC", then it is possible to rearrange the characters to form "ABC"; otherwise, it is not possible.
3. **Algorithm Selection**: The algorithm involves sorting the characters of the input string and then checking if the sorted result equals the list ['A', 'B', 'C']. This approach efficiently verifies the required condition with minimal computational overhead.

### Solution Code
```python
def main():
    S = input().strip()
    if sorted(S) == ['A', 'B', 'C']:
        print("Yes")
    else:
        print("No")

if __name__ == '__main__':
    main()
```

### Explanation
1. **Reading Input**: The input string `S` is read from standard input.
2. **Sorting and Comparison**: The characters of `S` are sorted into a list. This sorted list is then compared against the list ['A', 'B', 'C'].
3. **Output Result**: If the sorted list matches ['A', 'B', 'C'], the output is "Yes"; otherwise, the output is "No". This approach efficiently checks whether the input string can be rearranged into "ABC" by leveraging sorting to handle any permutation of the characters. 

This solution efficiently addresses the problem with a time complexity of O(1) for sorting (since the string length is fixed at 3) and O(1) for the comparison, making it optimal for the given constraints.
