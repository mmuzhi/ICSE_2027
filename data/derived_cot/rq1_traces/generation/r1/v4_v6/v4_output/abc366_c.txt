=== ANSWER ===
To solve this problem, we need to efficiently process a series of queries on a bag of balls, where each query either adds a ball, removes a ball, or requests the count of distinct integers present in the bag. The challenge lies in maintaining the count of distinct integers dynamically as balls are added and removed.

### Approach
1. **Initialization**: We start with an empty bag, which we represent using a dictionary to keep track of the frequency of each integer present in the bag. Additionally, we maintain a variable `distinct` to store the current count of distinct integers.
2. **Processing Queries**:
   - **Type 1 (Add a ball)**: For each integer `x` added, we increment its frequency in the dictionary. If this is the first occurrence of `x` (i.e., its frequency becomes 1), we increment the `distinct` count.
   - **Type 2 (Remove a ball)**: For each integer `x` removed, we decrement its frequency in the dictionary. If the frequency of `x` drops to zero after removal, we decrement the `distinct` count.
   - **Type 3 (Query distinct count)**: We record the current value of `distinct` as the answer for this query.
3. **Output**: After processing all queries, we output the recorded answers for each Type 3 query in order.

This approach efficiently tracks the distinct integers in the bag by leveraging the frequency dictionary and the `distinct` counter, ensuring optimal performance with each operation being O(1) on average.

### Solution Code
```python
def main():
    import sys
    data = sys.stdin.read().split()
    q = int(data[0])
    freq = {}
    distinct = 0
    ans = []
    index = 1
    for _ in range(q):
        t = data[index]
        index += 1
        if t == '1':
            x = int(data[index])
            index += 1
            freq[x] = freq.get(x, 0) + 1
            if freq[x] == 1:
                distinct += 1
        elif t == '2':
            x = int(data[index])
            index += 1
            freq[x] -= 1
            if freq[x] == 0:
                distinct -= 1
        elif t == '3':
            ans.append(str(distinct))
    print("\n".join(ans))

if __name__ == "__main__":
    main()
```

### Explanation
- **Initialization**: The `freq` dictionary is used to count occurrences of each integer in the bag, and `distinct` is initialized to zero.
- **Processing Queries**:
  - **Type 1 (Add)**: When adding a ball with integer `x`, `freq[x]` is incremented. If `x` was not previously present (i.e., `freq[x]` becomes 1), `distinct` is incremented.
  - **Type 2 (Remove)**: When removing a ball with integer `x`, `freq[x]` is decremented. If `x`'s count drops to zero, `distinct` is decremented.
  - **Type 3 (Query)**: The current value of `distinct` is recorded as the answer for the query.
- **Output**: After processing all queries, the answers to Type 3 queries are printed in the order they were received.

This method ensures that each operation is handled in constant time on average, making the solution efficient for large input sizes as specified in the problem constraints.
