To solve this problem, we need to find the minimum number of moves required to travel from the tile containing the point (S_x + 0.5, S_y + 0.5) to the tile containing the point (T_x + 0.5, T_y + 0.5) on a two-dimensional plane covered with tiles arranged according to specific rules. The tiles are defined by integer triples (i, j, k) where 0 ≤ k < K. The arrangement of tiles depends on the parity of i and j.

### Approach
1. **Tile Identification**: For each point (x, y), we determine the tile (i, j, k) that contains it. The tile is identified based on the parity of i and j:
   - If i and j have the same parity, the tile covers the area where \(iK \leq x \leq (i+1)K\) and \(jK + k \leq y \leq jK + k + 1\).
   - If i and j have different parity, the tile covers the area where \(iK + k \leq x \leq iK + k + 1\) and \(jK \leq y \leq (j+1)K\).

2. **Coordinate Transformation**: To avoid floating-point inaccuracies, we use integer arithmetic to compute the tile indices (i, j) and the offset k. Specifically:
   - For a given coordinate (x, y), we compute \(i = \left\lfloor \frac{2x + 1}{2K} \right\rfloor\) and \(j = \left\lfloor \frac{2y + 1}{2K} \right\rfloor\).
   - Depending on the parity of i and j, k is computed as either \(\left\lfloor \frac{2y + 1 - 2jK}{2} \right\rfloor\) (same parity) or \(\left\lfloor \frac{2x + 1 - 2iK}{2} \right\rfloor\) (different parity).

3. **Distance Calculation**: 
   - If the source and destination tiles are the same (i.e., same i and j), the minimum moves is simply the absolute difference between their k values.
   - Otherwise, the minimum moves is the sum of the Manhattan distance between the tile indices (i and j) and an additional adjustment cost. The adjustment cost is the minimum of two values: 
     - The sum of the distances from the source k to 0 and from the destination k to (K-1).
     - The sum of the distances from the source k to (K-1) and from the destination k to 0.

import sys

def main():
    data = sys.stdin.read().split()
    if not data:
        return
    t = int(data[0])
    index = 1
    results = []
    for _ in range(t):
        K = int(data[index]); index += 1
        Sx = int(data[index]); index += 1
        Sy = int(data[index]); index += 1
        Tx = int(data[index]); index += 1
        Ty = int(data[index]); index += 1
        
        i_s = (2 * Sx + 1) // (2 * K)
        j_s = (2 * Sy + 1) // (2 * K)
        i_t = (2 * Tx + 1) // (2 * K)
        j_t = (2 * Ty + 1) // (2 * K)
        
        if (i_s % 2 == j_s % 2):
            k_s = (2 * Sy + 1 - 2 * j_s * K) // 2
        else:
            k_s = (2 * Sx + 1 - 2 * i_s * K) // 2
        
        if (i_t % 2 == j_t % 2):
            k_t = (2 * Ty + 1 - 2 * j_t * K) // 2
        else:
            k_t = (2 * Tx + 1 - 2 * i_t * K) // 2
        
        if (i_s, j_s) == (i_t, j_t):
            results.append(str(abs(k_s - k_t)))
        else:
            di = abs(i_s - i_t)
            dj = abs(j_s - j_t)
            extra = min(
                abs(k_s - 0) + abs(k_t - (K - 1)),
                abs(k_s - (K - 1)) + abs(k_t - 0)
            )
            ans = di + dj + extra
            results.append(str(ans))
    
    print("\n".join(results))

if __name__ == '__main__':
