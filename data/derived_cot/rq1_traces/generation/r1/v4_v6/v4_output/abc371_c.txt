You are given simple undirected graphs G and H, each with N vertices: vertices 1, 2, \ldots, N.
Graph G has M_G edges, and its i-th edge (1\leq i\leq M_G) connects vertices u_i and v_i.
Graph H has M_H edges, and its i-th edge (1\leq i\leq M_H) connects vertices a_i and b_i.
You can perform the following operation on graph H any number of times, possibly zero.

- Choose a pair of integers (i,j) satisfying 1\leq i<j\leq N. Pay A_{i,j} yen, and if there is no edge between vertices i and j in H, add one; if there is, remove it.

Find the minimum total cost required to make G and H isomorphic.
What is a simple undirected graph?
 A simple undirected graph is a graph without self-loops or multi-edges, where edges have no direction.

What does it mean for graphs to be isomorphic?
 Two graphs G and H with N vertices are isomorphic if and only if there exists a permutation (P_1,P_2,\ldots,P_N) of (1,2,\ldots,N) such that for all 1\leq i\lt j\leq N:

-  an edge exists between vertices i and j in G if and only if an edge exists between vertices P_i and P_j in H.

To solve this problem, we need to determine the minimum cost required to make two undirected graphs, G and H, isomorphic by performing edge flip operations on graph H. The cost of flipping an edge (i, j) in H is given by a matrix A_{i,j}. The solution involves checking all possible permutations of vertices in H to match the vertices in G, and for each permutation, calculating the cost of flipping edges in H such that the adjacency matrices of G and H (after relabeling) become identical.

### Approach
1. **Problem Analysis**: The problem requires transforming graph H into a graph that is isomorphic to graph G by flipping edges in H. Each flip operation on an edge (i, j) in H incurs a cost A_{i,j}. The goal is to find the permutation of vertices in H that minimizes the total flip cost to match the adjacency structure of G.

2. **Key Insight**: For any permutation of vertices in H, the cost is computed by comparing each edge in G with the corresponding edge in the permuted H. If the edge status (present or absent) differs between G and H for a pair (i, j), the cost A_{i,j} is added to the total cost for that permutation.

3. **Algorithm Selection**: Given the constraint N ≤ 8, we can generate all possible permutations of vertices (N! permutations) and for each permutation, compute the total flip cost by comparing all edge pairs between G and the permuted H. The minimum cost across all permutations is the answer.

4. **Complexity Analysis**: With N ≤ 8, there are 8! = 40320 permutations. For each permutation, we compare C(8,2) = 28 edge pairs. Thus, the total operations are 40320 * 28 ≈ 1.13e6, which is computationally feasible.

### Solution Code
```python
import itertools
import sys

def main():
    data = sys.stdin.read().splitlines()
    if not data:
        return
    n = int(data[0].strip())
    mg_line = data[1].split()
    if not mg_line:
        return
    mg = int(mg_line[0])
    adjG = [[0] * (n+1) for _ in range(n+1)]
    idx = 2
    for _ in range(mg):
        u, v = map(int, data[idx].split())
        idx += 1
        adjG[u][v] = 1
        adjG[v][u] = 1
        
    mh_line = data[idx].split()
    mh = int(mh_line[0])
    idx += 1
    adjH = [[0] * (n+1) for _ in range(n+1)]
    for _ in range(mh):
        a, b = map(int, data[idx].split())
        idx += 1
        adjH[a][b] = 1
        adjH[b][a] = 1
        
    cost = [[0] * (n+1) for _ in range(n+1)]
    for i in range(1, n):
        nums = list(map(int, data[idx].split()))
        idx += 1
        k = 0
        for j in range(i+1, n+1):
            cost[i][j] = nums[k]
            cost[j][i] = nums[k]
            k += 1
            
    perms = itertools.permutations(range(1, n+1))
    ans = float('inf')
    
    for p in perms:
        total_cost = 0
        for i in range(1, n+1):
            for j in range(i+1, n+1):
                u = p[i-1]
                v = p[j-1]
                if adjG[i][j] != adjH[u][v]:
                    total_cost += cost[u][v]
        if total_cost < ans:
            ans = total_cost
            
    print(ans)

if __name__ == "__main__":
    main()
```
