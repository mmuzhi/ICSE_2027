To solve this problem, we need to count the number of "good" arrays of size `n` where each element is in the range `[1, m]` and exactly `k` adjacent pairs of elements are equal. The solution involves combinatorial mathematics and modular arithmetic to handle large numbers efficiently.

### Approach
1. **Problem Analysis**: The key observation is that an array with exactly `k` equal adjacent pairs can be viewed as having `n - k` segments (contiguous blocks of the same value). Each segment must have a value from `[1, m]`, and adjacent segments must have different values.
2. **Combinatorial Insight**: 
   - **Segment Partitioning**: The number of ways to partition the array into `n - k` segments is given by the binomial coefficient `C(n-1, n-k-1)`, which is equivalent to `C(n-1, k)`.
   - **Color Assignment**: The first segment can be colored in `m` ways, and each subsequent segment can be colored in `m-1` ways (different from the previous segment). Thus, the total color assignments is `m * (m-1)^(n-k-1)`.
3. **Modular Arithmetic**: Since the result can be very large, all calculations are performed modulo `10^9 + 7`.
4. **Efficiency**: Precompute factorials and their modular inverses up to the maximum possible value (100,000) to efficiently compute binomial coefficients using precomputed values.

### Solution Code
```python
class Solution:
    mod = 10**9 + 7
    max_n_val = 100000
    fact = [1] * (max_n_val + 1)
    inv_fact = [1] * (max_n_val + 1)
    for i in range(1, max_n_val + 1):
        fact[i] = fact[i-1] * i % mod
    inv_fact[max_n_val] = pow(fact[max_n_val], mod - 2, mod)
    for i in range(max_n_val, 0, -1):
        inv_fact[i-1] = inv_fact[i] * i % mod

    def countGoodArrays(self, n: int, m: int, k: int) -> int:

        a = n - 1
        b = exp
        if b < 0 or b > a:
            binom_coef = 0
        else:

            binom_coef = binom_coef * Solution.inv_fact[a - b] % Solution.mod
        color_part = m * pow(m - 1, exp, Solution.mod) % Solution.mod
        return binom_coef * color_part % Solution.mod
```
