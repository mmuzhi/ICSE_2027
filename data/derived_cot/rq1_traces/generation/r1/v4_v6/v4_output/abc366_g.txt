- For every vertex v with a degree of at least 1, the total XOR of the numbers written on its adjacent vertices (excluding v itself) is 0.


What is XOR?

The XOR of two non-negative integers A and B, denoted as A \oplus B, is defined as follows:


- In the binary representation of A \oplus B, the bit at position 2^k \, (k \geq 0) is 1 if and only if exactly one of the bits at position 2^k in the binary representations of A and B is 1. Otherwise, it is 0.


For example, 3 \oplus 5 = 6 (in binary: 011 \oplus 101 = 110).

In general, the bitwise XOR of k integers p_1, \dots, p_k is defined as (\cdots ((p_1 \oplus p_2) \oplus p_3) \oplus \cdots \oplus p_k).  It can be proved that this is independent of the order of p_1, \dots, p_k.

Otherwise, let X_v be the integer written on vertex v, and print your solution in the following format. If multiple solutions exist, any of them will be accepted.
Yes

- 1 \leq u_i < v_i \leq N
- (u_i, v_i) \neq (u_j, v_j) for i \neq j.

The integers must be between 1 and \(2^{60} - 1\). The key insight is recognizing that the problem can be modeled using linear algebra over the binary field GF(2), where each bit of the assigned numbers must satisfy the same linear constraints independently.

2. **Matrix Construction**: Construct an adjacency matrix \(A\) where \(A[i][j] = 1\) if there is an edge between vertices \(i\) and \(j\) (zero otherwise). The system of equations derived from the problem conditions is \(A \cdot x = 0\) for each vertex, where \(x\) is the solution vector in GF(2).

3. **Kernel Computation**: Compute the kernel (null space) of matrix \(A\) using Gaussian elimination over GF(2). The kernel consists of all vectors \(x\) such that \(A \cdot x = 0\). The basis vectors of the kernel form the solution space.

4. **Feasibility Check**: If the kernel is trivial (only the zero vector), output "No" because no valid assignment exists. Additionally, for each vertex, if all basis vectors have a zero in its position, output "No" since that vertex cannot be assigned a non-zero number.

5. **Solution Assignment**: For each vertex, compute the integer value by combining the basis vectors. Specifically, for each vertex \(j\), the assigned integer is the bitwise OR of \(1 \ll (i + \text{offset})\) for each basis vector \(i\) where the \(j\)-th element is 1.
