=== ANSWER ===
To solve this problem, we need to determine if we can make either two horizontal or two vertical cuts on an n x n grid such that each of the three resulting sections contains at least one rectangle, and every rectangle belongs to exactly one section. The rectangles are non-overlapping and defined by their bottom-left and top-right coordinates.

### Approach
1. **Problem Analysis**:
   - The grid is divided into three sections by two horizontal or vertical cuts.
   - Each section must contain at least one rectangle entirely within it.
   - Rectangles must not span across sections (i.e., each rectangle must lie entirely within one section).

2. **Key Insights**:
   - **Horizontal Cuts**: The grid is divided into three horizontal strips. The rectangles must be partitioned such that:
     - The top edge of every rectangle in the bottom strip is below the first cut.
     - The bottom edge of every rectangle in the middle strip is above the first cut and its top edge is below the second cut.
     - The bottom edge of every rectangle in the top strip is above the second cut.
   - **Vertical Cuts**: Similarly, the grid is divided into three vertical strips with analogous conditions for the rectangles' left and right edges.

3. **Algorithm Selection**:
   - **Sorting**: For horizontal cuts, sort rectangles by their top-edge y-coordinate (ey). For vertical cuts, sort by their right-edge x-coordinate (ex).
   - **Suffix Arrays**: Compute suffix arrays to store the minimum start y-coordinate (sy) for horizontal cuts and minimum start x-coordinate (sx) for vertical cuts from each index to the end of the sorted list.
   - **Validation Arrays**: For each possible partition point, check if the minimum start coordinate of the remaining rectangles is greater than or equal to the maximum end coordinate of the current partition. This ensures no rectangle spans across sections.
   - **Efficient Checking**: Use prefix and suffix arrays to efficiently validate partitions without nested loops, ensuring linear time complexity.

4. **Complexity Analysis**:
   - **Time Complexity**: O(m log m) for sorting the rectangles (where m is the number of rectangles) and O(m) for computing suffix arrays and validation checks. Thus, overall O(m log m).
   - **Space Complexity**: O(m) for storing suffix arrays and validation arrays.

### Solution Code
```python
class Solution:
    def checkValidCuts(self, n: int, rectangles: List[List[int]]) -> bool:
        m = len(rectangles)
        if m < 3:
            return False
        
        # Check horizontal cuts
        sorted_ey = sorted(rectangles, key=lambda x: x[3])
        min_sy_suffix = [0] * m
        min_sy_suffix[-1] = sorted_ey[-1][1]
        for i in range(m-2, -1, -1):
            min_sy_suffix[i] = min(sorted_ey[i][1], min_sy_suffix[i+1])
        
        valid_j = [False] * m
        for j in range(m-1):
            valid_j[j] = (min_sy_suffix[j+1] >= sorted_ey[j][3])
        
        has_valid_suffix = [False] * m
        has_valid_suffix[m-1] = False
        for j in range(m-2, -1, -1):
            has_valid_suffix[j] = valid_j[j] or has_valid_suffix[j+1]
        
        for i in range(m-2):
            if min_sy_suffix[i+1] >= sorted_ey[i][3]:
                if has_valid_suffix[i+1]:
                    return True
        
        # Check vertical cuts
        sorted_ex = sorted(rectangles, key=lambda x: x[2])
        min_sx_suffix = [0] * m
        min_sx_suffix[-1] = sorted_ex[-1][0]
        for i in range(m-2, -1, -1):
            min_sx_suffix[i] = min(sorted_ex[i][0], min_sx_suffix[i+1])
        
        valid_j = [False] * m
        for j in range(m-1):
            valid_j[j] = (min_sx_suffix[j+1] >= sorted_ex[j][2])
        
        has_valid_suffix = [False] * m
        has_valid_suffix[m-1] = False
        for j in range(m-2, -1, -1):
            has_valid_suffix[j] = valid_j[j] or has_valid_suffix[j+1]
        
        for i in range(m-2):
            if min_sx_suffix[i+1] >= sorted_ex[i][2]:
                if has_valid_suffix[i+1]:
                    return True
        
        return False
```
