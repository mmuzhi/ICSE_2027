To solve this problem, we need to determine the minimum number of characters to insert into a given string such that the resulting string alternates between 'i' and 'o' starting with 'i' at odd positions and 'o' at even positions. The solution involves dynamic programming to efficiently compute the minimal insertions required.

### Approach
1. **Problem Analysis**: The goal is to transform the input string into a string where every odd-positioned character is 'i' and every even-positioned character is 'o'. Characters can be inserted at any position without changing the order of the original characters.
2. **Dynamic Programming Setup**: We use two arrays, `dp0` and `dp1`, where:
   - `dp0[i]` represents the minimum insertions needed starting from the `i-th` character when the next expected character is 'i' (state 0).
   - `dp1[i]` represents the minimum insertions needed starting from the `i-th` character when the next expected character is 'o' (state 1).
3. **Base Case**: For the end of the string (i = n), `dp0[n]` is 0 (no insertions needed if we are in state 0) and `dp1[n]` is 1 (one insertion of 'o' needed if we are in state 1 to reach state 0).
4. **Recursive Case**: For each character from the end to the start:
   - If the current character matches the expected character in the current state, we proceed to the next state without insertion.
   - If it doesn't match, we consider inserting the expected character, flipping the state, and then processing the current character again.
5. **State Transitions**: The transitions are derived by comparing the cost of using the current character (if it matches) versus inserting the expected character and then processing the current character in the new state.

### Solution Code
```python
def main():
    S = input().strip()
    n = len(S)
    dp0 = [10**9] * (n + 1)
    dp1 = [10**9] * (n + 1)
    dp0[n] = 0
    dp1[n] = 1
    for i in range(n - 1, -1, -1):
        if S[i] == 'i':
            c0 = dp1[i + 1]
        else:
            c0 = 10**9
        if S[i] == 'o':
            c1 = dp0[i + 1]
        else:
            c1 = 10**9
        dp0[i] = min(c0, c1 + 1)
        dp1[i] = min(c1, c0 + 1)
    print(dp0[0])

if __name__ == '__main__':
    main()
```

### Explanation
1. **Initialization**: The `dp0` and `dp1` arrays are initialized to a large number (10^9) to represent infinity, except for the base cases at the end of the string.
2. **Base Case Handling**: At the end of the string (i = n), `dp0[n]` is set to 0 because no insertions are needed if we are in state 0. `dp1[n]` is set to 1 because one insertion of 'o' is required to transition from state 1 to state 0.
3. **Processing Backwards**: The loop processes each character from the end of the string to the start. For each character:
   - If it matches the expected character in the current state (e.g., 'i' for state 0), the cost is the value from the next state (flipped state).
   - If it doesn't match, the cost is derived by inserting the expected character (adding 1 insertion) and then processing the current character in the new state.
4. **Result Extraction**: The answer is found in `dp0[0]`, representing the minimal insertions needed starting from the beginning of the string in state 0.

This approach efficiently computes the minimal insertions by leveraging dynamic programming to explore all possible states and transitions, ensuring optimal performance even for the maximum input size.
