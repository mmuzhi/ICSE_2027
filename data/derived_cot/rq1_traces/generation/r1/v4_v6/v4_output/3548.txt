To solve this problem, we need to count the number of n-digit integers that can be rearranged to form a k-palindromic integer. A k-palindromic integer is a palindrome that is divisible by k. The solution involves generating all possible digit frequency vectors that sum to n, filtering those that can form a palindrome, and then checking if any permutation of the digits (without leading zeros) forms a k-palindromic integer.

### Approach
1. **Generate Frequency Vectors**: For each possible digit frequency vector (f0, f1, ..., f9) that sums to n, ensure that the vector satisfies the palindrome condition (i.e., for even n, all frequencies must be even; for odd n, exactly one frequency must be odd).
2. **Count Valid Numbers**: For each valid frequency vector, compute the number of distinct n-digit numbers (without leading zeros) that can be formed from the digit multiset.
3. **Check k-Palindromic Condition**: For each digit multiset, determine if there exists a permutation (without leading zeros) that forms a palindrome divisible by k. This involves:
   - **Middle Digit Handling (for odd n)**: Identify the digit with an odd frequency (if n is odd) and compute its contribution to the remainder modulo k.
   - **Non-Middle Positions**: Use dynamic programming to assign digits to non-middle positions in the first half of the palindrome, tracking the remainder modulo k. The weights for each position are precomputed based on their positional contributions.
4. **Sum Valid Counts**: Sum the counts of all valid digit multisets that can form a k-palindromic integer.

### Solution Code
```python
import math
from collections import defaultdict

class Solution:
    def countGoodIntegers(self, n: int, k: int) -> int:
        if n % 2 == 0:
            half = n // 2
            b_vectors = []
            self.gen_vectors(half, 0, [0]*10, b_vectors)
            a_vectors = [[2 * b for b in vec] for vec in b_vectors]
        else:
            a_vectors = []
            half = (n - 1) // 2
            for d0 in range(10):
                b_vectors = []
                self.gen_vectors(half, 0, [0]*10, b_vectors)
                for b_vec in b_vectors:
                    a_vec = [2 * b for b in b_vec]
                    a_vec[d0] += 1
                    a_vectors.append(a_vec)
        
        m_prime = n // 2
        w = [0] * m_prime
        for i in range(m_prime):
            w[i] = (pow(10, i, k) + pow(10, n-1-i, k)) % k
        
        ans = 0
        for a_vec in a_vectors:
            if n - a_vec[0] == 0:
                continue
            denom = 1
            for f in a_vec:
                denom *= math.factorial(f)
            count_multiset = (math.factorial(n-1) * (n - a_vec[0])) // denom
            if count_multiset == 0:
                continue
            
            if n % 2 == 0:
                r_mid = 0
                x = [a_vec[d] // 2 for d in range(10)]
            else:
                d0 = None
                for d in range(10):
                    if a_vec[d] % 2 == 1:
                        d0 = d
                        break
                r_mid = (d0 * pow(10, (n-1)//2, k)) % k
                x = [a_vec[d] // 2 for d in range(10)]
            
            dp = {0: {tuple(x)}}
            for i in range(m_prime):
                new_dp = defaultdict(set)
                for r, states in dp.items():
                    for state in states:
                        for d in range(10):
                            if state[d] == 0:
                                continue
                            if i == 0 and d == 0:
                                continue
                            new_state = list(state)
                            new_state[d] -= 1
                            new_state = tuple(new_state)
                            new_r = (r + d * w[i]) % k
                            new_dp[new_r].add(new_state)
                dp = new_dp
            target_r = (-r_mid) % k
            if any(state for r, states in dp.items() if r == target_r):
                ans += count_multiset
        return ans

    def gen_vectors(self, n, idx, path, results):
        if idx == 9:
            path[idx] = n
            results.append(path[:])
            return
        for i in range(0, n+1):
            path[idx] = i
            self.gen_vectors(n - i, idx+1, path, results)
```

### Explanation
1. **Frequency Vector Generation**: The solution generates all valid digit frequency vectors that sum to `n` and satisfy the palindrome condition (even frequencies for even `n`, exactly one odd frequency for odd `n`). This is done by generating vectors of half the values (for even `n`) or adjusting for one odd digit (for odd `n`).
2. **Count Calculation**: For each valid vector, the number of distinct n-digit numbers (without leading zeros) is computed using combinatorial mathematics: `( (n-1)! * (n - f0) ) / (∏ f_d! )`.
3. **k-Palindromic Check**: Using dynamic programming, the solution checks if any permutation of the digit multiset forms a palindrome divisible by `k`. The DP tracks the remainder modulo `k` while assigning digits to positions in the first half of the palindrome, considering positional weights.
4. **Result Aggregation**: The counts of all valid multisets that can form a k-palindromic integer are summed to produce the final result.

This approach efficiently narrows down the problem to manageable computations by leveraging combinatorial generation and dynamic programming, ensuring correctness while handling constraints effectively.
