=== ANSWER ===
To solve this problem, we need to determine the number of empty squares in an 8x8 grid where we can place a piece such that it cannot be captured by any existing pieces. The existing pieces are represented by '#' and empty squares by '.'. A piece placed on a square (i, j) can be captured by any existing piece in the same row i or the same column j. 

### Approach
1. **Problem Analysis**: The goal is to count all empty squares (i, j) where there are no existing pieces in row i or column j. If there is any existing piece in row i or column j, placing a piece at (i, j) would make it vulnerable to capture.
2. **Intuition**: For each empty square (i, j), we need to check if the entire row i and column j are free of any existing pieces. If both conditions are satisfied, the square is a safe placement.
3. **Algorithm Selection**: 
   - **Precompute Row Safety**: For each row, check if it contains any '#' and mark it as unsafe if it does.
   - **Precompute Column Safety**: Similarly, for each column, check if it contains any '#' and mark it as unsafe if it does.
   - **Count Safe Squares**: Iterate over each square in the grid. For each empty square (i, j), if both row i and column j are marked as safe, increment the count.
4. **Complexity Analysis**: The solution involves processing each row and column once, leading to a time complexity of O(8 + 8 + 64) = O(1) since the grid size is fixed (8x8). The space complexity is O(1) for storing row and column safety flags.

### Solution Code
```python
def main():
    grid = []
    for _ in range(8):
        grid.append(input().strip())
    
    row_has_piece = [False] * 8
    col_has_piece = [False] * 8
    
    for i in range(8):
        if '#' in grid[i]:
            row_has_piece[i] = True
    
    for j in range(8):
        for i in range(8):
            if grid[i][j] == '#':
                col_has_piece[j] = True
                break
    
    count = 0
    for i in range(8):
        for j in range(8):
            if grid[i][j] == '.' and (not row_has_piece[i]) and (not col_has_piece[j]):
                count += 1
    print(count)

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The grid is read as 8 strings, each representing a row.
2. **Row Safety Check**: For each row, we check if it contains any '#' and update the `row_has_piece` array accordingly.
3. **Column Safety Check**: For each column, we check all rows for any '#' in that column and update the `col_has_piece` array.
4. **Counting Safe Squares**: For each empty square (i, j), if both `row_has_piece[i]` and `col_has_piece[j]` are False, the square is safe, and we increment the count.
5. **Output Result**: The total count of safe squares is printed.

This approach efficiently checks all necessary conditions to determine safe placements by leveraging precomputed row and column safety flags, ensuring optimal performance for the given problem constraints.
