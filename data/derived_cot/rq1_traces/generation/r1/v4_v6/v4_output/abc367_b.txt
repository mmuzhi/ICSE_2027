To solve this problem, we need to format a given real number such that the decimal part has no trailing zeros and there is no unnecessary trailing decimal point. The input is provided as a string representing a real number with up to three decimal places. The constraints specify that the number is between 0 (inclusive) and 100 (exclusive).

### Approach
1. **Input Handling**: Read the input string which represents the real number.
2. **Check for Decimal Point**: Determine if the input string contains a decimal point. This helps in deciding whether to process the number as having a fractional part or as an integer.
3. **Normalize Integer Part**: If the input has a decimal point, split the string into the integer part and the fractional part. The integer part is normalized by converting it to an integer and back to a string to remove any leading zeros.
4. **Strip Trailing Zeros**: The fractional part (if present) is processed to remove any trailing zeros. If the fractional part becomes empty after stripping zeros, only the normalized integer part is printed.
5. **Output Handling**: If the input does not contain a decimal point, the string is treated as an integer, normalized by converting to an integer and back to a string to remove leading zeros, and then printed.

### Solution Code
```python
s = input().strip()
if '.' in s:
    parts = s.split('.', 1)
    a = parts[0]
    b = parts[1]
    if a == '':
        a = '0'
    else:
        a = str(int(a))
    b_stripped = b.rstrip('0')
    if b_stripped == '':
        print(a)
    else:
        print(a + '.' + b_stripped)
else:
    if s == '':
        print('0')
    else:
        print(str(int(s)))
```

### Explanation
1. **Input Reading**: The input string is read and stripped of any leading or trailing whitespace.
2. **Decimal Point Check**: The code checks if the input string contains a decimal point. If it does, the string is split into two parts: the integer part (before the decimal) and the fractional part (after the decimal).
3. **Integer Normalization**: The integer part is checked for being empty (which would imply a value of 0) and normalized by converting it to an integer and back to a string. This step ensures any leading zeros are removed.
4. **Fractional Processing**: The fractional part is processed to remove trailing zeros using the `rstrip('0')` method. If the resulting fractional part is empty, only the normalized integer part is printed. Otherwise, the normalized integer part and the stripped fractional part are combined with a decimal point and printed.
5. **Integer Handling**: If the input does not contain a decimal point, it is directly converted to an integer and back to a string to remove any leading zeros, then printed.

This approach efficiently formats the real number according to the specified conditions, ensuring no trailing zeros in the decimal part and no unnecessary trailing decimal point.
