To solve this problem, we need to find the length of the longest contiguous subarray in an array of positive integers where the product of all elements in the subarray equals the product of the greatest common divisor (GCD) and the least common multiple (LCM) of the subarray. The solution involves checking all possible contiguous subarrays and determining whether each subarray meets the specified condition.

### Approach
1. **Problem Analysis**: The problem requires checking each contiguous subarray of the given array to see if it is "product equivalent". A subarray is product equivalent if the product of its elements equals the product of its GCD and LCM. 
2. **Key Insight**: 
   - For a subarray of length 1, the condition holds only if the single element is 1.
   - For a subarray of length 2, the condition always holds true due to the mathematical identity that the product of two numbers equals the product of their GCD and LCM.
   - For subarrays of length 3 or more, the condition holds if and only if all elements in the subarray are pairwise coprime (i.e., the GCD of every pair of elements is 1).
3. **Algorithm Selection**: We iterate over all possible contiguous subarrays. For each subarray:
   - If the length is 1, check if the element is 1.
   - If the length is 2, it is automatically valid.
   - If the length is 3 or more, check all pairs of elements to ensure they are pairwise coprime. If any pair fails this condition, the subarray is invalid.
4. **Complexity Analysis**: Given the constraints (array length up to 100), the approach involves checking all possible subarrays, leading to a time complexity of O(n^3) in the worst case. This is feasible because the maximum number of operations (around 100,000) is manageable for the given constraints.

### Solution Code
```python
import math
from typing import List

class Solution:
    def maxLength(self, nums: List[int]) -> int:
        n = len(nums)
        max_len = 0
        for i in range(n):
            window = []
            for j in range(i, n):
                window.append(nums[j])
                L = len(window)
                if L == 1:
                    if window[0] == 1:
                        if L > max_len:
                            max_len = L
                elif L == 2:
                    if L > max_len:
                        max_len = L
                else:
                    valid = True
                    for x in range(L):
                        for y in range(x+1, L):
                            if math.gcd(window[x], window[y]) != 1:
                                valid = False
                                break
                        if not valid:
                            break
                    if valid:
                        if L > max_len:
                            max_len = L
                    else:
                        break
        return max_len
```

### Explanation
1. **Initialization**: We start by initializing `max_len` to 0, which will store the length of the longest valid subarray found.
2. **Iterate Over Subarrays**: For each starting index `i` in the array, we consider all contiguous subarrays starting at `i` by extending the window to include subsequent elements (index `j` from `i` to the end of the array).
3. **Check Validity of Subarray**:
   - **Length 1**: The subarray is valid only if the single element is 1.
   - **Length 2**: The subarray is always valid.
   - **Length 3 or More**: We check all pairs of elements in the subarray. If any pair has a GCD greater than 1, the subarray is invalid. If valid, we update `max_len` if the current subarray length is greater.
4. **Early Termination**: If a subarray of length 3 or more is found invalid, we break the inner loop early since extending the window further will not make it valid.
5. **Result**: After processing all subarrays, `max_len` holds the length of the longest valid subarray, which is returned as the result.

This approach efficiently checks all possible subarrays while leveraging early termination to optimize performance, ensuring correctness within feasible computational limits.
