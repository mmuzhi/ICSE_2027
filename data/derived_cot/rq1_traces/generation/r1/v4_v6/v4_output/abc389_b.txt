=== ANSWER ===
To solve this problem, we need to find the positive integer \( N \) such that \( N! = X \), where \( X \) is a given integer not less than 2. The solution involves computing factorials incrementally until we find the factorial that matches the given \( X \).

### Approach
1. **Initialization**: Start with \( n = 1 \) and \( f = 1 \) (since \( 1! = 1 \)).
2. **Iterative Factorial Calculation**: For each subsequent integer \( n \), compute the factorial \( f \) by multiplying the current factorial by \( n \).
3. **Comparison with \( X \)**: After each computation, check if the current factorial \( f \) is greater than or equal to \( X \). If \( f \) equals \( X \), the corresponding \( n \) is the answer. Since \( X \) is guaranteed to be a factorial of some integer \( N \), the loop will terminate when \( f \) matches \( X \).

This approach efficiently narrows down the possible values of \( N \) by leveraging the fact that factorials grow rapidly, ensuring that the loop runs a minimal number of times (up to \( N = 20 \) for \( X \) up to \( 3 \times 10^{18} \)).

### Solution Code
```python
def main():
    X = int(input().strip())
    n = 1
    f = 1
    while f < X:
        n += 1
        f = f * n
    print(n)

if __name__ == '__main__':
    main()
```

### Explanation
1. **Initialization**: The variables `n` (current integer) and `f` (factorial of `n`) are initialized to 1.
2. **Loop Execution**: The loop continues as long as the computed factorial `f` is less than the input value \( X \).
3. **Factorial Update**: Inside the loop, `n` is incremented by 1, and `f` is updated by multiplying it with the new `n`.
4. **Termination Condition**: The loop exits once `f` becomes greater than or equal to \( X \). Given the problem constraints, `f` will exactly match \( X \) at this point, and the current value of `n` is printed as the result.

This approach efficiently computes the solution by leveraging the rapid growth of factorial values, ensuring minimal computational steps while accurately identifying the target \( N \).
