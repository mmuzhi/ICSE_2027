=== ANSWER ===
To solve this problem, we need to find the maximum difference between the frequencies of two distinct characters in a given string, where one character has an odd frequency and the other has an even frequency. The difference is calculated as the frequency of the character with an odd frequency minus the frequency of the character with an even frequency.

### Approach
1. **Frequency Calculation**: First, we count the frequency of each character in the string using a counter. This helps us understand how many times each character appears.
2. **Segregation of Frequencies**: We separate the frequencies into two lists: one for characters with odd frequencies and another for characters with even frequencies.
3. **Maximizing the Difference**: The maximum difference between an odd frequency and an even frequency is obtained by selecting the largest odd frequency and the smallest even frequency. The result is simply the difference between these two values (largest odd frequency minus smallest even frequency).

### Solution Code
```python
from collections import Counter

class Solution:
    def maxDifference(self, s: str) -> int:
        freq = Counter(s)
        odd_freqs = []
        even_freqs = []
        for count in freq.values():
            if count % 2 == 0:
                even_freqs.append(count)
            else:
                odd_freqs.append(count)
        return max(odd_freqs) - min(even_freqs)
```

### Explanation
1. **Frequency Calculation**: The `Counter` from the `collections` module is used to count occurrences of each character in the string. For example, for the string "aaaaabbc", the frequencies are 'a': 5, 'b': 2, and 'c': 1.
2. **Segregation of Frequencies**: We iterate over the frequency values and segregate them into two lists: `odd_freqs` for frequencies that are odd (like 5 and 1) and `even_freqs` for frequencies that are even (like 2).
3. **Maximizing the Difference**: The solution then computes the maximum value in `odd_freqs` (which is 5) and the minimum value in `even_freqs` (which is 2) and returns their difference (5 - 2 = 3). This approach ensures that we get the largest possible difference by leveraging the highest odd frequency and the lowest even frequency available in the string.

This method efficiently addresses the problem constraints and guarantees the optimal solution by focusing on the extreme values within the segregated frequency lists.
