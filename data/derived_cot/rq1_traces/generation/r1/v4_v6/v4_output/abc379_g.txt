To solve this problem, we need to count the number of ways to replace each '?' in a given grid with digits 1, 2, or 3 such that any two adjacent cells (sharing an edge) have different digits. The solution involves dynamic programming (DP) and careful state management to efficiently handle the constraints.

### Approach
1. **Problem Analysis**: The grid has cells containing digits 1, 2, 3, or '?'. The task is to replace all '?' with digits such that no two adjacent cells (horizontally or vertically) have the same digit. The challenge is to count all valid configurations modulo 998244353.

2. **Key Insight**: The constraints (H × W ≤ 200) allow us to consider the grid's smaller dimension (either height or width) to manage the state space efficiently. By transposing the grid if necessary, we ensure the width (W) is the smaller dimension (W ≤ 14), making it feasible to generate all valid row configurations.

3. **Dynamic Programming**: 
   - **Valid Masks Generation**: For each row, generate all possible digit sequences (masks) that satisfy horizontal adjacency constraints (no two adjacent digits in a row are the same) and fixed digit constraints (if a cell is not '?', it must match the given digit).
   - **DP State Transition**: For each row, compute the number of valid configurations by considering compatible masks from the previous row. Two masks are compatible if, for every column, the digits in the previous and current row differ (ensuring vertical adjacency constraints).

4. **Complexity Optimization**: 
   - **Transpose Grid**: If the original width is larger than height, transpose the grid to minimize the number of valid masks per row.
   - **Efficient Compatibility Check**: For each pair of masks (from previous and current rows), check vertical constraints in O(W) time. The total operations are manageable due to the small width (W ≤ 14).

### Solution Code
```python
MOD = 998244353

def main():
    import sys
    data = sys.stdin.read().splitlines()
    if not data: 
        print(0)
        return
    H, W = map(int, data[0].split())
    grid = []
    for i in range(1, 1 + H):
        grid.append(data[i].strip())
    
    if W > H:
        new_grid = []
        for j in range(W):
            new_row = []
            for i in range(H):
                new_row.append(grid[i][j])
            new_grid.append(''.join(new_row))
        grid = new_grid
        H, W = W, H
    else:
        grid = [list(row) for row in grid]
    
    valid_masks = []
    for i in range(H):
        masks = [[]]
        for j in range(W):
            new_masks = []
            choices = []
            if grid[i][j] == '?':
                choices = [1, 2, 3]
            else:
                choices = [int(grid[i][j])]
            for ass in masks:
                for c in choices:
                    if j > 0 and ass[-1] == c:
                        continue
                    new_masks.append(ass + [c])
            masks = new_masks
        valid_masks.append(masks)
    
    if not valid_masks:
        print(0)
        return
        
    dp = [1] * len(valid_masks[0])
    for i in range(1, H):
        new_dp = [0] * len(valid_masks[i])
        A = valid_masks[i-1]
        B = valid_masks[i]
        for ia, a in enumerate(A):
            for ib, b in enumerate(B):
                compatible = True
                for j in range(W):
                    if a[j] == b[j]:
                        compatible = False
                        break
                if compatible:
                    new_dp[ib] = (new_dp[ib] + dp[ia]) % MOD
        dp = new_dp
    
    ans = sum(dp) % MOD
    print(ans)

if __name__ == '__main__':
```
