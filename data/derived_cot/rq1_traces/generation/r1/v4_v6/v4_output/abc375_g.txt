- All input values are integers.

To solve this problem, we need to determine for each road in a given undirected graph whether the shortest path from city 1 to city N changes when that road is removed. The solution involves leveraging Dijkstra's algorithm to compute shortest paths and then analyzing the necessity of each road in maintaining the shortest path distance.

### Approach
1. **Graph Representation**: The cities and roads are represented as an adjacency list where each node (city) maintains a list of connected nodes (roads) with their respective lengths.
2. **Forward Dijkstra's Algorithm**: Compute the shortest distances from city 1 (node 0) to all other cities. Additionally, count the number of distinct shortest paths from city 1 to each city using dynamic updates during the Dijkstra's execution.
3. **Backward Dijkstra's Algorithm**: Similarly, compute the shortest distances from city N (node N-1) to all other cities and count the number of distinct shortest paths from each city to city N.
4. **Critical Road Check**: For each road (u, v, c), check if it lies on any shortest path from city 1 to city N. This is done by verifying if either direction (u to v or v to u) satisfies the condition where the sum of the forward distance to u, the road length, and the backward distance from v equals the total shortest path distance (D). If satisfied, the road is part of a shortest path.
5. **Necessity Check**: For roads part of any shortest path, determine if they are necessary by checking if the number of shortest paths that include the road (in either direction) equals the total number of shortest paths from city 1 to city N. If they are equal, removing the road will increase the shortest path distance (or make city N unreachable), so the answer is "Yes"; otherwise, it's "No".

### Solution Code
```python
import heapq
mod = 10**9 + 7

def main():
    import sys
    data = sys.stdin.read().split()
    n = int(data[0])
    m = int(data[1])
    graph = [[] for _ in range(n)]
    edges = []
    index = 2
    for _ in range(m):
        a = int(data[index])
        b = int(data[index+1])
        c = int(data[index+2])
        index += 3
        a0 = a - 1
        b0 = b - 1
        graph[a0].append((b0, c))
        graph[b0].append((a0, c))
        edges.append((a0, b0, c))
    
    INF = 10**18
    dist_forward = [INF] * n
    ways_forward = [0] * n
    dist_forward[0] = 0
    ways_forward[0] = 1
    heap = [(0, 0)]
    
    while heap:
        d, u = heapq.heappop(heap)
        if d != dist_forward[u]:
            continue
        for v, c in graph[u]:
            nd = d + c
            if nd < dist_forward[v]:
                dist_forward[v] = nd
                ways_forward[v] = ways_forward[u]
                heapq.heappush(heap, (nd, v))
            elif nd == dist_forward[v]:
                ways_forward[v] = (ways_forward[v] + ways_forward[u]) % mod
    
    dist_backward = [INF] * n
    ways_backward = [0] * n
    dist_backward[n-1] = 0
    ways_backward[n-1] = 1
    heap = [(0, n-1)]
    
    while heap:
        d, u = heapq.heappop(heap)
        if d != dist_backward[u]:
            continue
        for v, c in graph[u]:
            nd = d + c
            if nd < dist_backward[v]:
                dist_backward[v] = nd
                ways_backward[v] = ways_backward[u]
                heapq.heappush(heap, (nd, v))
            elif nd == dist_backward[v]:
                ways_backward[v] = (ways_backward[v] + ways_backward[u]) % mod
    
    D = dist_forward[n-1]
    results = []
    for a, b, c in edges:
        cond1 = dist_forward[a] + c + dist_backward[b] == D
        cond2 = dist_forward[b] + c + dist_backward[a] == D
        if not (cond1 or cond2):
            results.append("No")
            continue
        
        cnt = 0
        if cond1:
            cnt = (ways_forward[a] * ways_backward[b]) % mod
        else:
            cnt = (ways_forward[b] * ways_backward[a]) % mod
        
        total = ways_forward[n-1] % mod
        if cnt == total:
            results.append("Yes")
        else:
            results.append("No")
    
    for res in results:
        print(res)

if __name__ == "__main__":
    main()
```

### Explanation
1. **Graph Representation**: The input is parsed into an adjacency list where each node points to its neighbors along with the road length connecting them.
2. **Forward Dijkstra**: Computes the shortest distances from city 1 (node 0) to all other cities and counts the number of distinct shortest paths using a priority queue. During relaxation, if a new shorter path is found, the distance and path count are updated; if an equally short path is found, the path count is incremented modulo \(10^9 + 7\).
3. **Backward Dijkstra**: Similarly computes shortest distances from city N (node N-1) to all other cities and counts distinct shortest paths in reverse.
4. **Critical Road Check**: For each road, checks if it lies on any shortest path by verifying if the sum of forward distance to one end, the road length, and backward distance from the other end equals the total shortest path distance \(D\).
5. **Necessity Check**: For roads on shortest paths, checks if the count of paths using the road equals the total number of shortest paths. If so, removing the road affects the shortest path distance, resulting in "Yes"; otherwise, "No".

This approach efficiently checks each road's necessity by leveraging Dijkstra's algorithm and dynamic path counting, ensuring optimal performance for large graphs.
