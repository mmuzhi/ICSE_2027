=== ANSWER ===
To solve this problem, we need to determine whether Snuke wants to eat takoyaki based on which hand he is raising. The solution involves checking the given inputs (L and R) to decide the appropriate output according to the problem's instructions.

### Approach
1. **Problem Analysis**: The problem provides two integers, L and R, each being either 0 or 1. These integers indicate whether Snuke is raising his left hand (L = 1), right hand (R = 1), both hands, or neither.
2. **Key Insight**: 
   - If only the left hand is raised (L = 1 and R = 0), Snuke wants to eat takoyaki, so the output should be "Yes".
   - If only the right hand is raised (L = 0 and R = 1), Snuke does not want to eat takoyaki, so the output should be "No".
   - If both hands are raised (L = 1 and R = 1) or neither hand is raised (L = 0 and R = 0), the output should be "Invalid" as Snuke is not following instructions.
3. **Algorithm Selection**: The solution involves straightforward conditional checks to cover all possible combinations of L and R values. The conditions directly map the input pairs to the required outputs.

### Solution Code
```python
def main():
    L, R = map(int, input().split())
    if L == 1 and R == 0:
        print("Yes")
    elif L == 0 and R == 1:
        print("No")
    else:
        print("Invalid")

if __name__ == '__main__':
    main()
```

### Explanation
1. **Reading Input**: The input consists of two integers, L and R, which are read from standard input.
2. **Condition Checks**:
   - **Left Hand Only (L=1, R=0)**: This indicates Snuke wants to eat takoyaki, so "Yes" is printed.
   - **Right Hand Only (L=0, R=1)**: This indicates Snuke does not want to eat takoyaki, so "No" is printed.
   - **Both Hands or No Hands (Other Cases)**: If both hands are raised (L=1, R=1) or neither is raised (L=0, R=0), the situation is invalid, and "Invalid" is printed.
3. **Output**: Based on the conditions above, the program prints the corresponding result.

This approach efficiently handles all possible input combinations with minimal checks, ensuring clarity and correctness.
