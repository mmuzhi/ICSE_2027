=== ANSWER ===
To solve this problem, we need to sort a given list of strings in ascending order based on their lengths and then concatenate them into a single string. The solution involves reading the input, processing the strings, sorting them, and finally combining them in the specified order.

### Approach
1. **Read Input:** The first line of input is an integer `N` representing the number of strings. The subsequent `N` lines are the strings themselves.
2. **Store Strings:** Store these strings in a list for further processing.
3. **Sort by Length:** Sort the list of strings based on their lengths in ascending order. Since the problem constraints guarantee that all string lengths are distinct, we don't need to handle ties.
4. **Concatenate Strings:** Combine the sorted strings into a single string by joining them in order.
5. **Output Result:** Print the resulting concatenated string.

### Solution Code
```python
def main():
    n = int(input().strip())
    strings = []
    for _ in range(n):
        strings.append(input().strip())
    strings.sort(key=len)
    result = ''.join(strings)
    print(result)

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input:** The program starts by reading the integer `N` which indicates the number of strings to follow.
2. **Storing Strings:** Each subsequent line is read and stored in a list called `strings`.
3. **Sorting Strings:** The `sort` method is used on the `strings` list with the `key` parameter set to `len`. This sorts the strings based on their lengths in ascending order.
4. **Concatenating Strings:** The sorted list of strings is then joined into a single string using the `join` method, which concatenates all elements of the list without any separators.
5. **Output:** The resulting concatenated string is printed.

This approach efficiently processes the input, sorts the strings by length, and produces the desired output by concatenating them in the sorted order. The solution is optimal with a time complexity of O(N log N) due to the sorting step, which is efficient given the constraints (N ≤ 50).
