class Solution:
    def countSubstrings(self, s: str) -> int:
        n = len(s)
        total = 0
        # Loop over d from 1 to 9
        for d in range(1, 10):
            # Precompute pow10_d for exponents 0 to n (mod d)
            pow10_d = [1] * (n+1)
            # For exponent 0: 1 % d
            pow10_d[0] = 1 % d
            for i in range(1, n+1):
                pow10_d[i] = (pow10_d[i-1] * 10) % d
            
            # dp_d: prefix mod d for s[0:i]
            dp = [0] * (n+1)
            # dp[0] = 0
            for i in range(1, n+1):
                dp[i] = (dp[i-1] * 10 + int(s[i-1])) % d
            
            # H[i] = (dp[i] * pow10_d[n-i]) % d
            H = [0] * (n+1)
            for i in range(n+1):
                H[i] = (dp[i] * pow10_d[n-i]) % d  # n-i: because exponent for 10 is n-i
            
            # Frequency map for H[0] to H[j] (as we iterate j from 1 to n)
            freq = {}
            # Start by putting H[0]
            freq[H[0]] = 1
            res_d = 0
            # Iterate j from 1 to n (j is the end index of the prefix s[0:j], and the last digit of the substring is s[j-1])
            for j in range(1, n+1):
                # Check if the last digit is the digit d (i.e., s[j-1] == str(d))
                if s[j-1] == str(d):
                    # Add the frequency of H[j] in the map
                    res_d += freq.get(H[j], 0)
                # Update the frequency map for H[j]
                freq[H[j]] = freq.get(H[j], 0) + 1
            total += res_d
        return total

class Solution:
    def countSubstrings(self, s: str) -> int:
        n = len(s)
        total = 0
        for d in range(1, 10):
            # Precompute pow10_d for exponents 0 to n (mod d)
            pow10_d = [1] * (n+1)
            pow10_d[0] = 1 % d
            for i in range(1, n+1):
                pow10_d[i] = (pow10_d[i-1] * 10) % d
            
            # dp_d: prefix mod d for s[0:i]
            dp = [0] * (n+1)
            for i in range(1, n+1):
                dp[i] = (dp[i-1] * 10 + int(s[i-1])) % d
            
            # H[i] = (dp[i] * pow10_d[n-i]) % d
            H = [0] * (n+1)
            for i in range(n+1):
                H[i] = (dp[i] * pow10_d[n-i]) % d
            
            # Frequency map for H[0] to H[j] (as we iterate j from 1 to n)
            freq = {}
            freq[H[0]] = 1
            res_d = 0
            for j in range(1, n+1):
                if s[j-1] == str(d):
                    res_d += freq.get(H[j], 0)
                freq[H[j]] = freq.get(H[j], 0) + 1
            total += res_d
        return total

To solve this problem, we need to count the number of substrings in a given string of digits that are divisible by their non-zero last digit. The solution involves efficiently checking each substring ending with a non-zero digit and verifying if the numeric value of the substring is divisible by its last digit. Given the constraints, a brute-force approach is infeasible, so we use modular arithmetic and dynamic programming to optimize the solution.

### Approach
1. **Problem Analysis**: The problem requires counting all substrings of a given string where the substring's numeric value is divisible by its last digit (which must be non-zero). A substring can include leading zeros, but substrings ending with '0' are excluded because their last digit must be non-zero.

2. **Key Insight**: For each digit `d` (1-9), we can use modular arithmetic to check divisibility. Specifically, for each substring ending with digit `d`, we need to verify if the substring's value modulo `d` is zero. 

3. **Optimization**: 
   - **Prefix Moduli Array**: For each digit `d`, compute a prefix array where each entry at index `i` represents the numeric value of the substring `s[0..i-1]` modulo `d`.
   - **Power of 10 Moduli**: Precompute an array where each entry at index `i` represents \(10^i \mod d\) for efficient computation of the substring's value.
   - **Hashing**: Transform the prefix values using the power of 10 moduli to form a hash value `H[i]` for each prefix. This allows us to use a frequency map to count valid substrings ending at each position efficiently.

4. **Frequency Map**: For each digit `d`, iterate through the string, maintaining a frequency map of hash values encountered so far. For each position `j` where the digit matches `d`, the count of valid substrings ending at `j` is the frequency of the hash value `H[j]` in the map. Update the map with `H[j]` after processing each position.

5. **Complexity**: The algorithm processes each digit (1-9) independently, leading to a time complexity of \(O(n \times 9) = O(n)\), which is efficient for the constraint \(n \leq 10^5\).

```python
class Solution:
    def countSubstrings(self, s: str) -> int:
        n = len(s)
        total = 0
        for d in range(1, 10):
            pow10_d = [1] * (n + 1)
            pow10_d[0] = 1 % d
            for i in range(1, n + 1):
                pow10_d[i] = (pow10_d[i - 1] * 10) % d
            
            dp = [0] * (n + 1)
            for i in range(1, n + 1):
                dp[i] = (dp[i - 1] * 10 + int(s[i - 1])) % d
            
            H = [0] * (n + 1)
            for i in range(n + 1):
                H[i] = (dp[i] * pow10_d[n - i]) % d
            
            freq = {H[0]: 1}
            res_d = 0
            for j in range(1, n + 1):
                if s[j - 1] == str(d):
                    res_d += freq.get(H[j], 0)
                freq[H[j]] = freq.get(H[j], 0) + 1
            total += res_d
        return total
```

### Explanation
1. **Initialization**: The solution initializes the total count of valid substrings to zero.
2. **Digit Processing**: For each digit `d` from 1 to 9:
   - **Power of 10 Moduli**: Computes an array `pow10_d` where each element at index `i` is \(10^i \mod d\).
   - **Prefix Moduli Array**: Computes an array `dp` where each element at index `i` is the numeric value of the substring `s[0..i-1]` modulo `d`.
   - **Hash Array**: Computes an array `H` where each element `H[i]` is `(dp[i] * pow10_d[n - i]) % d`, transforming the prefix values into a hash for efficient comparison.
3. **Frequency Map**: Uses a frequency map to count occurrences of each hash value encountered. For each position `j` in the string:
   - If the digit at `j-1` matches `d`, increments the count of valid substrings by the frequency of `H[j]` in the map.
   - Updates the frequency map with `H[j]` after processing each position.
4. **Result Accumulation**: The counts for each digit `d` are summed to get the total number of valid substrings, which is returned as the result.

This approach efficiently checks all possible substrings ending with each non-zero digit using modular arithmetic and dynamic programming, ensuring optimal performance for large input sizes.
