=== PROMPT ===
You will be given a competitive programming problem.
You are given an integer array a of size 4 and another integer array b of size at least 4.
You need to choose 4 indices i_0, i_1, i_2, and i_3 from the array b such that i_0 < i_1 < i_2 < i_3. Your score will be equal to the value a[0] * b[i_0] + a[1] * b[i_1] + a[2] * b[i_2] + a[3] * b[i_3].
Return the maximum score you can achieve.
 
Example 1:

Input: a = [3,2,5,6], b = [2,-6,4,-5,-3,2,-7]
Output: 26
Explanation:
We can choose the indices 0, 1, 2, and 5. The score will be 3 * 2 + 2 * (-6) + 5 * 4 + 6 * 2 = 26.

Output: -1
Explanation:

class Solution:
    def maxScore(self, a: List[int], b: List[int]) -> int:
        n = len(b)
        MIN_VAL = -10**15
        
        max_val = [a[3] * x for x in b]
        for i in range(n-2, -1, -1):
            max_val[i] = max(max_val[i], max_val[i+1])
        
        old = max_val.copy()
        if n-2 >= 0:
            max_val[n-2] = a[2] * b[n-2] + old[n-1]
        for i in range(n-3, -1, -1):
            candidate = a[2] * b[i] + old[i+1]
            max_val[i] = max(candidate, max_val[i+1])
        
        old = max_val.copy()
        if n-3 >= 0:
            max_val[n-3] = a[1] * b[n-3] + old[n-2]
        for i in range(n-4, -1, -1):
            candidate = a[1] * b[i] + old[i+1]
            max_val[i] = max(candidate, max_val[i+1])
        
        res = MIN_VAL
        for i in range(n-3):
            candidate = a[0] * b[i] + max_val[i+1]
            if candidate > res:
                res = candidate
        
        return res
