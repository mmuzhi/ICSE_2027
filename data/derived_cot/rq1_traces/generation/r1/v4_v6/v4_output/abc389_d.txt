To solve this problem, we need to count the number of 1x1 squares completely contained within a circle of radius R centered at the origin. The squares are tiled infinitely on the 2D plane, and each square is identified by its integer coordinates (i, j). The key insight is that a square is completely inside the circle if all its four corners are within the circle's boundary. 

### Approach
1. **Problem Analysis**: The problem requires checking each square (i, j) to determine if all its corners lie within or on the circle of radius R. The corners of a square (i, j) are at (i±0.5, j±0.5). The farthest corner from the origin for any square (i, j) is at (|i| + 0.5, |j| + 0.5) in terms of Euclidean distance. Thus, the condition for the square to be inside the circle is:
   \[
   (|i| + 0.5)^2 + (|j| + 0.5)^2 \leq R^2
   \]
2. **Mathematical Transformation**: To avoid floating-point operations and ensure precision, we transform the condition into integer arithmetic. Multiplying both sides by 4 gives:
   \[
   (2|i| + 1)^2 + (2|j| + 1)^2 \leq 4R^2
   \]
   Here, \(A = 2|i| + 1\) and \(B = 2|j| + 1\) are odd integers.
3. **Efficient Counting**: For each possible \(x = |i|\) (from 0 to R), we compute \(A = 2x + 1\). If \(A^2 > 4R^2\), we break the loop early. Otherwise, we compute the maximum possible \(B\) such that \(B^2 \leq 4R^2 - A^2\). The number of valid odd integers \(B\) (which correspond to \(y = |j|\)) is \((B_{\text{max}} + 1) // 2\).
4. **Symmetry Handling**: For each valid (x, y) pair:
   - If \(x = 0\) (i.e., \(i = 0\)), the count is \(2 \times \text{count}_y - 1\) (accounting for the origin and symmetric points on the y-axis).
   - If \(x > 0\), the count is \(4 \times \text{count}_y - 2\) (accounting for symmetric points in all quadrants).

### Solution Code
```python
import math

def main():
    R = int(input().strip())
    R2 = R * R
    total = 0
    for x in range(0, R + 1):
        A = 2 * x + 1
        A2 = A * A
        if A2 > 4 * R2:
            break
        rem = 4 * R2 - A2
        B_max = math.isqrt(rem)
        count_y = (B_max + 1) // 2
        if count_y == 0:
            continue
        if x == 0:
            total += 2 * count_y - 1
        else:
            total += 4 * count_y - 2
    print(total)

if __name__ == "__main__":
    main()
```

1. **Input Handling**: The radius \(R\) is read from standard input.
2. **Initialization**: \(R^2\) is computed once to avoid repeated calculations. The variable `total` is initialized to zero to accumulate the count of valid squares.
3. **Loop Through x Values**: For each \(x\) (from 0 to \(R\)):
   - Compute \(A = 2x + 1\) and \(A^2\). If \(A^2\) exceeds \(4R^2\), the loop breaks early since further \(x\) values will only be larger.
   - The remainder \(4R^2 - A^2\) is computed. The integer square root of this remainder gives \(B_{\text{max}}\), the largest integer \(B\) such that \(B^2 \leq 4R^2 - A^2\).
   - The number of valid \(y\) values (odd integers \(B\)) is \((B_{\text{max}} + 1) // 2\).
4. **Count Valid Squares**: Depending on whether \(x\) is zero or positive, the count of valid squares is adjusted to account for symmetries (origin, axes, and quadrants).
5. **Output Result**: The accumulated count of valid squares is printed.

This approach efficiently checks all potential squares by leveraging mathematical transformations and symmetry, ensuring optimal performance even for large \(R\).
