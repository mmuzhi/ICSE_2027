=== ANSWER ===
To solve this problem, we need to generate a unique string of length `N` that satisfies specific conditions. The string must be a palindrome composed of characters '-' and '=', contain exactly one or two '=' characters, and if there are two '=' characters, they must be adjacent. 

### Approach
1. **Problem Analysis**: The key observation is that the string must be a palindrome, meaning it reads the same forwards and backwards. This symmetry dictates the positions of the '=' characters. 
2. **Conditions Handling**:
   - **Odd N**: For an odd-length string, the center character must be '=' to satisfy the condition of exactly one '='. The remaining characters are '-', ensuring the palindrome property.
   - **Even N**: For an even-length string, the two center characters must be '=' (adjacent) to satisfy the condition of exactly two '='. The rest of the characters are '-', maintaining the palindrome structure.
3. **Uniqueness**: Given the constraints, the solution is unique for each `N` because any deviation would violate the conditions (e.g., placing '=' characters elsewhere would break the palindrome or the count condition).

### Solution Code
```python
def main():
    N = int(input().strip())
    s = ['-'] * N
    if N % 2 == 1:
        mid = N // 2
        s[mid] = '='
    else:
        mid_left = N // 2 - 1
        mid_right = N // 2
        s[mid_left] = '='
        s[mid_right] = '='
    print(''.join(s))

if __name__ == '__main__':
    main()
```

### Explanation
1. **Initialization**: We start by creating a list of `N` characters, all initialized to '-'.
2. **Odd Length Handling**: If `N` is odd, the center index (calculated as `N // 2`) is set to '='. This ensures exactly one '=' at the center, maintaining the palindrome property.
3. **Even Length Handling**: If `N` is even, the two center indices (positions `N//2 - 1` and `N//2`) are set to '='. This ensures exactly two adjacent '=' characters, preserving the palindrome structure.
4. **Output**: The list is converted to a string and printed, providing the unique solution that meets all given conditions.

This approach efficiently constructs the required string by leveraging the symmetry of palindromes and the specific conditions on the number and placement of '=' characters. The solution is optimal with a time complexity of O(N) due to initializing the list and setting specific indices, which is efficient for the given constraints (N ≤ 100).
