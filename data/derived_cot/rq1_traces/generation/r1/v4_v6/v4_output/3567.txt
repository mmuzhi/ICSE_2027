=== ANSWER ===
To solve this problem, we need to convert a given date in the Gregorian calendar format (yyyy-mm-dd) into its binary representation. The binary representation is obtained by converting the year, month, and day into their respective binary forms without leading zeros and then combining them in the same year-month-day format using hyphens.

### Approach
1. **Parse the Input Date**: The input date string is split into three parts: year, month, and day. This is done using the `split('-')` method, which separates the string at each hyphen.
2. **Convert to Integers**: Each part (year, month, day) is converted from a string to an integer. This allows us to handle leading zeros correctly (e.g., "02" becomes 2).
3. **Convert to Binary**: Each integer is converted to its binary representation using Python's built-in `bin()` function. The `bin()` function returns a string that starts with '0b', so we slice off the first two characters to get the binary digits.
4. **Combine Binary Strings**: The binary strings for the year, month, and day are combined into a single string separated by hyphens, forming the final result.

### Solution Code
```python
class Solution:
    def convertDateToBinary(self, date: str) -> str:
        year_str, month_str, day_str = date.split('-')
        year = int(year_str)
        month = int(month_str)
        day = int(day_str)
        year_bin = bin(year)[2:]
        month_bin = bin(month)[2:]
        day_bin = bin(day)[2:]
        return f"{year_bin}-{month_bin}-{day_bin}"
```

### Explanation
1. **Splitting the Date**: The input date string is split into three components (year, month, day) using the `split('-')` method. For example, "2080-02-29" is split into ["2080", "02", "29"].
2. **Integer Conversion**: Each component is converted from a string to an integer. This step removes any leading zeros (e.g., "02" becomes 2).
3. **Binary Conversion**: The integers are converted to their binary equivalents using the `bin()` function. The result of `bin()` includes a '0b' prefix, which is removed by slicing the string from the third character onward.
4. **Result Construction**: The binary strings for the year, month, and day are combined into a single string separated by hyphens. For instance, the binary representations of 2080, 2, and 29 are "100000100000", "10", and "11101", respectively, resulting in "100000100000-10-11101".

This approach efficiently processes the input date into the required binary format by leveraging basic string manipulation and conversion functions, ensuring clarity and correctness.
