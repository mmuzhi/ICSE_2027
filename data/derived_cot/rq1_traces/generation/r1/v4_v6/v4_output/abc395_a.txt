=== ANSWER ===
To solve this problem, we need to determine whether a given sequence of integers is strictly increasing. A sequence is strictly increasing if each element is less than the next element in the sequence. 

### Approach
1. **Input Reading**: The first input line contains an integer `N`, which is the length of the sequence. The second input line contains `N` integers separated by spaces.
2. **Sequence Check**: Iterate through the sequence from the first element to the second-to-last element. For each element, compare it with the next element. If any element is not less than the next element, the sequence is not strictly increasing.
3. **Output Result**: If all consecutive pairs satisfy the condition (each element is less than the next), output "Yes"; otherwise, output "No".

### Solution Code
```python
def main():
    n = int(input().strip())
    arr = list(map(int, input().split()))
    
    for i in range(n - 1):
        if arr[i] >= arr[i + 1]:
            print("No")
            return
    
    print("Yes")

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The first line of input is read as an integer `N`, representing the length of the sequence. The second line is split into a list of integers.
2. **Checking Sequence**: A loop runs from the first element to the second-to-last element of the list. For each element at index `i`, it checks if the element is greater than or equal to the next element (at index `i+1`). If this condition is met at any point, it immediately prints "No" and exits the function.
3. **Output Result**: If the loop completes without finding any invalid pair, the sequence is strictly increasing, and "Yes" is printed.

This approach efficiently checks each consecutive pair in the sequence in linear time, making it optimal for the given constraints. The solution is straightforward and ensures correctness by verifying each necessary condition.
