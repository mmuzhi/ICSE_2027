To solve this problem, we need to find the maximum sum of a contiguous subarray after performing arbitrary deletions on the given array such that all elements in the subarray are unique. The key insight is that the optimal solution involves selecting a subset of distinct positive elements from the array, as including any negative elements would reduce the sum. If no positive elements exist, we consider zero if present; otherwise, we select the maximum (least negative) element in the array.

### Approach
1. **Collect Distinct Positive Elements**: Iterate through the array and collect all distinct positive elements. This helps in ensuring that each element in the subarray is unique and positive, contributing maximally to the sum.
2. **Calculate Sum of Distinct Positives**: Sum all distinct positive elements. If this sum is greater than zero, it represents the maximum possible sum for the subarray.
3. **Check for Zero**: If there are no distinct positive elements (sum is zero), check if the array contains any zero. If zero is present, return zero as the maximum sum.
4. **Handle All Negative Elements**: If there are no positive elements and no zero, return the maximum element in the array, which will be the least negative element.

### Solution Code
```python
class Solution:
    def maxSum(self, nums: List[int]) -> int:
        distinct_positives = set()
        for num in nums:
            if num > 0:
                distinct_positives.add(num)
        s = sum(distinct_positives)
        if s > 0:
            return s
        if 0 in nums:
            return 0
        return max(nums)
```

### Explanation
1. **Collecting Distinct Positives**: The code iterates through each number in the array, adding positive numbers to a set. This ensures that each positive number is considered only once, maintaining uniqueness.
2. **Sum Calculation**: The sum of these distinct positive numbers is computed. If this sum is positive, it is returned immediately as it represents the maximum possible sum under the given constraints.
3. **Zero Handling**: If the sum of distinct positives is zero (indicating no positive numbers), the code checks if zero exists in the array. If zero is found, it returns zero, as selecting a subarray with zero would yield a sum of zero, which is better than any negative sum.
4. **Negative Handling**: If no positive numbers or zero exist, the maximum element (which will be the least negative) is returned, ensuring the solution adheres to the problem constraints by selecting at least one element.

This approach efficiently addresses the problem by leveraging the properties of distinct elements and their contributions to the sum, ensuring optimal performance with a linear pass through the array.
