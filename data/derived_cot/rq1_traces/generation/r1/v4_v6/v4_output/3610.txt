=== ANSWER ===
To solve this problem, we need to compute the x-sum for every subarray of size `k` in the given array `nums`. The x-sum of a subarray is defined as the sum of all elements in the subarray that belong to the top `x` most frequent elements, where ties in frequency are broken by selecting the larger element.

### Approach
1. **Sliding Window Technique**: We use a sliding window of size `k` to process each subarray efficiently. The window starts from the beginning of the array and moves one element at a time to the right.
2. **Frequency Tracking**: For each window, we maintain a frequency array that counts occurrences of each number (since numbers are between 1 and 50, an array of size 51 suffices).
3. **Top x Selection**: For each window, we generate a list of distinct elements along with their frequencies. This list is sorted primarily by frequency in descending order and secondarily by the element value in descending order. The top `x` elements from this sorted list are selected.
4. **Sum Calculation**: The x-sum is computed by summing the products of each selected element's frequency and its value. This sum is then stored in the result array.
5. **Window Update**: As the window slides, we update the frequency array by decrementing the count of the element that exits the window and incrementing the count of the new element that enters the window.

### Solution Code
```python
class Solution:
    def findXSum(self, nums: List[int], k: int, x: int) -> List[int]:
        n = len(nums)
        freq = [0] * 51
        for num in nums[:k]:
            freq[num] += 1
        
        result = []
        for i in range(0, n - k + 1):
            if i > 0:
                freq[nums[i - 1]] -= 1
                freq[nums[i + k - 1]] += 1
            distinct = []
            for num_val in range(1, 51):
                if freq[num_val] > 0:
                    distinct.append((freq[num_val], num_val))
            distinct.sort(key=lambda item: (-item[0], -item[1]))
            s_val = 0
            count = min(x, len(distinct))
            for j in range(count):
                s_val += distinct[j][0] * distinct[j][1]
            result.append(s_val)
        return result
```

### Explanation
1. **Initialization**: The frequency array `freq` is initialized to zero. The first window of size `k` is processed to populate the initial frequencies.
2. **Sliding Window Processing**: For each subsequent window, the frequency of the outgoing element (leftmost element of the previous window) is decremented, and the frequency of the incoming element (rightmost element of the current window) is incremented.
3. **Top x Selection**: For each window, distinct elements (those with non-zero frequency) are collected into a list. This list is sorted such that elements with higher frequencies come first, and elements with the same frequency are ordered by their values in descending order.
4. **Sum Calculation**: The sum of the top `x` elements (or all elements if fewer than `x` distinct elements exist) is computed by multiplying each element's frequency by its value and summing these products.
5. **Result Construction**: The computed sum for each window is stored in the result array, which is returned after processing all windows.

This approach efficiently processes each subarray using a sliding window and frequency tracking, ensuring optimal performance given the problem constraints. The solution leverages sorting to handle tie-breaking and top element selection, providing the correct x-sum for each window.
