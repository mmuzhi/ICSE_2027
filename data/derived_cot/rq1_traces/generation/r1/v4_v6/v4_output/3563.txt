### Approach
1. **Problem Analysis**: The problem can be modeled as an assignment problem where each row can be assigned at most one cell (value), and all assigned values must be distinct. The challenge is to maximize the sum of these values while adhering to the constraints.
2. **Key Insight**: The problem resembles the maximum weight bipartite matching problem, where rows form one set of nodes and distinct values form the other set. Each row can be connected to values present in that row, with the weight being the value itself. The solution involves finding a matching where the sum of weights is maximized.
3. **Algorithm Selection**: The Hungarian algorithm is chosen for solving the assignment problem efficiently. However, since the problem allows skipping rows, we introduce dummy values for each row to represent skipping. The matrix is padded to a square matrix (110x110) to facilitate the Hungarian algorithm.
4. **Transformation**: The original matrix is transformed into a cost matrix where each entry is adjusted to ensure minimization by the Hungarian algorithm. Specifically, each value in the matrix is subtracted from a large number (101) to convert the maximization problem into a minimization problem.
5. **Result Calculation**: After solving the minimization problem using the Hungarian algorithm, the maximum score is derived by reversing the transformation: subtracting the computed minimum cost from the total possible cost (110 * 101).

class Solution:
    def maxScore(self, grid: List[List[int]]) -> int:
        n_rows = len(grid)
        if n_rows == 0:
            return 0
        distinct_vals = sorted(set(val for row in grid for val in row))
        m = len(distinct_vals)
        N = 110
        mat = [[0] * N for _ in range(N)]
        for i in range(n_rows):
            for j in range(m):
                if distinct_vals[j] in grid[i]:
                    mat[i][j] = distinct_vals[j]
                else:
                    mat[i][j] = -10**9
            for j in range(m, m + n_rows):
                if j - m == i:
                    mat[i][j] = 0
                else:
                    mat[i][j] = -10**9
            for j in range(m + n_rows, N):
                mat[i][j] = -10**9
        for i in range(n_rows, N):
            for j in range(N):
                mat[i][j] = 0
        big = 101
        cost_matrix = [[0] * N for _ in range(N)]
        for i in range(N):
            for j in range(N):
                cost_matrix[i][j] = big - mat[i][j]
        cost_matrix_1_indexed = [[0] * (N + 1) for _ in range(N + 1)]
        for i in range(1, N + 1):
            for j in range(1, N + 1):
                cost_matrix_1_indexed[i][j] = cost_matrix[i - 1][j - 1]
        hungarian = HungarianAlgorithm(cost_matrix_1_indexed)
        min_cost = hungarian.solve()
        return N * big - min_cost
