=== ANSWER ===
To solve this problem, we need to determine the maximum number of times we can perform an operation where we choose two balls of the same color and discard them. The input consists of four integers, each representing the color of a ball. The colors are integers between 1 and 4 inclusive.

### Approach
1. **Problem Analysis**: The key observation here is that each operation requires two balls of the same color. Therefore, the maximum number of operations we can perform is determined by the number of pairs of same-colored balls we can form from the four given balls.
2. **Intuition**: For each color, the number of pairs we can form is the integer division of the count of that color by 2. Summing these values for all colors gives the total number of operations possible.
3. **Algorithm Selection**: We use a frequency array to count occurrences of each color (1 to 4). Then, for each color, we calculate the number of pairs (count // 2) and sum these values to get the answer.
4. **Complexity Analysis**: The solution involves a single pass through the input to count frequencies and another pass through the frequency array (which has a fixed size of 4), resulting in an O(1) time complexity. The space complexity is also O(1) due to the fixed-size frequency array.

### Solution Code
```python
def main():
    data = list(map(int, input().split()))
    freq = [0] * 5
    for x in data:
        freq[x] += 1
    pairs = 0
    for i in range(1, 5):
        pairs += freq[i] // 2
    print(pairs)

if __name__ == '__main__':
    main()
```

### Explanation
1. **Reading Input**: The input is read as a list of four integers representing the colors of the balls.
2. **Frequency Counting**: A frequency array of size 5 (indices 0 to 4) is initialized to zero. The array is then populated by iterating over the input list, incrementing the count at the index corresponding to each color.
3. **Calculating Pairs**: For each color (1 to 4), the number of pairs is calculated as the integer division of the count by 2. The sum of these values gives the total number of operations possible.
4. **Output Result**: The total number of operations is printed.

This approach efficiently counts the pairs by leveraging the frequency of each color, ensuring optimal performance with constant time and space complexity.
