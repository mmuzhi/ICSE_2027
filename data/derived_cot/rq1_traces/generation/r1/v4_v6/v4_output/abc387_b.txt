=== ANSWER ===
To solve this problem, we need to find the sum of all integers in a 9x9 multiplication table that are not equal to a given integer X. The multiplication table consists of products of row and column indices (from 1 to 9). The solution involves calculating the total sum of all elements in the table and then subtracting the sum of all occurrences of X (if any).

### Approach
1. **Total Sum Calculation**: The total sum of all elements in the 9x9 multiplication table can be computed efficiently. Since each element at position (i, j) is i * j, the sum of all elements is the product of the sum of row indices (1 through 9) and the sum of column indices (1 through 9). Both sums are 45 (since 1+2+...+9 = 45), so the total sum is 45 * 45 = 2025.
2. **Counting Occurrences of X**: For each integer X, we need to count how many times it appears in the multiplication table. This is done by checking all possible divisors i of X such that when X is divided by i, the quotient j is an integer between 1 and 9. For each valid pair (i, j) where i * j = X, we increment the count.
3. **Adjusting Total Sum**: The final result is obtained by subtracting the product of the count of X and X itself from the total sum. If X does not appear in the table, the count is zero, and the result remains the total sum (2025).

### Solution Code
```python
def main():
    total = 2025  # 45 * 45
    X = int(input().strip())
    count = 0
    for i in range(1, 10):
        if X % i == 0:
            j = X // i
            if 1 <= j <= 9:
                count += 1
    result = total - count * X
    print(result)

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The input is read as an integer X, which is the number to be excluded from the sum.
2. **Initialization**: The total sum of all elements in the 9x9 multiplication table is precomputed as 2025 (since the sum of indices 1 to 9 is 45, and 45 * 45 = 2025).
3. **Counting Occurrences**: For each index i from 1 to 9, we check if X is divisible by i. If it is, we compute j = X / i. If j is an integer between 1 and 9, it means the pair (i, j) (and possibly (j, i) if i ≠ j) contributes to the count of X in the table.
4. **Result Calculation**: The result is derived by subtracting the total value contributed by all occurrences of X (count * X) from the total sum of the table (2025). If X does not appear in the table, the result remains 2025.
5. **Output**: The computed result is printed.

This approach efficiently computes the desired sum by leveraging mathematical insights and straightforward iteration, ensuring optimal performance even for the upper constraint values.
