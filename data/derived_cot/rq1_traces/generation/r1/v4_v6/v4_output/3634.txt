=== ANSWER ===
To solve this problem, we need to calculate the total score obtained by pairing characters in a string with their mirror characters. The mirror of a character is defined as the corresponding character when the alphabet is reversed (e.g., 'a' mirrors to 'z', 'b' to 'y', etc.). The pairing process involves iterating through the string from left to right, and for each character, finding the closest unmarked character to its left that is its mirror. Once found, both characters are marked, and the distance between their indices is added to the score.

### Approach
1. **Mirror Character Calculation**: For each character in the string, its mirror character is computed using the formula `chr(ord('a') + ord('z') - ord(ch))`.
2. **Stack Management**: We maintain a stack for each character encountered in the string. Each stack keeps track of the indices of unmarked characters of that type.
3. **Iterative Processing**: As we iterate through each character in the string:
   - If the stack for its mirror character is non-empty, we pop the most recent index (the closest to the current index) from that stack. The difference between the current index and the popped index is added to the score.
   - If the stack is empty, the current character's index is pushed onto its own stack, indicating it remains unmarked for potential future pairings.
4. **Efficiency**: The algorithm efficiently processes each character in O(1) per operation (stack push/pop), leading to an overall O(n) time complexity, where n is the length of the string. The space complexity is O(n) due to the storage required for the stacks.

### Solution Code
```python
from collections import defaultdict

class Solution:
    def calculateScore(self, s: str) -> int:
        stacks = defaultdict(list)
        score = 0
        
        for i, ch in enumerate(s):
            mirror_char = chr(ord('a') + ord('z') - ord(ch))
            if stacks[mirror_char]:
                j = stacks[mirror_char].pop()
                score += (i - j)
            else:
                stacks[ch].append(i)
        
        return score
```

### Explanation
1. **Initialization**: A `defaultdict` is used to manage stacks for each character encountered. The `score` variable is initialized to accumulate the total score.
2. **Character Processing**: For each character `ch` at index `i`:
   - The mirror character of `ch` is computed. For example, if `ch` is 'a', its mirror is 'z'.
   - The stack corresponding to this mirror character is checked. If it contains indices, the most recent (top) index is popped. The difference `i - j` (where `j` is the popped index) is added to `score`.
   - If the mirror character's stack is empty, the current index `i` is pushed onto the stack for `ch`, marking it as available for future pairings.
3. **Result**: After processing all characters, the accumulated `score` is returned, representing the sum of distances from all valid pairings during the iteration.

This approach efficiently processes the string in linear time by leveraging stack operations to track and pair characters dynamically as the iteration progresses. The solution ensures optimal performance even for large input sizes.
