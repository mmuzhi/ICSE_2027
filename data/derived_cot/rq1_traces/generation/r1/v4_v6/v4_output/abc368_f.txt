=== ANSWER ===
To solve this problem, we need to determine the winner of a game played by Anna and Bruno, who take turns performing operations on a sequence of integers. The game involves replacing each integer with one of its proper divisors (excluding the integer itself). The player who cannot make a move loses. The solution involves analyzing the game using the Sprague-Grundy theorem, which is used for impartial games.

### Approach
1. **Game Analysis**: The game can be modeled using Grundy numbers (or mex values), which represent the state of each integer in the sequence. The Grundy number for an integer is the minimum excludant (mex) of the Grundy numbers of all its proper divisors.
2. **Precomputation**: We precompute the divisors for all integers up to the maximum value in the input (100,000) using a sieve method. This involves iterating through each integer and collecting all its divisors.
3. **Grundy Number Calculation**: For each integer from 1 to 100,000, we compute its Grundy number. The Grundy number for 1 is 0 (terminal state). For other integers, the Grundy number is the mex of the Grundy numbers of their proper divisors.
4. **Determine Winner**: The winner is determined by the XOR (nim-sum) of the Grundy numbers of all integers in the sequence. If the nim-sum is non-zero, the first player (Anna) wins; otherwise, the second player (Bruno) wins.

### Solution Code
```python
import sys

def main():
    data = sys.stdin.read().split()
    n = int(data[0])
    A = list(map(int, data[1:1+n]))
    
    max_val = 100000
    divisors = [[] for _ in range(max_val + 1)]
    for i in range(1, max_val + 1):
        j = i
        while j <= max_val:
            divisors[j].append(i)
            j += i
    
    g = [0] * (max_val + 1)
    g[1] = 0
    for n_val in range(2, max_val + 1):
        divs = divisors[n_val]
        s = set()
        for d in divs:
            if d == n_val:
                continue
            s.add(g[d])
        mex = 0
        while mex in s:
            mex += 1
        g[n_val] = mex
    
    xor_val = 0
    for a in A:
        xor_val ^= g[a]
    
    print("Anna" if xor_val != 0 else "Bruno")

if __name__ == '__main__':
    main()
```

### Explanation
1. **Reading Input**: The input is read from standard input, which includes the number of integers and the sequence itself.
2. **Precomputing Divisors**: Using a sieve method, we generate a list of divisors for each integer up to 100,000. This is done efficiently by iterating through each integer and marking its multiples.
3. **Calculating Grundy Numbers**: For each integer from 2 to 100,000, we compute its Grundy number by considering the mex of the Grundy numbers of its proper divisors. The mex is the smallest non-negative integer not present in the set of Grundy numbers of the divisors.
4. **Determining the Winner**: The XOR (nim-sum) of the Grundy numbers of all integers in the sequence is computed. If the result is non-zero, Anna wins; otherwise, Bruno wins. This decision is based on the principles of the Sprague-Grundy theorem for impartial games.
