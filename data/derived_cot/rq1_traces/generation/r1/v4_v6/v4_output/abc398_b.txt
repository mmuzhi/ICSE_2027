=== ANSWER ===
To solve this problem, we need to determine whether we can select five cards from a given set of seven cards such that the selected cards form a full house. A full house consists of three cards of one number and two cards of a different number. 

### Approach
1. **Frequency Counting**: First, we count the frequency of each distinct number in the given seven cards. This helps us understand how many times each number appears.
2. **Check Conditions**: We then check two conditions:
   - **Triple Condition**: There must be at least one number that appears three or more times (to form the triple).
   - **Pair Condition**: There must be at least two distinct numbers that appear two or more times. This ensures that one of these numbers (different from the triple number) can form the pair.
3. **Decision Making**: If both conditions are satisfied, we can form a full house; otherwise, we cannot.

### Solution Code
```python
def main():
    cards = list(map(int, input().split()))
    freq = {}
    for card in cards:
        freq[card] = freq.get(card, 0) + 1
    
    distincts = freq.keys()
    count_triple = 0
    count_pair = 0
    for num in distincts:
        if freq[num] >= 3:
            count_triple += 1
        if freq[num] >= 2:
            count_pair += 1
    
    if count_triple >= 1 and count_pair >= 2:
        print("Yes")
    else:
        print("No")

if __name__ == "__main__":
    main()
```

### Explanation
1. **Frequency Counting**: The code reads the seven card values and uses a dictionary to count occurrences of each distinct number.
2. **Checking Conditions**: The code iterates over each distinct number in the dictionary:
   - **Triple Check**: For each number, if it appears three or more times, it increments the `count_triple` variable.
   - **Pair Check**: For each number, if it appears two or more times, it increments the `count_pair` variable.
3. **Decision Logic**: After processing all numbers, the code checks if there is at least one triple candidate (`count_triple >= 1`) and at least two distinct pair candidates (`count_pair >= 2`). If both conditions are met, it prints "Yes"; otherwise, it prints "No".

This approach efficiently checks the necessary conditions to form a full house by leveraging frequency counting and simple conditional checks, ensuring optimal performance even for the upper constraint limits.
