There is no substring matching the pattern.

Explanation:
The substrings "u", "uc", and "uck" match the pattern.

To solve this problem, we need to determine if a given pattern string `p` containing exactly one '*' can be matched as a substring of another string `s`. The '*' in the pattern can be replaced by any sequence of zero or more characters, including an empty string. The solution involves checking all possible substrings of `s` that could match the pattern by splitting the pattern into a prefix and suffix around the '*' and verifying if the substring starts with the prefix and ends with the suffix.

### Approach
1. **Split the Pattern**: The pattern `p` is split into two parts: the prefix (all characters before the '*') and the suffix (all characters after the '*').
2. **Check Validity of Substrings**: For each possible starting index `i` in the string `s`, check if the substring starting at `i` matches the prefix. If it does, proceed to check possible ending indices `j` for the substring.
3. **Check Suffix Match**: For each valid ending index `j`, ensure that the substring ends with the suffix. The minimal length of the substring must be at least the combined length of the prefix and suffix (or 1 if both are empty to ensure the substring is non-empty).
4. **Early Termination**: If at any point a valid substring is found that matches both the prefix and suffix, return `true`. If no such substring is found after all checks, return `false`.

### Solution Code
```python
class Solution:
    def hasMatch(self, s: str, p: str) -> bool:
        n = len(s)
        star_index = p.index('*')
        prefix = p[:star_index]
        suffix = p[star_index+1:]
        L_prefix = len(prefix)
        L_suffix = len(suffix)
        L_min = L_prefix + L_suffix
        
        for i in range(n):
            if i + L_prefix > n:
                continue
            if s[i:i+L_prefix] != prefix:
                continue
            min_j = i + max(1, L_min)
            if min_j > n:
                continue
            for j in range(min_j, n+1):
                if s[j - L_suffix:j] == suffix:
                    return True
        return False
```

### Explanation
1. **Pattern Splitting**: The pattern `p` is split into `prefix` (all characters before the '*') and `suffix` (all characters after the '*').
2. **Iterate Over Starting Indices**: For each starting index `i` in `s`, check if the substring starting at `i` of length `L_prefix` (length of the prefix) matches the prefix. If not, skip to the next starting index.
3. **Check Ending Indices**: For each valid starting index `i`, calculate the minimal ending index `min_j` which ensures the substring length is at least the combined length of the prefix and suffix (or 1 if both are empty). If `min_j` exceeds the string length, skip to the next starting index.
4. **Suffix Verification**: For each ending index `j` from `min_j` to the end of the string, check if the substring ending at `j` of length `L_suffix` matches the suffix. If a match is found, return `true` immediately.
5. **No Match Found**: If no valid substring is found after all iterations, return `false`.

This approach efficiently checks all possible substrings of `s` that could match the pattern by leveraging the constraints provided, ensuring optimal performance for the given problem constraints.
