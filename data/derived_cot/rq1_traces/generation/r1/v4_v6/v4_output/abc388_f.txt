- Let your current square be x. Choose an integer i that satisfies all of the following conditions, and move to square x + i.
- A \leq i \leq B
- x + i \leq N
- Square x + i is not bad.

Input

N M A B
L_1 R_1
L_2 R_2
\vdots
L_M R_M

Output

- 1 \leq A \leq B \leq 20

- R_i < L_{i+1} \ (1 \leq i \leq M - 1)
- All input values are integers.

To solve this problem, we need to determine whether it's possible to move from square 1 to square N in a row of squares, avoiding certain "bad" squares, by jumping forward a distance between A and B (inclusive) at each step. The bad squares are defined by given intervals, and we can only land on squares that are not bad.

### Approach
1. **Identify Safe Intervals**: The safe squares are the complement of the union of the given bad intervals. Since the bad intervals are disjoint and sorted, we can compute the safe intervals as the gaps between these bad intervals, including the area before the first bad interval and after the last bad interval.
2. **BFS Over Safe Intervals**: We use a BFS approach to traverse the safe intervals. Starting from the first safe interval (which contains square 1), we check if we can jump from the current safe interval to any subsequent safe intervals. The jump is valid if the range `[current_start + A, current_end + B]` overlaps with the next safe interval `[next_start, next_end]`.
3. **Check Reachability to N**: After processing all reachable safe intervals, we check if any square within a reachable safe interval can jump directly to square N. Specifically, for each possible jump distance `i` (from A to B), we check if the square `N - i` is within any reachable safe interval.

### Solution Code
```python
import sys
from collections import deque

def main():
    data = sys.stdin.read().split()
    if not data:
        print("No")
        return
    
    it = iter(data)
    N = int(next(it))
    M = int(next(it))
    A = int(next(it))
    B = int(next(it))
    
    intervals = []
    for _ in range(M):
        L = int(next(it))
        R = int(next(it))
        intervals.append((L, R))
    
    safe_intervals = []
    if M == 0:
        safe_intervals.append((1, N))
    else:
        safe_intervals.append((1, intervals[0][0] - 1))
        for i in range(M - 1):
            start = intervals[i][1] + 1
            end = intervals[i+1][0] - 1
            if start <= end:
                safe_intervals.append((start, end))
        safe_intervals.append((intervals[-1][1] + 1, N))
    
    n_safe = len(safe_intervals)
    if n_safe == 0:
        print("No")
        return
    
    visited = [False] * n_safe
    q = deque()
    q.append(0)
    visited[0] = True
    
    while q:
        idx = q.popleft()
        s_i, e_i = safe_intervals[idx]
        j = idx + 1
        while j < n_safe:
            s_j, e_j = safe_intervals[j]
            if s_i + A <= e_j and e_i + B >= s_j:
                if not visited[j]:
                    visited[j] = True
                    q.append(j)
            if s_j > e_i + B:
                break
            j += 1
    
    steps = range(A, B + 1)
    for step in steps:
        x = N - step
        if x < 1:
            break
        for idx in range(n_safe):
            if visited[idx]:
                s, e = safe_intervals[idx]
                if s <= x <= e:
                    print("Yes")
                    return
    print("No")
```
