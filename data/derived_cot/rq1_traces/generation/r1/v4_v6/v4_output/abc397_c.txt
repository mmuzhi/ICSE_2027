=== ANSWER ===
To solve this problem, we need to find the maximum sum of distinct integers in two contiguous subarrays obtained by splitting the given array at any position. The solution involves efficiently computing the distinct counts for all possible left and right subarrays and then determining the optimal split position that maximizes their sum.

### Approach
1. **Problem Analysis**: The problem requires splitting an array into two non-empty contiguous subarrays such that the sum of distinct integers in the left and right subarrays is maximized. The challenge is to compute this efficiently for large arrays (up to 300,000 elements) without resorting to a brute-force O(N^2) approach.
2. **Key Insight**: The distinct counts for the left and right subarrays can be precomputed using two passes over the array:
   - **Forward Pass**: Compute an array `left_arr` where each element `left_arr[i]` represents the distinct count in the subarray from the start to index `i`.
   - **Backward Pass**: Compute an array `right_arr` where each element `right_arr[i]` represents the distinct count in the subarray from index `i` to the end.
3. **Optimal Split**: After precomputing `left_arr` and `right_arr`, iterate over all possible split positions (from 1 to N-1 in 0-based indexing) and compute the sum `left_arr[i-1] + right_arr[i]`. The maximum value encountered during this iteration is the answer.

### Solution Code
```python
import sys

def main():
    data = sys.stdin.read().split()
    n = int(data[0])
    A = list(map(int, data[1:1+n]))
    
    left_arr = [0] * n
    freq_left = [0] * (n + 1)
    distinct_left = 0
    for i in range(n):
        num = A[i]
        freq_left[num] += 1
        if freq_left[num] == 1:
            distinct_left += 1
        left_arr[i] = distinct_left
        
    right_arr = [0] * n
    freq_right = [0] * (n + 1)
    distinct_right = 0
    for i in range(n - 1, -1, -1):
        num = A[i]
        freq_right[num] += 1
        if freq_right[num] == 1:
            distinct_right += 1
        right_arr[i] = distinct_right
        
    ans = 0
    for i in range(1, n):
        candidate = left_arr[i - 1] + right_arr[i]
        if candidate > ans:
            ans = candidate
            
    print(ans)

if __name__ == '__main__':
    main()
```

### Explanation
1. **Reading Input**: The input is read from standard input, where the first value is the array length `N`, followed by the array elements.
2. **Forward Pass (left_arr)**: 
   - Initialize `freq_left` to keep track of element frequencies and `distinct_left` to count distinct elements.
   - Traverse the array from left to right. For each element, update its frequency. If the frequency becomes 1 (indicating a new distinct element), increment `distinct_left`. Store the current distinct count in `left_arr[i]`.
3. **Backward Pass (right_arr)**:
   - Similarly, initialize `freq_right` and `distinct_right`.
   - Traverse the array from right to left. For each element, update its frequency. If the frequency becomes 1, increment `distinct_right`. Store the current distinct count in `right_arr[i]`.
4. **Finding Maximum Sum**: Iterate over all possible split positions (from 1 to N-1). For each position `i`, the sum of distinct counts is `left_arr[i-1]` (left subarray) and `right_arr[i]` (right subarray). Track the maximum sum encountered.
5. **Output Result**: Print the maximum sum obtained.

This approach efficiently precomputes necessary values in O(N) time and space, making it suitable for large input sizes.
