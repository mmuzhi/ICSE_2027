=== ANSWER ===
To solve this problem, we need to determine if a given 6-digit integer meets specific conditions regarding the counts of digits 1, 2, and 3. The conditions are:
1. The digit 1 must appear exactly once.
2. The digit 2 must appear exactly twice.
3. The digit 3 must appear exactly three times.

### Approach
1. **Input Handling**: Convert the input integer into a string to easily process each digit.
2. **Digit Counting**: Count the occurrences of each digit (1, 2, and 3) in the string.
3. **Condition Check**: Verify if the counts of digits 1, 2, and 3 are exactly 1, 2, and 3 respectively.
4. **Output Result**: Print "Yes" if all conditions are satisfied; otherwise, print "No".

The solution leverages the fact that the total number of digits is fixed at 6. Therefore, if the counts of digits 1, 2, and 3 sum to 6 (1 + 2 + 3 = 6), there cannot be any other digits in the number. This simplifies the check to only verifying the counts of these specific digits.

### Solution Code
```python
def main():
    N = input().strip()
    count_1 = N.count('1')
    count_2 = N.count('2')
    count_3 = N.count('3')
    if count_1 == 1 and count_2 == 2 and count_3 == 3:
        print("Yes")
    else:
        print("No")

if __name__ == "__main__":
    main()
```

### Explanation
1. **Input Handling**: The input integer is read as a string to facilitate digit-by-digit processing.
2. **Digit Counting**: The `count` method is used to tally the occurrences of digits 1, 2, and 3 in the string.
3. **Condition Check**: The solution checks if the counts of digits 1, 2, and 3 are exactly 1, 2, and 3 respectively. If all conditions are met, it prints "Yes"; otherwise, it prints "No".
4. **Efficiency**: The approach efficiently checks the conditions in linear time relative to the length of the string (which is constant at 6 digits), making it optimal for the problem constraints.
