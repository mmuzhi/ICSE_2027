Input

N

Print S_1, S_2, \ldots, S_N in this order, separated by a single space.

- The values of P_i are distinct.
- 1 \leq Q_i \leq N
- The values of Q_i are distinct.
- All input values are integers.

=== ANSWER ===
To solve this problem, we need to determine, for each bib number from 1 to N, the bib number of the person that the wearer of the bib is staring at. The solution involves efficiently mapping bib numbers to person identities and then using these mappings to find the required bib numbers.

### Approach
1. **Problem Analysis**: 
   - We are given N people, each wearing a distinct bib number.
   - Each person is staring at another person, specified by the array P.
   - The array Q provides the bib number each person is wearing.
   - For each bib number `i`, we need to find the bib number of the person that the wearer of bib `i` is staring at.

2. **Key Insight**:
   - To solve this, we first map each bib number to the identity of the person wearing it. This allows us to quickly determine which person wears a specific bib.
   - Once we have this mapping, for each bib `i`, we:
     - Find the identity `A` of the person wearing bib `i`.
     - Use the array `P` to find the identity of the person `A` is staring at.
     - Use the array `Q` to find the bib number of the person being stared at.

3. **Algorithm Selection**:
   - **Mapping Bibs to Identities**: Create an array `bib_to_identity` where each index (representing a bib number) stores the identity of the person wearing that bib.
   - **Processing Each Bib**: For each bib `i` from 1 to N, use the mappings to find the target bib number as described above.

4. **Complexity Analysis**:
   - **Time Complexity**: O(N) for building the `bib_to_identity` array and O(N) for processing each bib. Thus, the overall complexity is O(N), which is efficient for the given constraints (N ≤ 3×10^5).
   - **Space Complexity**: O(N) for storing the `bib_to_identity` array and other auxiliary data structures.

### Solution Code
```python
import sys

def main():
    data = sys.stdin.read().split()
    n = int(data[0])
    P_list = list(map(int, data[1:1+n]))
    Q_list = list(map(int, data[1+n:1+2*n]))
    
    bib_to_identity = [0] * (n+1)
    for idx in range(n):
        bib = Q_list[idx]
        bib_to_identity[bib] = idx + 1
    
    res = []
    for i in range(1, n+1):
        identity_A = bib_to_identity[i]
        target_identity = P_list[identity_A - 1]
        bib_target = Q_list[target_identity - 1]
        res.append(str(bib_target))
    
    print(" ".join(res))

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The input is read all at once for efficiency, especially given the constraints. The first value is `N`, followed by the arrays `P` and `Q`.
2. **Mapping Bibs to Identities**: The loop processes each entry in `Q` to build the `bib_to_identity` array. This array allows O(1) access to the identity of the person wearing any given bib.
3. **Processing Each Bib**: For each bib number `i` from 1 to `N`:
   - The identity `A` of the person wearing bib `i` is retrieved from `bib_to_identity`.
   - The person `A` is staring at the person whose identity is `P_list[A-1]`.
   - The bib number of this target person is `Q_list[target_identity-1]`.
4. **Output**: The results for each bib are collected in a list and printed as a space-separated string.

This approach efficiently leverages direct array accesses and mappings to solve the problem in linear time, making it optimal for large inputs.
