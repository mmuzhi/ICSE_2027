=== ANSWER ===
To solve this problem, we need to replace every character in a given string that is not equal to a specified character `c_1` with another specified character `c_2`. The solution involves reading the input values, processing the string, and generating the output string as per the given requirements.

### Approach
1. **Read Input:** The first line of input contains an integer `N` (length of the string) and two characters `c_1` and `c_2`. The second line contains the string `S` of length `N`.
2. **Process String:** For each character in the string `S`, check if it is not equal to `c_1`. If it is not, replace it with `c_2`; otherwise, keep the character as is.
3. **Output Result:** Print the resulting string after processing all characters.

### Solution Code
```python
def main():
    data = input().split()
    n = int(data[0])
    c1 = data[1]
    c2 = data[2]
    s = input().strip()
    res = []
    for char in s:
        if char == c1:
            res.append(char)
        else:
            res.append(c2)
    print(''.join(res))

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input:** The first line of input is split into three parts: the integer `N` (which is the length of the string), and the characters `c_1` and `c_2`. The second line is the string `S` itself.
2. **Processing Characters:** We iterate over each character in the string `S`. For each character, if it matches `c_1`, it is added to the result list as is. If it does not match, it is replaced with `c_2` in the result list.
3. **Generating Output:** The result list is joined into a single string, which is then printed. This string is the original string with all non-`c_1` characters replaced by `c_2`.

This approach efficiently processes each character in the string exactly once, leading to an optimal solution with a time complexity of O(N), where N is the length of the string. The space complexity is also O(N) to store the result.
