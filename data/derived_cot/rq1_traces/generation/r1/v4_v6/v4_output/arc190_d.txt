To solve this problem, we need to compute the sum of the matrix \( B^p \) over all possible matrices \( B \) obtained by replacing each zero in the given matrix \( A \) with an integer between 1 and \( p-1 \). The result should be returned modulo \( p \), where \( p \) is a prime number. The solution involves leveraging algebraic properties and matrix operations to efficiently compute the result without explicitly enumerating all possible matrices \( B \), especially since the number of such matrices can be very large.

### Approach
1. **Problem Analysis**:
   - Given an \( N \times N \) matrix \( A \) with elements in \( [0, p-1] \), and a prime \( p \).
   - Each zero in \( A \) can be replaced by any integer in \( [1, p-1] \), leading to \( (p-1)^K \) possible matrices \( B \), where \( K \) is the number of zeros.
   - The task is to compute the sum of \( B^p \) for all such \( B \), modulo \( p \).

2. **Key Insights**:
   - For \( p = 2 \), the solution involves direct computation using matrix operations. Specifically, we compute:
     - \( A0^2 \), where \( A0 \) is \( A \) with zeros at free positions.
     - Matrices \( M1 \) (indicating free positions) and \( M2 \) (counting valid paths through free positions).
     - The result is derived as \( A0^2 + A0 \cdot M1 + M1 \cdot A0 + M2 \), modulo 2.
   - For \( p > 2 \):
     - Compute \( A0^p \) using matrix exponentiation modulo \( p \).
     - Construct matrix \( T \) by considering contributions from each free position:
       - For positions \( (i, i) \), add the \( i \)-th column and row of \( A0 \) to \( T \).
       - For \( p = 3 \) and positions \( (i, j) \) (where \( i \neq j \)), add \( A0[j][i] \) to \( T[i][j] \).
     - The final result is \( (-1)^K \cdot (A0^p + T) \) modulo \( p \), where \( (-1)^K \) is \( 1 \) if \( K \) is even, else \( p-1 \).

3. **Algorithm Selection**:
   - **Matrix Multiplication and Exponentiation**: Efficiently compute \( A0^p \) using modular exponentiation.
   - **Handling Free Positions**: For each free position, update \( T \) based on whether the position is on the diagonal or off-diagonal (only for \( p = 3 \)).

4. **Complexity Analysis**:
   - **Matrix Multiplication**: \( O(N^3) \) per multiplication.
   - **Matrix Exponentiation**: \( O(N^3 \log p) \) using exponentiation by squaring.
   - **Constructing \( T \)**: \( O(N^2) \) for free positions, and \( O(N^3) \) for \( M2 \) when \( p = 2 \).
   - Overall complexity is manageable for \( N \leq 100 \) and large \( p \) (up to \( 10^9 \)).

import sys

def mat_mult(A, B, mod=None):
    n = len(A)
    m = len(B)
    l = len(B[0])
    res = [[0] * l for _ in range(n)]
    for i in range(n):
        for k in range(m):
            if A[i][k]:
                for j in range(l):
                    res[i][j] = (res[i][j] + A[i][k] * B[k][j])
    if mod is not None:
        for i in range(n):
            for j in range(l):
                if mod:
                    res[i][j] %= mod
    return res

def matrix_power(matrix, power, mod):
    n = len(matrix)
    result = [[1 if i == j else 0 for j in range(n)] for i in range(n)]
    base = matrix
    while power:
        if power & 1:
            result = mat_mult(result, base, mod)
        base = mat_mult(base, base, mod)
        power //= 2
    return result
