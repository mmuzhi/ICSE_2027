=== ANSWER ===
To solve this problem, we need to determine for each baby born in the Kingdom of AtCoder whether they are given the name "Taro". The name "Taro" is exclusively given to the eldest son (the first male child) in each family. 

### Approach
1. **Problem Analysis**: The problem involves processing a sequence of births in chronological order across multiple families. Each birth record includes the family identifier and the gender of the baby. The key observation is that only the first male child in each family is named "Taro". Subsequent male children in the same family or any female child are not named "Taro".
2. **Intuition**: For each family, we need to track whether a male child has already been born. The first time a male child is born in a family, he is named "Taro". Any subsequent male children in the same family are not named "Taro". Female children are never named "Taro".
3. **Algorithm Selection**: We use a boolean array `has_male` of size `N+1` (where `N` is the number of families) to keep track of whether a male child has been born in each family. The array is initialized to `False`, indicating no male children initially. For each birth record:
   - If the baby is female, output "No".
   - If the baby is male, check the `has_male` array for the family:
     - If `has_male[family]` is `False`, this is the first male child in the family. Output "Yes" and set `has_male[family]` to `True`.
     - If `has_male[family]` is `True`, output "No".

### Solution Code
```python
def main():
    import sys
    data = sys.stdin.read().splitlines()
    if not data:
        return
    
    n, m = map(int, data[0].split())
    babies = []
    for i in range(1, m + 1):
        parts = data[i].split()
        family = int(parts[0])
        gender = parts[1]
        babies.append((family, gender))
    
    has_male = [False] * (n + 1)
    
    output = []
    for family, gender in babies:
        if gender == 'F':
            output.append("No")
        else:
            if not has_male[family]:
                output.append("Yes")
                has_male[family] = True
            else:
                output.append("No")
    
    for res in output:
        print(res)

if __name__ == '__main__':
    main()
```

### Explanation
1. **Reading Input**: The input is read from standard input. The first line contains two integers, `N` (number of families) and `M` (number of babies). The subsequent `M` lines each contain a family identifier and a gender ('M' for male, 'F' for female).
2. **Tracking Male Births**: A boolean array `has_male` is used to track whether a male child has been born in each family. The array is indexed by family identifiers (1 to `N`).
3. **Processing Each Baby**: For each baby:
   - If the baby is female, "No" is immediately added to the output.
   - If the baby is male, the corresponding family's entry in `has_male` is checked. If it is `False`, the baby is the first male in the family, so "Yes" is added to the output, and `has_male[family]` is set to `True`. Otherwise, "No" is added.
4. **Output**: The results for each baby are printed in the order they were processed.

This approach efficiently checks each baby's eligibility for the name "Taro" by maintaining a simple state for each family, ensuring optimal performance with a time complexity of O(M), where M is the number of babies. The space complexity is O(N) for tracking the state of each family.
