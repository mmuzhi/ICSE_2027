You are given a horizontally written text. Convert it to vertical writing, filling spaces with *.

You are given N strings S_1, S_2, \dots, S_N consisting of lowercase English letters. Let M be the maximum length of these strings.
Print M strings T_1, T_2, \dots, T_M that satisfy the following conditions:

- Each T_i consists of lowercase English letters and *.
- Each T_i does not end with *.
- For each 1 \leq i \leq N, the following holds:
- For each 1 \leq j \leq |S_i|, the (N-i+1)-th character of T_j exists, and the concatenation of the (N-i+1)-th characters of T_1, T_2, \dots, T_{|S_i|} in this order equals S_i.
- For each |S_i| + 1 \leq j \leq M, the (N-i+1)-th character of T_j either does not exist or is *.



Here, |S_i| denotes the length of the string S_i.

Input

The input is given from Standard Input in the following format:
N
S_1
S_2
\vdots
S_N

Output

Print the answer in the following format:
T_1
T_2
\vdots
T_M

Constraints


- N is an integer between 1 and 100, inclusive.
- Each S_i is a string of lowercase English letters with length between 1 and 100, inclusive.

To solve this problem, we need to convert a given set of horizontally written strings into vertically written strings, filling any missing positions with asterisks (`*`). The key challenge is to align each character from the original strings in the correct vertical columns while ensuring that each resulting vertical string does not end with an asterisk.

### Approach
1. **Problem Analysis**: The input consists of `N` strings, each of varying lengths. The goal is to create `M` vertical strings, where `M` is the maximum length among the input strings. Each vertical string `T_i` should be constructed such that:
   - The characters from the original strings are placed in specific columns. The first column corresponds to the last input string, the second column to the second last, and so on.
   - For each position in the vertical string, if the original string has a character at that position, it is included; otherwise, an asterisk is placed.
   - After constructing each vertical string, any trailing asterisks are removed to ensure the string does not end with an asterisk.

2. **Intuition**: The solution involves processing each row (from 0 to `M-1`) and each column (from 0 to `N-1`). For each column, the corresponding original string is determined (the last string for column 0, the second last for column 1, etc.). For each row, if the row index is within the length of the original string, the character is added; otherwise, an asterisk is added. After constructing each row, trailing asterisks are removed before output.

3. **Algorithm Selection**: The algorithm iterates over each row (0 to `M-1`) and each column (0 to `N-1`). For each column, it checks if the current row index is within the bounds of the corresponding original string. If so, it adds the character; otherwise, it adds an asterisk. After processing all columns for a row, it removes any trailing asterisks from the row before converting it to a string and outputting it.

4. **Complexity Analysis**: The algorithm processes each of the `M` rows and, for each row, processes `N` columns. The worst-case time complexity is O(N*M), where `N` is the number of strings (up to 100) and `M` is the maximum string length (up to 100). This results in a manageable complexity of O(10,000) operations, which is efficient for the given constraints.

### Solution Code
```python
def main():
    import sys
    data = sys.stdin.read().splitlines()
    N = int(data[0].strip())
    S = []
    for i in range(1, N+1):
        S.append(data[i].strip())
    
    M = max(len(s) for s in S) if N > 0 else 0
    
    for r in range(M):
        row_chars = []
        for c in range(N):
            s_index = N - 1 - c
            if r < len(S[s_index]):
                row_chars.append(S[s_index][r])
            else:
                row_chars.append('*')
        while row_chars and row_chars[-1] == '*':
            row_chars.pop()
        print(''.join(row_chars))

if __name__ == "__main__":
    main()
```

1. **Reading Input**: The input is read from standard input, where the first line is the number of strings `N`, followed by `N` strings.
2. **Finding Maximum Length**: The maximum length `M` among the input strings is determined to know how many vertical strings (`T_1` to `T_M`) need to be generated.
3. **Constructing Vertical Strings**: For each row `r` (from 0 to `M-1`), the algorithm processes each column `c` (from 0 to `N-1`):
   - The corresponding original string for column `c` is `S[N-1-c]`.
   - If the row index `r` is within the bounds of this string, the character `S[N-1-c][r]` is added to the row; otherwise, an asterisk is added.
4. **Trimming Trailing Asterisks**: After constructing each row, any trailing asterisks are removed to ensure the row does not end with an asterisk.
5. **Output**: Each processed row is printed as a string.
