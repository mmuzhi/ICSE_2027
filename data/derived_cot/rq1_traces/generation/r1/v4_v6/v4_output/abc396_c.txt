Input

The input is given from Standard Input in the following format:
N M
B_1 B_2 \ldots B_N
W_1 W_2 \ldots W_M

Output

- 1 \leq N,M \leq 2\times 10^5
- -10^9 \leq B_i, W_j \leq 10^9
- All input values are integers.

To solve this problem, we need to maximize the sum of values of chosen black and white balls such that the number of black balls chosen is at least the number of white balls chosen. The solution involves sorting the balls by their values in descending order, computing prefix sums, and then efficiently determining the optimal combination of black and white balls.

### Approach
1. **Sorting and Prefix Sums**: 
   - Sort the black balls in descending order of their values and compute a prefix sum array where each entry at index `i` represents the sum of the top `i` black balls.
   - Similarly, sort the white balls in descending order and compute a prefix sum array for the top `i` white balls.

2. **Maximizing Black Ball Contribution**:
   - For each possible number of black balls (from 0 to N), compute the maximum prefix sum achievable by taking at least that many black balls. This is done by traversing the prefix sum array from the end to the start, keeping track of the maximum sum encountered so far.

3. **Combining Results**:
   - For each possible number of white balls (from 0 to the minimum of N and M), calculate the total sum by adding the prefix sum of the top `p` white balls to the maximum prefix sum of black balls (considering at least `p` black balls). The maximum value obtained from these combinations is the answer.

### Solution Code
```python
import sys

def main():
    data = sys.stdin.read().split()
    if not data:
        print(0)
        return
    n = int(data[0])
    m = int(data[1])
    blacks = list(map(int, data[2:2+n]))
    whites = list(map(int, data[2+n:2+n+m]))
    
    blacks.sort(reverse=True)
    whites.sort(reverse=True)
    
    prefix_black = [0] * (n + 1)
    for i in range(1, n + 1):
        prefix_black[i] = prefix_black[i - 1] + blacks[i - 1]
    
    prefix_white = [0] * (m + 1)
    for j in range(1, m + 1):
        prefix_white[j] = prefix_white[j - 1] + whites[j - 1]
    
    max_black = [0] * (n + 1)
    max_black[n] = prefix_black[n]
    for i in range(n - 1, -1, -1):
        max_black[i] = max(prefix_black[i], max_black[i + 1])
    
    ans = 0
    max_p = min(n, m)
    for p in range(0, max_p + 1):
        candidate = prefix_white[p] + max_black[p]
        if candidate > ans:
            ans = candidate
    print(ans)

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The input is read from standard input, which includes the number of black and white balls followed by their respective values.
2. **Sorting**: Both black and white balls are sorted in descending order to prioritize higher values.
3. **Prefix Sum Arrays**: Prefix sum arrays for both black and white balls are computed. These arrays store the cumulative sums of the top `i` balls.
4. **Maximizing Black Ball Sums**: An auxiliary array `max_black` is created where each entry at index `i` holds the maximum sum achievable by taking at least `i` black balls. This is computed by traversing the prefix sum array from the end to the start.
5. **Finding Optimal Combination**: For each possible count of white balls (from 0 to the minimum of N and M), the total sum is calculated by adding the sum of the top `p` white balls to the maximum sum of black balls (considering at least `p` black balls). The highest value encountered during this process is the solution.
6. **Output**: The maximum value obtained is printed, which represents the optimal sum under the given constraints. If no balls are chosen, the sum is zero, which is handled naturally by the algorithm.
