=== ANSWER ===
To solve this problem, we need to compute the Hamming distance between two strings of equal length. The Hamming distance is defined as the number of positions at which the corresponding characters in the two strings differ.

### Approach
1. **Problem Analysis**: The problem requires comparing each character in the two strings at the same position and counting how many times they differ. The strings are of the same length, which simplifies the comparison process.
2. **Intuition**: By iterating through each character position in the strings, we can check if the characters at the same position in both strings are different. If they are, we increment a counter.
3. **Algorithm Selection**: The straightforward approach is to loop through each character index from 0 to N-1 (where N is the length of the strings) and compare the corresponding characters in the two strings. This approach is efficient with a time complexity of O(N), which is optimal for this problem given the constraints (N ≤ 100).
4. **Complexity Analysis**: The algorithm runs in linear time O(N) since each character in the strings is checked exactly once. The space complexity is O(1) as no additional space is used apart from a few variables for counting.

### Solution Code
```python
def main():
    N = int(input().strip())
    S = input().strip()
    T = input().strip()
    
    count = 0
    for i in range(N):
        if S[i] != T[i]:
            count += 1
            
    print(count)

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The first line of input is read as an integer `N`, representing the length of the strings. The next two lines are read as strings `S` and `T`.
2. **Initialization**: A counter variable `count` is initialized to zero. This variable will keep track of the number of differing characters between the two strings.
3. **Comparison Loop**: A loop runs from `0` to `N-1`. For each index `i`, the characters `S[i]` and `T[i]` are compared. If they are different, the counter is incremented.
4. **Output Result**: After processing all characters, the value of `count` is printed, which represents the Hamming distance between the two strings.

This approach efficiently computes the Hamming distance by leveraging a simple loop and character comparison, ensuring optimal performance for the given problem constraints.
