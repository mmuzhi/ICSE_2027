To solve this problem, we need to determine the total number of possible ways an event can be conducted for `n` performers, where each performer is assigned to one of `x` stages, and each non-empty stage (band) is awarded a score between 1 and `y`. The solution involves considering all possible assignments of performers to stages and scores to bands, ensuring that the stages are distinguishable and the scores are independent of the stage assignments.

### Approach
1. **Problem Analysis**: The problem requires counting all distinct ways to assign `n` performers to `x` stages (some stages might be empty) and then assign a score from 1 to `y` to each non-empty stage (band). The key insight is that the assignment of performers to stages and the subsequent scoring of bands must consider all possible combinations, including which stages are used and the scores assigned to them.

2. **Key Insight**: The solution can be broken down into two main steps:
   - **Stage Assignment**: For each possible number of non-empty stages `k` (from 1 to `min(n, x)`), calculate the number of ways to assign `n` performers to exactly `k` out of `x` stages such that each of the `k` stages is non-empty. This involves combinatorial mathematics, specifically using inclusion-exclusion principles to count surjective functions (assignments where each of the `k` stages has at least one performer).
   - **Score Assignment**: For each of the `k` non-empty stages, assign a score from 1 to `y`, which contributes a multiplicative factor of `y^k`.

3. **Combinatorial Calculation**: For each `k`:
   - Compute the term using inclusion-exclusion: `sum_{j=0}^{k} (-1)^j * C(k, j) * (k - j)^n`, where `C(k, j)` is the binomial coefficient. This term counts the number of surjective functions from `n` performers to `k` stages.
   - Multiply by the number of ways to choose `k` stages from `x` stages, given by `C(x, k)`.
   - Multiply by `y^k` for the score assignments.
   - Sum all these values for `k` from 1 to `min(n, x)` to get the total number of ways.

4. **Modulo Handling**: Since the result can be very large, all calculations are performed modulo \(10^9 + 7\).

### Solution Code
```python
MOD = 10**9 + 7

class Solution:
    def numberOfWays(self, n: int, x: int, y: int) -> int:
        max_val = 1000
        comb = [[0] * (max_val + 1) for _ in range(max_val + 1)]
        for i in range(max_val + 1):
            comb[i][0] = 1
            for j in range(1, i + 1):
                comb[i][j] = (comb[i - 1][j] + comb[i - 1][j - 1]) % MOD
        
        m = min(n, x)
        total = 0
        for k in range(1, m + 1):
            term = 0
            for j in range(0, k + 1):
                sign = 1 if j % 2 == 0 else -1
                t = comb[k][j] * pow(k - j, n, MOD) % MOD
                term = (term + sign * t) % MOD
            term = (term % MOD + MOD) % MOD
            ways_for_k = comb[x][k] * pow(y, k, MOD) % MOD
            ways_for_k = (ways_for_k * term) % MOD
            total = (total + ways_for_k) % MOD
        return total
```

   - **Inclusion-Exclusion Term**: Calculate the number of ways to assign `n` performers to exactly `k` stages such that no stage is empty. This is done using the inclusion-exclusion principle, summing over subsets of stages that could be empty.
   - **Adjust for Negative Values**: Ensure the term is non-negative by adjusting modulo \(10^9 + 7\).
   - **Combine with Stage Selection and Scoring**: Multiply the term by the number of ways to choose `k` stages from `x` (`C(x, k)`) and the number of ways to assign scores to the `k` bands (`y^k`).
