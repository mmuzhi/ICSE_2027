### Approach
1. **Problem Analysis**: The problem involves minimizing the sum of coordinates of N distinct pieces on a number line after performing operations that swap gaps between consecutive pieces in a specific manner. The operations can be performed any number of times, and the goal is to find the minimal possible sum.
2. **Key Insight**: The operations effectively allow us to rearrange the gaps between consecutive pieces. The minimal sum is achieved by assigning the smallest gaps to the positions with the largest weights. The weights for the gaps are determined by their positions: the weight for the i-th gap is (N - i).
3. **Separation of Gaps**: The gaps are separated into two groups based on their positions (odd-indexed and even-indexed in the 0-based gap array). Each group is sorted in increasing order.
4. **Weight Assignment**: The largest weights in each group are assigned to the smallest gaps in that group. Specifically:
   - For the odd-indexed gaps, the weights are (N-1), (N-3), (N-5), etc.
   - For the even-indexed gaps, the weights are (N-2), (N-4), (N-6), etc.
5. **Sum Calculation**: The total minimal sum is computed as the sum of the initial coordinate multiplied by N plus the weighted sums of the sorted odd and even gaps.

### Solution Code
```python
def main():
    import sys
    data = sys.stdin.read().split()
    n = int(data[0])
    X = list(map(int, data[1:1+n]))
    
    gaps = []
    for i in range(1, n):
        gaps.append(X[i] - X[i-1])
    
    odd_gaps = []
    even_gaps = []
    for i in range(len(gaps)):
        if i % 2 == 0:
            odd_gaps.append(gaps[i])
        else:
            even_gaps.append(gaps[i])
    
    odd_gaps.sort()
    even_gaps.sort()
    
    len_odd = len(odd_gaps)
    weights_odd = [n - 1 - 2 * i for i in range(len_odd)]
    
    len_even = len(even_gaps)
    weights_even = [n - 2 - 2 * i for i in range(len_even)]
    
    sum_odd = 0
    for i in range(len_odd):
        sum_odd += odd_gaps[i] * weights_odd[i]
    
    sum_even = 0
    for i in range(len_even):
        sum_even += even_gaps[i] * weights_even[i]
    
    total_sum = X[0] * n + sum_odd + sum_even
    print(total_sum)

if __name__ == "__main__":
```
