To solve this problem, we need to count the number of valid substrings in `word1` such that, when rearranged, they have `word2` as a prefix. A substring is valid if it contains at least the same frequency of each character as in `word2`. 

### Approach
1. **Problem Analysis**: The problem requires checking each substring of `word1` to determine if it can be rearranged to start with `word2`. The key insight is that a substring is valid if it contains at least the same count of each character present in `word2`. The length of the substring must be at least the length of `word2`.
2. **Sliding Window Technique**: We use a sliding window approach to efficiently check substrings. The window expands to include characters until the counts of all characters in `word2` are met. Once the condition is satisfied, all substrings starting at the current left index and ending at any index from the current right index to the end of the string are valid.
3. **Frequency Tracking**: We maintain frequency counts of characters in the current window and compare them with the required counts from `word2`. The `satisfied_count` variable tracks how many distinct characters in `word2` have their required counts met in the current window.
4. **Efficiency**: The algorithm processes each character in `word1` at most twice (once when expanding the window and once when shrinking it), resulting in an O(n) time complexity, where n is the length of `word1`.

### Solution Code
```python
class Solution:
    def validSubstringCount(self, word1: str, word2: str) -> int:
        n1 = len(word1)
        req_arr = [0] * 26
        for c in word2:
            idx = ord(c) - ord('a')
            req_arr[idx] += 1
        
        total_distinct = sum(1 for x in req_arr if x > 0)
        
        cur_arr = [0] * 26
        l = 0
        r = 0
        ans = 0
        satisfied_count = 0
        
        for l in range(n1):
            while r < n1 and satisfied_count < total_distinct:
                c = word1[r]
                idx = ord(c) - ord('a')
                if req_arr[idx] > 0:
                    cur_arr[idx] += 1
                    if cur_arr[idx] == req_arr[idx]:
                        satisfied_count += 1
                r += 1
            
            if satisfied_count == total_distinct:
                ans += (n1 - r + 1)
            
            c = word1[l]
            idx = ord(c) - ord('a')
            if req_arr[idx] > 0:
                if cur_arr[idx] == req_arr[idx]:
                    satisfied_count -= 1
                cur_arr[idx] -= 1
        
        return ans
```

### Explanation
1. **Initialization**: We initialize `req_arr` to store the frequency counts of each character in `word2`. The variable `total_distinct` counts how many distinct characters in `word2` have non-zero counts.
2. **Sliding Window Setup**: We use `cur_arr` to track the frequency counts of characters in the current window defined by pointers `l` (left) and `r` (right). The `satisfied_count` variable keeps track of how many characters in `word2` have their required counts met in the current window.
3. **Window Expansion**: For each left index `l`, we expand the right pointer `r` until the current window meets the character count requirements of `word2`. If the requirements are met, all substrings starting at `l` and ending at any index from `r-1` to the end of `word1` are counted as valid.
4. **Window Shrinking**: After processing the current left index, we move `l` forward by one, adjusting the frequency counts and `satisfied_count` accordingly if the removed character was part of `word2`.
5. **Result Calculation**: The total count of valid substrings is accumulated in `ans` and returned after processing all possible left indices.

This approach efficiently checks all valid substrings by leveraging the sliding window technique and frequency tracking, ensuring optimal performance even for large input sizes.
