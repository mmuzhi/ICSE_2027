N

To solve this problem, we need to determine the maximum height at coordinate 0 from which it is not possible to see all buildings. The solution involves checking visibility conditions for each building and efficiently computing critical heights using geometric properties and convex hull techniques.

### Approach
1. **Initial Check at Height 0**: 
   - For each building, compute the maximum value of `(X_i / X_j) * H_j` for all buildings `j` to the left of `i`. This value, denoted as `M_i`, must be strictly less than the height `H_i` of building `i` to be visible from height 0 at coordinate 0. If all buildings satisfy this condition, we immediately return -1.

2. **Convex Hull Construction**:
   - For buildings not visible at height 0, we proceed to compute the maximum height `h` where at least one building remains invisible. This involves maintaining a convex hull of points `(1/X_j, H_j/X_j)` for buildings processed so far. The convex hull helps in efficiently finding the building `j` that maximizes the critical height `h` for each building `i`.

3. **Ternary Search on Convex Hull**:
   - For each building `i`, we perform a ternary search on the convex hull to find the point `j` that maximizes the slope `(v_j - v_i)/(u_j - u_i)`, which corresponds to the critical height `h` where building `i` becomes visible. The maximum slope found is used to update the answer.

4. **Precision Handling**:
   - To ensure precision, after narrowing down candidate points using ternary search, we compute the exact critical height using integer arithmetic for the final small segment of points. This avoids floating-point precision issues during the initial slope calculations.
