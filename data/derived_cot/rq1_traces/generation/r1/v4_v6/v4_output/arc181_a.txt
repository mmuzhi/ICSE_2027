To solve this problem, we need to determine the minimum number of operations required to transform a given permutation into the identity permutation (where each element at position \(i\) is \(i\)) using a specific operation. The operation involves selecting an integer \(k\), sorting the elements before \(k\) (if \(k \geq 2\)) and sorting the elements after \(k\) (if \(k \leq N-1\)).

### Approach
1. **Check for Sorted Permutation**: If the permutation is already the identity permutation, the answer is 0.
2. **Precompute Prefix and Suffix Arrays**: For efficient checks, we precompute:
   - `prefix_max` and `prefix_min`: Arrays storing the maximum and minimum values up to each index from the start.
   - `suffix_max` and `suffix_min`: Arrays storing the maximum and minimum values from each index to the end.
3. **Check for One Operation Solution**: We check if there exists a \(k\) such that:
   - For \(k = 1\): The first element is 1, and the rest of the elements form a consecutive sequence from 2 to \(N\).
   - For \(k = N\): The last element is \(N\), and the first \(N-1\) elements form a consecutive sequence from 1 to \(N-1\).
   - For \(2 \leq k \leq N-1\): The element at \(k\) is \(k\), the maximum of the elements before \(k\) is \(k-1\), and the minimum of the elements after \(k\) is \(k+1\).
4. **Output Result**: If any \(k\) satisfies the conditions for a one-operation solution, output 1. Otherwise, output 2, as it can be proven that at most two operations suffice for any permutation under the given constraints.

import sys

def main():
    data = sys.stdin.read().split()
    t = int(data[0])
    index = 1
    results = []
    for _ in range(t):
        n = int(data[index])
        index += 1
        P = list(map(int, data[index:index+n]))
        index += n
        
        sorted_flag = True
        for i in range(n):
            if P[i] != i+1:
                sorted_flag = False
                break
        if sorted_flag:
            results.append("0")
            continue
            
        prefix_max = [0] * n
        prefix_min = [0] * n
        suffix_max = [0] * n
        suffix_min = [0] * n
        
        prefix_max[0] = P[0]
        prefix_min[0] = P[0]
        for i in range(1, n):
            prefix_max[i] = max(prefix_max[i-1], P[i])
            prefix_min[i] = min(prefix_min[i-1], P[i])
            
        suffix_max[n-1] = P[n-1]
        suffix_min[n-1] = P[n-1]
        for i in range(n-2, -1, -1):
            suffix_max[i] = max(suffix_max[i+1], P[i])
            suffix_min[i] = min(suffix_min[i+1], P[i])
            
        found = False
        if P[0] == 1:

                found = True

                    found = True

                if prefix_min[n-2] == 1 and prefix_max[n-2] == n-1:
