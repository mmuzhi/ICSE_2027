To solve this problem, we need to compute the sum of the maximum and minimum elements of all subsequences of an array `nums` that have at most `k` elements. Given the constraints, a brute-force approach is infeasible, so we use combinatorial mathematics and sorting to efficiently compute the result.

### Approach
1. **Sorting the Array**: First, we sort the array to facilitate the computation of min and max values in subsequences.
2. **Precompute Factorials and Inverses**: We precompute factorials and their modular inverses up to the length of the array to efficiently compute binomial coefficients modulo \(10^9 + 7\).
3. **Precompute Cumulative Binomial Sums**: For each possible subset size up to `k`, we compute the sum of binomial coefficients \(C(m, j)\) for \(j\) from 0 to \(\min(m, k)\). This helps in quickly determining the number of valid subsequences for any given set size.
4. **Iterate Over Distinct Values**: For each distinct value in the sorted array, we determine the number of subsequences where the current value is the minimum or the maximum:
   - **Minimum Calculation**: For a value `v` to be the minimum, the subsequence must include at least one occurrence of `v` and no elements smaller than `v`. The count of such subsequences is derived from the cumulative binomial sums of elements greater than or equal to `v` minus those strictly greater than `v`.
   - **Maximum Calculation**: Similarly, for `v` to be the maximum, the subsequence must include at least one occurrence of `v` and no elements larger than `v`. The count is derived from the cumulative binomial sums of elements less than or equal to `v` minus those strictly less than `v`.
5. **Summing Contributions**: For each distinct value, we sum its contributions to the total min and max sums, adjusting for negative values using modulo arithmetic.
6. **Result Calculation**: The final result is the sum of the total min and max sums modulo \(10^9 + 7\).

### Solution Code
```python
import bisect

MOD = 10**9 + 7

class Solution:
    def minMaxSums(self, nums: list, k: int) -> int:
        n = len(nums)
        if n == 0:
            return 0
        
        fact = [1] * (n + 1)
        inv_fact = [1] * (n + 1)
        
        for i in range(1, n + 1):
            fact[i] = fact[i - 1] * i % MOD
        
        inv_fact[n] = pow(fact[n], MOD - 2, MOD)
        for i in range(n, 0, -1):
            inv_fact[i - 1] = inv_fact[i] * i % MOD
        
        F_arr = [0] * (n + 1)
        for m in range(0, n + 1):
            total = 0
            j_end = min(m, k)
            for j in range(0, j_end + 1):
                comb = fact[m] * inv_fact[j] % MOD * inv_fact[m - j] % MOD
                total = (total + comb) % MOD
            F_arr[m] = total
        
        A = sorted(nums)
        distinct_v = sorted(set(A))
        min_sum = 0
        max_sum = 0
        
        for v in distinct_v:
            left_index = bisect.bisect_left(A, v)
            right_index = bisect.bisect_right(A, v)
            
            size_S = n - left_index
            size_T = n - right_index
            count_min = F_arr[size_S] - F_arr[size_T]
            count_min %= MOD
            if count_min < 0:
                count_min += MOD
            min_sum = (min_sum + v * count_min) % MOD
            
            size_S_prime = right_index
            size_T_prime = left_index
            count_max = F_arr[size_S_prime] - F_arr[size_T_prime]
            count_max %= MOD
            if count_max < 0:
                count_max += MOD
            max_sum = (max_sum + v * count_max) % MOD
        
        return (min_sum + max_sum) % MOD
```
