=== ANSWER ===
To solve this problem, we need to extract the digits of a given three-digit integer and rearrange them to form two new integers as specified. The hundreds, tens, and ones digits of the input number are rearranged to form one integer with the digits in the order tens, ones, hundreds, and another integer with the digits in the order ones, hundreds, tens.

### Approach
1. **Extract Digits**: The input three-digit integer is split into its hundreds, tens, and ones digits. This can be done using integer division and modulus operations.
2. **Form New Integers**: Using the extracted digits, two new integers are formed:
   - The first integer is formed by placing the tens digit in the hundreds place, the ones digit in the tens place, and the hundreds digit in the ones place.
   - The second integer is formed by placing the ones digit in the hundreds place, the hundreds digit in the tens place, and the tens digit in the ones place.
3. **Output the Result**: The two newly formed integers are printed separated by a space.

### Solution Code
```python
def main():
    n = int(input().strip())
    a = n // 100
    b = (n % 100) // 10
    c = n % 10
    num1 = b * 100 + c * 10 + a
    num2 = c * 100 + a * 10 + b
    print(f"{num1} {num2}")

if __name__ == '__main__':
    main()
```

### Explanation
1. **Reading Input**: The input is read as an integer. For example, if the input is `379`, it is stored as the integer `379`.
2. **Digit Extraction**: The hundreds digit (`a`) is obtained by integer division of the number by 100 (e.g., `379 // 100 = 3`). The tens digit (`b`) is obtained by taking the remainder when divided by 100 (e.g., `379 % 100 = 79`) and then integer dividing by 10 (e.g., `79 // 10 = 7`). The ones digit (`c`) is the remainder when divided by 10 (e.g., `379 % 10 = 9`).
3. **Forming New Integers**: 
   - The first integer (`num1`) is formed as `b * 100 + c * 10 + a`. For `379`, this is `7 * 100 + 9 * 10 + 3 = 700 + 90 + 3 = 793`.
   - The second integer (`num2`) is formed as `c * 100 + a * 10 + b`. For `379`, this is `9 * 100 + 3 * 10 + 7 = 900 + 30 + 7 = 937`.
4. **Output**: The two integers are printed separated by a space (e.g., `793 937` for the input `379`).

This approach efficiently extracts the digits and rearranges them to form the required integers, ensuring the solution is both clear and optimal.
