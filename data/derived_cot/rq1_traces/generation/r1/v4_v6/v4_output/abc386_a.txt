=== ANSWER ===
To solve this problem, we need to determine whether adding one card to a set of four cards can form a Full House. A Full House is defined as a set of five cards where three cards have the same integer and the other two cards have another distinct integer. 

### Approach
1. **Problem Analysis**: The input consists of four integers representing the numbers on the cards. We need to check if adding a fifth card (any integer between 1 and 13) can result in a Full House. 
2. **Key Insight**: The existing four cards must meet specific conditions to form a Full House when a fifth card is added:
   - **Case 1**: The four cards have exactly two distinct integers with frequencies (3, 1). Adding the card that appears once will result in frequencies (3, 2), forming a Full House.
   - **Case 2**: The four cards have exactly two distinct integers with frequencies (2, 2). Adding either of the two integers will result in frequencies (3, 2), forming a Full House.
   - **Other Cases**: If the four cards have one distinct integer (all four cards are the same), three distinct integers, or four distinct integers, it is impossible to form a Full House by adding any fifth card.
3. **Algorithm**:
   - Count the frequency of each distinct integer in the four cards.
   - Determine the number of distinct integers:
     - If there is only one distinct integer, output "No".
     - If there are two distinct integers, check their frequencies. If the sorted frequencies are [1, 3] or [2, 2], output "Yes"; otherwise, "No".
     - If there are three or four distinct integers, output "No".

### Solution Code
```python
def main():
    data = list(map(int, input().split()))
    count = {}
    for card in data:
        count[card] = count.get(card, 0) + 1
    distinct_count = len(count)
    if distinct_count == 1:
        print("No")
    elif distinct_count == 2:
        counts_sorted = sorted(count.values())
        if counts_sorted == [1, 3] or counts_sorted == [2, 2]:
            print("Yes")
        else:
            print("No")
    else:
        print("No")

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The input is read as four integers.
2. **Frequency Counting**: A dictionary is used to count occurrences of each distinct integer in the input.
3. **Distinct Count Check**: The number of distinct integers is determined:
   - **One Distinct Integer**: All four cards are the same. Adding any fifth card cannot form a Full House, so output "No".
   - **Two Distinct Integers**: The frequencies of the integers are sorted. If they are [1, 3] or [2, 2], adding a fifth card can form a Full House (output "Yes"); otherwise, output "No".
   - **Three or Four Distinct Integers**: It's impossible to form a Full House by adding one card, so output "No".

This approach efficiently checks the necessary conditions to determine if a Full House can be formed by adding one card to the given four cards.
