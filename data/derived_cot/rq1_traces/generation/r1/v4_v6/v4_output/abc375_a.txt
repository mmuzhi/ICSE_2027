=== ANSWER ===
To solve this problem, we need to count the number of triplets of consecutive seats in a row where the first and third seats are occupied ('#') and the middle seat is unoccupied ('.'). The triplets must be within the valid range of seats from 1 to N-2 (1-indexed), which translates to checking indices from 0 to N-3 in the 0-indexed string representation.

### Approach
1. **Problem Analysis**: The problem requires checking every set of three consecutive seats (i, i+1, i+2) for a specific pattern where the first and third seats are occupied, and the middle seat is unoccupied. The challenge is to efficiently scan through the string representing the seats and count all such valid triplets.
2. **Key Insight**: For each starting index `i` (0-indexed) from 0 to N-3, check if the characters at positions `i`, `i+1`, and `i+2` form the pattern '#', '.', '#'.
3. **Algorithm Selection**: A simple linear scan through the string is sufficient. For each valid starting index, check the three consecutive characters. If they match the required pattern, increment the count.
4. **Complexity Analysis**: The algorithm runs in O(N) time, where N is the length of the string, because it checks each triplet in constant time and scans the string once. The space complexity is O(1) as no additional data structures are used beyond the input string.

### Solution Code
```python
def main():
    n = int(input().strip())
    s = input().strip()
    if n < 3:
        print(0)
        return
    count = 0
    for i in range(n - 2):
        if s[i] == '#' and s[i+1] == '.' and s[i+2] == '#':
            count += 1
    print(count)

if __name__ == '__main__':
    main()
```

### Explanation
1. **Reading Input**: The input consists of an integer `n` (number of seats) and a string `s` representing the occupancy status of each seat.
2. **Edge Case Handling**: If the number of seats `n` is less than 3, it's impossible to have any valid triplets, so the answer is immediately 0.
3. **Linear Scan**: For each starting index `i` from 0 to `n-3`, the code checks if the characters at positions `i`, `i+1`, and `i+2` form the pattern '#', '.', '#'. If they do, the count is incremented.
4. **Output Result**: After processing all possible triplets, the total count of valid patterns is printed.

This approach efficiently checks all possible triplets in linear time, ensuring optimal performance even for large input sizes as specified in the constraints.
