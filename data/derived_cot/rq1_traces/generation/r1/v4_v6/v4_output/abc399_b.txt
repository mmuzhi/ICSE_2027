=== ANSWER ===
To solve this problem, we need to assign ranks to N participants based on their scores according to a specific procedure. The procedure involves iteratively determining the rank for the highest remaining score, assigning the current rank value to all participants with that score, and then updating the rank value by adding the number of participants just ranked. This process repeats until all participants have been assigned a rank.

### Approach
1. **Problem Analysis**: The problem requires us to assign ranks to participants such that participants with the highest remaining scores get the current rank, and ties (participants with the same score) are handled by assigning them the same rank. The next rank value is then incremented by the number of participants just ranked.
2. **Intuition**: By sorting the participants' scores in descending order, we can efficiently process each group of participants with the same score. For each distinct score encountered in the sorted list, we assign the current rank to all participants with that score, then update the rank by adding the count of these participants.
3. **Algorithm Selection**: The algorithm involves:
   - Reading and storing each participant's score along with their original index.
   - Sorting these scores in descending order.
   - Traversing the sorted list to group participants by their scores.
   - Assigning the current rank to each group and updating the rank value by the size of the group.
4. **Complexity Analysis**: Sorting the list of scores takes O(N log N) time, and traversing the sorted list to assign ranks takes O(N) time. Thus, the overall complexity is O(N log N), which is efficient given the constraints (N ≤ 100).

### Solution Code
```python
def main():
    import sys
    data = sys.stdin.read().split()
    n = int(data[0])
    scores = list(map(int, data[1:1+n]))
    
    indexed_scores = [(scores[i], i) for i in range(n)]
    indexed_scores.sort(key=lambda x: x[0], reverse=True)
    
    res = [0] * n
    current_rank = 1
    i = 0
    while i < n:
        current_score = indexed_scores[i][0]
        count = 0
        j = i
        while j < n and indexed_scores[j][0] == current_score:
            count += 1
            j += 1
        for idx in range(i, j):
            original_index = indexed_scores[idx][1]
            res[original_index] = current_rank
        current_rank += count
        i = j
    
    for rank in res:
        print(rank)

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The input is read from standard input, where the first value is the number of participants (N), followed by their scores.
2. **Storing Scores with Indices**: Each score is paired with its original index to keep track of participants after sorting.
3. **Sorting**: The list of scores is sorted in descending order to process higher scores first.
4. **Assigning Ranks**: The sorted list is traversed to group participants with the same score. For each group, the current rank is assigned to all participants in the group, and the rank is incremented by the group size.
5. **Output**: The ranks are printed in the original order of participants, ensuring each participant's rank is displayed correctly.

This approach efficiently handles the ranking process by leveraging sorting and grouping, ensuring optimal performance and correctness.
