- Operation 1: Choose two identical characters in the string and delete them (cannot be performed if there are not two or more identical characters).
- Operation 2: Choose one A, one B, and one C in the string and delete them (cannot be performed if there are not one or more of each of A, B, and C).

For example, ABACA is a good string because it can be turned into an empty string by performing the operations as follows:

You are given a string S of length N consisting of A, B, C, and ?. How many ways are there to replace each ? with A, B, or C to form a string that contains at least K good strings as contiguous substrings? Substrings are counted separately if they are at different positions in the original string, even if they are identical strings.

- 0 \leq K \leq \frac{N(N+1)}{2}
- N and K are integers.
- |S| = N
- S is a string consisting of A, B, C, and ?.

To solve this problem, we need to count the number of ways to replace each '?' in a given string S with 'A', 'B', or 'C' such that the resulting string contains at least K good contiguous substrings. A substring is considered good if it can be reduced to an empty string by repeatedly removing two identical characters or one 'A', one 'B', and one 'C' in any order. 

### Approach
1. **Problem Analysis**: The key insight is that a substring is good if the counts of 'A', 'B', and 'C' modulo 2 are either all even or all odd. This can be represented using a bitmask where each bit corresponds to the parity of the count of 'A', 'B', and 'C'. The valid states are 0 (000) and 7 (111) in binary.
2. **Dynamic Programming (DP) Setup**: We use a DP approach where the state consists of the current prefix state (a bitmask representing the parities of 'A', 'B', and 'C' counts up to the current position) and a frequency vector (counts of each state encountered so far in the prefix array).
3. **State Transitions**: For each character in the string (including handling '?' by considering all three possibilities), we update the current state by XOR-ing it with the bitmask of the character. The frequency vector is updated by incrementing the count of the new state.
4. **Final Count Calculation**: After processing all characters, we aggregate the frequency vectors. For each frequency vector, we compute the total number of good substrings. This is done by counting pairs of prefixes with the same state or states that XOR to 7 (i.e., complementary states). If the total meets or exceeds K, we include the count of such vectors in the result.

### Solution Code
```python
mod = 998244353

import sys

def main():
    data = sys.stdin.read().splitlines()
    if not data:
        return
    nk = data[0].split()
    N = int(nk[0])
    K = int(nk[1])
    S = data[1].strip()
    
    dp = {}
    initial_fv = (1, 0, 0, 0, 0, 0, 0, 0)
    dp[(0, initial_fv)] = 1

    for i in range(N):
        next_dp = {}
        for (state, fv), cnt in dp.items():
            if S[i] == '?':
                chars = ['A', 'B', 'C']
            else:
                chars = [S[i]]
            for char in chars:
                if char == 'A':
                    v = 1
                elif char == 'B':
                    v = 2
                elif char == 'C':
                    v = 4
                new_state = state ^ v
                lst = list(fv)
                lst[new_state] += 1
                new_fv = tuple(lst)
                key = (new_state, new_fv)
                next_dp[key] = (next_dp.get(key, 0) + cnt) % mod
        dp = next_dp

    fv_count = {}
    for (state, fv), cnt in dp.items():
        if fv in fv_count:
            fv_count[fv] = (fv_count[fv] + cnt) % mod
        else:
            fv_count[fv] = cnt

    ans = 0
    for fv, cnt in fv_count.items():
        total = 0
        for i in range(8):
            c = fv[i]
            total += c * (c - 1) // 2
        total += fv[0] * fv[7]
        total += fv[1] * fv[6]
        total += fv[2] * fv[5]
        total += fv[3] * fv[4]
        if total >= K:
            ans = (ans + cnt) % mod

    print(ans)

if __name__ == "__main__":
    main()
```

### Explanation
1. **Initialization**: The DP starts with the initial state (bitmask 0) and a frequency vector indicating that the initial prefix (empty string) has state 0.
2. **Processing Each Character**: For each character in the string, if it's a '?', we consider all three possible characters. For each character, we update the current state by XOR-ing it with the character's bitmask. The frequency vector is updated to reflect the new state.
3. **State Tracking**: The DP tracks the number of ways to reach each (state, frequency vector) pair at each step.
4. **Result Calculation**: After processing all characters, we aggregate the results by frequency vector. For each vector, we compute the total good substrings by counting valid pairs of prefixes. If the total meets or exceeds K, the count for that vector is added to the result.
5. **Output**: The final result is printed modulo 998244353.

This approach efficiently counts the valid string configurations by leveraging dynamic programming and bitmask techniques to manage state transitions and frequency counts.
