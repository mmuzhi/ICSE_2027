=== ANSWER ===
To solve this problem, we need to classify Takahashi's body temperature into one of three categories based on predefined thresholds. The solution involves reading the input temperature, converting it to a numerical value, and then checking against the given conditions to determine the appropriate classification.

### Approach
1. **Read Input**: The input is a string representing the body temperature to one decimal place. We convert this string to a float for numerical comparison.
2. **Classify Temperature**: Using conditional checks, we determine which category the temperature falls into:
   - **High fever**: If the temperature is 38.0°C or higher, output `1`.
   - **Fever**: If the temperature is between 37.5°C (inclusive) and 38.0°C (exclusive), output `2`.
   - **Normal**: If the temperature is below 37.5°C, output `3`.
3. **Output Result**: Print the corresponding integer based on the classification.

### Solution Code
```python
def main():
    X = float(input().strip())
    if X >= 38.0:
        print(1)
    elif X >= 37.5:
        print(2)
    else:
        print(3)

if __name__ == '__main__':
    main()
```

### Explanation
1. **Input Handling**: The input is read as a string and immediately converted to a floating-point number to facilitate numerical comparisons.
2. **Condition Checks**: The code checks the temperature against the thresholds in descending order:
   - First, it checks if the temperature is 38.0°C or higher. If true, it prints `1`.
   - If not, it checks if the temperature is 37.5°C or higher (but less than 38.0°C). If true, it prints `2`.
   - If neither condition is met, the temperature must be below 37.5°C, so it prints `3`.
3. **Efficiency**: The approach efficiently categorizes the temperature with straightforward conditional checks, ensuring optimal performance even for edge cases within the given constraints (30 ≤ X ≤ 50). The solution handles all valid inputs as specified.
