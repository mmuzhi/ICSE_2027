You will be given a competitive programming problem.
You are given integers N and M.
Print all integer sequences (A_1, A_2, \ldots, A_N) of length N that satisfy all of the following conditions, in lexicographical order.

- 1 \leq A_i
- A_{i - 1} + 10 \leq A_i for each integer i from 2 through N
- A_N \leq M

What is lexicographical order?
A sequence S = (S_1, S_2, \ldots, S_N) of length N is smaller in lexicographical order than a sequence T = (T_1, T_2, \ldots, T_N) of length N if and only if there exists an integer 1 \leq i \leq N such that both of the following hold:

-  (S_1, S_2, \ldots, S_{i-1}) = (T_1, T_2, \ldots, T_{i-1})
-  S_i is less than T_i (as a number).

Input

The input is given from Standard Input in the following format:
N M

Output

Let X be the number of integer sequences that satisfy the conditions, and print X + 1 lines.
The first line should contain the value of X.
The (i + 1)-th line (1 \leq i \leq X) should contain the i-th smallest integer sequence in lexicographical order, with elements separated by spaces.

Constraints


- 2 \leq N \leq 12
- 10N - 9 \leq M \leq 10N
- All input values are integers.

### Approach

2. **Key Insight**: The sequence can be determined by the first element \( x_1 \) and a series of non-negative integers \( d_1, d_2, \ldots, d_{N-1} \) such that each \( d_i \) represents the excess over the minimal increment of 10. 3. **Algorithm Selection**: We use a recursive approach to generate all valid sequences:
   - Iterate over possible values of \( x_1 \) (from 1 to \( M - 10(N-1) \)).
   - For each \( x_1 \), compute the remaining sum \( R = M - (x_1 + 10(N-1)) \), which must be non-negative.
   - Generate all possible gap vectors \( (d_1, d_2, \ldots, d_{N-1}) \) such that their sum is \( \leq R \). This is done using depth-first search (DFS).
   - For each gap vector, construct the sequence starting with \( x_1 \) and each subsequent element as \( A_i = A_{i-1} + 10 + d_{i-1} \).
