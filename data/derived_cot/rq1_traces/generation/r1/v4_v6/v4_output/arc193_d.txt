   - **Binary Search for Minimum Operations**: For feasible cases, we use binary search to determine the minimum number of operations `D`. For each candidate `D`, we check if every piece in `A` can be moved to some position in `B` within `D` steps (ensuring coverage of all required positions in `B`).

        hi = n
        
        def feasible(D):
            j = 0

            mid = (lo + hi) // 2

            else:
