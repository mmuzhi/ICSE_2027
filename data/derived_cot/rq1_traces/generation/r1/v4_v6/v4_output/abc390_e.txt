=== PROMPT ===
You will be given a competitive programming problem.
There are N foods, each containing exactly one of vitamins 1, 2, and 3.
Specifically, eating the i-th food gives you A_i units of vitamin V_i, and C_i calories.
Takahashi can choose any subset of these N foods as long as the total calorie consumption does not exceed X.
Find the maximum possible value of this: the minimum intake among vitamins 1, 2, and 3.

Input

The input is given from Standard Input in the following format:
N X
V_1 A_1 C_1
V_2 A_2 C_2
\vdots
V_N A_N C_N

Output

Print the maximum possible value of "the minimum intake among vitamins 1, 2, and 3" when the total calories consumed is at most X.

Constraints


- 1 \leq N \leq 5000
- 1 \leq X \leq 5000
- 1 \leq V_i \leq 3
- 1 \leq A_i \leq 2 \times 10^5
- 1 \leq C_i \leq X
- All input values are integers.

Sample Input 1

5 25
1 8 5
2 3 5
2 7 10
3 2 5
3 3 10

Sample Output 1

3

Each food provides the following if eaten:

- 1st food: 8 units of vitamin 1, and 5 calories
- 2nd food: 3 units of vitamin 2, and 5 calories
- 3rd food: 7 units of vitamin 2, and 10 calories
- 4th food: 2 units of vitamin 3, and 5 calories
- 5th food: 3 units of vitamin 3, and 10 calories

Eating the 1st, 2nd, 4th, and 5th foods gives 8 units of vitamin 1, 3 units of vitamin 2, 5 units of vitamin 3, and 25 calories.
In this case, the minimum among the three vitamin intakes is 3 (vitamin 2).
It is impossible to get 4 or more units of each vitamin without exceeding 25 calories, so the answer is 3.

Sample Input 2

2 5000
1 200000 1
2 200000 1

Sample Output 2

0

To solve this problem, we need to maximize the minimum intake among three vitamins (1, 2, and 3) by selecting a subset of foods such that the total calorie consumption does not exceed a given limit X. Each food provides exactly one type of vitamin, along with a certain amount of that vitamin and associated calories.

### Approach
1. **Problem Analysis**: The problem involves selecting foods such that the total calories do not exceed X, and the minimum of the total vitamins (1, 2, and 3) is maximized. Each food contributes to only one vitamin type, so we can categorize foods by their vitamin type.
2. **Key Insight**: For each vitamin type, we need to determine the minimum calories required to achieve at least a certain amount of that vitamin. This can be modeled using a dynamic programming approach similar to the 0-1 knapsack problem.
3. **Dynamic Programming Setup**: For each vitamin type, we compute an array `max_vit` where `max_vit[cal]` represents the maximum vitamin amount achievable with calorie consumption `cal`. This is done using a knapsack algorithm for each type.
4. **Binary Search for Optimal Minimum Vitamin**: We perform a binary search over possible minimum vitamin values (from 0 to the maximum achievable minimum vitamin among the three types). For each candidate value `m`, we check if it's feasible to achieve at least `m` vitamins of each type with total calories ≤ X. The feasibility is determined by summing the minimum calories required for each vitamin type to achieve `m` and comparing it to X.
5. **Complexity Analysis**: The knapsack computation for each type is O(N*X), where N is the number of foods and X is the calorie limit. The binary search runs in O(log(min_vit)), where min_vit is the maximum achievable minimum vitamin. The overall complexity is O(N*X + log(min_vit)), which is feasible given the constraints (N, X ≤ 5000).

import sys

def main():
    data = sys.stdin.read().split()
    it = iter(data)
    N = int(next(it))

    for _ in range(N):

        C = int(next(it))

            type3.append((A, C))
    
    def compute_type(type_foods):
        dp = [0] * (X + 1)
        for A, C in type_foods:
            for j in range(X, C - 1, -1):
                if dp[j - C] + A > dp[j]:
                    dp[j] = dp[j - C] + A
        max_vit = [0] * (X + 1)
        cur_max = 0
        for j in range(X + 1):
            if dp[j] > cur_max:
                cur_max = dp[j]
            max_vit[j] = cur_max
        return max_vit
    
    max_vit1 = compute_type(type1)
    max_vit2 = compute_type(type2)
    max_vit3 = compute_type(type3)

    total2 = max_vit2[X]
    total3 = max_vit3[X]
    high_m = min(total1, total2, total3)
    
    def get_min_cal(m, max_vit_arr):
        lo, hi = 0, X
        res = X + 1
        while lo <= hi:

            if max_vit_arr[mid_cal] >= m:
                res = mid_cal

        return res
    
    low_m = 0

2. **Knapsack for Each Vitamin Type**: For each vitamin type, we compute the maximum vitamin amount achievable for every possible calorie value (0 to X) using dynamic programming. This involves iterating over each food item and updating the DP array in reverse to avoid reprocessing items.
3. **Cumulative Maximum Array**: After processing all items for a vitamin type, we create a cumulative maximum array where each entry at index `cal` stores the maximum vitamin achievable with calorie consumption up to `cal`.
4. **Binary Search for Optimal Vitamin Minimum**: We perform a binary search over possible minimum vitamin values.
