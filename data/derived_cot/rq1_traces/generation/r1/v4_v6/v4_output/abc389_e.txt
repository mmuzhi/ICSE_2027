To solve this problem, we need to maximize the total number of units of products purchased such that the total cost does not exceed a given budget M. The cost to buy k units of the i-th product is k² multiplied by P_i. 

### Approach
1. **Problem Analysis**: The problem requires maximizing the total units purchased (sum of k_i) across all products, given that the total cost (sum of k_i² * P_i) is at most M. The challenge lies in efficiently determining the optimal distribution of units across products to maximize the total units without exceeding the budget.

2. **Key Insight**: The minimal cost for purchasing T units can be derived by selecting the T smallest marginal costs. The marginal cost for the j-th unit of product i is (2j - 1) * P_i. The minimal total cost for T units is the sum of the T smallest marginal costs across all products.

3. **Binary Search on Total Units (T)**: We perform a binary search over the possible values of T (from 0 to M // min(P_i)) to find the maximum T such that the minimal cost F(T) is ≤ M. 

4. **Computing F(T)**:
   - **Binary Search on λ**: For a given T, we find the smallest λ such that the count of marginal costs ≤ λ is at least T. This λ is the T-th smallest marginal cost.
   - **Sum Calculation**: Using λ, we compute the sum of all marginal costs strictly less than λ (sum_less) and the count of such costs (count_less). The total cost F(T) is then sum_less plus (T - count_less) * λ.

5. **Complexity**: The binary search over T runs in O(log(M)) iterations. For each T, the binary search over λ runs in O(log(2*T*max(P))) iterations, each involving a pass over all products (O(N)). Thus, the overall complexity is O(N log(M) log(T*max(P))), which is feasible given the constraints.

import sys

def main():
    data = sys.stdin.read().split()
    n = int(data[0])
    M_val = int(data[1])
    P_list = list(map(int, data[2:2+n]))
    
    if n == 0:
        print(0)
        return
        
    minP = min(P_list)
    maxP = max(P_list)
    
    T_max = M_val // minP
    
    def F(T):
        if T == 0:
            return 0
        low_lam = 0
        high_lam = (2 * T) * maxP
        while low_lam < high_lam:
            mid_lam = (low_lam + high_lam) // 2
            cnt = 0
            for p in P_list:
                cnt += (mid_lam + p) // (2 * p)
            if cnt >= T:
                high_lam = mid_lam
            else:
                low_lam = mid_lam + 1
        lam_star = low_lam
        count_less = 0
        sum_less = 0
        for p in P_list:
            n_i_less = (lam_star - 1 + p) // (2 * p)
            count_less += n_i_less
            sum_less += p * (n_i_less * n_i_less)
        remaining = T - count_less
        return sum_less + remaining * lam_star

    lowT = 0
    highT = T_max
    ans = 0
    while lowT <= highT:
        midT = (lowT + highT) // 2
        if midT == 0:
            ans = midT
            lowT = midT + 1
            continue
        cost = F(midT)
        if cost <= M_val:
            ans = midT
            lowT = midT + 1
        else:
            highT = midT - 1
            
    print(ans)

if __name__ == "__main__":
