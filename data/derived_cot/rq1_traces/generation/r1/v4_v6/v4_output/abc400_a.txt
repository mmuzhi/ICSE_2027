=== ANSWER ===
To solve this problem, we need to determine if it's possible to arrange 400 people in a rectangular formation with exactly A rows without any gaps. The solution involves checking if 400 is divisible by the given integer A. If it is, then the number of columns B is simply 400 divided by A. Otherwise, if 400 is not divisible by A, no such arrangement exists, and we should return -1.

### Approach
1. **Problem Analysis**: The problem requires arranging 400 people in A rows and B columns such that A * B = 400. The key insight is that B must be a positive integer, which means 400 must be divisible by A without any remainder.
2. **Algorithm Selection**: The solution involves checking the divisibility condition. If A divides 400 evenly, then B is computed as 400 divided by A. If not, the solution returns -1.
3. **Complexity Analysis**: The solution involves a constant-time check (O(1)) since it only performs a division and modulus operation once.

### Solution Code
```python
def main():
    A = int(input().strip())
    if 400 % A == 0:
        print(400 // A)
    else:
        print(-1)

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The input is read as an integer A, which represents the number of rows.
2. **Divisibility Check**: The code checks if 400 is divisible by A using the modulus operator. If the remainder is zero, then 400 divided by A gives the number of columns B, which is printed.
3. **Handling Non-Divisible Cases**: If 400 is not divisible by A, the modulus operation results in a non-zero remainder, and the code prints -1, indicating no valid arrangement exists.

This approach efficiently solves the problem by leveraging basic arithmetic operations to check for divisibility, ensuring optimal performance.
