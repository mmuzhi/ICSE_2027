To solve this problem, we need to determine whether it's possible to transform string `S` into string `T` using a series of operations where each operation replaces all occurrences of a chosen character `x` with another character `y`. If possible, we also need to find the minimum number of operations required.

### Approach
1. **Consistency Check**: For each distinct character `c` in `S`, all corresponding characters in `T` at positions where `c` appears in `S` must be the same. If this condition is not met, it's impossible to transform `S` into `T`, and we return `-1`.
2. **Immediate Match**: If `S` is already identical to `T`, no operations are needed, so we return `0`.
3. **Count Non-Self-Loop Edges**: For each distinct character `c` in `S` that maps to a different character `d` in `T`, count these mappings as edges in a directed graph. The total number of such edges is `total_edges`.
4. **Cycle Detection**: Construct a directed graph where nodes are distinct characters from `S` and edges are mappings from `c` to `d` (if `d` is also in `S`). Detect cycles in this graph. Each cycle requires one additional operation to resolve.
5. **Calculate Minimum Operations**: The minimum number of operations required is the sum of `total_edges` (each edge requires one operation) and the number of cycles (each cycle requires one additional operation to break).

### Solution Code
```python
def main():
    import sys
    data = sys.stdin.read().splitlines()
    n = int(data[0])
    S = data[1].strip()
    T = data[2].strip()
    
    if S == T:
        print(0)
        return
        
    mapping = {}
    for i in range(n):
        char = S[i]
        if char in mapping:
            if mapping[char] != T[i]:
                print(-1)
                return
        else:
            mapping[char] = T[i]
    
    distinct_S = set(S)
    total_edges = 0
    for c in distinct_S:
        if c != mapping[c]:
            total_edges += 1
            
    graph = {}
    for c in distinct_S:
        if c == mapping[c]:
            continue
        if mapping[c] in distinct_S:
            graph[c] = mapping[c]
    
    visited = set()
    cycles = 0
    for node in graph:
        if node not in visited:
            seen = set()
            cur = node
            while cur not in visited and cur in graph:
                visited.add(cur)
                seen.add(cur)
                cur = graph[cur]
            if cur in seen:
                cycles += 1
                
    result = total_edges + cycles
    print(result)

if __name__ == "__main__":
    main()
```

### Explanation
1. **Consistency Check**: The code first checks if all occurrences of each character in `S` map to the same character in `T`. If not, it immediately returns `-1`.
2. **Immediate Match**: If `S` and `T` are identical, the code returns `0` since no operations are needed.
3. **Edge Counting**: The code counts the number of edges (non-self-loop mappings) from characters in `S` to characters in `T`. Each edge represents a necessary operation.
4. **Cycle Detection**: The code constructs a directed graph where nodes are characters from `S` and edges are mappings to other characters in `S`. It detects cycles in this graph, as each cycle requires an extra operation to resolve (using a temporary character).
5. **Result Calculation**: The total operations required are the sum of edges (non-self-loop mappings) and detected cycles. This sum is printed as the result.

This approach efficiently checks for feasibility and computes the minimal operations by leveraging graph theory concepts to handle character mappings and cycles.
