To solve this problem, we need to compute the sum of the scores of all good sequences, where a good sequence is defined as an integer sequence of length between 1 and N (inclusive) with each element between 1 and M (inclusive). The score of a sequence is the number of positive divisors of the product of its elements. Given the constraints (N can be as large as 10^18 and M up to 16), a direct approach is infeasible, so we use mathematical insights and combinatorial techniques to optimize the solution.

### Approach
1. **Prime Factorization Insight**: The score (number of divisors) of the product of sequence elements can be expressed as the product of (exponent + 1) for each prime in the prime factorization of the product. This multiplicative property allows us to handle each prime independently.
2. **Generating Functions and Inclusion-Exclusion Principle**: For each prime, we model the problem using generating functions to track the exponents contributed by each element in the sequence. The inclusion-exclusion principle helps in efficiently counting sequences that meet specific exponent requirements for each prime.
3. **Precomputation of Key Values**: 
   - **V(T)**: For each subset of primes T, compute the sum over elements (1 to M) of the product of exponents of primes in T.
   - **W(S, m)**: For each subset of primes S and integer m, compute the sum over all set partitions of S into m non-empty subsets of the product of V(T) for each subset T in the partition.
   - **U(m)**: Precompute values related to the sum of combinations and geometric series, which are used in the inclusion-exclusion calculations.
4. **Efficient Calculation**: Using precomputed values, we efficiently compute the total score by iterating over all subsets of primes and applying combinatorial formulas, leveraging modular arithmetic to handle large numbers.

mod = 998244353

import sys
from itertools import combinations

def main():
    data = sys.stdin.read().split()
    N = int(data[0])
    M_val = int(data[1])
    N_val = N
    M = M_val

    primes = (2, 3, 5, 7, 11, 13)
    nprimes = len(primes)
    
    V_dict = {}
    all_subsets = []
    for r in range(0, nprimes+1):
        for subset in combinations(primes, r):
            T = tuple(sorted(subset))
            all_subsets.append(T)
            V_dict[T] = 0

    for x in range(1, M+1):
        expo = {}
        for p in primes:
            cnt = 0
            tmp = x
            while tmp % p == 0:
                cnt += 1
                tmp //= p
            expo[p] = cnt
        
        for T in all_subsets:
            prod = 1
            for p in T:
                prod *= expo[p]
            V_dict[T] = (V_dict[T] + prod) % mod

    fact_small = [1] * 8
    for i in range(1, 8):
        fact_small[i] = fact_small[i-1] * i

    U = [0] * 7
    if M == 1:
        for m in range(0, 7):
            r = m + 1
            numerator = 1
            for i in range(r):
                term = (N_val + 1 - i) % mod
                numerator = (numerator * term) % mod
            denominator = fact_small[r] % mod
            inv_denom = pow(denominator, mod-2, mod)
            U[m] = numerator * inv_denom % mod
    else:
        Mn1 = pow(M, N_val+1, mod)
        U0 = (1 - Mn1) % mod
        inv_denom0 = pow((1 - M) % mod, mod-2, mod)
        U0 = U0 * inv_denom0 % mod
        U[0] = U0
        for m in range(1, 7):
            binom = 1
            if m > (N_val+1):
                binom = 0
            else:
                numerator_b = 1
                for i in range(m):
                    term = (N_val + 1 - i) % mod
                    numerator_b = (numerator_b * term) % mod
                denominator_b = fact_small[m] % mod
                binom = numerator_b * pow(denominator_b, mod-2, mod) % mod
            
            term_val = (M * U[m-1] - Mn1 * binom) % mod
            term_val = (term_val % mod + mod) % mod
            inv_denom = pow((1 - M) % mod, mod-2, mod)
            U_m = term_val * inv_denom % mod
            U[m] = U_m

    def set_partitions(elements, m):
        if m == 1:
            return [ [elements] ]
        n = len(elements)
        if n < m:
            return []
        if n == 0:
            return []
        first = elements[0]
        rest = elements[1:]
        results = []
        parts_minus = set_partitions(rest, m-1)
        for p in parts_minus:
            new_p = [[first]] + p
            results.append(new_p)
        parts_same = set_partitions(rest, m)
        for p in parts_same:
            for i in range(len(p)):
                new_block = p[i] + [first]
                new_p = p[:i] + [new_block] + p[i+1:]
                results.append(new_p)
        return results

    W_dict = {}
    for S in all_subsets:
        if len(S) == 0:
            continue
        n = len(S)
        W_list = [0] * (n+1)
        list_S = list(S)
        for m in range(1, n+1):
            partitions = set_partitions(list_S, m)
            total = 0
            for part in partitions:
                prod = 1
                for block in part:
                    T = tuple(sorted(block))
                    prod = (prod * V_dict[T]) % mod
                total = (total + prod) % mod
            W_list[m] = total

            term = term * U[m] % mod
