=== ANSWER ===
To solve this problem, we need to count the number of ordered pairs (s, t) where s ≠ t, and the clockwise distance from rest area s to rest area t is a multiple of a given integer M. The rest areas are arranged in a circular fashion, and the clockwise distance between consecutive rest areas is provided.

### Approach
1. **Problem Analysis**:
   - The rest areas are numbered 1 to N in clockwise order around a lake.
   - The clockwise distance from rest area i to i+1 is given by A_i (with rest area N+1 being rest area 1).
   - For any pair (s, t) where s ≠ t, the clockwise distance from s to t is:
     - If s < t: Sum of A_i from s to t-1.
     - If s > t: Sum of A_i from s to N plus the sum from 1 to t-1.
   - We need to count all such pairs where this distance is a multiple of M.

2. **Key Insight**:
   - The clockwise distance can be expressed using prefix sums. Let P be the prefix sum array where P[0] = 0, P[i] = P[i-1] + A[i-1] for i from 1 to N.
   - The clockwise distance from s to t is:
     - If s < t: P[t-1] - P[s-1]
     - If s > t: total_sum - (P[s-1] - P[t-1])
   - This distance must be a multiple of M, i.e., (distance) mod M = 0.

3. **Algorithm Selection**:
   - **Case 1 (s < t)**: The condition simplifies to (P[t-1] - P[s-1]) mod M = 0, which means P[t-1] mod M = P[s-1] mod M. We count all pairs (i, j) where i < j and residues R[i] = R[j].
   - **Case 2 (s > t)**: The condition simplifies to (total_sum + P[t-1] - P[s-1]) mod M = 0. This can be rewritten as (P[s-1] - P[t-1]) mod M = total_sum mod M. Equivalently, R[s-1] = (R[t-1] + total_sum mod M) mod M. We count all pairs (i, j) where i > j and R[i] = (R[j] + T_mod) mod M (where T_mod = total_sum mod M).

4. **Complexity Analysis**:
   - **Time Complexity**: O(N) for computing prefix sums and residues, and O(N) for counting valid pairs in both cases. The overall complexity is O(N).
   - **Space Complexity**: O(N) for storing residues and O(M) for the frequency array, which is feasible given M ≤ 10^6.

### Solution Code
```python
import sys
from collections import defaultdict

def main():
    data = sys.stdin.read().split()
    n = int(data[0])
    m = int(data[1])
    A = list(map(int, data[2:2+n]))
    
    total_sum = sum(A)
    T_mod = total_sum % m
    
    R = [0] * n
    if n > 0:
        R[0] = 0
        for i in range(1, n):
            R[i] = (R[i-1] + A[i-1]) % m
    
    freq_all = defaultdict(int)
    for r in R:
        freq_all[r] += 1
    
    ans1 = 0
    for count in freq_all.values():
        ans1 += count * (count - 1) // 2
        
    freq = [0] * m
    for r in R:
        freq[r] += 1
    
    ans2 = 0
    for j in range(n):
        r_val = R[j]
        freq[r_val] -= 1
        target = (r_val + T_mod) % m
        ans2 += freq[target]
    
    print(ans1 + ans2)

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The input is read and parsed into the number of rest areas (N), the modulus (M), and the list of distances between consecutive rest areas (A).
2. **Total Sum and Modulus**: The total sum of all distances is computed, and its modulus with M (T_mod) is calculated.
3. **Residue Array (R)**: The residue array R is constructed where each element R[i] represents the prefix sum of distances up to the i-th rest area modulo M.
4. **Case 1 Counting (ans1)**: Using a frequency dictionary, we count all pairs (i, j) where i < j and R[i] = R[j]. This is done by summing combinations of frequencies.
5. **Case 2 Counting (ans2)**: Using a frequency array of size M, we iterate through each residue, remove it from the frequency array, and count residues that satisfy R[i] = (R[j] + T_mod) mod M for i > j.
6. **Result**: The results from both cases are summed and printed as the final answer.

This approach efficiently counts all valid pairs (s, t) by leveraging prefix sums and modular arithmetic, ensuring optimal performance even for large inputs.
