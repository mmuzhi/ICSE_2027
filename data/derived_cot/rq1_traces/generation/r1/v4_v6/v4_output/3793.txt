```python
                            if c_left != c_right:
                                valid = False
                                break
                            left += 1
                            right -= 1
                        if valid:
                            ans = total_len
        return ans
```

### Explanation
1. **Initialization**: The solution starts by initializing the lengths of strings `s` and `t` and the answer `ans` to zero.
2. **Generating Substrings**: Using nested loops, all possible contiguous substrings of `s` and `t` are generated. The outer loops iterate over the start and end indices for substrings of `s`, and the inner loops do the same for `t`.
3. **Palindrome Check**: For each combination of substrings from `s` and `t`, the concatenated string's total length is checked. If it's zero (both substrings empty), it's skipped. If the total length is not greater than the current longest palindrome (`ans`), it's also skipped.
4. **Character Comparison**: The algorithm checks if the concatenated string is a palindrome by comparing characters from the start and end moving towards the center. If any mismatch is found, the check stops early.
5. **Update Answer**: If the entire concatenated string is a palindrome, its length is compared with `ans`, updating `ans` if it's longer.
6. **Result**: After checking all combinations, the longest palindrome length found (`ans`) is returned.

This approach efficiently checks all possible substring combinations while optimizing by early termination during palindrome checks and skipping shorter combinations once a longer palindrome is found. The constraints ensure the solution runs in a feasible time.
