To solve this problem, we need to determine the minimum number of operations required to make an array non-decreasing. Each operation involves selecting an element from the array and dividing it by its greatest proper divisor. The greatest proper divisor of a number \( x \) is the largest divisor of \( x \) that is strictly less than \( x \). 

### Approach
1. **Precompute Smallest Prime Factors (SPF):** 
   - We precompute the smallest prime factor for every number up to \( 10^6 \) using the Sieve of Eratosthenes. This helps in efficiently determining the greatest proper divisor for any number in the array. The greatest proper divisor of a number \( x \) is \( x \) divided by its smallest prime factor.

2. **Dynamic Programming (DP) Setup:**
   - For each element in the array, we consider two states:
     - **State 0:** The element remains unchanged (no operation performed).
     - **State 1:** The element is divided by its greatest proper divisor (operation performed), reducing it to its smallest prime factor.
   - The cost of transitioning to State 1 is 1 if the element is composite (i.e., it can be divided), otherwise, it remains 0 (for primes or 1).

3. **DP Transitions:**
   - For each element starting from the second element, we check transitions from the previous element's states (State 0 and State 1) to the current element's states (State 0 and State 1). The transitions are valid only if the previous element's value (from either state) is less than or equal to the current element's value (from either state).
   - The DP values are updated to reflect the minimum operations required to achieve a non-decreasing sequence up to the current element.

4. **Result Calculation:**
   - After processing all elements, the result is the minimum value between the two states (State 0 and State 1) of the last element. If both states are invalid (i.e., the sequence cannot be made non-decreasing), return -1.

import math

class Solution:
    spf = None

    def minOperations(self, nums: List[int]) -> int:

                    for j in range(i * i, max_n + 1, i):
                        if spf_arr[j] == j:
                            spf_arr[j] = i

        dp0 = 0

            a_prev1 = Solution.spf[nums[i-1]]
            a_curr0 = nums[i]
            a_curr1 = Solution.spf[nums[i]]
            
            cur0 = INF
            if a_prev0 <= a_curr0:
                cur0 = min(cur0, dp0)
            if a_prev1 <= a_curr0:
                cur0 = min(cur0, dp1)

                cur1 = min(cur1, dp0)

                if cur1 != INF:

            else:
