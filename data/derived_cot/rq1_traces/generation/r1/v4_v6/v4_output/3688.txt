A subarray is a contiguous non-empty sequence of elements within an array.
 
Example 1:

Input: nums = [-3,2,-2,-1,3,-2,3]
Output: 7
Explanation:
We can have the following arrays after at most one operation:

        mapping = defaultdict(list)

            if len(indices) == n:

            # Revert the updates: set the indices back to x
            for i in indices:

        return ans

To solve this problem, we need to find the maximum subarray sum after removing all occurrences of any single integer from the array, provided the array remains non-empty. The solution involves considering each distinct integer in the array, removing all its occurrences, and then computing the maximum subarray sum of the resulting array. The challenge is to efficiently compute these maximum subarray sums for each candidate integer without resorting to a brute-force approach, which would be too slow for large arrays.

### Approach
1. **Initial Setup**: We start by computing the maximum subarray sum of the original array using Kadane's algorithm. This serves as the initial candidate for the maximum subarray sum.
2. **Preprocessing**: We create a mapping from each distinct integer in the array to the list of indices where it appears. This helps in efficiently accessing all occurrences of each integer.
3. **Segment Tree Construction**: We build a segment tree over the original array. Each node in the segment tree stores four values: the total sum of the segment, the maximum prefix sum, the maximum suffix sum, and the maximum subarray sum within the segment. This structure allows efficient updates and queries.
4. **Processing Each Candidate Integer**: For each distinct integer in the array, we check if removing all its occurrences would leave the array non-empty. If so, we update the segment tree by setting all occurrences of this integer to zero. We then query the segment tree for the maximum subarray sum of the modified array. After recording this value, we revert the updates to restore the original array values.
5. **Result Calculation**: The maximum value obtained from the original array and all modified arrays (after removing each distinct integer) is the desired result.

### Solution Code
```python
from collections import defaultdict

class SegmentTree:
    def __init__(self, data):
        self.n = len(data)
        self.data = data
        self.tree = [(0, 0, 0, 0)] * (4 * self.n)
        self.build(0, 0, self.n - 1)
    
    def build(self, idx, l, r):
        if l == r:
            val = self.data[l]
            self.tree[idx] = (val, val, val, val)
            return
        mid = (l + r) // 2
        left_child = 2 * idx + 1
        right_child = 2 * idx + 2
        self.build(left_child, l, mid)
        self.build(right_child, mid + 1, r)
        self.tree[idx] = self.merge_nodes(self.tree[left_child], self.tree[right_child])
    
    def merge_nodes(self, left, right):
        total = left[0] + right[0]
        max_prefix = max(left[1], left[0] + right[1])
        max_suffix = max(right[2], right[0] + left[2])
        max_subarray = max(left[3], right[3], left[2] + right[1])
        return (total, max_prefix, max_suffix, max_subarray)
    
    def update_point(self, idx, l, r, pos, val):
        if l == r:
            self.tree[idx] = (val, val, val, val)
            return
        mid = (l + r) // 2
        if pos <= mid:
            self.update_point(2 * idx + 1, l, mid, pos, val)
        else:
            self.update_point(2 * idx + 2, mid + 1, r, pos, val)
        self.tree[idx] = self.merge_nodes(self.tree[2 * idx + 1], self.tree[2 * idx + 2])
    
    def update(self, pos, val):
        self.update_point(0, 0, self.n - 1, pos, val)
    
    def get_max_subarray(self):
        return self.tree[0][3]

class Solution:
    def maxSubarraySum(self, nums):
        n = len(nums)
        seg_tree = SegmentTree(nums)
        ans = seg_tree.get_max_subarray()
        mapping = defaultdict(list)
        for i, num in enumerate(nums):
            mapping[num].append(i)
        
        for x, indices in mapping.items():
            if len(indices) == n:
                continue
            for i in indices:
                seg_tree.update(i, 0)
            current_max = seg_tree.get_max_subarray()
            ans = max(ans, current_max)
            for i in indices:
                seg_tree.update(i, x)
        
        return ans
```

### Explanation
1. **Segment Tree**: The segment tree is used to efficiently compute the maximum subarray sum after dynamically updating elements (setting occurrences of a candidate integer to zero). Each node in the tree stores information that allows merging segments while maintaining the necessary properties (total sum, max prefix, max suffix, and max subarray sum).
2. **Processing Candidates**: For each distinct integer in the array, we temporarily set all its occurrences to zero in the segment tree. This simulates removing the integer from the array. We then query the tree for the maximum subarray sum of this modified array.
3. **Efficiency**: The segment tree updates and queries take logarithmic time per operation. Since we process each distinct integer and update/revert all its occurrences, the total operations are O(n log n) in the worst case, making the solution efficient for large arrays (n ≤ 10^5).
4. **Result**: The solution compares the maximum subarray sums obtained from the original array and each modified array (after removing each distinct integer) and returns the highest value encountered. This ensures optimal performance while adhering to the problem constraints.
