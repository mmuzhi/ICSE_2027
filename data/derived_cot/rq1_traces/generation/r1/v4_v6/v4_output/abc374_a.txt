=== ANSWER ===
To solve this problem, we need to determine if a given string ends with the suffix "san". The solution involves checking the last three characters of the string to see if they match "san". 

### Approach
1. **Problem Analysis**: The problem requires checking if the input string ends with "san". Given the constraints that the string length is between 4 and 30, we can safely check the last three characters without worrying about the string being too short.
2. **Intuition**: The solution leverages Python's string slicing or the `endswith` method to efficiently check the suffix of the string.
3. **Algorithm Selection**: We can use either:
   - String slicing to extract the last three characters and compare them with "san".
   - The built-in `endswith` method which directly checks if the string ends with the specified suffix.
4. **Complexity Analysis**: Both approaches operate in constant time O(1) since checking the last three characters or using `endswith` is a constant-time operation regardless of the string length.

### Solution Code
```python
def main():
    S = input().strip()
    if S.endswith('san'):
        print("Yes")
    else:
        print("No")

if __name__ == "__main__":
    main()
```

### Explanation
1. **Input Handling**: The input string `S` is read from standard input and stripped of any leading or trailing whitespace.
2. **Check Suffix**: The solution uses the `endswith` method to check if the string `S` ends with "san".
3. **Output Result**: If the condition is met, "Yes" is printed; otherwise, "No" is printed.
This approach efficiently checks the suffix and provides the correct output based on the given problem constraints.
