=== ANSWER ===
To solve this problem, we need to count the number of substrings in a given string that contain every vowel ('a', 'e', 'i', 'o', 'u') at least once and exactly `k` consonants. 

### Approach
1. **Problem Analysis**: The task involves checking all possible substrings of the given string to determine if they meet two conditions: 
   - The substring must include all five vowels at least once.
   - The substring must contain exactly `k` consonants (non-vowel characters).
   
2. **Intuition**: For each possible starting index in the string, we expand the window to include subsequent characters, tracking the count of distinct vowels and the number of consonants. Once all five vowels are present in the current window, we check if the number of consonants matches `k`. If it does, we increment our result count. If the consonant count exceeds `k`, we break early for that starting index since further expansion will only increase the consonant count.

3. **Algorithm Selection**: 
   - **Nested Loop**: The outer loop iterates over each character as the starting index of the substring.
   - **Inner Loop**: Expands the window from the starting index, tracking distinct vowels and consonants. Once all vowels are found, it checks for the exact consonant count `k`. If the consonant count exceeds `k`, it breaks early to avoid unnecessary checks.

4. **Complexity Analysis**: 
   - **Time Complexity**: O(n^2) in the worst case, where `n` is the length of the string. This is because for each starting index, the inner loop might process up to `n` characters.
   - **Space Complexity**: O(1) since we use a fixed-size frequency array for vowels and a few variables for counts.

### Solution Code
```python
class Solution:
    def countOfSubstrings(self, word: str, k: int) -> int:
        n = len(word)
        vowels = "aeiou"
        result = 0
        for start in range(n):
            freq = [0] * 5
            distinct = 0
            cons = 0
            for end in range(start, n):
                ch = word[end]
                if ch in vowels:
                    idx = vowels.index(ch)
                    if freq[idx] == 0:
                        distinct += 1
                    freq[idx] += 1
                else:
                    cons += 1
                if distinct == 5:
                    if cons == k:
                        result += 1
                    if cons > k:
                        break
        return result
```

### Explanation
1. **Initialization**: We initialize the result count to zero and define the vowels string for quick reference.
2. **Outer Loop**: Iterates over each character in the string as the starting index of a potential substring.
3. **Inner Loop**: For each starting index, expands the window to include subsequent characters:
   - **Tracking Vowels**: Uses a frequency array to count occurrences of each vowel and tracks distinct vowels encountered.
   - **Tracking Consonants**: Counts non-vowel characters in the current window.
4. **Condition Check**: Once all five vowels are found in the current window:
   - If the consonant count equals `k`, the result count is incremented.
   - If the consonant count exceeds `k`, the inner loop breaks early since further expansion won't reduce the count.
5. **Result**: After processing all possible substrings, the result count is returned, representing the number of valid substrings meeting the specified conditions.
