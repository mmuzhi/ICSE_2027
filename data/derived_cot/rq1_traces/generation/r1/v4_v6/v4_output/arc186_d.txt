Whether a non-empty sequence of non-negative integers (V_1, V_2, \dots, V_M) is Polish or not is recursively defined as follows:

- We say (V_1, V_2, \dots, V_M) is Polish if there exist V_1 Polish sequences W_1, W_2, \dots, W_{V_1} such that the concatenation of sequences (V_1), W_1, W_2, \dots, W_{V_1} in this order equals (V_1, V_2, \dots, V_M).

In particular, the sequence (0) is Polish.
Given a sequence of non-negative integers (A_1, A_2, \dots, A_N) of length N, find the number of Polish sequences of length N that are lexicographically not greater than (A_1, A_2, \dots, A_N), modulo 998244353.
 What is lexicographical order on sequences?
We say that sequence S = (S_1,S_2,\ldots,S_{|S|}) is lexicographically less than sequence T = (T_1,T_2,\ldots,T_{|T|}) if either condition 1. or 2. below holds.
Here, |S|, |T| represent the lengths of S, T respectively.

-  |S| \lt |T| and (S_1,S_2,\ldots,S_{|S|}) = (T_1,T_2,\ldots,T_{|S|}). 
-  There exists an integer 1 \leq i \leq \min\lbrace |S|, |T| \rbrace such that both of the following hold:

-  (S_1,S_2,\ldots,S_{i-1}) = (T_1,T_2,\ldots,T_{i-1})
-  S_i is (numerically) less than T_i.

Input

The input is given from Standard Input in the following format:

A_1 A_2 \dots A_N

Output

Print the number of sequences satisfying the conditions, modulo 998244353.

Constraints


- 1\leq N \leq 3\times 10^5
- 0\leq A_i \lt N
- All input values are integers.

The solution involves understanding the recursive definition of Polish sequences and efficiently counting them using dynamic programming and combinatorial mathematics.

### Approach
1. **Understanding Polish Sequences**: A sequence is Polish if it can be decomposed into a head (the first element) followed by exactly V1 Polish sequences. The base case is the sequence (0). This structure implies that the entire sequence must satisfy certain constraints related to the lengths of the subsequences.

2. **Dynamic Programming Setup**: We use a dynamic programming (DP) approach where `dp[i]` represents the number of Polish sequences of length i. The recurrence relation is derived from the observation that a Polish sequence of length i must start with a value k (0 ≤ k < i) and be followed by k Polish sequences whose total length sums to i-1.

3. **Generating Function**: To efficiently compute the DP array, we use generating functions. The generating function for the DP array is defined as a polynomial where the coefficient of x^i is `dp[i]`. The recurrence relation translates into a functional equation for this generating function, which we solve using convolution and combinatorial techniques.

4. **Lexicographical Order Handling**: To count sequences lexicographically not greater than A, we iterate through each position, considering sequences that match the prefix of A up to that position and then extend with any valid Polish sequence. We use the precomputed DP array to quickly count valid continuations.

5. **Modulo Handling**: Given the constraints, all operations are performed modulo 998244353 to avoid overflow and meet the problem requirements.

MOD = 998244353

import sys
import numpy as np
sys.setrecursionlimit(10**6)

def main():
    data = sys.stdin.read().split()
    n = int(data[0])
    A = list(map(int, data[1:1+n]))
    
    if n == 1:
        print(1 if A[0] == 0 else 0)
        return

    dp = [0] * (n+1)
    dp[1] = 1
    for i in range(2, n+1):
        dp[i] = 0
        for k in range(1, i):
            j = i - k - 1
            if j < 0:
                break
            dp[i] = (dp[i] + dp[k] * dp[j]) % MOD
    
    F = np.zeros(n+1, dtype=np.int64)
    F[1] = 1
    for i in range(2, n+1):
        F[i] = dp[i]
        for j in range(1, i):
            F[i] = (F[i] + F[j] * dp[i - j]) % MOD
    
    ans = 0
    for i in range(n):
        if i == 0:
            if A[0] == 0:
                ans = (ans + 1) % MOD
                continue
            for x in range(1, min(A[0], n)+1):
                if n - x - 1 >= 0:
                    ans = (ans + dp[n - x]) % MOD
        else:
            for x in range(A[i]):
                if n - i - 1 >= 0:
                    ans = (ans + F[n - i]) % MOD
            if A[i] > 0:
