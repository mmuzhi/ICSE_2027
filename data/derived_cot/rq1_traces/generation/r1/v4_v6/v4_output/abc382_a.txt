You will be given a competitive programming problem.
There are N boxes arranged in a row, and some of these boxes contain cookies.
The state of these boxes is represented by a string S of length N.
Specifically, the i-th box (1\leq i \leq N) from the left contains one cookie if the i-th character of S is @, and is empty if it is ..
Over the next D days, Takahashi will choose and eat one cookie per day from among the cookies in these boxes.
Determine how many of the N boxes will be empty after D days have passed.
(It can be proved that this value does not depend on which cookies Takahashi chooses each day.)
It is guaranteed that S contains at least D occurrences of @.

Input

The input is given from Standard Input in the following format:
N D
S

Output

Print the number of boxes that will be empty after D days have passed among the N boxes.

Constraints


- 1 \leq D \leq N \leq 100
- N and D are integers.
- S is a string of length N consisting of @ and ..
- S contains at least D occurrences of @.

Sample Input 1

5 2
.@@.@

Sample Output 1

4

For example, Takahashi might act as follows:

- Day 1: There are cookies in the 2nd, 3rd, and 5th boxes from the left. He chooses the cookie in the 2nd box to eat.
- Day 2: There are cookies in the 3rd and 5th boxes. He chooses the cookie in the 5th box to eat.
- After two days have passed, only the 3rd box from the left contains a cookie. Therefore, four out of the five boxes are empty.

Even though Takahashi might choose differently on each day than in this example, there will still be four empty boxes after two days.
Therefore, the answer is 4.

Sample Input 2

3 3
@@@

Sample Output 2

3

Sample Input 3

10 4
@@@.@@.@@.

Sample Output 3

7

=== ANSWER ===
To solve this problem, we need to determine the number of empty boxes after Takahashi has eaten D cookies over D days. The key observation here is that each cookie removal turns one box from having a cookie to being empty. The solution leverages this observation to compute the result efficiently without simulating each day's cookie removal.

### Approach
1. **Problem Analysis**: The problem involves N boxes arranged in a row, each either containing a cookie ('@') or being empty ('.'). Over D days, Takahashi eats one cookie per day from any box that currently has a cookie. The challenge is to find out how many boxes will be empty after D days.
2. **Insight**: The solution hinges on recognizing that each cookie removal converts one box from having a cookie to being empty. Therefore, the final number of empty boxes is the sum of the initially empty boxes and the number of cookies removed (D).
3. **Algorithm Selection**: The algorithm involves counting the number of initially empty boxes (those with '.') and adding D to this count. This approach works because each of the D removals turns a cookie-containing box into an empty one, and the initially empty boxes remain empty.
4. **Complexity Analysis**: The solution requires a single pass through the string to count the initial empty boxes, resulting in a time complexity of O(N), where N is the length of the string. This is optimal for the given constraints.

### Solution Code
```python
def main():
    import sys
    data = sys.stdin.read().splitlines()
    n_d = data[0].split()
    D = int(n_d[1])
    S = data[1].strip()
    initial_empty = S.count('.')
    result = initial_empty + D
    print(result)

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The input is read from standard input, where the first line contains N (number of boxes) and D (number of days), and the second line is the string S representing the initial state of the boxes.
2. **Counting Initial Empty Boxes**: The number of initially empty boxes is determined by counting the occurrences of '.' in the string S.
3. **Computing Result**: The result is computed by adding D (the number of cookies to be removed) to the count of initially empty boxes. This gives the total number of empty boxes after D days, as each cookie removal turns one box from having a cookie to being empty.
4. **Output**: The computed result is printed.

This approach efficiently solves the problem by leveraging the observation that the order of cookie removal does not affect the final count of empty boxes, thus simplifying the solution to a straightforward calculation.
