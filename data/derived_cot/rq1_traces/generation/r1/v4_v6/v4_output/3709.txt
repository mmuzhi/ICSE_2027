1. The substring must consist of only one distinct character (e.g., "aaa" or "bbb").
2. If there is a character immediately before the substring, it must be different from the character in the substring.
3. If there is a character immediately after the substring, it must also be different from the character in the substring.

### Approach
1. **Iterate through the string**: For each possible starting index `i` of a substring of length `k`, check the substring `s[i:i+k]`.
2. **Check homogeneity**: Verify if all characters in the substring `s[i:i+k]` are the same. This can be efficiently checked by comparing the substring to a string of `k` identical characters (i.e., `s[i] * k`).
3. **Check boundaries**: 
   - The character immediately before the substring (if it exists) must be different from the substring's character. This is checked by comparing `s[i-1]` with `s[i]` when `i > 0`.
   - The character immediately after the substring (if it exists) must be different from the substring's character. This is checked by comparing `s[i+k]` with `s[i]` when `i + k < len(s)`.
4. **Return early**: If a valid substring is found that satisfies all conditions, return `True` immediately. If no such substring is found after checking all possible substrings, return `False`.

class Solution:
