You will be given a competitive programming problem.
You are given a grid of H rows and W columns.
Let (i,j) denote the cell at row i (1 \leq i \leq H) from the top and column j (1 \leq j \leq W) from the left.
The state of the grid is represented by H strings S_1, S_2, \ldots, S_H, each of length W, as follows:

- If the j-th character of S_i is #, cell (i,j) is painted black.
- If the j-th character of S_i is ., cell (i,j) is painted white.
- If the j-th character of S_i is ?, cell (i,j) is not yet painted.

Takahashi wants to paint each not-yet-painted cell white or black so that all the black cells form a rectangle.
More precisely, he wants there to exist a quadruple of integers (a,b,c,d) (1 \leq a \leq b \leq H, 1 \leq c \leq d \leq W) such that:

For each cell (i,j) (1 \leq i \leq H, 1 \leq j \leq W),
if a \leq i \leq b and c \leq j \leq d, the cell is black;
otherwise, the cell is white.

Determine whether this is possible.

Input

The input is given from Standard Input in the following format:
H W
S_1
S_2
\vdots
S_H

Output

If it is possible to paint all the not-yet-painted cells so that the black cells form a rectangle, print Yes; otherwise, print No.

Constraints


- 1 \leq H, W \leq 1000
- H and W are integers.
- Each S_i is a string of length W consisting of #, ., ?.
- There is at least one cell that is already painted black.

To solve this problem, we need to determine whether it's possible to paint all the '?' cells in a grid such that all the black cells form a rectangle. The key insight is that the minimal rectangle covering all existing black cells must not contain any white cells ('.'), and all existing black cells must lie within this rectangle. 

### Approach
1. **Identify Existing Black Cells**: Traverse the grid to collect the positions of all existing black cells ('#').
2. **Determine Minimal Rectangle**: Calculate the minimal rectangle that encompasses all existing black cells. This rectangle is defined by the minimum and maximum row and column indices of these black cells.
3. **Check for White Cells in Rectangle**: For each cell within the minimal rectangle, if any cell is white ('.'), it's impossible to form the desired rectangle (output "No"). Otherwise, it's possible (output "Yes").

### Solution Code
```python
def main():
    import sys
    data = sys.stdin.read().splitlines()
    if not data:
        print("No")
        return
    H, W = map(int, data[0].split())
    grid = []
    for i in range(1, 1 + H):
        grid.append(data[i].strip())
    
    blacks = []
    for i in range(H):
        for j in range(W):
            if grid[i][j] == '#':
                blacks.append((i, j))
    
    if not blacks:
        print("No")
        return
    
    min_row = min(i for i, j in blacks)
    max_row = max(i for i, j in blacks)
    min_col = min(j for i, j in blacks)
    max_col = max(j for i, j in blacks)
    
    for i in range(min_row, max_row + 1):
        for j in range(min_col, max_col + 1):
            if grid[i][j] == '.':
                print("No")
                return
    
    print("Yes")

if __name__ == '__main__':
    main()
```

### Explanation
1. **Reading Input**: The input is read from standard input, where the first line specifies the grid dimensions (H rows and W columns), followed by H lines representing the grid.
2. **Collecting Black Cells**: The grid is scanned to collect the coordinates (i, j) of all cells containing '#' (existing black cells).
3. **Minimal Rectangle Calculation**: The minimal rectangle is determined by finding the smallest and largest row and column indices among all existing black cells. This rectangle must cover all black cells.
4. **Checking for White Cells**: The algorithm checks each cell within the minimal rectangle. If any cell is white ('.'), it immediately outputs "No" because such a cell cannot be painted black. If no white cells are found within the rectangle, it outputs "Yes", indicating that all '?' cells within the rectangle can be painted black and those outside can be painted white to form the desired rectangle. 

This approach efficiently checks the feasibility of forming a black rectangle by leveraging the minimal bounding rectangle of existing black cells and ensuring no conflicting white cells exist within it.
