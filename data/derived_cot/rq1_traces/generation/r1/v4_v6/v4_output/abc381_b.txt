=== ANSWER ===
To solve this problem, we need to determine if a given string meets the criteria of a "1122 string". The criteria are as follows:
1. The length of the string must be even.
2. For each pair of consecutive characters (i.e., the 1st and 2nd, 3rd and 4th, etc.), the characters must be identical.
3. Each distinct character in the string must appear exactly twice.

### Approach
1. **Check String Length**: The first condition requires that the length of the string is even. If the length is odd, the string cannot be a 1122 string, and we immediately return "No".
2. **Check Pair Equality**: The second condition requires that every pair of consecutive characters (i.e., characters at positions 0 and 1, 2 and 3, etc.) are identical. If any pair fails this condition, we return "No".
3. **Check Character Frequency**: The third condition requires that each distinct character in the string appears exactly twice. This can be verified by checking that the number of distinct characters in the string is exactly half the length of the string. This is because each distinct character should form exactly one pair (two characters), so the total number of distinct characters should be equal to the total number of pairs (which is half the string length).

### Solution Code
```python
def main():
    S = input().strip()
    n = len(S)
    if n % 2 != 0:
        print("No")
        return
    
    for i in range(0, n, 2):
        if S[i] != S[i+1]:
            print("No")
            return
    
    distinct_count = len(set(S))
    if distinct_count == n // 2:
        print("Yes")
    else:
        print("No")

if __name__ == '__main__':
    main()
```

### Explanation
1. **Input Handling**: The input string is read and stored in the variable `S`.
2. **Length Check**: The length of `S` is checked to see if it is even. If not, "No" is printed and the program exits.
3. **Pair Equality Check**: The loop iterates over the string in steps of 2, checking each pair of consecutive characters. If any pair consists of different characters, "No" is printed and the program exits.
4. **Distinct Character Check**: The number of distinct characters in the string is counted. If this count equals half the length of the string, it means each distinct character appears exactly twice, satisfying the third condition. Thus, "Yes" is printed; otherwise, "No" is printed.

This approach efficiently checks all three conditions in sequence, ensuring optimal performance with a time complexity of O(n), where n is the length of the string. The space complexity is O(1) for storing the distinct characters, as the number of distinct characters is bounded by the alphabet size (26 lowercase letters).
