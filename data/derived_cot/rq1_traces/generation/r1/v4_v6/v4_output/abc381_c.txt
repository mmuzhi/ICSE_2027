You will be given a competitive programming problem.
The definition of an 11/22 string in this problem is the same as in Problems A and E.

A string T is called an 11/22 string when it satisfies all of the following conditions:

- |T| is odd. Here, |T| denotes the length of T.
- The 1-st through (\frac{|T|+1}{2} - 1)-th characters are all 1.
- The (\frac{|T|+1}{2})-th character is /.
- The (\frac{|T|+1}{2} + 1)-th through |T|-th characters are all 2.

For example, 11/22, 111/222, and / are 11/22 strings, but 1122, 1/22, 11/2222, 22/11, and //2/2/211 are not.
You are given a string S of length N consisting of 1, 2, and /, where S contains at least one /.
Find the maximum length of a (contiguous) substring of S that is an 11/22 string.

Input

The input is given from Standard Input in the following format:
N
S

Output

Print the maximum length of a (contiguous) substring of S that is an 11/22 string.

Constraints


- 1 \leq N \leq 2 \times 10^5
- S is a string of length N consisting of 1, 2, and /.
- S contains at least one /.

Sample Input 1

211/2212

Sample Output 1

5

The substring from the 2-nd to 6-th character of S is 11/22, which is an 11/22 string. Among all substrings of S that are 11/22 strings, this is the longest. Therefore, the answer is 5.

=== ANSWER ===
To solve this problem, we need to find the maximum length of a contiguous substring in a given string that forms an 11/22 string. An 11/22 string is defined as a string where the length is odd, the first half (excluding the middle character) consists of all '1's, the middle character is '/', and the second half (excluding the middle character) consists of all '2's.

### Approach
1. **Problem Analysis**: The problem requires checking all possible substrings of the given string that meet the criteria of an 11/22 string. The key observation is that any valid substring must have a '/' as its middle character. Therefore, we can iterate over each occurrence of '/' in the string and consider it as the potential center of a valid substring.
2. **Precomputation**: To efficiently determine the maximum possible substring length centered at each '/', we precompute two arrays:
   - `left_ones`: This array stores the number of consecutive '1's ending at each position (from left to right).
   - `right_twos`: This array stores the number of consecutive '2's starting at each position (from right to left).
3. **Center Processing**: For each '/' in the string, we use the precomputed arrays to determine the number of consecutive '1's immediately to the left and the number of consecutive '2's immediately to the right. The maximum valid substring length centered at this '/' is then calculated as `2 * min(left_count, right_count) + 1`.
4. **Result Calculation**: The maximum length encountered during the iteration over all '/' positions is the solution to the problem.

### Solution Code
```python
def main():
    import sys
    data = sys.stdin.read().splitlines()
    n = int(data[0].strip())
    s = data[1].strip()
    
    left_ones = [0] * n
    if n > 0:
        left_ones[0] = 1 if s[0] == '1' else 0
        for i in range(1, n):
            if s[i] == '1':
                left_ones[i] = left_ones[i-1] + 1
            else:
                left_ones[i] = 0
    
    right_twos = [0] * n
    if n > 0:
        right_twos[-1] = 1 if s[-1] == '2' else 0
        for i in range(n-2, -1, -1):
            if s[i] == '2':
                right_twos[i] = right_twos[i+1] + 1
            else:
                right_twos[i] = 0
    
    ans = 1
    for i in range(n):
        if s[i] == '/':
            left_count = left_ones[i-1] if i - 1 >= 0 else 0
            right_count = right_twos[i+1] if i + 1 < n else 0
            current_length = 2 * min(left_count, right_count) + 1
            if current_length > ans:
                ans = current_length
    print(ans)

if __name__ == "__main__":
    main()
```

### Explanation
1. **Reading Input**: The input is read from standard input, where the first line is the length of the string, and the second line is the string itself.
2. **Precomputing Arrays**:
   - `left_ones` is computed by iterating from left to right. For each position, if the character is '1', it increments the count from the previous position; otherwise, it resets to zero.
   - `right_twos` is computed by iterating from right to left. For each position, if the character is '2', it increments the count from the next position; otherwise, it resets to zero.
3. **Finding Maximum Substring**: For each '/' in the string, the number of consecutive '1's to the left and '2's to the right are retrieved from the precomputed arrays. The length of the valid substring centered at this '/' is `2 * min(left_count, right_count) + 1`. The maximum length encountered during this iteration is printed as the result.

This approach efficiently checks all potential valid substrings by leveraging precomputed arrays, ensuring optimal performance even for large input sizes.
