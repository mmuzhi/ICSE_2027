package org.example;

import java.util.List;
import java.util.stream.DoubleStream;

public class DataStatistics2 {
    private double[] data;

    public DataStatistics2(List<Integer> data) {
        this.data = data.stream().mapToDouble(Integer::doubleValue).toArray();
    }

    public double getSum() {
        return DoubleStream.of(data).sum();
    }

    public double getMin() {
        return DoubleStream.of(data).min().orElse(Double.NaN);
    }

    public double getMax() {
        return DoubleStream.of(data).max().orElse(Double.NaN);
    }

    public double getVariance() {
        double mean = getSum() / data.length;
        return DoubleStream.of(data)
                .map(val -> (val - mean) * (val - mean))
                .average()
                .orElse(Double.NaN);
    }

    public double getStdDeviation() {
        return Math.sqrt(getVariance());
    }

    public double getCorrelation() {
        return 1.0;
    }
}

To solve this problem, we need to translate a Java class that performs statistical operations on a dataset into an equivalent Python class. The Java class includes methods to compute the sum, minimum, maximum, variance, standard deviation, and correlation of a list of integers converted to doubles. The Python implementation must maintain identical behavior, including handling edge cases such as empty inputs and ensuring the same numerical results.

### Approach
1. **Constructor Conversion**: The Java constructor converts a list of integers into an array of doubles. In Python, we convert each integer in the input list to a float and store them in a list.
2. **Sum Calculation**: The sum of all elements in the dataset is computed using Python's built-in `sum` function.
3. **Minimum and Maximum Values**: These are computed using Python's `min` and `max` functions. If the dataset is empty, we return `float('nan')` to match Java's behavior of returning `Double.NaN`.
4. **Variance Calculation**: The variance is computed as the average of the squared differences from the mean. For an empty dataset, we return `float('nan')`.
5. **Standard Deviation**: This is the square root of the variance. We use the `math.sqrt` function for this computation.
6. **Correlation**: As per the Java code, this method always returns 1.0.

### Solution Code
```python
import math

class DataStatistics2:
    def __init__(self, data):
        self.data = [float(x) for x in data]
    
    def getSum(self):
        return sum(self.data)
    
    def getMin(self):
        if len(self.data) == 0:
            return float('nan')
        return min(self.data)
    
    def getMax(self):
        if len(self.data) == 0:
            return float('nan')
        return max(self.data)
    
    def getVariance(self):
        n = len(self.data)
        if n == 0:
            return float('nan')
        mean = self.getSum() / n
        squared_diffs = [(x - mean) ** 2 for x in self.data]
        return sum(squared_diffs) / n
    
    def getStdDeviation(self):
        variance = self.getVariance()
        return math.sqrt(variance)
    
    def getCorrelation(self):
        return 1.0
```
