package org.example;

import java.util.ArrayList;
import java.util.List;

public class IPAddress {
    private String ipAddress;

    public IPAddress(String ipAddress) {
        this.ipAddress = ipAddress;
    }

    public boolean isValid() {
        String[] octets = ipAddress.split("\\.");
        if (octets.length != 4) {
            return false;
        }
        for (String octet : octets) {
            try {
                int num = Integer.parseInt(octet);
                if (num < 0 || num > 255) {
                    return false;
                }
            } catch (NumberFormatException e) {
                return false;
            }
        }
        return true;
    }

    public List<String> getOctets() {
        List<String> octetsList = new ArrayList<>();
        if (isValid()) {
            String[] octets = ipAddress.split("\\.");
            for (String octet : octets) {
                octetsList.add(octet);
            }
        }
        return octetsList;
    }

    public String getBinary() {
        if (isValid()) {
            StringBuilder binaryString = new StringBuilder();
            String[] octets = ipAddress.split("\\.");
            for (String octet : octets) {
                int num = Integer.parseInt(octet);
                String binaryOctet = String.format("%08d", Integer.parseInt(Integer.toBinaryString(num)));
                binaryString.append(binaryOctet).append(".");
            }
            return binaryString.substring(0, binaryString.length() - 1);
        }
        return "";
    }
}

To solve this problem, we need to translate a Java class that handles IP address validation, octet extraction, and binary representation into Python while maintaining identical behavior. The Python class should replicate the functionality of the Java class, including input validation, output formats, and edge case handling.

### Approach
1. **Class Initialization**: The `IPAddress` class is initialized with an IP address string.
2. **Validation Method (`isValid`)**:
   - Split the IP address string into octets using the dot as a delimiter.
   - Check if there are exactly 4 octets.
   - Validate each octet to ensure it is an integer between 0 and 255 inclusive.
3. **Octet Extraction Method (`getOctets`)**:
   - If the IP address is valid, return the list of octet strings; otherwise, return an empty list.
4. **Binary Representation Method (`getBinary`)**:
   - If the IP address is valid, convert each octet to its binary representation, ensuring each binary string is 8 digits long (padded with leading zeros if necessary), and join them with dots; otherwise, return an empty string.

### Solution Code
```python
class IPAddress:
    def __init__(self, ip_address):
        self.ip_address = ip_address

    def isValid(self):
        octets = self.ip_address.split('.')
        if len(octets) != 4:
            return False
        for octet in octets:
            try:
                num = int(octet)
                if num < 0 or num > 255:
                    return False
            except ValueError:
                return False
        return True

    def getOctets(self):
        if self.isValid():
            return self.ip_address.split('.')
        else:
            return []

    def getBinary(self):
        if self.isValid():
            octets = self.ip_address.split('.')
            binary_octets = []
            for octet in octets:
                num = int(octet)
                bin_str = bin(num)[2:]
                padded_bin = bin_str.zfill(8)
                binary_octets.append(padded_bin)
            return ".".join(binary_octets)
        else:
            return ""
```

### Explanation
1. **Initialization**: The `__init__` method stores the IP address string provided during object creation.
2. **Validation (`isValid`)**:
   - The IP address is split into octets. If there aren't exactly 4 octets, it returns `False`.
   - Each octet is converted to an integer. If conversion fails (non-integer) or the integer is outside the range 0-255, it returns `False`.
3. **Octet Extraction (`getOctets`)**:
   - If the IP is valid, it returns the list of octet strings; otherwise, an empty list is returned.
4. **Binary Conversion (`getBinary`)**:
   - If the IP is valid, each octet is converted to a binary string (without the '0b' prefix), padded to 8 digits with leading zeros, and joined into a single string with dots separating the octets. If invalid, an empty string is returned.

This approach ensures that the Python implementation behaves identically to the original Java code, handling all edge cases and maintaining the same input/output expectations.
