import java.util.ArrayList;
import java.util.HashMap;
import java.util.List;
import java.util.Map;

public class BoyerMooreSearch {
    private final String text;
    private final String pattern;
    private final int textLen;
    private final int patLen;

    public BoyerMooreSearch(String text, String pattern) {
        this.text = text;
        this.pattern = pattern;
        this.textLen = text.length();
        this.patLen = pattern.length();
    }

    int matchInPattern(char ch) {
        return pattern.lastIndexOf(ch);
    }

    int mismatchInText(int currentPos) {
        for (int i = patLen - 1; i >= 0; i--) {
            if (pattern.charAt(i) != text.charAt(currentPos + i)) {
                return currentPos + i;
            }
        }
        return -1;
    }

    public List<Integer> badCharacterHeuristic() {
        List<Integer> positions = new ArrayList<>();
        int i = 0;

        Map<Character, Integer> badCharHeuristic = new HashMap<>();
        for (int j = 0; j < patLen; j++) {
            badCharHeuristic.put(pattern.charAt(j), j);
        }

        if(patLen==0)
        {
            for(int j=0;j<=textLen;++j)
            {
                positions.add(j);
            }
            return positions;
        }

        while (i <= textLen - patLen) {
            int mismatchIndex = mismatchInText(i);
            if (mismatchIndex == -1) {
                positions.add(i);
                i += patLen;
            } else {
                char mismatchChar = text.charAt(mismatchIndex);
                int matchIndex = badCharHeuristic.getOrDefault(mismatchChar, -1);
                if (matchIndex >= 0) {
                    i += Math.max(1, mismatchIndex - i - matchIndex);
                }
                else {
                    i += mismatchIndex - i + 1;
                }
            }
        }
        return positions;
    }

The Boyer-Moore algorithm efficiently finds all occurrences of a pattern within a text by leveraging the bad character heuristic, which allows skipping sections of the text that cannot possibly match the pattern.

### Approach
1. **Initialization**: The `BoyerMooreSearch` class is initialized with the text and pattern strings, along with their lengths.
2. **Mismatch Detection**: The `mismatchInText` method checks for a mismatch between the pattern and a substring of the text starting at a given position. It scans the pattern from right to left and returns the position of the first mismatch encountered in the text.
3. **Bad Character Heuristic**: The `badCharacterHeuristic` method precomputes a dictionary that maps each character in the pattern to its last occurrence index. This dictionary helps in determining how far the pattern can be shifted when a mismatch occurs.
4. **Pattern Matching**: The main loop processes the text to find all starting positions where the pattern matches:
   - **Exact Match**: If no mismatch is found (`mismatchIndex` is -1), the starting position is recorded, and the pattern is shifted by its full length to avoid overlapping matches.
   - **Mismatch Handling**: If a mismatch is found:
     - The bad character heuristic is used to compute the shift amount. If the mismatched character exists in the pattern, the shift is calculated to align the last occurrence of this character in the pattern with the mismatched position in the text. If the character is not in the pattern, the pattern is shifted past the mismatched character.

```python
        patLen = self.patLen
        textLen = self.textLen
        if patLen == 0:
            return list(range(textLen + 1))
        
        badChar = {}
        for idx, char in enumerate(self.pattern):
            badChar[char] = idx
        
        positions = []
        i = 0
        while i <= textLen - patLen:
            mismatchIndex = self.mismatchInText(i)
            if mismatchIndex == -1:
                positions.append(i)
                i += patLen
            else:
                j = mismatchIndex - i
                char_text = self.text[mismatchIndex]
                matchIndex = badChar.get(char_text, -1)
                if matchIndex >= 0:
                    shift = j - matchIndex
                    i += max(1, shift)
                else:
                    i = mismatchIndex + 1
        return positions
```

### Explanation
- **Initialization**: The `__init__` method initializes the text and pattern strings along with their lengths.
- **Mismatch Detection**: The `mismatchInText` method checks for mismatches by comparing the pattern against a substring of the text starting at `currentPos`. It scans from the end of the pattern to the beginning and returns the position of the first mismatch in the text, or -1 if no mismatch is found.
- **Bad Character Heuristic Setup**: The `badCharacterHeuristic` method first handles the edge case where the pattern is empty, returning all possible positions in the text. For non-empty patterns, it builds a dictionary (`badChar`) that records the last occurrence index of each character in the pattern.
- **Pattern Matching Loop**: The loop processes the text:
  - **Exact Match**: When `mismatchInText` returns -1, the current position `i` is added to the result list, and `i` is incremented by the pattern length to skip overlapping matches.
  - **Mismatch Handling**: If a mismatch occurs:
    - The position `j` within the pattern is calculated.
    - The mismatched character from the text is looked up in `badChar`. If found, the shift is computed as `j - matchIndex` (with a minimum shift of 1). If not found, the pattern is shifted past the mismatched character (`i = mismatchIndex + 1`).

This approach efficiently locates all occurrences of the pattern in the text by leveraging the bad character heuristic to minimize unnecessary comparisons, maintaining the same behavior as the original Java implementation.
