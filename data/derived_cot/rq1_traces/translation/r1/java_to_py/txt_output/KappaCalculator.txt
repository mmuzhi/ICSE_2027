package org.example;

import java.util.Arrays;

public class KappaCalculator {

    public static double kappa(int[][] testData, int k) {
        int[][] dataMat = testData;
        double P0 = 0.0;
        for (int i = 0; i < k; i++) {
            P0 += dataMat[i][i] * 1.0;
        }
        int[] xsum = new int[k];
        int[] ysum = new int[k];
        int sum = 0;
        for (int i = 0; i < k; i++) {
            for (int j = 0; j < k; j++) {
                xsum[i] += dataMat[i][j];
                ysum[j] += dataMat[i][j];
                sum += dataMat[i][j];
            }
        }
        double Pe = 0.0;
        for (int i = 0; i < k; i++) {
            Pe += ysum[i] * xsum[i];
        }
        Pe = Pe / (sum * sum);
        P0 = P0 / sum;
        return (P0 - Pe) / (1 - Pe);
    }

    public static double fleissKappa(int[][] testData, int N, int k, int n) {
        int[][] dataMat = testData;
        double[] P = new double[N];
        double sum = 0.0;
        for (int i = 0; i < N; i++) {
            double temp = 0.0;
            for (int j = 0; j < k; j++) {
                sum += dataMat[i][j];
                temp += dataMat[i][j] * dataMat[i][j];
            }
            temp -= n;
            temp /= (n - 1) * n;
            P[i] = temp;
        }
        double P0 = Arrays.stream(P).sum() / N;
        double[] pj = new double[k];
        for (int j = 0; j < k; j++) {
            for (int i = 0; i < N; i++) {
                pj[j] += dataMat[i][j];
            }
            pj[j] /= sum;
        }
        double Pe = Arrays.stream(pj).map(p -> p * p).sum();
        return (P0 - Pe) / (1 - Pe);
    }

    public static void main(String[] args) {
        System.out.println(kappa(new int[][]{{2, 1, 1}, {1, 2, 1}, {1, 1, 2}}, 3)); // 0.25
        System.out.println(fleissKappa(new int[][]{{0, 0, 0, 0, 14},
                {0, 2, 6, 4, 2},
                {0, 0, 3, 5, 6},
                {0, 3, 9, 2, 0},
                {2, 2, 8, 1, 1},
                {7, 7, 0, 0, 0},
                {3, 2, 6, 3, 0},
                {2, 5, 3, 2, 2},
                {6, 5, 2, 1, 0},
                {0, 2, 2, 3, 7}}, 10, 5, 14));
    }
}

To solve this problem, we need to translate two Java functions (`kappa` and `fleissKappa`) that calculate different types of kappa coefficients (Cohen's Kappa and Fleiss' Kappa) into Python. The goal is to maintain identical behavior, including inputs, outputs, side effects, edge cases, and exceptions, while using idiomatic Python only when it doesn't alter the behavior.

### Approach
1. **Cohen's Kappa (`kappa`)**:
   - **Problem Analysis**: Cohen's Kappa measures inter-rater agreement for categorical items. It uses a confusion matrix where rows represent the raters' decisions and columns represent the ground truth.
   - **Steps**:
     - Calculate `P0` as the sum of diagonal elements divided by the total number of observations.
     - Compute row sums (`xsum`) and column sums (`ysum`) of the confusion matrix.
     - Calculate `Pe` (expected agreement) as the sum of the products of corresponding row and column sums divided by the square of the total observations.
     - The kappa coefficient is then derived as `(P0 - Pe) / (1 - Pe)`.

2. **Fleiss' Kappa (`fleissKappa`)**:
   - **Problem Analysis**: Fleiss' Kappa measures inter-rater agreement among multiple raters when categorizing items. It uses a matrix where each row represents a subject and each column represents a category.
   - **Steps**:
     - For each subject, compute `P[i]` which is the proportion of rater pairs that agree.
     - Calculate `P0` as the average of all `P[i]` values.
     - Compute `pj`, the proportion of all assignments to each category.
     - Calculate `Pe` as the sum of squares of `pj` values.
     - The kappa coefficient is derived similarly as `(P0 - Pe) / (1 - Pe)`.

def kappa(testData, k):
    dataMat = testData
    P0 = 0.0
    for i in range(k):
        P0 += dataMat[i][i] * 1.0
    
    xsum = [0] * k
    ysum = [0] * k
    total_sum = 0
    for i in range(k):
        for j in range(k):
            xsum[i] += dataMat[i][j]
            ysum[j] += dataMat[i][j]
            total_sum += dataMat[i][j]
            
    Pe = 0.0
    for i in range(k):
        Pe += ysum[i] * xsum[i]
    Pe = Pe / (total_sum * total_sum)
    
    P0 = P0 / total_sum
    return (P0 - Pe) / (1 - Pe)

    P = [0.0] * N

### Explanation
- **Cohen's Kappa (`kappa`)**:
  - The function starts by initializing `P0` to accumulate the sum of diagonal elements of the confusion matrix.
  - It then computes row sums (`xsum`) and column sums (`ysum`) while accumulating the total observations (`total_sum`).
  - The expected agreement (`Pe`) is calculated by summing the products of corresponding row and column sums, divided by the square of the total observations.
  - The kappa coefficient is derived by normalizing `P0` and applying the formula `(P0 - Pe) / (1 - Pe)`.

- **Fleiss' Kappa (`fleissKappa`)**:
  - For each subject (row), the function computes the proportion of agreeing rater pairs (`P[i]`) by leveraging the sum of squares of category counts.
  - The overall observed agreement (`P0`) is the average of all `P[i]` values.
  - The proportion of assignments to each category (`pj`) is computed by summing each column and dividing by the total assignments.
  - The expected agreement (`Pe`) is the sum of squares of `pj` values.
  - The kappa coefficient is then derived similarly as `(P0 - Pe) / (1 - Pe)`.
