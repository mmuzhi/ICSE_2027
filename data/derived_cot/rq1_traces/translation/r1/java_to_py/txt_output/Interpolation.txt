package org.example;

import java.util.ArrayList;
import java.util.List;

public class Interpolation {

    public static List<Double> interpolate1d(List<Double> x, List<Double> y, List<Double> xInterp) {
        List<Double> yInterp = new ArrayList<>();
        for (Double xi : xInterp) {
            for (int i = 0; i < x.size() - 1; i++) {
                if (x.get(i) <= xi && xi <= x.get(i + 1)) {
                    double yi = y.get(i) + (y.get(i + 1) - y.get(i)) * (xi - x.get(i)) / (x.get(i + 1) - x.get(i));
                    yInterp.add(yi);
                    break;
                }
            }
        }
        return yInterp;
    }

    public static List<Double> interpolate2d(List<Double> x, List<Double> y, List<List<Double>> z, List<Double> xInterp, List<Double> yInterp) {
        List<Double> zInterp = new ArrayList<>();
        for (int k = 0; k < xInterp.size(); k++) {
            Double xi = xInterp.get(k);
            Double yi = yInterp.get(k);
            for (int i = 0; i < x.size() - 1; i++) {
                if (x.get(i) <= xi && xi <= x.get(i + 1)) {
                    for (int j = 0; j < y.size() - 1; j++) {
                        if (y.get(j) <= yi && yi <= y.get(j + 1)) {
                            double z00 = z.get(i).get(j);
                            double z01 = z.get(i).get(j + 1);
                            double z10 = z.get(i + 1).get(j);
                            double z11 = z.get(i + 1).get(j + 1);
                            double zi = (z00 * (x.get(i + 1) - xi) * (y.get(j + 1) - yi) +
                                    z10 * (xi - x.get(i)) * (y.get(j + 1) - yi) +
                                    z01 * (x.get(i + 1) - xi) * (yi - y.get(j)) +
                                    z11 * (xi - x.get(i)) * (yi - y.get(j))) /
                                    ((x.get(i + 1) - x.get(i)) * (y.get(j + 1) - y.get(j)));
                            zInterp.add(zi);
                            break;
                        }
                    }
                    break;
                }
            }
        }
        return zInterp;
    }
}

=== RAW ANSWER ===

To solve this problem, we need to translate two Java methods for interpolation into Python while maintaining identical behavior. The first method performs 1D linear interpolation, and the second method performs 2D bilinear interpolation. The key aspects to preserve are the input-output behavior, handling of edge cases, and loop control structures.

### Approach
1. **1D Linear Interpolation (`interpolate1d`)**:
   - For each interpolation point `xi` in `xInterp`, find the interval `[x[i], x[i+1]]` that contains `xi`.
   - Compute the interpolated value `yi` using linear interpolation between `y[i]` and `y[i+1]`.
   - Add `yi` to the result list and break out of the loop once the interval is found.

2. **2D Bilinear Interpolation (`interpolate2d`)**:
   - For each pair of interpolation points `(xi, yi)` in `xInterp` and `yInterp`, find the grid cell `[x[i], x[i+1]] x [y[j], y[j+1]]` that contains `(xi, yi)`.
   - Compute the interpolated value `zi` using bilinear interpolation with the four corner values of the grid cell.
   - Add `zi` to the result list and break out of the loops once the cell is found.

def interpolate1d(x, y, xInterp):
    yInterp = []

### Explanation
- **1D Interpolation**: The method iterates over each point in `xInterp`. For each point, it checks each interval in `x` to find the enclosing segment. Once found, it computes the interpolated value using linear interpolation and breaks out of the loop to move to the next point.
- **2D Interpolation**: The method processes each pair of points `(xi, yi)` from `xInterp` and `yInterp`. It searches for the grid cell defined by `x` and `y` that contains the point. Upon finding the cell, it performs bilinear interpolation using the four corner values of the cell, appends the result, and breaks out of the loops to proceed to the next point.
- **Edge Cases**: If no interval or grid cell is found for a point, the point is skipped, and no value is added to the result list, mirroring the Java behavior. This ensures that the output lists may be shorter than the input lists if some points are outside the defined interpolation ranges.
