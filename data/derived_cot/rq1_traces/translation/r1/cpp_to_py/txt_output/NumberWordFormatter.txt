std::string NumberWordFormatter::trans_two(const std::string& s) {
    std::string ss = s;
    if (ss.size() == 1) ss = "0" + ss;

    if (ss[0] == '0') {
        return NUMBER[ss[1] - '0'];
    }
    else if (ss[0] == '1') {
        return NUMBER_TEEN[std::stoi(ss) - 10];
    }
    else if (ss[1] == '0') {
        return NUMBER_TEN[ss[0] - '1'];
    }
    else {
        return NUMBER_TEN[ss[0] - '1'] + " " + NUMBER[ss[1] - '0'];
    }
}

std::string NumberWordFormatter::trans_three(const std::string& s) {
    if (s[0] == '0') {
        return trans_two(s.substr(1));
    }
    else if (s.substr(1) == "00") {
        return NUMBER[s[0] - '0'] + " HUNDRED";
    }
    else {
        return NUMBER[s[0] - '0'] + " HUNDRED AND " + trans_two(s.substr(1));
    }
}

std::string NumberWordFormatter::parse_more(int i) {
    return NUMBER_MORE[i];
}

### Explanation
1. **Initialization**: The `__init__` method sets up lists for number words, teen numbers, tens, and larger units (thousand, million, etc.), mirroring the C++ constants.
2. **Format Dispatch**: The `format` method checks the input type and routes it to the appropriate handler (`format_int`, `format_double`, `format_string`, or `format_null`).
3. **String Processing**: The `format_string` method splits the input into integer and fractional parts. The integer part is reversed, padded to a multiple of three digits, and split into chunks. Each chunk is processed into words and combined with units (thousand, million). Trailing spaces are trimmed from the result.
4. **Fractional Handling**: The fractional part (if present) is converted to words with "AND CENTS" and appended to the result.
5. **Helper Methods**: 
   - `trans_two` converts two-digit strings to words, handling teens and tens.
   - `trans_three` processes three-digit strings, incorporating hundreds and tens.
   - `parse_more` provides the unit suffix (e.g., "THOUSAND") for each chunk index.
6. **Edge Cases**: Zero integer parts result in "ZERO ONLY", and `None` inputs return an empty string, consistent with the C++ behavior.
