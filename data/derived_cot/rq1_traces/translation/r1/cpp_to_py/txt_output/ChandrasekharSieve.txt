#include <vector>

class ChandrasekharSieve {
public:
    ChandrasekharSieve(int n);
    std::vector<int> get_primes() const;

private:
    int n;
    std::vector<int> primes;

    std::vector<int> generate_primes();
};

ChandrasekharSieve::ChandrasekharSieve(int n) : n(n) {
    primes = generate_primes();
}

std::vector<int> ChandrasekharSieve::generate_primes() {
    if (n < 2) {
        return {};
    }

    std::vector<bool> sieve(n + 1, true);
    sieve[0] = sieve[1] = false;

    int p = 2;
    while (p * p <= n) {
        if (sieve[p]) {
            for (int i = p * p; i <= n; i += p) {
                sieve[i] = false;
            }
        }
        p++;
    }

    std::vector<int> primes;
    for (int i = 2; i <= n; i++) {
        if (sieve[i]) {
            primes.push_back(i);
        }
    }

    return primes;
}

std::vector<int> ChandrasekharSieve::get_primes() const {
    return primes;
}

### Approach
The Sieve of Eratosthenes algorithm efficiently finds all prime numbers up to a specified integer `n` by iteratively marking the multiples of each prime starting from the first prime number (2). The steps involved are:
1. **Initialization**: Create a boolean list `sieve` of size `n+1` initialized to `True`, where each index represents whether the number is prime.
2. **Mark Non-Primes**: Set `sieve[0]` and `sieve[1]` to `False` since 0 and 1 are not primes.
3. **Sieve Process**: Starting from the first prime number (2), mark all multiples of each prime as non-prime. This is done by iterating through each number up to the square root of `n`.
4. **Collect Primes**: After processing, collect all indices from 2 to `n` that remain marked as `True` in the sieve list, which are the prime numbers.

### Solution Code
```python
class ChandrasekharSieve:
    def __init__(self, n):
        self.n = n
        self.primes = self.generate_primes()
    
    def generate_primes(self):
        if self.n < 2:
            return []
        
        sieve = [True] * (self.n + 1)
        sieve[0] = False
        sieve[1] = False
        
        p = 2
        while p * p <= self.n:
            if sieve[p]:
                for i in range(p * p, self.n + 1, p):
                    sieve[i] = False
            p += 1
        
        primes = []
        for i in range(2, self.n + 1):
            if sieve[i]:
                primes.append(i)
        return primes
    
    def get_primes(self):
        return self.primes
```

1. **Initialization**: The `ChandrasekharSieve` class is initialized with an integer `n`. The constructor immediately calls the `generate_primes` method to populate the `primes` list.
2. **Prime Generation**:
   - **Edge Case Handling**: If `n` is less than 2, return an empty list since there are no primes below 2.
   - **Sieve Setup**: A boolean list `sieve` of size `n+1` is created, initialized to `True`. The first two elements (0 and 1) are set to `False`.
   - **Marking Multiples**: For each number `p` starting from 2 up to the square root of `n`, if `p` is marked as prime, all multiples of `p` starting from `p*p` are marked as non-prime.
   - **Collecting Primes**: After processing, iterate through the sieve list from 2 to `n`, collecting all indices that remain `True` into the `primes` list.
3. **Access Method**: The `get_primes` method returns the generated list of primes.

This approach ensures that the Python implementation behaves identically to the original C++ code, handling all edge cases and producing the same output for any given input `n`. The algorithm efficiently generates primes using the Sieve of Eratosthenes method, optimized by starting the marking of multiples from `p*p`.
