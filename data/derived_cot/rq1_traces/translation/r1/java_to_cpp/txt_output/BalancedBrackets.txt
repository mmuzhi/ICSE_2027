=== SOURCE CODE (java) ===
package org.example;

import java.util.Stack;

public class BalancedBrackets {
    private Stack<Character> stack;
    private String leftBrackets;
    private String rightBrackets;
    private String expr;

    public BalancedBrackets(String expr) {
        this.stack = new Stack<>();
        this.leftBrackets = "({[";
        this.rightBrackets = ")}]";
        this.expr = expr;
    }

    public void clearExpr() {
        StringBuilder cleanedExpr = new StringBuilder();
        for (char c : expr.toCharArray()) {
            if (leftBrackets.indexOf(c) != -1 || rightBrackets.indexOf(c) != -1) {
                cleanedExpr.append(c);
            }
        }
        this.expr = cleanedExpr.toString();
    }

    public boolean checkBalancedBrackets() {
        clearExpr();
        for (char Brkt : expr.toCharArray()) {
            if (leftBrackets.indexOf(Brkt) != -1) {
                stack.push(Brkt);
            } else {
                if (stack.isEmpty()) {
                    return false;
                }
                char currentBrkt = stack.pop();
                if (currentBrkt == '(' && Brkt != ')') {
                    return false;
                }
                if (currentBrkt == '{' && Brkt != '}') {
                    return false;
                }
                if (currentBrkt == '[' && Brkt != ']') {
                    return false;
                }
            }
        }
        return stack.isEmpty();
    }

    public String getExpr() {
        return expr;
    }

    public static void main(String[] args) {
        BalancedBrackets b = new BalancedBrackets("a(b)c");
        System.out.println(b.checkBalancedBrackets());
    }
}

=== RAW ANSWER ===

To solve this problem, we need to translate a Java class that checks for balanced brackets in a given string into an equivalent C++ class. The solution must preserve the exact behavior, including input/output handling, side effects, and edge cases. 

### Approach
1. **Class Structure**: The C++ class `BalancedBrackets` mirrors the Java class structure, with private member variables for the stack, left and right bracket characters, and the input expression.
2. **Cleaning Expression**: The `clearExpr` method processes the input string to retain only bracket characters (both left and right brackets), removing all other characters.
3. **Checking Balanced Brackets**: The `checkBalancedBrackets` method processes the cleaned expression:
   - **Left Brackets**: When encountering a left bracket (`(`, `{`, `[`), it is pushed onto the stack.
   - **Right Brackets**: When encountering a right bracket (`)`, `}`, `]`), the method checks if the stack is empty (indicating unbalanced brackets) or if the top of the stack does not match the corresponding left bracket.
4. **Stack Emptiness Check**: After processing all characters, the method checks if the stack is empty to ensure all opened brackets have been closed properly.
