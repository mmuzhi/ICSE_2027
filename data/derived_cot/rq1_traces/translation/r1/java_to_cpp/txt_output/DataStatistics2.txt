=== SOURCE CODE (java) ===
package org.example;

import java.util.List;
import java.util.stream.DoubleStream;

public class DataStatistics2 {
    private double[] data;

    public DataStatistics2(List<Integer> data) {
        this.data = data.stream().mapToDouble(Integer::doubleValue).toArray();
    }

    public double getSum() {
        return DoubleStream.of(data).sum();
    }

    public double getMin() {
        return DoubleStream.of(data).min().orElse(Double.NaN);
    }

    public double getMax() {
        return DoubleStream.of(data).max().orElse(Double.NaN);
    }

    public double getVariance() {
        double mean = getSum() / data.length;
        return DoubleStream.of(data)
                .map(val -> (val - mean) * (val - mean))
                .average()
                .orElse(Double.NaN);
    }

    public double getStdDeviation() {
        return Math.sqrt(getVariance());
    }

    public double getCorrelation() {
        return 1.0;
    }
}

=== RAW ANSWER ===

To solve this problem, we need to translate a Java class into a C++ class while maintaining identical behavior. The Java class performs various statistical operations on a list of integers converted to doubles. The key operations include calculating the sum, minimum, maximum, variance, standard deviation, and correlation of the data.

### Approach
1. **Data Storage**: Convert the input list of integers into a vector of doubles for efficient processing.
2. **Sum Calculation**: Iterate through the vector and accumulate the sum of all elements.
3. **Minimum and Maximum**: Use standard library functions (`std::min_element` and `std::max_element`) to find the minimum and maximum values in the vector.
4. **Variance Calculation**: Compute the mean of the data, then calculate the average of the squared differences from the mean.
5. **Standard Deviation**: Simply take the square root of the variance.
6. **Correlation**: As per the original Java code, always return 1.0.

### Solution Code
```cpp
#include <vector>
#include <cmath>
#include <algorithm>
#include <numeric>

class DataStatistics2 {
private:
    std::vector<double> data;

public:
    DataStatistics2(const std::vector<int>& input) {
        data.reserve(input.size());
        for (int val : input) {
            data.push_back(static_cast<double>(val));
        }
    }

    double getSum() const {
        return std::accumulate(data.begin(), data.end(), 0.0);
    }

    double getMin() const {
        if (data.empty()) {
            return std::nan("");
        }
        return *std::min_element(data.begin(), data.end());
    }

    double getMax() const {
        if (data.empty()) {
            return std::nan("");
        }
        return *std::max_element(data.begin(), data.end());
    }

    double getVariance() const {
        if (data.empty()) {
            return std::nan("");
        }
        double mean = getSum() / data.size();
        double sum_sq = 0.0;
        for (double val : data) {
            double diff = val - mean;
            sum_sq += diff * diff;
        }
        return sum_sq / data.size();
    }

    double getStdDeviation() const {
        return std::sqrt(getVariance());
    }

    double getCorrelation() const {
        return 1.0;
    }
};
```

### Explanation
1. **Constructor**: Converts the input list of integers into a vector of doubles using `static_cast<double>` for each element.
2. **getSum**: Uses `std::accumulate` to sum all elements in the vector. The initial value is `0.0` to ensure floating-point arithmetic.
3. **getMin and getMax**: Check if the vector is empty and return `NaN` if true. Otherwise, use `std::min_element` and `std::max_element` to find the respective values.
4. **getVariance**: Computes the mean of the data. For each element, calculates the squared difference from the mean and averages these squared differences.
5. **getStdDeviation**: Simply returns the square root of the variance.
6. **getCorrelation**: Returns 1.0 as specified in the original Java code.

This approach ensures that the C++ implementation behaves identically to the original Java code, handling edge cases such as empty input data appropriately.
