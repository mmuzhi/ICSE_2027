package org.example;

import java.util.ArrayList;
import java.util.HashMap;
import java.util.List;
import java.util.Map;

public class VectorUtil {

    public static double similarity(double[] vector1, double[] vector2) {
        double dotProduct = 0.0;
        double norm1 = 0.0;
        double norm2 = 0.0;
        for (int i = 0; i < vector1.length; i++) {
            dotProduct += vector1[i] * vector2[i];
            norm1 += Math.pow(vector1[i], 2);
            norm2 += Math.pow(vector2[i], 2);
        }
        double denominator = Math.sqrt(norm1) * Math.sqrt(norm2);
        return denominator == 0 ? 0.0 : dotProduct / denominator;
    }

    public static List<Double> cosineSimilarities(double[] vector1, List<double[]> vectorsAll) {
        List<Double> similarities = new ArrayList<>();
        double norm1 = Math.sqrt(dotProduct(vector1, vector1));
        if (norm1 == 0) {
            for (int i = 0; i < vectorsAll.size(); i++) {
                similarities.add(0.0);
            }
            return similarities;
        }
        for (double[] vector2 : vectorsAll) {
            double norm2 = Math.sqrt(dotProduct(vector2, vector2));
            if (norm2 == 0) {
                similarities.add(0.0);
            } else {
                double similarity = dotProduct(vector1, vector2) / (norm1 * norm2);
                similarities.add(similarity);
            }
        }
        return similarities;
    }

    public static double nSimilarity(List<double[]> vectorList1, List<double[]> vectorList2) {
        if (vectorList1.isEmpty() || vectorList2.isEmpty()) {
            throw new IllegalArgumentException("At least one of the passed lists is empty.");
        }
        double[] avgVector1 = averageVector(vectorList1);
        double[] avgVector2 = averageVector(vectorList2);
        return similarity(avgVector1, avgVector2);
    }

    public static Map<String, Double> computeIdfWeightDict(int totalNum, Map<String, Double> numberDict) {
        Map<String, Double> result = new HashMap<>();
        for (Map.Entry<String, Double> entry : numberDict.entrySet()) {
            double count = entry.getValue();
            double weight = Math.log((totalNum + 1) / (count + 1));
            result.put(entry.getKey(), weight);
        }
        return result;
    }

    private static double dotProduct(double[] vector1, double[] vector2) {
        double result = 0.0;
        for (int i = 0; i < vector1.length; i++) {
            result += vector1[i] * vector2[i];
        }
        return result;
    }

    private static double[] averageVector(List<double[]> vectors) {
        double[] avgVector = new double[vectors.get(0).length];
        for (double[] vector : vectors) {
            for (int i = 0; i < vector.length; i++) {
                avgVector[i] += vector[i];
            }
        }
        for (int i = 0; i < avgVector.length; i++) {
            avgVector[i] /= vectors.size();
        }
        return avgVector;
    }
}

=== RAW ANSWER ===

To solve this problem, we need to translate Java code that calculates vector similarities and other vector operations into equivalent C++ code. The goal is to maintain identical behavior, including handling edge cases, inputs/outputs, and exceptions exactly as in the original Java code.

### Approach
1. **Data Structures Translation**: 
   - Replace Java `double[]` with `std::vector<double>` for dynamic arrays.
   - Replace Java `List<double[]>` with `std::vector<std::vector<double>>` for lists of vectors.
   - Replace Java `Map<String, Double>` with `std::unordered_map<std::string, double>` for key-value pairs.

2. **Functionality Preservation**:
   - Implement static methods within a `VectorUtil` class to mirror the Java static methods.
   - Ensure mathematical operations (like `pow`, `sqrt`, `log`) use C++'s `<cmath>` equivalents.
   - Handle edge cases such as zero denominators by returning 0.0, and throw exceptions for invalid inputs.

3. **Algorithm Consistency**:
   - Maintain the same algorithms for vector operations (dot product, cosine similarity, IDF weight calculation) without introducing optimizations that could alter behavior.

### Explanation
1. **similarity**: Computes the cosine similarity between two vectors. If the denominator (product of norms) is zero, returns 0.0 to avoid division by zero.
2. **cosineSimilarities**: Computes cosine similarities between a given vector and a list of vectors. If the norm of the given vector is zero, returns a list of zeros.
3. **nSimilarity**: Computes the cosine similarity between the average vectors of two lists of vectors. Throws an exception if either list is empty.
4. **computeIdfWeightDict**: Calculates IDF weights for terms based on their counts in a corpus.
5. **dotProduct**: Helper function to compute the dot product of two vectors.
6. **averageVector**: Helper function to compute the average vector from a list of vectors, assuming all vectors are of the same length.

This C++ implementation faithfully replicates the behavior of the original Java code, ensuring identical results for all inputs and edge cases.
