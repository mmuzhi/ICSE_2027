package org.example;

import java.util.Arrays;

public class KappaCalculator {

    public static double kappa(int[][] testData, int k) {
        int[][] dataMat = testData;
        double P0 = 0.0;
        for (int i = 0; i < k; i++) {
            P0 += dataMat[i][i] * 1.0;
        }
        int[] xsum = new int[k];
        int[] ysum = new int[k];
        int sum = 0;
        for (int i = 0; i < k; i++) {
            for (int j = 0; j < k; j++) {
                xsum[i] += dataMat[i][j];
                ysum[j] += dataMat[i][j];
                sum += dataMat[i][j];
            }
        }
        double Pe = 0.0;
        for (int i = 0; i < k; i++) {
            Pe += ysum[i] * xsum[i];
        }
        Pe = Pe / (sum * sum);
        P0 = P0 / sum;
        return (P0 - Pe) / (1 - Pe);
    }

    public static double fleissKappa(int[][] testData, int N, int k, int n) {
        int[][] dataMat = testData;
        double[] P = new double[N];
        double sum = 0.0;
        for (int i = 0; i < N; i++) {
            double temp = 0.0;
            for (int j = 0; j < k; j++) {
                sum += dataMat[i][j];
                temp += dataMat[i][j] * dataMat[i][j];
            }
            temp -= n;
            temp /= (n - 1) * n;
            P[i] = temp;
        }
        double P0 = Arrays.stream(P).sum() / N;
        double[] pj = new double[k];
        for (int j = 0; j < k; j++) {
            for (int i = 0; i < N; i++) {
                pj[j] += dataMat[i][j];
            }
            pj[j] /= sum;
        }
        double Pe = Arrays.stream(pj).map(p -> p * p).sum();
        return (P0 - Pe) / (1 - Pe);
    }

    public static void main(String[] args) {
        System.out.println(kappa(new int[][]{{2, 1, 1}, {1, 2, 1}, {1, 1, 2}}, 3)); // 0.25
        System.out.println(fleissKappa(new int[][]{{0, 0, 0, 0, 14},
                {0, 2, 6, 4, 2},
                {0, 0, 3, 5, 6},
                {0, 3, 9, 2, 0},
                {2, 2, 8, 1, 1},
                {7, 7, 0, 0, 0},
                {3, 2, 6, 3, 0},
                {2, 5, 3, 2, 2},
                {6, 5, 2, 1, 0},
                {0, 2, 2, 3, 7}}, 10, 5, 14));
    }
}

=== RAW ANSWER ===

To solve this problem, we need to translate Java code for calculating Cohen's Kappa and Fleiss' Kappa into C++ while maintaining identical behavior. The solution involves handling 2D arrays, performing arithmetic operations, and ensuring the output matches the Java implementation.

### Approach
1. **Problem Analysis**: The Java code consists of two functions:
   - `kappa`: Computes Cohen's Kappa for a square matrix of size `k x k`.
   - `fleissKappa`: Computes Fleiss' Kappa for an `N x k` matrix where `N` is the number of subjects, `k` is the number of categories, and `n` is the number of raters per subject.
2. **Key Steps**:
   - **Cohen's Kappa**: 
     - Calculate `P0` as the sum of diagonal elements divided by the total sum of all elements.
     - Compute row and column sums (`xsum` and `ysum`).
     - Calculate `Pe` as the sum of products of corresponding row and column sums divided by the square of the total sum.
     - Return `(P0 - Pe) / (1 - Pe)`.
   - **Fleiss' Kappa**:
     - For each subject, compute the sum of squares of counts, adjust by subtracting `n`, and scale by `(n-1)*n` to get `P[i]`.
     - Average `P[i]` across subjects to get `P0`.
     - Compute category proportions `pj` by summing counts per category and dividing by the total sum.
     - Calculate `Pe` as the sum of squares of `pj`.
     - Return `(P0 - Pe) / (1 - Pe)`.
3. **Edge Cases**: Handle division by zero by relying on C++ floating-point behavior (e.g., returning NaN).

### Explanation
1. **Cohen's Kappa**:
   - **P0 Calculation**: Sum of diagonal elements divided by total elements.
   - **Row and Column Sums**: Computed to understand the distribution of agreements.
   - **Pe Calculation**: Expected agreement based on row and column sums.
   - **Kappa Formula**: `(P0 - Pe) / (1 - Pe)` measures agreement beyond chance.
2. **Fleiss' Kappa**:
   - **Per Subject Calculation**: For each subject, compute agreement among raters adjusted by the number of raters.
   - **P0 Calculation**: Average agreement across subjects.
   - **Category Proportions (pj)**: Proportion of assignments per category.
   - **Pe Calculation**: Expected agreement based on category proportions.
   - **Kappa Formula**: Same as Cohen's but applied to multiple raters per subject.
3. **Main Function**: Tests both functions with sample inputs, printing results identical to the Java outputs. The use of vectors ensures dynamic array handling similar to Java arrays.
