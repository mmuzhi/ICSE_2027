    @staticmethod
    def add(states, stat):
        BitStatusUtil.check([states, stat])
        return states | stat

    @staticmethod
    def has(states, stat):
        BitStatusUtil.check([states, stat])
        return (states & stat) == stat

    @staticmethod
    def remove(states, stat):
        BitStatusUtil.check([states, stat])
        if BitStatusUtil.has(states, stat):
            return states ^ stat
        return states

    @staticmethod
    def check(args):
        for arg in args:
            if arg < 0:
                raise ValueError(f"{arg} must be greater than or equal to 0")
            if arg % 2 != 0:
                raise ValueError(f"{arg} not even")

To solve this problem, we need to translate a Python utility class that performs bitwise operations into C++ while maintaining identical behavior. The class includes methods to add, check, and remove statuses using bitwise operations, along with a method to validate input parameters.

### Approach
1. **Class Structure**: The `BitStatusUtil` class in C++ will contain static methods similar to the Python class.
2. **Parameter Validation**: The `check` method validates that all parameters are non-negative and even. If any parameter fails these checks, it throws an exception.
3. **Bitwise Operations**:
   - **Add**: Uses bitwise OR (`|`) to combine statuses.
   - **Has**: Uses bitwise AND (`&`) to check if a status is present.
   - **Remove**: Uses bitwise XOR (`^`) to remove a status if it is present.
4. **Exception Handling**: The `check` method throws `std::invalid_argument` exceptions for invalid parameters, mirroring Python's `ValueError`.

### Solution Code
```cpp
#include <vector>
#include <stdexcept>
#include <initializer_list>
#include <string>

class BitStatusUtil {
public:
    static int add(int states, int stat) {
        check({states, stat});
        return states | stat;
    }

    static bool has(int states, int stat) {
        check({states, stat});
        return (states & stat) == stat;
    }

    static int remove(int states, int stat) {
        check({states, stat});
        if (has(states, stat)) {
            return states ^ stat;
        }
        return states;
    }

    static void check(std::initializer_list<int> args) {
        for (int arg : args) {
            if (arg < 0) {
                throw std::invalid_argument(std::to_string(arg) + " must be greater than or equal to 0");
            }
            if (arg % 2 != 0) {
                throw std::invalid_argument(std::to_string(arg) + " not even");
            }
        }
    }
};
```

- **Class Methods**: All methods (`add`, `has`, `remove`, `check`) are static, allowing them to be called without instantiating the class.
- **Parameter Validation**: The `check` method uses an initializer list to accept multiple parameters. It checks each parameter for non-negativity and evenness, throwing appropriate exceptions if checks fail.
- **Bitwise Operations**:
  - **Add**: Combines two statuses using bitwise OR.
  - **Has**: Checks if a status is present using bitwise AND.
  - **Remove**: Removes a status using bitwise XOR if it is present; otherwise, returns the original status.
- **Exception Handling**: Exceptions include detailed messages indicating the cause of failure, ensuring behavior matches the Python implementation.

This approach ensures the C++ implementation behaves identically to the Python original, handling edge cases and exceptions in the same manner.
