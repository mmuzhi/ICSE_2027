    def add_dish(self, dish):
        for menu_dish in self.menu:
            if dish["dish"] == menu_dish["dish"]:
                if menu_dish["count"] < dish["count"]:
                    return False
                else:
                    menu_dish["count"] -= dish["count"]
                    break
        self.selected_dishes.append(dish)
        return True

    def calculate_total(self):
        total = 0
        for dish in self.selected_dishes:
            total += dish["price"] * dish["count"] * self.sales[dish["dish"]]
        return total

    def checkout(self):
        if len(self.selected_dishes) == 0:
            return False
        total = self.calculate_total()
        self.selected_dishes = []
        return total

### Approach
1. **Data Structures**: 
   - **DishItem Struct**: Represents a dish with attributes for name, price, and count.
   - **Order Class**: Manages the restaurant's menu, selected dishes, and sales discounts.
2. **Public Members**: 
   - `menu`: A list of available dishes.
   - `selected_dishes`: A list of dishes currently in the order.
   - `sales`: A map of dish names to their discount factors.
3. **Methods**:
   - **add_dish**: Adds a dish to the order if it is available in sufficient quantity in the menu. Adjusts the menu count accordingly.
   - **calculate_total**: Computes the total cost of the order by applying sales discounts to each dish.
   - **checkout**: Finalizes the order by returning the total cost if there are items; otherwise, returns a boolean `false`.
