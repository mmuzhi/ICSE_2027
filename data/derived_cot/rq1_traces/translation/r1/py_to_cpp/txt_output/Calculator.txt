        3

To solve this problem, we need to translate a Python calculator class that evaluates arithmetic expressions into C++. The calculator supports basic arithmetic operations (+, -, *, /, ^) and handles parentheses. The solution involves using stacks to process operands and operators according to their precedence, similar to the shunting yard algorithm.

### Approach
1. **Initialization**: The calculator initializes a map of operators to their corresponding arithmetic functions.
2. **Precedence Handling**: Each operator has a precedence level (priority), where higher values indicate higher priority. For example, exponentiation (^) has the highest precedence, followed by multiplication and division, then addition and subtraction.
3. **Expression Parsing**: The input expression is processed character by character:
   - **Digits and Decimal Points**: These are accumulated into a buffer to form complete numbers.
   - **Operators**: When encountered, the calculator processes higher or equal precedence operators from the operator stack before pushing the current operator.
   - **Parentheses**: Opening parentheses are pushed onto the stack. Closing parentheses trigger processing of all operators until the matching opening parenthesis is found.
4. **Operator Application**: Operators are applied by popping the top two operands and the top operator from their respective stacks, performing the operation, and pushing the result back onto the operand stack.
5. **Final Processing**: After parsing all characters, any remaining number buffer is pushed onto the operand stack. Remaining operators are processed to complete the evaluation.
6. **Result Extraction**: The result is the top value of the operand stack if the stack is not empty; otherwise, `std::nullopt` is returned.

### Solution Code
```cpp
#include <optional>
#include <stack>
#include <string>
#include <cctype>
#include <cmath>
#include <map>
#include <functional>
#include <vector>
#include <iostream>

class Calculator {
private:
    std::map<char, std::function<double(double, double)>> operators;

    int precedence(char op) {
        static std::map<char, int> prec = {
            {'+', 1}, {'-', 1},
            {'*', 2}, {'/', 2},
            {'^', 3}
        };
        auto it = prec.find(op);
        return (it != prec.end()) ? it->second : 0;
    }

    void apply_operator(std::stack<double>& operands, std::stack<char>& ops) {
        char op = ops.top();
        ops.pop();
        double operand2 = operands.top();
        operands.pop();
        double operand1 = operands.top();
        operands.pop();
        double result = operators[op](operand1, operand2);
        operands.push(result);
    }

public:
    Calculator() {
        operators = {
            {'+', [](double a, double b) { return a + b; }},
            {'-', [](double a, double b) { return a - b; }},
            {'*', [](double a, double b) { return a * b; }},
            {'/', [](double a, double b) { return a / b; }},
            {'^', [](double a, double b) { return std::pow(a, b); }}
        };
    }

    std::optional<double> calculate(const std::string& expression) {
        std::stack<double> operand_stack;
        std::stack<char> operator_stack;
        std::string num_buffer;

        for (char c : expression) {
            if (std::isdigit(static_cast<unsigned char>(c)) || c == '.') {
                num_buffer += c;
            } else {
                if (!num_buffer.empty()) {
                    operand_stack.push(std::stod(num_buffer));
                    num_buffer.clear();
                }

                if (operators.find(c) != operators.end()) {
                    while (!operator_stack.empty() && 
                           operator_stack.top() != '(' && 
                           precedence(operator_stack.top()) >= precedence(c)) {
                        apply_operator(operand_stack, operator_stack);
                    }
                    operator_stack.push(c);
                } else if (c == '(') {
                    operator_stack.push(c);
                } else if (c == ')') {
                    while (!operator_stack.empty() && operator_stack.top() != '(') {
                        apply_operator(operand_stack, operator_stack);
                    }
                    if (!operator_stack.empty() && operator_stack.top() == '(') {
                        operator_stack.pop();
                    }
                }
            }
        }

        if (!num_buffer.empty()) {
            operand_stack.push(std::stod(num_buffer));
        }

        while (!operator_stack.empty()) {
            apply_operator(operand_stack, operator_stack);
        }

        if (operand_stack.empty()) {
            return std::nullopt;
        }
        return operand_stack.top();
    }
};
```

2. **Precedence Handling**: The `precedence` method returns the priority level of each operator, ensuring correct order of operations during evaluation.
3. **Operator Application**: The `apply_operator` method pops the top operator and the top two operands from their stacks, applies the operator, and pushes the result back onto the operand stack.
4. **Expression Parsing**: The `calculate` method processes each character in the input string:
   - **Number Formation**: Digits and decimal points are accumulated into `num_buffer` until an operator or parenthesis is encountered, at which point the buffer is converted to a double and pushed onto the operand stack.
   - **Operator Processing**: Operators are pushed onto the operator stack only after processing any higher or equal precedence operators already on the stack.
   - **Parentheses Handling**: Opening parentheses are pushed onto the stack. Closing parentheses trigger processing of all operators until the matching opening parenthesis is found and removed.
5. **Final Processing**: After processing all characters, any remaining number in `num_buffer` is pushed onto the operand stack. Remaining operators are processed to complete the evaluation.
