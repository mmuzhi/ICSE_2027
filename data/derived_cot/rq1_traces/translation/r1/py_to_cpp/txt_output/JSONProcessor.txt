import json
import os


class JSONProcessor:
    def read_json(self, file_path):
        if not os.path.exists(file_path):
            return 0
        try:
            with open(file_path, 'r') as file:
                data = json.load(file)
            return data
        except:
            return -1

    def write_json(self, data, file_path):
        try:
            with open(file_path, 'w') as file:
                json.dump(data, file)
            return 1
        except:
            return -1

    def process_json(self, file_path, remove_key):
        data = self.read_json(file_path)
        if data == 0 or data == -1:
            return 0
        if remove_key in data:
            del data[remove_key]
            self.write_json(data, file_path)
            return 1
        else:
            return 0

### Explanation
1. **Reading JSON Files**: The `read_json` method uses `std::filesystem::exists` to check if the file exists. If not, it returns 0. It then attempts to parse the file using `nlohmann::json::parse`. If parsing fails, it catches any exception and returns -1. Successful parsing returns the JSON data.
2. **Writing JSON Files**: The `write_json` method opens the file for writing and outputs the JSON data in a compact format. If writing fails, it catches any exception and returns -1; otherwise, it returns 1.
3. **Processing JSON Files**: The `process_json` method first reads the JSON file. If the read operation fails (file doesn't exist or parsing error), it returns 0. It then checks if the JSON data is an object and contains the specified key. If so, it removes the key, writes the modified data back to the file (ignoring write success or failure), and returns 1. If the key is not found, it returns 0.

This approach ensures that the C++ implementation behaves identically to the original Python code, handling all edge cases and errors as specified. The use of `std::variant` allows the `read_json` method to return either an integer status code or the parsed JSON data, mimicking the dynamic typing of Python. #include <nlohmann/json.hpp>
#include <fstream>
#include <filesystem>
#include <variant>
#include <string>

namespace fs = std::filesystem;
using json = nlohmann::json;

class JSONProcessor {
public:
    std::variant<json, int> read_json(const std::string& file_path) {
        if (!fs::exists(file_path)) {
            return 0;
        }
        try {
            std::ifstream file(file_path);
            json data = json::parse(file);
            return data;
        } catch (...) {
            return -1;
        }
    }

    int write_json(const json& data, const std::string& file_path) {
        try {
            std::ofstream file(file_path);
            file << data;
            return 1;
        } catch (...) {
            return -1;
        }
    }

    int process_json(const std::string& file_path, const std::string& remove_key) {
        auto result = read_json(file_path);
        if (std::holds_alternative<int>(result)) {
            return 0;
        }
        json data = std::get<json>(result);
        if (data.is_object() && data.contains(remove_key)) {
            data.erase(remove_key);
            write_json(data, file_path);
            return 1;
        } else {
            return 0;
        }
    }
};
