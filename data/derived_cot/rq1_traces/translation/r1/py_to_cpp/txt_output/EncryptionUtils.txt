class EncryptionUtils:
    def __init__(self, key):
        self.key = key

    def caesar_cipher(self, plaintext, shift):
        ciphertext = ""
        for char in plaintext:
            if char.isalpha():
                if char.isupper():
                    ascii_offset = 65
                else:
                    ascii_offset = 97
                shifted_char = chr((ord(char) - ascii_offset + shift) % 26 + ascii_offset)
                ciphertext += shifted_char
            else:
                ciphertext += char
        return ciphertext
    
    def vigenere_cipher(self, plain_text):
        encrypted_text = ""
        key_index = 0
        for char in plain_text:
            if char.isalpha():
                shift = ord(self.key[key_index % len(self.key)].lower()) - ord('a')
                encrypted_char = chr((ord(char.lower()) - ord('a') + shift) % 26 + ord('a'))
                encrypted_text += encrypted_char.upper() if char.isupper() else encrypted_char
                key_index += 1
            else:
                encrypted_text += char
        return encrypted_text

    def rail_fence_cipher(self, plain_text, rails):
        fence = [['\n' for _ in range(len(plain_text))] for _ in range(rails)]
        direction = -1
        row, col = 0, 0

        for char in plain_text:
            if row == 0 or row == rails-1:
                direction = -direction

            fence[row][col] = char
            col += 1
            row += direction

        encrypted_text = ''
        for i in range(rails):
            for j in range(len(plain_text)):
                if fence[i][j] != '\n':
                    encrypted_text += fence[i][j]

        return encrypted_text

### Approach
1. **Caesar Cipher**: Each alphabetic character in the plaintext is shifted by a fixed number of positions down the alphabet. Non-alphabetic characters remain unchanged.
2. **Vigenere Cipher**: Each alphabetic character in the plaintext is shifted by a number of positions corresponding to the key character's position in the alphabet. The key repeats cyclically. Non-alphabetic characters remain unchanged and do not consume a key character.
3. **Rail Fence Cipher**: The plaintext is written in a zigzag pattern across a specified number of rails (rows) and then read row-wise to form the ciphertext.

        for (char c : plain_text) {
            if (std::isalpha(c)) {
                char key_char = std::tolower(this->key[key_index % this->key.length()]);
                int shift = key_char - 'a';
                char base = std::isupper(c) ? 'A' : 'a';

                if (std::isupper(c)) {

                encrypted_text += c;

        if (rails <= 1) {
            return plain_text;

        std::vector<std::vector<char>> fence(rails, std::vector<char>(plain_text.length(), '\n'));

                    encrypted_text += fence[i][j];
                }
            }
        }

### Explanation
1. **Caesar Cipher**:
   - For each character in the plaintext, if it is alphabetic, determine its case (upper or lower) to set the appropriate ASCII offset ('A' for uppercase, 'a' for lowercase).
   - Shift the character within the bounds of the alphabet (26 letters), handling negative shifts by adjusting modulo 26.
   - Non-alphabetic characters are added to the ciphertext unchanged.

2. **Vigenere Cipher**:
   - Iterate through each character in the plaintext. For alphabetic characters, use the corresponding key character (converted to lowercase) to determine the shift value.
   - Preserve the case of the plaintext character in the ciphertext. Non-alphabetic characters are added unchanged without advancing the key index.

3. **Rail Fence Cipher**:
   - Create a 2D vector (rails × length of plaintext) initialized with placeholder characters ('\n').
   - Traverse the plaintext in a zigzag pattern: move down the rails until the bottom rail is reached, then reverse direction to move up, and so on.
   - Construct the ciphertext by reading characters row-wise from the 2D vector, skipping placeholder characters.
