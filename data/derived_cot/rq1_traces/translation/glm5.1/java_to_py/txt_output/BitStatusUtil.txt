package org.example;

public class BitStatusUtil {

    public static int add(int states, int stat) {
        check(states, stat);
        return states | stat;
    }

    public static boolean has(int states, int stat) {
        check(states, stat);
        return (states & stat) == stat;
    }

    public static int remove(int states, int stat) {
        check(states, stat);
        if (has(states, stat)) {
            return states ^ stat;
        }
        return states;
    }

    public static void check(int... args) {
        for (int arg : args) {
            if (arg < 0) {
                throw new IllegalArgumentException(arg + " must be greater than or equal to 0");
            }
            if (arg % 2 != 0) {
                throw new IllegalArgumentException(arg + " not even");
            }
        }
    }

    public static void main(String[] args) {
        System.out.println(add(2, 4));
        System.out.println(has(6, 2));
        System.out.println(remove(6, 2));
        try {
            check(2, 3, 4);
        } catch (IllegalArgumentException e) {
            System.out.println(e.getMessage());
        }
    }
}

```python
def add(states: int, stat: int) -> int:
    check(states, stat)
    return states | stat


def has(states: int, stat: int) -> bool:
    check(states, stat)
    return (states & stat) == stat


def remove(states: int, stat: int) -> int:
    check(states, stat)
    if has(states, stat):
        return states ^ stat
    return states


def check(*args: int) -> None:
    for arg in args:
        if arg < 0:
            raise ValueError(f"{arg} must be greater than or equal to 0")
        if arg % 2 != 0:
            raise ValueError(f"{arg} not even")


def main() -> None:
    print(add(2, 4))
    print(has(6, 2))
    print(remove(6, 2))
    try:
        check(2, 3, 4)
    except ValueError as e:
        print(e)


if __name__ == "__main__":
    main()
```

### Key Translation Details:
- **Utility Class vs. Module Functions**: Java requires a class to hold static utility methods. In Python, it is idiomatic to use module-level functions, which preserves the exact same calling behavior without unnecessary boilerplate.
- **Varargs (`int... args`)**: Translated to Python's `*args`, which collects positional arguments into a tuple.
- **Exceptions**: Java's `IllegalArgumentException` maps perfectly to Python's built-in `ValueError`.
- **String Concatenation**: Java's `arg + " string"` is translated to Python's idiomatic f-strings `f"{arg} string"`, which guarantees the same output and type coercion.
- **Booleans**: Java's `true`/`false` becomes Python's `True`/`False`. The bitwise logic remains identical.
- **Exception Message Extraction**: Java's `e.getMessage()` translates to Python's `str(e)` (or just passing `e` to `print`), which prints the string message passed to the `ValueError` constructor.
