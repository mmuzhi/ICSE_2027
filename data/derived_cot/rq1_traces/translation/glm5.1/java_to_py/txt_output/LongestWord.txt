package org.example;

import java.util.ArrayList;
import java.util.List;
import java.util.regex.Pattern;

public class LongestWord {
    List<String> wordList;

    public LongestWord() {
        wordList = new ArrayList<>();
    }

    public void addWord(String word) {
        wordList.add(word);
    }

    public String findLongestWord(String sentence) {
        String longestWord = "";
        sentence = sentence.toLowerCase();
        sentence = sentence.replaceAll("[" + Pattern.quote("!\"#$%&'()*+,-./:;<=>?@[\\]^_`{|}~") + "]", "");
        String[] words = sentence.split(" ");
        for (String word : words) {
            if (wordList.contains(word) && word.length() > longestWord.length()) {
                longestWord = word;
            }
        }
        return longestWord;
    }

    public static void main(String[] args) {
        LongestWord longestWord = new LongestWord();
        longestWord.addWord("A");
        longestWord.addWord("aM");
        System.out.println(longestWord.findLongestWord("I am a student."));
    }
}

**Key translation notes:**

- **`wordList`** â†’ `self.word_list`: Python instance attribute, using snake_case convention.
- **`Pattern.quote(...)` + `replaceAll`**: Java removes all ASCII punctuation via regex. Python's `string.punctuation` contains the exact same character set (`!"#$%&'()*+,-./:;<=>?@[\]^_`{|}~`), and `str.translate` with `str.maketrans` removes them identically.
- **`split(" ")`**: Preserved as `split(" ")` (single space) to match Java's behavior of producing empty strings for consecutive spaces, rather than Python's default `split()` which collapses whitespace.
- **`wordList.contains(word)`** â†’ `word in self.word_list`: Python uses linear search for `list`, same as Java's `ArrayList.contains()`.
- **Case sensitivity preserved**: The Java code lowercases the sentence but *not* the words in `wordList`, so `"am"` won't match `"aM"`. This behavior is kept exactly.
