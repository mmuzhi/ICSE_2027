=== SOURCE CODE (java) ===
package org.example;

import java.util.*;
import java.util.stream.Collectors;

public class DataStatistics {

    public double mean(int[] data) {
        double sum = Arrays.stream(data).sum();
        return Math.round((sum / data.length) * 100.0) / 100.0;
    }

    public double median(int[] data) {
        int[] sortedData = Arrays.copyOf(data, data.length);
        Arrays.sort(sortedData);
        int n = sortedData.length;

        if (n % 2 == 0) {
            int middle = n / 2;
            return Math.round(((sortedData[middle - 1] + sortedData[middle]) / 2.0) * 100.0) / 100.0;
        } else {
            int middle = n / 2;
            return sortedData[middle];
        }
    }

    public List<Integer> mode(int[] data) {
        Map<Integer, Long> frequencyMap = Arrays.stream(data)
                .boxed()
                .collect(Collectors.groupingBy(e -> e, Collectors.counting()));

        long maxCount = Collections.max(frequencyMap.values());

        return frequencyMap.entrySet()
                .stream()
                .filter(entry -> entry.getValue() == maxCount)
                .map(Map.Entry::getKey)
                .sorted()
                .collect(Collectors.toList());
    }
}

### Key translation decisions:

- **`int[]` → `std::vector<int>`**: The idiomatic C++ equivalent. `mean` and `mode` take `const` references (no copy needed), while `median` takes by value — this naturally creates the copy that Java's `Arrays.copyOf` did explicitly, then sorts in-place.

- **`Arrays.stream(data).sum()` → `std::accumulate(..., 0.0)`**: Accumulating into a `double` (via the `0.0` initial value) matches Java's assignment of the int sum into a `double` variable.

- **`Math.round(x * 100.0) / 100.0` → `std::round(x * 100.0) / 100.0`**: Both use "round half away from zero" semantics, producing identical results.

- **`Collectors.groupingBy` + `Collectors.counting()` → `std::unordered_map` loop**: Manually counting frequencies replaces the Java stream pipeline. `long long` is used for counts to match Java's 64-bit `Long`.

- **Stream filter/map/sort/collect → explicit loop + `std::sort`**: The mode extraction logic is expressed imperatively, which is idiomatic C++ for this pattern.
