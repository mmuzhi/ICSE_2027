        } else {
            query.append("*");
        }
        query.append(" FROM ").append(table);

                    query.append(" AND ");

                first = false;

Key translation decisions:

1. **`LinkedHashMap` → `std::vector<std::pair<std::string, std::string>>`**: Preserves insertion order (the key behavioral property of `LinkedHashMap`), unlike `std::map` which sorts by key.

2. **Nullable `Map` parameters → `const OrderedMap*`**: Pointers can be `nullptr`, matching Java's null reference semantics. Nullable `String columns` → `const char*` for the same reason.

3. **`delete` → `delete_`**: `delete` is a C++ reserved keyword, so the method must be renamed.

4. **`String.split(",\\s*")` → `regexSplit`**: Uses `std::sregex_token_iterator` with `-1` submatch to extract non-matching segments. Trailing empty strings are removed to match Java's `String.split()` (limit=0) behavior.

5. **`StringBuilder` → `std::string` with `+=`**: Equivalent concatenation behavior; `std::string` has efficient appending in modern C++.

6. **`String.join` → manual `join` helper**: Replicates the delimiter-joined output.
