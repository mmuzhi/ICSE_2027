from collections import Counter


class DataStatistics:
    def mean(self, data):
        return round(sum(data) / len(data), 2)

    def median(self, data):
        sorted_data = sorted(data)
        n = len(sorted_data)
        if n % 2 == 0:
            middle = n // 2
            return round((sorted_data[middle - 1] + sorted_data[middle]) / 2, 2)
        else:
            middle = n // 2
            return sorted_data[middle]

    def mode(self, data):
        counter = Counter(data)
        mode_count = max(counter.values())
        mode = [x for x, count in counter.items() if count == mode_count]
        return mode

**Key translation notes:**

1. **`mean`**: Uses `std::accumulate` for summation and replicates Python's `round(x, 2)` via `std::round(x * 100.0) / 100.0`. (Note: Python uses banker's rounding while C++ uses round-half-away-from-zero; they differ only on exact `.005` boundaries.)

2. **`median`**: Takes `data` by value to allow in-place sorting (equivalent to Python's `sorted()` creating a copy). For even-length lists, averages the two middle elements and rounds to 2 decimals. 3. **`mode`**: Uses `std::unordered_map<double, int>` to replicate Python's `Counter`. Finds the max count, then collects all keys matching that count. The iteration order of `unordered_map` differs from Python's `dict`, so the output order of modes may vary (Python's `Counter` also has non-deterministic ordering across versions).
