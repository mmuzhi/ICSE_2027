class AvgPartition {
public:
    AvgPartition(const std::vector<int>& lst, int limit);
    std::pair<int, int> set_num() const;
    std::vector<int> get(int index) const;

private:
    std::vector<int> lst;
    int limit;
};
AvgPartition::AvgPartition(const std::vector<int>& lst, int limit)
    : lst(lst), limit(limit) {}

std::pair<int, int> AvgPartition::set_num() const {
    int size = lst.size() / limit;
    int remainder = lst.size() % limit;
    return { size, remainder };
}

std::vector<int> AvgPartition::get(int index) const {
    auto [size, remainder] = set_num();
    int start = index * size + std::min(index, remainder);
    int end = start + size;
    if (index + 1 <= remainder) {
        end += 1;
    }
    return std::vector<int>(lst.begin() + start, lst.begin() + end);
}

=== RAW ANSWER ===

To solve this problem, we need to translate a C++ class into Python while maintaining the same behavior, including handling edge cases and exceptions identically. The class `AvgPartition` manages a list of integers and a limit, providing methods to determine the partition size and retrieve specific partitions.

### Approach
1. **Initialization**: The Python class constructor takes a list and a limit, storing a copy of the list to avoid modifying the original input.
2. **Partition Calculation**: The `set_num` method calculates the number of elements per partition (`size`) and the remaining elements (`remainder`).
3. **Partition Retrieval**: The `get` method retrieves a specific partition based on the index. It calculates the start and end indices of the partition, adjusting for any remainder elements distributed across the partitions.

### Solution Code
```python
class AvgPartition:
    def __init__(self, lst, limit):
        self.lst = lst.copy()
        self.limit = limit

    def set_num(self):
        n = len(self.lst)
        if self.limit == 0:
            raise ZeroDivisionError("Limit cannot be zero")
        size = n // self.limit
        remainder = n % self.limit
        return (size, remainder)

    def get(self, index):
        size, remainder = self.set_num()
        start = index * size + min(index, remainder)
        end = start + size
        if index + 1 <= remainder:
            end += 1
        return self.lst[start:end]
```

### Explanation
1. **Initialization**: The constructor stores a copy of the input list and the limit. This ensures the original list remains unmodified.
2. **Partition Calculation**: The `set_num` method computes the partition size and remainder. If the limit is zero, it raises a `ZeroDivisionError`, mirroring undefined behavior in C++.
3. **Partition Retrieval**: The `get` method calculates the start and end indices for the requested partition. The start index is computed by multiplying the partition index by the partition size and adding the minimum of the index and remainder. The end index is adjusted if the partition is among the first remainder partitions. The result is a slice of the stored list from the calculated start to end indices.

This approach ensures the Python class behaves identically to the C++ counterpart, handling all edge cases and exceptions as specified.
