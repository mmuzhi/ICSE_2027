    public static double mrr(Object data) {
        if (!(data instanceof List) && !(data instanceof Tuple)) {
            throw new IllegalArgumentException("the input must be a tuple([0,...,1,...],int) or a iteration of list of tuple");
        }

        if (data instanceof Tuple) {
            Tuple tuple = (Tuple) data;
            List<Integer> subList = tuple.getList();
            int totalNum = tuple.getTotalNum();

            if (totalNum == 0) {
                return 0.0;
            }

            double mr = 0.0;
            for (int i = 0; i < subList.size(); i++) {
                if (subList.get(i) == 1) {
                    mr = 1.0 / (i + 1);
                    break;
                }
            }
            return mr;
        }
        else {
            List<Tuple> tupleList = (List<Tuple>) data;
            List<Double> separateResult = new ArrayList<>();

            for (Tuple tuple : tupleList) {
                List<Integer> subList = tuple.getList();
                int totalNum = tuple.getTotalNum();

                if (totalNum == 0) {
                    separateResult.add(0.0);
                } else {
                    double mr = 0.0;
                    for (int i = 0; i < subList.size(); i++) {
                        if (subList.get(i) == 1) {
                            mr = 1.0 / (i + 1);
                            break;
                        }
                    }
                    separateResult.add(mr);
                }
            }
            return separateResult.stream().mapToDouble(Double::doubleValue).average().orElse(0.0);
        }
    }

    public static double map(Object data) {
        if (!(data instanceof List) && !(data instanceof Tuple)) {
            throw new IllegalArgumentException("the input must be a tuple([0,...,1,...],int) or a iteration of list of tuple");
        }

        if (data instanceof Tuple) {
            Tuple tuple = (Tuple) data;
            List<Integer> subList = tuple.getList();
            int totalNum = tuple.getTotalNum();

            if (totalNum == 0) {
                return 0.0;
            }

            double ap = 0.0;
            int count = 0;
            for (int i = 0; i < subList.size(); i++) {
                if (subList.get(i) == 1) {
                    count++;
                    ap += count / (i + 1.0);
                }
            }
            return ap / totalNum;
        } else {
            List<Tuple> tupleList = (List<Tuple>) data;
            List<Double> separateResult = new ArrayList<>();

            for (Tuple tuple : tupleList) {
                List<Integer> subList = tuple.getList();
                int totalNum = tuple.getTotalNum();

                if (totalNum == 0) {
                    separateResult.add(0.0);
                } else {
                    double ap = 0.0;
                    int count = 0;
                    for (int i = 0; i < subList.size(); i++) {
                        if (subList.get(i) == 1) {
                            count++;
                            ap += count / (i + 1.0);
                        }
                    }
                    separateResult.add(ap / totalNum);
                }
            }
            return separateResult.stream().mapToDouble(Double::doubleValue).average().orElse(0.0);
        }
    }

    public static class Tuple {
        private List<Integer> list;
        private int totalNum;

        public Tuple(List<Integer> list, int totalNum) {
            this.list = list;
            this.totalNum = totalNum;
        }

        public List<Integer> getList() {
            return list;
        }

        public int getTotalNum() {
            return totalNum;
        }
    }

The Java code defines two static methods, `mrr` and `map`, which operate on either a single `Tuple` or a list of `Tuple` objects. The C++ solution should replicate this functionality using idiomatic C++ constructs.

### Approach
1. **Define the Tuple Class**: Create a `Tuple` class in C++ that holds a vector of integers and a total number. This mirrors the Java `Tuple` class.
2. **Handle Input Types**: Use function overloading to handle both a single `Tuple` and a vector of `Tuples`. This avoids the need for a variant type and keeps the interface similar to Java.
3. **Compute MRR and MAP**: For each `Tuple`, compute the MRR (Monthly Recurring Revenue) by finding the first occurrence of 1 in the list. For MAP (Mean Average Precision), sum the precision values at each relevant point and average them.
4. **Edge Cases**: Handle cases where the total number is zero or the list is empty by returning zero.

1. **Tuple Class**: The `Tuple` class encapsulates a list of integers and a total number, providing getters for these values.
2. **mrr Function**: The `mrr` function computes the MRR by finding the first occurrence of 1 in the list and calculating its reciprocal. For a list of tuples, it averages the MRR values of all tuples.
3. **map Function**: The `map` function calculates the average precision by summing the precision values at each relevant point and dividing by the total number of elements. For a list of tuples, it averages the MAP values of all tuples.
4. **Edge Handling**: Both functions handle cases where the total number is zero or the list is empty by returning zero, ensuring consistent behavior with the Java counterpart.
