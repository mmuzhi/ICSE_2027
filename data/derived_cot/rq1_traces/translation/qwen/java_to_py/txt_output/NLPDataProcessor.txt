package org.example;

import java.util.ArrayList;
import java.util.Arrays;
import java.util.List;

public class NLPDataProcessor {

    public List<String> constructStopWordList() {
        return Arrays.asList("a", "an", "the");
    }

    public List<List<String>> removeStopWords(List<String> stringList, List<String> stopWordList) {
        List<List<String>> result = new ArrayList<>();
        for (String string : stringList) {
            List<String> words = new ArrayList<>(Arrays.asList(string.split(" ")));
            words.removeAll(stopWordList);
            result.add(words);
        }
        return result;
    }

    public List<List<String>> process(List<String> stringList) {
        List<String> stopWordList = constructStopWordList();
        return removeStopWords(stringList, stopWordList);
    }
}

3. **String Splitting**: The Java method splits strings using `split(" ")`, which includes empty strings if present. The Python equivalent uses `split(' ')` to preserve this behavior.
4. **Stop Word Removal**: The Java method `removeStopWords` uses `removeAll` to filter out stop words. In Python, we use list comprehension to achieve the same result efficiently.
5. **Edge Cases**: The solution handles edge cases such as empty strings and multiple spaces correctly, ensuring identical behavior to the Java counterpart.

### Solution Code
```python
class NLPDataProcessor:

    def construct_stop_word_list(self):
        return ("a", "an", "the")

    def remove_stop_words(self, string_list, stop_word_list):
        result = []
        for string in string_list:
            words = string.split(' ')
            filtered_words = [word for word in words if word not in stop_word_list]
            result.append(filtered_words)
        return result

    def process(self, string_list):
        stop_word_list = self.construct_stop_word_list()
        return self.remove_stop_words(string_list, stop_word_list)
```

1. **construct_stop_word_list**: This method returns a tuple of stop words, ensuring immutability similar to Java's `Arrays.asList`.
2. **remove_stop_words**: This method processes each string in the input list by splitting it into words (preserving empty strings) and filtering out any words present in the stop word list. The result is a list of filtered words for each input string.
3. **process**: This method orchestrates the processing by first constructing the stop word list and then applying the stop word removal to the input strings.

This translation ensures that all behaviors, including edge cases and exceptions, are preserved, making the Python class function identically to its Java counterpart.
