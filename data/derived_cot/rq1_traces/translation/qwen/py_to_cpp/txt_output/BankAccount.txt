class BankAccount:
    def __init__(self, balance=0):
        self.balance = balance

    def deposit(self, amount):
        if amount < 0:
            raise ValueError("Invalid amount")
        self.balance += amount
        return self.balance

    def withdraw(self, amount):
        if amount < 0:
            raise ValueError("Invalid amount")
        if amount > self.balance:
            raise ValueError("Insufficient balance.")
        self.balance -= amount
        return self.balance

    def view_balance(self):
        return self.balance

    def transfer(self, other_account, amount):
        self.withdraw(amount)
        other_account.deposit(amount)

=== RAW ANSWER ===

To solve this problem, we need to translate a Python class representing a bank account system into an equivalent C++ class. The goal is to maintain the same behavior, including handling edge cases and exceptions, while using idiomatic C++ practices where appropriate.

### Approach
1. **Class Structure**: Define a `BankAccount` class in C++ with a private member `balance` to encapsulate the account balance.
2. **Constructors**: Implement a default constructor that initializes the balance to 0 and a parameterized constructor that allows setting the initial balance.
3. **Deposit Method**: Add funds to the account, ensuring the amount is non-negative. If invalid, throw an exception.
4. **Withdraw Method**: Remove funds from the account, checking for sufficient balance and non-negative amount. Throw exceptions for invalid operations.
5. **View Balance Method**: Return the current balance without any modifications.
6. **Transfer Method**: Transfer funds from the current account to another account by withdrawing from the current account and depositing into the other account. This method leverages the existing deposit and withdraw methods to ensure consistency.

### Solution Code
```cpp
#include <stdexcept>

class BankAccount {
private:
    int balance;

public:
    // Default constructor
    BankAccount() : balance(0) {}

    // Parameterized constructor
    explicit BankAccount(int initial_balance) : balance(initial_balance) {}

    // Deposit money into the account
    void deposit(int amount) {
        if (amount < 0) {
            throw std::invalid_argument("Invalid amount");
        }
        balance += amount;
    }

    // Withdraw money from the account
    void withdraw(int amount) {
        if (amount < 0) {
            throw std::invalid_argument("Invalid amount");
        }
        if (amount > balance) {
            throw std::out_of_range("Insufficient balance.");
        }
        balance -= amount;
    }

    // View current balance
    int view_balance() const {
        return balance;
    }

    // Transfer money to another account
    void transfer(BankAccount& other_account, int amount) {
        withdraw(amount);
        other_account.deposit(amount);
    }
};
```

### Explanation
- **Constructors**: The class includes a default constructor initializing the balance to 0 and a parameterized constructor for setting an initial balance.
- **Deposit Method**: Adds the specified amount to the balance if it is non-negative. Throws an `std::invalid_argument` exception if the amount is negative.
- **Withdraw Method**: Deducts the specified amount from the balance if there are sufficient funds and the amount is non-negative. Throws `std::invalid_argument` for negative amounts and `std::out_of_range` for insufficient funds.
- **View Balance Method**: Returns the current balance without any modifications.
- **Transfer Method**: Uses the withdraw method to deduct the amount from the current account and the deposit method to add the amount to the target account, ensuring atomicity and consistency.

This implementation adheres to C++ best practices by using appropriate exception types, encapsulation, and constructor initialization, while maintaining the exact behavior of the original Python code.
