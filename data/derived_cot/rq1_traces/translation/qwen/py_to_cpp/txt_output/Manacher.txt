class Manacher:
    def __init__(self, input_string) -> None:

        :param input_string: The input_string to be searched, str.
        """
        self.input_string = input_string

    def palindromic_length(self, center, diff, string):
        """
        Recursively calculates the length of the palindromic substring based on a given center, difference value, and input string.
        :param center: The center of the palindromic substring, int.
        :param diff: The difference between the center and the current position, int.
        :param string: The string to be searched, str.
        :return: The length of the palindromic substring, int.
        >>> manacher = Manacher('ababa')
        >>> manacher.palindromic_length(2, 1, 'a|b|a|b|a')
        2

        """

    def palindromic_string(self):
        """
        Finds the longest palindromic substring in the given string.
        :return: The longest palindromic substring, str.
        >>> manacher = Manacher('ababaxse')
        >>> manacher.palindromic_string()
        'ababa'

        """
'''

class Manacher:
    def __init__(self, input_string) -> None:
        self.input_string = input_string

    def palindromic_length(self, center, diff, string):
        if (center - diff == -1 or center + diff == len(string)
                or string[center - diff] != string[center + diff]):
            return 0
        return 1 + self.palindromic_length(center, diff + 1, string)


    def palindromic_string(self):
        max_length = 0

        new_input_string = ""
        output_string = ""

        for i in self.input_string[:len(self.input_string) - 1]:
            new_input_string += i + "|"
        new_input_string += self.input_string[-1]

        for i in range(len(new_input_string)):

            length =self.palindromic_length(i, 1, new_input_string)

            if max_length < length:
                max_length = length
                start = i

        for i in new_input_string[start - max_length:start + max_length + 1]:
            if i != "|":
                output_string += i

        return output_string

To solve this problem, we need to implement the Manacher algorithm to find the longest palindromic substring in a given string. The solution involves transforming the input string by inserting separators between characters, then using a recursive function to determine the length of palindromic substrings centered at each position in the transformed string. The longest palindrome is then extracted by removing the separators from the corresponding substring in the transformed string.

### Approach
1. **String Transformation**: The input string is transformed by inserting a separator ('|') between each character and appending the last character without a trailing separator. This transformation helps in handling both even and odd-length palindromes uniformly.
2. **Recursive Palindromic Length Calculation**: For each character in the transformed string, the algorithm checks for the longest palindromic substring centered at that character by expanding outwards (comparing characters at increasing distances from the center).
3. **Tracking Maximum Length**: During the iteration, the algorithm keeps track of the maximum length of the palindromic substring found and the corresponding center.
4. **Substring Extraction**: The longest palindromic substring is extracted from the transformed string by removing all separators, ensuring the result matches the original substring.

### Solution Code

#include <string>

class Manacher {
private:
    std::string input_string;

public:
    Manacher(const std::string& s) : input_string(s) {}

    int palindromic_length(int center, int diff, const std::string& str) {
        int left = center - diff;
        int right = center + diff;
        if (left < 0 || right >= static_cast<int>(str.length()) || str[left] != str[right]) {
            return 0;
        }
        return 1 + palindromic_length(center, diff + 1, str);
    }

    std::string palindromic_string() {
        if (input_string.empty()) {
            return "";
        }

        std::string transformed;
        for (size_t i = 0; i < input_string.length() - 1; ++i) {
            transformed += input_string[i];
            transformed += '|';
        }
        transformed += input_string.back();

        int max_length = 0;
        int best_center = 0;

        for (size_t i = 0; i < transformed.length(); ++i) {
            int length = palindromic_length(i, 1, transformed);
            if (length > max_length) {
                max_length = length;
                best_center = i;
            }
        }

        std::string result;
        for (int i = best_center - max_length; i <= best_center + max_length; ++i) {
            if (transformed[i] != '|') {
                result += transformed[i];
            }
        }

        return result;
    }
