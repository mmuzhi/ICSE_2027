class NumberConverter:
    @staticmethod
    def decimal_to_binary(decimal_num):
        binary_num = bin(decimal_num)[2:]
        return binary_num

    @staticmethod
    def binary_to_decimal(binary_num):
        decimal_num = int(binary_num, 2)
        return decimal_num

    @staticmethod
    def decimal_to_octal(decimal_num):
        octal_num = oct(decimal_num)[2:]
        return octal_num

    @staticmethod
    def octal_to_decimal(octal_num):
        decimal_num = int(octal_num, 8)
        return decimal_num

    @staticmethod
    def decimal_to_hex(decimal_num):
        hex_num = hex(decimal_num)[2:]
        return hex_num

    @staticmethod
    def hex_to_decimal(hex_num):
        decimal_num = int(hex_num, 16)
        return decimal_num

To solve this problem, we need to translate Python code for number conversion into an equivalent C++ implementation. The Python code uses static methods for converting between decimal, binary, octal, and hexadecimal formats. The goal is to replicate these functionalities in C++ while maintaining the same behavior, inputs, outputs, and edge cases.

### Approach
The approach involves manually implementing the conversion algorithms since C++ does not have built-in functions for these conversions like Python's `bin()`, `oct()`, and `hex()`. Each conversion method is implemented using loops and mathematical operations to ensure consistency and correctness:

1. **Decimal to Binary**: Repeatedly divide the number by 2 and collect remainders to form the binary string.
2. **Binary to Decimal**: Iterate over each character in the binary string, convert it to its numerical value, and sum the values multiplied by the appropriate power of 2.
3. **Decimal to Octal**: Repeatedly divide the number by 8 and collect remainders to form the octal string.
4. **Octal to Decimal**: Iterate over each character in the octal string, convert it to its numerical value, and sum the values multiplied by the appropriate power of 8.
5. **Decimal to Hexadecimal**: Repeatedly divide the number by 16 and map remainders to hexadecimal digits.
6. **Hexadecimal to Decimal**: Iterate over each character in the hexadecimal string, convert it to its numerical value, and sum the values multiplied by the appropriate power of 16.

### Solution Code

```cpp
#include <string>
#include <cmath>
#include <cctype>
#include <iostream>

class NumberConverter {
public:
    static std::string decimal_to_binary(int decimal_num) {
        if (decimal_num == 0) {
            return "0";
        }
        std::string binary = "";
        int n = decimal_num;
        while (n > 0) {
            int remainder = n % 2;
            binary = std::to_string(remainder) + binary;
            n = n / 2;
        }
        return binary;
    }

    static int binary_to_decimal(const std::string& binary_num) {
        int decimal = 0;
        int length = binary_num.length();
        for (int i = 0; i < length; i++) {
            if (binary_num[i] == '1') {
                decimal += std::pow(2, length - i - 1);
            }
        }
        return decimal;
    }

    static std::string decimal_to_octal(int decimal_num) {
        if (decimal_num == 0) {
            return "0";
        }
        std::string octal = "";
        int n = decimal_num;
        while (n > 0) {
            int remainder = n % 8;
            octal = std::to_string(remainder) + octal;
            n = n / 8;
        }
        return octal;
    }

    static int octal_to_decimal(const std::string& octal_num) {
        int decimal = 0;
        int length = octal_num.length();
        for (int i = 0; i < length; i++) {
            if (octal_num[i] >= '0' && octal_num[i] <= '7') {
                decimal += (octal_num[i] - '0') * std::pow(8, length - i - 1);
            }
        }
        return decimal;
    }

    static std::string decimal_to_hex(int decimal_num) {
        if (decimal_num == 0) {
            return "0";
        }
        std::string hex = "";
        int n = decimal_num;
        while (n > 0) {
            int remainder = n % 16;
            if (remainder < 10) {
                hex = std::to_string(remainder) + hex;
            } else {
                char hex_digit = 'a' + (remainder - 10);
                hex = std::string(1, hex_digit) + hex;
            }
            n = n / 16;
        }
        return hex;
    }

    static int hex_to_decimal(const std::string& hex_num) {
        int decimal = 0;
        int length = hex_num.length();
        for (int i = 0; i < length; i++) {
            char c = hex_num[i];
            int value;
            if (c >= '0' && c <= '9') {
                value = c - '0';
            } else if (c >= 'a' && c <= 'f') {
                value = c - 'a' + 10;
            } else if (c >= 'A' && c <= 'F') {
                value = c - 'A' + 10;
            } else {
                // Handle invalid characters if necessary
                continue;
            }
            decimal += value * std::pow(16, length - i - 1);
        }
        return decimal;
    }
};
```

- **Binary to Decimal**: Each character in the binary string is converted to its numerical value, and the values are summed with appropriate powers of 2.
- **Decimal to Octal**: Similar to binary conversion, but division is by 8, and remainders are collected.
- **Octal to Decimal**: Each character is converted to its numerical value, and the values are summed with appropriate powers of 8.
- **Decimal to Hexadecimal**: Division by 16 is used, with remainders mapped to hexadecimal digits (0-9, a-f).
- **Hexadecimal to Decimal**: Each character is converted to its numerical value, handling both lowercase and uppercase letters, and summed with appropriate powers of 16.
